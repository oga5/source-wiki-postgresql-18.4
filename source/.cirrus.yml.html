.cirrus.yml
1# CI configuration file for CI utilizing cirrus-ci.org
2#
3# For instructions on how to enable the CI integration in a repository and
4# further details, see src/tools/ci/README
5#
6#
7# The actual CI tasks are defined in .cirrus.tasks.yml. To make the compute
8# resources for CI configurable on a repository level, the "final" CI
9# configuration is the combination of:
10#
11# 1) the contents of this file
12#
13# 2) computed environment variables
14#
15#    Used to enable/disable tasks based on the execution environment. See
16#    .cirrus.star: compute_environment_vars()
17#
18# 3) if defined, the contents of the file referenced by the, repository
19#    level, REPO_CI_CONFIG_GIT_URL variable (see
20#    https://cirrus-ci.org/guide/programming-tasks/#fs for the accepted
21#    format)
22#
23#    This allows running tasks in a different execution environment than the
24#    default, e.g. to have sufficient resources for cfbot.
25#
26# 4) .cirrus.tasks.yml
27#
28# This composition is done by .cirrus.star
29
