3pg_amcheck is a command-line tool for running the amcheck extension.
4
5Running the regression tests
6============================
7
8NOTE: You must have given the --enable-tap-tests argument to configure.
9Also, to use "make installcheck", you must have built and installed
10contrib/amcheck and contrib/pageinspect in addition to the core code.
11
12Run
13    make check
14or
15    make installcheck
16You can use "make installcheck" if you previously did "make install".
17In that case, the code in the installation tree is tested.  With
18"make check", a temporary installation tree is built from the current
19sources and then tested.
20
21See src/test/perl/README for more info about running these tests.
