1Test programs and libraries for OAuth
2-------------------------------------
3
4This folder contains tests for the client- and server-side OAuth
5implementations. Most tests are run end-to-end to test both simultaneously. The
6tests in t/001_server use a mock OAuth authorization server, implemented jointly
7by t/OAuth/Server.pm and t/oauth_server.py, to run the libpq Device
8Authorization flow. The tests in t/002_client exercise custom OAuth flows and
9don't need an authorization server.
10
11Tests in this folder require 'oauth' to be present in PG_TEST_EXTRA, since
12HTTPS servers listening on localhost with TCP/IP sockets will be started. A
13Python installation is required to run the mock authorization server.
