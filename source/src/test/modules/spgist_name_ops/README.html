src/test/modules/spgist_name_ops/README
1spgist_name_ops implements an SP-GiST operator class that indexes
2columns of type "name", but with storage identical to that used
3by SP-GiST text_ops.
4
5This is not terribly useful in itself, perhaps, but it allows
6testing cases where the indexed data type is different from the leaf
7data type and yet we can reconstruct the original indexed value.
8That situation is not tested by any built-in SP-GiST opclass.
