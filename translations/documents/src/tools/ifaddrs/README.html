test_ifaddrs
============

このプログラムは、ローカルマシン上のすべてのIPv4およびIPv6インターフェースのアドレスとネットマスクを表示します。これは、この機能がさまざまなプラットフォームで動作することをテストするのに役立ちます。pg_hba.confの "samehost" や "samenet" が正しく動作していないと思われる場合は、このプログラムを実行して何が起きているかを確認してください。

使用法: test_ifaddrs
