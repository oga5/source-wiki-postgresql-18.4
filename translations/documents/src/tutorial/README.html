tutorial
========

このディレクトリにはSQLチュートリアルのスクリプトが含まれています。これらを確認するには、まず
	% make
を実行して、ユーザー定義関数および型のためのすべてのスクリプトとCファイルをコンパイルしてください。（makeにはGNU makeが必要です。お使いのシステムでは別の名前、例えば 'gmake' となっている場合があります）

次に、-s (シングルステップ) フラグを付けて psql を実行します。
	% psql -s

psql 内では、psql の \i <filename> コマンドを使用して、各スクリプトファイルを個別に試すことができます。
