src/pl/plperl/README

PL/Perl を使用すると、PostgreSQL の関数やプロシージャを Perl で記述できます。PL/Perl をビルドに含めるには、'./configure --with-perl' を使用してください。このディレクトリからビルドするには、'make all; make install' を使用します。libperl は共有ライブラリとしてビルドされている必要がありますが、標準的なインストールでは通常そうなっていません。

詳細については、『PostgreSQL ユーザーズガイド』を参照してください。
