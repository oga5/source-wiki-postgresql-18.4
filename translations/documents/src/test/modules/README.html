テスト用拡張機能およびライブラリ
=============================

src/test/modules には、主に、あるいは完全に PostgreSQL のテストを目的とした、またはサンプルコードとして機能することを目的とした PostgreSQL 拡張機能が含まれています。ここにある拡張機能は、本番環境のサーバーにインストールすることを意図しておらず、「実務」には適していません。

さらに、このディレクトリやそのサブディレクトリで "make install" や "make installcheck" を実行することは可能ですが、重要なデータを含むサーバーに対してこれを行うことは推奨されません。これらのテストの一部は、テスト対象のサーバー内のロールやその他のグローバルオブジェクトに対して、望ましくない副作用を及ぼす可能性があります。トップレベルでの "make installcheck-world" は、このディレクトリには再帰的に適用されません。

ほとんどの拡張機能には、独自の pg_regress テストまたは isolationtester 仕様が含まれています。一部は、ツリー内の他の場所にあるテストでも使用されます。

新しいフックや C レベルの API として公開されるその他の機能を追加する場合は、ここにテストを追加してください。
