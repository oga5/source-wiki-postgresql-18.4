src/test/modules/test_rls_hooks/README

test_rls_hooks は、RLS 用に提供されたフックを使用して、使用する追加のポリシーを定義する方法の例です。

関数
=========
test_rls_hooks_permissive(CmdType cmdtype, Relation relation)
    RETURNS List*

リレーション上の既存のポリシーに追加され、OR で結合されるべきポリシーのリストを返します。

test_rls_hooks_restrictive(CmdType cmdtype, Relation relation)
    RETURNS List*

リレーション上の既存のポリシーに追加され、AND で結合されるべきポリシーのリストを返します。
