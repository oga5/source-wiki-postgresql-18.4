モジュール `test_json_parser`
=========================

このモジュールには、jsonパーサーをテストするための2つのプログラムが含まれています。

- `test_json_parser_incremental` は、インクリメンタルパーサーをテストするためのものです。ファイルを読み込み、非常に小さなチャンク（デフォルトは一度に60バイト）に分割してインクリメンタルパーサーに渡します。これは速度テストを目的としたものではなく、インクリメンタルパーサーの正確性をテストするためのものです。オプション "-c nn" は別のチャンクサイズを指定し、"-r nn" は同じ入力に対して1バイトまでのチャンクサイズの範囲を実行します（出力はnullバイトで区切られます）。"-s" はセマンティックルーチンを使用することを指定します。セマンティックルーチンはjsonを再出力しますが、整形された形式ではありません。必須の非オプション引数は入力ファイル名です。
- `test_json_parser_perf` は、標準の再帰下降パーサーと非再帰的なインクリメンタルパーサーの両方の速度をテストするためのものです。`-i` フラグが指定された場合は非再帰的パーサーを使用し、それ以外の場合は標準パーサーを使用します。残りのフラグは、解析の繰り返し回数と入力を含むファイルです。非再帰的パーサーを使用する場合でも、入力は単一のチャンクとしてパーサーに渡されます。したがって、結果は標準パーサーの結果と比較可能です。

サンプル入力ファイルは、数年前に取得された `delicious` ブックマークのリストから抽出された、小さなサニタイズ済みの抜粋であり、すべて単一のjson配列でラップされています。
