src/test/modules/test_rbtree/README

test_rbtree は、赤黒木の操作の正確性を確認するためのテストモジュールです。

これらのテストは、整数を格納する赤黒木に対して実行されます。
rbtree のロジックは比較関数をブラックボックスとして扱うため、キーの型が正確に何であるかは重要ではありません。

走査の正確性の確認は、赤黒木が二分探索木であるという事実に基づいています。したがって、要素は昇順（左-現在-右の場合）または降順（右-現在-左の場合）でアクセスされる必要があります。

また、このモジュールでは、find、delete、および leftmost 操作の正確性についてもいくつかチェックを行います。
