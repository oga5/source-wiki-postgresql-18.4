OAuth 用のテストプログラムおよびライブラリ
-------------------------------------

このフォルダーには、クライアント側およびサーバー側の OAuth 実装のテストが含まれています。ほとんどのテストは、両方を同時にテストするためにエンドツーエンドで実行されます。t/001_server 内のテストでは、t/OAuth/Server.pm と t/oauth_server.py によって共同で実装されたモック OAuth 認可サーバーを使用して、libpq Device Authorization フローを実行します。t/002_client 内のテストはカスタム OAuth フローを検証するものであり、認可サーバーを必要としません。

このフォルダー内のテストを実行するには、localhost 上で TCP/IP ソケットをリッスンする HTTPS サーバーが起動されるため、PG_TEST_EXTRA に 'oauth' が含まれている必要があります。モック認可サーバーを実行するには Python のインストールが必要です。
