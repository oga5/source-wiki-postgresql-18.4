認証の回帰テスト
===================================

このディレクトリには、認証用のテストスイートが含まれています。SSL証明書認証のテストは、より複雑であり、マルチユーザーシステムで実行するのが安全ではないため、src/test/ssl/ に分けて配置されています。


テストの実行
=================

注意: configure に --enable-tap-tests 引数を指定しておく必要があります。

実行するには
    make check
または
    make installcheck
を実行してください。以前に "make install" を実行した場合は "make installcheck" を使用できます。その場合、インストールツリー内のコードがテストされます。"make check" を使用した場合は、現在のソースから一時的なインストールツリーが構築され、それがテストされます。

いずれの場合も、このテストではテスト用の Postgres クラスタの初期化、起動、停止が行われます。

これらのテストの実行に関する詳細については、src/test/perl/README を参照してください。
