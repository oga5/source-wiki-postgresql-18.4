postmaster の回帰テスト
=======================

このディレクトリには、postmaster の接続処理、接続制限、および起動/停止シーケンスに関するテストスイートが含まれています。


テストの実行
============

注意: configure 実行時に --enable-tap-tests 引数を指定している必要があります。

実行方法:
    make check
または
    make installcheck
以前に "make install" を実行している場合は "make installcheck" を使用できます。
その場合、インストールツリー内のコードがテストされます。"make check" を使用した場合は、現在のソースから一時的なインストールツリーが構築され、それがテストされます。

いずれの場合も、これらのテストはテスト用の Postgres クラスタを初期化、起動、および停止します。

これらのテストの実行に関する詳細については、src/test/perl/README を参照してください。
