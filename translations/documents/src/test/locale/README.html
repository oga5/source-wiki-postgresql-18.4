ロケール
=======

このディレクトリには、ロケールに関する一連のテストが含まれています。libcのCTYPEサポートとインストールされたロケールデータをテストするために、Cプログラム test-ctype.c を用意しました。また、照合順序をテストするための test-sort.pl と test-sort.py もあります。

特定のロケールのテストを実行するには、
    make check-$locale
を実行してください。例：
    make check-koi8-r

現在、利用可能なテストは少数のロケールのみです。スクリプト 'runall' は、libcとロケールデータをテストするために test-ctype を呼び出し、test-sort.pl を実行します（Pythonインタプリタがインストールされている場合は test-sort.py のコメントを外してください）。また、提供されているSQLスクリプトファイルを使用してPostgreSQLのテストも行います。

ロケールのテストを追加するには、$locale ディレクトリを作成し、koi8-r/* と同様の Makefile（およびその他のファイル）を作成する必要があります。実際、（個人的な考えですが）最も簡単な方法は、koi8-r ディレクトリをコピーしてファイルを編集/置換することです。
