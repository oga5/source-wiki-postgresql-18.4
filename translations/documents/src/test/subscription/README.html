サブスクリプション/論理レプリケーションの回帰テスト
=====================================================

このディレクトリには、サブスクリプション/論理レプリケーションのテストスイートが含まれています。

テストの実行
=================

注意: configure に --enable-tap-tests 引数を指定している必要があります。
また、"make installcheck" を使用するには、コアコードに加えて contrib/hstore をビルドおよびインストールしておく必要があります。

実行方法
    make check
または
    make installcheck
以前に "make install" を実行している場合は "make installcheck" を使用できます。
その場合、インストールツリー内のコードがテストされます。"make check" の場合は、現在のソースから一時的なインストールツリーが構築され、それがテストされます。

いずれの場合も、このテストでは複数のテスト用 Postgres クラスタの初期化、起動、停止が行われます。

これらのテストの実行に関する詳細については、src/test/perl/README を参照してください。
