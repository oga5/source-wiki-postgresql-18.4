ICU 機能の回帰テスト
======================================

このディレクトリには、ICU 機能のテストスイートが含まれています。

テストの実行
=================

注意: configure に --enable-tap-tests 引数を指定しておく必要があります。

実行するには
    make check
または
    make installcheck
を実行してください。以前に "make install" を実行している場合は "make installcheck" を使用できます。
その場合、インストールツリー内のコードがテストされます。"make check" では、現在のソースから一時的なインストールツリーが構築され、それがテストされます。

いずれの場合も、このテストでは複数のテスト用 Postgres クラスタの初期化、起動、停止が行われます。

これらのテストの実行に関する詳細については、src/test/perl/README を参照してください。
