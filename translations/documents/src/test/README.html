PostgreSQL テスト
================

このディレクトリには、さまざまなテストインフラストラクチャと、PostgreSQL のテストの一部が含まれています。すべてのテストがここにあるわけではありません。特に、個別の contrib/ モジュールや src/bin にもさらに多くのテストが存在します。

これらのテストすべてが "make check" で実行されるわけではありません。どのテストが自動的に実行されるかについては、src/test/Makefile を確認してください。

authentication/
  認証用テスト（下記も参照）

examples/
  "make check" を通じて回帰テストも兼ねる libpq 用のデモンストレーションプログラム

isolation/
  SQL レベルでの並行動作のテスト

kerberos/
  Kerberos/GSSAPI 認証および暗号化のテスト

ldap/
  LDAP ベースの認証のテスト

locale/
  ロケールデータ、エンコーディングなどの健全性チェック

mb/
  マルチバイトエンコーディング（UTF-8）サポートのテスト

modules/
  テスト目的のみ、または主にテスト目的で使用される拡張機能。通常、本番データベースへのインストールには適していません

perl/
  Perl ベースの TAP テスト用インフラストラクチャ

recovery/
  リカバリおよびレプリケーション用のテストスイート

regress/
  PostgreSQL のメイン回帰テストスイート、pg_regress

ssl/
  SSL 証明書の取り扱いを検証および確認するためのテスト

subscription/
  論理レプリケーションのテスト
