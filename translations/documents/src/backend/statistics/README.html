拡張統計情報
===================

様々な量（例：条件の選択率）を推定する際、デフォルトのアプローチは独立性の仮定に基づいています。実際にはこれが当てはまらないことが多く、推定誤差が生じる原因となります。

拡張統計情報は、列間の様々な種類の依存関係を追跡し、推定を改善してより良いプランを作成することを目的としています。


統計情報の種類
-------------------

現在、いくつかの種類の拡張統計情報が存在します。

    (a) ndistinct 係数

    (b) ソフト関数従属性 (README.dependencies)

    (c) MCV リスト (README.mcv)


互換性のある句の型
-----------------------

各統計情報の種類は、特定のサブセットの句の型を推定するために使用できます。

    (a) 関数従属性 - 等価句 (AND)、場合により IS NULL

    (b) MCV リスト - 等価句および不等価句 (AND, OR, NOT)、IS [NOT] NULL

現在、Var op Const または Const op Var の形式の OpExpr のみがサポートされていますが、将来的に Var op Var のような句の選択率も推定できるようにコードを拡張することは可能です。


複雑な句
----------------

より複雑な句の推定もサポートしています。基本的には、参照されるすべての属性が単一の統計情報オブジェクトによってカバーされている限り、(Var op Const) を末端とする AND/OR 句を扱えます。

例えば、以下の条件は

    (a=1) AND ((b=2) OR ((c=3) AND (d=4)))

(a,b,c,d) に関する統計情報を使用して推定できます。(b,c,d) に関する統計情報しかない場合は、後半部分を推定し、(a=1) は単純な統計情報を使用して推定します。

(a,b,c) に関する統計情報しかない場合、現時点では適用できませんが、clauselist_selectivity() は再帰的に動作するため、後半部分（OR 句）を処理する際には統計情報を適用できることに留意してください。

注: マルチ統計情報推定パッチにより、式ツリーのより深い部分に一部の句を「条件」として渡すことも可能になります。


選択率の推定
----------------------

プランナ全体において、ほとんどの選択率推定要求は引き続き clauselist_selectivity() が担当します。clauselist_selectivity() は、以下の条件を満たす場合に、指定された RelOptInfo 上の拡張統計情報を利用するように指示されます。

    (a) 実際に有効な RelOptInfo が与えられていること。結合リレーションは NULL として渡されるため、無効です。

    (b) 与えられたリレーションに、実際に構築済みの拡張統計情報が定義されていること。

上記の条件が満たされると、clauselist_selectivity() はまず、句リストを拡張統計情報の選択率推定関数に渡そうと試みます。この関数は、推定を行える句を見つけられない場合があります。その場合、これらの句は単に無視されます。これらの関数で実際に推定作業が行われる場合、他の推定関数がどの句をスキップすべきかを知ることができるよう、推定を行った句にマークを付けることが求められます。


ANALYZE におけるサンプルサイズ
-------------------------

ANALYZE を実行する際、サンプリングする行数は以下のように決定されます。

    (300 * statistics_target)

これは個々の列に対する統計情報としては十分に機能しますが、拡張統計情報には不十分かもしれません。推定誤差を分析した論文では、いずれもテーブルに比例したサンプルサイズを使用しています（通常、テーブルの 1〜3% で正確な統計情報を構築するのに十分であるとされています）。

要求される精度（MCV 項目の数やヒストグラムのビン数）もサンプルサイズを決定する際に考慮されるべきですが、拡張統計情報においてこれらは必ずしも statistics_target によって制限されるわけではありません。

しかし、サンプルの収集は非常にコストが高く、さらに増やすと ANALYZE がより困難になるため、これについてはさらなる議論が必要です。現在の実装での実験から判断すると、固定サイズで今のところは十分に機能しているように見えるため、これは将来の課題として残しておきます。
