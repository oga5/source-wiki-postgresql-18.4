エンコーディング
=========

conv.c:		コード変換のための静的関数および公開テーブル
mbutils.c:	バックエンド専用の公開関数
stringinfo_mb.c: バックエンド専用のマルチバイト対応stringinfo関数
wstrcmp.c:	mb用strcmp
wstrncmp.c:	mb用strncmp

src/common/ も参照のこと:

encnames.c:	エンコーディング名のための公開関数
wchar.c:	主にmb文字列およびマルチバイト変換のための静的関数と公開テーブル
