pgevent
=======

MSG00001.bin は Microsoft MC コンパイラによって生成されたバイナリファイルです。MC コンパイラは MS Core SDK に無料で同梱されていますが、MSYS ツールには含まれておらず、MC ファイルをコンパイルする代替手段も見つかりませんでした。

要約すると、コマンド "MC pgmsgevent.mc" は pgmsgevent.h、pgmsgevent.rc、および MSG00001.bin ファイルを生成します。MC ファイル内で %s 形式の文字列を宣言しているため、将来的にこの文字列の定義を変更することなく、任意の文字列を書き込むことができます。

最後に、DllUnregisterServer と DllRegisterServer はシステム定義のエントリポイントであるため、これら 2 つの関数を「装飾（decoration）」なしの名前でエクスポートする必要があります。そのため、手作業で修正を行わずに自動生成された .def ファイルを使用することはできません。
