src/bin/pg_amcheck/README

pg_amcheck は、amcheck 拡張を実行するためのコマンドラインツールです。

リグレッションテストの実行
==========================

注意: configure に --enable-tap-tests 引数を指定しておく必要があります。
また、"make installcheck" を使用するには、コアコードに加えて contrib/amcheck および contrib/pageinspect をビルドしてインストールしておく必要があります。

実行するには
    make check
または
    make installcheck
を実行してください。以前に "make install" を実行済みであれば "make installcheck" を使用できます。その場合、インストールツリー内のコードがテストされます。"make check" の場合は、現在のソースから一時的なインストールツリーが構築され、それがテストされます。

これらのテストの実行に関する詳細については、src/test/perl/README を参照してください。
