# 重要な meson ターゲットの説明。'help' ターゲットおよび installation.sgml (generate-targets-meson.pl 経由) で使用されます。
# 現時点では、パーサーは非常に単純です。どちらのパーサーもコメントを無視します。
# help ターゲットはそれ以外のすべてを出力します。
# XML の場合、先頭に改行がない行はグループと見なされ、残りの行は空白で区切られたターゲットとその説明となります。
#
コードターゲット:
  all                           ドキュメント以外のすべてをビルドする
  backend                       バックエンドおよび関連モジュールをビルドする
  bin                           フロントエンドバイナリをビルドする
  contrib                       contrib モジュールをビルドする
  pl                            手続き型言語をビルドする

開発者ターゲット:
  reformat-dat-files            カタログデータファイルを標準フォーマットに書き直す
  expand-dat-files              すべてのデータファイルを展開してデフォルト値を含める
  update-unicode                Unicode データを新しいバージョンに更新する

ドキュメントターゲット:
  html                          マルチページ HTML 形式でドキュメントをビルドする
  man                           man ページ形式でドキュメントをビルドする
  docs                          マルチページ HTML および man ページ形式でドキュメントをビルドする
  doc/src/sgml/postgres-A4.pdf  A4 ページサイズの PDF 形式でドキュメントをビルドする
  doc/src/sgml/postgres-US.pdf  US レターページサイズの PDF 形式でドキュメントをビルドする
  doc/src/sgml/postgres.html    シングルページ HTML 形式でドキュメントをビルドする
  alldocs                       サポートされているすべての形式でドキュメントをビルドする

インストールターゲット:
  install                       ドキュメントを除き、Postgres をインストールする
  install-docs                  マルチページ HTML および man ページ形式でドキュメントをインストールする
  install-html                  マルチページ HTML 形式でドキュメントをインストールする
  install-man                   man ページ形式でドキュメントをインストールする
  install-quiet                 "install" と同様だが、インストールされたファイルは表示されない
  install-world                 マルチページ HTML および man ページのドキュメントを含め、Postgres をインストールする
  uninstall                     インストールされたファイルを削除する

その他のターゲット:
  clean                         すべてのビルド成果物を削除する
  test                          有効なすべてのテストを実行する (contrib を含む)
  world                         ドキュメントを含むすべてをビルドする
  help                          重要なターゲットを一覧表示する
