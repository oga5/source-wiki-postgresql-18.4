PostgreSQL データベース管理システム =====================================

このディレクトリには、PostgreSQL データベース管理システムのソースコード配布物が含まれています。

PostgreSQL は、SQL 標準の拡張サブセットをサポートする高度なオブジェクト関係データベース管理システムであり、トランザクション、外部キー、サブクエリ、トリガー、ユーザー定義型、関数などを備えています。この配布物には C 言語バインディングも含まれています。

著作権およびライセンス情報については、COPYRIGHT ファイルを参照してください。

このバージョンの PostgreSQL に関する一般的なドキュメントは、<https://www.postgresql.org/docs/18/> にあります。特に、ソースコードから PostgreSQL をビルドする方法については、<https://www.postgresql.org/docs/18/installation.html> を参照してください。

本ソフトウェアおよび関連ソフトウェアの最新バージョンは、<https://www.postgresql.org/download/> から入手できます。詳細については、当社のウェブサイト <https://www.postgresql.org/> をご覧ください。
