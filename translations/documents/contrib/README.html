PostgreSQL contrib ツリー
---------------------------

このサブツリーには、移植ツール、分析ユーティリティ、およびプラグイン機能が含まれています。これらは、対象ユーザーが限定的であるか、実験的すぎるという理由から、PostgreSQLのコアシステムの一部には含まれていません。ただし、これらが有用でないという意味ではありません。

各モジュールのユーザー向けドキュメントは、メインのSGMLドキュメント内に記載されています。

ソース配布物からビルドする場合、"world"ターゲットをビルドしない限り、これらのモジュールは自動的にはビルドされません。このディレクトリで "make all" および "make install" を実行することで、すべてをビルドおよびインストールすることも可能です。また、特定のモジュールのみをビルドおよびインストールしたい場合は、そのモジュールのサブディレクトリで同様の操作を行ってください。

一部のディレクトリでは、新しいユーザー定義関数、演算子、または型が提供されています。これらのモジュールを利用するには、コードをインストールした後、CREATE EXTENSIONコマンドを実行して、データベースシステムに新しいSQLオブジェクトを登録する必要があります。新しいデータベースでは、単に以下を実行するだけです。

    CREATE EXTENSION module_name;

この手順の詳細については、PostgreSQLのドキュメントを参照してください。
