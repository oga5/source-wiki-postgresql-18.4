test_empty

概念

test_empty は、PostgreSQL のテストモジュールにおいて、対象となるデータ構造（IntegerSet または適応型基数木 rt_radix_tree）が「空」の状態であるときの基本的な振る舞いを検証するためのテスト関数です。

なぜ重要か

データ構造の初期化直後や要素が何も登録されていない状態において、検索、削除、イテレーションなどの基本操作が安全かつ正しく機能すること（クラッシュやメモリリーク、不正な値の返却が発生しないこと）を保証する最小限のテストとして機能するためです。

関与するモジュール

  • module:src/test/modules/test_integerset
  • module:src/test/modules/test_radixtree

主要エンティティ

  • symbol:src/test/modules/test_integerset/test_integerset.c#test_empty: IntegerSet が空の状態におけるメンバ判定（intset_is_member）およびイテレーション（intset_begin_iterate, intset_iterate_next）を検証する関数。
  • symbol:src/test/modules/test_radixtree/test_radixtree.c#test_empty: 適応型基数木（ART）が空の状態における検索（rt_find）、削除（rt_delete）、エントリ数確認（rt_num_entries）、およびイテレーション（rt_begin_iterate, rt_iterate_next）を検証する関数。

エンドツーエンドの流れ

これら2つのテスト関数はそれぞれ独立したテストモジュール内で個別に実行され、モジュール間を跨ぐエンドツーエンドの相互作用や制御フローは存在しません。各モジュール内での実行フローは以下の通りです。

1. IntegerSet の検証 (test_integerset.c)

1. intset_create() により空の IntegerSet を生成します。 2. 0、1、および PG_UINT64_MAX に対する intset_is_member の呼び出しがすべて false を返すことを検証します。 3. intset_begin_iterate でイテレータを初期化し、intset_iterate_next が直ちに false を返すことを検証します。

2. 適応型基数木（ART）の検証 (test_radixtree.c)

1. rt_create により空の基数木を生成します（TEST_SHARED_RT マクロの定義に応じて、共有メモリまたはローカルメモリ上に生成）。 2. 任意のキー（0、1、PG_UINT64_MAX）に対する rt_find が NULL を返し、rt_delete が false を返し、rt_num_entries が 0 であることを検証します。 3. rt_begin_iterate でイテレータを作成し、rt_iterate_next が直ちに NULL を返すことを検証します。 4. rt_end_iterate でイテレータを破棄し、rt_free で基数木を解放します。

重要な不変条件

  • **空集合のメンバシップ**: 空の IntegerSet に対する intset_is_member は、いかなる値に対しても常に false を返さなければなりません。
  • **空の基数木の検索・削除**: 空の基数木に対する rt_find は常に NULL を返し、rt_delete は常に false を返し、rt_num_entries は常に 0 でなければなりません。
  • **空状態のイテレーション**: 空のデータ構造に対してイテレータを初期化した直後の最初の next 取得処理（intset_iterate_next または rt_iterate_next）は、値を返さずに終了（false または NULL を返却）しなければなりません。

よくある落とし穴

  • **メモリ管理モードの差異**: test_radixtree.c の test_empty は、TEST_SHARED_RT マクロの定義有無によって共有メモリ（DSA）を使用するかローカルメモリを使用するかが切り替わります。修正時には両方のメモリ管理パスが正しく動作することを確認する必要があります。
  • **検証失敗時のアボート**: いずれのテストも、期待される不変条件が満たされない場合は elog(ERROR) または EXPECT_* マクロ経由でエラーを発生させ、トランザクションを即座にアボートします。
