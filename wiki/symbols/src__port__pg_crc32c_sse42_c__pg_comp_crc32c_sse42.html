pg_comp_crc32c_sse42

定義

  • 種別: function
  • 定義箇所: src/port/pg_crc32c_sse42.c:26

目的

Intel SSE 4.2 命令セットを利用して、データバッファの CRC-32C チェックサムを高速に計算する。

シグネチャまたは形状

pg_crc32c pg_comp_crc32c_sse42(pg_crc32c crc, const void *data, size_t len)

入力と出力

  • **入力**:
  • crc: 初期 CRC 値。
  • data: 計算対象のデータバッファへのポインタ。
  • len: データバッファのバイト長。
  • **出力**:
  • 計算された CRC-32C 値を返す。

前提条件と不変条件

  • 実行環境の CPU が SSE 4.2 命令セットをサポートしている必要がある。
  • 非対応 CPU で呼び出した場合、不正命令例外が発生する。
  • 入力データのアライメントは不問（Intel アーキテクチャの特性上、アライメントによる性能向上は確認されていない）。

アルゴリズムまたは振る舞い

1. **64bit アーキテクチャの場合**:

  • 8 バイト単位で _mm_crc32_u64 を使用して計算。
  • 残余が 4 バイト以上ある場合、_mm_crc32_u32 を使用して計算。

    • 2. **32bit アーキテクチャの場合**:

  • 4 バイト単位で _mm_crc32_u32 を使用して計算。

    • 3. **共通処理**:

  • 最後に残った 1〜3 バイトを _mm_crc32_u8 を用いて 1 バイトずつ処理し、最終的な CRC 値を算出する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/port/pg_crc32c_sse42_choose.c: CPU 機能フラグに基づき、本関数を pg_comp_crc32c 関数ポインタに割り当てる。
  • **呼び出し先**:
  • _mm_crc32_u64 (SSE 4.2)
  • _mm_crc32_u32 (SSE 4.2)
  • _mm_crc32_u8 (SSE 4.2)

関連データ構造

  • pg_crc32c: チェックサム値を保持する型（port/pg_crc32c.h で定義）。

失敗モードと境界条件

  • CPU が SSE 4.2 をサポートしていない環境での実行は致命的なエラーとなる。
  • len が 0 の場合、初期値 crc をそのまま返す。

保守担当者向けメモ

  • 本関数は直接呼び出さず、必ず src/port/pg_crc32c_sse42_choose.c 等のラッパーを介して、実行時の CPU 機能判定を経てから使用すること。
  • 32bit 環境では 8 バイト命令が利用できないため、条件コンパイル (__x86_64__) による分岐が維持されている。
