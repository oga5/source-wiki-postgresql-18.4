r_i_verb_suffix

定義

  • **種別**: function
  • **定義箇所**: src/backend/snowball/libstemmer/stem_ISO_8859_1_french.c:995

目的

フランス語のステミング処理において、特定の動詞接尾辞を単語末尾から検索し、条件を満たす場合に削除する。

シグネチャまたは形状

static int r_i_verb_suffix(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（Snowballランタイム環境構造体）
  • **出力**: int 型。接尾辞の削除に成功した場合は 1、失敗または条件不適合の場合は 0、エラー発生時は負の値を返す。

前提条件と不変条件

  • z->I[2]（領域境界）より現在のカーソル位置 z->c が後ろにあること。
  • 処理対象の単語末尾が特定の文字グループ（a_5 テーブルに関連）に該当すること。

アルゴリズムまたは振る舞い

1. **領域チェック**: 現在のカーソル位置が z->I[2] より前であれば終了する。 2. **検索**: find_among_b を使用し、a_5 テーブルに基づいて単語末尾から接尾辞を検索する。 3. **条件判定**:

  • マッチした接尾辞の直前に 'H' が存在する場合、処理を中断する。
  • g_v（母音グループ）に該当する文字が直前に存在することを確認する。

    • 4. **削除**: 条件を満たせば slice_del を呼び出し、該当する接尾辞を削除する。 5. **状態復元**: 処理の前後で z->lb を適切に管理し、環境を維持する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_ISO_8859_1_french.c:1211 および src/backend/snowball/libstemmer/stem_UTF_8_french.c:1222 からの呼び出しが確認されている。
  • **呼び出し先**: find_among_b, out_grouping_b, slice_del（Snowballランタイム関数）。

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_5: 検索対象となる接尾辞パターンを格納した among 構造体配列。
  • g_v: 母音判定用のグループ定義。

失敗モードと境界条件

  • カーソル位置が領域境界より前にある場合、即座に 0 を返す。
  • find_among_b がマッチしない場合、0 を返す。
  • slice_del が内部エラー（負の値）を返した場合、その値をそのまま伝播させる。
