unicodeStyleBorderFormat

定義

  • 種別: struct
  • 定義箇所: src/fe_utils/print.c:116

目的

Unicode罫線を用いた表形式出力において、表の枠線や交差点を構成する文字セットを定義するために使用されます。

シグネチャまたは形状

typedef struct unicodeStyleBorderFormat
{
	const char *up_and_right;
	const char *vertical;
	const char *down_and_right;
	const char *horizontal;
	const char *down_and_left;
	const char *left_and_right;
} unicodeStyleBorderFormat;

入力と出力

本構造体はデータ構造であり、関数ではありません。各フィールドは、表の特定の部位（垂直線、水平線、およびそれらの交差点）に対応する文字列（const char *）を保持します。

前提条件と不変条件

  • 各フィールドは、Unicode罫線文字を表す有効な文字列である必要があります。
  • print.c 内の表出力ロジックにおいて、この構造体で定義された文字を使用して表の枠組みが描画されます。

アルゴリズムまたは振る舞い

本構造体自体はデータ保持のみを行います。実際の描画処理では、この構造体のインスタンスが参照され、表の各行・列の境界線を描画する際に各フィールドの文字が選択的に出力されます。

呼び出し元と呼び出し先

  • 参照元: src/fe_utils/print.c:130 （border_style 配列の型として定義）
  • 参照元: src/fe_utils/print.c:3695 （border ポインタ変数として使用）
  • 呼び出し先: 情報なし。

関連データ構造

  • unicodeStyleFormat

失敗モードと境界条件

  • 構造体の各フィールドに NULL や不正なエンコーディングの文字列が設定された場合、出力処理時に予期せぬ表示崩れやクラッシュが発生する可能性があります。

保守担当者向けメモ

  • 本構造体は psql 等のフロントエンドにおける Unicode 表出力のスタイルを決定します。新しい罫線スタイルを追加する場合は、この構造体のインスタンスを定義し、出力ロジック側で適切に選択されるようにしてください。
