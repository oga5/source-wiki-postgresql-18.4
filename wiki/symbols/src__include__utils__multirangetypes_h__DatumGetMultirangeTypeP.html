DatumGetMultirangeTypeP

定義

  • 種別: static inline 関数
  • 定義箇所: src/include/utils/multirangetypes.h:47

目的

Datum 型として保持されているマルチレンジ型の値を、メモリ上の MultirangeType 構造体へのポインタとして取得する。

シグネチャまたは形状

static inline MultirangeType *DatumGetMultirangeTypeP(Datum X)

入力と出力

  • **入力**: Datum X（マルチレンジ型を保持する Datum）
  • **出力**: MultirangeType *（デトーストされたマルチレンジ型構造体へのポインタ）

前提条件と不変条件

  • 入力 X は有効なマルチレンジ型の Datum である必要がある。
  • PG_DETOAST_DATUM を使用しているため、必要に応じてメモリ上のデータが展開（デトースト）される。

アルゴリズムまたは振る舞い

1. 引数 X に対して PG_DETOAST_DATUM マクロを適用する。 2. 展開された結果を MultirangeType * 型にキャストして返却する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/executor/execIndexing.c
  • src/backend/utils/adt/multirangetypes_selfuncs.c
  • src/backend/utils/adt/rangetypes_gist.c
  • src/backend/utils/adt/rangetypes_typanalyze.c
  • src/include/utils/multirangetypes.h (マクロ PG_GETARG_MULTIRANGE_P 内)
  • **呼び出し先**:
  • PG_DETOAST_DATUM（外部マクロ）

関連データ構造

  • MultirangeType: マルチレンジ型のメモリ表現。
  • Datum: PostgreSQLの汎用データ型表現。

失敗モードと境界条件

  • PG_DETOAST_DATUM の仕様に依存する。入力が不正なメモリ領域を指している場合や、型が不整合な場合はランタイムエラーが発生する可能性がある。

保守担当者向けメモ

  • 本関数はインライン関数として定義されており、パフォーマンスを考慮して直接メモリへのアクセスを提供している。
  • 構造体のメンバを直接操作せず、必ず提供されているアクセサ関数やマクロを使用すること。
