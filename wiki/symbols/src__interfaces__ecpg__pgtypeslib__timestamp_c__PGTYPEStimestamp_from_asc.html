PGTYPEStimestamp_from_asc

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/pgtypeslib/timestamp.c:201

目的

文字列形式のタイムスタンプを解析し、内部表現である timestamp 型（マイクロ秒単位の int64）へ変換します。

シグネチャまたは形状

timestamp PGTYPEStimestamp_from_asc(char *str, char **endptr);

入力と出力

  • **入力**:
  • str: 解析対象のタイムスタンプ文字列。
  • endptr: 解析終了位置を受け取るポインタ（NULL指定可能）。
  • **出力**:
  • 成功時: 解析された timestamp 値。
  • 失敗時: 0 を返し、グローバル変数 errno に PGTYPES_TS_BAD_TIMESTAMP を設定。

前提条件と不変条件

  • 入力文字列 str の長さは MAXDATELEN 以下である必要があります。
  • 内部的な日付・時刻解析関数（ParseDateTime, DecodeDateTime）が正常に完了する必要があります。
  • 変換結果が有効範囲外の場合、エラーとして扱われます。

アルゴリズムまたは振る舞い

1. 文字列長をチェックし、超過している場合はエラーを返します。 2. ParseDateTime で文字列をフィールドに分割し、DecodeDateTime で日付・時刻要素へデコードします。 3. デコードされた型（dtype）に応じて処理を分岐します：

  • DTK_DATE: tm2timestamp を呼び出し、構造体から timestamp へ変換。
  • DTK_EPOCH: SetEpochTimestamp を呼び出し、エポック値を設定。
  • DTK_LATE: TIMESTAMP_NOEND を使用して無限大（未来）を設定。
  • DTK_EARLY: TIMESTAMP_NOBEGIN を使用して無限大（過去）を設定。

    • 4. 成功時は errno を 0 にリセットして結果を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/interfaces/ecpg/compatlib/informix.c
  • src/interfaces/ecpg/ecpglib/data.c
  • src/interfaces/ecpg/test/expected/pgtypeslib-dt_test.c
  • src/interfaces/ecpg/test/expected/pgtypeslib-dt_test2.c
  • **呼び出し先**:
  • ParseDateTime, DecodeDateTime
  • tm2timestamp
  • SetEpochTimestamp
  • TIMESTAMP_NOEND, TIMESTAMP_NOBEGIN (マクロ)

関連データ構造

  • timestamp: int64 型。
  • struct tm: 日付・時刻の構成要素を保持。
  • fsec_t: 小数秒を保持。

失敗モードと境界条件

  • 文字列が長すぎる場合、または解析に失敗した場合は errno に PGTYPES_TS_BAD_TIMESTAMP が設定され、0 が返されます。
  • 未知の dtype が返された場合もエラーとなります。

保守担当者向けメモ

  • 本関数は errno を使用してエラー状態を通知するため、呼び出し側で errno の確認を行うことが推奨されます。
  • endptr が NULL の場合は内部で一時的なポインタが使用されます。
