目的

スウェーデン語のステミング処理において、単語末尾の主要な接尾辞を特定し、条件に応じて削除または正規化を行う。

シグネチャまたは形状

static int r_main_suffix(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング実行環境の状態を保持する構造体）
  • 出力: int（成功時は 1、失敗または処理対象外の場合は 0、エラー発生時は負の値を返す）

前提条件と不変条件

  • z->I[1] に格納された領域境界（R1）以降の文字を処理対象とする。
  • 処理対象の文字が特定のビットマスク条件（1851442）を満たす必要がある。

アルゴリズムまたは振る舞い

1. **境界チェック**: 現在のカーソル位置 z->c が z->I[1] 未満であれば終了する。 2. **マッチング**: a_0 配列を用いて、単語末尾から find_among_b を実行し、接尾辞を検索する。 3. **条件分岐**:

  • among_var == 1: マッチした接尾辞を slice_del で削除する。
  • among_var == 2: g_s_ending グループに含まれる文字が直前にある場合のみ、slice_del で削除する。

    • 4. **状態復元**: 処理の前後で z->lb（左境界）を一時的に退避・復元し、処理範囲を保護する。

呼び出し元と呼び出し先

  • 呼び出し元: swedish_UTF_8_stem（src/backend/snowball/libstemmer/stem_UTF_8_swedish.c）
  • 呼び出し先: find_among_b, slice_del, in_grouping_b_U

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_0: 接尾辞マッチングに使用される among 構造体の配列。
  • g_s_ending: among_var == 2 の判定に使用される文字グループ。

失敗モードと境界条件

  • z->c <= z->lb: カーソルが左境界を超えている場合は処理を中断する。
  • find_among_b がマッチしない場合、または slice_del が負の値を返した場合は、その値をそのまま呼び出し元へ伝播させる。
