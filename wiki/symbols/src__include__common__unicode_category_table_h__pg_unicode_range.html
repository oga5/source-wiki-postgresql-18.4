pg_unicode_range

定義

  • 種別: struct
  • 定義箇所: src/include/common/unicode_category_table.h:28

目的

Unicodeコードポイントの範囲（開始点と終了点）を表現し、特定のカテゴリやプロパティに該当する文字集合を定義するために使用されます。

シグネチャまたは形状

typedef struct
{
	uint32		first;			/* Unicode codepoint */
	uint32		last;			/* Unicode codepoint */
} pg_unicode_range;

入力と出力

  • **入力**: first および last メンバに、Unicodeコードポイントの範囲を指定します。
  • **出力**: 構造体として、特定のプロパティテーブル（unicode_alphabetic 等）の一部として保持されます。

前提条件と不変条件

  • first は範囲の開始コードポイント、last は終了コードポイント（包含的）を表します。
  • この構造体を用いたテーブルは、二分探索が可能なようにソートされていることが前提となります。

アルゴリズムまたは振る舞い

  • 本構造体はデータ定義のみを持ち、ロジックは含みません。
  • 外部の range_search 関数（src/common/unicode_category.c）が、この構造体の配列を引数として受け取り、指定されたコードポイントが範囲内に含まれるかを二分探索で判定します。

呼び出し元と呼び出し先

  • **呼び出し元（参照元）**:
  • src/common/unicode_category.c: range_search 関数がこの構造体を参照します。
  • src/include/common/unicode_category_table.h: unicode_alphabetic, unicode_lowercase, unicode_uppercase, unicode_case_ignorable, unicode_white_space, unicode_hex_digit, unicode_join_control などの静的配列定義で使用されています。
  • **呼び出し先**:
  • 構造体定義のため、呼び出し先はありません。

関連データ構造

  • pg_category_range: カテゴリ情報を保持する関連構造体。
  • pg_unicode_properties: プロパティフラグを保持する関連構造体。

失敗モードと境界条件

  • first > last となるような不正な範囲が定義された場合、検索関数 range_search の動作が保証されません。
  • 範囲外のコードポイントが検索された場合、range_search は該当なしを返します。
