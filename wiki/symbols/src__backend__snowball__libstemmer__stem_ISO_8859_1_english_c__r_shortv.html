r_shortv

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_english.c:461

目的

単語の末尾が「短音節（short syllable）」構造であるかを判定する。

シグネチャまたは形状

static int r_shortv(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング環境構造体。現在のカーソル位置 z->c を含む）
  • 出力: int（条件を満たす場合は 1、満たさない場合は 0）

前提条件と不変条件

  • z が有効な SN_env 構造体を指していること。
  • g_v_WXY および g_v グルーピング定義に基づき、母音および子音の判定が行われる。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置から後方に向かって、以下のいずれかのパターンを検索する。

  • パターンA: [非母音(WXY)] + [母音] + [母音]
  • パターンB: [母音] + [非母音] + [母音]（ただし、カーソル位置が文字列の先頭境界 z->lb を超えないこと）

    • 2. パターンAに一致する場合、成功として 1 を返す。 3. パターンAに一致しない場合、カーソル位置を復元し、パターンBを試行する。 4. いずれにも一致しない場合、0 を返す。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_ISO_8859_1_english.c および stem_ISO_8859_1_porter.c 等の各ステップ関数から参照されている。
  • 呼び出し先: out_grouping_b, in_grouping_b（Snowballランタイムのユーティリティ関数）。

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • g_v_WXY, g_v: 母音判定用の文字グループ。

失敗モードと境界条件

  • 文字列の先頭（z->lb）に達した場合、パターンBの判定において 0 を返す。
  • 検索対象の文字がグループ定義に含まれない場合、判定は失敗する。

保守担当者向けメモ

  • 本関数は自動生成コードの一部である。ロジックの変更が必要な場合は、直接編集せず元の Snowball スクリプト（.sbl）を修正すること。
  • _b 接尾辞が付く関数は後方一致検索を行うため、カーソル操作の副作用に注意が必要である。
