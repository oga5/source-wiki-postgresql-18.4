PQgetgssctx

定義

  • 種別: function
  • 定義箇所: src/interfaces/libpq/fe-secure.c:482

目的

GSSAPI認証で使用されるセキュリティコンテキストを取得するためのインターフェースを提供します。

シグネチャまたは形状

void *PQgetgssctx(PGconn *conn);

入力と出力

  • **入力**: PGconn *conn （接続状態を保持する構造体へのポインタ）
  • **出力**: void * （現在の実装では常に NULL を返します）

前提条件と不変条件

  • 引数 conn が有効な PGconn 構造体を指している必要があります。
  • 本関数は常に NULL を返すため、呼び出し側は戻り値が NULL であることを前提とした処理を行う必要があります。

アルゴリズムまたは振る舞い

関数は引数として受け取った conn を使用せず、即座に NULL を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/interfaces/libpq/fe-secure-gssapi.c:759
  • src/interfaces/libpq/fe-secure.c:483
  • **呼び出し先**:
  • 該当なし（本関数は他の関数を呼び出しません）

関連データ構造

  • PGconn: libpq の接続状態を管理する主要な構造体。

失敗モードと境界条件

  • 本関数自体はエラーを発生させず、常に NULL を返します。

保守担当者向けメモ

  • 本関数は現在ダミー実装であり、GSSAPIコンテキストの取得機能を提供していません。将来的にGSSAPIコンテキストへのアクセスが必要となった場合は、PGconn 構造体内に適切なフィールドを追加し、本関数を実装し直す必要があります。
