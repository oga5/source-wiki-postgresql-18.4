json_manifest_object_end

定義

  • 種別: function
  • 定義箇所: src/common/parse_manifest.c:316

目的

バックアップマニフェストのJSON解析において、オブジェクトの終了イベントを処理し、解析状態の遷移とデータの確定を行う。

シグネチャまたは形状

static JsonParseErrorType json_manifest_object_end(void *state)

入力と出力

  • **入力**: void *state（JsonManifestParseState 構造体へのポインタ）
  • **出力**: JSON_SUCCESS（成功時）、または解析失敗時（json_manifest_parse_failure 経由で異常終了）

前提条件と不変条件

  • state は JsonManifestParseState 型としてキャスト可能であること。
  • 解析器の状態（parse->state）が、オブジェクト終了を許容する特定のフェーズ（JM_EXPECT_TOPLEVEL_END, JM_EXPECT_THIS_FILE_FIELD, JM_EXPECT_THIS_WAL_RANGE_FIELD）にあること。

アルゴリズムまたは振る舞い

1. parse->state に基づき以下の遷移を行う：

  • JM_EXPECT_TOPLEVEL_END: 解析を終了フェーズ（JM_EXPECT_EOF）へ移行する。
  • JM_EXPECT_THIS_FILE_FIELD: json_manifest_finalize_file を呼び出し、蓄積されたファイル情報を確定させ、次のファイル待ち状態（JM_EXPECT_FILES_NEXT）へ移行する。
  • JM_EXPECT_THIS_WAL_RANGE_FIELD: json_manifest_finalize_wal_range を呼び出し、WAL範囲情報を確定させ、次のWAL範囲待ち状態（JM_EXPECT_WAL_RANGES_NEXT）へ移行する。

    • 2. 上記以外の状態の場合は json_manifest_parse_failure を呼び出し、エラーを報告する。

呼び出し元と呼び出し先

  • **呼び出し元**: JSONパーサー（jsonapi.c 経由のコールバック）。
  • **呼び出し先**:
  • json_manifest_finalize_file
  • json_manifest_finalize_wal_range
  • json_manifest_parse_failure

関連データ構造

  • JsonManifestParseState: 解析中の状態を保持する構造体。

失敗モードと境界条件

  • 予期しないタイミングでオブジェクトが終了した場合、json_manifest_parse_failure が呼び出され、処理が中断される。

保守担当者向けメモ

  • 本関数は JsonManifestParseState の状態遷移を厳密に管理している。新しいJSONフィールドやオブジェクト構造を追加する場合は、switch 文のケースを適切に拡張すること。
