CreateWalTarMethod

定義

  • 種別: function
  • 定義箇所: src/bin/pg_basebackup/walmethods.c:1354

目的

WAL（Write-Ahead Log）データを単一の tar ファイルとして保存するための WalWriteMethod インスタンスを生成・初期化する。

シグネチャまたは形状

WalWriteMethod *CreateWalTarMethod(const char *tarbase,
                                   pg_compress_algorithm compression_algorithm,
                                   int compression_level, bool sync);

入力と出力

  • **入力**:
  • tarbase: 生成する tar ファイルのベースパス。
  • compression_algorithm: 使用する圧縮アルゴリズム（PG_COMPRESSION_GZIP 等）。
  • compression_level: 圧縮レベル。
  • sync: 書き込み時に fsync を実行するかどうかのフラグ。
  • **出力**:
  • 初期化された WalWriteMethod 構造体へのポインタ。

前提条件と不変条件

  • tarbase は有効なパス文字列であること。
  • compression_algorithm が PG_COMPRESSION_GZIP の場合、.tar.gz 拡張子が自動的に付与される。それ以外は .tar が付与される。
  • tar メソッドは一度に1つのファイルのみオープン可能という制約を持つ。

アルゴリズムまたは振る舞い

1. TarMethodData 構造体をメモリ確保し、WalTarMethodOps をメソッドテーブルとして割り当てる。 2. 圧縮設定および同期フラグを構造体に格納し、エラー状態をクリアする。 3. tarbase と拡張子を結合し、tarfilename を生成する。 4. ファイル記述子 fd を -1 に初期化する。 5. HAVE_LIBZ が定義されており、かつ PG_COMPRESSION_GZIP が指定されている場合、zlibOut 用のバッファを確保する。 6. 初期化済みの WalWriteMethod を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/bin/pg_basebackup/pg_basebackup.c:572
  • **呼び出し先**:
  • pg_malloc0 / pg_malloc (メモリ確保)
  • clear_error (エラー状態の初期化)

関連データ構造

  • TarMethodData: tar メソッド固有のデータ保持構造体。
  • WalWriteMethod: 抽象基底構造体。
  • WalTarMethodOps: tar メソッド用の関数ポインタテーブル。

失敗モードと境界条件

  • メモリ確保失敗時は pg_malloc 経由でプロセスが終了する可能性がある。
  • tarbase が NULL または不正なパスの場合、sprintf でのメモリ破壊やエラーが発生する可能性がある。

保守担当者向けメモ

  • 本メソッドは Gzip 圧縮のみをサポートしている。LZ4 圧縮はディレクトリメソッドでのみサポートされている点に注意すること。
  • zlibOut バッファの確保は HAVE_LIBZ マクロに依存しているため、ビルド環境による挙動の違いに留意すること。
