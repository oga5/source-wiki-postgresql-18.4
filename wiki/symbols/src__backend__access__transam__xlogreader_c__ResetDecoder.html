ResetDecoder

定義

  • 種別: function
  • 定義箇所: src/backend/access/transam/xlogreader.c:1614

目的

XLogReaderState が保持するデコード済みレコードのキュー、デコード用バッファ、およびエラー状態を初期化（リセット）する。

シグネチャまたは形状

static void ResetDecoder(XLogReaderState *state)

入力と出力

  • **入力**: state (XLogReaderState *) - リセット対象のリーダー状態構造体。
  • **出力**: なし（副作用として state の内部状態が更新される）。

前提条件と不変条件

  • state は有効な XLogReaderState 構造体へのポインタであること。
  • 本関数実行後、デコードキューは空になり、デコードバッファのヘッドとテールはバッファの開始位置にリセットされる。

アルゴリズムまたは振る舞い

1. **デコードキューの解放**: decode_queue_head から順にレコードを辿り、oversized フラグが立っているレコードを pfree で解放する。その後、キュー関連のポインタをすべて NULL に設定する。 2. **デコードバッファのクリア**: decode_buffer_head および decode_buffer_tail を decode_buffer の開始位置にリセットする。 3. **エラー状態のクリア**: errormsg_buf を空文字列にし、errormsg_deferred フラグを false に設定する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/access/transam/xlogreader.c 内の特定の箇所（ソースコード上の直接的な参照として確認済み）。
  • **呼び出し先**:
  • pfree（メモリ解放用関数）。

関連データ構造

  • XLogReaderState: リーダーの状態を保持するメイン構造体。
  • DecodedXLogRecord: デコードされたレコードを保持する構造体。

失敗モードと境界条件

  • state が NULL の場合、未定義の動作を引き起こす可能性がある（ソースコード上でのチェックは明示されていない）。
  • キュー内に oversized ではないレコードが存在する場合、それらは解放されずにキューから外される（メモリ管理の不変条件に依存）。

保守担当者向けメモ

  • 本関数は XLogReaderState の内部状態を強制的に初期化するため、リーダーの再利用時やエラー回復時に呼び出される。
  • pfree を使用しているため、フロントエンドツールで使用する際は、適切なメモリコンテキストが設定されていることを確認すること。
