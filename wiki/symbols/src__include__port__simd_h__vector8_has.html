vector8_has

定義

  • 種別: function
  • 定義箇所: src/include/port/simd.h:161

目的

ベクトル v 内に特定の 8 ビット値 c が含まれているかを判定する。

シグネチャまたは形状

static inline bool vector8_has(const Vector8 v, const uint8 c)

入力と出力

  • **入力**:
  • v: 検索対象の Vector8 型データ。
  • c: 検索する 8 ビットのターゲット値。
  • **出力**:
  • bool: ベクトル内に c が存在すれば true、そうでなければ false を返す。

前提条件と不変条件

  • USE_ASSERT_CHECKING が有効な場合、SIMD 演算の結果と、メモリ上の各要素を逐次比較した結果が一致することが保証される。

アルゴリズムまたは振る舞い

1. **検証（デバッグ時）**: USE_ASSERT_CHECKING が定義されている場合、ループを用いて v の各要素と c を比較し、期待値を算出する。 2. **検索**:

  • USE_NO_SIMD が定義されている場合: vector8_broadcast(c) で生成したベクトルと v の XOR を取り、vector8_has_zero を用いてゼロ要素の有無を判定する。
  • SIMD 利用可能な場合: vector8_eq で v と c を比較し、その結果に対して vector8_is_highbit_set を適用して判定する。

    • 3. **検証（終了時）**: 算出した結果と検証用結果を Assert で比較し、値を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/json.c
  • src/include/port/pg_lfind.h
  • src/include/port/simd.h (内部的な自己参照)
  • **呼び出し先**:
  • vector8_broadcast
  • vector8_has_zero (SIMD 非使用時)
  • vector8_eq (SIMD 使用時)
  • vector8_is_highbit_set (SIMD 使用時)

関連データ構造

  • Vector8: プラットフォーム依存のベクトル型（__m128i や uint8x16_t 等）。

失敗モードと境界条件

  • 特になし。入力された Vector8 の全要素を走査する。

保守担当者向けメモ

  • 本関数は simd.h 内の他の関数（vector8_has_zero 等）と密接に関連している。
  • SIMD 非対応環境ではビット演算によるエミュレーションが行われるため、移植性を維持しつつパフォーマンスを確保する設計となっている。
