pgssGlobalStats

定義

  • 種別: struct
  • 定義箇所: contrib/pg_stat_statements/pg_stat_statements.c:218

目的

pg_stat_statements 拡張機能全体に関わるグローバルな統計情報を保持するための構造体です。

シグネチャまたは形状

typedef struct pgssGlobalStats
{
    int64       dealloc;        /* エントリが解放された回数 */
    TimestampTz stats_reset;    /* 全ての統計がリセットされた時刻 */
} pgssGlobalStats;

入力と出力

  • **入力**: 統計情報の永続化ファイルからの読み込み、または統計リセット操作時の現在時刻。
  • **出力**: 統計情報の永続化ファイルへの書き出し。

前提条件と不変条件

  • dealloc は、ハッシュテーブルのエントリがメモリ不足等により解放されるたびにインクリメントされます。
  • stats_reset は、統計情報が初期化またはリセットされた時点のタイムスタンプを保持します。

アルゴリズムまたは振る舞い

本構造体は、pgssSharedState 内のメンバとして保持され、以下の処理に関与します。

  • **永続化**: サーバーのシャットダウン時や定期的な処理において、ファイルへの書き出し (fwrite) および読み込み (fread) が行われます。
  • **管理**: 統計情報のライフサイクル（リセットやエントリの破棄）を追跡するために使用されます。

呼び出し元と呼び出し先

  • **呼び出し元（参照箇所）**:
  • contrib/pg_stat_statements/pg_stat_statements.c:255: pgssSharedState 構造体内で stats として定義。
  • contrib/pg_stat_statements/pg_stat_statements.c:673: 統計ファイル読み込み時に使用。
  • contrib/pg_stat_statements/pg_stat_statements.c:798: 統計ファイル書き出し時に使用。
  • contrib/pg_stat_statements/pg_stat_statements.c:2024: 統計表示関数内でローカル変数として使用。
  • **呼び出し先**:
  • 特定の関数呼び出しはソースから直接確認できませんが、fread/fwrite を介したファイルシステム操作が行われます。

関連データ構造

  • pgssSharedState: 本構造体を保持する共有メモリ上のメイン構造体。

失敗モードと境界条件

  • **ファイルI/Oエラー**: fread または fwrite が期待されるサイズ（sizeof(pgssGlobalStats)）を読み書きできない場合、統計情報の永続化に失敗します。

保守担当者向けメモ

  • 本構造体は共有メモリ上に配置されるため、更新時には適切なロック（pgssSharedState のロック）が取得されていることを確認してください。
  • 構造体のメンバを追加・変更する場合は、統計ファイルのフォーマット互換性に影響するため、バージョン管理に注意が必要です。
