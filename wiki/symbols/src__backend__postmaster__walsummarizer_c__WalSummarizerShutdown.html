WalSummarizerShutdown

定義

  • 種別: function
  • 定義箇所: src/backend/postmaster/walsummarizer.c:791

目的

WAL Summarizer プロセスの終了処理として、共有メモリ上のプロセス状態を無効化する。

シグネチャまたは形状

static void WalSummarizerShutdown(int code, Datum arg)

入力と出力

  • code: 終了コード（on_shmem_exit のコールバック規約に従う）。
  • arg: 補助データ（未使用）。
  • 出力: なし。

前提条件と不変条件

  • WALSummarizerLock が利用可能であること。
  • WalSummarizerCtl が初期化済みであること。

アルゴリズムまたは振る舞い

1. WALSummarizerLock を排他モードで取得する。 2. 共有メモリ上の WalSummarizerCtl->summarizer_pgprocno を INVALID_PROC_NUMBER に設定し、プロセスが終了したことを示す。 3. WALSummarizerLock を解放する。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/postmaster/walsummarizer.c 内の on_shmem_exit 登録箇所（src/backend/postmaster/walsummarizer.c:260）。
  • 呼び出し先: LWLockAcquire, LWLockRelease。

関連データ構造

  • WalSummarizerCtl: 共有メモリ上の制御構造体。
  • WALSummarizerLock: プロセス状態を保護するためのロック。

失敗モードと境界条件

  • ロック取得時にデッドロックが発生する可能性がある（他のプロセスが WALSummarizerLock を保持したまま異常終了した場合など）。

保守担当者向けメモ

  • 本関数は on_shmem_exit を介して登録される終了ハンドラである。プロセスが正常または異常終了する際に、共有メモリの状態をクリーンアップするために呼び出される。
