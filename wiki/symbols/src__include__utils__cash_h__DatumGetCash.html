DatumGetCash

定義

  • 種別: static inline 関数
  • 定義箇所: src/include/utils/cash.h:20

目的

PostgreSQL の Datum 型から Cash 型（通貨データ）への変換を行う。

シグネチャまたは形状

static inline Cash DatumGetCash(Datum X)

入力と出力

  • **入力**: Datum X（変換対象のデータ）
  • **出力**: Cash（int64 のエイリアスである通貨データ）

前提条件と不変条件

  • Cash 型は int64 として定義されている。
  • 入力 Datum X は、適切に Cash 型として解釈可能な値である必要がある。

アルゴリズムまたは振る舞い

内部的に DatumGetInt64(X) を呼び出し、Datum を 64 ビット整数として抽出して返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/btree_gist/btree_utils_num.c:81
  • src/include/utils/cash.h:32（PG_GETARG_CASH マクロ内）
  • **呼び出し先**:
  • DatumGetInt64（PostgreSQL の FMGR インターフェース関数）

関連データ構造

  • Cash: 通貨データの内部表現（int64）。
  • Datum: PostgreSQL の汎用データ型。

失敗モードと境界条件

  • Datum からの変換において、型が不整合な場合の動作は DatumGetInt64 の仕様に依存する。
  • 64 ビット整数としての範囲外の値が渡された場合の挙動は、呼び出し側のコンテキストに依存する。

保守担当者向けメモ

  • 本関数はインライン化されており、パフォーマンスを重視している。
  • money 型の演算ロジック自体は src/backend/utils/adt/cash.c に実装されているため、演算時のオーバーフローチェック等が必要な場合はそちらを参照すること。
