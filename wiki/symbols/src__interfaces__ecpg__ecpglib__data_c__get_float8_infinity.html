get_float8_infinity

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/ecpglib/data.c:79

目的

プラットフォームに依存しない方法で、浮動小数点数の「無限大（Infinity）」を表す double 値を取得する。

シグネチャまたは形状

static double get_float8_infinity(void);

入力と出力

  • 入力: なし
  • 出力: double 型の無限大値

前提条件と不変条件

  • math.h がインクルードされている環境を前提とする。
  • INFINITY マクロが定義されている場合はそれを使用し、未定義の場合は HUGE_VAL の二乗を用いて無限大を生成する。

アルゴリズムまたは振る舞い

1. INFINITY マクロが定義されているか判定する。 2. 定義されている場合、INFINITY を double にキャストして返す。 3. 定義されていない場合、HUGE_VAL * HUGE_VAL を計算し、その結果（無限大）を double にキャストして返す。

呼び出し元と呼び出し先

  • 呼び出し先: math.h の INFINITY または HUGE_VAL。
  • 呼び出し元: ツールによる索引では、contrib/btree_gin や src/backend/access/ 配下のバックエンドコードからの参照が多数検出されているが、これらは src/interfaces/ecpg/ecpglib/data.c 内の関数を直接呼び出しているものではなく、同名の関数が各所に存在している可能性が高い。ecpglib 内での具体的な呼び出し元は、提供されたソース情報からは特定できない。

関連データ構造

  • なし

失敗モードと境界条件

  • HUGE_VAL が定義されていない環境ではコンパイルエラーとなる可能性がある。
  • HUGE_VAL が非常に大きい値である場合、その二乗はオーバーフローして無限大となることが期待されるが、環境の浮動小数点演算の実装に依存する。

保守担当者向けメモ

  • 本関数は static であり、src/interfaces/ecpg/ecpglib/data.c 内でのみ使用される。
  • バックエンド側の同名関数と混同しないよう注意すること。ECPG 固有のデータ変換ロジックの一部として機能する。
