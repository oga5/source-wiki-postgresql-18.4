usage

定義

  • 種別: function
  • 定義箇所: src/bin/pg_checksums/pg_checksums.c:66

目的

pg_checksums コマンドの利用方法、オプション一覧、および環境変数 PGDATA に関する情報を標準出力へ表示する。

シグネチャまたは形状

static void usage(void);

入力と出力

  • 入力: なし（グローバル変数 progname を参照）
  • 出力: なし（標準出力へヘルプメッセージを出力）

前提条件と不変条件

  • progname が適切に初期化されていること。
  • PACKAGE_BUGREPORT、PACKAGE_NAME、PACKAGE_URL がビルド時に定義されていること。

アルゴリズムまたは振る舞い

1. progname を用いてツールの概要を表示する。 2. 利用可能なコマンドラインオプション（-D, -c, -d, -e, -f, -N, -P, --sync-method, -v, -V, -?）を列挙する。 3. データディレクトリの指定方法（引数または PGDATA 環境変数）を説明する。 4. バグ報告先およびプロジェクトのホームページ情報を表示する。

呼び出し元と呼び出し先

  • 呼び出し元: 提供されたソースコード内には、本関数を呼び出している箇所は明示されていません。
  • 呼び出し先: printf（標準ライブラリ）

※ツールによる索引結果に表示されている contrib 等の他のファイル内の usage に関する言及は、本関数とは無関係な識別子への参照であり、本関数の呼び出し元ではありません。

関連データ構造

  • なし

失敗モードと境界条件

  • 標準出力への書き込みが失敗した場合の挙動は、標準ライブラリの printf に依存します。

保守担当者向けメモ

  • オプションを追加または変更した際は、本関数内のメッセージも更新してください。
  • 国際化対応のため、メッセージは _() マクロで囲まれています。
