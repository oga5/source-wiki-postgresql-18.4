shell_archive_configured

定義

  • 種別: function
  • 定義箇所: src/backend/archive/shell_archive.c:45

目的

アーカイブモジュールが実行可能状態にあるか、すなわち archive_command が設定されているかを検証する。

シグネチャまたは形状

static bool shell_archive_configured(ArchiveModuleState *state)

入力と出力

  • **入力**: ArchiveModuleState *state（アーカイブモジュールの状態構造体。本関数内では未使用）
  • **出力**: bool 型。archive_command が設定されていれば true、そうでなければ false を返す。

前提条件と不変条件

  • グローバル変数 XLogArchiveCommand が、PostgreSQLの構成設定に基づいて適切に初期化されていること。

アルゴリズムまたは振る舞い

1. グローバル変数 XLogArchiveCommand の先頭文字を確認する。 2. 文字列が空（'\0'）でない場合、true を返す。 3. 文字列が空の場合、arch_module_check_errdetail を呼び出してエラー詳細を記録し、false を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: shell_archive_callbacks 構造体の初期化時（src/backend/archive/shell_archive.c:34）に、check_configured_cb フィールドとして登録される。
  • **呼び出し先**: arch_module_check_errdetail（エラー詳細の記録用）。

関連データ構造

  • ArchiveModuleState: アーカイブモジュールの状態を保持する構造体。
  • XLogArchiveCommand: archive_command GUCパラメータの値を保持するグローバル文字列。

失敗モードと境界条件

  • archive_command が未設定（空文字列）の場合、arch_module_check_errdetail を通じてエラーメッセージが生成され、アーカイブ処理の構成が不完全であることが報告される。

保守担当者向けメモ

  • 本関数は ArchiveModuleCallbacks インターフェースの一部として呼び出される。
  • state 引数は現在のインターフェース定義に従って受け取っているが、現在の実装では XLogArchiveCommand を直接参照しているため使用されていない。将来的な拡張で状態依存のチェックが必要になった場合に利用可能である。
