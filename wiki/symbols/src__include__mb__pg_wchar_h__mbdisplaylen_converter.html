mbdisplaylen_converter

定義

  • 種別: typedef
  • 定義箇所: src/include/mb/pg_wchar.h:384

目的

マルチバイト文字列の先頭文字が占める表示幅（カラム数）を計算するための関数ポインタ型です。

シグネチャまたは形状

typedef int (*mbdisplaylen_converter) (const unsigned char *mbstr);

入力と出力

  • **入力**: mbstr - 評価対象のマルチバイト文字列へのポインタ。
  • **出力**: int - 文字列先頭の文字が占める表示幅。

前提条件と不変条件

  • 入力される mbstr は、対象となるエンコーディングにおいて有効なマルチバイト文字の先頭を指している必要があります。
  • 返される値は、その文字が端末や画面上で占めるカラム数（通常は 0, 1, 2 のいずれか）を返します。

アルゴリズムまたは振る舞い

この型は、特定のエンコーディングにおける文字の表示幅を算出する関数を抽象化します。各エンコーディングの実装において、この型に適合する関数が定義され、pg_wchar_tbl 構造体を通じて利用されます。

呼び出し元と呼び出し先

  • **呼び出し元**: src/include/mb/pg_wchar.h 内の pg_wchar_tbl 構造体において、dsplen フィールドとして保持されています。具体的な呼び出し箇所については、提供されたソース excerpt には含まれていません。
  • **呼び出し先**: 不明（各エンコーディング固有の実装関数が代入されます）。

関連データ構造

  • pg_wchar_tbl: 各エンコーディングの変換・検証関数を保持するテーブル構造体であり、本型はそのメンバとして使用されます。

失敗モードと境界条件

  • 入力ポインタが NULL の場合の挙動については、本定義からは規定されていません。
  • 不正なマルチバイトシーケンスが渡された場合の戻り値は、各実装関数に依存します。

保守担当者向けメモ

  • 本型は pg_wchar_tbl の一部として、PostgreSQL の文字幅計算の抽象化レイヤーを構成しています。新しいエンコーディングを追加する際は、この型に適合する関数を実装し、テーブルに登録する必要があります。
