PLpgSQL_type_type

定義

  • 種別: enum
  • 定義箇所: src/pl/plpgsql/src/plpgsql.h:93

目的

PL/pgSQL におけるデータ型の分類を定義し、コンパイルおよび実行時に型情報を識別するために使用されます。

シグネチャまたは形状

typedef enum PLpgSQL_type_type
{
    PLPGSQL_TTYPE_SCALAR,
    PLPGSQL_TTYPE_REC,
    PLPGSQL_TTYPE_PSEUDO,
} PLpgSQL_type_type;

入力と出力

  • 本シンボル自体は列挙型定義であり、入力や出力は持ちません。
  • 実行時には、PLpgSQL_type 構造体内の ttype フィールドとして値が保持されます。

前提条件と不変条件

  • PLpgSQL_type 構造体の ttype フィールドに格納される値は、必ずこの列挙型のいずれかである必要があります。

アルゴリズムまたは振る舞い

  • PLPGSQL_TTYPE_SCALAR: スカラー型およびドメイン型を識別します。
  • PLPGSQL_TTYPE_REC: 複合型（RECORD 型を含む）を識別します。
  • PLPGSQL_TTYPE_PSEUDO: 擬似型（anyelement 等）を識別します。

呼び出し元と呼び出し先

  • 呼び出し元: src/pl/plpgsql/src/plpgsql.h:214 にて PLpgSQL_type 構造体のメンバとして参照されています。
  • 呼び出し先: 情報なし。

関連データ構造

  • PLpgSQL_type: この列挙型をメンバとして保持する構造体。

失敗モードと境界条件

  • 定義外の値を ttype に代入した場合の動作は未定義です。

保守担当者向けメモ

  • 新しい型カテゴリを追加する必要がある場合は、この列挙型を拡張し、それを利用する pl_comp.c や pl_exec.c 内のスイッチ文を更新してください。
