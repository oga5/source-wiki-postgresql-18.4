help

定義

  • 種別: function
  • 定義箇所: src/bin/scripts/reindexdb.c:894

目的

reindexdb コマンドの利用方法、オプション一覧、および接続設定に関する情報を標準出力に表示する。

シグネチャまたは形状

void help(const char *progname);

入力と出力

  • **入力**: progname (実行中のプログラム名を示す文字列)
  • **出力**: なし（標準出力へヘルプメッセージを出力して終了する）

前提条件と不変条件

  • progname は有効な文字列ポインタであること。
  • printf が標準出力に対して書き込み可能であること。

アルゴリズムまたは振る舞い

1. プログラム名を含む使用方法の概要を表示する。 2. reindexdb で利用可能なコマンドラインオプション（再構築対象の指定、並列実行、接続設定など）を列挙する。 3. REINDEX SQL コマンドのドキュメント参照を案内する。 4. バグ報告先およびプロジェクトのホームページ情報を出力する。

呼び出し元と呼び出し先

  • **呼び出し元**: ソースコード内で明示的な呼び出し元は特定されていない。
  • **呼び出し先**: printf (標準ライブラリ)

※ツールが提示した contrib/oid2name/oid2name.c 等の参照は、同名の別関数を指しており、本シンボルとは無関係である。

関連データ構造

  • なし

失敗モードと境界条件

  • 標準出力がクローズされている場合、出力に失敗する可能性がある。

保守担当者向けメモ

  • オプションの追加や変更を行った際は、本関数内のメッセージも同期して更新すること。
  • PACKAGE_BUGREPORT、PACKAGE_NAME、PACKAGE_URL マクロに依存しているため、ビルド環境の構成変更時には注意が必要。
