pg_atomic_exchange_u32_impl

定義

  • 種別: static inline 関数
  • 定義箇所: src/include/port/atomics/generic-gcc.h:195

目的

pg_atomic_uint32 型の値を新しい値で原子的に置き換え、置き換え前の値を返す。

シグネチャまたは形状

static inline uint32 pg_atomic_exchange_u32_impl(volatile pg_atomic_uint32 *ptr, uint32 newval)

入力と出力

  • **入力**:
  • ptr: 操作対象となる pg_atomic_uint32 構造体へのポインタ（volatile 修飾あり）。
  • newval: 書き込む新しい uint32 値。
  • **出力**:
  • 置き換え操作が行われる直前の、ptr が指すメモリ位置に格納されていた uint32 値。

前提条件と不変条件

  • ptr は有効な pg_atomic_uint32 構造体を指している必要がある。
  • 本関数は atomics.h を経由して呼び出されることを前提としている。

アルゴリズムまたは振る舞い

GCC の組み込み関数 __atomic_exchange_n を使用して、メモリ順序 __ATOMIC_SEQ_CST（逐次一貫性）で値を交換する。この操作は不可分（atomic）であり、他のスレッドからの干渉を受けずに値の更新と旧値の取得を行う。

呼び出し元と呼び出し先

  • **呼び出し元**: src/include/port/atomics.h:334 および src/include/port/atomics/generic.h 内の複数の箇所から呼び出されている。
  • **呼び出し先**: GCC 組み込み関数 __atomic_exchange_n を呼び出す。

関連データ構造

  • pg_atomic_uint32: 原子操作対象の値を保持する構造体。

失敗モードと境界条件

  • 本関数自体はメモリの整合性を保証するが、ptr が NULL である場合や適切にアライメントされていない場合は未定義動作となる可能性がある。

保守担当者向けメモ

  • 本関数は generic-gcc.h に定義されており、GCC 互換コンパイラ環境での汎用的な実装である。
  • メモリモデルとして __ATOMIC_SEQ_CST を指定しており、最も厳密な順序付けを行っている。
