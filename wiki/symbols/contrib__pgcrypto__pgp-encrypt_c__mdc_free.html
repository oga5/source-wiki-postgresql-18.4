mdc_free

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/pgp-encrypt.c:134

目的

MDC（Modification Detection Code）計算に使用されたハッシュコンテキストのリソースを解放します。

シグネチャまたは形状

static void mdc_free(void *priv)

入力と出力

  • **入力**: void *priv - 解放対象となる PX_MD 型のハッシュコンテキストへのポインタ。
  • **出力**: なし。

前提条件と不変条件

  • priv は px_md_free が安全に処理できる有効な PX_MD コンテキストを指している必要があります。

アルゴリズムまたは振る舞い

1. 引数 priv を PX_MD * 型にキャストします。 2. px_md_free(md) を呼び出し、ハッシュコンテキストに関連付けられたメモリおよびリソースを解放します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/pgcrypto/pgp-decrypt.c（複数箇所）
  • contrib/pgcrypto/pgp-encrypt.c（フィルタチェーンの終了処理時）
  • **呼び出し先**:
  • px_md_free（px.h で定義される共通暗号化プリミティブ）

関連データ構造

  • PX_MD: ハッシュ計算の状態を保持する構造体。

失敗モードと境界条件

  • priv が NULL の場合、px_md_free の実装に依存します。

保守担当者向けメモ

  • 本関数は PushFilter パターンにおけるクリーンアップ関数として使用されます。フィルタチェーンの破棄時に呼び出されることを想定しています。
