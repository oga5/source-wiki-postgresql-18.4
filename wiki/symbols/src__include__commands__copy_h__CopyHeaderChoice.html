CopyHeaderChoice

定義

  • 種別: enum
  • 定義箇所: src/include/commands/copy.h:26

目的

COPY コマンドにおけるヘッダー行の扱い（有無および検証）を指定するための列挙型です。

シグネチャまたは形状

typedef enum CopyHeaderChoice
{
	COPY_HEADER_FALSE = 0,
	COPY_HEADER_TRUE,
	COPY_HEADER_MATCH,
} CopyHeaderChoice;

入力と出力

  • 入力: COPY コマンドのオプション解析時に使用されます。
  • 出力: CopyFormatOptions 構造体の header_line フィールドとして保持されます。

前提条件と不変条件

  • COPY_HEADER_FALSE (0) がデフォルト値として想定されます。
  • COPY_HEADER_MATCH は、ファイル内のヘッダー行がテーブルの列定義と一致するかを検証するモードを示します。

アルゴリズムまたは振る舞い

  • COPY コマンドのオプション解析プロセスにおいて、ユーザー指定の HEADER オプションを内部表現に変換するために使用されます。
  • 実行時、この値に基づいてデータの読み込みまたは書き出し時にヘッダー行をスキップするか、あるいは検証を行うかが決定されます。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/commands/copy.c 内の DefElem から値を抽出する関数群（例: src/backend/commands/copy.c:365 付近）。
  • 呼び出し先: 該当なし（列挙型のため、直接的な関数呼び出しは発生しません）。

関連データ構造

  • CopyFormatOptions: header_line フィールドを通じて本型を参照します。

失敗モードと境界条件

  • 不正なオプション値が指定された場合、解析フェーズでエラーが発生する可能性があります。

保守担当者向けメモ

  • 本型は COPY コマンドのオプション拡張に伴い、新しい選択肢が追加される可能性があります。
  • COPY_HEADER_MATCH の実装詳細については src/backend/commands/copy.c のロジックを参照してください。
