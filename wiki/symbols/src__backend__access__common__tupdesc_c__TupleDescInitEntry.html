TupleDescInitEntry

定義

  • 種別: function
  • 定義箇所: src/backend/access/common/tupdesc.c:841

目的

TupleDesc（タプル記述子）内の特定の属性（列）定義を、システムカタログ（pg_type）から取得した情報に基づいて初期化します。

シグネチャまたは形状

void TupleDescInitEntry(TupleDesc desc,
                        AttrNumber attributeNumber,
                        const char *attributeName,
                        Oid oidtypeid,
                        int32 typmod,
                        int attdim)

入力と出力

  • **入力**:
  • desc: 初期化対象の TupleDesc 構造体。
  • attributeNumber: 設定する属性のインデックス（1始まり）。
  • attributeName: 属性名（NULLの場合は空文字列で初期化）。
  • oidtypeid: 属性のデータ型に対応する OID。
  • typmod: 型修飾子（atttypmod）。
  • attdim: 配列の次元数。
  • **出力**: なし（desc が指すメモリ領域を直接更新）。

前提条件と不変条件

  • desc は有効なポインタであること。
  • attributeNumber は 1 以上かつ desc->natts 以下であること。
  • attdim は 0 以上かつ PG_INT16_MAX 以下であること。
  • 指定された oidtypeid がシステムカタログに存在すること。

アルゴリズムまたは振る舞い

1. **属性の特定**: TupleDescAttr を使用して、指定されたインデックスの Form_pg_attribute 構造体を取得します。 2. **基本情報の初期化**: attrelid を 0 に設定し、属性名、型修飾子、次元数を設定します。 3. **フラグの初期化**: attnotnull、atthasdef、attisdropped などの属性フラグをデフォルト値（主に false または \0）にリセットします。 4. **型情報の取得**: SearchSysCache1 を用いて pg_type システムカタログから型情報を検索します。見つからない場合はエラーを発生させます。 5. **属性の更新**: 取得した Form_pg_type から、attlen、attbyval、attalign、attstorage、attcollation などの物理属性をコピーします。 6. **最適化**: populate_compact_attribute を呼び出し、高速化のための CompactAttribute 情報を同期します。 7. **クリーンアップ**: ReleaseSysCache でシステムキャッシュを解放します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/dblink/dblink.c
  • contrib/pg_buffercache/pg_buffercache_pages.c
  • （その他、動的にタプル記述子を構築する各モジュール）
  • **呼び出し先**:
  • SearchSysCache1 / ReleaseSysCache（システムカタログ参照）
  • populate_compact_attribute（最適化メタデータの更新）
  • namestrcpy（属性名のコピー）

関連データ構造

  • TupleDesc: タプルの構造を保持するメイン構造体。
  • Form_pg_attribute: 属性のメタデータを保持する構造体。
  • Form_pg_type: システムカタログ pg_type の行構造。

失敗モードと境界条件

  • oidtypeid が無効な場合、elog(ERROR) によりトランザクションがアボートされます。
  • attributeNumber が範囲外の場合、Assert によりデバッグビルドで停止します。

保守担当者向けメモ

  • FormData_pg_attribute のフィールドを直接変更した場合は、必ず populate_compact_attribute を呼び出して CompactAttribute 配列との整合性を保つ必要があります。
  • attributeName が NULL の場合、NAMEDATALEN 分の領域をゼロ埋めして初期化します。
