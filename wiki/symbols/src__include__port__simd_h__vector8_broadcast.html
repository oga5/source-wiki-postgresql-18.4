vector8_broadcast

定義

  • 種別: function
  • 定義箇所: src/include/port/simd.h:134

目的

単一の 8 ビット整数値をベクトル内の全要素に複製（ブロードキャスト）し、SIMD レジスタ（Vector8）を生成します。

シグネチャまたは形状

static inline Vector8 vector8_broadcast(const uint8 c)

入力と出力

  • **入力**: c (複製対象の uint8 型の値)
  • **出力**: c を全要素に格納した Vector8 型のベクトル

前提条件と不変条件

  • Vector8 型は、コンパイル時の環境（SSE2, NEON, またはフォールバック）に応じて適切な SIMD 型または整数型に解決されます。
  • フォールバック環境では、64 ビット整数演算を用いてベクトルをエミュレートします。

アルゴリズムまたは振る舞い

コンパイル時の機能フラグに基づき、以下のいずれかの実装が選択されます。 1. **SSE2 (USE_SSE2)**: _mm_set1_epi8(c) を使用。 2. **NEON (USE_NEON)**: vdupq_n_u8(c) を使用。 3. **フォールバック**: ~UINT64CONST(0) / 0xFF * c を計算し、64 ビット整数内に 8 ビット値を 8 個並べた値を生成します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/include/lib/radixtree.h
  • src/include/port/simd.h (内部の比較・演算関数)
  • src/include/utils/ascii.h
  • **呼び出し先**:
  • ハードウェア固有の組み込み関数（_mm_set1_epi8, vdupq_n_u8）または標準的な算術演算子。

関連データ構造

  • Vector8: 本関数が生成するベクトル型。

失敗モードと境界条件

  • 特になし。入力値 c は uint8 の範囲内であれば常に安全に処理されます。

保守担当者向けメモ

  • 本関数はインライン関数として定義されており、ヘッダーファイル内で完結しています。
  • フォールバック実装は 64 ビット環境を前提としています。環境のビット幅が異なる場合の挙動については、simd.h 全体の移植性要件を確認してください。
