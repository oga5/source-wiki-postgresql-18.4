makeBitString

定義

  • 種別: function
  • 定義箇所: src/backend/nodes/value.c:76

目的

ビット列を保持する BitString ノードを生成し、初期化するファクトリ関数です。

シグネチャまたは形状

BitString *makeBitString(char *str)

入力と出力

  • **入力**: str (ビット列を表す文字列へのポインタ)
  • **出力**: 初期化された BitString 構造体へのポインタ

前提条件と不変条件

  • 引数 str は、呼び出し元によって palloc 等で適切にメモリ確保されている必要があります。
  • makeNode(BitString) マクロにより、PostgreSQL のメモリコンテキスト上でノードのメモリが確保されます。

アルゴリズムまたは振る舞い

1. makeNode(BitString) を呼び出し、BitString 型のノード構造体をメモリ上に確保します。 2. 引数として受け取った文字列ポインタ str を、ノードの bsval メンバに代入します。 3. 初期化済みのノードポインタを返却します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/nodes/read.c:502 にて呼び出されています。
  • **呼び出し先**: makeNode マクロ（PostgreSQL のノードメモリ確保用）。

関連データ構造

  • BitString: src/include/nodes/value.h で定義される、ビット列を保持するためのノード構造体。

失敗モードと境界条件

  • makeNode がメモリ不足により失敗した場合、PostgreSQL の標準的なエラーハンドリング（palloc の挙動に準拠）によりプロセスが中断される可能性があります。

保守担当者向けメモ

  • 本関数は文字列の所有権をノードに委譲する設計です。渡される文字列のライフサイクルがノードの寿命と一致していることを確認してください。
  • Float や String ノードと同様のファクトリパターンに従っています。
