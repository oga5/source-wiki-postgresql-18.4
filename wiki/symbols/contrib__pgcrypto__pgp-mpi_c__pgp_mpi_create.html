pgp_mpi_create

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/pgp-mpi.c:55

目的

指定されたバイナリデータから PGP_MPI 構造体を新規作成し、データをコピーして初期化する。

シグネチャまたは形状

int pgp_mpi_create(uint8 *data, int bits, PGP_MPI **mpi);

入力と出力

  • **入力**:
  • data: MPI の本体データを含むバイト配列。
  • bits: MPI のビット長。
  • mpi: 作成された PGP_MPI 構造体へのポインタを格納するアドレス。
  • **出力**:
  • 成功時は 0 を返し、*mpi に確保された構造体が格納される。
  • 失敗時は負の整数（エラーコード）を返す。

前提条件と不変条件

  • bits に基づいて pgp_mpi_alloc がメモリを確保できる必要がある。
  • data は少なくとも (bits + 7) / 8 バイト以上の有効なデータ領域を指している必要がある。

アルゴリズムまたは振る舞い

1. pgp_mpi_alloc(bits, &n) を呼び出し、指定ビット数分のメモリを確保する。 2. 確保に失敗した場合は、そのエラーコードを即座に返す。 3. 確保に成功した場合、n->data に対して入力 data から n->bytes 分のデータを memcpy でコピーする。 4. 成功した構造体ポインタを *mpi に代入し、0 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/pgcrypto/pgp-pubenc.c:115
  • **呼び出し先**: pgp_mpi_alloc

関連データ構造

  • PGP_MPI: MPI データを保持する構造体（ビット数、バイト数、データ本体へのポインタを保持）。

失敗モードと境界条件

  • pgp_mpi_alloc がメモリ不足等で失敗した場合、そのエラーコードがそのまま返される。

保守担当者向けメモ

  • 本関数はメモリ確保とデータコピーを行うため、呼び出し元は不要になった時点で pgp_mpi_free を使用してメモリを解放する必要がある。
