pgstat_backend_reset_timestamp_cb

定義

  • 種別: function
  • 定義箇所: src/backend/utils/activity/pgstat_backend.c:399

目的

バックエンド統計情報の統計リセット時刻（stat_reset_timestamp）を更新するためのコールバック関数です。

シグネチャまたは形状

void pgstat_backend_reset_timestamp_cb(PgStatShared_Common *header, TimestampTz ts)

入力と出力

  • **入力**:
  • header: 更新対象の共有統計構造体の共通ヘッダー（PgStatShared_Common *）。
  • ts: 設定する新しいリセット時刻（TimestampTz）。
  • **出力**: なし（副作用として共有メモリ上の値を更新）。

前提条件と不変条件

  • header は PgStatShared_Backend 型の構造体としてキャスト可能である必要があります。

アルゴリズムまたは振る舞い

1. 引数 header を PgStatShared_Backend * 型へキャストします。 2. キャストした構造体内の stats.stat_reset_timestamp フィールドに、引数 ts で渡された時刻を代入します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/activity/pgstat.c:379 において、統計管理用のコールバックテーブル（reset_timestamp_cb）に登録されています。
  • **呼び出し先**: 直接的な関数呼び出しはありません。

関連データ構造

  • PgStatShared_Common: 統計共有メモリ構造体の共通ヘッダー。
  • PgStatShared_Backend: バックエンド統計を保持する共有メモリ構造体。

失敗モードと境界条件

  • header が NULL である場合や、PgStatShared_Backend 型以外の構造体を指している場合に未定義の動作を引き起こす可能性があります。

保守担当者向けメモ

  • 本関数は統計サブシステムのコールバックインターフェースの一部です。統計リセット処理の拡張が必要な場合は、このコールバックが正しく呼び出されているか確認してください。
