unicode_casefold_simple

定義

  • 種別: function
  • 定義箇所: src/common/unicode_case.c:73

目的

Unicode 文字に対して、正規化比較のためのケースフォールディング（Case Folding）を単一文字単位で適用する。

シグネチャまたは形状

pg_wchar unicode_casefold_simple(pg_wchar code)

入力と出力

  • **入力**: pg_wchar code（変換対象の Unicode コードポイント）
  • **出力**: pg_wchar（変換後のコードポイント。変換ルールが存在しない場合は元の code を返す）

前提条件と不変条件

  • find_case_map 関数が case_map_fold テーブルを参照して変換を試みる。
  • 変換テーブルに該当するマッピングが存在しない場合、入力値がそのまま返される。

アルゴリズムまたは振る舞い

1. find_case_map を呼び出し、case_map_fold テーブルを用いて code に対応するフォールディング後のコードポイント cp を検索する。 2. cp が 0 でない（マッピングが見つかった）場合は cp を返す。 3. cp が 0 の場合は、変換不要と判断し、入力された code をそのまま返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/common/unicode/case_test.c:85
  • **呼び出し先**:
  • find_case_map（内部関数）

関連データ構造

  • pg_wchar: PostgreSQL で使用されるワイド文字型。
  • case_map_fold: src/common/unicode_case_table.h で定義されるケースフォールディング用変換テーブル。

失敗モードと境界条件

  • 入力値が変換テーブルに存在しない場合、エラーにはならず入力値をそのまま返す。
  • 変換結果が 0 の場合は「マッピングなし」として扱われる。

保守担当者向けメモ

  • 本関数は単純な 1 対 1 の文字変換を行う。文字列全体のフォールディングや、文脈依存の複雑な変換が必要な場合は、unicode_strfold 等のより上位の API を検討すること。
