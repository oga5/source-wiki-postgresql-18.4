pgstat_is_kind_builtin

定義

  • 種別: static inline function
  • 定義箇所: src/include/utils/pgstat_kind.h:60

目的

指定された統計種別（PgStat_Kind）が、PostgreSQLの標準組み込み統計種別であるかどうかを判定する。

シグネチャまたは形状

static inline bool pgstat_is_kind_builtin(PgStat_Kind kind)

入力と出力

  • **入力**: kind (PgStat_Kind) - 判定対象の統計種別ID。
  • **出力**: bool - 組み込み種別であれば true、それ以外（カスタム種別や無効値）であれば false を返す。

前提条件と不変条件

  • PgStat_Kind は uint32 型のエイリアスである。
  • 組み込み統計種別は PGSTAT_KIND_BUILTIN_MIN 以上、かつ PGSTAT_KIND_BUILTIN_MAX 以下の範囲に定義されている必要がある。

アルゴリズムまたは振る舞い

引数 kind が PGSTAT_KIND_BUILTIN_MIN 以上かつ PGSTAT_KIND_BUILTIN_MAX 以下であるかを比較し、その真偽値を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/activity/pgstat.c (複数箇所: 統計情報のルーティングや処理分岐に使用)
  • src/backend/utils/activity/pgstat_shmem.c
  • **呼び出し先**:
  • 比較演算子のみを使用しており、他の関数呼び出しは含まれない。

関連データ構造

  • PgStat_Kind: 統計種別を識別するための型。
  • PGSTAT_KIND_BUILTIN_MIN / PGSTAT_KIND_BUILTIN_MAX: 組み込み統計種別の範囲を定義する定数。

失敗モードと境界条件

  • kind が PGSTAT_KIND_INVALID (0) の場合、範囲外となるため false を返す。
  • 拡張機能によって定義されたカスタム統計種別は、この関数の範囲外となるため false を返す。

保守担当者向けメモ

  • 本関数はインライン関数として定義されており、統計システムのパフォーマンスに影響を与えるため、ロジックの変更は最小限に留めること。
  • 統計種別の範囲（MIN/MAX）を変更する場合は、pgstat_kind.h 内の定数定義と整合性を保つ必要がある。
