my_bloom_power

定義

  • 種別: function
  • 定義箇所: src/backend/lib/bloomfilter.c:209

目的

指定されたビット数（target_bitset_bits）を収容可能な最小の 2 のべき乗指数（power）を算出します。

シグネチャまたは形状

static int my_bloom_power(uint64 target_bitset_bits)

入力と出力

  • **入力**: target_bitset_bits (uint64) - 必要なビット数。
  • **出力**: int - 2 のべき乗指数。ビット数が 0 の場合は -1 を返します。

前提条件と不変条件

  • 戻り値の指数は最大 32 に制限されます。
  • 入力値が 0 の場合、初期値の -1 が返されます。

アルゴリズムまたは振る舞い

1. bloom_power を -1 で初期化します。 2. target_bitset_bits が 0 より大きく、かつ bloom_power が 32 未満である間、以下の処理を繰り返します。

  • bloom_power を 1 ずつインクリメントします。
  • target_bitset_bits を右に 1 ビットシフト（2 で除算）します。

    • 3. 最終的な bloom_power を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/lib/bloomfilter.c:108（bloom_create 内でビットセットサイズを決定するために呼び出されます）。
  • **呼び出し先**: なし。

関連データ構造

  • bloom_filter: 本関数で算出された指数を用いて、ビットセットのサイズが決定されます。

失敗モードと境界条件

  • target_bitset_bits が 0 の場合、-1 を返します。
  • target_bitset_bits が非常に大きい場合、bloom_power は 32 で打ち切られます。

保守担当者向けメモ

  • 本関数はビットセットのサイズを 2 のべき乗に正規化するために使用されます。
  • 戻り値が 32 に達した後の挙動は、ループ条件により固定されます。
