r_shortv

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_porter.c:225

目的

Porter Stemming アルゴリズムにおいて、単語末尾が「子音-母音-子音」のパターン（かつ最後の文字が W, X, Y 以外）であるかを確認する。

シグネチャまたは形状

static int r_shortv(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（ステミング環境の状態を保持する構造体）
  • **出力**: int 型。条件を満たす場合は 1 を返し、満たさない場合は 0 を返す。

前提条件と不変条件

  • SN_env 構造体が初期化されており、有効な文字列バッファとカーソル位置を保持していること。
  • 内部的に g_v_WXY（母音および W, X, Y）および g_v（母音）のグループ定義が利用可能であること。

アルゴリズムまたは振る舞い

以下の順序で条件判定を行い、すべてに適合した場合に 1 を返す。 1. out_grouping_b_U(z, g_v_WXY, 89, 121, 0): 現在位置の文字が母音または W, X, Y ではないことを確認する。 2. in_grouping_b_U(z, g_v, 97, 121, 0): 現在位置の文字が母音であることを確認する。 3. out_grouping_b_U(z, g_v, 97, 121, 0): 現在位置の文字が母音ではない（子音である）ことを確認する。 各ステップで条件を満たさない場合は直ちに 0 を返して終了する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/snowball/libstemmer/stem_ISO_8859_1_english.c
  • src/backend/snowball/libstemmer/stem_ISO_8859_1_porter.c
  • src/backend/snowball/libstemmer/stem_UTF_8_english.c
  • src/backend/snowball/libstemmer/stem_UTF_8_porter.c
  • **呼び出し先**:
  • out_grouping_b_U（Snowball ランタイム関数）
  • in_grouping_b_U（Snowball ランタイム関数）

関連データ構造

  • SN_env: ステミング処理の環境状態を保持する構造体。
  • g_v, g_v_WXY: 母音および特定の文字グループを定義する静的配列（Snowball コンパイラにより生成）。

失敗モードと境界条件

  • 文字列が短すぎて判定に必要な 3 文字分を遡れない場合、out_grouping_b_U 等のランタイム関数が失敗し、本関数も 0 を返す。
