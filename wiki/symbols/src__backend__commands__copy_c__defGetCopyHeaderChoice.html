defGetCopyHeaderChoice

定義

  • 種別: function
  • 定義箇所: src/backend/commands/copy.c:368

目的

COPY コマンドの HEADER オプションに指定された値を解析し、内部的な CopyHeaderChoice 列挙型へ変換する。

シグネチャまたは形状

CopyHeaderChoice defGetCopyHeaderChoice(DefElem *def, bool is_from)

入力と出力

  • **入力**:
  • def: 解析対象の DefElem 構造体。
  • is_from: COPY FROM の場合は true、COPY TO の場合は false。
  • **出力**:
  • COPY_HEADER_TRUE, COPY_HEADER_FALSE, COPY_HEADER_MATCH のいずれかを返す。

前提条件と不変条件

  • def->arg が NULL の場合、デフォルトで COPY_HEADER_TRUE とみなされる。
  • match は COPY FROM でのみ許可される。

アルゴリズムまたは振る舞い

1. def->arg が NULL なら COPY_HEADER_TRUE を返す。 2. def->arg が整数の場合、0 なら COPY_HEADER_FALSE、1 なら COPY_HEADER_TRUE を返す。 3. それ以外（文字列等）の場合、true, on は COPY_HEADER_TRUE、false, off は COPY_HEADER_FALSE に対応させる。 4. match が指定された場合、is_from が false（COPY TO）であればエラーを報告し、そうでなければ COPY_HEADER_MATCH を返す。 5. 上記のいずれにも該当しない場合、構文エラーとして ereport(ERROR) を発行する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/commands/copy.c 内の ProcessCopyOptions 関数から呼び出される。
  • **呼び出し先**: nodeTag, intVal, defGetString, pg_strcasecmp, ereport 等のユーティリティ関数。

関連データ構造

  • DefElem: COPY コマンドのオプション要素を保持する構造体。
  • CopyHeaderChoice: HEADER オプションの状態を表す列挙型。

失敗モードと境界条件

  • COPY TO で HEADER 'match' を指定した場合、ERRCODE_FEATURE_NOT_SUPPORTED でエラーとなる。
  • 認識できない値や不正な型が指定された場合、ERRCODE_SYNTAX_ERROR でエラーとなる。

保守担当者向けメモ

  • 本関数が受け入れる文字列のセットは、文法定義（opt_boolean_or_string）と一致させる必要がある。
  • 新たなヘッダー選択肢を追加する場合は、CopyHeaderChoice 列挙型の定義と本関数の switch/if 文の両方を更新すること。
