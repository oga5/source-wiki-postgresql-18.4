gbt_macad8ge

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_macaddr8.c:34

目的

macaddr8 型の2つの値に対して「以上（greater than or equal to）」の比較を行い、その結果を真偽値で返す。

シグネチャまたは形状

bool gbt_macad8ge(const void *a, const void *b, FmgrInfo *flinfo)

入力と出力

  • **入力**:
  • a: 比較対象の macaddr8 値へのポインタ。
  • b: 比較対象の macaddr8 値へのポインタ。
  • flinfo: 関数呼び出し情報（DirectFunctionCall2 で使用）。
  • **出力**:
  • bool: a >= b であれば true、そうでなければ false。

前提条件と不変条件

  • 入力ポインタ a および b は、有効な macaddr8 型のデータ構造を指している必要がある。
  • flinfo は DirectFunctionCall2 が要求する形式で提供される必要がある。

アルゴリズムまたは振る舞い

1. PointerGetDatum を使用して、入力ポインタ a および b を Datum 型に変換する。 2. DirectFunctionCall2 を介して、PostgreSQL のネイティブ関数 macaddr8_ge を呼び出す。 3. DatumGetBool を使用して、戻り値を bool 型に変換して返す。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/btree_gist/btree_macaddr8.c 内の静的変数 tinfo の初期化において参照されている。
  • **呼び出し先**: macaddr8_ge（PostgreSQL ネイティブの比較関数）。

関連データ構造

  • macaddr8: PostgreSQL のネットワークアドレス型。

失敗モードと境界条件

  • 入力ポインタが NULL の場合、DirectFunctionCall2 の挙動に依存する（通常はセグメンテーション違反等の未定義動作となる）。

保守担当者向けメモ

  • 本関数は btree_gist モジュールにおける macaddr8 型の比較アダプターの一部である。
  • 比較ロジックは完全に macaddr8_ge に委譲されているため、比較の定義自体を変更する場合は macaddr8 型の定義を確認すること。
