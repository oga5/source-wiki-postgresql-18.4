radius_packet

定義

  • **種別**: struct
  • **定義箇所**: src/backend/libpq/auth.c:2791

目的

RADIUS認証プロトコルにおけるパケット構造を保持するためのデータ構造です。

シグネチャまたは形状

typedef struct
{
    uint8       code;
    uint8       id;
    uint16      length;
    uint8       vector[RADIUS_VECTOR_LENGTH];
    char        pad[RADIUS_BUFFER_SIZE - RADIUS_VECTOR_LENGTH];
} radius_packet;

入力と出力

本構造体はデータ保持用であり、関数としての入出力は持ちません。

前提条件と不変条件

  • offsetof(radius_packet, vector) が 4 であることがアサーションにより保証されています。

アルゴリズムまたは振る舞い

RADIUSパケットの送受信時に、送信バッファおよび受信バッファとしてメモリ領域を確保し、各フィールドに値を格納または読み出すために使用されます。

呼び出し元と呼び出し先

  • **呼び出し元（参照箇所）**:
  • src/backend/libpq/auth.c: radius_add_attribute 関数が引数として受け取ります。
  • src/backend/libpq/auth.c: ローカル変数 radius_send_pack および radius_recv_pack として定義されています。
  • **呼び出し先**:
  • 情報なし。

関連データ構造

  • radius_attribute: RADIUSパケット内の属性情報を扱う構造体。

失敗モードと境界条件

  • pad フィールドは RADIUS_BUFFER_SIZE に基づいて定義されており、パケットサイズがこのバッファサイズを超過する場合の挙動については、本構造体定義単体では規定されていません。

保守担当者向けメモ

  • 本構造体は src/backend/libpq/auth.c 内のRADIUS認証処理専用です。
  • vector フィールドのオフセットが固定されているため、構造体のメンバ順序を変更する際は注意が必要です。
