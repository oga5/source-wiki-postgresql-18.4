build_EvalXFuncInt

定義

  • 種別: function
  • 定義箇所: src/backend/jit/llvm/llvmjit_expr.c:3059

目的

PostgreSQL の式評価において、汎用的な式評価関数を呼び出すための LLVM IR 命令を生成する。

シグネチャまたは形状

static LLVMValueRef build_EvalXFuncInt(
    LLVMBuilderRef b, 
    LLVMModuleRef mod, 
    const char *funcname,
    LLVMValueRef v_state, 
    ExprEvalStep *op,
    int nargs, 
    LLVMValueRef *v_args
)

入力と出力

  • **入力**:
  • b: LLVM ビルダー。
  • mod: LLVM モジュール。
  • funcname: 呼び出す関数名。
  • v_state: ExprState への LLVM 値。
  • op: 現在の ExprEvalStep へのポインタ。
  • nargs: 引数の数。
  • v_args: 引数となる LLVM 値の配列。
  • **出力**:
  • 呼び出し結果を表す LLVMValueRef。

前提条件と不変条件

  • v_fn（funcname から取得した関数）が期待するパラメータ数（nargs + 2）と、実際に渡された引数数が一致している必要がある。

アルゴリズムまたは振る舞い

1. llvm_pg_func を用いて指定された funcname に対応する関数を取得する。 2. LLVMCountParams を使用して、関数が期待する引数の数と nargs + 2 を比較し、不一致の場合は elog(ERROR) を発生させる。 3. palloc を用いて、v_state、op へのポインタ、および v_args を格納するためのパラメータ配列を確保する。 4. l_call を使用して、構築したパラメータ配列を用いて関数呼び出しの IR を生成する。 5. 確保したメモリを pfree で解放し、呼び出し結果を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/jit/llvm/llvmjit_expr.c 内の式評価ステップ生成処理（行60, 71付近）。
  • **呼び出し先**: llvm_pg_func, LLVMCountParams, palloc, l_ptr_const, l_call, LLVMGetFunctionType, pfree。

関連データ構造

  • ExprEvalStep: 式評価の各ステップを定義する構造体。
  • LLVMValueRef, LLVMBuilderRef, LLVMModuleRef: LLVM API の主要型。

失敗モードと境界条件

  • LLVMCountParams(v_fn) != (nargs + 2) の場合、パラメータ数不一致としてエラー終了する。

保守担当者向けメモ

  • 本関数は l_ptr_const を使用して op を定数ポインタとして IR に埋め込んでいる。
  • l_ 接頭辞を持つ関数は jit/llvmjit_emit.h で定義されたラッパーであるため、詳細な IR 生成ロジックを確認する場合はそちらを参照すること。
