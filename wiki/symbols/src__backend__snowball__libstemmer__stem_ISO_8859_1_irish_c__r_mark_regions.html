r_mark_regions

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_irish.c:223

目的

アイルランド語のステミング処理において、単語内の母音と子音の配置に基づき、接尾辞削除の対象となる領域（R1, R2, RV）の境界インデックスを計算し、SN_env 構造体に格納する。

シグネチャまたは形状

static int r_mark_regions(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z（ステミング環境の状態を保持する構造体）
  • **出力**: 常に 1 を返す。副作用として z->I[0]（R2の開始位置）、z->I[1]（R1の開始位置）、z->I[2]（RVの開始位置）が更新される。

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • g_v（母音グループ）が定義されていること。
  • 処理対象の文字列は ISO-8859-1 エンコーディングであること。

アルゴリズムまたは振る舞い

1. 初期状態として、領域の開始位置（I[0], I[1], I[2]）を文字列の末尾（z->l）に設定する。 2. 母音と子音の切り替わりを検出して領域を決定する：

  • out_grouping（子音）と in_grouping（母音）を交互に適用し、単語内の特定の構造を探索する。
  • 最初の母音の後の子音の開始位置を I[2]（RV）に設定する。
  • 最初の母音の後の子音の後の母音の開始位置を I[1]（R1）に設定する。
  • さらに続く子音の後の母音の開始位置を I[0]（R2）に設定する。

    • 3. 探索に失敗した場合は、現在のカーソル位置を初期値（c1）に戻す。

呼び出し元と呼び出し先

  • **呼び出し元**: irish_ISO_8859_1_stem 関数から呼び出される（直接の呼び出し関係はソースコードの制御フローより推論）。
  • **呼び出し先**: out_grouping, in_grouping（Snowball ランタイムのユーティリティ関数）。

関連データ構造

  • struct SN_env: ステミングの実行環境。I 配列（整数型作業領域）、c（カーソル位置）、l（文字列長）を保持する。

失敗モードと境界条件

  • 文字列が空、または母音・子音のパターンが定義された領域条件を満たさない場合、領域インデックスは更新されず、初期値（文字列の末尾）のまま維持される。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードである。ロジックの変更が必要な場合は、直接編集せず元の .sbl ファイルを修正すること。
  • I[0], I[1], I[2] のインデックスは、後続の接尾辞削除ルールにおいて r_R1, r_R2, r_RV として参照される。
