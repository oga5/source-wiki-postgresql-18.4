pg_isblank

定義

  • 種別: function
  • 定義箇所: src/backend/libpq/hba.c:145

目的

与えられた文字が空白文字（スペース、タブ、または復帰）であるかを判定します。

シグネチャまたは形状

bool pg_isblank(const char c)

入力と出力

  • **入力**: c (判定対象の文字)
  • **出力**: c がスペース (' ')、タブ ('\t')、または復帰 ('\r') のいずれかであれば true を返し、それ以外であれば false を返します。

前提条件と不変条件

  • 入力文字 c は char 型です。
  • 本関数は副作用を持ちません。

アルゴリズムまたは振る舞い

入力された文字 c に対して、以下の論理和条件を評価します。 c == ' ' || c == '\t' || c == '\r'

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/libpq/auth.c: 認証関連の文字列解析処理において複数回呼び出されています。
  • src/backend/libpq/hba.c: 設定ファイルのトークン化および解析処理において呼び出されています。
  • **呼び出し先**:
  • なし（組み込みの比較演算子のみを使用）。

関連データ構造

  • なし

失敗モードと境界条件

  • 入力値が '\0'（ヌル文字）の場合、本関数は false を返します。
  • 改行文字 ('\n') は本関数の判定対象（空白文字）には含まれません。

保守担当者向けメモ

  • 本関数は isblank() などの標準ライブラリ関数とは異なり、'\r' を明示的に含める一方で '\n' を除外する、PostgreSQLのHBA設定ファイル解析に特化した定義となっています。変更時には解析ロジックへの影響を考慮してください。
