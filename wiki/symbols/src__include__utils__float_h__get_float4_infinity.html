get_float4_infinity

定義

  • 種別: function
  • 定義箇所: src/include/utils/float.h:73

目的

プラットフォーム間で一貫した float4 型の無限大値を取得する。

シグネチャまたは形状

static inline float4 get_float4_infinity(void)

入力と出力

  • 入力: なし
  • 出力: float4 型の無限大値

前提条件と不変条件

  • C99準拠の環境では INFINITY マクロを使用する。
  • INFINITY が利用できない環境では、HUGE_VAL の積によるオーバーフローを利用して無限大を生成する。

アルゴリズムまたは振る舞い

1. INFINITY マクロが定義されている場合、それを float4 にキャストして返す。 2. INFINITY が定義されていない場合、HUGE_VAL * HUGE_VAL を計算し、その結果を float4 にキャストして返す。

  • この手法は、プラットフォーム依存の HUGE_VAL の挙動（最大値か無限大か）に関わらず、強制的なオーバーフローにより無限大を得ることを意図している。
  • Microsoft Visual C++ (_MSC_VER) 環境では、計算に伴う警告を抑制する設定が含まれる。

  • 呼び出し先:

関連データ構造

  • float4: PostgreSQLにおける単精度浮動小数点型。

失敗モードと境界条件

  • プラットフォームの浮動小数点実装が IEEE 754 に準拠していない場合、無限大の生成が期待通りに動作しない可能性がある。

保守担当者向けメモ

  • 本関数はインライン関数として定義されており、ヘッダーファイル内で完結している。
  • float8 版の get_float8_infinity も同様の目的で存在するため、修正時は整合性に注意すること。
