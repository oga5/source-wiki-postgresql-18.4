uuid_ns_url

定義

  • 種別: function
  • 定義箇所: contrib/uuid-ossp/uuid-ossp.c:447

目的

名前空間ベースの UUID 生成（v3/v5）において、URL 名前空間を識別するための定数 UUID 値を返します。

シグネチャまたは形状

Datum uuid_ns_url(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS（PostgreSQL の関数呼び出し規約に基づく引数）。
  • **出力**: Datum 型として、URL 名前空間を表す UUID 値を返します。

前提条件と不変条件

  • コンパイル時の環境設定（HAVE_UUID_OSSP の有無）により、内部的な取得経路が異なります。
  • RFC 4122 で定義された URL 名前空間の UUID 値（6ba7b811-9dad-11d1-80b4-00c04fd430c8）を保持または生成します。

アルゴリズムまたは振る舞い

1. HAVE_UUID_OSSP が定義されている場合: special_uuid_value("ns:URL") を呼び出し、OSSP ライブラリ経由で値を取得します。 2. それ以外の場合: uuid_generate_internal を呼び出し、ハードコードされた URL 名前空間の UUID 文字列を内部形式に変換して返します。

呼び出し元と呼び出し先

  • **呼び出し元**: PG_FUNCTION_INFO_V1(uuid_ns_url) マクロにより、SQL インターフェースから呼び出し可能として登録されています。
  • **呼び出し先**:
  • special_uuid_value（OSSP 使用時）
  • uuid_generate_internal（OSSP 非使用時）

関連データ構造

  • Datum: PostgreSQL の汎用データ型。

失敗モードと境界条件

  • 外部ライブラリ（OSSP）のリンク状況や、内部的な UUID 生成関数の実行結果に依存します。具体的なエラーハンドリングの詳細は、呼び出し先の special_uuid_value または uuid_generate_internal の実装に依存します。

保守担当者向けメモ

  • 本関数は名前空間の定数値を返すためのラッパーです。RFC 4122 に準拠した固定値を使用しているため、名前空間の定義を変更する必要がない限り、ロジックの修正は不要です。
