setFmtEncoding

定義

  • 種別: function
  • 定義箇所: src/fe_utils/string_utils.c:68

目的

識別子をクォートする関数（fmtId 等）が使用する文字エンコーディングをグローバルに設定します。

シグネチャまたは形状

void setFmtEncoding(int encoding)

入力と出力

  • **入力**: encoding (int型) - 設定するエンコーディングID。
  • **出力**: なし（void）。

前提条件と不変条件

  • 本関数で設定されたエンコーディングは、後続の fmtId などの識別子処理関数において、マルチバイト文字の適切な処理のために参照されます。
  • fmtId などの関数を呼び出す前に、適切なエンコーディングが設定されている必要があります。

アルゴリズムまたは振る舞い

引数として受け取った encoding を、モジュール内のグローバル変数 fmtIdEncoding に代入します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/bin/pg_dump/pg_backup_archiver.c
  • src/bin/pg_dump/pg_dump.c
  • src/bin/pg_dump/pg_dumpall.c
  • src/bin/psql/command.c
  • src/bin/scripts/createdb.c
  • src/bin/scripts/createuser.c
  • src/test/modules/test_escape/test_escape.c
  • **呼び出し先**: なし。

関連データ構造

  • fmtIdEncoding: 本関数によって更新される内部グローバル変数。

失敗モードと境界条件

  • 特筆すべき失敗モードはありません。無効なエンコーディングIDが渡された場合の動作は、呼び出し側の責任となります。

保守担当者向けメモ

  • 本関数はスレッドセーフではありません。マルチスレッド環境で利用する場合、グローバル変数 fmtIdEncoding へのアクセス競合に注意してください。
  • 可能な限り、エンコーディングを引数として直接受け取る fmtIdEnc() などの関数を使用することが推奨されます。
