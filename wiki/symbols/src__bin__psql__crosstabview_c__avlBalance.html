avlBalance

定義

  • 種別: function
  • 定義箇所: src/bin/psql/crosstabview.c:494

目的

AVL木のノードにおける左右の子ノードの高さの差（バランス係数）を計算します。

シグネチャまたは形状

static int avlBalance(avl_node *n);

入力と出力

  • **入力**: n (バランス係数を算出する対象の avl_node へのポインタ)
  • **出力**: 左の子ノードの高さから右の子ノードの高さを引いた整数値を返します。

前提条件と不変条件

  • 引数 n は有効な avl_node を指している必要があります。
  • n->children[0] および n->children[1] が存在することを前提としています（NULLチェックは関数内で行われていません）。

アルゴリズムまたは振る舞い

1. ノード n の左子ノードの height から、右子ノードの height を減算します。 2. 計算結果をそのまま返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/bin/psql/crosstabview.c:495
  • src/bin/psql/crosstabview.c:509
  • src/bin/psql/crosstabview.c:515
  • **呼び出し先**:
  • 本関数は直接他の関数を呼び出しません。ノード構造体 avl_node のメンバにアクセスします。

関連データ構造

  • avl_node: AVL木の各ノードを構成する構造体。height メンバおよび children 配列を持ちます。

失敗モードと境界条件

  • n が NULL の場合、あるいは n->children[0] や n->children[1] が NULL の場合、セグメンテーション違反が発生する可能性があります。

保守担当者向けメモ

  • 本関数は AVL 木の再平衡化処理の一部として使用されます。
  • 戻り値が正であれば左側が、負であれば右側が重いことを示します。
