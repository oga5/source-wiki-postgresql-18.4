makeFloat

定義

  • 種別: function
  • 定義箇所: src/backend/nodes/value.c:36

目的

数値文字列を保持する Float ノードを生成し、初期化するファクトリ関数です。

シグネチャまたは形状

Float *makeFloat(char *numericStr)

入力と出力

  • **入力**: numericStr (数値表現を含む文字列へのポインタ)
  • **出力**: 初期化された Float 構造体へのポインタ

前提条件と不変条件

  • 引数 numericStr は、呼び出し元によって palloc されたメモリ領域を指している必要があります。
  • 生成されたノードは、numericStr の所有権を保持します。

アルゴリズムまたは振る舞い

1. makeNode(Float) マクロを使用して、PostgreSQL のメモリコンテキスト上に Float 構造体のメモリを確保します。 2. 引数として受け取った numericStr を、確保した構造体の fval メンバに代入します。 3. 初期化済みのノードポインタを返却します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/catalog/objectaddress.c
  • src/backend/commands/sequence.c
  • src/backend/commands/tsearchcmds.c
  • src/backend/nodes/read.c
  • **呼び出し先**:
  • makeNode (マクロ)

関連データ構造

  • Float: 数値文字列を保持するためのノード構造体（src/include/nodes/value.h で定義）。

失敗モードと境界条件

  • メモリ不足時には makeNode マクロの内部実装（palloc）によりエラーが発生します。
  • numericStr が NULL の場合、fval に NULL が代入されます。

保守担当者向けメモ

  • Float ノードは内部的に double 型ではなく文字列として値を保持します。これはパーサー段階での精度保持を目的としています。
  • 渡された文字列のメモリ管理はノードの寿命と同期させる必要があります。
