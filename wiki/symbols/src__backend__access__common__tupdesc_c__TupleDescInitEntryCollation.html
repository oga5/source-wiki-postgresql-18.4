TupleDescInitEntryCollation

定義

  • 種別: function
  • 定義箇所: src/backend/access/common/tupdesc.c:1025

目的

タプル記述子（TupleDesc）内の指定された属性（列）に対して、照合順序（Collation）IDを割り当てる。

シグネチャまたは形状

void TupleDescInitEntryCollation(TupleDesc desc, AttrNumber attributeNumber, Oid collationid)

入力と出力

  • desc (TupleDesc): 対象となるタプル記述子。
  • attributeNumber (AttrNumber): 設定対象の属性番号（1始まり）。
  • collationid (Oid): 設定する照合順序のOID。
  • 戻り値: なし（副作用として desc の内容を更新）。

前提条件と不変条件

  • desc が有効なポインタであること。
  • attributeNumber が 1 以上かつ desc->natts（属性総数）以下であること。

アルゴリズムまたは振る舞い

1. desc が有効か、attributeNumber が範囲内かをアサーションで検証する。 2. TupleDescAttr マクロを使用して、指定された属性インデックス（attributeNumber - 1）に対応する属性定義構造体にアクセスする。 3. 該当属性の attcollation フィールドに collationid を代入する。

関連データ構造

  • TupleDesc: タプル記述子。
  • FormData_pg_attribute: 属性定義を保持する構造体（TupleDescAttr を通じて参照）。

失敗モードと境界条件

  • attributeNumber が範囲外の場合、アサーション違反によりプロセスが異常終了する（デバッグビルド時）。

保守担当者向けメモ

  • 本関数は属性の照合順序のみを更新する。他の属性メタデータを変更した場合は、必要に応じて populate_compact_attribute を呼び出す必要がある点に注意すること。
