gbt_oid_dist

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_oid.c:71

目的

2つの Oid 値の間の距離を計算し、GiSTインデックスにおける近傍検索やペナルティ計算の基礎となる数値（float8）を返します。

シグネチャまたは形状

float8 gbt_oid_dist(const void *a, const void *b, FmgrInfo *flinfo)

入力と出力

  • **入力**:
  • a, b: 比較対象の Oid 値へのポインタ。
  • flinfo: 関数管理情報（本関数内では未使用）。
  • **出力**:
  • 2つの Oid 値の絶対差分を float8 型として返します。

前提条件と不変条件

  • 入力ポインタ a および b は、有効な Oid 値を指している必要があります。
  • Oid は符号なし整数として扱われます。

アルゴリズムまたは振る舞い

1. ポインタ a および b が指す値を Oid 型としてキャストし、それぞれ aa, bb に格納します。 2. aa < bb の場合、bb - aa を計算します。 3. それ以外の場合、aa - bb を計算します。 4. 計算結果を float8 型に変換して返します。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/btree_gist/btree_oid.c 内のGiSTサポート関数群から参照されています（具体的な関数名はソースコードの構造上、本関数を直接呼び出すGiSTのインターフェース関数群を指します）。
  • **呼び出し先**: 本関数は標準的な算術演算のみを行い、他の関数を呼び出しません。

関連データ構造

  • Oid: PostgreSQLのオブジェクト識別子型。

失敗モードと境界条件

  • Oid 値の差分が float8 の表現範囲を超えることはありません。
  • 入力ポインタが NULL の場合の挙動は定義されていません。

保守担当者向けメモ

  • 本関数は Oid 型の単純な距離計算を行うユーティリティです。より複雑なGiSTインデックスのロジックについては、contrib/btree_gist/btree_utils_num.c を参照してください。
