pg_atomic_unlocked_test_flag_impl

定義

  • 種別: function
  • 定義箇所: src/include/port/atomics/generic-gcc.h:130

目的

pg_atomic_flag が現在ロックされていない（値が 0 である）かどうかを、原子操作を伴わずに確認します。

シグネチャまたは形状

static inline bool pg_atomic_unlocked_test_flag_impl(volatile pg_atomic_flag *ptr)

入力と出力

  • **入力**: ptr - テスト対象の pg_atomic_flag 構造体へのポインタ。
  • **出力**: ptr->value が 0 であれば true、それ以外であれば false を返します。

前提条件と不変条件

  • ptr は有効なメモリ領域を指している必要があります。
  • 本関数は原子的な読み込みを保証するものではなく、単なる値のチェックを行います。

アルゴリズムまたは振る舞い

1. 引数として受け取った ptr が指す構造体の value メンバにアクセスします。 2. その値が 0 であるかを評価し、結果をブール値として返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/include/port/atomics.h:198 からの呼び出しが確認されています。
  • **呼び出し先**: 本関数は組み込み関数や他の関数を呼び出しません。

関連データ構造

  • pg_atomic_flag: 原子操作対象のフラグを保持する構造体。

失敗モードと境界条件

  • ptr が NULL の場合、未定義の動作を引き起こします。
  • 本関数は非原子的なチェックであるため、呼び出し直後に他のスレッドによってフラグの状態が変更される可能性があります。

保守担当者向けメモ

  • 本関数は static inline として定義されており、ヘッダーファイル内で直接展開されます。
  • atomics.h を経由して使用されることを前提としています。
