encrypt_init

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/pgp-encrypt.c:157

目的

暗号化フィルタチェーンの初期化を行い、CFBモードでの暗号化コンテキストを準備する。

シグネチャまたは形状

static int encrypt_init(PushFilter *next, void *init_arg, void **priv_p)

入力と出力

  • **入力**:
  • next: 後続の PushFilter オブジェクト。
  • init_arg: PGP_Context 構造体へのポインタ。
  • priv_p: 初期化された EncStat 構造体を格納するための出力用ポインタ。
  • **出力**: 成功時は ENCBUF を返し、失敗時は負の整数（エラーコード）を返す。

前提条件と不変条件

  • init_arg は有効な PGP_Context を指している必要がある。
  • MDC（Modification Detection Code）が有効な場合、ストリームの先頭にバージョンバイト 0x01 が書き込まれる。

アルゴリズムまたは振る舞い

1. ctx->disable_mdc が 0 の場合、MDC 使用を示すバージョンバイト 0x01 を next フィルタへ書き込む。 2. pgp_cfb_create を呼び出し、ctx 内のアルゴリズムとセッション鍵を用いて PGP_CFB コンテキストを生成する。 3. EncStat 構造体をメモリ確保し、生成した ciph を格納する。 4. priv_p に EncStat へのポインタをセットし、ENCBUF を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: ソースコード内の直接的な呼び出し元は明示されていないが、フィルタチェーン構築時に使用される。
  • **呼び出し先**:
  • pushf_write: MDC 有効時にバージョンバイトを書き込む。
  • pgp_cfb_create: 暗号化コンテキストの生成。
  • palloc0: EncStat 構造体のメモリ確保。

関連データ構造

  • PushFilter: フィルタチェーンの構成要素。
  • PGP_Context: 暗号化設定を保持する構造体。
  • EncStat: 本フィルタのプライベート状態を保持する構造体（ciph を含む）。
  • PGP_CFB: CFB モードの暗号化状態。

失敗モードと境界条件

  • pushf_write が失敗した場合、そのエラーコードをそのまま返す。
  • pgp_cfb_create が失敗した場合、そのエラーコードをそのまま返す。
  • メモリ確保失敗時は palloc0 がエラーを発生させる可能性がある。

保守担当者向けメモ

  • 本関数は PushFilter インターフェースの一部として実装されている。
  • MDC の有無によってストリームの先頭バイトが変化するため、復号側との整合性に注意すること。
