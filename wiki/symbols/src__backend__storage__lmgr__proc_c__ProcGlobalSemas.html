ProcGlobalSemas

定義

  • 種別: function
  • 定義箇所: src/backend/storage/lmgr/proc.c:156

目的

システム全体で必要となるセマフォの総数を算出します。

シグネチャまたは形状

int ProcGlobalSemas(void)

入力と出力

  • **入力**: なし
  • **出力**: int 型。必要なセマフォの合計数。

前提条件と不変条件

  • MaxBackends（最大バックエンド数）および NUM_AUXILIARY_PROCS（補助プロセス数）が定義されている環境で呼び出される必要があります。

アルゴリズムまたは振る舞い

全バックエンドプロセス（autovacuumを含む）の数と、補助プロセスの数を加算した値を返します。 計算式: MaxBackends + NUM_AUXILIARY_PROCS

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/storage/ipc/ipci.c:95 にて、システム初期化時にセマフォ数を決定するために呼び出されます。
  • **呼び出し先**: なし。

関連データ構造

  • MaxBackends（グローバル設定値）
  • NUM_AUXILIARY_PROCS（定数）

失敗モードと境界条件

  • 戻り値は常に正の整数となります。
  • 呼び出し元でのセマフォ割り当て処理において、この値がシステムリソース制限を超えないことが前提となります。

保守担当者向けメモ

  • 本関数はセマフォの総数を定義する単一のソースです。プロセスモデルの変更や補助プロセスの増減を行う際は、本関数の計算式が適切か確認してください。
