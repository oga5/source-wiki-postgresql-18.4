l_int8_const

定義

  • 種別: function
  • 定義箇所: src/include/jit/llvmjit_emit.h:47

目的

LLVM IR 生成において、指定された 8 ビット整数値を表す定数ノード（LLVMValueRef）を生成するヘルパー関数です。

シグネチャまたは形状

static inline LLVMValueRef l_int8_const(LLVMContextRef lc, int8 i)

入力と出力

  • **入力**:
  • lc (LLVMContextRef): 定数を作成する対象の LLVM コンテキスト。
  • i (int8): 生成する定数値。
  • **出力**:
  • LLVMValueRef: LLVM IR 上の 8 ビット整数定数。

前提条件と不変条件

  • lc は有効な LLVMContextRef である必要があります。
  • 本関数はインライン関数として定義されており、呼び出し側のコンテキストで展開されます。

アルゴリズムまたは振る舞い

1. LLVMInt8TypeInContext(lc) を呼び出し、指定されたコンテキストにおける 8 ビット整数型を取得します。 2. LLVMConstInt を使用して、値 i を持つ定数を作成します。この際、符号なし（false）として生成されます。 3. 生成された LLVMValueRef を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/jit/llvm/llvmjit_deform.c (複数箇所)
  • src/backend/jit/llvm/llvmjit_expr.c:1749
  • **呼び出し先**:
  • LLVMInt8TypeInContext (LLVM C API)
  • LLVMConstInt (LLVM C API)

関連データ構造

  • LLVMValueRef: LLVM IR の値を表す型。
  • LLVMContextRef: LLVM のコンテキストを管理する型。

失敗モードと境界条件

  • LLVM C API の内部的な制約により、コンテキストが無効な場合は未定義の動作を引き起こす可能性があります。

保守担当者向けメモ

  • 本関数は USE_LLVM が定義されている環境でのみ利用可能です。
  • LLVM C API のラッパーであるため、API のバージョンアップ時に変更が必要となる場合があります。
  • 符号付き整数として定数を生成する必要がある場合は、別の関数を検討するか、本関数の引数処理を確認してください。
