generate_series_timestamptz_fctx

定義

  • 種別: struct
  • 定義箇所: src/backend/utils/adt/timestamp.c:67

目的

generate_series 関数において、タイムゾーン付きタイムスタンプ（timestamptz）の生成処理中に状態を保持するためのコンテキスト構造体です。

シグネチャまたは形状

typedef struct {
    TimestampTz current;    /* 現在の生成値 */
    TimestampTz finish;     /* 終了値 */
    Interval    step;       /* ステップ間隔 */
    int         step_sign;  /* ステップの符号 */
    pg_tz      *attimezone; /* タイムゾーン情報 */
} generate_series_timestamptz_fctx;

入力と出力

本構造体は関数内部のメモリ管理（palloc）を通じて生成され、generate_series の実行状態を維持するために使用されます。

前提条件と不変条件

  • current および finish は TimestampTz 型（マイクロ秒単位の int64）で保持されます。
  • step_sign は step の方向性を決定するために使用されます。

アルゴリズムまたは振る舞い

generate_series 実行時に palloc でメモリ確保され、反復処理の各ステップで current を更新しながら値を生成します。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/utils/adt/timestamp.c 内の generate_series 関連関数（行6759, 6780, 6781付近で参照およびメモリ確保を確認）。
  • 呼び出し先: 情報なし。

関連データ構造

  • TimestampTz: タイムゾーン付きタイムスタンプの内部表現。
  • Interval: 時間間隔を保持する構造体。
  • pg_tz: タイムゾーン情報を保持する構造体。

失敗モードと境界条件

  • メモリ不足時に palloc が失敗する可能性があります。
  • TimestampTz の範囲外の値や、infinity（無限大）の扱いについては、timestamp.c の算術演算ルールに従います。

保守担当者向けメモ

  • 本構造体は generate_series の反復状態を保持する専用のコンテキストです。
  • タイムゾーン変換を伴うため、attimezone フィールドのライフサイクル管理に注意してください。
