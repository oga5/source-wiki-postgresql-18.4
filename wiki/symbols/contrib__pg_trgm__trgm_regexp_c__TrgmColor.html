TrgmColor

定義

  • 種別: typedef int
  • 定義箇所: contrib/pg_trgm/trgm_regexp.c:285

目的

正規表現から変換されたグラフ構造において、状態遷移のラベルやトリグラムの分類を識別するための整数型識別子です。

シグネチャまたは形状

int 型のエイリアスです。

入力と出力

  • 入力: グラフ走査やトリグラム選択処理において、特定の状態や色を指し示す値として使用されます。
  • 出力: printTrgmColor 関数を通じて、デバッグ用の文字列バッファに書き出されます。

前提条件と不変条件

  • グラフ内の特定のノードやエッジに関連付けられた識別子として機能します。
  • TrgmStateKey 構造体や TrgmColorInfo 構造体と組み合わせて、正規表現の展開グラフを構成します。

アルゴリズムまたは振る舞い

  • グラフ構築時、正規表現の NFA から抽出されたトリグラムや状態遷移を分類するために使用されます。
  • validArcLabel 関数において、特定の TrgmStateKey に対してこの色が有効なラベルであるかを判定する際に利用されます。

呼び出し元と呼び出し先

  • 呼び出し元: contrib/pg_trgm/trgm_regexp.c 内のグラフ構築、走査、およびデバッグ出力処理。
  • 呼び出し先: printTrgmColor 関数（デバッグ出力用）、validArcLabel 関数（ラベル検証用）。

関連データ構造

  • TrgmStateKey: 状態遷移のキーとして TrgmColor を保持します。
  • TrgmColorInfo: TrgmColor に関連するメタ情報を管理します。
  • TrgmNFA: 正規表現解析の過程で TrgmColor を利用してグラフを構築します。

失敗モードと境界条件

  • 整数型であるため、定義された範囲外の値が渡された場合の動作は、使用されるコンテキスト（配列インデックス等）に依存します。

保守担当者向けメモ

  • 本シンボルは int の別名であり、主にグラフのノードやエッジのラベル付けに使用されます。
  • TRGM_REGEXP_DEBUG が有効な場合、printTrgmColor を通じて変換過程の可視化が行われます。
