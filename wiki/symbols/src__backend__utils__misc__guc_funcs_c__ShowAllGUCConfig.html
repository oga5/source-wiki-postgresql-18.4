ShowAllGUCConfig

定義

  • 種別: function
  • 定義箇所: src/backend/utils/misc/guc_funcs.c:455

目的

システム内の全GUC（Grand Unified Configuration）パラメータを収集し、名前、現在の設定値、および説明文を DestReceiver を介して出力する。

シグネチャまたは形状

static void ShowAllGUCConfig(DestReceiver *dest)

入力と出力

  • **入力**: DestReceiver *dest（タプルを受け取るための出力先インターフェース）
  • **出力**: なし（副作用として dest を通じて (name, setting, description) の3列からなるタプル群を送信する）

前提条件と不変条件

  • get_guc_variables により取得されるパラメータリストはソート済みであること。
  • 出力されるタプルは TEXT 型の3列構成であること。
  • ConfigOptionIsVisible による可視性チェックを通過したパラメータのみが出力されること。

アルゴリズムまたは振る舞い

1. get_guc_variables を呼び出し、全GUC変数のリストを取得する。 2. 3列（name, setting, description）の TupleDesc を作成し、begin_tup_output_tupdesc で出力準備を行う。 3. 各パラメータを順次走査し、以下の条件でフィルタリングする：

  • GUC_NO_SHOW_ALL フラグが立っているものは除外する。
  • ConfigOptionIsVisible で現在のユーザーに権限がないものは除外する。

    • 4. 各パラメータの値を cstring_to_text で変換し、do_tup_output で出力する。 5. 処理に使用したメモリ（Datum 等）を pfree で解放する。 6. 最後に end_tup_output を呼び出し、出力ストリームを閉じる。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/misc/guc_funcs.c 内の関数（GetPGVariable 等からの参照が確認されている）。
  • **呼び出し先**:
  • get_guc_variables
  • CreateTemplateTupleDesc, TupleDescInitBuiltinEntry
  • begin_tup_output_tupdesc, do_tup_output, end_tup_output
  • ConfigOptionIsVisible
  • ShowGUCOption
  • cstring_to_text, pfree

関連データ構造

  • DestReceiver: タプル出力先。
  • config_generic: GUCパラメータの基本構造体。
  • TupleDesc: 出力タプルのスキーマ定義。

失敗モードと境界条件

  • パラメータ値が取得できない場合（ShowGUCOption が NULL を返す場合）、setting 列には NULL が設定される。
  • 説明文（short_desc）が存在しない場合、description 列には NULL が設定される。

保守担当者向けメモ

  • NUM_PG_SETTINGS_ATTS や GetConfigOptionValues との整合性に注意すること。
  • セキュリティ上の理由から、権限のないユーザーに対しては設定値のソース情報などが隠蔽される仕様となっている。
  • pfree によるメモリ解放がループ内で行われているため、新しい属性を追加する際はメモリリークに注意すること。
