decrypt_elgamal

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/pgp-pubdec.c:96

目的

ElGamal公開鍵暗号方式を用いて、PGPセッション鍵パケットから暗号化されたデータを復号します。

シグネチャまたは形状

static int decrypt_elgamal(PGP_PubKey *pk, PullFilter *pkt, PGP_MPI **m_p)

入力と出力

  • **入力**:
  • pk: 復号に使用する公開鍵情報（PGP_PubKey）。
  • pkt: 暗号化されたデータが格納された入力フィルタ（PullFilter）。
  • m_p: 復号結果の多倍長整数（PGP_MPI）を格納するためのポインタ。
  • **出力**:
  • 成功時は 0 を返し、m_p に復号されたMPIが格納されます。
  • 失敗時は負の整数（エラーコード）を返します。

前提条件と不変条件

  • pk->algo が PGP_PUB_ELG_ENCRYPT である必要があります。それ以外の場合は PXE_PGP_WRONG_KEY を返します。
  • pkt から読み込まれるデータは、ElGamal暗号の構成要素である2つのMPI（c1, c2）の順序に従っている必要があります。

アルゴリズムまたは振る舞い

1. 鍵アルゴリズムの検証を行い、不一致であれば即座に終了します。 2. pgp_mpi_read を用いて、入力フィルタから2つのMPI（c1, c2）を順次読み込みます。 3. 読み込みに成功した場合、pgp_elgamal_decrypt を呼び出して復号を実行し、結果を m_p に格納します。 4. 処理の終了時（成功・失敗を問わず）、一時的に確保した c1 および c2 のメモリを pgp_mpi_free で解放します。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/pgcrypto/pgp-pubdec.c 内の pgp_parse_pubenc_sesskey から呼び出されます。
  • **呼び出し先**:
  • pgp_mpi_read: MPI形式のデータ読み込み。
  • pgp_elgamal_decrypt: ElGamal復号の実行。
  • pgp_mpi_free: メモリ解放。

関連データ構造

  • PGP_PubKey: 公開鍵情報を保持する構造体。
  • PGP_MPI: 多倍長整数を扱うための構造体。
  • PullFilter: データ読み込み用のフィルタ構造体。

失敗モードと境界条件

  • pk->algo が ElGamal ではない場合: PXE_PGP_WRONG_KEY を返します。
  • MPIの読み込み（pgp_mpi_read）に失敗した場合: その時点でのエラーコードを返し、確保済みのメモリを解放して終了します。
  • 復号処理（pgp_elgamal_decrypt）が失敗した場合: そのエラーコードを返します。

保守担当者向けメモ

  • 本関数はPKCS#1 v1.5パディングに関連する復号フローの一部です。
  • c1 および c2 のメモリリークを防ぐため、goto out による終了処理が徹底されています。修正を加える際は、この解放フローを維持してください。
