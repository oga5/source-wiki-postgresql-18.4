JsonValueListIterator

定義

  • 種別: struct
  • 定義箇所: src/backend/utils/adt/jsonpath_exec.c:153

目的

JsonValueList に格納された要素を反復処理するためのイテレータ状態を保持します。

シグネチャまたは形状

typedef struct JsonValueListIterator
{
	JsonbValue *value;
	List	   *list;
	ListCell   *next;
} JsonValueListIterator;

入力と出力

  • 入力: JsonValueList 構造体（JsonValueListInitIterator を介して参照）。
  • 出力: 反復処理の過程で、現在の要素を指す JsonbValue *value が更新されます。

前提条件と不変条件

  • JsonValueListInitIterator によって初期化される必要があります。
  • JsonValueListNext を呼び出すことで、リスト内の次の要素へ進みます。

アルゴリズムまたは振る舞い

  • JsonValueList が保持する List 構造を走査します。
  • next フィールドを使用してリストの現在の位置を追跡し、value フィールドに現在の JsonbValue を格納します。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/utils/adt/jsonpath_exec.c 内の複数の関数（executeItem 等）で、マッチした要素のリストを走査するために使用されています。
  • 呼び出し先: JsonValueListInitIterator および JsonValueListNext 関数によって操作されます。

関連データ構造

  • JsonValueList: 反復対象となるリスト構造体。
  • JsonbValue: リスト内の各要素の型。
  • List / ListCell: PostgreSQL の標準的なリスト管理構造。

失敗モードと境界条件

  • リストが空の場合、または走査が終了した後の動作については、JsonValueListNext の実装に依存します。

保守担当者向けメモ

  • 本構造体は jsonpath_exec.c 内でのみ使用される内部的なイテレータです。
  • JsonValueList の実装変更に伴い、本構造体のフィールド構成も影響を受ける可能性があります。
