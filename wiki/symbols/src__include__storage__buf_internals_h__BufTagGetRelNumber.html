BufTagGetRelNumber

定義

  • 種別: function
  • 定義箇所: src/include/storage/buf_internals.h:115

目的

BufferTag 構造体からリレーション番号（RelFileNumber）を抽出して返します。

シグネチャまたは形状

static inline RelFileNumber BufTagGetRelNumber(const BufferTag *tag)

入力と出力

  • **入力**: const BufferTag *tag （対象のバッファタグへのポインタ）
  • **出力**: RelFileNumber （タグに含まれるリレーション番号）

前提条件と不変条件

  • 引数 tag は有効な BufferTag 構造体を指している必要があります。

アルゴリズムまたは振る舞い

  • tag が指す構造体の relNumber フィールドを直接返します。インライン関数として定義されており、オーバーヘッドを最小限に抑えたアクセサとして機能します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/pg_buffercache/pg_buffercache_pages.c
  • contrib/pg_prewarm/autoprewarm.c
  • src/backend/storage/buffer/bufmgr.c
  • src/include/storage/buf_internals.h（内部での参照）
  • **呼び出し先**:
  • 本関数は単純なフィールドアクセスを行うため、他の関数を呼び出しません。

関連データ構造

  • BufferTag: バッファを識別するためのタグ構造体。
  • RelFileNumber: リレーションを識別するための数値型。

失敗モードと境界条件

  • tag が NULL の場合に発生する動作については、本ソース excerpt からは定義されていません。

保守担当者向けメモ

  • 本関数は static inline で定義されているため、変更時はヘッダーファイルの再コンパイルが必要です。
  • BufferTag 構造体のレイアウト変更に伴い、本関数の実装も追従する必要があります。
