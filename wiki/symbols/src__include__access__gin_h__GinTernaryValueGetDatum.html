GinTernaryValueGetDatum

定義

  • 種別: static inline function
  • 定義箇所: src/include/access/gin.h:87

目的

GinTernaryValue 型の値を、PostgreSQL の内部データ表現である Datum 型に変換する。

シグネチャまたは形状

static inline Datum GinTernaryValueGetDatum(GinTernaryValue X)

入力と出力

  • **入力**: GinTernaryValue X (検索判定に使用される3値: TRUE/FALSE/MAYBE)
  • **出力**: Datum (変換後の値)

前提条件と不変条件

  • GinTernaryValue は bool と同じサイズであることが設計上保証されている。
  • 本関数は単なる型キャストとして実装されており、値の変換や検証は行わない。

アルゴリズムまたは振る舞い

引数として受け取った GinTernaryValue 型の値を、明示的に Datum 型へキャストして返却する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/include/access/gin.h:88 (直接参照)
  • PG_RETURN_GIN_TERNARY_VALUE(x) マクロ (定義箇所: src/include/access/gin.h:93)
  • **呼び出し先**:
  • 呼び出し先に関する情報は提供されていない。

関連データ構造

  • GinTernaryValue: GIN インデックスの検索条件判定に用いられる列挙型。
  • Datum: PostgreSQL における汎用的なデータ表現型。

失敗モードと境界条件

  • 本関数は単純なキャストであるため、関数内部での失敗モードは存在しない。
  • GinTernaryValue のサイズが bool と異なる環境で利用された場合、予期せぬ動作を引き起こす可能性がある。

保守担当者向けメモ

  • GinTernaryValue は bool とのキャストを前提としているため、型定義を変更する際は StaticAssertDecl 等による制約チェックが維持されているか確認すること。
