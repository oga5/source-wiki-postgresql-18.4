pgstat_get_io_context_name

定義

  • 種別: function
  • 定義箇所: src/backend/utils/activity/pgstat_io.c:239

目的

IOContext 列挙型の値を、対応する人間が判読可能な文字列（I/Oコンテキスト名）に変換します。

シグネチャまたは形状

const char *pgstat_get_io_context_name(IOContext io_context)

入力と出力

  • **入力**: IOContext io_context（変換対象のI/Oコンテキスト値）
  • **出力**: const char *（"bulkread", "bulkwrite", "init", "normal", "vacuum" のいずれかの文字列リテラル）

前提条件と不変条件

  • 入力される io_context は、定義済みの IOContext 列挙型に含まれる値である必要があります。

アルゴリズムまたは振る舞い

1. switch 文を用いて io_context の値を評価します。 2. 対応するケースに応じて、以下の文字列を返します。

  • IOCONTEXT_BULKREAD -> "bulkread"
  • IOCONTEXT_BULKWRITE -> "bulkwrite"
  • IOCONTEXT_INIT -> "init"
  • IOCONTEXT_NORMAL -> "normal"
  • IOCONTEXT_VACUUM -> "vacuum"

    • 3. 上記のいずれにも該当しない場合、elog(ERROR) を呼び出し、エラーを報告します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/pgstatfuncs.c:1464
  • **呼び出し先**:
  • elog (エラー報告用マクロ)

関連データ構造

  • IOContext (列挙型)

失敗モードと境界条件

  • 定義されていない IOContext 値が渡された場合、elog(ERROR) により処理が中断されます。

保守担当者向けメモ

  • 新しい IOContext が追加された場合は、本関数の switch 文にもケースを追加する必要があります。
