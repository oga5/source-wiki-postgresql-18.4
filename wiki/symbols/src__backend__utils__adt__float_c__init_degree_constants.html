init_degree_constants

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/float.c:2018

目的

三角関数の度数法計算で使用される定数を初期化します。コンパイラの最適化による精度低下を回避し、計算結果の一貫性を保つために使用されます。

シグネチャまたは形状

static void init_degree_constants(void)

入力と出力

  • 入力: なし
  • 出力: なし（副作用としてグローバルな定数変数を更新）

前提条件と不変条件

  • 本関数は、度数法関連の計算を行う前に一度だけ実行されることを想定しています。
  • 実行後、グローバル変数 degree_consts_set が true に設定されます。

アルゴリズムまたは振る舞い

1. sin_30, one_minus_cos_60, asin_0_5, acos_0_5, atan_1_0 を標準数学ライブラリの関数を用いて計算し、保持します。 2. tan_45 および cot_45 を、内部的な度数法用関数 sind_q1 および cosd_q1 を用いて計算します。 3. 最後に degree_consts_set フラグを true に更新し、初期化完了を示します。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/utils/adt/float.c 内の度数法計算関数から呼び出されます（直接的な呼び出し元関数の詳細はソース抜粋からは特定できません）。
  • 呼び出し先: sin, cos, asin, acos, atan, sind_q1, cosd_q1（標準数学ライブラリおよび内部ヘルパー関数）。

関連データ構造

  • degree_consts_set (bool): 初期化状態を管理するフラグ。
  • sin_30, one_minus_cos_60, asin_0_5, acos_0_5, atan_1_0, tan_45, cot_45: 計算結果を保持する静的変数。

失敗モードと境界条件

  • 本関数自体はエラーを返しませんが、依存する数学ライブラリの挙動に依存します。
  • 初期化が二重に行われないよう、呼び出し側で degree_consts_set のチェックを行うことが推奨されます。

保守担当者向けメモ

  • 本関数は、プラットフォーム間での浮動小数点演算の差異を吸収するための重要な役割を担っています。定数の計算ロジックを変更する際は、float.c 内の他の度数法関連関数への影響を十分に検証してください。
