check_agg_arguments

定義

  • 種別: function
  • 定義箇所: src/backend/parser/parse_agg.c:644

目的

集約関数の引数、フィルタ式、および直接引数（directargs）を走査し、集約関数のセマンティックレベル（agglevel）を決定するとともに、ネストされた集約や不正な配置、不適切なCTE参照を検証する。

シグネチャまたは形状

int check_agg_arguments(ParseState *pstate, List *directargs, List *args, Expr *filter, int agglocation)

入力と出力

  • **入力**:
  • pstate: パーサーの状態。
  • directargs: 集約関数の直接引数リスト。
  • args: 集約関数の引数リスト。
  • filter: 集約関数のフィルタ式。
  • agglocation: エラー報告用の構文位置。
  • **出力**:
  • 戻り値: 決定された集約関数のセマンティックレベル（int）。

前提条件と不変条件

  • check_agg_arguments_walker を使用して式ツリーを走査し、変数、集約、CTEの最小レベルを特定する。
  • 集約関数のネストは禁止されている。
  • 集約関数のセマンティックレベルより下のレベルにあるCTEは使用できない。

アルゴリズムまたは振る舞い

1. check_agg_arguments_context を初期化し、args および filter に対して check_agg_arguments_walker を実行する。 2. 検出された変数レベルと集約レベルに基づき、集約関数のセマンティックレベル（agglevel）を算出する。 3. args または filter 内に、決定された agglevel と一致する集約関数が存在する場合、ネストエラーとして報告する。 4. context 内の min_ctelevel が agglevel より小さい場合、不正なCTE参照としてエラーを報告する。 5. directargs が存在する場合、同様に走査を行い、以下の条件でエラーを報告する：

  • 変数レベルが agglevel より小さい場合。
  • 集約レベルが agglevel 以下の場合（ネストとみなす）。
  • CTEレベルが agglevel より小さい場合。

    • 6. 最終的な agglevel を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/parser/parse_agg.c 内の transformAggregateCall（行342付近）から呼び出される。
  • **呼び出し先**:
  • check_agg_arguments_walker
  • locate_agg_of_level
  • locate_var_of_level
  • ereport (エラー発生時)

関連データ構造

  • ParseState: パーサーの状態を保持。
  • check_agg_arguments_context: 走査中のレベル情報を保持する構造体。

失敗モードと境界条件

  • **ERRCODE_GROUPING_ERROR**: 集約関数のネスト、または集約関数の直接引数に下位レベルの変数や集約が含まれる場合に発生。
  • **ERRCODE_FEATURE_NOT_SUPPORTED**: 集約関数のセマンティックレベルより下位のCTEを参照した場合に発生。

保守担当者向けメモ

  • 本関数は集約関数の妥当性検証の核心部である。sublevels_up の概念を理解した上で修正を行うこと。
  • directargs に対するチェックは、標準SQLの要件および実行時の順序依存性を考慮して厳格に実装されている。
