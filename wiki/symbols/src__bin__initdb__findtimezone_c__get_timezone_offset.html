get_timezone_offset

定義

  • 種別: function
  • 定義箇所: src/bin/initdb/findtimezone.c:174

目的

指定された struct tm 構造体から、現在のタイムゾーンのオフセット（UTCからの秒数）を取得します。

シグネチャまたは形状

static int get_timezone_offset(struct tm *tm);

入力と出力

  • **入力**: struct tm *tm （時刻情報を含む構造体へのポインタ）
  • **出力**: int （UTCからのオフセット秒数）

前提条件と不変条件

  • プラットフォームが HAVE_STRUCT_TM_TM_ZONE または HAVE_INT_TIMEZONE のいずれかをサポートしている必要があります。いずれも定義されていない場合、コンパイルエラーとなります。

アルゴリズムまたは振る舞い

1. HAVE_STRUCT_TM_TM_ZONE が定義されている場合、tm->tm_gmtoff を返します。 2. 上記が定義されておらず HAVE_INT_TIMEZONE が定義されている場合、グローバル変数 TIMEZONE_GLOBAL の負の値を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/bin/initdb/findtimezone.c:462 にて std_ofs の算出に使用されています。
  • **呼び出し先**: 外部関数への依存はありません。

関連データ構造

  • struct tm: C標準ライブラリの時刻構造体。

失敗モードと境界条件

  • プラットフォーム固有の定義（tm_gmtoff や TIMEZONE_GLOBAL）が存在しない環境では、コンパイル時にエラーが発生します。

保守担当者向けメモ

  • 本関数はプラットフォームの差異を吸収するためのラッパーです。新しい環境への移植時にタイムゾーン取得方法が変更された場合は、プリプロセッサ条件の更新が必要です。
