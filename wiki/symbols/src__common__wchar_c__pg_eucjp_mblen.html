pg_eucjp_mblen

定義

  • 種別: function
  • 定義箇所: src/common/wchar.c:189

目的

EUC-JP エンコーディングにおけるマルチバイト文字のバイト長を判定します。

シグネチャまたは形状

int pg_eucjp_mblen(const unsigned char *s)

入力と出力

  • **入力**: s - 判定対象のマルチバイト文字列へのポインタ。
  • **出力**: 指定された文字列の先頭文字のバイト長（整数）。

前提条件と不変条件

  • 入力ポインタ s は有効なメモリ領域を指している必要があります。
  • 内部的に pg_euc_mblen を呼び出すため、EUC-JP のエンコーディング規則に従ったバイトシーケンスであることを前提とします。

アルゴリズムまたは振る舞い

1. 入力された文字列ポインタ s をそのまま pg_euc_mblen 関数へ渡します。 2. pg_euc_mblen の戻り値をそのまま呼び出し元へ返します。

呼び出し元と呼び出し先

  • **呼び出し先**: pg_euc_mblen
  • **呼び出し元**:
  • src/common/wchar.c 内の pg_wchar_table 配列において、PG_EUC_JP および PG_EUC_JIS_2004 エンコーディングの操作関数テーブルの一部として参照されています。

関連データ構造

  • pg_wchar_table: 各エンコーディングの操作関数を保持するグローバルな関数テーブル。

失敗モードと境界条件

  • 入力文字列が不正なエンコーディングである場合の挙動は、内部で呼び出される pg_euc_mblen の実装に依存します。

保守担当者向けメモ

  • 本関数は pg_euc_mblen への単なるラッパーです。EUC-JP 固有のロジックを追加する必要が生じた場合は、この関数内で処理を分岐させるか、pg_euc_mblen 自体の修正を検討してください。
