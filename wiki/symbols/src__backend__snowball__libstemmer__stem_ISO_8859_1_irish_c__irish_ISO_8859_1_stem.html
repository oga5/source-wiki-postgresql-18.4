目的

ISO-8859-1 エンコーディングのアイルランド語テキストに対し、形態論的変化の除去および接尾辞の削除を行い、単語の語幹を抽出する。

シグネチャまたは形状

int irish_ISO_8859_1_stem(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z（ステミング環境を保持する構造体へのポインタ）
  • **出力**: 成功時に 1 を返し、エラー発生時に負の値を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • 内部処理において z->c（現在のカーソル位置）および z->l（文字列長）が適切に管理される。

アルゴリズムまたは振る舞い

1. **初期変異の処理**: r_initial_morph を呼び出し、単語先頭の変異を処理する。 2. **領域定義**: r_mark_regions を呼び出し、ステミング対象となる領域（R1, R2, RV）を計算する。 3. **接尾辞の削除**: 以下の順序で各関数を適用し、それぞれの処理前後でカーソル位置を復元する。

  • r_noun_sfx（名詞接尾辞の削除）
  • r_deriv（派生語の処理）
  • r_verb_sfx（動詞接尾辞の削除）

    • 4. **終了**: 処理完了後、カーソルを初期位置に戻し 1 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: ソースコード内の直接的な呼び出し元は明示されていないが、libstemmer の API を介して外部から呼び出される設計である。
  • **呼び出し先**: r_initial_morph, r_mark_regions, r_noun_sfx, r_deriv, r_verb_sfx。

関連データ構造

  • struct SN_env: ステミングの実行状態を保持する構造体。

失敗モードと境界条件

  • 各サブ関数（r_initial_morph 等）が負の値を返した場合、その値をそのまま呼び出し元へ伝播させ終了する。
