AlterPublicationTables

定義

  • 種別: function
  • 定義箇所: src/backend/commands/publicationcmds.c:1198

目的

パブリケーションに関連付けられたテーブルリストの追加、削除、または置換（SET）を行う。

シグネチャまたは形状

static void AlterPublicationTables(AlterPublicationStmt *stmt, HeapTuple tup, List *tables, const char *queryString, bool publish_schema)

入力と出力

  • **入力**:
  • stmt: ALTER PUBLICATION 文の構文解析結果。
  • tup: 対象パブリケーションの pg_publication カタログタプル。
  • tables: 操作対象のテーブルリスト。
  • queryString: 元のSQLクエリ文字列（行フィルタ解析用）。
  • publish_schema: スキーマパブリケーションかどうかのフラグ。
  • **出力**: なし（副作用としてカタログを更新）。

前提条件と不変条件

  • tup は有効な pg_publication タプルであること。
  • tables が空で stmt->action が AP_SetObjects 以外の場合、何もせず終了する。

アルゴリズムまたは振る舞い

1. **初期化**: OpenTableList でテーブルリストを開く。 2. **アクション別処理**:

  • AP_AddObjects: TransformPubWhereClauses で行フィルタを検証し、CheckPubRelationColumnList でカラムリストを検証後、PublicationAddTables を呼び出す。
  • AP_DropObjects: PublicationDropTables を呼び出す。
  • AP_SetObjects:
  • 現在のパブリケーションに含まれる全リレーションを取得。
  • 新旧のWHERE句およびカラムリストを比較し、変更がないものは維持、それ以外は削除対象リスト（delrels）に追加。
  • PublicationDropTables で不要なリレーションを削除。
  • PublicationAddTables で新しいリレーションリストを適用。

    • 3. **終了**: CloseTableList でリソースを解放。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/commands/publicationcmds.c 内の AlterPublication（推測されるディスパッチャ）。
  • **呼び出し先**: OpenTableList, TransformPubWhereClauses, CheckPubRelationColumnList, PublicationAddTables, PublicationDropTables, GetPublicationRelations, SearchSysCache2, SysCacheGetAttr, ReleaseSysCache, pub_collist_to_bitmapset, pub_collist_validate, equal, bms_equal, table_open, CloseTableList。

関連データ構造

  • AlterPublicationStmt: SQL文の解析構造体。
  • PublicationRelInfo: パブリケーションとリレーションの関連情報。
  • pg_publication: パブリケーションのカタログ定義。

失敗モードと境界条件

  • AP_SetObjects 時に既存のWHERE句やカラムリストが一致しない場合、リレーションは再作成（削除および追加）の対象となる。
  • 行フィルタやカラムリストの検証に失敗した場合、エラーが発生する。

保守担当者向けメモ

  • 本関数は pg_publication_rel カタログの更新を伴うため、キャッシュ無効化の整合性に注意すること。
  • AP_SetObjects における比較ロジックは、既存の WHERE 句やカラムリストの変更を正確に検知するために equal() および bms_equal() を使用している。
