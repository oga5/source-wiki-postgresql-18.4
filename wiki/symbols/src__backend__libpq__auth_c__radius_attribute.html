radius_attribute

定義

  • 種別: struct
  • 定義箇所: src/backend/libpq/auth.c:2784

目的

RADIUSプロトコルにおける属性（Attribute）を表現するためのデータ構造です。

シグネチャまたは形状

typedef struct
{
	uint8		attribute;
	uint8		length;
	uint8		data[FLEXIBLE_ARRAY_MEMBER];
} radius_attribute;

入力と出力

  • 入力: RADIUSパケット内のバイナリデータ。
  • 出力: 構造体メンバとして属性タイプ、長さ、および可変長のデータフィールドを保持します。

前提条件と不変条件

  • data フィールドは FLEXIBLE_ARRAY_MEMBER を使用しており、構造体の末尾に可変長のデータが続くことを前提としています。

アルゴリズムまたは振る舞い

  • RADIUSパケットの解析時に、パケット内のオフセット計算を通じて属性データへアクセスするために使用されます。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/libpq/auth.c 内のRADIUS認証処理ロジック。
  • 呼び出し先: 本構造体自体はデータ保持用であり、直接的な関数呼び出しは行いません。

関連データ構造

  • radius_packet: RADIUSパケット全体を表現する構造体。

失敗モードと境界条件

  • length フィールドの値が不正な場合、パケット解析時にメモリ境界外アクセスが発生するリスクがあります。

保守担当者向けメモ

  • data フィールドが可変長であるため、本構造体を扱う際は必ずパケット全体のサイズと length フィールドの整合性を検証してください。
