r_possessive

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_finnish.c:333

目的

フィンランド語のステミング処理において、単語末尾の所有接尾辞を識別し、必要に応じて削除または置換を行う。

シグネチャまたは形状

static int r_possessive(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（Snowball環境構造体）
  • 出力: 成功時に 1、失敗または適用対象外の場合に 0、メモリ操作エラー時に負の値を返す。

前提条件と不変条件

  • z->I[1] に格納された境界位置以降の文字列を処理対象とする。
  • 処理対象領域が境界より前にある場合、即座に 0 を返す。

アルゴリズムまたは振る舞い

1. find_among_b を使用し、所有接尾辞のパターン（a_4）を末尾から検索する。 2. マッチしたパターン（among_var）に応じて以下の処理を行う：

  • **ケース1**: 直前に 'k' が存在しない場合のみ削除。
  • **ケース2**: 削除後、さらに特定の文字列（s_0）を検索し、別の文字列（s_1）に置換する。
  • **ケース3**: 単純削除。
  • **ケース4-6**: 特定の文字（'a', '228', 'e'）を先行条件として確認し、それぞれのパターン（a_1, a_2, a_3）に合致する場合に削除。

    • 3. 処理の各段階で slice_del または slice_from_s を呼び出し、文字列バッファを更新する。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_ISO_8859_1_finnish.c および stem_UTF_8_finnish.c 内のメインステミング処理フロー。
  • 呼び出し先: find_among_b, slice_del, slice_from_s, eq_s_b（いずれもSnowballランタイム関数）。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_1 ～ a_4: 語尾パターンを定義する among 構造体配列。

失敗モードと境界条件

  • z->c（現在のカーソル位置）が z->lb（境界）以下の場合、処理を中断する。
  • slice_del 等のランタイム関数が負の値を返した場合、そのエラーコードを伝播させる。

保守担当者向けメモ

  • 本関数はSnowballコンパイラによる自動生成コードである。ロジックの変更が必要な場合は、直接編集せず元の .sbl スクリプトを修正すること。
  • ISO-8859-1版とUTF-8版の両方で参照されているため、修正時は両方の整合性に注意が必要。
