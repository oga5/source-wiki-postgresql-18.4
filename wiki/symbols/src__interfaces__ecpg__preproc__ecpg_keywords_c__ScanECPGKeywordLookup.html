ScanECPGKeywordLookup

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/preproc/ecpg_keywords.c:38

目的

入力された文字列がSQLキーワードまたはECPG固有のキーワードであるかを判定し、対応するトークン値を返します。

シグネチャまたは形状

int ScanECPGKeywordLookup(const char *text)

入力と出力

  • **入力**: text (検索対象の文字列)
  • **出力**: 一致したキーワードのトークンID（整数）、または一致しない場合は -1

前提条件と不変条件

  • 入力文字列 text は有効なC文字列である必要があります。
  • 検索は ScanKeywordLookup を介して行われ、大文字・小文字を区別しない照合が行われます。

アルゴリズムまたは振る舞い

1. ScanKeywordLookup を使用し、まず ScanKeywords（標準SQLキーワード）を検索します。一致した場合は SQLScanKeywordTokens から対応するトークン値を返します。 2. 標準SQLキーワードで見つからない場合、ScanECPGKeywords（ECPG固有キーワード）を検索します。一致した場合は ECPGScanKeywordTokens から対応するトークン値を返します。 3. いずれにも該当しない場合、-1 を返します。

呼び出し元と呼び出し先

  • **呼び出し先**: ScanKeywordLookup
  • **呼び出し元**: ソースコード内の明示的な呼び出し元は提供されていませんが、src/interfaces/ecpg/preproc/preproc_extern.h で外部インターフェースとして宣言されています。

関連データ構造

  • ScanKeywords: 標準SQLキーワードのリスト。
  • ScanECPGKeywords: ECPG固有キーワードのリスト。
  • SQLScanKeywordTokens: 標準SQLキーワードに対応するトークン値の配列。
  • ECPGScanKeywordTokens: ECPG固有キーワードに対応するトークン値の配列。

失敗モードと境界条件

  • 入力文字列がどのキーワードリストにも存在しない場合、-1 を返します。
  • 検索ロジックはバックエンドのキーワード検索ルールに依存しており、仕様変更の影響を受ける可能性があります。

保守担当者向けメモ

  • 本関数はX-Macroパターンを利用してキーワードリストを管理しています。キーワードの追加や変更を行う際は、関連するヘッダーファイル（ecpg_kwlist.h 等）の整合性に注意してください。
