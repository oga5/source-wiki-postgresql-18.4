pgstat_get_xact_stack_level

定義

  • 種別: function
  • 定義箇所: src/backend/utils/activity/pgstat_xact.c:237

目的

指定されたトランザクションのネストレベルに対応する PgStat_SubXactStatus 構造体を取得または作成し、統計情報のトランザクション管理スタックを維持する。

シグネチャまたは形状

PgStat_SubXactStatus *pgstat_get_xact_stack_level(int nest_level)

入力と出力

  • **入力**: nest_level (int) - 取得対象のトランザクションネストレベル。
  • **出力**: PgStat_SubXactStatus * - 指定レベルに対応するスタックエントリへのポインタ。

前提条件と不変条件

  • pgStatXactStack は現在のトランザクション状態を保持するスタックの先頭を指す。
  • 新規作成されるエントリは TopTransactionContext 上に割り当てられる。
  • スタックは nest_level の降順（最新が先頭）で構成される。

アルゴリズムまたは振る舞い

1. 現在のスタック先頭 (pgStatXactStack) が指定された nest_level と一致するか確認する。 2. 一致しない場合（スタックが空、またはレベルが異なる場合）、TopTransactionContext から新しい PgStat_SubXactStatus を割り当てる。 3. 新規エントリの初期化を行う：

  • pending_drops リストを初期化。
  • nest_level を設定。
  • prev ポインタを現在の pgStatXactStack に設定。
  • first を NULL に初期化。
  • pgStatXactStack を新しいエントリで更新する。

    • 4. 該当する（または作成された）エントリを返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/activity/pgstat_relation.c: 657, 946
  • src/backend/utils/activity/pgstat_xact.c: 146, 342
  • **呼び出し先**:
  • MemoryContextAlloc (メモリ割り当て)
  • dclist_init (リスト初期化)

関連データ構造

  • PgStat_SubXactStatus: トランザクションごとの統計変更状態を保持する構造体。
  • pgStatXactStack: モジュール内のグローバル変数（スタックの先頭）。

失敗モードと境界条件

  • TopTransactionContext が存在しない環境で呼び出された場合、メモリ割り当てによりエラーが発生する可能性がある。
  • nest_level が現在のトランザクション状態と整合しない場合、スタックが動的に拡張される。

保守担当者向けメモ

  • 本関数は TopTransactionContext を使用するため、トランザクション終了時に自動的にメモリが解放される設計となっている。
  • 複雑なサブトランザクションのネストが発生する場合、スタックのポインタ操作が正しく行われているか注意すること。
