r_izenak

定義

  • **定義箇所**: src/backend/snowball/libstemmer/stem_UTF_8_basque.c:1050

目的

バスク語のステミング処理において、名詞（izenak）に関連する接尾辞を単語末尾から探索し、削除または置換を行う。

シグネチャまたは形状

static int r_izenak(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（ステミング環境の状態を保持する構造体）
  • **出力**: 成功時に 1、処理対象外または失敗時に 0、エラー時に負の値を返す。

前提条件と不変条件

  • z->c は単語の末尾位置を指している必要がある。
  • find_among_b を用いて a_1 テーブル（295個の要素）から接尾辞を後方一致検索する。
  • 処理対象の接尾辞が特定された場合、r_RV、r_R1、r_R2 のいずれかの領域制約を満たす必要がある。

アルゴリズムまたは振る舞い

1. z->ket に現在のカーソル位置を記録する。 2. 単語末尾の文字が特定の条件（ビットマスク 71162402 による判定）を満たさない場合、即座に 0 を返す。 3. find_among_b で接尾辞を検索し、一致した among_var に応じて以下の処理を行う：

  • **ケース 1**: r_RV を確認後、slice_del で削除。
  • **ケース 2**: r_R2 を確認後、slice_del で削除。
  • **ケース 4**: r_R1 を確認後、slice_del で削除。
  • **ケース 3, 5-10**: slice_from_s を用いて、対応する文字列（s_3〜s_9）に置換。

    • 4. 処理が正常に完了すれば 1 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_UTF_8_basque.c および src/backend/snowball/libstemmer/stem_ISO_8859_1_basque.c 内のメインステミング処理ルーチンから呼び出される。
  • **呼び出し先**: find_among_b、r_RV、r_R1、r_R2、slice_del、slice_from_s。

関連データ構造

  • struct SN_env: ステミングの実行状態を管理する構造体。
  • a_1: 名詞接尾辞の探索用テーブル。

失敗モードと境界条件

  • z->c <= z->lb の場合、または接尾辞がテーブルに見つからない場合は 0 を返す。
  • r_RV、r_R1、r_R2 などの領域チェック関数が失敗した場合、その戻り値をそのまま返す。
  • slice_del や slice_from_s がメモリ不足等の理由で負の値を返した場合、そのエラーを伝播させる。
