ExecEndGroup

定義

  • 種別: function
  • 定義箇所: src/backend/executor/nodeGroup.c:225

目的

Group ノードの実行を終了し、関連するリソースを解放する。

シグネチャまたは形状

void ExecEndGroup(GroupState *node)

入力と出力

  • **入力**: GroupState *node - 終了対象となるグループ化ノードの実行状態構造体。
  • **出力**: なし（void）。

前提条件と不変条件

  • 引数 node は有効な GroupState 構造体を指している必要がある。
  • outerPlanState(node) を介して取得される子プランの状態が初期化済みであること。

アルゴリズムまたは振る舞い

1. outerPlanState(node) を呼び出し、GroupState に含まれる子プランの PlanState を取得する。 2. 取得した PlanState を引数として ExecEndNode を呼び出し、子プランの終了処理（リソース解放等）を実行する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/executor/execProcnode.c:722 からの呼び出しが確認されている。
  • **呼び出し先**: ExecEndNode（子プランの終了処理）。

関連データ構造

  • GroupState: Group ノードの実行状態を保持する構造体。
  • PlanState: 子プランの実行状態を保持する構造体。

失敗モードと境界条件

  • node が NULL の場合、outerPlanState マクロ内で不正なメモリ参照が発生する可能性がある。

保守担当者向けメモ

  • 本関数は Group ノード固有の終了処理を担うが、現状では子プランの終了処理を委譲するのみである。将来的に GroupState 固有のメモリ解放やリソース管理が必要となった場合は、ExecEndNode の呼び出し前後で処理を追加すること。
