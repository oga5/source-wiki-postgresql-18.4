pg_atomic_exchange_u32_impl

定義

  • 種別: function
  • 定義箇所: src/include/port/atomics/generic-msvc.h:60

目的

MSVC環境において、32ビット符号なし整数に対してアトミックな値の交換操作を提供します。

シグネチャまたは形状

static inline uint32 pg_atomic_exchange_u32_impl(volatile pg_atomic_uint32 *ptr, uint32 newval)

入力と出力

  • **入力**:
  • ptr: 操作対象となる pg_atomic_uint32 構造体へのポインタ。
  • newval: ptr に書き込む新しい 32ビット値。
  • **出力**:
  • 交換前の ptr が保持していた元の 32ビット値を返します。

前提条件と不変条件

  • ptr は volatile 修飾されており、コンパイラによる最適化で読み書きが省略されないことが保証されます。
  • InterlockedExchange を使用するため、対象のアドレスは適切にアライメントされている必要があります。

アルゴリズムまたは振る舞い

1. Windows API の InterlockedExchange を呼び出します。 2. ptr->value の現在値をアトミックに newval で置き換えます。 3. 置き換え前の値を戻り値として返します。この操作はメモリバリアを内包します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/include/port/atomics.h:334 からの呼び出しが確認されています。
  • **呼び出し先**: Windows の InterlockedExchange 関数を呼び出します。

関連データ構造

  • pg_atomic_uint32: アトミック操作対象の値を保持する構造体。

失敗モードと境界条件

  • アドレスが適切にアライメントされていない場合、InterlockedExchange の仕様に基づきシステムエラーや未定義動作が発生する可能性があります。

保守担当者向けメモ

  • 本関数は generic-msvc.h 内で定義されており、atomics.h を通じて利用されることを前提としています。
  • Interlocked API はメモリバリアを内包しているため、追加のバリア操作は不要です。
