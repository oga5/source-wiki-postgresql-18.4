GetTimerFrequency

定義

  • 種別: function
  • 定義箇所: src/include/portability/instr_time.h:147

目的

Windows環境における高精度パフォーマンスカウンタの周波数（1秒あたりのカウント数）を取得し、double 型の浮動小数点数として返します。

シグネチャまたは形状

static inline double GetTimerFrequency(void)

入力と出力

  • **入力**: なし（void）
  • **出力**: double 型の周波数値。

前提条件と不変条件

  • 本関数は Windows プラットフォームの QueryPerformanceFrequency API に依存しています。
  • 戻り値は、システムが提供するパフォーマンスカウンタの周波数に基づきます。

アルゴリズムまたは振る舞い

1. Windows API の QueryPerformanceFrequency を呼び出し、LARGE_INTEGER 型の変数 f に周波数を格納します。 2. f.QuadPart を double 型にキャストして返します。

呼び出し元と呼び出し先

  • **呼び出し先**: Windows API QueryPerformanceFrequency
  • **呼び出し元**:
  • 同ファイル内の時間変換処理（instr_time 構造体のティック値をナノ秒に変換する計算式内）で参照されています。

関連データ構造

  • LARGE_INTEGER: Windows API で使用される 64 ビット整数型。

失敗モードと境界条件

  • Windows 以外のプラットフォームでの動作は定義されていません（本ヘッダのポータビリティ設計に依存）。
  • QueryPerformanceFrequency が失敗した場合の挙動については、本ソースコード内では明示的なエラーハンドリングは行われていません。

保守担当者向けメモ

  • 本関数は static inline で定義されており、ヘッダファイル内で直接利用されることを想定しています。
  • 内部で QueryPerformanceFrequency を呼び出しているため、Windows 以外の環境でコンパイルされる場合は、条件付きコンパイル等の適切なガードが適用されているか確認してください。
