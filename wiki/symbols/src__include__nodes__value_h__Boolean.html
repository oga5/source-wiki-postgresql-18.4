Boolean

定義

  • 種別: struct
  • 定義箇所: src/include/nodes/value.h:55

目的

SQLクエリ内の真偽値リテラルを、PostgreSQLの構文解析ツリー（Parse Tree）内で保持するためのデータコンテナ。

シグネチャまたは形状

typedef struct Boolean
{
	pg_node_attr(special_read_write)

	NodeTag		type;
	bool		boolval;
} Boolean;

入力と出力

  • **入力**: bool 型の値。
  • **出力**: Boolean 構造体としてラップされたノード。

前提条件と不変条件

  • type フィールドには、PostgreSQLのノードシステムで定義された適切な NodeTag が設定されている必要がある。
  • pg_node_attr(special_read_write) 属性により、ノードの読み書きに関する特殊な処理が適用される。

アルゴリズムまたは振る舞い

  • パーサーが字句解析中に真偽値リテラルを検出した際、本構造体を用いてメモリ上にノードを生成する。
  • 構文解析の各フェーズにおいて、List 等の構造に格納されて受け渡される。
  • 意味解析や計画作成の段階で、内部の boolval が抽出され、実行可能な形式へ変換される。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/postgres_fdw/postgres_fdw.c において、リモートクエリのフラグ（RETURNING句の有無や処理済みフラグなど）を保持するために使用されている。
  • **呼び出し先**: 提供されたソースからは、特定の関数呼び出し関係は特定できない。

関連データ構造

  • NodeTag: ノードの型識別子。
  • List: 本構造体が格納される一般的なコンテナ。

失敗モードと境界条件

  • src/backend/commands/copy.c において、入力値が真偽値として解釈できない場合にエラーメッセージを出力する境界条件が存在する。

保守担当者向けメモ

  • 本構造体は Value 構造体から分離された個別のノード型である。
  • 浮動小数点数や文字列を保持する他のノード構造体と同様に、PostgreSQLのノードシステムの一部として統合されている。
