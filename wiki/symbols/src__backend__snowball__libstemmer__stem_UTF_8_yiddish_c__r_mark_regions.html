r_mark_regions

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_yiddish.c:619

目的

イディッシュ語のステミング処理において、単語内の領域（R1など）を計算し、SN_env 構造体の内部状態（I[0], I[1]）を更新する。

シグネチャまたは形状

static int r_mark_regions(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z （ステミング環境へのポインタ）
  • 出力: 成功時に 1、失敗または処理中断時に 0 を返す。内部状態 z->I[0] および z->I[1] が更新される。

前提条件と不変条件

  • z は有効な SN_env 構造体であること。
  • z->p（文字列バッファ）および z->c（カーソル位置）が適切に初期化されていること。

アルゴリズムまたは振る舞い

1. **初期化**: z->I[1] を文字列長 z->l に設定。 2. **接頭辞処理**: 特定の文字列パターンを検索し、一致した場合は slice_from_s を用いて置換を行う。 3. **領域特定**:

  • find_among を使用して特定の接尾辞パターンを検索し、条件に応じて置換を行う。
  • skip_utf8 を用いてカーソルを移動し、z->I[0] に基準位置を記録する。
  • in_grouping_U を用いて子音の連続を判定し、z->I[1] を更新する。
  • 母音の出現位置を走査し、最終的な z->I[1] を決定する。

    • 4. **境界調整**: z->I[1] が z->I[0] より小さい場合、z->I[1] を z->I[0] に補正する。

呼び出し元と呼び出し先

  • 呼び出し元: 提供されたソースコード内には明示的な呼び出し元は記載されていない。
  • 呼び出し先: eq_s, slice_from_s, find_among, skip_utf8, in_grouping_U, out_grouping_U （Snowball ランタイム関数）。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • among 構造体: 接尾辞検索用のトライ木データ構造（a_1, a_2）。

失敗モードと境界条件

  • slice_from_s が負の値を返した場合、その値をそのまま戻り値として終了する。
  • skip_utf8 が失敗した場合、0 を返す。
  • 文字列の末尾を超えたアクセスが発生しないよう、z->c < z->l 等のガード条件が含まれている。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードであるため、直接編集せず元の .sbl スクリプトを修正すること。
  • g_consonant および g_vowel はイディッシュ語の文字グループ定義を参照している。
