SyncCommitLevel

定義

  • 種別: enum
  • 定義箇所: src/include/access/xact.h:68

目的

トランザクションコミット時における同期レベル（WALの書き込みおよびフラッシュの待機範囲）を定義します。

シグネチャまたは形状

typedef enum {
    SYNCHRONOUS_COMMIT_OFF,
    SYNCHRONOUS_COMMIT_LOCAL_FLUSH,
    SYNCHRONOUS_COMMIT_REMOTE_WRITE,
    SYNCHRONOUS_COMMIT_REMOTE_FLUSH,
    SYNCHRONOUS_COMMIT_REMOTE_APPLY,
} SyncCommitLevel;

入力と出力

本シンボル自体は列挙型定義であり、入力や出力は持ちません。

前提条件と不変条件

  • 各列挙値は、トランザクションの永続化保証レベルを昇順に表します。
  • SYNCHRONOUS_COMMIT_OFF は非同期コミットを意味します。
  • SYNCHRONOUS_COMMIT_REMOTE_* 系は、レプリケーション環境におけるリモートノードの処理完了を待機する設定です。

アルゴリズムまたは振る舞い

トランザクションのコミット処理において、この列挙値に基づいてWALのフラッシュ待機やリモートノードからの応答待機が行われます。

呼び出し元と呼び出し先

  • 呼び出し元および呼び出し先の具体的な関数情報は、提供されたソース excerpt には含まれていません。

関連データ構造

  • src/include/access/xact.h 内で定義されるトランザクション管理構造体や、WAL関連のレコード構造体と密接に関連します。

失敗モードと境界条件

  • 設定値が範囲外の場合の挙動については、本定義からは特定できません。

保守担当者向けメモ

  • 本列挙型は、PostgreSQLの同期レプリケーション設定（synchronous_commit パラメータ）と直接対応しています。
  • 新しい同期レベルを追加する場合は、関連するWAL書き込みロジックおよびレプリケーション待機ロジックの更新が必要です。
