目的

入力文字列内のアクセント記号付き文字や特定の文字を正規化（置換または削除）し、ステミング処理に適した形式に変換する。

入力と出力

  • **入力**: struct SN_env * z （Snowballステミング環境）
  • **出力**: int （成功時に 1、失敗時に負の値を返す）

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • a_0 テーブル（13個の要素）が定義されている必要がある。

アルゴリズムまたは振る舞い

1. 文字列全体に対してループを実行する。 2. find_among を使用して a_0 テーブルに基づき、現在のカーソル位置からマッチする接尾辞や文字を探索する。 3. マッチしたケースに応じて slice_from_s を呼び出し、対応する文字列に置換する。 4. マッチしない場合、現在のカーソル位置の文字をスキップ（削除）する。 5. 文字列の末尾に達するまでこの処理を繰り返す。
