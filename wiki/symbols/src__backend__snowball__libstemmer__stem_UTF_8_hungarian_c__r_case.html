r_case

定義

  • **種別**: function
  • **定義箇所**: src/backend/snowball/libstemmer/stem_UTF_8_hungarian.c:589

目的

ハンガリー語のステミング処理において、単語末尾の特定の接尾辞（ケースマーカー）を特定し、条件を満たす場合にそれを削除する。

シグネチャまたは形状

static int r_case(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z（Snowball ステミング実行環境）
  • **出力**: int（成功時に 1、失敗時に 0、エラー時に負の値を返す）

前提条件と不変条件

  • z->c が単語の末尾を指していること。
  • a_4 配列に定義された 44 個の接尾辞のいずれかが単語末尾に存在すること。
  • 削除対象の接尾辞が R1 領域（r_R1 で定義）内に含まれていること。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を z->ket に設定する。 2. find_among_b を使用し、末尾から a_4 配列の接尾辞を検索する。一致しない場合は 0 を返して終了。 3. 一致した接尾辞の開始位置を z->bra に設定する。 4. r_R1 を呼び出し、一致した箇所が R1 領域内にあるか検証する。検証失敗時はその戻り値を返す。 5. slice_del を呼び出し、一致した接尾辞を単語から削除する。 6. r_v_ending を呼び出し、削除後の単語に対して母音終了処理を行う。 7. すべてのステップが成功した場合に 1 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/snowball/libstemmer/stem_ISO_8859_2_hungarian.c:809
  • src/backend/snowball/libstemmer/stem_UTF_8_hungarian.c:815
  • **呼び出し先**:
  • find_among_b

  • r_v_ending

関連データ構造

  • struct SN_env: ステミング処理の実行状態を保持する構造体。
  • a_4: 削除対象となる接尾辞を格納した among 構造体の配列。

失敗モードと境界条件

  • find_among_b が一致を見つけられない場合、0 を返す。
  • r_R1 または slice_del、r_v_ending が負の値を返した場合、そのエラーコードを伝播させる。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードであるため、直接編集せず、元の Snowball 定義ファイルを修正すること。
  • a_4 配列の定義変更は、ステミングのルール全体に影響を与えるため注意が必要。
