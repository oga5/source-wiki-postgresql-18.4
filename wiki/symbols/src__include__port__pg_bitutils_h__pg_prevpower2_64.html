pg_prevpower2_64

定義

  • 種別: function
  • 定義箇所: src/include/port/pg_bitutils.h:247

目的

与えられた 64 ビット符号なし整数以下の最大の 2 のべき乗を算出する。

シグネチャまたは形状

static inline uint64 pg_prevpower2_64(uint64 num)

入力と出力

  • **入力**: num (uint64) - 対象となる 64 ビット整数。
  • **出力**: num 以下の最大の 2 のべき乗（uint64）。

前提条件と不変条件

  • pg_leftmost_one_pos64 が正常に動作する環境であること。
  • 入力値 num が 0 の場合、pg_leftmost_one_pos64 の仕様に依存する（一般的に 0 は未定義動作の対象となるため、呼び出し側で 0 を避ける必要がある）。

アルゴリズムまたは振る舞い

1. pg_leftmost_one_pos64(num) を呼び出し、num の最上位ビット（MSB）の位置を取得する。 2. 1 をその位置まで左シフト（1 << pos）することで、MSB のみを保持する 2 のべき乗値を生成して返す。

呼び出し元と呼び出し先

  • **呼び出し先**: pg_leftmost_one_pos64
  • **呼び出し元**: 直接的な呼び出し元は提供されたソース内には明示されていないが、pg_prevpower2_size_t マクロが本関数を指すように定義されている。

関連データ構造

  • なし。

失敗モードと境界条件

  • num が 0 の場合、pg_leftmost_one_pos64 の挙動に依存し、予期せぬ結果やアサーションエラーが発生する可能性がある。

保守担当者向けメモ

  • 本関数は static inline で定義されており、ヘッダーファイル内で完結する。
  • pg_prevpower2_size_t マクロを通じて、プラットフォームのワードサイズに応じた汎用的な利用が想定されている。
