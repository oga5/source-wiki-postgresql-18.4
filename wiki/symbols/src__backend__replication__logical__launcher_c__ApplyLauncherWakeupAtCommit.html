ApplyLauncherWakeupAtCommit

定義

  • 種別: function
  • 定義箇所: src/backend/replication/logical/launcher.c:1114

目的

トランザクションのコミット時に論理レプリケーションランチャーを起動（ウェイクアップ）させるためのフラグをセットする。

シグネチャまたは形状

void ApplyLauncherWakeupAtCommit(void)

入力と出力

  • 入力: なし
  • 出力: なし（副作用として静的変数 on_commit_launcher_wakeup を true に設定する）

前提条件と不変条件

  • 本関数は、トランザクションのコミット処理に関連するコンテキストで呼び出されることを想定している。

アルゴリズムまたは振る舞い

1. 静的ブール変数 on_commit_launcher_wakeup を参照する。 2. 当該変数が false の場合、true に更新する。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/backend/commands/subscriptioncmds.c:810
  • src/backend/commands/subscriptioncmds.c:1355
  • src/backend/commands/subscriptioncmds.c:2022
  • 呼び出し先: なし

関連データ構造

  • on_commit_launcher_wakeup（ファイルスコープの静的変数）

失敗モードと境界条件

  • 特筆すべき失敗モードはない。複数回呼び出された場合でも、フラグは true に維持される。

保守担当者向けメモ

  • 本関数はフラグを立てるのみであり、実際のランチャーの起動処理は、このフラグを監視している別のトランザクション終了処理（AtEOXact_ApplyLauncher 等）によって実行される。
