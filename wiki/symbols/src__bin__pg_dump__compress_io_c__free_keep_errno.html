free_keep_errno

定義

  • 種別: function
  • 定義箇所: src/bin/pg_dump/compress_io.c:177

目的

メモリ解放処理（free）を実行する際、その副作用として発生しうる errno の書き換えを抑制し、呼び出し直前の errno の値を維持するために使用されます。

シグネチャまたは形状

static void free_keep_errno(void *p)

入力と出力

  • **入力**: void *p（解放対象のメモリ領域へのポインタ）
  • **出力**: なし（void）

前提条件と不変条件

  • p は free() に渡すことが可能な有効なポインタ、または NULL である必要があります。
  • 本関数実行後、呼び出し元が参照する errno は、本関数呼び出し直前の値と同一であることが保証されます。

アルゴリズムまたは振る舞い

1. 現在の errno の値をローカル変数 save_errno に退避します。 2. 標準ライブラリの free(p) を呼び出し、メモリを解放します。 3. 退避しておいた save_errno を errno に書き戻します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/bin/pg_dump/compress_io.c 内の各所（fname の解放、CFH 構造体の解放など）。
  • **呼び出し先**: 標準ライブラリの free()。

関連データ構造

  • 特になし（汎用的なポインタを扱うユーティリティ関数です）。

失敗モードと境界条件

  • p が NULL の場合、標準の free() の仕様に従い、何も行われません。
  • free() 自体が内部で errno を変更する可能性がある環境において、その変更を隠蔽します。

保守担当者向けメモ

  • 本関数はエラーハンドリングの整合性を保つためのユーティリティです。メモリ解放を伴うクリーンアップ処理において、直前のエラー状態を保持する必要がある箇所で利用してください。
