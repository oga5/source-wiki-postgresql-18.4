print2

定義

目的

ECPG（Embedded SQL for C）のテストにおいて、SQLエラー発生時に標準エラー出力へ通知を行うためのカスタムエラーハンドラ関数です。

シグネチャまたは形状

static void print2(void)

入力と出力

  • 入力: なし
  • 出力: なし（副作用として標準エラー出力へ文字列を出力）

前提条件と不変条件

  • sqlprint() 関数が利用可能であること。
  • 本関数は、SQL実行後に sqlca.sqlcode が負の値（エラー）であることを確認した後に呼び出されることを想定しています。

アルゴリズムまたは振る舞い

1. 標準エラー出力 (stderr) に "Found another error\n" という文字列を出力します。 2. sqlprint() を呼び出し、ECPGライブラリが保持する詳細なSQLエラー情報を出力します。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/interfaces/ecpg/test/expected/preproc-whenever.c:162 (SQL実行後の条件分岐内)
  • src/interfaces/ecpg/test/expected/preproc-whenever.c:171 (SQL実行後の条件分岐内)
  • プリプロセッサ指令 exec sql whenever sqlerror call print2 ( ) ; の変換結果として参照されています。
  • 呼び出し先:
  • sqlprint() (ECPGライブラリ関数)

関連データ構造

  • sqlca: SQL通信領域。sqlprint() がエラー内容を読み取るために使用します。

失敗モードと境界条件

  • 本関数自体にエラーハンドリングは含まれていません。sqlprint() の実行結果に依存します。
