ItemPointerCopy

定義

  • 種別: static inline function
  • 定義箇所: src/include/storage/itemptr.h:171

目的

ある ItemPointerData 構造体の内容を別の ItemPointerData 構造体へコピーする。

シグネチャまたは形状

static inline void ItemPointerCopy(const ItemPointerData *fromPointer, ItemPointerData *toPointer)

入力と出力

  • fromPointer: コピー元の ItemPointerData へのポインタ（入力）。
  • toPointer: コピー先の ItemPointerData へのポインタ（出力）。
  • 戻り値: なし。

前提条件と不変条件

  • fromPointer および toPointer は、PointerIsValid マクロによって有効なメモリ領域を指していることが検証される（Assert による）。

アルゴリズムまたは振る舞い

1. toPointer が有効なポインタであるか検証する。 2. fromPointer が有効なポインタであるか検証する。 3. fromPointer が指す ItemPointerData の内容を、toPointer が指すメモリ領域へ直接代入（コピー）する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/amcheck/verify_nbtree.c
  • src/backend/access/heap/heapam.c
  • src/backend/access/heap/heapam_handler.c
  • **呼び出し先**:
  • PointerIsValid（マクロ）

関連データ構造

  • ItemPointerData: 物理的なタプル位置を保持する 6 バイトの構造体。

失敗モードと境界条件

  • Assert が有効なビルドにおいて、引数に NULL または無効なポインタが渡された場合、プログラムは異常終了する。

保守担当者向けメモ

  • 本関数はインライン化されており、パフォーマンスが重視されるパスで使用される。
  • ItemPointerData 構造体のメモリレイアウト（6 バイト）に依存したコピーを行うため、構造体の定義変更時には本関数の動作に影響がないか確認が必要である。
