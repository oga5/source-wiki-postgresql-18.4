textregexreplace_extended_no_flags

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/regexp.c:751

目的

SQLの regexp_replace 関数において、フラグ引数が指定されない場合の正規表現置換処理を実行するラッパー関数です。

シグネチャまたは形状

Datum textregexreplace_extended_no_flags(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS（SQL関数呼び出し規約に基づく引数群）。
  • **出力**: Datum 型。置換後の文字列（text型）を返します。

前提条件と不変条件

  • fcinfo に有効な関数呼び出し情報が含まれている必要があります。
  • 内部的に呼び出される textregexreplace_extended が要求する前提条件に依存します。

アルゴリズムまたは振る舞い

1. 受け取った fcinfo をそのまま textregexreplace_extended 関数へ転送します。 2. textregexreplace_extended の戻り値をそのまま呼び出し元へ返却します。

呼び出し元と呼び出し先

  • **呼び出し元**: SQLエンジン（SQLレベルの regexp_replace 関数呼び出し時）。
  • **呼び出し先**: textregexreplace_extended（src/backend/utils/adt/regexp.c 内）。

関連データ構造

  • Datum: PostgreSQLの汎用データ型表現。
  • PG_FUNCTION_ARGS: 関数引数を受け取るためのマクロ定義。

失敗モードと境界条件

  • 内部で呼び出される textregexreplace_extended の失敗モード（正規表現のコンパイルエラーやメモリ不足など）に準じます。

保守担当者向けメモ

  • 本関数は単なる転送用ラッパーです。ロジックの変更が必要な場合は、実体である textregexreplace_extended を修正してください。
