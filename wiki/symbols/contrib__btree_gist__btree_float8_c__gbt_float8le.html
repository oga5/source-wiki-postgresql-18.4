gbt_float8le

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_float8.c:44

目的

2つの float8 値を比較し、左辺が右辺以下であるか（<=）を判定する。

シグネチャまたは形状

bool gbt_float8le(const void *a, const void *b, FmgrInfo *flinfo)

入力と出力

  • **入力**:
  • a: 比較対象の float8 値へのポインタ。
  • b: 比較対象の float8 値へのポインタ。
  • flinfo: 関数呼び出し情報（本関数内では未使用）。
  • **出力**:
  • bool: *a <= *b が真であれば true、そうでなければ false を返す。

前提条件と不変条件

  • a および b は、有効な float8 型の値を指すポインタである必要がある。
  • 本関数は btree_gist モジュール内で数値比較を行うためのユーティリティとして機能する。

アルゴリズムまたは振る舞い

1. ポインタ a および b を float8 型としてキャストし、デリファレンスする。 2. C言語の標準演算子 <= を用いて比較を行い、その結果を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/btree_gist/btree_float8.c 内の tinfo 構造体（gbtree_ninfo）の定義において、比較関数として参照されている。
  • **呼び出し先**: なし（標準の比較演算子のみを使用）。

関連データ構造

  • float8: PostgreSQLの倍精度浮動小数点数型。

失敗モードと境界条件

  • 入力ポインタが NULL の場合、セグメンテーション違反が発生する可能性がある。
  • float8 の比較において NaN が含まれる場合の挙動は、標準の浮動小数点比較規則に従う。

保守担当者向けメモ

  • 本関数は btree_utils_num が提供する汎用的な数値処理ロジックと組み合わせて使用される。
  • btree_float8.c 内の tinfo 構造体を通じて、GiSTインデックスの構築や検索処理から間接的に呼び出される設計となっている。
