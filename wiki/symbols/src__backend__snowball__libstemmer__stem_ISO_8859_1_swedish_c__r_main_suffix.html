r_main_suffix

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_swedish.c:174

目的

スウェーデン語のステミング処理において、単語の末尾から主要な接尾辞を検索し、条件に応じて削除または正規化を行う。

シグネチャまたは形状

static int r_main_suffix(struct SN_env * z)

入力と出力

  • 入力: SN_env * z （Snowballステミング環境構造体）
  • 出力: int （成功時に 1、失敗または処理対象外の場合に 0、エラー発生時に負の値を返す）

前提条件と不変条件

  • z->I[1] に格納された領域境界（R1等）より後ろの文字列を対象とする。
  • z->c（カーソル位置）が z->I[1] 以上である必要がある。

アルゴリズムまたは振る舞い

1. **境界チェック**: カーソル位置が z->I[1] 未満であれば即座に 0 を返す。 2. **接尾辞検索**: a_0 配列を用いて、現在のカーソル位置から後方に向かって接尾辞を検索する。 3. **条件分岐**:

  • among_var == 1: マッチした接尾辞を削除する。
  • among_var == 2: g_s_ending グループに含まれる文字が直前に存在しない場合のみ、接尾辞を削除する。

    • 4. **状態復帰**: 処理中、一時的に z->lb を z->I[1] に変更し、終了後に元の値へ戻す。

呼び出し元と呼び出し先

  • 呼び出し元: swedish_ISO_8859_1_stem（src/backend/snowball/libstemmer/stem_ISO_8859_1_swedish.c 内）
  • 呼び出し先: find_among_b, slice_del, in_grouping_b（いずれもSnowballランタイム関数）

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_0: 接尾辞マッチングに使用される among 構造体の配列。
  • g_s_ending: among_var == 2 の条件判定に使用される文字グループ。

失敗モードと境界条件

  • z->c < z->I[1]: 処理対象領域外のため 0 を返す。
  • find_among_b がマッチしない場合: 0 を返す。
  • slice_del が失敗した場合: 負の値をそのまま伝播させる。

保守担当者向けメモ

  • 本関数は自動生成コードであるため、直接編集せず、元のSnowballスクリプト（.sbl）を修正すること。
  • z->p（文字列バッファ）のビット操作による高速な文字判定が行われているため、変更時はランタイムの仕様を確認すること。
