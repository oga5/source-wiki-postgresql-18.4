r_undouble

定義

  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_hungarian.c:552

目的

単語の末尾にある重複した文字（二重子音など）を削除し、単語を正規化する。

シグネチャまたは形状

static int r_undouble(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z （ステミング実行環境）
  • 出力: 成功時に 1、失敗時に 0 または負の値を返す。

前提条件と不変条件

  • z が有効なステミング環境構造体であること。
  • skip_b_utf8 が単語の末尾から2文字分を正常に遡れること。

アルゴリズムまたは振る舞い

1. skip_b_utf8 を使用して、現在のカーソル位置から末尾へ向かって1文字分遡る。 2. 遡った位置を z->ket（削除対象の末尾）に設定する。 3. 再度 skip_b_utf8 を使用して、さらに1文字分遡る。 4. 遡った位置を z->bra（削除対象の先頭）に設定する。 5. slice_del(z) を呼び出し、z->bra から z->ket までの範囲を削除する。

呼び出し元と呼び出し先

  • 呼び出し元: 複数の言語（ハンガリー語、デンマーク語、オランダ語等）のステミング処理関数から呼び出されることが示唆されている。
  • 呼び出し先: skip_b_utf8 および slice_del（Snowball ランタイム関数）。

関連データ構造

  • struct SN_env: ステミング処理の状態を保持する構造体。

失敗モードと境界条件

  • skip_b_utf8 が負の値を返した場合（単語の先頭に達した等）、処理を中断し 0 を返す。
  • slice_del が負の値を返した場合、その値をそのまま返す。
