ConstraintExclusionType

定義

  • 種別: enum
  • 定義箇所: src/include/optimizer/cost.h:36

目的

クエリ最適化プロセスにおいて、制約排除（Constraint Exclusion）の適用範囲を制御するために使用される列挙型です。

シグネチャまたは形状

typedef enum
{
    CONSTRAINT_EXCLUSION_OFF,       /* 制約排除を使用しない */
    CONSTRAINT_EXCLUSION_ON,        /* すべてのリレーションに制約排除を適用する */
    CONSTRAINT_EXCLUSION_PARTITION, /* otherrels（継承された子テーブル等）のみに適用する */
} ConstraintExclusionType;

入力と出力

  • 入力: なし（列挙定数）
  • 出力: なし（列挙定数）

前提条件と不変条件

  • 本列挙型は、プランナがクエリ実行計画を生成する際の構成パラメータとして参照されます。

アルゴリズムまたは振る舞い

  • CONSTRAINT_EXCLUSION_OFF: 制約排除機能を無効化します。
  • CONSTRAINT_EXCLUSION_ON: 全てのリレーションに対して制約排除を試みます。
  • CONSTRAINT_EXCLUSION_PARTITION: パーティションや継承された子テーブル（otherrels）に対してのみ制約排除を適用します。

呼び出し元と呼び出し先

  • 呼び出し元および呼び出し先の具体的な関数情報は、提供されたソース抜粋からは特定できません。

関連データ構造

  • PlannerInfo（プランナの制御構造）において、制約排除の挙動を決定する設定値として利用されることが想定されます。

失敗モードと境界条件

  • 該当なし。
