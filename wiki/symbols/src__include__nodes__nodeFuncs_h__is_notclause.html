is_notclause

定義

  • 種別: function
  • 定義箇所: src/include/nodes/nodeFuncs.h:126

目的

与えられたノードが、論理否定（NOT）を表す BoolExpr ノードであるかを判定します。

シグネチャまたは形状

static inline bool is_notclause(const void *clause)

入力と出力

  • **入力**: clause (任意のノードへのポインタ)
  • **出力**: bool 型。ノードが NOT 演算子を持つ BoolExpr であれば true、それ以外（NULL を含む）であれば false を返します。

前提条件と不変条件

  • clause が指すメモリ領域は、Node 構造体として有効である必要があります。
  • IsA マクロにより、ノードの型タグが T_BoolExpr であることが検証されます。

アルゴリズムまたは振る舞い

1. clause が NULL でないことを確認します。 2. IsA(clause, BoolExpr) を用いて、ノードが BoolExpr 型であることを確認します。 3. BoolExpr 構造体にキャストし、boolop フィールドが NOT_EXPR と一致するかを評価して結果を返します。

関連データ構造

  • BoolExpr: NOT 演算子を含む論理式を保持する構造体。
  • Node: PostgreSQLのツリー構造の基本型。

失敗モードと境界条件

  • clause が NULL の場合、安全に false を返します。
  • BoolExpr 以外のノード型が渡された場合、IsA チェックにより false を返します。
