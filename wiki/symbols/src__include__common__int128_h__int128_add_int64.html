int128_add_int64

定義

  • 種別: function
  • 定義箇所: src/include/common/int128.h:49

目的

INT128 型の変数に対し、64ビット整数を加算するインライン関数です。

シグネチャまたは形状

static inline void int128_add_int64(INT128 *i128, int64 v)

入力と出力

  • **入力**:
  • i128: 加算対象となる INT128 型変数へのポインタ。
  • v: 加算する int64 型の値。
  • **出力**: なし（i128 が指す値を直接更新します）。

前提条件と不変条件

  • i128 は有効な INT128 型のメモリ領域を指している必要があります。
  • INT128 型は、プラットフォームに応じてネイティブな int128 型、または hi/lo 構造体として定義されます。

アルゴリズムまたは振る舞い

  • i128 が指す値に対して、v を加算します。
  • ネイティブ環境（USE_NATIVE_INT128 が有効）では、コンパイラの組み込み演算子 += を使用します。
  • ソフトウェア実装環境では、lo（下位64ビット）への加算を行い、キャリーが発生した場合は hi（上位64ビット）を調整する処理が行われます。

呼び出し元と呼び出し先

  • src/include/common/int128.h:50

  • **呼び出し先**:
  • 本関数はインライン関数であり、特定の関数を呼び出すのではなく、コンパイラによる演算処理へ直接展開されます。

関連データ構造

  • INT128: 128ビット整数を保持する型。

失敗モードと境界条件

  • 128ビットの範囲を超える加算が発生した場合のオーバーフロー動作は、符号付き整数のオーバーフローに関するC言語の標準的な挙動（未定義動作の可能性がある）に従います。

保守担当者向けメモ

  • 本関数はパフォーマンスを重視し static inline として定義されています。
  • ソフトウェア実装環境では、エンディアンを考慮した hi/lo の配置がなされています。移植性を維持するため、構造体定義の変更時には注意が必要です。
