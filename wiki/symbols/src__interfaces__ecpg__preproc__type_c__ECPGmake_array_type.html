ECPGmake_array_type

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/preproc/type.c:86

目的

ECPGプリプロセッサにおいて、C言語の配列型を表現するための ECPGtype 構造体を生成するファクトリ関数です。

シグネチャまたは形状

struct ECPGtype *ECPGmake_array_type(struct ECPGtype *type, const char *size);

入力と出力

  • **入力**:
  • type: 配列の要素型を定義する ECPGtype 構造体へのポインタ。
  • size: 配列のサイズを表す文字列。
  • **出力**:
  • 配列型として初期化された ECPGtype 構造体へのポインタ。

前提条件と不変条件

  • ECPGmake_simple_type が内部で呼び出され、ECPGt_array 型タグを持つ構造体が割り当てられます。
  • 戻り値の構造体は、u.element メンバに引数 type を保持します。

アルゴリズムまたは振る舞い

1. ECPGmake_simple_type(ECPGt_array, size, 0) を呼び出し、配列用の基本構造体を生成します。 2. 生成された構造体の u.element メンバに、引数で渡された要素型 type を代入します。 3. 構築された ECPGtype 構造体を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/interfaces/ecpg/preproc/type.c:36
  • src/interfaces/ecpg/preproc/variable.c:47, 94, 242
  • **呼び出し先**:
  • ECPGmake_simple_type

関連データ構造

  • ECPGtype: 型情報を保持する中心的な構造体。

失敗モードと境界条件

  • メモリ割り当て失敗時の挙動については、mm_alloc 等のラッパー関数に依存します（ソースコード上での明示的なエラーハンドリングは記述されていません）。

保守担当者向けメモ

  • 本関数は ECPGtype の構築プロセスの一部です。生成された構造体は、最終的に ECPGfree_type を用いて再帰的に解放される必要があります。
  • size 引数は文字列として扱われるため、プリプロセッサが解析した配列サイズがそのまま渡されます。
