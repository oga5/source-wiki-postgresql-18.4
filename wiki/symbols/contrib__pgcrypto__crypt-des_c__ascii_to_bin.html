ascii_to_bin

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/crypt-des.c:202

目的

Base64 風の文字セット（., /, 0-9, A-Z, a-z）を対応する 6 ビットのバイナリ値に変換します。

シグネチャまたは形状

static inline int ascii_to_bin(char ch)

入力と出力

  • **入力**: char ch （変換対象の文字）
  • **出力**: int （0 から 63 までの変換後の値、または無効な文字の場合は 0）

前提条件と不変条件

  • 入力文字が定義された文字セット外である場合、戻り値は 0 となります。
  • 文字セットの対応関係:
  • . → 0
  • 0-9 → 2-11
  • A-Z → 12-37
  • a-z → 38-63

アルゴリズムまたは振る舞い

1. 文字 ch が 'z' より大きい場合は 0 を返します。 2. 'a' 以上であれば ch - 'a' + 38 を返します。 3. 'Z' より大きい場合は 0 を返します。 4. 'A' 以上であれば ch - 'A' + 12 を返します。 5. '9' より大きい場合は 0 を返します。 6. '.' 以上であれば ch - '.' を返します。 7. 上記のいずれにも該当しない場合は 0 を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/pgcrypto/crypt-des.c 内の複数箇所（行 700, 703, 747, 748）で、ソルトや設定値のデコードに使用されています。
  • **呼び出し先**:
  • なし（標準ライブラリ関数等の呼び出しはありません）。

関連データ構造

  • なし。

失敗モードと境界条件

  • 文字セットに含まれない文字が入力された場合、エラーを通知せず 0 を返します。
  • '.' より小さい文字（例: ! や ）が入力された場合も 0 を返します。

保守担当者向けメモ

  • 本関数は _crypt_a64 文字セットを前提としています。この文字セットの仕様変更がない限り、ロジックの変更は不要です。
  • inline 指定されているため、コンパイル時に呼び出し元へ展開されます。
