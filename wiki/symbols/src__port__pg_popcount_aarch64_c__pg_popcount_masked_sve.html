pg_popcount_masked_sve

定義

  • 種別: function
  • 定義箇所: src/port/pg_popcount_aarch64.c:192

目的

AArch64 SVE（Scalable Vector Extension）命令セットを利用し、指定されたマスクでフィルタリングされたバッファ内のビットカウント（popcount）を高速に計算します。

シグネチャまたは形状

uint64 pg_popcount_masked_sve(const char *buf, int bytes, bits8 mask)

入力と出力

  • **入力**:
  • buf: カウント対象のデータバッファへのポインタ。
  • bytes: バッファのサイズ（バイト単位）。
  • mask: 各バイトに適用するビットマスク。
  • **出力**:
  • マスクされたデータ内のセットビット（1）の総数を uint64 で返します。

前提条件と不変条件

  • POPCNT_AARCH64 が定義された環境で利用可能です。
  • 実行時にSVE命令セットがサポートされている必要があります。

アルゴリズムまたは振る舞い

1. **初期化**: SVEレジスタ幅（svcntb()）に基づき、4倍のレジスタ幅を一度に処理するループを展開します。 2. **メインループ**: 4つのアキュムレータ（accum1〜accum4）を使用し、命令レベルの並列性を高めて処理します。各反復で svand_n_u64_x によりマスクを適用し、svcnt_u64_x でビット数を加算します。 3. **中間処理**: 残りのデータが2レジスタ幅以上ある場合、同様の処理を2レジスタ分行います。 4. **集計**: 4つのアキュムレータの値を svaddv_u64 で合計します。 5. **端数処理**: 残りのデータに対して svwhilelt_b8_s32 で述語（predicate）を生成し、逐次的にビットカウントを行い、最終結果に加算します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/port/pg_popcount_aarch64.c 内で pg_popcount_masked_optimized 関数ポインタに代入されています。
  • **呼び出し先**: SVE命令セット（svptrue_b64, svdup_u64, svcntb, svand_n_u64_x, svld1_u64, svadd_u64_x, svcnt_u64_x, svaddv_u64, svwhilelt_b8_s32, svand_n_u8_x, svld1_u8, svaddv_u8, svcnt_u8_x）を直接呼び出します。

関連データ構造

  • svbool_t, svuint64_t, svuint8_t: SVE命令セットで使用されるベクトル型。

失敗モードと境界条件

  • bytes が 0 の場合、ループ処理はスキップされ 0 を返します。
  • SVE非対応環境で呼び出された場合の動作は、本関数単体では保護されておらず、上位のディスパッチャ（pg_popcount_masked_choose 等）による判定に依存します。

保守担当者向けメモ

  • 本関数は pg_popcount_masked_optimized を介して動的に選択されます。
  • ループ展開によるパイプライン効率化が図られているため、修正時はレジスタ依存関係に注意してください。
