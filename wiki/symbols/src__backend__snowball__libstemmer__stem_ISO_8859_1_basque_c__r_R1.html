目的

現在のカーソル位置が、単語内の「R1」領域の開始位置以降にあるかどうかを判定します。

シグネチャまたは形状

static int r_R1(struct SN_env * z)

  • **入力**: SN_env * z（ステミング環境構造体）。z->I[1] にR1領域の開始インデックスが、z->c に現在のカーソル位置が格納されています。
  • **出力**: 判定結果を整数で返します。条件を満たす場合は非ゼロ（真）、満たさない場合はゼロ（偽）を返します。

前提条件と不変条件

  • r_mark_regions 関数が事前に呼び出され、z->I[1] に有効なR1領域の境界値が設定されている必要があります。

1. ステミング環境 z 内の整数配列 I のインデックス1（R1領域の開始位置）と、現在のカーソル位置 c を比較します。 2. z->I[1] <= z->c が成立する場合、現在のカーソル位置はR1領域内またはその境界上にあります。 3. 比較結果をそのまま戻り値として返します。
