mic2euc_cn

定義

  • 種別: function
  • 定義箇所: src/backend/utils/mb/conversion_procs/euc_cn_and_mic/euc_cn_and_mic.c:122

目的

MULE_INTERNAL形式の文字列をEUC_CN形式に変換する内部変換関数。

シグネチャまたは形状

static int mic2euc_cn(const unsigned char *mic, unsigned char *p, int len, bool noError)

入力と出力

  • **入力**:
  • mic: 変換元のMULE_INTERNAL形式文字列へのポインタ。
  • p: 変換後のEUC_CN形式を格納するバッファへのポインタ。
  • len: 変換対象のバイト長。
  • noError: エラー発生時に例外を投げず処理を中断するかどうかを示すフラグ。
  • **出力**:
  • 戻り値: 処理したバイト数（int）。

前提条件と不変条件

  • mic は有効なMULE_INTERNAL形式のバイト列であること。
  • p は変換後の結果を格納するのに十分な領域を確保していること。
  • 変換ループ終了後、p の末尾にはヌル終端文字 \0 が付与される。

アルゴリズムまたは振る舞い

1. len が0になるまで以下のループを実行する。 2. 高ビットが立っている場合（マルチバイト文字）:

  • 識別子が LC_GB2312_80 であるか確認し、異なる場合はエラー処理を行う。
  • 後続の2バイトが共に高ビットであることを確認する。
  • 識別子をスキップし、後続の2バイトを p にコピーして進める。

    • 3. 高ビットが立っていない場合（ASCII文字）:

  • 0（ヌル文字）でないことを確認し、そのまま p にコピーする。

    • 4. noError が true の場合、不正なシーケンスや変換不可文字に遭遇した時点でループを中断する。 5. 最後に p に \0 を書き込み、処理した総バイト数を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: mic_to_euc_cn 関数（src/backend/utils/mb/conversion_procs/euc_cn_and_mic/euc_cn_and_mic.c:70）。
  • **呼び出し先**:
  • report_untranslatable_char（エラー時）
  • report_invalid_encoding（エラー時）

関連データ構造

  • LC_GB2312_80: MULE_INTERNALにおけるGB2312文字セットの識別子。
  • PG_MULE_INTERNAL, PG_EUC_CN: エンコーディング識別子。

失敗モードと境界条件

  • noError が false の場合、不正なエンコーディングや変換不可能な文字に遭遇すると例外を発生させる。
  • noError が true の場合、エラー発生時に処理を中断し、それまでに変換できた分のみを返す。
  • 入力文字列が LC_GB2312_80 で始まるが、後続のバイトが不足している場合や高ビットが立っていない場合は不正なエンコーディングとして扱われる。

保守担当者向けメモ

  • 本関数は静的関数であり、モジュール内での変換ロジックを担う。
  • IS_HIGHBIT_SET マクロを使用してバイトの属性を判定している。
  • 変換後のバッファ p の末尾に常にヌル終端を付与する仕様である点に注意すること。
