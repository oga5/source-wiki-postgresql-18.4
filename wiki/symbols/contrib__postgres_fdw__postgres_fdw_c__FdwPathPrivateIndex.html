FdwPathPrivateIndex

定義

  • 種別: enum
  • 定義箇所: contrib/postgres_fdw/postgres_fdw.c:285

目的

postgres_fdw において、ForeignPath ノードの fdw_private リストに格納される要素のインデックスを定義します。

シグネチャまたは形状

enum FdwPathPrivateIndex {
    FdwPathPrivateHasFinalSort,
    FdwPathPrivateHasLimit,
};

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: fdw_private リスト内の特定位置を指す整数値（0 または 1）

前提条件と不変条件

  • fdw_private リストに格納される要素の順序は、この列挙型の定義順序と厳密に一致している必要があります。

アルゴリズムまたは振る舞い

  • FdwPathPrivateHasFinalSort: リモート側で最終的なソートが実行されるかを示す Boolean ノードを保持します。
  • FdwPathPrivateHasLimit: リモート側で LIMIT 句が適用されるかを示す Boolean ノードを保持します。

呼び出し元と呼び出し先

  • 呼び出し元: contrib/postgres_fdw/postgres_fdw.c 内のパス生成ロジック（7010行目、7263行目付近）において、fdw_private リストへのアクセス時に使用されます。
  • 呼び出し先: 情報なし。

関連データ構造

  • ForeignPath（PostgreSQL プランナのノード構造体）: fdw_private フィールドを通じて本列挙型が参照されます。

失敗モードと境界条件

  • リストの順序が定義と一致しない場合、誤ったデータ型（Boolean 以外のノード）を読み込むリスクがあります。

保守担当者向けメモ

  • fdw_private リストの要素を追加・変更する場合は、必ずこの列挙型を更新し、リスト操作を行う箇所（特に postgres_fdw.c 内のリスト構築および参照箇所）との整合性を確認してください。
