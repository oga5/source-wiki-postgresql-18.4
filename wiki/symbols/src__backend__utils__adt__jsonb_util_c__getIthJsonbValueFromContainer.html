getIthJsonbValueFromContainer

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/jsonb_util.c:471

目的

JsonbContainer（JSONB 配列）から、指定されたインデックス i に位置する要素を抽出し、メモリ上の JsonbValue 構造体として返します。

シグネチャまたは形状

JsonbValue *getIthJsonbValueFromContainer(JsonbContainer *container, uint32 i)

入力と出力

  • **入力**:
  • container: 対象となる JsonbContainer へのポインタ。
  • i: 取得したい要素のインデックス（0 始まり）。
  • **出力**:
  • 指定したインデックスの要素を格納した JsonbValue 構造体へのポインタ。
  • インデックスが範囲外の場合は NULL を返します。

前提条件と不変条件

  • container は JsonContainerIsArray マクロにより、JSONB 配列であることが保証されていなければなりません。配列でない場合はエラー（elog(ERROR)）となります。

アルゴリズムまたは振る舞い

1. JsonContainerIsArray を用いて、入力が配列であることを検証します。 2. JsonContainerSize で配列の要素数 nelements を取得します。 3. インデックス i が nelements 以上であれば、範囲外として NULL を返します。 4. palloc を使用して JsonbValue 構造体用のメモリを確保します。 5. getJsonbOffset を用いて対象要素の物理的なオフセットを特定し、fillJsonbValue を呼び出して JsonbValue 構造体へ値を展開します。 6. 構築した JsonbValue へのポインタを返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/jsonfuncs.c
  • src/backend/utils/adt/jsonpath_exec.c
  • **呼び出し先**:
  • JsonContainerIsArray (マクロ)
  • JsonContainerSize (マクロ)
  • palloc
  • getJsonbOffset
  • fillJsonbValue

関連データ構造

  • JsonbContainer: JSONB のバイナリコンテナ構造。
  • JsonbValue: メモリ上で JSONB の各要素（スカラー、オブジェクト、配列）を表現する構造体。

失敗モードと境界条件

  • **エラー**: container が配列型でない場合、elog(ERROR) によりトランザクションが中断されます。
  • **境界条件**: i >= nelements の場合、関数は NULL を返します。

保守担当者向けメモ

  • 本関数は palloc を使用してメモリを確保するため、呼び出し側で適切にメモリ管理（またはコンテキストの解放）を行う必要があります。
  • fillJsonbValue への依存により、JSONB の内部バイナリ形式（JEntry 配列とデータ部）の変更に影響を受けます。
