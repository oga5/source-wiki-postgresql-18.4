check_uescapechar

定義

  • 種別: function
  • 定義箇所: src/backend/parser/parser.c:351

目的

Unicodeエスケープ構文（U&'...'）で使用されるエスケープ文字として、指定された文字が妥当かどうかを判定します。

シグネチャまたは形状

static bool check_uescapechar(unsigned char escape)

入力と出力

  • **入力**: escape (unsigned char) - 判定対象の文字。
  • **出力**: bool - 指定された文字がエスケープ文字として使用可能な場合は true、不適切な場合は false を返します。

前提条件と不変条件

  • 入力文字が16進数（0-9, a-f, A-F）、+、'、"、または空白文字（scanner_isspace が真となる文字）である場合、エスケープ文字としては使用できません。

アルゴリズムまたは振る舞い

1. 入力された escape が以下のいずれかに該当するかを確認します。

  • isxdigit(escape)（16進数）
  • +
  • '
  • "
  • scanner_isspace(escape)（空白文字）

    • 2. 上記のいずれかに該当する場合は false を返し、それ以外の場合は true を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/parser/parser.c:277（エスケープ文字の妥当性検証）
  • src/interfaces/ecpg/preproc/parser.c:198（ECPGプリプロセッサ内での検証）
  • **呼び出し先**:
  • isxdigit (標準ライブラリ)
  • scanner_isspace (ソース内の補助関数)

関連データ構造

  • 特になし。

失敗モードと境界条件

  • 入力文字が上記「前提条件」で挙げた禁止文字セットに含まれる場合、関数は false を返します。これは呼び出し側でエラーハンドリングのトリガーとなります。
