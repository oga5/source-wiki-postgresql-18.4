multirange_get_bounds_offset

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/multirangetypes.c:674

目的

マルチレンジ型データ内の特定のインデックス i に対応する境界値（bounds）のオフセットを計算します。

シグネチャまたは形状

static uint32 multirange_get_bounds_offset(const MultirangeType *multirange, int32 i)

入力と出力

  • **入力**:
  • multirange: 対象となる MultirangeType 構造体へのポインタ。
  • i: 境界値のオフセットを求める対象のインデックス。
  • **出力**:
  • uint32: 指定されたインデックスまでの境界値の累積オフセット長。

前提条件と不変条件

  • multirange は有効な MultirangeType 構造体を指している必要があります。
  • マルチレンジ型は可変長データであるため、境界値へのアクセスには本関数のようなオフセット計算が不可欠です。

アルゴリズムまたは振る舞い

1. MultirangeGetItemsPtr マクロを使用して、マルチレンジ内のアイテムポインタ配列を取得します。 2. インデックス i から開始し、0 に達するまでループします。 3. 各ステップで MULTIRANGE_ITEM_GET_OFFLEN を使用してアイテムのオフセット長を累積します。 4. MULTIRANGE_ITEM_HAS_OFF マクロにより、オフセットが明示的に保持されている箇所に到達した時点でループを終了し、累積値を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/multirangetypes.c:711
  • src/backend/utils/adt/multirangetypes.c:761
  • **呼び出し先**:
  • MultirangeGetItemsPtr (マクロ)
  • MULTIRANGE_ITEM_GET_OFFLEN (マクロ)
  • MULTIRANGE_ITEM_HAS_OFF (マクロ)

関連データ構造

  • MultirangeType: マルチレンジ型のメモリ上の表現。
  • RangeBound: レンジの境界値を定義する構造体。

失敗モードと境界条件

  • インデックス i が負の値や範囲外である場合の挙動については、ソースコード内に明示的なガード条件は記述されていません。

保守担当者向けメモ

  • 本関数は、マルチレンジの内部レイアウトが可変長であることに起因するオフセット計算を抽象化しています。
  • 境界値の抽出ロジックを変更する際は、MULTIRANGE_ITEM 関連マクロの定義と整合性が取れていることを確認してください。
