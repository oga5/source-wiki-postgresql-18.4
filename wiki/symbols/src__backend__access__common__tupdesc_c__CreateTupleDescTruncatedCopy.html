CreateTupleDescTruncatedCopy

定義

  • 種別: function
  • 定義箇所: src/backend/access/common/tupdesc.c:295

目的

既存のタプル記述子（TupleDesc）から、指定した列数（natts）分だけを切り出した新しいタプル記述子を作成する。制約やデフォルト値はコピーされない。

シグネチャまたは形状

TupleDesc CreateTupleDescTruncatedCopy(TupleDesc tupdesc, int natts)

入力と出力

  • **入力**:
  • tupdesc: 元となるタプル記述子。
  • natts: 切り出す列数。
  • **出力**:
  • 新規に割り当てられた TupleDesc 構造体。

前提条件と不変条件

  • natts <= tupdesc->natts であること（Assert により検証）。
  • 戻り値の TupleDesc は、呼び出し元で適切に解放する必要がある。

アルゴリズムまたは振る舞い

1. CreateTemplateTupleDesc(natts) を呼び出し、新しい記述子用のメモリを確保する。 2. 元の記述子から属性配列（FormData_pg_attribute）の先頭 natts 分を memcpy でコピーする。 3. コピーされた各属性について、制約やデフォルト値に関連するフィールド（attnotnull, atthasdef, atthasmissing, attidentity, attgenerated）を初期化（クリア）する。 4. 各属性に対して populate_compact_attribute を呼び出し、最適化用メタデータを更新する。 5. 元の記述子から型識別子（tdtypeid, tdtypmod）をコピーし、新しい記述子を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/pageinspect/gistfuncs.c
  • src/backend/access/common/indextuple.c
  • src/backend/access/gist/gist.c
  • **呼び出し先**:
  • CreateTemplateTupleDesc
  • TupleDescAttr
  • populate_compact_attribute

関連データ構造

  • TupleDesc: タプルの構造を定義するメタデータ構造体。
  • FormData_pg_attribute: 各列の属性情報を保持する構造体。

失敗モードと境界条件

  • natts が元の記述子の列数を超えている場合、Assert が有効なビルドではアボートする。
  • メモリ不足時には CreateTemplateTupleDesc 経由でエラーが発生する可能性がある。

保守担当者向けメモ

  • 本関数は制約（TupleConstr）をコピーしないため、制約情報が必要な場合は別の関数を検討すること。
  • populate_compact_attribute を呼び出しているため、属性情報の変更後は常に最新の最適化用メタデータが同期される。
