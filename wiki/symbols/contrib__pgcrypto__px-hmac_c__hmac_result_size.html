hmac_result_size

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/px-hmac.c:39

目的

HMAC 計算に使用されているハッシュアルゴリズムの出力サイズ（バイト単位）を取得します。

シグネチャまたは形状

static unsigned hmac_result_size(PX_HMAC *h)

入力と出力

  • **入力**: PX_HMAC *h （HMAC コンテキスト構造体へのポインタ）
  • **出力**: unsigned （ハッシュアルゴリズムのダイジェストサイズ）

前提条件と不変条件

  • 引数 h は有効な PX_HMAC 構造体を指している必要があります。
  • 構造体内の md メンバ（メッセージダイジェストコンテキスト）が初期化されている必要があります。

アルゴリズムまたは振る舞い

内部で保持しているハッシュアルゴリズムのコンテキスト h->md を参照し、px_md_result_size 関数を呼び出してその結果を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/pgcrypto/px-hmac.c:165 （px_find_hmac 関数内にて、PX_HMAC 構造体の result_size フィールドに関数ポインタとして代入される）
  • **呼び出し先**: px_md_result_size （px.h で定義された抽象化レイヤーの関数）

関連データ構造

  • PX_HMAC: HMAC 操作の状態を保持する構造体。md メンバを通じてハッシュアルゴリズムのコンテキストを管理します。

失敗モードと境界条件

  • h が NULL の場合、デリファレンスによりクラッシュする可能性があります。

保守担当者向けメモ

  • 本関数は PX_HMAC 構造体のメソッドとして機能するように設計されています。
  • px_md_result_size が返す値は、使用するハッシュアルゴリズム（SHA-256, MD5 等）に依存します。
