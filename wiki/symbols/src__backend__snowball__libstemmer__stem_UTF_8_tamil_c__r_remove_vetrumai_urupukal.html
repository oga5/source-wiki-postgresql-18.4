r_remove_vetrumai_urupukal

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_tamil.c:1108

目的

タミル語の格変化接尾辞（Vetrumai Urupukal）を単語末尾から除去し、語幹を正規化する。

シグネチャまたは形状

static int r_remove_vetrumai_urupukal(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング環境構造体。処理対象の文字列を含む）
  • 出力: int（成功時に 1、失敗または処理中断時に 0 または負の値を返す）

前提条件と不変条件

  • r_has_min_length(z) が正の値を返すこと（最小長を満たすこと）。
  • z->lb および z->c（カーソル位置）が適切に設定されていること。
  • 処理終了時に z->I[0] が 1 に設定される。

アルゴリズムまたは振る舞い

1. r_has_min_length を実行し、条件を満たさない場合は即座に終了する。 2. カーソルを単語末尾に移動し、find_among_b を用いて a_20 配列内のパターンと照合する。 3. マッチしたパターンに応じて slice_del（削除）または slice_from_s（置換）を実行する。 4. 特定のケースでは、find_among_b を用いた追加の条件判定（a_18, a_19 等）を行い、マッチしない場合はバックトラックする。 5. a_20 の照合に失敗した場合、s_32 との照合および a_21, a_22 を用いた条件分岐を試みる。 6. 最後に s_35 との照合を行い、必要に応じて置換を行う。 7. 最後に r_fix_endings(z) を呼び出し、語幹の整合性を確保して終了する。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_tamil.c:1387（tamil_UTF_8_stem 内から呼び出される）
  • 呼び出し先: r_has_min_length, find_among_b, slice_del, slice_from_s, eq_s_b, len_utf8, r_fix_endings

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_18 ～ a_22: 接尾辞パターンを定義した among 構造体の配列。

失敗モードと境界条件

  • r_has_min_length が失敗した場合、処理は中断される。
  • slice_del や slice_from_s が負の値を返した場合、エラーとして呼び出し元へ伝播する。
  • UTF-8 文字列の長さが不足している場合（len_utf8 チェック）、特定の置換処理がスキップされる。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードである。ロジックの変更は .sbl ソースファイルに対して行うこと。
  • among 配列のインデックスや s_* 定数はタミル語の UTF-8 バイトシーケンスに依存しているため、直接編集しないこと。
