Jsonb_to_SV

定義

  • 種別: function
  • 定義箇所: contrib/jsonb_plperl/jsonb_plperl.c:61

目的

PostgreSQL の JsonbContainer 構造体を、Perl のデータ構造（SV）へ再帰的に変換します。

シグネチャまたは形状

static SV *Jsonb_to_SV(JsonbContainer *jsonb)

入力と出力

  • **入力**: JsonbContainer *jsonb（変換対象の JSONB データ）
  • **出力**: SV *（変換された Perl のスカラー、配列リファレンス、またはハッシュリファレンス）

前提条件と不変条件

  • dTHX マクロにより Perl インタプリタのコンテキストが初期化されている必要があります。
  • JsonbIterator を使用して JSONB の各ノードを走査します。

アルゴリズムまたは振る舞い

1. JsonbIteratorInit でイテレータを初期化し、最初のトークンを取得します。 2. トークンの種類に応じて以下の処理を行います：

  • **WJB_BEGIN_ARRAY**:
  • rawScalar の場合: 配列内の単一要素を取り出し、JsonbValue_to_SV を呼び出してスカラーとして返します。
  • 通常の配列の場合: newAV() で Perl 配列を作成し、各要素を JsonbValue_to_SV で変換して av_push し、リファレンスを返します。
  • **WJB_BEGIN_OBJECT**:
  • newHV() で Perl ハッシュを作成します。
  • WJB_KEY トークンを検出し、続く WJB_VALUE を JsonbValue_to_SV で変換して hv_store によりハッシュに格納し、リファレンスを返します。

    • 3. 予期しないトークンが検出された場合は elog(ERROR) を発生させます。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/jsonb_plperl/jsonb_plperl.c 内の関数（jsonb_to_plperl 等）から呼び出されます。
  • **呼び出し先**:
  • JsonbIteratorInit, JsonbIteratorNext (PostgreSQL JSONB API)
  • JsonbValue_to_SV (再帰的な変換用ヘルパー)
  • newAV, av_push, newHV, hv_store, newRV (Perl API)

関連データ構造

  • JsonbContainer: PostgreSQL 側の JSONB 内部表現。
  • JsonbIterator: JSONB 走査用イテレータ。
  • SV, AV, HV: Perl のスカラー、配列、ハッシュ構造体。

失敗モードと境界条件

  • 予期しない JSONB トークンが検出された場合、elog(ERROR) により処理が中断されます。
  • 再帰的な構造を持つため、極端に深いネストを持つ JSONB データに対してはスタックオーバーフローのリスクがあります。

保守担当者向けメモ

  • 本関数は JsonbValue_to_SV と相互再帰的に動作する設計です。
  • Perl のメモリ管理（newAV, newHV 等）と PostgreSQL のメモリ管理が混在するため、メモリリークには注意が必要です。
