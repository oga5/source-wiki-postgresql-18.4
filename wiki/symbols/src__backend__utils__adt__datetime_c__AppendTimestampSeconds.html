AppendTimestampSeconds

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/datetime.c:520

目的

タイムスタンプの秒および小数秒部分を文字列バッファに書き込み、更新されたバッファポインタを返します。

シグネチャまたは形状

char *AppendTimestampSeconds(char *cp, struct pg_tm *tm, fsec_t fsec)

入力と出力

  • **入力**:
  • cp: 書き込み先の文字列バッファポインタ。
  • tm: 秒情報（tm_sec）を含む pg_tm 構造体へのポインタ。
  • fsec: 小数秒を表す fsec_t 型の値。
  • **出力**:
  • 書き込み後のバッファ位置を示す char * ポインタ。

前提条件と不変条件

  • tm は有効な pg_tm 構造体を指している必要があります。
  • MAX_TIMESTAMP_PRECISION が秒の精度として使用されます。

アルゴリズムまたは振る舞い

内部的に AppendSeconds 関数を呼び出します。tm->tm_sec（秒）と fsec（小数秒）を、MAX_TIMESTAMP_PRECISION の精度で cp が指すバッファに書き込みます。この際、小数秒の表示を有効にするフラグ（true）が渡されます。

呼び出し元と呼び出し先

  • **呼び出し先**: AppendSeconds
  • **呼び出し元**: src/backend/utils/adt/datetime.c 内の複数箇所（行 4492, 4519, 4552, 4593 など）から呼び出されています。

関連データ構造

  • struct pg_tm: 日付・時刻の構成要素を保持する構造体。
  • fsec_t: 小数秒を保持するための型。

失敗モードと境界条件

  • cp が指すバッファが、書き込まれる秒文字列（および小数秒）を格納するのに十分なサイズであることを前提としています。バッファオーバーフローのチェックは本関数内では行われません。

保守担当者向けメモ

  • 本関数は AppendSeconds のラッパーです。秒の書式化ロジックを変更する場合は AppendSeconds を確認してください。
  • 精度定数 MAX_TIMESTAMP_PRECISION は utils/datetime.h で定義されています。
