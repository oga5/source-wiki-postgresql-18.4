SetEpochTimestamp

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/pgtypeslib/timestamp.c:65

目的

システムのエポック（Unixエポック）に対応する timestamp 値を取得します。

シグネチャまたは形状

static timestamp SetEpochTimestamp(void)

入力と出力

  • **入力**: なし
  • **出力**: timestamp 型（内部的には int64 のマイクロ秒単位）。エポック時刻を表現する値。

前提条件と不変条件

  • GetEpochTime() が正常に時刻を取得できる必要があります。
  • 内部的な変換関数 tm2timestamp() が成功することが前提となります。

アルゴリズムまたは振る舞い

1. GetEpochTime() を呼び出し、現在のエポック時刻を struct tm 構造体に格納します。 2. GetEpochTime() が負の値を返した場合は、エラーとして 0 を返します。 3. 成功した場合、tm2timestamp() を使用して struct tm を timestamp 型に変換します。 4. 変換された timestamp 値を返します。

呼び出し元と呼び出し先

  • **呼び出し先**: GetEpochTime(), tm2timestamp()
  • **呼び出し元**:
  • 同一ファイル内: src/interfaces/ecpg/pgtypeslib/timestamp.c:241, src/interfaces/ecpg/pgtypeslib/timestamp.c:547
  • 外部（ツールによる推論）: src/backend/utils/adt/timestamp.c 内の複数箇所（注：これらはバックエンド側の実装であり、ECPGライブラリの static 関数である本シンボルを直接呼び出しているかはソースの文脈上確認が必要です）。

関連データ構造

  • timestamp: int64（マイクロ秒単位）
  • struct tm: C標準の日時構造体

失敗モードと境界条件

  • GetEpochTime() が失敗（負の値を返却）した場合、関数は 0 を返します。
  • tm2timestamp() の計算結果が有効範囲外の場合、その挙動は tm2timestamp() の仕様に依存します。

保守担当者向けメモ

  • 本関数は static であり、timestamp.c 内でのみ利用されることを意図しています。
  • 戻り値の 0 はエラー状態を示すため、呼び出し側ではこの値を有効なタイムスタンプとして扱わないよう注意が必要です。
