transformMergeStmt

定義

  • 種別: function
  • 定義箇所: src/backend/parser/parse_merge.c:106

目的

MERGE 文の構文解析結果（MergeStmt）を、実行可能なクエリツリー（Query）へ変換します。

シグネチャまたは形状

Query *transformMergeStmt(ParseState *pstate, MergeStmt *stmt)

入力と出力

  • **入力**:
  • pstate: 解析状態を保持する ParseState 構造体。
  • stmt: 解析対象の MergeStmt 構文木。
  • **出力**: 変換後の Query 構造体。

前提条件と不変条件

  • pstate->p_ctenamespace が空（NIL）であること（MERGE 文の外側に WITH 句が存在しないことを前提とする）。
  • ターゲットリレーションがテーブル、パーティションテーブル、またはビューのいずれかであること。
  • ターゲットテーブルとソースリレーションのエイリアス名が重複していないこと。

アルゴリズムまたは振る舞い

1. **初期化**: Query ノードを作成し、WITH 句を処理（再帰的 WITH はエラー）。 2. **検証**: 各 WHEN 句の権限（ACL_INSERT, UPDATE, DELETE, SELECT）を収集し、到達不能な句がないかチェック。 3. **ターゲット設定**: setTargetTable を呼び出し、ターゲットリレーションを確定。 4. **ソース変換**: transformFromClause でソースリレーションを解析し、名前空間に追加。 5. **結合条件の変換**: addNSItemToQuery でターゲットを名前空間に追加後、transformExpr で結合条件を変換。 6. **RETURNING 句の変換**: transformReturningClause を実行。 7. **アクション変換**: 各 WHEN 句に対し、setNamespaceForMergeWhen で可視性を切り替えながら MergeAction を生成。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/parser/analyze.c:364
  • **呼び出し先**:
  • transformWithClause, setTargetTable, transformFromClause, GetNSItemByRangeTablePosn, addNSItemToQuery, transformExpr, makeFromExpr, transformReturningClause, setNamespaceForMergeWhen

関連データ構造

  • Query: 変換後のクエリツリー。
  • MergeStmt: 入力となる構文木。
  • MergeAction: 各 WHEN 句に対応するアクション情報。
  • ParseState: 解析状態管理用。

失敗モードと境界条件

  • WITH RECURSIVE が指定された場合、エラーを報告。
  • ターゲットリレーションがサポート外の型（テーブル/ビュー以外）である場合、エラーを報告。
  • ターゲットとソースのエイリアス名が重複している場合、ERRCODE_DUPLICATE_ALIAS でエラーを報告。
  • 到達不能な WHEN 句（条件なしの句の後に別の句がある場合）はエラーを報告。

保守担当者向けメモ

  • MERGE は INSERT ON CONFLICT と異なり EXCLUDED 疑似リレーションを持ちません。
  • setNamespaceForMergeWhen は名前空間の可視性を直接操作するため、副作用に注意してください。
  • 最終的な結合ツリーの構築は本関数内ではなく、後続の transform_MERGE_to_join で行われます。
