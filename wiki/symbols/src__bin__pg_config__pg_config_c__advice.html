advice

定義

  • 種別: function
  • 定義箇所: src/bin/pg_config/pg_config.c:108

目的

ユーザーに対して、詳細なヘルプ情報の確認方法を標準エラー出力へ提示する。

シグネチャまたは形状

static void advice(void)

入力と出力

  • 入力: なし
  • 出力: なし（副作用として標準エラー出力へメッセージを書き出す）

前提条件と不変条件

  • グローバル変数 progname が適切に初期化されていること。

アルゴリズムまたは振る舞い

1. fprintf を使用し、progname を含む「Try "progname --help" for more information.」という形式のメッセージを標準エラー出力（stderr）に書き出す。

呼び出し元と呼び出し先

  • 呼び出し元: src/bin/pg_config/pg_config.c:182（main 関数内）
  • 呼び出し先: なし（標準ライブラリの fprintf を使用）
  • 注記: ツールが検出した src/backend/... 等の他のファイルにおける「advice」という単語を含む言及は、本関数とは無関係なコメントや別機能への言及であり、本関数の呼び出し元・呼び出し先ではありません。

関連データ構造

  • なし

失敗モードと境界条件

  • progname が未設定または NULL の場合、fprintf の動作に依存する。

保守担当者向けメモ

  • 本関数は pg_config コマンドの引数解析エラー時などに呼び出される。
  • メッセージのフォーマットを変更する場合は、help() 関数内の出力形式との整合性に注意すること。
