r_adjectival

定義

  • 定義箇所: src/backend/snowball/libstemmer/stem_KOI8_R_russian.c:427

目的

ロシア語のステミング処理において、形容詞的語尾（adjectival endings）の判定および削除を行う。

シグネチャまたは形状

static int r_adjectival(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング環境の状態を保持する構造体）
  • 出力: 成功時に 1、失敗または処理中断時に 0 または負の値を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • 本関数は r_adjective の実行結果に依存しており、r_adjective が正の値を返した場合のみ後続の処理を実行する。

アルゴリズムまたは振る舞い

1. r_adjective(z) を呼び出し、結果が 0 以下であれば即座に終了する。 2. 単語の末尾から find_among_b を使用し、配列 a_2 に定義されたパターンを検索する。 3. マッチした場合、among_var の値に応じて以下の処理を行う。

  • case 1: 直前の文字が 0xC1 または 0xD1 であることを確認し、該当すれば slice_del で削除する。
  • case 2: 条件判定なしで slice_del を実行し、該当部分を削除する。

    • 4. いずれのケースでも、削除処理が成功すれば 1 を返す。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_KOI8_R_russian.c および src/backend/snowball/libstemmer/stem_UTF_8_russian.c 内のステミング処理フロー。
  • 呼び出し先: r_adjective（内部関数）、find_among_b（ランタイムユーティリティ）、slice_del（ランタイムユーティリティ）。

関連データ構造

  • struct SN_env: ステミング環境を管理する構造体。
  • a_2: 形容詞的語尾パターンを格納した among 構造体の配列。

失敗モードと境界条件

  • r_adjective が失敗した場合、本関数もその戻り値を引き継いで終了する。
  • find_among_b でパターンが見つからない場合、または slice_del が失敗した場合は、処理を中断またはエラーを返す。
  • 文字列の境界（z->lb）や、特定の文字コード（0xC1, 0xD1）との不一致が発生した場合は、処理をスキップ（lab0 へジャンプ）する。
