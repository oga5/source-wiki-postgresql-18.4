XLogDumpPrivate

定義

  • 種別: struct
  • 定義箇所: src/bin/pg_waldump/pg_waldump.c:47

目的

pg_waldump において、WAL 読み込み処理の実行状態（タイムラインおよび LSN 範囲）を管理するためのプライベートデータ構造です。

シグネチャまたは形状

typedef struct XLogDumpPrivate
{
	TimeLineID	timeline;
	XLogRecPtr	startptr;
	XLogRecPtr	endptr;
	bool		endptr_reached;
} XLogDumpPrivate;

入力と出力

  • 本構造体は XLogReader の private_data として保持され、WAL 読み込みの制御に使用されます。
  • startptr および endptr は読み込み対象の LSN 範囲を定義します。

前提条件と不変条件

  • XLogReader の初期化時に、この構造体のインスタンスが private_data に割り当てられる必要があります。
  • endptr_reached は、指定された endptr に到達したかどうかを示すフラグとして機能します。

アルゴリズムまたは振る舞い

  • WAL 読み込みループにおいて、現在の読み込み位置が endptr を超えたかどうかを判定するために参照されます。
  • main 関数内で初期化され、読み込み処理の進行に伴い状態が更新されます。

呼び出し元と呼び出し先

  • 呼び出し元: main 関数（pg_waldump.c:797 および 852）にてインスタンス化および初期化が行われます。
  • 呼び出し先: WALDumpReadPage などのコールバック関数内で、state->private_data を通じてキャストされ、読み込み範囲の検証に使用されます（pg_waldump.c:392）。

関連データ構造

  • XLogReader: 本構造体を private_data として保持する WAL リーダー構造体。

失敗モードと境界条件

  • endptr が設定されていない場合や、無効な LSN が指定された場合の挙動については、本ソース抜粋からは判断できません。

保守担当者向けメモ

  • 本構造体は pg_waldump の実行コンテキストに強く依存しています。変更を加える際は、XLogReader のコールバック関数（特に WALDumpReadPage）への影響を確認してください。
