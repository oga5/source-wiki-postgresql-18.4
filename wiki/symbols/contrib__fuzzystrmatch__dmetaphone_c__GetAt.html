GetAt

定義

  • 種別: function
  • 定義箇所: contrib/fuzzystrmatch/dmetaphone.c:324

目的

metastring 構造体が保持する文字列バッファから、指定された位置の文字を安全に取得します。

シグネチャまたは形状

static char GetAt(metastring *s, int pos)

入力と出力

  • **入力**:
  • s: 対象となる metastring 構造体へのポインタ。
  • pos: 取得対象の文字インデックス（整数）。
  • **出力**:
  • 指定位置の文字（char）。範囲外の場合はヌル文字 '\0' を返します。

前提条件と不変条件

  • s は有効な metastring 構造体を指している必要があります。
  • s->str には有効な文字列データが格納されている必要があります。

アルゴリズムまたは振る舞い

1. pos が 0 未満、または s->length 以上であるかを確認します。 2. 範囲外であれば '\0' を返します。 3. 範囲内であれば、s->str の先頭ポインタに pos を加算したアドレスから文字を読み取り、返します。

呼び出し元と呼び出し先

  • **呼び出し元**: DoubleMetaphone 関数内の各発音ルール判定ロジック（例: contrib/fuzzystrmatch/dmetaphone.c:424, 437, 460 等）。
  • **呼び出し先**: 本関数は他の関数を呼び出しません。

関連データ構造

  • metastring: 文字列のバッファ、現在の長さ、および最大サイズを保持する構造体。

失敗モードと境界条件

  • pos が負の値、または文字列の長さ以上である場合、エラーを発生させるのではなく '\0' を返すことで安全に境界を処理します。

保守担当者向けメモ

  • 本関数は DoubleMetaphone アルゴリズムにおける文字走査の基本インターフェースです。インデックスの境界チェックを自動化しているため、ルール実装時に個別の範囲チェックを省略可能です。
