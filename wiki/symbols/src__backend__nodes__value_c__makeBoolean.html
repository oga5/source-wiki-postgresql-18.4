makeBoolean

定義

  • 種別: function
  • 定義箇所: src/backend/nodes/value.c:48

目的

真偽値（bool）を保持する Boolean 型のノードを生成し、初期化する。

シグネチャまたは形状

Boolean *makeBoolean(bool val)

入力と出力

  • **入力**: val (真偽値)
  • **出力**: 初期化された Boolean 構造体へのポインタ

前提条件と不変条件

  • makeNode() マクロが利用可能であること。
  • 戻り値のノードは、PostgreSQL のメモリコンテキスト上で確保される。

アルゴリズムまたは振る舞い

1. makeNode(Boolean) マクロを呼び出し、Boolean 構造体に必要なメモリを確保する。 2. 引数 val を、確保した構造体の boolval メンバに代入する。 3. 初期化済みのノードポインタを返却する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/postgres_fdw/postgres_fdw.c
  • src/backend/commands/sequence.c
  • src/backend/commands/tsearchcmds.c
  • src/backend/nodes/read.c
  • **呼び出し先**:
  • makeNode() マクロ（PostgreSQL のノードメモリ管理機能）

関連データ構造

  • Boolean: src/include/nodes/value.h で定義される、真偽値を保持するためのノード構造体。

失敗モードと境界条件

  • メモリ不足時には makeNode() 経由でエラーが発生する可能性がある。

保守担当者向けメモ

  • 本関数は Value ノードの一種を生成するファクトリ関数である。
  • Float や String と異なり、bool 型は値そのものを保持するため、メモリの所有権管理（palloc された文字列の管理など）は不要である。
