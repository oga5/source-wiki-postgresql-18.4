FromCharDateMode

定義

  • 種別: enum
  • 定義箇所: src/backend/utils/adt/formatting.c:137

目的

TO_CHAR や TO_DATE 等の関数において、解析対象の文字列がどのような日付形式（グレゴリオ暦または ISO 8601 週日付）に基づいているかを識別するために使用されます。

シグネチャまたは形状

typedef enum
{
    FROM_CHAR_DATE_NONE = 0,
    FROM_CHAR_DATE_GREGORIAN,
    FROM_CHAR_DATE_ISOWEEK,
} FromCharDateMode;

入力と出力

  • 本シンボルは列挙型であり、関数への引数として渡されるか、構造体（TmFromChar 等）のメンバとして保持されます。

前提条件と不変条件

  • FROM_CHAR_DATE_NONE は 0 に初期化され、日付モードが未設定であることを示します。
  • 日付変換処理において、解析されたフォーマットノードの内容に応じて、この列挙値が適切に設定される必要があります。

アルゴリズムまたは振る舞い

  • from_char_set_mode() 関数において、現在の解析状態（TmFromChar）に対して日付モードを適用・検証するために使用されます。
  • フォーマット文字列の解析過程で、グレゴリオ暦特有の指定子や ISO 週日付特有の指定子が出現した際に、このモードが切り替わります。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/utils/adt/formatting.c 内のフォーマット解析および変換処理。
  • 呼び出し先: from_char_set_mode() 関数がこの型を引数として受け取ります。

関連データ構造

  • TmFromChar: 日付変換処理中の状態を保持する構造体。この構造体内に date_mode メンバとして保持されます。

失敗モードと境界条件

  • 矛盾する日付モード（例: グレゴリオ暦と ISO 週日付が混在するフォーマット）が指定された場合、from_char_set_mode() 内でエラーハンドリングが行われる可能性があります。

保守担当者向けメモ

  • 新しい日付形式やカレンダーシステムをサポートする場合、この列挙型に新しいメンバを追加し、from_char_set_mode() のロジックを更新する必要があります。
