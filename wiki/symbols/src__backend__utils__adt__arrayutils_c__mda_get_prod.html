mda_get_prod

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/arrayutils.c:166

目的

多次元配列の各次元における要素数の積（ストライド）を計算し、配列の線形オフセット計算を支援します。

シグネチャまたは形状

void mda_get_prod(int n, const int *range, int *prod)

入力と出力

  • n (int): 配列の次元数。
  • range (const int *): 各次元の要素数（範囲）を格納した配列。
  • prod (int *): 計算結果を格納する出力用配列。各次元の要素数の累積積が格納されます。

前提条件と不変条件

  • prod 配列は、少なくとも n 個の要素を保持できるサイズである必要があります。
  • range 配列には各次元のサイズが格納されている必要があります。

アルゴリズムまたは振る舞い

1. prod[n - 1] に 1 を代入します。 2. n - 2 から 0 までのインデックス i について、以下の計算を逆順に行います。

  • prod[i] = prod[i + 1] * range[i + 1]

    • 3. これにより、prod[i] には i+1 次元目以降の要素数の積が格納されます。

失敗モードと境界条件

  • n が 0 または負の値である場合、配列アクセス違反が発生する可能性があります。
  • range の値が非常に大きい場合、prod の計算結果が int 型の範囲を超えてオーバーフローする可能性があります。本関数自体にはオーバーフローチェックは含まれていません。
