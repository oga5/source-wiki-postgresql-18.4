r_verb

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_armenian.c:485

目的

アルメニア語のステミング処理において、単語末尾から動詞に関連する接尾辞を検索し、一致した場合はそれを削除する。

シグネチャまたは形状

static int r_verb(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング環境構造体）
  • 出力: int（成功時に 1、失敗時に 0、エラー発生時に負の値を返す）

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • a_1 配列（71個の接尾辞パターン）が定義されている必要がある。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を z->ket に設定する。 2. find_among_b を使用し、a_1 配列に基づいて単語末尾から接尾辞を後方検索する。 3. 一致しなかった場合は 0 を返す。 4. 一致した場合、その開始位置を z->bra に設定する。 5. slice_del を呼び出し、該当する接尾辞を単語から削除する。 6. slice_del の結果を返し、成功した場合は 1 を返す。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_armenian.c 内のメイン処理（armenian_UTF_8_stem 経由）。
  • 呼び出し先: find_among_b（接尾辞検索）、slice_del（文字列削除）。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_1: 動詞接尾辞のパターンを保持する among 構造体の配列。

失敗モードと境界条件

  • find_among_b が一致を見つけられない場合、0 を返す。
  • slice_del がメモリ不足や内部エラー等で負の値を返した場合、その値をそのまま伝播させる。
