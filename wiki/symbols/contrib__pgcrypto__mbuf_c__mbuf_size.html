mbuf_size

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/mbuf.c:55

目的

MBuf 構造体が現在保持しているデータ量（バイト数）を算出します。

シグネチャまたは形状

int mbuf_size(MBuf *mbuf);

入力と出力

  • **入力**: MBuf *mbuf - サイズを測定する対象のバッファへのポインタ。
  • **出力**: int - バッファ内に格納されているデータの合計バイト数。

前提条件と不変条件

  • mbuf は有効な MBuf 構造体を指している必要があります。
  • mbuf->data_end はバッファ内のデータ終了位置を、mbuf->data はデータ開始位置を指しており、data_end >= data であることが前提です。

アルゴリズムまたは振る舞い

バッファのデータ終了ポインタ（data_end）からデータ開始ポインタ（data）を減算することで、格納されているデータのサイズを計算して返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/pgcrypto/mbuf.c:164 にて、バッファ内のデータ長を取得するために呼び出されています。
  • **呼び出し先**:
  • 本関数は計算のみを行うため、他の関数を呼び出しません。

関連データ構造

  • MBuf: contrib/pgcrypto/mbuf.c で定義される可変長メモリバッファ構造体。

失敗モードと境界条件

  • mbuf が NULL の場合、セグメンテーション違反が発生する可能性があります。
  • バッファが空の場合、data_end と data が等しいため、戻り値は 0 となります。

保守担当者向けメモ

  • 本関数は単純な算術演算であるため、副作用はありません。
  • mbuf.h にてプロトタイプ宣言されており、外部モジュールからも利用可能です。
