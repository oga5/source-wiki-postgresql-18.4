bbsink_forward_begin_archive

定義

  • 種別: function
  • 定義箇所: src/backend/backup/basebackup_sink.c:36

目的

bbsink（バックアップシンク）チェーンにおいて、アーカイブ開始の通知を現在のシンクから後続のシンクへ転送する。

シグネチャまたは形状

void bbsink_forward_begin_archive(bbsink *sink, const char *archive_name)

入力と出力

  • sink: 現在の bbsink オブジェクトへのポインタ。
  • archive_name: 開始するアーカイブの名前。
  • 戻り値: なし。

前提条件と不変条件

  • sink->bbs_next が NULL であってはならない（後続のシンクが存在すること）。

アルゴリズムまたは振る舞い

1. sink->bbs_next が存在することをアサーションで確認する。 2. sink->bbs_next が指す後続シンクの begin_archive メソッド（bbsink_begin_archive マクロ経由）を呼び出し、archive_name を渡す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/basebackup_to_shell/basebackup_to_shell.c
  • src/backend/backup/basebackup_server.c
  • **呼び出し先**:
  • bbsink_begin_archive（sink->bbs_next のメソッド）

関連データ構造

  • bbsink: バックアップデータの転送先を抽象化した構造体。bbs_next メンバによりシンクの連結リストを形成する。

失敗モードと境界条件

  • sink->bbs_next が NULL の場合、アサーション違反によりプログラムが停止する。

保守担当者向けメモ

  • 本関数は転送専用のヘルパーである。独自のシンクを実装する際、bbsink_ops 構造体の begin_archive コールバックに直接割り当てることが可能である。
