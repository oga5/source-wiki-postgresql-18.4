leftmostvalue_int8

定義

  • 種別: function
  • 定義箇所: contrib/btree_gin/btree_gin.c:225

目的

int8 型のデータに対して、GINインデックスを用いた範囲検索（BTLess 等）を行う際に、スキャン開始点として使用する最小値を返します。

シグネチャまたは形状

static Datum leftmostvalue_int8(void)

入力と出力

  • 入力: なし
  • 出力: int8 型の最小値（PG_INT64_MIN）を Datum 型として返します。

前提条件と不変条件

  • int8 型の範囲検索において、インデックスの先頭からスキャンを開始するための基準値として利用されます。

アルゴリズムまたは振る舞い

PG_INT64_MIN を Int64GetDatum マクロで変換し、呼び出し元へ返します。

呼び出し元と呼び出し先

  • 呼び出し元: contrib/btree_gin/btree_gin.c 内の GIN_SUPPORT(int8, ...) マクロ定義において参照されています。
  • 呼び出し先: Int64GetDatum（PostgreSQLのデータ型変換マクロ）。

関連データ構造

  • int8 型（PostgreSQLの64ビット整数型）

失敗モードと境界条件

  • 境界条件: int8 型が取り得る最小値（-9223372036854775808）を正確に返すことが保証されています。

保守担当者向けメモ

  • 本関数は GIN_SUPPORT マクロを通じて登録されます。新しい整数型を追加する際は、同様のパターンで最小値を返す関数を定義してください。
