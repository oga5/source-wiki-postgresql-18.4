z_free

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/pgp-compress.c:63

目的

zlib ライブラリが PostgreSQL のメモリ管理機構（pfree）を使用してメモリを解放するためのラッパー関数です。

シグネチャまたは形状

static void z_free(void *priv, void *addr);

入力と出力

  • priv: zlib の内部状態用ポインタ（本関数では使用されません）。
  • addr: 解放対象のメモリ領域へのポインタ。
  • 戻り値: なし。

前提条件と不変条件

  • addr は、PostgreSQL の palloc によって確保された有効なメモリ領域を指している必要があります。

アルゴリズムまたは振る舞い

1. 受け取ったポインタ addr に対して pfree() を呼び出し、メモリを解放します。

呼び出し元と呼び出し先

  • 呼び出し元:
  • contrib/pgcrypto/pgp-compress.c: 圧縮および伸長ストリームの初期化時に、z_stream 構造体の zfree フィールドへ本関数を代入します。
  • 呼び出し先:
  • pfree (PostgreSQL メモリ管理関数)

関連データ構造

  • z_stream (zlib ライブラリの構造体)

失敗モードと境界条件

  • addr が NULL の場合、pfree の仕様に従い動作します。

保守担当者向けメモ

  • 本関数は zlib が PostgreSQL のメモリ管理システムと統合されることを保証し、メモリリークを防止します。z_alloc と対で使用されます。
