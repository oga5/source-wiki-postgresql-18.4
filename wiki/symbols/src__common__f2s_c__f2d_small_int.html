f2d_small_int

定義

  • 種別: function
  • 定義箇所: src/common/f2s.c:688

目的

浮動小数点数が整数として正確に表現可能か判定し、その値を floating_decimal_32 構造体に格納する。Ryuアルゴリズムにおける高速パスとして機能する。

シグネチャまたは形状

static inline bool f2d_small_int(const uint32 ieeeMantissa, const uint32 ieeeExponent, floating_decimal_32 *v)

入力と出力

  • **入力**:
  • ieeeMantissa: IEEE 754 単精度浮動小数点数の仮数部（23ビット）。
  • ieeeExponent: IEEE 754 単精度浮動小数点数の指数部。
  • v: 結果を格納する floating_decimal_32 構造体へのポインタ。
  • **出力**:
  • 戻り値: 整数として表現可能な場合は true、それ以外は false。
  • v: true の場合、mantissa に整数値が、exponent に 0 が設定される。

前提条件と不変条件

  • FLOAT_BIAS および FLOAT_MANTISSA_BITS が定義されている環境であること。
  • 入力された浮動小数点数が、整数として表現可能な範囲（2^24 未満）にある場合にのみ true を返す。

アルゴリズムまたは振る舞い

1. 指数 e2 を計算し、数値が整数として表現可能な範囲（-FLOAT_MANTISSA_BITS <= e2 <= 0）にあるか確認する。 2. 範囲内であれば、仮数部の下位ビットをマスクして、小数部分がゼロであることを確認する。 3. 小数部分がゼロであれば、暗黙の1ビットを考慮した仮数 m2 を算出し、e2 に基づいてシフトして整数値 v->mantissa を求める。 4. 成功時は v->exponent = 0 とし、true を返す。それ以外は false を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/common/f2s.c 内の浮動小数点変換処理（float_to_shortest_decimal_bufn 等の文脈）から呼び出される。
  • **呼び出し先**: 外部関数への直接的な呼び出しはない。

関連データ構造

  • floating_decimal_32: 変換結果（仮数と指数）を保持する構造体。

失敗モードと境界条件

  • e2 が範囲外の場合、または仮数部に小数成分が含まれる場合は false を返す。
  • 0.5 のようなケース（ieeeMantissa == 0, e2 == -24）は、条件判定により正しく処理される。

保守担当者向けメモ

  • 複数の return false を避ける実装となっている。これはコンパイラによる f2d 関数の過度なインライン化を防ぐための設計である。
  • m2 の計算において 1U << FLOAT_MANTISSA_BITS を使用しているため、ビット操作の定数定義に変更を加える際は注意が必要。
