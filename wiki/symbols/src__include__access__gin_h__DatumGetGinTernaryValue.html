DatumGetGinTernaryValue

定義

  • 種別: function
  • 定義箇所: src/include/access/gin.h:81

目的

Datum 型として保持されている値を GinTernaryValue 型へキャストして取得します。

シグネチャまたは形状

static inline GinTernaryValue DatumGetGinTernaryValue(Datum X)

入力と出力

  • **入力**: Datum X - 変換対象となるデータ。
  • **出力**: GinTernaryValue - X をキャストした結果。

前提条件と不変条件

  • GinTernaryValue は bool と同じサイズであることが設計上の前提です。
  • 本関数は単なる型キャストを行うため、入力値 X が有効な GinTernaryValue の範囲内（TRUE/FALSE/MAYBE）にあることを呼び出し側で保証する必要があります。

アルゴリズムまたは振る舞い

入力された Datum 型の値を、インライン展開可能なキャスト操作により GinTernaryValue 型へ変換して返します。

呼び出し元と呼び出し先

  • **呼び出し元**: 以下のソースコード内で利用されています。
  • src/backend/access/gin/ginlogic.c:91
  • src/backend/access/gin/ginlogic.c:112
  • **呼び出し先**: 外部関数への依存はありません。

関連データ構造

  • GinTernaryValue: GIN インデックスの検索条件判定に使用される3値論理型。

失敗モードと境界条件

  • 本関数自体は型キャストのみを行うため、関数内部での失敗モードはありません。
  • 不適切な値が Datum に格納されている場合、キャスト後の GinTernaryValue が未定義の挙動を引き起こす可能性があります。

保守担当者向けメモ

  • GinTernaryValue は bool とのキャストを前提としているため、サイズ変更を行う際は StaticAssertDecl による制約チェックを維持してください。
