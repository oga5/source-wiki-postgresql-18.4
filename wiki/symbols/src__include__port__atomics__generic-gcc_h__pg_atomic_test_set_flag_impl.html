pg_atomic_test_set_flag_impl

定義

  • 種別: function
  • 定義箇所: src/include/port/atomics/generic-gcc.h:117

目的

GCCの組み込み関数を利用し、アトミックなフラグのテストおよびセット操作（Test-and-Set）を実行します。

シグネチャまたは形状

static inline bool pg_atomic_test_set_flag_impl(volatile pg_atomic_flag *ptr)

入力と出力

  • **入力**: ptr - 操作対象となる pg_atomic_flag 構造体へのポインタ。
  • **出力**: bool - フラグが以前「未セット（0）」であった場合に true を返し、既に「セット（1）」されていた場合に false を返します。

前提条件と不変条件

  • ptr は volatile 修飾された pg_atomic_flag を指している必要があります。
  • 本関数は atomics.h を経由して呼び出されることを前提としています。

アルゴリズムまたは振る舞い

1. GCCの組み込み関数 __sync_lock_test_and_set を呼び出し、ptr->value に対してアトミックに 1 を書き込みます。 2. 書き込み前の値が 0 であったかどうかを判定し、その結果を返します。 3. メモリバリアとして「Acquireバリア」の性質を持ちます。

呼び出し元と呼び出し先

  • **呼び出し元**: src/include/port/atomics.h:185 からの呼び出しが確認されています。
  • **呼び出し先**: コンパイラ組み込み関数 __sync_lock_test_and_set を直接呼び出します。

関連データ構造

  • pg_atomic_flag: 原子操作対象のフラグ値を保持する構造体。

失敗モードと境界条件

  • 一部のプラットフォームでは、__sync_lock_test_and_set に渡せる値が 1 に制限されている場合があります。
  • 本実装はフルバリアではなく、Acquireバリアとして機能します。

保守担当者向けメモ

  • 本関数は static inline で定義されており、呼び出し元でインライン展開されます。
  • generic-gcc.h 内の他の関数と同様、HAVE_GCC__SYNC_* などのマクロ環境下で動作します。
