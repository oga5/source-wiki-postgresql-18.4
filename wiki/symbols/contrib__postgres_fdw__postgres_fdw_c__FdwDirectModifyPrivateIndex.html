FdwDirectModifyPrivateIndex

定義

  • 種別: enum
  • 定義箇所: contrib/postgres_fdw/postgres_fdw.c:124

目的

外部テーブルに対する直接的な変更操作（Direct Modification）において、プランノードの fdw_private リストに格納される各要素のインデックスを定義します。

シグネチャまたは形状

enum FdwDirectModifyPrivateIndex {
    FdwDirectModifyPrivateUpdateSql,      /* String node: リモート実行用SQL */
    FdwDirectModifyPrivateHasReturning,   /* Boolean node: RETURNING句の有無 */
    FdwDirectModifyPrivateRetrievedAttrs, /* Integer list: RETURNINGで取得する属性番号 */
    FdwDirectModifyPrivateSetProcessed,   /* Boolean node: セット処理済みフラグ */
};

入力と出力

本シンボル自体は列挙型であり、入力や出力を持つ関数ではありません。プランノードの fdw_private リストの各インデックスとして使用されます。

前提条件と不変条件

  • fdw_private リスト内の各要素は、この列挙型の定義順序と一致している必要があります。

アルゴリズムまたは振る舞い

  • FdwDirectModifyPrivateUpdateSql: リモートサーバーへ送信する SQL 文を保持します。
  • FdwDirectModifyPrivateHasReturning: RETURNING 句が指定されているかどうかを示すフラグを保持します。
  • FdwDirectModifyPrivateRetrievedAttrs: RETURNING 句によって取得される属性のリストを保持します。
  • FdwDirectModifyPrivateSetProcessed: 変更操作のセット処理が完了したかを示すフラグを保持します。

呼び出し元と呼び出し先

  • 呼び出し元: contrib/postgres_fdw/postgres_fdw.c:2615 付近にて、fdw_private リストの要素アクセス時に参照されています。
  • 呼び出し先: 本シンボルは列挙型であるため、直接的な呼び出し先はありません。

関連データ構造

  • PgFdwDirectModifyState: 直接変更操作の実行状態を保持する構造体。
  • fdw_private (プランノードのメンバ): 本列挙型がインデックスとして使用されるリスト。

失敗モードと境界条件

  • fdw_private リストの長さが本列挙型の要素数と一致しない場合、リストアクセス時に不正なメモリ参照や型不整合が発生する可能性があります。

保守担当者向けメモ

  • リストの要素を追加・変更する場合は、必ずこの列挙型の定義と、それを使用しているコード（特に fdw_private を構築・参照する箇所）の両方を更新してください。
