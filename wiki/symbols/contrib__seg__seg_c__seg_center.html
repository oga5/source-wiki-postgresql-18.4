seg_center

定義

  • 種別: function
  • 定義箇所: contrib/seg/seg.c:164

目的

SEG 構造体で表現された区間の中間値（中心値）を計算し、float4 型として返します。

シグネチャまたは形状

Datum seg_center(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS を介して受け取る SEG 型のポインタ（PG_GETARG_SEG_P(0)）。
  • **出力**: float4 型の計算結果（Datum として返却）。

前提条件と不変条件

  • 入力された SEG 構造体の lower および upper メンバが有効な数値であることを前提とします。

アルゴリズムまたは振る舞い

1. PG_GETARG_SEG_P(0) を使用して、引数から SEG 構造体へのポインタを取得します。 2. seg->lower と seg->upper をそれぞれ float にキャストし、加算します。 3. 加算結果を 2.0 で除算し、その値を PG_RETURN_FLOAT4 マクロを通じて返します。

呼び出し元と呼び出し先

  • **呼び出し元**: PostgreSQL の関数マネージャ（PG_FUNCTION_INFO_V1(seg_center) により登録）。
  • **呼び出し先**: PG_GETARG_SEG_P および PG_RETURN_FLOAT4 マクロ。

関連データ構造

  • SEG: segdata.h で定義される、区間の境界値や有効桁数を保持する構造体。

失敗モードと境界条件

  • 入力ポインタが NULL の場合や、SEG 構造体の内容が不正な場合は、PostgreSQL の標準的なエラーハンドリングに依存します。

保守担当者向けメモ

  • 本関数は float へのキャストを行って計算しているため、極端に大きな値や精度を要する数値の計算時には浮動小数点演算の特性に留意してください。
