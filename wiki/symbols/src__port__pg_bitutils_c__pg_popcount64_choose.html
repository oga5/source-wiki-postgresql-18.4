pg_popcount64_choose

定義

  • 種別: function
  • 定義箇所: src/port/pg_bitutils.c:194

目的

64ビット整数のポピュレーションカウント（1のビット数の計数）を行う関数の初期化と、実行時の最適化実装への切り替えを行う。

シグネチャまたは形状

static int pg_popcount64_choose(uint64 word);

入力と出力

  • 入力: uint64 word（計数対象の64ビット整数）
  • 出力: int（word 内の1のビット数）

前提条件と不変条件

  • 本関数は、pg_popcount64 関数ポインタが初期状態（本関数を指している状態）のときに呼び出される。
  • 呼び出しにより、内部の関数ポインタがハードウェア最適化版または汎用版へ更新される。

アルゴリズムまたは振る舞い

1. choose_popcount_functions() を呼び出し、CPU機能判定に基づいた関数ポインタの書き換えを実行する。 2. 書き換え後の pg_popcount64 を呼び出し、その結果を返す。

呼び出し元と呼び出し先

  • 呼び出し元: pg_popcount64 関数ポインタ（初期化時）
  • 呼び出し先: choose_popcount_functions()

関連データ構造

  • pg_popcount64: 本関数が初期値として設定されている関数ポインタ。

失敗モードと境界条件

  • 特記事項なし。CPU機能判定の結果に関わらず、必ず有効なカウント結果を返す。

保守担当者向けメモ

  • 本関数は pg_popcount64 の動的ディスパッチ機構の入り口である。
  • choose_popcount_functions() が実行された後は、以降の呼び出しで本関数はバイパスされる。
