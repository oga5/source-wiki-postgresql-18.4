astreamer_lz4_compressor_finalize

定義

  • 種別: function
  • 定義箇所: src/fe_utils/astreamer_lz4.c:200

目的

LZ4圧縮ストリームの終端処理を行い、残存する圧縮データをフラッシュして後続のストリーマーへ転送する。

シグネチャまたは形状

static void astreamer_lz4_compressor_finalize(astreamer *streamer);

入力と出力

  • **入力**: astreamer *streamer（astreamer_lz4_frame 構造体としてキャストして使用）
  • **出力**: なし（副作用として後続のストリーマーへデータを転送し、終了処理を伝播させる）

前提条件と不変条件

  • streamer は astreamer_lz4_frame 型のインスタンスであること。
  • mystreamer->cctx（LZ4圧縮コンテキスト）が有効であること。
  • bbs_next（後続のストリーマー）が初期化されていること。

アルゴリズムまたは振る舞い

1. **フッター領域の確保**: LZ4F_compressBound(0) を用いて必要なフッターサイズを計算する。 2. **バッファ確認**: 現在の出力バッファにフッターを書き込む十分な空きがない場合、既存のデータを astreamer_content で後続へフラッシュし、必要に応じて enlargeStringInfo でバッファを拡張する。 3. **圧縮終了処理**: LZ4F_compressEnd を呼び出し、残りの圧縮データをバッファに書き込む。エラー発生時は pg_fatal で終了する。 4. **データ転送**: 最終的な圧縮データを astreamer_content で後続へ転送する。 5. **連鎖終了**: astreamer_finalize を呼び出し、後続のストリーマーの終了処理をトリガーする。

呼び出し元と呼び出し先

  • **呼び出し元**: astreamer インターフェースの finalize 関数ポインタ経由で呼び出される（直接の呼び出し元はソースコード上では明示されていない）。
  • **呼び出し先**:
  • astreamer_content（後続ストリーマーへのデータ転送）
  • enlargeStringInfo（バッファ拡張）
  • LZ4F_compressEnd / LZ4F_isError / LZ4F_getErrorName（LZ4ライブラリ関数）
  • astreamer_finalize（後続ストリーマーの終了処理）

関連データ構造

  • astreamer_lz4_frame: 圧縮状態を保持する構造体。
  • astreamer: ストリーマーの基底構造体。

失敗モードと境界条件

  • **バッファ不足**: フッター書き込み時にバッファが不足する場合、自動的に拡張を試みる。
  • **LZ4エラー**: LZ4F_compressEnd がエラーを返した場合、pg_fatal によりプロセスが異常終了する。

保守担当者向けメモ

  • 本関数は USE_LZ4 が定義されている環境でのみコンパイルされる。
  • bbs_buffer の管理は astreamer フレームワークの規約に従う必要がある。
