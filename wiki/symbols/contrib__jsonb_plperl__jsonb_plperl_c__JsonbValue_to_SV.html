JsonbValue_to_SV

定義

  • 種別: function
  • 定義箇所: contrib/jsonb_plperl/jsonb_plperl.c:19

目的

PostgreSQL の JsonbValue 構造体を、Perl インタプリタで扱える SV (Scalar Value) 型へ変換する。

シグネチャまたは形状

static SV *JsonbValue_to_SV(JsonbValue *jbv)

入力と出力

  • **入力**: JsonbValue *jbv （変換対象の JSONB 値へのポインタ）
  • **出力**: SV * （変換後の Perl スカラー値）

前提条件と不変条件

  • dTHX マクロにより、Perl インタプリタのコンテキストが初期化されている必要がある。
  • jbv は有効な JsonbValue を指していること。

アルゴリズムまたは振る舞い

jbv->type に基づき、以下の通り変換を行う： 1. **jbvBinary**: Jsonb_to_SV を再帰的に呼び出す。 2. **jbvNumeric**: numeric_out を介して文字列化し、さらに SvNV を用いて数値型へ変換する。 3. **jbvString**: pnstrdup で文字列を複製し、cstr2sv で Perl 文字列へ変換する。 4. **jbvBool**: PL_sv_yes または PL_sv_no を用いて Perl の真偽値へ変換する。 5. **jbvNull**: newSV(0) を用いて未定義値（undef）を生成する。 6. **default**: 未知の型が渡された場合、elog(ERROR) を発生させる。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/jsonb_plperl/jsonb_plperl.c 内の複数箇所（Jsonb_to_SV 等の再帰処理やループ内）。
  • **呼び出し先**:
  • Jsonb_to_SV
  • numeric_out (PostgreSQL 内部関数)
  • newSVnv, cstr2sv, newSV (Perl API)
  • pfree, elog (PostgreSQL メモリ管理・エラー処理)

関連データ構造

  • JsonbValue: PostgreSQL 側の JSONB 内部表現。
  • SV: Perl のスカラー値構造体。

失敗モードと境界条件

  • **未知の型**: jbv->type が定義外の値の場合、elog(ERROR) により処理が中断される。
  • **メモリ**: 文字列変換時に pfree を行いメモリ解放を試みるが、Perl 側への変換失敗時はメモリリークの可能性がある。

保守担当者向けメモ

  • 本関数は再帰的な構造変換の一部を担う。ネストが深い JSONB データに対してはスタックオーバーフローのリスクがあるため注意すること。
  • jbvNumeric 変換時に numeric_out を経由しているため、数値の精度やフォーマットは PostgreSQL の numeric 型の出力仕様に依存する。
