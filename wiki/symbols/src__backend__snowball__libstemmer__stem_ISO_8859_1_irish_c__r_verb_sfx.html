r_verb_sfx

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_irish.c:405

目的

アイルランド語のステミング処理において、単語末尾の動詞接尾辞を特定し、条件に応じて削除する。

シグネチャまたは形状

static int r_verb_sfx(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z（Snowball ステミング環境へのポインタ）
  • 出力: int（成功時に 1、失敗または処理対象外の場合に 0、エラー発生時に負の値を返す）

前提条件と不変条件

  • z->c（現在のカーソル位置）が有効な範囲内であること。
  • find_among_b を使用するため、単語の末尾から前方に向かって検索が行われる。
  • 処理対象の文字コードは ISO-8859-1 である。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を z->ket に設定する。 2. 単語の長さおよび特定のビットマスク条件（a_3 辞書に関連）に基づき、接尾辞の候補が存在するかを判定する。 3. find_among_b を用いて、定義された動詞接尾辞リスト a_3 からマッチするものを検索する。 4. マッチした場合、z->bra を現在のカーソル位置に設定し、among_var の値に応じて以下のいずれかの処理を行う：

  • case 1: r_RV(z)（RV領域内であること）を確認後、slice_del(z) で削除。
  • case 2: r_R1(z)（R1領域内であること）を確認後、slice_del(z) で削除。

    • 5. 処理成功時に 1 を返す。

呼び出し元と呼び出し先

  • 呼び出し元: irish_ISO_8859_1_stem（src/backend/snowball/libstemmer/stem_ISO_8859_1_irish.c:459 および stem_UTF_8_irish.c:459）
  • 呼び出し先: find_among_b, r_RV, r_R1, slice_del

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_3: 動詞接尾辞を定義した among 構造体の配列。

失敗モードと境界条件

  • 単語が短すぎる場合や、接尾辞が辞書に存在しない場合は 0 を返す。
  • r_RV や r_R1 のチェックに失敗した場合、または slice_del がエラーを返した場合は、その戻り値をそのまま返す。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードである。ロジックの変更が必要な場合は、直接編集せず元の .sbl ファイルを修正すること。
  • a_3 辞書の内容や領域定義（R1, RV）の変更が本関数の挙動に直接影響する。
