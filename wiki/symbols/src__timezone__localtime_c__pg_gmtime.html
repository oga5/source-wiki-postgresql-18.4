pg_gmtime

定義

  • 種別: function
  • 定義箇所: src/timezone/localtime.c:1388

目的

UNIX タイムスタンプ（pg_time_t）を、UTC（協定世界時）に基づいた分解された時刻構造体（pg_tm）へ変換します。

シグネチャまたは形状

struct pg_tm *pg_gmtime(const pg_time_t *timep)

入力と出力

  • **入力**: timep - 変換対象の pg_time_t 型の時刻値へのポインタ。
  • **出力**: 変換結果を格納した pg_tm 構造体へのポインタを返します。

前提条件と不変条件

  • 入力ポインタ timep は有効な pg_time_t を指している必要があります。
  • 戻り値の構造体は、内部的に保持されている静的領域（tm）を指しているため、呼び出しごとに内容が上書きされる可能性があります。

アルゴリズムまたは振る舞い

1. 内部関数 gmtsub を呼び出します。 2. gmtsub に対して、オフセット値 0（UTCを示す）と、結果を格納する静的変数 &tm を渡します。 3. gmtsub の計算結果をそのまま呼び出し元へ返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/backup/backup_manifest.c:175
  • src/backend/utils/adt/timestamp.c:2177
  • **呼び出し先**:
  • gmtsub（内部関数）

関連データ構造

  • pg_time_t: UNIX タイムスタンプを表す型。
  • pg_tm: 年、月、日、時、分、秒などに分解された時刻情報を持つ構造体。

失敗モードと境界条件

  • 入力値が pg_time_t の表現範囲外である場合の挙動については、gmtsub の実装に依存します。
  • 静的変数 tm を使用しているため、マルチスレッド環境下での同時呼び出しには注意が必要です。

保守担当者向けメモ

  • 本関数は gmtsub のラッパーです。UTC 変換のロジックを変更する場合は gmtsub を参照してください。
  • src/include/pgtime.h:59 に外部インターフェースとして宣言されています。
