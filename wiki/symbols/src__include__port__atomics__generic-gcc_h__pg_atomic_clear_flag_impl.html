pg_atomic_clear_flag_impl

定義

  • 種別: function
  • 定義箇所: src/include/port/atomics/generic-gcc.h:139

目的

pg_atomic_flag の値を原子的にクリア（リセット）するために使用されます。

シグネチャまたは形状

static inline void pg_atomic_clear_flag_impl(volatile pg_atomic_flag *ptr)

入力と出力

  • **入力**: ptr - 操作対象となる pg_atomic_flag 構造体へのポインタ（volatile 修飾あり）。
  • **出力**: なし（void）。

前提条件と不変条件

  • ptr は有効な pg_atomic_flag 構造体を指している必要があります。
  • 本関数は atomics.h を経由して呼び出されることを前提としています。

アルゴリズムまたは振る舞い

GCC の組み込み関数 __sync_lock_release を呼び出し、指定されたメモリ領域の値を解放します。この操作は原子的に実行されます。

呼び出し元と呼び出し先

  • **呼び出し元**: src/include/port/atomics.h:209 からの呼び出しが確認されています。
  • **呼び出し先**: GCC 組み込み関数 __sync_lock_release を直接呼び出します。

関連データ構造

  • pg_atomic_flag: 原子操作対象のフラグ値を保持する構造体。

失敗モードと境界条件

  • 本関数は volatile なポインタを要求するため、非揮発性メモリや不正なアドレスを渡した場合の動作は未定義です。
  • 64bit 原子操作が無効化されている環境下での動作については、generic-gcc.h の全体的な制約に従います。

保守担当者向けメモ

  • 本関数は static inline で定義されており、コンパイル時に呼び出し元へ展開されます。
  • __sync 系 API を使用しているため、より新しい __atomic API への移行が検討される場合がありますが、現状はフォールバック実装として機能しています。
