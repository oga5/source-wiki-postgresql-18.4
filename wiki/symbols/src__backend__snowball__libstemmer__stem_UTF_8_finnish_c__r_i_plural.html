r_i_plural

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_finnish.c:513

目的

フィンランド語のステミング処理において、単語の末尾が特定の複数形（'i' または 'j' を含む接尾辞）である場合に、その接尾辞を削除して語幹を正規化する。

シグネチャまたは形状

static int r_i_plural(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（Snowball ステミング環境構造体）
  • 出力: 成功時に 1、失敗または条件不一致時に 0、内部エラー時に負の値を返す。

前提条件と不変条件

  • z->I[1] に格納された領域 R1 の開始位置以降で処理を行う。
  • 処理対象の単語末尾が 'i' (ASCII 105) または 'j' (ASCII 106) である必要がある。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c が領域 R1 (z->I[1]) より前にある場合は終了する。 2. 領域の境界を R1 に設定し、末尾の文字が 'i' または 'j' であるかを確認する。 3. a_8 配列で定義された接尾辞パターンを find_among_b を用いて後方検索する。 4. マッチした場合、該当する接尾辞を slice_del で削除する。 5. 処理終了後、一時的に変更した境界設定を復元する。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_finnish.c および stem_ISO_8859_1_finnish.c、stem_UTF_8_estonian.c 内のメインステミング処理フローから呼び出される。
  • 呼び出し先: find_among_b（接尾辞検索）、slice_del（文字列削除）。

関連データ構造

  • SN_env: ステミング環境を保持する構造体。z->I[1]（領域 R1 のインデックス）および z->c（現在のカーソル位置）を使用。
  • a_8: 複数形接尾辞を定義した among 構造体の配列。

失敗モードと境界条件

  • カーソル位置が領域 R1 より前にある場合、何もせず 0 を返す。
  • 末尾が 'i' または 'j' でない場合、0 を返す。
  • find_among_b がパターンに一致しない場合、0 を返す。
  • slice_del が失敗した場合、その戻り値（負の値）をそのまま返す。

保守担当者向けメモ

  • z->I[1] は領域 R1 の開始位置を保持しており、ステミングの有効範囲を制御する重要な不変条件である。
