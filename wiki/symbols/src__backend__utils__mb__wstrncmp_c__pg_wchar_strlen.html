pg_wchar_strlen

定義

  • 種別: function
  • 定義箇所: src/backend/utils/mb/wstrncmp.c:69

目的

pg_wchar 型の配列（ワイド文字文字列）の長さを計算し、ヌル文字を除いた文字数を返します。

シグネチャまたは形状

size_t pg_wchar_strlen(const pg_wchar *str)

入力と出力

  • **入力**: str - ヌル終端された pg_wchar 型の配列へのポインタ。
  • **出力**: size_t 型の整数値。文字列の長さ（ヌル文字を含まない要素数）。

前提条件と不変条件

  • str はヌル文字（0）で終端されている必要があります。
  • pg_wchar 型の定義は mb/pg_wchar.h に依存します。

アルゴリズムまたは振る舞い

1. ポインタ s を入力 str で初期化します。 2. *s が非ゼロである間、ポインタ s をインクリメントし続けます。 3. ヌル文字に到達した時点でループを終了します。 4. 最終的なポインタ位置 s と開始位置 str の差分を計算し、その値を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/mb/mbutils.c:1012
  • **呼び出し先**:
  • 本関数は標準的なポインタ演算のみを行い、他の関数を呼び出しません。

関連データ構造

  • pg_wchar: PostgreSQL内部でマルチバイト文字を表現するための型。

失敗モードと境界条件

  • str が NULL ポインタである場合、未定義の動作（セグメンテーション違反など）が発生します。
  • 空の文字列（*str == 0）が渡された場合、ループは実行されず 0 を返します。

保守担当者向けメモ

  • 標準 C ライブラリの strlen と同様のロジックですが、対象が pg_wchar 型である点に注意してください。
  • 非常に軽量な関数であり、複雑な依存関係はありません。
