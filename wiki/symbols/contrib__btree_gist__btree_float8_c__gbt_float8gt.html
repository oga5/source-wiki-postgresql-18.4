gbt_float8gt

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_float8.c:29

目的

GiSTインデックスにおける float8 型の「より大きい（greater than）」比較演算を提供します。

シグネチャまたは形状

static bool gbt_float8gt(const void *a, const void *b, FmgrInfo *flinfo)

入力と出力

  • **入力**:
  • a, b: 比較対象の float8 型値へのポインタ。
  • flinfo: 関数マネージャ情報（本関数内では未使用）。
  • **出力**:
  • bool: *a > *b が真であれば true、それ以外は false を返します。

前提条件と不変条件

  • a および b は有効な float8 型の値を指している必要があります。

アルゴリズムまたは振る舞い

1. ポインタ a および b を float8 型としてキャストします。 2. C言語の標準的な比較演算子 > を用いて大小比較を行い、その結果を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/btree_gist/btree_float8.c 内の gbtree_ninfo 構造体（tinfo）の初期化時に参照されています。
  • **呼び出し先**: 外部関数への呼び出しはありません。

関連データ構造

  • float8: PostgreSQLの倍精度浮動小数点数型。

失敗モードと境界条件

  • a または b が NULL ポインタである場合、セグメンテーション違反が発生する可能性があります。
  • 浮動小数点数の比較であるため、NaN（非数）が含まれる場合の挙動は標準の比較演算子に依存します。

保守担当者向けメモ

  • 本関数は btree_utils_num が提供する汎用的なGiSTインデックスロジックから呼び出される比較関数の一つです。
  • btree_float8.c 内の tinfo 構造体を通じて、インデックスの構築や検索処理に組み込まれています。
