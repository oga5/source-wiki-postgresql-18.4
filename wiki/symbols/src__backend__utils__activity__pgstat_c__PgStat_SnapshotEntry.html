PgStat_SnapshotEntry

定義

  • 種別: struct
  • 定義箇所: src/backend/utils/activity/pgstat.c:150

目的

統計情報のスナップショットを保持するためのハッシュテーブル要素として機能します。

シグネチャまたは形状

typedef struct PgStat_SnapshotEntry
{
	PgStat_HashKey key;
	char		status;			/* for simplehash use */
	void	   *data;			/* the stats data itself */
} PgStat_SnapshotEntry;

入力と出力

  • 入力: PgStat_HashKey 型のキー、および統計データ本体へのポインタ (void *data)。
  • 出力: なし（構造体定義）。

前提条件と不変条件

  • status フィールドは simplehash ライブラリの内部管理用として使用されます。
  • data ポインタは、対応する統計種別のデータ構造を指す必要があります。

アルゴリズムまたは振る舞い

  • 本構造体は simplehash を用いたハッシュテーブルの要素として利用されます（SH_ELEMENT_TYPE として定義）。
  • 統計スナップショットの構築時、キーに基づいて統計データがこの構造体に格納され、ハッシュテーブルを通じて管理されます。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/utils/activity/pgstat.c 内の統計スナップショット生成および検索処理（行960, 984, 1013, 1149 等）。
  • 呼び出し先: 情報なし。

関連データ構造

  • PgStat_HashKey: 統計エントリを一意に識別するためのキー構造体。
  • simplehash: 本構造体を要素として管理するハッシュテーブル実装。

失敗モードと境界条件

  • data ポインタが NULL の場合、または key が無効な場合、統計情報の取得や参照時に不整合が発生する可能性があります。

保守担当者向けメモ

  • 本構造体は pgstat.c 内の統計スナップショット管理の基盤です。変更を加える際は、simplehash の仕様および PgStat_HashKey の定義との整合性に注意してください。
