resetQueryRepresentation

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/tsrank.c:597

目的

QueryRepresentation 構造体内の各オペランドの状態を初期化し、再利用可能な状態にする。

シグネチャまたは形状

static void resetQueryRepresentation(QueryRepresentation *qr, bool reverseinsert)

入力と出力

  • **入力**:
  • qr: 初期化対象の QueryRepresentation 構造体へのポインタ。
  • reverseinsert: 各オペランドの reverseinsert フィールドに設定するブール値。
  • **出力**: なし（void）。

前提条件と不変条件

  • qr は有効な QueryRepresentation 構造体を指している必要がある。
  • qr->query が初期化されており、qr->query->size が有効なオペランド数を示していること。

アルゴリズムまたは振る舞い

1. qr->query->size で指定された回数だけループを実行する。 2. 各オペランド（qr->operandData[i]）に対して以下の操作を行う。

  • operandexists を false に設定する。
  • reverseinsert を引数で渡された値に設定する。
  • npos を 0 にリセットする。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/tsrank.c:663
  • src/backend/utils/adt/tsrank.c:692
  • **呼び出し先**:
  • 直接的な関数呼び出しはなし。

関連データ構造

  • QueryRepresentation: クエリの表現を保持する構造体。
  • QueryRepresentationOperand: QueryRepresentation 内の各オペランドの状態を管理する構造体。

失敗モードと境界条件

  • qr が NULL の場合、セグメンテーションフォールトが発生する可能性がある。
  • qr->query が NULL の場合、qr->query->size へのアクセスによりクラッシュする。

保守担当者向けメモ

  • 本関数は Cover 関数等のランキング計算処理において、クエリの再評価を行う直前に状態をクリアするために使用される。
  • reverseinsert フラグの切り替えにより、文書内の位置情報の走査順序を制御する役割を持つ。
