print_msg

定義

  • 種別: function
  • 定義箇所: src/bin/pg_ctl/pg_ctl.c:235

目的

pg_ctl ユーティリティの実行状況や進捗メッセージを標準出力へ表示する。

シグネチャまたは形状

static void print_msg(const char *msg)

入力と出力

  • **入力**: msg (表示する文字列へのポインタ)
  • **出力**: なし（標準出力への書き込みのみ）

前提条件と不変条件

  • silent_mode グローバル変数が false である場合にのみ出力が行われる。

アルゴリズムまたは振る舞い

1. silent_mode が偽であることを確認する。 2. fputs を使用して msg を標準出力 (stdout) に書き出す。 3. fflush(stdout) を呼び出し、出力を即座にフラッシュする。

呼び出し元と呼び出し先

  • **呼び出し元**: src/bin/pg_ctl/pg_ctl.c 内の各所（サーバー起動待機時の進捗表示や、状態変化の通知など）。
  • **呼び出し先**: 標準ライブラリの fputs および fflush。

関連データ構造

  • silent_mode (グローバル変数: 出力の抑制状態を制御)

失敗モードと境界条件

  • silent_mode が真の場合、関数は何も行わずに即座に復帰する。
  • 標準出力がクローズされている場合や書き込みエラーが発生した場合の挙動は、標準ライブラリの fputs に依存する。

保守担当者向けメモ

  • 本関数は pg_ctl の標準的な出力経路である。エラーメッセージの出力には write_stderr が別途用意されているため、用途に応じて使い分けること。
