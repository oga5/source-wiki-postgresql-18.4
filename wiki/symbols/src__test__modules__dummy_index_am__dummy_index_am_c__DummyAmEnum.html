DummyAmEnum

定義

  • 種別: enum

目的

dummy_index_am モジュールにおいて、リレーションオプションとして使用可能な列挙型値を定義するために存在します。

シグネチャまたは形状

typedef enum DummyAmEnum
{
	DUMMY_AM_ENUM_ONE,
	DUMMY_AM_ENUM_TWO,
} DummyAmEnum;

入力と出力

  • 入力: なし（列挙型の定義）
  • 出力: DUMMY_AM_ENUM_ONE および DUMMY_AM_ENUM_TWO という識別子を提供します。

前提条件と不変条件

  • 本列挙型は DummyIndexOptions 構造体内で使用されることを前提としています。

アルゴリズムまたは振る舞い

  • インデックス作成時に指定可能なカスタムオプションの値を制限する列挙型として機能します。

呼び出し元と呼び出し先

  • 呼び出し元: src/test/modules/dummy_index_am/dummy_index_am.c 内の DummyIndexOptions 構造体定義（43行目）において、フィールド option_enum の型として参照されています。
  • 呼び出し先: なし。

関連データ構造

  • DummyIndexOptions: 本列挙型をメンバとして保持する構造体。

失敗モードと境界条件

  • 該当なし。

保守担当者向けメモ

  • 本列挙型は、dummy_index_am におけるリレーションオプションの定義と検証の仕組みを実証するためのサンプルです。新しいオプション値を追加する場合は、この列挙型を拡張してください。
