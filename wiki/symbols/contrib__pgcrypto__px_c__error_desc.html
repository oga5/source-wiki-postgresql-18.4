error_desc

定義

  • 種別: struct
  • 定義箇所: contrib/pgcrypto/px.c:36

目的

pgcrypto モジュール内で使用される内部エラーコードと、それに対応する人間が読めるエラーメッセージ文字列を関連付けるためのデータ構造です。

シグネチャまたは形状

struct error_desc
{
	int			err;
	const char *desc;
};

入力と出力

  • err: PXE_* で定義される整数型のエラーコード。
  • desc: エラーコードに対応する説明文を保持する文字列リテラルへのポインタ。

前提条件と不変条件

  • px_strerror 関数において、エラーコードの検索テーブルとして使用されます。
  • desc は静的に割り当てられた文字列を指すことが想定されています。

アルゴリズムまたは振る舞い

本構造体は、エラーコードからメッセージを解決するためのルックアップテーブル（px_err_list）の要素として機能します。px_strerror はこのリストを走査し、一致する err を持つ error_desc を見つけることで、対応する desc を返します。

呼び出し元と呼び出し先

  • 呼び出し元: contrib/pgcrypto/px.c 内の px_err_list 定義（42行目）および px_strerror 関数（114行目付近）。
  • 呼び出し先: なし（データ構造のため）。
  • 注意: src/test/modules/oauth_validator/t/oauth_server.py に同名の参照が見られますが、これは本構造体とは無関係な外部のテストコードです。

関連データ構造

  • px_err_list: error_desc 型の静的配列。エラーコードとメッセージの全マッピングを保持します。

失敗モードと境界条件

  • 該当するエラーコードが px_err_list に存在しない場合、px_strerror はデフォルトのメッセージを返すか、NULL を返す可能性があります（実装依存）。

保守担当者向けメモ

  • 新しいエラーコードを px.h に追加した際は、必ず px.c 内の px_err_list に対応する error_desc エントリを追加してください。
