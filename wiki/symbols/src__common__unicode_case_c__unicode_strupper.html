unicode_strupper

定義

  • 種別: function
  • 定義箇所: src/common/unicode_case.c:164

目的

入力された UTF-8 文字列を大文字に変換し、結果をバッファに格納する。

シグネチャまたは形状

size_t unicode_strupper(char *dst, size_t dstsize, const char *src, ssize_t srclen, bool full);

入力と出力

  • **入力**:
  • dst: 変換結果を格納するバッファ。
  • dstsize: dst バッファのサイズ（バイト単位）。
  • src: 変換対象の UTF-8 文字列。
  • srclen: src の長さ（バイト単位）。-1 の場合はヌル終端文字列として扱う。
  • full: Unicode 特殊ケースマッピング（ギリシャ語の最終シグマ等）を適用するかどうかのフラグ。
  • **出力**:
  • 変換後の文字列に必要なバッファサイズを size_t で返す。

前提条件と不変条件

  • dstsize が 0 の場合、変換に必要なバッファサイズを計算し、書き込みは行わない。
  • src は有効な UTF-8 文字列である必要がある。

アルゴリズムまたは振る舞い

内部関数 convert_case を呼び出し、CaseUpper モードで変換を実行する。文字列を走査し、各文字に対して casemap を適用して大文字へ変換する。full が真の場合、文脈依存の特殊マッピング（check_special_conditions）が適用される。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/pg_locale_builtin.c:110
  • src/common/unicode/case_test.c:129, 320
  • **呼び出し先**:
  • convert_case (内部関数)

関連データ構造

  • CaseMapResult (変換結果の状態を示す列挙型)

失敗モードと境界条件

  • dstsize が変換結果を格納するのに不十分な場合、切り詰められた結果が格納される可能性がある（convert_case の実装に依存）。
  • srclen が負の値の場合、src がヌル終端されていることを前提とする。

保守担当者向けメモ

  • 本関数は src/include/common/unicode_case.h で公開されている API である。
  • パフォーマンス向上のため、ASCII 文字に対しては高速パスが内部的に利用される。
  • ギリシャ語の最終シグマのような文脈依存の変換が必要な場合は、full フラグを適切に設定すること。
