sepgsql_get_debug_audit

定義

  • 種別: function
  • 定義箇所: contrib/sepgsql/hooks.c:76

目的

SE-PostgreSQL モジュールにおけるデバッグ監査（debug audit）設定の現在の有効状態を取得する。

シグネチャまたは形状

bool sepgsql_get_debug_audit(void)

入力と出力

  • **入力**: なし
  • **出力**: bool 型。デバッグ監査が有効な場合は true、無効な場合は false を返す。

前提条件と不変条件

  • 本関数は、モジュール内のグローバル変数 sepgsql_debug_audit の値を読み取るためのアクセサである。

アルゴリズムまたは振る舞い

1. モジュール内で保持されている sepgsql_debug_audit 変数の値をそのまま返却する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/sepgsql/uavc.c:368: アクセスベクタキャッシュ（AVC）の処理内で、監査ログ出力の判断に使用されている。
  • **呼び出し先**:
  • なし（直接的な関数呼び出しは行わない）。

関連データ構造

  • sepgsql_debug_audit: 本関数が参照するモジュール内の GUC 変数（bool 型）。

失敗モードと境界条件

  • 特筆すべき失敗モードはない。常に現在の設定値を返す。

保守担当者向けメモ

  • 本関数は sepgsql.h で extern 宣言されており、モジュール内の他コンポーネントから監査設定を参照する際の標準的なインターフェースとして機能する。
