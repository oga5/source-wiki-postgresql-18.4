r_Step_1a

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_english.c:485

目的

Porterステミングアルゴリズムの第1段階（Step 1a）を実行し、単語末尾の複数形や所有格の接尾辞を正規化または削除する。

シグネチャまたは形状

static int r_Step_1a(struct SN_env * z)

入力と出力

  • 入力: SN_env 構造体（現在のステミング環境、カーソル位置、文字列データを含む）。
  • 出力: 成功時に 1、失敗または処理対象外の場合に 0、メモリ操作等のエラー発生時に負の値を返す。

前提条件と不変条件

  • z が有効な SN_env 構造体を指していること。
  • find_among_b を使用して、文字列の後方から特定の接尾辞を検索する。
  • 処理対象の文字が SN_env の境界条件（lb）を超えないこと。

アルゴリズムまたは振る舞い

1. **所有格・複数形の処理**:

  • ' (アポストロフィ) または s で終わる単語に対し、a_1 テーブルを用いて後方一致検索を行う。一致した場合は slice_del で削除する。

    • 2. **語尾の置換・削除**:

  • d または s で終わる単語に対し、a_2 テーブルを用いて後方一致検索を行う。
  • among_var の値に応じて以下の処理を行う:
  • 1: ss を ss に置換（実質的な変更なし）。
  • 2: ies または ied を i または ie に置換。
  • 3: s で終わる単語において、直前に母音が存在する場合に s を削除する。

呼び出し元と呼び出し先

  • 呼び出し元: english_ISO_8859_1_stem（stem_ISO_8859_1_english.c 内）および関連する Porter ステマー実装ファイル。
  • 呼び出し先: find_among_b（後方検索）、slice_del（削除）、slice_from_s（置換）、out_grouping_b（母音判定）。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_1, a_2: 検索対象の接尾辞を定義した among 構造体の配列。
  • g_v: 母音グループを定義した grouping 構造体。

失敗モードと境界条件

  • slice_del や slice_from_s が内部で失敗した場合、負の戻り値をそのまま伝播させる。
  • 文字列の先頭（lb）に達した場合、検索や削除処理を中断し 0 を返す。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードである。ロジックの変更が必要な場合は、直接編集せず元の .sbl スクリプトを修正すること。
  • find_among_b による後方一致検索がアルゴリズムの基盤となっている。
