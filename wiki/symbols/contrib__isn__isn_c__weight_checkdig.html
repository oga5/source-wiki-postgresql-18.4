weight_checkdig

定義

  • 種別: function
  • 定義箇所: contrib/isn/isn.c:281

目的

国際標準番号（ISxN）の文字列に対して、重み付け計算を用いたチェックデジットの算出を行います。

シグネチャまたは形状

static unsigned weight_checkdig(char *isn, unsigned size)

入力と出力

  • **入力**:
  • isn: 数値文字列へのポインタ。
  • size: 重み付けの開始値（桁数に関連する係数）。
  • **出力**:
  • 計算されたチェックデジット（unsigned 型）。

前提条件と不変条件

  • isn はヌル終端文字列であること。
  • size は計算の初期重みとして使用され、ループ内でデクリメントされる。

アルゴリズムまたは振る舞い

1. 文字列 isn を走査し、数字（isdigit）である場合に限り、現在の size を重みとして (size * 数値) を weight に加算する。 2. 数字を処理するたびに size をデクリメントする。 3. 走査終了後、weight を 11 で割った余りを求める。 4. 余りが 0 でない場合、11 - 余り を結果として返す。余りが 0 の場合は 0 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/isn/isn.c 内の各入力関数（isbn_in, issn_in 等の内部処理）から、チェックデジットの検証目的で呼び出されます。
  • 具体的には、contrib/isn/isn.c:461, 488, 847, 853 での呼び出しが確認されています。
  • **呼び出し先**:
  • 標準 C ライブラリの isdigit を使用します。

関連データ構造

  • ean13 型（内部表現）の検証プロセスにおいて、入力文字列の妥当性を判定するために使用されます。

失敗モードと境界条件

  • 文字列内に数字以外の文字が含まれる場合、その文字はスキップされ、重み計算には影響しません。
  • size が 1 に達した時点でループは終了します。

保守担当者向けメモ

  • 本関数はチェックデジットの計算ロジックをカプセル化しています。ISBN-10 や ISSN など、重み付けアルゴリズムが共通する識別子に対して再利用されています。
  • 戻り値の範囲は 0 から 10 です。
