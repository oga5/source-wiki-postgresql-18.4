JsonbExtractScalar

定義

  • **種別**: function
  • **定義箇所**: src/backend/utils/adt/jsonb.c:1967

目的

JsonbContainer がスカラー値（JSONB のルート要素として格納された単一の値）である場合に、その値を抽出して JsonbValue 構造体に格納する。

シグネチャまたは形状

bool JsonbExtractScalar(JsonbContainer *jbc, JsonbValue *res)

入力と出力

  • **入力**:
  • jbc: 対象となる JsonbContainer へのポインタ。
  • res: 抽出されたスカラー値を格納するための JsonbValue へのポインタ。
  • **出力**:
  • 戻り値: 抽出に成功した場合は true、失敗した場合は false。
  • res: 成功時は抽出された値が設定される。失敗時は res->type に実際のコンテナ型（jbvArray または jbvObject）が設定される。

前提条件と不変条件

  • jbc は有効な JsonbContainer であること。
  • ルートスカラーは内部的に「要素数1の配列」として格納されているという不変条件に基づき動作する。

アルゴリズムまたは振る舞い

1. JsonContainerIsArray および JsonContainerIsScalar を用いて、対象がスカラー値を含む配列かどうかを検証する。条件を満たさない場合は false を返す。 2. JsonbIteratorInit でイテレータを初期化する。 3. JsonbIteratorNext を順次呼び出し、以下のシーケンスを検証する：

  • WJB_BEGIN_ARRAY: 配列の開始を確認。
  • WJB_ELEM: 唯一の要素を res に抽出。
  • WJB_END_ARRAY: 配列の終了を確認。
  • WJB_DONE: イテレータの終了を確認。

    • 4. 成功時に true を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/jsonb.c 内の複数箇所（主に JSONB 構造の解析時）。
  • src/backend/utils/adt/jsonpath_exec.c（JSONPath 実行時）。
  • **呼び出し先**:
  • JsonbIteratorInit
  • JsonbIteratorNext
  • JsonContainerIsArray (マクロ)
  • JsonContainerIsScalar (マクロ)
  • IsAJsonbScalar (マクロ)

関連データ構造

  • JsonbContainer: JSONB のバイナリ表現を保持する構造体。
  • JsonbValue: JSONB の各ノード（スカラー、オブジェクト、配列）を表現する構造体。
  • JsonbIterator: JsonbContainer を走査するためのイテレータ。

失敗モードと境界条件

  • 対象がスカラー値ではない（オブジェクトや空の配列など）場合、false を返し、res->type に実際の型をセットして終了する。
  • 内部的なアサーション（Assert）により、期待される構造（要素数1の配列など）と異なる場合はデバッグビルド時に停止する。

保守担当者向けメモ

  • 本関数は、JSONB のルートがスカラーであるという特殊な内部表現（rawScalar フラグ付きの配列）を前提としているため、将来的にこの内部表現が変更された場合は修正が必要となる。
