AggSplit

定義

  • 種別: enum
  • 定義箇所: src/include/nodes/nodes.h:380

目的

集約処理（Aggregation）を複数のフェーズに分割して実行する際のモードを識別するために使用されます。

シグネチャまたは形状

typedef enum AggSplit
{
    AGGSPLIT_SIMPLE = 0,
    AGGSPLIT_INITIAL_SERIAL = AGGSPLITOP_SKIPFINAL | AGGSPLITOP_SERIALIZE,
    AGGSPLIT_FINAL_DESERIAL = AGGSPLITOP_COMBINE | AGGSPLITOP_DESERIALIZE,
} AggSplit;

入力と出力

  • 入力: なし（列挙型定数）
  • 出力: なし（列挙型定数）

前提条件と不変条件

  • AGGSPLIT_SIMPLE は分割を行わない標準的な集約処理を示します。
  • AGGSPLIT_INITIAL_SERIAL および AGGSPLIT_FINAL_DESERIAL は、並列集約などの文脈でシリアライズ/デシリアライズを伴う分割処理を定義します。

アルゴリズムまたは振る舞い

集約処理の各フェーズ（初期段階でのシリアライズ、最終段階でのデシリアライズと結合）を制御するためのフラグセットとして機能します。

失敗モードと境界条件

  • 列挙型の値は、集約処理の各フェーズにおけるビットフラグ（AGGSPLITOP_*）の組み合わせとして定義されており、無効なビット組み合わせを指定した場合の動作は定義されていません。

保守担当者向けメモ

  • 本シンボルは、クエリプランナーおよび実行エンジンにおいて集約の並列化戦略を決定する重要な役割を担います。
  • AggSplit の値は、src/include/nodes/primnodes.h 内の Aggref 構造体などで保持されます。
