PGP_PUB_ALGO_TYPE

定義

  • 種別: enum
  • 定義箇所: contrib/pgcrypto/pgp.h:66

目的

OpenPGP (RFC 4880) 規格で定義される公開鍵暗号アルゴリズムを識別するための列挙型です。

シグネチャまたは形状

enum PGP_PUB_ALGO_TYPE
{
    PGP_PUB_RSA_ENCRYPT_SIGN = 1,
    PGP_PUB_RSA_ENCRYPT = 2,
    PGP_PUB_RSA_SIGN = 3,
    PGP_PUB_ELG_ENCRYPT = 16,
    PGP_PUB_DSA_SIGN = 17,
};

入力と出力

  • 入力: 該当なし（定数定義）
  • 出力: PGP 処理において、使用する公開鍵アルゴリズムを示す整数値。

前提条件と不変条件

  • RFC 4880 に準拠した値が割り当てられています。
  • 各列挙値は、PGP 鍵パケット内のアルゴリズム識別子と直接対応します。

アルゴリズムまたは振る舞い

PGP 処理において、暗号化や署名を行う際にどのアルゴリズムを使用するかを決定するための識別子として機能します。

呼び出し元と呼び出し先

  • 呼び出し元: 提供されたソース情報からは特定の呼び出し元を特定できません。
  • 呼び出し先: 該当なし（定数定義のため）。

関連データ構造

  • PGP_PubKey: 公開鍵情報を保持する構造体で、このアルゴリズム型を保持するフィールドを持つことが想定されます。

失敗モードと境界条件

  • 定義されていないアルゴリズム値が指定された場合、PGP 処理ロジック側でエラーハンドリングが必要です。

保守担当者向けメモ

  • 本列挙型は RFC 4880 に基づいています。新しいアルゴリズムの追加が必要な場合は、RFC の仕様を確認してください。
  • pgp-pubenc.c や pgp-pubdec.c などの公開鍵処理ロジックが、この列挙型をどのように参照しているかを確認することを推奨します。
