TCPatternKind

定義

  • 種別: enum
  • 定義箇所: src/bin/psql/tab-complete.in.c:1348

目的

psql のタブ補完機能において、入力された単語と補完パターンを照合する際の「マッチングの種類」を定義します。

シグネチャまたは形状

enum TCPatternKind
{
	Match,
	MatchCS,
	HeadMatch,
	HeadMatchCS,
	TailMatch,
	TailMatchCS,
};

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: なし

前提条件と不変条件

  • psql のタブ補完ロジックにおいて、パターンマッチングの挙動を制御するために使用されます。
  • CS サフィックスが付与されたメンバ（MatchCS 等）は、大文字小文字を区別する（Case-Sensitive）マッチングを意図していると推測されます。

アルゴリズムまたは振る舞い

  • Match: 完全一致による照合。
  • HeadMatch: 先頭部分一致による照合。
  • TailMatch: 末尾部分一致による照合。
  • 各種 CS バリアントは、上記マッチングにおいて大文字小文字の区別を適用します。

呼び出し元と呼び出し先

  • 呼び出し元: src/bin/psql/tab-complete.in.c:1367 にて、TCPattern 構造体のメンバ kind として参照されています。
  • 呼び出し先: 提供されたソース excerpt には、この列挙型を利用する関数やロジックは含まれていません。

関連データ構造

  • TCPattern: この列挙型をメンバとして保持する構造体。

失敗モードと境界条件

  • 列挙型であるため、定義外の値を割り当てた場合の挙動は未定義です。

保守担当者向けメモ

  • 本シンボルは gen_tabcomplete.pl によって生成されるコードの一部として利用される可能性があります。
  • MatchCS 等の各メンバが具体的にどの関数で評価されるかについては、match_previous_words() 等の関連ロジックを調査してください。
