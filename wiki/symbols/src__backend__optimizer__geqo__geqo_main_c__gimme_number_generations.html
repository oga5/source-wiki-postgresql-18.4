gimme_number_generations

定義

  • 種別: function
  • 定義箇所: src/backend/optimizer/geqo/geqo_main.c:351

目的

遺伝的アルゴリズム（GEQO）における世代数（number of generations）を決定します。

シグネチャまたは形状

static int gimme_number_generations(int pool_size)

入力と出力

  • **入力**: pool_size (int) - 個体群のサイズ。
  • **出力**: (int) - 決定された世代数。

前提条件と不変条件

  • グローバル変数 Geqo_generations が 0 より大きい場合、その値が優先的に返されます。

アルゴリズムまたは振る舞い

1. グローバル設定変数 Geqo_generations を確認します。 2. Geqo_generations > 0 であれば、その値をそのまま返します。 3. それ以外の場合、引数として受け取った pool_size をそのまま世代数として返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/optimizer/geqo/geqo_main.c 内の geqo() 関数から呼び出されます。
  • **呼び出し先**: 外部関数への呼び出しはありません。

関連データ構造

  • Geqo_generations (グローバル変数): ユーザー定義の世代数設定。

失敗モードと境界条件

  • Geqo_generations が 0 または負の値である場合、個体群サイズがそのまま世代数として使用されます。

保守担当者向けメモ

  • 本関数は geqo() 内で個体群サイズが決定された直後に呼ばれ、進化ループの反復回数を制御するために使用されます。
