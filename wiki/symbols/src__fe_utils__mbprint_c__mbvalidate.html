mbvalidate

定義

  • 種別: function
  • 定義箇所: src/fe_utils/mbprint.c:391

目的

文字列内の不正なマルチバイト文字を検出し、除去することで、端末表示用の文字列を正規化します。

シグネチャまたは形状

unsigned char *mbvalidate(unsigned char *pwcs, int encoding);

入力と出力

  • **入力**:
  • pwcs: 検証および修正対象となる文字列へのポインタ。
  • encoding: 文字列のエンコーディングID（例: PG_UTF8）。
  • **出力**:
  • 入力として受け取った pwcs ポインタをそのまま返します。

前提条件と不変条件

  • 現在、UTF-8 (PG_UTF8) 以外のエンコーディングに対する検証ロジックは実装されておらず、該当する場合は何もしません。
  • 入力文字列は、関数内で直接修正（不正なバイトの除去）が行われます。

アルゴリズムまたは振る舞い

1. encoding が PG_UTF8 であるかを確認します。 2. PG_UTF8 の場合、mb_utf_validate を呼び出し、文字列内の不正なバイトシーケンスを処理します。 3. その他のエンコーディングの場合は、特段の処理を行いません。 4. 最後に、入力されたポインタ pwcs を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/fe_utils/print.c: content->header および content->cell の処理時に呼び出されています。
  • **呼び出し先**:
  • mb_utf_validate: encoding が PG_UTF8 の場合に呼び出されます。

関連データ構造

  • 特になし。

失敗モードと境界条件

  • UTF-8 以外のエンコーディングが指定された場合、検証は行われず、入力文字列はそのまま返されます。
  • 入力ポインタ pwcs が NULL の場合の挙動については、ソースコード上では明示的なガードは確認できません。

保守担当者向けメモ

  • 本関数は現在 UTF-8 に特化しています。他のエンコーディングで検証が必要な場合は、else ブロック内に適切なルーチンを追加する必要があります。
  • mbvalidate は文字列を直接書き換えるため、呼び出し元で元の文字列の整合性が損なわれないよう注意してください。
