gbt_macad8gt

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_macaddr8.c:29

目的

GiST インデックスにおいて、macaddr8 型の「より大きい（greater than）」比較を行うためのアダプター関数です。

シグネチャまたは形状

bool gbt_macad8gt(const void *a, const void *b, FmgrInfo *flinfo)

入力と出力

  • **入力**:
  • a, b: 比較対象となる macaddr8 型のポインタ。
  • flinfo: 関数呼び出し情報（FmgrInfo）。
  • **出力**:
  • a > b であれば true、そうでなければ false を返します。

前提条件と不変条件

  • a および b は有効な macaddr8 型のデータである必要があります。
  • 内部的に macaddr8_gt 関数を呼び出すため、PostgreSQL の型システムにおける macaddr8 の比較セマンティクスに従います。

アルゴリズムまたは振る舞い

1. a と b を PointerGetDatum を用いて Datum 型に変換します。 2. DirectFunctionCall2 を介して、PostgreSQL ネイティブの macaddr8_gt 関数を呼び出します。 3. 戻り値を DatumGetBool で bool 型に変換して返します。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/btree_gist/btree_macaddr8.c 内の静的構造体（tinfo）の初期化時に参照されています。
  • **呼び出し先**: macaddr8_gt（PostgreSQL ネイティブの比較関数）。

関連データ構造

  • mac8KEY: 本ファイルで定義される macaddr8 の範囲を保持する構造体。
  • tinfo: gbtree_ninfo 型の静的変数。本関数を含む比較関数群を GiST インデックスに提供します。

失敗モードと境界条件

  • a または b が NULL の場合、DirectFunctionCall2 の挙動に依存します。
  • 比較対象の型が macaddr8 ではない場合、型不整合によるエラーが発生する可能性があります。

保守担当者向けメモ

  • 本関数は btree_gist モジュールにおける macaddr8 型の比較ロジックの橋渡し役です。
  • ロジックの変更が必要な場合は、macaddr8_gt 自体の仕様を確認してください。
