gbt_float8_dist

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_float8.c:72

目的

2つの float8 値の間の絶対距離を計算し、GiSTインデックスにおける最近傍検索（KNN）をサポートします。

シグネチャまたは形状

float8 gbt_float8_dist(const void *a, const void *b, FmgrInfo *flinfo)

入力と出力

  • **入力**:
  • a, b: 比較対象の float8 値へのポインタ。
  • flinfo: 関数管理情報（使用されません）。
  • **出力**:
  • 2値間の絶対距離（float8）。

前提条件と不変条件

  • 入力ポインタ a および b は、有効な float8 値を指している必要があります。
  • 減算結果が無限大（inf）になった場合、元の値が無限大でなければオーバーフローとみなされます。

アルゴリズムまたは振る舞い

1. a と b が指す値を float8 として読み込みます。 2. arg1 - arg2 を計算します。 3. 計算結果が inf であり、かつ元の入力値がいずれも inf ではない場合、float_overflow_error() を呼び出してエラーを発生させます。 4. 計算結果の絶対値（fabs(r)）を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: ソースコード内の直接的な呼び出し元は確認されていません。
  • **呼び出し先**:
  • float_overflow_error()（エラー発生時）
  • fabs()（標準ライブラリ）

関連データ構造

  • float8: PostgreSQLの倍精度浮動小数点型。

失敗モードと境界条件

  • **オーバーフロー**: 減算結果が inf になるが、入力値が有限である場合、エラーが発生します。
  • **入力値**: 入力値自体が inf である場合、isinf チェックによりオーバーフローエラーは回避されます。

保守担当者向けメモ

  • 本関数は btree_gist モジュールにおける float8 型の距離計算ロジックです。
  • contrib/btree_gist/btree_float8.c 内の他の関数から参照される可能性がありますが、具体的な呼び出し元はソースコードの静的解析範囲では特定されていません。
