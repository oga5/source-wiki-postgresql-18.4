gbt_macad8lt

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_macaddr8.c:52

目的

macaddr8 型の2つの値に対して「より小さい（less than）」という比較を行い、その結果を真偽値で返す。

シグネチャまたは形状

bool gbt_macad8lt(const void *a, const void *b, FmgrInfo *flinfo)

入力と出力

  • **入力**:
  • a, b: 比較対象となる macaddr8 型のポインタ。
  • flinfo: 関数呼び出し情報（DirectFunctionCall2 で使用）。
  • **出力**:
  • a < b であれば true、そうでなければ false を返す。

前提条件と不変条件

  • 入力ポインタ a および b は有効な macaddr8 型のデータであること。
  • macaddr8_lt 関数がシステムカタログに存在し、呼び出し可能であること。

アルゴリズムまたは振る舞い

1. PointerGetDatum を使用して、入力ポインタ a および b を Datum 型に変換する。 2. DirectFunctionCall2 を介して、PostgreSQL のネイティブ関数 macaddr8_lt を呼び出す。 3. macaddr8_lt の実行結果（Datum）を DatumGetBool で bool 型に変換して返す。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/btree_gist/btree_macaddr8.c 内の静的変数 tinfo（gbtree_ninfo 型）の初期化時に参照されている。
  • **呼び出し先**: macaddr8_lt（PostgreSQL ネイティブ関数）。

関連データ構造

  • macaddr8: 比較対象となるネットワークアドレス型。
  • FmgrInfo: 関数呼び出しのメタデータ。

失敗モードと境界条件

  • 入力ポインタが NULL の場合、PointerGetDatum および DirectFunctionCall2 の挙動に依存する（通常はセグメンテーション違反等のエラーとなる）。
  • macaddr8_lt が内部でエラーを発生させた場合、その例外が伝播する。

保守担当者向けメモ

  • 本関数は macaddr8 型の比較ロジックを btree_gist モジュールに適合させるためのラッパーである。
  • 比較ロジックの詳細は macaddr8_lt に依存しているため、比較仕様の変更が必要な場合は utils/inet.h 関連の実装を確認すること。
