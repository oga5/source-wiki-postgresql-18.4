pg_ucol_open

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/pg_locale_icu.c:229

目的

ICUの ucol_open をラップし、PostgreSQLの環境に適したロケール設定の正規化と、ICUバージョン間の挙動差異を吸収した UCollator オブジェクトを生成する。

シグネチャまたは形状

UCollator *pg_ucol_open(const char *loc_str)

入力と出力

  • **入力**: loc_str (ロケール名を示す文字列)
  • **出力**: 成功時は初期化された UCollator* を返す。
  • **エラー**: loc_str が NULL の場合、またはICUによる照合器の生成に失敗した場合は ereport(ERROR, ...) によりトランザクションを中断する。

前提条件と不変条件

  • loc_str は NULL であってはならない（デフォルト照合器のオープンはサポート外）。
  • ICUのバージョンが55未満の場合、言語コードが "und" であれば "root" に置換される。
  • ICUのバージョンが54未満の場合、照合属性の適用（icu_set_collation_attributes）が必須となる。

アルゴリズムまたは振る舞い

1. **入力検証**: loc_str が NULL であればエラーを投げる。 2. **ロケール名の正規化**: ICU 54以前の環境において、言語コードが "und" の場合は "root" に置換する。 3. **照合器の生成**: ucol_open を呼び出し、失敗した場合はエラーを報告する。 4. **バージョン依存の属性設定**: ICU 53以前の環境では、icu_set_collation_attributes を呼び出し、属性設定に失敗した場合は生成した照合器を解放してエラーを報告する。 5. **クリーンアップ**: 正規化のために確保したメモリ（fixed_str）があれば解放する。 6. **戻り値**: 生成された UCollator ポインタを返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/pg_locale.c
  • src/backend/utils/adt/pg_locale_icu.c (複数箇所)
  • **呼び出し先**:
  • ucol_open (ICU API)
  • uloc_getLanguage (ICU API)
  • icu_set_collation_attributes (内部関数)
  • ucol_close (ICU API)
  • palloc / pfree (PostgreSQLメモリ管理)

関連データ構造

  • UCollator: ICUの照合器オブジェクト。
  • UErrorCode: ICUの処理結果を示すステータスコード。

失敗モードと境界条件

  • loc_str が NULL の場合: elog(ERROR) で停止。
  • ICUのロケール解析失敗: ereport(ERROR) で停止。
  • ucol_open 失敗: ereport(ERROR) で停止。
  • ICU 54未満での属性設定失敗: ucol_close を実行した上で ereport(ERROR) で停止。

保守担当者向けメモ

  • ICUのバージョン判定（U_ICU_VERSION_MAJOR_NUM）に基づき、古いバージョン特有の回避策が実装されている。
  • ICU 54未満の環境では、ucol_open 後の属性設定が必須であるため、エラーハンドリング時に照合器のリークが発生しないよう注意すること。
