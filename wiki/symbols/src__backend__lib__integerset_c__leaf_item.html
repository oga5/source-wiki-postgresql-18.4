leaf_item

定義

  • 種別: struct
  • 定義箇所: src/backend/lib/integerset.c:160

目的

B-tree のリーフノード内において、整数の集合を圧縮して保持するための最小単位として機能します。

シグネチャまたは形状

typedef struct
{
	uint64		first;			/* このアイテム内の最初の整数 */
	uint64		codeword;		/* 'first' からの差分を Simple-8b でエンコードしたもの */
} leaf_item;

入力と出力

  • 本構造体はデータ保持用であり、関数ではないため直接的な入出力はありません。
  • 内部の codeword は、simple8b_encode によって生成され、simple8b_decode によって復元される対象となります。

前提条件と不変条件

  • first は、そのアイテムが保持する整数の集合における最小値です。
  • codeword は、first 以降の整数値との差分を Simple-8b アルゴリズムでパッキングしたものです。

アルゴリズムまたは振る舞い

  • リーフノード内で複数の leaf_item が配列として保持されます。
  • 検索時には、intset_binsrch_leaf を用いて、目的の値を含む可能性のある leaf_item を二分探索で特定します。
  • 特定後、simple8b_contains 等を用いて codeword 内の差分を走査し、値の存在を確認します。

呼び出し元と呼び出し先

  • **呼び出し元（参照箇所）**:
  • src/backend/lib/integerset.c:176 (intset_leaf_node 構造体内の配列)
  • src/backend/lib/integerset.c:267 (intset_binsrch_leaf の引数)
  • src/backend/lib/integerset.c:431, 559, 658 (ローカル変数やポインタとして利用)
  • src/backend/lib/integerset.c:746 (intset_binsrch_leaf のシグネチャ)
  • **呼び出し先**:
  • 直接的な関数呼び出しはありませんが、本構造体のフィールドは Simple-8b 関連のエンコード/デコード関数によって操作されます。

関連データ構造

  • intset_leaf_node: leaf_item の配列を保持するリーフノード構造体。
  • IntegerSet: 本構造体を含む B-tree 全体を管理するルート構造体。

失敗モードと境界条件

  • データのクラスタリング（連続性）が低い場合、Simple-8b による圧縮効率が低下し、8 バイト/整数に近づきます。
  • 順序付けられた挿入が前提であり、昇順以外の値が入力された場合の動作は保証されません。

保守担当者向けメモ

  • 本構造体は integerset.c のリーフノードのメモリレイアウトに直結しています。フィールドの順序や型を変更する場合は、関連する simple8b 処理および二分探索ロジックへの影響を考慮してください。
