ResetLogicalStreamingState

定義

  • 種別: function
  • 定義箇所: src/backend/replication/logical/logical.c:1943

目的

論理ストリーミングに関連するグローバルな状態変数を初期化し、トランザクション処理の境界で状態をリセットするために使用されます。

シグネチャまたは形状

void ResetLogicalStreamingState(void)

入力と出力

  • 入力: なし
  • 出力: なし（副作用としてグローバル変数を更新）

前提条件と不変条件

  • 本関数は、トランザクションの終了処理やリセットが必要なタイミングで呼び出されることを前提としています。

アルゴリズムまたは振る舞い

以下のグローバル変数を初期値に設定します。 1. CheckXidAlive を InvalidTransactionId に設定。 2. bsysscan を false に設定。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/backend/access/transam/xact.c:2902
  • src/backend/access/transam/xact.c:5297
  • 呼び出し先:
  • 本関数内での直接的な関数呼び出しはありません。

関連データ構造

  • InvalidTransactionId (トランザクションIDの無効値)
  • CheckXidAlive (グローバル変数)
  • bsysscan (グローバル変数)

失敗モードと境界条件

  • 本関数は値を代入するのみであり、失敗する条件はありません。
