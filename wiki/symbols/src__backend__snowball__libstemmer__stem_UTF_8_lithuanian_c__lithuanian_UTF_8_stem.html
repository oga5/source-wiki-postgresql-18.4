lithuanian_UTF_8_stem

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_lithuanian.c:752

目的

UTF-8 エンコーディングされたリトアニア語の単語に対し、接尾辞の除去および綴りの正規化を行い、語幹を抽出する。

シグネチャまたは形状

int lithuanian_UTF_8_stem(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（ステミング環境構造体。処理対象の文字列およびカーソル位置 c、文字列長 l を保持）
  • **出力**: int（成功時に 1、失敗時に負の値を返す）

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • 処理開始時に、母音の位置に基づきステミング対象領域（I[0]）が決定される。
  • 処理中、カーソル位置 z->c は文字列の範囲内（0 から z->l）で操作される。

アルゴリズムまたは振る舞い

1. **領域決定**: 母音の位置を特定し、ステミング対象領域の境界を z->I[0] に設定する。 2. **綴り修正（前処理）**: r_fix_conflicts を呼び出し、競合する綴りを補正する。 3. **接尾辞除去**: r_step1 および r_step2 を順次実行し、定義されたテーブルに基づき接尾辞を削除する。 4. **綴り修正（後処理）**: r_fix_chdz および r_fix_gd を呼び出し、除去過程で生じた不自然な綴りを修正する。 5. **状態復元**: 処理終了後、カーソル位置を初期状態に戻し、成功値 1 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: 提供されたソース excerpt 内に明示的な呼び出し元は記載されていない。
  • **呼び出し先**: r_fix_conflicts, r_step1, r_fix_chdz, r_step2, r_fix_gd を呼び出す。また、out_grouping_U, in_grouping_U, len_utf8, skip_utf8 等のランタイム関数を利用する。

関連データ構造

  • struct SN_env: ステミング処理の実行環境を保持する構造体。

失敗モードと境界条件

  • 各ステップ（r_step1 等）で負の値が返された場合、その値をそのまま呼び出し元へ伝播させ終了する。
  • 文字列長が短い場合や、特定の条件（len_utf8 のチェック等）を満たさない場合は、一部の処理をスキップする。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードである。直接編集せず、元の .sbl スクリプトを修正すること。
  • 処理は z->c（カーソル）と z->l（長さ）を直接操作する低レベルなポインタ演算に依存しているため、デバッグ時は SN_env の状態遷移を追跡すること。
