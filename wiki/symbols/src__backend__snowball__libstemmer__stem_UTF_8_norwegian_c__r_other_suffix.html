r_other_suffix

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_norwegian.c:227

目的

ノルウェー語のステミング処理において、特定の接尾辞を単語末尾から削除する。

シグネチャまたは形状

static int r_other_suffix(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング実行環境）
  • 出力: 成功時に 1、失敗時に 0、または slice_del の実行結果（負の値）を返す。

前提条件と不変条件

  • z->I[1] に格納された領域境界（R1等）よりカーソル位置 z->c が後ろにあること。
  • find_among_b を用いた後方検索が可能な状態であること。

アルゴリズムまたは振る舞い

1. カーソル位置 z->c が領域境界 z->I[1] より前であれば即座に終了する。 2. 検索範囲を z->I[1] から現在のカーソル位置までに限定する。 3. 単語末尾の文字が特定のビットマスク条件（4718720 に基づく判定）を満たすか確認する。 4. a_2 配列を用いて、単語末尾から接尾辞を後方検索する。 5. マッチした接尾辞の開始位置を z->bra に設定し、slice_del を呼び出して該当部分を削除する。 6. 処理終了後、検索範囲の制限を元に戻す。

呼び出し元と呼び出し先

  • 呼び出し元: norwegian_UTF_8_stem（src/backend/snowball/libstemmer/stem_UTF_8_norwegian.c 内）
  • 呼び出し先: find_among_b（検索用）、slice_del（削除用）

関連データ構造

  • SN_env: ステミング実行環境。
  • a_2: 接尾辞マッチングに使用される among 構造体の配列。

失敗モードと境界条件

  • カーソル位置が領域境界より前にある場合、何もせず 0 を返す。
  • find_among_b が接尾辞を見つけられない場合、0 を返す。
  • slice_del がメモリ不足や不正な操作で失敗した場合、そのエラーコードを伝播させる。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードであるため、直接編集しないこと。
  • UTF-8 エンコーディングを前提としており、文字単位ではなくバイト単位の操作を行っている点に注意。
  • z->lb のスタック操作（mlimit1）は、後方検索の範囲を正しく制御するために不可欠である。
