brinRevmapTerminate

定義

  • 種別: function
  • 定義箇所: src/backend/access/brin/brin_revmap.c:99

目的

BrinRevmap 構造体が保持するリソースを解放し、アクセス状態を終了させる。

シグネチャまたは形状

void brinRevmapTerminate(BrinRevmap *revmap)

入力と出力

  • **入力**: revmap (BrinRevmap *) - 終了対象のアクセス状態を保持する構造体へのポインタ。
  • **出力**: なし。

前提条件と不変条件

  • revmap ポインタは有効であること。
  • revmap->rm_metaBuf は有効なバッファを参照していること。

アルゴリズムまたは振る舞い

1. revmap->rm_metaBuf に保持されているメタページのバッファを解放する (ReleaseBuffer)。 2. revmap->rm_currBuf が InvalidBuffer でない場合、現在キャッシュしているリバースマップページのバッファを解放する (ReleaseBuffer)。 3. revmap 構造体自体をメモリから解放する (pfree)。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/access/brin/brin.c (複数箇所)
  • src/backend/access/brin/brin_revmap.c (複数箇所)
  • **呼び出し先**:
  • ReleaseBuffer (PostgreSQL バッファマネージャ)
  • pfree (PostgreSQL メモリ管理)

関連データ構造

  • BrinRevmap: revmap へのアクセス状態を保持する構造体。

失敗モードと境界条件

  • revmap が NULL の場合、メモリ保護違反が発生する可能性がある。
  • rm_metaBuf が無効なバッファである場合、ReleaseBuffer 内部でエラーが発生する可能性がある。

保守担当者向けメモ

  • 本関数は brinRevmapInitialize で確保されたリソースのクリーンアップを担う。
  • BrinRevmap 構造体のライフサイクル管理において、呼び出し元が本関数を呼び出した後に revmap ポインタを参照しないよう注意すること。
