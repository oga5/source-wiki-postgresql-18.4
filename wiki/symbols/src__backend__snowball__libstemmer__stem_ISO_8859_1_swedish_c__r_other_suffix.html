r_other_suffix

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_swedish.c:225

目的

スウェーデン語のステミング処理において、単語末尾の特定の接尾辞を削除または置換する。

シグネチャまたは形状

static int r_other_suffix(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（Snowballステミング環境構造体）
  • 出力: 成功時に 1、失敗または処理対象外の場合に 0、エラー発生時に負の値を返す。

前提条件と不変条件

  • z->I[1]（領域R1の開始位置）が設定されていること。
  • 現在のカーソル位置 z->c が z->I[1] 以上であること。

アルゴリズムまたは振る舞い

1. **領域チェック**: カーソル位置がR1領域内にあるか確認する。 2. **マッチング**: a_2 配列を用いて、単語末尾から接尾辞を検索する。 3. **条件分岐**:

  • case 1: マッチした接尾辞を削除する。
  • case 2: g_ost_ending グループに含まれない場合、接尾辞を s_0 に置換する。
  • case 3: 接尾辞を s_1 に置換する。

    • 4. 処理の過程で z->lb（下限）を一時的に変更し、処理終了後に復元する。

呼び出し元と呼び出し先

  • 呼び出し元: swedish_ISO_8859_1_stem（および同等のDanish/Norwegian用ステミング関数）から呼び出される。
  • 呼び出し先: find_among_b, slice_del, in_grouping_b, slice_from_s（Snowballランタイム関数）。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_2: 接尾辞マッチングに使用される among 構造体の配列。
  • g_ost_ending: case 2 の条件判定に使用される文字グループ。

失敗モードと境界条件

  • カーソル位置がR1領域外の場合、直ちに 0 を返す。
  • find_among_b で接尾辞が見つからない場合、0 を返す。
  • slice_del や slice_from_s が失敗（負の値を返却）した場合、その値をそのまま伝播させる。

保守担当者向けメモ

  • 本関数は自動生成コードであるため、直接編集せず、元のSnowballスクリプト（.sbl）を修正すること。
  • z->p（文字列バッファ）への直接アクセスはランタイムの仕様に依存するため、変更時は注意が必要。
