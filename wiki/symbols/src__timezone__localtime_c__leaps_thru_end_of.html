leaps_thru_end_of

定義

  • 種別: function
  • 定義箇所: src/timezone/localtime.c:1405

目的

指定された年（y）までの閏年の総数を計算します。

シグネチャまたは形状

static int leaps_thru_end_of(const int y);

入力と出力

  • **入力**: y (int型) - 閏年の計算対象となる年。
  • **出力**: int型 - 紀元（0年）から指定された年までの閏年の累計数。

前提条件と不変条件

  • 負の年（紀元前）と正の年（紀元後）の両方を扱うことができます。
  • 内部的に leaps_thru_end_of_nonneg を利用して計算を行います。

アルゴリズムまたは振る舞い

1. 入力された年 y が負の場合、y を正の数に変換し、leaps_thru_end_of_nonneg の結果に対して負のオフセット補正を適用します。 2. y が 0 以上の場合、直接 leaps_thru_end_of_nonneg(y) を呼び出します。

関連データ構造

  • 特になし。

失敗モードと境界条件

  • y が非常に小さい負の値や大きい正の値である場合、整数オーバーフローの可能性があるため、呼び出し元での入力値の範囲に依存します。

保守担当者向けメモ

  • 本関数は leaps_thru_end_of_nonneg のラッパーとして機能しています。計算ロジックの変更が必要な場合は、非負の計算を行う leaps_thru_end_of_nonneg 側を確認してください。
