mda_get_range

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/arrayutils.c:152

目的

多次元配列の各次元における範囲（要素数）を計算します。

シグネチャまたは形状

void mda_get_range(int n, int *span, const int *st, const int *endp)

入力と出力

  • n (int): 配列の次元数。
  • span (int*): 出力用。各次元の要素数（範囲）を格納する配列。
  • st (const int*): 各次元の開始インデックスを格納した配列。
  • endp (const int*): 各次元の終了インデックスを格納した配列。
  • 戻り値: なし（span 配列を書き換えることで結果を返却）。

前提条件と不変条件

  • span, st, endp は少なくとも n 個の要素を持つ有効なメモリ領域を指している必要があります。
  • 各次元において endp[i] >= st[i] であることが期待されます。

アルゴリズムまたは振る舞い

1. 0 から n-1 までの各次元 i についてループを実行します。 2. 各次元の範囲を span[i] = endp[i] - st[i] + 1 の式で計算し、span 配列に格納します。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/backend/utils/adt/arrayfuncs.c 内の複数の箇所（例: 2122, 3012, 5059, 5132, 5218 行目）から呼び出されています。
  • 呼び出し先:
  • 本関数は算術演算のみを行うため、他の関数を呼び出しません。

関連データ構造

  • ArrayType (PostgreSQL の配列内部表現に関連)

失敗モードと境界条件

  • n が負の値や不適切な値である場合、メモリ破壊や不正なアクセスが発生する可能性があります。本関数自体には境界チェックは含まれていません。

保守担当者向けメモ

  • mda_ は「Multi-Dimensional Array」の略称です。
  • 本関数は次元の範囲計算という数学的な処理のみを担うため、呼び出し元で次元数やインデックスの妥当性を検証した上で使用してください。
