JsonPathBool

定義

  • 種別: enum
  • 定義箇所: src/backend/utils/adt/jsonpath_exec.c:127

目的

SQL/JSON パス言語における述語評価の結果を表現する三値論理型です。

シグネチャまたは形状

typedef enum JsonPathBool
{
	jpbFalse = 0,
	jpbTrue = 1,
	jpbUnknown = 2
} JsonPathBool;

入力と出力

  • **出力**: 述語評価関数（executeBoolItem 等）の戻り値として使用されます。
  • **変換**: 評価結果は、最終的に jsonb 型の真偽値または NULL に変換されます。

前提条件と不変条件

  • 述語評価において、エラーが発生した場合は jpbUnknown が返されることがあります。

アルゴリズムまたは振る舞い

  • SQL/JSON 標準に準拠した三値論理（True, False, Unknown）を保持します。
  • executeBoolItem や executePredicate などの述語評価エンジンにおいて、条件判定の論理結果として利用されます。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • executeBoolItem
  • executeNestedBoolItem
  • executePredicate
  • executeStartsWith
  • executeLikeRegex
  • executeComparison
  • compareItems
  • **呼び出し先**:
  • 本シンボル自体は列挙型であり、関数呼び出しは行いません。

関連データ構造

  • JsonPathItem: 述語評価の対象となるパスアイテム。
  • JsonPathExecContext: 実行時の状態を保持するコンテキスト。
  • JsonbValue: 比較対象となる JSON 値。

失敗モードと境界条件

  • 述語評価中にエラーが発生した場合、jpbUnknown が返される仕様となっています。

保守担当者向けメモ

  • 本列挙型は jsonpath の述語評価における論理状態を定義するものです。論理演算の追加や変更を行う際は、この三値論理の整合性を維持してください。
