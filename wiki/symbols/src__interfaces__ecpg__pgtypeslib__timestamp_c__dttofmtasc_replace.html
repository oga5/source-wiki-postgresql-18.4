dttofmtasc_replace

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/pgtypeslib/timestamp.c:299

目的

timestamp 型の値を、指定されたフォーマット文字列（fmtstr）に従って文字列へ変換します。strftime に似た形式で日付・時刻要素をバッファへ書き込みます。

シグネチャまたは形状

static int dttofmtasc_replace(timestamp *ts, date dDate, int dow, struct tm *tm, char *output, int *pstr_len, const char *fmtstr);

入力と出力

  • **入力**:
  • ts: 変換対象のタイムスタンプ値。
  • dDate, dow, tm: 日付、曜日、時刻の分解情報。
  • fmtstr: フォーマット指定文字列。
  • **出力**:
  • output: 変換結果を格納するバッファ。
  • pstr_len: 残りのバッファサイズ（更新される）。
  • **戻り値**: 成功時は 0、エラー時は -1 またはその他の非ゼロ値。

前提条件と不変条件

  • pstr_len は output の有効な残りサイズを指している必要があります。
  • tm 構造体は有効な時刻情報を含んでいる必要があります。
  • 一部のフォーマット指定子（%a, %A, %b, %B, %c）はロケール対応が不完全です。

アルゴリズムまたは振る舞い

1. fmtstr を走査し、% プレースホルダーを検出します。 2. プレースホルダーの種類に応じて、tm 構造体や関連する定数配列から値を抽出します。 3. PGTYPES_TYPE_* 型に基づき、値をバッファへ書き込みます。 4. %D などの複合的な指定子や、%E（代替フォーマット）などの特殊なケースでは、再帰呼び出しや strftime を利用して処理します。 5. 処理中、pstr_len を減算し、バッファの残り容量を管理します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/interfaces/ecpg/pgtypeslib/timestamp.c 内の各所（PGTYPEStimestamp_fmt_asc 等から再帰的または直接呼び出し）。
  • **呼び出し先**:
  • 自身（再帰呼び出し: %D 等の処理時）。
  • 標準ライブラリ関数: strftime。

関連データ構造

  • timestamp: int64 型（マイクロ秒単位）。
  • struct tm: C標準の日付・時刻構造体。
  • union un_fmt_comb: フォーマット変換用の一時的な値保持構造。

失敗モードと境界条件

  • fmtstr が空、または不正な指定子が含まれる場合。
  • バッファサイズが不足する場合（pstr_len が適切に管理されない場合）。
  • strftime が失敗した場合（%E, %G, %g 等の処理時）、-1 を返します。

保守担当者向けメモ

  • %a, %A, %b, %B は現在ロケール非依存です。将来的なロケール対応時にはこれらの定数配列の参照箇所を修正する必要があります。
  • %E や %G などの strftime を利用する箇所では、tm->tm_mon の調整（1ベースから0ベースへの変換）が必須です。修正時はこの前後関係を維持してください。
