array_boundary

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/ecpglib/data.c:32

目的

SQL 配列のパース処理において、現在の文字が配列の終端（境界）であるかを判定します。

シグネチャまたは形状

static bool array_boundary(enum ARRAY_TYPE isarray, char c);

入力と出力

  • **入力**:
  • isarray: 配列の型を示す列挙型 (enum ARRAY_TYPE)。
  • c: 判定対象の文字。
  • **出力**:
  • bool: isarray が ECPG_ARRAY_ARRAY であり、かつ c が } である場合に true を返します。それ以外の場合は false を返します。

前提条件と不変条件

  • isarray が ECPG_ARRAY_ARRAY であることが、境界判定の前提となります。

アルゴリズムまたは振る舞い

1. isarray が ECPG_ARRAY_ARRAY であるかを確認します。 2. 文字 c が配列の終了文字である } と一致するかを評価します。 3. 両方の条件を満たす場合にのみ true を返し、配列の終端であることを示します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/interfaces/ecpg/ecpglib/data.c 内の複数箇所（配列の走査ループや条件分岐において、要素の区切りや終端を判定するために呼び出されています）。
  • **呼び出し先**:
  • 外部関数への呼び出しはありません。

関連データ構造

  • enum ARRAY_TYPE: 配列の型を定義する列挙型。

失敗モードと境界条件

  • isarray が ECPG_ARRAY_ARRAY 以外の場合、文字の内容に関わらず常に false を返します。
  • 文字列が \0（終端）に達した場合の挙動は、呼び出し側のループ条件に依存します。

保守担当者向けメモ

  • 本関数は array_delimiter と対になって使用されることが多く、SQL 配列のパースロジックの根幹を成します。
  • ECPG_ARRAY_ARRAY 以外の配列型に対する拡張が必要な場合は、本関数の条件式を更新する必要があります。
