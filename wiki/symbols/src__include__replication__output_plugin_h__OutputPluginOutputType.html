OutputPluginOutputType

定義

  • 種別: enum
  • 定義箇所: src/include/replication/output_plugin.h:17

目的

論理デコーディングにおいて、出力プラグインが生成するデータの形式（バイナリまたはテキスト）を識別するために使用されます。

シグネチャまたは形状

typedef enum OutputPluginOutputType
{
	OUTPUT_PLUGIN_BINARY_OUTPUT,
	OUTPUT_PLUGIN_TEXTUAL_OUTPUT,
} OutputPluginOutputType;

入力と出力

  • 本シンボル自体は列挙型であり、入力や出力を持つ関数ではありません。
  • 設定値として OutputPluginOptions 構造体内の output_type フィールドに格納されます。

前提条件と不変条件

  • OUTPUT_PLUGIN_BINARY_OUTPUT または OUTPUT_PLUGIN_TEXTUAL_OUTPUT のいずれか一つを選択する必要があります。

アルゴリズムまたは振る舞い

  • 論理デコーディングのコアエンジンは、この値に基づいて出力データの処理方法を決定します。

呼び出し元と呼び出し先

  • 呼び出し元: src/include/replication/output_plugin.h 内の OutputPluginOptions 構造体で参照されています。
  • 呼び出し先: 情報なし。

関連データ構造

  • OutputPluginOptions: 出力プラグインの設定を保持する構造体であり、本型をメンバとして含みます。

失敗モードと境界条件

  • 定義外の値を設定した場合の挙動は未定義です。

保守担当者向けメモ

  • 新しい出力形式を追加する必要がある場合は、この列挙型を拡張してください。
