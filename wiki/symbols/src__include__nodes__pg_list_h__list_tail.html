list_tail

定義

  • 種別: function
  • 定義箇所: src/include/nodes/pg_list.h:134

目的

リストの末尾要素を保持する ListCell へのポインタを取得します。

シグネチャまたは形状

static inline ListCell *list_tail(const List *l)

入力と出力

  • **入力**: l (リスト構造体へのポインタ)
  • **出力**: リストの末尾要素を指す ListCell へのポインタ。リストが空（NIL）の場合は NULL を返します。

前提条件と不変条件

  • l が NULL でない場合、l->length は 1 以上である必要があります。
  • リストのヘッダ構造体 List は、elements 配列を保持していることが前提です。

アルゴリズムまたは振る舞い

1. 入力されたリスト l が NULL かどうかを判定します。 2. l が NULL の場合、NULL を返します。 3. l が有効な場合、l->elements 配列のインデックス l->length - 1 にある要素のアドレスを返します。

呼び出し元と呼び出し先

  • 呼び出し元: 提供されたソース情報には具体的な呼び出し元の記述はありません。
  • 呼び出し先: 内部で List 構造体のメンバ (elements, length) に直接アクセスします。

関連データ構造

  • List: リストのヘッダ構造体。
  • ListCell: リスト内の各要素を保持する共用体。

失敗モードと境界条件

  • l が NIL（NULL）の場合、安全に NULL を返します。
  • l が空リスト（length == 0）の場合、elements[-1] を参照しようとするため、未定義の動作を引き起こす可能性があります。

保守担当者向けメモ

  • 本関数は static inline で定義されており、ヘッダーファイル内で完結しています。
  • リストの末尾要素へのアクセスは、リストの長さが 0 でないことを呼び出し側で保証する必要があります。
