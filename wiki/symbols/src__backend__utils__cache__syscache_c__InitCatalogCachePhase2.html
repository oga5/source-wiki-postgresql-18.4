InitCatalogCachePhase2

定義

  • 種別: function
  • 定義箇所: src/backend/utils/cache/syscache.c:179

目的

システムカタログキャッシュの初期化プロセスを完了させるために呼び出されます。

シグネチャまたは形状

void InitCatalogCachePhase2(void)

入力と出力

  • 入力: なし
  • 出力: なし（副作用としてシステムキャッシュの状態を更新）

前提条件と不変条件

  • CacheInitialized が真であること（Assert により検証）。

アルゴリズムまたは振る舞い

1. SysCacheSize で定義された全キャッシュエントリをループ処理します。 2. 各キャッシュに対して InitCatCachePhase2(SysCache[cacheId], true) を呼び出し、初期化の第2フェーズを実行します。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/utils/cache/relcache.c:4377
  • 呼び出し先: InitCatCachePhase2（catcache.c 内の関数と推測されるが、本ソース excerpt に定義は含まれない）

関連データ構造

  • SysCache[]: システムキャッシュを保持する CatCache 構造体の配列。

失敗モードと境界条件

  • CacheInitialized が偽の場合、Assert によりプロセスが停止します。

保守担当者向けメモ

  • 本関数はシステム起動時の初期化シーケンスの一部です。キャッシュの初期化を2段階に分けることで、依存関係のあるリレーションキャッシュ等の構築順序を制御しています。
