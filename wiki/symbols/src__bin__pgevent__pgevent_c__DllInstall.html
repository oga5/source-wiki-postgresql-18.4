DllInstall

定義

  • 種別: function
  • 定義箇所: src/bin/pgevent/pgevent.c:37

目的

Windows環境において、regsvr32 コマンド経由でイベントソース名を指定してPostgreSQLのイベントログ登録を制御します。

シグネチャまたは形状

HRESULT DllInstall(BOOL bInstall, LPCWSTR pszCmdLine);

入力と出力

  • **入力**:
  • bInstall: インストール時は TRUE、アンインストール時は FALSE。
  • pszCmdLine: イベントソース名を指定するコマンドライン文字列。
  • **出力**:
  • S_OK: 常に成功を返します。

前提条件と不変条件

  • 管理者権限で実行される必要があります（レジストリ操作のため）。
  • regsvr32 の仕様上の挙動を回避するため、インストール時には /n オプション（regsvr32 /n /i）を併用することが推奨されます。

アルゴリズムまたは振る舞い

1. pszCmdLine が空でない場合、その値をグローバル変数 event_source にコピーします。 2. bInstall が TRUE の場合、DllRegisterServer() を呼び出してレジストリへの登録処理を実行します。 3. regsvr32 /i の仕様により、インストール時に DllRegisterServer が先に呼び出される問題を回避するため、/n オプションなしでの実行には注意が必要です。

呼び出し元と呼び出し先

  • **呼び出し元**: regsvr32 コマンド（外部ツール）。
  • **呼び出し先**: DllRegisterServer()（bInstall が真の場合）。

関連データ構造

  • event_source: イベントソース名を保持するグローバル変数。

失敗モードと境界条件

  • regsvr32 /i を単独で実行した場合、DllRegisterServer が先に呼び出されるため、デフォルトのイベントソース登録が意図せず上書きされる可能性があります。

保守担当者向けメモ

  • 本関数は regsvr32 の特殊な挙動に対するハックを含んでいます。修正を行う際は、インストールとアンインストール時の呼び出し順序（DllRegisterServer / DllUnregisterServer との前後関係）に十分注意してください。
