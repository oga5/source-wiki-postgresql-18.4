VarsupShmemSize

定義

  • 種別: function
  • 定義箇所: src/backend/access/transam/varsup.c:40

目的

トランザクションIDおよびOIDの生成状態を保持する共有メモリ構造体（TransamVariablesData）に必要なメモリサイズを算出します。

シグネチャまたは形状

Size VarsupShmemSize(void)

入力と出力

  • 入力: なし
  • 出力: Size 型（TransamVariablesData 構造体のバイトサイズ）

前提条件と不変条件

  • 共有メモリの初期化プロセスにおいて、他の共有メモリ領域と合計して割り当てられる前に呼び出される必要があります。

アルゴリズムまたは振る舞い

  • sizeof(TransamVariablesData) を返します。この値は、システム起動時に共有メモリを確保する際の計算に使用されます。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/storage/ipc/ipci.c 内の共有メモリ初期化処理（ShmemCreate 関連のサイズ計算時）。
  • 呼び出し先: なし。

関連データ構造

  • TransamVariablesData: ID生成に必要な全状態（次XID、次OID、Wraparound閾値など）を保持する共有メモリ上の構造体。

失敗モードと境界条件

  • 特になし。常に固定値を返します。

保守担当者向けメモ

  • TransamVariablesData 構造体の定義が変更された場合、この関数が返すサイズも自動的に更新されますが、共有メモリレイアウトの変更時には注意が必要です。
