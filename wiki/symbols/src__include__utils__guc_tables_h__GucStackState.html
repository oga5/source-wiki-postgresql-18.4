GucStackState

定義

  • 種別: enum
  • 定義箇所: src/include/utils/guc_tables.h:111

目的

PostgreSQLのGUC（Grand Unified Configuration）システムにおいて、設定変数の変更履歴を管理するスタック（GucStack）の各エントリが、どのような操作によって生成されたかを識別するために使用されます。

シグネチャまたは形状

typedef enum
{
    GUC_SAVE,
    GUC_SET,
    GUC_LOCAL,
    GUC_SET_LOCAL,
} GucStackState;

入力と出力

  • 本シンボルは列挙型であり、関数ではありません。
  • GucStack 構造体内の state フィールドの型として使用されます。

前提条件と不変条件

  • GucAction 列挙型と類似していますが、SET と LOCAL が組み合わさった状態を表現するために、本列挙型には4番目の状態（GUC_SET_LOCAL）が含まれています。

アルゴリズムまたは振る舞い

各列挙値は以下の操作に対応します：

  • GUC_SAVE: 関数実行に伴う SET オプションによるエントリ。
  • GUC_SET: 通常の SET コマンドによるエントリ。
  • GUC_LOCAL: SET LOCAL コマンドによるエントリ。
  • GUC_SET_LOCAL: SET の後に SET LOCAL が実行されたことによるエントリ。

呼び出し元と呼び出し先

  • 呼び出し元および呼び出し先の具体的な関数情報は、提供されたソース抜粋からは特定できません。
  • src/include/utils/guc_tables.h:124 にて、GucStack 構造体のメンバとして参照されています。

関連データ構造

  • GucStack: 設定変更のスタック構造。本列挙型を保持します。
  • config_generic: GUC変数の基底構造体。

失敗モードと境界条件

  • 該当なし（列挙型定義のため）。

保守担当者向けメモ

  • GucAction との混同を避けてください。本列挙型はスタック管理専用の拡張状態を含んでいます。
