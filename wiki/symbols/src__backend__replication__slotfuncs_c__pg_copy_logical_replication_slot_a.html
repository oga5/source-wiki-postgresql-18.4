pg_copy_logical_replication_slot_a

定義

  • 種別: function
  • 定義箇所: src/backend/replication/slotfuncs.c:860

目的

論理レプリケーションスロットをコピーするためのSQLインターフェースを提供します。

シグネチャまたは形状

Datum pg_copy_logical_replication_slot_a(PG_FUNCTION_ARGS)

入力と出力

  • 入力: PG_FUNCTION_ARGS（SQL経由で渡される引数）
  • 出力: Datum（コピーされたスロットの情報を含むタプル）

前提条件と不変条件

  • 内部ヘルパー関数 copy_replication_slot の制約に従います。
  • ソーススロットが無効化されている場合や、WAL予約が完了していない場合にはコピーが拒否されます。

アルゴリズムまたは振る舞い

1. copy_replication_slot 関数を呼び出します。 2. その際、論理スロットであることを示すフラグ true を引数として渡します。 3. copy_replication_slot の戻り値をそのまま呼び出し元へ返します。

呼び出し元と呼び出し先

  • 呼び出し元: SQLエンジン（SQLコマンド経由での実行）
  • 呼び出し先: copy_replication_slot

関連データ構造

  • PG_FUNCTION_ARGS: PostgreSQLの関数呼び出し規約で使用される引数構造体。

失敗モードと境界条件

  • copy_replication_slot の実装に依存します。スロットの不整合を防ぐため、条件を満たさない場合はエラーを発生させます。

保守担当者向けメモ

  • 本関数は copy_replication_slot への単なるラッパーです。ロジックの変更が必要な場合は、主に copy_replication_slot を参照してください。
