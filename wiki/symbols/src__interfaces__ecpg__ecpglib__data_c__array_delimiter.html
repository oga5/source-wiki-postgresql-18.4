array_delimiter

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/ecpglib/data.c:19

目的

ECPGにおける配列またはベクトルデータのパース処理において、現在の文字が要素の区切り文字であるかを判定します。

シグネチャまたは形状

static bool array_delimiter(enum ARRAY_TYPE isarray, char c);

入力と出力

  • **入力**:
  • isarray: 配列の型（ECPG_ARRAY_ARRAY または ECPG_ARRAY_VECTOR）。
  • c: 判定対象の文字。
  • **出力**:
  • bool: 指定された型において c が区切り文字であれば true、そうでなければ false を返します。

前提条件と不変条件

  • isarray が ECPG_ARRAY_ARRAY の場合、区切り文字はカンマ (,) です。
  • isarray が ECPG_ARRAY_VECTOR の場合、区切り文字はスペース (' ') です。

アルゴリズムまたは振る舞い

1. isarray が ECPG_ARRAY_ARRAY かつ c が ',' であれば true を返します。 2. isarray が ECPG_ARRAY_VECTOR かつ c が ' ' であれば true を返します。 3. 上記以外の場合は false を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/interfaces/ecpg/ecpglib/data.c 内のデータパース処理（ecpg_get_data 等）から、配列要素の境界を特定するために呼び出されます。
  • **呼び出し先**:
  • 本関数は他の関数を呼び出しません。

関連データ構造

  • enum ARRAY_TYPE: 配列の形式を定義する列挙型。

失敗モードと境界条件

  • isarray に定義外の値が渡された場合、常に false を返します。
  • c が '\0'（終端文字）の場合、区切り文字とは見なされず false を返します。

保守担当者向けメモ

  • 本関数は data.c 内の配列パースロジックにおいて、array_boundary と対になって使用されます。新しい配列形式や区切り文字のルールを追加する場合は、この関数と array_boundary の両方を更新する必要があります。
