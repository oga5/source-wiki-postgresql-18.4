rot13_passphrase

定義

  • 種別: function
  • 定義箇所: src/test/modules/ssl_passphrase_callback/ssl_passphrase_func.c:66

目的

OpenSSLのパスフレーズ要求に対し、静的変数 ssl_passphrase に格納された文字列をROT13変換して提供するコールバック関数です。

シグネチャまたは形状

static int rot13_passphrase(char *buf, int size, int rwflag, void *userdata)

入力と出力

  • **入力**:
  • buf: パスフレーズを書き込むためのバッファ。
  • size: buf のサイズ。
  • rwflag: 読み取り/書き込みフラグ（OpenSSL仕様）。
  • userdata: ユーザー定義データ（OpenSSL仕様）。
  • **出力**:
  • 戻り値: buf に書き込まれたパスフレーズの長さ（int）。

前提条件と不変条件

  • ssl_passphrase が NULL でないことが Assert により保証されています。

アルゴリズムまたは振る舞い

1. ssl_passphrase の内容を buf にコピーします。 2. buf 内の各文字に対し、以下のROT13変換を適用します。

  • 'a'-'m' または 'A'-'M' の場合、13文字進めます。
  • 'n'-'z' または 'N'-'Z' の場合、13文字戻します。

    • 3. 変換後の文字列長を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/test/modules/ssl_passphrase_callback/ssl_passphrase_func.c 内の SSL_CTX_set_default_passwd_cb を介してOpenSSLライブラリから呼び出されます。
  • **呼び出し先**: strlcpy および strlen（標準Cライブラリ関数）。
  • 注: src/test/modules/ldap_password_func/ldap_password_func.c にも同名のシンボルが存在しますが、本関数とはシグネチャおよび目的が異なる別個の実装です。

関連データ構造

  • ssl_passphrase: モジュール内で保持される静的文字列変数。

失敗モードと境界条件

  • size を超える文字列は strlcpy により切り詰められます。
  • アルファベット以外の文字（数字や記号）は変換されず、そのまま保持されます。

保守担当者向けメモ

  • 本関数はテスト目的の難読化であり、セキュリティ上の保護機能はありません。
  • OpenSSLのコールバック仕様に従っているため、シグネチャの変更には注意が必要です。
