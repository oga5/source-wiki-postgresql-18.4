LDAP_TIMEVAL

定義

  • 種別: struct (typedef)
  • 定義箇所: src/interfaces/libpq/fe-connect.c:72

目的

LDAP 関連の操作におけるタイムアウト時間を保持するためのデータ構造です。

シグネチャまたは形状

typedef struct timeval LDAP_TIMEVAL;

標準の struct timeval と同一の形状を持ちます。

入力と出力

  • 入力: 初期化時に秒数およびマイクロ秒数を指定してインスタンス化されます。
  • 出力: LDAP 接続や通信処理のタイムアウト制限として利用されます。

前提条件と不変条件

  • sys/time.h 等で定義される標準的な struct timeval と互換性があります。
  • タイムアウト値として正の整数が設定されることが想定されます。

アルゴリズムまたは振る舞い

LDAP 接続試行時の待機時間を制御するために使用されます。具体的には、PGLDAP_TIMEOUT 定数を用いて秒数を設定し、マイクロ秒を 0 に初期化する形で利用されます。

呼び出し元と呼び出し先

  • 呼び出し元: src/interfaces/libpq/fe-connect.c:5492 にて、LDAP タイムアウト設定用のローカル変数としてインスタンス化されています。
  • 呼び出し先: 提供されたソース情報からは、この構造体を利用する具体的な関数呼び出し先は特定できません。

関連データ構造

  • struct timeval: 本構造体の基底となる標準ライブラリの構造体。

失敗モードと境界条件

  • タイムアウト値に負の値を設定した場合の挙動は、基盤となるシステムコール（LDAP ライブラリの関数）の仕様に依存します。

保守担当者向けメモ

  • 本シンボルは fe-connect.c 内での LDAP 関連処理の局所的な型定義です。
  • LDAP 関連の機能拡張を行う際は、この構造体が使用されるコンテキスト（タイムアウトの適用範囲）を fe-connect.c 内で追跡してください。
