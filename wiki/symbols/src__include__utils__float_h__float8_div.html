float8_div

定義

  • 種別: static inline function
  • 定義箇所: src/include/utils/float.h:237

目的

PostgreSQLの要件に基づき、エラーチェック（ゼロ除算、オーバーフロー、アンダーフロー）を伴う float8 型の除算を実行する。

シグネチャまたは形状

static inline float8 float8_div(const float8 val1, const float8 val2)

入力と出力

  • **入力**:
  • val1: 被除数 (float8)
  • val2: 除数 (float8)
  • **出力**: 演算結果 (float8)

前提条件と不変条件

  • 演算結果が標準的なC言語の浮動小数点演算の範囲を超える場合、PostgreSQLの内部エラー報告関数を呼び出し、処理を中断する。
  • NaNの扱いはPostgreSQLの浮動小数点演算ルールに従う。

アルゴリズムまたは振る舞い

1. **ゼロ除算チェック**: val2 が 0.0 であり、かつ val1 が NaN でない場合、float_zero_divide_error() を呼び出す。 2. **除算実行**: val1 / val2 を計算する。 3. **オーバーフローチェック**: 結果が無限大 (isinf) であり、かつ val1 が無限大でない場合、float_overflow_error() を呼び出す。 4. **アンダーフローチェック**: 結果が 0.0 であり、かつ val1 が 0.0 ではなく、かつ val2 が無限大でない場合、float_underflow_error() を呼び出す。 5. 結果を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/access/gist/gistproc.c
  • src/backend/utils/adt/cash.c
  • src/backend/utils/adt/float.c
  • src/backend/utils/adt/geo_ops.c
  • **呼び出し先**:
  • float_zero_divide_error()
  • float_overflow_error()
  • float_underflow_error()
  • 標準Cライブラリ関数: isnan(), isinf()

関連データ構造

  • float8 (PostgreSQLの8バイト浮動小数点型)

失敗モードと境界条件

  • val2 が 0.0 の場合: float_zero_divide_error を発生させる。
  • 演算結果がオーバーフローした場合: float_overflow_error を発生させる。
  • 演算結果がアンダーフローした場合: float_underflow_error を発生させる。

保守担当者向けメモ

  • 本関数はインライン関数として定義されており、パフォーマンスを最適化している。
  • エラー報告関数（float_overflow_error 等）は pg_noinline 属性を持つため、エラー発生時のみ呼び出される設計となっている。
