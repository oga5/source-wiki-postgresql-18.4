r_attached_pronoun

目的

単語の末尾に付着したイタリア語の代名詞を検出し、除去または置換を行うことで、語幹抽出の前処理を行う。

シグネチャまたは形状

static int r_attached_pronoun(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング環境構造体）
  • 出力: 処理が成功した場合は 1、失敗した場合は 0、エラー発生時は負の値を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • 内部で r_RV を呼び出し、対象領域が RV（語幹の残余部分）内にあることを検証する。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置（z->c）を ket に設定する。 2. 文字コードのビット演算による高速な事前チェックを行い、条件を満たさない場合は終了する。 3. find_among_b を使用して、配列 a_2（37要素）から接尾辞を後方検索する。 4. bra を設定し、特定の文字（'o' または 'r'）で終わるかを確認する。 5. find_among_b を使用して、配列 a_3（5要素）から代名詞を検索する。 6. r_RV を呼び出し、対象が RV 領域内であることを確認する。 7. 検索結果（among_var）に応じて以下のいずれかを実行する:

  • 1: slice_del で接尾辞を削除する。
  • 2: slice_from_s で接尾辞を別の文字列（s_10）に置換する。

呼び出し元と呼び出し先

  • 呼び出し元: italian_ISO_8859_1_stem（イタリア語ステマーのメインフロー内）
  • 呼び出し先: find_among_b, r_RV, slice_del, slice_from_s

関連データ構造

  • SN_env: ステミング処理の環境を保持する構造体。
  • a_2, a_3: 接尾辞検索に使用される among 構造体の配列。

失敗モードと境界条件

  • 検索対象の接尾辞が見つからない場合、または r_RV のチェックに失敗した場合は 0 を返す。
  • メモリ操作（slice_del 等）でエラーが発生した場合は負の値を返す。
  • 文字列の境界（z->lb）に達した場合、処理を中断する。
