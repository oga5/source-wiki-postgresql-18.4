PLy_result_length

定義

  • 種別: function
  • 定義箇所: src/pl/plpython/plpy_resultobject.c:235

目的

Pythonの len() 関数やシーケンス操作を通じて、PLyResult オブジェクトが保持する行数（クエリ結果のレコード数）を取得する。

シグネチャまたは形状

static Py_ssize_t PLy_result_length(PyObject *arg)

入力と出力

  • **入力**: arg (Pythonオブジェクトとして渡される PLyResultObject へのポインタ)
  • **出力**: Py_ssize_t 型の整数値（ob->rows リストの要素数）

前提条件と不変条件

  • arg は PLyResultObject 型へキャスト可能であること。
  • PLyResultObject 構造体内の rows メンバは、有効なPythonリストオブジェクトを指していること。

アルゴリズムまたは振る舞い

1. 引数 arg を PLyResultObject * にキャストする。 2. 内部メンバ ob->rows（クエリ結果の行を保持するPythonリスト）に対して PyList_Size() を呼び出す。 3. 取得したリストのサイズを戻り値として返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/pl/plpython/plpy_resultobject.c 内の Py_sq_length スロットおよび Py_mp_length スロットから参照されている。
  • **呼び出し先**:
  • PyList_Size (Python C API)

関連データ構造

  • PLyResultObject: クエリ結果を保持するC構造体。rows メンバを通じて行データにアクセスする。

失敗モードと境界条件

  • arg が PLyResultObject ではない場合、キャストにより不正なメモリ参照が発生する可能性がある。
  • ob->rows が NULL または無効なオブジェクトである場合、PyList_Size の呼び出しによりPython例外が発生する可能性がある。

保守担当者向けメモ

  • 本関数はPythonのシーケンスおよびマッピングプロトコルの長さ取得メソッドとして登録されている。
  • PyList_Size の戻り値は Py_ssize_t であるため、大規模な結果セットに対してもPythonの標準的なサイズ制限に従う。
