cmpQTN

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/tsquery_util.c:152

目的

QTNode ポインタの配列をソートするために、qsort が要求する比較関数インターフェースを提供します。

シグネチャまたは形状

static int cmpQTN(const void *a, const void *b)

入力と出力

  • **入力**:
  • a, b: 比較対象である QTNode * 型のポインタを指す void * 型のポインタ。
  • **出力**:
  • QTNodeCompare の戻り値（int 型）。a が b より小さい場合は負、等しい場合は 0、大きい場合は正の値を返します。

前提条件と不変条件

  • a および b は、有効な QTNode * を指すポインタの配列要素である必要があります。
  • 内部で呼び出される QTNodeCompare は、比較対象のノードが正規化（ソート）済みであることを前提とする場合があります。

アルゴリズムまたは振る舞い

1. a および b を QTNode *const * 型にキャストします。 2. 間接参照により取得した QTNode * を QTNodeCompare に渡し、その結果をそのまま返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/adt/tsquery_util.c 内の qsort 呼び出し（QTNSort 関数内など）。
  • **呼び出し先**: QTNodeCompare。

関連データ構造

  • QTNode: tsquery を表現するメモリ上のツリー構造体。

失敗モードと境界条件

  • a または b が NULL ポインタである場合、QTNodeCompare の実装に依存した未定義の動作が発生します。

保守担当者向けメモ

  • 本関数は qsort のコールバックとしてのみ使用されます。
  • 比較ロジックを変更する場合は、QTNodeCompare を修正してください。
