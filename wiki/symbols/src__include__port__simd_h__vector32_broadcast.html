vector32_broadcast

定義

  • 種別: function
  • 定義箇所: src/include/port/simd.h:147

目的

単一の 32 ビット整数値を、すべての要素がその値である Vector32 型のベクトルへ複製（ブロードキャスト）します。

シグネチャまたは形状

static inline Vector32 vector32_broadcast(const uint32 c)

入力と出力

  • **入力**: c (uint32) - ベクトル全体に展開する 32 ビット整数値。
  • **出力**: Vector32 - 入力値 c で満たされたベクトル。

前提条件と不変条件

  • 本関数は USE_SSE2 または USE_NEON が定義されている環境でのみ有効です。
  • SIMD が利用できない環境では実装されていません。

アルゴリズムまたは振る舞い

コンパイル時の機能フラグに応じて以下のハードウェア命令を呼び出します。

  • **SSE2 (USE_SSE2)**: _mm_set1_epi32(c) を使用して 128 ビットレジスタに 32 ビット値を 4 つ配置します。
  • **NEON (USE_NEON)**: vdupq_n_u32(c) を使用して 128 ビットレジスタに 32 ビット値を 4 つ配置します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/include/port/pg_lfind.h:162 にて、検索キーをベクトル化するために呼び出されています。
  • **呼び出し先**: ターゲットアーキテクチャに応じて emmintrin.h (SSE2) または arm_neon.h (NEON) の組み込み関数を呼び出します。

関連データ構造

  • Vector32: プラットフォームに応じて __m128i または uint32x4_t 等にマッピングされるベクトル型。

失敗モードと境界条件

  • SIMD 非対応環境で本関数を呼び出した場合、コンパイルエラーとなります。

保守担当者向けメモ

  • Vector32 関連の関数は、SIMD が利用できない環境では 64 ビット整数でのエミュレーションが効率的ではないため、意図的に実装されていません。
  • 移植性を維持するため、直接ハードウェア命令を記述せず、必ず本関数を経由して利用してください。
