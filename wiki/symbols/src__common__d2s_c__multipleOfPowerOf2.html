multipleOfPowerOf2

定義

  • 種別: function
  • 定義箇所: src/common/d2s.c:105

目的

指定された整数 value が 2 の p 乗で割り切れるか（すなわち、下位 p ビットがすべて 0 であるか）を判定します。

シグネチャまたは形状

static inline bool multipleOfPowerOf2(const uint64 value, const uint32 p)

入力と出力

  • **入力**:
  • value: 判定対象の 64 ビット符号なし整数。
  • p: 判定基準となる 2 のべき乗の指数。
  • **出力**:
  • true: value が 2^p で割り切れる場合。
  • false: それ以外の場合。

前提条件と不変条件

  • p は value のビット幅（64）を超えない範囲で指定されることが想定されます。

アルゴリズムまたは振る舞い

1. (1 << p) - 1 を計算し、下位 p ビットが 1 であるマスクを作成します。 2. value と当該マスクのビットごとの論理積（AND）をとります。 3. 結果が 0 であれば、下位 p ビットがすべて 0 であると判断し true を返します。 4. ソースコード内のコメントによれば、__builtin_ctzll(value) >= p と同等の論理を実装しています。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/common/d2s.c:488
  • src/common/f2s.c:356
  • **呼び出し先**:
  • 特になし（組み込み演算のみ）。

関連データ構造

  • なし。

失敗モードと境界条件

  • p が 0 の場合、マスクは 0 となり、常に true を返します。
  • value が 0 の場合、すべてのビットが 0 であるため、任意の p に対して true を返します。

保守担当者向けメモ

  • 本関数は d2s.c（倍精度）と f2s.c（単精度）の両方から利用されています。
  • 移植性を考慮し、コンパイラ組み込み関数（__builtin_ctzll）ではなくビットマスク演算が採用されています。変更時はこのビット演算の正確性を維持してください。
