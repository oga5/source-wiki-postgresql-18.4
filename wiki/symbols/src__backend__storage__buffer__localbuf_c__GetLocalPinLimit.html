GetLocalPinLimit

定義

  • 種別: function
  • 定義箇所: src/backend/storage/buffer/localbuf.c:305

目的

現在のバックエンドプロセスが同時にピン留め可能なローカルバッファの最大数を取得します。

シグネチャまたは形状

uint32 GetLocalPinLimit(void)

入力と出力

  • 入力: なし
  • 出力: num_temp_buffers（設定されている一時バッファの総数）を uint32 型で返します。

前提条件と不変条件

  • 各バックエンドは自身の一時バッファを管理しており、そのすべてを同時にピン留めすることが可能です。

アルゴリズムまたは振る舞い

  • グローバル変数 num_temp_buffers の値をそのまま返します。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/backend/storage/aio/read_stream.c:618
  • 呼び出し先:
  • なし（直接的な関数呼び出しは含まれていません）

関連データ構造

  • num_temp_buffers: 一時バッファの最大数を保持するグローバル変数。

失敗モードと境界条件

  • 特筆すべき失敗モードはありません。常に設定されたバッファ数を返します。

保守担当者向けメモ

  • 本関数は一時テーブルのバッファ管理における上限値を定義する単純なアクセサです。
  • src/include/storage/bufmgr.h にて外部公開されています。
