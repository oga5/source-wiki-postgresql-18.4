InitArchiveFmt_Null

定義

  • 種別: function
  • 定義箇所: src/bin/pg_dump/pg_backup_null.c:47

目的

pg_dump のアーカイブハンドル（ArchiveHandle）を「Null アーカイブ」形式用に初期化し、プレーンテキスト形式の SQL 出力を行うための関数ポインタを設定します。

シグネチャまたは形状

void InitArchiveFmt_Null(ArchiveHandle *AH)

入力と出力

  • 入力: ArchiveHandle *AH（初期化対象のアーカイブハンドル構造体）
  • 出力: なし（AH の各関数ポインタフィールドを直接書き換える）

前提条件と不変条件

  • AH は有効な ArchiveHandle 構造体へのポインタであること。
  • 本関数は読み込みモード（archModeRead）をサポートしておらず、その場合はエラーとして終了します。

アルゴリズムまたは振る舞い

1. ArchiveHandle の各ポインタフィールド（WriteDataPtr, EndDataPtr, WriteBytePtr, WriteBufPtr, ClosePtr, PrintTocDataPtr 等）に、pg_backup_null.c 内の静的関数を割り当てます。 2. ラージオブジェクト関連のポインタ（StartLOsPtr, StartLOPtr, EndLOPtr, EndLOsPtr）を初期化します。 3. ClonePtr および DeClonePtr を NULL に設定します。 4. AH->mode が archModeRead である場合、pg_fatal を呼び出して処理を停止します。

呼び出し元と呼び出し先

  • 呼び出し元: src/bin/pg_dump/pg_backup_archiver.c:2474
  • 呼び出し先:
  • _WriteData, _EndData, _WriteByte, _WriteBuf, _CloseArchive, _PrintTocData
  • _StartLOs, _StartLO, _EndLO, _EndLOs
  • pg_fatal

関連データ構造

  • ArchiveHandle (pg_backup_archiver.h で定義)

失敗モードと境界条件

  • archModeRead モードで呼び出された場合、pg_fatal によりプログラムが異常終了します。

保守担当者向けメモ

  • 本関数は書き込み専用の初期化エントリポイントです。読み込み機能の追加はサポートされていません。
  • _WriteByte や _WriteBuf は空実装であり、メタデータの書き込みが不要なプレーンテキスト出力の特性を反映しています。
