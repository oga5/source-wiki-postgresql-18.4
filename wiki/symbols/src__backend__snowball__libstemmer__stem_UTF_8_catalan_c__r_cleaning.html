r_cleaning

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_catalan.c:1219

目的

入力文字列に対してアクセント記号の除去や正規化を行い、ステミング処理に適した形式へ変換する。

シグネチャまたは形状

static int r_cleaning(struct SN_env * z)

入力と出力

  • 入力: SN_env 構造体（ステミング環境）。現在のカーソル位置 z->c から文字列の末尾までを処理対象とする。
  • 出力: 成功時に 1 を返す。内部的な文字列操作でエラーが発生した場合は負の値を返す。

前提条件と不変条件

  • z が有効な SN_env 構造体を指していること。
  • z->p にUTF-8エンコーディングされた文字列が格納されていること。

アルゴリズムまたは振る舞い

1. 文字列全体をループで走査する。 2. find_among を使用して、定義済みの置換ルール（a_0 配列）に一致する部分文字列を検索する。 3. 一致したルールに応じて slice_from_s を呼び出し、該当箇所を正規化後の文字列に置換する。 4. ルールに一致しない文字（アクセント記号等）は skip_utf8 を用いてスキップし、次の文字へ進む。 5. 文字列の終端に達するまでこの処理を繰り返す。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_catalan.c および src/backend/snowball/libstemmer/stem_ISO_8859_1_catalan.c 内のステミング処理フローから呼び出される。
  • 呼び出し先: find_among, slice_from_s, skip_utf8（いずれもSnowballランタイム関数）。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_0: 置換ルールを定義した among 構造体の配列。

失敗モードと境界条件

  • slice_from_s が失敗した場合、その戻り値（負の値）をそのまま呼び出し元へ伝播させる。
  • 文字列の末尾に達した際、skip_utf8 が失敗した場合は lab0 ラベルへジャンプし、ループを終了する。

保守担当者向けメモ

  • 本関数はSnowballコンパイラにより自動生成されたコードである。ロジックの変更が必要な場合は、直接編集せず元のSnowballスクリプトを修正すること。
  • a_0 配列の定義内容が、カタルーニャ語の正規化ルールを規定している。
