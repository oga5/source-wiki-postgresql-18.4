jsonb_lt

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/jsonb_op.c:165

目的

2つの jsonb 値を比較し、左側の値が右側の値よりも小さいかどうかを判定する（< 演算子の実装）。

シグネチャまたは形状

Datum jsonb_lt(PG_FUNCTION_ARGS)

入力と出力

  • 入力: PG_FUNCTION_ARGS を介して2つの jsonb 型ポインタ（jba, jbb）を受け取る。
  • 出力: bool 型の Datum。jba < jbb であれば true、そうでなければ false を返す。

前提条件と不変条件

  • 入力された jsonb ポインタは PG_GETARG_JSONB_P マクロにより適切に取得される必要がある。
  • 比較処理の終了後、PG_FREE_IF_COPY を用いてメモリの解放を行う必要がある。

アルゴリズムまたは振る舞い

1. PG_GETARG_JSONB_P を使用して、引数から2つの Jsonb 構造体を取得する。 2. compareJsonbContainers 関数を呼び出し、両者の root 構造を比較する。 3. 比較結果が 0 未満（jba < jbb）である場合に true を、それ以外の場合に false を res に格納する。 4. PG_FREE_IF_COPY で入力引数のメモリを解放する。 5. PG_RETURN_BOOL を介して判定結果を返す。

呼び出し元と呼び出し先

  • 呼び出し元: PostgreSQLの関数マネージャ（fmgr）経由でSQLから呼び出される。
  • 呼び出し先: compareJsonbContainers（jsonb コンテナの比較を行う補助関数）。

関連データ構造

  • Jsonb: jsonb 型の内部表現。
  • JsonbContainer: Jsonb 構造体内の root フィールドが指すデータ構造。

失敗モードと境界条件

  • 入力が NULL の場合、PG_GETARG_JSONB_P の仕様に従い、呼び出し元で適切に処理される必要がある。
  • 比較対象の jsonb 構造が不正な場合、compareJsonbContainers 内でエラーが発生する可能性がある。

保守担当者向けメモ

  • 本関数はB-Treeインデックスのソート順を決定する比較ロジックの一部として機能する。
  • 比較ロジックの変更が必要な場合は、compareJsonbContainers の実装を確認すること。
