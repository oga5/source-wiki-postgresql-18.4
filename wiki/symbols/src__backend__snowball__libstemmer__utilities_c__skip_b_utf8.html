skip_b_utf8

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/utilities.c:52

目的

UTF-8 エンコードされた文字列バッファにおいて、現在のカーソル位置から後方（逆方向）へ指定された文字数分だけカーソルを移動させる。

シグネチャまたは形状

int skip_b_utf8(const symbol * p, int c, int limit, int n)

入力と出力

  • **入力**:
  • p: 文字列バッファへのポインタ。
  • c: 現在のカーソル位置（インデックス）。
  • limit: 後方移動の境界値（これより小さくはならない）。
  • n: スキップする文字数。
  • **出力**:
  • 成功時: 移動後の新しいカーソル位置（int）。
  • 失敗時: -1（n < 0 の場合、または移動中に limit に到達した場合）。

前提条件と不変条件

  • n は 0 以上であること。
  • c はバッファ内の有効な位置を指していること。
  • UTF-8 のマルチバイトシーケンスが正しく構成されていることを前提とする。

アルゴリズムまたは振る舞い

1. n が負であれば -1 を返す。 2. n 回のループを実行し、各回で以下の処理を行う：

  • 現在の c が limit 以下であれば -1 を返す。
  • c をデクリメントし、その位置のバイト b を取得する。
  • b が 0x80 以上（マルチバイト文字の継続バイトまたは先頭バイト）の場合：
  • c が limit より大きい間、b が 0xC0 以上（UTF-8 の先頭バイト）になるまで c をデクリメントし続ける。

    • 3. 最終的な c の値を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • stem_UTF_8_arabic.c, stem_UTF_8_danish.c, stem_UTF_8_dutch.c, stem_UTF_8_english.c, stem_UTF_8_estonian.c, stem_UTF_8_finnish.c 等の各言語ステミング実装ファイルから呼び出されている。
  • **呼び出し先**:
  • 本関数は直接的な関数呼び出しを行わない。

関連データ構造

  • symbol: 文字列バッファの要素型。
  • SN_env (間接的): 呼び出し元で z->p (バッファ), z->c (カーソル), z->lb (境界) として参照される。

失敗モードと境界条件

  • n < 0: 即座に -1 を返す。
  • c <= limit: 移動中に境界に達した場合、処理を中断し -1 を返す。
  • 不正な UTF-8 シーケンス: 本関数はバイト単位のスキップを行うため、UTF-8 の構造が壊れている場合の挙動は未定義。

保守担当者向けメモ

  • 本関数は後方検索用である。前方検索には skip_utf8 を使用すること。
  • 0x80 および 0xC0 の判定は、UTF-8 のマルチバイト文字構造（先頭バイトは 11xxxxxx、継続バイトは 10xxxxxx）に基づいている。
