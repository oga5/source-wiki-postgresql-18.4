PLyUnicode_FromJsonbValue

定義

  • 種別: function
  • 定義箇所: contrib/jsonb_plpython/jsonb_plpython.c:68

目的

JsonbValue 構造体から文字列データを取り出し、対応する Python の Unicode 文字列オブジェクト（PyObject）へ変換する。

シグネチャまたは形状

static PyObject *PLyUnicode_FromJsonbValue(JsonbValue *jbv)

入力と出力

  • **入力**: JsonbValue *jbv - 変換対象の JSONB 値へのポインタ。
  • **出力**: PyObject * - 変換された Python の文字列オブジェクト。

前提条件と不変条件

  • 入力される jbv の type フィールドが jbvString である必要がある（Assert により検証される）。

アルゴリズムまたは振る舞い

1. jbv->type が jbvString であることをアサーションで確認する。 2. jbv->val.string.val（文字列のポインタ）と jbv->val.string.len（文字列の長さ）を引数として、PLyUnicode_FromStringAndSize を呼び出す。 3. 生成された Python 文字列オブジェクトを呼び出し元へ返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/jsonb_plpython/jsonb_plpython.c:117
  • contrib/jsonb_plpython/jsonb_plpython.c:209
  • **呼び出し先**:
  • PLyUnicode_FromStringAndSize（plpython 拡張が提供する関数）

関連データ構造

  • JsonbValue: PostgreSQL の JSONB 内部表現。
  • PyObject: Python のオブジェクト型。

失敗モードと境界条件

  • jbv->type が jbvString 以外の場合、アサーション違反によりプログラムが停止する可能性がある。
  • PLyUnicode_FromStringAndSize がメモリ不足等で失敗した場合の挙動は、当該関数の実装に依存する。

保守担当者向けメモ

  • 本関数は plpython 拡張が提供する PLyUnicode_FromStringAndSize に依存している。_PG_init で解決される関数ポインタ経由の呼び出しである点に注意すること。
