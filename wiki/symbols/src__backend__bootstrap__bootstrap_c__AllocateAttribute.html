AllocateAttribute

定義

  • 種別: function
  • 定義箇所: src/backend/bootstrap/bootstrap.c:913

目的

システムカタログの属性定義（pg_attribute）を保持するためのメモリ領域を確保し、ゼロ初期化された構造体として提供する。

シグネチャまたは形状

static Form_pg_attribute AllocateAttribute(void)

入力と出力

  • 入力: なし
  • 出力: Form_pg_attribute 型のポインタ（ATTRIBUTE_FIXED_PART_SIZE 分のメモリが確保された領域）。

前提条件と不変条件

  • TopMemoryContext が初期化されていること。
  • 戻り値は MemoryContextAllocZero によって確保されるため、呼び出し側で解放の要否を管理する必要がある。

アルゴリズムまたは振る舞い

1. TopMemoryContext をメモリコンテキストとして指定する。 2. ATTRIBUTE_FIXED_PART_SIZE のサイズ分だけメモリを確保する。 3. 確保したメモリ領域をゼロクリアして返す。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/backend/bootstrap/bootstrap.c:465
  • src/backend/bootstrap/bootstrap.c:533
  • 呼び出し先:
  • MemoryContextAllocZero (PostgreSQLメモリ管理API)

関連データ構造

  • Form_pg_attribute: システムカタログ pg_attribute の行構造体。

失敗モードと境界条件

  • メモリ不足時には MemoryContextAllocZero の仕様に従い、エラーが発生する（通常はプロセス終了）。

保守担当者向けメモ

  • 本関数はブートストラップ処理中に属性定義を動的に生成する際に使用される。
  • ATTRIBUTE_FIXED_PART_SIZE の定義変更が本関数の確保サイズに直接影響するため、構造体の定義変更時には注意が必要。
