calc_length_hist_frac

定義

  • **種別**: function
  • **定義箇所**: src/backend/utils/adt/rangetypes_selfuncs.c:854

目的

範囲型の長さヒストグラムに基づき、指定された範囲 [length1, length2] 内に収まる確率（累積分布関数の平均値）を算出します。

シグネチャまたは形状

static double calc_length_hist_frac(Datum *length_hist_values, int length_hist_nvalues,
                                    double length1, double length2, bool equal)

入力と出力

  • **入力**:
  • length_hist_values: 範囲の長さのヒストグラム境界値の配列。
  • length_hist_nvalues: ヒストグラムの要素数。
  • length1, length2: ターゲットとなる長さの範囲。
  • equal: 上限値の包含判定フラグ。
  • **出力**:
  • double: 計算された選択率（確率）。

前提条件と不変条件

  • length2 >= length1 であること。
  • ヒストグラムのビンは台形として近似され、各ビン間の累積確率は線形補間されます。
  • length1 == length2 の場合、除算によるゼロ除算を避けるため、その点における累積確率を直接返します。

アルゴリズムまたは振る舞い

1. **初期化**: length_hist_bsearch を用いて length1 が含まれるヒストグラムのビンを特定します。 2. **補間**: get_len_position を使用し、ビン内での length1 および length2 の相対位置を算出します。 3. **積分**: ヒストグラムのビンを台形と見なし、各ビンの面積（0.5 * (PB + PA) * (B - A)）を累積します。 4. **正規化**: 累積した面積を範囲の幅（length2 - length1）で除算し、平均確率を求めます。 5. **例外処理**: length2 が無限大かつ equal が真の場合は 1.0 を返します。また、無限大同士の除算が発生する場合は 0.5 を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/rangetypes_selfuncs.c 内の選択率計算関数群。
  • src/backend/utils/adt/multirangetypes_selfuncs.c 内のマルチレンジ型選択率計算関数群。
  • **呼び出し先**:
  • length_hist_bsearch: ヒストグラムの探索。
  • get_len_position: ビン内での位置計算。
  • DatumGetFloat8: Datum から double への変換。

関連データ構造

  • Datum: PostgreSQL の汎用データ表現。
  • pg_statistic: 範囲型の長さヒストグラムが格納されるシステムカタログ。

失敗モードと境界条件

  • length2 < 0.0: 0.0 を返します。
  • length1 == length2: ゼロ除算を回避し、その点での累積確率を返します。
  • area および length2 が共に無限大の場合: 0.5 を返します。
  • ヒストグラムのビンが空または不正な場合: get_len_position の挙動に依存します。

保守担当者向けメモ

  • 本関数は「範囲の長さ」と「下限値」が独立しているという仮定に基づいています。
  • PA と PB が共に 0 の場合、NaN を避けるためのガード（if (PA > 0 || PB > 0)）が実装されています。
  • 幾何学的な面積計算ロジックは、累積分布関数の積分を近似するものです。修正時は台形公式の整合性に注意してください。
