gist_translate_cmptype_btree

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_gist.c:64

目的

GiST 索引の内部比較型（CompareType）を、PostgreSQL の B-tree 索引における戦略番号（Strategy Number）へ変換するマッピング関数です。

シグネチャまたは形状

Datum gist_translate_cmptype_btree(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_GETARG_INT32(0) を介して CompareType 列挙値を受け取ります。
  • **出力**: 対応する B-tree 戦略番号（uint16）を PG_RETURN_UINT16 で返します。

前提条件と不変条件

  • 入力値は CompareType 列挙型に準拠している必要があります。

アルゴリズムまたは振る舞い

入力された CompareType に基づき、以下の通り変換を行います。

  • COMPARE_EQ → BTEqualStrategyNumber
  • COMPARE_LT → BTLessStrategyNumber
  • COMPARE_LE → BTLessEqualStrategyNumber
  • COMPARE_GT → BTGreaterStrategyNumber
  • COMPARE_GE → BTGreaterEqualStrategyNumber
  • 上記以外 → InvalidStrategy

呼び出し元と呼び出し先

  • **呼び出し元**: ソースコード上での直接的な呼び出し元は、提供された情報からは特定できません。
  • **呼び出し先**: PostgreSQL のマクロ PG_GETARG_INT32 および PG_RETURN_UINT16 を使用します。

関連データ構造

  • CompareType (定義: access/cmptype.h)
  • B-tree 戦略番号定数 (定義: access/stratnum.h)

失敗モードと境界条件

  • 定義されていない CompareType が渡された場合、InvalidStrategy を返します。

保守担当者向けメモ

  • 本関数は GiST 索引が B-tree の順序付けロジックを模倣するための「翻訳機」として機能します。
  • PG_FUNCTION_INFO_V1(gist_translate_cmptype_btree) により、PostgreSQL の関数マネージャに登録されています。
