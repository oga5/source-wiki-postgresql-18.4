SlotSyncShmemSize

定義

  • 種別: function
  • 定義箇所: src/backend/replication/logical/slotsync.c:1791

目的

論理レプリケーションスロット同期機能で使用される共有メモリ領域のサイズを算出します。

シグネチャまたは形状

Size SlotSyncShmemSize(void)

入力と出力

  • 入力: なし
  • 出力: Size 型の共有メモリサイズ（バイト単位）

前提条件と不変条件

  • 共有メモリの割り当て前に呼び出される必要があります。
  • 返されるサイズは SlotSyncCtxStruct 構造体のサイズと一致します。

アルゴリズムまたは振る舞い

SlotSyncCtxStruct 構造体のサイズを sizeof 演算子で取得し、その値をそのまま返します。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/backend/replication/logical/slotsync.c:1803 (SlotSyncShmemInit 内)
  • src/backend/storage/ipc/ipci.c:151 (CreateSharedMemoryAndSemaphores 内)
  • 呼び出し先: なし

関連データ構造

  • SlotSyncCtxStruct: 本関数がサイズを算出する対象の共有メモリ構造体。

失敗モードと境界条件

  • 特になし。常に sizeof(SlotSyncCtxStruct) を返します。

保守担当者向けメモ

  • SlotSyncCtxStruct の定義が変更された場合、本関数は自動的に正しいサイズを返すため、修正は不要です。
  • 共有メモリの割り当てサイズ計算において、他のモジュールと整合性を保つために ipci.c から参照されています。
