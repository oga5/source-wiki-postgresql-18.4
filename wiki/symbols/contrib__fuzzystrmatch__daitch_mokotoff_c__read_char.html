read_char

定義

  • 種別: function
  • 定義箇所: contrib/fuzzystrmatch/daitch_mokotoff.c:395

目的

入力文字列から UTF-8 文字を読み取り、Daitch-Mokotoff アルゴリズムで利用可能な単一の ASCII 文字（または予約された特殊文字）に正規化・変換する。

シグネチャまたは形状

static char read_char(const unsigned char *str, int *ix)

入力と出力

  • **入力**:
  • str: 対象となる UTF-8 文字列のポインタ。
  • ix: 現在の読み取り位置を示すインデックス（関数内で更新される）。
  • **出力**:
  • 変換後の char 型文字。変換不能または予約範囲の場合は \x1a を返す。

前提条件と不変条件

  • ix は有効な文字列インデックスを指している必要がある。
  • utf8_to_unicode および pg_utf_mblen が利用可能であること。

アルゴリズムまたは振る舞い

1. *ix を基点に utf8_to_unicode を呼び出し、UTF-8 文字を Unicode コードポイントに変換する。 2. 文字が読み取れた場合、pg_utf_mblen を用いて *ix を次の文字位置へ進める。 3. コードポイントに応じた変換処理を行う:

  • [ から ] の範囲: 予約済みのため \x1a を返す。
  • 0x60 未満: そのまま char として返す。
  • 0x100 未満: iso8859_1_to_ascii_upper テーブルを用いて大文字 ASCII に変換する。
  • 0x100 以上: 特定の拡張ラテン文字（A/E/Tの異体字）を [、\、] のいずれかにマッピングする。
  • その他: \x1a を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/fuzzystrmatch/daitch_mokotoff.c 内のループ処理（while ((c = read_char(...)) != '\0')）から呼び出される。
  • **呼び出し先**: utf8_to_unicode, pg_utf_mblen。

関連データ構造

  • iso8859_1_to_ascii_upper: ISO-8859-1 から ASCII への変換テーブル。

失敗モードと境界条件

  • 文字列の終端に達した場合、c が 0 となり *ix は更新されない。
  • 変換対象外の文字や予約範囲の文字は \x1a に置換される。

保守担当者向けメモ

  • 本関数は daitch_mokotoff.h で定義される音韻ルールと密接に関連している。特定の文字マッピングを変更する場合は、アルゴリズム全体への影響を考慮すること。
