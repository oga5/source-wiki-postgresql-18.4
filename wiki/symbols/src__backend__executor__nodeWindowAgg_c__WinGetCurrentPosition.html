WinGetCurrentPosition

定義

  • 種別: function
  • 定義箇所: src/backend/executor/nodeWindowAgg.c:3298

目的

ウィンドウ関数実行時において、現在のパーティション内での処理対象行のインデックス（位置）を取得する。

シグネチャまたは形状

int64 WinGetCurrentPosition(WindowObject winobj)

入力と出力

  • **入力**: WindowObject winobj（ウィンドウ関数の実行コンテキスト）
  • **出力**: int64 型の現在の行位置（0から始まるインデックス）

前提条件と不変条件

  • winobj は WindowObjectIsValid マクロにより有効であることが保証されていなければならない。

アルゴリズムまたは振る舞い

1. winobj が指す winstate 構造体から、現在の行位置を示す currentpos フィールドを読み取る。 2. 読み取った値を呼び出し元に返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/windowfuncs.c（複数の箇所でランク計算等に使用）
  • **呼び出し先**:
  • WindowObjectIsValid（マクロによる検証）

関連データ構造

  • WindowObject: ウィンドウ関数APIで使用されるコンテキスト構造体。
  • WindowAggState: winobj->winstate を介して参照される、ウィンドウ集約ノードの実行時状態。

失敗モードと境界条件

  • winobj が無効な場合、Assert によりプログラムが停止する可能性がある。

保守担当者向けメモ

  • 本関数は src/include/windowapi.h で公開されており、ウィンドウ関数APIの一部として提供されている。
  • 戻り値はパーティション内での相対位置であるため、パーティションが切り替わるとリセットされる点に注意すること。
