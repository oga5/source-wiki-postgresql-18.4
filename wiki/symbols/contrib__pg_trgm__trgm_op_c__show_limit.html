show_limit

定義

  • 種別: function
  • 定義箇所: contrib/pg_trgm/trgm_op.c:278

目的

現在設定されている類似度閾値（similarity_threshold）を SQL インターフェース経由で取得する。

シグネチャまたは形状

Datum show_limit(PG_FUNCTION_ARGS)

入力と出力

  • 入力: PG_FUNCTION_ARGS（PostgreSQL の関数呼び出し規約に基づく引数）
  • 出力: float4 型の類似度閾値（PG_RETURN_FLOAT4 を使用して返却）

前提条件と不変条件

  • similarity_threshold がグローバル変数として初期化されていること。
  • PostgreSQL の関数呼び出し規約（PG_FUNCTION_ARGS）に従って呼び出されること。

アルゴリズムまたは振る舞い

1. グローバル変数 similarity_threshold を参照する。 2. その値を float4 型として呼び出し元に返す。

呼び出し元と呼び出し先

  • 呼び出し元: PostgreSQL の SQL エンジン（PG_FUNCTION_INFO_V1(show_limit) マクロにより登録されているため、SQL からの呼び出しが可能）。
  • 呼び出し先: なし（直接的な関数呼び出しは行わない）。

関連データ構造

  • similarity_threshold: pg_trgm モジュール内で保持される GUC（Grand Unified Configuration）変数。

失敗モードと境界条件

  • 特になし。常に現在の similarity_threshold の値を返す。

保守担当者向けメモ

  • 本関数は単なるゲッターであり、ロジックの変更は不要。
  • 類似度閾値を変更する場合は set_limit 関数を参照すること。
