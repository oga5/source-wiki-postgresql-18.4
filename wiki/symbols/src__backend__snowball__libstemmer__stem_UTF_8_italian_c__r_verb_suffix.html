r_verb_suffix

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_italian.c:918

目的

イタリア語のステミング処理において、単語末尾の動詞接尾辞を特定し、削除することで語幹を抽出する。

シグネチャまたは形状

static int r_verb_suffix(struct SN_env * z)

入力と出力

  • 入力: SN_env 構造体（ステミング対象の文字列や現在のカーソル位置 c、領域境界 I[2] を保持）。
  • 出力: 成功時に 1、失敗または処理対象外の場合に 0、内部エラー発生時に負の値を返す。

前提条件と不変条件

  • z->I[2] に格納された領域境界（通常は RV 領域の開始位置）よりカーソル位置 z->c が後ろにあること。
  • a_7 配列に定義された動詞接尾辞パターンが対象となる。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c が領域境界 z->I[2] より前であれば即座に 0 を返す。 2. 現在の境界設定を退避し、z->lb を z->I[2] に設定して検索範囲を限定する。 3. find_among_b を使用し、a_7 配列に基づき後方から接尾辞を検索する。 4. 接尾辞が見つかった場合、z->bra から z->ket までの範囲を slice_del で削除する。 5. 退避した境界設定を復元し、結果を返す。

呼び出し元と呼び出し先

  • 呼び出し元: italian_UTF_8_stem（直接の呼び出し関係はソースコードの制御フローより推論）。
  • 呼び出し先: find_among_b, slice_del。
  • 補足: 他の言語（Catalan, French, Portuguese, Spanish 等）のステマーにも同名の関数が存在するが、本ページは stem_UTF_8_italian.c 内の定義に限定している。

関連データ構造

  • struct SN_env: ステミング環境を管理する構造体。
  • a_7: 動詞接尾辞のパターンを保持する among 構造体の配列。

失敗モードと境界条件

  • z->c < z->I[2]: 処理対象領域に達していない場合は失敗する。
  • find_among_b が一致する接尾辞を見つけられない場合は 0 を返す。
  • slice_del がメモリ不足等の理由で負の値を返した場合は、そのエラーコードを伝播させる。

保守担当者向けメモ

  • a_7 配列の要素数（87）や構造に変更を加える際は、find_among_b の引数との整合性に注意が必要。
