psql_init

定義

  • 種別: function
  • 定義箇所: src/test/regress/pg_regress_main.c:103

目的

pg_regress のテスト実行環境における初期化処理として、デフォルトのデータベース名を登録する。

シグネチャまたは形状

static void psql_init(int argc, char **argv)

入力と出力

  • 入力:
  • argc: コマンドライン引数の数。
  • argv: コマンドライン引数の配列。
  • 出力: なし（void）。

前提条件と不変条件

  • dblist が初期化済みであること。
  • 本関数は regression_main からのコールバックとして呼び出されることを想定している。

アルゴリズムまたは振る舞い

1. add_stringlist_item を呼び出し、グローバル変数 dblist に文字列 "regression" を追加する。

呼び出し元と呼び出し先

  • 呼び出し元: src/test/regress/pg_regress_main.c 内の main 関数（regression_main への引数として渡される）。
  • 呼び出し先: add_stringlist_item（pg_regress.h で定義されたユーティリティ関数）。

関連データ構造

  • dblist: テスト対象のデータベース名を保持する文字列リスト。

失敗モードと境界条件

  • add_stringlist_item が内部でメモリ不足に陥った場合、プログラムは終了する可能性がある。

保守担当者向けメモ

  • 本関数は pg_regress フレームワークの初期化フェーズで実行される。
  • 現在はデフォルトのデータベース名設定のみを行っているが、将来的にテスト実行前の環境設定が必要になった場合はここにロジックを追加する。
