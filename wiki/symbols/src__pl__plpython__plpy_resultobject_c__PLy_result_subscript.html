PLy_result_subscript

定義

  • 種別: function
  • 定義箇所: src/pl/plpython/plpy_resultobject.c:267

目的

Pythonの PLyResult オブジェクトに対する添字アクセス（result[i]）を処理し、内部の行データリストから要素を取得します。

シグネチャまたは形状

static PyObject *PLy_result_subscript(PyObject *arg, PyObject *item)

入力と出力

  • **入力**:
  • arg: PLyResultObject 型のPythonオブジェクト（PLyResult インスタンス）。
  • item: 添字として指定されたPythonオブジェクト（整数やスライスなど）。
  • **出力**:
  • 成功時: item に対応する行データ（PyObject）。
  • 失敗時: NULL（Pythonの例外が設定されます）。

前提条件と不変条件

  • arg は PLyResultObject 構造体へキャスト可能である必要があります。
  • 内部の rows メンバは、Pythonのリストオブジェクトとして初期化済みである必要があります。

アルゴリズムまたは振る舞い

1. arg を PLyResultObject 型にキャストします。 2. 保持している rows（Pythonリスト）に対して、Python C APIの PyObject_GetItem を呼び出し、指定された item に対応する要素を取得して返します。

呼び出し元と呼び出し先

  • **呼び出し元**: Pythonインタープリタ（PLyResult 型の添字アクセス時）。
  • **呼び出し先**: PyObject_GetItem（Python C API）。

関連データ構造

  • PLyResultObject: クエリ結果を保持するC構造体。内部に rows（結果行のリスト）を保持します。

失敗モードと境界条件

  • item が範囲外の場合、PyObject_GetItem が IndexError を発生させ、本関数は NULL を返します。
  • arg が不正な型である場合、キャストにより未定義の動作を引き起こす可能性があります。

保守担当者向けメモ

  • 本関数は PLyResult_spec において Py_mp_subscript スロットに割り当てられています。
  • Python 3.8以降の参照カウント管理の変更については、ファイル全体のコンパイル条件を確認してください。
