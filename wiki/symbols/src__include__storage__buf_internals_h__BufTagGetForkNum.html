BufTagGetForkNum

定義

  • 種別: function
  • 定義箇所: src/include/storage/buf_internals.h:121

目的

BufferTag 構造体から、そのタグが指すリレーションのフォーク番号（ForkNumber）を取得します。

シグネチャまたは形状

static inline ForkNumber BufTagGetForkNum(const BufferTag *tag)

入力と出力

  • **入力**: tag (const BufferTag *) - 読み取り対象のバッファタグへのポインタ。
  • **出力**: ForkNumber - タグ内に格納されているフォーク番号。

前提条件と不変条件

  • tag は有効な BufferTag 構造体を指している必要があります。

アルゴリズムまたは振る舞い

tag 構造体のメンバである forkNum を直接返します。インライン関数として定義されており、オーバーヘッドなしで値にアクセスします。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/pg_buffercache/pg_buffercache_pages.c
  • contrib/pg_prewarm/autoprewarm.c
  • src/backend/storage/buffer/bufmgr.c (多数の箇所でバッファのフォーク番号比較やトレースに使用)
  • **呼び出し先**:
  • なし（構造体メンバへの直接アクセスのみ）

関連データ構造

  • BufferTag: バッファを識別するためのタグ構造体。
  • ForkNumber: リレーションのフォーク（メイン、FSM、VMなど）を識別する型。

失敗モードと境界条件

  • tag が NULL の場合、未定義の動作となります。

保守担当者向けメモ

  • 本関数は非常に軽量なアクセサであるため、パフォーマンスへの影響は最小限です。
  • BufferTag 構造体のレイアウト変更時には、本関数の実装も整合性を保つ必要があります。
