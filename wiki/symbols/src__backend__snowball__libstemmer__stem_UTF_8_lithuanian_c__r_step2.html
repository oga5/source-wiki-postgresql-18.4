r_step2

目的

リトアニア語のステミング処理において、定義された接尾辞テーブル（a_1）に基づき、単語末尾から接尾辞を繰り返し除去する。

シグネチャまたは形状

static int r_step2(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z （Snowball ステミング環境）
  • 出力: 成功時に 1、slice_del 失敗時に負の値を返す。

前提条件と不変条件

  • z->I[0] にステミング対象領域の境界（インデックス）が格納されていること。
  • a_1 は 62 個の要素を持つ among テーブルであること。

アルゴリズムまたは振る舞い

1. while(1) ループにより、マッチする接尾辞がなくなるまで以下の処理を繰り返す。 2. 現在のカーソル位置から z->I[0] までの範囲で、find_among_b を使用して a_1 テーブル内の接尾辞を後方検索する。 3. マッチした場合、z->bra から z->ket までの範囲を slice_del で削除する。 4. マッチしない場合はループを終了し、カーソル位置を復元して 1 を返す。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_lithuanian.c 内のメイン処理（806行目付近）。
  • 呼び出し先: find_among_b（接尾辞検索）、slice_del（文字列削除）。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_1: 接尾辞除去に使用される among テーブル。

失敗モードと境界条件

  • z->c < z->I[0] の場合、対象領域外のため処理をスキップする。
  • slice_del が負の値を返した場合、その値をそのまま呼び出し元へ伝播させる。
