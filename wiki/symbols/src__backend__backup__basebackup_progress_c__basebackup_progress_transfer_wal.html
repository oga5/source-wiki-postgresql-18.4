basebackup_progress_transfer_wal

定義

  • 種別: function
  • 定義箇所: src/backend/backup/basebackup_progress.c:228

目的

ベースバックアップの進捗状況を追跡するシステムビュー pg_stat_progress_basebackup において、現在のフェーズを「WAL転送中」に更新する。

シグネチャまたは形状

void basebackup_progress_transfer_wal(void)

入力と出力

  • 入力: なし
  • 出力: なし（副作用として統計情報を更新）

前提条件と不変条件

  • pgstat サブシステムが初期化されており、バックアップの進捗追跡が開始されていること。

アルゴリズムまたは振る舞い

1. pgstat_progress_update_param を呼び出す。 2. 進捗パラメータ PROGRESS_BASEBACKUP_PHASE を PROGRESS_BASEBACKUP_PHASE_TRANSFER_WAL に設定する。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/backup/basebackup.c:428
  • 呼び出し先: pgstat_progress_update_param（pgstat.h で定義）

関連データ構造

  • PROGRESS_BASEBACKUP_PHASE: 進捗フェーズを識別するための定数。
  • PROGRESS_BASEBACKUP_PHASE_TRANSFER_WAL: WAL転送フェーズを示す値。

失敗モードと境界条件

  • 統計情報収集が無効な環境では、pgstat_progress_update_param は何も行わない。

保守担当者向けメモ

  • 本関数は basebackup.c からの呼び出しにより、バックアップの進行に合わせて明示的にフェーズを遷移させるために使用される。
  • bbsink インターフェースのコールバックではなく、バックアップのメインロジックから直接呼び出される点に注意すること。
