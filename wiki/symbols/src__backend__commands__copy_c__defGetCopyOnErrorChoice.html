defGetCopyOnErrorChoice

定義

  • 種別: function
  • 定義箇所: src/backend/commands/copy.c:432

目的

COPY コマンドの ON_ERROR オプション値を解析し、内部的な列挙型 CopyOnErrorChoice に変換する。

シグネチャまたは形状

CopyOnErrorChoice defGetCopyOnErrorChoice(DefElem *def, ParseState *pstate, bool is_from)

入力と出力

  • **入力**:
  • def: 解析対象の DefElem 構造体。
  • pstate: パーサーの状態を保持する ParseState。
  • is_from: COPY FROM 操作であるかを示す真偽値。
  • **出力**:
  • 成功時: COPY_ON_ERROR_STOP または COPY_ON_ERROR_IGNORE を返す。

前提条件と不変条件

  • is_from が false（COPY TO）の場合、本関数はエラーを発生させる。

アルゴリズムまたは振る舞い

1. defGetString(def) を呼び出し、オプション値を文字列として取得する。 2. is_from が false の場合、COPY TO では ON_ERROR を使用できないため、ERRCODE_INVALID_PARAMETER_VALUE でエラーを報告する。 3. 文字列が "stop" の場合は COPY_ON_ERROR_STOP を返す。 4. 文字列が "ignore" の場合は COPY_ON_ERROR_IGNORE を返す。 5. 上記以外の場合は、認識できないオプション値として ERRCODE_INVALID_PARAMETER_VALUE でエラーを報告する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/commands/copy.c 内の ProcessCopyOptions（opts_out->on_error = defGetCopyOnErrorChoice(...) として呼び出される）。
  • **呼び出し先**: defGetString（DefElem から文字列を取得）、ereport（エラー報告）、pg_strcasecmp（文字列比較）。

関連データ構造

  • CopyOnErrorChoice: ON_ERROR オプションの選択肢を定義する列挙型。
  • DefElem: SQL 文のオプション定義を保持するノード。

失敗モードと境界条件

  • COPY TO で ON_ERROR が指定された場合、エラーとなる。
  • 指定された値が "stop" または "ignore" 以外の場合、エラーとなる。
  • エラー発生時は parser_errposition を使用して、SQL 文内の該当位置を特定する。

保守担当者向けメモ

  • 本関数は COPY FROM 専用のオプション解析ロジックである。将来的に ON_ERROR の選択肢が増える場合は、pg_strcasecmp の判定ロジックを拡張すること。
