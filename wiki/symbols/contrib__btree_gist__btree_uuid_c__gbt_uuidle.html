gbt_uuidle

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_uuid.c:54

目的

2つの pg_uuid_t 値を比較し、第一引数が第二引数以下（less than or equal to）であるかを判定します。

シグネチャまたは形状

static bool gbt_uuidle(const void *a, const void *b, FmgrInfo *flinfo)

入力と出力

  • **入力**:
  • a: 比較対象の UUID（const void * として渡されるが、内部で pg_uuid_t * にキャストされる）。
  • b: 比較対象の UUID。
  • flinfo: 関数管理情報（本関数内では使用されません）。
  • **出力**:
  • bool: a <= b であれば true、そうでなければ false を返します。

前提条件と不変条件

  • 入力ポインタ a および b は、有効な pg_uuid_t 構造体を指している必要があります。
  • UUID の比較は uuid_internal_cmp を介して行われ、バイト列としての辞書順比較に基づきます。

アルゴリズムまたは振る舞い

1. a と b を pg_uuid_t 型にキャストします。 2. uuid_internal_cmp を呼び出し、その戻り値が 0 以下であるかを確認します。 3. 比較結果をブール値として返します。

呼び出し元と呼び出し先

  • **呼び出し先**: uuid_internal_cmp
  • **呼び出し元**:
  • contrib/btree_gist/btree_uuid.c 内の演算子クラス定義（GiST インデックスのサポート関数として参照されています）。

関連データ構造

  • pg_uuid_t: PostgreSQL の UUID 型を表現する構造体。

失敗モードと境界条件

  • 入力ポインタが NULL の場合の挙動は定義されていません。
  • uuid_internal_cmp が内部的に行うバイト比較の結果に依存します。

保守担当者向けメモ

  • 本関数は btree_gist における UUID 型の比較演算子クラスの一部です。ロジックを変更する場合は、uuid_internal_cmp の仕様と整合性が取れていることを確認してください。
