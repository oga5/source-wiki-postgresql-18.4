CopyConvertBuf

定義

  • 種別: function
  • 定義箇所: src/backend/commands/copyfromparse.c:399

目的

COPY FROM コマンドにおいて、入力バッファ内のデータに対してエンコーディング変換および妥当性検証を行う。

シグネチャまたは形状

static void CopyConvertBuf(CopyFromState cstate)

入力と出力

  • **入力**: CopyFromState 構造体（cstate）。raw_buf に格納された未処理の入力データ、およびエンコーディング設定を参照する。
  • **出力**: なし（副作用として cstate 内のバッファ状態を更新する）。

前提条件と不変条件

  • cstate->need_transcoding の値に基づき、変換の有無が決定される。
  • 変換が不要な場合、raw_buf と input_buf は同一のメモリ領域を指す。
  • 変換が必要な場合、raw_buf から input_buf へ変換後のデータが書き込まれる。

アルゴリズムまたは振る舞い

1. **変換不要時**: pg_encoding_verifymbstr を呼び出し、raw_buf 内の未検証バイト列が現在のエンコーディングとして妥当かを確認する。 2. **変換必要時**:

  • input_buf 内の未処理データを先頭へ移動する。
  • pg_do_encoding_conversion_buf を呼び出し、raw_buf から input_buf へエンコーディング変換を行う。
  • 変換されたバイト数を input_buf_len に加算する。

    • 3. **EOF/エラー処理**:

  • データが空で raw_reached_eof が真であれば input_reached_eof を設定する。
  • 不完全なマルチバイトシーケンスや不正なバイト列を検出した場合、input_reached_error を設定する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/commands/copyfromparse.c 内の COPY FROM 処理ループ（例: CopyReadLine 等の内部）。
  • **呼び出し先**: pg_encoding_verifymbstr, pg_do_encoding_conversion_buf 等のエンコーディング関連ユーティリティ。

関連データ構造

  • CopyFromState: COPY FROM の状態を保持する構造体。raw_buf, input_buf, file_encoding 等を含む。

失敗モードと境界条件

  • **EOF**: 入力ソースの終端に達した場合、input_reached_eof が設定される。
  • **不正なバイト列**: エンコーディング検証または変換に失敗した場合、input_reached_error が設定される。
  • **不完全なシーケンス**: 入力バッファの末尾でマルチバイト文字が途切れている場合、変換/検証を中断し、後続のデータ読み込みを待機する。

保守担当者向けメモ

  • 本関数は COPY FROM のデータパイプラインにおける「Convert」層を担う。
  • input_buf のオーバーフローを防ぐため、変換処理はバッファの空き容量を考慮して実行される。
  • エラー報告は、すべての有効なバイトが消費されるまで遅延される設計となっている（end-of-input マーカーを正しく処理するため）。
