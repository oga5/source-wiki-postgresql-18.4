pgaio_io_get_owner

定義

  • 種別: function
  • 定義箇所: src/backend/storage/aio/aio.c:354

目的

指定された非同期I/Oハンドル（PgAioHandle）を所有しているバックエンドプロセスの識別番号（ProcNumber）を取得します。

シグネチャまたは形状

ProcNumber pgaio_io_get_owner(PgAioHandle *ioh)

入力と出力

  • **入力**: PgAioHandle *ioh - 情報を取得対象とするI/Oハンドルへのポインタ。
  • **出力**: ProcNumber - ハンドルを所有するプロセスの識別番号。

前提条件と不変条件

  • 引数 ioh は有効な PgAioHandle 構造体を指している必要があります。

アルゴリズムまたは振る舞い

PgAioHandle 構造体内の owner_procno メンバを直接参照し、その値を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/storage/buffer/bufmgr.c:7192: ハンドルの所有権を確認するための Assert 文で使用。
  • src/backend/storage/smgr/smgr.c:1079: プロセス番号の取得に使用。
  • **呼び出し先**: なし。

関連データ構造

  • PgAioHandle: 非同期I/O操作を管理する主要な構造体。owner_procno フィールドを保持します。
  • ProcNumber: PostgreSQLのバックエンドプロセスを識別するための型。

失敗モードと境界条件

  • ioh が NULL の場合の挙動については、提供されたソースコード内では明示されていません。

保守担当者向けメモ

  • 本関数は単なるゲッターであり、状態遷移やロックの取得は行いません。
  • src/include/storage/aio.h:288 に外部公開用のプロトタイプ宣言が存在します。
