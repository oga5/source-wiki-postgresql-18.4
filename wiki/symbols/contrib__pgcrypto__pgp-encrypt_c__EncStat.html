EncStat

定義

  • 種別: struct
  • 定義箇所: contrib/pgcrypto/pgp-encrypt.c:151

目的

暗号化フィルタ（encrypt_filter）の実行状態を保持するためのプライベートなデータ構造です。

シグネチャまたは形状

struct EncStat
{
	PGP_CFB    *ciph;
	uint8		buf[ENCBUF];
};

入力と出力

  • ciph: 暗号化処理に使用する PGP_CFB コンテキストへのポインタ。
  • buf: 暗号化処理中に使用される固定長（ENCBUF）の作業用バッファ。

前提条件と不変条件

  • 本構造体は encrypt_filter のプライベートデータとして使用されます。
  • ciph は初期化済みである必要があります。

アルゴリズムまたは振る舞い

暗号化フィルタのライフサイクルにおいて、CFB モードの暗号化状態を維持し、データ処理に必要な一時領域を提供します。

呼び出し元と呼び出し先

  • 呼び出し元: contrib/pgcrypto/pgp-encrypt.c 内の encrypt_init、encrypt_process、encrypt_free で参照されています。
  • 呼び出し先: 本構造体自体はデータ保持用であり、直接関数を呼び出すことはありません。

関連データ構造

  • PGP_CFB: 暗号化アルゴリズムのコンテキスト。
  • encrypt_filter: 本構造体を利用するフィルタ実装。

失敗モードと境界条件

  • buf のサイズ ENCBUF を超えるデータ処理は、フィルタのロジックに依存します。
  • 構造体の初期化に失敗した場合、ciph が NULL のままアクセスされる可能性があります。

保守担当者向けメモ

  • 本構造体は pgp-encrypt.c 内で完結しており、外部モジュールからは直接操作すべきではありません。
  • ENCBUF のサイズ変更が必要な場合は、バッファオーバーフローのリスクがないか確認してください。
