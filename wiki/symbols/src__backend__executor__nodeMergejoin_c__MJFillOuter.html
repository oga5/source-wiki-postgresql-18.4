MJFillOuter

定義

  • 種別: function
  • 定義箇所: src/backend/executor/nodeMergejoin.c:446

目的

外部結合（Outer Join）において、結合条件に一致する相手側タプルが存在しない場合に、NULL補完されたタプルを生成して返す。

シグネチャまたは形状

TupleTableSlot *MJFillOuter(MergeJoinState *node)

入力と出力

  • **入力**: MergeJoinState *node（マージ結合の実行状態を保持する構造体）
  • **出力**: 結合条件（otherqual）を満たした場合は投影後の TupleTableSlot * を返し、満たさない場合は NULL を返す。

前提条件と不変条件

  • node->mj_OuterTupleSlot に現在の外部タプルが保持されていること。
  • node->mj_NullInnerTupleSlot にNULL補完用の空タプルが準備されていること。
  • node->js.ps.ps_ExprContext が有効であること。

アルゴリズムまたは振る舞い

1. ResetExprContext を呼び出し、式評価コンテキストをリセットする。 2. コンテキストの外部タプルに node->mj_OuterTupleSlot を、内部タプルに node->mj_NullInnerTupleSlot を設定する。 3. ExecQual を用いて otherqual（結合条件）を評価する。 4. 条件が真であれば、ExecProject を実行して投影後のタプルを返す。 5. 条件が偽であれば、InstrCountFiltered2 を呼び出してフィルタリングされたカウントを更新し、NULL を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/executor/nodeMergejoin.c 内の複数箇所（行671, 962, 1224, 1402）。
  • **呼び出し先**: ResetExprContext, ExecQual, ExecProject, InstrCountFiltered2。

関連データ構造

  • MergeJoinState: マージ結合の実行状態を管理する主要構造体。
  • TupleTableSlot: タプルを保持するスロット。
  • ExprContext: 式評価のためのコンテキスト。

失敗モードと境界条件

  • otherqual が評価結果として偽を返す場合、この関数は NULL を返し、呼び出し側は次のタプル処理へ移行する。
  • 投影（ExecProject）が失敗した場合の挙動は、呼び出し側の ExecProject 実装に依存する。

保守担当者向けメモ

  • 本関数は外部結合のNULL補完専用である。内部結合のNULL補完には MJFillInner が使用される。
  • MJ_printf によるデバッグ出力が含まれており、トレースが必要な場合はマクロ定義を確認すること。
