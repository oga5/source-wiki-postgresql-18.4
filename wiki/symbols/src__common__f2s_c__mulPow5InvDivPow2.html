mulPow5InvDivPow2

定義

  • 種別: function
  • 定義箇所: src/common/f2s.c:161

目的

Ryuアルゴリズムに基づき、浮動小数点数の変換過程で5の累乗の逆数を用いた除算（ビットシフトによる近似）を実行する。

シグネチャまたは形状

static inline uint32 mulPow5InvDivPow2(const uint32 m, const uint32 q, const int32 j)

入力と出力

  • **入力**:
  • m: 被乗数（仮数に関連する値）。
  • q: FLOAT_POW5_INV_SPLIT テーブルのインデックス。
  • j: mulShift に渡されるビットシフト量。
  • **出力**:
  • uint32: m と FLOAT_POW5_INV_SPLIT[q] を乗算し、結果を j ビット右シフトした値。

前提条件と不変条件

  • FLOAT_POW5_INV_SPLIT テーブルが定義されていること。
  • mulShift 関数が利用可能であること。

アルゴリズムまたは振る舞い

1. FLOAT_POW5_INV_SPLIT 配列からインデックス q に対応する事前計算値を取得する。 2. mulShift 関数を呼び出し、m と取得した値の乗算およびビットシフト処理を行う。 3. その結果を戻り値として返す。

関連データ構造

  • FLOAT_POW5_INV_SPLIT: 5の累乗の逆数を保持する事前計算テーブル。

失敗モードと境界条件

  • j の値が mulShift の許容範囲外である場合、予期しないシフト結果となる可能性がある。
  • q が FLOAT_POW5_INV_SPLIT の有効なインデックス範囲外である場合、メモリ境界外アクセスが発生する。

保守担当者向けメモ

  • 本関数は static inline で定義されており、コンパイル時にインライン展開されることが期待されている。
  • mulShift の実装詳細に依存するため、変更時は ryu_common.h 等の関連定義を確認すること。
