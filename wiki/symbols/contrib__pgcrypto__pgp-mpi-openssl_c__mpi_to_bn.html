mpi_to_bn

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/pgp-mpi-openssl.c:38

目的

OpenPGP 形式の多倍長整数（PGP_MPI）を、OpenSSL の BIGNUM 形式に変換する。

シグネチャまたは形状

BIGNUM *mpi_to_bn(PGP_MPI *n)

入力と出力

  • **入力**: PGP_MPI *n（変換対象の MPI 構造体）
  • **出力**: 成功時は変換された BIGNUM * を返し、失敗時は NULL を返す。

前提条件と不変条件

  • 入力された PGP_MPI 構造体は有効である必要がある。
  • 変換後の BIGNUM のビット長は、元の PGP_MPI が保持するビット長（n->bits）と一致しなければならない。

アルゴリズムまたは振る舞い

1. BN_bin2bn を使用して、PGP_MPI のデータ部を BIGNUM に変換する。 2. 変換結果が NULL の場合はそのまま NULL を返す。 3. BN_num_bits を用いて、生成された BIGNUM のビット長が元の PGP_MPI のビット長と一致するか検証する。 4. ビット長が一致しない場合は、エラーログを出力し、BN_clear_free でメモリを解放した上で NULL を返す。 5. 一致する場合は、生成された BIGNUM ポインタを返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • pgp_elgamal_encrypt (ElGamal 暗号化時のパラメータ変換)
  • pgp_elgamal_decrypt (ElGamal 復号時のパラメータ変換)
  • pgp_rsa_encrypt (RSA 暗号化時のパラメータ変換)
  • **呼び出し先**:
  • BN_bin2bn (OpenSSL)
  • BN_num_bits (OpenSSL)
  • BN_clear_free (OpenSSL)
  • px_debug (内部ユーティリティ)

関連データ構造

  • PGP_MPI: PGP 形式の多倍長整数を保持する構造体。
  • BIGNUM: OpenSSL の多倍長整数構造体。

失敗モードと境界条件

  • BN_bin2bn がメモリ不足等で失敗した場合、NULL を返す。
  • 変換後のビット長が元の MPI のビット長と不一致の場合、変換は失敗とみなされ NULL を返す。

保守担当者向けメモ

  • 本関数が返す BIGNUM ポインタは動的に確保されるため、呼び出し側で適切に BN_clear_free を呼び出し、メモリリークを防ぐ必要がある。
  • エラー発生時のログ出力には px_debug が使用されている。
