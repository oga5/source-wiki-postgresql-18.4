pgstat_replslot_reset_timestamp_cb

定義

  • 種別: function
  • 定義箇所: src/backend/utils/activity/pgstat_replslot.c:217

目的

レプリケーションスロットの統計情報における「統計リセット時刻」を更新するためのコールバック関数です。

シグネチャまたは形状

void pgstat_replslot_reset_timestamp_cb(PgStatShared_Common *header, TimestampTz ts)

入力と出力

  • header: 更新対象となるレプリケーションスロットの共有統計ヘッダー（PgStatShared_Common 型）。
  • ts: 設定する新しいリセット時刻（TimestampTz 型）。
  • 戻り値: なし。

前提条件と不変条件

  • header は PgStatShared_ReplSlot 構造体としてキャスト可能である必要があります。

アルゴリズムまたは振る舞い

1. 引数 header を PgStatShared_ReplSlot 型へキャストします。 2. キャストした構造体内の stats.stat_reset_timestamp フィールドに、引数 ts で渡された値を代入します。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/utils/activity/pgstat.c:344 において、統計サブシステムのコールバック構造体の一部として参照されています。
  • 呼び出し先: 本関数内での関数呼び出しはありません。

関連データ構造

  • PgStatShared_Common: 統計共有メモリの共通ヘッダー。
  • PgStatShared_ReplSlot: レプリケーションスロットの統計情報を保持する共有メモリ上の構造体。

失敗モードと境界条件

  • header が NULL または PgStatShared_ReplSlot 以外の型を指している場合、メモリ破壊や不正なメモリアクセスが発生する可能性があります。

保守担当者向けメモ

  • 本関数は統計サブシステムの汎用的なリセット処理からコールバックとして呼び出される設計です。
  • PgStatShared_ReplSlot 構造体のレイアウト変更時には、キャストの整合性に注意してください。
