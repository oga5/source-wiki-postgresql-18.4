jbvType

定義

  • 種別: struct（匿名構造体）
  • 定義箇所: src/backend/utils/adt/jsonb.c:2010

目的

JsonbValue の型（enum jbvType）と、それに対応するエラーメッセージや型名をマッピングするための静的テーブルの要素として定義されています。

シグネチャまたは形状

static const struct
{
    enum jbvType type;
    const char *msg;
}

入力と出力

  • 入力: なし（静的定義）
  • 出力: なし（静的定義）

前提条件と不変条件

  • type フィールドは utils/jsonb.h で定義された enum jbvType の値をとります。
  • この構造体は jsonb.c 内で型変換エラー時のメッセージ生成に使用されることを前提としています。

アルゴリズムまたは振る舞い

この構造体自体はデータ保持用であり、ロジックは含みません。この構造体の配列を用いて、cannotCastJsonbValue 関数などが特定の jbvType に対応するエラーメッセージを検索するルックアップテーブルとして機能します。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/utils/adt/jsonb.c:2008 の cannotCastJsonbValue 関数が、この構造体を用いたテーブルを参照しています。
  • 呼び出し先: なし。

関連データ構造

  • enum jbvType: utils/jsonb.h に定義された列挙型。JSONB の各要素（数値、文字列、オブジェクト、配列など）の型を識別します。
  • JsonbValue: jsonb の各要素を保持する構造体。

失敗モードと境界条件

  • 構造体定義自体に失敗モードはありません。
  • ルックアップテーブルとして使用される際、定義されていない jbvType が渡された場合の挙動は、テーブルを走査する側の実装に依存します。

保守担当者向けメモ

  • 本シンボルは jsonb.c 内のローカルな静的構造体です。
  • enum jbvType に新しい型が追加された場合は、この構造体を利用しているテーブルにも対応するエントリを追加する必要があります。
