jspInit

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/jsonpath.c:979

目的

JsonPath 構造体からルートノードを抽出し、JsonPathItem を初期化します。

シグネチャまたは形状

void jspInit(JsonPathItem *v, JsonPath *js)

入力と出力

  • **入力**:
  • v: 初期化対象となる JsonPathItem 構造体へのポインタ。
  • js: 読み込み元の JsonPath 構造体へのポインタ。
  • **出力**: なし（v が指すメモリ領域が更新されます）。

前提条件と不変条件

  • js->header のバージョンビット（JSONPATH_LAX フラグを除いた部分）が JSONPATH_VERSION と一致している必要があります。一致しない場合、アサーションエラーが発生します。

アルゴリズムまたは振る舞い

1. js->header のバージョン整合性を検証します。 2. js->data（バイナリバッファの先頭）を起点として、jspInitByBuffer を呼び出し、v にルートノードの情報を設定します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/jsonb_gin.c
  • src/backend/utils/adt/jsonpath.c
  • src/backend/utils/adt/jsonpath_exec.c
  • **呼び出し先**:
  • jspInitByBuffer

関連データ構造

  • JsonPath: jsonpath 型のメモリ上の構造体。
  • JsonPathItem: パス式の各要素を表現する構造体。

失敗モードと境界条件

  • js が指す JsonPath 構造体のヘッダーが不正な場合、アサーションによりプロセスが停止します。

保守担当者向けメモ

  • 本関数は jspInitByBuffer のラッパーであり、バイナリバッファのオフセット 0 を指定して初期化を行います。
  • jsonpath.h にて extern 宣言されており、外部モジュールからも利用可能です。
