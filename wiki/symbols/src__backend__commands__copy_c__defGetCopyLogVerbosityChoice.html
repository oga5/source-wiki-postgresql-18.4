defGetCopyLogVerbosityChoice

定義

  • 種別: function
  • 定義箇所: src/backend/commands/copy.c:495

目的

COPY コマンドの LOG_VERBOSITY オプションに指定された文字列値を解析し、対応する内部列挙型 CopyLogVerbosityChoice に変換する。

シグネチャまたは形状

static CopyLogVerbosityChoice defGetCopyLogVerbosityChoice(DefElem *def, ParseState *pstate)

入力と出力

  • **入力**:
  • def: 解析対象の DefElem 構造体（LOG_VERBOSITY オプションの定義）。
  • pstate: エラー報告に使用する ParseState。
  • **出力**:
  • 成功時: COPY_LOG_VERBOSITY_SILENT, COPY_LOG_VERBOSITY_DEFAULT, または COPY_LOG_VERBOSITY_VERBOSE のいずれか。
  • 失敗時: ereport(ERROR, ...) により処理を中断。

前提条件と不変条件

  • 入力される def は LOG_VERBOSITY オプションに関連付けられている必要がある。
  • defGetString(def) が有効な文字列を返すこと。

アルゴリズムまたは振る舞い

1. defGetString を呼び出し、オプションの値を文字列として取得する。 2. pg_strcasecmp を使用して、値が "silent"、"default"、"verbose" のいずれであるかを大文字小文字を区別せずに判定する。 3. 一致する値があれば対応する列挙値を返す。 4. いずれにも一致しない場合、ERRCODE_INVALID_PARAMETER_VALUE を伴うエラーを発生させ、パーサーの現在の位置情報を報告する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/commands/copy.c 内の ProcessCopyOptions 関数から呼び出される（opts_out->log_verbosity = defGetCopyLogVerbosityChoice(defel, pstate);）。
  • **呼び出し先**: defGetString（ユーティリティ関数）、pg_strcasecmp（文字列比較）、ereport（エラー報告）、parser_errposition（エラー位置特定）。

関連データ構造

  • CopyLogVerbosityChoice: LOG_VERBOSITY の設定値を保持する列挙型。
  • DefElem: COPY コマンドのオプション定義を保持する構造体。
  • ParseState: 構文解析の状態を保持する構造体。

失敗モードと境界条件

  • **無効な値**: "silent", "default", "verbose" 以外の文字列が指定された場合、ereport によりエラーがスローされる。
  • **NULL 入力**: def が NULL の場合の挙動は定義されていない（呼び出し元に依存）。

保守担当者向けメモ

  • 本関数は COPY コマンドのオプション解析専用である。新しいログ冗長性レベルを追加する場合は、列挙型の定義と本関数の条件分岐の両方を更新する必要がある。
