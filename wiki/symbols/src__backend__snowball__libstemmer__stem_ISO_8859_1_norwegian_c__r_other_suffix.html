r_other_suffix

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_norwegian.c:223

目的

ノルウェー語のステミング処理において、単語末尾の特定の接尾辞を削除する。

シグネチャまたは形状

static int r_other_suffix(struct SN_env * z)

入力と出力

  • 入力: SN_env 構造体（単語バッファおよびステミング状態を保持）。
  • 出力: 成功時に 1、失敗または条件不一致時に 0、エラー発生時に負の値を返す。

前提条件と不変条件

  • z->I[1] に格納された領域境界より後ろの文字を処理対象とする。
  • 処理対象の文字が特定のビットマスク条件を満たす必要がある。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c が領域境界 z->I[1] より前であれば終了する。 2. 処理範囲を z->I[1] から単語末尾までに制限する。 3. 単語末尾の文字が特定のビットマスク（4718720）に合致するか確認する。 4. a_2 で定義された接尾辞リストを末尾から検索し、一致するものがあればその範囲を削除する。 5. 処理終了後に一時的な境界設定を復元する。

呼び出し元と呼び出し先

  • 呼び出し元: norwegian_ISO_8859_1_stem（stem_ISO_8859_1_norwegian.c 内）
  • 呼び出し先: find_among_b（接尾辞検索）、slice_del（文字列削除）

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_2: 削除対象となる接尾辞を格納した among 構造体の配列。

失敗モードと境界条件

  • カーソル位置が領域境界より前にある場合は 0 を返す。
  • 接尾辞が見つからない場合や、ビットマスク条件を満たさない場合は 0 を返す。
  • slice_del が失敗した場合は、その戻り値（負の値）をそのまま返す。

保守担当者向けメモ

  • 本関数はSnowballコンパイラによる自動生成コードである。ロジックの変更が必要な場合は、対応するSnowballソース（.sbl）を修正すること。
  • a_2 配列の要素数（11）は、Snowballソースの定義と同期している必要がある。
