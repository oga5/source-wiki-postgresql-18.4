PLy_result_init_type

定義

  • 種別: function
  • 定義箇所: src/pl/plpython/plpy_resultobject.c:80

目的

Pythonの型システムに対して PLyResult 型を登録し、PL/Python拡張内でクエリ結果オブジェクトを扱えるように初期化します。

シグネチャまたは形状

void PLy_result_init_type(void)

入力と出力

  • 入力: なし
  • 出力: なし（副作用としてグローバル変数 PLy_ResultType を設定）

前提条件と不変条件

  • PLyResult_spec が有効なPython型定義仕様として定義されている必要があります。
  • 本関数はPythonインタプリタが初期化された後に呼び出される必要があります。

アルゴリズムまたは振る舞い

1. PyType_FromSpec を呼び出し、静的定義 PLyResult_spec に基づいてPython型オブジェクトを作成します。 2. 作成された型オブジェクトをグローバル変数 PLy_ResultType に格納します。 3. 型オブジェクトの作成に失敗した場合、elog(ERROR, ...) を呼び出し、PostgreSQLの例外メカニズムを通じて処理を中断します。

呼び出し元と呼び出し先

  • 呼び出し元: src/pl/plpython/plpy_plpymodule.c:150
  • 呼び出し先: PyType_FromSpec (Python C API), elog (PostgreSQLエラー報告)

関連データ構造

  • PLyResult_spec: PyType_Spec 構造体。Python型のスロットやメソッドを定義します。
  • PLy_ResultType: PyTypeObject へのポインタ。初期化された型情報を保持します。

失敗モードと境界条件

  • PyType_FromSpec が NULL を返した場合、elog(ERROR) によりPostgreSQLのトランザクションがアボートされます。

保守担当者向けメモ

  • 本関数はPython 3の型定義APIを使用しています。Pythonのバージョンアップに伴う型定義仕様の変更がある場合は PLyResult_spec の内容を確認してください。
