AtStart_GUC

定義

  • 種別: function
  • 定義箇所: src/backend/utils/misc/guc.c:2214

目的

トランザクション開始時に GUC（Grand Unified Configuration）のネストレベルを初期化し、設定値のスタック操作を準備する。

シグネチャまたは形状

void AtStart_GUC(void)

入力と出力

  • 入力: なし
  • 出力: なし（副作用としてグローバル変数 GUCNestLevel を更新）

前提条件と不変条件

  • トランザクション間では GUCNestLevel が 0 であることが期待される。

アルゴリズムまたは振る舞い

1. 現在の GUCNestLevel が 0 でない場合、警告（WARNING）をログに出力する（前回のトランザクション終了処理が正しく行われていない可能性を示す）。 2. GUCNestLevel を 1 に設定する。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/access/transam/xact.c:2204
  • 呼び出し先: なし（内部的なログ出力関数 elog を除く）

関連データ構造

  • GUCNestLevel（グローバル変数）: 現在の GUC 設定のネスト深さを保持する。

失敗モードと境界条件

  • GUCNestLevel != 0: 正常な状態ではないが、処理は中断されず警告出力のみ行われる。

保守担当者向けメモ

  • 本関数はトランザクション開始のフックとして機能する。対となる終了処理（AtEOXact_GUC）との整合性が重要である。
