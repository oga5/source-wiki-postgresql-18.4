IsSyncingReplicationSlots

定義

  • 種別: function
  • 定義箇所: src/backend/replication/logical/slotsync.c:1782

目的

現在、論理レプリケーションスロットの同期処理が実行中であるかどうかを示すブール値を返します。

シグネチャまたは形状

bool IsSyncingReplicationSlots(void)

入力と出力

  • **入力**: なし。
  • **出力**: 同期中であれば true、そうでなければ false を返します。

前提条件と不変条件

  • この関数は、内部的な静的変数 syncing_slots の状態を反映します。
  • 呼び出し元は、この関数の戻り値に基づいて、リカバリ中のスロット操作やレプリケーションの挙動を制御します。

アルゴリズムまたは振る舞い

  • 内部変数 syncing_slots の値をそのまま返します。
