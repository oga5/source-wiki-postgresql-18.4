brin_free_desc

定義

  • 種別: function
  • 定義箇所: src/backend/access/brin/brin.c:1635

目的

BrinDesc 構造体に関連付けられたメモリコンテキストを解放し、リソースをクリーンアップします。

シグネチャまたは形状

void brin_free_desc(BrinDesc *bdesc)

入力と出力

  • **入力**: bdesc (BrinDesc *) - 解放対象の BRIN 記述子構造体へのポインタ。
  • **出力**: なし。

前提条件と不変条件

  • bdesc->bd_tupdesc->tdrefcount が 1 以上であることを保証するアサーションが存在します。

アルゴリズムまたは振る舞い

1. bdesc->bd_tupdesc の参照カウントが有効であることを確認します。 2. bdesc->bd_context に割り当てられたメモリコンテキストを MemoryContextDelete により削除します。個別のフィールドに対する pfree は不要です。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/pageinspect/brinfuncs.c (複数箇所)
  • src/backend/access/brin/brin.c (インデックススキャン終了時やビルド状態の破棄時など)
  • **呼び出し先**:
  • MemoryContextDelete (PostgreSQL メモリ管理 API)

関連データ構造

  • BrinDesc: BRIN インデックスの操作に必要な情報を保持する構造体。

失敗モードと境界条件

  • bdesc が NULL の場合、あるいは bd_tupdesc の参照カウントが不正な場合はアサーション違反により異常終了します。

保守担当者向けメモ

  • 本関数は BrinDesc が保持するメモリコンテキスト全体を破棄するため、呼び出し後に bdesc ポインタを参照することはできません。
