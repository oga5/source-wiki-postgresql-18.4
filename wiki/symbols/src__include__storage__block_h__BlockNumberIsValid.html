BlockNumberIsValid

定義

  • 種別: function
  • 定義箇所: src/include/storage/block.h:70

目的

指定されたブロック番号が有効な値であるかを判定します。

シグネチャまたは形状

static inline bool BlockNumberIsValid(BlockNumber blockNumber)

入力と出力

  • **入力**: BlockNumber 型の blockNumber（検証対象のブロック番号）。
  • **出力**: bool 型。blockNumber が InvalidBlockNumber（0xFFFFFFFF）と等しくない場合に true を返し、等しい場合に false を返します。

前提条件と不変条件

  • BlockNumber は 32ビット符号なし整数として定義されています。
  • InvalidBlockNumber は無効なブロックを示す定数として定義されています。

アルゴリズムまたは振る舞い

引数として受け取った blockNumber が、無効値を示す InvalidBlockNumber と一致するかを比較する単純なインライン判定関数です。

関連データ構造

  • BlockNumber: 本関数が検証対象とする基本型。
  • InvalidBlockNumber: 本関数が比較対象とする定数。

失敗モードと境界条件

  • InvalidBlockNumber（0xFFFFFFFF）が渡された場合、false を返します。
  • それ以外のすべての uint32 値に対しては true を返します。
