pg_wchar_tbl

定義

  • 種別: struct
  • 定義箇所: src/include/mb/pg_wchar.h:392

目的

PostgreSQLがサポートする各文字エンコーディングに対し、変換、検証、および長さ計算を行うための関数ポインタとメタデータを保持するテーブル構造体です。

シグネチャまたは形状

typedef struct
{
	mb2wchar_with_len_converter mb2wchar_with_len;
	wchar2mb_with_len_converter wchar2mb_with_len;
	mblen_converter mblen;
	mbdisplaylen_converter dsplen;
	mbchar_verifier mbverifychar;
	mbstr_verifier mbverifystr;
	int			maxmblen;
} pg_wchar_tbl;

入力と出力

本構造体は、特定のエンコーディングに対応する以下の操作を提供します。

  • **入力**: マルチバイト文字列、ワイド文字（pg_wchar）、または文字単位のデータ。
  • **出力**: 変換後の文字列、文字のバイト長、表示幅、または検証結果（成功/失敗）。

前提条件と不変条件

  • maxmblen は、そのエンコーディングにおいて1文字が取り得る最大バイト数を保持する必要があります。
  • 各関数ポインタは、対応するエンコーディングの仕様に従って実装されている必要があります。

アルゴリズムまたは振る舞い

PostgreSQLのエンコーディング変換処理において、現在のセッションエンコーディングに対応する pg_wchar_tbl のインスタンスが参照されます。pg_do_encoding_conversion 等の関数がこのテーブル内の関数ポインタを呼び出すことで、動的にエンコーディング固有の処理を実行します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/common/wchar.c 内の pg_wchar_table[] 定義にて、各エンコーディングの実装関数を格納するために使用されています。
  • **呼び出し先**: 構造体内の各メンバ（mb2wchar_with_len 等）は、各エンコーディング固有の変換・検証ロジックを指し示します。

関連データ構造

  • pg_wchar: PostgreSQL内部のワイド文字型。
  • pg_wchar_table[]: pg_wchar_tbl 型のインスタンスを保持するグローバルな配列。

失敗モードと境界条件

  • mbverifystr や mbverifychar は、不正なマルチバイトシーケンスが入力された場合にエラーを報告する役割を担います。
  • maxmblen を超えるバイト長を持つ文字が処理された場合、バッファオーバーフロー等のリスクがあるため、各実装関数での適切な境界チェックが前提となります。

保守担当者向けメモ

  • 本構造体は src/common/wchar.c で定義される pg_wchar_table 配列のスキーマを規定しています。
  • 新しいエンコーディングを追加する際は、この構造体の全メンバを適切に実装した上で pg_wchar_table に登録する必要があります。
