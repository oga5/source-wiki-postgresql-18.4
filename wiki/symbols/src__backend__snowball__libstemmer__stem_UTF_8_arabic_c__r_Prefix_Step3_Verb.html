r_Prefix_Step3_Verb

定義

  • **種別**: 関数
  • **定義箇所**: src/backend/snowball/libstemmer/stem_UTF_8_arabic.c:1154

目的

アラビア語のステミング処理において、動詞の接頭辞を特定のルールに基づいて除去または置換する。

シグネチャまたは形状

static int r_Prefix_Step3_Verb(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z （Snowball ステミング環境構造体）
  • **出力**: int （成功時に 1、一致しない場合や失敗時に 0 を返す）

前提条件と不変条件

  • z->p に格納された文字列が UTF-8 エンコーディングであること。
  • find_among を用いた検索が実行され、カーソル位置が適切に管理されていること。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を開始点 z->bra に設定する。 2. a_8 テーブルを使用して、4 つのパターンから一致する接頭辞を検索する。 3. 一致しない場合は即座に 0 を返す。 4. 一致した場合、終了点 z->ket を現在のカーソル位置に設定する。 5. 文字列全体の長さ（len_utf8）が 4 文字以下の場合、除去を行わず 0 を返す。 6. among_var の値（1〜4）に応じ、対応する文字列（s_61〜s_64）で該当箇所を置換する。 7. 置換処理が成功すれば 1 を返し、失敗すればエラーコードを返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_UTF_8_arabic.c:1638
  • **呼び出し先**: find_among, len_utf8, slice_from_s

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_8: 接頭辞マッチングに使用される among 構造体の配列。

失敗モードと境界条件

  • **一致なし**: find_among が一致を見つけられない場合、0 を返す。
  • **長さ制限**: 文字列長が 4 文字以下の場合は置換を行わず 0 を返す。
  • **メモリ/操作エラー**: slice_from_s が負の値を返した場合、その値をそのまま呼び出し元へ伝播させる。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードであるため、直接編集せず .sbl ソースファイルを修正すること。
  • UTF-8 マルチバイト文字を扱うため、文字数判定には必ず len_utf8 を使用している点に注意。
