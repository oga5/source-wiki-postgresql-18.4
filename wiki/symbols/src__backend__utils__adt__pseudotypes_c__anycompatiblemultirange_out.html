anycompatiblemultirange_out

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/pseudotypes.c:248

目的

anycompatiblemultirange 擬似型に対する出力関数として機能し、マルチレンジ型の文字列表現を生成します。

シグネチャまたは形状

Datum anycompatiblemultirange_out(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS (関数マネージャ経由で渡される MultirangeType の Datum)
  • **出力**: Datum (マルチレンジ型の文字列表現を含む cstring へのポインタ)

前提条件と不変条件

  • 本関数は anycompatiblemultirange 型の出力処理を担いますが、実体としては multirange_out に処理を委譲します。
  • 呼び出し元は PostgreSQL の関数呼び出し規約（fmgr）に従う必要があります。

アルゴリズムまたは振る舞い

1. 受け取った fcinfo (関数呼び出し情報) をそのまま multirange_out 関数へ渡します。 2. multirange_out の戻り値をそのまま呼び出し元へ返却します。

呼び出し元と呼び出し先

  • **呼び出し元**: PostgreSQL の型システム（pg_type の出力関数エントリ経由）。
  • **呼び出し先**: multirange_out (直接的な委譲先)。

関連データ構造

  • MultirangeType: マルチレンジ型の内部表現構造体。
  • Datum: PostgreSQL の汎用データ表現。

失敗モードと境界条件

  • multirange_out が内部で発生させるエラー（不正なマルチレンジデータなど）に依存します。
  • 呼び出し元情報および詳細なエラー条件については、本ソース excerpt には記載されていません。

保守担当者向けメモ

  • 本関数は単なるラッパーであり、ロジックの変更が必要な場合は multirange_out を確認してください。
  • 擬似型に対する出力処理として、将来的に型チェック等の追加が必要になった場合は本関数で制御可能です。
