r_VI

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_finnish.c:406

目的

フィンランド語のステミング処理において、特定の接尾辞パターン（'i' で終わり、その前が母音である場合）を検出し、条件を満たす場合にカーソルを後退させる。

シグネチャまたは形状

static int r_VI(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z（Snowball ステミング環境）
  • 出力: int（成功時に 1、失敗時に 0 を返す）

前提条件と不変条件

  • z->c（現在のカーソル位置）が z->lb（境界位置）より大きいこと。
  • 処理対象の文字が UTF-8 エンコーディングであること。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c が境界 z->lb 以下であるか、または直前の文字が 'i' でない場合は 0 を返して終了する。 2. カーソルを 1 文字分左へ移動（z->c--）させる。 3. 移動後の位置で g_V2（母音グループ）に含まれる文字かどうかを in_grouping_b_U で判定する。 4. 母音であれば 1 を返し、そうでなければ 0 を返す。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_finnish.c および stem_ISO_8859_1_finnish.c 内の among 構造体配列から参照されている。
  • 呼び出し先: in_grouping_b_U（Snowball ランタイム関数）。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • g_V2: 母音を定義するビットマップ配列。

失敗モードと境界条件

  • カーソルが境界 z->lb に達している場合、または直前の文字が 'i' でない場合は失敗する。
  • 'i' の直前が g_V2 に定義された母音グループに含まれない場合、失敗する。

  • g_V2 の定義内容がフィンランド語の母音ルールと一致しているか確認が必要な場合は、同ファイルの定義箇所を参照すること。
