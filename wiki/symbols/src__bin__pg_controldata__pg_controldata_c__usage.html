usage

定義

  • 種別: function
  • 定義箇所: src/bin/pg_controldata/pg_controldata.c:32

目的

pg_controldata コマンドの利用方法、オプション、およびサポート情報を標準出力に表示する。

シグネチャまたは形状

static void usage(const char *progname)

入力と出力

  • **入力**:
  • progname: 実行中のプログラム名（通常は argv[0]）。
  • **出力**:
  • なし（標準出力へヘルプメッセージを書き出す）。

前提条件と不変条件

  • progname は有効な文字列ポインタであること。
  • gettext（_() マクロ）が適切に初期化されている環境で呼び出されること。

アルゴリズムまたは振る舞い

1. プログラム名を含む概要を表示する。 2. コマンドライン引数の書式（[OPTION] [DATADIR]）を表示する。 3. 利用可能なオプション（-D, -V, -?）の一覧を表示する。 4. 環境変数 PGDATA のデフォルト動作について説明する。 5. バグ報告先およびプロジェクトのホームページ URL を表示する。

呼び出し元と呼び出し先

  • **呼び出し元**: ソースコード内で明示的な呼び出し元は特定されていません（main 関数内での呼び出しが想定されますが、提供されたソース抜粋には含まれていません）。
  • **呼び出し先**: printf および国際化対応のための _() マクロ。
  • **注意**: ツールが提示した contrib/ 配下の usage への言及は、本シンボルとは異なる文脈（変数名やコメント）であり、本関数の呼び出し元ではありません。

関連データ構造

  • なし。

失敗モードと境界条件

  • 標準出力への書き込みが失敗した場合の挙動は、標準ライブラリの printf に依存します。

保守担当者向けメモ

  • 本関数は pg_controldata のコマンドラインインターフェースを定義しています。オプションの追加や変更を行った際は、必ず本関数の出力内容を更新してください。
