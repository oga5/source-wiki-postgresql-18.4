compare_sort_item_count

定義

  • 種別: function
  • 定義箇所: src/backend/statistics/mcv.c:402

目的

SortItem 構造体の配列を、count フィールドの値に基づいて降順にソートするための比較関数を提供します。

シグネチャまたは形状

static int compare_sort_item_count(const void *a, const void *b, void *arg);

入力と出力

  • **入力**:
  • a, b: 比較対象の SortItem 構造体へのポインタ（void * として渡される）。
  • arg: 使用されていない補助引数（void *）。
  • **出力**:
  • a->count > b->count の場合: -1
  • a->count < b->count の場合: 1
  • a->count == b->count の場合: 0

前提条件と不変条件

  • 引数 a および b は、SortItem 構造体への有効なポインタである必要があります。
  • SortItem 構造体には count フィールドが存在することが前提です。

アルゴリズムまたは振る舞い

1. 引数 a と b を SortItem * にキャストします。 2. ia->count と ib->count を比較します。 3. 降順ソートを実現するため、ia->count が大きい場合に負の値を返し、小さい場合に正の値を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/statistics/mcv.c 内のソート処理（qsort 等の比較関数として使用）。
  • **呼び出し先**: なし。

関連データ構造

  • SortItem: count フィールドを持つ構造体（本関数がソートの基準として使用）。

失敗モードと境界条件

  • a または b が NULL の場合の動作は定義されていません。
  • count が等しい場合は 0 を返すため、ソートの安定性は保証されません。

保守担当者向けメモ

  • 本関数は qsort 互換のシグネチャを持っています。
  • 降順ソート専用であるため、昇順が必要な場合は別の比較関数が必要です。
