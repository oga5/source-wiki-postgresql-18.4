wal_level_str

定義

  • 種別: function
  • 定義箇所: src/bin/pg_controldata/pg_controldata.c:72

目的

WalLevel 列挙型の値を、人間が判読可能な文字列に変換するヘルパー関数です。

シグネチャまたは形状

static const char *wal_level_str(WalLevel wal_level)

入力と出力

  • **入力**: WalLevel 型の列挙値（WAL_LEVEL_MINIMAL, WAL_LEVEL_REPLICA, WAL_LEVEL_LOGICAL）。
  • **出力**: 対応する文字列（"minimal", "replica", "logical"）を指す定数文字列ポインタ。未知の値の場合は翻訳されたエラーメッセージを返します。

前提条件と不変条件

  • 入力される wal_level は WalLevel 列挙型で定義された有効な値であることが期待されます。

アルゴリズムまたは振る舞い

1. switch 文を用いて wal_level の値を評価します。 2. 各列挙値に対応する文字列リテラルを返します。 3. 定義されていない値が渡された場合、_("unrecognized \"wal_level\"") を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/bin/pg_controldata/pg_controldata.c:298 において、制御ファイルから読み取った wal_level を表示する際に呼び出されています。
  • **呼び出し先**: 内部で他の関数を呼び出していません。
  • **注記**: src/backend/access/rmgrdesc/xlogdesc.c に同名の変数や類似の処理が見られますが、本シンボルとの直接的な呼び出し関係は確認できません。

関連データ構造

  • WalLevel: access/transam.h 等で定義される、WALの記録レベルを保持する列挙型。

失敗モードと境界条件

  • 定義されていない WalLevel 値が渡された場合、プログラムはクラッシュせず、翻訳されたエラー文字列を返します。

保守担当者向けメモ

  • WalLevel 列挙型に新しいレベルが追加された場合、本関数の switch 文にも対応するケースを追加する必要があります。
