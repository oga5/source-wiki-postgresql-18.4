InitXLogReaderState

定義

  • 種別: function
  • 定義箇所: contrib/pg_walinspect/pg_walinspect.c:93

目的

WAL読み取り用の XLogReaderState を初期化し、指定されたLSN以降で最初に有効なWALレコードを検索して読み取り準備を整える。

シグネチャまたは形状

static XLogReaderState *InitXLogReaderState(XLogRecPtr lsn)

入力と出力

  • **入力**: lsn (読み取り開始位置の基準となる XLogRecPtr)
  • **出力**: 初期化済みの XLogReaderState 構造体へのポインタ。
  • **戻り値**: 成功時は XLogReaderState*、失敗時はエラーを報告しプロセスを終了する。

前提条件と不変条件

  • lsn は XLOG_BLCKSZ 以上である必要がある（ブートストラップWALページは読み取り不可）。
  • XLogReaderAllocate によるメモリ確保が成功していること。
  • XLogFindNextRecord により、指定した lsn 以降に有効なレコードが存在することが確認できること。

アルゴリズムまたは振る舞い

1. **入力検証**: lsn が XLOG_BLCKSZ 未満の場合、エラーを発生させる。 2. **メモリ確保**: ReadLocalXLogPageNoWaitPrivate 構造体を割り当てる。 3. **リーダー初期化**: XLogReaderAllocate を呼び出し、WALセグメント読み取り用のコールバック（read_local_xlog_page_no_wait 等）を設定する。 4. **レコード検索**: XLogFindNextRecord を使用し、指定された lsn 以降の最初の有効なレコード位置を特定する。 5. **検証**: 有効なレコードが見つからない場合はエラーを報告し、見つかった場合は初期化済みリーダーを返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/pg_walinspect/pg_walinspect.c 内の各解析関数（pg_get_wal_record_info, pg_get_wal_records_info, pg_get_wal_block_info, pg_get_wal_stats 等）。
  • **呼び出し先**:
  • XLogReaderAllocate
  • XLogFindNextRecord
  • palloc0
  • ereport

関連データ構造

  • XLogReaderState: WAL読み取り状態を保持するコア構造体。
  • ReadLocalXLogPageNoWaitPrivate: WAL読み取り時のプライベートデータを保持する構造体。

失敗モードと境界条件

  • lsn < XLOG_BLCKSZ: ERRCODE_INVALID_PARAMETER_VALUE でエラー。
  • メモリ不足: XLogReaderAllocate が NULL を返した場合、ERRCODE_OUT_OF_MEMORY でエラー。
  • 有効なレコードが見つからない: XLogRecPtrIsInvalid が真となる場合、エラーを報告。

保守担当者向けメモ

  • 本関数は pg_walinspect モジュールの各エントリポイントから共通して利用される。
  • wal_segment_size 等のグローバルなWAL設定に依存しているため、変更時は access/xlogreader.h の仕様と整合性を確認すること。
