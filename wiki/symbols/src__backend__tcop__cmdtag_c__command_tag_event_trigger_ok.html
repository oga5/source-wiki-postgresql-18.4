command_tag_event_trigger_ok

定義

  • 種別: function
  • 定義箇所: src/backend/tcop/cmdtag.c:65

目的

指定されたコマンドタグがイベントトリガーの対象として許可されているかどうかを判定します。

シグネチャまたは形状

bool command_tag_event_trigger_ok(CommandTag commandTag)

入力と出力

  • **入力**: CommandTag 型の列挙値。
  • **出力**: イベントトリガーが許可されている場合は true、そうでない場合は false を返します。

前提条件と不変条件

  • commandTag は有効な CommandTag 列挙値である必要があります。
  • tag_behavior 配列が初期化済みである必要があります。

アルゴリズムまたは振る舞い

tag_behavior 配列（全コマンドタグの属性テーブル）から、引数で指定された commandTag に対応するインデックスを参照し、その要素内の event_trigger_ok フィールドの値を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/commands/event_trigger.c:230
  • src/backend/commands/event_trigger.c:672
  • **呼び出し先**:
  • 直接的な関数呼び出しはありません（tag_behavior 配列へのアクセスのみ）。

関連データ構造

  • CommandTag: コマンドの種類を識別する列挙型。
  • CommandTagBehavior: 各コマンドタグの属性（イベントトリガー可否など）を保持する構造体。

失敗モードと境界条件

  • commandTag が tag_behavior 配列の範囲外である場合、未定義の動作を引き起こす可能性があります。

保守担当者向けメモ

  • 本関数は cmdtaglist.h で定義された属性テーブルに依存しています。新しいコマンドを追加し、それがイベントトリガーをサポートする必要がある場合は、cmdtaglist.h の定義を更新してください。
