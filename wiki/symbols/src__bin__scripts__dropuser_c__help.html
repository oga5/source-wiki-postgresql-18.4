help

定義

  • 種別: function
  • 定義箇所: src/bin/scripts/dropuser.c:167

目的

dropuser コマンドの利用方法およびサポートされているオプションの一覧を標準出力に表示する。

シグネチャまたは形状

static void help(const char *progname)

入力と出力

  • **入力**: progname (実行中のプログラム名を示す文字列へのポインタ)
  • **出力**: なし（標準出力へヘルプメッセージを出力して終了する）

前提条件と不変条件

  • progname は有効な文字列ポインタであること。
  • printf が標準出力に対して書き込み可能であること。

アルゴリズムまたは振る舞い

1. progname を使用して、ツールの概要と使用法（Usage）を表示する。 2. dropuser で利用可能なオプション（-e, -i, -V, --if-exists, -? 等）を列挙する。 3. 接続関連のオプション（-h, -p, -U, -w, -W）を列挙する。 4. PACKAGE_BUGREPORT および PACKAGE_URL を用いて、バグ報告先とプロジェクトのホームページを表示する。

呼び出し元と呼び出し先

  • **呼び出し元**: 提供されたソースコード内には、本関数を呼び出している箇所は明示されていません。
  • **呼び出し先**: 標準ライブラリの printf 関数を呼び出します。
  • **注意**: ツールが提示した contrib/oid2name/oid2name.c 内の help 関数は同名ですが、本シンボルとは別個の実装です。

関連データ構造

  • なし

失敗モードと境界条件

  • 標準出力への書き込みが失敗した場合（パイプのクローズ等）、printf の振る舞いに依存します。

保守担当者向けメモ

  • 本関数は dropuser のオプション仕様が変更された際に、必ず更新する必要があります。
  • 国際化対応のため、メッセージは _() マクロでラップされています。
