RoleRecurseType

定義

  • 種別: enum
  • 定義箇所: src/backend/utils/adt/acl.c:75

目的

ロールメンバーシップの再帰的探索において、探索の条件や目的を区別するために使用される列挙型です。

シグネチャまたは形状

enum RoleRecurseType
{
    ROLERECURSE_MEMBERS = 0,    /* 無条件に再帰 */
    ROLERECURSE_PRIVS = 1,      /* 継承可能な権限を通じて再帰 */
    ROLERECURSE_SETROLE = 2     /* set_option を持つ権限を通じて再帰 */
};

入力と出力

  • 入力: なし（列挙値の定義）
  • 出力: なし

前提条件と不変条件

  • roles_is_member_of 関数において、この型に基づいたキャッシュが個別に管理されます。

アルゴリズムまたは振る舞い

  • ROLERECURSE_MEMBERS: メンバーシップ関係を無条件に辿ります。
  • ROLERECURSE_PRIVS: 権限継承のルールに従い、継承可能な権限を持つロールを辿ります。
  • ROLERECURSE_SETROLE: SET ROLE コマンドに関連する、set_option が有効な権限を辿ります。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/utils/adt/acl.c 内の roles_is_member_of 関数（5153行目）が引数としてこの型を受け取ります。
  • 呼び出し先: なし（列挙型定義のため）。

関連データ構造

  • roles_is_member_of: この型を使用してロールの再帰的解決を行う関数。

失敗モードと境界条件

  • 該当なし。

保守担当者向けメモ

  • 本型は roles_is_member_of のキャッシュ戦略と密接に関連しています。新しい再帰タイプを追加する場合は、キャッシュの分離と整合性に注意してください。
