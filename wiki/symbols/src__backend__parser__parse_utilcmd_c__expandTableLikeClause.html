expandTableLikeClause

定義

  • 種別: function
  • 定義箇所: src/backend/parser/parse_utilcmd.c:1343

目的

CREATE TABLE ... (LIKE ...) 構文において、参照元テーブルから列のデフォルト値や CHECK 制約を抽出し、新規作成テーブルに適用するためのコマンドリストを生成する。

シグネチャまたは形状

List *expandTableLikeClause(RangeVar *heapRel, TableLikeClause *table_like_clause)

入力と出力

  • **入力**:
  • heapRel: 新規作成されるテーブルの RangeVar。
  • table_like_clause: 展開対象の LIKE 句情報（参照元テーブルの OID およびオプションを含む）。
  • **出力**:
  • List *: 実行すべき AlterTableCmd および CommentStmt のリスト。

前提条件と不変条件

  • table_like_clause->relationOid は有効な OID であること。
  • 呼び出し元で、参照元テーブルおよび新規テーブルに対する適切なロックが既に取得されていること。
  • transformTableLikeClause によって事前処理が完了していること。

アルゴリズムまたは振る舞い

1. 参照元テーブルと新規テーブルを relation_open で開く。 2. build_attrmap_by_name を使用し、参照元と新規テーブル間の属性マッピングを作成する。 3. **デフォルト値の処理**: CREATE_TABLE_LIKE_DEFAULTS または CREATE_TABLE_LIKE_GENERATED オプションが指定されている場合、参照元テーブルの各列を走査し、デフォルト式を新規テーブルの列番号に対応させて AT_CookedColumnDefault コマンドを生成する。 4. **制約の処理**: CREATE_TABLE_LIKE_CONSTRAINTS オプションが指定されている場合、参照元の CHECK 制約を走査し、属性番号を変換した上で AT_AddConstraint コマンドを生成する。 5. **コメントの処理**: CREATE_TABLE_LIKE_COMMENTS オプションが指定されている場合、制約に対するコメントを抽出し CommentStmt を生成する。 6. 生成されたコマンドのリストを返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/tcop/utility.c の 1229 行目付近から呼び出される。
  • **呼び出し先**: relation_open, relation_openrv, build_attrmap_by_name, TupleDescGetDefault, map_variable_attnos, makeNode, lappend, stringToNode, nodeToString, GetComment, get_relation_constraint_oid 等。

関連データ構造

  • RangeVar: テーブルの識別情報。
  • TableLikeClause: LIKE 句のオプションと参照元 OID。
  • AlterTableCmd: ALTER TABLE 操作を表現するノード。
  • AttrMap: 属性番号の変換テーブル。

失敗モードと境界条件

  • OidIsValid が偽の場合、エラーを投げる。
  • デフォルト式や制約内に「テーブル全体の参照（whole-row reference）」が含まれている場合、将来的なテーブル構造の乖離を防ぐため ERRCODE_FEATURE_NOT_SUPPORTED でエラーを投げる。
  • 参照元テーブルのデフォルト式が見つからない場合、エラーを投げる。

保守担当者向けメモ

  • 本関数は CREATE TABLE の実行フェーズで呼び出される。
  • map_variable_attnos を使用して属性番号を変換する際、whole-row 参照を拒否するロジックは、他の類似機能（ALTER TABLE 等）と一貫性を持たせる必要がある。
