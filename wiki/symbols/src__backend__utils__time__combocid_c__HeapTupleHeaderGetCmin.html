HeapTupleHeaderGetCmin

定義

  • 種別: function
  • 定義箇所: src/backend/utils/time/combocid.c:103

目的

タプルヘッダーから、そのタプルを挿入したトランザクションのコマンドID（cmin）を取得します。タプルが「Combo CID」を使用している場合、適切な変換を経て実値を返します。

シグネチャまたは形状

CommandId HeapTupleHeaderGetCmin(const HeapTupleHeaderData *tup)

入力と出力

  • **入力**: tup (対象タプルのヘッダー構造体へのポインタ)
  • **出力**: CommandId (タプルの挿入時コマンドID)

前提条件と不変条件

  • tup の t_infomask に HEAP_MOVED フラグが設定されていないこと。
  • tup の xmin が現在のトランザクションIDと一致していること。

アルゴリズムまたは振る舞い

1. HeapTupleHeaderGetRawCommandId を用いて、ヘッダーから生のコマンドIDを取得します。 2. タプルの t_infomask を確認し、HEAP_COMBOCID フラグが立っているか判定します。 3. フラグが立っている場合、GetRealCmin を呼び出し、Combo CIDを本来のコマンドIDに変換して返します。 4. フラグが立っていない場合、取得した生のコマンドIDをそのまま返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/access/heap/heapam.c
  • src/backend/access/heap/heapam_handler.c
  • src/backend/access/heap/heapam_visibility.c
  • src/backend/utils/time/combocid.c
  • **呼び出し先**:
  • HeapTupleHeaderGetRawCommandId (マクロ)
  • TransactionIdIsCurrentTransactionId (マクロ)
  • GetRealCmin (内部関数)

関連データ構造

  • HeapTupleHeaderData: タプルのヘッダー情報。
  • CommandId: コマンドIDを表す型。

失敗モードと境界条件

  • 前提条件（HEAP_MOVED の不在、xmin の一致）が満たされない場合、Assert によりプログラムが停止します。

保守担当者向けメモ

  • 本関数は HEAP_COMBOCID フラグの有無によって挙動が分岐します。Combo CIDのロジックを変更する際は、GetRealCmin との整合性に注意してください。
