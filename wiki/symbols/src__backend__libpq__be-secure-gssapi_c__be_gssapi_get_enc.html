be_gssapi_get_enc

定義

  • 種別: function
  • 定義箇所: src/backend/libpq/be-secure-gssapi.c:755

目的

指定された接続（Port）において、GSSAPIによる通信の暗号化が有効であるかどうかを判定し、その状態を返します。

シグネチャまたは形状

bool be_gssapi_get_enc(Port *port);

入力と出力

  • **入力**: Port *port - 接続情報を保持する構造体へのポインタ。
  • **出力**: bool - 暗号化が有効な場合は true、無効または接続情報が不完全な場合は false を返します。

前提条件と不変条件

  • port が NULL である場合、または port->gss が初期化されていない場合は false を返します。
  • 戻り値は port->gss->enc の値に依存します。

アルゴリズムまたは振る舞い

1. port ポインタが有効か、および port->gss 構造体が割り当てられているかを確認します。 2. いずれかが無効な場合は false を返します。 3. 上記のチェックを通過した場合、port->gss->enc の値をそのまま返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/activity/backend_status.c: lgssstatus.gss_enc の設定に使用。
  • src/backend/utils/init/postinit.c: 接続初期化時のログ出力に使用。
  • **呼び出し先**:
  • 本関数は直接的な関数呼び出しを行いません。

関連データ構造

  • Port: 接続状態を管理する構造体。gss メンバを通じてGSSAPI関連の情報を保持します。

失敗モードと境界条件

  • port が NULL の場合、安全に false を返します。
  • port->gss が NULL の場合、安全に false を返します。

保守担当者向けメモ

  • 本関数は、GSSAPIセッションの状態を外部から参照するためのアクセサです。
  • port->gss の構造定義については libpq/libpq-be.h および libpq/be-gssapi-common.h を参照してください。
