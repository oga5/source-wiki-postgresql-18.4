pg_hmac_errno

定義

  • 種別: enum
  • 定義箇所: src/common/hmac_openssl.c:51

目的

OpenSSL を使用した HMAC 計算処理において発生しうるエラー状態を識別するために定義された列挙型です。

シグネチャまたは形状

typedef enum pg_hmac_errno
{
    PG_HMAC_ERROR_NONE = 0,
    PG_HMAC_ERROR_DEST_LEN,
    PG_HMAC_ERROR_OPENSSL,
} pg_hmac_errno;

入力と出力

  • 本シンボル自体は列挙型の定義であり、関数ではありません。
  • HMAC 計算処理の結果として、この型の値がエラーコードとして返されます。

前提条件と不変条件

  • PG_HMAC_ERROR_NONE はエラーが発生していない正常な状態を示します。
  • PG_HMAC_ERROR_DEST_LEN は出力バッファのサイズ不足を示唆します。
  • PG_HMAC_ERROR_OPENSSL は OpenSSL 内部でエラーが発生したことを示します。

アルゴリズムまたは振る舞い

  • HMAC 計算の各ステップ（初期化、更新、終了）において、エラー発生時にこの列挙型のいずれかの値が設定されます。

呼び出し元と呼び出し先

  • 呼び出し元: src/common/hmac.c および src/common/hmac_openssl.c 内の HMAC 関連関数がこの型を使用してエラー状態を保持します。
  • 呼び出し先: 情報なし。

関連データ構造

  • pg_hmac_ctx: HMAC 計算の状態を保持する構造体。

失敗モードと境界条件

  • PG_HMAC_ERROR_DEST_LEN: 最終的な MAC 値を格納するバッファが不足している場合に発生します。
  • PG_HMAC_ERROR_OPENSSL: OpenSSL のエラーキューにエラーが記録されている場合に発生します。

保守担当者向けメモ

  • 本列挙型は src/common/hmac_openssl.c に特化したエラーコードを定義しています。
  • 新しいエラー条件を追加する場合は、この列挙型を拡張し、対応するエラーメッセージ変換処理を pg_hmac_error() に追加してください。
