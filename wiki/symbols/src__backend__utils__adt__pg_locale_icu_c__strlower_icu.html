strlower_icu

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/pg_locale_icu.c:384

目的

ICU（International Components for Unicode）ライブラリを使用して、指定されたロケールに基づき文字列を小文字に変換します。

シグネチャまたは形状

size_t strlower_icu(char *dest, size_t destsize, const char *src, ssize_t srclen, pg_locale_t locale)

入力と出力

  • **入力**:
  • dest: 変換結果を格納するバッファ。
  • destsize: dest バッファのサイズ。
  • src: 変換対象の文字列。
  • srclen: src の長さ（バイト単位）。
  • locale: ICUロケール情報を含む pg_locale_t 構造体。
  • **出力**:
  • 戻り値: 変換後の文字列の長さ（バイト単位）。

前提条件と不変条件

  • icu_to_uchar および icu_convert_case が内部で呼び出され、ICUの UChar 形式への変換とケース変換が行われます。
  • 変換に使用される一時バッファ（buff_uchar, buff_conv）は、処理終了後に pfree で解放されます。

アルゴリズムまたは振る舞い

1. icu_to_uchar を呼び出し、データベースエンコーディングの文字列を UChar 形式に変換します。 2. icu_convert_case を呼び出し、ICUの u_strToLower 関数を用いて小文字変換を実行します。 3. icu_from_uchar を呼び出し、変換結果をデータベースエンコーディングに戻して dest に格納します。 4. 確保した一時バッファを解放し、結果の長さを返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/adt/pg_locale.c 内のロケール管理ロジックから呼び出されます。
  • **呼び出し先**: icu_to_uchar, icu_convert_case (u_strToLower をラップ), icu_from_uchar, pfree。

関連データ構造

  • pg_locale_t: ロケール設定を保持する構造体。
  • UChar: ICUで使用されるUTF-16形式の文字型。

失敗モードと境界条件

  • ICU内部でエラーが発生した場合、icu_convert_case 等の内部関数が ereport(ERROR, ...) を発行し、PostgreSQLのトランザクションメカニズムを通じて処理が中断されます。

保守担当者向けメモ

  • 本関数は icu_convert_case を介して汎用的なケース変換ロジックを利用しています。
  • メモリ管理は palloc/pfree に依存しており、変換過程で生成される一時バッファのリークがないよう設計されています。
