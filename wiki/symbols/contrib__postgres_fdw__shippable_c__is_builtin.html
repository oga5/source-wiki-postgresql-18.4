is_builtin

定義

  • 種別: function
  • 定義箇所: contrib/postgres_fdw/shippable.c:151

目的

指定されたオブジェクトIDがPostgreSQLの組み込みオブジェクト（pg_catalog等）であるかを判定します。これにより、リモートサーバーへ安全にプッシュダウン可能かどうかの初期判断基準を提供します。

シグネチャまたは形状

bool is_builtin(Oid objectId)

入力と出力

  • **入力**: objectId (Oid) - 判定対象のデータベースオブジェクトID。
  • **出力**: bool - objectId が組み込み範囲内であれば true、それ以外であれば false を返します。

前提条件と不変条件

  • FirstGenbkiObjectId を境界値として使用します。
  • information_schema 等の動的に生成されるオブジェクトは、この判定では組み込みとして扱われません。

アルゴリズムまたは振る舞い

入力された objectId が FirstGenbkiObjectId 未満であるかを比較し、その結果を真偽値として返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/postgres_fdw/deparse.c:1159
  • contrib/postgres_fdw/shippable.c:168 (is_shippable 関数内)
  • **呼び出し先**:
  • 本関数は他の関数を呼び出しません。

関連データ構造

  • Oid: PostgreSQLのオブジェクト識別子型。

失敗モードと境界条件

  • objectId が無効な値であっても、単なる数値比較として処理されます。
  • FirstGenbkiObjectId を境界とするため、これ以上の値を持つオブジェクトは、たとえシステムカタログ由来であっても本関数では false と判定されます。

保守担当者向けメモ

  • 本関数は postgres_fdw における「出荷可能（shippable）」判定の第一段階です。
  • 判定ロジックを変更する場合は、FirstGenbkiObjectId の定義およびその意図する範囲を再確認してください。
