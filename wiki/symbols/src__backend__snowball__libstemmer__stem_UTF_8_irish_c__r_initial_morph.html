r_initial_morph

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_irish.c:261

目的

アイルランド語のステミング処理において、単語の先頭に現れる頭子音変異（initial mutation）を検出し、削除または置換することで正規化を行う。

シグネチャまたは形状

static int r_initial_morph(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング対象の文字列と現在のカーソル位置 c を含む環境構造体）
  • 出力: int（成功時に 1、マッチしなかった場合に 0、エラー発生時に負の値を返す）

前提条件と不変条件

  • z->c が単語の先頭を指している状態で呼び出されることを想定している。
  • a_0 テーブルに定義された24種類のパターンに基づいてマッチングを行う。

アルゴリズムまたは振る舞い

1. z->bra を現在のカーソル位置 z->c に設定する。 2. find_among(z, a_0, 24) を呼び出し、定義されたパターンとのマッチングを行う。 3. マッチした場合、z->ket を現在のカーソル位置に設定する。 4. among_var の値に基づき、以下のいずれかの操作を実行する：

  • case 1: slice_del(z) で該当部分を削除。
  • case 2～10: slice_from_s(z, 1, s_n) で該当部分を特定の文字列に置換。

    • 5. 処理が成功した場合は 1 を返し、マッチしなかった場合は 0 を返す。

呼び出し元と呼び出し先

  • 呼び出し元: irish_UTF_8_stem（src/backend/snowball/libstemmer/stem_UTF_8_irish.c:435 および stem_ISO_8859_1_irish.c:435）
  • 呼び出し先: find_among, slice_del, slice_from_s（いずれも Snowball ランタイムユーティリティ）

関連データ構造

  • SN_env: ステミング処理の状態を保持する構造体。
  • a_0: 頭子音変異のパターンを格納した静的テーブル。

失敗モードと境界条件

  • slice_del または slice_from_s が負の値を返した場合、そのエラーコードをそのまま呼び出し元に伝播させる。
  • find_among がマッチしなかった場合、文字列操作を行わずに 0 を返す。
