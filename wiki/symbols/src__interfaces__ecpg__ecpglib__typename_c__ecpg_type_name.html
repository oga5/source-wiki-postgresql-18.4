ecpg_type_name

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/ecpglib/typename.c:16

目的

ECPG内部の型列挙子（ECPGttype）を、対応するC言語の型名を表す文字列に変換します。

シグネチャまたは形状

const char *ecpg_type_name(enum ECPGttype typ)

入力と出力

  • **入力**: enum ECPGttype 型の列挙値（ECPG内部の型識別子）。
  • **出力**: 対応する型名を示す const char * 型の文字列（例: "int", "double", "varchar" など）。

前提条件と不変条件

  • 入力された typ が ECPGttype で定義された既知の列挙値である必要があります。
  • 未知の型が渡された場合、関数は abort() を呼び出し、プログラムを終了させます。

アルゴリズムまたは振る舞い

1. switch-case 文を使用して、入力された typ を評価します。 2. 各 ECPGttype に対応する文字列リテラルを返します。 3. ECPGt_char と ECPGt_string は共に "char" を返します。 4. 定義されていない列挙値が渡された場合、default ケースにより abort() が実行されます。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/interfaces/ecpg/ecpglib/data.c
  • src/interfaces/ecpg/ecpglib/execute.c
  • src/interfaces/ecpg/preproc/type.c
  • **呼び出し先**:
  • abort()（標準ライブラリ関数）

関連データ構造

  • enum ECPGttype（ecpgtype.h で定義）: ECPGで使用される型識別子の列挙体。

失敗モードと境界条件

  • **失敗モード**: 未定義の ECPGttype 値が渡された場合、abort() によりプロセスが異常終了します。
  • **境界条件**: ECPGt_char と ECPGt_char_variable はどちらも "char" を返します。

保守担当者向けメモ

  • 新しい型をECPGに追加した際は、本関数の switch 文に新しい case を追加する必要があります。
  • 本関数は状態を持たない純粋な変換関数です。
