MemoryContextAllocZero

定義

  • 種別: function
  • 定義箇所: src/backend/utils/mmgr/mcxt.c:1224

目的

指定されたメモリコンテキスト内でメモリを割り当て、その領域をゼロクリアして返す。

シグネチャまたは形状

void *MemoryContextAllocZero(MemoryContext context, Size size)

入力と出力

  • **入力**:
  • context: 割り当て対象の MemoryContext。
  • size: 割り当てるバイト数。
  • **出力**:
  • ゼロクリアされたメモリ領域へのポインタ。

前提条件と不変条件

  • context が有効であること（MemoryContextIsValid による検証）。
  • 現在の実行状態がクリティカルセクション内ではないこと（AssertNotInCriticalSection による検証）。

アルゴリズムまたは振る舞い

1. context->isReset を false に設定する。 2. context->methods->alloc を呼び出し、指定されたサイズのメモリを割り当てる。 3. Valgrind 向けにメモリ割り当てを通知する（VALGRIND_MEMPOOL_ALLOC）。 4. MemSetAligned を使用して、割り当てられたメモリ領域全体を 0 で埋める。 5. 確保したメモリ領域のポインタを返す。

呼び出し元と呼び出し先

  • **呼び出し先**:
  • context->methods->alloc: コンテキスト固有の割り当て実装。
  • **呼び出し元**:
  • contrib/pg_trgm/trgm_regexp.c
  • contrib/pgcrypto/openssl.c
  • contrib/test_decoding/test_decoding.c
  • src/backend/access/common/indextuple.c
  • src/backend/access/common/session.c
  • src/backend/access/gist/gistbuildbuffers.c
  • src/backend/access/spgist/spgutils.c
  • src/backend/access/transam/xact.c
  • src/backend/access/transam/xloginsert.c

関連データ構造

  • MemoryContext: メモリ割り当ての管理単位。
  • MemoryContextMethods: コンテキスト型ごとのメソッドテーブル。

失敗モードと境界条件

  • メモリ不足（OOM）が発生した場合、context->methods->alloc 内でエラーが報告される可能性がある。

保守担当者向けメモ

  • 本関数は palloc0 の内部実装の一部として機能する。
  • ゼロクリア処理には MemSetAligned を使用しており、アライメントを考慮した効率的な書き込みが行われる。
