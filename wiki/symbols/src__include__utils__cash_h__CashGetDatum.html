CashGetDatum

定義

  • 種別: static inline function
  • 定義箇所: src/include/utils/cash.h:26

目的

Cash 型の値を PostgreSQL の汎用データ型である Datum に変換する。

シグネチャまたは形状

static inline Datum CashGetDatum(Cash X)

入力と出力

  • **入力**: Cash X （通貨データを表す 64 ビット整数）
  • **出力**: Datum （PostgreSQL の関数マネージャが扱う汎用データ形式）

前提条件と不変条件

  • Cash 型は int64 のエイリアスである。
  • 本関数は Int64GetDatum を呼び出すことで変換を実現している。

アルゴリズムまたは振る舞い

引数として受け取った Cash 型の値を、Int64GetDatum マクロを介して Datum 型へキャストして返却する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/btree_gist/btree_utils_num.c:153
  • src/include/utils/cash.h:33 （PG_RETURN_CASH マクロ内）
  • **呼び出し先**:
  • Int64GetDatum （PostgreSQL の FMGR インターフェース）

関連データ構造

  • Cash: 通貨データの内部表現（int64）。
  • Datum: PostgreSQL における値の汎用表現。

失敗モードと境界条件

  • 本関数自体は単純な型変換を行うインライン関数であり、内部でエラーを発生させることはない。
  • Cash 型の範囲外の値が渡された場合の挙動は、呼び出し元のロジックに依存する。

保守担当者向けメモ

  • 本関数は src/include/utils/cash.h で定義されており、money 型の演算結果をシステムへ戻す際の標準的なインターフェースとして使用される。
  • 変更を加える際は、Datum 型の定義および Int64GetDatum の仕様変更に追従する必要がある。
