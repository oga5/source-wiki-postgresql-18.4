l_int64_const

定義

  • 種別: static inline function
  • 定義箇所: src/include/jit/llvmjit_emit.h:74

目的

LLVM IR 生成において、64ビット整数型の定数値を表現する LLVMValueRef を簡便に作成するためのヘルパー関数です。

シグネチャまたは形状

static inline LLVMValueRef l_int64_const(LLVMContextRef lc, int64 i)

入力と出力

  • **入力**:
  • lc (LLVMContextRef): 定数を作成する対象の LLVM コンテキスト。
  • i (int64): 生成する定数の値。
  • **出力**:
  • LLVMValueRef: 指定された値と64ビット整数型を持つ LLVM 定数オブジェクト。

前提条件と不変条件

  • lc は有効な LLVMContextRef である必要があります。
  • 本関数は USE_LLVM が定義されている環境でのみ利用可能です。

アルゴリズムまたは振る舞い

1. 指定されたコンテキスト lc に対して LLVMInt64TypeInContext を呼び出し、64ビット整数型を取得します。 2. LLVMConstInt を使用して、値 i を持つ定数オブジェクトを生成します。この際、符号なしフラグは false に設定されます。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/jit/llvm/llvmjit_expr.c: sizeof 演算の結果を LLVM 定数に変換する際に使用されています。
  • **呼び出し先**:
  • LLVMInt64TypeInContext (LLVM C API)
  • LLVMConstInt (LLVM C API)

関連データ構造

  • LLVMValueRef: LLVM IR 上の値を表す不透明なポインタ型。
  • LLVMContextRef: LLVM のコンパイルコンテキスト。

失敗モードと境界条件

  • LLVM C API の仕様に依存します。無効なコンテキストが渡された場合の動作は LLVM 側の規定に従います。

保守担当者向けメモ

  • 本関数はインライン関数であり、呼び出し側のコンテキストで直接展開されます。
  • LLVM C API のバージョンアップに伴い、定数生成の仕様が変更された場合は本関数を修正してください。
