AtEOXact_PgStat_Database

定義

  • **種別**: 関数
  • **定義箇所**: src/backend/utils/activity/pgstat_database.c:284

目的

トランザクション終了時（コミットまたはロールバック）に、データベース単位のトランザクション統計カウンタを更新する。

シグネチャまたは形状

void AtEOXact_PgStat_Database(bool isCommit, bool parallel)

入力と出力

  • **入力**:
  • isCommit (bool): true の場合コミットとして、false の場合ロールバックとして計上する。
  • parallel (bool): パラレルワーカーのトランザクションであるかを示すフラグ。
  • **出力**: なし（副作用としてグローバル変数を更新）。

前提条件と不変条件

  • parallel が true の場合、統計情報の更新は行われない。
  • 統計情報はブール値ではなくカウンタとして保持される。これは、報告メッセージが即座に送信されない可能性があるためである。

アルゴリズムまたは振る舞い

1. parallel 引数が false であることを確認する。 2. isCommit が true であれば、グローバル変数 pgStatXactCommit をインクリメントする。 3. isCommit が false であれば、グローバル変数 pgStatXactRollback をインクリメントする。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/activity/pgstat_xact.c:44
  • **呼び出し先**:
  • 本関数は直接的な関数呼び出しを行わない（グローバル変数の更新のみ）。

関連データ構造

  • pgStatXactCommit (グローバル変数)
  • pgStatXactRollback (グローバル変数)

失敗モードと境界条件

  • パラレルワーカーのトランザクションは統計対象外として除外される。

保守担当者向けメモ

  • 本関数はトランザクション終了処理の一部として呼び出される。
  • 統計情報の報告タイミングと実際のトランザクション終了タイミングが一致しない場合があるため、カウンタ形式での保持が必須となっている。
