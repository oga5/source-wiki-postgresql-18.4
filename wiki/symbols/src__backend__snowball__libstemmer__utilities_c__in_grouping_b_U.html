in_grouping_b_U

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/utilities.c:129

目的

Snowball ステミングアルゴリズムにおいて、現在のカーソル位置から後方（逆方向）へ向かって UTF-8 文字を読み込み、その文字が指定された文字集合（グループ）に含まれるかを判定する。

シグネチャまたは形状

int in_grouping_b_U(struct SN_env * z, const unsigned char * s, int min, int max, int repeat)

入力と出力

  • **入力**:
  • z: ステミング環境を保持する SN_env 構造体へのポインタ。
  • s: 文字集合をビットマップ形式で保持するバイト配列。
  • min, max: 判定対象となる文字コードの範囲。
  • repeat: 0 以外の場合、グループに一致し続ける限りカーソルを後方へ移動し続ける。
  • **出力**:
  • 成功時（一致しなかった場合）: 最後に読み込んだ文字のバイト長（正の整数）。
  • 失敗時（UTF-8 デコード失敗）: -1。
  • 成功時（グループに一致し、repeat が 0 の場合）: 0。

前提条件と不変条件

  • z->c は現在のカーソル位置を指し、z->lb は後方検索の境界を示す。
  • s は (max - min + 1) ビットをカバーする十分なサイズを持つ必要がある。

アルゴリズムまたは振る舞い

1. get_b_utf8 を呼び出し、現在のカーソル位置から後方へ 1 文字デコードする。 2. デコードに失敗（w == 0）した場合は -1 を返す。 3. 取得した文字 ch が min 未満または max を超える場合、あるいはビットマップ s 上で該当ビットが立っていない場合は、その文字のバイト長 w を返して終了する。 4. 一致した場合、カーソル z->c を w 分だけ後方へ進める。 5. repeat が真であれば、1 に戻り処理を繰り返す。 6. repeat が偽、または一致しなくなった時点で 0 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • stem_UTF_8_danish.c
  • stem_UTF_8_english.c
  • stem_UTF_8_estonian.c
  • **呼び出し先**:
  • get_b_utf8 (同一ファイル内)

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • among: 辞書検索用構造体（本関数は直接使用しないが、モジュール内で関連）。

失敗モードと境界条件

  • get_b_utf8 が不正な UTF-8 シーケンスを検出した場合、-1 を返して処理を中断する。
  • カーソルが z->lb に達して文字を読み込めない場合、get_b_utf8 の仕様に従い w が 0 となり、本関数も -1 を返す。

保守担当者向けメモ

  • _U サフィックスは UTF-8 対応版であることを示す。非 UTF-8 環境では使用しないこと。
  • s 配列のビット操作は ch >> 3（バイトオフセット）と ch & 0x7（ビットオフセット）で行われており、文字コードの範囲が min から max であることに依存している。
