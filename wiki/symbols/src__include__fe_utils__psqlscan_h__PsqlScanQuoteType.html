PsqlScanQuoteType

定義

  • 種別: enum
  • 定義箇所: src/include/fe_utils/psqlscan.h:52

目的

変数置換や値の取得を行う際、対象の値をどのような形式でクォート（引用符付け）するかを指定するために使用されます。

シグネチャまたは形状

typedef enum
{
    PQUOTE_PLAIN,        /* そのままの値を返す */
    PQUOTE_SQL_LITERAL,  /* 有効なSQLリテラルとなるようクォートを追加 */
    PQUOTE_SQL_IDENT,    /* SQL識別子として必要な場合にクォート */
    PQUOTE_SHELL_ARG,    /* シェルコマンド内で安全に使用できるようクォート */
} PsqlScanQuoteType;

入力と出力

  • 入力: 変数取得関数（psql_get_variable 等）の引数として渡されます。
  • 出力: 指定されたクォート規則に従って加工された文字列が、呼び出し元へ返されます。

前提条件と不変条件

  • 各列挙子は、特定のコンテキスト（SQLリテラル、識別子、シェル引数）におけるエスケープや引用符付与の要件を定義しています。

アルゴリズムまたは振る舞い

  • PQUOTE_PLAIN: 変換を行わず、値をそのまま返します。
  • PQUOTE_SQL_LITERAL: SQLリテラルとして解釈されるよう、適切にクォートを付与します。
  • PQUOTE_SQL_IDENT: 識別子として曖昧さがある場合や特殊文字が含まれる場合にクォートを付与します。
  • PQUOTE_SHELL_ARG: シェル環境で安全に引数として渡せるよう、クォート処理を行います。

呼び出し元と呼び出し先

  • 呼び出し元（直接的な証拠）:
  • src/bin/psql/common.c: psql_get_variable 関数
  • src/bin/psql/common.h: psql_get_variable の宣言
  • src/include/fe_utils/psqlscan.h: PsqlScanCallbacks 構造体内の get_variable 関数ポインタ
  • src/include/fe_utils/psqlscan_int.h: 内部的な関数定義
  • 呼び出し先: 本シンボル自体は列挙型であり、関数ではないため呼び出し先はありません。

関連データ構造

  • PsqlScanCallbacks: 変数置換処理において、本型を引数として受け取るコールバック関数を保持します。

失敗モードと境界条件

  • 該当する情報なし。

保守担当者向けメモ

  • 本型は psql の変数置換メカニズムと密接に関連しています。新しいクォート形式が必要な場合は、この列挙型を拡張し、対応する処理を psql_get_variable 等の実装側に反映させる必要があります。
