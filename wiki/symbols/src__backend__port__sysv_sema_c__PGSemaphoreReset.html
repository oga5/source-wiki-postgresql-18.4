PGSemaphoreReset

定義

  • 種別: function
  • 定義箇所: src/backend/port/sysv_sema.c:422

目的

指定されたセマフォの値を 0 にリセットします。

シグネチャまたは形状

void PGSemaphoreReset(PGSemaphore sema)

入力と出力

  • **入力**: sema (PGSemaphore) - リセット対象のセマフォ構造体へのポインタ。
  • **出力**: なし。

前提条件と不変条件

  • sema は有効な PGSemaphore 構造体を指している必要があります。
  • 内部的に sema->semId および sema->semNum が初期化済みである必要があります。

アルゴリズムまたは振る舞い

1. 引数として受け取った sema 構造体から、セマフォ ID (semId) とセマフォ番号 (semNum) を取得します。 2. IpcSemaphoreInitialize を呼び出し、当該セマフォの値を 0 に設定します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/storage/lmgr/proc.c (プロセス管理におけるセマフォのリセット)
  • src/backend/port/posix_sema.c および src/backend/port/win32_sema.c (各プラットフォーム実装におけるラッパー)
  • **呼び出し先**:
  • IpcSemaphoreInitialize (System V セマフォの初期化関数)

関連データ構造

  • PGSemaphore: セマフォの識別情報（ID とインデックス）を保持する構造体。

失敗モードと境界条件

  • IpcSemaphoreInitialize が内部で semctl システムコールを呼び出すため、カーネルリソースの枯渇や権限不足により失敗する可能性があります。

保守担当者向けメモ

  • 本関数はプラットフォーム非依存のインターフェースとして定義されていますが、SysV 実装では IpcSemaphoreInitialize への単なる転送処理となっています。
  • 他のプラットフォーム（POSIX, Win32）でも同名の関数が定義されており、ビルド時に適切な実装が選択されます。
