initReadOnlyStringInfo

定義

  • 種別: static inline function
  • 定義箇所: src/include/lib/stringinfo.h:156

目的

既存のメモリ領域を StringInfo 構造体でラップし、読み取り専用のバッファとして初期化します。

シグネチャまたは形状

void initReadOnlyStringInfo(StringInfo str, char *data, int len)

入力と出力

  • str: 初期化対象の StringInfoData 構造体へのポインタ。
  • data: 読み取り対象となる既存のバッファへのポインタ。
  • len: バッファの有効なデータ長。
  • 出力: なし（str が指す構造体の各フィールドが更新されます）。

前提条件と不変条件

  • str は有効なメモリ領域を指している必要があります。
  • maxlen が 0 に設定されるため、この関数で初期化された StringInfo に対して append 系関数や resetStringInfo を呼び出すと、アサーションエラーが発生します。

アルゴリズムまたは振る舞い

1. str->data に data を代入します。 2. str->len に len を代入します。 3. str->maxlen を 0 に設定し、読み取り専用状態を確立します。 4. str->cursor を 0 に初期化します。

  • 呼び出し先: なし（インライン関数であり、外部関数呼び出しを含みません）。

関連データ構造

  • StringInfoData: StringInfo が指す実体。
  • StringInfo: StringInfoData へのポインタ型。

失敗モードと境界条件

  • str が NULL の場合、セグメンテーション違反が発生します。
  • 読み取り専用モードであるため、書き込み操作を試みるとアサーションエラーが発生します。

保守担当者向けメモ

  • 本関数は maxlen = 0 を設定することで読み取り専用を強制します。既存のバッファをラップする際は、そのバッファの生存期間が StringInfo の使用期間よりも長いことを確認してください。
