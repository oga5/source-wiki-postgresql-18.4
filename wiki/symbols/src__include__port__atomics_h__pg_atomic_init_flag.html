pg_atomic_init_flag

定義

  • 種別: static inline function
  • 定義箇所: src/include/port/atomics.h:169

目的

アトミックフラグ（pg_atomic_flag）を初期化する。

シグネチャまたは形状

void pg_atomic_init_flag(volatile pg_atomic_flag *ptr)

入力と出力

  • 入力: ptr - 初期化対象となる pg_atomic_flag 型変数へのポインタ。
  • 出力: なし（void）。

前提条件と不変条件

  • ptr は volatile 修飾された pg_atomic_flag を指している必要がある。
  • 操作対象のメモリは適切にアライメントされている必要がある。

アルゴリズムまたは振る舞い

内部的に pg_atomic_init_flag_impl(ptr) を呼び出し、プラットフォーム固有の実装へ処理を委譲する。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/backend/postmaster/autovacuum.c:3409
  • src/test/regress/regress.c:501
  • 呼び出し先:
  • pg_atomic_init_flag_impl（プラットフォーム固有の実装）

関連データ構造

  • pg_atomic_flag: アトミック操作に使用されるフラグ型。

失敗モードと境界条件

  • ポインタ ptr が NULL の場合の挙動については、本ソース excerpt 内では定義されていない。

保守担当者向けメモ

  • 本関数は atomics.h における抽象化レイヤーであり、実際の初期化ロジックは pg_atomic_init_flag_impl に依存する。
  • 64bit アトミック操作のフォールバックが必要な環境では、src/backend/port/atomics.c の実装が参照される可能性がある。
