pg_sha512_init

定義

  • 種別: function
  • 定義箇所: src/common/sha2.c:604

目的

SHA-512 ハッシュ計算を行うためのコンテキスト構造体を初期化し、計算開始可能な状態にリセットします。

シグネチャまたは形状

void pg_sha512_init(pg_sha512_ctx *context)

入力と出力

  • **入力**: pg_sha512_ctx *context（初期化対象のコンテキスト構造体へのポインタ）
  • **出力**: なし（副作用として context が書き換えられます）

前提条件と不変条件

  • context が NULL の場合、関数は何も行わず直ちに復帰します。
  • 正常実行後、context->state は sha512_initial_hash_value で初期化されます。
  • context->buffer はゼロクリアされ、context->bitcount は 0 にリセットされます。

アルゴリズムまたは振る舞い

1. 引数 context が NULL かどうかを確認し、該当する場合は終了します。 2. memcpy を使用して、SHA-512 の初期ハッシュ値を context->state にコピーします。 3. memset を使用して、内部バッファ context->buffer を PG_SHA512_BLOCK_LENGTH 分だけゼロで埋めます。 4. ビットカウンタ context->bitcount の両要素を 0 に設定します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/common/cryptohash.c:123 からの呼び出しが確認されています。
  • **呼び出し先**: 標準ライブラリの memcpy および memset を使用します。

関連データ構造

  • pg_sha512_ctx: SHA-512 の計算状態を保持する構造体（src/common/sha2_int.h で定義）。

失敗モードと境界条件

  • context に NULL ポインタが渡された場合、安全に無視されます。

保守担当者向けメモ

  • 本関数は Aaron D. Gifford 氏による実装に基づいています。
  • pg_sha512_ctx の内部構造に変更を加える場合は、sha512_initial_hash_value の整合性と bitcount の管理方法に注意してください。
