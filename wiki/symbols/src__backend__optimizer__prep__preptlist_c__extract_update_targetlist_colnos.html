extract_update_targetlist_colnos

定義

  • 種別: function
  • 定義箇所: src/backend/optimizer/prep/preptlist.c:347

目的

UPDATE文のターゲットリスト（tlist）を正規化し、更新対象となる列の元の属性番号を抽出します。

シグネチャまたは形状

List *extract_update_targetlist_colnos(List *tlist)

入力と出力

  • **入力**: tlist (更新対象の TargetEntry を含むリスト)
  • **出力**: 更新対象列の元の属性番号（resno）を格納した整数リスト。

前提条件と不変条件

  • 入力される tlist は TargetEntry のリストである必要があります。
  • resjunk が false であるエントリのみが更新対象として抽出されます。

アルゴリズムまたは振る舞い

1. update_colnos リストを初期化し、再ナンバリング用のカウンタ nextresno を 1 に設定します。 2. tlist 内の各 TargetEntry を走査します。 3. resjunk フラグが立っていないエントリについて、その時点の resno を update_colnos に追加します。 4. 各エントリの resno を nextresno の値で上書きし、nextresno をインクリメントします。 5. 抽出した元の属性番号リストを返します。

呼び出し元と呼び出し先

  • **呼び出し先**:
  • 本関数内での直接的な関数呼び出しはありません（lfirst, lappend_int 等のリスト操作マクロ/関数のみ使用）。

関連データ構造

  • TargetEntry: ターゲットリストの各要素を構成する構造体。
  • List: PostgreSQLの汎用リスト構造。

失敗モードと境界条件

  • 空のリストが渡された場合、空のリストを返します。
  • resjunk のみが含まれるリストの場合、空のリストを返しますが、resno の再割り当ては行われます。

保守担当者向けメモ

  • 本関数はターゲットリストの物理的な順序を再定義するため、呼び出し後に元の属性番号を参照する必要がある場合は、戻り値の update_colnos を保持しておく必要があります。
