FuncLookupError

定義

  • 種別: enum
  • 定義箇所: src/backend/parser/parse_func.c:39

目的

関数検索処理において発生したエラーの種類を識別するために使用される列挙型です。

シグネチャまたは形状

typedef enum {
    FUNCLOOKUP_NOSUCHFUNC,
    FUNCLOOKUP_AMBIGUOUS,
} FuncLookupError;

入力と出力

  • 本シンボル自体は列挙値の定義であり、関数ではありません。
  • 検索処理の結果として、エラーが発生した場合に呼び出し元へエラー種別を伝達するために使用されます。

前提条件と不変条件

  • 検索対象の関数が見つからない場合は FUNCLOOKUP_NOSUCHFUNC が使用されます。
  • 複数の候補が見つかり、型解決のヒューリスティックで一意に特定できない場合は FUNCLOOKUP_AMBIGUOUS が使用されます。

アルゴリズムまたは振る舞い

  • 関数検索関数（LookupFuncWithArgs 等）の引数として FuncLookupError * 型のポインタを渡すことで、検索失敗時の詳細な理由を呼び出し元が受け取れるように設計されています。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/parser/parse_func.c 内の関数検索ロジック（LookupFuncWithArgs 等）によって参照されています。
  • 呼び出し先: 本シンボルは列挙型であるため、呼び出し先はありません。

関連データ構造

  • src/include/parser/parse_func.h: 本モジュールの公開インターフェースとして関連します。

失敗モードと境界条件

  • 検索対象が存在しない場合、または曖昧な場合にのみ値が設定されます。正常終了時にはこれらの値は設定されません。

保守担当者向けメモ

  • 新たな検索エラー種別を追加する必要がある場合は、この列挙型を拡張してください。
  • 呼び出し元でこのエラーコードをどのようにハンドリングしているか（エラーメッセージの生成など）を確認する際は、src/backend/parser/parse_func.c 内の参照箇所を参照してください。
