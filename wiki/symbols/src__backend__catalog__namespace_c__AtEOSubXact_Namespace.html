AtEOSubXact_Namespace

定義

  • 種別: function
  • 定義箇所: src/backend/catalog/namespace.c:4560

目的

サブトランザクションの終了（コミットまたはアボート）時に、一時名前空間の状態を管理・クリーンアップします。

シグネチャまたは形状

void AtEOSubXact_Namespace(bool isCommit, SubTransactionId mySubid, SubTransactionId parentSubid)

入力と出力

  • isCommit: サブトランザクションがコミットされる場合は true、アボートされる場合は false。
  • mySubid: 現在終了しようとしているサブトランザクションのID。
  • parentSubid: 親トランザクションまたは親サブトランザクションのID。
  • 出力: なし（副作用としてグローバル変数および MyProc の状態を更新）。

前提条件と不変条件

  • myTempNamespaceSubID が現在のサブトランザクションIDと一致する場合のみ、状態更新が行われます。

アルゴリズムまたは振る舞い

1. myTempNamespaceSubID が現在のサブトランザクションIDと一致するか確認します。 2. **コミット時 (isCommit == true)**:

  • myTempNamespaceSubID を parentSubid に更新し、一時名前空間の所有権を親へ引き継ぎます。

    • 3. **アボート時 (isCommit == false)**:

  • myTempNamespaceSubID を InvalidSubTransactionId にリセットします。
  • myTempNamespace および myTempToastNamespace を InvalidOid にリセットします。
  • baseSearchPathValid および searchPathCacheValid を false に設定し、検索パスの再構築を強制します。
  • MyProc->tempNamespaceId を InvalidOid にリセットします。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/access/transam/xact.c 内のサブトランザクション終了処理（コミット時およびアボート時）。
  • 呼び出し先: 直接的な関数呼び出しは含まれていませんが、グローバル変数および MyProc 構造体へのアクセスを行います。

関連データ構造

  • myTempNamespace (グローバル変数)
  • myTempToastNamespace (グローバル変数)
  • myTempNamespaceSubID (グローバル変数)
  • baseSearchPathValid (グローバル変数)
  • searchPathCacheValid (グローバル変数)
  • MyProc (プロセス制御ブロック)

失敗モードと境界条件

  • サブトランザクションのアボート時に一時名前空間の作成が失敗していた場合、状態が適切にリセットされることを保証します。
  • アボート時は、当該サブトランザクション内で作成された名前空間内のオブジェクトが他から不可視であることを前提としています。

保守担当者向けメモ

  • 本関数はサブトランザクションのライフサイクルに密接に関連しています。一時名前空間の管理ロジックを変更する際は、AtEOXact_Namespace との整合性に注意してください。
