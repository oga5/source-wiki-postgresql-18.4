pg_u_prop_lowercase

定義

  • 種別: function
  • 定義箇所: src/common/unicode_category.c:121

目的

指定された Unicode コードポイントが「小文字（Lowercase）」プロパティを持つかどうかを判定します。

シグネチャまたは形状

bool pg_u_prop_lowercase(pg_wchar code)

入力と出力

  • **入力**: pg_wchar code（判定対象の Unicode コードポイント）
  • **出力**: bool（小文字プロパティを持つ場合は true、それ以外は false）

前提条件と不変条件

  • 入力コードポイントは pg_wchar 型である必要があります。
  • Unicode 規格の安定性ポリシーに基づき、判定に使用されるカテゴリ値は固定されています。

アルゴリズムまたは振る舞い

1. **ASCII 範囲の判定**: コードポイントが 0x80 未満の場合、unicode_opt_ascii テーブルを参照し、PG_U_PROP_LOWERCASE ビットマスクを用いて定数時間で判定します。 2. **範囲外の判定**: 0x80 以上のコードポイントに対しては、unicode_lowercase テーブルに対して range_search を実行し、該当するかを二分探索で特定します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/common/unicode/category_test.c:69

  • src/common/unicode_category.c:251
  • **呼び出し先**:

関連データ構造

  • unicode_opt_ascii: ASCII 文字のプロパティを保持する静的テーブル。
  • unicode_lowercase: 小文字プロパティを持つコードポイントの範囲を保持する静的テーブル。
  • PG_U_PROP_LOWERCASE: プロパティ判定用のビットマスク。

失敗モードと境界条件

  • 0x80 未満の入力に対してはテーブル参照による高速な判定が行われます。
  • 0x80 以上の入力に対しては、テーブル内の範囲検索結果に依存します。
