VarBitPGetDatum

定義

  • 種別: static inline function
  • 定義箇所: src/include/utils/varbit.h:56

目的

VarBit 構造体へのポインタを、PostgreSQLの内部データ表現である Datum 型に変換します。

シグネチャまたは形状

static inline Datum VarBitPGetDatum(const VarBit *X)

入力と出力

  • **入力**: const VarBit *X - ビット列データを保持する構造体へのポインタ。
  • **出力**: Datum - 変換された Datum 型の値。

前提条件と不変条件

  • 入力ポインタ X は有効な VarBit 構造体を指している必要があります。
  • VarBit は varlena 構造体と互換性があるため、この変換により Datum としての取り扱いが可能になります。

アルゴリズムまたは振る舞い

PointerGetDatum(X) を呼び出し、ポインタ値を Datum 型へキャストして返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/varbit.c (複数箇所)
  • src/include/utils/varbit.h 内の PG_RETURN_VARBIT_P(x) マクロ
  • **呼び出し先**:
  • PointerGetDatum (PostgreSQLのコアマクロ)

関連データ構造

  • VarBit: ビット列を格納する構造体。
  • Datum: PostgreSQLにおける汎用データ型。

失敗モードと境界条件

  • 入力ポインタが NULL の場合、PointerGetDatum の仕様に従い (Datum) 0 が返されます。

保守担当者向けメモ

  • 本関数はインライン化されており、パフォーマンスを考慮して設計されています。
  • VarBit 構造体のメモリレイアウトを変更する場合は、本関数が varlena 互換性を維持しているか確認してください。
