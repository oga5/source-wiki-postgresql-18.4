AlterTableSpaceOptions

定義

  • 種別: function
  • 定義箇所: src/backend/commands/tablespace.c:1014

目的

既存のテーブルスペースに関連付けられたオプション（spcoptions）を変更します。

シグネチャまたは形状

Oid AlterTableSpaceOptions(AlterTableSpaceOptionsStmt *stmt)

入力と出力

  • **入力**: AlterTableSpaceOptionsStmt *stmt（変更対象のテーブルスペース名と新しいオプション定義を含む構造体）
  • **出力**: Oid（変更されたテーブルスペースの OID）

前提条件と不変条件

  • 対象のテーブルスペースが存在すること。
  • 呼び出し元が対象テーブルスペースの所有者であること。
  • pg_tablespace システムカタログに対する RowExclusiveLock を保持すること。

アルゴリズムまたは振る舞い

1. pg_tablespace カタログをスキャンし、指定された名前のテーブルスペースを検索します。存在しない場合はエラーを発生させます。 2. object_ownercheck を呼び出し、現在のユーザーがテーブルスペースの所有者であることを確認します。所有者でない場合は aclcheck_error を発生させます。 3. transformRelOptions を使用して、既存のオプションと新しいオプションをマージ（またはリセット）し、新しい spcoptions を生成します。 4. tablespace_reloptions を呼び出し、生成されたオプションの妥当性を検証します。 5. heap_modify_tuple を使用して、pg_tablespace のタプルを更新します。 6. CatalogTupleUpdate を実行してシステムカタログを更新し、InvokeObjectPostAlterHook でフックを呼び出します。 7. 最後にスキャンを終了し、ロックを解放して OID を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/tcop/utility.c:727
  • **呼び出し先**:
  • table_open, table_beginscan_catalog, heap_getnext, table_endscan, table_close
  • ScanKeyInit, heap_getattr, heap_modify_tuple, CatalogTupleUpdate
  • transformRelOptions, tablespace_reloptions
  • object_ownercheck, aclcheck_error, InvokeObjectPostAlterHook

関連データ構造

  • AlterTableSpaceOptionsStmt: 構文解析結果を保持する構造体。
  • pg_tablespace: テーブルスペース情報を格納するシステムカタログ。

失敗モードと境界条件

  • 指定されたテーブルスペース名が見つからない場合、ERRCODE_UNDEFINED_OBJECT でエラー終了します。
  • ユーザーが所有者でない場合、権限エラーが発生します。
  • tablespace_reloptions による検証で不正なオプションが検出された場合、エラーが発生します。

保守担当者向けメモ

  • 本関数は pg_tablespace カタログの spcoptions 列のみを更新します。物理的なディレクトリ構造やシンボリックリンクの変更は行いません。
  • stmt->isReset が真の場合、オプションはリセットされます。
