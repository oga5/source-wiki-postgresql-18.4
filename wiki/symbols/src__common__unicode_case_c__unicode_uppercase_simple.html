unicode_uppercase_simple

定義

  • 種別: function
  • 定義箇所: src/common/unicode_case.c:65

目的

単一の Unicode 文字（pg_wchar）を大文字に変換します。

シグネチャまたは形状

pg_wchar unicode_uppercase_simple(pg_wchar code);

入力と出力

  • **入力**: code (変換対象の pg_wchar 型 Unicode コードポイント)
  • **出力**: 変換後の pg_wchar コードポイント。変換ルールが存在しない場合は元の code をそのまま返します。

前提条件と不変条件

  • case_map_upper テーブルを使用してマッピングを検索します。
  • 変換結果が 0 の場合はマッピングなしと見なし、入力値をそのまま返します。

アルゴリズムまたは振る舞い

1. find_case_map 関数を呼び出し、case_map_upper テーブルから code に対応する大文字変換値を検索します。 2. 検索結果が有効（0 以外）であればその値を返し、無効であれば入力された code をそのまま返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/regex/regc_pg_locale.c:560
  • src/common/unicode/case_test.c:84
  • **呼び出し先**:
  • find_case_map (内部関数)

関連データ構造

  • pg_wchar: PostgreSQL で使用されるワイド文字型。
  • case_map_upper: src/common/unicode_case_table.h で定義されている大文字変換用テーブル。

失敗モードと境界条件

  • 変換対象の文字に大文字が存在しない場合、入力値がそのまま返されます。
  • find_case_map が 0 を返す場合、変換は行われません。

保守担当者向けメモ

  • 本関数は「単純な」ケース変換のみを扱います。文脈依存の特殊マッピング（ギリシャ語の最終シグマなど）が必要な場合は、convert_case 等のより高位な関数を使用してください。
