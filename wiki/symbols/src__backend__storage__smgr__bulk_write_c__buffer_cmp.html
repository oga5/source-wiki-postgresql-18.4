buffer_cmp

定義

  • 種別: function
  • 定義箇所: src/backend/storage/smgr/bulk_write.c:224

目的

PendingWrite 構造体の配列をブロック番号（blkno）に基づいて昇順にソートするための比較関数です。

シグネチャまたは形状

static int buffer_cmp(const void *a, const void *b)

入力と出力

  • **入力**: 比較対象となる PendingWrite 構造体へのポインタ a および b。
  • **出力**: a のブロック番号が b より大きい場合は 1、小さい場合は -1 を返します。

前提条件と不変条件

  • 入力される a と b は PendingWrite 型へのポインタとしてキャスト可能であること。
  • 同一のブロック番号を持つ書き込みが存在しないこと（Assert により検証）。

アルゴリズムまたは振る舞い

1. 引数を PendingWrite 型にキャストします。 2. bufa->blkno と bufb->blkno を比較します。 3. bufa->blkno == bufb->blkno の場合はアサーションエラーとなります。 4. bufa->blkno > bufb->blkno であれば 1 を、そうでなければ -1 を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/storage/smgr/bulk_write.c 内の qsort 関数から、pending_writes 配列をソートする際に呼び出されます。
  • **呼び出し先**: なし。

関連データ構造

  • PendingWrite: 書き込み対象のブロック番号等を保持する構造体。

失敗モードと境界条件

  • **アサーション失敗**: 重複するブロック番号が渡された場合、デバッグビルドでは Assert によりプログラムが停止します。

保守担当者向けメモ

  • 本関数は qsort の比較関数としてのみ使用されます。ソート順序を変更する場合は、このロジックを修正してください。
