CopyToTextOneRow

定義

  • 種別: function
  • 定義箇所: src/backend/commands/copyto.c:244

目的

COPY TO コマンドにおいて、テーブルの1行（タプル）をテキスト形式で出力するためのラッパー関数です。

シグネチャまたは形状

static void CopyToTextOneRow(CopyToState cstate, TupleTableSlot *slot)

入力と出力

  • **入力**:
  • cstate: COPY TO 操作の全状態を保持する CopyToState 構造体。
  • slot: 出力対象のデータを含む TupleTableSlot。
  • **出力**: なし（副作用として出力バッファへデータを書き込む）。

前提条件と不変条件

  • cstate が有効に初期化されていること。
  • slot に出力対象のタプルが格納されていること。

アルゴリズムまたは振る舞い

1. CopyToTextLikeOneRow を呼び出し、第3引数に false を渡してテキスト形式での処理を実行します。 2. 内部的には CopyToTextLikeOneRow が各属性値の文字列化およびエスケープ処理を行い、出力バッファへ書き込みます。

呼び出し元と呼び出し先

  • **呼び出し元**: CopyToRoutine 構造体の CopyToOneRow フィールドから間接的に呼び出されます。
  • **呼び出し先**: CopyToTextLikeOneRow を呼び出します。

関連データ構造

  • CopyToState: COPY TO 操作の実行状態を管理する構造体。
  • TupleTableSlot: クエリ実行エンジンがタプルを保持するために使用する構造体。
  • CopyToRoutine: 出力形式ごとの関数ポインタを保持するインターフェース構造体。

失敗モードと境界条件

  • メモリ不足や出力先（ファイル、パイプ、フロントエンド）への書き込みエラーが発生した場合、PostgreSQL のエラーハンドリング機構により例外がスローされます。

保守担当者向けメモ

  • 本関数は CopyToTextLikeOneRow の単なるラッパーです。テキスト形式の出力ロジックに変更を加える場合は、CopyToTextLikeOneRow を参照してください。
  • CopyToCSVOneRow と同様に CopyToTextLikeOneRow を共有しており、CSV 形式との共通化が図られています。
