pg_u_prop_hex_digit

定義

  • 種別: function
  • 定義箇所: src/common/unicode_category.c:180

目的

指定された Unicode コードポイントが、Unicode 規格における「16進数桁（Hex Digit）」プロパティを持つかどうかを判定します。

シグネチャまたは形状

bool pg_u_prop_hex_digit(pg_wchar code)

入力と出力

  • **入力**: pg_wchar code（判定対象の Unicode コードポイント）
  • **出力**: bool（該当する場合は true、それ以外は false）

前提条件と不変条件

  • 入力コードポイントは pg_wchar 型であること。
  • Unicode 規格の安定性ポリシーに基づき、判定基準となるカテゴリ値は固定されているものとします。

アルゴリズムまたは振る舞い

1. **ASCII 範囲の判定**: code が 0x80 未満の場合、unicode_opt_ascii テーブルを参照し、PG_U_PROP_HEX_DIGIT ビットが立っているかを定数時間で判定します。 2. **範囲外の判定**: 0x80 以上のコードポイントについては、unicode_hex_digit テーブルに対して range_search を実行し、該当範囲に含まれるかを二分探索で特定します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/common/unicode/category_test.c:74
  • src/common/unicode_category.c:325
  • **呼び出し先**:
  • range_search（unicode_hex_digit テーブルを引数として使用）

関連データ構造

  • unicode_opt_ascii: ASCII 文字のプロパティを保持する静的テーブル。
  • unicode_hex_digit: 16進数桁プロパティを持つコードポイント範囲を保持する静的テーブル。

失敗モードと境界条件

  • 0x80 未満のコードポイントに対してはテーブル参照による高速な判定が行われます。
  • 0x80 以上のコードポイントに対しては range_search による二分探索が行われるため、テーブルの定義範囲外の入力に対しては正しく false を返します。
