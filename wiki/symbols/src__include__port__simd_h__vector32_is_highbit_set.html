vector32_is_highbit_set

定義

  • 種別: function
  • 定義箇所: src/include/port/simd.h:293

目的

Vector32 型のベクトルに含まれる各要素の最上位ビット（high bit）が設定されているかを判定します。

シグネチャまたは形状

static inline bool vector32_is_highbit_set(const Vector32 v)

入力と出力

  • **入力**: v (Vector32) - 判定対象のベクトル。
  • **出力**: bool - ベクトルのいずれかの要素で最上位ビットがセットされている場合に true を返し、それ以外は false を返します。

前提条件と不変条件

  • 本関数は Vector32 型を引数に取ります。
  • SIMD が利用できない環境では Vector32 関連の関数は実装されていないため、本関数も利用できません。

アルゴリズムまたは振る舞い

1. コンパイル時のフラグに基づき、以下のいずれかの処理を行います。

  • USE_NEON が定義されている場合: v を Vector8 にキャストし、vector8_is_highbit_set を呼び出します。
  • それ以外の場合: v をそのまま vector8_is_highbit_set に渡します。

    • 2. 内部的には vector8_is_highbit_set の結果をそのまま返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/include/port/pg_lfind.h:142
  • **呼び出し先**: vector8_is_highbit_set

関連データ構造

  • Vector32: 32ビット要素を保持するSIMDベクトル型。
  • Vector8: 8ビット要素を保持するSIMDベクトル型。

失敗モードと境界条件

  • SIMD 非対応環境では、本関数は実装されていないためコンパイルエラーまたはリンクエラーとなる可能性があります。

保守担当者向けメモ

  • 本関数は vector8_is_highbit_set へのラッパーとして機能しています。
  • Vector32 から Vector8 へのキャストは、プラットフォーム固有の SIMD 命令セットの特性に依存しています。
