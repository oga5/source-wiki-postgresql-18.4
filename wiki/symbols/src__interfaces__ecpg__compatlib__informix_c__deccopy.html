deccopy

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/compatlib/informix.c:172

目的

Informix 互換の decimal 型構造体の値を、別の decimal 型構造体へコピーする。

シグネチャまたは形状

void deccopy(decimal *src, decimal *target)

入力と出力

  • **入力**:
  • src: コピー元の decimal 構造体へのポインタ。
  • target: コピー先の decimal 構造体へのポインタ。
  • **出力**: なし（戻り値なし）。

前提条件と不変条件

  • src および target は、有効な decimal 構造体を指している必要がある。
  • memcpy を使用しているため、src と target のメモリ領域が重なっている場合の動作は memcpy の仕様に依存する。

アルゴリズムまたは振る舞い

1. src が指すメモリ領域から sizeof(decimal) バイト分を、target が指すメモリ領域へコピーする。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/interfaces/ecpg/compatlib/informix.c 内で直接呼び出されている箇所がある（行173）。
  • **呼び出し先**:
  • 標準 C ライブラリの memcpy を呼び出す。

関連データ構造

  • decimal: Informix 互換の数値型構造体。

失敗モードと境界条件

  • src または target が NULL ポインタである場合、セグメンテーション違反が発生する可能性がある。

保守担当者向けメモ

  • 本関数は memcpy による単純なメモリコピーを行うラッパーである。decimal 構造体の内部構造に変更があった場合、このコピー処理が適切であるか再確認すること。
