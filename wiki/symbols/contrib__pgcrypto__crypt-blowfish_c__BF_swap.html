BF_swap

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/crypt-blowfish.c:440

目的

リトルエンディアン環境において、BF_word 型の配列のバイト順序を入れ替え、ビッグエンディアン形式との整合性を確保する。

シグネチャまたは形状

static void BF_swap(BF_word *x, int count)

入力と出力

  • x: 入れ替え対象となる BF_word 型配列へのポインタ。
  • count: 処理対象となる配列要素数。
  • 出力: なし（ポインタ x が指すメモリ領域を直接書き換える）。

前提条件と不変条件

  • WORDS_BIGENDIAN が定義されていない（リトルエンディアン）環境でのみ動作する。
  • ビッグエンディアン環境では、本関数は何も行わない（空の関数として振る舞う）。

アルゴリズムまたは振る舞い

1. WORDS_BIGENDIAN が未定義の場合、以下の処理を count 回繰り返す。 2. 対象の BF_word 値に対して、16ビット単位の入れ替えを行う。 3. さらに、8ビット単位での入れ替え（バイトスワップ）を行い、32ビットワード内のバイト順序を反転させる。 4. ポインタ x をインクリメントし、次の要素へ進む。

呼び出し元と呼び出し先

  • 呼び出し元:
  • contrib/pgcrypto/crypt-blowfish.c:641 (ソルトの変換)
  • contrib/pgcrypto/crypt-blowfish.c:746 (出力結果の変換)
  • 呼び出し先: なし。

関連データ構造

  • BF_word: Blowfish アルゴリズムで使用される32ビットワード型。

失敗モードと境界条件

  • count に 0 以下の値が渡された場合の挙動は未定義（ループ条件 while (--count) に依存）。
  • ポインタ x が指すメモリ領域が count 分確保されていない場合、バッファオーバーランが発生する。

保守担当者向けメモ

  • 本関数は bcrypt の移植性を維持するための重要なユーティリティである。
  • ビッグエンディアン環境では何もしないため、エンディアン依存のバグを調査する際は、コンパイル時の WORDS_BIGENDIAN 定義を確認すること。
