list_length

定義

  • 種別: function
  • 定義箇所: src/include/nodes/pg_list.h:151

目的

PostgreSQLの汎用リスト構造（List）に含まれる要素数を取得します。

シグネチャまたは形状

static inline int list_length(const List *l)

入力と出力

  • **入力**: const List *l （対象のリストへのポインタ。NIL も許容される）
  • **出力**: int （リスト内の要素数。空リストの場合は 0）

前提条件と不変条件

  • l が NULL（NIL）の場合、0 を返します。
  • List 構造体の length メンバが現在の要素数を正しく保持していることを前提とします。

アルゴリズムまたは振る舞い

1. 引数 l が真（非 NULL）であるかを確認します。 2. l が真であれば、l->length の値を返します。 3. l が偽（NIL）であれば、0 を返します。

  • **呼び出し先**:
  • 本関数は static inline であり、特定の関数を呼び出すのではなく、List 構造体のメンバアクセスを行います。

関連データ構造

  • List: 要素数 length を保持するリストのヘッダ構造体。

失敗モードと境界条件

  • l が NIL の場合、セグメンテーションフォールトを起こさず安全に 0 を返します。

保守担当者向けメモ

  • 本関数は頻繁に呼び出されるため、static inline として定義されています。
  • リストの要素数は List 構造体内で管理されているため、計算量は O(1) です。
