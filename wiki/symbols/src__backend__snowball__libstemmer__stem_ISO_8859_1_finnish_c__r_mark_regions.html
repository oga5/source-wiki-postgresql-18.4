r_mark_regions

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_finnish.c:277

目的

単語内の処理対象領域（R1, R2）を特定し、SN_env 構造体の内部インデックス I[0] および I[1] に境界位置を格納する。

シグネチャまたは形状

static int r_mark_regions(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z（ステミング環境の状態を保持する構造体）
  • **出力**: 成功時に 1、失敗時に 0 を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体へのポインタであること。
  • g_V1 グルーピング定義に基づき、母音の出現位置を基準に領域を計算する。

アルゴリズムまたは振る舞い

1. z->I[1]（R1の開始位置）と z->I[0]（R2の開始位置）を初期値として文字列の末尾に設定する。 2. out_grouping と in_grouping を使用して、母音の連続または母音・子音の切り替わりを検出し、単語内の特定のオフセットを計算する。 3. 最初の母音の後の子音の開始位置を z->I[1] に設定する。 4. さらにその後の母音の後の子音の開始位置を z->I[0] に設定する。 5. 処理の各段階で out_grouping または in_grouping が失敗した場合は 0 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: finnish_ISO_8859_1_stem（src/backend/snowball/libstemmer/stem_ISO_8859_1_finnish.c 内）。
  • **呼び出し先**: out_grouping, in_grouping（Snowballランタイム関数）。

関連データ構造

  • struct SN_env: ステミング対象の文字列、現在のカーソル位置、領域境界インデックスを保持する。

失敗モードと境界条件

  • 単語内に十分な母音・子音のパターンが存在しない場合、out_grouping または in_grouping が負の値を返し、関数は 0 を返して終了する。

保守担当者向けメモ

  • 本関数はSnowballコンパイラによって自動生成されたコードである。ロジックの変更が必要な場合は、直接編集せず元のSnowballスクリプト（.sbl）を修正すること。
  • ISO-8859-1エンコーディング専用であるため、UTF-8環境での動作には適さない。
