r_attached_pronoun

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_italian.c:725

目的

単語の末尾に付着したイタリア語の代名詞を検出し、削除または置換を行うことで語幹抽出を補助する。

シグネチャまたは形状

static int r_attached_pronoun(struct SN_env * z)

入力と出力

  • **入力**: SN_env 構造体（現在の単語データおよびカーソル位置 c を含む）。
  • **出力**: 処理が成功した場合は 1、失敗した場合は 0、エラー発生時は負の値を返す。

前提条件と不変条件

  • z->c が単語内の有効な位置を指していること。
  • r_RV 関数により、対象の接尾辞が RV 領域内にあることが検証される必要がある。

アルゴリズムまたは振る舞い

1. **末尾チェック**: 現在位置の文字が特定のビットマスク条件を満たすか確認する。 2. **接尾辞検索**: find_among_b を使用し、配列 a_2（37要素）から後方検索を行う。 3. **代名詞特定**: find_among_b を使用し、配列 a_3（5要素）から代名詞を特定する。 4. **領域検証**: r_RV(z) を呼び出し、対象が RV 領域内にあるか確認する。 5. **置換/削除**:

  • among_var == 1: slice_del(z) で接尾辞を削除。
  • among_var == 2: slice_from_s(z, 1, s_10) で接尾辞を別の文字列に置換。

呼び出し元と呼び出し先

  • **呼び出し元**: italian_UTF_8_stem（src/backend/snowball/libstemmer/stem_UTF_8_italian.c）および他言語のステマー（Catalan, Spanish 等）から呼び出される。
  • **呼び出し先**: find_among_b, r_RV, slice_del, slice_from_s。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_2, a_3: 接尾辞検索用の among 構造体配列。

失敗モードと境界条件

  • 単語の末尾が条件を満たさない場合、または find_among_b が一致しない場合は 0 を返す。
  • r_RV が失敗した場合、その戻り値をそのまま返す。
  • slice_del や slice_from_s がエラー（負の値）を返した場合、その値を伝播させる。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードであるため、直接編集せず .sbl ソースを修正すること。
  • a_2 および a_3 配列の定義変更が本関数の挙動に直接影響する。
