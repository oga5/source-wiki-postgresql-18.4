mbchar_verifier

定義

  • 種別: typedef
  • 定義箇所: src/include/mb/pg_wchar.h:388

目的

特定のマルチバイト文字エンコーディングにおいて、与えられたバイト列が有効な単一の文字を構成しているかを検証するための関数ポインタ型です。

シグネチャまたは形状

typedef int (*mbchar_verifier) (const unsigned char *mbstr, int len);

入力と出力

  • **入力**:
  • mbstr: 検証対象のマルチバイト文字列の先頭ポインタ。
  • len: 文字列の長さ（バイト数）。
  • **出力**:
  • int: 文字が有効であればその文字のバイト長を返し、無効であれば負の値を返します。

前提条件と不変条件

  • mbstr は有効なメモリ領域を指している必要があります。
  • 各エンコーディング固有の検証ロジックに従い、不正なバイトシーケンスを検出します。

アルゴリズムまたは振る舞い

  • 指定されたエンコーディングのルールに基づき、mbstr の先頭から始まるバイト列がそのエンコーディングにおいて妥当な文字であるかを判定します。
  • 妥当な場合は文字の長さを返し、妥当でない場合はエラーを示す負の値を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/mb/mbutils.c 内の関数にて、mbverify や mbverifychar といった変数名で参照されています。
  • **呼び出し先**:
  • 本シンボル自体は関数ポインタ型であり、具体的な実装関数は pg_wchar_tbl 構造体を通じて各エンコーディングごとに割り当てられます。

関連データ構造

  • pg_wchar_tbl: 各エンコーディングの検証関数を保持するテーブル構造体。
  • pg_enc: エンコーディング識別子。

失敗モードと境界条件

  • len が 0 または負の場合の挙動は、実装側の検証関数に依存します。
  • 不正なマルチバイトシーケンスが入力された場合、負の値を返すことで検証失敗を通知します。

保守担当者向けメモ

  • 本型定義は pg_wchar_tbl を介して利用されるため、新しいエンコーディングを追加する際は、このシグネチャに適合する検証関数を実装し、テーブルに登録する必要があります。
