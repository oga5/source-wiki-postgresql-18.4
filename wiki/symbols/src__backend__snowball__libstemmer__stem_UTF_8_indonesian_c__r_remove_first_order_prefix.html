r_remove_first_order_prefix

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_indonesian.c:193

目的

インドネシア語のステミング処理において、単語の先頭から第一順位の接頭辞を特定し、除去または置換を行う。

シグネチャまたは形状

static int r_remove_first_order_prefix(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（ステミング環境の状態を保持する構造体）。
  • **出力**: 成功時に 1、失敗または条件不一致時に 0 を返す。slice_del 等の内部関数がエラーを返した場合は負の値を返す。

前提条件と不変条件

  • z->c（現在のカーソル位置）が単語の先頭付近にあること。
  • 単語の2文字目（z->c + 1）が文字コード 105 ('i') または 101 ('e') である必要がある。
  • z->I[0] および z->I[1] は、ステミングの進行状態を追跡する整数配列として更新される。

アルゴリズムまたは振る舞い

1. z->bra を現在のカーソル位置に設定する。 2. 2文字目のチェックを行い、条件を満たさない場合は即座に 0 を返す。 3. find_among を使用して、a_3 配列（12個のパターン）から接頭辞を検索する。 4. マッチした among_var に応じて以下の処理を行う：

  • **ケース 1, 2**: slice_del で接頭辞を削除し、z->I[0] を更新（1または3）、z->I[1] をデクリメントする。
  • **ケース 3, 4**: slice_from_s で接頭辞を特定の文字列に置換し、z->I[0] を更新、z->I[1] をデクリメントする。
  • **ケース 5, 6**: z->I[0] を更新し、z->I[1] をデクリメントした後、母音の有無に応じて slice_del または slice_from_s を選択的に実行する。

    • 5. z->ket を現在のカーソル位置に設定し、処理を完了する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_UTF_8_indonesian.c および stem_ISO_8859_1_indonesian.c 内のメインステミング処理から呼び出される。
  • **呼び出し先**: find_among, slice_del, slice_from_s, in_grouping_U（いずれも Snowball ランタイム関数）。

関連データ構造

  • struct SN_env: ステミング環境を管理する構造体。
  • a_3: 第一順位の接頭辞パターンを定義する among 構造体の配列。

失敗モードと境界条件

  • z->c + 1 が文字列長 z->l を超える場合、または2文字目が指定の文字でない場合は 0 を返す。
  • find_among でマッチしない場合は 0 を返す。
  • slice_del や slice_from_s が負の値を返した場合、そのエラーコードをそのまま呼び出し元に伝播させる。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードであるため、直接編集せず、元の .sbl スクリプトを修正すること。
  • z->I[0] の値（1または3）は、後続の接頭辞除去処理における状態遷移を制御しているため、変更時は他の接頭辞除去関数との整合性に注意が必要。
