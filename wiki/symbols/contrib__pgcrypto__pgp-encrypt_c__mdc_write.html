mdc_write

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/pgp-encrypt.c:105

目的

PGP 暗号化パイプラインにおいて、データストリームのハッシュ値を計算しつつ、後続のフィルタへデータを転送する。

シグネチャまたは形状

static int mdc_write(PushFilter *dst, void *priv, const uint8 *data, int len)

入力と出力

  • dst: データを転送する先の PushFilter オブジェクト。
  • priv: MDC 計算用のハッシュコンテキスト (PX_MD *)。
  • data: 書き込み対象のデータバッファ。
  • len: 書き込むデータの長さ。
  • 戻り値: pushf_write の実行結果（成功時は 0、失敗時はエラーコード）。

前提条件と不変条件

  • priv は px_md_update が許容する有効なハッシュコンテキストであること。
  • dst は有効な PushFilter インスタンスであること。

アルゴリズムまたは振る舞い

1. priv に保持されたハッシュコンテキストに対し、px_md_update を呼び出して入力データ data を加算する。 2. pushf_write を呼び出し、入力データ data を次のフィルタ dst へ渡す。 3. pushf_write の戻り値をそのまま呼び出し元へ返す。

呼び出し元と呼び出し先

  • 呼び出し先: px_md_update (ハッシュ計算), pushf_write (フィルタチェーンへの転送)。
  • 呼び出し元: ソースコード内の直接的な呼び出し元は明示されていませんが、mdc_filter 構造体の一部としてフィルタチェーン内で使用されます。

関連データ構造

  • PushFilter: PGP 暗号化パイプラインを構成するフィルタ構造体。
  • PX_MD: px.h で定義されるハッシュ計算用コンテキスト。

失敗モードと境界条件

  • px_md_update または pushf_write がエラーを返した場合、そのエラーコードがそのまま伝播される。

保守担当者向けメモ

  • データの改ざん検知を目的としており、pgp-encrypt.c 内で定義される他の mdc_* 関数群（mdc_init, mdc_flush, mdc_free）とセットで管理される。
