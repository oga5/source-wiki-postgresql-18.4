r_verb_suffix

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_french.c:1020

目的

フランス語のステミング処理において、単語末尾の動詞接尾辞を検索し、条件に応じて削除または置換を行う。

シグネチャまたは形状

static int r_verb_suffix(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z （Snowballランタイム環境）
  • **出力**: int （成功時に1、失敗時に0、エラー発生時に負の値を返す）

前提条件と不変条件

  • z->I[2] に格納された領域境界（R2）より前方の文字は変更されない。
  • 処理対象の単語末尾が a_6 テーブル（38個のパターン）のいずれかに一致する必要がある。

アルゴリズムまたは振る舞い

1. **境界チェック**: 現在のカーソル位置 z->c が z->I[2] 未満であれば即座に終了する。 2. **検索**: find_among_b を使用し、a_6 テーブルに基づき単語末尾から接尾辞を検索する。 3. **条件分岐**:

  • **ケース1**: r_R2(z) を呼び出し、領域R2内であることを確認した上で slice_del を実行する。
  • **ケース2**: 無条件で slice_del を実行する。
  • **ケース3**: slice_del を実行した後、末尾が 'e' であれば追加で削除を試みる。

    • 4. **復元**: 処理終了時に z->lb を元の値 mlimit1 に戻す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_ISO_8859_1_french.c:1218（直接の証拠）。その他、Catalan, Italian, Portuguese, Spanish等の言語実装からも参照されている。
  • **呼び出し先**: find_among_b, r_R2, slice_del。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_6: 動詞接尾辞のパターンを定義した among 構造体の配列。

失敗モードと境界条件

  • find_among_b が一致を見つけられない場合、0を返して終了する。
  • r_R2 や slice_del が内部でエラー（負の値）を返した場合、その値を伝播させる。
  • カーソル位置が境界条件を満たさない場合、処理を行わず0を返す。

保守担当者向けメモ

  • 本関数はSnowballコンパイラによる自動生成コードである。ロジックの変更が必要な場合は、対応する .sbl ソースファイルを修正し、再生成すること。
  • a_6 テーブルのインデックスや case 文の分岐は、フランス語の文法規則と密接に関連している。
