InetPGetDatum

定義

  • 種別: function
  • 定義箇所: src/include/utils/inet.h:128

目的

inet 型の構造体ポインタを、PostgreSQL の関数マネージャ（fmgr）が扱う Datum 型へ変換します。

シグネチャまたは形状

static inline Datum InetPGetDatum(const inet *X)

入力と出力

  • **入力**: const inet *X （ネットワークアドレス情報を保持する inet 型構造体へのポインタ）
  • **出力**: Datum （PostgreSQL の内部データ表現形式）

前提条件と不変条件

  • 入力ポインタ X は有効な inet 構造体を指している必要があります。
  • 本関数は PointerGetDatum をラップしたインライン関数であり、メモリレイアウトの変換は行いません。

アルゴリズムまたは振る舞い

1. 引数として受け取った inet 構造体へのポインタ X を PointerGetDatum マクロに渡します。 2. PointerGetDatum が返す Datum 値をそのまま呼び出し元へ返却します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/network_gist.c
  • src/backend/utils/adt/network_spgist.c
  • src/include/utils/inet.h （PG_RETURN_INET_P マクロ内）
  • **呼び出し先**:
  • PointerGetDatum （PostgreSQL のコアマクロ）

関連データ構造

  • inet: ネットワークアドレスを格納する可変長データ構造体。
  • Datum: PostgreSQL における汎用的なデータ表現型。

失敗モードと境界条件

  • 本関数自体は単純なポインタ変換であるため、関数内部での失敗モードはありません。
  • 入力ポインタ X が NULL の場合、PointerGetDatum の振る舞いに依存します。

保守担当者向けメモ

  • 本関数は PG_RETURN_INET_P マクロの実装として利用されています。
  • inet 型のメモリレイアウトや varlena ヘッダーの操作については、src/include/utils/inet.h の概要および関連する SET_INET_VARSIZE マクロの定義を参照してください。
