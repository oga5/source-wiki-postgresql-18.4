build_column_default

定義

  • 種別: function
  • 定義箇所: src/backend/rewrite/rewriteHandler.c:1262

目的

指定されたリレーションの特定の列に対して、デフォルト値の式（Node）を生成または取得する。

シグネチャまたは形状

Node *build_column_default(Relation rel, int attrno)

入力と出力

  • **入力**:
  • rel: 対象となるリレーション（Relation）。
  • attrno: デフォルト値を取得する列番号（1始まり）。
  • **出力**:
  • デフォルト値の式を表す Node。デフォルトが存在しない場合は NULL を返す。

前提条件と不変条件

  • rel は有効なリレーション構造体であること。
  • attrno はリレーション内の有効な列番号であること。

アルゴリズムまたは振る舞い

1. **Identity列の処理**: 列が attidentity であれば、NextValueExpr を生成して返す。 2. **列デフォルトの取得**: atthasdef が真であれば、TupleDescGetDefault を用いて式を取得する。見つからない場合はエラーを投げる。 3. **型デフォルトの取得**: 列デフォルトがなく、かつ生成列（attgenerated）でない場合、get_typdefault を用いて型のデフォルト値を取得する。 4. **型変換**: 取得した式が列の型と一致しない可能性があるため、coerce_to_target_type を用いてターゲットの型（atttype, atttypmod）へキャストする。 5. **整合性チェック**: キャストに失敗した場合は ERRCODE_DATATYPE_MISMATCH エラーを報告する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/commands/copyfrom.c
  • src/backend/commands/tablecmds.c
  • src/backend/executor/nodeModifyTable.c
  • src/backend/replication/logical/worker.c
  • src/backend/rewrite/rewriteHandler.c (複数箇所)
  • **呼び出し先**:
  • TupleDescAttr, getIdentitySequence, makeNode, TupleDescGetDefault, RelationGetRelationName, get_typdefault, exprType, coerce_to_target_type, elog, ereport 等。

関連データ構造

  • Relation: リレーション情報。
  • TupleDesc: タプル記述子。
  • Form_pg_attribute: 属性定義。
  • NextValueExpr: Identity列用の式ノード。

失敗モードと境界条件

  • atthasdef が真であるにもかかわらずデフォルト式が取得できない場合、elog(ERROR) を発生させる。
  • 取得したデフォルト式の型と列の型に互換性がない場合、ereport(ERROR) を発生させる。

保守担当者向けメモ

  • 本関数は、パーサの transformAssignedExpr() と同様の型変換ロジックを維持する必要がある。
  • attidentity 列の処理は、他のデフォルト値取得ロジックよりも優先される。
