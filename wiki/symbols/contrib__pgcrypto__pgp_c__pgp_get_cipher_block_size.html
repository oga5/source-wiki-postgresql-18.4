pgp_get_cipher_block_size

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/pgp.c:146

目的

指定された暗号アルゴリズムコードに対応するブロックサイズ（バイト単位）を取得します。

シグネチャまたは形状

int pgp_get_cipher_block_size(int code)

入力と出力

  • **入力**: code (int) - 暗号アルゴリズムを識別する内部コード。
  • **出力**: (int) - アルゴリズムのブロックサイズ。該当するアルゴリズムが見つからない場合は 0 を返します。

前提条件と不変条件

  • get_cipher_info(code) が有効な cipher_info 構造体へのポインタ、または NULL を返すことを前提としています。

アルゴリズムまたは振る舞い

1. get_cipher_info(code) を呼び出し、指定されたコードに対応するメタデータ構造体を取得します。 2. 構造体が取得できた場合、その block_len フィールドの値を返します。 3. 構造体が取得できなかった（NULL）場合、0 を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/pgcrypto/pgp-decrypt.c:252
  • contrib/pgcrypto/pgp-encrypt.c:484
  • contrib/pgcrypto/pgp-pubkey.c:373
  • **呼び出し先**:
  • get_cipher_info (同ファイル内)

関連データ構造

  • cipher_info: 暗号アルゴリズムのメタデータを保持する構造体。block_len フィールドを介してブロックサイズを提供します。

失敗モードと境界条件

  • 不明な code が渡された場合、get_cipher_info が NULL を返し、本関数は 0 を返します。

保守担当者向けメモ

  • 本関数は pgp.h で宣言されており、公開インターフェースの一部です。
  • 新しい暗号アルゴリズムを追加する際は、get_cipher_info が参照するリストを更新することで、本関数も自動的に対応可能となります。
