is_opclause

定義

  • 種別: function
  • 定義箇所: src/include/nodes/nodeFuncs.h:77

目的

与えられたノードが演算子式（OpExpr）であるかを判定する。

シグネチャまたは形状

static inline bool is_opclause(const void *clause)

入力と出力

  • **入力**: clause (任意のノードへのポインタ)
  • **出力**: clause が NULL ではなく、かつ OpExpr 型のノードである場合に true を返し、それ以外の場合に false を返す。

前提条件と不変条件

  • clause は NULL である可能性がある。
  • IsA マクロを使用してノードの型タグを検証する。

アルゴリズムまたは振る舞い

1. clause が NULL でないことを確認する。 2. IsA(clause, OpExpr) を評価し、ノードの型が T_OpExpr であるかを確認する。 3. 両方の条件を満たす場合にのみ true を返す。

関連データ構造

  • OpExpr: 演算子式を表すノード構造体。
  • Node: PostgreSQLのツリー構造の基本単位。

失敗モードと境界条件

  • clause が NULL の場合、安全に false を返す。
  • clause が OpExpr 以外のノード型である場合、false を返す。

保守担当者向けメモ

  • 本関数はインライン関数として定義されており、クエリ最適化や実行時のノード走査において頻繁に呼び出されるため、パフォーマンスを考慮してヘッダーファイルに配置されている。
