目的

オランダ語のステミング処理の初期段階として、入力文字列に対するアクセント記号の正規化、特定の文字置換、および母音に挟まれた 'i' や 'y' の処理を行う。

シグネチャまたは形状

static int r_prelude(struct SN_env * z)

入力と出力

  • 入力: SN_env 構造体（現在のカーソル位置や文字列データを含む）。
  • 出力: 成功時に 1 を返し、失敗（メモリ不足やスライス操作の失敗）時に負の値を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • header.h で定義された Snowball ランタイム環境が初期化されていること。

アルゴリズムまたは振る舞い

1. **文字置換**: find_among を使用し、定義されたテーブルに基づいて特定の文字パターンを置換する。一致しない場合はカーソルを1つ進める。 2. **'y' の処理**: 文字列の先頭が 'y' である場合、それを別の文字（s_5）に置換する。 3. **母音間処理**: 文字列を走査し、母音（g_v グループ）に挟まれた 'i' を別の文字（s_6）に、または 'y' を別の文字（s_7）に置換する。 4. 処理中、bra（開始位置）と ket（終了位置）を更新し、slice_from_s を用いて文字列の書き換えを行う。

  • 呼び出し先: find_among, slice_from_s, in_grouping（いずれも Snowball ランタイム関数）。
