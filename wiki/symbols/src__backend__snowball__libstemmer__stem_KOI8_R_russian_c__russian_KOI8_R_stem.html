russian_KOI8_R_stem

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_KOI8_R_russian.c:572

目的

KOI8-Rエンコーディングで表現されたロシア語テキストに対し、Snowballアルゴリズムを用いて語幹抽出（ステミング）を行う。

シグネチャまたは形状

int russian_KOI8_R_stem(struct SN_env * z)

入力と出力

  • **入力**: SN_env 構造体（ステミング対象の文字列データおよび作業領域を含む）。
  • **出力**: 成功時に 1、失敗時に負の値を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • 入力文字列はKOI8-Rエンコーディングであること。
  • 内部で r_mark_regions を呼び出し、語幹抽出に必要な領域（R1, R2）を特定する。

アルゴリズムまたは振る舞い

1. **前処理**: 入力文字列内の特定の文字（0xA3）を置換する。 2. **領域特定**: r_mark_regions を実行し、ステミングの境界を計算する。 3. **接尾辞削除**: 以下の順序でルールを適用し、単語末尾から前方へ向かってマッチングと削除を行う。

  • 完了体分詞 (r_perfective_gerund)
  • 再帰動詞 (r_reflexive)
  • 形容詞・形容詞的語尾 (r_adjectival)
  • 動詞 (r_verb)
  • 名詞 (r_noun)

    • 4. **後処理**: 特定の文字コード（0xC9）の削除、派生語尾の削除 (r_derivational)、および最終的な調整 (r_tidy_up) を行う。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_KOI8_R_russian.c および src/include/snowball/libstemmer/stem_KOI8_R_russian.h に宣言が存在するが、具体的な呼び出し元コードは提供されていない。
  • **呼び出し先**: r_mark_regions, r_perfective_gerund, r_reflexive, r_adjectival, r_verb, r_noun, r_derivational, r_tidy_up, slice_from_s, slice_del。

関連データ構造

  • struct SN_env: Snowballランタイムの環境構造体。
  • among 構造体配列（a_0 ～ a_7）: パターンマッチングに使用される接尾辞定義。

失敗モードと境界条件

  • slice_from_s や各ルール関数が負の値を返した場合、その値をそのまま戻り値として終了する。
  • 文字列の境界や領域（R1, R2）の判定に失敗した場合、処理を中断またはスキップする。

保守担当者向けメモ

  • 本関数はSnowballコンパイラによる自動生成コードである。直接編集せず、元のSnowballスクリプト（.sbl）を修正すること。
  • KOI8-R特有の文字コード（0xA3, 0xC9 等）を扱うため、デバッグ時はエンコーディングの不一致に注意が必要。
