decrypt_rsa

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/pgp-pubdec.c:123

目的

RSAアルゴリズムを使用して、PGP公開鍵で暗号化されたセッション鍵を復号します。

シグネチャまたは形状

static int decrypt_rsa(PGP_PubKey *pk, PullFilter *pkt, PGP_MPI **m_p)

入力と出力

  • **入力**:
  • pk: 復号に使用する公開鍵構造体 (PGP_PubKey)。
  • pkt: 暗号化されたデータを含む入力フィルタ (PullFilter)。
  • m_p: 復号結果のMPI（多倍長整数）を格納するためのポインタ。
  • **出力**:
  • 成功時は 0 を返し、m_p に復号されたデータが格納されます。
  • 失敗時は負の整数（エラーコード）を返します。

前提条件と不変条件

  • pk->algo が PGP_PUB_RSA_ENCRYPT または PGP_PUB_RSA_ENCRYPT_SIGN である必要があります。
  • 復号処理の過程で生成される一時的なMPIデータは、関数終了前に解放される必要があります。

アルゴリズムまたは振る舞い

1. 鍵アルゴリズムの検証を行い、RSA以外の場合は PXE_PGP_WRONG_KEY を返します。 2. pgp_mpi_read を呼び出し、入力フィルタから暗号化されたMPIデータを読み込みます。 3. pgp_rsa_decrypt を呼び出し、RSA復号を実行して結果を m_p に格納します。 4. 読み込んだ暗号化データ（MPI）を pgp_mpi_free で解放します。 5. 復号処理の結果を呼び出し元へ返します。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/pgcrypto/pgp-pubdec.c 内の pgp_parse_pubenc_sesskey から呼び出されます。
  • **呼び出し先**:
  • pgp_mpi_read
  • pgp_rsa_decrypt
  • pgp_mpi_free

関連データ構造

  • PGP_PubKey: 公開鍵情報を保持する構造体。
  • PGP_MPI: 多倍長整数を扱うためのPGP内部構造体。
  • PullFilter: データ読み込み用のフィルタ構造体。

失敗モードと境界条件

  • 鍵アルゴリズムがRSAでない場合: PXE_PGP_WRONG_KEY を返します。
  • MPIの読み込みに失敗した場合: pgp_mpi_read が返すエラーコードをそのまま伝播します。
  • 復号処理自体が失敗した場合: pgp_rsa_decrypt が返すエラーコードを返します。

保守担当者向けメモ

  • 本関数はPKCS#1 v1.5パディングを前提とした復号フローの一部です。
  • 復号失敗時に返される PXE_PGP_WRONG_KEY は、パディングエラーやチェックサム不一致など、複数の原因を包括する汎用的なエラーコードとして扱われます。
