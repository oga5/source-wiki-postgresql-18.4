helpVariables

定義

  • 種別: function
  • 定義箇所: src/bin/psql/help.c:373

目的

psql の設定変数、表示設定、および環境変数に関するヘルプ情報を構築し、ユーザーへ表示する。

シグネチャまたは形状

void helpVariables(unsigned short int pager);

入力と出力

  • **入力**: pager (ページャの使用可否を制御するフラグ)
  • **出力**: なし（標準出力またはページャ経由でヘルプテキストを出力）

前提条件と不変条件

  • PQExpBuffer を使用してヘルプ内容をバッファリングする。
  • ページングが必要な場合、PageOutput() を介して外部ページャへ出力する。

アルゴリズムまたは振る舞い

1. PQExpBufferData を初期化する。 2. HELP0 および HELPN マクロを使用して、以下のカテゴリのヘルプテキストをバッファに追加する。

  • psql 変数（AUTOCOMMIT, ECHO 等）
  • 表示設定（border, format 等）
  • 環境変数（PGDATABASE, PGUSER 等）

    • 3. プラットフォーム依存の処理として、Windows環境か否かで環境変数の設定方法の記述を切り替える。 4. 構築したバッファを PageOutput() に渡し、ユーザー設定に従って出力する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/bin/psql/command.c:3613
  • src/bin/psql/startup.c:711
  • **呼び出し先**:
  • initPQExpBuffer (バッファ初期化)
  • HELP0, HELPN (文字列追加マクロ)
  • PageOutput (ページング制御)

関連データ構造

  • PQExpBufferData: ヘルプテキスト構築用バッファ。
  • pset: settings.h で管理される現在の設定状態（直接参照はしていないが、本関数が提供する情報は pset に関連する）。

失敗モードと境界条件

  • ページャが有効な場合、PageOutput の動作に依存する。
  • HELPN マクロでデフォルト値（DEFAULT_WATCH_INTERVAL 等）を埋め込む際、定義が欠落しているとコンパイルエラーとなる可能性がある。

保守担当者向けメモ

  • ヘルプテキストの内容を更新した際は、必ず ref/psql-ref.sgml（DocBook）と同期させること。
  • 新しい変数を追加した場合は、適切なカテゴリ（変数、表示設定、環境変数）に分類して記述すること。
