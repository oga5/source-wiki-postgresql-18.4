PQresultMemorySize

定義

  • 種別: function
  • 定義箇所: src/interfaces/libpq/fe-exec.c:668

目的

指定された PGresult オブジェクトが内部的に保持しているメモリ使用量（バイト単位）を取得します。

シグネチャまたは形状

size_t PQresultMemorySize(const PGresult *res);

入力と出力

  • **入力**: res - 調査対象の PGresult 構造体へのポインタ。
  • **出力**: res が保持するメモリサイズを size_t 型で返します。

前提条件と不変条件

  • res が NULL の場合、関数は 0 を返します。
  • res が有効なポインタである場合、その構造体内部の memorySize フィールドが返されます。

アルゴリズムまたは振る舞い

1. 引数 res が NULL かどうかを判定します。 2. NULL であれば 0 を即座に返します。 3. NULL でなければ、res->memorySize の値を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: 提供されたソース情報には、具体的な呼び出し元の関数名は記載されていません。
  • **呼び出し先**: 内部で他の関数を呼び出すことはありません。

関連データ構造

  • PGresult: クエリ結果を保持する主要な構造体。本関数はこの構造体の memorySize フィールドを参照します。

失敗モードと境界条件

  • res が NULL の場合は安全に 0 を返します。それ以外の境界条件や失敗モードは定義されていません。

保守担当者向けメモ

  • 本関数は libpq-fe.h で公開されている API です。
  • PGresult はメモリ断片化を防ぐために独自のアロケータ（pqResultAlloc 等）を使用しており、本関数はその管理下にある総メモリ量を報告するために使用されます。
