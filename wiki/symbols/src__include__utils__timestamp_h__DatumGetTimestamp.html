DatumGetTimestamp

定義

  • 種別: static inline 関数
  • 定義箇所: src/include/utils/timestamp.h:27

目的

PostgreSQLの内部データ型 Datum から、日時表現である Timestamp 型へ安全に変換する。

シグネチャまたは形状

static inline Timestamp DatumGetTimestamp(Datum X)

入力と出力

  • **入力**: Datum X （変換対象のデータ）
  • **出力**: Timestamp （int64 として表現される内部日時形式）

前提条件と不変条件

  • Timestamp 型は内部的に int64 として実装されている。
  • 本関数は DatumGetInt64(X) を呼び出し、その結果を Timestamp 型にキャストして返す。

アルゴリズムまたは振る舞い

1. 引数 X を DatumGetInt64 マクロに渡し、int64 値を取得する。 2. 取得した値を Timestamp 型にキャストして戻り値とする。 3. インライン関数として定義されており、呼び出し時のオーバーヘッドを最小化する設計となっている。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/btree_gist/btree_utils_num.c
  • src/backend/utils/adt/json.c
  • src/backend/utils/adt/jsonpath_exec.c
  • src/backend/utils/adt/selfuncs.c
  • src/backend/utils/adt/timestamp.c
  • **呼び出し先**:
  • DatumGetInt64（マクロ）

関連データ構造

  • Timestamp: 本関数が変換先とする日時表現型。
  • Datum: PostgreSQLの汎用データ型。

失敗モードと境界条件

  • 入力 Datum が正しく int64 表現を保持していない場合、キャスト後の値は不定となる。本関数自体は型チェックを行わない。

保守担当者向けメモ

  • 本関数は Timestamp 型の内部表現が int64 であることに依存している。将来的に Timestamp の定義が変更された場合、本関数の実装も修正が必要となる。
