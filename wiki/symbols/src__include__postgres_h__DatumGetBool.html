DatumGetBool

定義

  • 種別: static inline 関数
  • 定義箇所: src/include/postgres.h:94

目的

PostgreSQL の汎用データ型である Datum を、C 言語の bool 型へ変換するために使用されます。

シグネチャまたは形状

static inline bool DatumGetBool(Datum X)

入力と出力

  • **入力**: Datum X（変換対象の汎用データ値）
  • **出力**: bool（X が 0 でない場合は true、0 の場合は false）

前提条件と不変条件

  • Datum 型は sizeof(void *) と同じサイズであると仮定されています。
  • 本関数はインライン化されるため、関数呼び出しのオーバーヘッドなしで型変換を行います。

アルゴリズムまたは振る舞い

引数 X に対して X != 0 の比較演算を行い、その結果を bool 型として返します。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/btree_gist/btree_bit.c、contrib/btree_gist/btree_bytea.c、contrib/btree_gist/btree_date.c 等の GiST インデックス関連モジュールから、比較関数の結果を bool に変換する目的で呼び出されています。
  • **呼び出し先**: 本関数は単一の比較演算のみを行うため、他の関数を呼び出しません。

関連データ構造

  • Datum: PostgreSQL における汎用データ表現。

失敗モードと境界条件

  • Datum がポインタ型を保持している場合、その値が NULL（0）であれば false を返し、それ以外であれば true を返します。
  • Datum の値が未定義または不正なビットパターンである場合、比較結果は C 言語の標準的な整数比較ルールに従います。

保守担当者向けメモ

  • 本関数は postgres.h で定義されている一連の DatumGet* インライン関数群の一部です。
  • 移植性を維持するため、Datum から基本型への変換には必ずこれらの関数を使用してください。直接的なキャストは避けるべきです。
