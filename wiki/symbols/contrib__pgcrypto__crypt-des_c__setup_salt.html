setup_salt

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/crypt-des.c:372

目的

DES暗号化処理で使用するソルト値を内部的なビット形式に変換し、グローバルな saltbits 変数を更新する。

シグネチャまたは形状

static void setup_salt(long salt)

入力と出力

  • **入力**: long salt（変換対象のソルト値）
  • **出力**: なし（副作用としてグローバル変数 saltbits および old_salt を更新）

前提条件と不変条件

  • old_salt との比較により、同一のソルトが渡された場合は処理をスキップする最適化が行われる。
  • 24ビットのソルト情報を対象としてビット操作を行う。

アルゴリズムまたは振る舞い

1. 引数 salt が現在の old_salt と一致するか確認し、一致すれば即座に終了する。 2. 一致しない場合、old_salt を更新する。 3. saltbits を 0 に初期化する。 4. 24回のループを実行し、salt の各ビットを obit（0x800000 から開始し右シフト）へ転送する。

  • salt のビットが立っている場合、対応する obit の位置に saltbits を設定する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/pgcrypto/crypt-des.c:373 (定義内での自己再帰的参照の可能性、または初期化時)
  • contrib/pgcrypto/crypt-des.c:629
  • contrib/pgcrypto/crypt-des.c:761
  • **呼び出し先**: 外部関数呼び出しはなし。

関連データ構造

  • old_salt (グローバル変数): 前回のソルト値を保持。
  • saltbits (グローバル変数): 変換後のビットパターンを保持。

失敗モードと境界条件

  • 24ビットを超えるソルト値が渡された場合、上位ビットは無視される。

保守担当者向けメモ

  • 本関数は crypt(3) 互換の DES 実装の一部であり、パフォーマンス向上のためにソルトのキャッシュを行っている。
  • 割り込みチェック（CHECK_FOR_INTERRUPTS()）は本関数内には含まれていない。
