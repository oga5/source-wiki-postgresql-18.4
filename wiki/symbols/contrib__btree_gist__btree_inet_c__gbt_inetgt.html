gbt_inetgt

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_inet.c:28

目的

inet 型の GiST インデックスにおいて、2つのスカラ値（double に変換済み）を比較し、左辺が右辺より大きいかどうかを判定します。

シグネチャまたは形状

static bool gbt_inetgt(const void *a, const void *b, FmgrInfo *flinfo)

入力と出力

  • **入力**:
  • a: 比較対象の左辺値（double 型へのポインタ）。
  • b: 比較対象の右辺値（double 型へのポインタ）。
  • flinfo: 関数管理情報（本関数内では未使用）。
  • **出力**:
  • bool: *a > *b が真であれば true、それ以外は false を返します。

前提条件と不変条件

  • 入力ポインタ a および b は、有効な double 型の値を指している必要があります。
  • 本関数は inet 型から変換されたスカラ値の比較を前提としています。

アルゴリズムまたは振る舞い

1. ポインタ a が指す double 値をデリファレンスします。 2. ポインタ b が指す double 値をデリファレンスします。 3. 両者を比較し、a が b より大きい場合に true を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/btree_gist/btree_inet.c:77（tinfo 構造体の初期化時に参照されています）。
  • **呼び出し先**:
  • 直接的な関数呼び出しはありません。

関連データ構造

  • double: inet 型から変換されたスカラ表現として使用されます。

失敗モードと境界条件

  • ポインタが NULL の場合、デリファレンスによりセグメンテーション違反が発生します。
  • double 型の精度限界により、非常に近い値を持つ inet アドレス間での比較結果が期待通りにならない可能性があります。

保守担当者向けメモ

  • 本関数は btree_utils_num.c で定義される汎用的な GiST 比較ロジックの一部として利用されます。
  • flinfo 引数は現在の実装では使用されていませんが、シグネチャの互換性のために保持されています。
