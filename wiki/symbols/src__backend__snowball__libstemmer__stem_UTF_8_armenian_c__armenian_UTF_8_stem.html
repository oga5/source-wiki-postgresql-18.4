armenian_UTF_8_stem

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_armenian.c:518

目的

アルメニア語のテキストに対して、接尾辞の除去および正規化を行うステミング処理のメインエントリーポイントです。

シグネチャまたは形状

int armenian_UTF_8_stem(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z（ステミング環境を保持する構造体）
  • **出力**: 成功時に 1、失敗時に負の値を返します。

前提条件と不変条件

  • z は有効な SN_env 構造体である必要があります。
  • 内部で r_mark_regions を呼び出し、単語内の領域（R2）を特定します。

アルゴリズムまたは振る舞い

1. r_mark_regions(z) を実行し、単語内の領域を計算します。 2. ステミング対象の範囲を z->I[1]（R2領域の開始位置）以降に制限します。 3. 以下の順序で接尾辞除去関数を適用し、各ステップでカーソル位置を復元します。

  • r_ending
  • r_verb
  • r_adjective
  • r_noun

    • 4. 処理終了後、カーソルを初期位置に戻し、成功値 1 を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: ソースコード内の明示的な呼び出し元は提供されていませんが、stem_UTF_8_armenian.h で公開されています。
  • **呼び出し先**: r_mark_regions, r_ending, r_verb, r_adjective, r_noun を呼び出します。

関連データ構造

  • struct SN_env: ステミング環境を管理する構造体。
  • among 構造体（a_0 ～ a_3）: 接尾辞マッチングに使用されます。

失敗モードと境界条件

  • r_mark_regions が負の値を返した場合、その値をそのまま返して終了します。
  • 各接尾辞除去関数が負の値を返した場合、その値をそのまま返して終了します。
  • 単語の現在の位置が z->I[1] より小さい場合、ステミングを行わず 0 を返します。
