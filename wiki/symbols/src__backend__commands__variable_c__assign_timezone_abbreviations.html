assign_timezone_abbreviations

定義

  • 種別: function
  • 定義箇所: src/backend/commands/variable.c:518

目的

GUC（Grand Unified Configuration）パラメータ timezone_abbreviations の設定値をシステムに適用し、タイムゾーン省略形のテーブルをインストールします。

シグネチャまたは形状

void assign_timezone_abbreviations(const char *newval, void *extra)

入力と出力

  • newval: 設定される新しい値（文字列）。本関数内では使用されません。
  • extra: check_timezone_abbreviations によって検証・準備された TimeZoneAbbrevTable へのポインタ。
  • 戻り値: なし。

前提条件と不変条件

  • extra が NULL の場合、本関数は何も行わず即座に復帰します（デフォルト値の処理用）。
  • extra が非 NULL である場合、有効な TimeZoneAbbrevTable 構造体を指している必要があります。

アルゴリズムまたは振る舞い

1. extra が NULL かどうかを確認し、該当する場合は処理を終了します。 2. extra が存在する場合、InstallTimeZoneAbbrevTable（ソース内では InstallTimeZoneAbbrevs と表記）を呼び出し、指定されたタイムゾーン省略形テーブルを現在のセッションに適用します。

呼び出し元と呼び出し先

  • 呼び出し元: GUCサブシステム（src/backend/utils/misc/guc_tables.c にてフックとして登録）。
  • 呼び出し先: InstallTimeZoneAbbrevs（タイムゾーン省略形テーブルのインストールを実行）。

関連データ構造

  • TimeZoneAbbrevTable: タイムゾーンの省略形とオフセットの対応を保持する構造体。

失敗モードと境界条件

  • extra が NULL の場合は安全に無視されます。
  • extra が不正なポインタである場合、InstallTimeZoneAbbrevs 呼び出し時に未定義の動作が発生する可能性があります。

保守担当者向けメモ

  • 本関数は assign_hook として機能します。検証ロジックは対応する check_timezone_abbreviations に集約されています。
  • InstallTimeZoneAbbrevs の詳細な動作については、utils/tzparser.h および関連する実装を参照してください。
