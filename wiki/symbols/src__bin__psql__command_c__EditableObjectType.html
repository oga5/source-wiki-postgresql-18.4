EditableObjectType

定義

  • 種別: enum
  • 定義箇所: src/bin/psql/command.c:50

目的

psql において編集対象となるデータベースオブジェクトの型を識別するために使用されます。

シグネチャまたは形状

typedef enum EditableObjectType
{
	EditableFunction,
	EditableView,
} EditableObjectType;

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: EditableFunction または EditableView のいずれかの値。

前提条件と不変条件

  • psql 内でオブジェクトの編集や定義取得を行う際の型判別に使用されます。

アルゴリズムまたは振る舞い

  • EditableFunction は関数オブジェクトを指します。
  • EditableView はビューオブジェクトを指します。
  • 実行時に is_func フラグ等の条件判定に基づき、適切な列挙値が選択されます。

呼び出し元と呼び出し先

  • 呼び出し元（参照箇所）:
  • src/bin/psql/command.c:1463
  • src/bin/psql/command.c:2987
  • 呼び出し先:
  • lookup_object_oid (引数として使用)
  • get_create_object_cmd (引数として使用)

関連データ構造

  • なし。

失敗モードと境界条件

  • 列挙型に含まれない値を代入した場合の挙動は未定義です。

保守担当者向けメモ

  • 新たな編集可能オブジェクト型を追加する場合は、この列挙型を拡張し、lookup_object_oid および get_create_object_cmd の各関数内で対応する処理を追加してください。
