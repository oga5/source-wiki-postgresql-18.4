JsonbValueToJsonb

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/jsonb_util.c:91

目的

メモリ上の JsonbValue 構造体を、PostgreSQL の jsonb 型の物理的なバイナリ表現（Jsonb）に変換します。

シグネチャまたは形状

Jsonb *JsonbValueToJsonb(JsonbValue *val)

入力と出力

  • **入力**: JsonbValue *val - 変換対象の JSONB 値。
  • **出力**: Jsonb * - 変換後の jsonb バイナリデータ。

前提条件と不変条件

  • 入力 val が jbvBinary 型の場合、そのバイナリデータは有効な jsonb 形式である必要があります。
  • スカラー値が渡された場合、内部で一時的な配列として構築されます。

アルゴリズムまたは振る舞い

1. **スカラー値の場合**: IsAJsonbScalar(val) が真であれば、一時的な JsonbParseState を作成し、スカラー値を単一要素の配列としてラップした上で convertToJsonb を呼び出します。 2. **コンテナ（オブジェクト/配列）の場合**: val->type が jbvObject または jbvArray であれば、直接 convertToJsonb を呼び出してバイナリ化します。 3. **バイナリ値の場合**: val->type が jbvBinary であれば、メモリを確保し、既存のバイナリデータをコピーして返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/hstore/hstore_io.c
  • contrib/jsonb_plperl/jsonb_plperl.c
  • contrib/jsonb_plpython/jsonb_plpython.c
  • src/backend/executor/execExprInterp.c
  • src/backend/utils/adt/jsonb.c
  • **呼び出し先**:
  • pushJsonbValue (スカラー変換時)
  • convertToJsonb (コンテナ変換時)
  • palloc / memcpy (バイナリコピー時)

関連データ構造

  • JsonbValue: メモリ上の JSONB ノード表現。
  • Jsonb: ディスク上の JSONB バイナリ表現。
  • JsonbParseState: 構築中の JSONB 状態を保持するスタック。

失敗モードと境界条件

  • メモリ不足時には palloc によりエラーが発生します。
  • val が不正な型を持つ場合、Assert によりデバッグビルドで停止します。

保守担当者向けメモ

  • 本関数は jsonb 構築の最終段階で頻繁に使用されます。
  • スカラー値を配列としてラップする処理は、jsonb のバイナリ仕様がコンテナ（配列またはオブジェクト）をルートに持つ必要があることに起因しています。
