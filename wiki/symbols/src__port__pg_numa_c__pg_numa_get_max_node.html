pg_numa_get_max_node

定義

  • 種別: function
  • 定義箇所: src/port/pg_numa.c:115

目的

システム上で利用可能な最大NUMAノード番号を取得します。

シグネチャまたは形状

int pg_numa_get_max_node(void)

入力と出力

  • 入力: なし
  • 出力: システム上の最大NUMAノード番号（整数値）を返します。

前提条件と不変条件

  • USE_LIBNUMA が定義されている環境で有効です。
  • libnuma ライブラリがシステムに存在し、適切にリンクされている必要があります。

アルゴリズムまたは振る舞い

Linuxの libnuma が提供する numa_max_node() を呼び出し、その結果をそのまま返します。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/storage/ipc/shmem.c:569 にて、共有メモリ関連の初期化処理の一部として呼び出されています。
  • 呼び出し先: numa_max_node()（libnuma ライブラリ関数）。

関連データ構造

  • なし

失敗モードと境界条件

  • USE_LIBNUMA が定義されていない環境では、本関数は無効化されるか、ダミー動作となります。
  • libnuma が利用できない環境での挙動は、ビルド時の設定に依存します。

保守担当者向けメモ

  • 本関数は libnuma への単なるラッパーです。
  • USE_LIBNUMA が定義されていない環境での動作については、src/include/port/pg_numa.h の定義を確認してください。
