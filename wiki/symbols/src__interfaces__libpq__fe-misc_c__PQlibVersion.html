PQlibVersion

定義

  • 種別: function
  • 定義箇所: src/interfaces/libpq/fe-misc.c:62

目的

現在使用している libpq のバージョン番号を整数として取得する。

シグネチャまたは形状

int PQlibVersion(void);

入力と出力

  • 入力: なし
  • 出力: int 型の libpq バージョン番号

前提条件と不変条件

  • コンパイル時に定義される PG_VERSION_NUM マクロの値が返される。

アルゴリズムまたは振る舞い

  • 呼び出されると、定数 PG_VERSION_NUM をそのまま返す。

呼び出し元と呼び出し先

  • 呼び出し元: 提供されたソース情報からは特定できません。
  • 呼び出し先: なし。

関連データ構造

  • なし。

失敗モードと境界条件

  • 失敗することはありません。常に現在のライブラリのバージョンを返します。

保守担当者向けメモ

  • 本関数は libpq-fe.h で公開 API として宣言されています。バージョン番号の形式については PG_VERSION_NUM の定義に従います。
