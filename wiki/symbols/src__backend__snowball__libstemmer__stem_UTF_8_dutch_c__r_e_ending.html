r_e_ending

定義

目的

オランダ語のステミング処理において、単語末尾の 'e' を削除し、必要に応じて二重子音を解消する。

シグネチャまたは形状

static int r_e_ending(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z （Snowball ステミング環境）
  • 出力: 成功時に 1、失敗または条件不一致時に 0、エラー時に負の値を返す。

前提条件と不変条件

  • z->c が単語の末尾を指していること。
  • r_R1 が定義する領域内に接尾辞が存在すること。
  • 接尾辞の直前に母音（g_v グループ）が存在すること。

アルゴリズムまたは振る舞い

1. 環境内の整数配列 z->I[3] を 0 に初期化する。 2. 現在位置 z->c を z->ket に設定する。 3. 現在位置がバッファの先頭以降であり、かつ直前の文字が 'e' であることを確認する。 4. z->c をデクリメントし、z->bra を設定する。 5. r_R1(z) を呼び出し、削除対象が R1 領域内にあるか検証する。 6. out_grouping_b_U を使用し、削除対象の 'e' の直前に母音が存在するかを確認する。 7. slice_del(z) で 'e' を削除する。 8. z->I[3] を 1 に設定し、r_undouble(z) を呼び出して二重子音の解消を行う。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_dutch.c 内の r_standard_suffix 等のステミングロジックから呼び出される（詳細はソースコードの制御フローに依存）。
  • 呼び出し先: r_R1, out_grouping_b_U, slice_del, r_undouble。

関連データ構造

  • struct SN_env: ステミングの実行状態を保持する構造体。
  • g_v: 母音を定義するグループ（out_grouping_b_U で使用）。

失敗モードと境界条件

  • z->c <= z->lb: バッファの境界に達している場合は 0 を返す。
  • p[z->c - 1] != 'e': 末尾が 'e' でない場合は 0 を返す。
  • r_R1 または slice_del が負の値を返した場合、その値をそのまま伝播させる。
