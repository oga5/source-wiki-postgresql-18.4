pg_comp_crc32c_armv8

定義

  • 種別: function
  • 定義箇所: src/port/pg_crc32c_armv8.c:21

目的

ARMv8 プロセッサの CRC 拡張命令（CRC32C）を利用し、指定されたデータに対する CRC-32C チェックサムを高速に計算する。

シグネチャまたは形状

pg_crc32c pg_comp_crc32c_armv8(pg_crc32c crc, const void *data, size_t len)

入力と出力

  • **入力**:
  • crc: 計算の初期値（または前回の計算結果）。
  • data: チェックサム計算対象のデータポインタ。
  • len: データのバイト長。
  • **出力**: 計算後の CRC-32C 値を返す。

前提条件と不変条件

  • コンパイル時に ARMv8 CRC 拡張を有効にするフラグ（例: -march=armv8-a+crc）が必要。
  • data ポインタが指すメモリ領域は、少なくとも len バイト分有効であること。

アルゴリズムまたは振る舞い

1. **アライメント調整**: 入力ポインタが 8 バイト境界に揃っていない場合、先頭の 1, 2, 4 バイトを __crc32cb, __crc32ch, __crc32cw 命令で順次処理し、ポインタを 8 バイト境界に合わせる。 2. **メインループ**: 8 バイト単位で __crc32cd 命令を繰り返し実行し、高速に計算を行う。 3. **残余処理**: ループ終了後、残った 7 バイト以下のデータを 4, 2, 1 バイト単位で順次処理し、最終的な値を算出する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/port/pg_crc32c_armv8_choose.c: 実行時に本関数を選択するロジックで使用。
  • src/include/port/pg_crc32c.h: マクロ経由での呼び出し。
  • **呼び出し先**:
  • ARM C Language Extensions (arm_acle.h) で定義された組み込み関数: __crc32cb, __crc32ch, __crc32cw, __crc32cd。

関連データ構造

  • pg_crc32c: チェックサム値を保持する型（port/pg_crc32c.h で定義）。

失敗モードと境界条件

  • len が 0 の場合、入力された crc 値をそのまま返す。
  • メモリアライメントが 8 バイトでない場合でも、命令の仕様上は動作するが、アライメント調整処理を経て効率的に計算される。

保守担当者向けメモ

  • 本関数はハードウェアアクセラレーション層である。ソフトウェア実装（pg_crc32c_sb8.c）との切り替えロジックは pg_crc32c_armv8_choose.c に依存している。
  • PointerIsAligned マクロによるアライメント調整は、ハードウェアの仕様上必須ではないが、メモリバスの効率を最大化するために実装されている。
