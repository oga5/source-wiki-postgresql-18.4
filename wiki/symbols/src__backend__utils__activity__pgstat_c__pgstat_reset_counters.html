pgstat_reset_counters

定義

  • 種別: function
  • 定義箇所: src/backend/utils/activity/pgstat.c:833

目的

現在のデータベースに関連する統計カウンタをリセットします。

シグネチャまたは形状

void pgstat_reset_counters(void)

入力と出力

  • 入力: なし（内部的に MyDatabaseId を使用）
  • 出力: なし（副作用として共有メモリ上の統計情報が更新される）

前提条件と不変条件

  • 呼び出し時に有効なデータベースコンテキスト（MyDatabaseId）が存在している必要があります。

アルゴリズムまたは振る舞い

1. GetCurrentTimestamp() を呼び出し、現在時刻を取得します。 2. match_db_entries を条件として pgstat_reset_matching_entries を呼び出し、現在のデータベースに関連付けられた統計エントリをリセットします。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/backend/utils/adt/pgstatfuncs.c:1872
  • 呼び出し先:
  • GetCurrentTimestamp()
  • pgstat_reset_matching_entries()

関連データ構造

  • MyDatabaseId: 現在のバックエンドが接続しているデータベースの識別子。
  • match_db_entries: データベース単位で統計エントリを照合するための内部関数（pgstat.c 内で定義）。

失敗モードと境界条件

  • データベースに接続していない状態で呼び出された場合の挙動については、本ソース抜粋からは判断できません。

保守担当者向けメモ

  • 本関数は pgstat_reset_matching_entries を介して統計リセットの共通ロジックを利用しています。特定の統計種別のみをリセットしたい場合は、pgstat_reset_of_kind 等の他のインターフェースを検討してください。
