GetStartupBufferPinWaitBufId

定義

  • 種別: function
  • 定義箇所: src/backend/storage/lmgr/proc.c:765

目的

共有メモリ上のグローバルヘッダーから、現在スタートアッププロセスがバッファピンの解放を待機しているバッファIDを取得します。

シグネチャまたは形状

int GetStartupBufferPinWaitBufId(void)

入力と出力

  • **入力**: なし
  • **出力**: int 型のバッファID。

前提条件と不変条件

  • ProcGlobal が初期化済みである必要があります。
  • コードの再配置（コンパイラによる最適化）を防ぐため、ProcGlobal を volatile ポインタとして参照しています。

アルゴリズムまたは振る舞い

1. 共有メモリ上のグローバル変数 ProcGlobal を volatile ポインタとして取得します。 2. procglobal->startupBufferPinWaitBufId に格納されている値を読み取り、呼び出し元へ返却します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/storage/buffer/bufmgr.c:5824
  • src/backend/tcop/postgres.c:3132
  • **呼び出し先**:
  • 共有メモリ上の ProcGlobal 構造体（直接参照）

関連データ構造

  • PROC_HDR: ProcGlobal が指す共有メモリ上の構造体。startupBufferPinWaitBufId フィールドを保持します。

失敗モードと境界条件

  • ProcGlobal が未初期化の状態で呼び出された場合の動作は定義されていません。
  • 戻り値が負の値である場合、待機中のバッファが存在しないことを示す可能性があります（src/backend/tcop/postgres.c での比較ロジックより）。

保守担当者向けメモ

  • 本関数は SetStartupBufferPinWaitBufId と対になるアクセサです。
  • 共有メモリ上の値を直接読み取るため、マルチプロセス環境下での整合性は ProcGlobal の設計に依存します。
