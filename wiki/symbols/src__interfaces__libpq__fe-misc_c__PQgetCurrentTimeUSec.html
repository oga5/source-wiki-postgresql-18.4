PQgetCurrentTimeUSec

定義

  • 種別: function
  • 定義箇所: src/interfaces/libpq/fe-misc.c:1234

目的

マイクロ秒精度で現在時刻を取得します。

シグネチャまたは形状

pg_usec_time_t PQgetCurrentTimeUSec(void)

入力と出力

  • 入力: なし
  • 出力: pg_usec_time_t 型の現在時刻（マイクロ秒単位）

前提条件と不変条件

  • システムの gettimeofday が利用可能であること。
  • 返り値は tv_sec をマイクロ秒に換算し、tv_usec を加算した値となります。

アルゴリズムまたは振る舞い

1. gettimeofday を呼び出し、現在の時刻を struct timeval に格納します。 2. tv_sec に 1,000,000 を乗算してマイクロ秒に変換し、tv_usec を加算して返します。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/bin/psql/command.c:4418
  • src/interfaces/libpq/fe-connect.c:2821
  • src/interfaces/libpq/fe-misc.c:1167
  • src/interfaces/libpq/fe-misc.c:1208
  • 呼び出し先:
  • gettimeofday (システムコール)

関連データ構造

  • pg_usec_time_t: 時刻を保持するための型。
  • struct timeval: gettimeofday で使用される時刻構造体。

失敗モードと境界条件

  • gettimeofday の失敗については本関数内で明示的なハンドリングを行っていません。

保守担当者向けメモ

  • 本関数は libpq-fe.h で公開されており、タイムアウト計算等の基盤として使用されています。
  • PQsocketPoll と共に、libpq の時間計測機能の主要な構成要素です。
