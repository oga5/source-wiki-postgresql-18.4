EncodeSpecialTimestamp

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/timestamp.c:1586

目的

Timestamp 型の特殊値（無限大）を、対応する文字列形式に変換します。

シグネチャまたは形状

void EncodeSpecialTimestamp(Timestamp dt, char *str)

入力と出力

  • **入力**:
  • dt: 変換対象の Timestamp 値。
  • str: 結果を格納するためのバッファ（呼び出し元で十分な領域を確保している必要がある）。
  • **出力**: なし（str に文字列を書き込む）。

前提条件と不変条件

  • dt は TIMESTAMP_IS_NOBEGIN または TIMESTAMP_IS_NOEND のいずれかである必要があります。
  • str は EARLY または LATE の文字列を格納できる十分なサイズが必要です。

アルゴリズムまたは振る舞い

1. TIMESTAMP_IS_NOBEGIN(dt) が真の場合、str に EARLY（負の無限大）をコピーします。 2. TIMESTAMP_IS_NOEND(dt) が真の場合、str に LATE（正の無限大）をコピーします。 3. 上記のいずれでもない場合、elog(ERROR, ...) を呼び出し、エラーを発生させます。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/json.c
  • src/backend/utils/adt/timestamp.c
  • src/interfaces/ecpg/pgtypeslib/timestamp.c
  • **呼び出し先**:
  • strcpy (標準ライブラリ)
  • elog (PostgreSQL エラー報告)

関連データ構造

  • Timestamp: マイクロ秒単位の int64 で保持される日時型。

失敗モードと境界条件

  • dt が無限大（infinity / -infinity）以外の値である場合、elog(ERROR) によりトランザクションが中断されます。

保守担当者向けメモ

  • 本関数は EARLY および LATE マクロに依存しています。これらは datetime.h 等で定義されている文字列定数です。
  • str バッファのオーバーフロー防止策は呼び出し元に委ねられているため、変更を加える際は注意してください。
