ResetPromoteSignaled

定義

  • 種別: function
  • 定義箇所: src/backend/postmaster/startup.c:293

目的

スタンバイからプライマリへの昇格要求フラグ（promote_signaled）をリセットし、偽（false）に設定します。

シグネチャまたは形状

void ResetPromoteSignaled(void)

入力と出力

  • 入力: なし
  • 出力: なし（副作用として静的変数 promote_signaled を更新）

前提条件と不変条件

  • 本関数は、昇格処理の完了後、または昇格要求を無効化すべきタイミングで呼び出されることを想定しています。

アルゴリズムまたは振る舞い

1. 内部変数 promote_signaled に false を代入します。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/backend/access/transam/xlogrecovery.c:4484
  • 呼び出し先:
  • なし

関連データ構造

  • promote_signaled: 本関数が操作する静的フラグ変数。

失敗モードと境界条件

  • 特筆すべき失敗モードはありません。

保守担当者向けメモ

  • 本関数は promote_signaled の状態を管理する唯一の手段ではありません。昇格要求の発生（セット）については StartupProcTriggerHandler 等のシグナルハンドラを確認してください。
