ecpg_isspace

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/preproc/parser.c:297

目的

Flexスキャナが空白文字とみなす文字かどうかを判定し、字句解析時のフィルタリングを補助する。

シグネチャまたは形状

static bool ecpg_isspace(char ch)

入力と出力

  • **入力**: char ch （判定対象の文字）
  • **出力**: bool （対象文字が空白であれば true、それ以外は false）

前提条件と不変条件

  • Flexスキャナの空白定義と一致させる必要がある。

アルゴリズムまたは振る舞い

引数 ch が以下のいずれかの文字である場合に true を返し、それ以外の場合は false を返す。

  • 空白 (' ')
  • 水平タブ ('\t')
  • 改行 ('\n')
  • 復帰 ('\r')
  • 改ページ ('\f')

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/interfaces/ecpg/preproc/parser.c:288 (check_uescapechar 関数内)
  • **呼び出し先**: なし（標準的な比較演算のみを実行）

関連データ構造

  • なし

失敗モードと境界条件

  • 入力が char 型の範囲外である場合の動作は定義されていない。

保守担当者向けメモ

  • check_uescapechar() 関数と本関数は、Flexスキャナの定義と整合性を保つ必要がある。修正時は両者の定義が一致しているか確認すること。
