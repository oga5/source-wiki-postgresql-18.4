CreateReplicationSlot

定義

  • 種別: function
  • 定義箇所: src/backend/replication/walsender.c:1190

目的

クライアントからのリクエストに基づき、物理または論理レプリケーションスロットを作成し、必要に応じてスロット情報の永続化やスナップショットの生成・エクスポートを行う。

シグネチャまたは形状

static void CreateReplicationSlot(CreateReplicationSlotCmd *cmd);

入力と出力

  • **入力**: CreateReplicationSlotCmd *cmd（スロット名、プラグイン名、一時的フラグ、スナップショット操作等の設定を含む構造体）。
  • **出力**: なし（関数内で DestRemoteSimple を用いて結果をクライアントへ送信する）。

前提条件と不変条件

  • MyReplicationSlot が NULL であること（既にスロットを保持していないこと）。
  • 論理レプリケーションの場合、CheckLogicalDecodingRequirements() を満たすこと。
  • スナップショット操作（CRS_USE_SNAPSHOT）を行う場合、トランザクションブロック内であること、REPEATABLE READ 隔離レベルであること、読み取り専用トランザクションであること、クエリ実行前であること、サブトランザクションでないことが必須。

アルゴリズムまたは振る舞い

1. **オプション解析**: parseCreateReplSlotOptions でコマンド引数を解析。 2. **物理スロット作成**: ReplicationSlotCreate を呼び出し、必要に応じて ReplicationSlotReserveWal で WAL を予約し、ディスクへ保存。 3. **論理スロット作成**:

  • ReplicationSlotCreate で一時的またはエフェメラルなスロットを作成。
  • CreateInitDecodingContext でデコーディングコンテキストを初期化。
  • DecodingContextFindStartpoint で開始地点を特定。
  • スナップショット操作（エクスポートまたは使用）を実行。
  • FreeDecodingContext でコンテキストを解放し、永続スロットの場合は ReplicationSlotPersist で保存。

    • 4. **結果送信**: 作成されたスロット名、整合性ポイント（LSN）、スナップショット名、プラグイン名を含むタプルを構築し、DestRemoteSimple を介してクライアントへ送信。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/replication/walsender.c 内のコマンドディスパッチャ（exec_replication_command 等）から呼び出される。
  • **呼び出し先**: ReplicationSlotCreate, ReplicationSlotReserveWal, CreateInitDecodingContext, DecodingContextFindStartpoint, SnapBuildExportSnapshot, SnapBuildInitialSnapshot, RestoreTransactionSnapshot 等。

関連データ構造

  • CreateReplicationSlotCmd: コマンド引数。
  • ReplicationSlot: 共有メモリ上のスロット状態。
  • LogicalDecodingContext: 論理デコード用コンテキスト。
  • TupleDesc: 送信データの構造定義。

失敗モードと境界条件

  • トランザクション状態の不整合（SNAPSHOT 'use' 時の制約違反）により ereport(ERROR) が発生する。
  • スロット名が重複している場合や、論理デコードの前提条件を満たさない場合にエラーとなる。
  • 一時的スロットはエラー発生時に自動的に破棄される設計となっている。

保守担当者向けメモ

  • 本関数は物理と論理の両方のスロット作成を担うため、条件分岐が複雑である。
  • 論理レプリケーションの初期化プロセス（DecodingContextFindStartpoint）は時間がかかる可能性があるため、タイムアウト処理の無効化（last_reply_timestamp = 0）が行われている点に注意。
  • pg_basebackup 等の外部ツールから呼び出される CreateReplicationSlot（src/bin/pg_basebackup/streamutil.c 等）とは別の関数であることに留意すること。
