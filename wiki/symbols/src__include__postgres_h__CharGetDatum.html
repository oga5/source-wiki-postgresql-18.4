CharGetDatum

定義

  • 種別: static inline 関数
  • 定義箇所: src/include/postgres.h:126

目的

C 言語の char 型の値を、PostgreSQL の汎用データ表現である Datum 型に変換する。

シグネチャまたは形状

static inline Datum CharGetDatum(char X)

入力と出力

  • **入力**: char X （変換対象の文字値）
  • **出力**: Datum （X を保持する Datum 型の値）

前提条件と不変条件

  • Datum 型は sizeof(void *) と同じサイズであると仮定されており、本関数は char 型の値を Datum 型へキャストして返します。

アルゴリズムまたは振る舞い

引数として受け取った char 型の値 X を、Datum 型へ明示的にキャストして返します。インライン関数として定義されているため、関数呼び出しのオーバーヘッドなしに型変換を行います。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/btree_gin/btree_gin.c
  • src/backend/access/nbtree/nbtcompare.c
  • src/backend/catalog/aclchk.c
  • src/backend/catalog/heap.c
  • **呼び出し先**:
  • 本関数は単一のキャスト操作のみを行うため、他の関数を呼び出しません。

関連データ構造

  • Datum: PostgreSQL における汎用データ型。

失敗モードと境界条件

  • 本関数自体は単純なキャストであるため、関数内部での失敗モードはありません。
  • char 型のサイズが Datum 型のサイズを超える環境では、切り捨てが発生する可能性があります。

保守担当者向けメモ

  • 本関数は postgres.h 内の他の *GetDatum 関数群と同様、型安全性を確保しつつ効率的なデータ変換を提供します。
  • 移植性を維持するため、char 型の値を Datum に変換する際は、直接キャストせず必ず本関数を使用してください。
