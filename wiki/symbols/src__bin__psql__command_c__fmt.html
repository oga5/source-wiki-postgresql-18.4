fmt

定義

  • 種別: struct
  • 定義箇所: src/bin/psql/command.c:5081

目的

psql における出力フォーマットの名称と、それに対応する内部的な列挙型値をマッピングするために使用されます。

シグネチャまたは形状

struct fmt {
    const char *name;
    enum printFormat number;
};

入力と出力

  • 本構造体はデータ保持用であり、関数ではありません。
  • name: フォーマット名を表す文字列（例: "unaligned", "aligned" 等）。
  • number: fe_utils/print.h で定義される printFormat 列挙型の値。

前提条件と不変条件

  • src/bin/psql/command.c 内の静的配列 formats[] の要素として定義されています。
  • name は NULL 終端文字列である必要があります。

アルゴリズムまたは振る舞い

  • psql の出力形式設定コマンド（\pset format など）において、ユーザーが指定した文字列を formats[] 配列内で検索し、対応する printFormat 値を特定するために利用されます。

呼び出し元と呼び出し先

  • 呼び出し元および呼び出し先の具体的な関数情報は、提供されたソース抜粋からは特定できません。
  • ツールによる検索結果（contrib/dblink や xlogreader.c 等の fmt）は、本シンボルとは無関係な汎用的な書式指定文字列（format string）を指しており、本構造体とは別物です。

関連データ構造

  • enum printFormat: fe_utils/print.h で定義される、出力形式を規定する列挙型。
  • formats[]: src/bin/psql/command.c 内で定義される struct fmt 型の静的配列。

失敗モードと境界条件

  • 構造体自体に失敗モードはありません。
  • 検索時に一致する name が見つからない場合、呼び出し側のロジックでエラーハンドリングを行う必要があります。

保守担当者向けメモ

  • 本シンボルは psql の出力設定に関連する内部的なマッピングテーブルの一部です。
  • ツールが提示した他のファイル（dblink.c 等）の fmt は、変数名や引数名としての利用であり、本構造体とは一切関係ありません。混同しないよう注意してください。
