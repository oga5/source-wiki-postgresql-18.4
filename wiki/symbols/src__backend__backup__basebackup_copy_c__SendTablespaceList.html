SendTablespaceList

定義

  • 種別: function
  • 定義箇所: src/backend/backup/basebackup_copy.c:377

目的

バックアップ対象のテーブルスペース情報をクライアントへ送信する。具体的には、テーブルスペースのOID、パス、およびサイズを含む結果セットを構築し、COPYプロトコル経由で転送する。

シグネチャまたは形状

static void SendTablespaceList(List *tablespaces)

入力と出力

  • **入力**: tablespaces (テーブルスペース情報 tablespaceinfo を保持するリスト)
  • **出力**: なし（副作用としてクライアントへデータ行を送信）

前提条件と不変条件

  • DestRemoteSimple を用いた DestReceiver が作成可能であること。
  • tablespaces リスト内の各要素は tablespaceinfo 構造体へのポインタであること。
  • ti->size が負の値の場合、サイズ情報は NULL として送信される。

アルゴリズムまたは振る舞い

1. DestRemoteSimple を使用して DestReceiver を初期化する。 2. 3列（spcoid: OID, spclocation: TEXT, size: INT8）のタプル記述子を作成する。 3. begin_tup_output_tupdesc を呼び出し、出力ストリームを開始する。 4. tablespaces リストを走査し、各テーブルスペースについて以下の処理を行う：

  • パスが NULL の場合は OID とパスを NULL に設定する。
  • パスが存在する場合は OID とパスを変換して設定する。
  • サイズが 0 以上であれば、キロバイト単位（size / 1024）に変換して設定する。
  • do_tup_output を呼び出し、1行分のデータを送信する。

    • 5. end_tup_output を呼び出し、出力ストリームを終了する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/backup/basebackup_copy.c 内の関数から呼び出される（詳細はソースコードの文脈に依存）。
  • **呼び出し先**: CreateDestReceiver, CreateTemplateTupleDesc, TupleDescInitBuiltinEntry, begin_tup_output_tupdesc, do_tup_output, end_tup_output 等のバックエンドユーティリティ関数。

関連データ構造

  • tablespaceinfo: テーブルスペースの情報を保持する構造体。
  • TupOutputState: タプル出力状態を管理する構造体。
  • TupleDesc: タプルの列定義を保持する構造体。

失敗モードと境界条件

  • tablespaces が空の場合、ヘッダー情報のみが送信され、データ行は送信されない。
  • ti->path が NULL の場合、OID とパスの列は NULL として扱われる。
  • ti->size が負の場合、サイズ列は NULL として扱われる。

保守担当者向けメモ

  • 本関数は DestRemoteSimple を使用しているため、クライアントとの通信プロトコルに強く依存している。列定義やデータ型を変更する場合は、クライアント側の解釈と整合性を取る必要がある。
