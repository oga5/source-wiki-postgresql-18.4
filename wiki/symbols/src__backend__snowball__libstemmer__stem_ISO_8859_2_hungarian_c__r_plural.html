r_plural

定義

  • **種別**: function
  • **定義箇所**: src/backend/snowball/libstemmer/stem_ISO_8859_2_hungarian.c:674

目的

ハンガリー語のステミング処理において、単語末尾の複数形接尾辞を検出し、ルールに基づいて削除または置換を行う。

シグネチャまたは形状

static int r_plural(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（ステミング環境の状態を保持する構造体）
  • **出力**: int（成功時に 1、失敗または適用対象外の場合に 0、エラー発生時に負の値を返す）

前提条件と不変条件

  • z->c が有効な単語境界内にあること。
  • 単語の末尾（z->c - 1）が文字コード 107（'k'）であること。
  • r_R1 関数により、対象の接尾辞が R1 領域内にあることが検証されること。

アルゴリズムまたは振る舞い

1. find_among_b を使用し、a_8 配列（7つのパターン）に基づいて単語末尾から接尾辞を検索する。 2. マッチした場合、r_R1 を呼び出し、該当箇所が R1 領域内であることを確認する。 3. マッチしたパターン（among_var）に応じて以下のいずれかの操作を行う：

  • 1: s_6 に置換。
  • 2: s_7 に置換。
  • 3: 接尾辞を削除（slice_del）。

    • 4. 操作が成功すれば 1 を返し、失敗すればエラーコードを伝播する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_ISO_8859_2_hungarian.c:851 および src/backend/snowball/libstemmer/stem_UTF_8_hungarian.c:857 から呼び出される。
  • **呼び出し先**: find_among_b, r_R1, slice_from_s, slice_del（いずれも Snowball ランタイム関数）。

関連データ構造

  • SN_env: ステミング環境を管理する構造体。
  • a_8: 複数形接尾辞を定義した among 構造体の配列。

失敗モードと境界条件

  • 単語末尾が 'k' でない場合、または find_among_b でマッチしない場合は 0 を返す。
  • r_R1 が R1 領域外と判定した場合、または slice 操作中にメモリやバッファエラーが発生した場合は負の値を返す。

保守担当者向けメモ

  • among_var のケース分岐は a_8 配列の定義と密接に関連しているため、接尾辞リストを変更する際は整合性に注意が必要。
