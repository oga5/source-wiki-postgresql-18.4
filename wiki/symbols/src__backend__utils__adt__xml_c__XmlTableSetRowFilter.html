XmlTableSetRowFilter

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/xml.c:4813

目的

XMLTABLE 演算子において、行を抽出するための XPath 式をコンパイルし、実行コンテキストに設定します。

シグネチャまたは形状

static void XmlTableSetRowFilter(TableFuncScanState *state, const char *path)

入力と出力

  • state: TableFuncScanState 構造体へのポインタ。内部の XmlTableBuilderData を取得するために使用されます。
  • path: 行フィルタとして使用する XPath 式の文字列。
  • 出力: なし（副作用として xtCxt->xpathcomp にコンパイル済み XPath オブジェクトを格納）。

前提条件と不変条件

  • USE_LIBXML が定義されている環境であること。
  • XmlTableSetDocument が既に呼び出され、xtCxt->xpathcxt が初期化されていること。
  • path は空文字列であってはならない。

アルゴリズムまたは振る舞い

1. GetXmlTableBuilderPrivateData を介して、state から XmlTableBuilderData を取得します。 2. path が空文字列の場合、ERRCODE_INVALID_ARGUMENT_FOR_XQUERY エラーを発生させます。 3. path を libxml2 形式の文字列に複製します。 4. xmlXPathCtxtCompile を使用して XPath 式をコンパイルし、結果を xtCxt->xpathcomp に保存します。 5. コンパイルに失敗した場合、または libxml2 側でエラーが発生した場合は、xml_ereport を通じてエラーを報告します。 6. USE_LIBXML が未定義の場合は NO_XML_SUPPORT() を呼び出します。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/utils/adt/xml.c 内の XmlTableRoutine 構造体の .SetRowFilter フィールド経由で呼び出されます。
  • 呼び出し先: GetXmlTableBuilderPrivateData, pg_xmlCharStrndup, xmlXPathCtxtCompile, xml_ereport。

関連データ構造

  • TableFuncScanState: 実行時の状態を保持する構造体。
  • XmlTableBuilderData: XMLTABLE 実行に必要なコンテキスト（XPath コンテキストやコンパイル済み式など）を保持する構造体。

失敗モードと境界条件

  • path が空文字列の場合: ERROR をスロー。
  • XPath 式が不正な場合: xmlXPathCtxtCompile が失敗し、xml_ereport により ERROR をスロー。
  • XmlTableSetDocument 未実行時: Assert によりデバッグビルドで停止。

保守担当者向けメモ

  • 本関数は libxml2 のグローバルな状態に依存するため、エラーハンドリングは PgXmlErrorContext を通じて行われます。
  • xpathcomp のメモリ管理は XmlTableBuilderData のライフサイクルに依存します。
