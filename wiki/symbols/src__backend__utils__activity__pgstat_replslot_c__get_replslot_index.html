get_replslot_index

定義

  • 種別: static int
  • 定義箇所: src/backend/utils/activity/pgstat_replslot.c:223

目的

指定された名前のレプリケーションスロットを検索し、対応する内部インデックスを返します。

シグネチャまたは形状

static int get_replslot_index(const char *name, bool need_lock)

入力と出力

  • **入力**:
  • name: 検索対象のスロット名（const char *）。
  • need_lock: スロット検索時にロックを取得するかどうかを示すフラグ（bool）。
  • **出力**:
  • 成功時: スロットのインデックス（int）。
  • 失敗時: -1。

前提条件と不変条件

  • name は NULL であってはなりません（Assert により検証）。

アルゴリズムまたは振る舞い

1. SearchNamedReplicationSlot(name, need_lock) を呼び出し、指定された名前のスロットを検索します。 2. スロットが見つからない場合は -1 を返します。 3. スロットが見つかった場合は、ReplicationSlotIndex(slot) を使用してそのスロットのインデックスを取得し、返却します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/activity/pgstat_replslot.c:177
  • src/backend/utils/activity/pgstat_replslot.c:204
  • **呼び出し先**:
  • SearchNamedReplicationSlot
  • ReplicationSlotIndex

関連データ構造

  • ReplicationSlot: レプリケーションスロットの情報を保持する構造体。

失敗モードと境界条件

  • 指定された名前のスロットが存在しない場合、-1 を返します。

保守担当者向けメモ

  • 本関数は pgstat_replslot.c 内の統計処理において、スロット名からインデックスへの変換を行うためのユーティリティです。
  • 戻り値の -1 は呼び出し側で適切にハンドリングする必要があります。
