Lookahead

定義

  • 種別: function
  • 定義箇所: contrib/fuzzystrmatch/fuzzystrmatch.c:320

目的

文字列内の指定されたオフセット位置にある文字を安全に取得します。

シグネチャまたは形状

static char Lookahead(char *word, int how_far)

入力と出力

  • **入力**:
  • word: 走査対象の文字列（char *）。
  • how_far: 現在位置から先読みする距離（int）。
  • **出力**:
  • 指定位置の文字（char）。文字列の終端に達した場合はヌル文字 ('\0') を返します。

前提条件と不変条件

  • word はヌル終端文字列である必要があります。
  • how_far が 0 の場合、現在の文字を返します。

アルゴリズムまたは振る舞い

1. ループを使用して、word の先頭から how_far で指定されたインデックスまで、または文字列の終端 ('\0') に達するまでポインタを進めます。 2. 到達したインデックス位置の文字を特定します。 3. その文字を戻り値として返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/fuzzystrmatch/fuzzystrmatch.c:315 にて定義されているマクロ Look_Ahead_Letter(n) から呼び出されます。
  • **呼び出し先**:
  • 直接的な関数呼び出しはありません。

関連データ構造

  • 特になし。

失敗モードと境界条件

  • how_far が文字列の長さ以上の値を指定された場合、ループは文字列の終端で停止し、ヌル文字 ('\0') を返します。
  • word が空文字列の場合、常にヌル文字を返します。

保守担当者向けメモ

  • 本関数は Metaphone アルゴリズムの実装の一部として使用されています。
  • src/tools/pg_bsd_indent/io.c に同名の関数が存在しますが、本モジュールの Lookahead とは無関係です。
