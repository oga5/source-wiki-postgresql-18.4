check_unicode_value

定義

  • 種別: function
  • 定義箇所: src/backend/parser/parser.c:341

目的

Unicodeエスケープシーケンスから変換されたコードポイントが、有効なUnicode値であるかを検証します。

シグネチャまたは形状

static void check_unicode_value(pg_wchar c)

入力と出力

  • **入力**: pg_wchar c（検証対象のUnicodeコードポイント）
  • **出力**: なし（戻り値なし）

前提条件と不変条件

  • 入力されたコードポイントが有効な範囲外である場合、エラーを報告します。

アルゴリズムまたは振る舞い

1. is_valid_unicode_codepoint(c) を呼び出し、コードポイントの妥当性を確認します。 2. 無効な値である場合、ereport を使用して ERROR レベルの例外を発生させ、ERRCODE_SYNTAX_ERROR（構文エラー）として処理を中断します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/parser/parser.c 内の複数箇所（行429、468など）から呼び出されています。
  • **呼び出し先**: is_valid_unicode_codepoint（妥当性判定）、ereport（エラー報告）。

関連データ構造

  • pg_wchar: Unicodeコードポイントを保持する型。

失敗モードと境界条件

  • 無効なUnicodeコードポイントが渡された場合、即座に構文エラーとして処理が停止します。

保守担当者向けメモ

  • 本関数は str_udeescape 等のUnicodeエスケープ処理の一部として使用されます。
  • エラー発生時には ereport によりSQL解析プロセスが中断されるため、呼び出し側でトランザクション状態を考慮したエラーハンドリングがなされていることを前提としています。
