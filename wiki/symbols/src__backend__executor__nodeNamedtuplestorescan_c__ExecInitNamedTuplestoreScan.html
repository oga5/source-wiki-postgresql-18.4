ExecInitNamedTuplestoreScan

定義

  • 種別: function
  • 定義箇所: src/backend/executor/nodeNamedtuplestorescan.c:81

目的

名前付き一時タプルストア（ENR）をスキャンするための実行ノード（NamedTuplestoreScanState）を初期化し、スキャンに必要な読み取りポインタと実行環境を確立する。

シグネチャまたは形状

NamedTuplestoreScanState *ExecInitNamedTuplestoreScan(NamedTuplestoreScan *node, EState *estate, int eflags)

入力と出力

  • **入力**:
  • node: 初期化対象の NamedTuplestoreScan ノード。
  • estate: クエリ実行状態（EState）。
  • eflags: 実行フラグ（EXEC_FLAG_BACKWARD や EXEC_FLAG_MARK はサポート外）。
  • **出力**:
  • 初期化済みの NamedTuplestoreScanState 構造体へのポインタ。

前提条件と不変条件

  • eflags に EXEC_FLAG_BACKWARD または EXEC_FLAG_MARK が含まれていないこと。
  • node に子プラン（outerPlan / innerPlan）が存在しないこと。
  • estate->es_queryEnv 内に、node->enrname で指定された名前付きタプルストアが存在すること。

アルゴリズムまたは振る舞い

1. NamedTuplestoreScanState を割り当て、ExecNamedTuplestoreScan を実行関数として設定する。 2. get_ENR を使用してクエリ環境から対象の EphemeralNamedRelation を取得する。見つからない場合はエラーを投げる。 3. tuplestore_alloc_read_pointer を呼び出し、スキャン用の読み取りポインタを確保する。 4. tuplestore_select_read_pointer と tuplestore_rescan を呼び出し、読み取り位置を先頭にリセットする。 5. ExecAssignExprContext で式コンテキストを作成し、ExecInitScanTupleSlot でスキャン用タプルスロットを初期化する。 6. ExecInitResultTypeTL および ExecAssignScanProjectionInfo で射影情報を設定する。 7. ExecInitQual を呼び出し、ノードのクオリファイア（qual）を初期化する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/executor/execProcnode.c 内の ExecInitNode（推論）。
  • **呼び出し先**: makeNode, get_ENR, ENRMetadataGetTupDesc, tuplestore_alloc_read_pointer, tuplestore_select_read_pointer, tuplestore_rescan, ExecAssignExprContext, ExecInitScanTupleSlot, ExecInitResultTypeTL, ExecAssignScanProjectionInfo, ExecInitQual。

関連データ構造

  • NamedTuplestoreScanState: 本関数が生成する状態構造体。
  • EphemeralNamedRelation: クエリ環境から取得される名前付きタプルストアのメタデータ。
  • Tuplestorestate: 物理的なタプルストアの管理構造体。

失敗モードと境界条件

  • 指定された enrname がクエリ環境に見つからない場合、elog(ERROR) により処理を中断する。
  • サポートされていないフラグ（EXEC_FLAG_BACKWARD 等）が渡された場合、Assert によりデバッグビルドで停止する。

保守担当者向けメモ

  • 本関数は前方スキャンのみを想定している。
  • 読み取りポインタの解放処理は現在実装されておらず、クエリ終了時のクリーンアップに依存している。
