pa_reset_subtrans

定義

  • 種別: function
  • 定義箇所: src/backend/replication/logical/applyparallelworker.c:1417

目的

サブトランザクションリスト（subxactlist）をリセットし、状態を初期化する。

シグネチャまたは形状

void pa_reset_subtrans(void)

入力と出力

  • 入力: なし
  • 出力: なし（副作用としてグローバル変数 subxactlist を NIL に設定する）

前提条件と不変条件

  • 本関数は、サブトランザクションリストが保持するメモリがトランザクション終了時に解放されることを前提としているため、明示的なメモリ解放は行わない。

アルゴリズムまたは振る舞い

1. グローバル変数 subxactlist に NIL を代入する。

呼び出し元と呼び出し先

  • 呼び出し元（直接的な証拠）:
  • src/backend/replication/logical/applyparallelworker.c:1418
  • src/backend/replication/logical/applyparallelworker.c:1472
  • src/backend/replication/logical/worker.c:1392
  • src/backend/replication/logical/worker.c:2236
  • 呼び出し先: なし

関連データ構造

  • subxactlist: サブトランザクションを追跡するために使用されるリスト構造。

失敗モードと境界条件

  • 特筆すべき失敗モードはない。リストが既に空であっても安全に実行可能。

保守担当者向けメモ

  • 本関数は src/include/replication/worker_internal.h で外部公開されている。
  • メモリ管理はトランザクションのライフサイクルに依存しているため、本関数内で個別のメモリ解放処理を追加しないこと。
