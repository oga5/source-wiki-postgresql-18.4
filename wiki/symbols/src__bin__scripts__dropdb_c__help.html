help

定義

  • 種別: function
  • 定義箇所: src/bin/scripts/dropdb.c:167

目的

dropdb コマンドの利用方法、サポートされているオプション、および接続パラメータに関する情報を標準出力に表示する。

シグネチャまたは形状

static void help(const char *progname)

入力と出力

  • 入力: progname (実行中のプログラム名を示す文字列)
  • 出力: なし（標準出力へヘルプメッセージを書き出す）

前提条件と不変条件

  • progname は有効な文字列ポインタであること。
  • PACKAGE_BUGREPORT、PACKAGE_NAME、PACKAGE_URL がビルド環境で定義されていること。

アルゴリズムまたは振る舞い

1. プログラム名を含む概要を表示する。 2. コマンドライン引数の構文を表示する。 3. データベース削除に関連するオプション（-e, -f, -i, -V, --if-exists, -?）を列挙する。 4. 接続に関連するオプション（-h, -p, -U, -w, -W, --maintenance-db）を列挙する。 5. バグ報告先およびプロジェクトのホームページ情報を表示する。

呼び出し元と呼び出し先

  • 呼び出し元: 提供されたソースコード内には、この関数を呼び出す具体的な箇所は明示されていません。
  • 呼び出し先: printf (標準ライブラリ)

※ツールが検出した contrib/oid2name/oid2name.c 等の同名関数は、本ファイル (dropdb.c) の help とは異なる定義です。

関連データ構造

  • なし

失敗モードと境界条件

  • 標準出力への書き込みが失敗した場合（パイプのクローズ等）、動作は printf の仕様に依存する。

保守担当者向けメモ

  • 新しいコマンドラインオプションを追加した際は、本関数の出力内容にも必ず反映させること。
  • _() マクロを使用しているため、翻訳対象として適切に管理すること。
