LoadArchiveLibrary

定義

  • **種別**: 関数
  • **定義箇所**: src/backend/postmaster/pgarch.c:912

目的

WALアーカイブ処理に必要な外部ライブラリまたはシェルアーカイブ機能を初期化し、アーカイブコールバックを登録する。

シグネチャまたは形状

static void LoadArchiveLibrary(void)

入力と出力

  • **入力**: グローバル変数 XLogArchiveLibrary および XLogArchiveCommand を参照。
  • **出力**: なし（副作用としてグローバル変数 ArchiveCallbacks および archive_module_state を設定）。

前提条件と不変条件

  • archive_command と archive_library が同時に設定されている場合はエラーとなる。
  • ロードされるアーカイブモジュールは _PG_archive_module_init シンボルを定義している必要がある。
  • アーカイブモジュールは archive_file_cb コールバックを登録する必要がある。

アルゴリズムまたは振る舞い

1. 設定値の競合チェックを行い、両方が設定されていればエラーを報告する。 2. XLogArchiveLibrary が空であれば shell_archive_init を使用し、そうでなければ load_external_function を用いて外部ライブラリから _PG_archive_module_init をロードする。 3. ロードした初期化関数を呼び出し、ArchiveCallbacks を取得する。 4. archive_file_cb が存在することを確認する。 5. archive_module_state をメモリ確保し、モジュールに startup_cb が定義されていれば実行する。 6. プロセス終了時にモジュールのシャットダウンコールバックが呼ばれるよう before_shmem_exit を登録する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/postmaster/pgarch.c 内の PgArchiverMain から呼び出される。
  • **呼び出し先**: load_external_function（外部ライブラリロード用）、shell_archive_init、およびロードされたモジュールの startup_cb。

関連データ構造

  • ArchiveModuleInit: 初期化関数の型定義。
  • ArchiveCallbacks: アーカイブ処理のコールバック関数群を保持する構造体。
  • ArchiveModuleState: アーカイブモジュールの状態を保持する構造体。

失敗モードと境界条件

  • archive_command と archive_library の両方が設定されている場合、ERRCODE_INVALID_PARAMETER_VALUE でエラー終了する。
  • _PG_archive_module_init シンボルが見つからない場合、エラー終了する。
  • archive_file_cb が登録されていない場合、エラー終了する。

保守担当者向けメモ

  • 本関数は sigsetjmp による保護範囲外で実行される初期化処理であるため、ここでのエラーはプロセスを終了させる。
  • before_shmem_exit に登録される pgarch_call_module_shutdown_cb は、プロセス終了時のクリーンアップを保証する。
