help

定義

  • 種別: function
  • 定義箇所: src/bin/scripts/clusterdb.c:270

目的

clusterdb コマンドの利用方法、オプション一覧、および関連情報を標準出力に表示する。

シグネチャまたは形状

static void help(const char *progname)

入力と出力

  • **入力**: progname (実行ファイル名を示す文字列)
  • **出力**: なし（標準出力へのメッセージ出力のみ）

前提条件と不変条件

  • progname は有効な文字列ポインタであること。
  • gettext（_() マクロ）が環境に応じて適切にメッセージを翻訳できる状態であること。

アルゴリズムまたは振る舞い

1. progname を使用して、コマンドの概要と使用法（Usage）を表示する。 2. clusterdb がサポートする各オプション（-a, -d, -e, -q, -t, -v, -V, -?）および接続オプションの説明を順次出力する。 3. SQL コマンド CLUSTER の詳細参照を促すメッセージを表示する。 4. PACKAGE_BUGREPORT および PACKAGE_NAME、PACKAGE_URL を用いて、バグ報告先とホームぺージ情報を出力する。

呼び出し元と呼び出し先

  • **呼び出し元**: 提供されたソースコード内には、本関数を呼び出している箇所は明示されていません。
  • **呼び出し先**: printf（標準ライブラリ関数）。

関連データ構造

  • なし。

失敗モードと境界条件

  • 標準出力がクローズされている場合、printf の呼び出しは失敗する可能性がありますが、本関数内で明示的なエラーハンドリングは行われていません。

保守担当者向けメモ

  • 本関数は clusterdb のヘルプメッセージを定義しています。オプションを追加または変更した場合は、必ず本関数内の説明文も更新してください。
  • ツールによる検索結果で表示されている contrib/oid2name/oid2name.c 等の help 関数は、名前が同一ですが別個のシンボルであり、本関数とは無関係です。
