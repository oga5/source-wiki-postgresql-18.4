networkjoinsel_semi

定義

  • **種別**: function
  • **定義箇所**: src/backend/utils/adt/network_selfuncs.c:406

目的

セミ結合（EXISTS や IN 句など）における、ネットワーク型（inet/cidr）の選択率を推定します。

シグネチャまたは形状

static Selectivity networkjoinsel_semi(Oid operator, int opr_codenum, VariableStatData *vardata1, VariableStatData *vardata2)

入力と出力

  • **入力**:
  • operator: 評価対象の演算子OID。
  • opr_codenum: 演算子を包含関係の数値（-2〜2）に変換した値。
  • vardata1, vardata2: 結合対象となる2つのリレーションの統計情報（VariableStatData）。
  • **出力**:
  • Selectivity（double）: 推定された選択率（0.0〜1.0）。

前提条件と不変条件

  • vardata1 および vardata2 に含まれる統計情報（MCV、ヒストグラム）が有効であること。
  • MAX_CONSIDERED_ELEMS（1024）を超える統計要素は間引き（デシメーション）され、計算コストが抑制されます。

アルゴリズムまたは振る舞い

1. **統計情報の取得**: get_attstatsslot を使用して、両リレーションのMCVおよびヒストグラムを取得します。 2. **MCVの処理**: LHS（左辺）のMCVリストを走査し、各要素に対して inet_semi_join_sel を呼び出して選択率を累積します。 3. **ヒストグラムの処理**: LHSのヒストグラム（最初と最後の要素を除く）を走査します。要素数が多すぎる場合は間引きを行い、各要素に対して inet_semi_join_sel を適用して確率を累積します。 4. **デフォルト推定**: 統計情報が不足している場合、DEFAULT_SEL を用いたデフォルトの選択率を算出します。 5. **リソース解放**: free_attstatsslot を呼び出し、取得した統計スロットを解放します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/network_selfuncs.c 内の networkjoinsel 関数（直接の呼び出しを確認）。
  • **呼び出し先**:
  • mcv_population: MCVの母集団割合の算出。
  • inet_semi_join_sel: セミ結合の選択率計算コアロジック。
  • get_attstatsslot / free_attstatsslot: 統計情報の取得・解放。
  • fmgr_info / get_opcode: 演算子情報の取得。

関連データ構造

  • VariableStatData: 統計情報へのアクセス用構造体。
  • AttStatsSlot: pg_statistic から抽出された個別の統計データ（MCV、ヒストグラム）。
  • Form_pg_statistic: pg_statistic システムカタログの行構造。

失敗モードと境界条件

  • **統計情報の欠如**: vardata に統計情報が存在しない場合、デフォルトの選択率計算へフォールバックします。
  • **ヒストグラムのサイズ**: ヒストグラムの要素数が2以下の場合は、ヒストグラムによる推定をスキップします。
  • **計算コスト**: 結合推定は $O(N^2)$ になり得るため、MAX_CONSIDERED_ELEMS による間引きが適用されます。

保守担当者向けメモ

  • 包含関係の推定には inet_semi_join_sel が使用されています。推定精度が低い場合は、inet_hist_match_divider によるビット単位の確率補正ロジックを確認してください。
  • 統計情報の取得に失敗した場合のメモリリークを防ぐため、free_attstatsslot の呼び出しが必須です。
