pg_atomic_exchange_u64_impl

定義

  • 種別: function
  • 定義箇所: src/include/port/atomics/generic-sunpro.h:107

目的

Oracle Solaris Studio (SunPro) コンパイラ環境において、64ビット符号なし整数のアトミックな交換操作を提供します。

シグネチャまたは形状

static inline uint64 pg_atomic_exchange_u64_impl(volatile pg_atomic_uint64 *ptr, uint64 newval)

入力と出力

  • **入力**:
  • ptr: 操作対象となる pg_atomic_uint64 構造体へのポインタ。
  • newval: ptr が指すメモリ位置に書き込む新しい値。
  • **出力**:
  • 交換前のメモリ位置に格納されていた値を uint64 型で返します。

前提条件と不変条件

  • ptr は volatile 修飾されており、アトミック操作の対象として適切に配置されている必要があります。
  • 本関数は Solaris 環境でのみ有効です。

アルゴリズムまたは振る舞い

1. Solaris 標準の <atomic.h> で定義された atomic_swap_64 関数を呼び出します。 2. ptr が指す構造体の value フィールドに対して、newval をアトミックに書き込みます。 3. 書き込み前の値を即座に返却します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/include/port/atomics.h:508 からの呼び出しが確認されています。
  • **呼び出し先**: Solaris システム関数 atomic_swap_64 を呼び出します。

関連データ構造

  • pg_atomic_uint64: アトミック操作対象の値を保持する構造体。

失敗モードと境界条件

  • 供給されたソース情報には、特定の失敗モードや境界条件に関する記述はありません。

保守担当者向けメモ

  • 本関数は Solaris の atomic_swap_64 の単純なラッパーです。
  • pg_atomic_uint64 のアライメントについては、コンパイラのバージョンによる制約を受ける可能性があるため注意してください。
