get_len_position

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/multirangetypes_selfuncs.c:872

目的

レンジ長ヒストグラムの特定のビン内において、指定された値がどの程度の相対位置（0.0〜1.0）にあるかを算出します。

シグネチャまたは形状

static double get_len_position(double value, double hist1, double hist2)

入力と出力

  • **入力**:
  • value: 位置を特定したい対象の値。
  • hist1: ビンの下限値。
  • hist2: ビンの上限値。
  • **出力**:
  • double: ビン内での相対位置。0.0（下限）から 1.0（上限）の範囲で返されます。

前提条件と不変条件

  • hist1 および hist2 はヒストグラムの境界値です。
  • value が無限大（isinf）である場合、境界値の状態に応じて固定値（0.0, 0.5, 1.0）が返されます。

アルゴリズムまたは振る舞い

1. **有限値の場合**: hist1 と hist2 が共に有限であれば、線形補間により (value - hist1) / (hist2 - hist1) に相当する相対位置を計算します。 2. **無限値の処理**:

  • hist1 が -inf で hist2 が有限の場合: 1.0 を返します。
  • hist1 と hist2 が共に inf の場合: 0.0 を返します。
  • 上記以外（両方が無限大など）の場合: 0.5 を返します。

    • 3. **異常値の保護**: value が有限値の計算中に無限大であると判定された場合は、安全策として 0.5 を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/multirangetypes_selfuncs.c 内の関数（calc_length_hist_frac 等から参照）。
  • src/backend/utils/adt/rangetypes_selfuncs.c 内の関数。
  • **呼び出し先**:
  • math.h の isinf() マクロ。

関連データ構造

  • pg_statistic: レンジ長ヒストグラムのデータソース。

失敗モードと境界条件

  • hist1 == hist2 の場合、除算によるゼロ除算が発生する可能性がありますが、実装上は isinf チェックにより回避される設計となっています。
  • 呼び出し側で不正なヒストグラム境界が渡された場合でも、クラッシュを避けるためにアサーションを行わず、デフォルト値（0.5）を返す安全な設計になっています。

保守担当者向けメモ

  • 本関数は rangetypes_selfuncs.c と multirangetypes_selfuncs.c の両方で定義・使用されています。修正を行う際は両方のファイルでの整合性に注意してください。
  • 統計的な仮定として、レンジ長はヒストグラムのビン内で線形に分布していると見なしています。
