r_instrum

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_2_hungarian.c:563

目的

ハンガリー語のステミング処理において、単語末尾の楽器格（instrumental case）接尾辞を特定し、削除および正規化を行う。

シグネチャまたは形状

static int r_instrum(struct SN_env * z)

入力と出力

  • 入力: SN_env * z （ステミング環境を保持する構造体）
  • 出力: 成功時に 1、失敗または条件不一致時に 0、エラー時に負の値を返す。

前提条件と不変条件

  • z->c は処理対象の単語末尾位置を指している必要がある。
  • 処理対象の単語末尾の文字が 108（ASCII/ISO-8859-2における 'l'）であること。

アルゴリズムまたは振る舞い

1. z->ket を現在のカーソル位置 z->c に設定する。 2. 単語末尾が l でない場合、または境界条件を満たさない場合は 0 を返す。 3. a_3 配列に定義されたパターンに基づき、find_among_b を用いて末尾からマッチングを行う。 4. マッチした場合、z->bra をマッチした接尾辞の開始位置に設定する。 5. r_R1 を呼び出し、マッチした箇所が R1 領域内にあるか検証する。 6. r_double を呼び出し、必要に応じて子音の重複を処理する。 7. slice_del を呼び出し、接尾辞を削除する。 8. r_undouble を呼び出し、削除後の単語に対して重複の正規化を行う。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_ISO_8859_2_hungarian.c:803 および src/backend/snowball/libstemmer/stem_UTF_8_hungarian.c:809 からの呼び出しが確認されている。
  • 呼び出し先: find_among_b, r_R1, r_double, slice_del, r_undouble。

関連データ構造

  • SN_env: ステミング環境を管理する構造体。
  • a_3: 楽器格の接尾辞パターンを保持する among 構造体の配列。

失敗モードと境界条件

  • z->c - 1 <= z->lb: 単語の境界を超えて処理しようとした場合に 0 を返す。
  • find_among_b がマッチしない場合、処理を中断し 0 を返す。
  • r_R1, r_double, slice_del, r_undouble の各関数が負の値を返した場合、その値をそのまま伝播させる。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードである。ロジックの変更が必要な場合は、直接編集せず元の .sbl スクリプトを修正すること。
  • a_3 配列の定義内容が、現在のハンガリー語の文法ルールと一致しているか確認が必要な場合がある。
