TSQuerySignGetDatum

定義

  • 種別: static inline function
  • 定義箇所: src/include/tsearch/ts_utils.h:251

目的

TSQuerySign 型の値を、PostgreSQL の内部データ表現である Datum 型に変換する。

シグネチャまたは形状

static inline Datum TSQuerySignGetDatum(TSQuerySign X)

入力と出力

  • **入力**: TSQuerySign 型の値 X。
  • **出力**: X を int64 にキャストし、Int64GetDatum マクロを適用した Datum 型の値。

前提条件と不変条件

  • TSQuerySign 型が int64 へのキャストが可能であること。
  • 本関数はインライン関数として定義されており、呼び出し元で Datum への変換が保証される。

アルゴリズムまたは振る舞い

1. 引数 X を int64 型に明示的にキャストする。 2. Int64GetDatum マクロを使用して、キャストされた値を Datum 型に変換して返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/tsquery_gist.c: GiST インデックスの構築および分割処理において使用。
  • src/include/tsearch/ts_utils.h: PG_RETURN_TSQUERYSIGN マクロ内で使用。
  • **呼び出し先**:
  • Int64GetDatum (PostgreSQL のコアマクロ)

関連データ構造

  • TSQuerySign: 本関数が変換対象とする型。
  • Datum: PostgreSQL における汎用的なデータ表現型。

失敗モードと境界条件

  • TSQuerySign の値が int64 の範囲外である場合、キャスト時にデータ損失が発生する可能性がある（ただし、現在の実装では TSQuerySign の定義に依存する）。

保守担当者向けメモ

  • 本関数は tsquery_gist.c 等で頻繁に使用されるため、変更時は GiST インデックスのバイナリ互換性に影響がないか注意すること。
  • PG_RETURN_TSQUERYSIGN マクロの基盤となっているため、このマクロ経由で呼び出される関数群の戻り値の型変換に直接関与している。
