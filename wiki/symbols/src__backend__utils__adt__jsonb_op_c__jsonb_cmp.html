jsonb_cmp

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/jsonb_op.c:235

目的

B-Treeインデックスにおける jsonb 型のソート順を決定するための比較関数です。

シグネチャまたは形状

Datum jsonb_cmp(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS を介して2つの jsonb 型データ（jba, jbb）を受け取ります。
  • **出力**: 比較結果を int32 型で返します（jba < jbb なら負、jba == jbb なら0、jba > jbb なら正）。

前提条件と不変条件

  • 入力される jsonb ポインタは有効である必要があります。
  • PostgreSQLの関数マネージャ（fmgr）経由で呼び出されることを前提としています。

アルゴリズムまたは振る舞い

1. PG_GETARG_JSONB_P マクロを使用して、引数から2つの Jsonb 構造体を取得します。 2. compareJsonbContainers 関数を呼び出し、両者の root コンテナを比較します。 3. PG_FREE_IF_COPY を使用して、必要に応じてコピーされたメモリを解放します。 4. 比較結果を PG_RETURN_INT32 を通じて返却します。

呼び出し元と呼び出し先

  • **呼び出し先**: compareJsonbContainers（src/backend/utils/adt/jsonb_util.c で定義されていると推測される補助関数）。
  • **呼び出し元**: 提供されたソース情報には、本関数を直接呼び出している具体的な関数名は記載されていません。

関連データ構造

  • Jsonb: jsonb 型の内部表現。
  • JsonbContainer (root メンバ): 比較対象となる jsonb のルート構造。

失敗モードと境界条件

  • 入力が NULL の場合の挙動については、本関数内では明示的なチェックを行っていません（PostgreSQLの fmgr の規約に依存します）。

保守担当者向けメモ

  • 本関数はB-Treeインデックスのソート順を定義するため、比較ロジックの変更はインデックスの再構築を必要とする可能性があります。
  • compareJsonbContainers の実装詳細については、src/backend/utils/adt/jsonb_util.c を参照してください。
