pg_unicode_category

定義

  • 種別: enum
  • 定義箇所: src/include/common/unicode_category.h:30

目的

Unicode 標準における文字の「一般カテゴリ（General Category）」を表現するための列挙型です。ICU（International Components for Unicode）の UCharCategory と値を一致させており、文字の属性判定の基盤として利用されます。

シグネチャまたは形状

typedef enum pg_unicode_category {
    PG_U_UNASSIGNED = 0,
    PG_U_UPPERCASE_LETTER = 1,
    /* ... (中略) ... */
    PG_U_FINAL_PUNCTUATION = 29
} pg_unicode_category;

入力と出力

  • 本シンボル自体は列挙値の定義です。
  • 関連関数 unicode_category(pg_wchar code) の戻り値として使用され、指定されたコードポイントに対応するカテゴリを返します。

前提条件と不変条件

  • Unicode 安定性ポリシーに基づき、定義されたカテゴリ値は固定されており、将来的に変更されることはありません。

アルゴリズムまたは振る舞い

  • 文字の分類結果を保持します。
  • src/common/unicode_category.c 内では、これらの列挙値を用いてビットマスクを作成し、効率的なカテゴリ比較を行うために利用されます。

関連データ構造

  • src/include/common/unicode_category_table.h: 判定ロジックで使用される静的データテーブル。

失敗モードと境界条件

  • 該当なし（列挙型定義のため）。

保守担当者向けメモ

  • カテゴリ値の追加や変更を行う際は、ICU の UCharCategory との整合性を維持してください。
  • パフォーマンスが要求される箇所では、個別の列挙値比較ではなく、PG_U_*_MASK マクロによるビット演算での判定を検討してください。
