run_crypt_md5

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/px-crypt.c:50

目的

MD5 アルゴリズムを用いたパスワードハッシュ化処理を px-crypt インターフェースへ適合させるためのラッパー関数です。

シグネチャまたは形状

static char *run_crypt_md5(const char *psw, const char *salt, char *buf, unsigned len);

入力と出力

  • psw: ハッシュ化対象のパスワード文字列。
  • salt: 使用するソルト文字列。
  • buf: 結果を格納するためのバッファ。
  • len: buf のサイズ。
  • 戻り値: ハッシュ化された文字列へのポインタ（buf を指す）。

前提条件と不変条件

  • px_crypt_md5 が利用可能であること。
  • buf はハッシュ結果を格納するのに十分なサイズであること。

アルゴリズムまたは振る舞い

1. 内部関数 px_crypt_md5 を呼び出し、引数として受け取ったパスワード、ソルト、バッファ、長さをそのまま渡します。 2. px_crypt_md5 の実行結果をそのまま呼び出し元へ返します。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/pgcrypto/px-crypt.c 内の静的テーブル px_crypt_list から参照されています（"$1$" プレフィックスに対応）。
  • **呼び出し先**: px_crypt_md5 を呼び出します。

関連データ構造

  • px_crypt_list: ソルトのプレフィックスと本関数を紐付ける静的テーブル。

失敗モードと境界条件

  • px_crypt_md5 の実装に依存します。バッファサイズ len が不足している場合の挙動は、当該関数の仕様に従います。

保守担当者向けメモ

  • 本関数は単なるディスパッチ用ラッパーです。MD5 ハッシュ計算の具体的なロジックについては contrib/pgcrypto/crypt-md5.c を参照してください。
