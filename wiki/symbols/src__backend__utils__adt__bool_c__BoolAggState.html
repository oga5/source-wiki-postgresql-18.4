BoolAggState

定義

  • 種別: struct
  • 定義箇所: src/backend/utils/adt/bool.c:317

目的

集約関数（bool_and, bool_or 等）の実行中に、計算状態を保持するための内部構造体です。

シグネチャまたは形状

typedef struct BoolAggState
{
	int64		aggcount;		/* 集約された非NULL値の数 */
	int64		aggtrue;		/* 集約された値のうち真であるものの数 */
} BoolAggState;

入力と出力

  • 本構造体は、集約関数がメモリコンテキスト上で動的に割り当てて使用します。
  • bool_accum 等の関数において、PG_GETARG_POINTER(0) を通じてポインタとして受け渡されます。

前提条件と不変条件

  • aggcount は、集約対象となった非NULL値の総数を保持します。
  • aggtrue は、集約対象となった値のうち true であったものの総数を保持します。

アルゴリズムまたは振る舞い

  • 集約関数が呼び出されるたびに、この構造体のフィールドが更新されます。
  • 最終的に bool_alltrue や bool_anytrue などの関数がこの状態を参照し、集約結果（true または false）を算出します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/adt/bool.c 内の集約関連関数（bool_accum 等）が、状態の初期化や更新のために本構造体を操作します。
  • **呼び出し先**: 本構造体自体はデータ保持用であり、関数を呼び出すことはありません。

関連データ構造

  • bool_accum / bool_accum_inv: 本構造体を状態として利用する集約関数。
  • bool_alltrue / bool_anytrue: 本構造体の値を最終計算する関数。

失敗モードと境界条件

  • MemoryContextAlloc によるメモリ割り当て失敗の可能性があります。
  • 入力がNULLの場合、PG_ARGISNULL(0) チェックにより NULL として扱われます。

保守担当者向けメモ

  • 本構造体は、ウィンドウ関数やインクリメンタルな集約をサポートするために bool_accum_inv とペアで動作する設計となっています。
  • 構造体のフィールドは int64 で定義されており、大規模なデータセットの集約にも対応可能です。
