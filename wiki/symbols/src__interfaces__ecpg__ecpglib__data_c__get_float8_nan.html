get_float8_nan

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/ecpglib/data.c:89

目的

プラットフォームの差異を吸収し、標準化された NaN（Not-a-Number）値を double 型として取得する。

シグネチャまたは形状

static double get_float8_nan(void)

入力と出力

  • 入力: なし
  • 出力: double 型の NaN 値

前提条件と不変条件

  • 戻り値は常に浮動小数点の NaN として評価される。

アルゴリズムまたは振る舞い

1. NAN マクロが定義されており、かつ NetBSD/MIPS 環境ではない場合、NAN を double にキャストして返す。 2. それ以外（NetBSD/MIPS 環境を含む）の場合、0.0 / 0.0 の演算結果を double にキャストして返す。

呼び出し元と呼び出し先

  • 呼び出し元: ツールによる解析では、src/backend/access/hash/hashfunc.c、src/backend/access/spgist/spgproc.c、src/backend/utils/adt/float.c からの参照が示唆されている。ただし、これらは ecpglib 外部のバックエンドコードであり、本関数が static で定義されていることを考慮すると、これらは同名の別関数であるか、ツールによるヒューリスティックな誤検出の可能性がある。
  • 呼び出し先: なし（標準ライブラリの NAN マクロまたは浮動小数点演算を使用）。

関連データ構造

  • なし

失敗モードと境界条件

  • NetBSD/MIPS 環境における NAN マクロの不具合を回避するため、ゼロ除算による NaN 生成にフォールバックする設計となっている。

保守担当者向けメモ

  • 本関数は static で定義されているため、同一ファイル（src/interfaces/ecpg/ecpglib/data.c）内でのみ使用されることを想定している。
  • ツールが提示したバックエンド側の呼び出し元は、本関数のスコープ外である可能性が高いため、参照の際は注意すること。
