TuplesortSpaceType

定義

  • 種別: enum
  • 定義箇所: src/include/utils/tuplesort.h:87

目的

ソート処理において、データが現在どの領域（メモリまたはディスク）を使用して保持されているかを識別するために使用されます。

シグネチャまたは形状

typedef enum
{
    SORT_SPACE_TYPE_DISK,
    SORT_SPACE_TYPE_MEMORY,
} TuplesortSpaceType;

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: なし

前提条件と不変条件

  • Tuplesortstate 構造体内の spaceType フィールドにおいて、現在使用中の領域タイプを追跡するために利用されます。

アルゴリズムまたは振る舞い

  • SORT_SPACE_TYPE_MEMORY: ソート対象データが work_mem 内に収まっている状態を示します。
  • SORT_SPACE_TYPE_DISK: メモリ制限を超え、一時ファイル（テープ）へ書き出されている状態を示します。
  • tuplesort_space_type_name() 関数を通じて、デバッグやログ出力用に文字列へ変換可能です。

呼び出し元と呼び出し先

  • 呼び出し元: tuplesort.c 内のソート状態管理ロジック。
  • 呼び出し先: tuplesort_space_type_name(TuplesortSpaceType t) がこの型を引数として受け取ります。

関連データ構造

  • Tuplesortstate: 本列挙型を保持する主要な状態構造体。

失敗モードと境界条件

  • 定義外の値が代入された場合の挙動は未定義です。

保守担当者向けメモ

  • 新たなソート領域タイプ（例: 共有メモリ等）を追加する場合は、本列挙型および tuplesort_space_type_name の両方を更新する必要があります。
