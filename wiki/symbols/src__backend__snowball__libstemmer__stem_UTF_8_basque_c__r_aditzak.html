r_aditzak

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_basque.c:1007

目的

バスク語のステミング処理において、動詞（aditzak）に関連する接尾辞を単語末尾から探索し、削除または置換を行う。

シグネチャまたは形状

static int r_aditzak(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（ステミング対象の単語データと環境状態を保持する構造体）
  • **出力**: 処理が成功した場合は 1、失敗または対象外の場合は 0、エラー発生時は負の値を返す。

前提条件と不変条件

  • z->c が単語の末尾を指していること。
  • find_among_b による探索のため、z->lb（単語の開始境界）が正しく設定されていること。

アルゴリズムまたは振る舞い

1. z->ket を現在のカーソル位置 z->c に設定する。 2. 単語末尾の文字が特定の条件（a_0 テーブルの探索範囲）を満たすか確認し、満たさない場合は終了する。 3. find_among_b を用いて、a_0 テーブルから接尾辞を後方一致で探索する。 4. 一致した場合、z->bra を一致した接尾辞の開始位置に設定する。 5. among_var の値に応じて以下のいずれかのアクションを実行する：

  • **ケース1**: r_RV を確認後、slice_del で接尾辞を削除。
  • **ケース2**: r_R2 を確認後、slice_del で接尾辞を削除。
  • **ケース3, 4, 5**: slice_from_s を用いて、それぞれ定義された文字列に置換。

    • 6. 成功時に 1 を返して終了する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_UTF_8_basque.c および src/backend/snowball/libstemmer/stem_ISO_8859_1_basque.c 内のメインステミング関数から呼び出される。
  • **呼び出し先**: find_among_b, r_RV, r_R2, slice_del, slice_from_s（いずれも Snowball ランタイム関数）。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_0: 動詞接尾辞の探索用 among テーブル。

失敗モードと境界条件

  • find_among_b が一致しない場合、0 を返して処理を中断する。
  • r_RV や r_R2 のチェックに失敗した場合、その戻り値（負の値または0）をそのまま返す。
  • slice_del や slice_from_s がメモリ不足等のエラーを返した場合、その負の値を伝播させる。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードである。ロジックの変更が必要な場合は、直接編集せず対応する .sbl ファイルを修正すること。
  • a_0 テーブルのインデックスや among_var のケース分岐は、バスク語の文法規則に直結している。
