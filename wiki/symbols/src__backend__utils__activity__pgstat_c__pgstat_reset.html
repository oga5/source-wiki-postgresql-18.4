pgstat_reset

定義

  • 種別: function
  • 定義箇所: src/backend/utils/activity/pgstat.c:852

目的

指定された統計種別（kind）、データベース（dboid）、およびオブジェクト（objid）に対応する統計カウンタをリセットします。

シグネチャまたは形状

void pgstat_reset(PgStat_Kind kind, Oid dboid, uint64 objid)

入力と出力

  • **入力**:
  • kind: リセット対象の統計種別（PgStat_Kind）。
  • dboid: 対象のデータベース OID。
  • objid: 対象オブジェクトの識別子。
  • **出力**: なし（void）。

前提条件と不変条件

  • 対象の統計種別が「固定量（fixed amount）」の統計ではないこと（Assert(!pgstat_get_kind_info(kind)->fixed_amount) により検証）。

アルゴリズムまたは振る舞い

1. pgstat_get_kind_info(kind) を呼び出し、指定された種別のメタ情報を取得します。 2. GetCurrentTimestamp() を使用して現在時刻を取得します。 3. pgstat_reset_entry() を呼び出し、指定されたオブジェクトの単一カウンタをリセットします。 4. 対象の統計種別がデータベースを跨いでアクセスされない場合（!kind_info->accessed_across_databases）、pgstat_reset_database_timestamp() を呼び出し、当該データベースの統計リセット時刻を更新します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/activity/pgstat_replslot.c
  • src/backend/utils/activity/pgstat_xact.c
  • src/backend/utils/adt/pgstatfuncs.c
  • **呼び出し先**:
  • pgstat_get_kind_info
  • GetCurrentTimestamp
  • pgstat_reset_entry
  • pgstat_reset_database_timestamp

関連データ構造

  • PgStat_Kind: 統計種別を定義する列挙型。
  • PgStat_KindInfo: 統計種別の振る舞いを定義する構造体。

失敗モードと境界条件

  • fixed_amount が真である統計種別に対して呼び出した場合、アサーションエラーが発生します。

保守担当者向けメモ

  • 本関数は単一オブジェクトの統計リセットを担います。データベース全体や統計種別全体のリセットについては、他の専用関数（pgstat_reset_of_kind 等）を参照してください。
