r_step_s8

定義

  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_greek.c:2789

目的

ギリシャ語のステミング処理における、特定の語尾パターンに対する削除および置換ルールを適用する。

シグネチャまたは形状

static int r_step_s8(struct SN_env * z)

入力と出力

  • 入力: SN_env 構造体（ステミング対象の文字列と現在のカーソル位置 c を保持）。
  • 出力: 処理が成功した場合は 1、失敗した場合は 0、メモリ操作等でエラーが発生した場合は負の値を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体へのポインタであること。
  • find_among_b および slice_ 系関数が正常に動作する環境であること。

アルゴリズムまたは振る舞い

1. z->ket を現在位置に設定し、a_18 配列を用いて末尾からパターンを検索する。一致しない場合は 0 を返す。 2. 一致した部分を slice_del で削除する。 3. z->I[0] を 0 にリセットする。 4. a_17 配列を用いて末尾から別のパターンを検索する。

  • 一致した場合: among_var の値に応じて slice_from_s で文字列を置換する。
  • 一致しない場合: s_60 との完全一致を試み、一致すれば s_61 に置換する。

    • 5. 成功時は 1 を返して終了する。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_greek.c:3525
  • 呼び出し先: find_among_b, slice_del, slice_from_s, eq_s_b（いずれも Snowball ランタイム関数）

関連データ構造

  • struct SN_env: ステミング処理の状態管理構造体。
  • a_17, a_18: 語尾パターンを定義する among 構造体の配列。

失敗モードと境界条件

  • find_among_b が一致しない場合、処理は中断され 0 を返す。
  • slice_del や slice_from_s が負の値を返した場合、そのエラーコードをそのまま伝播させる。
  • among_var による分岐でいずれにも該当せず、かつ s_60 との比較にも失敗した場合は 0 を返す。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードである。ロジックの変更が必要な場合は、直接編集せず対応する .sbl ファイルを修正すること。
  • a_17 および a_18 配列の定義内容が本関数の挙動を決定づけるため、デバッグ時はこれらの配列と find_among_b の戻り値を確認すること。
