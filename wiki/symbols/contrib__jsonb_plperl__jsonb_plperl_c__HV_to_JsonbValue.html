HV_to_JsonbValue

定義

  • 種別: function
  • 定義箇所: contrib/jsonb_plperl/jsonb_plperl.c:150

目的

Perl のハッシュ構造 (HV) を PostgreSQL の Jsonb 形式へ変換し、JsonbParseState に追加します。

シグネチャまたは形状

static JsonbValue *HV_to_JsonbValue(HV *obj, JsonbParseState **jsonb_state)

入力と出力

  • **入力**:
  • obj: 変換対象の Perl ハッシュ (HV*)。
  • jsonb_state: Jsonb 構築状態を保持する JsonbParseState へのポインタ。
  • **出力**:
  • JsonbValue*: 構築されたオブジェクトの終了トークン (WJB_END_OBJECT) を含む JsonbValue へのポインタ。

前提条件と不変条件

  • dTHX マクロにより、Perl インタプリタのコンテキストが初期化されている必要があります。
  • jsonb_state は有効な JsonbParseState を指している必要があります。

アルゴリズムまたは振る舞い

1. pushJsonbValue を呼び出し、WJB_BEGIN_OBJECT トークンを jsonb_state に追加します。 2. hv_iterinit を使用して Perl ハッシュのイテレータを初期化します。 3. hv_iternextsv を用いてハッシュ内の全キーと値を順次走査します。

  • 各キーを pnstrdup で複製し、jbvString 型の JsonbValue として WJB_KEY トークンと共に jsonb_state に追加します。
  • 値に対して SV_to_JsonbValue を再帰的に呼び出し、jsonb_state を更新します。

    • 4. 最後に WJB_END_OBJECT トークンを jsonb_state に追加し、その結果を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: SV_to_JsonbValue からの呼び出しが確認されています（ソースコードの文脈より）。
  • **呼び出し先**:
  • pushJsonbValue (PostgreSQL jsonb API)
  • hv_iterinit, hv_iternextsv (Perl API)
  • SV_to_JsonbValue (再帰呼び出し)
  • pnstrdup (PostgreSQL メモリ管理)

関連データ構造

  • HV: Perl のハッシュ構造体。
  • JsonbParseState: Jsonb 構築過程を管理する状態構造体。
  • JsonbValue: Jsonb の各要素を表す構造体。

失敗モードと境界条件

  • Perl ハッシュ内のキーや値が複雑なネスト構造を持つ場合、再帰呼び出しによりスタックオーバーフローが発生する可能性があります。
  • pnstrdup によるメモリ割り当て失敗時は、PostgreSQL のエラーハンドリング機構によりアボートされます。

保守担当者向けメモ

  • 本関数は SV_to_JsonbValue からの再帰的な呼び出しを前提としています。
  • Perl のメモリ管理と PostgreSQL の palloc が混在するため、変換プロセスにおけるメモリリークの発生には注意が必要です。
