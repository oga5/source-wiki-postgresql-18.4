JsonManifestWALRangeField

定義

  • 種別: enum
  • 定義箇所: src/common/parse_manifest.c:57

目的

バックアップマニフェスト内のWAL範囲（WAL Range）オブジェクトに含まれる各フィールドを識別するために使用されます。

シグネチャまたは形状

typedef enum
{
    JMWRF_TIMELINE,
    JMWRF_START_LSN,
    JMWRF_END_LSN,
} JsonManifestWALRangeField;

入力と出力

  • 入力: JSON解析プロセスにおけるキーの識別子。
  • 出力: 解析中のWAL範囲オブジェクト内での現在のフィールド位置を示す列挙値。

前提条件と不変条件

  • src/common/parse_manifest.c 内のJSON解析ロジックにおいて、WAL範囲オブジェクトの処理中にのみ使用されることを想定しています。

アルゴリズムまたは振る舞い

  • JSON解析中にWAL範囲に関連するキー（Timeline、Start-LSN、End-LSN）を検出した際、現在の解析状態を追跡するためにこの列挙値が利用されます。

呼び出し元と呼び出し先

  • 呼び出し元: src/common/parse_manifest.c 内のJSONハンドラ関数。
  • 呼び出し先: 情報なし。

関連データ構造

  • JsonManifestParseState: 解析中の内部状態を保持する構造体であり、この列挙型をメンバとして保持する可能性があります。

失敗モードと境界条件

  • マニフェスト内のJSON構造が期待されるWAL範囲の形式と一致しない場合、解析エラーが発生します。

保守担当者向けメモ

  • 本シンボルはフロントエンド（postgres_fe.h）向けに設計されています。
  • フィールドの追加や変更を行う際は、src/common/parse_manifest.c 内の対応する解析ハンドラ（json_manifest_object_field_start 等）のロジックを更新してください。
