DebugLogicalRepStreamingMode

定義

  • 種別: enum
  • 定義箇所: src/include/replication/reorderbuffer.h:31

目的

論理レプリケーションにおけるストリーミング処理の挙動を制御・デバッグするためのモードを定義します。

シグネチャまたは形状

typedef enum
{
	DEBUG_LOGICAL_REP_STREAMING_BUFFERED,
	DEBUG_LOGICAL_REP_STREAMING_IMMEDIATE,
} DebugLogicalRepStreamingMode;

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: DEBUG_LOGICAL_REP_STREAMING_BUFFERED または DEBUG_LOGICAL_REP_STREAMING_IMMEDIATE のいずれかの値。

前提条件と不変条件

  • 本列挙型は、論理デコーディングにおけるストリーミングの動作モードを規定するために使用されます。

アルゴリズムまたは振る舞い

  • DEBUG_LOGICAL_REP_STREAMING_BUFFERED: ストリーミングデータをバッファリングして処理するモード。
  • DEBUG_LOGICAL_REP_STREAMING_IMMEDIATE: ストリーミングデータを即時処理するモード。

呼び出し元と呼び出し先

  • 呼び出し元および呼び出し先の具体的なコードパスに関する情報は、提供されたソース excerpt には含まれていません。

関連データ構造

  • ReorderBuffer: 本列挙型が利用される可能性のある、論理デコーディングのメイン構造体。

失敗モードと境界条件

  • 提供されたソース excerpt には、本シンボルに関連する失敗モードや境界条件に関する記述はありません。
