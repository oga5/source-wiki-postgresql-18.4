FreeCachedExpression

定義

  • 種別: function
  • 定義箇所: src/backend/utils/cache/plancache.c:1848

目的

CachedExpression オブジェクトを破棄し、関連するメモリリソースを解放する。

シグネチャまたは形状

void FreeCachedExpression(CachedExpression *cexpr)

入力と出力

  • **入力**: cexpr (CachedExpression*) - 破棄対象のキャッシュ済み式構造体へのポインタ。
  • **出力**: なし（void）。

前提条件と不変条件

  • cexpr は有効なポインタであること。
  • cexpr->magic が CACHEDEXPR_MAGIC と一致していること（サニティチェック）。

アルゴリズムまたは振る舞い

1. Assert を用いて cexpr の整合性を確認する。 2. グローバルリストから当該ノードを削除する (dlist_delete)。 3. cexpr->context が保持するメモリコンテキストを削除し、関連するすべてのストレージを解放する (MemoryContextDelete)。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/pl/plpgsql/src/pl_exec.c:7998 (確認済み)
  • **呼び出し先**:
  • dlist_delete
  • MemoryContextDelete
  • Assert

関連データ構造

  • CachedExpression: キャッシュされた式を保持する構造体。
  • dlist_node: グローバルリスト管理用のノード。
  • MemoryContext: 関連するメモリ管理コンテキスト。

失敗モードと境界条件

  • cexpr が NULL の場合、または magic フィールドが不正な場合は Assert によりプロセスが停止する可能性がある。

保守担当者向けメモ

  • 本関数はメモリコンテキスト全体を削除するため、呼び出し後に cexpr ポインタを参照することは避けること。
  • dlist_delete を使用しているため、本関数を呼び出す前に当該オブジェクトがグローバルリストに登録されていることが前提となる。
