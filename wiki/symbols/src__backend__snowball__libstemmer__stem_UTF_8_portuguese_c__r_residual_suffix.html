r_residual_suffix

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_portuguese.c:820

目的

ポルトガル語のステミング処理において、単語の末尾に残存する特定の接尾辞を検索し、条件を満たす場合に削除を行う。

シグネチャまたは形状

static int r_residual_suffix(struct SN_env * z)

入力と出力

  • **入力**: z (Snowball ステミング環境を保持する SN_env 構造体へのポインタ)
  • **出力**: 成功時に 1、失敗または条件不一致時に 0、エラー発生時に負の値を返す。

前提条件と不変条件

  • z が有効な SN_env 構造体を指していること。
  • a_7 テーブルに定義された接尾辞パターンが、現在の単語末尾に存在するかを判定する。
  • 削除対象の接尾辞が RV 領域内にあることが削除の必須条件である。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を z->ket に設定する。 2. find_among_b を使用し、a_7 テーブルに基づいた接尾辞を後方検索する。見つからない場合は 0 を返す。 3. 見つかった接尾辞の開始位置を z->bra に設定する。 4. r_RV(z) を呼び出し、接尾辞が RV 領域に含まれるか検証する。検証失敗時はその戻り値を返す。 5. slice_del(z) を呼び出し、該当する接尾辞を単語から削除する。 6. 削除処理の結果を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: portuguese_UTF_8_stem（直接の呼び出し元はソースコードの制御フローによる）。
  • **呼び出し先**: find_among_b, r_RV, slice_del。

関連データ構造

  • SN_env: ステミングの状態（カーソル位置、領域境界、文字列バッファ）を保持する構造体。
  • a_7: 削除対象となる残余接尾辞のパターンを格納したテーブル。

失敗モードと境界条件

  • find_among_b が一致する接尾辞を見つけられない場合、0 を返して終了する。
  • r_RV が接尾辞の有効性を否定した場合、そのエラーコードまたは不一致コードを伝播する。
  • slice_del がメモリ操作等で失敗した場合、負の値を返す。
