MJEvalOuterValues

定義

  • 種別: function
  • 定義箇所: src/backend/executor/nodeMergejoin.c:288

目的

マージ結合における外部（Outer）タプルの結合キーを評価し、そのタプルが結合可能か、あるいは結合を終了すべきかを判定する。

シグネチャまたは形状

MJEvalResult MJEvalOuterValues(MergeJoinState *mergestate)

入力と出力

  • **入力**: MergeJoinState *mergestate（現在のマージ結合の状態を保持する構造体）。
  • **出力**: MJEvalResult 型の列挙値。
  • MJEVAL_MATCHABLE: 結合可能。
  • MJEVAL_NONMATCHABLE: 結合不可能（NULL値が含まれる等）。
  • MJEVAL_ENDOFJOIN: 結合を終了すべき状態。

前提条件と不変条件

  • mergestate->mj_OuterTupleSlot が有効なタプルを指しているか、あるいは TupIsNull で判定可能であること。
  • mergestate->mj_OuterEContext（式コンテキスト）が初期化済みであること。

アルゴリズムまたは振る舞い

1. mj_OuterTupleSlot が空であれば MJEVAL_ENDOFJOIN を返す。 2. mj_OuterEContext をリセットし、現在のタプルを ecxt_outertuple に設定する。 3. 結合句（mj_Clauses）をループで走査し、ExecEvalExpr を用いて各キーの値を評価する。 4. 評価結果が NULL の場合:

  • 最初の結合句で、かつ ssup_nulls_first が偽、かつ mj_FillOuter が偽であれば、結合を早期終了（MJEVAL_ENDOFJOIN）とみなす。
  • それ以外の場合は、そのタプルを結合不可能（MJEVAL_NONMATCHABLE）とマークする。

    • 5. 評価終了後、元のメモリコンテキストに復帰し、最終的な判定結果を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/executor/nodeMergejoin.c 内のメインループ（ExecMergeJoin 等）から複数回呼び出される。
  • **呼び出し先**: ExecEvalExpr（式評価）、ResetExprContext、MemoryContextSwitchTo 等の実行エンジン基盤関数。

関連データ構造

  • MergeJoinState: 結合の進行状況や式コンテキストを保持。
  • MergeJoinClause: 結合キーの式や評価結果（ldatum, lisnull）を保持。
  • MJEvalResult: 評価結果を示す列挙型。

失敗モードと境界条件

  • 外部入力が終了している場合（TupIsNull）、MJEVAL_ENDOFJOIN を返す。
  • 結合キーに NULL が含まれる場合、結合条件の構成に応じて MJEVAL_NONMATCHABLE または MJEVAL_ENDOFJOIN を返す。

保守担当者向けメモ

  • 本関数は MJEvalInnerValues と対になるロジックを持つ。
  • NULL 許容の結合（Outer Join）の場合、NULL が含まれていても即座に終了せず、MJEVAL_NONMATCHABLE として処理を継続する点に注意すること。
