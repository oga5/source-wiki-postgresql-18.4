int8inc_any

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/int8.c:803

目的

集約関数等の実行において、64ビット整数（int8）のインクリメント処理を抽象化して提供します。

シグネチャまたは形状

Datum int8inc_any(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS を介して関数呼び出し情報（fcinfo）を受け取ります。
  • **出力**: Datum 型の値を返します。

前提条件と不変条件

  • PG_FUNCTION_ARGS マクロにより、PostgreSQLの関数呼び出し規約に従った引数スタックが提供されている必要があります。

アルゴリズムまたは振る舞い

1. 受け取った fcinfo をそのまま int8inc 関数へ転送します。 2. int8inc の戻り値をそのまま呼び出し元へ返却します。

呼び出し元と呼び出し先

  • **呼び出し先**: int8inc
  • **呼び出し元**: ソースコード内の直接的な呼び出し元は確認されていませんが、int8inc() および int8inc_any() のプロサポート関数（int8inc_support）に関連付けられていることが示唆されています。

関連データ構造

  • Datum: PostgreSQLの汎用データ型表現。
  • FunctionCallInfoData (PG_FUNCTION_ARGS): 関数呼び出しのコンテキスト情報。

失敗モードと境界条件

  • 内部で呼び出される int8inc の挙動に依存します。算術演算におけるオーバーフロー等の境界条件は int8inc 側の実装に従います。

保守担当者向けメモ

  • 本関数は int8inc への単なるパススルー関数です。将来的に int8inc との差異が生じるような拡張を行う場合は、この委譲関係を維持するか検討してください。
