InitPrivsType

定義

  • 種別: enum
  • 定義箇所: src/include/catalog/pg_init_privs.h:77

目的

データベースオブジェクトの「初期権限（Initial Privileges）」の由来を識別するために使用されます。

シグネチャまたは形状

typedef enum InitPrivsType
{
	INITPRIVS_INITDB = 'i',
	INITPRIVS_EXTENSION = 'e',
} InitPrivsType;

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: char 型の文字コード（'i' または 'e'）として表現される識別子。

前提条件と不変条件

  • pg_init_privs システムカタログにおいて、権限が initdb 時に設定されたものか、拡張機能によって追加されたものかを区別するために使用されます。

アルゴリズムまたは振る舞い

  • INITPRIVS_INITDB ('i'): データベース初期化時に設定されたデフォルト権限であることを示します。
  • INITPRIVS_EXTENSION ('e'): 拡張機能のインストール時に設定された権限であることを示します。

呼び出し元と呼び出し先

  • 呼び出し元および呼び出し先の具体的な関数情報は、提供されたソース excerpt には含まれていません。
  • 関連情報として、recordExtensionInitPriv() 関数が拡張機能インストール時にこのカタログへデータを書き込む際に利用される可能性があります。

関連データ構造

  • FormData_pg_init_privs: pg_init_privs テーブルの行構造を定義する構造体。本列挙型はこの構造体の一部として使用されます。

失敗モードと境界条件

  • 定義された値（'i', 'e'）以外の値が割り当てられた場合の挙動については、本ソース excerpt に記載がありません。
