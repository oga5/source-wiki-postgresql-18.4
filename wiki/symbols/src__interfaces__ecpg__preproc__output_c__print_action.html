print_action

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/preproc/output.c:34

目的

WHENEVER 文で指定されたアクション（エラー発生時や警告時の動作）に対応するCコードを、出力ストリーム base_yyout へ書き出します。

シグネチャまたは形状

static void print_action(struct when *w)

入力と出力

  • **入力**: struct when *w （実行すべきアクションの種類と、必要に応じてコマンド文字列を保持する構造体）
  • **出力**: なし（副作用として base_yyout にCコードを出力）

前提条件と不変条件

  • base_yyout が有効なファイルストリームとしてオープンされていること。
  • 引数 w が指す構造体が、W_SQLPRINT, W_GOTO, W_DO, W_STOP, W_BREAK, W_CONTINUE のいずれかの有効な code を保持していること。

アルゴリズムまたは振る舞い

w->code の値に基づき、以下のCコードを base_yyout に出力します。

  • W_SQLPRINT: sqlprint();
  • W_GOTO: goto <command>;
  • W_DO: <command>;
  • W_STOP: exit (1);
  • W_BREAK: break;
  • W_CONTINUE: continue;

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/interfaces/ecpg/preproc/output.c 内の whenever_action 関数から呼び出されます。
  • 具体的には when_nf、when_warn、when_error の各グローバル変数を引数として渡されます。
  • **呼び出し先**:
  • 標準ライブラリ関数 fprintf を使用して出力を行います。

関連データ構造

  • struct when: アクションの種類（code）と、goto や関数呼び出しに必要な文字列（command）を保持します。

失敗モードと境界条件

  • w->code が定義された列挙値のいずれにも該当しない場合、コメント形式でエラーコードを出力し、プログラムの停止や異常終了は行いません。

保守担当者向けメモ

  • 本関数は WHENEVER 句の生成コードを決定する中心的な役割を担います。新しいアクションタイプを追加する場合は、switch 文にケースを追加してください。
  • base_yyout は preproc_extern.h で宣言されているグローバルな出力先です。
