nepali_UTF_8_stem

定義

  • **種別**: function
  • **定義箇所**: src/backend/snowball/libstemmer/stem_UTF_8_nepali.c:376

目的

ネパール語の単語に対し、Snowballアルゴリズムを用いて接尾辞を削除し、語幹を抽出する。

シグネチャまたは形状

int nepali_UTF_8_stem(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z（Snowballランタイム環境。処理対象の文字列バッファとカーソル位置を含む）
  • **出力**: int（成功時は 1、エラー発生時は負の値を返す）

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • 処理対象の文字列はUTF-8エンコーディングであること。
  • 関数実行中、カーソル位置（z->c）は文字列の末尾から先頭方向へ移動する。

アルゴリズムまたは振る舞い

1. **初期化**: カーソルを文字列の末尾（z->l）に設定する。 2. **カテゴリ1の削除**: r_remove_category_1 を適用する。 3. **カテゴリ2・3の反復削除**:

  • r_check_category_2 で条件を確認しつつ、r_remove_category_2 および r_remove_category_3 をループ内で適用する。
  • 各ステップでマッチした接尾辞は slice_del により削除される。

    • 4. **終了処理**: カーソルを元の位置（z->lb）に戻し、1 を返して終了する。

呼び出し元と呼び出し先

  • **呼び出し元**: 外部インターフェース（src/include/snowball/libstemmer/stem_UTF_8_nepali.h で宣言）。具体的な呼び出し元コードの詳細は提供されていない。
  • **呼び出し先**:
  • r_remove_category_1
  • r_check_category_2
  • r_remove_category_2
  • r_remove_category_3

関連データ構造

  • struct SN_env: Snowballランタイムの共通環境構造体。

失敗モードと境界条件

  • 内部関数（r_remove_category_x 等）が負の値を返した場合、その値をそのまま呼び出し元へ伝播させる。
  • 文字列が空、または接尾辞に該当するパターンが存在しない場合は、何も削除せず終了する。

保守担当者向けメモ

  • 本関数はSnowballコンパイラによる自動生成コードである。直接編集せず、元のSnowballスクリプト（.sbl）を修正すること。
  • UTF-8のマルチバイト文字を扱うため、バイト単位の操作には注意が必要である。
