pgstat_reset_entries_of_kind

定義

  • 種別: function
  • 定義箇所: src/backend/utils/activity/pgstat_shmem.c:1156

目的

指定された統計種別（PgStat_Kind）に該当するすべての統計エントリをリセットし、リセット時刻を更新します。

シグネチャまたは形状

void pgstat_reset_entries_of_kind(PgStat_Kind kind, TimestampTz ts)

入力と出力

  • **入力**:
  • kind: リセット対象とする統計情報の種別。
  • ts: リセット時刻（TimestampTz）。
  • **出力**: なし。

前提条件と不変条件

  • 呼び出し元は、統計情報の共有メモリ領域が初期化済みであることを前提とします。

アルゴリズムまたは振る舞い

1. 内部関数 pgstat_reset_matching_entries を呼び出します。 2. その際、第一引数に match_kind（種別によるフィルタリング用関数）を渡し、第二引数には kind を Datum 型に変換して渡します。 3. 第三引数としてリセット時刻 ts をそのまま引き渡します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/activity/pgstat.c:883 からの呼び出しが確認されています。
  • **呼び出し先**: pgstat_reset_matching_entries を呼び出します。

関連データ構造

  • PgStat_Kind: 統計情報の種別を定義する列挙型。
  • TimestampTz: 時刻を表す型。
  • Datum: PostgreSQLの汎用データ型。

失敗モードと境界条件

  • 共有メモリ上の統計情報が破損している場合や、統計システムが未初期化の状態での呼び出しは未定義の動作を引き起こす可能性があります。

保守担当者向けメモ

  • 本関数は単なるラッパーであり、実質的なリセットロジックは pgstat_reset_matching_entries に集約されています。種別ごとのリセット処理を変更する場合は、対象の match_kind 実装を確認してください。
