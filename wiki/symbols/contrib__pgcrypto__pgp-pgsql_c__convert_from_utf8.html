convert_from_utf8

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/pgp-pgsql.c:84

目的

UTF-8 エンコーディングのテキストデータを、現在のデータベースエンコーディングに変換します。

シグネチャまたは形状

static text *convert_from_utf8(text *src)

入力と出力

  • **入力**: src (UTF-8 でエンコードされた text 型へのポインタ)
  • **出力**: 変換後のデータベースエンコーディングでエンコードされた text 型へのポインタを返します。

前提条件と不変条件

  • 入力 src は有効な UTF-8 文字列である必要があります。
  • データベースのエンコーディング設定が GetDatabaseEncoding() を通じて正しく取得可能である必要があります。

アルゴリズムまたは振る舞い

内部的に convert_charset 関数を呼び出し、ソースエンコーディングを PG_UTF8、ターゲットエンコーディングを GetDatabaseEncoding() として指定することで文字コード変換を実行します。

呼び出し元と呼び出し先

  • **呼び出し先**: convert_charset
  • **呼び出し元**:
  • contrib/pgcrypto/pgp-pgsql.c:85 (定義直後の内部参照)
  • contrib/pgcrypto/pgp-pgsql.c:537

関連データ構造

  • text: PostgreSQL の可変長文字列型。

失敗モードと境界条件

  • convert_charset の仕様に依存します。変換不可能な文字が含まれる場合やメモリ不足が発生した場合、PostgreSQL のエラーハンドリング機構により例外が発生する可能性があります。

保守担当者向けメモ

  • 本関数は pgcrypto モジュール内での文字コード変換のラッパーです。データベースのエンコーディングが UTF-8 の場合、実質的にコピーまたは透過的な処理が行われます。
  • 機密情報を含む文字列を扱う場合は、メモリの解放時に clear_and_pfree が適切に使用されているか確認してください。
