r_residual_form

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_portuguese.c:827

目的

ポルトガル語のステミング処理において、特定の残余接尾辞（a_8 配列で定義）をマッチングし、条件に応じて削除または置換を行う。

シグネチャまたは形状

static int r_residual_form(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（Snowball ステミング環境構造体）
  • 出力: 成功時に 1、失敗または処理対象外の場合に 0、エラー発生時に負の値を返す。

前提条件と不変条件

  • z が有効な SN_env 構造体を指していること。
  • a_8 配列が find_among_b で検索可能な状態で定義されていること。
  • 処理対象の単語が z 内にロードされていること。

アルゴリズムまたは振る舞い

1. z->ket を現在のカーソル位置に設定し、a_8 配列を用いて後方から接尾辞を検索する。 2. マッチしない場合は 0 を返す。 3. マッチした場合、z->bra を設定し、among_var の値に応じて分岐する。

  • **case 1**: r_RV(z) を呼び出し、RV領域内であれば slice_del(z) で削除する。その後、直前の文字が 'u' か 'i' かを判定し、特定の条件（'gu' または 'ci'）を満たす場合に再度 r_RV(z) を経て削除を行う。
  • **case 2**: slice_from_s(z, 1, s_10) を呼び出し、接尾辞を特定の文字列に置換する。

    • 4. 最終的に 1 を返す。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_ISO_8859_1_portuguese.c:945 および src/backend/snowball/libstemmer/stem_UTF_8_portuguese.c:951 からの呼び出しが確認されている。
  • 呼び出し先: find_among_b, r_RV, slice_del, slice_from_s（いずれも Snowball ランタイム関数）。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_8: 検索対象の接尾辞を保持する among 構造体の配列。

失敗モードと境界条件

  • find_among_b がマッチしない場合、処理を中断して 0 を返す。
  • r_RV や slice_del 等の内部関数が負の値を返した場合、その値をそのまま伝播させて終了する。
  • 文字列の先頭や境界（z->lb）に達した場合、条件判定により処理がスキップされる。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードである。ロジックの変更が必要な場合は、直接編集せず元の .sbl スクリプトを修正すること。
  • case 1 内の文字判定（'u', 'g', 'i', 'c'）は、ポルトガル語の綴りルールに基づく特殊な削除条件である。
