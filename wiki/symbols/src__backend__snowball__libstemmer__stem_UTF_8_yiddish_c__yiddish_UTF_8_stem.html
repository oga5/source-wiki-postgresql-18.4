yiddish_UTF_8_stem

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_yiddish.c:1212

目的

イディッシュ語のテキストに対してステミング（語幹抽出）処理を行い、単語の接尾辞を除去して正規化された語幹を生成する。

シグネチャまたは形状

int yiddish_UTF_8_stem(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（ステミング環境を保持する構造体。処理対象の文字列や領域情報を含む）
  • **出力**: 成功時に 1、エラー発生時に負の値を返す。

前提条件と不変条件

  • SN_env 構造体は、yiddish_UTF_8_create_env によって適切に初期化されている必要がある。
  • 処理対象の文字列は UTF-8 エンコーディングであること。

アルゴリズムまたは振る舞い

1. **前処理**: r_prelude(z) を呼び出し、文字の正規化や不要な記号の削除を行う。 2. **領域特定**: r_mark_regions(z) を呼び出し、ステミングのルール適用範囲（R1など）を計算する。この際、現在のカーソル位置を一時保存し、計算後に復元する。 3. **接尾辞除去**: r_standard_suffix(z) を呼び出し、単語末尾から定義済みの接尾辞を検索・除去する。 4. **終了処理**: カーソル位置を初期状態に戻し、成功を示す 1 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/include/snowball/libstemmer/stem_UTF_8_yiddish.h および src/backend/snowball/libstemmer/stem_UTF_8_yiddish.c 内で宣言されている（具体的な呼び出し箇所は提供されたソース内では特定不可）。
  • **呼び出し先**: r_prelude, r_mark_regions, r_standard_suffix。

関連データ構造

  • SN_env: ステミング環境を管理する構造体。
  • among: 接尾辞検索に使用されるトライ木的なデータ構造（r_standard_suffix 内で使用）。

失敗モードと境界条件

  • 各内部関数（r_prelude, r_mark_regions, r_standard_suffix）が負の値を返した場合、その値をそのまま呼び出し元へ伝播させ終了する。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードである。ロジックの変更が必要な場合は、直接編集せず元の Snowball スクリプト（.sbl）を修正すること。
  • r_R1 などのガード条件が正しく機能しているか確認する際は、r_mark_regions の実装を参照すること。
