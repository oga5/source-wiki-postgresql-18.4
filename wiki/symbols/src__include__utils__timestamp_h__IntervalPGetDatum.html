IntervalPGetDatum

定義

  • 種別: function
  • 定義箇所: src/include/utils/timestamp.h:57

目的

Interval 構造体へのポインタを、PostgreSQLの関数マネージャ（FMGR）で扱われる Datum 型に変換する。

シグネチャまたは形状

static inline Datum IntervalPGetDatum(const Interval *X)

入力と出力

  • **入力**: const Interval *X （変換対象の Interval 構造体へのポインタ）
  • **出力**: Datum （X を指し示す Datum 値）

前提条件と不変条件

  • 入力ポインタ X は有効な Interval 構造体を指している必要がある。
  • 本関数はインライン関数として定義されており、呼び出し元で PointerGetDatum マクロが解決可能であること。

アルゴリズムまたは振る舞い

内部的に PointerGetDatum(X) を呼び出し、ポインタ値を Datum 型へキャストして返却する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/btree_gin/btree_gin.c
  • contrib/btree_gist/btree_interval.c
  • **呼び出し先**:
  • PointerGetDatum （マクロ）

関連データ構造

  • Interval: PostgreSQLの期間型を表す構造体。
  • Datum: PostgreSQLの汎用データ型。

失敗モードと境界条件

  • 入力ポインタが NULL の場合、PointerGetDatum の仕様に従い (Datum) 0 が返される。

保守担当者向けメモ

  • 本関数は src/include/utils/timestamp.h で定義されたインライン関数であり、パフォーマンスを考慮して設計されている。
  • Interval 型のデータ操作において、SQL層と内部表現を橋渡しする標準的な手段である。
