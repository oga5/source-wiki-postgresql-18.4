DatumGetTSQuerySign

定義

  • 種別: static inline function
  • 定義箇所: src/include/tsearch/ts_utils.h:257

目的

PostgreSQL の Datum 型として保持されている TSQuerySign データを、本来の TSQuerySign 型へキャストして取り出すための変換関数です。

シグネチャまたは形状

static inline TSQuerySign DatumGetTSQuerySign(Datum X)

入力と出力

  • **入力**: Datum X （TSQuerySign を格納した Datum 型の値）
  • **出力**: TSQuerySign 型の値

前提条件と不変条件

  • 入力された Datum X は、内部的に int64 として表現可能な形式で格納されている必要があります。

アルゴリズムまたは振る舞い

1. DatumGetInt64(X) を呼び出し、Datum 型を int64 に変換します。 2. その結果を TSQuerySign 型にキャストして返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/tsquery_gist.c: GiST インデックスの処理において、GETENTRY マクロや key の取得時に使用。
  • src/include/tsearch/ts_utils.h: PG_GETARG_TSQUERYSIGN マクロ内で使用。
  • **呼び出し先**:
  • DatumGetInt64 （PostgreSQL のコアユーティリティ関数）

関連データ構造

  • TSQuerySign: 全文検索クエリのシグネチャを表現する型。
  • Datum: PostgreSQL の汎用データ型。

失敗モードと境界条件

  • Datum に格納されている値が int64 に収まらない場合、キャスト時にデータが切り詰められる可能性があります。

保守担当者向けメモ

  • 本関数は static inline で定義されており、ヘッダーファイル内で完結しています。
  • GiST インデックスのキー操作において頻繁に使用されるため、変更時はインデックスの整合性に影響がないか確認してください。
