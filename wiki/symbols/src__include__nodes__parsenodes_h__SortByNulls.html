SortByNulls

定義

  • 種別: enum
  • 定義箇所: src/include/nodes/parsenodes.h:52

目的

SQLの ORDER BY 句における NULLS FIRST または NULLS LAST の指定を表現するための列挙型です。

シグネチャまたは形状

typedef enum SortByNulls
{
    SORTBY_NULLS_DEFAULT,
    SORTBY_NULLS_FIRST,
    SORTBY_NULLS_LAST,
} SortByNulls;

入力と出力

  • 入力: SQLパーサーが構文解析時に NULLS 指定を検出した際、この列挙値が割り当てられます。
  • 出力: 解析後のノード構造体（SortBy 等）の一部として保持され、後続のクエリ解析やプラン生成プロセスで使用されます。

前提条件と不変条件

  • SORTBY_NULLS_DEFAULT は、明示的な指定がない場合のデフォルトのソート順序を示します。
  • SORTBY_NULLS_FIRST および SORTBY_NULLS_LAST は、それぞれ NULL 値をソート順の先頭または末尾に配置することを明示します。

アルゴリズムまたは振る舞い

本シンボル自体は状態を保持する定義体であり、具体的なソートアルゴリズムのロジックは含みません。パーサーが生成したこの値に基づき、プランナーが適切なソート順序を決定します。

呼び出し元と呼び出し先

  • 呼び出し元: src/include/nodes/parsenodes.h 内の構造体定義（SortBy 等）でフィールドとして参照されています。
  • 呼び出し先: 本シンボルは列挙型であるため、直接的な関数呼び出し先はありません。

関連データ構造

  • src/include/nodes/parsenodes.h: SortBy 構造体などで sortby_nulls や nulls_ordering フィールドの型として使用されています。

失敗モードと境界条件

  • 該当なし（列挙型定義のため、実行時の失敗モードは存在しません）。
