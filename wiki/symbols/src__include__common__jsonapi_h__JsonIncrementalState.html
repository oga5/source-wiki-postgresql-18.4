JsonIncrementalState

定義

  • 種別: struct
  • 定義箇所: src/include/common/jsonapi.h:64

目的

JSON の増分（ストリーミング）解析における状態を保持するための構造体です。

シグネチャまたは形状

typedef struct JsonIncrementalState JsonIncrementalState;

※構造体の内部メンバは提供されたソース抜粋には含まれていません。

入力と出力

  • 入力: JsonLexContext 内の inc_state メンバとして保持されます。
  • 出力: 解析プロセスにおける状態の永続化に使用されます。

前提条件と不変条件

  • JsonLexContext の一部として割り当てられ、管理されます。
  • 解析のライフサイクルを通じて、増分解析の進捗を追跡するために使用されます。

アルゴリズムまたは振る舞い

  • pg_parse_json_incremental を通じた断片的な JSON データの解析において、前回の解析終了時点の状態を保持し、次回の入力データに対する解析を継続可能にします。

呼び出し元と呼び出し先

  • 呼び出し元: src/common/jsonapi.c 内の解析ロジックにおいて、JsonLexContext の初期化時に ALLOC0 を介してメモリが割り当てられます。
  • 呼び出し先: 提供されたソースからは、本構造体自体が直接関数を呼び出す関係は確認できません。

関連データ構造

  • JsonLexContext: 本構造体を保持する親構造体。
  • JsonSemAction: 解析時のコールバック定義。

失敗モードと境界条件

  • メモリ不足時: src/common/jsonapi.c に failed_inc_oom という静的変数が定義されており、メモリ割り当て失敗時のエラー状態として利用されている可能性があります。

保守担当者向けメモ

  • 構造体の詳細なメンバ定義は src/common/jsonapi.c の実装側に依存している可能性があります。必要に応じて実装ファイルを確認してください。
