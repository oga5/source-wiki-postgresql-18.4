r_v_ending

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_hungarian.c:518

目的

ハンガリー語のステミング処理において、特定の接尾辞（a_1 配列で定義）を単語末尾から検出し、R1 領域内であれば置換または削除を行う。

シグネチャまたは形状

static int r_v_ending(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z（ステミング実行環境）
  • 出力: 成功時に 1、失敗または条件不一致時に 0、エラー発生時に負の値を返す。

前提条件と不変条件

  • z->c（現在のカーソル位置）が有効な範囲にあること。
  • r_R1(z) が呼び出され、対象の接尾辞が R1 領域内にあることが確認されること。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を z->ket に設定する。 2. 単語末尾の文字が特定のバイト値（161 または 169）であるかを確認し、一致しなければ終了する。 3. find_among_b を使用して、末尾が a_1 配列の接尾辞と一致するか判定する。 4. 一致した場合、z->bra を設定し、r_R1(z) を呼び出して接尾辞が R1 領域内にあるか検証する。 5. 検証成功後、among_var の値に応じて slice_from_s を実行し、接尾辞を置換する。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_hungarian.c および src/backend/snowball/libstemmer/stem_ISO_8859_2_hungarian.c 内のステミング処理フローから呼び出される。
  • 呼び出し先: find_among_b, r_R1, slice_from_s。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_1: 接尾辞マッチングに使用される among 構造体の配列。

失敗モードと境界条件

  • 単語末尾の文字が条件（161, 169）に合致しない場合は 0 を返す。
  • find_among_b が一致を見つけられない場合は 0 を返す。
  • r_R1 が領域外と判断した場合、その戻り値をそのまま返す。
  • slice_from_s がエラーを返した場合、その負の値をそのまま返す。
