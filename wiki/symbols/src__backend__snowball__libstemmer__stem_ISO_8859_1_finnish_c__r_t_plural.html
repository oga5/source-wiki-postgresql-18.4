r_t_plural

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_finnish.c:528

目的

フィンランド語のステミング処理において、特定の複数形語尾（'t' で終わるもの）を単語から除去する。

シグネチャまたは形状

static int r_t_plural(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z（Snowballのステミング環境状態）
  • 出力: int（成功時に 1、失敗または条件不一致時に 0、エラー時に負の値を返す）

前提条件と不変条件

  • z->I[1]（領域R1の開始位置）および z->I[0]（領域R2の開始位置）が適切に設定されていること。
  • g_V1（母音グループ）が定義されていること。

アルゴリズムまたは振る舞い

1. **第一段階（'t' の除去）**:

  • 現在位置が z->I[1] 以降であることを確認。
  • 単語末尾が 't' であり、かつその直前に母音が存在する場合、't' を削除する。

    • 2. **第二段階（語尾の照合と除去）**:

  • 現在位置が z->I[0] 以降であることを確認。
  • a_9 配列を用いて、単語末尾の特定の語尾パターンを逆方向に検索する。
  • マッチした語尾が特定の条件（s_4 との比較）を満たさない場合、その語尾を削除する。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_ISO_8859_1_finnish.c:703 および src/backend/snowball/libstemmer/stem_UTF_8_finnish.c:707 から呼び出される。
  • 呼び出し先: slice_del（文字列削除）、in_grouping_b（母音判定）、find_among_b（語尾パターン照合）、eq_s_b（文字列比較）。

関連データ構造

  • struct SN_env: ステミングの実行状態を保持する構造体。
  • a_9: 削除対象の語尾パターンを格納した among 構造体配列。
  • g_V1: 母音判定用のグループ定義。

失敗モードと境界条件

  • z->c が境界（z->lb）を超えた場合や、slice_del がメモリ不足等で負の値を返した場合は、その値をそのまま伝播させる。
  • 検索対象の語尾が見つからない場合や、条件を満たさない場合は 0 を返す。

保守担当者向けメモ

  • 本関数はSnowballコンパイラによる自動生成コードである。ロジックの変更が必要な場合は、直接編集せず元の .sbl スクリプトを修正すること。
  • ISO-8859-1版とUTF-8版の両方で同名の関数が定義されているため、修正時は両方の整合性に注意が必要。
