temp_tablespaces_extra

定義

  • 種別: struct
  • 定義箇所: src/backend/commands/tablespace.c:1189

目的

一時テーブルスペースのリストを保持し、SetTempTablespaces() へ渡すためのデータ構造です。

シグネチャまたは形状

typedef struct
{
    int     numSpcs;
    Oid     tblSpcs[FLEXIBLE_ARRAY_MEMBER];
} temp_tablespaces_extra;

入力と出力

  • 入力: numSpcs（テーブルスペース数）、tblSpcs（テーブルスペース OID の配列）。
  • 出力: なし（データ保持用構造体）。

前提条件と不変条件

  • tblSpcs は FLEXIBLE_ARRAY_MEMBER を使用しているため、構造体インスタンスは offsetof(temp_tablespaces_extra, tblSpcs) に配列サイズを加算したメモリ領域を確保する必要があります。

アルゴリズムまたは振る舞い

  • GUC 変数 temp_tablespaces の解析結果を格納し、後続のトランザクション処理で一時テーブルスペースの設定を伝播させるために使用されます。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/commands/tablespace.c 内の GUC 設定関連処理（guc_malloc を用いたメモリ確保箇所など）。
  • 呼び出し先: SetTempTablespaces()（本構造体から OID リストを渡す）。

関連データ構造

  • Oid: テーブルスペースを識別するオブジェクト ID。

失敗モードと境界条件

  • メモリ確保失敗: guc_malloc によるメモリ確保が失敗した場合の挙動は、呼び出し元のエラーハンドリングに依存します。

保守担当者向けメモ

  • 本構造体は可変長配列を含むため、インスタンス生成時には必ず offsetof を用いた適切なサイズ計算を行ってください。
