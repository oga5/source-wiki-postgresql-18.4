integer_digits

定義

  • 種別: function
  • 定義箇所: src/fe_utils/print.c:277

目的

文字列として与えられた数値の整数部分に含まれる桁数をカウントします。

シグネチャまたは形状

static int integer_digits(const char *my_str)

入力と出力

  • **入力**: my_str (数値の文字列表現)
  • **出力**: 整数部分の桁数を示す int 値。

前提条件と不変条件

  • my_str はヌル終端文字列である必要があります。
  • 符号（+ または -）は桁数としてカウントされません。

アルゴリズムまたは振る舞い

1. 文字列の先頭が + または - の場合、ポインタを1つ進めて符号をスキップします。 2. 標準ライブラリ関数 strspn を使用し、文字列の先頭から連続する数字（0〜9）の長さを計算して返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/fe_utils/print.c 内の数値フォーマット処理に関連する箇所（291行目、333行目など）。
  • **呼び出し先**: 標準ライブラリ関数 strspn。

関連データ構造

特になし。

失敗モードと境界条件

  • 文字列が数字で始まらない場合、戻り値は 0 となります。
  • 符号のみの文字列や空文字列の場合、戻り値は 0 となります。

保守担当者向けメモ

  • 本関数は static であり、src/fe_utils/print.c 内でのみ使用されます。
  • 小数点以下の桁数は考慮されません。小数点が含まれる場合、strspn は小数点の手前までの長さを返します。
