r_particle_etc

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_finnish.c:307

目的

フィンランド語のステミング処理において、単語末尾から粒子（particle）や特定の接尾辞を識別し、条件に応じて削除する。

シグネチャまたは形状

static int r_particle_etc(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（Snowball ステミング環境）
  • **出力**: 成功時に 1、失敗または処理対象外の場合に 0、エラー発生時に負の値を返す。

前提条件と不変条件

  • z->I[1]（領域 R1 の開始位置）が設定されていること。
  • 現在のカーソル位置 z->c が z->I[1] 以降にあること。

アルゴリズムまたは振る舞い

1. **範囲制限**: z->I[1] を下限として、単語末尾から a_0 配列に基づいた接尾辞の検索を行う。 2. **マッチング**: find_among_b を使用して接尾辞を特定する。 3. **条件分岐**:

  • case 1: g_particle_end グループに含まれる文字が直前に存在する場合に削除対象とする。
  • case 2: r_R2 関数を呼び出し、領域 R2 内でのマッチングが確認できた場合に削除対象とする。

    • 4. **削除**: 特定された接尾辞を slice_del により削除する。

呼び出し元と呼び出し先

  • **呼び出し元**: finnish_UTF_8_stem（src/backend/snowball/libstemmer/stem_UTF_8_finnish.c）
  • **呼び出し先**:
  • find_among_b（接尾辞検索）
  • in_grouping_b_U（文字グループ判定）
  • r_R2（領域判定）
  • slice_del（文字列削除）

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_0: 粒子および接尾辞を定義した among 構造体の配列。
  • g_particle_end: 粒子判定に使用される文字グループ。

失敗モードと境界条件

  • z->c < z->I[1] の場合、処理を行わず 0 を返す。
  • find_among_b でマッチしない場合、0 を返す。
  • r_R2 や slice_del が負の値を返した場合、その値をそのまま伝播させる。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードであるため、直接編集せず、元の Snowball 言語ソースを修正すること。
  • a_0 配列の定義変更が本関数の挙動に直接影響する。
