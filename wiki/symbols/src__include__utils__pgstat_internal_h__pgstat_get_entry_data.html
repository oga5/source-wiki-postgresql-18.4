pgstat_get_entry_data

定義

  • 種別: function
  • 定義箇所: src/include/utils/pgstat_internal.h:903

目的

共有メモリ上の統計エントリ（PgStatShared_Common）から、特定の統計種別（PgStat_Kind）に対応するデータ領域へのポインタを取得します。

シグネチャまたは形状

static inline void *pgstat_get_entry_data(PgStat_Kind kind, PgStatShared_Common *entry);

入力と出力

  • **入力**:
  • kind: 統計の種別を指定する PgStat_Kind 型の値。
  • entry: 統計エントリの共通ヘッダーを指す PgStatShared_Common 型のポインタ。
  • **出力**:
  • 指定された種別に対応する統計データ本体のメモリ位置を指す void * 型のポインタ。

前提条件と不変条件

  • pgstat_get_kind_info(kind) が返すメタデータ構造体の shared_data_off が 0 ではないこと。
  • shared_data_off が PG_UINT32_MAX 未満であること。
  • entry は有効な共有メモリ領域を指していること。

アルゴリズムまたは振る舞い

1. pgstat_get_kind_info(kind) を呼び出し、当該統計種別のメタデータから共有データへのオフセット（shared_data_off）を取得します。 2. 取得したオフセットが有効範囲内であることを Assert で検証します。 3. entry のアドレスにオフセットを加算し、計算されたアドレスを返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/activity/pgstat.c (複数箇所)
  • src/backend/utils/activity/pgstat_shmem.c (複数箇所)
  • **呼び出し先**:
  • pgstat_get_kind_info (インライン展開されるメタデータ取得関数)

関連データ構造

  • PgStat_Kind: 統計種別を識別する列挙型。
  • PgStatShared_Common: 共有統計エントリの共通ヘッダー構造体。
  • PgStat_KindInfo: 統計種別ごとのメタデータ（オフセット情報を含む）。

失敗モードと境界条件

  • kind に対応するメタデータが存在しない場合や、shared_data_off が正しく設定されていない場合、Assert によりデバッグビルドで停止します。
  • オフセットが 0 の場合、アサーション違反が発生します。

保守担当者向けメモ

  • 本関数は static inline で定義されており、ヘッダーファイル内で完結しています。
  • 共有メモリ上のレイアウト変更に伴い shared_data_off の計算ロジックが変更される可能性があるため、統計種別の追加時には PgStat_KindInfo の定義と整合性を確認してください。
