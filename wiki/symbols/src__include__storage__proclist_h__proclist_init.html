proclist_init

定義

  • 種別: function
  • 定義箇所: src/include/storage/proclist.h:28

目的

proclist_head 構造体を初期化し、リストが空であることを示す状態に設定します。

シグネチャまたは形状

static inline void proclist_init(proclist_head *list)

入力と出力

  • **入力**: list - 初期化対象となる proclist_head 構造体へのポインタ。
  • **出力**: なし（副作用として list が指す構造体のメンバを更新）。

前提条件と不変条件

  • list は有効な proclist_head 構造体を指している必要があります。
  • 初期化後、リストの head および tail メンバは INVALID_PROC_NUMBER に設定されます。

アルゴリズムまたは振る舞い

1. 引数として受け取った list の head メンバに INVALID_PROC_NUMBER を代入します。 2. 同様に list の tail メンバに INVALID_PROC_NUMBER を代入します。 3. これにより、リストが空であることを示す標準的な状態となります。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/storage/lmgr/condition_variable.c:38
  • src/backend/storage/lmgr/lwlock.c:726
  • src/backend/storage/lmgr/lwlock.c:939
  • src/backend/storage/lmgr/lwlock.c:1745
  • **呼び出し先**: なし（インライン関数であり、他の関数を呼び出しません）。

関連データ構造

  • proclist_head: リストの先頭と末尾の procno を保持する構造体。
  • INVALID_PROC_NUMBER: リストが空であることを示す定数。

失敗モードと境界条件

  • list が NULL の場合、未定義の動作（セグメンテーション違反など）が発生します。

保守担当者向けメモ

  • 本関数は共有メモリ上のデータ構造を扱うため、絶対ポインタではなく procno を使用してリストを管理する設計の一環です。
  • proclist_types.h で定義された型に依存しています。
