pg_valid_server_encoding_id

定義

  • 種別: function
  • 定義箇所: src/common/encnames.c:515

目的

指定された整数値が、PostgreSQL サーバーでサポートされている有効なエンコーディング ID であるかを判定します。

シグネチャまたは形状

int pg_valid_server_encoding_id(int encoding)

入力と出力

  • **入力**: encoding (int) - 検証対象のエンコーディング ID。
  • **出力**: int - 有効な場合は真（非ゼロ）、無効な場合は偽（0）を返します。

前提条件と不変条件

  • mb/pg_wchar.h で定義されている PG_VALID_BE_ENCODING マクロの検証ロジックに依存しています。

アルゴリズムまたは振る舞い

1. 引数として受け取った encoding を、PG_VALID_BE_ENCODING マクロに渡します。 2. マクロの評価結果（有効か否か）をそのまま呼び出し元へ返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/bin/initdb/initdb.c:2739 において、データベース初期化時のエンコーディング検証に使用されています。
  • **呼び出し先**: PG_VALID_BE_ENCODING マクロ（src/include/mb/pg_wchar.h で定義）。

関連データ構造

  • pg_enc 型（mb/pg_wchar.h で定義されるエンコーディング ID の列挙型）。

失敗モードと境界条件

  • 無効なエンコーディング ID が渡された場合、マクロの定義に従い 0 を返します。

保守担当者向けメモ

  • 本関数は単なるラッパーです。検証ロジックの変更が必要な場合は、src/include/mb/pg_wchar.h 内の PG_VALID_BE_ENCODING マクロ定義を確認してください。
