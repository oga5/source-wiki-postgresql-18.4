目的

単語内の領域「R1」が現在のカーソル位置以降に存在するかを判定するガード条件として機能します。

static int r_R1(struct SN_env * z)

  • **入力**: SN_env * z（ステミング環境構造体）
  • **出力**: int 型。条件を満たす場合は非ゼロ（真）、満たさない場合はゼロ（偽）を返します。

前提条件と不変条件

  • z->I[1] には、事前に r_mark_regions 等で計算された R1 領域の開始位置が格納されている必要があります。
  • z->c は現在のカーソル位置を示します。

現在のカーソル位置 z->c が、定義済みの R1 領域の開始位置 z->I[1] 以上であるかを比較します。

  • z->I[1] <= z->c が真であれば、現在の位置は R1 領域内またはそれ以降にあると判断されます。
