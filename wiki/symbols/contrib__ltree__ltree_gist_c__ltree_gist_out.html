ltree_gist_out

定義

  • 種別: function
  • 定義箇所: contrib/ltree/ltree_gist.c:31

目的

ltree_gist 型の値を人間が読める形式に変換しようとする試みを拒否し、エラーを報告します。

シグネチャまたは形状

Datum ltree_gist_out(PG_FUNCTION_ARGS)

入力と出力

  • 入力: PG_FUNCTION_ARGS（GiST インデックス内部の ltree_gist データ）
  • 出力: なし（関数内でエラーを発生させ、処理を中断します）

前提条件と不変条件

  • 本関数は ltree_gist 型の出力関数として登録されていますが、当該型は直接表示することを意図していません。

アルゴリズムまたは振る舞い

1. ereport を呼び出し、ERRCODE_FEATURE_NOT_SUPPORTED（機能非サポート）エラーを発生させます。 2. エラーメッセージとして「cannot display a value of type ltree_gist」を出力します。 3. PG_RETURN_VOID() を呼び出しますが、実際にはエラーにより制御が戻ることはありません。

呼び出し元と呼び出し先

  • 呼び出し元: PostgreSQL の型システム（PG_FUNCTION_INFO_V1 マクロにより登録済み）。
  • 呼び出し先: ereport（PostgreSQL のエラー報告メカニズム）。

関連データ構造

  • ltree_gist: 本関数が対象とする内部インデックス構造体。

失敗モードと境界条件

  • 本関数は呼び出された時点で常に失敗（エラー発生）するように設計されています。

保守担当者向けメモ

  • 本関数は ltree_gist 型の表示を意図的に禁止しています。デバッグ目的であっても、この型を直接表示する機能は提供されていません。
