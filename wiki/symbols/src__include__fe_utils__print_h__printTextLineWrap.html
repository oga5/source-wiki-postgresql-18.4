printTextLineWrap

定義

  • 種別: enum
  • 定義箇所: src/include/fe_utils/print.h:61

目的

テキスト出力時における各行の折り返し状態（ラップ状態）を識別・管理するために使用されます。

シグネチャまたは形状

typedef enum printTextLineWrap
{
    PRINT_LINE_WRAP_NONE,
    PRINT_LINE_WRAP_WRAP,
    PRINT_LINE_WRAP_NEWLINE,
} printTextLineWrap;

入力と出力

  • 本シンボルは列挙型であり、関数ではありません。
  • 出力値として、以下のいずれかの状態を保持します。
  • PRINT_LINE_WRAP_NONE: 折り返しなし。
  • PRINT_LINE_WRAP_WRAP: 長すぎる行による折り返し。
  • PRINT_LINE_WRAP_NEWLINE: データ内に含まれる改行。

前提条件と不変条件

  • 特になし。

アルゴリズムまたは振る舞い

  • テキスト整形処理において、各カラムや行がどのような理由で改行・折り返しされたかを記録するために利用されます。

呼び出し元と呼び出し先

  • 呼び出し元: src/fe_utils/print.c:668 にて、各カラムのラップ状態を保持する変数として参照されています。
  • 呼び出し先: 本シンボルは列挙型であるため、直接的な呼び出し先はありません。

関連データ構造

  • printTableContent（src/include/fe_utils/print.h 内で定義）: テーブルの出力内容を管理する構造体であり、本列挙型はこれに関連する整形処理で使用されます。

失敗モードと境界条件

  • 列挙型であるため、定義された値以外の値が代入された場合の挙動は未定義です。

保守担当者向けメモ

  • 本シンボルは print.c におけるカラムの表示状態追跡に使用されています。新しい出力フォーマットやラップロジックを追加する際は、この列挙型に状態を追加する必要があるか検討してください。
