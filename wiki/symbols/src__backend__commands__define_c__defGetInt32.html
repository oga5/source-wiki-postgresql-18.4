defGetInt32

定義

  • 種別: function
  • 定義箇所: src/backend/commands/define.c:148

目的

DefElem ノードから整数値（int32）を安全に抽出する。

シグネチャまたは形状

int32 defGetInt32(DefElem *def)

入力と出力

  • **入力**: DefElem *def（解析対象の定義要素）
  • **出力**: int32 型の変換された値。
  • **副作用**: 変換失敗時や値が欠落している場合に ereport(ERROR, ...) を発行し、処理を中断する。

前提条件と不変条件

  • def ポインタは有効であること。
  • def->arg が T_Integer 型のノードであること。

アルゴリズムまたは振る舞い

1. def->arg が NULL の場合、ERRCODE_SYNTAX_ERROR でエラーを報告する。 2. nodeTag(def->arg) を評価する。

  • T_Integer の場合: intVal() マクロを使用して値を抽出し、int32 にキャストして返す。
  • それ以外の場合: ERRCODE_SYNTAX_ERROR でエラーを報告する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/commands/aggregatecmds.c
  • src/backend/commands/dbcommands.c
  • src/backend/commands/tablecmds.c
  • src/backend/commands/vacuum.c
  • **呼び出し先**:
  • intVal() (マクロ)
  • ereport() (エラー報告用)

関連データ構造

  • DefElem: SQL コマンドのオプションや定義を保持する構造体。
  • Node: def->arg が指す抽象構文木ノード。

失敗モードと境界条件

  • def->arg が NULL の場合: 構文エラー。
  • def->arg が T_Integer 以外のノード型（例: T_String）である場合: 構文エラー。

保守担当者向けメモ

  • 本関数は int32 への変換専用です。大きな数値や浮動小数点数を含む可能性がある場合は、defGetInt64 や defGetNumeric の利用を検討してください。
  • エラーメッセージは「%s requires an integer value」で統一されており、def->defname がエラー箇所を特定するために使用されます。
