MemoryChunkGetValue

定義

  • 種別: static inline 関数
  • 定義箇所: src/include/utils/memutils_memorychunk.h:221

目的

メモリチャンクのヘッダー情報（hdrmask）から、エンコードされた値（サイズ情報等）を抽出して返します。

シグネチャまたは形状

static inline Size MemoryChunkGetValue(MemoryChunk *chunk)

入力と出力

  • **入力**: MemoryChunk *chunk - 対象となるメモリチャンクへのポインタ。
  • **出力**: Size - チャンクの hdrmask から抽出された値。

前提条件と不変条件

  • 対象のチャンクが「外部管理モード（External）」ではないこと（Assert により検証）。
  • hdrmask は HdrMaskGetValue マクロを通じて正しくデコード可能であること。

アルゴリズムまたは振る舞い

1. HdrMaskIsExternal を用いて、チャンクが外部管理フラグを持っていないことを確認します。 2. HdrMaskGetValue を呼び出し、chunk->hdrmask からメタデータを抽出して返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/mmgr/alignedalloc.c
  • src/backend/utils/mmgr/aset.c
  • src/backend/utils/mmgr/bump.c
  • src/backend/utils/mmgr/generation.c
  • **呼び出し先**:
  • HdrMaskIsExternal (マクロ)
  • HdrMaskGetValue (マクロ)

関連データ構造

  • MemoryChunk: メモリコンテキストが割り当てるチャンクのヘッダー構造体。hdrmask フィールドを保持します。

失敗モードと境界条件

  • 外部管理チャンクに対して呼び出した場合、Assert が有効なビルドではアサーション失敗により停止します。

保守担当者向けメモ

  • 本関数は hdrmask のビット操作に依存しています。hdrmask のエンコード方式を変更する場合は、本関数および HdrMaskGetValue の定義を整合させる必要があります。
