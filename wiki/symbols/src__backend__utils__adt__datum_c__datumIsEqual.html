datumIsEqual

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/datum.c:222

目的

2つの Datum 値が等しいかどうかを、型情報（typByVal および typLen）に基づいて判定します。

シグネチャまたは形状

bool datumIsEqual(Datum value1, Datum value2, bool typByVal, int typLen)

入力と出力

  • **入力**:
  • value1, value2: 比較対象の Datum 値。
  • typByVal: 値渡し（true）か参照渡し（false）かを示すフラグ。
  • typLen: 型のバイト長。
  • **出力**:
  • 両者が等しい場合は true、それ以外は false を返します。

前提条件と不変条件

  • 呼び出し元は、対象となるデータ型に対して正しい typByVal および typLen を提供する必要があります。誤った値を提供した場合、メモリ破壊や不正なポインタ参照を引き起こす可能性があります。
  • 本関数は NULL 値の比較には対応していません。

アルゴリズムまたは振る舞い

1. **値渡しの場合 (typByVal が true)**:

  • value1 と value2 を直接比較します。データ型が Datum 内の余剰ビットをどのように埋めるかについて一貫性があることを前提としています。

    • 2. **参照渡しの場合 (typByVal が false)**:

  • datumGetSize を使用して両方の値のサイズを取得します。
  • サイズが異なる場合は直ちに false を返します。
  • サイズが等しい場合、DatumGetPointer で取得したポインタが指すメモリ領域を memcmp でバイト単位で比較します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/access/common/tupdesc.c
  • src/backend/access/heap/heapam.c
  • src/backend/nodes/equalfuncs.c
  • src/backend/optimizer/prep/prepagg.c
  • src/backend/parser/parse_coerce.c
  • src/backend/partitioning/partbounds.c
  • src/backend/storage/page/bufpage.c
  • **呼び出し先**:
  • datumGetSize
  • DatumGetPointer (マクロ)
  • memcmp (標準ライブラリ)

関連データ構造

  • Datum: PostgreSQL の汎用データ型表現。

失敗モードと境界条件

  • **TOAST データ**: TOAST されたデータや、異なる内部表現を持つ同一値に対しては、単純なバイト比較を行うため期待通りの結果にならない可能性があります。
  • **NULL 値**: 本関数は NULL 値を適切に処理しません。

保守担当者向けメモ

  • 本関数はバイナリイメージの同一性を判定するものであり、論理的な等価性とは異なる場合があります。
  • 拡張オブジェクト（Expanded Object）が含まれる場合、呼び出し元で事前にフラット化されている必要があります。
