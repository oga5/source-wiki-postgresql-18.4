目的

Porterステミングアルゴリズムのステップ1cを実装し、単語末尾の 'y' または 'Y' を 'i' に置換する処理を行う。

シグネチャまたは形状

static int r_Step_1c(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z（ステミング環境を保持する構造体）
  • **出力**: int（成功時に 1、失敗または条件不一致時に 0、エラー時に負の値を返す）

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • z->c は現在のカーソル位置を指す。
  • 処理対象の単語末尾が 'y' または 'Y' であることが前提となる。

アルゴリズムまたは振る舞い

1. **末尾チェック**: 現在のカーソル位置から後方へ 'y' または 'Y' を検索する。見つからない場合は終了する。 2. **母音チェック**: out_grouping_b を使用し、末尾の 'y'/'Y' の直前が母音（g_v）ではないことを確認する。 3. **置換**: 条件を満たす場合、slice_from_s を使用して末尾の文字を 'i' に置換する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_ISO_8859_1_english.c および stem_ISO_8859_1_porter.c、stem_UTF_8_english.c、stem_UTF_8_porter.c の各メインステミング関数から呼び出される。
  • **呼び出し先**: out_grouping_b（母音判定）、slice_from_s（文字列置換）。

関連データ構造

  • struct SN_env: ステミング環境（カーソル位置、文字列バッファ、領域境界など）。
  • g_v: 母音を定義するグループ構造体。

失敗モードと境界条件

  • 単語末尾が 'y'/'Y' でない場合、0 を返して終了する。
  • 直前が母音である場合、置換を行わず 0 を返して終了する。
  • slice_from_s が失敗した場合、その戻り値（負の値）をそのまま返す。
