name_bpchar

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/varchar.c:406

目的

name 型の値を bpchar (char(n)) 型の値に変換します。

シグネチャまたは形状

Datum name_bpchar(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS を介して name 型のデータを受け取ります。
  • **出力**: BpChar * 型（char(n) の内部表現）を Datum として返します。

前提条件と不変条件

  • 入力引数は Name 型として取得されます。
  • cstring_to_text を使用して変換が行われるため、結果は text 型と互換性のある varlena 構造体となります。

アルゴリズムまたは振る舞い

1. PG_GETARG_NAME(0) を使用して入力引数を Name 型として取得します。 2. NameStr マクロで Name 構造体から C 文字列を取り出し、cstring_to_text を用いて BpChar 型へ変換します。 3. PG_RETURN_BPCHAR_P マクロを使用して、変換された BpChar データを返します。

呼び出し元と呼び出し先

  • **呼び出し元**: ソースコード内のコメント（src/backend/utils/adt/varchar.c:400）に言及がありますが、具体的な呼び出し箇所は提供されていません。
  • **呼び出し先**:
  • PG_GETARG_NAME
  • NameStr
  • cstring_to_text
  • PG_RETURN_BPCHAR_P

関連データ構造

  • Name: PostgreSQL のシステム識別子等に使用される固定長文字列型。
  • BpChar: char(n) 型の内部表現。

失敗モードと境界条件

  • メモリ不足時など、cstring_to_text が失敗した場合は例外が発生します。

保守担当者向けメモ

  • 本関数は name から bpchar への単純な型キャストを提供します。
  • text 型への変換を介しているため、text 型のメモリ管理ルールに従います。
