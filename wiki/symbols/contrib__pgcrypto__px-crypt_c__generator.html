generator

定義

  • 種別: struct
  • 定義箇所: contrib/pgcrypto/px-crypt.c:126

目的

pgcrypto モジュールにおいて、パスワードハッシュ生成時に使用するソルト生成アルゴリズムのメタデータおよび実行関数を保持するための構造体です。

シグネチャまたは形状

struct generator
{
    char       *name;
    char       *(*gen) (unsigned long count, const char *input, int size,
                        char *output, int output_size);
    int         input_len;
    int         def_rounds;
    int         min_rounds;
    int         max_rounds;
};

入力と出力

  • **入力**: 構造体の各フィールドは、アルゴリズム名、ソルト生成関数ポインタ、入力長、およびラウンド数の範囲（デフォルト・最小・最大）を定義します。
  • **出力**: gen フィールドに指定された関数は、生成されたソルト文字列を output バッファに格納し、そのポインタを返します。

前提条件と不変条件

  • gen 関数ポインタは、指定されたラウンド数と入力データに基づいてソルトを生成できる実装である必要があります。
  • min_rounds は max_rounds 以下である必要があります。

アルゴリズムまたは振る舞い

  • 本構造体は gen_list 配列の要素として定義され、px_gen_salt() 関数がアルゴリズムを検索・特定するために使用されます。
  • px_gen_salt() は、この構造体の min_rounds および max_rounds を参照して、ユーザー指定のラウンド数が妥当であるかを検証します。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/pgcrypto/px-crypt.c 内の px_gen_salt() 関数が、この構造体の配列（gen_list）を走査して使用します。
  • **呼び出し先**: gen フィールドに格納された関数ポインタ（_crypt_gensalt_*_rn 等）が実行されます。
  • **注記**: ツールが検出した contrib/bloom/blutils.c や src/backend/ 等の他のファイルにおける「generator」への言及は、本構造体とは異なる概念（乱数生成器やOID生成器）を指している可能性が高く、直接的な関係は確認できません。

関連データ構造

  • gen_list: struct generator 型の静的配列。px-crypt.c 内で定義され、利用可能な全ソルト生成アルゴリズムを保持します。

失敗モードと境界条件

  • px_gen_salt() において、指定されたアルゴリズム名が gen_list に存在しない場合、処理は失敗します。
  • ラウンド数が min_rounds 未満または max_rounds を超える場合、バリデーションエラーとなります。

保守担当者向けメモ

  • 本構造体は px-crypt.c 内のソルト生成ロジックと密結合しています。新しいハッシュアルゴリズムを追加する際は、gen_list に本構造体のインスタンスを追加してください。
  • gen フィールドの関数ポインタが参照する具体的な実装は、crypt-gensalt.c 等の関連ファイルを確認してください。
