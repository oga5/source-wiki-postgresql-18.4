parallel_vacuum_init

定義

  • 種別: function
  • 定義箇所: src/backend/commands/vacuumparallel.c:242

目的

並列 VACUUM 実行のための環境構築および共有メモリ（DSM）の初期化を行う。

シグネチャまたは形状

ParallelVacuumState *parallel_vacuum_init(Relation rel, Relation *indrels, int nindexes,
                                          int nrequested_workers, int vac_work_mem,
                                          int elevel, BufferAccessStrategy bstrategy)

入力と出力

  • **入力**:
  • rel: 対象テーブルのリレーション。
  • indrels: インデックスリレーションの配列。
  • nindexes: インデックス数。
  • nrequested_workers: 要求されたワーカー数。
  • vac_work_mem: VACUUM 用のメモリ制限。
  • elevel: エラーレベル。
  • bstrategy: バッファアクセス戦略。
  • **出力**:
  • 成功時: 初期化された ParallelVacuumState 構造体へのポインタ。
  • 失敗時（並列実行不可と判断された場合）: NULL。

前提条件と不変条件

  • nrequested_workers >= 0 であること。
  • nindexes > 0 であること。
  • parallel_vacuum_compute_workers により、並列実行可能なワーカー数が 1 以上であると判定されること。

アルゴリズムまたは振る舞い

1. parallel_vacuum_compute_workers を呼び出し、並列実行の可否とワーカー数を決定する。 2. ParallelContext を作成し、共有メモリ上の各領域（インデックス統計、共有制御情報、BufferUsage/WalUsage、クエリ文字列）のサイズを見積もる。 3. InitializeParallelDSM で共有メモリを確保する。 4. インデックスごとの並列処理オプションを確認し、PVIndStats を初期化して共有メモリに配置する。 5. PVShared 構造体を初期化し、maintenance_work_mem の分配、TidStore（デッドタプル管理用）の作成、アトミックカウンタの初期化を行う。 6. 各ワーカー用の BufferUsage および WalUsage 領域を確保する。 7. デバッグ用クエリ文字列が存在する場合、共有メモリにコピーする。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/access/heap/vacuumlazy.c (Lazy Vacuum のメインロジック)
  • **呼び出し先**:
  • parallel_vacuum_compute_workers
  • CreateParallelContext
  • shm_toc_estimate_chunk / shm_toc_estimate_keys
  • InitializeParallelDSM
  • shm_toc_allocate / shm_toc_insert
  • TidStoreCreateShared

関連データ構造

  • ParallelVacuumState: 並列 VACUUM の状態を保持するメイン構造体。
  • PVShared: 全ワーカーで共有される制御情報。
  • PVIndStats: インデックスごとの統計情報。
  • TidStore: 削除対象タプルを管理する共有データ構造。

失敗モードと境界条件

  • 並列実行が不可能な場合（parallel_workers <= 0）、NULL を返す。
  • インデックスが並列処理に対応していない場合、そのインデックスは並列処理対象から除外される。

保守担当者向けメモ

  • 本関数は ParallelContext のエントリポイントとして "parallel_vacuum_main" を指定している。
  • maintenance_work_mem は並列処理可能なインデックス数に基づいてワーカー間で分割される。
  • TidStore の DSA ハンドルは PVShared を介してワーカーに共有される。
