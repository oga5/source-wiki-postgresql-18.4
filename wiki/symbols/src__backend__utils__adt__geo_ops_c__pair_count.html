pair_count

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/geo_ops.c:391

目的

文字列内の指定された区切り文字の出現回数をカウントし、それに基づいて座標ペア（点）の数を検証・算出します。

シグネチャまたは形状

static int pair_count(char *s, char delim);

入力と出力

  • **入力**:
  • s: 解析対象の文字列。
  • delim: カウント対象の区切り文字（例: ','）。
  • **出力**:
  • 成功時: 区切り文字の数から算出されたペア数（(ndelim + 1) / 2）。
  • 失敗時: -1（区切り文字が奇数個の場合など、ペアとして不完全な場合）。

前提条件と不変条件

  • 入力文字列 s はヌル終端されている必要があります。
  • 区切り文字の数が偶数であることは、ペアが正しく構成されているための必要条件として扱われます。

アルゴリズムまたは振る舞い

1. strchr を用いて、文字列 s 内の delim の出現回数 ndelim を走査してカウントします。 2. カウントされた ndelim が偶数であるかを確認します。 3. ndelim が偶数の場合、ペア数として (ndelim + 1) / 2 を返します。 4. ndelim が奇数の場合、ペアとして成立しないと判断し -1 を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/geo_ops.c:1414
  • src/backend/utils/adt/geo_ops.c:3425
  • **呼び出し先**:
  • 標準ライブラリ関数 strchr を呼び出します。

関連データ構造

  • 特になし。

失敗モードと境界条件

  • **区切り文字が奇数**: ndelim % 2 が真となるため、-1 が返されます。
  • **区切り文字が存在しない**: ndelim が 0 となり、(0 % 2) は偽であるため、(0 + 1) / 2 すなわち 0 が返されます。
  • **空文字列**: ndelim が 0 となり、結果は 0 となります。

保守担当者向けメモ

  • 本関数は、幾何データ型の入力解析において、座標文字列が期待される形式（ペアの数）を満たしているかを検証するために使用されています。
  • 戻り値が -1 の場合、呼び出し側でエラーハンドリングが行われる前提となっています。
