pg_u_prop_join_control

定義

  • 種別: function
  • 定義箇所: src/common/unicode_category.c:191

目的

指定された Unicode コードポイントが「結合制御文字（Join Control）」プロパティを持つかどうかを判定します。

シグネチャまたは形状

bool pg_u_prop_join_control(pg_wchar code)

入力と出力

  • **入力**: pg_wchar code（判定対象の Unicode コードポイント）
  • **出力**: bool（プロパティに該当する場合は true、それ以外は false）

前提条件と不変条件

  • 入力されるコードポイントは pg_wchar 型であること。
  • Unicode 規格の安定性ポリシーに基づき、カテゴリおよびプロパティ値は固定されているものとして扱われます。

アルゴリズムまたは振る舞い

1. **ASCII 最適化**: code が 0x80 未満の場合、unicode_opt_ascii テーブルを参照し、PG_U_PROP_JOIN_CONTROL ビットマスクを用いて定数時間で判定します。 2. **テーブル検索**: code が 0x80 以上の場合、unicode_join_control テーブルに対して range_search を実行し、該当範囲に含まれるかを判定します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/common/unicode/category_test.c:75
  • src/common/unicode_category.c:239
  • **呼び出し先**:
  • range_search（src/common/unicode_category.c 内のユーティリティ）

関連データ構造

  • unicode_opt_ascii: ASCII 文字のプロパティを保持する静的テーブル。
  • unicode_join_control: 結合制御文字の範囲を定義する静的テーブル（unicode_category_table.h 由来）。

失敗モードと境界条件

  • 0x80 未満のコードポイントに対してはテーブル参照による高速な判定が行われます。
  • 0x80 以上のコードポイントに対しては、unicode_join_control テーブルの範囲外であれば false を返します。
