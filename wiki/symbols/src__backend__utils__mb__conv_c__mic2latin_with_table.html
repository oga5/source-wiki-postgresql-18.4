mic2latin_with_table

定義

  • 種別: function
  • 定義箇所: src/backend/utils/mb/conv.c:256

目的

Mule Internal Code (MIC) 文字列を、指定された変換テーブルを用いて特定のシングルバイト文字セット（Latin系など）へ変換します。

シグネチャまたは形状

int mic2latin_with_table(const unsigned char *mic,
                         unsigned char *p,
                         int len,
                         int lc,
                         int encoding,
                         const unsigned char *tab,
                         bool noError)

入力と出力

  • **入力**:
  • mic: 変換元のMIC文字列。
  • p: 変換後の文字列を格納するバッファ。
  • len: 入力文字列のバイト長。
  • lc: 変換対象のMule文字セット識別子。
  • encoding: 変換先のエンコーディング識別子。
  • tab: 変換用ルックアップテーブル。
  • noError: 変換失敗時にエラーを報告せず中断するかどうかを指定するフラグ。
  • **出力**:
  • 戻り値: 処理された入力バイト数。
  • p には変換後の文字列が格納され、末尾に \0 が付与されます。

前提条件と不変条件

  • 入力 mic は有効なMule Internal Codeである必要があります。
  • tab は mic[1] - HIGHBIT をインデックスとして使用するため、適切なサイズとオフセット調整がなされている必要があります。

アルゴリズムまたは振る舞い

1. 入力文字列をバイト単位で走査します。 2. ASCII文字（HIGHBIT がセットされていない）はそのまま出力バッファへコピーします。 3. マルチバイト文字の場合、pg_mule_mblen で長さを確認します。 4. 指定された文字セット lc と一致し、かつ変換テーブル tab に対応するエントリが存在する場合、変換後の値を出力します。 5. 変換不可または不正なエンコーディングが検出された場合、noError が false ならばエラーを報告し、true ならば処理を中断します。 6. 最後に p の末尾にヌル終端を書き込み、消費したバイト数を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/mb/conversion_procs/cyrillic_and_mic/cyrillic_and_mic.c
  • src/backend/utils/mb/conversion_procs/latin2_and_win1250/latin2_and_win1250.c
  • **呼び出し先**:
  • pg_mule_mblen
  • report_invalid_encoding
  • report_untranslatable_char

関連データ構造

  • pg_wchar.h で定義されるエンコーディング関連定数およびマクロ（PG_MULE_INTERNAL, IS_HIGHBIT_SET 等）。

失敗モードと境界条件

  • noError が true の場合、エラー報告を行わずに処理を中断し、その時点までの変換結果を返します。

保守担当者向けメモ

  • 本関数は変換テーブル tab に強く依存しています。新しい文字セットを追加する際は、テーブルのインデックス計算が mic[1] - HIGHBIT と整合しているか確認してください。
  • 戻り値は「消費した入力バイト数」であり、出力バッファの長さではない点に注意してください。
