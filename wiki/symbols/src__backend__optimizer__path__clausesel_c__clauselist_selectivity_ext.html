clauselist_selectivity_ext

定義

  • 種別: function
  • 定義箇所: src/backend/optimizer/path/clausesel.c:116

目的

クエリプランナがコスト見積もりのために使用する、AND で結合された条件リスト全体の選択率（Selectivity）を計算します。拡張統計の適用や、同一変数に対する範囲クエリの重なりを考慮した補正を行います。

シグネチャまたは形状

Selectivity clauselist_selectivity_ext(PlannerInfo *root,
                                       List *clauses,
                                       int varRelid,
                                       JoinType jointype,
                                       SpecialJoinInfo *sjinfo,
                                       bool use_extended_stats)

入力と出力

  • **入力**:
  • root: プランナのグローバル情報。
  • clauses: 評価対象の式リスト。
  • varRelid: 関連するリレーションID。
  • jointype, sjinfo: 結合の種類と情報。
  • use_extended_stats: 拡張統計の使用可否フラグ。
  • **出力**:
  • Selectivity (double): 計算された選択率（0.0〜1.0）。

前提条件と不変条件

  • clauses が単一要素の場合、直接 clause_selectivity_ext を呼び出します。
  • 拡張統計が適用可能な場合、statext_clauselist_selectivity が優先的に使用されます。
  • 範囲クエリの補正は、addRangeClause で収集された境界情報に基づいて行われます。

アルゴリズムまたは振る舞い

1. **単一条件の最適化**: リストが1要素であれば即座に clause_selectivity_ext を呼び出して終了します。 2. **拡張統計の適用**: 単一リレーションに対する条件であり、かつ拡張統計が利用可能な場合、statext_clauselist_selectivity を用いて可能な限り多くの条件を推定します。 3. **個別条件の推定**: 拡張統計でカバーされない条件に対し、clause_selectivity_ext を再帰的に呼び出します。 4. **範囲クエリの収集**: 擬似定数を含む比較演算子（<, <=, >, >=）を検出し、addRangeClause を通じて rqlist に登録します。 5. **範囲クエリの補正**: rqlist を走査し、上限・下限の重なりを考慮した選択率の補正（hibound + lobound - 1.0）および NULL 値の補正を行います。 6. **集約**: 範囲クエリ以外の条件は単純な積として計算されます。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/optimizer/path/clausesel.c (内部再帰呼び出し)
  • src/backend/statistics/dependencies.c
  • src/backend/statistics/extended_stats.c
  • **呼び出し先**:
  • clause_selectivity_ext
  • statext_clauselist_selectivity
  • addRangeClause
  • nulltestsel
  • find_single_rel_for_clauses

関連データ構造

  • RangeQueryClause: 範囲クエリの境界を保持する構造体。
  • RestrictInfo: 条件式とメタデータを保持する構造体。
  • PlannerInfo: プランナのコンテキスト情報。

失敗モードと境界条件

  • 範囲クエリの境界値がデフォルト値（DEFAULT_INEQ_SEL）の場合、DEFAULT_RANGE_INEQ_SEL を使用してフォールバックします。
  • 拡張統計が適用できない、または条件が範囲クエリの形式に合致しない場合は、単純な積（s1 * s2）として処理されます。

保守担当者向けメモ

  • 本関数は clauselist_selectivity の拡張版であり、統計情報の適用範囲を制御する use_extended_stats フラグを引数に持ちます。
  • 範囲クエリの補正ロジックは、nulltestsel を用いた NULL 値の二重除外処理を含みます。
