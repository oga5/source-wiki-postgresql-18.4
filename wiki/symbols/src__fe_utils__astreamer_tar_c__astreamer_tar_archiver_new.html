astreamer_tar_archiver_new

定義

  • 種別: function
  • 定義箇所: src/fe_utils/astreamer_tar.c:391

目的

tarアーカイブ処理を行う astreamer_tar_archiver インスタンスを生成し、後続のストリーマーと接続する。

シグネチャまたは形状

astreamer *astreamer_tar_archiver_new(astreamer *next)

入力と出力

  • **入力**: next (後続の処理を行う astreamer 構造体へのポインタ)
  • **出力**: 初期化された astreamer 構造体へのポインタ

前提条件と不変条件

  • next は有効な astreamer オブジェクトであること。
  • 戻り値の astreamer オブジェクトの操作関数（bbs_ops）は astreamer_tar_archiver_ops に設定される。

アルゴリズムまたは振る舞い

1. palloc0 を使用して astreamer_tar_archiver 構造体分のメモリをゼロクリアで確保する。 2. 確保した構造体の base.bbs_ops に astreamer_tar_archiver_ops を割り当てる。 3. 引数 next を構造体の base.bbs_next に格納し、パイプラインを構築する。 4. 構築した構造体の base メンバを返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/bin/pg_basebackup/pg_basebackup.c:1226
  • **呼び出し先**: palloc0 (メモリ確保)

関連データ構造

  • astreamer_tar_archiver: アーカイバーの状態を保持する構造体。
  • astreamer: ストリーマーの基本インターフェース。
  • astreamer_ops: ストリーマーの操作関数テーブル。

失敗モードと境界条件

  • メモリ不足時には palloc0 がエラーを発生させ、処理が中断される可能性がある。

保守担当者向けメモ

  • 本関数は astreamer パイプラインの一部として機能する。
  • astreamer_tar_archiver_ops の定義が変更された場合、本関数による初期化ロジックも整合性を確認する必要がある。
