IsTidEqualAnyClause

定義

  • 種別: function
  • 定義箇所: src/backend/optimizer/path/tidpath.c:172

目的

与えられた RestrictInfo が、ctid = ANY(ARRAY[...]) 形式のTID等価比較条件であるかを判定する。

シグネチャまたは形状

static bool IsTidEqualAnyClause(PlannerInfo *root, RestrictInfo *rinfo, RelOptInfo *rel)

入力と出力

  • **入力**:
  • root: プランナのグローバル情報。
  • rinfo: 評価対象の RestrictInfo。
  • rel: 対象となるリレーション情報。
  • **出力**:
  • bool: 条件が ctid に対する ANY 演算子による等価比較であり、かつ右辺が定数（または擬似定数）である場合に true を返す。

前提条件と不変条件

  • rinfo->clause は ScalarArrayOpExpr 型である必要がある。
  • 演算子は TIDEqualOperator である必要がある。
  • node->useOr が true である必要がある。
  • node->args は2つの要素を持つ必要がある。

アルゴリズムまたは振る舞い

1. rinfo->clause が ScalarArrayOpExpr かを確認する。 2. 演算子が TIDEqualOperator か、かつ useOr が有効かを確認する。 3. 第1引数（arg1）が対象リレーションの ctid 列であるかを IsCTIDVar で検証する。 4. 第2引数（arg2）が対象リレーションの変数を含まないこと、および揮発性関数を含まないことを確認する（擬似定数であることの検証）。 5. すべての条件を満たせば true を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/optimizer/path/tidpath.c 内の TidQualFromRestrictInfoList 等から参照されている。
  • **呼び出し先**:
  • IsA (マクロ)
  • IsCTIDVar
  • pull_varnos
  • contain_volatile_functions
  • bms_is_member

関連データ構造

  • RestrictInfo: 評価対象の制約条件。
  • ScalarArrayOpExpr: ANY 演算子を含む式ノード。
  • RelOptInfo: 対象リレーションのメタデータ。

失敗モードと境界条件

  • rinfo->clause が ScalarArrayOpExpr でない場合は false を返す。
  • 演算子が TIDEqualOperator でない場合や、useOr が false の場合は false を返す。
  • 第1引数が ctid 列でない場合や、第2引数が定数でない（変数や揮発性関数を含む）場合は false を返す。

保守担当者向けメモ

  • 本関数は TID スキャンパス生成における条件抽出の一部であり、ANY 句を用いた効率的なタプルアクセスをサポートするために使用される。
  • arg2 の評価において pull_varnos を使用しているため、サブクエリや他テーブルへの依存関係がある場合は正しく除外される。
