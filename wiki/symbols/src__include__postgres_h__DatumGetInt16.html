DatumGetInt16

定義

  • 種別: static inline 関数
  • 定義箇所: src/include/postgres.h:166

目的

PostgreSQL の汎用データ型である Datum を、C 言語の int16 型へ変換する。

シグネチャまたは形状

static inline int16 DatumGetInt16(Datum X)

入力と出力

  • **入力**: Datum X（変換対象のデータ）
  • **出力**: int16（Datum からキャストされた 16 ビット符号付き整数）

前提条件と不変条件

  • Datum 型は、プラットフォームに応じて sizeof(void *) と同じサイズ（64bit 環境では 8 バイト、32bit 環境では 4 バイト）であると仮定される。
  • 本関数は、Datum に格納されている値が int16 として解釈可能であることを前提とする。

アルゴリズムまたは振る舞い

入力された Datum 型の引数 X を、明示的に int16 型へキャストして返す。インライン関数として定義されており、関数呼び出しのオーバーヘッドを排除する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/btree_gist/btree_utils_num.c
  • contrib/pageinspect/btreefuncs.c
  • contrib/pageinspect/gistfuncs.c
  • src/backend/access/nbtree/nbtcompare.c
  • src/backend/access/spgist/spgtextproc.c
  • src/backend/commands/analyze.c
  • src/backend/optimizer/path/costsize.c
  • **呼び出し先**:
  • 情報なし（C 言語の組み込みキャスト演算子のみを使用）。

関連データ構造

  • Datum: PostgreSQL の汎用データ表現型。

失敗モードと境界条件

  • Datum に格納された値が int16 の範囲（-32,768 ～ 32,767）を超えている場合、キャストによる切り捨てが発生する可能性がある。本関数自体は範囲チェックを行わない。

保守担当者向けメモ

  • 本関数は postgres.h で定義される型変換関数群の一つである。
  • 移植性を維持するため、Datum からの変換には必ずこの関数を使用すること。直接的なキャストは避けること。
