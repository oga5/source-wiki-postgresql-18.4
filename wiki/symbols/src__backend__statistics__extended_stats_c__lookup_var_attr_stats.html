lookup_var_attr_stats

定義

  • 種別: function
  • 定義箇所: src/backend/statistics/extended_stats.c:689

目的

拡張統計情報の構築に必要な、指定された列および式に対応する VacAttrStats 構造体の配列を取得する。

シグネチャまたは形状

static VacAttrStats **lookup_var_attr_stats(Bitmapset *attrs, List *exprs, int nvacatts, VacAttrStats **vacatts)

入力と出力

  • **入力**:
  • attrs: 統計対象の列を示す Bitmapset。
  • exprs: 統計対象の式リスト。
  • nvacatts: vacatts 配列の要素数。
  • vacatts: 利用可能な VacAttrStats の配列。
  • **出力**:
  • 成功時: 取得した VacAttrStats へのポインタ配列。
  • 失敗時: NULL（必要な列の統計が見つからない、または式の解析に失敗した場合）。

前提条件と不変条件

  • vacatts 配列内の各要素は有効な tupattnum を保持している必要がある。
  • 式の解析には examine_attribute を使用する。
  • 戻り値の配列は呼び出し元で解放する必要がある。

アルゴリズムまたは振る舞い

1. attrs（列）と exprs（式）の合計数に基づき、VacAttrStats 配列をメモリ確保する。 2. attrs に含まれる各列番号に対し、vacatts 内から一致する tupattnum を検索する。見つからない場合は NULL を返して終了する。 3. exprs に含まれる各式に対し、examine_attribute を呼び出して統計情報を取得する。解析不能な場合は NULL を返して終了する。 4. 式の統計情報に対して、vacatts[0] から取得した tupDesc を設定する。 5. 収集した全ての統計情報ポインタを配列として返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/statistics/extended_stats.c 内の統計構築ロジック（BuildRelationExtStatistics 等から参照）。
  • **呼び出し先**: examine_attribute（式解析用）。

関連データ構造

  • VacAttrStats: 属性ごとの統計情報を保持する構造体。
  • Bitmapset: 列番号の集合。
  • List: 式のリスト。

失敗モードと境界条件

  • 指定された列の統計情報が vacatts 内に存在しない場合、NULL を返す。
  • 式が解析不能（examine_attribute が NULL を返す）な場合、NULL を返す。
  • メモリ確保失敗時は palloc によりエラーが発生する。

保守担当者向けメモ

  • 式の tupDesc 設定において、vacatts[0] を参照しているが、vacatts が空の場合の安全性についてはソースコード上で明示的なガードがないため注意が必要。
