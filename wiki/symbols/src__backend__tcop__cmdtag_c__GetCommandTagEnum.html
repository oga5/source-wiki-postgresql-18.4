GetCommandTagEnum

定義

  • 種別: function
  • 定義箇所: src/backend/tcop/cmdtag.c:82

目的

文字列形式のコマンド名から、対応する CommandTag 列挙型への変換を行う。

シグネチャまたは形状

CommandTag GetCommandTagEnum(const char *commandname)

入力と出力

  • **入力**: commandname (文字列へのポインタ)
  • **出力**: 変換後の CommandTag 列挙値。一致するものがない場合や入力が空の場合は CMDTAG_UNKNOWN を返す。

前提条件と不変条件

  • tag_behavior 配列は、cmdtaglist.h を通じて静的に定義されており、コマンド名でソートされている必要がある。
  • pg_strcasecmp を使用するため、大文字小文字を区別せずに比較を行う。

アルゴリズムまたは振る舞い

1. 入力 commandname が NULL または空文字列の場合、直ちに CMDTAG_UNKNOWN を返す。 2. 静的配列 tag_behavior に対してバイナリサーチを実行する。 3. pg_strcasecmp を用いて比較を行い、一致した場合はそのインデックスを CommandTag 型にキャストして返す。 4. 検索範囲内で一致が見つからない場合は CMDTAG_UNKNOWN を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/commands/event_trigger.c
  • src/backend/utils/cache/evtcache.c
  • **呼び出し先**:
  • pg_strcasecmp (文字列比較用)

関連データ構造

  • CommandTag: コマンドの種類を表す列挙型。
  • CommandTagBehavior: 各コマンドタグの属性を保持する構造体。
  • tag_behavior: 全コマンドタグの属性を格納する静的配列。

失敗モードと境界条件

  • 入力が NULL または空文字列の場合、安全に CMDTAG_UNKNOWN を返す。
  • 検索対象のコマンド名が tag_behavior に存在しない場合、CMDTAG_UNKNOWN を返す。

保守担当者向けメモ

  • 新しいコマンドを追加する場合は src/include/tcop/cmdtaglist.h を編集すること。本関数は自動的に更新された tag_behavior 配列を参照する。
  • バイナリサーチの前提として、tag_behavior 配列が名前順にソートされていることを維持する必要がある。
