StatsBuildData

定義

  • **種別**: struct
  • **定義箇所**: src/include/statistics/extended_stats_internal.h:61

目的

拡張統計情報（Extended Statistics）を構築する際、対象となるテーブルの複数列にわたるデータセットを保持するための中間データ構造です。

シグネチャまたは形状

typedef struct StatsBuildData
{
    int            numrows;      /* 行数 */
    int            nattnums;     /* 対象列数 */
    AttrNumber    *attnums;      /* 対象列の属性番号配列 */
    VacAttrStats **stats;        /* 各列の統計情報へのポインタ配列 */
    Datum        **values;       /* 各行・各列のデータ値（Datum） */
    bool         **nulls;        /* 各行・各列のNULLフラグ */
} StatsBuildData;

入力と出力

  • **入力**: 統計構築対象のテーブルデータ（行数、列属性、各列の統計情報、値、NULL状態）。
  • **出力**: 本構造体自体が、statext_*_build 関数群（ndistinct, dependencies, mcv の構築）への入力として機能します。

前提条件と不変条件

  • values および nulls は、numrows × nattnums の二次元配列として構成される必要があります。
  • attnums および stats は、nattnums の要素数を持つ配列である必要があります。

アルゴリズムまたは振る舞い

1. **収集**: make_build_data 関数により、対象テーブルから統計計算に必要なデータが本構造体に格納されます。 2. **構築**: statext_dependencies_build や build_sorted_items 等の関数が本構造体を参照し、相関関係やソート済みアイテムを生成します。 3. **計算**: dependency_degree 関数などが、本構造体内のデータを用いて統計的な計算を行います。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/statistics/extended_stats.c: make_build_data にてインスタンス化および初期化が行われます。
  • **呼び出し先**:
  • src/backend/statistics/dependencies.c: dependency_degree および statext_dependencies_build が本構造体を参照します。
  • src/backend/statistics/extended_stats.c: build_sorted_items が本構造体を参照します。

関連データ構造

  • VacAttrStats: 各列の統計情報を保持する構造体。
  • MultiSortSupport: 複数列のソート順序を管理するための構造体。

失敗モードと境界条件

  • numrows が 0 の場合、統計計算ロジックにおいて空のデータセットとして扱われます。
  • メモリ割り当て失敗時、make_build_data 経由でエラーが発生する可能性があります。

保守担当者向けメモ

  • 本構造体は統計構築の基盤となるため、フィールドの追加や変更を行う際は、src/backend/statistics/extended_stats.c 内のメモリ確保ロジック（sizeof(StatsBuildData) を使用している箇所）と整合性を確認してください。
  • values および nulls の二次元配列のメモリレイアウトは、統計計算のパフォーマンスに直結します。
