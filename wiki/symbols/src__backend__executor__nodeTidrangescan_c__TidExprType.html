TidExprType

定義

  • 種別: enum
  • 定義箇所: src/backend/executor/nodeTidrangescan.c:38

目的

TID範囲スキャンにおいて、クエリ内の比較演算子が ctid の下限（Lower Bound）を規定するものか、上限（Upper Bound）を規定するものかを識別するために使用されます。

シグネチャまたは形状

typedef enum
{
	TIDEXPR_UPPER_BOUND,
	TIDEXPR_LOWER_BOUND,
} TidExprType;

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: TIDEXPR_UPPER_BOUND または TIDEXPR_LOWER_BOUND のいずれかの値。

前提条件と不変条件

  • TidOpExpr 構造体内のフィールド exprtype として保持され、スキャン実行時の範囲評価に使用されます。

アルゴリズムまたは振る舞い

  • TIDEXPR_LOWER_BOUND: ctid に対する >= や > などの比較条件を処理する際に割り当てられます。
  • TIDEXPR_UPPER_BOUND: ctid に対する <= や < などの比較条件を処理する際に割り当てられます。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/executor/nodeTidrangescan.c 内の式解析ロジック（TidExprListCreate 等）から参照されます。
  • 呼び出し先: 情報なし。

関連データ構造

  • TidOpExpr: 本列挙型をメンバとして保持する構造体。

失敗モードと境界条件

  • 該当なし（列挙型のため）。

保守担当者向けメモ

  • 本列挙型は nodeTidrangescan.c 内部のロジックに特化しています。比較演算子の追加や変更を行う際は、この列挙型が適切に割り当てられているか確認してください。
