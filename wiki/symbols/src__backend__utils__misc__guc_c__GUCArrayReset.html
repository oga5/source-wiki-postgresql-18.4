GUCArrayReset

定義

  • 種別: function
  • 定義箇所: src/backend/utils/misc/guc.c:6641

目的

設定パラメータの配列から、現在のユーザーが変更権限を持たない項目のみを抽出し、新しい配列として再構築する。

シグネチャまたは形状

ArrayType *GUCArrayReset(ArrayType *array)

入力と出力

  • **入力**: ArrayType *array（設定パラメータを含むテキスト配列）
  • **出力**: ArrayType *（権限のない項目のみで構成された新しい配列、または全て削除された場合は NULL）

前提条件と不変条件

  • 入力配列が NULL の場合、関数は直ちに NULL を返す。
  • 呼び出し元がスーパーユーザーである場合、関数は常に NULL を返す（全項目削除）。
  • 配列内の各要素は name=value 形式の文字列であることを前提とする。

アルゴリズムまたは振る舞い

1. 入力が NULL であれば NULL を返す。 2. 呼び出し元がスーパーユーザーであれば NULL を返す。 3. 入力配列の各要素を走査する。

  • array_ref を使用して要素を取得し、TextDatumGetCString で文字列化する。
  • = を区切り文字としてパラメータ名を抽出する。
  • validate_option_array_item を呼び出し、現在のユーザーがそのパラメータを削除（リセット）する権限を持つか判定する。
  • 権限がある場合はその項目をスキップする。
  • 権限がない場合は、array_set または construct_array_builtin を使用して新しい配列に項目を追加する。

    • 4. 構築された新しい配列を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/catalog/pg_db_role_setting.c:73
  • **呼び出し先**:
  • superuser()
  • array_ref
  • TextDatumGetCString
  • validate_option_array_item
  • array_set
  • construct_array_builtin
  • pfree

関連データ構造

  • ArrayType (PostgreSQL の配列型)
  • TEXTOID

失敗モードと境界条件

  • 入力配列が空の場合、NULL が返される。
  • 全ての項目が削除可能な場合、結果は NULL となる。
  • メモリ不足時には palloc 等の内部関数経由でエラーが発生する可能性がある。

保守担当者向けメモ

  • 本関数は pg_db_role_setting の更新処理において、権限のない設定項目を保持しつつリセットを行うために使用される。
  • validate_option_array_item の挙動が本関数のフィルタリング結果を直接左右するため、権限チェックの仕様変更時には注意が必要である。
