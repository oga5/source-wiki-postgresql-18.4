ExplainPropertyText

定義

  • 種別: function
  • 定義箇所: src/backend/commands/explain_format.c:213

目的

EXPLAIN コマンドの出力において、指定されたラベルに関連付けられた文字列型のプロパティを出力バッファへ書き込む。

シグネチャまたは形状

void ExplainPropertyText(const char *qlabel, const char *value, ExplainState *es)

入力と出力

  • qlabel (入力): プロパティの名称（ラベル）。
  • value (入力): 出力する文字列値。
  • es (入出力): EXPLAIN コマンドの出力状態を保持する ExplainState 構造体。

前提条件と不変条件

  • es は初期化済みであること。
  • qlabel および value は有効な文字列ポインタであること。

アルゴリズムまたは振る舞い

内部的に ExplainProperty 関数を呼び出し、第2引数（数値型データ用）に NULL を、第4引数（数値フラグ）に false を渡すことで、文字列としてフォーマット処理を実行する。これにより、es->format に応じた適切なエスケープや引用符の付与が行われる。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/file_fdw/file_fdw.c
  • contrib/pg_overexplain/pg_overexplain.c
  • **呼び出し先**:
  • ExplainProperty (同一ファイル内)

関連データ構造

  • ExplainState: 出力形式や現在のネスト状態を管理する構造体。

失敗モードと境界条件

  • es が NULL の場合、セグメンテーション違反が発生する可能性がある。
  • value が NULL の場合の挙動は、内部の ExplainProperty の実装に依存する。

保守担当者向けメモ

  • 本関数は文字列出力のラッパーである。フォーマット固有の出力ロジックを変更する場合は、ExplainProperty を参照すること。
