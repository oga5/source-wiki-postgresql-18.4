r_adjective

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_KOI8_R_russian.c:416

目的

ロシア語のステミング処理において、単語末尾の形容詞的接尾辞を特定し、削除する。

シグネチャまたは形状

static int r_adjective(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z （Snowballステミング環境）
  • **出力**: int 型。成功時は 1、失敗またはマッチしない場合は 0、内部エラー発生時は負の値を返す。

前提条件と不変条件

  • z->c が単語の末尾を指していること。
  • find_among_b を使用するため、エンコーディング（KOI8-R）に基づいた適切な文字境界が維持されていること。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を z->ket に保存する。 2. 文字列の末尾から前方へ向かい、特定のビットマスクおよび a_1 配列（26個のパターン）を用いて形容詞接尾辞を検索する。 3. マッチした場合、その位置を z->bra に設定する。 4. slice_del(z) を呼び出し、該当する接尾辞を文字列から削除する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_KOI8_R_russian.c 内のメインステミングロジック（russian_KOI8_R_stem からの呼び出しフローに含まれる）。
  • **呼び出し先**: find_among_b（接尾辞検索）、slice_del（文字列削除）。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_1: 形容詞接尾辞パターンを格納した among 構造体の配列。

失敗モードと境界条件

  • z->c - 1 <= z->lb: 文字列の先頭に達した場合、検索を中止し 0 を返す。
  • slice_del が負の値を返した場合、そのエラーコードをそのまま伝播させる。
  • パターンマッチングに失敗した場合は 0 を返す。

  • a_1 配列の定義は同一ファイル内の上位行を参照すること。
