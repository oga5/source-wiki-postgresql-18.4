FdwModifyPrivateIndex

定義

  • 種別: enum
  • 定義箇所: contrib/postgres_fdw/postgres_fdw.c:101

目的

postgres_fdw において、外部テーブルの変更操作（INSERT/UPDATE/DELETE）時に PgFdwModifyState の fdw_private リストへ格納される各要素のインデックスを定義します。

シグネチャまたは形状

enum FdwModifyPrivateIndex {
    FdwModifyPrivateUpdateSql,      /* SQL文 (Stringノード) */
    FdwModifyPrivateTargetAttnums,  /* ターゲット属性番号のリスト (Integerノード) */
    FdwModifyPrivateLen,            /* VALUES句の末尾までの長さ (Integerノード) */
    FdwModifyPrivateHasReturning,   /* RETURNING句の有無 (Booleanノード) */
    FdwModifyPrivateRetrievedAttrs  /* RETURNINGで取得する属性番号のリスト (Integerノード) */
};

入力と出力

本シンボル自体は列挙型であり、入力や出力を持つ関数ではありません。各列挙子は、実行時に List 構造体の特定のインデックス位置を指し示すために使用されます。

前提条件と不変条件

  • PgFdwModifyState の fdw_private リストは、この列挙型の順序に従って構築される必要があります。
  • 各インデックスに対応するリスト要素は、コメントで指定された型（String, Integer, Boolean 等）のノードである必要があります。

アルゴリズムまたは振る舞い

外部テーブルの変更処理において、プランナが生成した情報をエグゼキュータへ受け渡すためのキーとして機能します。例えば、FdwModifyPrivateUpdateSql はリモートサーバーで実行すべき SQL 文字列を特定し、FdwModifyPrivateRetrievedAttrs は RETURNING 句で返される列を特定するために参照されます。

呼び出し元と呼び出し先

  • 呼び出し元: contrib/postgres_fdw/postgres_fdw.c:1906 付近にて、リスト内の要素がこの列挙型と一致している必要がある旨の言及があります。
  • 呼び出し先: 本シンボル自体はデータ構造の定義であり、関数ではないため呼び出し先はありません。

関連データ構造

  • PgFdwModifyState: この列挙型を使用して fdw_private フィールドのデータにアクセスします。

失敗モードと境界条件

  • リストのインデックスがこの列挙型の定義と一致しない場合、誤った型へのキャストやメモリ不正アクセスが発生する可能性があります。

保守担当者向けメモ

  • 新しいプライベート情報を PgFdwModifyState に追加する場合は、必ずこの列挙型の末尾に新しい要素を追加し、リスト構築および参照箇所を更新してください。
