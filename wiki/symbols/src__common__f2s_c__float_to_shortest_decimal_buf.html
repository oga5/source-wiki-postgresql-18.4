float_to_shortest_decimal_buf

定義

  • 種別: function
  • 定義箇所: src/common/f2s.c:779

目的

単精度浮動小数点数（float）を、その値を正確に表現する最短の10進数文字列に変換し、指定されたバッファに書き込みます。

シグネチャまたは形状

int float_to_shortest_decimal_buf(float f, char *result)

入力と出力

  • **入力**:
  • f: 変換対象の単精度浮動小数点数。
  • result: 変換後の文字列を格納するバッファ（呼び出し元で十分な領域を確保する必要がある）。
  • **出力**:
  • 戻り値: 書き込まれた文字列の長さ（終端文字を含まない）。

前提条件と不変条件

  • result バッファは、変換結果を格納するために十分なサイズ（FLOAT_SHORTEST_DECIMAL_LEN）が必要です。
  • Assert により、書き込まれた長さが FLOAT_SHORTEST_DECIMAL_LEN 未満であることを保証します。

アルゴリズムまたは振る舞い

1. float_to_shortest_decimal_bufn を呼び出し、Ryuアルゴリズムを用いて数値を10進数表現に変換し、バッファへ書き込みます。 2. 戻り値として得られたインデックス位置にヌル終端文字（\0）を書き込み、文字列を完成させます。 3. 最後に書き込んだ文字数（インデックス）を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/float.c:327
  • **呼び出し先**:
  • float_to_shortest_decimal_bufn (同ファイル内)

関連データ構造

  • FLOAT_SHORTEST_DECIMAL_LEN: バッファサイズの定数。

失敗モードと境界条件

  • バッファサイズが不足している場合、Assert によりデバッグビルドで停止します。
  • 特殊値（NaN, Infinity, 0）の処理は float_to_shortest_decimal_bufn の内部ロジックに依存します。

保守担当者向けメモ

  • 本関数は float_to_shortest_decimal_bufn のラッパーであり、文字列の終端処理のみを担います。
  • 変換ロジックの変更が必要な場合は float_to_shortest_decimal_bufn を参照してください。
