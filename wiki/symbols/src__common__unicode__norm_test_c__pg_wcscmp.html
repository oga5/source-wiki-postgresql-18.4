pg_wcscmp

定義

  • 種別: function
  • 定義箇所: src/common/unicode/norm_test.c:43

目的

pg_wchar 型の文字列を比較し、辞書順に基づいた比較結果を返す補助関数です。

シグネチャまたは形状

static int pg_wcscmp(const pg_wchar *s1, const pg_wchar *s2)

入力と出力

  • **入力**: 比較対象となる2つの pg_wchar 型文字列へのポインタ (s1, s2)。
  • **出力**:
  • s1 < s2 の場合: -1
  • s1 > s2 の場合: 1
  • s1 == s2 の場合: 0

前提条件と不変条件

  • 入力される文字列は pg_wchar 型のヌル終端文字列である必要があります。
  • 本関数は norm_test.c 内のテスト実行において、正規化結果と期待値の比較に使用されます。

アルゴリズムまたは振る舞い

1. 文字列の先頭から順に各要素を比較します。 2. *s1 が *s2 より小さい場合は -1 を返し、大きい場合は 1 を返します。 3. 両方の文字が等しく、かつ *s1 が 0（終端）であれば、文字列全体が一致したとみなして 0 を返します。 4. 一致し続ける限りポインタをインクリメントし、比較を継続します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/common/unicode/norm_test.c 内の main 関数にて、正規化処理の検証結果と期待値を比較するために呼び出されます。
  • **呼び出し先**: 外部関数は呼び出しません。

関連データ構造

  • pg_wchar: PostgreSQL で使用されるワイド文字型。

失敗モードと境界条件

  • 文字列がヌル終端されていない場合、メモリ境界外アクセスが発生する可能性があります。
  • 比較対象の文字列が NULL ポインタである場合の挙動は定義されていません。

保守担当者向けメモ

  • 本関数は norm_test.c 専用の補助関数であり、標準ライブラリの wcscmp と同様のロジックを pg_wchar 型に対して実装しています。
  • テストプログラムの実行効率に影響するため、ロジックを変更する際は比較処理の正確性を維持してください。
