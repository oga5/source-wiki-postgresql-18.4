oid_dist

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_oid.c:100

目的

2つの Oid 型の値の間の距離（絶対差分）を計算します。

シグネチャまたは形状

Datum oid_dist(PG_FUNCTION_ARGS)

入力と出力

  • 入力: PG_GETARG_OID(0) および PG_GETARG_OID(1) により、2つの Oid 値を受け取ります。
  • 出力: PG_RETURN_OID(res) により、2つの値の差の絶対値を Oid 型として返します。

前提条件と不変条件

  • 入力された2つの Oid 値は、数値として比較可能である必要があります。
  • 戻り値は常に非負の Oid 値となります。

アルゴリズムまたは振る舞い

1. 第1引数 a と第2引数 b を取得します。 2. a < b の場合、b - a を計算します。 3. それ以外（a >= b）の場合、a - b を計算します。 4. 計算結果を Oid 型として返します。

呼び出し元と呼び出し先

  • 呼び出し元: ソースコード上の直接的な呼び出し元は提供されていません。
  • 呼び出し先: PostgreSQLの関数管理インターフェースである PG_GETARG_OID および PG_RETURN_OID マクロを使用しています。

関連データ構造

  • Oid: PostgreSQLのオブジェクト識別子型。

失敗モードと境界条件

  • Oid 型の範囲を超えるような計算結果が発生した場合の挙動については、本ソースコード内では明示されていません。

保守担当者向けメモ

  • 本関数は Oid を数値として扱うためのユーティリティです。GiSTインデックスにおける距離計算のロジックとして利用されます。
