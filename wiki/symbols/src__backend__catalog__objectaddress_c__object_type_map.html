object_type_map

定義

  • 種別: struct
  • 定義箇所: src/backend/catalog/objectaddress.c:650

目的

オブジェクトタイプ名（文字列）と内部的な ObjectType 列挙型を関連付けるためのマッピング構造体です。

シグネチャまたは形状

static const struct object_type_map
{
    const char *tm_name;
    ObjectType  tm_type;
}

入力と出力

本構造体はデータ定義であり、関数ではないため直接的な入力・出力はありません。

前提条件と不変条件

  • tm_name はヌル終端文字列である必要があります。
  • tm_type は ObjectType 列挙型の有効な値である必要があります。

アルゴリズムまたは振る舞い

この構造体は、主に文字列形式のオブジェクトタイプ名を内部的な列挙型に変換するための静的配列の要素として使用されます。

呼び出し元と呼び出し先

  • 呼び出し元: 提供されたソース情報からは、本構造体を利用する具体的な関数や配列の参照先は特定できません。
  • 呼び出し先: なし（データ構造のため）。

関連データ構造

  • ObjectType: tm_type フィールドで使用される列挙型。

失敗モードと境界条件

  • 構造体自体に失敗モードはありませんが、この構造体を用いた検索処理において、一致する tm_name が見つからない場合のハンドリングが呼び出し側に求められます。

保守担当者向けメモ

  • 新しいオブジェクトタイプをシステムに追加する際は、この構造体を用いたマッピング配列にエントリを追加する必要があります。
  • 参照先となる配列の定義箇所については、ソースコードの文脈からさらなる調査が必要です。
