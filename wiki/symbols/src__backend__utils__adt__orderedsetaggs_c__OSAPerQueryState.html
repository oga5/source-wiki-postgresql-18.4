OSAPerQueryState

定義

  • 種別: struct
  • 定義箇所: src/backend/utils/adt/orderedsetaggs.c:49

目的

順序集合集約関数（Ordered-set aggregate functions）の実行において、クエリ全体で共有されるメタデータおよび実行状態を保持する。

シグネチャまたは形状

typedef struct OSAPerQueryState
{
    Aggref     *aggref;
    MemoryContext qcontext;
    ExprContext *econtext;
    bool        rescan_needed;

    /* タプル蓄積用フィールド */
    TupleDesc   tupdesc;
    TupleTableSlot *tupslot;
    int         numSortCols;
    AttrNumber *sortColIdx;
    Oid        *sortOperators;
    Oid        *eqOperators;
    Oid        *sortCollations;
    bool       *sortNullsFirsts;
    ExprState  *compareTuple;

    /* Datum蓄積用フィールド */
    Oid         sortColType;
    int16       typLen;
    bool        typByVal;
    char        typAlign;
    Oid         sortOperator;
    Oid         eqOperator;
    Oid         sortCollation;
    bool        sortNullsFirst;
    FmgrInfo    equalfn;
} OSAPerQueryState;

入力と出力

  • 入力: ordered_set_startup 等の関数内で palloc0 を用いてメモリ確保され、fn_extra にキャッシュされる。
  • 出力: 集約関数の実行過程で、ソート順序、型情報、および式評価コンテキストとして参照される。

前提条件と不変条件

  • qcontext は本構造体およびクエリ関連データを含むメモリコンテキストであること。
  • タプル蓄積用フィールドと Datum 蓄積用フィールドは、集約対象のデータ形式に応じて使い分けられる。
  • compareTuple および equalfn は必要に応じて遅延生成される。

アルゴリズムまたは振る舞い

1. **初期化**: ordered_set_startup により、集約関数のメタデータ（Aggref）や評価コンテキストが設定される。 2. **状態保持**: 実行中の集約において、ソートに必要な演算子情報や型情報を保持し、tuplesort 処理の準備を支援する。 3. **リソース管理**: クエリのライフサイクルに紐付いており、集約終了時には関連するリソースが適切に解放される必要がある。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/utils/adt/orderedsetaggs.c 内の ordered_set_startup（初期化）、ordered_set_transition 等（状態参照）。
  • 呼び出し先: 構造体内部で ExprState や FmgrInfo を保持し、式評価や比較演算を実行する。

関連データ構造

  • OSAPerGroupState: グループ単位の実行状態を保持する構造体。
  • Aggref: 集約関数のノード情報。
  • TupleTableSlot: ソート対象のタプル保持に使用。
  • tuplesort: 本構造体の情報を利用してソートを実行するエンジン。

失敗モードと境界条件

  • メモリ不足: palloc0 失敗時にエラーが発生する可能性がある。
  • 型情報の不整合: sortColType 等の定義と実際の入力データが一致しない場合、ソートや比較処理で予期せぬ動作を引き起こす可能性がある。

保守担当者向けメモ

  • 本構造体はクエリのライフサイクルに依存する。メモリリークを防ぐため、AggRegisterCallback を介したクリーンアップが必須である。
  • AGGKIND_HYPOTHETICAL の場合、仮説的な行を区別するための追加情報がソート処理に影響を与える点に留意すること。
