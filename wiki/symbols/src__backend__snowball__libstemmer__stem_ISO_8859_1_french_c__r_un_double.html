目的

単語末尾の二重子音を解消（削除）する。フランス語のステミング処理における後処理の一部として機能する。

シグネチャまたは形状

static int r_un_double(struct SN_env * z)

入力と出力

  • 入力: SN_env 構造体（ステミング環境の状態を保持）。
  • 出力: 処理が成功した場合は 1、条件を満たさない場合やエラー発生時は 0 または負の値を返す。

前提条件と不変条件

  • z->c（現在のカーソル位置）が有効な範囲内であること。
  • a_8 テーブルに定義された特定の二重子音パターンが単語末尾に存在すること。

アルゴリズムまたは振る舞い

1. **パターンマッチング**: 現在のカーソル位置から後方へ向かって find_among_b を実行し、a_8 テーブル内のパターンと照合する。 2. **条件判定**:

  • 単語の境界条件（z->lb）および文字コードのビット演算による検証を行う。
  • マッチした場合は、その位置を ket（削除開始位置）に設定する。

    • 3. **削除処理**:

  • カーソルを1つ戻し、その位置を bra（削除終了位置）に設定する。
  • slice_del(z) を呼び出し、該当する文字を削除する。

呼び出し元と呼び出し先

  • 呼び出し先: find_among_b（パターン検索）、slice_del（文字列削除）。

関連データ構造

  • struct SN_env: ステミング環境を管理する構造体。
  • a_8: 二重子音パターンを保持する among 構造体の配列。

失敗モードと境界条件

  • z->c - 2 <= z->lb: カーソル位置が不十分な場合、処理を中断し 0 を返す。
  • slice_del が負の値を返した場合、その値をそのまま伝播させる。
