pq_block_sigpipe

定義

  • 種別: function
  • 定義箇所: src/interfaces/libpq-oauth/oauth-utils.c:171

目的

ネットワーク通信中に発生する SIGPIPE シグナルを一時的にブロックし、シグナルマスクを保存することで、予期せぬプロセス終了を防ぎます。

シグネチャまたは形状

int pq_block_sigpipe(sigset_t *osigset, bool *sigpipe_pending);

入力と出力

  • **入力**:
  • osigset: 現在のシグナルマスクを保存するための sigset_t へのポインタ。
  • sigpipe_pending: SIGPIPE が既に保留中であるかを格納するための bool へのポインタ。
  • **出力**:
  • 成功時は 0 を返し、失敗時は -1 を返します。

前提条件と不変条件

  • pthread_sigmask が利用可能であること。
  • osigset および sigpipe_pending は有効なメモリ領域を指していること。

アルゴリズムまたは振る舞い

1. SIGPIPE を含むシグナルセットを作成します。 2. pthread_sigmask を使用して SIGPIPE をブロックし、元のシグナルマスクを osigset に保存します。 3. pthread_sigmask が失敗した場合は -1 を返します。 4. 元のマスクで既に SIGPIPE がブロックされていた場合、sigpending を呼び出して実際に保留中の SIGPIPE が存在するかを確認し、その結果を sigpipe_pending に書き込みます。 5. それ以外の場合、sigpipe_pending を false に設定します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/interfaces/libpq-oauth/oauth-curl.c
  • src/interfaces/libpq/fe-print.c
  • src/interfaces/libpq/fe-secure.c
  • **呼び出し先**:
  • sigemptyset, sigaddset, pthread_sigmask, sigismember, sigpending（いずれも標準 C ライブラリ/POSIX 関数）。

関連データ構造

  • sigset_t: シグナルセットを保持する型。

失敗モードと境界条件

  • pthread_sigmask がエラーを返した場合、関数は -1 を返します。
  • sigpending が失敗した場合、関数は -1 を返します。

保守担当者向けメモ

  • 本関数は SIGPIPE のブロック状態を管理するため、必ず対となる pq_reset_sigpipe を呼び出してシグナルマスクを復元してください。
  • SOCK_ERRNO マクロを使用してエラー状態を判定しています。
