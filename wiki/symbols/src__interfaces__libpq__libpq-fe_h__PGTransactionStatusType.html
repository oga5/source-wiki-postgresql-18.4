PGTransactionStatusType

定義

  • 種別: enum
  • 定義箇所: src/interfaces/libpq/libpq-fe.h:145

目的

PostgreSQL データベース接続における現在のトランザクション状態を表現するために使用されます。

シグネチャまたは形状

typedef enum
{
    PQTRANS_IDLE,      /* 接続はアイドル状態 */
    PQTRANS_ACTIVE,    /* コマンド実行中 */
    PQTRANS_INTRANS,   /* アイドル状態、トランザクションブロック内 */
    PQTRANS_INERROR,   /* アイドル状態、失敗したトランザクション内 */
    PQTRANS_UNKNOWN    /* 状態判定不能 */
} PGTransactionStatusType;

入力と出力

  • 本シンボルは列挙型であり、関数ではありません。
  • PQtransactionStatus(const PGconn *conn) 関数の戻り値として使用されます。

前提条件と不変条件

  • libpq 内部の xactStatus メンバは PQTRANS_ACTIVE に変化しないという不変条件が存在します（src/interfaces/libpq/libpq-int.h による）。
  • 接続状態が正常でない場合、PQTRANS_UNKNOWN が返される可能性があります。

アルゴリズムまたは振る舞い

  • PQTRANS_IDLE: トランザクション外で、かつコマンドを実行していない状態。
  • PQTRANS_ACTIVE: クライアントからサーバーへコマンドが送信され、処理中である状態。
  • PQTRANS_INTRANS: BEGIN コマンド発行後、トランザクションブロック内で待機している状態。
  • PQTRANS_INERROR: トランザクション内でエラーが発生し、ROLLBACK 等の処理が必要な状態。

関連データ構造

  • PGconn: 接続状態を保持する不透明構造体。
  • libpq-int.h 内の xactStatus メンバ。

失敗モードと境界条件

  • 接続が切断されている、または通信エラーが発生している場合、PQTRANS_UNKNOWN が返されます。
