目的

ハンガリー語のステミング処理において、特定の接尾辞パターン（a_5 で定義されるもの）を単語末尾から検出し、条件を満たす場合に置換を行う。

シグネチャまたは形状

static int r_case_special(struct SN_env * z);

入力と出力

  • **入力**: struct SN_env * z（ステミング環境の状態を保持する構造体）
  • **出力**: int 型。処理が成功した場合は 1、マッチしなかった場合やエラー時は 0 または負の値を返す。

前提条件と不変条件

  • z->c（現在のカーソル位置）が有効な範囲にあること。
  • r_R1 関数による領域判定が、置換実行の前提条件として必須である。

アルゴリズムまたは振る舞い

1. **末尾チェック**: 単語末尾の文字が 'n' (110) または 't' (116) であることを確認する。 2. **パターンマッチング**: find_among_b を使用し、末尾から a_5 配列に含まれる接尾辞を検索する。 3. **領域判定**: マッチした場合、r_R1 を呼び出し、対象箇所が R1 領域内にあるか検証する。 4. **置換**: among_var の値（1 または 2）に応じて、slice_from_s を用いて接尾辞を特定の文字列（s_2 または s_3）に置換する。

関連データ構造

  • struct SN_env: ステミング環境を管理する構造体。
  • a_5: among 構造体の配列（検索対象の接尾辞パターン）。

失敗モードと境界条件

  • 単語末尾が 'n' または 't' でない場合、即座に 0 を返す。
  • find_among_b でマッチしない場合、0 を返す。
  • r_R1 が失敗（戻り値が 0 以下）した場合、その値をそのまま返す。
  • slice_from_s がメモリ不足等で失敗した場合、負の値を返す。
