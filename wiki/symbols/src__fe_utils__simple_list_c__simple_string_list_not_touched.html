simple_string_list_not_touched

定義

  • 種別: function
  • 定義箇所: src/fe_utils/simple_list.c:143

目的

SimpleStringList 内で、一度も検索（touched）されていない最初の文字列を特定して返します。

シグネチャまたは形状

const char *simple_string_list_not_touched(SimpleStringList *list);

入力と出力

  • **入力**: SimpleStringList *list（走査対象の文字列リストへのポインタ）
  • **出力**: const char *（未検索の文字列へのポインタ。存在しない場合は NULL）

前提条件と不変条件

  • list は有効な SimpleStringList 構造体を指している必要があります。
  • リスト内の各セルは touched フラグを保持しており、simple_string_list_member 等の検索関数によってこのフラグが更新されることを前提としています。

アルゴリズムまたは振る舞い

1. リストの head から順に各セルを走査します。 2. 各セルにおいて touched フラグを確認します。 3. touched が false である最初のセルを見つけた場合、そのセルの保持する文字列（val）を即座に返します。 4. リストの末尾まで走査しても touched が false のセルが見つからない場合、NULL を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/bin/pg_dump/pg_backup_archiver.c 内の複数箇所（schemaNames, tableNames, indexNames, functionNames, triggerNames の各リストに対して呼び出されています）。
  • **呼び出し先**: 内部的な関数呼び出しはありません。

関連データ構造

  • SimpleStringList: 文字列を保持する単方向連結リストの管理構造体。
  • SimpleStringListCell: リストの各要素を構成する構造体。touched フラグと文字列値 val を保持します。

失敗モードと境界条件

  • リストが空の場合、ループは実行されず NULL を返します。
  • すべての要素が既に検索済み（touched == true）の場合、NULL を返します。

保守担当者向けメモ

  • 本関数は $O(N)$ の線形探索を行います。要素数が多いリストに対して頻繁に呼び出すとパフォーマンスに影響する可能性があります。
  • touched フラグは simple_string_list_member 等の他の関数によって管理されるため、本関数単体では状態を変更しません。
