pg_get_utf8_id

定義

  • 種別: function
  • 定義箇所: src/fe_utils/mbprint.c:33

目的

UTF-8 エンコーディングに対応する内部 ID を取得し、キャッシュします。

シグネチャまたは形状

static int pg_get_utf8_id(void)

入力と出力

  • 入力: なし
  • 出力: int 型のエンコーディング ID。

前提条件と不変条件

  • 初回呼び出し時に pg_char_to_encoding("utf8") を実行し、結果を静的変数 utf8_id に保持します。
  • 2回目以降の呼び出しでは、キャッシュされた値を返します。

アルゴリズムまたは振る舞い

1. 静的変数 utf8_id が初期値（-1）であるかを確認します。 2. 初期値であれば、pg_char_to_encoding("utf8") を呼び出し、その結果を utf8_id に代入します。 3. utf8_id の値を返します。

呼び出し元と呼び出し先

  • 呼び出し元: src/fe_utils/mbprint.c:43 にて PG_UTF8 マクロとして定義されています。
  • 呼び出し先: pg_char_to_encoding（外部関数）。

関連データ構造

  • なし（内部的な静的変数 utf8_id を使用）。

失敗モードと境界条件

  • pg_char_to_encoding が失敗した場合の挙動は、当該関数の仕様に依存します。

保守担当者向けメモ

  • 本関数は PG_UTF8 マクロを通じて利用されることが想定されています。
  • pg_wchar.h を直接インクルードしない設計方針により、エンコーディング ID の不整合を回避しています。
