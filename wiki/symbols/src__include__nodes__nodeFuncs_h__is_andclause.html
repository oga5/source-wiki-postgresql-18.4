is_andclause

定義

  • 種別: static inline function
  • 定義箇所: src/include/nodes/nodeFuncs.h:108

目的

与えられたノードが AND 演算を表す BoolExpr であるかを判定します。

シグネチャまたは形状

bool is_andclause(const void *clause)

入力と出力

  • **入力**: clause (任意のノードへのポインタ)
  • **出力**: true (ノードが AND 演算子を持つ BoolExpr の場合)、それ以外は false

前提条件と不変条件

  • clause が NULL の場合、安全に false を返します。
  • clause が指すメモリ領域は、少なくとも Node 構造体として有効である必要があります。

アルゴリズムまたは振る舞い

1. clause が NULL でないことを確認します。 2. IsA マクロを使用して、ノードの型が T_BoolExpr であるかを確認します。 3. ノードを BoolExpr 構造体にキャストし、boolop フィールドが AND_EXPR と一致するかを評価します。

関連データ構造

  • BoolExpr: AND, OR, NOT 演算子を保持するノード構造体。
  • Node: PostgreSQLのツリー構造の基底型。

失敗モードと境界条件

  • clause が NULL の場合は false を返します。
  • BoolExpr 以外のノード型が渡された場合、IsA マクロにより false が返されます。
