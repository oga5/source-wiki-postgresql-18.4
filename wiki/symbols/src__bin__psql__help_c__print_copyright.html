print_copyright

定義

  • 種別: function
  • 定義箇所: src/bin/psql/help.c:754

目的

PostgreSQLの著作権情報および免責事項を標準出力に表示する。

シグネチャまたは形状

void print_copyright(void)

入力と出力

  • 入力: なし
  • 出力: なし（標準出力へ著作権テキストを書き出す）

前提条件と不変条件

  • 呼び出し時に標準出力が利用可能であること。

アルゴリズムまたは振る舞い

1. PostgreSQLの名称および著作権保持者（PostgreSQL Global Development Groupおよびカリフォルニア大学）に関する情報を出力する。 2. ソフトウェアの使用、複製、改変、配布に関する許諾条件を表示する。 3. カリフォルニア大学による損害賠償責任の免責および保証の否認に関する法的文言を表示する。

呼び出し元と呼び出し先

  • 呼び出し元: src/bin/psql/command.c:988
  • 呼び出し先: 標準ライブラリ関数 puts

関連データ構造

  • なし

失敗モードと境界条件

  • 標準出力への書き込みが失敗した場合の挙動は puts に依存する。

保守担当者向けメモ

  • 本関数で表示される著作権の年号や免責事項の文言を変更する場合は、法務上の要件を確認すること。
  • 本関数は src/bin/psql/help.h で宣言されている。
