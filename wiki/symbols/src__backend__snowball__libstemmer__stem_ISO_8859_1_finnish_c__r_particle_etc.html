r_particle_etc

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_finnish.c:305

目的

フィンランド語のステミング処理において、単語末尾の接辞（particle）や特定の語尾を識別し、条件に応じて削除する。

シグネチャまたは形状

static int r_particle_etc(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング環境の状態を保持する構造体）
  • 出力: 処理が成功した場合は 1、失敗または対象外の場合は 0、エラー発生時は負の値を返す。

前提条件と不変条件

  • z->I[1] に格納された境界位置以降の文字列を対象とする。
  • find_among_b を使用して、後方から a_0 配列内のパターンを照合する。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置が z->I[1] より前であれば終了する。 2. a_0 配列を用いて単語末尾のパターンを検索する。 3. マッチしたパターンに応じて分岐する:

  • case 1: g_particle_end グループに含まれる文字が直前にある場合のみ削除対象とする。
  • case 2: r_R2 関数を呼び出し、R2領域内であるという条件を満たす場合のみ削除対象とする。

    • 4. 条件を満たした場合、slice_del を実行して該当部分を削除する。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_ISO_8859_1_finnish.c および stem_UTF_8_finnish.c 内のメインステミング処理フローから呼び出される。
  • 呼び出し先: find_among_b（パターン照合）、r_R2（領域判定）、slice_del（文字列削除）、in_grouping_b（文字グループ判定）。

関連データ構造

  • SN_env: ステミング環境を管理する構造体。
  • a_0: 照合対象となる among 構造体の配列。
  • g_particle_end: case 1 の判定に使用される文字グループ。

失敗モードと境界条件

  • z->c < z->I[1] の場合、処理を行わず 0 を返す。
  • find_among_b でパターンが見つからない場合、0 を返す。
  • slice_del が失敗（負の値を返却）した場合、その値をそのまま伝播させる。
