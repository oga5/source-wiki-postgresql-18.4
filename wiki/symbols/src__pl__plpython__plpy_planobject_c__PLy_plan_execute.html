PLy_plan_execute

定義

  • 種別: function
  • 定義箇所: src/pl/plpython/plpy_planobject.c:128

目的

Python側から呼び出されたプリペアドステートメント（PLyPlan）を実行し、その結果をPythonオブジェクトとして返します。

シグネチャまたは形状

static PyObject *PLy_plan_execute(PyObject *self, PyObject *args)
  • self: PLyPlanObject インスタンス。
  • args: Python側から渡される引数タプル（オプションの引数リストと実行制限数を含む）。

入力と出力

  • **入力**:
  • args: PyArg_ParseTuple を介して解析される引数。|Ol 形式により、オプションの引数リスト（list）と実行制限数（limit）を受け取ります。
  • **出力**:
  • 成功時: PLy_spi_execute_plan の実行結果である PyObject*。
  • 失敗時: NULL（引数の解析失敗時、または内部実行エラー時）。

前提条件と不変条件

  • self は PLyPlanObject 型の有効なインスタンスである必要があります。
  • PyArg_ParseTuple による引数解析が成功することが実行の前提となります。

アルゴリズムまたは振る舞い

1. PyArg_ParseTuple を使用して、Pythonから渡された引数（引数リストと実行制限数）を解析します。 2. 解析した引数をそのまま PLy_spi_execute_plan 関数へ渡し、実際のSPI実行処理を委譲します。 3. PLy_spi_execute_plan の戻り値を呼び出し元へ返します。

呼び出し元と呼び出し先

  • **呼び出し元**: Pythonインタープリタ（メソッドテーブル {"execute", PLy_plan_execute, METH_VARARGS, NULL} を介して登録）。
  • **呼び出し先**: PLy_spi_execute_plan（src/pl/plpython/plpy_spi.c 内の関数と推測されるが、本ソース内での直接的な定義は未確認）。

関連データ構造

  • PLyPlanObject: 実行対象の計画を保持する構造体。
  • PyObject: Pythonの汎用オブジェクト型。

失敗モードと境界条件

  • 引数の型が期待するものと異なる場合、PyArg_ParseTuple が失敗し NULL を返します。
  • PLy_spi_execute_plan 内部でエラーが発生した場合、そのエラーが伝播し NULL が返されます。

保守担当者向けメモ

  • 本関数は単なるラッパーであり、実処理は PLy_spi_execute_plan に依存しています。
  • limit パラメータは long 型として解析されており、SPI実行時の行数制限として機能します。
