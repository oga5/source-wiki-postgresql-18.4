GetRealCmin

定義

  • 種別: function
  • 定義箇所: src/backend/utils/time/combocid.c:278

目的

Combo CID（コマンドIDのペアを抽象化したID）から、対応する元の cmin 値を復元する。

シグネチャまたは形状

static CommandId GetRealCmin(CommandId combocid)

入力と出力

  • **入力**: combocid (CommandId) - 変換対象のCombo CID。
  • **出力**: CommandId - 復元された cmin 値。

前提条件と不変条件

  • combocid は、現在有効な usedComboCids（現在割り当て済みのCombo CID数）より小さい値である必要がある。この条件は Assert によって検証される。

アルゴリズムまたは振る舞い

1. combocid が有効範囲内にあることを Assert で確認する。 2. comboCids 配列の combocid インデックスに格納されている cmin 値を直接参照し、その値を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/time/combocid.c:112（HeapTupleHeaderGetCmin 内での参照が確認されている）
  • **呼び出し先**:
  • なし（配列アクセスのみ）

関連データ構造

  • comboCids: Combo CIDをインデックスとして cmin/cmax のペアを保持する配列。

失敗モードと境界条件

  • combocid >= usedComboCids の場合、アサーション失敗（デバッグビルド時）またはメモリ領域外アクセスのリスクがある。

保守担当者向けメモ

  • 本関数は static であり、モジュール内でのみ使用される。
  • comboCids 配列の整合性は、HeapTupleHeaderAdjustCmax 等のCombo CID生成ロジックによって維持されている。
