config_group

定義

  • 種別: enum
  • 定義箇所: src/include/utils/guc_tables.h:55

目的

PostgreSQLの「Grand Unified Configuration (GUC)」システムにおいて、設定変数を機能単位で分類するための列挙型です。

シグネチャまたは形状

enum config_group
{
    UNGROUPED,
    FILE_LOCATIONS,
    CONN_AUTH_SETTINGS,
    /* ... (中略) ... */
    DEVELOPER_OPTIONS,
};

入力と出力

  • 入力: なし（定数定義）
  • 出力: 設定変数のグループ識別子

前提条件と不変条件

  • 各設定変数は、config_generic 構造体の group フィールドを通じて、この列挙型のいずれかの値に割り当てられます。
  • UNGROUPED は pg_settings ビューに表示されないオプションに使用されます。

アルゴリズムまたは振る舞い

  • GUCシステム内で設定変数を論理的に整理し、管理や表示を行う際の分類キーとして機能します。

呼び出し元と呼び出し先

  • 呼び出し元（参照元）:
  • src/include/utils/guc_tables.h:176: config_generic 構造体内のメンバとして使用。
  • src/backend/utils/misc/guc_tables.c:697: 各グループに対応する表示用名称の定義において使用。
  • 呼び出し先: なし（本シンボルはデータ定義であり、関数を呼び出しません）。

関連データ構造

  • config_generic: GUC変数の基底構造体。この構造体が config_group を保持します。

失敗モードと境界条件

  • 該当なし（列挙型定義のため）。

保守担当者向けメモ

  • 新しい設定カテゴリを追加する場合は、本列挙型に値を追加し、対応する表示用名称を src/backend/utils/misc/guc_tables.c に追加する必要があります。
