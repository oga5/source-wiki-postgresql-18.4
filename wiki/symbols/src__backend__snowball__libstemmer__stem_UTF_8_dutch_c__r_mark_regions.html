r_mark_regions

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_dutch.c:249

目的

オランダ語のステミング処理において、語幹抽出の基準となる単語内の領域（R1, R2）を計算し、Snowball 環境の内部インデックスに格納する。

シグネチャまたは形状

static int r_mark_regions(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z（Snowball ステミング環境）
  • **出力**: 成功時に 1、失敗時に 0 を返す。
  • **副作用**: 環境 z 内の整数配列 z->I[0]（R1の開始位置の候補）、z->I[1]（R2の開始位置）、z->I[2]（R1の開始位置）を更新する。

前提条件と不変条件

  • z が有効な SN_env 構造体を指していること。
  • g_v（母音グループ）が定義されていること。
  • 処理対象の単語が z->p に格納されていること。

アルゴリズムまたは振る舞い

1. z->I[2]（R1）と z->I[1]（R2）を単語の末尾位置で初期化する。 2. 単語の先頭から3文字目以降の位置を z->I[0] に記録する。 3. 母音（g_v）の直後の子音位置を検索し、その位置を z->I[2]（R1）として設定する。 4. R1 が z->I[0] より小さい場合は、R1 を z->I[0] に補正する。 5. さらにその後の母音の直後の子音位置を検索し、その位置を z->I[1]（R2）として設定する。

呼び出し元と呼び出し先

  • **呼び出し元**: dutch_UTF_8_stem（直接の呼び出し関係はソースコードの制御フローから推論されるが、本抜粋には明示的な呼び出し元コードは含まれていない）。
  • **呼び出し先**: skip_utf8, out_grouping_U, in_grouping_U（Snowball ランタイム関数）。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • g_v: 母音を定義するグループテーブル。

失敗モードと境界条件

  • skip_utf8 やグループ検索関数が負の値を返した場合、処理を中断し 0 を返す。
  • 単語が短く、R1 や R2 を定義するための母音・子音の並びが見つからない場合、初期値のまま処理が終了する。

保守担当者向けメモ

  • z->I 配列のインデックス（0, 1, 2）は Snowball の内部仕様に依存しているため、変更時は注意が必要。
