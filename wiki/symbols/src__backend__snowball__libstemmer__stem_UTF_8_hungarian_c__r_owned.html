目的

ハンガリー語のステミング処理において、単語の末尾から所有格に関連する接尾辞を特定し、削除または置換を行う。

static int r_owned(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z （ステミング実行環境）
  • **出力**: int （処理が成功した場合は 1、適用されなかった場合や失敗した場合は 0 または負の値を返す）

前提条件と不変条件

  • z->c が単語の末尾を指していること。
  • z->p に格納された文字列の末尾が特定の文字（UTF-8で i または í に相当するバイト値）であること。
  • r_R1 関数による領域チェックが成功すること。

アルゴリズムまたは振る舞い

1. 現在位置 z->c を z->ket に設定する。 2. 文字列末尾が i (105) または í (169) であるかを確認し、一致しなければ終了する。 3. find_among_b を使用し、接尾辞リスト a_9（12要素）に対して後方マッチングを行う。 4. マッチした場合、z->bra を現在位置に設定し、r_R1 を呼び出して対象が R1 領域内にあるか検証する。 5. マッチした接尾辞のインデックス（among_var）に基づき、以下のいずれかを行う：

  • 1: slice_del で接尾辞を削除。
  • 2: slice_from_s で接尾辞を s_8 に置換。
  • 3: slice_from_s で接尾辞を s_9 に置換。

    • 6. 成功時に 1 を返し、失敗時に 0 またはエラーコードを返す。

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_UTF_8_hungarian.c および src/backend/snowball/libstemmer/stem_ISO_8859_2_hungarian.c 内のメイン処理フローから呼び出される。

失敗モードと境界条件

  • z->c - 1 <= z->lb: 文字列の境界に達している場合は処理を中断する。
  • find_among_b が一致しない場合、処理を中断する。
  • r_R1 が負の値を返した場合、その値をそのまま呼び出し元へ伝播させる。
