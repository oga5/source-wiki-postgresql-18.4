PQmblenBounded

定義

  • 種別: function
  • 定義箇所: src/interfaces/libpq/fe-misc.c:1265

目的

指定されたエンコーディングにおいて、マルチバイト文字列の先頭1文字の長さを、文字列の終端を考慮して安全に取得します。

シグネチャまたは形状

int PQmblenBounded(const char *s, int encoding)

入力と出力

  • **入力**:
  • s: 検査対象の文字列へのポインタ。
  • encoding: 文字列のエンコーディング識別子。
  • **出力**:
  • 文字列の先頭1文字のバイト数。文字列が終端に達している場合は、その時点までの長さが返されます。

前提条件と不変条件

  • encoding は pg_encoding_mblen がサポートする有効な値である必要があります。
  • s はヌル終端された文字列である必要があります。

アルゴリズムまたは振る舞い

1. pg_encoding_mblen(encoding, s) を呼び出し、現在のエンコーディング規則に基づいて先頭文字のバイト数を算出します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/bin/psql/common.c 内の複数の箇所（クエリ文字列の解析や単語単位の処理において頻繁に使用）。
  • **呼び出し先**: pg_encoding_mblen（mb/pg_wchar.h 関連の関数と推測される）、strnlen。

関連データ構造

  • 特になし。

失敗モードと境界条件

  • s が空文字列の場合、strnlen の挙動により 0 が返されます。
  • 不正なマルチバイトシーケンスが含まれる場合、pg_encoding_mblen の実装に依存した挙動となります。

保守担当者向けメモ

  • 本関数は psql クライアント側での文字列操作において、マルチバイト文字を壊さずに走査するために多用されています。修正を行う際は、pg_encoding_mblen の仕様変更が本関数の戻り値に与える影響を考慮してください。
