scram_H

定義

  • 種別: function
  • 定義箇所: src/common/scram-common.c:111

目的

指定されたハッシュアルゴリズムを使用して、入力データに対するハッシュ値を計算する汎用関数です。

シグネチャまたは形状

int scram_H(const uint8 *input, pg_cryptohash_type hash_type, int key_length, uint8 *result, const char **errstr);

入力と出力

  • **入力**:
  • input: ハッシュ化対象のデータ。
  • hash_type: 使用するハッシュアルゴリズムの種類（pg_cryptohash_type）。
  • key_length: 入力データの長さ。
  • **出力**:
  • result: 計算されたハッシュ値を格納するバッファ。
  • errstr: エラー発生時にエラーメッセージが格納されるポインタ。
  • **戻り値**: 成功時に 0、失敗時に -1 を返します。

前提条件と不変条件

  • input および result は有効なメモリ領域を指している必要があります。
  • 内部で使用する pg_cryptohash コンテキストの生成および計算処理が正常に完了する必要があります。

アルゴリズムまたは振る舞い

1. pg_cryptohash_create を呼び出し、指定された hash_type に基づくコンテキストを生成します。 2. コンテキストの生成に失敗した場合、pg_cryptohash_error を取得して -1 を返します。 3. pg_cryptohash_init で初期化し、pg_cryptohash_update で入力データを処理した後、pg_cryptohash_final で結果を生成します。 4. 処理の成否に関わらず、最後に pg_cryptohash_free を呼び出してコンテキストを解放します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/libpq/auth-scram.c
  • src/common/scram-common.c
  • src/interfaces/libpq/fe-auth-scram.c
  • **呼び出し先**:
  • pg_cryptohash_create
  • pg_cryptohash_init
  • pg_cryptohash_update
  • pg_cryptohash_final
  • pg_cryptohash_free
  • pg_cryptohash_error

関連データ構造

  • pg_cryptohash_ctx: 暗号学的ハッシュ計算の状態を保持する構造体。
  • pg_cryptohash_type: 使用可能なハッシュアルゴリズムを定義する列挙型。

失敗モードと境界条件

  • コンテキスト生成失敗（メモリ不足等）時に -1 を返し、errstr にエラー内容を設定します。
  • ハッシュ計算の各ステップ（init, update, final）でエラーが発生した場合、コンテキストを解放した上で -1 を返します。

保守担当者向けメモ

  • 本関数は環境非依存（フロントエンド/バックエンド共通）で動作するように設計されています。
  • errstr は呼び出し元でエラーログ出力等に利用されることを想定しています。
