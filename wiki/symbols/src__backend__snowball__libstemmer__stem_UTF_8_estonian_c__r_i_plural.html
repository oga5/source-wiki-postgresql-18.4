r_i_plural

定義

  • **種別**: function
  • **定義箇所**: src/backend/snowball/libstemmer/stem_UTF_8_estonian.c:969

目的

エストニア語のステミング処理において、特定の複数形接尾辞（i で始まるもの）を単語末尾から削除する。

シグネチャまたは形状

static int r_i_plural(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（Snowball ランタイム環境構造体）
  • **出力**: int（成功時に 1、失敗または条件不一致時に 0、エラー時に負の値を返す）

前提条件と不変条件

  • z->I[0] に格納された境界位置よりも現在のカーソル位置 z->c が後ろにあること。
  • 削除対象の接尾辞が a_3 配列に定義されていること。
  • 削除対象の文字が g_RV（RV領域）に含まれていること。

アルゴリズムまたは振る舞い

1. **境界チェック**: 現在のカーソル位置 z->c が z->I[0] 未満であれば終了する。 2. **領域設定**: z->lb を z->I[0] に一時的に設定し、処理範囲を制限する。 3. **接尾辞検索**: z->p[z->c - 1] が文字 i (ASCII 105) であることを確認し、a_3 配列に基づいて後方検索 (find_among_b) を行う。 4. **領域検証**: 削除対象の接尾辞が g_RV グループに含まれるかを確認する。 5. **削除**: 条件を満たす場合、slice_del を呼び出して該当部分を削除する。 6. **復元**: 変更した z->lb を元の値に戻す。

関連データ構造

  • SN_env: ステミング処理の環境状態を保持する構造体。
  • a_3: r_i_plural が使用する接尾辞の among 配列。
  • g_RV: 処理対象領域を定義する文字グループ。

失敗モードと境界条件

  • z->c <= z->lb: カーソルが境界より前にある場合、処理を中断する。
  • find_among_b の不一致: 接尾辞が見つからない場合、0 を返す。
  • slice_del のエラー: メモリ操作等で負の値が返された場合、その値をそのまま伝播させる。
