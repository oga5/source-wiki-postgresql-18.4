ResetAllOptions

定義

  • 種別: function
  • 定義箇所: src/backend/utils/misc/guc.c:2002

目的

現在設定されているすべての GUC（Grand Unified Configuration）パラメータを、デフォルト値にリセットします。

シグネチャまたは形状

void ResetAllOptions(void)

入力と出力

  • 入力: なし（内部の guc_nondef_list を参照）
  • 出力: なし（副作用としてグローバルな設定変数を更新）

前提条件と不変条件

  • guc_nondef_list に登録されている、デフォルト値から変更されたパラメータのみを対象とします。
  • PGC_SUSET または PGC_USERSET コンテキストのパラメータのみがリセット対象となります。
  • GUC_NO_RESET_ALL フラグが設定されているパラメータはリセットされません。

アルゴリズムまたは振る舞い

1. guc_nondef_list を反復処理し、リセット対象のパラメータを特定します。 2. トランザクションのアボートに備え、push_old_value を呼び出して現在の値を保存します。 3. パラメータの型（PGC_BOOL, PGC_INT, PGC_REAL, PGC_STRING, PGC_ENUM）に応じて、以下の処理を行います。

  • assign_hook が定義されていれば実行します。
  • 変数値を reset_val で上書きします。
  • extra フィールドを reset_extra で更新します。

    • 4. パラメータのソース情報を reset_source に戻し、セッションコンテキストおよびロール情報をリセットします。 5. GUC_REPORT フラグを持つパラメータで、かつ未報告の状態であれば、guc_report_list に追加して報告対象とします。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/backend/commands/discard.c:71 (DISCARD ALL コマンド実行時)
  • src/backend/utils/misc/guc_funcs.c:151
  • 呼び出し先:
  • push_old_value
  • 各パラメータの assign_hook
  • set_string_field / set_extra_field
  • set_guc_source
  • slist_push_head

関連データ構造

  • config_generic: パラメータの基底構造体。
  • guc_nondef_list: デフォルト値から変更されたパラメータのリスト。
  • guc_report_list: 変更をクライアントへ報告するためのリスト。

失敗モードと境界条件

  • GUC_NO_RESET_ALL フラグが付与されたパラメータは、明示的にリセットから除外されます。
  • PGC_S_OVERRIDE 以下のソースを持つパラメータは、既にデフォルトに近い状態とみなされリセットされません。

保守担当者向けメモ

  • 本関数は DISCARD ALL 等の重要なクリーンアップ処理で使用されます。
  • 新しい GUC パラメータ型を追加した場合は、本関数の switch 文に該当するケースを追加する必要があります。
  • assign_hook の実行順序や、リセット時のメモリ管理（特に文字列型）については guc_internal.h の定義と整合性を保つ必要があります。
