gbt_uuidge

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_uuid.c:42

目的

2つのUUID値を比較し、第1引数が第2引数以上であるか（>=）を判定します。

シグネチャまたは形状

static bool gbt_uuidge(const void *a, const void *b, FmgrInfo *flinfo)

入力と出力

  • **入力**:
  • a, b: 比較対象の pg_uuid_t 型へのポインタ（void *として渡される）。
  • flinfo: 関数呼び出し情報（本関数内では未使用）。
  • **出力**:
  • bool: a >= b であれば true、そうでなければ false を返します。

前提条件と不変条件

  • a および b は有効な pg_uuid_t 構造体を指している必要があります。
  • UUIDの比較はバイト列としての比較に基づきます。

アルゴリズムまたは振る舞い

内部関数 uuid_internal_cmp を呼び出し、その戻り値が 0 以上（>= 0）であるかどうかを評価します。

呼び出し元と呼び出し先

  • **呼び出し先**: uuid_internal_cmp
  • **呼び出し元**: ソースコード内の静的定義（contrib/btree_gist/btree_uuid.c:86）において、GiST演算子クラスのサポート関数として登録されています。直接的な呼び出し元コードの詳細は提供されていません。

関連データ構造

  • pg_uuid_t: PostgreSQLのUUID型を表現するデータ構造。

失敗モードと境界条件

  • 入力ポインタが NULL の場合の動作は定義されていません。
  • uuid_internal_cmp の仕様に依存します。

保守担当者向けメモ

  • 本関数は btree_gist 拡張におけるUUID型の比較ロジックの一部です。同様の比較関数（gbt_uuidlt, gbt_uuidgt, gbt_uuidle）と対になって実装されています。
