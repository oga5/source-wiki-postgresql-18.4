range_table_walker_impl

定義

  • 種別: function
  • 定義箇所: src/backend/nodes/nodeFuncs.c:2796

目的

クエリの範囲テーブル（Range Table）に含まれる各 RangeTblEntry に対して、指定されたコールバック関数を再帰的に適用し、走査を行う。

シグネチャまたは形状

bool range_table_walker_impl(List *rtable, tree_walker_callback walker, void *context, int flags)

入力と出力

  • **入力**:
  • rtable: 走査対象の RangeTblEntry を保持する List。
  • walker: 各ノードに対して呼び出されるコールバック関数。
  • context: コールバック関数に渡される任意のコンテキストデータ。
  • flags: 走査の挙動を制御するフラグ。
  • **出力**:
  • bool: コールバック関数が true を返した場合は直ちに true を返し、走査を中断する。すべての要素を走査し終えた場合は false を返す。

前提条件と不変条件

  • rtable は RangeTblEntry ノードのリストである必要がある。
  • walker 関数は、RangeTblEntry を適切に処理できるシグネチャを持つ必要がある。

アルゴリズムまたは振る舞い

1. rtable リスト内の各要素を foreach マクロで順次取り出す。 2. 各要素を RangeTblEntry 型としてキャストする。 3. range_table_entry_walker を呼び出し、現在の RangeTblEntry に対して walker を適用する。 4. range_table_entry_walker が true を返した場合は、その時点で走査を終了し true を呼び出し元へ返す。 5. リストの最後まで走査が完了した場合は false を返す。

呼び出し元と呼び出し先

  • **呼び出し先**: range_table_entry_walker
  • **呼び出し元**:
  • src/include/nodes/nodeFuncs.h にてマクロ定義を通じて参照されている。
  • 具体的な呼び出し元関数の詳細は、提供されたソース範囲内では特定できない。

関連データ構造

  • List: 範囲テーブルを保持するリスト構造。
  • RangeTblEntry: 範囲テーブルの各要素を表すノード構造体。
  • tree_walker_callback: 走査時に実行される関数ポインタ型。

失敗モードと境界条件

  • rtable が空リストの場合、ループは実行されず false を返す。
  • walker 関数内でエラーが発生した場合の挙動は、その関数の実装に依存する。

保守担当者向けメモ

  • 本関数は range_table_entry_walker を介して間接的にノードを走査する。ノード構造に変更があった場合は、range_table_entry_walker 側の実装も確認すること。
  • 走査の停止条件（true を返すと中断）は、他の walker 系関数と一貫している。
