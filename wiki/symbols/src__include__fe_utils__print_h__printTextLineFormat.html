printTextLineFormat

定義

  • 種別: struct
  • 定義箇所: src/include/fe_utils/print.h:43

目的

テーブル形式の出力において、罫線や区切り線を描画するために使用する文字セットを保持する構造体です。

シグネチャまたは形状

typedef struct printTextLineFormat
{
    const char *hrule;      /* 水平線文字 */
    const char *leftvrule;  /* 左端の垂直線（水平線を含む） */
    const char *midvrule;   /* 列間の垂直線（水平線を含む） */
    const char *rightvrule; /* 右端の垂直線（水平線を含む） */
} printTextLineFormat;

入力と出力

本構造体はデータ構造であり、関数ではありません。各メンバは const char * 型の文字列ポインタを保持し、出力処理時に罫線描画用として参照されます。

前提条件と不変条件

  • printTextFormat 構造体内の lrule 配列（printTextRule 列挙型でインデックスされる）の一部として使用されます。
  • 各メンバは、出力フォーマットに応じた適切な描画文字を指す必要があります。

アルゴリズムまたは振る舞い

テーブルの各行や区切り線を描画する際、現在の出力位置やルール（printTextRule）に応じて、この構造体から適切な文字を取得してストリームへ書き出します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/fe_utils/print.c 内のテーブル出力関数群（printTable 等）が、スタイル定義から本構造体を参照しています。
  • **呼び出し先**: 本構造体自体はデータ保持用であり、直接関数を呼び出すことはありません。

関連データ構造

  • printTextFormat: 本構造体を配列として保持する親構造体。
  • printTextRule: printTextFormat 内の lrule 配列をインデックスするために使用される列挙型。

失敗モードと境界条件

  • メンバが NULL に設定されている場合、出力処理側で適切にハンドリングされないとセグメンテーション違反を引き起こす可能性があります。

保守担当者向けメモ

  • 本構造体は printTable 等の出力ロジックにおいて、罫線のスタイルを切り替えるための基盤です。新しい罫線スタイルを追加する場合は、この構造体のメンバ構成を維持したまま、printTextFormat の初期化データに追加してください。
