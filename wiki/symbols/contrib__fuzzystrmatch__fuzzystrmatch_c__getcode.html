getcode

定義

  • 種別: function
  • 定義箇所: contrib/fuzzystrmatch/fuzzystrmatch.c:122

目的

文字をMetaphoneアルゴリズムで使用されるビットフラグ形式のコードへ変換します。

シグネチャまたは形状

static int getcode(char c)

入力と出力

  • **入力**: char c （評価対象の文字）
  • **出力**: int （文字に対応するビットフラグ値、または該当しない場合は 0）

前提条件と不変条件

  • 入力文字は unsigned char にキャストされて処理されます。
  • ASCIIのアルファベット（A-Z）のみが有効なコードを持ちます。非ASCII文字や記号が入力された場合、関数は 0 を返します。

アルゴリズムまたは振る舞い

1. 入力文字がアルファベットであるか判定します。 2. アルファベットの場合、大文字に変換します。 3. 大文字が 'A' から 'Z' の範囲内であれば、静的配列 _codes から対応するインデックスの値を返します。 4. 上記以外の場合は 0 を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • isvowel マクロ
  • NOCHANGE マクロ
  • AFFECTH マクロ
  • MAKESOFT マクロ
  • NOGHTOF マクロ
  • **呼び出し先**:
  • isalpha (標準ライブラリ)
  • toupper (標準ライブラリ)

関連データ構造

  • _codes: ファイルスコープで定義された静的配列（具体的な内容はソースコードの他の箇所で定義されています）。

失敗モードと境界条件

  • 非ASCII文字が入力された場合、isalpha の挙動に依存しつつ、最終的に 0 を返すことで安全に処理されます。
  • 大文字・小文字は toupper により正規化されるため、入力のケースは結果に影響しません。

保守担当者向けメモ

  • 本関数はMetaphoneアルゴリズムの判定ロジック（母音判定や特定の文字グループ判定）の基礎となります。
  • _codes 配列の定義が変更された場合、本関数の戻り値の意味も連動して変化するため注意してください。
