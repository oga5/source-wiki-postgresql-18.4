unicode_titlecase_simple

定義

  • 種別: function
  • 定義箇所: src/common/unicode_case.c:57

目的

単一の Unicode 文字（pg_wchar）に対して、タイトルケース変換を適用します。

シグネチャまたは形状

pg_wchar unicode_titlecase_simple(pg_wchar code)

入力と出力

  • **入力**: code (変換対象の pg_wchar 型 Unicode コードポイント)
  • **出力**: 変換後の pg_wchar。変換ルールが存在しない場合は元の code をそのまま返します。

前提条件と不変条件

  • find_case_map 関数が case_map_title テーブルを参照して変換を試みます。
  • 変換結果が 0 の場合はマッピングが存在しないとみなされ、入力値が維持されます。

アルゴリズムまたは振る舞い

1. find_case_map を呼び出し、case_map_title テーブルから code に対応するタイトルケースを探します。 2. 戻り値が 0 以外であれば、その値をタイトルケースとして返します。 3. 戻り値が 0 であれば、入力された code をそのまま返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/common/unicode/case_test.c:83 でテスト目的の呼び出しが確認されています。
  • **呼び出し先**: find_case_map を内部で呼び出します。

関連データ構造

  • pg_wchar: PostgreSQL で使用されるワイド文字型。
  • case_map_title: タイトルケース変換用のマッピングテーブル（src/common/unicode_case_table.h に定義）。

失敗モードと境界条件

  • マッピングテーブルに存在しない文字が入力された場合、変換を行わず入力値をそのまま返します。
  • 本関数は単一文字の変換のみを扱うため、文字列全体のタイトルケース変換に必要な単語境界判定は行いません。

保守担当者向けメモ

  • 本関数は単純なマッピング変換のみを実装しています。文脈依存の特殊ケース（ギリシャ語の最終シグマ等）が必要な場合は、convert_case 等のより高次な関数を利用してください。
