トルコ語のステミング処理において、特定の接尾辞パターン（a_4 に関連するもの）を識別し、母音調和のルールに基づいた除去を行う。

入力と出力

  • **入力**: struct SN_env * z （Snowball ランタイム環境）
  • **出力**: int 型。成功時は 1、失敗時は 0、またはエラー発生時は負の値を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • 処理対象の単語末尾が母音（'a' または 'e'）である必要がある。

アルゴリズムまたは振る舞い

1. r_check_vowel_harmony(z) を呼び出し、母音調和の妥当性を検証する。検証失敗時はその結果を返す。 2. 現在のカーソル位置 z->c がバッファ境界 z->lb より後ろにあり、かつ末尾の文字が 'a' (97) または 'e' (101) であることを確認する。条件を満たさない場合は 0 を返す。 3. find_among_b(z, a_4, 2) を使用し、後方検索で a_4 配列内のパターンとマッチングを行う。マッチしない場合は 0 を返す。 4. r_mark_suffix_with_optional_y_consonant(z) を呼び出し、接尾辞の処理を完了させる。この関数の戻り値が 0 以下であればそれを返し、成功時は 1 を返す。
