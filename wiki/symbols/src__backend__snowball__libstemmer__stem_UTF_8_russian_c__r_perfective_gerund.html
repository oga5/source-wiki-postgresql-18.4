目的

ロシア語のステミング処理において、単語末尾から完了分詞（perfective gerund）の接尾辞を検出し、削除することで語幹を抽出します。

シグネチャまたは形状

static int r_perfective_gerund(struct SN_env * z);

入力と出力

  • **入力**: SN_env * z（Snowballステミング環境構造体）
  • **出力**: int 型。接尾辞が削除された場合、または処理が正常に完了した場合は 1 を返し、マッチしない場合は 0 を返します。エラー発生時は負の値を返します。

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要があります。
  • 内部的に find_among_b を使用するため、単語の末尾から前方に向かって処理が行われます。

アルゴリズムまたは振る舞い

1. z->ket を現在のカーソル位置 z->c に設定します。 2. a_0 配列を使用して、後方から接尾辞のパターンマッチング（find_among_b）を行います。 3. マッチした接尾辞の種類（among_var）に応じて分岐します。

  • **ケース1**: 特定の接尾辞（s_0 または s_1）との照合を試み、成功した場合に slice_del で削除します。
  • **ケース2**: そのまま slice_del で削除します。

    • 4. 削除処理が成功した場合、1 を返して終了します。

関連データ構造

  • a_0: 完了分詞の接尾辞パターンを格納した among 構造体の配列。

失敗モードと境界条件

  • find_among_b がマッチしない場合、0 を返して終了します。
  • slice_del が内部でエラー（負の値）を返した場合、その値をそのまま呼び出し元へ伝播させます。
