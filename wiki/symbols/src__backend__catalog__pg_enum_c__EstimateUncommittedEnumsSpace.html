EstimateUncommittedEnumsSpace

定義

  • 種別: function
  • 定義箇所: src/backend/catalog/pg_enum.c:812

目的

トランザクション内で作成された未コミットの列挙型および列挙値の情報を、並列クエリ等のためにシリアライズする際に必要なメモリサイズを算出します。

シグネチャまたは形状

Size EstimateUncommittedEnumsSpace(void)

入力と出力

  • **入力**: なし。
  • **出力**: Size 型。未コミットの列挙型および列挙値の総数に、終端子（terminator）分として 2 つの Oid サイズを加算したバイト数を返します。

前提条件と不変条件

  • uncommitted_enum_types および uncommitted_enum_values ハッシュテーブルが初期化されているか、または NULL であることを前提とします。
  • 返されるサイズは、SerializeUncommittedEnums で使用されるメモリ領域の要件と一致する必要があります。

アルゴリズムまたは振る舞い

1. uncommitted_enum_types ハッシュテーブルが存在する場合、そのエントリ数を取得します。 2. uncommitted_enum_values ハッシュテーブルが存在する場合、そのエントリ数を取得します。 3. 合計エントリ数に 2 を加算し、sizeof(Oid) を乗算した値を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/access/transam/parallel.c:292
  • src/backend/catalog/pg_enum.c:835 (アサーション内)
  • **呼び出し先**:
  • hash_get_num_entries (ハッシュテーブルのエントリ数取得)

関連データ構造

  • uncommitted_enum_types: 未コミットの列挙型を追跡するハッシュテーブル。
  • uncommitted_enum_values: 未コミットの列挙値を追跡するハッシュテーブル。

失敗モードと境界条件

  • ハッシュテーブルが NULL の場合、そのテーブルのエントリ数は 0 として計算されます。
  • 戻り値は常に sizeof(Oid) * 2 以上となります（終端子分）。

保守担当者向けメモ

  • 本関数は SerializeUncommittedEnums と対になる計算ロジックです。シリアライズ形式に変更を加える場合は、本関数の計算式も必ず更新してください。
