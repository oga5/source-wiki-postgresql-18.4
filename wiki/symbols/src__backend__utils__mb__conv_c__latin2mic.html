latin2mic

定義

  • 種別: function
  • 定義箇所: src/backend/utils/mb/conv.c:88

目的

ISO-8859 などのシングルバイト文字セット（LATIN）から Mule Internal Code (MIC) 形式へ変換します。

シグネチャまたは形状

int latin2mic(const unsigned char *l, unsigned char *p, int len, int lc, int encoding, bool noError)

入力と出力

  • **入力**:
  • l: 変換元文字列（ポインタ）
  • p: 変換先バッファ（ポインタ）
  • len: 変換対象のバイト長
  • lc: MIC の文字セット識別子
  • encoding: 現在のエンコーディング識別子
  • noError: 変換エラー時にエラー報告を抑制するかを示すフラグ
  • **出力**:
  • 戻り値: 処理されたバイト数（int）
  • p が指すバッファに変換後の文字列が書き込まれ、末尾に \0 が付与されます。

前提条件と不変条件

  • p は変換後の文字列を格納するのに十分なサイズを持つ必要があります。
  • len が 0 の場合、処理は行われず 0 を返します。

アルゴリズムまたは振る舞い

1. 入力文字列をバイト単位で走査します。 2. 各バイト c1 について以下を実行します。

  • c1 が 0 の場合、noError が偽であれば report_invalid_encoding を呼び出し、真であればループを終了します。
  • IS_HIGHBIT_SET(c1) が真（上位ビットが立っている）の場合、MIC のエスケープシーケンスとして lc を出力バッファに書き込みます。
  • c1 を出力バッファに書き込みます。

    • 3. 処理したバイト数をカウントし、最後にバッファをヌル終端します。 4. 最終的に消費した入力バイト数を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/mb/conversion_procs/cyrillic_and_mic/cyrillic_and_mic.c
  • src/backend/utils/mb/conversion_procs/latin2_and_win1250/latin2_and_win1250.c
  • src/backend/utils/mb/conversion_procs/latin_and_mic/latin_and_mic.c
  • **呼び出し先**:
  • report_invalid_encoding（エラー発生時）

関連データ構造

  • pg_wchar.h で定義されるエンコーディング識別子およびマクロ（IS_HIGHBIT_SET）。

失敗モードと境界条件

  • 入力文字列中に 0 バイトが含まれる場合、noError フラグの状態に応じてエラー報告または処理中断が発生します。
  • len を超えて読み込むことはありません。

保守担当者向けメモ

  • 本関数は latin_and_mic などの変換プロシージャから呼び出される低レベルユーティリティです。
  • lc（文字セット識別子）の扱いは、MIC 形式の仕様に依存するため、変更時は関連するエンコーディング定義を確認してください。
