sizebitvec

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/tsquery_gist.c:118

目的

TSQuerySign 型のビットマップに含まれるセットビット（1であるビット）の総数をカウントする。

シグネチャまたは形状

int sizebitvec(TSQuerySign sign)

入力と出力

  • **入力**: TSQuerySign sign（ビットマップ値）
  • **出力**: int（ビットマップ内の 1 の個数）

前提条件と不変条件

  • TSQS_SIGLEN 定数に基づき、ビットマップの全範囲を走査する。

アルゴリズムまたは振る舞い

1. size を 0 に初期化する。 2. 0 から TSQS_SIGLEN - 1 までループを回す。 3. 各イテレーションで、sign を右シフトし、最下位ビットが 1 かどうかを 0x01 とのビット積（&）で判定する。 4. 1 であれば size に加算する。 5. 最終的な size を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: 提供されたソースコード内では直接の呼び出し元は特定されていない。
  • **呼び出し先**: 本関数は算術演算のみを行い、他の関数を呼び出さない。
  • **注記**: 外部モジュール（contrib 配下の hstore, intarray, ltree, pg_trgm）において、同名の関数が定義・利用されているが、本シンボルとの直接的な呼び出し関係はソース内では確認できない。

関連データ構造

  • TSQuerySign: tsquery のインデックス用圧縮表現。

失敗モードと境界条件

  • TSQS_SIGLEN が 0 の場合、ループは実行されず 0 を返す。
  • 入力値が負数や特定のビットパターンであっても、論理シフト相当の処理によりビットカウントは正しく行われる。

保守担当者向けメモ
