int4_bool

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/int.c:390

目的

int4 型の値を boolean 型へ変換します。

シグネチャまたは形状

Datum int4_bool(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS を介して渡される int4 型の Datum。
  • **出力**: bool 型の Datum。値が 0 の場合は false、それ以外の場合は true を返します。

前提条件と不変条件

  • 入力引数は PG_GETARG_INT32(0) を通じて取得されます。
  • 戻り値は PG_RETURN_BOOL マクロを使用して構築されます。

アルゴリズムまたは振る舞い

1. PG_GETARG_INT32(0) で入力値を取得します。 2. 入力値が 0 と等しいか比較します。 3. 0 であれば false を返し、それ以外（非ゼロ）であれば true を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: SQLインターフェース経由で呼び出されます。具体的な呼び出し元のコードパスは提供されていません。
  • **呼び出し先**: PG_GETARG_INT32 および PG_RETURN_BOOL マクロを使用します。

関連データ構造

  • Datum: PostgreSQLの汎用データ表現形式。

失敗モードと境界条件

  • 入力値が int4 の範囲内であれば、特別なエラー条件はありません。

保守担当者向けメモ

  • 本関数は単純な真偽判定を行うキャスト関数です。
  • 逆の変換を行う関数として bool_int4 が存在します。
