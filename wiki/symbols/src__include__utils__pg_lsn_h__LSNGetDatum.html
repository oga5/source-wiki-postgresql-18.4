LSNGetDatum

定義

  • 種別: static inline function
  • 定義箇所: src/include/utils/pg_lsn.h:27

目的

PostgreSQLの内部的なログシーケンス番号（LSN）型である XLogRecPtr を、SQLエンジンが扱う汎用データ型 Datum に変換します。

シグネチャまたは形状

static inline Datum LSNGetDatum(XLogRecPtr X)

入力と出力

  • **入力**: XLogRecPtr X （WAL内の位置を示す64ビット識別子）
  • **出力**: Datum （SQL層で利用可能な汎用データ表現）

前提条件と不変条件

  • XLogRecPtr は uint64 として定義されています。
  • 本関数は Int64GetDatum を利用して変換を行うため、XLogRecPtr の値は int64 にキャストされます。

アルゴリズムまたは振る舞い

1. 引数として受け取った XLogRecPtr 型の値を int64 にキャストします。 2. Int64GetDatum マクロを呼び出し、Datum 型に変換して返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/pageinspect/gistfuncs.c
  • contrib/pageinspect/rawpage.c
  • contrib/pg_logicalinspect/pg_logicalinspect.c
  • contrib/pg_walinspect/pg_walinspect.c
  • **呼び出し先**:
  • Int64GetDatum（fmgr.h 経由で提供される汎用マクロ）

関連データ構造

  • XLogRecPtr: LSNの基底型（access/xlogdefs.h で定義）。
  • Datum: PostgreSQLの汎用データ型（fmgr.h で定義）。

失敗モードと境界条件

  • 本関数自体は単純なキャストと変換を行うため、関数内部での失敗モードはありません。
  • 入力値が InvalidXLogRecPtr であっても、そのまま Datum に変換されます。

保守担当者向けメモ

  • 本関数はインライン関数として定義されており、パフォーマンスを重視する箇所で利用されます。
  • LSNをSQL層へ戻す際は、本関数を直接呼ぶ代わりに PG_RETURN_LSN(x) マクロを使用することが推奨されます。
