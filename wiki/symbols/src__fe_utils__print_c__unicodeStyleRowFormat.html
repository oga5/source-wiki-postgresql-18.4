unicodeStyleRowFormat

定義

  • 種別: struct
  • 定義箇所: src/fe_utils/print.c:101

目的

Unicode罫線を用いた表形式出力において、行の境界線や交差部分を構成する文字セットを定義するために使用されます。

シグネチャまたは形状

typedef struct unicodeStyleRowFormat
{
    const char *horizontal;
    const char *vertical_and_right[2];
    const char *vertical_and_left[2];
} unicodeStyleRowFormat;

入力と出力

本構造体はデータ構造であり、関数ではありません。各メンバはUnicode罫線文字（文字列）を保持します。

前提条件と不変条件

  • vertical_and_right および vertical_and_left は、それぞれ2つの要素を持つ配列であり、異なる罫線スタイル（例：単線・二重線など）を切り替えるために使用されます。

アルゴリズムまたは振る舞い

本構造体は、print.c 内の表出力ロジックにおいて、Unicodeスタイルでの描画時に参照される定数データとして機能します。

呼び出し元と呼び出し先

  • 呼び出し元: src/fe_utils/print.c 内の表出力処理（row_style 配列の定義や、ヘッダー描画処理 header ポインタへの代入など）で参照されています。
  • 呼び出し先: 本構造体自体はデータ保持用であり、関数を呼び出すことはありません。

関連データ構造

  • unicodeStyleFormat: Unicode出力スタイル全体を定義する構造体の一部として利用される可能性があります。

失敗モードと境界条件

  • 構造体メンバが NULL で初期化された場合、出力処理側でセグメンテーション違反が発生する可能性があります。

保守担当者向けメモ

  • 本構造体は src/fe_utils/print.c 内で定義されています。Unicode罫線の種類を追加・変更する場合は、この構造体の定義と、それを参照している出力関数のロジックを併せて確認してください。
