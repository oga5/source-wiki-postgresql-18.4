PGTYPEStimestamp_sub_interval

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/pgtypeslib/timestamp.c:912

目的

timestamp 型の値から指定された interval 型の期間を減算し、結果を別の timestamp 型変数に格納します。

シグネチャまたは形状

int PGTYPEStimestamp_sub_interval(timestamp * tin, interval * span, timestamp * tout)

入力と出力

  • **入力**:
  • tin: 減算対象となる timestamp へのポインタ。
  • span: 減算する期間を表す interval へのポインタ。
  • **出力**:
  • tout: 計算結果を格納する timestamp へのポインタ。
  • 戻り値: 成功時に 0、エラー時に非ゼロの値を返します（内部で呼び出す PGTYPEStimestamp_add_interval の戻り値に依存）。

前提条件と不変条件

  • tin、span、tout は有効なメモリ領域を指している必要があります。
  • 内部的に interval の各フィールド（month および time）の符号を反転させるため、span が指す元のデータ構造は変更されません。

アルゴリズムまたは振る舞い

1. 入力された span の month および time フィールドの符号を反転させた一時的な interval 構造体（tspan）を作成します。 2. 作成した tspan を引数として PGTYPEStimestamp_add_interval を呼び出し、加算処理として減算を実行します。 3. PGTYPEStimestamp_add_interval の戻り値をそのまま返します。

呼び出し元と呼び出し先

  • **呼び出し先**: PGTYPEStimestamp_add_interval
  • **呼び出し元**: ソースコード内の直接的な呼び出し元情報は提供されていませんが、src/interfaces/ecpg/include/pgtypes_timestamp.h にて外部公開用として宣言されています。

関連データ構造

  • timestamp: 内部的にマイクロ秒単位の int64 として管理される型。
  • interval: 月単位および時間単位の期間を保持する構造体。

失敗モードと境界条件

  • 入力された timestamp が特殊値（infinity 等）である場合や、計算結果が timestamp 型の有効範囲を超える場合、PGTYPEStimestamp_add_interval の仕様に従いエラーが返されます。

保守担当者向けメモ

  • 本関数は PGTYPEStimestamp_add_interval のラッパーです。減算ロジックの変更が必要な場合は、interval の符号反転処理が適切か確認してください。
