ExecEndSampleScan

定義

  • 種別: function
  • 定義箇所: src/backend/executor/nodeSamplescan.c:178

目的

SampleScan ノードの実行を終了し、関連するリソースを解放する。

シグネチャまたは形状

void ExecEndSampleScan(SampleScanState *node)

入力と出力

  • **入力**: SampleScanState *node - 終了対象となるスキャン状態構造体へのポインタ。
  • **出力**: なし（void）。

前提条件と不変条件

  • node は有効な SampleScanState を指している必要がある。
  • node->tsmroutine が初期化されていること。

アルゴリズムまたは振る舞い

1. node->tsmroutine->EndSampleScan が定義されている場合、これを呼び出し、サンプリング手法固有の終了処理を実行する。 2. node->ss.ss_currentScanDesc が存在する場合、table_endscan を呼び出してテーブルスキャン記述子をクローズする。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/executor/execProcnode.c:628 から呼び出されることが確認されている。
  • **呼び出し先**:
  • node->tsmroutine->EndSampleScan（サンプリング手法固有の関数）
  • table_endscan（テーブルアクセスメソッドの終了関数）

関連データ構造

  • SampleScanState: スキャンノードの実行状態を保持する構造体。
  • TsmRoutine: サンプリング手法のインターフェースを定義する構造体。

失敗モードと境界条件

  • node->tsmroutine->EndSampleScan が NULL の場合、サンプリング手法固有の終了処理はスキップされる。
  • node->ss.ss_currentScanDesc が NULL の場合、table_endscan は呼び出されない。

保守担当者向けメモ

  • 本関数は TABLESAMPLE 句を用いたスキャン終了時のクリーンアップを担当する。
  • table_endscan の呼び出しは、node->ss.ss_currentScanDesc の存在確認を伴うため安全である。
