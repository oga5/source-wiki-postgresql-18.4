r_verb

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_KOI8_R_russian.c:477

目的

KOI8-Rエンコーディングのロシア語単語に対し、Snowballアルゴリズムのルールに基づいて動詞語尾の削除または置換を行う。

シグネチャまたは形状

static int r_verb(struct SN_env * z)

入力と出力

  • **入力**: z (Snowball環境構造体へのポインタ)
  • **出力**: 成功時に 1、処理対象外または失敗時に 0、メモリ不足等のエラー時に負の値を返す。

前提条件と不変条件

  • z->c が単語の末尾位置を指していること。
  • z->p にKOI8-Rエンコーディングの文字列が格納されていること。
  • 内部で find_among_b を使用するため、a_4 配列が定義されていること。

アルゴリズムまたは振る舞い

1. 現在位置 z->c を z->ket に設定する。 2. 文字列末尾の文字コードに基づき、事前定義された条件（a_4 配列のパターン）に合致するかを確認する。 3. find_among_b で接尾辞を検索し、マッチした場合は z->bra を設定する。 4. マッチした接尾辞の種類（among_var）に応じて以下の処理を行う:

  • **ケース1**: 特定の文字（0xC1 または 0xD1）が直前にあるかを確認し、条件を満たせば slice_del で削除する。
  • **ケース2**: 無条件で slice_del を実行する。

    • 5. 処理完了後、成功として 1 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_KOI8_R_russian.c 内のメインステミング処理フローから呼び出される。
  • **呼び出し先**: find_among_b (パターン検索), slice_del (文字列操作)。

関連データ構造

  • struct SN_env: ステミング処理の状態を保持する構造体。
  • a_4: 動詞語尾のパターンを格納した among 構造体の配列。

失敗モードと境界条件

  • 文字列が空、または検索範囲外（z->lb）に達した場合は 0 を返す。
  • slice_del が内部でエラー（負の値）を返した場合、その値をそのまま伝播させる。
  • マッチする接尾辞がない場合は 0 を返す。

保守担当者向けメモ

  • 本関数はSnowballコンパイラによる自動生成コードである。ロジックの変更が必要な場合は、直接編集せず元のSnowballスクリプト（.sbl）を修正すること。
  • KOI8-Rの文字コード（0xC1, 0xD1 等）を直接扱っているため、エンコーディングの仕様変更時には注意が必要である。
