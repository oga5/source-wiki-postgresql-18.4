NumericShort

定義

  • 種別: struct
  • 定義箇所: src/backend/utils/adt/numeric.c:136

目的

NUMERIC 型の値を格納する際、ディスク容量を節約するための短縮形式（Short form）を提供します。

シグネチャまたは形状

struct NumericShort
{
    uint16      n_header;
    NumericDigit n_data[FLEXIBLE_ARRAY_MEMBER];
};

入力と出力

  • **入力**: n_header（符号、表示スケール、重みを含む16ビットのヘッダ）、n_data（可変長配列としての数値桁データ）。
  • **出力**: なし（データ格納用の構造体）。

前提条件と不変条件

  • NumericShort 形式は、特定の条件を満たす小さな数値に対して使用されます。
  • ヘッダの残りの14ビットは、数値の属性（符号やスケール等）を保持するために使用されます。

アルゴリズムまたは振る舞い

  • NUMERIC 型の内部表現において、通常の NumericData よりもコンパクトな形式として機能します。
  • 数値の表現において、NumericShort 形式であるかどうかの判定は、ヘッダ情報のビットパターンに基づいて行われます。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/adt/numeric.c 内の NUMERIC 型処理ロジック。
  • **呼び出し先**: 構造体定義のため、直接的な関数呼び出しはありません。
  • 注: 具体的な関数レベルの呼び出し関係は、提供されたソース excerpt には含まれていません。

関連データ構造

  • NumericData: NUMERIC 型の標準的なオンディスク表現。
  • NumericDigit: 数値の各桁を保持する型。
  • NumericVar: 算術演算時に使用されるメモリ上の作業用構造体。

失敗モードと境界条件
