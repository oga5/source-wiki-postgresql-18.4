GetCurrentDateTime

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/datetime.c:375

目的

現在の日時を取得し、指定された構造体に格納する。

シグネチャまたは形状

void GetCurrentDateTime(struct pg_tm *tm)

入力と出力

  • **入力**: struct pg_tm *tm（現在日時を格納するための構造体へのポインタ）
  • **出力**: なし（引数 tm を介して結果を書き戻す）

前提条件と不変条件

  • 引数 tm は有効な struct pg_tm へのポインタであること。

アルゴリズムまたは振る舞い

1. GetCurrentTimeUsec(tm, &fsec, NULL) を呼び出す。 2. GetCurrentTimeUsec の内部処理により、現在の日時情報が tm に格納される。 3. 本関数自体は、取得された小数秒（fsec）およびタイムゾーン情報（NULL）を無視し、tm 構造体への書き込みのみを行う。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/date.c
  • src/backend/utils/adt/datetime.c
  • src/interfaces/ecpg/pgtypeslib/datetime.c
  • **呼び出し先**:
  • GetCurrentTimeUsec

関連データ構造

  • struct pg_tm: 日時情報を保持する構造体。
  • fsec_t: 小数秒を扱う型（本関数内ではローカル変数として宣言されるが使用されない）。

失敗モードと境界条件

  • システム時刻の取得に失敗した場合の挙動については、GetCurrentTimeUsec の実装に依存する。

保守担当者向けメモ

  • 本関数は GetCurrentTimeUsec のラッパーである。日時情報の取得ロジックを変更する場合は GetCurrentTimeUsec を参照すること。
  • src/interfaces/ecpg/pgtypeslib/dt.h にも同名の関数が存在するが、シグネチャが struct tm *tm となっている点に注意が必要。
