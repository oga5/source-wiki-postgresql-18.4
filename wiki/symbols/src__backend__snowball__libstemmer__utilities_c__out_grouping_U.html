out_grouping_U

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/utilities.c:141

目的

UTF-8 エンコードされた文字列において、指定された文字集合（グループ）に含まれない文字を検索し、カーソル位置を進める。

シグネチャまたは形状

int out_grouping_U(struct SN_env * z, const unsigned char * s, int min, int max, int repeat)

入力と出力

  • z: ステミング環境を保持する SN_env 構造体へのポインタ。
  • s: 文字集合をビットマップ形式で保持する配列。
  • min, max: 判定対象となる文字コードの範囲。
  • repeat: 0 以外の場合、グループ外の文字が続く限りカーソルを繰り返し進める。
  • 戻り値: 成功時は進めたバイト数（w）、失敗時は 0 または -1。

前提条件と不変条件

  • z->p（バッファ）、z->c（カーソル）、z->l（長さ）が有効なステミング環境として初期化されていること。
  • s は min から max までの範囲をカバーするビットマップであること。

アルゴリズムまたは振る舞い

1. get_utf8 を用いて現在のカーソル位置から文字コード ch をデコードし、そのバイト数 w を取得する。 2. w が 0 の場合はエラーとして -1 を返す。 3. ch が min 未満、または max を超える場合、あるいはビットマップ s 上で該当ビットが 0（グループ外）である場合、その文字はグループ外とみなされ、バイト数 w を返す。 4. グループ内の文字である場合、カーソル z->c を w だけ進め、repeat が真であれば 1 に戻り処理を継続する。 5. repeat が偽でグループ内の文字に当たった場合、またはループ終了時に 0 を返す。

呼び出し元と呼び出し先

  • 呼び出し元: stem_UTF_8_armenian.c, stem_UTF_8_basque.c, stem_UTF_8_catalan.c, stem_UTF_8_danish.c 等の各言語ステミング実装。
  • 呼び出し先: get_utf8（utilities.c 内のユーティリティ）。

関連データ構造

  • SN_env: ステミング環境を保持する構造体。

失敗モードと境界条件

  • get_utf8 が不正なシーケンスを検出した場合、-1 を返す。
  • 文字列の終端に達した場合、ループは終了し 0 を返す。

保守担当者向けメモ

  • _U サフィックスは UTF-8 対応を示しており、マルチバイト文字のスキップを正しく処理する。
  • s[ch >> 3] & (0X1 << (ch & 0X7)) は、ビットマップから特定の文字コードの存在有無を定数時間で判定する手法である。
