NewDumpOptions

定義

  • 種別: function
  • 定義箇所: src/bin/pg_dump/pg_backup_archiver.c:133

目的

ダンプ処理で使用される DumpOptions 構造体のメモリを確保し、初期化を行う。

シグネチャまたは形状

DumpOptions *NewDumpOptions(void)

入力と出力

  • 入力: なし
  • 出力: 初期化済みの DumpOptions 構造体へのポインタ。

前提条件と不変条件

  • pg_malloc を使用しているため、メモリ不足時にはプロセスが終了する。
  • 返される DumpOptions は InitDumpOptions によってデフォルト値が設定されている。

アルゴリズムまたは振る舞い

1. pg_malloc を呼び出し、DumpOptions 構造体サイズ分のメモリを動的に確保する。 2. InitDumpOptions を呼び出し、確保した領域に対してデフォルトのオプション値を設定する。 3. 初期化済みのポインタを呼び出し元へ返す。

呼び出し元と呼び出し先

  • 呼び出し元: src/bin/pg_dump/pg_backup_archiver.c:165 (および pg_backup.h で宣言されているため外部からの呼び出しの可能性あり)
  • 呼び出し先: pg_malloc (メモリ確保), InitDumpOptions (初期化)

関連データ構造

  • DumpOptions: ダンプ処理の挙動を制御する設定値群を保持する構造体。

失敗モードと境界条件

  • メモリ割り当て失敗時: pg_malloc の仕様に従い、エラー終了する。

保守担当者向けメモ

  • 本関数は DumpOptions のライフサイクル管理の起点です。設定値のデフォルトを変更する場合は InitDumpOptions を修正してください。
