printFormat

定義

  • 種別: enum
  • 定義箇所: src/include/fe_utils/print.h:28

目的

クエリ結果をフロントエンド（psql 等）で出力する際のフォーマット形式を定義します。

シグネチャまたは形状

enum printFormat {
    PRINT_NOTHING = 0,
    PRINT_ALIGNED,
    PRINT_ASCIIDOC,
    PRINT_CSV,
    PRINT_HTML,
    PRINT_LATEX,
    PRINT_LATEX_LONGTABLE,
    PRINT_TROFF_MS,
    PRINT_UNALIGNED,
    PRINT_WRAPPED
};

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: printTableOpt 構造体などの設定項目として使用され、出力処理の挙動を決定します。

前提条件と不変条件

  • PRINT_NOTHING は初期化漏れを検知するためのデフォルト値として使用されます。

アルゴリズムまたは振る舞い

printTable() などの出力関数において、この列挙値に基づいてテーブルの罫線スタイル、区切り文字、マークアップタグの付与などの整形ロジックが切り替えられます。

呼び出し元と呼び出し先

  • 呼び出し元（直接的な参照）:
  • src/bin/psql/command.c: _align2string 関数および変数定義にて使用。
  • src/include/fe_utils/print.h: printTableOpt 構造体内の format メンバとして使用。
  • 呼び出し先: なし（列挙型のため）。

関連データ構造

  • printTableOpt: 出力設定を保持する構造体で、本列挙型をメンバとして含みます。

失敗モードと境界条件

  • PRINT_NOTHING が設定されたまま出力処理が行われた場合、意図しない動作やエラーが発生する可能性があります。

保守担当者向けメモ

  • 新しい出力フォーマットを追加する場合は、本列挙型に値を追加し、src/fe_utils/print.c 内の出力ロジック（printTable 等）に該当する処理を実装してください。
