CompiledExprState

定義

  • 種別: struct
  • 定義箇所: src/backend/jit/llvm/llvmjit_expr.c:48

目的

式評価のJITコンパイルプロセスにおいて、コンパイル対象のコンテキストと生成される関数名を保持するために使用されます。

シグネチャまたは形状

typedef struct CompiledExprState
{
	LLVMJitContext *context;
	const char *funcname;
} CompiledExprState;

入力と出力

  • 入力: LLVMJitContext へのポインタおよび関数名文字列。
  • 出力: なし（構造体定義）。

前提条件と不変条件

  • context は有効な LLVMJitContext を指している必要があります。
  • funcname はコンパイルされた関数を識別するための文字列として保持されます。

アルゴリズムまたは振る舞い

本構造体は、式評価のJITコンパイル時にメモリ確保（palloc0）され、コンパイル状態を管理するためのプライベートデータとして利用されます。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/jit/llvm/llvmjit_expr.c 内の式コンパイル処理（palloc0 によるインスタンス化および state->evalfunc_private への代入）。
  • 呼び出し先: 情報なし。

関連データ構造

  • LLVMJitContext: JITコンパイルの実行環境を管理する構造体。

失敗モードと境界条件

  • メモリ不足時に palloc0 が失敗する可能性があります。

保守担当者向けメモ

  • 本構造体は llvmjit_expr.c 内のプライベートな状態管理用です。新しいフィールドを追加する場合は、コンパイル時のライフサイクル管理に影響がないか確認してください。
