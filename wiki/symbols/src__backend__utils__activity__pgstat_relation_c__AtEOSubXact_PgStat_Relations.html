AtEOSubXact_PgStat_Relations

定義

  • src/backend/utils/activity/pgstat_relation.c:608

目的

サブトランザクション終了時（コミットまたはアボート）に、リレーションの統計情報を親トランザクションレベルへ伝播または破棄し、統計状態を整理する。

シグネチャまたは形状

void AtEOSubXact_PgStat_Relations(PgStat_SubXactStatus *xact_state, bool isCommit, int nestDepth)

入力と出力

  • xact_state: 処理対象のサブトランザクション状態。
  • isCommit: コミット時は true、アボート時は false。
  • nestDepth: 現在のサブトランザクションのネストレベル。

前提条件と不変条件

  • trans->nest_level == nestDepth であること。
  • tabstat->trans == trans であること。
  • 統計情報は TopTransactionContext 上で管理されていることを前提とする。

アルゴリズムまたは振る舞い

サブトランザクション内の各リレーション統計エントリを走査し、以下の処理を行う。

1. **コミット時 (isCommit == true)**:

  • 親トランザクションが存在する場合、統計値（挿入・更新・削除数）を親へ加算する。truncdropped が設定されている場合は、親の値を上書きする。
  • 親が存在しない場合、現在のエントリを一つ上の階層へ再リンクし、ネストレベルを更新する。

    • 2. **アボート時 (isCommit == false)**:

  • restore_truncdrop_counters を呼び出し、TRUNCATE/DROP で失われた値を復元する。
  • 試行された操作数をトップレベルの tabstat->counts に加算し、デッドタプル数を更新する。

    • 3. **共通**:

  • 処理完了後、サブトランザクション固有の trans 構造体を pfree で解放する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/activity/pgstat_xact.c
  • **呼び出し先**: save_truncdrop_counters, restore_truncdrop_counters, pgstat_get_xact_stack_level, pfree

関連データ構造

  • PgStat_SubXactStatus: サブトランザクションの統計状態スタック。
  • PgStat_TableXactStatus: リレーションごとのトランザクション内統計。
  • PgStat_TableStatus: リレーションの永続的な統計情報。

失敗モードと境界条件

  • 親トランザクションが存在しない場合、スタックの再構築を行うため、メモリ管理（palloc/pfree）の整合性が重要となる。
  • truncdrop 状態の伝播において、親の統計値が正しく更新されない場合、統計の不整合が発生する可能性がある。

保守担当者向けメモ

  • 本関数はサブトランザクションのネスト構造を深く操作するため、trans->upper の参照が正しい階層を指しているか注意すること。
  • truncdropped フラグが立っている場合の統計加算ロジックは、通常の加算とは異なるため変更時は慎重に検討すること。
