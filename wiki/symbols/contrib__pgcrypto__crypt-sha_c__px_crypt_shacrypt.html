px_crypt_shacrypt

定義

  • **種別**: 関数
  • **定義箇所**: contrib/pgcrypto/crypt-sha.c:68

目的

SHA-256 または SHA-512 アルゴリズムを用いた UNIX 形式のパスワードハッシュを生成します。

シグネチャまたは形状

char *px_crypt_shacrypt(const char *pw, const char *salt, char *passwd, unsigned dstlen)

入力と出力

  • **入力**:
  • pw: ハッシュ化対象のパスワード文字列。
  • salt: $5$ (SHA-256) または $6$ (SHA-512) で始まるソルト文字列。オプションで rounds=<N>$ を含めることが可能。
  • passwd: 結果を格納するバッファ。
  • dstlen: passwd バッファのサイズ。
  • **出力**: 成功時はハッシュ文字列を格納した passwd へのポインタを返し、失敗時はエラーをスローします。

前提条件と不変条件

  • passwd が NULL でないこと。
  • pw および salt が NULL でないこと（NULL の場合は elog(ERROR) を発生）。
  • dstlen が PX_SHACRYPT_BUF_LEN 以上であること。
  • ソルト文字列の先頭 3 バイトが $5$ または $6$ であること。

アルゴリズムまたは振る舞い

1. **検証**: 入力値の妥当性を確認し、ソルトからアルゴリズム種別（SHA256/SHA512）とラウンド数を抽出します。 2. **ラウンド数調整**: rounds オプションが指定されている場合、許容範囲外であれば警告（NOTICE）を出力し、最小値または最大値に補正します。 3. **初期化**: 指定されたアルゴリズムに基づき、px_find_digest を用いてダイジェストオブジェクトを準備します。 4. **計算**: パスワードとソルトを組み合わせ、反復的なダイジェスト計算を行います。 5. **終了処理**: 計算結果を Base64 形式に変換してバッファに格納し、機密情報（一時バッファ）をゼロクリアして終了します。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/pgcrypto/px-crypt.c:76 (px_crypt ディスパッチャ経由)
  • **呼び出し先**: px_find_digest (ダイジェスト計算用抽象化レイヤー)

関連データ構造

  • PGCRYPTO_SHA_t: アルゴリズム種別を識別する列挙型。
  • StringInfo: ハッシュ生成およびソルト解析用の一時バッファ。
  • PX_MD: ダイジェスト計算用オブジェクト。

失敗モードと境界条件

  • **エラー**:
  • pw または salt が NULL の場合、elog(ERROR) を発生。
  • dstlen が不足している場合、elog(ERROR) を発生。
  • ソルトの形式が不正（$5$ または $6$ 以外）な場合、ereport(ERROR) を発生。
  • rounds の解析に失敗した場合、ereport(ERROR) を発生。
  • ダイジェストオブジェクトの初期化に失敗した場合、エラーを返して終了。

保守担当者向けメモ

  • 本関数は Ulrich Drepper 氏のリファレンス実装に基づいています。
  • セキュリティ上の理由から、計算終了後に memset を使用して一時バッファを確実に消去しています。改修時には機密情報の取り扱いに注意してください。
  • rounds の範囲外指定に対する NOTICE 出力は、互換性維持のための仕様です。
