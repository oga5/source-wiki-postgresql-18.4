r_R1

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_german.c:328

目的

単語内の現在のカーソル位置が、ステミング処理における R1 領域の開始位置以降にあるかを判定する。

シグネチャまたは形状

static int r_R1(struct SN_env * z)

入力と出力

  • **入力**: SN_env 構造体へのポインタ z。z->I[2] には R1 領域の開始インデックスが、z->c には現在のカーソル位置が格納されている。
  • **出力**: 整数値を返す。現在のカーソル位置 z->c が R1 の開始位置 z->I[2] 以上であれば 1（真）、そうでなければ 0（偽）を返す。

前提条件と不変条件

  • z->I[2] は、事前に r_mark_regions 関数によって正しく計算・設定されている必要がある。
  • z は有効な SN_env 構造体を指していること。

アルゴリズムまたは振る舞い

1. SN_env 構造体内の整数配列 I のインデックス 2（R1 の境界値）と、現在のカーソル位置 c を比較する。 2. z->I[2] <= z->c の評価結果を戻り値として返す。

呼び出し元と呼び出し先

  • **呼び出し元**: 外部のステミングルール関数から呼び出される。提供されたソース情報に基づくと、本関数は stem_ISO_8859_1_basque.c、stem_ISO_8859_1_catalan.c、stem_ISO_8859_1_dutch.c 等の他言語実装ファイルからも参照されている。
  • **呼び出し先**: なし（純粋な比較演算のみを実行）。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。I 配列（領域境界等の整数値）および c（現在のカーソル位置）を含む。

失敗モードと境界条件

  • z が NULL の場合、デリファレンスによりクラッシュする可能性がある。
  • r_mark_regions が未実行の場合、z->I[2] に不定値が含まれる可能性がある。
