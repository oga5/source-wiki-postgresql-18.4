pg_popcount32_slow

定義

  • 種別: function
  • 定義箇所: src/port/pg_bitutils.c:356

目的

32ビット整数に含まれる「1」のビット数（ポピュレーションカウント）を計算する汎用的なフォールバック実装を提供します。

シグネチャまたは形状

static inline int pg_popcount32_slow(uint32 word)

入力と出力

  • **入力**: uint32 word （カウント対象の32ビット整数）
  • **出力**: int （入力値に含まれる1のビット数）

前提条件と不変条件

  • HAVE__BUILTIN_POPCOUNT が定義されていない環境では、pg_number_of_ones ルックアップテーブルが初期化済みである必要があります。

アルゴリズムまたは振る舞い

1. コンパイル時に HAVE__BUILTIN_POPCOUNT が定義されている場合、組み込み関数 __builtin_popcount(word) を直接呼び出します。 2. 定義されていない場合、8ビット単位で値を処理するループを実行します。

  • word が0になるまで、下位8ビット（word & 255）をインデックスとして pg_number_of_ones テーブルを参照し、その値を result に加算します。
  • word を8ビット右シフトします。

    • 3. 最終的な合計値を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/port/pg_bitutils.c 内の関数ポインタ初期化処理（pg_popcount32 = pg_popcount32_slow;）
  • pg_popcount32_slow を直接利用する各計算処理箇所
  • **呼び出し先**:
  • pg_number_of_ones（ルックアップテーブル）
  • __builtin_popcount（コンパイラ組み込み関数、環境による）

関連データ構造

  • pg_number_of_ones: 0〜255の各値に対するビット数を保持するルックアップテーブル。

失敗モードと境界条件

  • 入力が 0 の場合、ループは実行されず 0 を返します。
  • 32ビットを超える値は入力できません（uint32 型のため）。

保守担当者向けメモ

  • 本関数はハードウェア命令（POPCNT 等）が利用できない環境でのフォールバックとして機能します。
  • パフォーマンスが重要な場合は、実行時に pg_popcount32_fast が選択されるようディスパッチ機構が適切に動作していることを確認してください。
