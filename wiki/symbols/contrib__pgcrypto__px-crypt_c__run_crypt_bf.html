run_crypt_bf

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/px-crypt.c:60

目的

Blowfish アルゴリズムを用いたパスワードハッシュ化処理を pgcrypto モジュールの統一インターフェースに適合させるためのラッパー関数です。

シグネチャまたは形状

static char *run_crypt_bf(const char *psw, const char *salt, char *buf, unsigned len);

入力と出力

  • **入力**:
  • psw: ハッシュ化対象のパスワード文字列。
  • salt: 使用するソルト文字列。
  • buf: 結果を格納するためのバッファ。
  • len: buf のサイズ。
  • **出力**:
  • ハッシュ化された文字列へのポインタ（buf を指す）。

前提条件と不変条件

  • buf は、Blowfish アルゴリズムの出力結果を格納するのに十分なサイズである必要があります。
  • _crypt_blowfish_rn が正常に動作する環境であること。

アルゴリズムまたは振る舞い

1. 内部関数 _crypt_blowfish_rn を呼び出し、パスワードとソルトからハッシュ値を生成します。 2. 生成された結果を buf に格納し、そのポインタを返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/pgcrypto/px-crypt.c 内の静的テーブル px_crypt_list から、$2a$ および $2x$ プレフィックスに関連付けられた関数として参照されています。
  • **呼び出し先**:
  • _crypt_blowfish_rn (具体的な Blowfish ハッシュ計算を実行する関数)

関連データ構造

  • px_crypt_list: px-crypt.c 内で定義された、ソルトのプレフィックスと実行関数を紐付けるテーブル。

失敗モードと境界条件

  • _crypt_blowfish_rn の挙動に依存します。バッファサイズ len が不足している場合や、入力値が不正な場合の挙動は当該関数に準じます。

保守担当者向けメモ

  • 本関数は crypt-blowfish.c で実装されている低レベルな Blowfish 実装へのブリッジです。アルゴリズム自体の修正が必要な場合は crypt-blowfish.c を確認してください。
