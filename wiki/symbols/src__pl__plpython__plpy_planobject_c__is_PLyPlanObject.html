is_PLyPlanObject

定義

  • 種別: function
  • 定義箇所: src/pl/plpython/plpy_planobject.c:84

目的

与えられたPythonオブジェクトが、PL/Pythonの PLyPlan 型であるかどうかを判定します。

シグネチャまたは形状

bool is_PLyPlanObject(PyObject *ob)

入力と出力

  • **入力**: PyObject *ob（判定対象のPythonオブジェクト）
  • **出力**: bool（ob の型が PLy_PlanType と一致する場合は true、それ以外は false）

前提条件と不変条件

  • ob は有効な PyObject ポインタである必要があります。
  • PLy_PlanType が初期化済みであることを前提とします。

アルゴリズムまたは振る舞い

1. 引数 ob の ob_type フィールドを参照します。 2. その値が PLy_PlanType と等しいかを比較し、結果を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/pl/plpython/plpy_spi.c:163
  • **呼び出し先**:
  • 特になし（C言語の構造体フィールド参照のみ）

関連データ構造

  • PLyPlanObject: 本関数が判定対象とする実行計画オブジェクト。
  • PLy_PlanType: Pythonの型システムにおける PLyPlan 型定義。

失敗モードと境界条件

  • ob が NULL の場合、本関数は ob->ob_type へのアクセスによりセグメンテーション違反を引き起こす可能性があります。

保守担当者向けメモ

  • 本関数は型チェックのユーティリティとして機能します。PLy_PlanType の定義変更やPython C APIの仕様変更時には、この判定ロジックが正しく機能するか確認してください。
