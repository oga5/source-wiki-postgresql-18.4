r_KER

定義

目的

インドネシア語のステミング処理において、特定の文字パターン（母音に続く特定の2文字）を照合し、条件を満たす場合にステミングのルール適用を判定する。

シグネチャまたは形状

static int r_KER(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z（Snowball ステミング環境へのポインタ）
  • **出力**: int 型。条件に合致した場合は 1、それ以外の場合は 0 を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体である必要がある。
  • g_vowel（母音グループ）が定義されている環境下で実行される。

アルゴリズムまたは振る舞い

1. out_grouping_U を使用し、現在のカーソル位置の文字が母音（g_vowel）でないことを確認する（範囲: 97-117）。 2. eq_s を使用し、現在のカーソル位置から特定の2文字（s_0）が一致するかを確認する。 3. 上記の条件を両方満たした場合に 1 を返し、それ以外は 0 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_UTF_8_indonesian.c および src/backend/snowball/libstemmer/stem_ISO_8859_1_indonesian.c 内の among 構造体テーブルから参照されている。
  • **呼び出し先**: out_grouping_U および eq_s（Snowball ランタイム関数）。

関連データ構造

  • struct SN_env: ステミング処理の状態を保持する構造体。
  • among: ステミングルールを定義する構造体配列。

失敗モードと境界条件

  • カーソル位置がバッファの末尾に近い場合、eq_s による文字列比較が失敗し 0 を返す。
  • out_grouping_U が母音を検出した場合、即座に 0 を返して終了する。
