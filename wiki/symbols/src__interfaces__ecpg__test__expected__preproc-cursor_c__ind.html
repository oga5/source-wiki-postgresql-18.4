目的

ECPG（Embedded SQL for C）において、SQLのインジケータ変数として使用される共用体型を定義します。

シグネチャまたは形状

typedef union { int integer; short smallint; } ind;

入力と出力

  • **入力**: int 型または short 型の整数値。
  • **出力**: 共用体として保持されたいずれかの型の値。

前提条件と不変条件

  • int 型と short 型のいずれか一方のみが有効な値として使用されることを前提としています。
  • ECPGプリプロセッサが生成するコードにおいて、SQLのNULL値判定やデータ長情報の保持に使用されます。

  • C言語の union 構造を利用し、メモリ領域を共有することで、異なる整数サイズのインジケータ値を柔軟に格納します。
