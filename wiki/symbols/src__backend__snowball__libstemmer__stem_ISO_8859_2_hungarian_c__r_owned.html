目的

ハンガリー語のステミング処理において、単語末尾の所有格接尾辞を特定し、削除または置換を行う。

シグネチャまたは形状

static int r_owned(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング環境構造体）
  • 出力: 処理が成功または該当なしの場合は 1、エラー発生時は負の値、条件不一致時は 0 を返す。

前提条件と不変条件

  • z->c が単語の末尾を指していること。
  • r_R1 関数により、対象の接尾辞が R1 領域内にあることが検証される。

アルゴリズムまたは振る舞い

1. 現在位置（z->c）がバッファの境界外でないか、かつ末尾の文字が 'i' (105) または 'é' (233) であるかを確認する。 2. find_among_b を使用し、配列 a_9（12要素）に基づき末尾から接尾辞を検索する。 3. マッチした場合、z->bra を設定し、r_R1 を呼び出して対象領域を検証する。 4. マッチした接尾辞の種類（among_var）に応じて以下の操作を行う：

  • 1: slice_del で接尾辞を削除。
  • 2 または 3: slice_from_s で接尾辞を特定の文字列（s_8 または s_9）に置換。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_ISO_8859_2_hungarian.c:833 および src/backend/snowball/libstemmer/stem_UTF_8_hungarian.c:839 からの呼び出しが確認されている。
  • 呼び出し先: find_among_b, r_R1, slice_del, slice_from_s。

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_9: 所有格接尾辞を定義した among 構造体の配列。

失敗モードと境界条件

  • z->c <= z->lb: バッファの先頭に達している場合は 0 を返す。
  • find_among_b がマッチしない場合、0 を返す。
  • r_R1 またはスライス操作（slice_del, slice_from_s）が負の値を返した場合、その値をそのまま伝播させる。
