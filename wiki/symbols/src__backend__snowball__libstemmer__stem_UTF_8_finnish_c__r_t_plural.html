r_t_plural

定義

  • **種別**: function
  • **定義箇所**: src/backend/snowball/libstemmer/stem_UTF_8_finnish.c:530

目的

フィンランド語のステミング処理において、特定の複数形接尾辞（'t' を含む語尾）を単語から削除し、語幹を正規化する。

シグネチャまたは形状

static int r_t_plural(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（Snowball ステミング環境へのポインタ）。
  • **出力**: 成功時に 1、失敗または条件不一致時に 0、メモリ不足等のエラー発生時に負の値を返す。

前提条件と不変条件

  • z->I[1]（領域 R1 の開始位置）および z->I[0]（領域 R2 の開始位置）が r_mark_regions によって適切に設定されていること。
  • z->p に処理対象の UTF-8 文字列が格納されていること。

アルゴリズムまたは振る舞い

1. **第一段階（'t' の削除）**:

  • 現在位置が R1 (z->I[1]) 以降であることを確認する。
  • 文字列末尾が 't' であるかを確認し、該当すれば削除する。
  • 削除直前の文字が母音グループ g_V1 に属していることを条件とする。

    • 2. **第二段階（接尾辞の削除）**:

  • 現在位置が R2 (z->I[0]) 以降であることを確認する。
  • 末尾が 'a' で終わるかを確認し、a_9 配列（接尾辞リスト）に基づいてマッチングを行う。
  • マッチした接尾辞を削除する。

    • 3. **後処理**:

  • among_var の値に基づき、特定の条件下で追加のチェック（eq_s_b による文字列比較）を行い、条件を満たせば削除を確定させる。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_UTF_8_finnish.c および src/backend/snowball/libstemmer/stem_ISO_8859_1_finnish.c 内のメインステミング処理フローから呼び出される。
  • **呼び出し先**: in_grouping_b_U（母音判定）、find_among_b（接尾辞検索）、slice_del（文字列削除）、eq_s_b（文字列比較）。

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_9: 削除対象の接尾辞を定義した among 構造体の配列。
  • g_V1: 母音を定義したビットマップ配列。

失敗モードと境界条件

  • z->c < z->I[1] または z->c < z->I[0] の場合、領域外として処理を中断し 0 を返す。
  • find_among_b で接尾辞がマッチしない場合、処理を中断し 0 を返す。
  • slice_del が内部エラー（負の値）を返した場合、その値をそのまま伝播させる。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードであるため、直接の修正は避けること。ロジック変更が必要な場合は、対応する Snowball 言語ソースを修正し、再生成を行う。
  • z->I[0] および z->I[1] は、それぞれ R2 および R1 領域のインデックスとして使用されている。
