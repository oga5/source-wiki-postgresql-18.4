convert_to_utf8

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/pgp-pgsql.c:90

目的

PostgreSQLのデータベースエンコーディングで表現された text 型データを、PGP処理に適した UTF-8 エンコーディングへ変換します。

シグネチャまたは形状

text *convert_to_utf8(text *src)

入力と出力

  • **入力**: src (変換対象の text 型データ)
  • **出力**: UTF-8 に変換された新しい text 型データ。変換が不要な場合は元のデータと等価な内容が返されます。

前提条件と不変条件

  • GetDatabaseEncoding() が現在のデータベースの文字コードを正しく返す環境であること。
  • 内部で呼び出される convert_charset が、指定されたエンコーディング間の変換をサポートしていること。

アルゴリズムまたは振る舞い

1. 現在のデータベースエンコーディングを GetDatabaseEncoding() で取得します。 2. 取得したエンコーディングとターゲットである PG_UTF8 を引数として convert_charset を呼び出します。 3. convert_charset の戻り値をそのまま呼び出し元へ返します。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/pgcrypto/pgp-pgsql.c 内の関数（例: 390行目付近）。
  • **呼び出し先**: convert_charset（同ファイル内のユーティリティ関数）。

関連データ構造

  • text: PostgreSQLの可変長文字列型。

失敗モードと境界条件

  • convert_charset が内部でエラーを発生させた場合、PostgreSQLの標準的なエラー処理メカニズムにより処理が中断されます。

保守担当者向けメモ

  • 本関数は convert_charset への単なるラッパーです。文字コード変換のロジック自体を変更する場合は convert_charset を確認してください。
  • 戻り値のメモリ管理は convert_charset の実装に依存します。
