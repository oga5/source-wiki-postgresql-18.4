PQsslAttributeNames

定義

  • 種別: function
  • 定義箇所: src/interfaces/libpq/fe-secure.c:445

目的

接続に関連付けられたSSL属性名のリストを取得するためのインターフェースを提供します。

シグネチャまたは形状

const char *const *PQsslAttributeNames(PGconn *conn);

入力と出力

  • **入力**: PGconn *conn （接続状態を保持する構造体へのポインタ）。
  • **出力**: const char *const * （NULL終端された文字列配列へのポインタ）。

前提条件と不変条件

  • 現在の実装では、常に NULL のみを要素として持つ静的な文字列配列を返します。
  • 渡される conn ポインタが有効である必要がありますが、関数内部ではこの引数は使用されていません。

アルゴリズムまたは振る舞い

1. 関数が呼び出されると、NULL のみを格納した静的な配列 result を定義します。 2. その配列のアドレスを即座に返します。

関連データ構造

  • PGconn: libpqの接続状態を管理する主要な構造体。

失敗モードと境界条件

  • 本関数は常に有効なポインタを返すため、実行時の失敗モードはありません。
  • 戻り値の配列は常に空（NULL のみ）であるため、呼び出し側で要素を走査する際は即座に終了します。

保守担当者向けメモ

  • 現在の実装はプレースホルダー的な挙動です。将来的にSSL属性のサポートを追加する場合は、この関数が適切な属性名のリストを返すように修正する必要があります。
