cash_mul_int64

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/cash.c:142

目的

通貨型（Cash）の値に対して、64ビット整数（int64）を乗算する。

シグネチャまたは形状

static inline Cash cash_mul_int64(Cash c, int64 i)

入力と出力

  • **入力**:
  • c: 乗算対象の通貨値（Cash 型）。
  • i: 乗数（int64 型）。
  • **出力**: 乗算結果を Cash 型で返す。

前提条件と不変条件

  • 内部表現は int64 である。
  • 乗算結果が int64 の範囲を超える場合、エラーを報告する。

アルゴリズムまたは振る舞い

1. pg_mul_s64_overflow を使用して、c と i の積を計算し、オーバーフローが発生するかを確認する。 2. オーバーフローが発生した場合は ereport を呼び出し、ERRCODE_NUMERIC_VALUE_OUT_OF_RANGE エラーを発生させる。 3. オーバーフローが発生しない場合、計算結果を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/cash.c 内の複数の算術演算関数（PG_RETURN_CASH マクロを介した呼び出しが確認されている）。
  • **呼び出し先**:
  • pg_mul_s64_overflow（オーバーフローチェック用）
  • ereport（エラー報告用）

関連データ構造

  • Cash: int64 の別名。通貨型の内部表現。

失敗モードと境界条件

  • **オーバーフロー**: 乗算結果が int64 の最大値または最小値を超える場合、money out of range というメッセージと共にエラー終了する。

保守担当者向けメモ

  • 本関数は static inline で定義されており、cash.c 内の算術演算関数から頻繁に利用される。
  • ロケール設定による影響は受けない（純粋な整数演算であるため）。
