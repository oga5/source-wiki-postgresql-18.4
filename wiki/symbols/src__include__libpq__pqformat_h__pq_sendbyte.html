pq_sendbyte

定義

  • 種別: static inline 関数
  • 定義箇所: src/include/libpq/pqformat.h:159

目的

StringInfo バッファの末尾に単一のバイト（uint8）を追加します。

シグネチャまたは形状

static inline void pq_sendbyte(StringInfo buf, uint8 byt)

入力と出力

  • **入力**:
  • buf: データを追加する対象の StringInfo バッファへのポインタ。
  • byt: バッファに追加する 8 ビットのデータ。
  • **出力**: なし（戻り値なし）。

前提条件と不変条件

  • buf は有効な StringInfo 構造体を指している必要があります。
  • 内部で pq_sendint8 を呼び出すため、バッファの容量が不足している場合は自動的に拡張されます。

アルゴリズムまたは振る舞い

1. 引数として受け取った byt を pq_sendint8 関数に渡します。 2. pq_sendint8 がバッファへの書き込みおよび必要に応じたバッファ拡張を実行します。

呼び出し元と呼び出し先

  • **呼び出し先**: pq_sendint8
  • **呼び出し元**:
  • src/backend/backup/basebackup_copy.c
  • src/backend/commands/copyfromparse.c
  • src/backend/commands/copyto.c
  • src/backend/replication/logical/proto.c

関連データ構造

  • StringInfo: 通信データのバッファを管理する構造体（lib/stringinfo.h で定義）。

失敗モードと境界条件

  • メモリ不足が発生した場合、pq_sendint8 内部でエラーが発生し、処理が中断される可能性があります。

保守担当者向けメモ

  • 本関数は pq_sendint8 の単なるラッパーです。
  • 頻繁に呼び出されるため、インライン化によるオーバーヘッド削減が意図されています。
