multipleOfPowerOf2

定義

  • 種別: function
  • 定義箇所: src/common/f2s.c:108

目的

指定された整数 value が 2 の p 乗で割り切れるか（すなわち、下位 p ビットがすべて 0 であるか）を判定します。

シグネチャまたは形状

static inline bool multipleOfPowerOf2(const uint32 value, const uint32 p)

入力と出力

  • **入力**:
  • value: 判定対象の 32 ビット符号なし整数。
  • p: 2 のべき乗の指数（判定するビット数）。
  • **出力**:
  • bool: value が 2^p の倍数であれば true、そうでなければ false を返します。

前提条件と不変条件

  • p は value のビット幅（32）の範囲内である必要があります。

アルゴリズムまたは振る舞い

  • (1u << p) - 1 を用いて下位 p ビットのみが 1 であるマスクを作成します。
  • value とそのマスクのビットごとの論理積（AND）をとり、結果が 0 であることを確認します。
  • コメントアウトされたコードとして __builtin_ctz を利用する可能性が示唆されています。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/common/f2s.c:356
  • src/common/d2s.c:488
  • **呼び出し先**:
  • なし（組み込み演算のみ）。
  • **注記**: src/common/d2s.c:106 にも同名の関数定義が存在しますが、引数の型（uint64）が異なります。

関連データ構造

  • なし。

失敗モードと境界条件

  • p が 0 の場合、マスクは (1u << 0) - 1 = 0 となり、value & 0 は常に 0 となるため、常に true を返します。

保守担当者向けメモ

  • 本関数は static inline で定義されており、Ryu アルゴリズムの実装において末尾のゼロを判定するために使用されます。
  • d2s.c に存在する同名関数との混同に注意してください。
