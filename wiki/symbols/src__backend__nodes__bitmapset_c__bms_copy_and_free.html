bms_copy_and_free

定義

  • 種別: function
  • 定義箇所: src/backend/nodes/bitmapset.c:108

目的

入力された Bitmapset を複製し、元の Bitmapset を解放（free）することで、メモリ管理を伴う集合の置換を効率的に行う。

シグネチャまたは形状

static Bitmapset *bms_copy_and_free(Bitmapset *a)

入力と出力

  • **入力**: a (Bitmapset *) - 複製および解放対象のビットマップセット。
  • **出力**: Bitmapset * - a の内容をコピーした新しいビットマップセット。

前提条件と不変条件

  • 入力 a は NULL または有効な Bitmapset ポインタであること。
  • bms_copy および bms_free の仕様に従い、a が NULL の場合は NULL を返す。

アルゴリズムまたは振る舞い

1. bms_copy(a) を呼び出し、入力集合のコピーを作成する。 2. bms_free(a) を呼び出し、元の集合が占有していたメモリを解放する。 3. 複製された新しい集合へのポインタを返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/nodes/bitmapset.c 内の複数の関数（bms_add_member, bms_del_member, bms_union, bms_intersect, bms_difference 等）から、既存の集合を更新する際に呼び出されている。
  • **呼び出し先**:
  • bms_copy
  • bms_free

関連データ構造

  • Bitmapset: 本関数が操作する主要な構造体。

失敗モードと境界条件

  • a が NULL の場合、bms_copy は NULL を返し、bms_free は何もしないため、結果として NULL が返される。

保守担当者向けメモ

  • 本関数は static であり、bitmapset.c 内でのみ使用される。
  • 呼び出し元は、戻り値のポインタを必ず受け取り、元の変数に再代入する必要がある（a = bms_copy_and_free(a) の形式で使用される）。
