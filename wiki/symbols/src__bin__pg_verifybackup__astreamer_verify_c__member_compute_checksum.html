member_compute_checksum

定義

  • 種別: function
  • 定義箇所: src/bin/pg_verifybackup/astreamer_verify.c:253

目的

アーカイブメンバーのデータストリームを逐次処理し、バックアップマニフェストと照合するためのチェックサムをインクリメンタルに計算する。

シグネチャまたは形状

static void member_compute_checksum(astreamer *streamer, astreamer_member *member, const char *data, int len)

入力と出力

  • streamer: 現在の検証状態を保持する astreamer 構造体。
  • member: 現在処理中のアーカイブメンバー情報。
  • data: 計算対象となるデータバッファ。
  • len: data のバイト長。
  • 出力: なし（副作用として astreamer_verify 構造体内のチェックサムコンテキストおよびバイトカウンタを更新）。

前提条件と不変条件

  • mystreamer->verify_checksum が真であること。
  • マニフェストに記録されたチェックサムタイプと、現在のチェックサムコンテキストのタイプが一致していること。

アルゴリズムまたは振る舞い

1. mystreamer->checksum_bytes に処理したデータ長 len を加算し、ファイルサイズとの照合に備える。 2. pg_checksum_update を呼び出し、提供された data をチェックサム計算コンテキストに供給する。 3. 計算に失敗した場合、report_backup_error を通じてエラーを報告し、mystreamer->verify_checksum フラグを偽に設定して以降の検証を停止する。

呼び出し元と呼び出し先

  • 呼び出し元: astreamer_verify_content（src/bin/pg_verifybackup/astreamer_verify.c:109）
  • 呼び出し先: pg_checksum_update、report_backup_error

関連データ構造

  • astreamer_verify: 検証状態を保持するメイン構造体。
  • pg_checksum_context: チェックサム計算の内部状態を保持。
  • manifest_file: マニフェスト上のファイル情報。

失敗モードと境界条件

  • pg_checksum_update が負の値を返した場合、エラー報告を行い検証フラグを無効化する。

保守担当者向けメモ

  • 本関数は astreamer インターフェースの一部として呼び出されます。
  • checksum_bytes はファイル全体のサイズ検証に使用されるため、正確な加算が必須です。
