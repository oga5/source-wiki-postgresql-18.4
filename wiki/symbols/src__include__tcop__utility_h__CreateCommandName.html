CreateCommandName

定義

  • 種別: static inline 関数
  • 定義箇所: src/include/tcop/utility.h:102

目的

パースツリーから、そのコマンドを識別するための標準的なコマンド名（文字列）を取得します。

シグネチャまたは形状

static inline const char *CreateCommandName(Node *parsetree)

入力と出力

  • **入力**: Node *parsetree （SQL文のパースツリー）
  • **出力**: const char * （コマンド名を表す静的な文字列）

前提条件と不変条件

  • 入力される parsetree は有効なノードである必要があります。
  • 内部で CreateCommandTag を呼び出すため、パースツリーがコマンドタグを生成可能な形式である必要があります。

アルゴリズムまたは振る舞い

1. parsetree を引数として CreateCommandTag() を呼び出し、コマンドタグ（CommandTag 型）を生成します。 2. 生成されたコマンドタグを GetCommandTagName() に渡し、対応する文字列形式に変換して返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/commands/event_trigger.c
  • src/backend/executor/execMain.c
  • src/backend/executor/functions.c
  • src/backend/executor/spi.c
  • src/test/modules/test_ddl_deparse/test_ddl_deparse.c
  • **呼び出し先**:
  • CreateCommandTag()
  • GetCommandTagName()

関連データ構造

  • Node: パースツリーの基底構造。
  • CommandTag: コマンドの種類を識別する列挙型または識別子。

失敗モードと境界条件

  • parsetree が NULL の場合や、コマンドタグを生成できないノード型が渡された場合の挙動については、内部で呼び出される CreateCommandTag の実装に依存します。

保守担当者向けメモ

  • 本関数はインライン関数であり、頻繁に呼び出されるため、パフォーマンスへの影響を考慮してください。
  • コマンド名の定義を変更または追加する場合は、tcop/cmdtag.h および関連するタグ生成ロジックを確認してください。
