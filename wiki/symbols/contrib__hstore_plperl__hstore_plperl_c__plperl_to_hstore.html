plperl_to_hstore

定義

  • 種別: function
  • 定義箇所: contrib/hstore_plperl/hstore_plperl.c:98

目的

Perl のハッシュリファレンスを PostgreSQL の hstore 型へ変換する SQL 関数です。

シグネチャまたは形状

Datum plperl_to_hstore(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS を介して渡される Perl のスカラー値（SV*）。
  • **出力**: HStore* 構造体へのポインタを Datum として返却。

前提条件と不変条件

  • 入力値は Perl のハッシュリファレンスである必要があります（再帰的にデリファレンスされます）。
  • Perl のハッシュ以外の型が渡された場合、ERRCODE_FEATURE_NOT_SUPPORTED でエラーを報告します。
  • メモリ割り当てには PostgreSQL のメモリコンテキスト（palloc）を使用します。

アルゴリズムまたは振る舞い

1. dTHX マクロで Perl インタープリタのコンテキストを取得します。 2. 入力値がリファレンスであれば、SvRV を用いて再帰的にデリファレンスし、ハッシュ（SVt_PVHV）を取り出します。 3. hv_iterinit と hv_iternext を使用して Perl ハッシュの全要素を走査します。 4. 各キーと値を pstrdup でコピーし、Pairs 配列を構築します。この際、hstoreCheckKeyLen および hstoreCheckValLen でサイズ検証を行います。 5. hstoreUniquePairs でキーの重複排除を行い、最終的な hstore 構造体を hstorePairs で生成します。 6. 生成された HStore ポインタを PG_RETURN_POINTER で返します。

呼び出し元と呼び出し先

  • **呼び出し元**: PostgreSQL の関数マネージャ（PG_FUNCTION_INFO_V1 マクロにより登録）。
  • **呼び出し先**: hstore モジュールの内部関数（hstoreUniquePairs, hstorePairs, hstoreCheckKeyLen, hstoreCheckValLen）および Perl API（SvROK, SvRV, hv_iterinit 等）。

関連データ構造

  • HStore: PostgreSQL の hstore 内部表現。
  • Pairs: hstore 構築用の中間データ構造。
  • HV, SV: Perl のハッシュおよびスカラー構造体。

失敗モードと境界条件

  • 入力がハッシュでない場合: ereport によりエラー終了します。
  • メモリ不足時: palloc 経由で PostgreSQL のエラーハンドリングが起動します。
  • Perl の値が未定義（undef）の場合: hstore の値として NULL が設定されます。

保守担当者向けメモ

  • hstore モジュールの関数は動的リンクで解決されるため、実行環境で hstore モジュールが正しくロードされている必要があります。
  • Perl API の使用にあたっては、Perl インタープリタのコンテキスト（dTHX）が正しく設定されていることを確認してください。
