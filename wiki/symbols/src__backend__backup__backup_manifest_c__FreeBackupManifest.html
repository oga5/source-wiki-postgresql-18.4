FreeBackupManifest

定義

  • 種別: function
  • 定義箇所: src/backend/backup/backup_manifest.c:91

目的

バックアップ・マニフェスト生成に使用された暗号ハッシュコンテキストを解放し、リソースをクリーンアップします。

シグネチャまたは形状

void FreeBackupManifest(backup_manifest_info *manifest)

入力と出力

  • **入力**: manifest (backup_manifest_info *) - 解放対象のマニフェスト情報構造体へのポインタ。
  • **出力**: なし（void）。

前提条件と不変条件

  • 引数 manifest は有効なポインタである必要があります。
  • 内部の manifest_ctx は pg_cryptohash_free で解放可能な状態である必要があります。

アルゴリズムまたは振る舞い

1. manifest->manifest_ctx に対して pg_cryptohash_free を呼び出し、ハッシュ計算用コンテキストを解放します。 2. manifest->manifest_ctx を NULL に設定し、二重解放や不正なアクセスを防ぎます。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/backup/basebackup.c:673
  • **呼び出し先**:
  • pg_cryptohash_free（外部ライブラリ関数）

関連データ構造

  • backup_manifest_info: マニフェスト生成の状態を保持する構造体。本関数はこの構造体内の manifest_ctx メンバを操作します。

失敗モードと境界条件

  • manifest が NULL の場合、構造体メンバへのアクセスによりセグメンテーション違反が発生する可能性があります。

保守担当者向けメモ

  • 本関数はマニフェスト生成プロセスの終了処理の一部です。manifest 構造体自体のメモリ解放は呼び出し元で行われるため、本関数内ではコンテキストの解放のみに専念しています。
