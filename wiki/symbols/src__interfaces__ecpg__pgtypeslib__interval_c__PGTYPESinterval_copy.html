PGTYPESinterval_copy

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/pgtypeslib/interval.c:1081

目的

interval 構造体の内容を、ソースポインタからデスティネーションポインタへコピーする。

シグネチャまたは形状

int PGTYPESinterval_copy(interval * intvlsrc, interval * intvldest)

入力と出力

  • **入力**:
  • intvlsrc: コピー元の interval 構造体へのポインタ。
  • intvldest: コピー先の interval 構造体へのポインタ。
  • **出力**:
  • 戻り値: 常に 0 を返す。

前提条件と不変条件

  • intvlsrc および intvldest は有効な interval 構造体を指している必要がある。
  • 本関数はメモリの割り当てを行わないため、呼び出し側で intvldest 用のメモリ領域を確保しておく必要がある。

アルゴリズムまたは振る舞い

1. intvlsrc が保持する time（マイクロ秒単位の時間）を intvldest に代入する。 2. intvlsrc が保持する month（月単位の期間）を intvldest に代入する。 3. 0 を返して終了する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/interfaces/ecpg/ecpglib/data.c:842
  • src/interfaces/ecpg/test/expected/pgtypeslib-dt_test.c:126
  • src/interfaces/ecpg/test/expected/pgtypeslib-dt_test2.c:177
  • **呼び出し先**:
  • 特になし（構造体のメンバ代入のみを実行）。

関連データ構造

  • interval: ECPG で使用される時間間隔を表現する構造体（src/interfaces/ecpg/include/pgtypes_interval.h で定義）。

失敗モードと境界条件

  • ポインタが NULL の場合、セグメンテーションフォールトが発生する可能性がある（ソースコード上に明示的なチェックは存在しない）。

保守担当者向けメモ

  • 本関数は単純なメンバコピーを行うユーティリティである。interval 構造体の定義に変更があった場合は、本関数の実装も更新する必要がある。
