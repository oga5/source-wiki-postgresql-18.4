LinkageTypes

定義

  • 種別: typedef
  • 定義箇所: src/backend/jit/llvm/llvmjit_inline.cpp:420

目的

LLVM の GlobalValue::LinkageTypes 型へのエイリアスを提供し、JIT コンパイルにおける関数インポート時のリンク属性指定を簡略化するために使用されます。

シグネチャまたは形状

typedef llvm::GlobalValue::LinkageTypes LinkageTypes;

入力と出力

  • 入力: なし（型定義）
  • 出力: llvm::GlobalValue::LinkageTypes 型

前提条件と不変条件

  • LLVM の GlobalValue 名前空間が利用可能であること。
  • 主にインライン化対象の関数を現在のモジュールへ統合する際の属性設定に使用されます。

アルゴリズムまたは振る舞い

本シンボル自体は型定義であり、実行時の振る舞いは持ちません。インポート処理において、インポートされた関数に対して AvailableExternallyLinkage などの属性を付与する際に参照されます。

呼び出し元と呼び出し先

  • 呼び出し元（参照箇所）:
  • src/backend/jit/llvm/llvmjit_inline.cpp:448: valueToImport->setLinkage(LinkageTypes::AvailableExternallyLinkage);
  • src/backend/jit/llvm/llvmjit_inline.cpp:863: LinkageTypes::AvailableExternallyLinkage の指定に使用。
  • 呼び出し先: LLVM API (llvm::GlobalValue::LinkageTypes)

関連データ構造

  • llvm::GlobalValue
  • llvm::Function (インポート対象の関数オブジェクト)

失敗モードと境界条件

  • 型定義であるため、本シンボル自体に失敗モードはありません。
  • 誤ったリンク属性を指定した場合、LLVM のモジュール結合時（IRMover 使用時）にリンカエラーや不正なシンボル解決が発生する可能性があります。

保守担当者向けメモ

  • 本シンボルは llvm::GlobalValue::LinkageTypes の単なる別名です。LLVM のバージョンアップに伴い、リンク属性の定義や挙動が変更された場合は、LLVM の仕様に従う必要があります。
