cmpLexeme

定義

  • 種別: function
  • 定義箇所: src/backend/tsearch/dict_thesaurus.c:355

目的

2つの TheLexeme 構造体を比較し、辞書内の語彙の順序付けや重複判定を行う。

シグネチャまたは形状

static int cmpLexeme(const TheLexeme *a, const TheLexeme *b)

入力と出力

  • **入力**: 比較対象となる TheLexeme 構造体へのポインタ a および b。
  • **出力**:
  • 0: 両方の lexeme が NULL、または文字列として等しい場合。
  • 正の整数: a->lexeme が NULL で b->lexeme が非 NULL の場合、あるいは strcmp の結果が正の場合。
  • 負の整数: a->lexeme が非 NULL で b->lexeme が NULL の場合、あるいは strcmp の結果が負の場合。

前提条件と不変条件

  • a および b は有効な TheLexeme 構造体を指している必要がある。
  • lexeme メンバは文字列（char *）または NULL である。

アルゴリズムまたは振る舞い

1. 両方の lexeme が NULL の場合は 0 を返す。 2. a->lexeme のみが NULL の場合は 1 を返す。 3. b->lexeme のみが NULL の場合は -1 を返す。 4. 両方が非 NULL の場合、strcmp(a->lexeme, b->lexeme) の結果を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/tsearch/dict_thesaurus.c 内の複数箇所（ソートや検索処理で使用）。
  • **呼び出し先**:
  • 標準ライブラリ関数 strcmp。

関連データ構造

  • TheLexeme: シソーラス辞書における語彙情報を保持する構造体。

失敗モードと境界条件

  • lexeme が NULL であるケースを明示的に処理しており、セグメンテーション違反を回避している。
  • strcmp に依存しているため、ロケール設定に影響を受ける可能性がある。

保守担当者向けメモ

  • 本関数は TheLexeme の比較専用である。LexemeInfo 構造体の比較には cmpLexemeInfo が別途存在する。
