is_funcclause

定義

  • 種別: static inline 関数
  • 定義箇所: src/include/nodes/nodeFuncs.h:70

目的

与えられたノードが FuncExpr 型の式節であるかを判定する。

シグネチャまたは形状

static inline bool is_funcclause(const void *clause)

入力と出力

  • **入力**: clause (任意のノードへのポインタ)
  • **出力**: bool 型。ノードが FuncExpr であれば true、それ以外（NULL を含む）であれば false を返す。

前提条件と不変条件

  • clause は NULL である可能性がある。
  • IsA マクロを使用してノードの型タグを検証するため、clause が指すメモリ領域は有効な Node 構造体（またはその派生）である必要がある。

アルゴリズムまたは振る舞い

1. clause が NULL でないことを確認する。 2. IsA(clause, FuncExpr) を評価し、ノードの型タグが T_FuncExpr と一致するかを判定する。 3. 両方の条件を満たす場合にのみ true を返す。

関連データ構造

  • FuncExpr: 本関数が判定対象とする式ノードの型。src/include/nodes/primnodes.h で定義されている。

失敗モードと境界条件

  • clause が NULL の場合、安全に false を返す。
  • clause が Node 構造体として不正なメモリを指している場合、IsA マクロの挙動により未定義動作が発生する可能性がある。

保守担当者向けメモ

  • 本関数は頻繁に呼び出されるため、パフォーマンスを考慮してヘッダーファイル内で static inline として定義されている。
  • 多くの呼び出し元で「paranoid（用心深い）」なチェックとして使用されており、ノードの型が期待通りであるかを実行時に確認する役割を担っている。
