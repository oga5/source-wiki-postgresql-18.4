calc_arraycontsel

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/array_selfuncs.c:336

目的

配列型に対する演算子（@>、<@、&&）の選択率（Selectivity）を、統計情報（MCELEM、DECHIST）を用いて推定する。

シグネチャまたは形状

static Selectivity calc_arraycontsel(VariableStatData *vardata, Datum constval, Oid elemtype, Oid operator)

入力と出力

  • **入力**:
  • vardata: 統計情報を含む変数データ。
  • constval: 比較対象の定数配列。
  • elemtype: 配列要素の型OID。
  • operator: 評価対象の演算子OID。
  • **出力**:
  • Selectivity（0.0〜1.0の範囲の推定値）。

前提条件と不変条件

  • constval は elemtype と同じ要素型を持つ配列であること。
  • 要素型に対して有効な比較関数が存在すること（存在しない場合は DEFAULT_SEL を返す）。
  • 統計情報が存在する場合、statistic_proc_security_check を通過すること。

アルゴリズムまたは振る舞い

1. **初期化**: lookup_type_cache を使用して要素型の比較関数を取得。 2. **統計情報の取得**: pg_statistic から STATISTIC_KIND_MCELEM（最頻要素）を取得。<@ 演算子の場合は併せて STATISTIC_KIND_DECHIST（要素数ヒストグラム）を取得。 3. **推定計算**: mcelem_array_selec を呼び出し、統計情報（または統計がない場合は NULL）を渡して選択率を算出。 4. **補正**: 統計情報が存在する場合、stanullfrac（NULL率）を用いて selec *= (1.0 - stanullfrac) の補正を行う。 5. **後処理**: constval が TOAST されていた場合、内部で作成したコピーを pfree して解放。

呼び出し元と呼び出し先

  • **呼び出し元**: arraycontsel（src/backend/utils/adt/array_selfuncs.c 内）。
  • **呼び出し先**: mcelem_array_selec、get_attstatsslot、free_attstatsslot、lookup_type_cache、DatumGetArrayTypeP。

関連データ構造

  • VariableStatData: 統計情報へのアクセスポイント。
  • AttStatsSlot: pg_statistic から抽出された統計データ格納用。
  • ArrayType: 配列の内部表現。
  • TypeCacheEntry: 型情報のキャッシュ。

失敗モードと境界条件

  • **統計情報欠如**: pg_statistic に情報がない場合、mcelem_array_selec に NULL を渡してフォールバック計算を行う。
  • **型不整合**: 比較関数が取得できない場合、直ちに DEFAULT_SEL を返す。
  • **メモリ管理**: constval が TOAST されている場合、DatumGetArrayTypeP で作成されたコピーを適切に解放する。

保守担当者向けメモ

  • 本関数は「各要素の出現は独立である」という仮定に基づいた推定を行う。
  • <@ 演算子におけるヒストグラム補正は、統計情報が利用可能な場合にのみ実行される。
  • 統計情報の更新状況（ANALYZE）が推定精度に直結するため、プランナの挙動が不自然な場合は統計情報の確認を推奨する。
