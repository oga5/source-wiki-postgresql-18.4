LockInfoData

定義

  • 種別: struct
  • 定義箇所: src/include/utils/rel.h:44

目的

PostgreSQL のロックマネージャがリレーションに対するロックを管理するために使用する識別情報を保持する。

シグネチャまたは形状

typedef struct LockInfoData
{
	LockRelId	lockRelId;
} LockInfoData;

入力と出力

  • 入力: LockRelId 型のフィールド lockRelId を保持する。
  • 出力: なし（データ構造体）。

前提条件と不変条件

  • RelationData 構造体の一部として組み込まれることを前提としている。

アルゴリズムまたは振る舞い

  • 本構造体自体はデータ保持に特化しており、特定のアルゴリズムは定義されていない。リレーションのロック状態を識別するためのコンテナとして機能する。

呼び出し元と呼び出し先

  • 呼び出し元: src/include/utils/rel.h 内で RelationData 構造体のメンバ rd_lockInfo として定義されている。
  • 呼び出し先: 情報なし。

関連データ構造

  • LockRelId: ロック対象のリレーションを識別するための構造体。
  • RelationData: 本構造体をメンバとして保持するリレーション記述子。
  • LockInfo: LockInfoData へのポインタ型。

失敗モードと境界条件

  • 情報なし。

保守担当者向けメモ

  • RelationData 構造体の一部として定義されているため、本構造体の変更はリレーションキャッシュのメモリレイアウトに直接影響する。
  • rd_lockInfo フィールドを介してアクセスされる。
