BF_word

定義

  • typedef unsigned int BF_word;
  • 定義箇所: contrib/pgcrypto/crypt-gensalt.c:19

目的

Blowfish 暗号アルゴリズムの実装において、32ビットのデータ単位を表現するための型定義です。

シグネチャまたは形状

unsigned int のエイリアスです。

入力と出力

該当なし（型定義のため）。

前提条件と不変条件

Blowfish アルゴリズムの内部処理において、32ビット幅のデータとして扱われることを前提としています。

アルゴリズムまたは振る舞い

Blowfish の計算ロジック（Sボックス、キー、一時変数など）において、数値演算およびビット操作の基本単位として使用されます。

呼び出し元と呼び出し先

  • **呼び出し元/参照元**: contrib/pgcrypto/crypt-blowfish.c 内の広範な関数（BF_decode, BF_encode, BF_swap 等）および内部変数（Sボックス、salt、output 等）で利用されています。
  • **呼び出し先**: なし（型定義のため）。

関連データ構造

  • BF_key: BF_word の配列として定義される Blowfish キー構造。
  • BF_ctx: Blowfish のコンテキスト構造体（本シンボルを利用して構成される）。

失敗モードと境界条件

該当なし（型定義のため）。

保守担当者向けメモ

  • 本型定義は crypt-blowfish.c がコンパイルされない環境でも Blowfish 関連のソルト生成機能が動作するよう、crypt-gensalt.c に配置されています。
  • 32ビット幅を明示的に扱う必要があるため、プラットフォーム依存の int サイズに注意してください。
