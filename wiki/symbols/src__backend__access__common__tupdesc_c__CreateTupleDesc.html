CreateTupleDesc

定義

  • 種別: function
  • 定義箇所: src/backend/access/common/tupdesc.c:228

目的

指定された属性定義の配列に基づき、新しい TupleDesc（タプル記述子）を作成および初期化します。

シグネチャまたは形状

TupleDesc CreateTupleDesc(int natts, Form_pg_attribute *attrs)

入力と出力

  • **入力**:
  • natts: 作成するタプルの属性数。
  • attrs: Form_pg_attribute 構造体へのポインタ配列。各要素は個別の属性定義を保持します。
  • **出力**:
  • 初期化済みの TupleDesc 構造体へのポインタ。

前提条件と不変条件

  • attrs 配列には少なくとも natts 個の有効な属性定義が含まれている必要があります。
  • 戻り値の TupleDesc は、内部的に CompactAttribute が各属性に対して同期された状態となります。

アルゴリズムまたは振る舞い

1. CreateTemplateTupleDesc(natts) を呼び出し、指定された属性数を持つ空の TupleDesc を割り当てます。 2. natts 回のループ処理を行い、以下の操作を繰り返します。

  • memcpy を使用して、入力された attrs[i] から TupleDesc 内の対応する属性スロットへ ATTRIBUTE_FIXED_PART_SIZE 分のデータをコピーします。
  • populate_compact_attribute(desc, i) を呼び出し、コピーした属性情報に基づいて最適化用の CompactAttribute を生成・更新します。

    • 3. 初期化が完了した desc を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/bootstrap/bootstrap.c: ブートストラップ処理中に呼び出されます。
  • src/backend/catalog/heap.c: システムカタログの初期化時に呼び出されます。
  • **呼び出し先**:
  • CreateTemplateTupleDesc: 構造体のメモリ確保に使用。
  • populate_compact_attribute: 属性情報の最適化に使用。

関連データ構造

  • TupleDesc: タプルの構造を保持する主要なメタデータ構造体。
  • Form_pg_attribute: 属性の定義情報を保持する構造体。
  • CompactAttribute: タプル解釈を高速化するための内部構造体。

失敗モードと境界条件

  • natts が 0 以下の値である場合、CreateTemplateTupleDesc の仕様に依存します。
  • メモリ不足時には、CreateTemplateTupleDesc 内でエラーが発生し、処理が中断される可能性があります。

保守担当者向けメモ

  • 本関数は属性定義のコピーのみを行い、制約（TupleConstr）のコピーは行いません。制約が必要な場合は CreateTupleDescCopyConstr 等の利用を検討してください。
  • FormData_pg_attribute の構造に変更を加えた場合、populate_compact_attribute が正しく最適化情報を再生成しているか確認してください。
