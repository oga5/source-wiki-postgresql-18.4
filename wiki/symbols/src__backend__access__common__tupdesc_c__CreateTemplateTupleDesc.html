CreateTemplateTupleDesc

定義

  • 種別: function
  • 定義箇所: src/backend/access/common/tupdesc.c:181

目的

指定された属性数（列数）を持つ、初期化済みの空のタプル記述子（TupleDesc）をメモリ上に割り当て、構築します。

シグネチャまたは形状

TupleDesc CreateTemplateTupleDesc(int natts)

入力と出力

  • **入力**: natts (int) - 作成するタプル記述子が保持する属性の数。
  • **出力**: TupleDesc - 割り当ておよび初期化されたタプル記述子構造体へのポインタ。

前提条件と不変条件

  • natts >= 0 であること（Assert により検証）。
  • 戻り値の TupleDesc は、CompactAttribute 配列と FormData_pg_attribute 配列を連続したメモリ領域として保持する。

アルゴリズムまたは振る舞い

1. natts が負でないことを確認します。 2. TupleDescData 構造体の固定部分、CompactAttribute 配列（natts 分）、および FormData_pg_attribute 配列（natts 分）を格納するために必要な合計メモリサイズを palloc で確保します。 3. 構造体の各フィールドを初期化します。

  • natts: 指定された属性数。
  • constr: NULL に設定。
  • tdtypeid: RECORDOID に設定。
  • tdtypmod: -1 に設定。
  • tdrefcount: -1（参照カウント管理対象外であることを示す）に設定。

    • 4. 構築した TupleDesc を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/dblink/dblink.c
  • contrib/pg_buffercache/pg_buffercache_pages.c
  • contrib/pg_visibility/pg_visibility.c
  • src/backend/access/brin/brin_tuple.c
  • src/backend/access/common/tupdesc.c（CreateTupleDesc 等の内部関数から）
  • **呼び出し先**:
  • palloc（メモリ確保）

関連データ構造

  • TupleDesc: タプルの構造を定義する主要な構造体。
  • CompactAttribute: タプル解釈を高速化するための簡略化された属性情報。
  • FormData_pg_attribute: システムカタログ pg_attribute の行データ構造。

失敗モードと境界条件

  • natts に負の値を指定した場合、Assert が有効なビルドではアボートします。
  • メモリ不足時には palloc がエラーを発生させます。

保守担当者向けメモ

  • 本関数はメモリを確保するのみであり、個別の属性定義（TupleDescInitEntry 等）は含まれていません。
  • FormData_pg_attribute 配列の各要素は、ATTRIBUTE_FIXED_PART_SIZE バイト分のみが有効であるとみなされます。
  • tdrefcount が -1 に初期化されるため、参照カウント管理が必要な場合は別途設定が必要です。
