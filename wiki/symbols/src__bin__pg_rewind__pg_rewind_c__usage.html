usage

定義

  • 種別: function
  • 定義箇所: src/bin/pg_rewind/pg_rewind.c:91

目的

pg_rewind コマンドの利用方法、利用可能なオプション、およびバグ報告先や公式ドキュメントのURLを標準出力に表示する。

シグネチャまたは形状

static void usage(const char *progname);

入力と出力

  • **入力**: progname (実行ファイル名を示す文字列)
  • **出力**: なし（標準出力へのテキスト表示のみ）

前提条件と不変条件

  • progname は有効な文字列ポインタであること。
  • PACKAGE_BUGREPORT、PACKAGE_NAME、PACKAGE_URL がビルド環境で定義されていること。

アルゴリズムまたは振る舞い

1. printf を使用し、プログラムの概要と基本的な使用法（Usage: ...）を表示する。 2. pg_rewind でサポートされている各コマンドラインオプション（-c, -D, --source-pgdata, --source-server, -n, -N, -P, -R, --config-file, --debug, --no-ensure-shutdown, --sync-method, -V, -?）の用途を順次出力する。 3. 最後にバグ報告用アドレスとプロジェクトのホームページURLを表示して終了する。

呼び出し元と呼び出し先

  • **呼び出し元**: 提供されたソース excerpt 内には、本関数を呼び出している箇所は含まれていません。
  • **呼び出し先**: 標準ライブラリの printf を呼び出します。

関連データ構造

  • なし

失敗モードと境界条件

  • 標準出力がクローズされている場合、出力は行われません。
  • 本関数自体はエラーを返さないため、失敗モードは存在しません。
