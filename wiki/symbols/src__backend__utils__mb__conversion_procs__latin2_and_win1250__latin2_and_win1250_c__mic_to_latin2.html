mic_to_latin2

定義

  • 種別: function
  • 定義箇所: src/backend/utils/mb/conversion_procs/latin2_and_win1250/latin2_and_win1250.c:101

目的

MULE_INTERNAL（PostgreSQL内部エンコーディング）から LATIN2（ISO-8859-2）への文字エンコーディング変換を実行します。

シグネチャまたは形状

Datum mic_to_latin2(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS を介して以下の引数を受け取ります。
  • src: 変換元の文字列（unsigned char *）
  • dest: 変換先のバッファ（unsigned char *）
  • len: 変換対象の長さ（int）
  • noError: エラーハンドリングのフラグ（bool）
  • **出力**: 変換されたバイト数を int32 として返します。

前提条件と不変条件

  • CHECK_ENCODING_CONVERSION_ARGS(PG_MULE_INTERNAL, PG_LATIN2) により、変換元が PG_MULE_INTERNAL、変換先が PG_LATIN2 であることが検証されます。

アルゴリズムまたは振る舞い

1. PG_GETARG_* マクロを使用して、関数引数から変換元ポインタ、変換先ポインタ、長さ、エラーフラグを取得します。 2. CHECK_ENCODING_CONVERSION_ARGS を呼び出し、エンコーディングの整合性を確認します。 3. mic2latin ユーティリティ関数を呼び出し、LC_ISO8859_2 を指定して変換を実行します。 4. 変換されたバイト数を PG_RETURN_INT32 で返します。

呼び出し元と呼び出し先

  • **呼び出し元**: PostgreSQLの変換マネージャ（直接的な呼び出し元はソースコード上では明示されていませんが、PG_FUNCTION_INFO_V1(mic_to_latin2) により関数として登録されています）。
  • **呼び出し先**: mic2latin（src/backend/utils/mb/wchar.c で定義された変換ユーティリティ）。

関連データ構造

  • PG_FUNCTION_ARGS: PostgreSQLの関数引数インターフェース。

失敗モードと境界条件

  • CHECK_ENCODING_CONVERSION_ARGS の検証に失敗した場合、エラーが発生します。
  • mic2latin の内部処理において、変換不能な文字や不正なシーケンスに遭遇した場合の挙動は noError 引数に依存します。

保守担当者向けメモ

  • 本関数は単純なラッパーであり、実際の変換ロジックは mb/wchar.c の mic2latin に委譲されています。
  • 変換テーブルの変更が必要な場合は、本ファイル内の静的テーブル定義を確認してください。
