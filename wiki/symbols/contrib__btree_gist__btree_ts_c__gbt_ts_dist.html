gbt_ts_dist

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_ts.c:114

目的

2つの Timestamp 値間の距離を計算し、float8 型の秒数として返します。

シグネチャまたは形状

float8 gbt_ts_dist(const void *a, const void *b, FmgrInfo *flinfo)

入力と出力

  • **入力**:
  • a, b: 比較対象の Timestamp へのポインタ（void * として渡される）。
  • flinfo: 関数管理情報（使用されません）。
  • **出力**:
  • 2つのタイムスタンプ間の差分を秒単位に換算した絶対値（float8）。

前提条件と不変条件

  • 入力された Timestamp が有限値である必要があります。
  • timestamp_mi 関数が正常に動作する環境である必要があります。

アルゴリズムまたは振る舞い

1. 入力値 a または b のいずれかが有限値でない（TIMESTAMP_NOT_FINITE）場合、無限大（get_float8_infinity()）を返します。 2. timestamp_mi 関数を呼び出し、2つのタイムスタンプの差分を Interval 型として取得します。 3. INTERVAL_TO_SEC マクロを使用して差分を秒単位に変換します。 4. fabs を適用し、その絶対値を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: ソースコード内の直接的な呼び出し元は明示されていませんが、GiSTインデックスの距離計算ロジックから参照されることが想定されます。
  • **呼び出し先**:
  • timestamp_mi: タイムスタンプの減算を実行。
  • TimestampGetDatumFast: タイムスタンプを Datum に変換。
  • DatumGetIntervalP: Datum を Interval ポインタに変換。
  • INTERVAL_TO_SEC: 差分を秒に変換。
  • fabs: 絶対値を計算。
  • get_float8_infinity: 無限大値を生成。

関連データ構造

  • Timestamp: 比較対象となるタイムスタンプ型。
  • Interval: 差分を保持するデータ構造。

失敗モードと境界条件

  • 入力値が無限大（infinity または -infinity）の場合、計算を行わずに無限大を返します。

保守担当者向けメモ

  • 本関数は timestamp 型専用です。timestamptz 型については、内部的にGMTへ正規化された後に同様のロジックが適用されます。
  • btree_utils_num.c で定義されている共通の距離計算ロジックと連携して動作します。
