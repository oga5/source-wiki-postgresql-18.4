uuid_generate_v35_internal

定義

  • 種別: function
  • 定義箇所: contrib/uuid-ossp/uuid-ossp.c:234

目的

名前空間（namespace）と名前（name）を基にした、UUID v3 または v5 の生成処理を仲介する内部関数です。

シグネチャまたは形状

static Datum uuid_generate_v35_internal(int mode, pg_uuid_t *ns, text *name)

入力と出力

  • **入力**:
  • mode: 生成する UUID のバージョン（UUID_MAKE_V3 または UUID_MAKE_V5）。
  • ns: 名前空間を表す pg_uuid_t 型のポインタ。
  • name: UUID 生成のシードとなる text 型の文字列。
  • **出力**:
  • 生成された UUID を含む Datum 型。

前提条件と不変条件

  • get_cached_uuid_t(1) により、セッション内で再利用可能な uuid_t オブジェクトが取得可能であること。
  • ns（名前空間）は有効な UUID として変換可能であること。

アルゴリズムまたは振る舞い

1. get_cached_uuid_t(1) を呼び出し、名前空間格納用の uuid_t オブジェクトを取得します。 2. uuid_out 関数と string_to_uuid を使用して、入力された pg_uuid_t 型の ns を uuid_t 形式に変換します。 3. 変換した名前空間と、text_to_cstring で変換した name を引数として uuid_generate_internal を呼び出し、UUID を生成して返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/uuid-ossp/uuid-ossp.c 内の v3/v5 生成関数（UUID_MAKE_V3 および UUID_MAKE_V5 を指定して呼び出される）。
  • **呼び出し先**:
  • get_cached_uuid_t
  • string_to_uuid
  • DirectFunctionCall1 (uuid_out)
  • text_to_cstring
  • uuid_generate_internal

関連データ構造

  • pg_uuid_t: PostgreSQL 内部の UUID 型。
  • uuid_t: OSSP UUID ライブラリの UUID 構造体。

失敗モードと境界条件

  • 入力された name が text 型として不正な場合、text_to_cstring 等でエラーが発生する可能性があります。
  • 外部ライブラリ（OSSP UUID）の内部エラーについては、uuid_generate_internal の挙動に依存します。

保守担当者向けメモ

  • 本関数は名前空間の変換と生成ロジックの橋渡しを行っています。名前空間のキャッシュ管理は get_cached_uuid_t に依存しているため、キャッシュの競合や状態管理に変更を加える際は注意が必要です。
