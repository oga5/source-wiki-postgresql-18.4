is_infinite

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/float.c:117

目的

浮動小数点値が無限大（正または負）であるかを判定し、その符号を正規化された整数値として返します。

シグネチャまたは形状

int is_infinite(double val)

入力と出力

  • **入力**: val (double型) - 判定対象の浮動小数点数。
  • **出力**: int 型の整数値。
  • 0: 有限数（無限大ではない）。
  • 1: 正の無限大。
  • -1: 負の無限大。

前提条件と不変条件

  • 標準ライブラリの isinf() 関数が利用可能であること。
  • 入力値 val は IEEE 754 浮動小数点形式に従うものと想定されます。

アルゴリズムまたは振る舞い

1. 標準の isinf(val) を呼び出し、無限大かどうかを判定します。 2. isinf が 0（偽）を返した場合、関数は 0 を返します。 3. isinf が 0 以外（真）を返した場合、val > 0 かどうかを評価します。

  • val > 0 であれば 1 を返します。
  • それ以外（val < 0）であれば -1 を返します。

呼び出し元と呼び出し先

  • **呼び出し先**: 標準ライブラリ関数 isinf()。
  • **呼び出し元**: ソースコード内の直接的な呼び出し元は src/backend/utils/adt/float.c:118 です。また、src/include/utils/float.h:43 にて外部公開用プロトタイプが宣言されています。

関連データ構造

  • 特になし。

失敗モードと境界条件

  • NaN（非数）が入力された場合の挙動は、プラットフォームの isinf() 実装に依存します。
  • val が 0 の場合、isinf(0) は 0 を返すため、本関数は正しく 0 を返します。

保守担当者向けメモ

  • 本関数はプラットフォーム間の浮動小数点挙動の差異を吸収するために使用されます。
  • float8 型の演算において、無限大の符号を判定する際の標準的な手段として利用してください。
