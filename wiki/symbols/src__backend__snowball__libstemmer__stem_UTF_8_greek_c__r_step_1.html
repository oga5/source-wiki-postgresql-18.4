目的

ギリシャ語のステミング処理における第1段階として、定義されたルールに基づき単語末尾の語尾を削除または置換する。

シグネチャまたは形状

static int r_step_1(struct SN_env * z)

入力と出力

  • 入力: SN_env 構造体（ステミング対象の文字列および状態を保持）。
  • 出力: 成功時に 1、ルールに一致しない場合に 0、メモリ操作等のエラー発生時に負の値を返す。

前提条件と不変条件

  • z が有効な SN_env 構造体を指していること。
  • 処理終了後、z->I[0] が 0 にリセットされる。

アルゴリズムまたは振る舞い

1. z->ket を現在のカーソル位置 z->c に設定する。 2. find_among_b を使用し、配列 a_1（サイズ40）に対して文字列末尾からパターンマッチングを行う。 3. 一致した among_var に応じて、以下の処理を行う：

  • z->bra を現在のカーソル位置に設定する。
  • slice_from_s を呼び出し、一致した語尾を対応する文字列に置換する。

    • 4. いずれのケースにも該当しない場合は 0 を返して終了する。 5. 成功時は z->I[0] を 0 に設定し、1 を返す。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_greek.c 内のメインステミング処理フローから呼び出される。
  • 呼び出し先: find_among_b（パターン検索）、slice_from_s（文字列置換）。

関連データ構造

  • SN_env: ステミング処理の状態管理構造体。
  • a_1: r_step_1 で使用される among 構造体の配列（ルール定義）。

失敗モードと境界条件

  • find_among_b が一致を見つけられない場合は 0 を返す。
  • slice_from_s が負の値を返した場合（メモリ不足や不正な操作）、その値をそのまま呼び出し元へ伝播させる。
