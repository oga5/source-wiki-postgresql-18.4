string_to_uuid

定義

  • 種別: function
  • 定義箇所: contrib/uuid-ossp/uuid-ossp.c:191

目的

文字列形式の UUID を OSSP UUID ライブラリの内部表現（uuid_t）に変換します。

シグネチャまたは形状

static void string_to_uuid(const char *str, uuid_t *uuid)

入力と出力

  • **入力**:
  • str: 変換対象の UUID 文字列（const char *）。
  • uuid: 変換結果を格納する uuid_t 型のポインタ。
  • **出力**: なし（戻り値は void）。

前提条件と不変条件

  • uuid_import 関数が成功することを前提としています。
  • 変換に失敗した場合は pguuid_complain を呼び出し、内部でエラー処理（例外送出等）が行われます。

アルゴリズムまたは振る舞い

1. uuid_import を呼び出し、str を UUID_FMT_STR 形式として uuid オブジェクトにインポートします。 2. 戻り値 rc が UUID_RC_OK でない場合、pguuid_complain(rc) を呼び出してエラーを報告します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/uuid-ossp/uuid-ossp.c 内の関数から呼び出されています。
  • **呼び出し先**:
  • uuid_import（OSSP UUID ライブラリ関数）
  • pguuid_complain（エラー処理用関数）
  • 注意: src/backend/utils/adt/uuid.c に同名の関数が存在しますが、本シンボルとは別の実装です。

関連データ構造

  • uuid_t: OSSP UUID ライブラリが使用する UUID オブジェクト型。
  • uuid_rc_t: OSSP UUID ライブラリの戻り値ステータス型。

失敗モードと境界条件

  • 入力文字列 str が不正な UUID 形式である場合、uuid_import がエラーを返し、pguuid_complain が実行されます。

保守担当者向けメモ

  • 本関数は OSSP UUID ライブラリに依存しています。ビルド環境で OSSP UUID が有効な場合にのみ使用されます。
  • UUID_LEN_STR は OSSP ライブラリが期待する文字列長を定義しています。
