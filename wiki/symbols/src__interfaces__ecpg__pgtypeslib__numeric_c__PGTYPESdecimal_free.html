PGTYPESdecimal_free

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/pgtypeslib/numeric.c:391

目的

decimal 型の変数に割り当てられたメモリを解放する。

シグネチャまたは形状

void PGTYPESdecimal_free(decimal *var)

入力と出力

  • 入力: decimal *var （解放対象の decimal 構造体へのポインタ）
  • 出力: なし（void）

前提条件と不変条件

  • 引数 var は、PGTYPESdecimal_new 等によって動的に確保された有効なメモリ領域を指している必要がある。
  • NULL ポインタが渡された場合の挙動は、標準ライブラリの free() の仕様に準拠する。

アルゴリズムまたは振る舞い

標準の free() 関数を呼び出し、指定された decimal 構造体のメモリ領域を解放する。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/interfaces/ecpg/test/expected/compat_informix-dec_test.c
  • src/interfaces/ecpg/test/expected/pgtypeslib-num_test2.c
  • 呼び出し先:
  • free() (標準ライブラリ)

関連データ構造

  • decimal: numeric 型に関連するクライアントサイドの数値構造体。

失敗モードと境界条件

  • 解放済みのポインタを渡した場合（二重解放）、未定義の動作を引き起こす。

保守担当者向けメモ

  • 本関数は decimal 型専用のラッパーである。numeric 型の解放には PGTYPESnumeric_free を使用すること。
