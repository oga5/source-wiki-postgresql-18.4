uuid_ns_dns

定義

  • 種別: function
  • 定義箇所: contrib/uuid-ossp/uuid-ossp.c:435

目的

DNS 名前空間を表す UUID 値を PostgreSQL の Datum 型として取得します。

シグネチャまたは形状

Datum uuid_ns_dns(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS（SQL インターフェース経由の引数）
  • **出力**: DNS 名前空間に対応する UUID を格納した Datum 型の値。

前提条件と不変条件

  • コンパイル時の環境設定（HAVE_UUID_OSSP の有無）により、内部的な取得ロジックが分岐します。

アルゴリズムまたは振る舞い

1. **OSSP UUID ライブラリ使用時**: special_uuid_value("ns:DNS") を呼び出し、OSSP ライブラリが管理する DNS 名前空間 UUID を取得します。 2. **OSSP 未使用時**: uuid_generate_internal を呼び出し、RFC 4122 で定義された DNS 名前空間 UUID (6ba7b810-9dad-11d1-80b4-00c04fd430c8) を生成して返します。

呼び出し元と呼び出し先

  • **呼び出し元**: PG_FUNCTION_INFO_V1(uuid_ns_dns) マクロにより、SQL 関数として登録されています。
  • **呼び出し先**:
  • special_uuid_value（HAVE_UUID_OSSP 定義時）
  • uuid_generate_internal（HAVE_UUID_OSSP 未定義時）

関連データ構造

  • Datum: PostgreSQL の汎用データ型。

失敗モードと境界条件

  • 提供されたソースコード内には、明示的なエラーハンドリングや境界条件の記述はありません。

保守担当者向けメモ

  • 本関数は名前空間 UUID を提供する定数的な役割を持ちます。OSSP ライブラリの有無で実装が分かれるため、変更時は両方のパスが整合性を保っているか確認してください。
