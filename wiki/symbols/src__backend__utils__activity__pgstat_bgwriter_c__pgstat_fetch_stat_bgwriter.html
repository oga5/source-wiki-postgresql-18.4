pgstat_fetch_stat_bgwriter

定義

  • 種別: function
  • 定義箇所: src/backend/utils/activity/pgstat_bgwriter.c:71

目的

バックグラウンドライター（bgwriter）の統計情報のスナップショットを取得し、SQLインターフェース等から参照可能な形式で提供します。

シグネチャまたは形状

PgStat_BgWriterStats *pgstat_fetch_stat_bgwriter(void)

入力と出力

  • 入力: なし
  • 出力: PgStat_BgWriterStats 構造体へのポインタ。この構造体には、バッファ書き出し回数、割り当て回数、統計リセット時刻などが含まれます。

前提条件と不変条件

  • 呼び出し前に統計サブシステムが初期化されている必要があります。
  • 返されるポインタは pgStatLocal.snapshot.bgwriter を指しており、呼び出しごとに最新のスナップショット内容が反映されます。

アルゴリズムまたは振る舞い

1. pgstat_snapshot_fixed(PGSTAT_KIND_BGWRITER) を呼び出し、共有メモリ上の最新の統計情報をローカルスナップショットへ同期させます。 2. 同期完了後、pgStatLocal.snapshot.bgwriter のアドレスを返します。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/backend/utils/adt/pgstatfuncs.c (SQL関数経由で統計情報を取得する複数の箇所)
  • 呼び出し先:
  • pgstat_snapshot_fixed (統計サブシステムの内部関数)

関連データ構造

  • PgStat_BgWriterStats: バックグラウンドライターの統計カウンタを保持する構造体。
  • pgStatLocal: 統計情報のローカルキャッシュを保持するグローバル変数。

失敗モードと境界条件

  • 統計サブシステムが未初期化の状態で呼び出された場合の挙動については、本ソースコード内では明示されていません。

保守担当者向けメモ

  • 本関数は pgstat_snapshot_fixed を介して共有メモリからデータを取得します。統計フィールドの追加や変更を行う場合は、PgStat_BgWriterStats 構造体および pgstat_bgwriter_snapshot_cb コールバックの定義を併せて確認してください。
