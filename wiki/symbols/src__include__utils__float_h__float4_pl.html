float4_pl

定義

  • 種別: function
  • 定義箇所: src/include/utils/float.h:145

目的

2つの float4 型の値を加算し、結果がオーバーフローした際にPostgreSQLのエラーハンドリング機構を介して通知を行う。

シグネチャまたは形状

static inline float4 float4_pl(const float4 val1, const float4 val2)

入力と出力

  • **入力**: val1 (float4), val2 (float4)
  • **出力**: 加算結果を返す。

前提条件と不変条件

  • math.h の isinf() を使用して結果を検証する。
  • 演算結果が無限大（isinf）であっても、入力値 val1 または val2 のいずれかが元々無限大である場合は、オーバーフローとはみなさない。

アルゴリズムまたは振る舞い

1. val1 と val2 を加算し、結果を result に格納する。 2. result が無限大であり、かつ val1 と val2 の双方が無限大ではない場合、演算結果がオーバーフローしたと判断する。 3. オーバーフロー時は float_overflow_error() を呼び出し、処理を中断する。 4. 正常な場合は result を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/adt/float.c:733 (PG_RETURN_FLOAT4 マクロ内)
  • **呼び出し先**: float_overflow_error() (エラー発生時)

関連データ構造

  • float4: PostgreSQLの単精度浮動小数点型。

失敗モードと境界条件

  • **オーバーフロー**: 加算結果が float4 の表現範囲を超えて無限大となった場合、float_overflow_error() によりエラーが報告される。

保守担当者向けメモ

  • 本関数はインライン関数として定義されており、パフォーマンスを重視している。
  • エラー報告関数 float_overflow_error は src/backend/utils/adt/float.c に実装されている。
