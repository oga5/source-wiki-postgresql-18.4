r_remove_particle

定義

  • **種別**: function
  • **定義箇所**: src/backend/snowball/libstemmer/stem_ISO_8859_1_indonesian.c:123

目的

インドネシア語のステミング処理において、単語末尾から特定の「粒子（particle）」を除去し、語幹抽出を促進する。

シグネチャまたは形状

static int r_remove_particle(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z（Snowballステミング環境構造体）
  • **出力**: int 型。成功時は 1、除去対象が見つからない場合や処理が適用されない場合は 0、エラー発生時は負の値を返す。

前提条件と不変条件

  • z->c（現在のカーソル位置）が有効な範囲内であること。
  • a_0 配列に定義された粒子パターンが対象となる。
  • 処理後、環境内の整数ワークスペース z->I[1]（単語内の母音数を示すフラグ）がデクリメントされる。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置を z->ket に設定する。 2. 単語の末尾が特定の文字（ASCII 104='h' または 110='n'）であるか、または十分な長さがあるかを確認する。条件を満たさない場合は 0 を返す。 3. find_among_b を使用し、a_0 配列に基づいて末尾から粒子を検索する。 4. 粒子が見つかった場合、その位置を z->bra に設定し、slice_del で該当部分を削除する。 5. 削除成功後、単語内の母音数カウント z->I[1] を 1 減らし、1 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_ISO_8859_1_indonesian.c および src/backend/snowball/libstemmer/stem_UTF_8_indonesian.c 内のステミングメイン処理から呼び出される。
  • **呼び出し先**:
  • find_among_b（Snowballランタイムユーティリティ）
  • slice_del（Snowballランタイムユーティリティ）

関連データ構造

  • struct SN_env: ステミング処理の状態を保持する構造体。
  • a_0: 除去対象となる粒子パターンを格納した among 構造体の配列。

失敗モードと境界条件

  • 単語が短すぎる場合や、末尾が対象文字（'h', 'n'）でない場合は即座に 0 を返す。
  • slice_del が内部エラー（負の値）を返した場合、その値をそのまま伝播させる。

保守担当者向けメモ

  • 本関数は自動生成コードの一部である。アルゴリズムの変更が必要な場合は、直接編集せず元のSnowballスクリプト（.sbl）を修正すること。
  • z->I[1] の更新は、後続のステミング処理における母音数判定に影響を与えるため、整合性に注意が必要である。
