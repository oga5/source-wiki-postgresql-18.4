WindowStatePerFuncData

定義

  • 種別: struct
  • 定義箇所: src/backend/executor/nodeWindowAgg.c:79

目的

ウィンドウ関数実行ノードにおいて、各ウィンドウ関数の実行状態およびメタデータを保持するために使用される。

シグネチャまたは形状

typedef struct WindowStatePerFuncData
{
    WindowFuncExprState *wfuncstate;
    WindowFunc *wfunc;
    int         numArguments;
    FmgrInfo    flinfo;
    Oid         winCollation;
    int16       resulttypeLen;
    bool        resulttypeByVal;
    bool        plain_agg;
    int         aggno;
    WindowObject winobj;
} WindowStatePerFuncData;

入力と出力

  • **入力**: WindowFunc および WindowFuncExprState への参照を保持し、関数実行に必要な引数情報や照合順序（winCollation）を管理する。
  • **出力**: WindowObject を介してウィンドウ関数APIへコンテキストを提供し、計算結果の型情報（resulttypeLen, resulttypeByVal）に基づいて値のコピーや削除を制御する。

前提条件と不変条件

  • plain_agg が真の場合、aggno は対応する WindowStatePerAggData のインデックスを指す必要がある。
  • flinfo は関数呼び出しのための有効な FmgrInfo 構造体である必要がある。

アルゴリズムまたは振る舞い

  • ウィンドウ関数の実行時状態をカプセル化し、パーティション内での関数評価をサポートする。
  • 集約関数がウィンドウ関数として使用される場合、plain_agg フラグにより通常の集約処理との区別を行う。
  • WindowObject を通じて、ウィンドウ関数がパーティション内のデータにアクセスするためのAPIインターフェースを提供する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/executor/nodeWindowAgg.c 内の初期化処理（palloc0 によるメモリ確保）にて参照される。
  • **呼び出し先**: 構造体内部で WindowFunc や WindowFuncExprState を保持するが、具体的な関数呼び出しの連鎖についてはソース抜粋からは不明。

関連データ構造

  • WindowFunc: 式ツリー内のウィンドウ関数定義。
  • WindowFuncExprState: 実行時の式評価状態。
  • WindowStatePerAggData: 集約関数として実行される場合の関連データ。
  • WindowObject: ウィンドウ関数APIで使用されるコンテキストオブジェクト。

失敗モードと境界条件

  • numArguments が不整合な場合、関数呼び出し時にエラーが発生する可能性がある。
  • メモリ割り当て失敗時に palloc0 が例外を投げる可能性がある。

保守担当者向けメモ

  • 本構造体は WindowAggState の一部として管理される。
  • resulttypeLen および resulttypeByVal は、関数結果のメモリ管理（コピー/削除）に不可欠であるため、型変更時には注意が必要。
