macaddr_lt

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/mac.c:209

目的

2つの macaddr 型の値が「より小さい（less than）」という関係にあるかを判定する。

シグネチャまたは形状

Datum macaddr_lt(PG_FUNCTION_ARGS)

入力と出力

  • 入力: PG_FUNCTION_ARGS を介して2つの macaddr 型へのポインタ（a1, a2）を受け取る。
  • 出力: a1 < a2 であれば true、そうでなければ false を Datum 型として返す。

前提条件と不変条件

  • 入力される macaddr ポインタは有効であること。
  • 内部比較ロジックとして macaddr_cmp_internal を使用する。

アルゴリズムまたは振る舞い

1. PG_GETARG_MACADDR_P マクロを使用して、引数から2つの macaddr 構造体へのポインタを取得する。 2. macaddr_cmp_internal(a1, a2) を呼び出し、比較結果を取得する。 3. 比較結果が 0 未満（a1 < a2）である場合に true を、それ以外の場合に false を返す。

呼び出し元と呼び出し先

  • 呼び出し元:
  • contrib/btree_gist/btree_macaddr.c:56（直接呼び出し）
  • 呼び出し先:
  • macaddr_cmp_internal

関連データ構造

  • macaddr: 6バイト（EUI-48）のMACアドレスを保持する構造体。

失敗モードと境界条件

  • 入力が NULL の場合、PostgreSQL の関数呼び出し規約に従い、呼び出し元で適切に処理される必要がある。

保守担当者向けメモ

  • 本関数は標準的な比較演算子 < を実装している。比較ロジックの変更が必要な場合は、macaddr_cmp_internal を修正すること。
