searchstoplist

定義

  • 種別: function
  • 定義箇所: src/backend/tsearch/ts_utils.c:140

目的

指定された単語がストップワードリストに含まれているかを高速に判定する。

シグネチャまたは形状

bool searchstoplist(StopList *s, char *key)

入力と出力

  • **入力**:
  • s: 検索対象の StopList 構造体へのポインタ。
  • key: 検索する単語（文字列）。
  • **出力**:
  • bool: 単語がリスト内に存在する場合は true、存在しない場合は false を返す。

前提条件と不変条件

  • s->stop に格納された単語リストは、bsearch によるバイナリサーチが可能なようにソート済みである必要がある。
  • s が指す StopList 構造体は有効であること。

アルゴリズムまたは振る舞い

1. s->stop が非 NULL かつ s->len > 0 であることを確認する。 2. bsearch 関数を用い、pg_qsort_strcmp を比較関数として指定して、key がリスト内に存在するかを検索する。 3. 検索結果が NULL でなければ true を、NULL であれば false を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/snowball/dict_snowball.c
  • src/backend/tsearch/dict_ispell.c
  • src/backend/tsearch/dict_simple.c
  • **呼び出し先**:
  • bsearch (標準ライブラリ)
  • pg_qsort_strcmp (比較関数)

関連データ構造

  • StopList: src/include/tsearch/ts_public.h で定義される、ストップワードリストを保持する構造体。

失敗モードと境界条件

  • s->stop が NULL の場合、または s->len が 0 の場合は、検索を行わず直ちに false を返す。

保守担当者向けメモ

  • 本関数はバイナリサーチに依存しているため、リストのソート順序が pg_qsort_strcmp の期待する順序と一致していることが不可欠である。リストの構築については readstoplist を参照のこと。
