ecpg_hex_dec_len

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/ecpglib/data.c:133

目的

16進数エンコードされた文字列の長さを基に、デコード後のバイナリデータの長さを算出します。

シグネチャまたは形状

unsigned ecpg_hex_dec_len(unsigned srclen)

入力と出力

  • **入力**: srclen (unsigned) - 16進数文字列の長さ。
  • **出力**: unsigned - デコード後のバイト数。

前提条件と不変条件

  • 入力値 srclen は、16進数表現の文字列長として与えられることを想定しています。

アルゴリズムまたは振る舞い

入力された長さ srclen を右に1ビットシフト（2で除算）することで、16進数2文字が1バイトに変換されることを前提としたデコード後の長さを返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/interfaces/ecpg/ecpglib/data.c:536 にて、size - 2 を引数として呼び出されています。
  • **呼び出し先**:
  • なし。

関連データ構造

  • なし。

失敗モードと境界条件

  • 入力が奇数の場合、右シフトにより端数が切り捨てられます。

保守担当者向けメモ

  • ecpg_hex_enc_len と対になる関数です。16進数文字列からバイナリへの変換サイズを計算する際に使用されます。
