show_item

定義

  • 種別: function
  • 定義箇所: src/bin/pg_config/pg_config.c:114

目的

指定された設定項目名（configname）に対応する設定値を ConfigData 配列から検索し、標準出力へ書き出す。

シグネチャまたは形状

static void show_item(const char *configname, ConfigData *configdata, size_t configdata_len)

入力と出力

  • **入力**:
  • configname: 検索対象の設定項目名（文字列）。
  • configdata: 設定情報の配列（ConfigData 構造体のポインタ）。
  • configdata_len: configdata 配列の要素数。
  • **出力**: なし（戻り値は void）。一致した項目がある場合、その設定値（setting）を標準出力に書き出す。

前提条件と不変条件

  • configdata は有効な ConfigData 配列を指している必要がある。
  • configdata_len は配列の実際のサイズと一致している必要がある。

アルゴリズムまたは振る舞い

1. configdata_len 回のループを実行する。 2. 各反復において、configdata[i].name と configname を strcmp で比較する。 3. 一致した場合、対応する configdata[i].setting を printf で出力する。 4. 一致する項目が複数存在する場合、すべて出力される。

呼び出し元と呼び出し先

  • **呼び出し元**: src/bin/pg_config/pg_config.c 内の main 関数から呼び出される（173行目付近）。
  • **呼び出し先**: 標準ライブラリの strcmp および printf を使用。

関連データ構造

  • ConfigData: common/config_info.h で定義される、設定項目名と設定値を保持する構造体。

失敗モードと境界条件

  • configname に一致する項目が configdata 内に存在しない場合、何も出力されずに終了する。
  • configdata_len が 0 の場合、ループは実行されず何も出力されない。

保守担当者向けメモ

  • 本関数は info_items テーブルと連携して動作する。新しい設定項目を追加する際は、configdata の生成ロジックと整合性が取れていることを確認すること。
