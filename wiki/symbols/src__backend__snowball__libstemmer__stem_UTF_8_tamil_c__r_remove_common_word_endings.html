r_remove_common_word_endings

定義

  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_tamil.c:1066

目的

タミル語の単語末尾から一般的な接尾辞を除去し、語幹抽出を促進する。

シグネチャまたは形状

static int r_remove_common_word_endings(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング環境構造体）
  • 出力: 成功時に 1、失敗または処理中断時に 0 または負の整数（エラーコード）を返す。

前提条件と不変条件

  • r_has_min_length(z) が真であること。
  • SN_env 構造体の lb（下限）および c（カーソル位置）が適切に設定されていること。

アルゴリズムまたは振る舞い

1. r_has_min_length を呼び出し、最小長を満たさない場合は終了する。 2. カーソルを単語末尾に移動し、a_17 配列を用いて後方一致検索（find_among_b）を行う。 3. マッチした結果（among_var）に応じて以下の処理を行う：

  • 1: s_24 に置換。
  • 2: a_16 配列との照合を試み、マッチしなければ s_25 に置換。
  • 3: 文字列を削除（slice_del）。

    • 4. 最後に r_fix_endings を呼び出し、語幹の整合性を調整する。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_tamil.c 内のメイン処理フローから参照されている。
  • 呼び出し先: r_has_min_length, find_among_b, slice_from_s, slice_del, r_fix_endings。

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_17, a_16: 接尾辞パターンを定義した among 構造体の配列。

失敗モードと境界条件

  • r_has_min_length が 0 以下を返した場合、処理を中断する。
  • find_among_b がマッチしない場合、0 を返して終了する。
  • slice_from_s や slice_del が負の値を返した場合、そのエラーコードを伝播させる。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによって自動生成されたコードである。ロジックの変更が必要な場合は、直接編集せず元の .sbl ファイルを修正すること。
  • among 配列内の数値は UTF-8 バイトシーケンスに対応しているため、デバッグ時はバイト単位の照合状況を確認すること。
