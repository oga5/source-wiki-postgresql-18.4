RemoveRoleFromInitPriv

定義

  • 種別: function
  • 定義箇所: src/backend/catalog/aclchk.c:4867

目的

システムカタログ pg_init_privs に格納された初期権限情報から、指定されたロールに関連する権限エントリを削除または更新する。

シグネチャまたは形状

void RemoveRoleFromInitPriv(Oid roleid, Oid classid, Oid objid, int32 objsubid)

入力と出力

  • **入力**:
  • roleid: 削除対象のロールOID。
  • classid: 対象オブジェクトのカタログOID。
  • objid: 対象オブジェクトのOID。
  • objsubid: 対象オブジェクトのサブID（列番号等）。
  • **出力**: なし（副作用として pg_init_privs カタログおよび共有依存関係を更新）。

前提条件と不変条件

  • pg_init_privs テーブルに対する RowExclusiveLock を保持していること。
  • 対象オブジェクトの所有者情報をシステムキャッシュから取得可能であること。

アルゴリズムまたは振る舞い

1. pg_init_privs をスキャンし、指定された objid, classid, objsubid に合致するエントリを検索する。 2. エントリが存在しない場合は処理を終了する。 3. 既存の ACL 情報を取得し、merge_acl_with_grant を用いて指定ロールの権限を剥奪した新しい ACL を生成する。 4. 新しい ACL が空であれば CatalogTupleDelete でエントリを削除し、空でなければ CatalogTupleUpdate で更新する。 5. updateInitAclDependencies を呼び出し、共有依存関係（pg_shdepend）を更新する。 6. CommandCounterIncrement() を実行し、変更を確定させる。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/catalog/pg_shdepend.c:1496
  • **呼び出し先**:
  • table_open, table_close
  • systable_beginscan, systable_getnext, systable_endscan
  • merge_acl_with_grant
  • updateInitAclDependencies
  • CatalogTupleDelete, CatalogTupleUpdate
  • CommandCounterIncrement

関連データ構造

  • pg_init_privs (システムカタログ)
  • Acl (アクセス制御リスト構造体)

失敗モードと境界条件

  • 対象オブジェクトのシステムキャッシュ検索に失敗した場合、elog(ERROR) を発生させる。
  • 指定されたオブジェクトに対する pg_init_privs エントリが存在しない場合、エラーにはならず静かに終了する。

保守担当者向けメモ

  • 本関数は pg_upgrade 等の特殊な状況下で初期権限を管理するために使用される。
  • merge_acl_with_grant の呼び出しにおいて、所有者を権限付与者として固定している点に注意すること。
