hmac_block_size

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/px-hmac.c:45

目的

HMAC 計算に使用されるハッシュアルゴリズムのブロックサイズを取得します。

シグネチャまたは形状

static unsigned hmac_block_size(PX_HMAC *h)

入力と出力

  • **入力**: PX_HMAC *h （HMAC コンテキストへのポインタ）
  • **出力**: unsigned （ハッシュアルゴリズムのブロックサイズ）

前提条件と不変条件

  • 引数 h は有効な PX_HMAC 構造体を指している必要があります。
  • 内部の h->md（メッセージダイジェストコンテキスト）が初期化されている必要があります。

アルゴリズムまたは振る舞い

PX_HMAC 構造体内に保持されているダイジェストコンテキスト h->md を参照し、px_md_block_size 関数を呼び出してそのブロックサイズを返します。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/pgcrypto/px-hmac.c:166 において、PX_HMAC 構造体の関数ポインタフィールド block_size に本関数が割り当てられます。
  • **呼び出し先**: px_md_block_size （px レイヤーのダイジェスト抽象化関数）。

関連データ構造

  • PX_HMAC: HMAC 操作の状態を保持する構造体。本関数はこの構造体内の md フィールドを利用します。

失敗モードと境界条件

  • h が NULL の場合、デリファレンスによりクラッシュする可能性があります。

保守担当者向けメモ

  • 本関数は PX_HMAC インターフェースの一部として実装されています。
  • px_md_block_size が返す値は、使用するハッシュアルゴリズム（SHA-1, SHA-256 等）に依存します。
