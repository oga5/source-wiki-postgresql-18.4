PQgetSSLKeyPassHook_OpenSSL

定義

  • 種別: function
  • 定義箇所: src/interfaces/libpq/fe-secure-openssl.c:1921

目的

現在設定されている OpenSSL 用の SSL 秘密鍵パスワード取得フック関数を取得します。

シグネチャまたは形状

PQsslKeyPassHook_OpenSSL_type PQgetSSLKeyPassHook_OpenSSL(void)

入力と出力

  • 入力: なし
  • 出力: 現在有効な PQsslKeyPassHook_OpenSSL_type 型の関数ポインタを返します。

前提条件と不変条件

  • PQsslKeyPassHook グローバル変数が初期化されている必要があります。

アルゴリズムまたは振る舞い

  • グローバル変数 PQsslKeyPassHook の値をそのまま返します。

呼び出し元と呼び出し先

  • 呼び出し元: src/interfaces/libpq/fe-secure.c:461 からの呼び出しが確認されています。
  • 呼び出し先: なし（直接的な関数呼び出しは行いません）。

関連データ構造

  • PQsslKeyPassHook_OpenSSL_type: パスワード取得フック関数の型定義。

失敗モードと境界条件

  • 特記事項なし。
