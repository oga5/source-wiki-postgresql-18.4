UUIDPGetDatum

定義

  • 種別: static inline function
  • 定義箇所: src/include/utils/uuid.h:26

目的

pg_uuid_t 構造体へのポインタを、PostgreSQLの関数マネージャ（fmgr）が扱う Datum 型へ変換する。

シグネチャまたは形状

static inline Datum UUIDPGetDatum(const pg_uuid_t *X)

入力と出力

  • **入力**: const pg_uuid_t *X （UUIDデータを保持する構造体へのポインタ）
  • **出力**: Datum （PostgreSQLの実行エンジンで受け渡し可能な汎用データ型）

前提条件と不変条件

  • 入力ポインタ X は有効な pg_uuid_t 構造体を指している必要がある。
  • pg_uuid_t は固定長（16バイト）のデータ構造である。

アルゴリズムまたは振る舞い

内部的に PointerGetDatum(X) を呼び出し、ポインタ値を Datum 型にキャストして返却する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/btree_gin/btree_gin.c
  • contrib/uuid-ossp/uuid-ossp.c
  • src/backend/utils/adt/uuid.c
  • src/include/utils/uuid.h （PG_RETURN_UUID_P マクロ経由）
  • **呼び出し先**:
  • PointerGetDatum （PostgreSQLのコアマクロ）

関連データ構造

  • pg_uuid_t: UUIDデータを保持する構造体。
  • Datum: PostgreSQLの関数間でデータを受け渡すための汎用型。

失敗モードと境界条件

  • 入力ポインタが NULL の場合、PointerGetDatum の挙動に依存する。

保守担当者向けメモ

  • 本関数はインライン定義されており、パフォーマンスを考慮して設計されている。
  • UUID型の実装を変更する場合、uuid.c 側のロジックを優先し、本関数のシグネチャ変更は避けること。
