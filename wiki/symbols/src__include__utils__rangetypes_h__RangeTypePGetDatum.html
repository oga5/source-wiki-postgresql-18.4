RangeTypePGetDatum

定義

  • 種別: static inline 関数
  • 定義箇所: src/include/utils/rangetypes.h:85

目的

RangeType 構造体へのポインタを、PostgreSQLの汎用データ型である Datum 形式に変換する。

シグネチャまたは形状

static inline Datum RangeTypePGetDatum(const RangeType *X)

入力と出力

  • **入力**: const RangeType *X （範囲型のメモリ表現へのポインタ）
  • **出力**: Datum （変換されたデータム値）

前提条件と不変条件

  • 入力ポインタ X は有効な RangeType 構造体を指している必要がある。
  • 本関数は PointerGetDatum をラップしたものであり、メモリ上の配置やアライメントは RangeType の仕様に従う。

アルゴリズムまたは振る舞い

PointerGetDatum マクロを呼び出し、RangeType ポインタを Datum 型へキャストして返却する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/multirangetypes.c (複数箇所)
  • src/backend/utils/adt/rangetypes.c
  • src/backend/utils/adt/rangetypes_gist.c (複数箇所)
  • **呼び出し先**:
  • PointerGetDatum (マクロ)

関連データ構造

  • RangeType: 範囲型のメモリ表現（可変長データ）。

失敗モードと境界条件

  • 入力ポインタが NULL の場合、PointerGetDatum の挙動に依存する。

保守担当者向けメモ

  • 本関数は単なる型変換のラッパーであるため、ロジックの変更は不要。
  • RangeType は可変長データ（varlena）として扱われるため、本関数で変換した Datum を使用する際は、必要に応じてデトースト処理が行われるコンテキストであることを確認すること。
