bbsink_shell_archive_contents

定義

  • 種別: function
  • 定義箇所: contrib/basebackup_to_shell/basebackup_to_shell.c:321

目的

バックアップのアーカイブデータを外部シェルコマンドへ転送し、同時に後続のシンクへデータを転送する。

シグネチャまたは形状

static void bbsink_shell_archive_contents(bbsink *sink, size_t len)

入力と出力

  • sink: bbsink 構造体へのポインタ。bbsink_shell 型としてキャストして使用される。
  • len: 転送するデータのバイト長。
  • 出力: なし（戻り値なし）。

前提条件と不変条件

  • sink は bbsink_shell 構造体として適切に初期化されている必要がある。
  • shell_send_data が内部で呼び出されるため、パイプが既に開かれている状態である必要がある。

アルゴリズムまたは振る舞い

1. sink を bbsink_shell 型にキャストする。 2. shell_send_data を呼び出し、指定された長さのデータを外部プロセスへパイプ経由で書き込む。 3. bbsink_forward_archive_contents を呼び出し、データを後続のシンクへ転送する。

呼び出し元と呼び出し先

  • 呼び出し元: bbsink_shell_ops 構造体（関数ポインタテーブル）を通じて、バックアップ処理のメインループから呼び出される。
  • 呼び出し先:
  • shell_send_data
  • bbsink_forward_archive_contents

関連データ構造

  • bbsink: バックアップターゲットフレームワークの基底構造体。
  • bbsink_shell: 外部コマンド実行状態を保持する拡張構造体。

失敗モードと境界条件

  • shell_send_data 内部でパイプへの書き込みエラーが発生した場合、バックアップ全体がエラーとして扱われる可能性がある。

保守担当者向けメモ

  • 本関数は bbsink インターフェースの一部であり、チェーンされたシンクへのデータ伝播を担う。変更時は bbsink_forward_archive_contents の呼び出し順序を維持すること。
