PGTYPEStimestamp_fmt_asc

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/pgtypeslib/timestamp.c:777

目的

timestamp 型の値を、指定されたフォーマット文字列に従って文字列へ変換します。

シグネチャまたは形状

int PGTYPEStimestamp_fmt_asc(timestamp *ts, char *output, int str_len, const char *fmtstr);

入力と出力

  • **入力**:
  • ts: 変換対象の timestamp 型ポインタ。
  • output: 結果を格納する文字バッファ。
  • str_len: output バッファのサイズ。
  • fmtstr: フォーマットを指定する文字列。
  • **出力**:
  • 成功時は 0 を返し、output にフォーマットされた文字列が格納されます。失敗時の戻り値についてはソースコードから明示されていません。

前提条件と不変条件

  • ts は有効な timestamp 値を指している必要があります。
  • output は str_len 分の書き込みが可能な領域である必要があります。

アルゴリズムまたは振る舞い

1. PGTYPESdate_from_timestamp を呼び出し、タイムスタンプから日付（date）を抽出します。 2. PGTYPESdate_dayofweek を呼び出し、曜日（dow）を算出します。 3. timestamp2tm を呼び出し、タイムスタンプを struct tm 構造体および小数秒（fsec_t）に分解します。 4. dttofmtasc_replace を呼び出し、抽出した各情報とフォーマット文字列を用いて output バッファへ文字列を書き込みます。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/interfaces/ecpg/compatlib/informix.c
  • src/interfaces/ecpg/test/expected/pgtypeslib-dt_test.c
  • **呼び出し先**:
  • PGTYPESdate_from_timestamp
  • PGTYPESdate_dayofweek
  • timestamp2tm
  • dttofmtasc_replace

関連データ構造

  • timestamp: 内部的に int64（マイクロ秒単位）として扱われるデータ型。
  • struct tm: C標準の日時構造体。
  • fsec_t: 小数秒を保持する型。
  • date: 日付を保持する型。

失敗モードと境界条件

  • str_len が不足している場合の挙動は dttofmtasc_replace の実装に依存します。
  • タイムスタンプ値が特殊な場合（infinity 等）の処理は timestamp2tm の内部ロジックに依存します。

保守担当者向けメモ

  • 本関数は dttofmtasc_replace を介してフォーマット処理を行っています。ロケール依存のフォーマット指定子（%a, %A 等）の挙動については、モジュール概要の注意書きを参照してください。
