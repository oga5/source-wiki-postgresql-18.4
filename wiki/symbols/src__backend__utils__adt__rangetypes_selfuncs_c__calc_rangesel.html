calc_rangesel

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/rangetypes_selfuncs.c:230

目的

範囲型に対するクエリの選択率（Selectivity）を、統計情報（ヒストグラム、NULL率、空範囲率）に基づいて推定する。

シグネチャまたは形状

static double calc_rangesel(TypeCacheEntry *typcache, VariableStatData *vardata, const RangeType *constval, Oid operator);

入力と出力

  • **入力**:
  • typcache: 型固有のキャッシュ情報。
  • vardata: 対象列の統計情報（pg_statistic を含む）。
  • constval: 比較対象の定数範囲値。
  • operator: 評価対象の範囲演算子（OID）。
  • **出力**:
  • double: 推定された選択率（0.0 から 1.0 の範囲）。

前提条件と不変条件

  • 範囲の長さと下限値は独立していると仮定して計算される。
  • 戻り値は CLAMP_PROBABILITY マクロにより必ず [0.0, 1.0] に収まる。
  • 統計情報が不足している場合、NULL率や空範囲率は 0.0 として扱われる。

アルゴリズムまたは振る舞い

1. **統計取得**: vardata から NULL 率と空範囲率を取得する。 2. **空範囲の処理**: constval が空の場合、演算子に応じて固定値（0.0, 1.0, または empty_frac）を返す。 3. **非空範囲の処理**:

  • calc_hist_selectivity を呼び出し、境界ヒストグラムを用いた選択率を計算する。
  • 計算に失敗した場合は default_range_selectivity にフォールバックする。
  • 演算子が OID_RANGE_CONTAINED_OP の場合、空範囲の割合を考慮して結果を補正する。

    • 4. **最終補正**: NULL 率を考慮して最終的な選択率を調整する。

呼び出し元と呼び出し先

  • **呼び出し元**: rangesel（src/backend/utils/adt/rangetypes_selfuncs.c 内の別箇所）。
  • **呼び出し先**:
  • get_attstatsslot / free_attstatsslot
  • RangeIsEmpty
  • calc_hist_selectivity
  • default_range_selectivity

関連データ構造

  • TypeCacheEntry: 型の比較関数やサブ差分関数を保持。
  • VariableStatData: 統計情報のコンテナ。
  • AttStatsSlot: pg_statistic から抽出されたスロットデータ。

失敗モードと境界条件

  • sslot.nnumbers != 1: 統計情報が予期せぬ形式の場合、elog(ERROR) を発生させる。
  • operator が未知の場合: elog(ERROR) を発生させる。
  • 統計情報が欠如している場合: デフォルト値を使用して推定を続行する。

保守担当者向けメモ

  • 本関数は「範囲の長さ」と「下限値」が独立しているという仮定に依存している。
  • カスタム範囲型を追加する際は、型キャッシュ内の subdiff 関数が正しく実装されているか確認すること。
  • 統計情報がない場合でも、定数が空範囲かどうかによる最低限の推定は行われる。
