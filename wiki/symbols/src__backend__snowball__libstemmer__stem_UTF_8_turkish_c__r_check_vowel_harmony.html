目的

トルコ語の母音調和ルールに基づき、単語末尾の母音が適切なグループに属しているかを検証します。

入力と出力

  • **入力**: SN_env * z（Snowball ステミング環境構造体）。z->c（カーソル位置）および z->p（文字列バッファ）を操作対象とします。
  • **出力**: int 型。母音調和の条件を満たす場合は 1 を返し、満たさない場合は 0 を返します。

前提条件と不変条件

  • z は有効な SN_env 構造体である必要があります。
  • 関数実行前後で、カーソル位置 z->c は元の位置に復元されます（m_test1 による保存と復元）。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置から後方（z->lb 方向）に向かって、母音グループ（g_vowel）を検索します。 2. 見つかった母音に応じて、以下のいずれかの条件を検証します：

  • 直前の文字が 'a' であり、かつ g_vowel1 に属する母音があるか。
  • 直前の文字が 'e' であり、かつ g_vowel2 に属する母音があるか。
  • 特定の文字列（s_0, s_1, s_2）との一致を確認し、対応する母音グループ（g_vowel3 ～ g_vowel6）を検証する。

    • 3. 複数の分岐（lab1 ～ lab7）を順次試し、いずれかの母音調和ルールに合致すれば成功（1）を返します。 4. どのルールにも合致しない場合は失敗（0）を返します。

  • g_vowel ～ g_vowel6: 母音のグループ定義。

失敗モードと境界条件

  • カーソルがバッファの境界（z->lb）に達した場合、検索は停止し失敗（0）を返します。
  • 該当する母音グループが見つからない場合、または指定された文字列と一致しない場合は 0 を返します。
