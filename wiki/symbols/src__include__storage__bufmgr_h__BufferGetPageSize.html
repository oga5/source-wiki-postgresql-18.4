BufferGetPageSize

定義

  • 種別: function
  • 定義箇所: src/include/storage/bufmgr.h:405

目的

バッファが保持するページのサイズ（バイト単位）を取得します。

シグネチャまたは形状

static inline Size BufferGetPageSize(Buffer buffer)

入力と出力

  • **入力**: buffer (型: Buffer) - 対象となるバッファの識別子。
  • **出力**: Size - ページサイズ。定数 BLCKSZ を返します。

前提条件と不変条件

  • buffer は有効なバッファである必要があります（BufferIsValid(buffer) が真であること）。

アルゴリズムまたは振る舞い

1. Assert マクロにより、引数 buffer が有効であることを検証します。 2. システム定義の定数 BLCKSZ を Size 型にキャストして返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/access/gin/ginutil.c
  • src/backend/access/hash/hash_xlog.c
  • src/backend/access/hash/hashovfl.c
  • src/backend/access/hash/hashpage.c
  • src/backend/access/heap/heapam_xlog.c
  • **呼び出し先**:
  • BufferIsValid (マクロ/インライン関数)

関連データ構造

  • Buffer: バッファを識別する整数型。
  • BLCKSZ: ページサイズを規定する定数。

失敗モードと境界条件

  • buffer が無効な場合、Assert が有効なビルドではプログラムが停止します。

保守担当者向けメモ

  • 本関数はインライン化されており、主にページ初期化処理（PageInit や _hash_pageinit）の引数として利用されます。
  • BLCKSZ はコンパイル時に決定される定数であるため、実行時のバッファ状態に関わらず常に一定の値を返します。
