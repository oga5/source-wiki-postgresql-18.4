sigset_t

定義

  • typedef int sigset_t;
  • 定義箇所: src/include/libpq/pqsignal.h:20

目的

Windows環境において、POSIX準拠のシグナルセット操作APIをエミュレートするために使用される整数型です。

シグネチャまたは形状

  • int 型のエイリアス。

入力と出力

  • 入力: シグナルマスクのビット表現を保持する整数値。
  • 出力: pqsigprocmask 等のシグナル操作関数において、シグナルセットの状態を保持・伝達するために使用されます。

前提条件と不変条件

  • Windows環境でのシグナル処理の抽象化レイヤーとして機能します。
  • POSIX環境における sigset_t と同等の役割を果たすことが期待されます。

アルゴリズムまたは振る舞い

  • 本シンボル自体は型定義であり、具体的なロジックは pqsigprocmask などの関数によって実装されます。
  • Windows環境下でシグナルをブロックまたは許可する際のビットマスクとして利用されます。

呼び出し元と呼び出し先

  • 呼び出し元（参照箇所）:
  • src/backend/libpq/pqsignal.c
  • src/backend/port/win32/signal.c
  • src/backend/postmaster/fork_process.c
  • src/backend/storage/ipc/dsm_impl.c
  • src/backend/storage/ipc/waiteventset.c
  • src/bin/psql/command.c
  • 呼び出し先: 情報なし。

関連データ構造

  • struct sigaction: シグナルハンドラ設定用構造体。内部で sigset_t sa_mask を保持します。

失敗モードと境界条件

  • Windows環境以外での挙動については、本ソースの文脈からは明示されていません。
  • 整数型であるため、保持できるシグナルの数は int 型のビット幅に依存します。

保守担当者向けメモ

  • 本シンボルはWindows環境での移植性確保を主目的としています。
  • pqsignal.h 内のコメントにある通り、POSIX APIのエミュレーション用である点に留意してください。
