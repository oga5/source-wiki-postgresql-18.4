pg_popcount32_choose

定義

  • 種別: function
  • 定義箇所: src/port/pg_bitutils.c:187

目的

実行時のCPU機能判定に基づき、最適な32ビットポピュレーションカウント実装へ切り替えるための初期化関数です。

シグネチャまたは形状

int pg_popcount32_choose(uint32 word)

入力と出力

  • 入力: uint32 word （ビット数をカウントする対象の32ビット整数）
  • 出力: int （入力値に含まれる1のビット数）

前提条件と不変条件

  • 本関数は、pg_popcount32 関数ポインタが初期状態（pg_popcount32_choose を指している状態）の時に呼び出されます。
  • 呼び出し後、pg_popcount32 ポインタはハードウェア最適化版または汎用版のいずれかに再割り当てされます。

アルゴリズムまたは振る舞い

1. choose_popcount_functions() を呼び出し、CPUの POPCNT 命令サポート状況を判定して、関数ポインタ pg_popcount32 を最適な実装へ書き換えます。 2. 書き換えられた pg_popcount32 を使用して、引数 word のポピュレーションカウントを実行し、その結果を返します。

呼び出し元と呼び出し先

  • 呼び出し元: pg_popcount32 関数ポインタ経由で呼び出されます（直接の呼び出し元はソースコード上では明示されていません）。
  • 呼び出し先:
  • choose_popcount_functions()
  • pg_popcount32 (再割り当て後の関数ポインタ)

関連データ構造

  • pg_popcount32: 本関数への初期値を持つ関数ポインタ。

失敗モードと境界条件

  • 特になし。入力値がどのような32ビット整数であっても、常に正しいビット数を返します。

保守担当者向けメモ

  • 本関数は動的ディスパッチのトリガーとして機能します。初回呼び出し以降は、関数ポインタが直接最適化実装を指すため、本関数はバイパスされます。
