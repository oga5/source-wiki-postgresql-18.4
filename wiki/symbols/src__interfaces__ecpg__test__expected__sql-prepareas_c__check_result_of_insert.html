目的

ECPG回帰テストにおいて、データベースの test テーブルからデータを取得し、その結果を標準出力へ表示して検証を行うヘルパー関数です。

シグネチャまたは形状

static void check_result_of_insert(void)

入力と出力

  • **入力**: なし（関数内で ivar1, ivar2 を宣言し、SQLクエリの結果を格納する）。
  • **出力**: なし（副作用として printf による標準出力への書き出しを行う）。

前提条件と不変条件

  • データベース接続が確立されており、test テーブルが存在すること。
  • test テーブルの列 c1, c2 が整数型（int）として定義されていること。

アルゴリズムまたは振る舞い

1. ivar1, ivar2 を 0 で初期化する。 2. ECPGdo を使用して select c1, c2 from test を実行し、結果を ivar1, ivar2 に格納する。 3. sqlca.sqlcode を確認し、負の値（エラー）であれば sqlprint() を呼び出してエラー情報を出力する。 4. 取得した ivar1 と ivar2 の値を printf("%d %d\n", ivar1, ivar2) で標準出力に書き出す。

呼び出し元と呼び出し先

  • **呼び出し元**: main 関数内の各テストケース（src/interfaces/ecpg/test/expected/sql-prepareas.c 内の複数箇所から計12回呼び出されている）。
  • **呼び出し先**: ECPGdo（ECPGランタイムライブラリ）、sqlprint（ECPGライブラリ）。

関連データ構造

  • sqlca: SQL通信領域（SQLCA）。エラーコードの判定に使用。

失敗モードと境界条件

  • sqlca.sqlcode < 0: SQL実行時にエラーが発生した場合、sqlprint() が実行され、エラー内容が標準エラー出力に記録される。
