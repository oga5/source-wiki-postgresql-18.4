strtitle_icu

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/pg_locale_icu.c:404

目的

ICU（International Components for Unicode）ライブラリを使用して、指定されたロケールに基づき入力文字列をタイトルケース（各単語の先頭を大文字、それ以外を小文字）に変換します。

シグネチャまたは形状

size_t strtitle_icu(char *dest, size_t destsize, const char *src, ssize_t srclen, pg_locale_t locale)

入力と出力

  • **入力**:
  • dest: 変換結果を格納するバッファ。
  • destsize: dest バッファのサイズ。
  • src: 変換対象のソース文字列。
  • srclen: src の長さ（バイト単位）。
  • locale: ICUロケール情報を含む pg_locale_t 構造体。
  • **出力**:
  • 変換後の文字列長（バイト単位）を size_t で返します。

前提条件と不変条件

  • icu_to_uchar および icu_convert_case が成功し、有効な UChar バッファが生成される必要があります。
  • 処理終了時には、内部で確保された buff_uchar および buff_conv が pfree により解放されることが保証されています。

アルゴリズムまたは振る舞い

1. icu_to_uchar を呼び出し、データベースエンコーディングの文字列を ICU 内部形式（UTF-16）に変換します。 2. icu_convert_case を介して ICU の u_strToTitle_default_BI 関数を呼び出し、タイトルケース変換を実行します。 3. icu_from_uchar を使用して、変換後の UChar データをデータベースエンコーディングに戻し、dest バッファへ書き込みます。 4. 変換に使用した一時的な UChar バッファを解放します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/pg_locale.c:1297
  • **呼び出し先**:
  • icu_to_uchar
  • icu_convert_case
  • u_strToTitle_default_BI
  • icu_from_uchar
  • pfree

関連データ構造

  • pg_locale_t: ロケール設定を保持する構造体。
  • UChar: ICU で使用される UTF-16 文字型。

失敗モードと境界条件

  • ICU 内部でエラーが発生した場合、icu_convert_case 等の内部処理を通じて ereport(ERROR, ...) が呼び出され、PostgreSQL のトランザクションが中断される可能性があります。
  • destsize が変換結果を格納するのに不十分な場合、icu_from_uchar の挙動に依存します。

保守担当者向けメモ

  • 本関数は pg_locale_t を介してロケール依存の変換を行うため、ロケール設定が正しく初期化されていることを前提としています。
  • メモリ管理は palloc/pfree に依存しているため、例外発生時のメモリリークには注意が必要です。
