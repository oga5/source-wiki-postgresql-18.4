TimestampTz

定義

  • 種別: typedef
  • 定義箇所: src/interfaces/ecpg/include/pgtypes_timestamp.h:11

目的

ECPG（Embedded C for PostgreSQL）アプリケーションにおいて、タイムゾーン付きのタイムスタンプ値を保持するためのデータ型を提供します。

シグネチャまたは形状

typedef int64 TimestampTz;

入力と出力

  • **入力**: 64ビット整数（int64）として表現されるタイムスタンプ値。
  • **出力**: データベースのタイムゾーン付きタイムスタンプ型と互換性のあるデータ表現。

前提条件と不変条件

  • 内部的には int64 型として扱われます。
  • ECPGライブラリを通じて、データベースのタイムスタンプ型とC言語の変数間で相互運用されることを前提としています。

アルゴリズムまたは振る舞い

本シンボルは型定義であり、それ自体に実行時のアルゴリズムは含まれません。ECPGライブラリ内の関数（PGTYPEStimestamp_from_asc 等）が、この型を使用して文字列とタイムスタンプ値の変換や算術演算を行います。

呼び出し元と呼び出し先

  • **呼び出し元（参照箇所）**:
  • contrib/btree_gist/btree_ts.c
  • contrib/dblink/dblink.c
  • contrib/pg_prewarm/autoprewarm.c
  • contrib/pg_stat_statements/pg_stat_statements.c
  • **呼び出し先**:
  • 本シンボルは型定義であるため、直接的な呼び出し先はありません。

関連データ構造

  • timestamp: タイムゾーンなしのタイムスタンプを保持する int64 型のエイリアス。
  • interval: タイムスタンプ演算に使用される期間型。

失敗モードと境界条件

  • int64 型の範囲を超える値が代入された場合、オーバーフローが発生する可能性があります。
  • タイムゾーンの解釈は、実行時のデータベース設定や環境変数に依存します。

保守担当者向けメモ

  • 本型は int64 のエイリアスであるため、算術演算や比較を行う際は int64 としての特性に従います。
  • C++から利用する場合は extern "C" ブロック内での使用を考慮してください。
