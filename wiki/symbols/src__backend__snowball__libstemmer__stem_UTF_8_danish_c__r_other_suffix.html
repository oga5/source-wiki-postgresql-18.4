r_other_suffix

定義

  • **種別**: function
  • **定義箇所**: src/backend/snowball/libstemmer/stem_UTF_8_danish.c:216

目的

デンマーク語のステミング処理において、特定の接尾辞の削除および置換を行う。

シグネチャまたは形状

static int r_other_suffix(struct SN_env * z);

入力と出力

  • **入力**: SN_env * z（ステミング環境構造体。現在のカーソル位置 c、境界 lb、文字列バッファ p を含む）。
  • **出力**: 成功時に 1、失敗または条件不一致時に 0、メモリ操作等のエラー発生時に負の値を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • 処理は単語の後方から前方へ向かって行われる。
  • z->I[1] に格納された領域境界（R1相当）が処理の制約として使用される。

アルゴリズムまたは振る舞い

1. **第一段階**: 特定の接尾辞（s_0 と s_1 の組み合わせ）を検索し、一致すれば削除する。 2. **第二段階**: z->I[1] 以降の領域において、a_2 テーブルに基づく接尾辞を検索する。 3. **分岐処理**:

  • among_var == 1: 一致した接尾辞を削除し、その後 r_consonant_pair を呼び出して子音ペアの調整を行う。
  • among_var == 2: 一致した接尾辞を s_2 で定義された文字列に置換する。

    • 4. 処理中、カーソル位置や境界情報は一時的に退避・復元され、整合性が保たれる。

呼び出し元と呼び出し先

  • **呼び出し元**: danish_UTF_8_stem（直接の呼び出し元として stem_UTF_8_danish.c 内で確認）。
  • **呼び出し先**: slice_del, slice_from_s, eq_s_b, find_among_b, r_consonant_pair。

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_2: 除去・置換対象の接尾辞を定義した among テーブル。

失敗モードと境界条件

  • slice_del や slice_from_s がエラー（負の値）を返した場合、その値をそのまま伝播させる。
  • 検索対象の接尾辞が見つからない場合や、領域境界（z->I[1]）の制約を満たさない場合は 0 を返す。

保守担当者向けメモ

  • 本関数は自動生成コードの一部である。ロジックの変更が必要な場合は、元のSnowballソース（.sbl）を修正すること。
  • among_var の値は a_2 テーブルのインデックスに対応している。
  • z->p[z->c - 1] >> 5 != 3 等のビット演算は、UTF-8エンコーディングにおける文字境界や特定の文字グループを高速に判定するための最適化である。
