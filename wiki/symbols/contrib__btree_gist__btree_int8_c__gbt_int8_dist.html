gbt_int8_dist

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_int8.c:72

目的

GiSTインデックスにおける int8 型の2点間の距離を算出します。

シグネチャまたは形状

float8 gbt_int8_dist(const void *a, const void *b, FmgrInfo *flinfo)

入力と出力

  • **入力**:
  • a, b: 距離計算対象となる int8 型の値（またはキー）へのポインタ。
  • flinfo: 関数呼び出し情報（PostgreSQLの関数マネージャ用）。
  • **出力**:
  • 2点間の距離を float8 型で返します。

前提条件と不変条件

  • GET_FLOAT_DISTANCE マクロを使用して計算が行われます。
  • 入力ポインタ a および b は、int64 型として解釈可能なデータ構造を指している必要があります。

アルゴリズムまたは振る舞い

1. GET_FLOAT_DISTANCE(int64, a, b) を呼び出します。 2. 内部的に int64 型の差分を計算し、その絶対値を float8 に変換して返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/btree_gist/btree_int8.c 内の静的変数 tinfo の初期化において参照されています（行90）。
  • **呼び出し先**:
  • GET_FLOAT_DISTANCE マクロ（btree_utils_num.h で定義されていると推測される共通ユーティリティ）。

関連データ構造

  • int64KEY: int8 型の範囲を保持する構造体。
  • tinfo: gbtree_ninfo 型の静的変数。本関数を距離計算関数として保持します。

失敗モードと境界条件

  • 入力値が int64 の範囲外である場合や、計算過程でオーバーフローが発生する場合の挙動は、GET_FLOAT_DISTANCE マクロの実装に依存します。

保守担当者向けメモ

  • 本関数は int8 型専用のラッパーです。ロジックの変更が必要な場合は、共通の数値処理ロジックである btree_utils_num.c を確認してください。
  • 呼び出し元に関する情報は、ソースコード内の静的変数 tinfo への登録箇所から推測されるものに限定されています。
