GetSubscriptionRelState

定義

  • 種別: function
  • 定義箇所: src/backend/catalog/pg_subscription.c:386

目的

特定のサブスクリプションとテーブルの組み合わせ（マッピング）における、現在の同期状態（srsubstate）と最終更新 LSN を取得します。

シグネチャまたは形状

char GetSubscriptionRelState(Oid subid, Oid relid, XLogRecPtr *sublsn)

入力と出力

  • **入力**:
  • subid (Oid): 対象サブスクリプションの識別子。
  • relid (Oid): 対象テーブルのリレーション識別子。
  • sublsn (XLogRecPtr *): 取得した LSN を格納するための出力用ポインタ。
  • **出力**:
  • 戻り値 (char): テーブルの同期状態（srsubstate）。マッピングが見つからない場合は SUBREL_STATE_UNKNOWN を返します。

前提条件と不変条件

  • pg_subscription_rel システムカタログに対する AccessShareLock を内部で取得・解放します。
  • SUBSCRIPTIONRELMAP システムキャッシュを使用して検索を行います。

アルゴリズムまたは振る舞い

1. SubscriptionRelRelationId に対して AccessShareLock を取得し、AlterSubscription 等による競合を回避します。 2. SUBSCRIPTIONRELMAP キャッシュを relid と subid で検索します。 3. タプルが存在しない場合、sublsn に InvalidXLogRecPtr を設定し、SUBREL_STATE_UNKNOWN を返して終了します。 4. タプルが存在する場合、srsubstate フィールドから状態を取得します。 5. srsublsn 属性をキャッシュから読み取り、値が NULL であれば InvalidXLogRecPtr を、そうでなければ LSN 値を sublsn に格納します。 6. キャッシュタプルを解放し、リレーションをクローズして状態を返します。

関連データ構造

  • pg_subscription_rel (システムカタログ)
  • Subscription (関連する構造体定義は src/include/catalog/pg_subscription.h に存在)

失敗モードと境界条件

  • 指定された subid と relid の組み合わせがカタログに存在しない場合、エラーにはならず SUBREL_STATE_UNKNOWN を返します。
  • srsublsn が NULL の場合、sublsn には InvalidXLogRecPtr が設定されます。

保守担当者向けメモ

  • 本関数は SysCache を利用しているため、カタログの直接的な変更を伴わない読み取りに適しています。
  • AccessShareLock を使用しているため、サブスクリプションの削除や変更操作との排他制御が考慮されています。
