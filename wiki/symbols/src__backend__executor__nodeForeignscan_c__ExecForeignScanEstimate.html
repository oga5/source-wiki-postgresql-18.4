ExecForeignScanEstimate

定義

  • 種別: function
  • 定義箇所: src/backend/executor/nodeForeignscan.c:355

目的

並列クエリ実行時において、外部テーブルスキャン（Foreign Scan）が共有メモリ（DSM）上で必要とする領域サイズを見積もる。

シグネチャまたは形状

void ExecForeignScanEstimate(ForeignScanState *node, ParallelContext *pcxt)

入力と出力

  • **入力**:
  • node: 実行中の ForeignScanState 構造体へのポインタ。
  • pcxt: 並列実行コンテキスト ParallelContext へのポインタ。
  • **出力**: なし（副作用として node->pscan_len の更新および pcxt->estimator への見積もり値の加算を行う）。

前提条件と不変条件

  • node->fdwroutine が有効な FdwRoutine を指していること。
  • fdwroutine->EstimateDSMForeignScan が定義されている場合のみ、見積もり処理が実行される。

アルゴリズムまたは振る舞い

1. node->fdwroutine から EstimateDSMForeignScan 関数ポインタを取得する。 2. 関数が定義されている場合、それを呼び出して必要な共有メモリサイズを計算し、node->pscan_len に格納する。 3. shm_toc_estimate_chunk を呼び出し、計算されたサイズを pcxt->estimator に登録する。 4. shm_toc_estimate_keys を呼び出し、1つのキーを共有メモリ領域の管理用に確保する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/executor/execParallel.c 内の並列クエリ実行準備処理から呼び出される。
  • **呼び出し先**: node->fdwroutine->EstimateDSMForeignScan（FDW実装依存の関数）。

関連データ構造

  • ForeignScanState: 外部スキャンの実行状態を保持する構造体。
  • FdwRoutine: FDWが提供する関数ポインタ群。
  • ParallelContext: 並列実行のための共有メモリ見積もりや管理を行うコンテキスト。

失敗モードと境界条件

  • fdwroutine->EstimateDSMForeignScan が NULL の場合、本関数は何もしない（エラーにはならない）。

保守担当者向けメモ

  • 本関数は、FDWが並列スキャンをサポートしている場合にのみ機能する。
  • 共有メモリのサイズ見積もりは、後続の ExecForeignScanInitializeDSM で実際に領域を割り当てるための準備段階である。
