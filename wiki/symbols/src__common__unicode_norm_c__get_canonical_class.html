get_canonical_class

定義

  • 種別: function
  • 定義箇所: src/common/unicode_norm.c:112

目的

指定された Unicode 文字（pg_wchar）の結合クラス（Canonical Combining Class）を取得します。

シグネチャまたは形状

static uint8 get_canonical_class(pg_wchar code)

入力と出力

  • **入力**: pg_wchar code（対象の Unicode 文字コード）
  • **出力**: uint8（文字の結合クラス値）

前提条件と不変条件

  • 内部で get_code_entry を呼び出し、文字の分解情報を検索します。
  • 結合クラスは Unicode 標準の定義に従います。

アルゴリズムまたは振る舞い

1. get_code_entry(code) を呼び出し、該当文字の分解情報エントリを取得します。 2. エントリが存在しない場合（ハングル文字や、結合クラスが0で分解情報を持たない文字の場合）、0 を返します。 3. エントリが存在する場合、そのエントリ内の comb_class メンバを返します。

呼び出し元と呼び出し先

  • **呼び出し先**: get_code_entry
  • **呼び出し元**:
  • src/common/unicode_norm.c 内の正規化処理において、文字の並べ替え順序を判定するために複数箇所から呼び出されています（行473, 474, 524, 638）。

関連データ構造

  • pg_unicode_decomposition: 文字の分解情報と結合クラスを保持する構造体。

失敗モードと境界条件

  • 該当する文字エントリが見つからない場合は、デフォルト値として 0 を返します。これはエラーではなく、結合クラスを持たない文字としての正常な振る舞いです。

保守担当者向けメモ

  • 本関数は static であり、src/common/unicode_norm.c 内でのみ使用されます。
  • ハングル文字の特殊な扱いについては、get_code_entry の実装および unicode_norm.c の概要にある「ハングル処理」の記述を参照してください。
