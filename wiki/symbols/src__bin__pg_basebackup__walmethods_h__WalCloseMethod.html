WalCloseMethod

定義

  • 種別: enum
  • 定義箇所: src/bin/pg_basebackup/walmethods.h:31

目的

WAL（Write-Ahead Log）ファイルを閉じる際の挙動を制御するための列挙型です。

シグネチャまたは形状

typedef enum
{
	CLOSE_NORMAL,
	CLOSE_UNLINK,
	CLOSE_NO_RENAME,
} WalCloseMethod;

入力と出力

  • 入力: なし（列挙定数）
  • 出力: なし（列挙定数）

前提条件と不変条件

  • Walfile 構造体に対するクローズ処理の引数として使用されます。

アルゴリズムまたは振る舞い

  • CLOSE_NORMAL: 通常のクローズ処理を実行します。
  • CLOSE_UNLINK: ファイルを閉じた後、削除（unlink）します。
  • CLOSE_NO_RENAME: ファイルの名称変更を行わずにクローズします。

呼び出し元と呼び出し先

  • 呼び出し元: src/bin/pg_basebackup/walmethods.c 内の dir_close および tar_close 関数がこの型を引数として受け取ります。
  • 呼び出し先: 情報なし。

関連データ構造

  • Walfile: walmethods.h 内で定義される構造体で、close 関数ポインタの引数として WalCloseMethod を使用します。

失敗モードと境界条件

  • 列挙型定義自体に失敗モードはありません。各メソッドを実装する関数（dir_close, tar_close）側でのエラーハンドリングに依存します。

保守担当者向けメモ

  • 本列挙型は walmethods.c で定義される各種ストレージメソッド（ディレクトリ、tar等）のクローズ処理のインターフェースを共通化するために使用されています。新しいクローズ挙動を追加する場合は、各実装関数側の修正が必要です。
