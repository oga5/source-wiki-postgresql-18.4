GetPublicationsStr

定義

  • 種別: function
  • 定義箇所: src/backend/catalog/pg_subscription.c:40

目的

パブリケーション名のリストをカンマ区切りの文字列に変換し、指定された StringInfo バッファへ書き出します。

シグネチャまたは形状

void GetPublicationsStr(List *publications, StringInfo dest, bool quote_literal)

入力と出力

  • **入力**:
  • publications: パブリケーション名（文字列）を保持する List。
  • dest: 出力先となる StringInfo 構造体へのポインタ。
  • quote_literal: true の場合は quote_literal_cstr を使用してエスケープし、false の場合は二重引用符（"）で囲みます。
  • **出力**: なし（dest が指すバッファを更新）。

前提条件と不変条件

  • publications リストは NIL であってはなりません（Assert により検証）。

アルゴリズムまたは振る舞い

1. publications リストを走査します。 2. 各要素の間にカンマとスペース（, ）を挿入します。 3. quote_literal が true の場合、quote_literal_cstr を適用してパブリケーション名をバッファに追加します。 4. quote_literal が false の場合、パブリケーション名を二重引用符で囲んでバッファに追加します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/commands/subscriptioncmds.c (複数箇所)
  • src/backend/replication/logical/tablesync.c
  • **呼び出し先**:
  • appendStringInfoString / appendStringInfoChar
  • quote_literal_cstr (条件付き)
  • strVal (リスト要素の抽出)

関連データ構造

  • List: パブリケーション名の保持に使用。
  • StringInfo: 出力バッファとして使用。

失敗モードと境界条件

  • publications が空リストの場合、Assert によりプログラムが停止します。

保守担当者向けメモ

  • 本関数は主に SQL コマンドの生成やログ出力のために使用されます。
  • quote_literal フラグの切り替えにより、用途に応じた適切な引用符付けが可能です。
