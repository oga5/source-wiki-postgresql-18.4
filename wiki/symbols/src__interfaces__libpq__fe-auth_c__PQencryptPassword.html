PQencryptPassword

定義

  • 種別: function
  • 定義箇所: src/interfaces/libpq/fe-auth.c:1362

目的

MD5アルゴリズムを使用してパスワードを暗号化し、認証用の文字列を生成する。

シグネチャまたは形状

char *PQencryptPassword(const char *passwd, const char *user);

入力と出力

  • **入力**:
  • passwd: 暗号化対象の平文パスワード。
  • user: ユーザー名（MD5ハッシュ生成時のソルトとして使用）。
  • **出力**:
  • 成功時: 暗号化されたパスワード文字列（呼び出し元で free() が必要）。
  • 失敗時: NULL。

前提条件と不変条件

  • passwd および user は有効な文字列ポインタであること。
  • 内部で malloc を使用するため、メモリ不足時には NULL を返す。

アルゴリズムまたは振る舞い

1. MD5_PASSWD_LEN + 1 バイトのメモリを確保する。 2. pg_md5_encrypt を呼び出し、passwd と user を用いてMD5ハッシュを生成する。 3. 暗号化に失敗した場合は確保したメモリを解放し、NULL を返す。 4. 成功した場合は、生成された暗号化文字列を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: 提供されたソース excerpt 内には具体的な呼び出し元の記述なし。
  • **呼び出し先**: malloc, free, pg_md5_encrypt, strlen。

関連データ構造

  • MD5_PASSWD_LEN: 暗号化後の文字列長を規定する定数。

失敗モードと境界条件

  • メモリ割り当て失敗時に NULL を返す。
  • pg_md5_encrypt が失敗した場合に NULL を返す。

保守担当者向けメモ

  • 本関数はレガシーなMD5認証用です。現代的な認証フローについては PQencryptPasswordConn の利用を検討してください。
  • 戻り値の文字列は呼び出し元で解放する責任があるため、メモリリークに注意してください。
