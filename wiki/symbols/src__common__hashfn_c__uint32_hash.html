uint32_hash

定義

  • 種別: function
  • 定義箇所: src/common/hashfn.c:687

目的

uint32 または int32 型のキーをハッシュ化し、ハッシュテーブルで使用するための uint32 型のハッシュ値を生成します。

シグネチャまたは形状

uint32 uint32_hash(const void *key, Size keysize)

入力と出力

  • **入力**:
  • key: ハッシュ対象のデータへのポインタ。
  • keysize: データのサイズ（バイト単位）。
  • **出力**:
  • uint32: 計算されたハッシュ値。

前提条件と不変条件

  • keysize は sizeof(uint32) と等しくなければなりません。これに違反した場合、Assert により停止します。

アルゴリズムまたは振る舞い

1. keysize が uint32 のサイズと一致することを確認します。 2. key が指す値を uint32 として読み込みます。 3. hash_bytes_uint32 関数を呼び出し、その結果を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/hash/dynahash.c: ハッシュテーブルのハッシュ関数として設定されるケースがあります。
  • **呼び出し先**:
  • hash_bytes_uint32 (直接の計算処理)

関連データ構造

  • uint32 (PostgreSQL の基本型)

失敗モードと境界条件

  • keysize が sizeof(uint32) 以外の場合、デバッグビルドではアサーション失敗が発生します。

保守担当者向けメモ

  • 本関数は uint32 型専用のラッパーです。汎用的なバイト列のハッシュ化には hash_bytes を使用してください。
  • oid_hash は本関数の別名として定義されています。
