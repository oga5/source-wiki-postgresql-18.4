gbt_float4_dist

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_float4.c:71

目的

GiSTインデックスにおける float4 型の2点間の距離を計算し、検索時の優先順位付けを支援します。

シグネチャまたは形状

static float8 gbt_float4_dist(const void *a, const void *b, FmgrInfo *flinfo)

入力と出力

  • **入力**:
  • a, b: 距離を計算する対象の float4 値へのポインタ。
  • flinfo: 関数呼び出し情報（PostgreSQLの関数マネージャ用）。
  • **出力**:
  • 2点間の距離を float8 型で返します。

前提条件と不変条件

  • 入力ポインタ a および b は、有効な float4 型のデータ領域を指している必要があります。

アルゴリズムまたは振る舞い

  • GET_FLOAT_DISTANCE マクロを使用して、float4 型の差分（絶対値）を計算します。
  • 内部的に btree_utils_num.c で定義された数値処理ロジックを介して計算を実行します。

呼び出し元と呼び出し先

  • **呼び出し元**: ソースコード内の静的解析では、本関数を直接呼び出している具体的な関数名は特定されていません。
  • **呼び出し先**: GET_FLOAT_DISTANCE マクロ（btree_utils_num.h 経由で提供される数値計算ロジック）。

関連データ構造

  • float4: 本関数が扱う対象のデータ型。
  • float8: 戻り値の型。

失敗モードと境界条件

  • 入力ポインタが NULL の場合の挙動は未定義です。
  • 浮動小数点数のオーバーフローやアンダーフローが発生する可能性があります。

保守担当者向けメモ

  • 本関数は btree_utils_num.c の汎用ロジックを利用するラッパーです。詳細な計算ロジックについては btree_utils_num.c を参照してください。
  • float4_dist 関数とは異なり、本関数は static であり、モジュール内部での利用に限定されています。
