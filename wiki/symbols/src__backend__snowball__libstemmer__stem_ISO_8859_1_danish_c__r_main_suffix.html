r_main_suffix

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_danish.c:160

目的

デンマーク語のステミング処理において、単語の末尾から主要な接尾辞を検索し、条件に応じて削除または置換を行う。

シグネチャまたは形状

static int r_main_suffix(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z （Snowballランタイム環境）
  • 出力: int （成功時に 1、失敗または処理不要時に 0、エラー発生時に負の値を返す）

前提条件と不変条件

  • z->I[1] に格納された領域境界（R1等）より後ろの文字列を対象とする。
  • z->c が z->I[1] 未満の場合、処理を中断する。
  • find_among_b を使用して、a_0 配列に定義された接尾辞を後方から検索する。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c が領域境界 z->I[1] より前であれば終了する。 2. z->lb を z->I[1] に一時的に設定し、検索範囲を制限する。 3. find_among_b を用いて a_0 配列から接尾辞をマッチングする。 4. マッチした among_var の値に応じて以下の処理を行う:

  • case 1: マッチした接尾辞を削除する。
  • case 2: g_s_ending グループに含まれる文字が直前にないことを確認した上で、接尾辞を削除する。

    • 5. 処理終了後、z->lb を元の値に戻す。

呼び出し元と呼び出し先

  • 呼び出し元: danish_ISO_8859_1_stem （src/backend/snowball/libstemmer/stem_ISO_8859_1_danish.c 内）
  • 呼び出し先: find_among_b, slice_del, in_grouping_b （いずれもSnowballランタイム関数）

関連データ構造

  • struct SN_env: ステミング環境の状態（カーソル位置、領域境界、文字列バッファ等）を保持。
  • a_0: 接尾辞マッチングに使用される among 構造体の配列。
  • g_s_ending: case 2 の条件判定に使用される文字グループ。

失敗モードと境界条件

  • find_among_b がマッチしない場合、0 を返して終了する。
  • slice_del が内部エラー（負の値）を返した場合、その値をそのまま伝播させる。
  • z->c <= z->lb の場合、検索範囲外として終了する。

保守担当者向けメモ

  • 本関数はSnowballコンパイラによる自動生成コードである。ロジックの変更が必要な場合は、直接編集せず元のSnowballスクリプト（.sbl）を修正すること。
  • a_0 配列の定義や among_var のケース分岐は、デンマーク語の文法ルールに直結している。
