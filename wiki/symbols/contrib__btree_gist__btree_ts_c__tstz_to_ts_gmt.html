tstz_to_ts_gmt

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_ts.c:197

目的

timestamptz 型の値を、インデックス内での比較を統一するために GMT ベースの timestamp 型へ変換する。

シグネチャまたは形状

static inline Timestamp tstz_to_ts_gmt(TimestampTz ts)

入力と出力

  • **入力**: TimestampTz ts (タイムゾーン付きタイムスタンプ)
  • **出力**: Timestamp (GMT に正規化されたタイムスタンプ)

前提条件と不変条件

  • GMT は定義上オフセットが 0 であるため、タイムゾーンの補正処理は不要である。
  • 入力された TimestampTz 値は、そのまま Timestamp 型へキャストされる。

アルゴリズムまたは振る舞い

1. 引数として受け取った TimestampTz を Timestamp 型にキャストする。 2. 変換された値を戻り値として返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/btree_gist/btree_ts.c 内の以下の箇所から呼び出されていることが確認されている:
  • 225行目: gmt = tstz_to_ts_gmt(ts);
  • 304行目: qqq = tstz_to_ts_gmt(query);
  • 323行目: qqq = tstz_to_ts_gmt(query);
  • **呼び出し先**:
  • 外部関数への呼び出しはない。

関連データ構造

  • Timestamp: PostgreSQL のタイムスタンプ型。
  • TimestampTz: PostgreSQL のタイムゾーン付きタイムスタンプ型。

失敗モードと境界条件

  • 特筆すべき失敗モードはない。型キャストのみを行う単純な変換関数である。

保守担当者向けメモ

  • 本関数は timestamptz を扱う GiST インデックスの比較ロジックにおいて、内部的な正規化のために使用される。
  • inline 関数として定義されており、パフォーマンスへの影響は最小限である。
