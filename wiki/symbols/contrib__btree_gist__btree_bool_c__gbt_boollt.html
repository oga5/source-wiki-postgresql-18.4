gbt_boollt

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_bool.c:46

目的

2つの bool 型値の「より小さい（less than）」比較を行い、GiSTインデックスの検索やソート処理における比較ロジックを提供します。

シグネチャまたは形状

static bool gbt_boollt(const void *a, const void *b, FmgrInfo *flinfo)

入力と出力

  • **入力**:
  • a: 比較対象の bool 値へのポインタ。
  • b: 比較対象の bool 値へのポインタ。
  • flinfo: 関数管理情報（本関数内では未使用）。
  • **出力**:
  • bool: *a < *b が真であれば true、そうでなければ false を返します。

前提条件と不変条件

  • a および b は有効な bool 型の値へのポインタである必要があります。
  • bool 型は内部的に数値（0または1）として扱われます。

アルゴリズムまたは振る舞い

  • 入力されたポインタを bool 型としてキャストし、C言語の標準的な小なり演算子（<）を用いて比較結果を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/btree_gist/btree_bool.c 内の比較テーブル定義（gbt_boollt,）にて参照されています。
  • **呼び出し先**:
  • 直接的な関数呼び出しはありません。

関連データ構造

  • bool: 比較対象となるPostgreSQLのブール型。

失敗モードと境界条件

  • ポインタが NULL の場合、セグメンテーション違反が発生する可能性があります。

保守担当者向けメモ

  • 本関数は btree_utils_num.c 等の共通ユーティリティから呼び出される比較関数群の一部です。
  • bool 型の比較は単純な数値比較と同等であるため、実装は極めて軽量です。
