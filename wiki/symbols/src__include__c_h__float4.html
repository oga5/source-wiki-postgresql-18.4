float4

定義

  • 種別: typedef
  • 定義箇所: src/include/c.h:608

目的

PostgreSQL における単精度浮動小数点数（32ビット浮動小数点数）を表現するための型定義です。

シグネチャまたは形状

typedef float float4;

入力と出力

  • 入力: C 言語の標準型 float に準拠します。
  • 出力: 算術演算や比較演算の結果として float4 型の値を返します。

前提条件と不変条件

  • C 言語の float 型が IEEE 754 単精度浮動小数点数であることを前提としています。
  • PostgreSQL のデータ構造として使用される場合、プラットフォーム固有のアライメント要件に従う必要があります。

アルゴリズムまたは振る舞い

  • float 型のエイリアスとして機能し、メモリ上のサイズおよび算術演算の挙動はコンパイラの float 実装に依存します。
  • 比較演算や算術演算において、標準的な浮動小数点演算が適用されます。

呼び出し元と呼び出し先

  • 呼び出し元（参照箇所）:
  • contrib/btree_gist/btree_float4.c: 比較関数、距離計算、および PG_GETARG_FLOAT4 マクロによる引数取得に使用。
  • contrib/btree_gin/btree_gin.c: GIN インデックスのサポート関数定義に使用。
  • 呼び出し先:
  • 本シンボルは型定義であるため、直接的な呼び出し先はありません。

関連データ構造

  • float8: PostgreSQL における倍精度浮動小数点数型。
  • PG_GETARG_FLOAT4: float4 型の値を関数引数から取得するためのマクロ。

失敗モードと境界条件

  • 浮動小数点演算特有の精度制限（丸め誤差）や、オーバーフロー・アンダーフローが発生する可能性があります。
  • ゼロ除算や非数（NaN）の取り扱いは、実行環境の浮動小数点演算ユニット（FPU）の仕様に依存します。

保守担当者向けメモ

  • 本型は c.h で定義されており、PostgreSQL 全体で一貫して使用されます。
  • 型のサイズを変更したり、別の型に置き換えたりする場合は、インデックスやバイナリ互換性に影響を与えるため注意が必要です。
