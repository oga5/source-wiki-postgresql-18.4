macaddr8_cmp_internal

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/mac8.c:309

目的

2つの macaddr8 型（EUI-64 MACアドレス）の値を比較し、ソート順序を決定するための内部比較関数です。

シグネチャまたは形状

static int32 macaddr8_cmp_internal(macaddr8 *a1, macaddr8 *a2)

入力と出力

  • **入力**: 比較対象となる2つの macaddr8 構造体へのポインタ a1, a2。
  • **出力**: 比較結果を示す int32 値。
  • -1: a1 < a2 の場合
  • 1: a1 > a2 の場合
  • 0: a1 == a2 の場合

前提条件と不変条件

  • hibits および lobits マクロが、macaddr8 構造体のメモリレイアウトに基づいて8バイトのデータをそれぞれ上位4バイトと下位4バイトの整数として正しく抽出できること。

アルゴリズムまたは振る舞い

1. hibits(a1) と hibits(a2) を比較し、大小関係があれば即座に結果を返します。 2. 上位4バイトが等しい場合、lobits(a1) と lobits(a2) を比較し、その結果を返します。 3. 両方のビット領域が等しい場合は 0 を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/mac8.c 内の SQL 比較演算子関数（macaddr8_cmp, macaddr8_lt, macaddr8_le, macaddr8_eq, macaddr8_ge, macaddr8_gt, macaddr8_ne に相当する各関数）。
  • **呼び出し先**:
  • hibits() マクロ
  • lobits() マクロ

関連データ構造

  • macaddr8: 8バイトのMACアドレスを保持する構造体。

失敗モードと境界条件

  • 構造体のメモリレイアウトが想定と異なる場合、比較結果が不正になる可能性があります。
  • 入力ポインタが NULL の場合の挙動は定義されていません。

保守担当者向けメモ

  • 本関数は hibits および lobits マクロに強く依存しています。構造体の定義やメモリレイアウトを変更する際は、これらのマクロの定義と整合性が取れているか確認してください。
