r_mark_regions

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_KOI8_R_russian.c:344

目的

Snowballステミングアルゴリズムにおいて、語幹抽出の制限範囲となるR1およびR2領域の境界位置を計算し、環境構造体 SN_env に格納する。

シグネチャまたは形状

static int r_mark_regions(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z（ステミング環境の状態を保持する構造体）
  • **出力**: 常に 1 を返す。
  • **副作用**: z->I[0] および z->I[1] に計算された境界位置（オフセット）が書き込まれる。

前提条件と不変条件

  • z が有効な SN_env 構造体を指していること。
  • g_v（母音グループ）が定義されていること。
  • 処理対象の文字列がKOI8-Rエンコーディングで適切に初期化されていること。

アルゴリズムまたは振る舞い

1. z->I[1]（R1）と z->I[0]（R2）を初期値として文字列長 z->l に設定する。 2. 文字列の先頭から以下の順序でパターンマッチングを行う：

  • 母音（out_grouping）
  • 非母音（in_grouping）
  • 母音（out_grouping）
  • 非母音（in_grouping）

    • 3. 最初の母音の後の位置を z->I[1]（R1）に、その後の非母音の後の位置を z->I[0]（R2）に設定する。 4. マッチングに失敗した場合は、カーソル位置を初期状態に戻し、領域を文字列末尾に維持する。

呼び出し元と呼び出し先

  • **呼び出し元**: russian_KOI8_R_stem（本ファイル内）。
  • **呼び出し先**: out_grouping, in_grouping（Snowballランタイムのユーティリティ関数）。
  • 注: 他の言語ファイル（Basque, Catalan等）にも同名の関数が存在するが、本シンボルは stem_KOI8_R_russian.c 内の定義を指す。

関連データ構造

  • struct SN_env: ステミング処理の環境変数、カーソル位置、領域境界（I 配列）を保持する。

失敗モードと境界条件

  • 文字列が短く、指定された母音・非母音のパターンに適合しない場合、lab0 ラベルへジャンプし、領域境界は更新されず初期値（文字列長）のままとなる。

保守担当者向けメモ

  • 本関数はSnowballコンパイラによって自動生成されたコードである。ロジックの変更が必要な場合は、直接編集せず元のSnowballスクリプト（.sbl）を修正すること。
  • 192 から 220 までの数値はKOI8-Rにおける母音の範囲を示す。
