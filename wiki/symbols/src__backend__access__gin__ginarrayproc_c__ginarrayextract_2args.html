ginarrayextract_2args

定義

  • 種別: function
  • 定義箇所: src/backend/access/gin/ginarrayproc.c:67

目的

古いバージョンの PostgreSQL との互換性を維持するためのラッパー関数です。

シグネチャまたは形状

Datum ginarrayextract_2args(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS (関数呼び出し情報)
  • **出力**: Datum 型の配列（ginarrayextract の戻り値に依存）

前提条件と不変条件

  • 呼び出し時に PG_NARGS() が 3 以上であることを要求します。これに満たない場合はエラーを発生させます。

アルゴリズムまたは振る舞い

1. 引数の数が 3 未満でないかを確認します。 2. 内部的に ginarrayextract(fcinfo) を呼び出し、その結果をそのまま返します。

呼び出し元と呼び出し先

  • **呼び出し元**: 提供されたソース情報からは特定されていません。
  • **呼び出し先**: ginarrayextract を直接呼び出します。

関連データ構造

  • PG_FUNCTION_ARGS: PostgreSQL の関数呼び出しインターフェースで使用されるマクロおよび構造体。

失敗モードと境界条件

  • PG_NARGS() < 3 の場合、elog(ERROR, ...) により処理が中断されます。

保守担当者向けメモ

  • 本関数は古いバージョンの互換性のために残されています。新規実装では使用せず、直接 ginarrayextract を利用してください。
