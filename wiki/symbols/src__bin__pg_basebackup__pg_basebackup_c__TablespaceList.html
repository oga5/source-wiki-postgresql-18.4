TablespaceList

定義

  • 種別: struct
  • 定義箇所: src/bin/pg_basebackup/pg_basebackup.c:50

目的

ユーザーが指定したテーブルスペースの配置変更（マッピング）を管理するためのリスト構造体です。

シグネチャまたは形状

typedef struct TablespaceList
{
	TablespaceListCell *head;
	TablespaceListCell *tail;
} TablespaceList;

入力と出力

  • 入力: TablespaceListCell 型の要素をリストの末尾に追加することで構成されます。
  • 出力: リストの先頭（head）および末尾（tail）へのポインタを保持します。

前提条件と不変条件

  • head および tail は、リストが空の場合には NULL で初期化される必要があります（例: static TablespaceList tablespace_dirs = {NULL, NULL};）。

アルゴリズムまたは振る舞い

  • 連結リストの先頭と末尾を保持することで、テーブルスペースのマッピング情報を順次管理します。

呼び出し元と呼び出し先

  • 呼び出し元: src/bin/pg_basebackup/pg_basebackup.c 内の tablespace_dirs 変数として静的に定義されています。
  • 呼び出し先: 提供されたソース情報からは、本構造体自体を操作する具体的な関数呼び出しの詳細は特定できません。

関連データ構造

  • TablespaceListCell: リストの各要素を構成するデータ構造。

失敗モードと境界条件

  • リストが空の状態（head == NULL かつ tail == NULL）は正常な初期状態として扱われます。

保守担当者向けメモ

  • 本構造体は pg_basebackup におけるテーブルスペースの再配置ロジックの中核です。リスト操作を行う際は、head と tail の整合性が保たれていることを確認してください。
