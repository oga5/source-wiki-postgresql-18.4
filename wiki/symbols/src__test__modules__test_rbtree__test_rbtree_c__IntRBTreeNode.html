IntRBTreeNode

定義

  • **種別**: struct
  • **定義箇所**: src/test/modules/test_rbtree/test_rbtree.c:27

目的

PostgreSQL の赤黒木ライブラリ (lib/rbtree.h) の動作検証を行うための、テスト用ノード構造体です。

シグネチャまたは形状

typedef struct IntRBTreeNode
{
    RBTNode     rbtnode;
    int         key;
} IntRBTreeNode;

入力と出力

  • **入力**: 構造体内の key フィールドに整数値を保持します。
  • **出力**: 赤黒木ライブラリのノードとして機能し、RBTNode を継承することでライブラリの操作対象となります。

前提条件と不変条件

  • RBTNode を構造体の先頭メンバとして保持する必要があります。
  • 本構造体は palloc を用いてメモリ確保されることを前提としています。

アルゴリズムまたは振る舞い

  • 赤黒木ライブラリが提供する比較関数や結合関数において、RBTNode へのポインタを IntRBTreeNode へキャストすることで、格納された key 値にアクセスします。

呼び出し元と呼び出し先

  • **呼び出し元**: irbt_cmp, irbt_combine, irbt_alloc, create_int_rbtree, testleftright, testrightleft, testfind 等のテスト関数から、ノードの生成、比較、走査のために参照されています。
  • **呼び出し先**: 継承元の RBTNode 型を通じて、lib/rbtree.h の API と相互作用します。

関連データ構造

  • RBTNode: 赤黒木ライブラリの基底ノード構造体。

失敗モードと境界条件

  • メモリ不足時には palloc がエラーを発生させます。
  • キャスト時に IntRBTreeNode 以外の型を誤って指定した場合、メモリ破壊や不正なキー値の読み取りが発生する可能性があります。

保守担当者向けメモ

  • 本構造体はテスト専用です。実運用コードで赤黒木を使用する場合は、同様の継承パターンを用いて独自のノード構造体を定義してください。
