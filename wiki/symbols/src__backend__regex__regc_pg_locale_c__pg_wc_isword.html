pg_wc_isword

定義

  • 種別: function
  • 定義箇所: src/backend/regex/regc_pg_locale.c:379

目的

与えられた文字が「単語構成文字（word character）」であるかを判定する。

シグネチャまたは形状

static int pg_wc_isword(pg_wchar c)

入力と出力

  • **入力**: pg_wchar c （判定対象の文字）
  • **出力**: int （1: 単語構成文字である場合、0: それ以外の場合）

前提条件と不変条件

  • pg_wchar 型は符号なしとして扱われる。
  • 本関数は regcomp.c からインクルードされる環境下で動作する。

アルゴリズムまたは振る舞い

1. 入力文字 c がアンダースコア（CHR('_')）と一致するかを確認する。一致する場合は 1 を返す。 2. それ以外の場合、pg_wc_isalnum(c) を呼び出し、その結果を返す。

  • つまり、単語構成文字は「英数字（alnum）」または「アンダースコア」と定義される。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/regex/regc_locale.c:606 (pg_ctype_get_cache への引数として使用)
  • src/backend/regex/regc_locale.c:688 (条件判定に使用)
  • **呼び出し先**:
  • pg_wc_isalnum

関連データ構造

  • pg_wchar: 文字コードを表す型。

失敗モードと境界条件

  • 特になし。入力値に対して常に 0 または 1 を返す。

保守担当者向けメモ

  • 本関数は regcomp.c にインクルードされるため、単体でのコンパイルは想定されていない。
  • 単語構成文字の定義を変更する場合は、pg_wc_isalnum の挙動と整合性が取れているか確認すること。
