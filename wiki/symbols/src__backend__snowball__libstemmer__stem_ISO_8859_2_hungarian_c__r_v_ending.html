r_v_ending

定義

目的

ハンガリー語のステミング処理において、特定の母音（á または é）で終わる単語の接尾辞を、R1領域内でのマッチングに基づき置換または削除する。

シグネチャまたは形状

static int r_v_ending(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z （ステミング環境の状態を保持する構造体）
  • 出力: int （成功時に 1、失敗または条件不一致時に 0、エラー時に負の値を返す）

前提条件と不変条件

  • z->c が単語の末尾を指していること。
  • z->lb が単語の開始境界を指していること。
  • find_among_b を使用するため、単語の末尾から前方に向かって処理が行われる。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を z->ket に設定する。 2. 単語の末尾の文字が á (225) または é (233) であるかを確認し、一致しない場合は終了する。 3. a_1 配列を使用して、末尾から接尾辞のマッチングを行う。 4. マッチした場合、z->bra を現在のカーソル位置に設定し、r_R1(z) を呼び出して対象が R1 領域内にあるか検証する。 5. 検証成功後、among_var の値（1または2）に応じて slice_from_s を実行し、接尾辞を対応する文字列に置換する。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_ISO_8859_2_hungarian.c:593 および src/backend/snowball/libstemmer/stem_UTF_8_hungarian.c:599 からの呼び出しが確認されている。
  • 呼び出し先: find_among_b, r_R1, slice_from_s

関連データ構造

  • struct SN_env: ステミング環境を管理する構造体。
  • a_1: 接尾辞マッチングに使用される among 構造体の配列。

失敗モードと境界条件

  • 単語の末尾が指定された母音でない場合、または find_among_b でマッチしない場合は 0 を返す。
  • r_R1 が領域外と判定した場合、その戻り値をそのまま返す。
  • slice_from_s がメモリ不足等のエラーを返した場合、その負の値をそのまま返す。
