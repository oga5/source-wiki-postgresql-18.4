gcd

定義

  • 種別: function
  • 定義箇所: contrib/tsm_system_rows/tsm_system_rows.c:302

目的

2つの符号なし32ビット整数 a と b の最大公約数（Greatest Common Divisor）を計算します。

シグネチャまたは形状

static uint32 gcd(uint32 a, uint32 b)

入力と出力

  • **入力**:
  • a (uint32): 比較対象の数値。
  • b (uint32): 比較対象の数値。
  • **出力**:
  • uint32: a と b の最大公約数。

前提条件と不変条件

  • ユークリッドの互除法に基づき、計算過程で a が 0 になるまでループを繰り返します。

アルゴリズムまたは振る舞い

1. a が 0 でない間、以下の処理を繰り返します。

  • c に現在の a を退避します。
  • a を b % a で更新します。
  • b に退避した c を代入します。

    • 2. ループ終了後、最終的な b の値を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: 同一ファイル内の random_relative_prime 関数から呼び出されています（contrib/tsm_system_rows/tsm_system_rows.c:339）。
  • **呼び出し先**: 組み込みの剰余演算子 % を使用しています。

関連データ構造

  • なし。

失敗モードと境界条件

  • a が 0 の場合、ループは実行されず、入力された b がそのまま返されます。
  • a と b が共に 0 の場合、0 を返します。
