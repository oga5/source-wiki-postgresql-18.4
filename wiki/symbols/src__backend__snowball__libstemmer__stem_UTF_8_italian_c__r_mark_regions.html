r_mark_regions

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_italian.c:590

目的

イタリア語のステミング処理において、語幹抽出の基準となる領域（RV, R1, R2）を単語内で特定し、SN_env 構造体の整数配列 I にその境界位置を記録する。

シグネチャまたは形状

static int r_mark_regions(struct SN_env * z)

入力と出力

  • **入力**: z (Snowball ランタイム環境構造体)
  • **出力**: 常に 1 を返す。
  • **副作用**: z->I[0] (R2), z->I[1] (R1), z->I[2] (RV) に、単語内の各領域の開始位置（オフセット）が格納される。

前提条件と不変条件

  • z は有効な SN_env 構造体を指していること。
  • g_v（母音グループ）が定義されていること。
  • 処理対象の単語は z->p に格納されていること。

アルゴリズムまたは振る舞い

1. **初期化**: I[0], I[1], I[2] を単語の末尾位置 z->l で初期化する。 2. **RV 領域の特定**:

  • 母音の後に子音が続く箇所、または子音の後に母音が続く箇所を探索し、その後の位置を RV とする。
  • 該当しない場合は、単語の先頭から2文字目以降の適切な位置を探索する。

    • 3. **R1, R2 領域の特定**:

  • 母音と子音の遷移（V-C または C-V）を順次探索する。
  • 最初の遷移後の位置を R1 とし、さらにその後の遷移後の位置を R2 とする。

    • 4. **復元**: 処理中に変更したカーソル位置 z->c を元の位置に復元する。

呼び出し元と呼び出し先

  • **呼び出し元**: 提供されたソース excerpt には明示的な呼び出し元は記載されていない。
  • **呼び出し先**: in_grouping_U, out_grouping_U, skip_utf8 を呼び出し、文字の属性判定とカーソル移動を行う。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。I 配列（整数値）、c（カーソル）、l（文字列長）、p（文字列データ）を含む。
  • g_v: 母音を定義するグループ。

失敗モードと境界条件

  • 文字列が空、または領域を特定できない場合、各領域のオフセットは初期値（文字列末尾）のままとなる。
  • skip_utf8 が負の値を返した場合、領域特定処理を中断し、現在のカーソル位置を保持して終了する。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードである。ロジックの変更が必要な場合は、直接編集せず対応する .sbl ソースファイルを修正すること。
  • I[0]=R2, I[1]=R1, I[2]=RV というマッピングは、他の言語用ステマーと共通の規約に基づいている。
