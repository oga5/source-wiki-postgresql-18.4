GetForeignServer

定義

  • 種別: function
  • 定義箇所: src/backend/foreign/foreign.c:111

目的

指定された OID に対応する外部サーバー（Foreign Server）の定義情報を取得します。

シグネチャまたは形状

ForeignServer *GetForeignServer(Oid serverid)

入力と出力

  • **入力**: serverid (Oid) - 取得対象の外部サーバーの OID。
  • **出力**: ForeignServer * - 外部サーバーの定義情報を含む構造体へのポインタ。

前提条件と不変条件

  • 指定された serverid がシステムカタログ pg_foreign_server に存在する必要があります。存在しない場合、内部で呼び出される GetForeignServerExtended によりエラーが発生します。

アルゴリズムまたは振る舞い

1. 引数として受け取った serverid を使用して、GetForeignServerExtended(serverid, 0) を呼び出します。 2. GetForeignServerExtended が返す ForeignServer 構造体へのポインタをそのまま呼び出し元へ返します。

呼び出し元と呼び出し先

  • **呼び出し先**: GetForeignServerExtended
  • **呼び出し元**:
  • contrib/file_fdw/file_fdw.c
  • contrib/postgres_fdw/connection.c
  • contrib/postgres_fdw/postgres_fdw.c
  • src/backend/catalog/objectaddress.c
  • src/backend/commands/tablecmds.c

関連データ構造

  • ForeignServer: 外部サーバーの定義情報を保持する構造体（foreign/foreign.h で定義）。

失敗モードと境界条件

  • 指定した serverid が存在しない場合、GetForeignServerExtended の仕様に従いエラーが発生します。

保守担当者向けメモ

  • 本関数は GetForeignServerExtended のラッパーであり、エラー制御フラグ（MISSING_OK）を無効にした状態で呼び出します。
  • 外部サーバーの存在が不確実な場合は、直接 GetForeignServerExtended を使用して MISSING_OK フラグを制御することを検討してください。
