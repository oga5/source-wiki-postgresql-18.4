action_to_str

定義

  • 種別: function
  • 定義箇所: src/bin/pg_rewind/filemap.c:473

目的

file_action_t 列挙型の値を、デバッグログや出力メッセージに適した人間が読める文字列に変換する。

シグネチャまたは形状

static const char *action_to_str(file_action_t action)

入力と出力

  • **入力**: file_action_t 型の列挙値。
  • **出力**: 対応するアクション名を表す const char * 型の文字列。

前提条件と不変条件

  • 入力値が file_action_t で定義された有効な列挙値であること。
  • 未定義の列挙値が渡された場合は、文字列 "unknown" を返す。

アルゴリズムまたは振る舞い

switch 文を使用して、以下のマッピングに従い文字列を返す：

  • FILE_ACTION_NONE → "NONE"
  • FILE_ACTION_COPY → "COPY"
  • FILE_ACTION_TRUNCATE → "TRUNCATE"
  • FILE_ACTION_COPY_TAIL → "COPY_TAIL"
  • FILE_ACTION_CREATE → "CREATE"
  • FILE_ACTION_REMOVE → "REMOVE"
  • その他 → "unknown"

呼び出し元と呼び出し先

  • **呼び出し元**: src/bin/pg_rewind/filemap.c 内のログ出力処理（例: entry->action の表示時）。
  • **呼び出し先**: なし。

関連データ構造

  • file_action_t: 同期アクションを定義する列挙型。

失敗モードと境界条件

  • 定義されていない列挙値が渡された場合、プログラムの異常終了は発生せず、安全に "unknown" を返す。

保守担当者向けメモ

  • 新しい file_action_t の値が追加された場合は、必ずこの関数に case 文を追加すること。
