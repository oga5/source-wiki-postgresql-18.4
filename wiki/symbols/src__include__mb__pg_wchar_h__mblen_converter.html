mblen_converter

定義

  • 種別: typedef
  • 定義箇所: src/include/mb/pg_wchar.h:382

目的

マルチバイト文字列の先頭文字のバイト長を計算するための関数ポインタ型です。

シグネチャまたは形状

typedef int (*mblen_converter) (const unsigned char *mbstr);

入力と出力

  • **入力**: const unsigned char *mbstr - 評価対象となるマルチバイト文字列の先頭ポインタ。
  • **出力**: int - 文字列の先頭にある単一文字のバイト長。

前提条件と不変条件

  • 入力ポインタは有効なメモリ領域を指している必要があります。
  • 返される値は、対象エンコーディングにおける文字のバイト長（通常は1以上）となります。

アルゴリズムまたは振る舞い

特定のエンコーディングルールに従い、mbstr が指す先頭のマルチバイト文字が何バイトで構成されているかを判定して返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/mb/mbutils.c:1222 にて変数定義として参照されています。
  • src/include/mb/pg_wchar.h:398 にて pg_wchar_tbl 構造体のメンバ mblen として定義されています。
  • **呼び出し先**:
  • 提供されたソース情報からは、具体的な呼び出し先関数は特定できません。

関連データ構造

  • pg_wchar_tbl: 各エンコーディングの変換関数を保持するテーブル構造体であり、本型はそのメンバとして使用されます。

失敗モードと境界条件

  • 入力文字列が不正なバイト列である場合や、NULLポインタが渡された場合の挙動については、各エンコーディング固有の実装に依存します。

保守担当者向けメモ

  • 本型は pg_wchar_tbl を通じて動的に選択されるため、新しいエンコーディングを追加する際は、このシグネチャに適合する関数を実装し、テーブルに登録する必要があります。
