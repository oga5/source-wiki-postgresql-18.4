DestroyMetaString

定義

  • 種別: function
  • 定義箇所: contrib/fuzzystrmatch/dmetaphone.c:259

目的

metastring 構造体およびその内部で保持されている文字列バッファを解放し、メモリをクリーンアップします。

シグネチャまたは形状

static void DestroyMetaString(metastring *s)

入力と出力

  • **入力**: metastring *s （解放対象の構造体ポインタ）
  • **出力**: なし（void）

前提条件と不変条件

  • 引数 s が NULL の場合、関数は何も行わず正常に終了します。
  • s->free_string_on_destroy が真である場合、s->str が指すメモリ領域が解放されます。

アルゴリズムまたは振る舞い

1. 引数 s が NULL かどうかを確認し、NULL であれば即座にリターンします。 2. s->free_string_on_destroy フラグが有効かつ s->str が NULL でない場合、META_FREE マクロを使用して文字列バッファを解放します。 3. 最後に META_FREE マクロを使用して、構造体 s 自体のメモリ領域を解放します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/fuzzystrmatch/dmetaphone.c 内の DoubleMetaphone 関数（original, primary, secondary の各構造体の破棄に使用）。
  • **呼び出し先**:
  • META_FREE マクロ（メモリ解放用）。

関連データ構造

  • metastring: 本関数が管理・破棄を行う対象の構造体。

失敗モードと境界条件

  • s が NULL の場合、安全に処理を終了します。
  • s->str が NULL の場合、文字列バッファの解放はスキップされます。

保守担当者向けメモ

  • 本関数は PostgreSQL のメモリコンテキスト管理に依存する META_FREE マクロを使用しています。メモリリークを防ぐため、NewMetaString で確保された構造体は必ず本関数で破棄してください。
