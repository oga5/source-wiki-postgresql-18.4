gbt_macadgt

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_macaddr.c:30

目的

macaddr 型の2つの値に対して「より大きい（greater than）」という比較を行い、その結果をGiSTインデックスの検索処理へ提供する。

シグネチャまたは形状

bool gbt_macadgt(const void *a, const void *b, FmgrInfo *flinfo)

入力と出力

  • **入力**:
  • a, b: 比較対象の macaddr 値へのポインタ。
  • flinfo: 関数呼び出し情報（PostgreSQLの fmgr インターフェース用）。
  • **出力**:
  • a > b であれば true、そうでなければ false を返す。

前提条件と不変条件

  • 入力ポインタ a および b は有効な macaddr 型のデータであること。
  • PostgreSQLの macaddr_gt 関数が利用可能であること。

アルゴリズムまたは振る舞い

1. PointerGetDatum を使用して、入力されたポインタをPostgreSQLの Datum 型に変換する。 2. DirectFunctionCall2 を介して、PostgreSQL標準の macaddr_gt 関数を呼び出す。 3. 戻り値を DatumGetBool で bool 型に変換して返す。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/btree_gist/btree_macaddr.c 内の静的構造体 tinfo（gbtree_ninfo 型）から参照されている。
  • **呼び出し先**: PostgreSQL内部の macaddr_gt 関数。

関連データ構造

  • macaddr: 比較対象となるPostgreSQLのネットワークMACアドレス型。
  • gbtree_ninfo: 本関数が登録されている比較設定テーブル。

失敗モードと境界条件

  • 入力が NULL の場合、DirectFunctionCall2 の仕様に従い、呼び出し側で適切にハンドリングされる必要がある。
  • macaddr_gt が内部的にエラーを発生させた場合、その例外が伝播する。

保守担当者向けメモ

  • 本関数は btree_utils_num が期待する比較関数のシグネチャに適合させている。
  • macaddr 型の比較ロジックを変更する場合は、src/backend/utils/adt/mac.c の標準関数との整合性に注意すること。
