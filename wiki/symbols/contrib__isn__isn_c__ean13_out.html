ean13_out

定義

  • 種別: function
  • 定義箇所: contrib/isn/isn.c:972

目的

PostgreSQL の内部型 ean13 を、人間が読みやすい文字列形式（C-string）に変換して出力する。

シグネチャまたは形状

Datum ean13_out(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS を介して ean13 型の値を 1 つ受け取る。
  • **出力**: Datum 型として、変換された文字列（char *）を返す。

前提条件と不変条件

  • PG_GETARG_EAN13(0) を使用して入力値を取得する。
  • 出力バッファ buf は MAXEAN13LEN + 1 のサイズを確保している。
  • ean2string 関数を呼び出し、ハイフンなしの形式で文字列化を行う。

アルゴリズムまたは振る舞い

1. PG_GETARG_EAN13(0) で ean13 値を取得する。 2. ean2string(val, false, buf, false) を呼び出し、val を buf に文字列として書き出す。 3. pstrdup(buf) を使用して、メモリコンテキスト上に文字列のコピーを作成する。 4. PG_RETURN_CSTRING(result) を介して結果を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: PostgreSQL の関数管理システム（PG_FUNCTION_INFO_V1 マクロにより登録）。具体的な SQL レベルの呼び出し元はソースコード上では不明。
  • **呼び出し先**: ean2string（文字列変換）、pstrdup（メモリ確保）。

関連データ構造

  • ean13: 内部ストレージ形式（uint64）。
  • MAXEAN13LEN: 文字列変換時の最大長を規定する定数。

失敗モードと境界条件

  • ean2string の内部処理に依存する。
  • メモリ不足時には pstrdup がエラーを発生させる可能性がある。

保守担当者向けメモ

  • 本関数は ean13 型の出力関数として登録されている。
  • ean2string の第 2 引数および第 4 引数に false を渡しており、ハイフン挿入を行わない形式で出力される仕様である。
