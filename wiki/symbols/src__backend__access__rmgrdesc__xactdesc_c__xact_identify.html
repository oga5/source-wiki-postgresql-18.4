xact_identify

定義

  • 種別: function
  • 定義箇所: src/backend/access/rmgrdesc/xactdesc.c:486

目的

WALレコードの info ビットから、トランザクションに関連する操作種別を識別し、対応するラベル文字列を返します。

シグネチャまたは形状

const char *xact_identify(uint8 info)

入力と出力

  • **入力**: uint8 info （WALレコードの info フィールド）
  • **出力**: const char * （識別された操作種別を表す文字列定数。該当しない場合は NULL）

前提条件と不変条件

  • 入力 info は XLOG_XACT_OPMASK でマスクされ、トランザクション関連の操作ビットのみが評価されます。

アルゴリズムまたは振る舞い

1. info を XLOG_XACT_OPMASK でビットマスクし、操作種別を抽出します。 2. switch 文を用いて以下のいずれかの定数と照合します。

  • XLOG_XACT_COMMIT -> "COMMIT"
  • XLOG_XACT_PREPARE -> "PREPARE"
  • XLOG_XACT_ABORT -> "ABORT"
  • XLOG_XACT_COMMIT_PREPARED -> "COMMIT_PREPARED"
  • XLOG_XACT_ABORT_PREPARED -> "ABORT_PREPARED"
  • XLOG_XACT_ASSIGNMENT -> "ASSIGNMENT"
  • XLOG_XACT_INVALIDATIONS -> "INVALIDATION"

    • 3. 一致した文字列へのポインタを返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/include/access/rmgrlist.h 内の PG_RMGR マクロ定義において、リソースマネージャの識別子として登録されています。
  • **呼び出し先**:
  • 本関数は直接他の関数を呼び出しません。

関連データ構造

  • uint8 (型)
  • XLOG_XACT_* 定数群（access/xact.h で定義）

失敗モードと境界条件

  • info が定義されたいずれの XLOG_XACT_* フラグにも合致しない場合、NULL を返します。

保守担当者向けメモ

  • 本関数は pg_waldump 等のツールから参照される共有コードです。新しいトランザクション関連のWALレコードタイプを追加した際は、必ず本関数の switch 文にもケースを追加してください。
