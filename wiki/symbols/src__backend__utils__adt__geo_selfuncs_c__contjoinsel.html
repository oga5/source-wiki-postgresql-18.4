contjoinsel

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/geo_selfuncs.c:91

目的

幾何演算子における包含関係（contain）の選択率（selectivity）を推定し、クエリプランナに対して固定の推定値を返却する。

シグネチャまたは形状

Datum contjoinsel(PG_FUNCTION_ARGS)

入力と出力

  • 入力: PG_FUNCTION_ARGS（SQL関数呼び出し規約に基づく引数群）。
  • 出力: float8 型の固定値 0.001 を PG_RETURN_FLOAT8 マクロ経由で返却する。

前提条件と不変条件

  • 入力されたクエリの統計情報やデータ分布は一切参照しない。
  • 常に固定値 0.001 を返却する。

アルゴリズムまたは振る舞い

1. 関数が呼び出されると、統計的な計算を行わずに即座に 0.001 を返却する。 2. この値は、プランナが幾何インデックス（GiST等）の利用を検討する際に、インデックス利用を促すためのヒューリスティックな値として機能する。

呼び出し元と呼び出し先

  • 呼び出し元: 提供されたソース情報からは特定不可。
  • 呼び出し先: なし（内部で他の関数を呼び出さない）。

関連データ構造

  • Datum: PostgreSQLの汎用データ型表現。
  • PG_FUNCTION_ARGS: 関数マネージャ（fmgr）が使用する引数構造体。

失敗モードと境界条件

  • 統計情報に基づかない固定値返却であるため、実際のデータ分布と乖離する可能性がある。
  • 境界条件（入力値の範囲や型）による挙動の変化はない。

保守担当者向けメモ

  • 本実装は統計情報に基づかない「完全にデタラメ（totally bogus）」な固定値である。
  • 幾何学的なデータ分布を考慮した真の選択率推定アルゴリズムが実装されるまでの暫定的な処置である。
