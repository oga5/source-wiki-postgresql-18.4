pg_euckr2wchar_with_len

定義

  • 種別: function
  • 定義箇所: src/common/wchar.c:214

目的

EUC-KR エンコーディングの文字列を、PostgreSQL 内部で使用されるワイド文字形式（pg_wchar）へ変換します。

シグネチャまたは形状

static int pg_euckr2wchar_with_len(const unsigned char *from, pg_wchar *to, int len)

入力と出力

  • from: 変換元となる EUC-KR エンコーディングの文字列（unsigned char *）。
  • to: 変換後のワイド文字を格納するバッファ（pg_wchar *）。
  • len: 変換対象のバイト長。
  • 戻り値: 変換されたワイド文字数（int）。

前提条件と不変条件

  • from が指すメモリ領域は、少なくとも len バイト以上の有効なデータを含んでいる必要があります。
  • to が指すバッファは、変換結果を格納するために十分なサイズ（len 以上の pg_wchar 要素数）を確保している必要があります。

アルゴリズムまたは振る舞い

内部的に pg_euc2wchar_with_len を呼び出し、EUC-KR 特有のエンコーディング規則に従ってワイド文字への変換を実行します。

呼び出し元と呼び出し先

  • 呼び出し先: pg_euc2wchar_with_len
  • 呼び出し元: src/common/wchar.c 内の pg_wchar_table（PG_EUC_KR エントリとして登録されています）。

関連データ構造

  • pg_wchar: PostgreSQL 内部で文字を表現するための型。
  • pg_wchar_table: 各エンコーディングの操作関数を保持するグローバルな関数テーブル。

失敗モードと境界条件

  • len が 0 の場合、変換は行われず 0 を返します。
  • 不正なバイトシーケンスが含まれる場合の挙動は、内部で呼び出される pg_euc2wchar_with_len の実装に依存します。

保守担当者向けメモ

  • 本関数は pg_euc2wchar_with_len への単なるラッパーです。EUC-KR 固有の変換ロジックを変更する必要がある場合は、呼び出し先の関数を確認してください。
