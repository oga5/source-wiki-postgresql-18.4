finalize_manifest

定義

  • 種別: function
  • 定義箇所: src/bin/pg_combinebackup/write_manifest.c:141

目的

バックアップマニフェストの生成を完了させ、WAL レンジの記録、最終的なチェックサムの計算、およびマニフェストファイルのクローズを行う。

シグネチャまたは形状

void finalize_manifest(manifest_writer *mwriter, manifest_wal_range *first_wal_range);

入力と出力

  • **入力**:
  • mwriter: マニフェスト生成の状態を保持する manifest_writer 構造体へのポインタ。
  • first_wal_range: 記録対象となる WAL レンジのリストの先頭要素。
  • **出力**: なし（副作用として mwriter が保持するバッファおよびファイルへ書き出しを行う）。

前提条件と不変条件

  • mwriter は create_manifest_writer によって初期化済みであること。
  • mwriter->manifest_ctx は、マニフェストの内容を正確に反映するために、これまでのデータ書き込みと同期して更新されていること。

アルゴリズムまたは振る舞い

1. ファイルリストの JSON 配列を閉じる（\n],\n を追記）。 2. "WAL-Ranges" キーを開始し、first_wal_range リストを走査して各要素（Timeline, Start-LSN, End-LSN）を JSON 形式でバッファに追記する。 3. WAL レンジの JSON 配列を閉じる。 4. flush_manifest を呼び出し、バッファ内のデータをディスクへフラッシュする。 5. mwriter->still_checksumming を false に設定し、チェックサム計算を終了する。 6. pg_checksum_final を用いて最終的な SHA-256 チェックサムを計算し、16 進数文字列に変換して "Manifest-Checksum" フィールドとしてバッファに追記する。 7. JSON オブジェクトを閉じる（}\n を追記）。

呼び出し元と呼び出し先

  • **呼び出し元**: src/bin/pg_combinebackup/pg_combinebackup.c:417
  • **呼び出し先**:
  • appendStringInfoString / appendStringInfo (lib/stringinfo.h)
  • flush_manifest (src/bin/pg_combinebackup/write_manifest.c)
  • pg_checksum_final (common/checksum_helper.h)
  • hex_encode (src/bin/pg_combinebackup/write_manifest.c)

関連データ構造

  • manifest_writer: マニフェスト生成のコンテキスト（バッファ、チェックサム計算用コンテキストを含む）。
  • manifest_wal_range: WAL レンジ情報を保持するリンクリスト構造体。

失敗モードと境界条件

  • enlargeStringInfo によるメモリ確保失敗時は、StringInfo の内部仕様に従いプログラムが終了する可能性がある。
  • pg_checksum_final の結果長が PG_SHA256_DIGEST_LENGTH と一致しない場合、Assert によりデバッグビルドで停止する。

保守担当者向けメモ

  • 本関数はマニフェストの最終的な整合性を保証する重要なステップである。チェックサム計算の範囲は flush_manifest を通じて管理されているため、この関数内での追加書き込みがチェックサム計算後に漏れないよう注意すること。
