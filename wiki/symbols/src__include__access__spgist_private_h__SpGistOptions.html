SpGistOptions

定義

  • 種別: struct
  • 定義箇所: src/include/access/spgist_private.h:26

目的

SP-GiSTインデックスのオプション（ストレージパラメータ）を保持するための構造体です。

シグネチャまたは形状

typedef struct SpGistOptions
{
    int32       varlena_header_;
    int         fillfactor;
} SpGistOptions;

入力と出力

  • **入力**: インデックス作成時や ALTER INDEX 時に指定される fillfactor パラメータ。
  • **出力**: インデックスの物理ページにおける充填率（0〜100%）の制御に使用されます。

前提条件と不変条件

  • varlena_header_ は直接操作してはなりません。
  • fillfactor は 0 から 100 の範囲である必要があります。

アルゴリズムまたは振る舞い

インデックスのメタデータとして保持され、インデックスページへのタプル挿入時にページ内の空き領域をどの程度残すかを決定するために参照されます。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/access/spgist/spgutils.c 内で、インデックスのオプション設定やサイズ計算時に参照されています。
  • **呼び出し先**: 直接的な関数呼び出し関係はソースから確認できませんが、リレーションの rd_options を介してインデックスの物理レイアウト制御に利用されます。

関連データ構造

  • Relation: インデックスのリレーションキャッシュ構造体。rd_options フィールドを通じて本構造体にアクセスします。

失敗モードと境界条件

  • fillfactor が 0〜100 の範囲外である場合、インデックスのページ管理において予期せぬ動作を引き起こす可能性があります。

保守担当者向けメモ

  • 本構造体は varlena 型として扱われるため、メモリレイアウトの変更には注意が必要です。
  • src/include/access/spgist_private.h:36 にあるマクロ定義を通じて、リレーションから fillfactor を抽出する実装が提供されています。
