TypeCategory

定義

  • 種別: function
  • 定義箇所: src/backend/parser/parse_coerce.c:2975

目的

指定されたデータ型（Oid）に対応する型カテゴリ（TYPCATEGORY）を取得します。

シグネチャまたは形状

TYPCATEGORY TypeCategory(Oid type)

入力と出力

  • **入力**: type (Oid) - 調査対象のデータ型ID。
  • **出力**: TYPCATEGORY - 型カテゴリを表す列挙値。

前提条件と不変条件

  • get_type_category_preferred を介して取得したカテゴリが TYPCATEGORY_INVALID ではないことを保証するアサーションが含まれています。

アルゴリズムまたは振る舞い

1. get_type_category_preferred を呼び出し、対象の型IDからカテゴリと優先フラグを取得します。 2. 取得したカテゴリが有効であることを確認します。 3. 取得したカテゴリを TYPCATEGORY 型にキャストして返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/commands/indexcmds.c
  • src/backend/parser/parse_coerce.c
  • src/backend/parser/parse_expr.c
  • src/backend/parser/parse_func.c
  • **呼び出し先**:
  • get_type_category_preferred (内部で呼び出し)

関連データ構造

  • TYPCATEGORY: 型カテゴリを定義する列挙型。
  • Oid: PostgreSQLのシステムカタログにおけるオブジェクト識別子。

失敗モードと境界条件

  • get_type_category_preferred が無効なカテゴリを返した場合、Assert によりプロセスが停止する可能性があります。

保守担当者向けメモ

  • 本関数は src/include/parser/parse_coerce.h で公開されています。
