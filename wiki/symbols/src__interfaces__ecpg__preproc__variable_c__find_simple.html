find_simple

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/preproc/variable.c:176

目的

静的な変数リスト allvariables から、指定された名前と完全に一致する変数エントリを検索します。

シグネチャまたは形状

static struct variable *find_simple(const char *name)

入力と出力

  • **入力**: name (検索対象の変数名を示す文字列)
  • **出力**: 見つかった場合は該当する struct variable へのポインタ、見つからない場合は NULL を返します。

前提条件と不変条件

  • allvariables は、ECPGプリプロセッサで宣言された全変数を保持するリンクリストとして初期化されている必要があります。
  • 検索対象の name は、構造体メンバや配列アクセスを含まない単純な変数名であることを想定しています。

アルゴリズムまたは振る舞い

1. 静的変数 allvariables を先頭から末尾まで走査します。 2. 各要素の name フィールドと引数の name を strcmp で比較します。 3. 一致する要素が見つかった時点でそのポインタを返し、リストの終端に達した場合は NULL を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/interfaces/ecpg/preproc/variable.c 内の find_variable 関数から呼び出されています。
  • **呼び出し先**: 標準ライブラリの strcmp を使用しています。

関連データ構造

  • struct variable: 変数の名前や型情報を保持する構造体。
  • allvariables: 変数リストを管理する静的なリンクリスト。

失敗モードと境界条件

  • リストが空の場合、または一致する名前が存在しない場合は NULL を返します。
  • name に NULL が渡された場合の挙動は、標準ライブラリの strcmp の仕様に依存します。

保守担当者向けメモ

  • 本関数は単純な名前一致のみを扱うため、構造体メンバの解決が必要な場合は find_struct_member 等の再帰的な検索関数を使用してください。
  • allvariables のリスト構造に変更を加える際は、本関数による検索結果が正しく維持されるか確認してください。
