out_grouping_b_U

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/utilities.c:153

目的

UTF-8 エンコードされた文字列において、現在のカーソル位置から後方（逆方向）へ向かって、指定された文字集合（グループ）に**含まれない**文字を検索し、カーソルを移動させる。

シグネチャまたは形状

int out_grouping_b_U(struct SN_env * z, const unsigned char * s, int min, int max, int repeat)

入力と出力

  • z: ステミング環境を保持する SN_env 構造体へのポインタ。
  • s: 文字グループをビットマップ形式で保持する配列。
  • min, max: 判定対象となる文字コードの範囲。
  • repeat: 0 以外の場合、条件に合致する間、連続して後方へカーソルを移動させる。
  • **戻り値**: 成功時は移動したバイト数（正の整数）、失敗時は 0 を返す。UTF-8 デコードに失敗した場合は -1 を返す。

前提条件と不変条件

  • z->c は現在のカーソル位置を指す。
  • s は min から max までの文字範囲をカバーするビットマップである必要がある。
  • get_b_utf8 を使用して後方へ文字をデコードするため、バッファの境界（z->lb）が正しく設定されている必要がある。

アルゴリズムまたは振る舞い

1. get_b_utf8 を呼び出し、現在のカーソル位置から 1 文字分後方の文字コード ch を取得する。 2. ch が min 未満または max より大きい場合、あるいはビットマップ s 上で該当ビットが立っていない（＝グループに含まれない）場合、その文字は「グループ外」と判定される。 3. グループ外と判定された場合、その文字のバイト数 w を戻り値として返し、カーソル z->c を w 分だけ後方へ移動させる。 4. repeat が真であれば、この処理をループして継続する。 5. グループに含まれる文字に遭遇した時点で処理を終了し、0 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: stem_UTF_8_dutch.c, stem_UTF_8_english.c, stem_UTF_8_french.c 等の各言語ステミング実装。
  • **呼び出し先**: get_b_utf8（src/backend/snowball/libstemmer/utilities.c 内）。

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • among: 辞書検索等で使用される構造体（本関数では直接使用しないが、モジュール内で関連）。

失敗モードと境界条件

  • get_b_utf8 が 0 を返した場合（不正な UTF-8 シーケンスやバッファ境界）、本関数も -1 を返し処理を中断する。
  • repeat が 0 の場合、1 文字だけ判定して終了する。

保守担当者向けメモ

  • 本関数は UTF-8 専用である。非 UTF-8 環境ではサフィックスなしの out_grouping_b を使用すること。
  • z->c は副作用として更新されるため、呼び出し順序がステミングロジックの正確性に直結する。
