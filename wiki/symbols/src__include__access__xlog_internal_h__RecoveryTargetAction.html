RecoveryTargetAction

定義

  • 種別: enum
  • 定義箇所: src/include/access/xlog_internal.h:322

目的

リカバリ処理がターゲット（特定の時点や状態）に到達した際に、システムが実行すべきアクションを定義します。

シグネチャまたは形状

typedef enum
{
	RECOVERY_TARGET_ACTION_PAUSE,
	RECOVERY_TARGET_ACTION_PROMOTE,
	RECOVERY_TARGET_ACTION_SHUTDOWN,
} RecoveryTargetAction;

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: なし

前提条件と不変条件

  • リカバリプロセスにおいて、ターゲット到達時の挙動を決定するために使用されます。

アルゴリズムまたは振る舞い

  • RECOVERY_TARGET_ACTION_PAUSE: リカバリを一時停止します。
  • RECOVERY_TARGET_ACTION_PROMOTE: リカバリを終了し、スタンバイからプライマリへ昇格します。
  • RECOVERY_TARGET_ACTION_SHUTDOWN: リカバリを終了し、シャットダウンします。

呼び出し元と呼び出し先

  • 呼び出し元および呼び出し先の具体的な関数情報は、提供されたソース excerpt には含まれていません。

関連データ構造

  • 本シンボルは src/include/access/xlog_internal.h で定義されており、WALリカバリに関連する他の構造体や処理と組み合わせて使用されます。

失敗モードと境界条件

  • 定義された3つの値以外の値が設定された場合の挙動については、本ソース excerpt に情報がありません。

保守担当者向けメモ

  • 本列挙型はリカバリの制御フローに直結するため、新しいアクションを追加する場合は、リカバリ処理側のロジック（xlog.c 等）の更新が必要です。
