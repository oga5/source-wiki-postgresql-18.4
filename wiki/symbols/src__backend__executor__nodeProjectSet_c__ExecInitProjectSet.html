ExecInitProjectSet

定義

  • 種別: function
  • 定義箇所: src/backend/executor/nodeProjectSet.c:226

目的

ProjectSet 実行ノードの初期化を行い、ターゲットリストに含まれる集合を返す関数（SRF）の評価環境を構築する。

シグネチャまたは形状

ProjectSetState *ExecInitProjectSet(ProjectSet *node, EState *estate, int eflags)

入力と出力

  • **入力**:
  • node: 初期化対象の ProjectSet プランノード。
  • estate: 実行状態を保持する EState。
  • eflags: 実行フラグ（EXEC_FLAG_MARK および EXEC_FLAG_BACKWARD はサポート外）。
  • **出力**:
  • 初期化済みの ProjectSetState 構造体へのポインタ。

前提条件と不変条件

  • eflags に EXEC_FLAG_MARK または EXEC_FLAG_BACKWARD が含まれていないこと。
  • node の innerPlan が NULL であること。
  • node の plan.qual が空（NIL）であること。
  • SRF はターゲットリストのトップレベルに存在することを前提とする。

アルゴリズムまたは振る舞い

1. ProjectSetState 構造体を割り当て、ExecProcNode に ExecProjectSet を設定する。 2. ExecAssignExprContext で式評価コンテキストを作成する。 3. ExecInitNode を呼び出し、子ノード（outerPlan）を初期化する。 4. ExecInitResultTupleSlotTL で結果タプルスロットを初期化する。 5. ターゲットリストの各要素を走査し、SRF（FuncExpr または OpExpr で retset が真のもの）であれば ExecInitFunctionResultSet を、それ以外は ExecInitExpr を使用してコンパイルする。 6. SRF の引数評価用に専用のメモリコンテキスト argcontext を作成する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/executor/execProcnode.c（ExecInitNode 経由での呼び出しが確認されている）。
  • **呼び出し先**:
  • ExecInitNode（子ノードの初期化）
  • ExecInitFunctionResultSet（SRF の初期化）
  • ExecInitExpr（通常の式評価の初期化）
  • AllocSetContextCreate（メモリコンテキストの作成）

関連データ構造

  • ProjectSetState: ノードの実行時状態を保持する構造体。
  • ProjectSet: プランノード。
  • ExprDoneCond: SRF の評価状態を追跡する配列。
  • argcontext: SRF 引数評価用のメモリコンテキスト。

失敗モードと境界条件

  • eflags にサポート外のフラグが含まれている場合、Assert により停止する。
  • innerPlan が存在する場合、Assert により停止する。
  • qual が設定されている場合、Assert により停止する。

保守担当者向けメモ

  • argcontext はタプルごとのコンテキストとはライフサイクルが異なるため、リセットのタイミングに注意すること。
  • 本関数は ExecBuildProjectionInfo を使用できないため、ターゲットリストの各要素を個別にコンパイルしている点に留意すること。
