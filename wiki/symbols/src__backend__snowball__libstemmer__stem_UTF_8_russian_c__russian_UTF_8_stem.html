russian_UTF_8_stem

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_russian.c:568

目的

UTF-8エンコーディングされたロシア語テキストに対し、Snowballアルゴリズムを用いて語幹抽出（ステミング）を行う。

シグネチャまたは形状

int russian_UTF_8_stem(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（Snowball実行環境。入力文字列および処理状態を保持）
  • **出力**: int（成功時に 1、失敗時に負の値を返す）

前提条件と不変条件

  • z は russian_UTF_8_create_env 等で初期化された有効な環境であること。
  • 内部状態 z->I 配列には、r_mark_regions によって計算された語幹抽出用の境界情報が格納されている必要がある。

アルゴリズムまたは振る舞い

1. **正規化**: 入力文字列内の特定の文字（ёなど）を正規化する。 2. **領域設定**: r_mark_regions を呼び出し、ロシア語の文法規則に基づいた領域（R1, R2）を定義する。 3. **接尾辞削除**: 以下の順序で後方からマッチングを行い、該当する接尾辞を削除する。

  • 完了分詞 (r_perfective_gerund)
  • 再帰動詞 (r_reflexive)
  • 形容詞 (r_adjectival)
  • 動詞 (r_verb)
  • 名詞 (r_noun)

    • 4. **後処理**: 派生語の処理 (r_derivational) および最終的な語尾調整 (r_tidy_up) を実行する。 5. **状態復元**: 処理中に変更されたカーソル位置や境界情報を適切に復元して終了する。

呼び出し元と呼び出し先

  • **呼び出し元**: 直接的な呼び出し元はソースコード上では明示されていないが、libstemmer のAPI経由で呼び出されることが想定される。
  • **呼び出し先**: r_mark_regions, r_perfective_gerund, r_reflexive, r_adjectival, r_verb, r_noun, r_derivational, r_tidy_up, slice_del, slice_from_s, eq_s, eq_s_b, skip_utf8。

関連データ構造

  • struct SN_env: Snowballランタイムの共通環境構造体。

失敗モードと境界条件

  • slice_from_s や slice_del などの内部関数が負の値を返した場合、そのエラーコードをそのまま伝播させる。
  • 領域設定が不十分な場合や、メモリ操作に関連するエラーが発生した場合は負の値を返す。

保守担当者向けメモ

  • 本関数はSnowballコンパイラによって自動生成されている。ロジックの変更が必要な場合は、直接Cファイルを編集せず、元のSnowballスクリプト（.sbl）を修正すること。
  • ロシア語のステミングは後方からのマッチング（find_among_b）が中心であるため、処理順序や領域の定義には注意が必要である。
