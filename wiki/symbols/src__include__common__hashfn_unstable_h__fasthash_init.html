fasthash_init

定義

  • 種別: function
  • 定義箇所: src/include/common/hashfn_unstable.h:109

目的

ハッシュ計算の状態を保持する fasthash_state 構造体を初期化し、指定されたシード値を用いてハッシュの初期値を設定します。

シグネチャまたは形状

static inline void fasthash_init(fasthash_state *hs, uint64 seed)

入力と出力

  • **入力**:
  • hs: 初期化対象となる fasthash_state 構造体へのポインタ。
  • seed: ハッシュ計算の開始点として使用する uint64 型のシード値。
  • **出力**: なし（hs が指すメモリ領域が直接更新されます）。

前提条件と不変条件

  • hs は有効な fasthash_state 構造体を指している必要があります。
  • 本関数は fasthash アルゴリズムの初期化フェーズとして使用されます。

アルゴリズムまたは振る舞い

1. memset を使用して hs が指す fasthash_state 構造体の全領域を 0 で初期化します。 2. 構造体内の hash メンバに、入力された seed と定数 0x880355f21e6d1965 との XOR 演算結果を格納します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/catalog/namespace.c:259
  • src/include/common/hashfn_unstable.h:61
  • src/include/common/hashfn_unstable.h:364
  • src/include/common/hashfn_unstable.h:396
  • **呼び出し先**:
  • 標準ライブラリの memset を使用します。

関連データ構造

  • fasthash_state: ハッシュ計算の進行状況を保持する構造体。

失敗モードと境界条件

  • hs が NULL の場合、memset の呼び出しにより未定義の動作が発生します。

保守担当者向けメモ

  • 本関数は「不安定」なハッシュ実装の一部です。プラットフォームやバージョン間で結果が一致しないため、永続化（ディスク保存）が必要なデータには使用しないでください。
  • fasthash_accum 等の他の fasthash 系関数と組み合わせて使用することを前提としています。
