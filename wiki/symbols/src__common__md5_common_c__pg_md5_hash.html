pg_md5_hash

定義

  • 種別: function
  • 定義箇所: src/common/md5_common.c:73

目的

入力バッファからMD5ハッシュを計算し、その結果を32文字の16進文字列として出力します。

シグネチャまたは形状

bool pg_md5_hash(const void *buff, size_t len, char *hexsum, const char **errstr)

入力と出力

  • **入力**:
  • buff: ハッシュ計算対象のデータバッファ。
  • len: buff のバイト長。
  • **出力**:
  • hexsum: 計算されたMD5ハッシュ（32文字の16進数文字列）が格納されるバッファ（呼び出し元で確保が必要）。
  • errstr: 失敗時にエラーメッセージが代入されるポインタ。
  • 戻り値: 成功時に true、失敗時に false を返します。

前提条件と不変条件

  • hexsum は、結果を格納するために十分なサイズ（少なくとも33バイト：32文字＋終端ヌル文字）が確保されている必要があります。
  • pg_cryptohash_ctx の生成および計算処理が成功することが前提です。

アルゴリズムまたは振る舞い

1. pg_cryptohash_create(PG_MD5) を呼び出し、ハッシュコンテキストを初期化します。 2. コンテキストの生成に失敗した場合、pg_cryptohash_error を取得して false を返します。 3. pg_cryptohash_init、pg_cryptohash_update、pg_cryptohash_final を順次実行し、バイナリ形式のダイジェストを生成します。 4. 計算中にエラーが発生した場合は、コンテキストを解放し、エラー情報を設定して false を返します。 5. 成功した場合、bytesToHex を用いてバイナリダイジェストを16進文字列に変換し、コンテキストを解放して true を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/cryptohashfuncs.c
  • src/common/md5_common.c (pg_md5_encrypt 内)
  • **呼び出し先**:
  • pg_cryptohash_create
  • pg_cryptohash_init
  • pg_cryptohash_update
  • pg_cryptohash_final
  • pg_cryptohash_error
  • pg_cryptohash_free
  • bytesToHex

関連データ構造

  • pg_cryptohash_ctx: ハッシュ計算の状態を保持する不透明なコンテキスト構造体。
  • MD5_DIGEST_LENGTH: MD5ハッシュのバイト長（16バイト）。

失敗モードと境界条件

  • コンテキストのメモリ割り当て失敗（OOM）が発生した場合、false を返し errstr にエラー内容が設定されます。
  • ハッシュ計算プロセス（init/update/final）のいずれかでエラーが発生した場合、処理を中断し false を返します。

保守担当者向けメモ

  • 本関数は cryptohash インターフェースを利用しており、具体的なMD5アルゴリズムの実装には依存していません。
  • 認証用途以外でのMD5利用は、セキュリティ上の脆弱性（衝突耐性の欠如）があるため、新規機能への採用には注意してください。
