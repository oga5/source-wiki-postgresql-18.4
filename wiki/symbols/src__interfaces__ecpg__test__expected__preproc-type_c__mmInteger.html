目的

ECPG（Embedded SQL for C）プリプロセッサの型定義機能およびカスタム型マッピングの動作を検証するための回帰テスト用エイリアス。

シグネチャまたは形状

typedef long mmInteger;

入力と出力

  • 入力: long 型の整数値。
  • 出力: mmInteger 型として定義された整数値。

前提条件と不変条件

  • C言語の long 型とバイナリ互換性を持つ。
  • 本シンボルはプリプロセッサによって生成されたテストコードの一部であり、手動編集を想定していない。

アルゴリズムまたは振る舞い

long 型の別名として機能し、ECPGプリプロセッサがカスタム型をどのようにC言語の型へ変換するかを検証する際に使用される。

呼び出し元と呼び出し先

  • 呼び出し元: src/interfaces/ecpg/test/expected/preproc-type.c:68 において、変数 idnum の型として参照されている。
  • 呼び出し先: 情報なし。

関連データ構造

  • TBempl: 本シンボルを利用して定義された構造体メンバを含むテスト用構造体。

失敗モードと境界条件

  • long 型の範囲を超える値の代入や、プラットフォーム間での long 型のサイズ差異（32bit/64bit）に依存する挙動を示す可能性がある。
