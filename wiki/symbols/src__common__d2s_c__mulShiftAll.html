mulShiftAll

定義

  • 種別: function
  • 定義箇所: src/common/d2s.c:170

目的

浮動小数点数の変換過程において、仮数部に対して乗算とビットシフトを適用し、丸め判定に必要な3つの値（中心値、下限値、上限値）を同時に計算する。

シグネチャまたは形状

static inline uint64 mulShiftAll(const uint64 m, const uint64 *const mul, const int32 j,
                                 uint64 *const vp, uint64 *const vm, const uint32 mmShift)

入力と出力

  • **入力**:
  • m: 変換対象の仮数。
  • mul: 乗算に使用する定数配列へのポインタ。
  • j: シフト量。
  • mmShift: 下限値計算用のオフセット値。
  • **出力**:
  • 戻り値: 4 * m に対して乗算とシフトを適用した結果（中心値）。
  • vp: 4 * m + 2 に基づく計算結果（上限値）が格納されるポインタ。
  • vm: 4 * m - 1 - mmShift に基づく計算結果（下限値）が格納されるポインタ。

前提条件と不変条件

  • mulShift 関数が利用可能であること。
  • vp および vm は、計算結果を書き込むために有効なメモリ領域を指していること。

アルゴリズムまたは振る舞い

1. vp に mulShift(4 * m + 2, mul, j) の結果を代入する。 2. vm に mulShift(4 * m - 1 - mmShift, mul, j) の結果を代入する。 3. mulShift(4 * m, mul, j) の結果を戻り値として返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/common/d2s.c 内の複数箇所（行 402, 448 など）から呼び出されている。
  • **呼び出し先**:
  • mulShift 関数を呼び出している。

関連データ構造

  • uint64: 64ビット符号なし整数。
  • int32: 32ビット符号付き整数。
  • uint32: 32ビット符号なし整数。

失敗モードと境界条件

  • mulShift 内部でのオーバーフローやシフト演算の挙動に依存する。
  • 入力 m や mmShift の値が極端な場合、計算結果が期待する範囲外になる可能性があるが、本関数自体は算術演算のみを行う。

保守担当者向けメモ

  • 本関数は Ryu アルゴリズムのコアとなる数値変換ロジックの一部であり、IEEE 754 浮動小数点数の正確な変換を保証するために使用される。
  • 変更を加える際は、mulShift の実装および呼び出し元での丸め処理との整合性を十分に検証すること。
