r_case_ending

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_finnish.c:411

目的

フィンランド語のステミング処理において、単語の末尾から特定の格語尾を照合し、該当する場合はそれを削除して正規化を行う。

シグネチャまたは形状

static int r_case_ending(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（Snowballステミング環境構造体）。現在の単語データおよび処理状態（カーソル位置、境界情報）を含む。
  • **出力**: int 型。処理が成功した場合は 1 を返し、失敗した場合は 0 を返す。エラー発生時は負の値を返す。

前提条件と不変条件

  • z->I[1] に格納された境界位置以降の文字列が処理対象となる。
  • 処理成功時、z->I[2] に 1 がセットされ、格語尾が削除されたことが記録される。

アルゴリズムまたは振る舞い

1. **境界チェック**: 現在のカーソル位置 z->c が z->I[1] 未満であれば終了する。 2. **パターン照合**: a_6 配列（30個のパターン）を用いて、単語末尾から格語尾を逆方向に検索する。 3. **条件分岐**:

  • case 1-6: 特定の母音（a, e, i, o, ä, ö）で終わる場合、カーソルを1つ戻す。
  • case 7: r_LONG 関数によるチェック、または特定の文字列（s_2）との一致を確認し、条件を満たせばカーソルを調整する。
  • case 8: g_V1（母音グループ）および g_C（子音グループ）の条件を満たすか確認する。

    • 4. **削除処理**: slice_del(z) を呼び出し、特定された格語尾部分を単語から削除する。 5. **状態更新**: 成功時に z->I[2] を 1 に設定し、1 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/snowball/libstemmer/stem_ISO_8859_1_finnish.c
  • src/backend/snowball/libstemmer/stem_UTF_8_estonian.c
  • src/backend/snowball/libstemmer/stem_UTF_8_finnish.c
  • **呼び出し先**:
  • find_among_b（パターン照合）
  • r_LONG（条件判定）
  • in_grouping_b（文字グループ判定）
  • eq_s_b（文字列比較）
  • slice_del（文字列削除）

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_6: 格語尾パターンを定義した among 構造体配列。
  • g_V1, g_C: 文字グループ定義。

失敗モードと境界条件

  • find_among_b で一致するパターンが見つからない場合、0 を返して終了する。
  • slice_del が失敗した場合、その戻り値（負の値）をそのまま返す。
  • カーソル位置が境界 z->lb を超える場合、処理を中断する。
