run_ifaddr_callback

定義

  • 種別: function
  • 定義箇所: src/backend/libpq/ifaddr.c:180

目的

ネットワークインターフェースのアドレスとマスクを検証・正規化し、ユーザー定義のコールバック関数を実行する。

シグネチャまたは形状

static void run_ifaddr_callback(PgIfAddrCallback callback, void *cb_data, struct sockaddr *addr, struct sockaddr *mask);

入力と出力

  • callback: 実行対象の PgIfAddrCallback 型関数ポインタ。
  • cb_data: コールバック関数に渡される任意のデータポインタ。
  • addr: ネットワークインターフェースのアドレス（struct sockaddr）。
  • mask: ネットワークマスク（struct sockaddr）。
  • 戻り値: なし。

前提条件と不変条件

  • addr が NULL の場合、関数は何もせず直ちに終了する。
  • mask が addr と異なるアドレスファミリーを持つ場合、または AF_INET/AF_INET6 においてマスクが未指定（INADDR_ANY または IN6_IS_ADDR_UNSPECIFIED）である場合、マスクは無効とみなされる。

アルゴリズムまたは振る舞い

1. addr が NULL であれば終了する。 2. mask が提供されている場合、その妥当性を検証する。ファミリーの不一致や未指定値が検出された場合、mask を NULL にリセットする。 3. mask が NULL（または無効）である場合、pg_sockaddr_cidr_mask を呼び出し、デフォルトのフルマスクを生成して適用する。 4. 最終的に確定した addr と mask を引数として、callback を実行する。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/libpq/ifaddr.c 内の pg_foreach_ifaddr 実装（OSごとのインターフェース列挙処理）。
  • 呼び出し先: pg_sockaddr_cidr_mask（マスク生成時）、および引数として渡された callback 関数。

関連データ構造

  • PgIfAddrCallback: コールバック関数の型定義。
  • struct sockaddr, struct sockaddr_storage: ネットワークアドレスを保持する標準構造体。

失敗モードと境界条件

  • addr が NULL の場合は処理をスキップする。
  • OSから提供されたマスクが不正な場合、自動的にフルマスクで補完されるため、呼び出し先のコールバックには常に有効なマスクが渡される。

保守担当者向けメモ

  • 本関数は pg_foreach_ifaddr の内部ヘルパーであり、OS依存のネットワーク列挙処理と上位層のコールバックを仲介する。
  • マスクの正規化ロジックは、ネットワーク設定が不完全な環境での堅牢性を担保している。
