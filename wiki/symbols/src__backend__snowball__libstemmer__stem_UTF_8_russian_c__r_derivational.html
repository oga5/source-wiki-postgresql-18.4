r_derivational

定義

  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_russian.c:520

目的

ロシア語のステミング処理において、単語末尾の派生語接尾辞を特定し、条件を満たす場合に削除を行う。

シグネチャまたは形状

static int r_derivational(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（Snowballステミング環境構造体）
  • 出力: 処理が成功または適用された場合は 1、適用条件を満たさない場合は 0、エラー発生時は負の値を返す。

前提条件と不変条件

  • z->c（現在のカーソル位置）が有効な範囲内であること。
  • a_6（派生語接尾辞のパターン配列）が定義されていること。
  • 削除対象の接尾辞が R2 領域内に存在すること。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を z->ket に設定する。 2. 単語末尾から前方へ向かって、特定の文字（UTF-8バイト値 130 または 140）をチェックする。条件を満たさない場合は終了する。 3. find_among_b を使用し、a_6 に定義された派生語接尾辞を後方検索する。 4. マッチした場合、その位置を z->bra に設定する。 5. r_R2(z) を呼び出し、マッチした接尾辞が R2 領域内にあるか検証する。 6. 検証に成功した場合、slice_del(z) を実行して接尾辞を削除する。

呼び出し元と呼び出し先

  • 呼び出し元: russian_UTF_8_stem（src/backend/snowball/libstemmer/stem_UTF_8_russian.c:661）
  • 呼び出し先: find_among_b, r_R2, slice_del

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_6: 派生語接尾辞を格納する among 構造体の配列。

失敗モードと境界条件

  • 単語長が不足している場合や、末尾の文字が特定の条件（130, 140）に一致しない場合は 0 を返す。
  • r_R2 が R2 領域外と判定した場合、または slice_del が失敗した場合は、その戻り値をそのまま返す。
