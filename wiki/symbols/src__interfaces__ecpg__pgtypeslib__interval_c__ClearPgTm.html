ClearPgTm

定義

  • 種別: static inline function
  • 定義箇所: src/interfaces/ecpg/pgtypeslib/interval.c:93

目的

日付・時刻計算に使用される struct tm 構造体および小数秒（fsec_t）変数をゼロ初期化する。

シグネチャまたは形状

static inline void ClearPgTm(struct tm *tm, fsec_t *fsec)

入力と出力

  • 入力:
  • tm: 初期化対象の struct tm へのポインタ。
  • fsec: 初期化対象の fsec_t 型変数へのポインタ。
  • 出力: なし（ポインタ経由で値を直接更新する）。

前提条件と不変条件

  • tm および fsec は有効なメモリ領域を指している必要がある。
  • 実行後、tm の各フィールド（年、月、日、時、分、秒）および fsec はすべて 0 に設定される。

アルゴリズムまたは振る舞い

1. tm->tm_year, tm->tm_mon, tm->tm_mday, tm->tm_hour, tm->tm_min, tm->tm_sec に 0 を代入する。 2. fsec が指す値に 0 を代入する。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/interfaces/ecpg/pgtypeslib/interval.c:119
  • src/interfaces/ecpg/pgtypeslib/interval.c:343
  • 呼び出し先: なし（標準的な代入操作のみを実行）。

関連データ構造

  • struct tm: C 標準の日付・時刻構造体。
  • fsec_t: 小数秒を保持するための型。

失敗モードと境界条件

  • ポインタが NULL の場合、未定義の動作（セグメンテーション違反）を引き起こす可能性がある。

保守担当者向けメモ

  • 本関数は interval.c 内で局所的に使用されるユーティリティである。
  • バックエンドの pg_tm ではなく、標準の struct tm を扱う点に注意すること。
