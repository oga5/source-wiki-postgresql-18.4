pgp_rsa_decrypt

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/pgp-mpi-openssl.c:250

目的

OpenPGP 形式の暗号文（MPI）を RSA 秘密鍵を用いて復号し、平文（MPI）として出力する。

シグネチャまたは形状

int pgp_rsa_decrypt(PGP_PubKey *pk, PGP_MPI *_c, PGP_MPI **m_p);

入力と出力

  • **入力**:
  • pk: 復号に使用する PGP_PubKey 構造体（秘密鍵成分 d および公開鍵成分 n を含む）。
  • _c: 暗号化されたメッセージ（PGP_MPI 形式）。
  • **出力**:
  • m_p: 復号された平文を格納する PGP_MPI へのポインタ。
  • 戻り値: 成功時に 0、失敗時に PXE_PGP_MATH_FAILED を返す。

前提条件と不変条件

  • pk には有効な RSA 秘密鍵（d）と公開鍵（n）が含まれている必要がある。
  • 内部で OpenSSL の BIGNUM を使用するため、OpenSSL ライブラリが初期化されている必要がある。
  • 関数終了時には、確保したすべての BIGNUM および BN_CTX が解放される。

アルゴリズムまたは振る舞い

1. mpi_to_bn を使用して、入力された PGP_MPI および鍵情報を OpenSSL の BIGNUM 形式に変換する。 2. BN_CTX_new で演算用コンテキストを確保する。 3. BN_mod_exp を用いて、モジュロべき乗演算 $m = c^d \pmod n$ を実行する。 4. 演算結果を bn_to_mpi で PGP_MPI 形式に変換し、m_p に格納する。 5. エラー発生時および正常終了時に、確保したリソース（BIGNUM ポインタ群および BN_CTX）を BN_clear_free 等で確実に解放する。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/pgcrypto/pgp-pubdec.c:139
  • **呼び出し先**:
  • mpi_to_bn (変換)
  • bn_to_mpi (変換)
  • BN_new, BN_CTX_new, BN_mod_exp, BN_CTX_free, BN_clear_free (OpenSSL API)

関連データ構造

  • PGP_PubKey: PGP 公開鍵および秘密鍵情報を保持する構造体。
  • PGP_MPI: OpenPGP の多倍長整数表現。
  • BIGNUM: OpenSSL の多倍長整数構造体。

失敗モードと境界条件

  • メモリ不足や OpenSSL の演算失敗が発生した場合、PXE_PGP_MATH_FAILED を返す。
  • 内部処理中にエラーが発生した場合、goto err ラベルへジャンプし、確保済みのリソースを解放してから終了する。

保守担当者向けメモ

  • 本関数は pgcrypto モジュールにおける RSA 復号の数学的処理のブリッジである。
  • OpenSSL の BIGNUM は動的確保されるため、エラーハンドリング時のメモリリーク防止が厳格に行われている。変更を加える際は err ラベル以降の解放処理が漏れないよう注意すること。
