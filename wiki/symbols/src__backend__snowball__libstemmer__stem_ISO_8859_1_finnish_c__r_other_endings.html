r_other_endings

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_finnish.c:483

目的

フィンランド語のステミング処理において、単語の末尾から特定の「その他の語尾」パターンを照合し、該当する場合は削除することで語幹を正規化する。

シグネチャまたは形状

static int r_other_endings(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（ステミング環境を保持する構造体）
  • **出力**: 処理が成功し語尾が削除された場合は 1、ルールに一致しない場合や処理が中断された場合は 0、エラー発生時は負の値を返す。

前提条件と不変条件

  • z->I[0] に設定された境界位置以降の文字列が処理対象となる。
  • find_among_b を使用して、末尾から a_7 配列に定義された14種類のパターンを照合する。

アルゴリズムまたは振る舞い

1. z->I[0] で指定された境界位置を z->lb に一時保存し、処理範囲を限定する。 2. find_among_b を呼び出し、a_7 配列内のパターンと単語末尾を照合する。 3. 一致したパターンが case 1（特定の条件付き語尾）である場合、s_3 との照合を試み、一致すれば削除をスキップする。 4. 照合された語尾を slice_del を用いて削除する。 5. 処理終了後、保存していた z->lb を復元する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_ISO_8859_1_finnish.c および src/backend/snowball/libstemmer/stem_UTF_8_finnish.c 内のメインステミング処理フローから呼び出される。
  • **呼び出し先**: find_among_b（パターン照合）、slice_del（文字列削除）、eq_s_b（文字列比較）。

関連データ構造

  • SN_env: ステミング環境を管理する構造体。
  • a_7: 照合対象となる語尾パターンを格納した among 構造体配列。

失敗モードと境界条件

  • z->c < z->I[0] の場合、処理範囲外のため直ちに 0 を返す。
  • find_among_b で一致が見つからない場合、0 を返す。
  • slice_del がメモリ不足等の理由で負の値を返した場合、その値をそのまま伝播させる。

  • a_7 配列の定義内容や、case 1 における s_3 の条件分岐は、フィンランド語の文法ルールに基づいている。
