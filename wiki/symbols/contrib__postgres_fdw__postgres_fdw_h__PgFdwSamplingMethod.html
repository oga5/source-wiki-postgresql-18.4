PgFdwSamplingMethod

定義

  • 種別: enum
  • 定義箇所: contrib/postgres_fdw/postgres_fdw.h:145

目的

リモートサーバー上の外部テーブルに対して ANALYZE を実行する際、統計情報収集のためのサンプリング手法を指定するために使用されます。

シグネチャまたは形状

typedef enum PgFdwSamplingMethod
{
    ANALYZE_SAMPLE_OFF,         /* リモートサンプリングを行わない */
    ANALYZE_SAMPLE_AUTO,        /* サーバーバージョンに基づき自動選択 */
    ANALYZE_SAMPLE_RANDOM,      /* リモートの random() 関数を使用 */
    ANALYZE_SAMPLE_SYSTEM,      /* TABLESAMPLE system を使用 */
    ANALYZE_SAMPLE_BERNOULLI,   /* TABLESAMPLE bernoulli を使用 */
} PgFdwSamplingMethod;

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: なし（列挙型定義）

前提条件と不変条件

  • ANALYZE_SAMPLE_AUTO がデフォルト値として使用されることが想定されています。

アルゴリズムまたは振る舞い

  • ANALYZE 実行時のサンプリング戦略を決定します。
  • TABLESAMPLE 句をサポートする手法（SYSTEM, BERNOULLI）と、関数ベースの手法（RANDOM）、およびサンプリング無効化（OFF）を切り替えます。

呼び出し元と呼び出し先

  • 呼び出し元（直接的な参照の証拠）:
  • contrib/postgres_fdw/deparse.c:2560
  • contrib/postgres_fdw/postgres_fdw.c:5099
  • contrib/postgres_fdw/postgres_fdw.h:233
  • 呼び出し先: なし（列挙型のため）

関連データ構造

  • PgFdwRelationInfo（プランナが使用する構造体内で、サンプリング設定を保持するために参照される可能性があります）

失敗モードと境界条件

  • サーバーバージョンが TABLESAMPLE をサポートしていない場合に ANALYZE_SAMPLE_SYSTEM や ANALYZE_SAMPLE_BERNOULLI を選択した場合の挙動は、呼び出し側の実装に依存します。

保守担当者向けメモ

  • 新しいサンプリング手法を追加する場合は、この列挙型に値を追加し、postgres_fdw.c 内のサンプリングロジックを更新してください。
