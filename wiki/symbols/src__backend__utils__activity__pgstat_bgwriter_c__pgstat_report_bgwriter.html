pgstat_report_bgwriter

定義

  • **種別**: function
  • **定義箇所**: src/backend/utils/activity/pgstat_bgwriter.c:30

目的

プロセスローカルに蓄積されたバックグラウンドライター（bgwriter）の統計情報を共有メモリへフラッシュし、統計システムへ反映させる。

シグネチャまたは形状

void pgstat_report_bgwriter(void)

入力と出力

  • **入力**: プロセスローカル変数 PendingBgWriterStats（蓄積された統計カウンタ）。
  • **出力**: 共有メモリ上の pgStatLocal.shmem->bgwriter への加算反映、および pgstat_flush_io(false) によるIO統計の報告。

前提条件と不変条件

  • pgStatLocal.shmem->is_shutdown が偽であること。
  • 統計システムが稼働状態（pgstat_assert_is_up()）であること。
  • 統計情報の更新は changecount を用いたロックフリーな手法で行われる。

アルゴリズムまたは振る舞い

1. PendingBgWriterStats がすべてゼロであれば、何もせず終了する。 2. pgstat_begin_changecount_write を呼び出し、共有メモリへの書き込みを開始する。 3. buf_written_clean、maxwritten_clean、buf_alloc の各カウンタを共有メモリへ加算する。 4. pgstat_end_changecount_write を呼び出し、書き込みを完了する。 5. MemSet を使用して PendingBgWriterStats をゼロクリアし、再利用可能にする。 6. pgstat_flush_io(false) を呼び出し、関連するIO統計を報告する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/postmaster/bgwriter.c:239（バックグラウンドライターのメインループから呼び出される）。
  • **呼び出し先**: pgstat_begin_changecount_write, pgstat_end_changecount_write, MemSet, pgstat_flush_io。

関連データ構造

  • PendingBgWriterStats: プロセスローカルな統計保持用構造体。
  • PgStatShared_BgWriter: 共有メモリ上の統計格納用構造体。

失敗モードと境界条件

  • 統計情報が更新されていない場合、共有メモリへの書き込みはスキップされる。
  • 共有メモリのシャットダウン中はアサーションエラーが発生する。

保守担当者向けメモ

  • BGWRITER_ACC マクロを使用して統計フィールドの加算を行っている。フィールド追加時はこのマクロ定義を確認すること。
  • changecount による更新はロック競合を避けるための設計であるため、このパターンを維持すること。
