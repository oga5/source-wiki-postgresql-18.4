r_step_s5

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_greek.c:2647

目的

ギリシャ語のステミング処理におけるステップ5を実行し、特定の語尾パターンを削除および置換する。

シグネチャまたは形状

static int r_step_s5(struct SN_env * z)

入力と出力

  • 入力: SN_env 構造体（ステミング対象の文字列および状態を保持）。
  • 出力: 成功時に 1、処理が適用されなかった場合やエラー発生時に 0 または負の値を返す。

前提条件と不変条件

  • SN_env 構造体が適切に初期化されていること。
  • 内部で使用される a_11（11要素）および a_10（40要素）の among 配列が有効であること。

アルゴリズムまたは振る舞い

1. find_among_b を使用して、文字列末尾から a_11 に一致するパターンを検索し、一致した場合は slice_del で削除する。 2. 整数配列 z->I[0] を 0 にリセットする。 3. find_among_b を使用して、文字列末尾から a_10 に一致するパターンを検索する。 4. 一致した場合、現在のカーソル位置が境界 z->lb より後ろにあることを確認する。 5. among_var の値に応じて、slice_from_s を用いて特定の文字列（s_43 または s_44）への置換を行う。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_greek.c:3507
  • 呼び出し先: find_among_b, slice_del, slice_from_s（いずれも Snowball ランタイム関数）

関連データ構造

  • struct SN_env: ステミング処理の状態管理用構造体。
  • a_10, a_11: 語尾削除ルールを定義する among 構造体の配列。

失敗モードと境界条件

  • find_among_b が一致を見つけられない場合、0 を返して終了する。
  • slice_del または slice_from_s が負の値を返した場合、そのエラーコードをそのまま伝播させる。
  • z->c > z->lb の条件を満たさない場合、処理を中断し 0 を返す。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードである。ロジックの変更が必要な場合は、対応する .sbl ファイルを修正し、再生成すること。
  • a_10 および a_11 の定義は、ギリシャ語のステミングルールに依存している。
