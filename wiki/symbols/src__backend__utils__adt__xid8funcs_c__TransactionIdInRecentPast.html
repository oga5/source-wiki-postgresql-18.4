TransactionIdInRecentPast

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/xid8funcs.c:96

目的

指定された64ビットトランザクションID（FullTransactionId）が、現在保持されているCLOG（Commit Log）の範囲内で参照可能かどうかを判定する。

シグネチャまたは形状

bool TransactionIdInRecentPast(FullTransactionId fxid, TransactionId *extracted_xid)

入力と出力

  • **入力**:
  • fxid: 判定対象の64ビットトランザクションID。
  • extracted_xid: fxid から抽出された32ビットの TransactionId を格納するためのポインタ（NULL可）。
  • **出力**:
  • bool: 指定されたIDがCLOGから参照可能な過去の範囲内であれば true、そうでなければ false。

前提条件と不変条件

  • 呼び出し元は、CLOGの切り捨てを防ぐために XactTruncationLock を保持していなければならない（Assert により検証される）。
  • fxid が現在より未来のトランザクションIDである場合、本関数はエラー（ERROR）を発生させる。

アルゴリズムまたは振る舞い

1. fxid から32ビットの xid を抽出する。 2. ReadNextFullTransactionId() を呼び出し、現在の最新トランザクションIDを取得する。 3. extracted_xid がNULLでなければ、抽出した xid を書き込む。 4. xid が無効な場合は false を返す。 5. xid が正規のトランザクションID（TransactionIdIsNormal）でない場合は true を返す。 6. fxid が現在より未来であればエラーを投げる。 7. TransamVariables->oldestClogXid を取得し、それに基づいて算出された oldest_clog_fxid と fxid を比較する。 8. fxid が oldest_clog_fxid より前（古い）でなければ true を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/xid8funcs.c:651
  • src/backend/replication/walsender.c（注: walsender.c 内の同名関数は引数構成が異なるため、本関数とは別定義の可能性がある）
  • **呼び出し先**:
  • XidFromFullTransactionId
  • ReadNextFullTransactionId
  • TransactionIdIsValid
  • TransactionIdIsNormal
  • FullTransactionIdPrecedes
  • FullTransactionIdFromAllowableAt
  • ereport

関連データ構造

  • FullTransactionId: 64ビットトランザクションID。
  • TransactionId: 32ビットトランザクションID。
  • TransamVariables: グローバルなトランザクション状態を保持する構造体。

失敗モードと境界条件

  • fxid が未来のIDである場合: ERRCODE_INVALID_PARAMETER_VALUE でエラー終了する。
  • XactTruncationLock が未取得の場合: Assert 失敗によりデバッグビルドで停止する。

保守担当者向けメモ

  • 本関数はCLOGの有効性を確認するヘルパーであり、ロックの管理責任を呼び出し元に委譲している。
  • walsender.c に存在する同名関数との関係性については、引数シグネチャが異なるため注意が必要である。
