toast_get_compression_id

定義

  • 種別: function
  • 定義箇所: src/backend/access/common/toast_compression.c:253

目的

varlena 構造体（TOASTデータ）から、適用されている圧縮アルゴリズムの識別子（ToastCompressionId）を取得します。

シグネチャまたは形状

ToastCompressionId toast_get_compression_id(struct varlena *attr)

入力と出力

  • **入力**: attr - 圧縮状態を確認する対象の varlena 構造体へのポインタ。
  • **出力**: 圧縮されている場合は対応する ToastCompressionId を返し、圧縮されていない場合や識別できない場合は TOAST_INVALID_COMPRESSION_ID を返します。

前提条件と不変条件

  • 入力される attr は有効な varlena 構造体である必要があります。
  • 圧縮方式の識別子は varlena ヘッダー内の2ビットに格納されているという制約に基づいています。

アルゴリズムまたは振る舞い

1. 初期値を TOAST_INVALID_COMPRESSION_ID に設定します。 2. VARATT_IS_EXTERNAL_ONDISK(attr) が真の場合、外部ストレージ上のポインタから圧縮情報を取得します。

  • VARATT_EXTERNAL_IS_COMPRESSED で圧縮の有無を確認し、真であれば VARATT_EXTERNAL_GET_COMPRESS_METHOD で ID を抽出します。

    • 3. 外部ストレージでない場合、VARATT_IS_COMPRESSED(attr) を確認し、インライン圧縮されている場合は VARDATA_COMPRESSED_GET_COMPRESS_METHOD を使用して ID を抽出します。 4. 最終的に特定された ID を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/varlena.c:5349 (直接の呼び出しを確認)
  • **呼び出し先**:
  • マクロ VARATT_IS_EXTERNAL_ONDISK, VARATT_EXTERNAL_GET_POINTER, VARATT_EXTERNAL_IS_COMPRESSED, VARATT_EXTERNAL_GET_COMPRESS_METHOD, VARATT_IS_COMPRESSED, VARDATA_COMPRESSED_GET_COMPRESS_METHOD を使用しています。

関連データ構造

  • struct varlena: PostgreSQLの可変長データ型。
  • ToastCompressionId: 圧縮アルゴリズムを識別するための列挙型または整数型。
  • struct varatt_external: 外部TOASTポインタの構造体。

失敗モードと境界条件

  • 圧縮されていないデータが渡された場合、TOAST_INVALID_COMPRESSION_ID を返します。
  • 外部ストレージ上のデータで圧縮フラグが立っていない場合も同様に無効な ID を返します。

保守担当者向けメモ

  • 本関数は src/include/access/toast_compression.h で宣言されています。
  • 将来的に圧縮アルゴリズムを拡張する場合、varlena ヘッダーの2ビット制限により最大4種類までという制約がある点に注意してください。
