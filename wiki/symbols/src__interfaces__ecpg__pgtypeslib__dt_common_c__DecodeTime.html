DecodeTime

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/pgtypeslib/dt_common.c:1436

目的

文字列形式の時刻（HH:MM:SS または HH:MM）を解析し、struct tm 構造体および秒以下の端数（fsec_t）に分解する。

シグネチャまたは形状

int DecodeTime(char *str, int *tmask, struct tm *tm, fsec_t *fsec)

入力と出力

  • **入力**:
  • str: 解析対象の時刻文字列。
  • tmask: 解析結果のマスク値を格納するポインタ。
  • tm: 時刻情報を格納する struct tm 構造体へのポインタ。
  • fsec: 秒以下の端数（マイクロ秒単位）を格納する fsec_t へのポインタ。
  • **出力**:
  • 成功時は 0 を返す。
  • 形式エラーや範囲外の値の場合は -1 を返す。

前提条件と不変条件

  • tmask には常に DTK_TIME_M が設定される。
  • tm->tm_hour, tm->tm_min, tm->tm_sec は非負であること。
  • tm->tm_min および tm->tm_sec は 0 から 59 の範囲内であること。
  • fsec は 1 秒未満（USECS_PER_SEC 未満）であること。

アルゴリズムまたは振る舞い

1. strtoint を使用して時、分、秒を順次抽出する。 2. コロン（:）を区切り文字として解析し、形式が一致しない場合はエラーを返す。 3. 秒以下の端数（. 以降）が存在する場合、最大6桁までを読み取り、fsec に変換する。 4. 最後に各フィールドの値が妥当な範囲内にあるかチェックを行う。

呼び出し元と呼び出し先

  • **呼び出し元**: src/interfaces/ecpg/pgtypeslib/dt_common.c:1912 で呼び出されている。
  • **呼び出し先**: strtoint（文字列から整数への変換関数）。
  • 注: src/backend/utils/adt/datetime.c にも同名の関数が存在するが、本ページは ecpg ライブラリ内の実装を対象としている。

関連データ構造

  • struct tm: 標準的な時刻構造体。
  • fsec_t: 秒以下の端数を保持する型。
  • DTK_TIME_M: 時刻解析を示すマスク定数。

失敗モードと境界条件

  • 時刻の区切り文字が正しくない場合（: 以外）は -1 を返す。
  • 各フィールドが負の値、または分・秒が 60 以上の場合、あるいは端数が 1 秒以上となる場合は -1 を返す。
  • 秒以下の端数が 7 桁以上ある場合、7 桁目以降は切り捨てられる（ソースコード内のコメントによる）。

保守担当者向けメモ

  • 本関数は ecpg ライブラリ用であり、バックエンド側の同名関数とは実装が異なる点に注意すること。
  • 秒以下の端数処理において、7 桁目以降を切り捨てる仕様は、バックエンド側の四捨五入処理と挙動が異なるため、将来的な整合性検討が必要。
