defGetBoolean

定義

  • 種別: function
  • 定義箇所: src/backend/commands/define.c:93

目的

DefElem ノードから真偽値（bool）を抽出・変換する。SQL コマンドのオプション解析において、柔軟な真偽値表現を標準化された bool 型へ変換するために使用される。

シグネチャまたは形状

bool defGetBoolean(DefElem *def)

入力と出力

  • **入力**: DefElem *def （解析対象の定義要素ノード）
  • **出力**: bool 型の値（true または false）
  • **副作用**: 変換不能な値が指定された場合、ereport により ERROR を発生させ処理を中断する。

前提条件と不変条件

  • def が NULL でないこと。
  • def->arg が NULL の場合、デフォルト値として true を返す。
  • def->arg が存在する場合、T_Integer 型または文字列として解釈可能な型である必要がある。

アルゴリズムまたは振る舞い

1. def->arg が NULL の場合、true を返す。 2. def->arg のノードタグを確認する。

  • T_Integer の場合: 値が 1 なら true、0 なら false を返す。それ以外はエラー。
  • その他の型の場合: defGetString() を呼び出し、文字列として取得する。
  • "true" または "on" の場合: true を返す。
  • "false" または "off" の場合: false を返す。

    • 3. 上記のいずれにも該当しない場合、ERRCODE_SYNTAX_ERROR を発生させる。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/dblink/dblink.c
  • contrib/dict_int/dict_int.c
  • contrib/dict_xsyn/dict_xsyn.c
  • contrib/file_fdw/file_fdw.c
  • **呼び出し先**:
  • defGetString (文字列抽出用)
  • intVal (整数抽出用)
  • ereport (エラー報告用)

関連データ構造

  • DefElem: src/include/nodes/parsenodes.h で定義される、SQL コマンドのオプションを保持する構造体。

失敗モードと境界条件

  • **構文エラー**: 期待される真偽値（0/1, true/false, on/off）以外の値が渡された場合、ereport が発行される。
  • **NULL 引数**: def->arg が NULL の場合は安全に true を返す。

保守担当者向けメモ

  • 本関数が受け入れる文字列のセットは、PostgreSQL の構文解析における opt_boolean_or_string プロダクションと一致させる必要がある。変更時は文法定義との整合性に注意すること。
