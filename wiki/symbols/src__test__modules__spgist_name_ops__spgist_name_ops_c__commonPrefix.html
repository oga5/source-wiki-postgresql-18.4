commonPrefix

定義

  • 種別: function
  • 定義箇所: src/test/modules/spgist_name_ops/spgist_name_ops.c:76

目的

2つの文字列（a と b）の先頭から一致するバイト数を計算し、共通のプレフィックス長を返します。

シグネチャまたは形状

static int commonPrefix(const char *a, const char *b, int lena, int lenb)

入力と出力

  • **入力**:
  • a, b: 比較対象の文字列ポインタ。
  • lena, lenb: それぞれの文字列の長さ。
  • **出力**:
  • 戻り値: 両方の文字列で一致した先頭からのバイト数（int）。

前提条件と不変条件

  • lena および lenb は、それぞれの文字列の有効な長さを指している必要があります。
  • 比較はバイト単位で行われます。

アルゴリズムまたは振る舞い

1. インデックス i を 0 に初期化します。 2. i が lena および lenb 未満であり、かつ a[i] と b[i] が等しい間、ループを繰り返します。 3. ループ内でポインタをインクリメントし、i を加算します。 4. 一致しなくなった時点、またはどちらかの文字列の終端に達した時点での i を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/test/modules/spgist_name_ops/spgist_name_ops.c:145
  • src/backend/access/spgist/spgtextproc.c:205, 347
  • **呼び出し先**:
  • 本関数は他の関数を呼び出しません。

関連データ構造

  • 特になし（プリミティブな文字列操作関数）。

失敗モードと境界条件

  • lena または lenb が 0 の場合、ループ条件を満たさないため 0 を返します。
  • 文字列が完全に一致する場合、短い方の文字列の長さが返されます。
