XmlPGetDatum

定義

  • 種別: static inline function
  • 定義箇所: src/include/utils/xml.h:56

目的

xmltype 型のポインタを、PostgreSQLの汎用的な Datum 型に変換する。

シグネチャまたは形状

static inline Datum XmlPGetDatum(const xmltype *X)

入力と出力

  • **入力**: const xmltype *X （XMLデータを保持する varlena 型のポインタ）
  • **出力**: Datum （PostgreSQLの内部データ表現形式）

前提条件と不変条件

  • 入力ポインタ X は有効な xmltype 構造体を指している必要がある。
  • PointerGetDatum マクロが期待する通り、X はメモリ上の有効なアドレスであること。

アルゴリズムまたは振る舞い

PointerGetDatum マクロを呼び出し、受け取った xmltype ポインタを Datum 型へキャストして返す。

呼び出し元と呼び出し先

  • **呼び出し元**: 提供されたソース情報からは、特定の呼び出し元は特定されていない。
  • **呼び出し先**: PointerGetDatum（PostgreSQLのコアマクロ）を呼び出す。

関連データ構造

  • xmltype: varlena 型のエイリアスであり、XMLデータを保持する。

失敗モードと境界条件

  • X が NULL の場合、PointerGetDatum の仕様に従い (Datum) 0 が返される。

保守担当者向けメモ

  • 本関数は xmltype を扱うための型安全な変換インターフェースの一部である。
  • xmltype の内部構造やメモリレイアウトに変更があった場合でも、PointerGetDatum を介しているため、本関数の変更は不要である可能性が高い。
