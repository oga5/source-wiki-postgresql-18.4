DatumGetUInt16

定義

  • 種別: static inline 関数
  • 定義箇所: src/include/postgres.h:186

目的

PostgreSQL の汎用データ型である Datum を、C 言語の符号なし 16 ビット整数型 (uint16) に変換する。

シグネチャまたは形状

static inline uint16 DatumGetUInt16(Datum X)

入力と出力

  • **入力**: Datum X（変換対象の汎用データ値）
  • **出力**: uint16（X を uint16 型にキャストした値）

前提条件と不変条件

  • Datum 型は sizeof(void *) と同じサイズであると仮定される。
  • 本関数は値の妥当性チェックを行わず、単純な型キャストを実行する。

アルゴリズムまたは振る舞い

1. 引数として受け取った Datum 型の値 X を、明示的に uint16 型へキャストする。 2. キャストされた値を戻り値として返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/access/gin/ginutil.c:253
  • src/backend/access/gist/gistutil.c:1110
  • src/include/fmgr.h:272（PG_GETARG_UINT16 マクロ経由）
  • **呼び出し先**:
  • 本関数はインライン関数であり、特定の関数を呼び出すことはない。

関連データ構造

  • Datum: PostgreSQL の実行エンジン内でデータを効率的に受け渡すための汎用型。

失敗モードと境界条件

  • 本関数自体はエラーチェックを行わない。入力値が uint16 の範囲外である場合、キャストの結果は C 言語の標準的な型変換規則に従う。

保守担当者向けメモ

  • 本関数は static inline で定義されており、関数呼び出しのオーバーヘッドを最小化している。
  • 移植性を維持するため、Datum からの変換には必ず本関数または対応する DatumGet* 系関数を使用すること。直接的なキャストは避けること。
