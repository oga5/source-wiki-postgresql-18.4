TypeGetTupleDesc

定義

  • 種別: function
  • 定義箇所: src/backend/utils/fmgr/funcapi.c:1902

目的

指定された型（typeoid）に基づき、その型が表す行構造を記述する TupleDesc を構築して返します。主に FROM 句で使用される関数の戻り値構造を決定するために使用されます。

シグネチャまたは形状

TupleDesc TypeGetTupleDesc(Oid typeoid, List *colaliases)

入力と出力

  • **入力**:
  • typeoid: 戻り値の型の OID。
  • colaliases: 列名の別名リスト（List）。
  • **出力**:
  • 成功時: 構築された TupleDesc 構造体へのポインタ。
  • 失敗時: エラーが発生した場合は ereport により処理を中断します。

前提条件と不変条件

  • TYPEFUNC_COMPOSITE_DOMAIN はサポートされていません。
  • TYPEFUNC_RECORD 型が渡された場合、typmod が不明であるためエラーとなります。
  • スカラー型の場合、colaliases は必須であり、かつ要素数は 1 である必要があります。

アルゴリズムまたは振る舞い

1. get_type_func_class を呼び出し、型の種類（複合型、スカラー型、レコード型）を判定します。 2. **複合型 (TYPEFUNC_COMPOSITE)**:

  • lookup_rowtype_tupdesc_copy で既存の型定義をコピーします。
  • colaliases が提供されている場合、列名（attname）を別名で上書きし、型を RECORDOID に設定します。

    • 3. **スカラー型 (TYPEFUNC_SCALAR)**:

  • colaliases が存在し、要素数が 1 であることを検証します。
  • CreateTemplateTupleDesc(1) で新しい記述子を作成し、TupleDescInitEntry で型情報を設定します。

    • 4. **レコード型 (TYPEFUNC_RECORD)**:

  • サポート外のためエラーを発生させます。

    • 5. その他、サポートされていない型の場合は elog でエラーを報告します。

呼び出し元と呼び出し先

  • **呼び出し元**: ソースコード内の直接的な呼び出し元は src/backend/utils/fmgr/funcapi.c:1888 です。
  • **呼び出し先**: get_type_func_class, lookup_rowtype_tupdesc_copy, list_length, strVal, list_nth, namestrcpy, CreateTemplateTupleDesc, TupleDescInitEntry 等を呼び出します。

関連データ構造

  • TupleDesc: タプルの構造を定義するデータ構造。
  • TypeFuncClass: 型の分類（TYPEFUNC_COMPOSITE, TYPEFUNC_SCALAR, TYPEFUNC_RECORD 等）。

失敗モードと境界条件

  • colaliases の長さが列数と一致しない場合、ERRCODE_DATATYPE_MISMATCH でエラーとなります。
  • スカラー型で colaliases が指定されていない場合、エラーとなります。
  • TYPEFUNC_RECORD が渡された場合、typmod 不足によりエラーとなります。

保守担当者向けメモ

  • 本関数は「レガシーな関数」と位置付けられており、ドメイン制約の適用などは考慮されていません。
  • 複合型の別名変更を行うと、型 ID が RECORDOID に強制変更される点に注意してください。
