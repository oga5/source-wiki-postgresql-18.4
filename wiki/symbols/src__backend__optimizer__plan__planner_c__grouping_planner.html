grouping_planner

定義

  • 種別: function
  • 定義箇所: src/backend/optimizer/plan/planner.c:1675

目的

クエリの「上層」プラン（集約、グループ化、ウィンドウ関数、DISTINCT、ORDER BY、LIMIT/OFFSET 等）を生成する。スキャン・結合後の処理を統括し、実行可能なプランツリーを構築する。

シグネチャまたは形状

static void grouping_planner(PlannerInfo *root, double tuple_fraction, SetOperationStmt *setops);

入力と出力

  • **入力**:
  • root: クエリ全体の最適化情報を保持する PlannerInfo 構造体。
  • tuple_fraction: 呼び出し元が期待するタプル取得率（0.0〜1.0）。
  • setops: UNION, INTERSECT, EXCEPT を伴うクエリの場合の SetOperationStmt。
  • **出力**: なし（root 内の processed_tlist やプランツリーを直接更新する）。

前提条件と不変条件

  • root->hasRecursion が真である場合（再帰的クエリ）、setops が存在しなければならない。
  • setOperations が存在する場合、commandType は CMD_SELECT である必要がある。
  • setOperations が存在する場合、distinctClause は空でなければならない。

アルゴリズムまたは振る舞い

1. **LIMIT/OFFSET 調整**: preprocess_limit を呼び出し、tuple_fraction を最適化する。 2. **集合演算処理**: setOperations がある場合、plan_set_operations を呼び出してパスを構築し、ソートキーを生成する。 3. **通常プランニング**: 集合演算がない場合、以下の順序で前処理を行う：

  • preprocess_grouping_sets または preprocess_groupclause によるグループ化の準備。
  • preprocess_targetlist によるターゲットリストの正規化。
  • preprocess_aggrefs による集約関数のマーク付けと重複排除。
  • find_window_functions によるウィンドウ関数の特定と最適化。

    • 4. **後続処理**: 抽出された情報に基づき、スキャン・結合後のプランノードを順次構築する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • standard_planner (src/backend/optimizer/plan/planner.c)
  • **呼び出し先**:
  • preprocess_limit, plan_set_operations, postprocess_setop_tlist, is_parallel_safe, make_pathkeys_for_sortclauses
  • preprocess_grouping_sets, preprocess_groupclause, preprocess_targetlist, preprocess_aggrefs
  • find_window_functions, optimize_window_clauses, select_active_windows, name_active_windows

関連データ構造

  • PlannerInfo: プランニング中のグローバル状態。
  • PathTarget: プランノードの出力列定義。
  • SetOperationStmt: 集合演算の構文木。
  • standard_qp_extra: クエリプランニング用の補助データ。

失敗モードと境界条件

  • FOR [KEY] UPDATE/SHARE 句が UNION/INTERSECT/EXCEPT と同時に指定された場合、ERRCODE_FEATURE_NOT_SUPPORTED でエラーを報告する。

保守担当者向けメモ

  • 本関数は非常に複雑なため、setOperations の有無による分岐を明確に区別して理解すること。
  • preprocess_aggrefs は、ターゲットリストを分割する前に実行する必要がある（Aggref ノードの整合性維持のため）。
  • tuple_fraction の変更は、下位のプラン生成ルーチンに影響を与えるため慎重に扱うこと。
