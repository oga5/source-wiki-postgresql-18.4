AbsoluteTime

定義

  • 種別: typedef
  • 定義箇所: src/interfaces/ecpg/pgtypeslib/dt_common.c:18

目的

ECPGライブラリ内において、絶対時刻を表現するための型定義です。

シグネチャまたは形状

typedef long AbsoluteTime;

入力と出力

  • 入力: long 型の整数値。
  • 出力: AbsoluteTime 型のエイリアス。

前提条件と不変条件

  • long 型として定義されており、システム依存のサイズ（通常は32ビットまたは64ビット）に依存します。

アルゴリズムまたは振る舞い

  • 本シンボル自体は型定義であり、具体的な計算ロジックは持ちません。
  • 日付・時刻処理において、特定の時点を示す数値表現として利用されます。

呼び出し元と呼び出し先

  • 呼び出し元: src/interfaces/ecpg/pgtypeslib/dt_common.c:973 の abstime2tm 関数が、引数として AbsoluteTime 型の _time を受け取ります。
  • 呼び出し先: 本シンボルは型定義であるため、直接的な呼び出し先はありません。

関連データ構造

  • struct tm: abstime2tm 関数において、AbsoluteTime から変換された時刻情報を格納するために使用されます。

失敗モードと境界条件

  • long 型の範囲を超える値が代入された場合の挙動は、プラットフォームの整数表現に依存します。

保守担当者向けメモ

  • 本型は、ECPGにおける日付・時刻処理の共通基盤の一部です。時刻計算の精度や範囲を変更する際は、この型定義がシステム全体に与える影響を考慮してください。
