CatalogTupleDelete

定義

  • 種別: function
  • 定義箇所: src/backend/catalog/indexing.c:364

目的

システムカタログのタプル削除処理を抽象化し、実行する。

シグネチャまたは形状

void CatalogTupleDelete(Relation heapRel, ItemPointer tid)

入力と出力

  • **入力**:
  • heapRel: 削除対象のシステムカタログのリレーション記述子。
  • tid: 削除するタプルの物理位置（ItemPointer）。
  • **出力**: なし（void）。

前提条件と不変条件

  • heapRel は有効なシステムカタログのリレーションであること。
  • tid は削除対象として有効なタプルを指していること。

アルゴリズムまたは振る舞い

1. simple_heap_delete を呼び出し、指定されたリレーション内のタプルを削除する。 2. カタログの削除操作においてはインデックス更新が不要であるため、本関数はインデックス更新処理を含まないラッパーとして動作する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/catalog/aclchk.c
  • src/backend/catalog/dependency.c
  • src/backend/catalog/heap.c
  • **呼び出し先**:
  • simple_heap_delete（src/backend/access/heap/heapam.c に定義）

関連データ構造

  • Relation: リレーション記述子。
  • ItemPointer: タプルの物理位置を示すデータ構造。

失敗モードと境界条件

  • simple_heap_delete が内部で発生させるエラー（無効な tid やロック競合など）に依存する。

保守担当者向けメモ

  • 本関数は現在、インデックス更新を伴わない単純な削除ラッパーである。
  • 将来的にカタログ削除時に特定のインデックス更新が必要となった場合、CatalogIndexState を利用した拡張が必要になる可能性がある。
