r_standard_suffix

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_catalan.c:1292

目的

カタルーニャ語のステミング処理において、単語末尾の標準的な接尾辞を識別し、削除または置換を行う。

シグネチャまたは形状

static int r_standard_suffix(struct SN_env * z)

入力と出力

  • 入力: SN_env 構造体（ステミング対象の単語データを含む）。
  • 出力: 成功時に 1、接尾辞が見つからない場合に 0、エラー発生時に負の値を返す。

前提条件と不変条件

  • z->c が単語の末尾を指していること。
  • find_among_b を用いて、a_2 テーブル（200個の接尾辞）から後方一致検索を行う。
  • 接尾辞の削除・置換には、単語内の領域定義（r_R1 または r_R2）の条件を満たす必要がある。

アルゴリズムまたは振る舞い

1. z->ket を現在のカーソル位置に設定し、a_2 テーブルを用いて後方から接尾辞を探索する。 2. マッチした接尾辞のインデックス（among_var）に応じて以下の処理を行う：

  • case 1, 5: r_R1 領域内であることを確認し、削除または置換を実行。
  • case 2, 3, 4: r_R2 領域内であることを確認し、削除または置換を実行。

    • 3. 処理成功時は 1 を返し、失敗時は 0 またはエラーコードを返す。

呼び出し元と呼び出し先

  • 呼び出し元: catalan_ISO_8859_1_stem（直接の呼び出しが確認されている）。
  • 呼び出し先: find_among_b, r_R1, r_R2, slice_del, slice_from_s。

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_2: 接尾辞探索用の among 構造体テーブル。

失敗モードと境界条件

  • find_among_b が一致を見つけられない場合、0 を返して終了する。
  • r_R1 または r_R2 の条件を満たさない場合、接尾辞の変更は行われない。
  • slice_del や slice_from_s がメモリ不足等の理由で失敗した場合、負の戻り値が伝播される。

保守担当者向けメモ

  • 本関数はSnowballコンパイラによる自動生成コードである。ロジックの変更が必要な場合は、直接編集せずソース定義（.sblファイル）を修正すること。
  • among_var の各ケースは、カタルーニャ語の文法ルールに基づいた接尾辞の優先順位と置換規則に対応している。
