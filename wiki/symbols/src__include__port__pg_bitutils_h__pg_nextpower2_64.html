pg_nextpower2_64

定義

  • 種別: static inline 関数
  • 定義箇所: src/include/port/pg_bitutils.h:211

目的

与えられた 64 ビット整数 num 以上で、最小の 2 のべき乗を計算して返す。

シグネチャまたは形状

static inline uint64 pg_nextpower2_64(uint64 num)

入力と出力

  • **入力**: uint64 num （計算対象の数値）
  • **出力**: uint64 （num 以上の最小の 2 のべき乗）

前提条件と不変条件

  • num > 0 であること。
  • num <= PG_UINT64_MAX / 2 + 1 であること（オーバーフロー防止のため）。
  • 上記条件は Assert により検証される。

アルゴリズムまたは振る舞い

1. num が既に 2 のべき乗であるかを確認する（num & (num - 1) == 0）。該当する場合は num をそのまま返す。 2. 2 のべき乗でない場合、pg_leftmost_one_pos64(num) を呼び出し、最上位ビットの位置を取得する。 3. 取得した位置に基づき、ビットシフト演算 1 << (pos + 1) を行い、次の 2 のべき乗を算出する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/include/lib/simplehash.h:319
  • src/include/lib/simplehash.h:503
  • **呼び出し先**:
  • pg_leftmost_one_pos64 （最上位ビット位置の取得）

関連データ構造

  • なし

失敗モードと境界条件

  • num が 0 の場合、Assert によりプログラムが停止する可能性がある。
  • num が PG_UINT64_MAX / 2 + 1 を超える場合、Assert によりプログラムが停止する可能性がある。

保守担当者向けメモ

  • 本関数は pg_nextpower2_size_t としてマクロ定義（src/include/port/pg_bitutils.h:441）されており、サイズ計算に広く利用されている。
  • パフォーマンス向上のため static inline で実装されている。変更時はインライン展開の効率に注意すること。
