plperl_to_jsonb

定義

  • 種別: function
  • 定義箇所: contrib/jsonb_plperl/jsonb_plperl.c:288

目的

Perl のデータ構造（SV）を PostgreSQL の jsonb 型へ変換する SQL 呼び出し可能なエントリポイントです。

シグネチャまたは形状

Datum plperl_to_jsonb(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS を介して受け取る Perl のスカラー値 (SV *)。
  • **出力**: jsonb 型の PostgreSQL データ (Datum) を返します。

前提条件と不変条件

  • dTHX マクロにより Perl インタプリタのコンテキストが初期化されている必要があります。
  • 変換対象の Perl データ構造は、SV_to_JsonbValue が処理可能な形式である必要があります。

アルゴリズムまたは振る舞い

1. dTHX で Perl コンテキストを取得します。 2. PG_GETARG_POINTER(0) を使用して、引数から Perl の SV ポインタを取得します。 3. SV_to_JsonbValue を呼び出し、Perl のデータ構造を再帰的に走査して JsonbValue 構造体へ変換します（第3引数 true はルートノードであることを示唆）。 4. JsonbValueToJsonb を使用して、構築された JsonbValue を Jsonb 構造体に変換します。 5. PG_RETURN_JSONB_P を介して結果を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: PostgreSQL の関数呼び出しインターフェース（PG_FUNCTION_INFO_V1 マクロにより登録）。
  • **呼び出し先**:
  • SV_to_JsonbValue（Perl オブジェクトの変換）
  • JsonbValueToJsonb（JsonbValue から Jsonb への変換）

関連データ構造

  • SV (Perl Scalar Value)
  • JsonbValue
  • JsonbParseState
  • Jsonb

失敗モードと境界条件

  • SV_to_JsonbValue 内で、jsonb が許容しない値（NaN や無限大など）が検出された場合、エラーが発生する可能性があります。
  • Perl のデータ構造が深くネストしている場合、再帰処理によりスタックオーバーフローのリスクがあります。

保守担当者向けメモ

  • 本関数は jsonb への変換の最上位層です。変換ロジックの詳細は SV_to_JsonbValue に依存しています。
  • メモリ管理は PostgreSQL の palloc と Perl のメモリ管理が混在するため、変換プロセス全体でのリークに注意してください。
