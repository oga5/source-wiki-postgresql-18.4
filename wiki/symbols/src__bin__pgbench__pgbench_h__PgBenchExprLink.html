PgBenchExprLink

定義

  • 種別: struct
  • 定義箇所: src/bin/pgbench/pgbench.h:107

目的

pgbench の式評価エンジンにおいて、関数呼び出しの引数リストを構成するための連結リストのノードとして機能します。

シグネチャまたは形状

struct PgBenchExprLink {
    struct PgBenchExpr *expr;
    struct PgBenchExprLink *next;
};

*(注: pgbench.h 内の関連定義に基づく構造)*

入力と出力

  • **入力**: PgBenchExpr へのポインタを保持し、リストの次の要素へのポインタを保持します。
  • **出力**: 関数評価時に引数として渡されるリスト構造を提供します。

前提条件と不変条件

  • PgBenchExprLink は PgBenchExprList 構造体によって管理されることが想定されています。
  • リストの終端は next ポインタが NULL であることで示されます。

アルゴリズムまたは振る舞い

  • 式の評価フェーズにおいて、関数呼び出しの引数を順次辿るために使用されます。
  • pgbench.c 内の関数評価ロジックにおいて、引数リストの先頭から順に PgBenchExpr を評価するために走査されます。

呼び出し元と呼び出し先

  • **呼び出し元**: src/bin/pgbench/pgbench.c 内の関数評価ロジック（例: eval_function 等）が、引数リストの走査のために参照しています。
  • **呼び出し先**: PgBenchExpr 構造体を参照し、リスト構造を維持するために PgBenchExprLink 自身を再帰的に参照します。

関連データ構造

  • PgBenchExpr: リストが保持する個々の式ノード。
  • PgBenchExprList: head と tail を保持し、PgBenchExprLink のリスト全体を管理する構造体。

失敗モードと境界条件

  • リストが空の場合、head および tail ポインタは NULL となります。
  • 循環参照が発生しないよう、パーサー側でリスト構築時に適切に制御される必要があります。

保守担当者向けメモ

  • pgbench.h 内で前方宣言 (typedef struct PgBenchExprLink PgBenchExprLink;) されているため、ポインタ経由での利用が推奨されます。
  • 新しい関数を追加する際は、引数リストの走査ロジックがこの構造体の連結順序に依存している点に注意してください。
