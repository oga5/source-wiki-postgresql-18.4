AggInfo

定義

  • 種別: struct
  • 定義箇所: src/bin/pg_dump/pg_dump.h:252

目的

データベースの集約関数（Aggregate）に関する情報を保持するための構造体です。

シグネチャまたは形状

typedef struct _aggInfo
{
	FuncInfo	aggfn;
} AggInfo;

入力と出力

  • 入力: FuncInfo 型のメンバー aggfn を保持します。
  • 出力: なし（データ保持用構造体）。

前提条件と不変条件

  • 現時点では FuncInfo 以外のフィールドは定義されていません。

アルゴリズムまたは振る舞い

  • 本構造体は、pg_dump において集約関数オブジェクトを表現するために使用されます。
  • 実行時に FuncInfo を介して集約関数のメタデータを管理します。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/optimizer/prep/prepagg.c および src/backend/optimizer/plan/planner.c 等のオプティマイザ関連コードから参照されています。
  • 呼び出し先: 本構造体自体はデータ保持用であり、直接的な関数呼び出しは行いません。

関連データ構造

  • FuncInfo: AggInfo の唯一のメンバーとして含まれる、関数情報を保持する構造体。

失敗モードと境界条件

  • 構造体自体に失敗モードはありません。メモリ割り当て失敗時は標準的なシステムエラーが発生します。

保守担当者向けメモ

  • pg_dump の文脈では、集約関数に関連する追加情報が必要になった場合にフィールドを拡張する設計となっています。
  • オプティマイザ側（src/backend/optimizer/）でも同名の構造体が使用されていますが、pg_dump 用の定義と混同しないよう注意してください。
