r_prelude

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_italian.c:494

目的

イタリア語のステミング処理における前処理を実行し、特殊文字の置換および母音（'u', 'i'）の正規化を行う。

シグネチャまたは形状

static int r_prelude(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング対象の単語と環境状態を保持する構造体）
  • 出力: 成功時に 1 を返し、エラー発生時に負の整数を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体へのポインタであること。
  • header.h で定義された Snowball ランタイム環境が初期化されていること。

アルゴリズムまたは振る舞い

1. **特殊文字置換**: a_0 配列に基づき、単語内の特定の文字列を find_among で検索し、一致した場合は対応する文字列に置換する。 2. **母音の正規化**: 単語を走査し、以下の条件で母音を置換する。

  • 'u' が母音に挟まれている場合、'u' を別の文字に置換する。
  • 'i' が母音に挟まれている場合、'i' を別の文字に置換する。

    • 3. 処理中、z->bra および z->ket を用いて置換範囲を特定し、slice_from_s で文字列の書き換えを行う。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_ISO_8859_1_italian.c 内のメイン処理（italian_ISO_8859_1_stem 等から間接的に参照される可能性あり）。
  • 呼び出し先: find_among（文字列検索）、slice_from_s（文字列置換）、in_grouping（文字グループ判定）。
  • 注: 他の言語ファイル（Dutch, English, French 等）にも同名のシンボルが存在するが、本ページは stem_ISO_8859_1_italian.c 内の定義を対象としている。

関連データ構造

  • struct SN_env: ステミングの実行環境。
  • a_0: 置換ルールを定義した among 構造体の配列。
  • g_v: 母音グループを定義した文字セット。

失敗モードと境界条件

  • slice_from_s が負の値を返した場合、そのエラーコードをそのまま呼び出し元へ伝播する。
  • 文字列の末尾に達した場合や、置換対象が見つからない場合は走査を終了する。
