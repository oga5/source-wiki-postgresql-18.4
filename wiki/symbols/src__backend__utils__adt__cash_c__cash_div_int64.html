cash_div_int64

定義

  • **種別**: static inline 関数
  • **定義箇所**: src/backend/utils/adt/cash.c:155

目的

通貨型（Cash）の値を int64 型の値で除算し、その結果を返す。

シグネチャまたは形状

static inline Cash cash_div_int64(Cash c, int64 i)

入力と出力

  • **入力**:
  • c: 被除数となる Cash 型（内部的には int64）の値。
  • i: 除数となる int64 型の値。
  • **出力**:
  • 除算結果を Cash 型として返す。

前提条件と不変条件

  • Cash 型は int64 の別名として定義されている。
  • 除数 i が 0 の場合、本関数はエラーを報告し、呼び出し元へは戻らない。

アルゴリズムまたは振る舞い

1. unlikely マクロを使用して、除数 i が 0 であるかを確認する。 2. i == 0 の場合、ERRCODE_DIVISION_BY_ZERO を指定して ereport を呼び出し、エラーを発生させる。 3. 0 でない場合、c / i の演算結果を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/cash.c:845
  • src/backend/utils/adt/cash.c:885
  • src/backend/utils/adt/cash.c:923
  • **呼び出し先**:
  • ereport（エラー発生時）

関連データ構造

  • Cash: 通貨型を表す内部表現（int64）。src/include/utils/cash.h で定義されている。

失敗モードと境界条件

  • **ゼロ除算**: 除数 i が 0 の場合、ereport(ERROR, ...) により処理が中断される。

保守担当者向けメモ

  • 本関数は static inline で定義されており、主に通貨型に対する算術演算のラッパーとして機能する。
  • 演算結果のオーバーフローチェックは本関数内では明示的に行われていないため、必要に応じて呼び出し側で考慮すること。
