PGTYPEStimestamp_current

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/pgtypeslib/timestamp.c:289

目的

現在のシステム日時を取得し、ECPGで利用可能な timestamp 型へ変換して格納します。

シグネチャまたは形状

void PGTYPEStimestamp_current(timestamp * ts)

入力と出力

  • **入力**: ts (出力先となる timestamp 型変数へのポインタ)
  • **出力**: なし（引数 ts を介して結果を書き戻す）

前提条件と不変条件

  • ts は有効なメモリ領域を指している必要があります。
  • 内部で errno を参照するため、呼び出し前に他の処理で errno が汚染されていないことが期待されます。

アルゴリズムまたは振る舞い

1. struct tm 型のローカル変数 tm を宣言します。 2. GetCurrentDateTime(&tm) を呼び出し、現在の日時情報を取得します。 3. errno が 0（成功）であることを確認します。 4. tm2timestamp(&tm, 0, NULL, ts) を呼び出し、取得した日時情報を timestamp 型（int64）に変換して ts に格納します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/interfaces/ecpg/compatlib/informix.c:614
  • src/interfaces/ecpg/test/expected/pgtypeslib-dt_test.c:252
  • **呼び出し先**:
  • GetCurrentDateTime
  • tm2timestamp

関連データ構造

  • timestamp: int64 として定義されるマイクロ秒単位の時刻型。
  • struct tm: C標準ライブラリの日時構造体。

失敗モードと境界条件

  • GetCurrentDateTime が失敗し errno が 0 以外となった場合、ts への書き込みは行われません。
  • tm2timestamp が内部で計算エラー（範囲外など）を検出した場合の挙動は、tm2timestamp の仕様に依存します。

保守担当者向けメモ

  • 本関数は errno の状態に依存しています。マルチスレッド環境等で errno が共有される場合、呼び出し直後の状態に注意が必要です。
  • tm2timestamp の第2引数に 0（fsec）、第3引数に NULL（timezone）を渡しているため、常にローカルタイムまたはシステム標準の解釈で変換されます。
