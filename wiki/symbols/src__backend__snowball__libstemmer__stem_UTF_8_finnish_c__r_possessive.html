r_possessive

定義

  • **種別**: function
  • **定義箇所**: src/backend/snowball/libstemmer/stem_UTF_8_finnish.c:335

目的

フィンランド語のステミング処理において、単語の所有接尾辞（possessive suffixes）を特定し、条件に応じて削除または置換を行う。

シグネチャまたは形状

static int r_possessive(struct SN_env * z);

入力と出力

  • **入力**: SN_env * z（Snowball ステミング環境構造体）。
  • **出力**: 成功時に 1、処理対象外または失敗時に 0、エラー時に負の値を返す。

前提条件と不変条件

  • z->I[1]（領域 R1 の開始位置）が設定されていること。
  • 現在のカーソル位置 z->c が z->I[1] 以上であること。
  • a_4 配列に定義された所有接尾辞のパターンマッチングを利用する。

アルゴリズムまたは振る舞い

1. **領域制限**: z->I[1] を下限として、所有接尾辞の検索範囲を限定する。 2. **パターンマッチング**: find_among_b を使用し、末尾から所有接尾辞を検索する。 3. **条件分岐**: マッチした接尾辞の種類（among_var）に応じて以下の処理を行う：

  • **ケース1**: 直前に 'k' がない場合のみ削除。
  • **ケース2**: 接尾辞を削除し、特定の文字列（s_1）に置換。
  • **ケース3**: 単純削除。
  • **ケース4-6**: 特定の文字（'a', 'ä', 'e'）が直前にある場合のみ、対応する接尾辞を削除。

    • 4. **状態更新**: 削除や置換が成功した場合、z->bra および z->ket を更新し、slice_del または slice_from_s を実行する。

呼び出し元と呼び出し先

  • **呼び出し元**: finnish_UTF_8_stem（src/backend/snowball/libstemmer/stem_UTF_8_finnish.c 内）。
  • **呼び出し先**: find_among_b, slice_del, slice_from_s, eq_s_b（いずれも Snowball ランタイム関数）。

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_4: 所有接尾辞を定義した among 構造体の配列。
  • a_1, a_2, a_3: ケース4-6で使用される補助的な接尾辞定義配列。

失敗モードと境界条件

  • z->c < z->I[1] の場合、処理を行わず 0 を返す。
  • slice_del や slice_from_s がメモリ不足等の理由で負の値を返した場合、その値をそのまま呼び出し元に伝播させる。
  • マッチング対象が見つからない場合は 0 を返す。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードである。アルゴリズムの変更が必要な場合は、直接編集せず Snowball ソースファイルを修正すること。
  • z->I[1] は領域 R1 を指すインデックスであり、ステミングの境界条件として重要である。
