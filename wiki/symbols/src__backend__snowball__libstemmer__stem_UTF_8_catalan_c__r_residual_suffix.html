目的

カタルーニャ語のステミング処理において、残余の接尾辞を特定し、必要に応じて削除または置換を行う。

シグネチャまたは形状

static int r_residual_suffix(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z（Snowballステミング環境）
  • **出力**: int（成功時は 1、失敗または処理対象外の場合は 0、エラー時は負の値）

前提条件と不変条件

  • z は有効な SN_env 構造体であること。
  • find_among_b を使用するため、文字列の末尾から前方に向かって検索が行われる。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を z->ket（削除対象の末尾）に設定する。 2. a_4 配列（22個の要素）を用いて、末尾から接尾辞を検索する。 3. 一致するものがない場合、0 を返して終了する。 4. 一致した場合、現在のカーソル位置を z->bra（削除対象の開始）に設定し、among_var の値に応じて以下の処理を行う：

  • **case 1**: r_R1(z) を呼び出し、R1領域内であれば slice_del(z) で削除する。
  • **case 2**: r_R1(z) を呼び出し、R1領域内であれば slice_from_s(z, 2, s_9) で指定文字列に置換する。

    • 5. 処理成功時に 1 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_UTF_8_catalan.c 内のステミングメイン処理フローから呼び出される。
  • **呼び出し先**: find_among_b, r_R1, slice_del, slice_from_s。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_4: 接尾辞検索用の among 構造体配列。

失敗モードと境界条件

  • r_R1 が失敗（戻り値が 0 以下）した場合、その戻り値をそのまま返して処理を中断する。
  • slice_del または slice_from_s が失敗（戻り値が負）した場合、その戻り値を返して処理を中断する。
