getrusage

定義

  • 種別: function
  • 定義箇所: src/port/win32getrusage.c:20

目的

Windows環境においてPOSIX標準の getrusage() をエミュレートし、現在のプロセス（RUSAGE_SELF）のリソース使用状況（カーネル時間およびユーザー時間）を取得する。

シグネチャまたは形状

int getrusage(int who, struct rusage *rusage);

入力と出力

  • **入力**:
  • who: 取得対象を指定するフラグ（RUSAGE_SELF のみをサポート）。
  • rusage: 結果を格納する struct rusage 構造体へのポインタ。
  • **出力**:
  • 成功時: 0 を返し、rusage 構造体に実行時間が格納される。
  • 失敗時: -1 を返し、errno にエラーコード（EINVAL または EFAULT）を設定する。

前提条件と不変条件

  • who 引数は RUSAGE_SELF である必要がある。
  • rusage 引数は有効なメモリ領域を指している必要がある。
  • Windows API GetProcessTimes() が正常に実行可能であること。

アルゴリズムまたは振る舞い

1. who が RUSAGE_SELF 以外の場合、errno を EINVAL に設定し -1 を返す。 2. rusage ポインタが NULL の場合、errno を EFAULT に設定し -1 を返す。 3. rusage 構造体をゼロクリアする。 4. GetProcessTimes() を呼び出し、現在のプロセスの実行時間を取得する。失敗時は _dosmaperr() を用いてWin32エラーをPOSIXエラーに変換し -1 を返す。 5. 取得した FILETIME（100ナノ秒単位）をマイクロ秒単位に変換し、struct timeval 形式（tv_sec, tv_usec）に分割して rusage に格納する。 6. 成功時は 0 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/tcop/postgres.c
  • src/backend/utils/misc/pg_rusage.c
  • **呼び出し先**:
  • GetProcessTimes() (Win32 API)
  • GetCurrentProcess() (Win32 API)
  • _dosmaperr() (Win32エラー変換用ユーティリティ)
  • memset, memcpy (標準Cライブラリ)

関連データ構造

  • struct rusage: リソース使用状況を保持する構造体。
  • FILETIME: Win32 APIで使用される時間形式。
  • struct timeval: POSIX標準の時間形式。

失敗モードと境界条件

  • RUSAGE_SELF 以外のフラグ指定: EINVAL を返す。
  • rusage ポインタが NULL: EFAULT を返す。
  • GetProcessTimes() の失敗: GetLastError() を取得し、_dosmaperr() を経由して対応するPOSIXエラーを errno に設定する。

保守担当者向けメモ

  • FILETIME からマイクロ秒への変換において、100ナノ秒単位を10で割ることでマイクロ秒に変換している。
