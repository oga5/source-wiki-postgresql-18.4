UInt8GetDatum

定義

  • 種別: static inline 関数
  • 定義箇所: src/include/postgres.h:156

目的

uint8 型の値を、PostgreSQL の汎用データ表現である Datum 型に変換する。

シグネチャまたは形状

static inline Datum UInt8GetDatum(uint8 X)

入力と出力

  • **入力**: uint8 X （変換対象の 8 ビット符号なし整数）
  • **出力**: Datum （変換後の汎用データ型）

前提条件と不変条件

  • Datum 型は sizeof(void *) と同じサイズであると仮定されており、本関数は uint8 を Datum へキャストして返します。

アルゴリズムまたは振る舞い

引数として受け取った uint8 型の値 X を、Datum 型へ明示的にキャストして返します。インライン関数として定義されているため、関数呼び出しのオーバーヘッドなしに変換が行われます。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/pageinspect/heapfuncs.c
  • src/backend/access/nbtree/nbtcompare.c
  • **呼び出し先**:
  • 本関数は単一のキャスト操作のみを行うため、他の関数を呼び出しません。

関連データ構造

  • Datum: PostgreSQL の実行エンジン内でデータを効率的に受け渡すための汎用型。
  • uint8: c.h で定義される 8 ビット符号なし整数型。

失敗モードと境界条件

  • 本関数自体は単純なキャストであるため、実行時の失敗モードはありません。
  • 入力値が uint8 の範囲（0〜255）を超えない限り、値は正確に保持されます。

保守担当者向けメモ

  • 本関数は postgres.h 内の他の *GetDatum 関数群と同様のパターンに従っています。変更を加える際は、他の変換関数との一貫性を維持してください。
  • 64bit 環境と 32bit 環境の両方で Datum のサイズが適切に扱われるよう、Datum 型の定義（sizeof(void *)）に依存している点に注意してください。
