vartag_external

定義

  • 種別: enum
  • 定義箇所: src/include/varatt.h:84

目的

TOAST（The Oversized-Attribute Storage Technique）ポインタが指し示すデータの格納形式や状態を識別するために使用されます。

シグネチャまたは形状

typedef enum vartag_external
{
    VARTAG_INDIRECT = 1,
    VARTAG_EXPANDED_RO = 2,
    VARTAG_EXPANDED_RW = 3,
    VARTAG_ONDISK = 18
} vartag_external;

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: なし（列挙型定義）

前提条件と不変条件

  • 各列挙値は、TOASTポインタのヘッダー情報と対応し、データがメモリ内間接参照、展開済みオブジェクト（読み取り専用/読み書き可能）、またはディスク上のいずれであるかを一意に特定します。

アルゴリズムまたは振る舞い

  • VARTAG_INDIRECT: メモリ上の別の場所にあるデータへの間接参照を示します。
  • VARTAG_EXPANDED_RO: 展開済みオブジェクトであり、読み取り専用であることを示します。
  • VARTAG_EXPANDED_RW: 展開済みオブジェクトであり、読み書き可能であることを示します。
  • VARTAG_ONDISK: データがTOASTテーブル（ディスク）上に存在することを示します。

呼び出し元と呼び出し先

  • 呼び出し元: 提供されたソース情報からは特定の呼び出し元を特定できません。
  • 呼び出し先: なし（列挙型定義のため）。

関連データ構造

  • varatt_external: TOASTポインタの構造体。本列挙型はこの構造体内で格納形式を識別するために利用されます。

失敗モードと境界条件

  • 定義外の値を設定した場合の挙動は未定義です。

保守担当者向けメモ

  • 本列挙型は varatt_external 構造体の一部として使用されます。値の追加や変更を行う際は、TOASTポインタを解釈するすべての関連マクロおよび関数（VARATT_IS_... 等）との整合性を確認してください。
