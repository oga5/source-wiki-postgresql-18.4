pg_prng_int64

定義

  • 種別: function
  • 定義箇所: src/common/pg_prng.c:172

目的

擬似乱数生成器の状態から、64ビット符号付き整数（int64）の乱数値を生成する。

シグネチャまたは形状

int64 pg_prng_int64(pg_prng_state *state)

入力と出力

  • **入力**: state - 乱数生成器の状態を保持する pg_prng_state 構造体へのポインタ。
  • **出力**: int64 型の乱数値。

前提条件と不変条件

  • state は初期化済みであること。
  • 本関数は暗号学的な安全性を提供しない。パスワードや暗号鍵の生成には使用不可。
  • pg_prng_state の全ビットが0の場合、生成器は0を返し続ける不動点となるため、初期化時に pg_prng_seed_check 等で回避されている必要がある。

アルゴリズムまたは振る舞い

1. 内部関数 xoroshiro128ss(state) を呼び出し、64ビットの乱数値を生成する。 2. 得られた値を int64 型にキャストして返却する。

呼び出し元と呼び出し先

  • **呼び出し先**: xoroshiro128ss
  • **呼び出し元**: 提供されたソース情報には、本関数を直接呼び出している具体的な関数名は記載されていない。

関連データ構造

  • pg_prng_state: 乱数生成器の状態を保持する構造体。

失敗モードと境界条件

  • 状態がすべて0で初期化された場合、常に0を返す。
  • 内部で使用する xoroshiro128ss の特性上、生成される値は一様分布に従うが、暗号学的な予測不可能性は保証されない。

保守担当者向けメモ

  • 本関数は xoroshiro128ss の結果をそのままキャストする単純なラッパーである。
  • 範囲指定が必要な場合は、pg_prng_int64_range の使用を検討すること。
