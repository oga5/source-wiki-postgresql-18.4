r_mark_regions

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_french.c:536

目的

フランス語のステミング処理において、単語内の領域（RV, R1, R2）の境界を決定し、SN_env 構造体の整数配列 I に格納する。

シグネチャまたは形状

static int r_mark_regions(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z（Snowballランタイム環境）
  • **出力**: 常に 1 を返す。
  • **副作用**: z->I[0]（R2境界）、z->I[1]（R1境界）、z->I[2]（RV境界）を更新する。

前提条件と不変条件

  • z は有効な SN_env 構造体を指していること。
  • g_v（母音グループ）が定義されていること。

アルゴリズムまたは振る舞い

1. **初期化**: I[0], I[1], I[2] を単語長 z->l で初期化する。 2. **RVの決定**:

  • 単語の先頭2文字が母音であるか、あるいは特定の接頭辞（a_0）にマッチするか、または子音であるかを判定し、RVの開始位置を z->I[2] に設定する。

    • 3. **R1, R2の決定**:

  • 母音と子音の遷移（out_grouping と in_grouping の組み合わせ）を走査する。
  • 最初の母音から子音への遷移後にR1の境界を z->I[1] に設定する。
  • さらに次の母音から子音への遷移後にR2の境界を z->I[0] に設定する。

    • 4. 処理終了後、カーソル位置を元の位置に復元する。

呼び出し元と呼び出し先

  • **呼び出し元**: ソースコード内の直接的な呼び出し関係は本抜粋からは特定できない。
  • **呼び出し先**: in_grouping, out_grouping, find_among（Snowballランタイム関数）。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • g_v: 母音を定義するグループ。
  • a_0: 接頭辞検索用の among テーブル。

失敗モードと境界条件

  • 単語が短すぎる場合や、母音・子音の遷移が見つからない場合、境界値は初期値（単語長）のまま維持される。
  • z->c（カーソル）が z->l（単語長）を超える操作はガードされている。

保守担当者向けメモ

  • 本関数はSnowballコンパイラによって自動生成されたコードである。ロジックの変更が必要な場合は、対応する .sbl ファイルを修正し、コンパイラで再生成すること。
  • 領域境界の計算ロジックはフランス語のステミング規則に依存している。
