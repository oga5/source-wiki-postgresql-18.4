statext_compute_stattarget

定義

  • src/backend/statistics/extended_stats.c:343

目的

拡張統計オブジェクトの計算に使用する統計ターゲット（精度）を決定する。

シグネチャまたは形状

int statext_compute_stattarget(int stattarget, int nattrs, VacAttrStats **stats)

入力と出力

  • **入力**:
  • stattarget: 統計オブジェクトに設定されたターゲット値（負の値は未設定を示す）。
  • nattrs: 統計対象となる属性の数。
  • stats: 各属性の統計情報を含む VacAttrStats 構造体の配列。
  • **出力**: 決定された統計ターゲット値（int）。

前提条件と不変条件

  • 戻り値は常に 0 以上 MAX_STATISTICS_TARGET 以下であること（Assert による検証）。

アルゴリズムまたは振る舞い

1. **オブジェクト設定の優先**: stattarget が 0 以上であれば、その値をそのまま返す（0 は計算無効化を意味する）。 2. **属性設定の参照**: stattarget が負の場合、対象となる全属性（stats）の attstattarget を走査し、その最大値を新たなターゲットとする。 3. **デフォルト値の適用**: 上記のいずれでも有効なターゲットが決定できない（負のままの）場合、グローバル変数 default_statistics_target を使用する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/statistics/extended_stats.c 内の複数箇所（BuildRelationExtStatistics 等の統計構築プロセス）。
  • **呼び出し先**: 本関数は計算ロジックのみを持ち、他の関数を呼び出さない。

関連データ構造

  • VacAttrStats: 属性ごとの統計情報を保持する構造体。

失敗モードと境界条件

  • stattarget が 0 の場合、統計計算が抑止される。
  • 全ての属性でターゲットが設定されていない場合、システム全体のデフォルト値にフォールバックする。

保守担当者向けメモ

  • 本関数は統計オブジェクトの精度決定における単一の真実源として機能する。
  • MAX_STATISTICS_TARGET を超える値が返ることはない。
