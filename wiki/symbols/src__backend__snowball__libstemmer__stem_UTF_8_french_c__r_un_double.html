r_un_double

目的

フランス語のステミング処理において、単語末尾の二重子音を解消（縮約）します。

シグネチャまたは形状

static int r_un_double(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング環境構造体）
  • 出力: 処理が成功した場合は 1、失敗または条件不一致の場合は 0、あるいは slice_del の戻り値（負数）を返します。

前提条件と不変条件

  • z が有効な SN_env 構造体を指していること。
  • z->c（カーソル位置）が単語の末尾付近にあること。

アルゴリズムまたは振る舞い

1. **条件判定**: 現在のカーソル位置から前方（逆方向）へ2文字以上離れているか、および直前の文字が特定の二重子音候補（a_8 配列で定義）に該当するかを確認します。 2. **マッチング**: find_among_b を使用して、末尾の二重子音パターンを検索します。 3. **削除範囲の特定**: マッチした二重子音のうち、1文字分を削除対象として z->ket（終了位置）と z->bra（開始位置）に設定します。 4. **削除実行**: slice_del(z) を呼び出し、該当する1文字をバッファから削除します。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_french.c および src/backend/snowball/libstemmer/stem_ISO_8859_1_french.c 内のメイン処理フローから呼び出されます。
  • 呼び出し先: find_among_b（パターンマッチング）、skip_b_utf8（カーソル移動）、slice_del（文字列削除）。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_8: 二重子音のパターンを定義する among 構造体の配列。

失敗モードと境界条件

  • カーソルがバッファの先頭に近い場合（z->c - 2 <= z->lb）、処理を中断し 0 を返します。
  • find_among_b がパターンに一致しない場合、0 を返します。
  • slice_del がメモリ操作等で失敗した場合、負の値を返します。
