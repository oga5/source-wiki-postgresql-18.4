PLy_procedure_name

定義

  • 種別: function
  • 定義箇所: src/pl/plpython/plpy_procedure.c:45

目的

指定された PLyProcedure 構造体から、対応するプロシージャ（関数）の名前を取得します。

シグネチャまたは形状

char *PLy_procedure_name(PLyProcedure *proc)

入力と出力

  • **入力**: proc - PLyProcedure 構造体へのポインタ。
  • **出力**: プロシージャ名を表す char * 型の文字列。proc が NULL の場合は " <unknown procedure>" を返します。

前提条件と不変条件

  • proc が有効な PLyProcedure 構造体を指している場合、その内部の proname フィールドがプロシージャ名として参照されます。

アルゴリズムまたは振る舞い

1. 引数 proc が NULL かどうかを判定します。 2. NULL であれば、静的な文字列 "<unknown procedure>" を返します。 3. NULL でなければ、proc 構造体内の proname フィールドを返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/pl/plpython/plpy_elog.c:289
  • src/pl/plpython/plpy_main.c:354
  • src/pl/plpython/plpy_main.c:357
  • **呼び出し先**: 該当なし（本関数は単純なフィールドアクセスのみを行います）。

関連データ構造

  • PLyProcedure: プロシージャの実行状態やメタデータを保持する構造体。本関数はこの構造体の proname フィールドに依存しています。

失敗モードと境界条件

  • proc が NULL の場合、プログラムの異常終了を避け、安全な代替文字列を返します。

保守担当者向けメモ

  • 本関数は単なるアクセサです。PLyProcedure 構造体の定義が変更された場合、proname フィールドの保持方法に影響がないか確認してください。
