Int16GetDatum

定義

  • 種別: static inline 関数
  • 定義箇所: src/include/postgres.h:176

目的

int16 型の値を、PostgreSQL の実行エンジン内で汎用的に受け渡される Datum 型へ変換します。

シグネチャまたは形状

static inline Datum Int16GetDatum(int16 X)

入力と出力

  • **入力**: int16 X （変換対象の 16 ビット整数値）
  • **出力**: Datum （int16 値を保持する Datum 型の表現）

前提条件と不変条件

  • Datum 型は sizeof(void *) と同じサイズであると仮定されています。
  • 本関数はインライン化されるため、関数呼び出しのオーバーヘッドなしに型変換を行います。

アルゴリズムまたは振る舞い

引数として渡された int16 型の値 X を、明示的に Datum 型へキャストして返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/btree_gin/btree_gin.c
  • contrib/btree_gist/btree_utils_num.c
  • contrib/pg_buffercache/pg_buffercache_pages.c
  • contrib/pg_walinspect/pg_walinspect.c
  • contrib/sepgsql/dml.c
  • contrib/sepgsql/relation.c
  • src/backend/access/brin/brin_inclusion.c
  • src/backend/access/brin/brin_minmax.c
  • src/backend/access/brin/brin_minmax_multi.c
  • src/backend/access/gist/gistget.c
  • **呼び出し先**:
  • なし（言語組み込みのキャスト演算のみを実行）。

関連データ構造

  • Datum: PostgreSQL における汎用データ表現型。

失敗モードと境界条件

  • 本関数自体は単純なキャストであるため、関数内部での失敗モードはありません。
  • int16 の範囲外の値を渡した場合の挙動は、C 言語のキャスト規則に従います。

保守担当者向けメモ

  • Datum 型のサイズはプラットフォーム（32bit/64bit）に依存するため、移植性を維持するために必ずこの変換関数を使用してください。直接的なキャストは避けるべきです。
