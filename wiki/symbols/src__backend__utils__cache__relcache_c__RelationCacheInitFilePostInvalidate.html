RelationCacheInitFilePostInvalidate

定義

  • 種別: function
  • 定義箇所: src/backend/utils/cache/relcache.c:6884

目的

リレーションキャッシュの初期化ファイルに関連する排他制御を解除します。

シグネチャまたは形状

void RelationCacheInitFilePostInvalidate(void)

入力と出力

  • 入力: なし
  • 出力: なし

前提条件と不変条件

  • 本関数を呼び出す前に、RelationCacheInitFilePreInvalidate 等によって RelCacheInitLock が獲得されている必要があります。

アルゴリズムまたは振る舞い

1. RelCacheInitLock（LWLock）を解放します。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/backend/access/transam/twophase.c
  • src/backend/utils/cache/inval.c
  • 呼び出し先:
  • LWLockRelease（PostgreSQL 内部のロック管理関数）

関連データ構造

  • RelCacheInitLock: リレーションキャッシュ初期化ファイルの整合性を保護するための軽量ロック（LWLock）。

失敗モードと境界条件

  • ロックが保持されていない状態で呼び出した場合の挙動は、LWLockRelease の仕様に依存します。

保守担当者向けメモ

  • 本関数は RelationCacheInitFilePreInvalidate と対になる関数です。初期化ファイルの操作が完了した後に必ず呼び出される必要があります。
