目的

現在のカーソル位置が、単語内の「R2」領域の開始位置以降にあるかどうかを判定する。

シグネチャまたは形状

static int r_R2(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z （Snowballランタイム環境）
  • 出力: int （条件を満たす場合は 1、それ以外は 0）

前提条件と不変条件

  • z->I[0] には、r_mark_regions によって計算された R2 領域の開始オフセットが格納されている必要がある。
  • z->c は現在のカーソル位置を示す。

アルゴリズムまたは振る舞い

1. z->I[0]（R2領域の開始位置）と z->c（現在のカーソル位置）を比較する。 2. z->I[0] <= z->c が真であれば 1 を返し、偽であれば 0 を返す。
