PostPrepare_Inval

定義

  • **種別**: function
  • **定義箇所**: src/backend/utils/cache/inval.c:992

目的

トランザクションの準備（PREPARE TRANSACTION）フェーズにおいて、キャッシュ無効化メッセージの処理を調整する。

シグネチャまたは形状

void PostPrepare_Inval(void)

入力と出力

  • **入力**: なし
  • **出力**: なし（副作用としてキャッシュ無効化メッセージの処理を実行）

前提条件と不変条件

  • 本関数はトランザクションの準備処理中に呼び出されることを想定している。

アルゴリズムまたは振る舞い

1. AtEOXact_Inval(false) を呼び出す。 2. AtEOXact_Inval に対して false を渡すことで、トランザクション終了時のメッセージ送出処理を準備フェーズ向けに実行する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/access/transam/xact.c:2737
  • **呼び出し先**: AtEOXact_Inval

関連データ構造

  • TransInvalidationInfo（AtEOXact_Inval 経由で参照される）

失敗モードと境界条件

  • 特筆すべき失敗モードはソース内に明示されていない。

保守担当者向けメモ

  • 本関数は AtEOXact_Inval のラッパーである。トランザクションの準備とコミットで無効化メッセージの扱いを分ける必要がある場合に、このインターフェースが利用される。
