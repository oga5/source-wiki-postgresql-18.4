gid_t

定義

  • 種別: typedef
  • 定義箇所: src/include/port/win32_port.h:235

目的

Windows 環境において、POSIX 準拠のグループ ID を表現するための型定義を提供します。

シグネチャまたは形状

typedef int gid_t;

入力と出力

  • 入力: 整数値（グループ ID）
  • 出力: int 型としてグループ ID を保持

前提条件と不変条件

  • Windows ネイティブビルドおよび MinGW ビルドで使用されます。
  • Perl が既に gid_t を定義している環境では、競合が発生する可能性があるため注意が必要です。

アルゴリズムまたは振る舞い

  • POSIX システムにおける gid_t と同等の役割を果たす整数型として機能します。
  • Windows API には直接対応する型がないため、int によるエイリアスとして実装されています。

呼び出し元と呼び出し先

  • 呼び出し元（参照箇所）:
  • src/backend/libpq/auth.c
  • src/backend/libpq/pqcomm.c
  • src/include/fe_utils/astreamer.h
  • src/include/pgtar.h
  • src/include/port.h
  • src/interfaces/libpq/fe-connect.c
  • src/port/getpeereid.c
  • src/port/tar.c
  • 呼び出し先: なし（型定義のため）

関連データ構造

  • uid_t: ユーザー ID を表す対となる型。
  • mode_t: ファイルモードを表す型。

失敗モードと境界条件

  • getpeereid 等の関数において、無効な値として (gid_t) (-1) が使用されるケースがあります。

保守担当者向けメモ

  • 本定義は Windows 環境専用です。Cygwin ビルドでは読み込まれません。
  • Perl との共存環境では定義の重複や競合に注意してください。
