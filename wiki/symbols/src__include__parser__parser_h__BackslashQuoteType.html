BackslashQuoteType

定義

  • 種別: enum
  • 定義箇所: src/include/parser/parser.h:48

目的

SQL 文字列リテラル内におけるバックスラッシュ（\）の扱いを制御するための設定値を定義します。

シグネチャまたは形状

typedef enum
{
    BACKSLASH_QUOTE_OFF,
    BACKSLASH_QUOTE_ON,
    BACKSLASH_QUOTE_SAFE_ENCODING,
} BackslashQuoteType;

入力と出力

  • 本シンボルは列挙型であり、関数ではありません。
  • GUC（Grand Unified Configuration）変数 backslash_quote の値として使用されます。

前提条件と不変条件

  • 字句解析器（scan.l）が SQL 文字列を解析する際、この列挙値に基づいてバックスラッシュの解釈を決定します。

アルゴリズムまたは振る舞い

  • BACKSLASH_QUOTE_OFF: バックスラッシュをエスケープ文字として扱わない（または無効化する）。
  • BACKSLASH_QUOTE_ON: バックスラッシュをエスケープ文字として扱う。
  • BACKSLASH_QUOTE_SAFE_ENCODING: 現在のクライアントエンコーディングにおいて、バックスラッシュが安全にエスケープ文字として使用できる場合にのみ許可する。

呼び出し元と呼び出し先

  • 呼び出し元および呼び出し先の詳細は、提供されたソース情報からは特定できません。
  • src/include/parser/parser.h 内で定義され、src/backend/parser/scan.l（字句解析器）によって参照されることが示唆されています。

関連データ構造

  • nodes/parsenodes.h: 構文解析に関連するノード定義が含まれます。

失敗モードと境界条件

  • 本シンボル自体に失敗モードはありません。
  • 設定値が不適切な場合、字句解析器が SQL 文字列を誤って解釈する可能性があります。

保守担当者向けメモ

  • 本列挙型は scan.l で使用されるグローバルな設定値と密接に関連しています。変更を加える際は、字句解析器の挙動への影響を考慮してください。
