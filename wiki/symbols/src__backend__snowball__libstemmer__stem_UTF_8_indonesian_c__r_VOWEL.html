目的

現在のカーソル位置にある文字が母音であるかを判定し、ステミング処理の条件分岐に利用する。

シグネチャまたは形状

static int r_VOWEL(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z（Snowball ステミング環境）
  • 出力: int 型。母音であれば 0 を返し、それ以外であれば 1 を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • g_vowel グローバル変数が母音の定義として初期化されていること。

アルゴリズムまたは振る舞い

1. in_grouping_U 関数を呼び出し、現在の文字が g_vowel グループ（ASCII 97から117の範囲）に含まれるかを確認する。 2. 文字が母音グループに含まれる場合、0 を返して終了する。 3. 文字が母音グループに含まれない場合、1 を返して終了する。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_indonesian.c 内の among 構造体配列（a_3 等）から間接的に呼び出される。
  • 呼び出し先: in_grouping_U（Snowball ランタイム関数）。

  • struct SN_env: ステミング処理の状態を保持する構造体。
  • g_vowel: 母音を定義する文字グループ。

失敗モードと境界条件

  • z が NULL の場合の挙動はランタイムの in_grouping_U 実装に依存する。
  • 文字列の終端に達している場合、in_grouping_U の仕様に従い判定が行われる。
