pg_atomic_unlocked_test_flag_impl

定義

  • 種別: function
  • 定義箇所: src/include/port/atomics/generic.h:90

目的

アトミックフラグが現在「設定されていない（クリアされている）」状態であるかを、ロックを使用せずに判定する。

シグネチャまたは形状

static inline bool pg_atomic_unlocked_test_flag_impl(volatile pg_atomic_flag *ptr)

入力と出力

  • **入力**: ptr - 検査対象の pg_atomic_flag 構造体へのポインタ。
  • **出力**: フラグの値が 0 であれば true、それ以外であれば false を返す。

前提条件と不変条件

  • ptr は有効な pg_atomic_flag を指している必要がある。
  • 本関数は「アンロック」操作であるため、アトミックな同期保証を伴わない読み取りを行う。

アルゴリズムまたは振る舞い

1. pg_atomic_read_u32_impl(ptr) を呼び出し、フラグの現在の値を読み取る。 2. 読み取った値が 0 と等しいかどうかを比較し、その結果をブール値として返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/include/port/atomics.h:198 からの呼び出しが確認されている。
  • **呼び出し先**: pg_atomic_read_u32_impl を内部で呼び出す。

関連データ構造

  • pg_atomic_flag: アトミック操作に使用されるフラグ型。

失敗モードと境界条件

  • 本関数は非アトミックな読み取りを行うため、他のスレッドが同時にフラグを更新している場合、返される値は最新の状態を反映していない可能性がある。

保守担当者向けメモ

  • 本関数は generic.h 内で定義されており、アーキテクチャ固有の実装が提供されていない場合のフォールバックとして機能する。
  • generic-gcc.h 等の他のヘッダーファイルでも同名の関数定義が見られるが、これらはコンパイラやアーキテクチャに応じた実装のバリエーションである。
