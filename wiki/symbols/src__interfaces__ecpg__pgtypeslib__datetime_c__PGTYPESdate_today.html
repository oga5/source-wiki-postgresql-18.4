PGTYPESdate_today

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/pgtypeslib/datetime.c:147

目的

現在の日付を取得し、ECPGの内部日付表現（date型）に変換して格納します。

シグネチャまたは形状

void PGTYPESdate_today(date *d)

入力と出力

  • **入力**: date *d （日付値を格納するためのポインタ）
  • **出力**: なし（引数 d が指すメモリ領域を更新）

前提条件と不変条件

  • d は有効な date 型のメモリ領域を指している必要があります。
  • 内部的な日付計算は、2000年1月1日を基準（date2j(2000, 1, 1)）としたオフセット値として扱われます。

アルゴリズムまたは振る舞い

1. GetCurrentDateTime(&ts) を呼び出し、現在のシステム日時を struct tm 構造体に取得します。 2. errno が 0（正常）であることを確認します。 3. date2j 関数を用いて、取得した年・月・日からユリウス日を計算します。 4. 計算したユリウス日から基準日（2000年1月1日）のユリウス日を減算し、その結果を *d に代入します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/interfaces/ecpg/compatlib/informix.c:537 からの呼び出しが確認されています。
  • **呼び出し先**: GetCurrentDateTime および date2j を呼び出します。

関連データ構造

  • date: ECPG内で使用される日付型（内部的には整数値）。
  • struct tm: C標準ライブラリの日時構造体。

失敗モードと境界条件

  • GetCurrentDateTime が失敗し errno が 0 以外となった場合、*d の値は更新されません。

保守担当者向けメモ

  • 本関数は src/interfaces/ecpg/include/pgtypes_date.h で宣言されています。
  • 基準日（2000-01-01）の変更は、計算ロジック全体に影響するため注意してください。
