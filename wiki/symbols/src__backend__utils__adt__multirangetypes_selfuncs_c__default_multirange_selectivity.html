default_multirange_selectivity

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/multirangetypes_selfuncs.c:77

目的

統計情報が利用できない、または計算不能な状況において、マルチレンジ型に関連する演算子の選択率（selectivity）のデフォルト値を返します。

シグネチャまたは形状

static double default_multirange_selectivity(Oid operator)

入力と出力

  • **入力**: operator (Oid) - 評価対象の演算子識別子。
  • **出力**: double - 推定された選択率（0.0 から 1.0 の範囲）。

前提条件と不変条件

  • 入力された operator はマルチレンジ型に関連する演算子であることを想定しています。
  • 統計情報が不足している場合や、ヒストグラムによる計算が適用できない場合のフォールバックとして使用されます。

アルゴリズムまたは振る舞い

指定された演算子に応じて以下の値を返します。 1. **0.01**: OVERLAPS 系の演算子（&&, @> 等の組み合わせ）。 2. **0.005**: CONTAINS または CONTAINED 系の演算子。 3. **DEFAULT_MULTIRANGE_INEQ_SEL**: マルチレンジと要素の包含演算子（@> または <@）。 4. **DEFAULT_INEQ_SEL**: 不等号演算子（<, <=, >, >=）および LEFT/RIGHT 系の演算子。 5. **0.01**: 上記のいずれにも該当しない場合のデフォルト値。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/multirangetypes_selfuncs.c 内の複数の箇所（統計計算のフォールバックとして使用）。
  • **呼び出し先**:
  • 外部関数への呼び出しは確認されていません。

関連データ構造

  • Oid: 演算子を識別するための型。
  • DEFAULT_MULTIRANGE_INEQ_SEL, DEFAULT_INEQ_SEL: 選択率の定数マクロ。

失敗モードと境界条件

  • 未知の operator が渡された場合、安全策として 0.01 を返します。
  • 統計情報が欠如している状態での推定であるため、実際のデータ分布とは乖離する可能性があります。

保守担当者向けメモ

  • 本関数は multirangesel 等のメインの選択率推定ロジックから、計算失敗時の安全な値として参照されます。
  • 新しいマルチレンジ演算子を追加した場合は、本関数の switch 文に適切なデフォルト値を追記してください。
