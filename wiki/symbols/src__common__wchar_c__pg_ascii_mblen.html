pg_ascii_mblen

定義

  • 種別: function
  • 定義箇所: src/common/wchar.c:87

目的

SQL_ASCII エンコーディングにおける、単一文字のバイト長を返します。

シグネチャまたは形状

int pg_ascii_mblen(const unsigned char *s);

入力と出力

  • **入力**: s (文字シーケンスの先頭を指すポインタ)
  • **出力**: 常に 1 を返します。

前提条件と不変条件

  • SQL_ASCII エンコーディングはシングルバイトエンコーディングであるため、文字長は常に 1 バイトとみなされます。

アルゴリズムまたは振る舞い

  • 入力引数 s の内容に関わらず、定数 1 を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/common/wchar.c 内の pg_wchar_table 配列初期化時に、PG_SQL_ASCII エンコーディング用の関数ポインタとして登録されています。
  • **呼び出し先**: 本関数は他の関数を呼び出しません。

関連データ構造

  • pg_wchar_table: 各エンコーディングの操作関数を保持するグローバルな関数テーブル。本関数はこのテーブルの一部として使用されます。

失敗モードと境界条件

  • 本関数は常に成功し、エラーを返しません。入力ポインタが NULL である場合の挙動については、本ソース excerpt からは判断できません。

保守担当者向けメモ

  • 本関数は SQL_ASCII の特性に基づいた単純な実装です。他のマルチバイトエンコーディング用関数（pg_utf8_mblen 等）とは異なり、引数の内容を検証するロジックは含まれていません。
