encrypt_and_write_elgamal

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/pgp-pubenc.c:132

目的

ElGamal公開鍵暗号方式を用いてセッション鍵を暗号化し、その結果をPGPパケットストリームへ書き出す。

シグネチャまたは形状

static int encrypt_and_write_elgamal(PGP_Context *ctx, PGP_PubKey *pk, PushFilter *pkt);

入力と出力

  • **入力**:
  • ctx: PGP処理のコンテキスト。
  • pk: 暗号化に使用するElGamal公開鍵構造体。
  • pkt: 暗号化データを出力するための PushFilter。
  • **出力**:
  • 成功時は 0 を返す。
  • 失敗時は負の整数（エラーコード）を返す。

前提条件と不変条件

  • pk->pub.elg.p が有効であり、そのバイト数から1を引いたサイズがパディング後のメッセージ長として使用される。
  • 処理の過程で生成される一時的なMPI（多倍長整数）および平文データは、関数終了時に必ず pgp_mpi_free によって解放される。

アルゴリズムまたは振る舞い

1. **メッセージ構築**: create_secmsg を呼び出し、セッション鍵とチェックサムを結合してパディングを施したMPI m を生成する。 2. **暗号化**: pgp_elgamal_encrypt を呼び出し、公開鍵 pk を用いて m を暗号化し、結果として2つのMPI c1 と c2 を得る。 3. **書き出し**: pgp_mpi_write を用いて、暗号化されたMPI c1 および c2 を順次 pkt ストリームへ書き出す。 4. **クリーンアップ**: 成功・失敗に関わらず、使用したMPIメモリを解放して終了する。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/pgcrypto/pgp-pubenc.c 内の公開鍵暗号セッション鍵パケット生成処理（pgp_write_pubenc_sesskey からの呼び出しが示唆されている）。
  • **呼び出し先**:
  • create_secmsg
  • pgp_elgamal_encrypt
  • pgp_mpi_write
  • pgp_mpi_free

関連データ構造

  • PGP_Context: PGP処理の全体状態を保持。
  • PGP_PubKey: 公開鍵情報（ElGamal鍵を含む）。
  • PushFilter: 出力ストリームの抽象化。
  • PGP_MPI: 多倍長整数（暗号化対象および結果）。

失敗モードと境界条件

  • create_secmsg または pgp_elgamal_encrypt が失敗した場合、エラーコードを返し、確保済みのMPIメモリを解放して終了する。
  • pgp_mpi_write による書き込みが失敗した場合も同様にエラーを伝播させる。

保守担当者向けメモ

  • 本関数はElGamal専用の暗号化フローを実装している。RSAの場合は encrypt_and_write_rsa が対照的な役割を果たす。
  • セキュリティ上の理由から、平文データを含むMPIは処理後に確実に解放される必要がある。現在の実装では goto err を用いた一元的な解放処理が行われている。
