r_factive

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_hungarian.c:660

目的

ハンガリー語のステミング処理において、特定の接尾辞（factive）を単語末尾から検出し、条件を満たす場合に削除および正規化を行う。

シグネチャまたは形状

static int r_factive(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z （ステミング実行環境）
  • 出力: 処理が成功した場合は 1、条件不一致やエラー発生時は 0 または負の値を返す。

前提条件と不変条件

  • z->c は現在の単語内のカーソル位置を指す。
  • a_7 配列に定義された接尾辞パターンとのマッチングを前提とする。
  • 処理対象の単語末尾が特定の文字コード（161 または 169）である必要がある。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を z->ket に設定する。 2. 単語末尾の文字が 161 または 169 であるかを確認し、条件を満たさない場合は終了する。 3. find_among_b を使用し、a_7 配列内の接尾辞と後方マッチングを行う。 4. マッチした場合、z->bra を現在のカーソル位置に設定する。 5. r_R1 を呼び出し、対象領域が R1 領域内にあるか検証する。 6. r_double を呼び出し、二重子音の処理を行う。 7. slice_del を呼び出し、マッチした接尾辞を削除する。 8. r_undouble を呼び出し、削除後の単語に対して二重子音の復元処理を行う。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_hungarian.c:833 および src/backend/snowball/libstemmer/stem_ISO_8859_2_hungarian.c:827 からの参照が確認されている。
  • 呼び出し先: find_among_b, r_R1, r_double, slice_del, r_undouble。

関連データ構造

  • struct SN_env: ステミング実行環境を保持する構造体。
  • a_7: 接尾辞マッチングに使用される among 構造体の配列。

失敗モードと境界条件

  • z->c - 1 <= z->lb: 単語の境界を超えた場合は 0 を返す。
  • find_among_b が一致しない場合、0 を返す。
  • r_R1, r_double, slice_del, r_undouble の各関数が負の値を返した場合、その値をそのまま伝播させて終了する。
