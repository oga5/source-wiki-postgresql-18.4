BufFileCreateFileSet

定義

  • 種別: function
  • 定義箇所: src/backend/storage/file/buffile.c:266

目的

FileSet 内に新しいバッファ付きファイル（BufFile）を作成し、初期化する。

シグネチャまたは形状

BufFile *BufFileCreateFileSet(FileSet *fileset, const char *name)

入力と出力

  • **入力**:
  • fileset: ファイルを格納する対象の FileSet へのポインタ。
  • name: 作成するファイルの名前。
  • **出力**:
  • 初期化された BufFile 構造体へのポインタ。

前提条件と不変条件

  • makeBufFileCommon(1) により、バッファ管理に必要なメモリが割り当てられていること。
  • name は pstrdup により BufFile 構造体内にコピーされる。
  • readOnly フラグは false に設定される。

アルゴリズムまたは振る舞い

1. makeBufFileCommon(1) を呼び出し、基本的な BufFile 構造体を生成する。 2. 指定された fileset を BufFile 構造体に割り当てる。 3. name を pstrdup で複製して保持する。 4. palloc を使用してファイル記述子配列を確保し、MakeNewFileSetSegment を呼び出して最初の物理セグメントを作成・割り当てる。 5. 読み書き可能な状態として初期化し、作成した BufFile を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/replication/logical/worker.c
  • src/backend/utils/sort/logtape.c
  • src/backend/utils/sort/sharedtuplestore.c
  • **呼び出し先**:
  • makeBufFileCommon
  • pstrdup
  • palloc
  • MakeNewFileSetSegment

関連データ構造

  • BufFile: 本関数が生成する主要な構造体。
  • FileSet: ファイルの集合を管理する構造体。
  • File: 仮想ファイル記述子。

失敗モードと境界条件

  • メモリ不足が発生した場合、palloc または pstrdup がエラーを発生させる可能性がある。
  • MakeNewFileSetSegment が失敗した場合、ファイル作成は中断される。

保守担当者向けメモ

  • 本関数は FileSet を介した共有ファイル作成の基点となる。
  • BufFile は palloc で割り当てられるため、呼び出し元のメモリコンテキストの寿命に依存する点に注意すること。
