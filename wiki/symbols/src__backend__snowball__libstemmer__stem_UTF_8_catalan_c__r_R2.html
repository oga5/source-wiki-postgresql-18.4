r_R2

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_catalan.c:1277

目的

単語内の現在のカーソル位置が、定義済みの「R2」領域内にあるかどうかを判定する。

シグネチャまたは形状

static int r_R2(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（Snowballステミング環境構造体）
  • 出力: int 型。現在のカーソル位置 z->c が z->I[0]（R2領域の開始位置）以上であれば 1 を返し、そうでなければ 0 を返す。

前提条件と不変条件

  • z->I[0] には、事前に r_mark_regions 関数によって計算された R2 領域の開始インデックスが格納されている必要がある。
  • z->c は現在の処理対象の文字位置を示す。

アルゴリズムまたは振る舞い

1. 環境構造体 z 内の整数配列 I のインデックス 0 に保持された R2 領域の開始位置を取得する。 2. 現在のカーソル位置 z->c と比較する。 3. z->I[0] <= z->c が真であれば、現在の位置は R2 領域内（または境界上）にあると判断し、成功（1）を返す。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_ISO_8859_1_basque.c、stem_ISO_8859_1_catalan.c、stem_ISO_8859_1_dutch.c 等の各言語ステマー実装から、接尾辞削除の条件判定として呼び出される。
  • 呼び出し先: なし（算術比較のみを実行）。

関連データ構造

  • SN_env: Snowballランタイム環境。I 配列（整数領域）および c（カーソル位置）を保持する。

失敗モードと境界条件

  • z が NULL の場合、プログラムはクラッシュする可能性がある（呼び出し元でのガードに依存）。
  • r_mark_regions が未実行の場合、z->I[0] の値は不定であり、判定結果は信頼できない。

保守担当者向けメモ

  • 本関数はSnowballコンパイラによって自動生成されたコードである。直接編集せず、元のSnowballスクリプト（.sbl）を修正すること。
  • I[0] の意味（R1かR2か）は各言語のスクリプト定義に依存するため、他言語への移植時にはインデックスの整合性に注意が必要である。
