ismn_cast_from_ean13

定義

  • 種別: function
  • 定義箇所: contrib/isn/isn.c:1070

目的

ean13 型の値を ISMN 型（内部的には ean13 型）へキャストするための PostgreSQL 関数です。

シグネチャまたは形状

Datum ismn_cast_from_ean13(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS を介して渡される ean13 型の値。
  • **出力**: Datum 型として返される、変換後の ean13（ISMN）値。

前提条件と不変条件

  • 入力値は ean13 型として正しく初期化されている必要があります。
  • 内部的に ean2isn 関数を呼び出し、変換処理を行います。

アルゴリズムまたは振る舞い

1. PG_GETARG_EAN13(0) を使用して、入力引数から ean13 値を取得します。 2. ean2isn 関数を呼び出し、第2引数に false（厳密な検証）、第4引数に ISMN 型を指定して変換を実行します。 3. 変換結果を PG_RETURN_EAN13 マクロを介して返します。

呼び出し元と呼び出し先

  • **呼び出し元**: PostgreSQL の関数マネージャ（PG_FUNCTION_INFO_V1 マクロにより登録）。
  • **呼び出し先**: ean2isn（contrib/isn/isn.c 内で定義されている変換関数）。

関連データ構造

  • ean13: uint64 に基づく内部ストレージ形式。
  • ISMN: ean2isn に渡される識別子タイプ。

失敗モードと境界条件

  • ean2isn 内部で変換が失敗した場合の挙動は、当該関数の実装に依存します。本関数自体はエラーハンドリングを明示的に記述していません。

保守担当者向けメモ

  • 本関数は PostgreSQL の型キャストシステムから呼び出されることを想定しています。
  • ean2isn の第2引数に false を渡しているため、変換時にチェックデジットの検証が厳密に行われます。
