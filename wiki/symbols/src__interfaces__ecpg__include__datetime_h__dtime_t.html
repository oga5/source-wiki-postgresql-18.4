dtime_t

定義

  • 種別: typedef
  • 定義箇所: src/interfaces/ecpg/include/datetime.h:10

目的

ECPG（Embedded SQL for C）において、Informix 互換の日時データを保持するための型定義を提供します。

シグネチャまたは形状

typedef timestamp dtime_t;

入力と出力

本シンボルは型定義であり、関数ではないため入出力は持ちません。timestamp 型の別名として機能します。

前提条件と不変条件

  • _ECPGLIB_H が定義されている場合、再定義を防ぐために本定義はスキップされます。
  • pgtypes_timestamp.h で定義される timestamp 型に依存しています。

アルゴリズムまたは振る舞い

本シンボル自体は宣言のみであり、実行時の振る舞いは持ちません。ECPG プリプロセッサが SQL 文を C コードに変換する際、日時変数の型として使用されます。

呼び出し元と呼び出し先

  • 呼び出し元: ECPG プリプロセッサが生成する C コード。
  • 呼び出し先: なし（型定義のため）。
  • 注記: 具体的な関数呼び出し関係は、本ソース抜粋からは特定できません。

関連データ構造

  • timestamp: 本型の基盤となる型。
  • intrvl_t: 同様に定義される期間データ用の型。

失敗モードと境界条件

  • 型定義のため、実行時の失敗モードはありません。
  • コンパイル時に ecpglib との重複定義が発生した場合、ガードマクロにより回避されます。

保守担当者向けメモ

  • Informix 互換モードでのコンパイル時に使用されます。
  • 型に対する操作が必要な場合は、src/interfaces/ecpg/pgtypeslib/datetime.c に実装されている PGTYPESdate_ 等の関数群を参照してください。
