目的

フィンランド語のステミング処理において、単語末尾の特定の母音パターン（'i' で終わり、その前が特定の母音グループ g_V2 に属する）を判定・削除する。

シグネチャまたは形状

static int r_VI(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z（Snowball環境構造体）
  • **出力**: 成功時に 1、失敗時に 0 を返す。

前提条件と不変条件

  • z->c（現在のカーソル位置）が z->lb（境界）より大きいこと。
  • 文字列の末尾（z->p[z->c - 1]）が 'i' であること。
  • 'i' の直前の文字が g_V2 グループに含まれること。

アルゴリズムまたは振る舞い

1. カーソル位置が境界内にあるか、かつ末尾文字が 'i' であるかを確認する。条件を満たさない場合は 0 を返す。 2. カーソルを1つ戻す（z->c--）。 3. in_grouping_b を使用し、現在のカーソル位置の文字が g_V2 グループに含まれるかを確認する。 4. 条件を満たせば 1 を返し、そうでなければ 0 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_ISO_8859_1_finnish.c 内の among 構造体配列（s_6_13, s_6_16, s_6_18）から参照されている。
  • **呼び出し先**: in_grouping_b（Snowballランタイム関数）。

関連データ構造

  • struct SN_env: ステミング処理の環境状態を保持する構造体。
  • g_V2: r_VI が判定に使用する文字グループ。

失敗モードと境界条件

  • カーソルが境界（z->lb）に達している場合、または末尾が 'i' でない場合は直ちに 0 を返す。
  • 'i' の直前の文字が g_V2 グループに該当しない場合、0 を返す。
