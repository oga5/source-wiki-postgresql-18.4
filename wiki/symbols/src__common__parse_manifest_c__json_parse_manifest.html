json_parse_manifest

定義

  • 種別: function
  • 定義箇所: src/common/parse_manifest.c:226

目的

PostgreSQLのバックアップマニフェスト（JSON形式）を一度に読み込み、解析および整合性検証を行うメイン関数です。

シグネチャまたは形状

void json_parse_manifest(JsonManifestParseContext *context, const char *buffer, size_t size);

入力と出力

  • **入力**:
  • context: 解析結果を受け取るためのコールバック関数群を保持する構造体。
  • buffer: マニフェスト内容が格納された文字列バッファ。
  • size: バッファのサイズ。
  • **出力**: なし（結果は context 経由で通知される）。

前提条件と不変条件

  • buffer は有効なJSON形式である必要があります。
  • 解析終了時にマニフェストの内部状態が JM_EXPECT_EOF に到達している必要があります。
  • マニフェスト末尾には、全内容を検証するためのSHA-256チェックサムが含まれている必要があります。

アルゴリズムまたは振る舞い

1. **初期化**: JsonManifestParseState を初期化し、解析状態を JM_EXPECT_TOPLEVEL_START に設定します。 2. **JSON解析**: makeJsonLexContextCstringLen でレキサーを作成し、pg_parse_json を用いてJSONを走査します。解析中、オブジェクトやスカラ値の開始・終了に応じて定義されたコールバック（json_manifest_object_start 等）を呼び出します。 3. **エラーハンドリング**: JSON解析エラーが発生した場合は json_manifest_parse_failure を呼び出し、処理を中断します。 4. **状態検証**: 解析完了後、期待される終了状態（JM_EXPECT_EOF）でない場合はエラーとして処理します。 5. **整合性検証**: verify_manifest_checksum を呼び出し、マニフェスト内のチェックサムと実際のデータ内容を照合します。 6. **終了処理**: freeJsonLexContext を呼び出し、レキサーのリソースを解放します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/bin/pg_combinebackup/load_manifest.c
  • src/bin/pg_verifybackup/pg_verifybackup.c
  • **呼び出し先**:
  • makeJsonLexContextCstringLen
  • pg_parse_json
  • json_manifest_parse_failure
  • verify_manifest_checksum
  • freeJsonLexContext
  • 各種コールバック関数（json_manifest_object_start 等）

関連データ構造

  • JsonManifestParseContext: 呼び出し元が提供するコールバックインターフェース。
  • JsonManifestParseState: 解析中の内部状態を保持する構造体。
  • JsonLexContext: JSON解析用のレキサー状態。
  • JsonSemAction: JSON解析時のセマンティックアクション定義。

失敗モードと境界条件

  • **JSON形式不正**: pg_parse_json がエラーを返し、json_manifest_parse_failure が実行されます。
  • **構造不整合**: マニフェストが期待される形式（トップレベルの開始や終了など）に従っていない場合、エラーとなります。
  • **チェックサム不一致**: verify_manifest_checksum 内で検証に失敗した場合、エラーとなります。
  • **復帰不可**: json_manifest_parse_failure は pg_unreachable() を呼び出すため、エラー発生時は呼び出し元へ復帰しません。

保守担当者向けメモ

  • 本関数はフロントエンド（postgres_fe.h）向けに設計されています。
  • 大規模なマニフェストを扱う場合は、本関数ではなくインクリメンタルAPI（json_parse_manifest_incremental_init 等）の使用を検討してください。
