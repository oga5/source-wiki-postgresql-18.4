GetXmlTableBuilderPrivateData

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/xml.c:4656

目的

TableFuncScanState の不透明なデータ領域（opaque）から、XMLTABLE 処理に必要な内部コンテキスト構造体（XmlTableBuilderData）を取得し、検証を行う。

シグネチャまたは形状

static inline XmlTableBuilderData *GetXmlTableBuilderPrivateData(TableFuncScanState *state, const char *fname)

入力と出力

  • **入力**:
  • state: TableFuncScanState 構造体へのポインタ。
  • fname: 呼び出し元の関数名（エラーメッセージ用）。
  • **出力**:
  • XmlTableBuilderData *: 取得した内部データ構造体へのポインタ。

前提条件と不変条件

  • state は TableFuncScanState 型である必要がある。
  • state->opaque に格納されているデータは、XMLTABLE_CONTEXT_MAGIC 値を持つ有効な XmlTableBuilderData 構造体でなければならない。

アルゴリズムまたは振る舞い

1. IsA(state, TableFuncScanState) を用いて、入力された state が正しい型であるかを確認する。型が一致しない場合は elog(ERROR) を発生させる。 2. state->opaque を XmlTableBuilderData * にキャストして取得する。 3. 取得した構造体の magic フィールドが XMLTABLE_CONTEXT_MAGIC と一致するかを確認する。一致しない場合は、呼び出し元の関数名 fname を含むエラーメッセージと共に elog(ERROR) を発生させる。 4. 検証を通過した場合、構造体へのポインタを返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • XmlTableSetDocument
  • XmlTableSetNamespace
  • XmlTableSetRowFilter
  • XmlTableSetColumnFilter
  • XmlTableFetchRow
  • XmlTableGetValue
  • XmlTableDestroyOpaque
  • **呼び出し先**:
  • IsA マクロ（型チェック）
  • elog（エラー報告）

関連データ構造

  • TableFuncScanState: 実行ノードの状態を保持する構造体。
  • XmlTableBuilderData: XMLTABLE 演算子の実行に必要なプライベートデータを保持する構造体。

失敗モードと境界条件

  • state が TableFuncScanState ではない場合、または opaque データが破損（magic 値の不一致）している場合、elog(ERROR) によりトランザクションが中断される。

保守担当者向けメモ

  • 本関数は static inline で定義されており、XMLTABLE 関連の各関数における共通の型安全チェックとして機能している。
  • magic 値の検証は、メモリ上のデータ構造が期待される状態にあることを保証するための重要なガードレールである。
