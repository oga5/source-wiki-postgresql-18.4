r_standard_suffix

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_dutch.c:389

目的

オランダ語のステミング処理において、定義されたルールに基づき単語の接尾辞を順次除去する。

シグネチャまたは形状

static int r_standard_suffix(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング環境構造体。カーソル位置や領域情報を含む）
  • 出力: int（成功時は 0 以外、失敗時は 0、エラー発生時は負の値を返す）

前提条件と不変条件

  • SN_env 構造体が初期化され、r_mark_regions によって R1, R2 領域が計算済みであること。
  • 処理対象の単語は ISO-8859-1 エンコーディングであること。

アルゴリズムまたは振る舞い

1. **接尾辞の検索と除去**: find_among_b を使用して後方から接尾辞を検索し、一致したルールに応じて以下の処理を行う。

  • R1 領域内での接尾辞置換や削除。
  • r_en_ending や r_e_ending を呼び出し、特定の語尾パターンを処理。
  • r_R1 や r_R2 を用いて、接尾辞除去が有効な領域内であることを検証。

    • 2. **条件付き削除**: 特定の文字（'c' や 'e' など）が直前に存在する場合の例外処理や、r_undouble による重複文字の解消を行う。 3. **状態の復元**: 各ステップで m1 から m9 までのマーカーを使用し、マッチング失敗時にカーソル位置をバックトラックする。

呼び出し元と呼び出し先

  • 呼び出し元: dutch_ISO_8859_1_stem（直接の呼び出し関係はソースコードの制御フローから推論可能）。
  • 呼び出し先: r_R1, r_R2, r_en_ending, r_e_ending, r_undouble, find_among_b, slice_from_s, slice_del, out_grouping_b, eq_s_b。

関連データ構造

  • struct SN_env: ステミングの実行状態を保持する構造体。
  • a_3, a_4, a_5: 接尾辞検索用の among 構造体配列。
  • g_v, g_v_j, g_v_I: 母音や特定の文字グループを定義するビットマップ。

失敗モードと境界条件

  • z->c（カーソル）が z->lb（下限）以下になった場合、または領域条件（R1, R2）を満たさない場合は処理を中断し、バックトラックする。
  • slice_from_s や slice_del 等のランタイム関数が負の値を返した場合、エラーとして伝播する。
