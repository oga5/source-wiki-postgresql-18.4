r_noun_sfx

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_irish.c:334

目的

アイルランド語のステミング処理において、単語末尾の名詞接尾辞を特定し、条件を満たす場合に削除を行う。

シグネチャまたは形状

static int r_noun_sfx(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング処理の実行環境状態）
  • 出力: int（処理が適用された場合は 1、適用されなかった場合は 0、エラー発生時は負の値を返す）

前提条件と不変条件

  • z->c が単語の末尾を指していること。
  • a_1 に定義された16個の接尾辞パターンに対してマッチングを行う。
  • 削除操作は、r_R1 または r_R2 によって定義される領域内でのみ許可される。

アルゴリズムまたは振る舞い

1. z->ket を現在のカーソル位置 z->c に設定する。 2. find_among_b を使用し、a_1 テーブルに基づき単語末尾から接尾辞を検索する。 3. マッチした接尾辞の種類（among_var）に応じて分岐する。

  • case 1: r_R1 を呼び出し、領域内であれば slice_del で削除する。
  • case 2: r_R2 を呼び出し、領域内であれば slice_del で削除する。

    • 4. 削除成功時に z->bra を更新し、処理結果を返す。

呼び出し元と呼び出し先

  • 呼び出し元: irish_UTF_8_stem（src/backend/snowball/libstemmer/stem_UTF_8_irish.c:447 等）
  • 呼び出し先: find_among_b, r_R1, r_R2, slice_del

関連データ構造

  • SN_env: ステミング処理の状態を保持する構造体。
  • a_1: 削除対象となる名詞接尾辞の定義テーブル。

失敗モードと境界条件

  • 接尾辞がマッチしない場合、0 を返し何も変更しない。
  • r_R1 または r_R2 のチェックで領域外と判定された場合、削除は行われず、そのチェック関数の戻り値が返される。
  • slice_del が失敗した場合（メモリ不足等）、負の戻り値が伝播される。
