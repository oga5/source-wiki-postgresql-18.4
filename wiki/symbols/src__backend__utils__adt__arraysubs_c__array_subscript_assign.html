array_subscript_assign

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/arraysubs.c:294

目的

配列型に対する要素代入操作を実行し、結果の Datum を更新する。

シグネチャまたは形状

static void array_subscript_assign(ExprState *state, ExprEvalStep *op, ExprContext *econtext)

入力と出力

  • **入力**:
  • op->resvalue: 代入対象の配列（Datum）。
  • op->resnull: 代入対象が NULL かどうかを示すフラグ。
  • sbsrefstate: 添字情報や代入値を含む SubscriptingRefState。
  • workspace: 型情報や境界値を含む ArraySubWorkspace。
  • **出力**:
  • op->resvalue: 更新後の配列（Datum）。
  • op->resnull: 更新後の NULL 状態（代入後は常に false）。

前提条件と不変条件

  • workspace は sbsrefstate->workspace から取得される。
  • 固定長配列型の場合、対象または代入値が NULL であれば代入を行わず終了する。
  • 可変長配列（varlena）の場合、対象が NULL であれば空の配列として初期化してから代入を行う。

アルゴリズムまたは振る舞い

1. **固定長配列のチェック**: refattrlength > 0 の場合、対象配列または代入値が NULL ならば処理を中断する。 2. **NULL 配列の初期化**: 対象配列が NULL の場合、construct_empty_array を用いて空の配列を生成し、arraySource とする。 3. **要素代入**: array_set_element を呼び出し、指定された添字位置に値を代入する。 4. **結果の格納**: 戻り値は常に非 NULL であるため、*op->resvalue を更新し、*op->resnull は変更しない。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/arraysubs.c:526（methods->sbs_assign への代入として参照）。
  • **呼び出し先**:
  • construct_empty_array
  • array_set_element

関連データ構造

  • ExprState, ExprEvalStep, ExprContext: 式評価用構造体。
  • SubscriptingRefState: 添字操作の実行時状態。
  • ArraySubWorkspace: 配列の型情報や次元情報を保持するワークスペース。

失敗モードと境界条件

  • 固定長配列において NULL が入力された場合、代入は無視される。
  • 可変長配列において NULL が入力された場合、空配列として扱われ、代入結果は単一要素の配列となる。

保守担当者向けメモ

  • 本関数は SubscriptingRef ノードの実行ステップとして呼び出される。
  • array_set_element に渡す引数は ArraySubWorkspace から提供される型情報（長さ、アライメント等）に依存するため、ワークスペースの整合性が重要である。
