vector8_has_zero

定義

  • 種別: function
  • 定義箇所: src/include/port/simd.h:194

目的

Vector8 型のベクトル内に 0 が含まれているかを判定します。

シグネチャまたは形状

static inline bool vector8_has_zero(const Vector8 v)

入力と出力

  • **入力**: v (Vector8) - 検査対象のベクトル。
  • **出力**: bool - ベクトル内に 0 が含まれていれば true、そうでなければ false を返します。

前提条件と不変条件

  • Vector8 型はプラットフォーム依存の SIMD 型（またはフォールバック用の整数型）です。
  • SIMD 非対応環境（USE_NO_SIMD 定義時）では、循環定義を避けるため vector8_has_le を使用します。

アルゴリズムまたは振る舞い

1. コンパイル時の設定（USE_NO_SIMD の有無）に基づき、以下のいずれかの処理を実行します。

  • **SIMD 非対応時**: vector8_has_le(v, 0) を呼び出し、ビット演算による判定を行います。
  • **SIMD 対応時**: vector8_has(v, 0) を呼び出し、ハードウェア固有の SIMD 命令を用いて判定します。

呼び出し元と呼び出し先

  • **呼び出し先**:
  • vector8_has_le (SIMD 非対応時)
  • vector8_has (SIMD 対応時)
  • **呼び出し元**:
  • src/include/port/simd.h 内の他のインライン関数（vector8_broadcast や vector8_ssub の結果に対する判定など）から呼び出されています。

関連データ構造

  • Vector8: ベクトル演算用の抽象型。

失敗モードと境界条件

  • 特になし。入力された Vector8 型の全要素に対して 0 の有無を網羅的に検査します。

保守担当者向けメモ

  • 本関数は vector8_has の特殊ケースです。SIMD 非対応環境での実装において vector8_has を直接呼び出すと循環定義が発生するため、必ず vector8_has_le を経由するように設計されています。
