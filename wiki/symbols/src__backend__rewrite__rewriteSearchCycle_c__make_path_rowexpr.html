make_path_rowexpr

定義

  • 種別: function
  • 定義箇所: src/backend/rewrite/rewriteSearchCycle.c:116

目的

再帰的CTEの SEARCH または CYCLE 句で使用されるパス記録用の基盤として、指定された列リストに基づいた RowExpr ノードを構築します。

シグネチャまたは形状

RowExpr *make_path_rowexpr(const CommonTableExpr *cte, const List *col_list)

入力と出力

  • **入力**:
  • cte: 対象となる CommonTableExpr 構造体。
  • col_list: パスに含める列名のリスト。
  • **出力**:
  • 構築された RowExpr 構造体。row_typeid は RECORDOID に設定されます。

前提条件と不変条件

  • col_list に含まれる各列名は、cte->ctecolnames に存在する必要があります。
  • 戻り値の RowExpr は、RECORDOID 型の行として初期化されます。

アルゴリズムまたは振る舞い

1. makeNode(RowExpr) でノードを生成し、row_typeid を RECORDOID に、row_format を COERCE_IMPLICIT_CAST に設定します。 2. col_list を走査し、各列名と一致する列を cte->ctecolnames から検索します。 3. 一致した列について、cte のメタデータ（型、型修飾子、照合順序）を使用して Var ノードを生成します。 4. 生成した Var ノードを rowexpr->args に、列名を rowexpr->colnames に追加します。 5. 構築した RowExpr を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/rewrite/rewriteSearchCycle.c:319 (search_col_list の処理)
  • src/backend/rewrite/rewriteSearchCycle.c:343 (cycle_col_list の処理)
  • **呼び出し先**:
  • makeNode (マクロ)
  • list_nth, list_length, lfirst, lappend (リスト操作関数)
  • makeVar (ノード生成関数)
  • makeString (ノード生成関数)
  • strcmp (標準ライブラリ)

関連データ構造

  • CommonTableExpr: CTE の定義情報を保持。
  • RowExpr: 行式を表すノード。
  • Var: クエリ内の変数を表すノード。

失敗モードと境界条件

  • col_list 内の列名が cte->ctecolnames に存在しない場合、その列は無視されます（エラーは発生しません）。

保守担当者向けメモ

  • 本関数は RECORDOID を使用して行式を構築するため、呼び出し側でこの行式を扱う際は、PostgreSQL のレコード型に関する制約に従う必要があります。
