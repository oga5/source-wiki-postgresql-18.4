DatumGetChar

定義

  • 種別: function
  • 定義箇所: src/include/postgres.h:116

目的

PostgreSQL の汎用データ型である Datum から、C 言語の char 型へ値を変換する。

シグネチャまたは形状

static inline char DatumGetChar(Datum X)

入力と出力

  • **入力**: Datum X（変換対象のデータ）
  • **出力**: char（Datum からキャストされた値）

前提条件と不変条件

  • Datum 型は sizeof(void *) と同じサイズであると仮定されている。
  • 本関数は static inline で定義されており、関数呼び出しのオーバーヘッドを最小化する。

アルゴリズムまたは振る舞い

受け取った Datum 型の引数 X を、明示的に char 型へキャストして返却する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/commands/policy.c
  • src/backend/replication/logical/tablesync.c
  • src/backend/rewrite/rewriteDefine.c
  • src/backend/utils/adt/datum.c
  • src/backend/utils/adt/ruleutils.c
  • src/backend/utils/adt/selfuncs.c
  • src/backend/utils/adt/tsvector_op.c
  • src/backend/utils/cache/catcache.c
  • **呼び出し先**: なし（C 言語のキャスト演算のみを実行）

関連データ構造

  • Datum: PostgreSQL の実行エンジン内でデータを保持する汎用型。

失敗モードと境界条件

  • Datum に格納されている値が char 型の範囲を超えている場合、キャストによる切り捨てが発生する可能性がある。

保守担当者向けメモ

  • 本関数は postgres.h で定義されており、バックエンドのソースコード全体で頻繁に使用される。変更時はバイナリ互換性に注意すること。
