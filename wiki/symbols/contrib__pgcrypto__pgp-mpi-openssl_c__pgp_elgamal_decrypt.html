pgp_elgamal_decrypt

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/pgp-mpi-openssl.c:163

目的

ElGamal 暗号方式を用いて、暗号化されたメッセージ（MPI形式）を秘密鍵を使用して復号する。

シグネチャまたは形状

int pgp_elgamal_decrypt(PGP_PubKey *pk, PGP_MPI *_c1, PGP_MPI *_c2, PGP_MPI **msg_p)

入力と出力

  • **入力**:
  • pk: 秘密鍵を含む PGP_PubKey 構造体。
  • _c1, _c2: 暗号化されたメッセージの構成要素（PGP_MPI）。
  • **出力**:
  • msg_p: 復号されたメッセージを格納する PGP_MPI へのポインタ。
  • 戻り値: 成功時に 0、失敗時に PXE_PGP_MATH_FAILED を返す。

前提条件と不変条件

  • pk->sec.elg.x（秘密鍵）および pk->pub.elg.p（素数パラメータ）が有効であること。
  • OpenSSL の BIGNUM 演算が成功すること。
  • 関数終了時には、内部で確保されたすべての BIGNUM および BN_CTX が BN_clear_free または BN_CTX_free により解放されること。

アルゴリズムまたは振る舞い

1. 入力された PGP_MPI を mpi_to_bn で OpenSSL の BIGNUM 形式に変換する。 2. $m = c_2 \cdot (c_1^x)^{-1} \pmod p$ の計算を実行する。

  • BN_mod_exp で $c_1^x \pmod p$ を算出。
  • BN_mod_inverse でその逆元を算出。
  • BN_mod_mul で $c_2$ との積を求め、平文 $m$ を得る。

    • 3. 計算結果を bn_to_mpi で PGP_MPI 形式に変換し、msg_p に格納する。 4. 処理の全過程において、エラー発生時は goto err を経由してリソースを解放する。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/pgcrypto/pgp-pubdec.c:115
  • **呼び出し先**:
  • mpi_to_bn / bn_to_mpi（内部ヘルパー）
  • OpenSSL BN_* 関数群（BN_new, BN_mod_exp, BN_mod_inverse, BN_mod_mul, BN_CTX_new 等）

関連データ構造

  • PGP_PubKey: 公開鍵および秘密鍵情報を保持する構造体。
  • PGP_MPI: PGP 形式の多倍長整数。
  • BIGNUM: OpenSSL の多倍長整数構造体。

失敗モードと境界条件

  • メモリ不足や OpenSSL の演算失敗時には PXE_PGP_MATH_FAILED を返す。
  • 内部ポインタのいずれかが初期化に失敗した場合、直ちにエラー処理へ移行する。

保守担当者向けメモ

  • 本関数は GnuPG の数学的処理を模倣している。
  • BIGNUM のメモリリークを防ぐため、エラー発生時を含めた厳格なクリーンアップ処理が実装されている。変更を加える際は goto err パターンの維持に注意すること。
