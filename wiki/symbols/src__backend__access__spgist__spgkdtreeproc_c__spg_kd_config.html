spg_kd_config

定義

  • 種別: function
  • 定義箇所: src/backend/access/spgist/spgkdtreeproc.c:27

目的

SP-GiSTインデックスにおけるk-d木実装の構成メタデータを設定します。

シグネチャまたは形状

Datum spg_kd_config(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS（spgConfigOut 構造体へのポインタが第2引数として渡されることを想定）
  • **出力**: void（cfg 引数を通じて設定を書き込む）

前提条件と不変条件

  • PG_GETARG_POINTER(1) が有効な spgConfigOut 構造体を指している必要があります。

アルゴリズムまたは振る舞い

以下の設定を spgConfigOut 構造体に適用します。 1. prefixType: FLOAT8OID に設定（k-d木がプレフィックスとして浮動小数点数を使用するため）。 2. labelType: VOIDOID に設定（ノードラベルを必要としないため）。 3. canReturnData: true に設定。 4. longValuesOK: false に設定。

呼び出し元と呼び出し先

  • **呼び出し元**: 提供されたソース情報からは、本関数を直接呼び出す具体的な関数名は特定できません。
  • **呼び出し先**: なし。

関連データ構造

  • spgConfigOut: SP-GiSTインデックスの構成情報を保持する構造体。

失敗モードと境界条件

  • 引数が正しく渡されない場合、メモリ不正アクセスの可能性があります。

保守担当者向けメモ

  • 本関数はk-d木の特性を定義する重要な設定関数です。変更を加える際は、spg_kd_picksplit や spg_kd_inner_consistent との整合性に注意してください。
