PGP_S2K_TYPE

定義

  • 種別: enum
  • 定義箇所: contrib/pgcrypto/pgp.h:37

目的

OpenPGP (RFC 4880) 規格における、パスフレーズから暗号鍵を生成する手法（String-to-Key: S2K）の種類を識別するために使用されます。

シグネチャまたは形状

enum PGP_S2K_TYPE
{
    PGP_S2K_SIMPLE = 0,
    PGP_S2K_SALTED = 1,
    PGP_S2K_ISALTED = 3,
};

入力と出力

  • 入力: なし（列挙型定数）
  • 出力: なし（列挙型定数）

前提条件と不変条件

  • RFC 4880 に準拠した値が割り当てられています。
  • 値 2 は本実装では定義されていません。

アルゴリズムまたは振る舞い

  • PGP_S2K_SIMPLE: ソルトを使用せず、パスフレーズから直接鍵を生成します。
  • PGP_S2K_SALTED: パスフレーズにソルトを付加して鍵を生成します。
  • PGP_S2K_ISALTED: 反復回数（Iterated）とソルトを組み合わせて鍵を生成します。

呼び出し元と呼び出し先

  • 呼び出し元および呼び出し先の具体的な関数情報は、提供されたソース抜粋からは特定できません。

関連データ構造

  • PGP_S2K: S2K の設定パラメータを保持する構造体であり、この列挙型をメンバとして使用します。

失敗モードと境界条件

  • 定義されていない値（例: 2 や 4 以上）が設定された場合の挙動については、本ソース抜粋からは不明です。

保守担当者向けメモ

  • 本列挙型は RFC 4880 の S2K 仕様に基づいています。実装の詳細については contrib/pgcrypto/pgp-s2k.c を参照してください。
