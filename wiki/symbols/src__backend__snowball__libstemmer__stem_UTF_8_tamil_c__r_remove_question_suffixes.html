r_remove_question_suffixes

目的

タミル語の単語末尾から疑問形に関連する接尾辞を除去し、語幹を正規化する。

シグネチャまたは形状

static int r_remove_question_suffixes(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング環境構造体）
  • 出力: 成功時に 1、失敗時に負の値を返す。

前提条件と不変条件

  • r_has_min_length(z) が真（戻り値 > 0）であることを要求する。
  • 処理中、SN_env のカーソル位置（c）および境界（lb, l）が操作される。

アルゴリズムまたは振る舞い

1. r_has_min_length を呼び出し、最小長を満たさない場合は即座に終了する。 2. 検索範囲を単語全体に設定し、find_among_b を用いて a_14 配列に定義されたパターンを単語末尾から照合する。 3. マッチした場合、slice_from_s を使用して該当箇所を s_21 で定義された文字列に置換する。 4. 最後に r_fix_endings を呼び出し、接尾辞除去後の語幹の整合性を調整する。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_tamil.c:1369（文脈から推測されるメインのステミング処理フローの一部）
  • 呼び出し先: r_has_min_length, find_among_b, slice_from_s, r_fix_endings

関連データ構造

  • SN_env: ステミング処理の環境状態を保持する構造体。
  • a_14: 疑問形接尾辞のパターンを格納した among 構造体の配列。

失敗モードと境界条件

  • r_has_min_length が 0 以下の値を返した場合、処理を中断する。
  • slice_from_s が負の値を返した場合、エラーとして伝播させる。
  • find_among_b がマッチしない場合、置換処理はスキップされる。
