r_mark_lArI

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_turkish.c:676

目的

トルコ語のステミング処理において、特定の接尾辞（lArI 関連）が単語末尾に存在するかを判定し、マッチングを行う。

シグネチャまたは形状

static int r_mark_lArI(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z（Snowball ステミング環境へのポインタ）
  • **出力**: int 型。条件を満たせば 1 を返し、満たさなければ 0 を返す。

前提条件と不変条件

  • z->c（現在のカーソル位置）が 3 以上であること。
  • 単語の末尾（z->p[z->c - 1]）が特定の文字コード（105 または 177）であること。

アルゴリズムまたは振る舞い

1. カーソル位置が境界条件（z->lb）に対して十分な余裕があるか、および末尾文字が指定された値と一致するかを確認する。一致しない場合は 0 を返す。 2. find_among_b 関数を呼び出し、a_1 配列に定義されたパターンと末尾からマッチングを行う。 3. マッチングに成功すれば 1 を、失敗すれば 0 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_UTF_8_turkish.c 内の複数箇所（行 1248, 1314, 1398, 1485, 1577, 1746）。
  • **呼び出し先**: find_among_b（Snowball ランタイム関数）。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_1: 接尾辞パターンを格納した among 構造体の配列。

失敗モードと境界条件

  • カーソル位置が z->lb + 3 以下の場合、処理を中断し 0 を返す。
  • 末尾文字が 105 または 177 以外の場合、処理を中断し 0 を返す。
  • find_among_b によるパターンマッチングが失敗した場合、0 を返す。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードである。直接編集せず、元の .sbl ファイルを修正すること。
  • 105 および 177 というマジックナンバーは、トルコ語の文字エンコーディングに関連する。
