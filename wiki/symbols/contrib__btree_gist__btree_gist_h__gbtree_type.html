gbtree_type

定義

  • 種別: enum
  • 定義箇所: contrib/btree_gist/btree_gist.h:14

目的

btree_gist 拡張モジュールにおいて、GiST インデックスでサポートされる各データ型を識別するための列挙型です。各データ型固有の実装関数が、自身の型を識別し、インデックスキーの生成や比較処理を適切に分岐するために使用します。

シグネチャまたは形状

enum gbtree_type
{
    gbt_t_var, gbt_t_int2, gbt_t_int4, gbt_t_int8, gbt_t_float4,
    gbt_t_float8, gbt_t_numeric, gbt_t_ts, gbt_t_cash, gbt_t_oid,
    gbt_t_time, gbt_t_date, gbt_t_intv, gbt_t_macad, gbt_t_macad8,
    gbt_t_text, gbt_t_bpchar, gbt_t_bytea, gbt_t_bit, gbt_t_bool,
    gbt_t_inet, gbt_t_uuid, gbt_t_enum
};

入力と出力

  • 入力: なし（列挙定数の定義）
  • 出力: なし（列挙定数の定義）

前提条件と不変条件

  • 新しいデータ型を btree_gist に追加する場合、本列挙型にエントリを追加する必要があります。

アルゴリズムまたは振る舞い

各データ型実装ファイル（例: btree_int4.c）が本ヘッダーをインクルードし、この列挙値を用いて実行時の型判定を行います。これにより、共通のユーティリティ関数（btree_utils_num.h や btree_utils_var.h）内で、型に応じた処理の切り替えが可能となります。

呼び出し元と呼び出し先

  • 呼び出し元（参照元）:
  • contrib/btree_gist/btree_utils_num.h:38
  • contrib/btree_gist/btree_utils_var.h:28
  • 呼び出し先: なし（定義のみのため）

関連データ構造

  • btree_utils_num.h および btree_utils_var.h で定義されるユーティリティ関数群。

失敗モードと境界条件

  • 該当なし（列挙型定義のため）。

保守担当者向けメモ

  • 本シンボルは btree_gist における「型カタログ」として機能します。
  • BtreeGistNotEqualStrategyNumber とは独立した型識別子ですが、演算子変換の際には併用されることが一般的です。
