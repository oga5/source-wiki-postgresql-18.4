yyguts_t

定義

  • 種別: struct
  • 定義箇所: src/pl/plpgsql/src/pl_scanner.c:140

目的

Flex によって生成されるスキャナの状態管理構造体において、PL/pgSQL 特有の拡張データへのアクセスポイントを提供します。

シグネチャまたは形状

struct yyguts_t
{
    struct plpgsql_yy_extra_type *yyextra_r;
};

入力と出力

  • 入力: なし（構造体定義）
  • 出力: yyextra_r メンバを通じて plpgsql_yy_extra_type 構造体へのポインタを保持します。

前提条件と不変条件

  • yyscanner ポインタを本構造体にキャストして使用することが前提となっています。
  • yyextra_r は、スキャナの初期化時に適切な plpgsql_yy_extra_type インスタンスで初期化されている必要があります。

アルゴリズムまたは振る舞い

  • 本構造体は、Flex のスキャナ状態（yyscanner）から yyextra マクロを通じて PL/pgSQL のスキャナ状態（plpgsql_yy_extra_type）を取り出すためのブリッジとして機能します。

呼び出し元と呼び出し先

  • 呼び出し元: src/pl/plpgsql/src/pl_scanner.c 内の #define yyextra マクロが、yyscanner を struct yyguts_t * にキャストして yyextra_r メンバにアクセスしています。
  • 呼び出し先: なし（データ構造定義）。

関連データ構造

  • plpgsql_yy_extra_type: yyextra_r が指し示す、PL/pgSQL スキャナの状態を保持する主要な構造体。

失敗モードと境界条件

  • yyscanner が NULL であるか、本構造体型として正しく初期化されていない状態で yyextra マクロを介してアクセスした場合、未定義動作が発生します。

保守担当者向けメモ

  • 本構造体は Flex の内部実装に依存しています。Flex のバージョンアップや生成オプションの変更により、構造体のレイアウトが変更される可能性がある点に注意してください。
