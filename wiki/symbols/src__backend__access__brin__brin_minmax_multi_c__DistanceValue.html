DistanceValue

定義

  • 種別: struct
  • 定義箇所: src/backend/access/brin/brin_minmax_multi.c:248

目的

BRINインデックスの「Multi Min/Max」演算において、間隔（Interval）同士の距離を計算し、マージ対象を決定するためのソート用データ構造として使用されます。

シグネチャまたは形状

typedef struct DistanceValue
{
	int			index;
	double		value;
} DistanceValue;

入力と出力

  • index: 比較対象となる間隔のインデックス番号。
  • value: 計算された距離（double型）。

前提条件と不変条件

  • qsort 等でソートされる際、value フィールドに基づいて昇順または降順に並べ替えられることが想定されています。

アルゴリズムまたは振る舞い

  • 複数の間隔の距離を計算する際、各間隔のインデックスと計算値をペアとして保持します。
  • qsort を用いて distances 配列をソートし、最も近い（あるいは遠い）間隔を特定するために利用されます。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/access/brin/brin_minmax_multi.c 内の複数の関数（1307, 1343, 1370, 1477, 1614, 1796, 2752 行付近）で、距離計算やマージ処理の過程で動的に割り当て・参照されています。
  • **呼び出し先**: qsort（標準ライブラリ）がこの構造体の配列をソートするために使用します。

関連データ構造

  • ExpandedRange: マージ処理のために正規化された間隔表現。
  • Ranges: メモリ上で間隔を管理する主要構造体。

失敗モードと境界条件

  • palloc0 によるメモリ割り当て失敗時にエラーが発生する可能性があります。
  • ndistances が 0 の場合、配列の割り当てやソート処理で境界条件に注意が必要です。

保守担当者向けメモ

  • この構造体は一時的な計算用であり、永続化はされません。
  • value の計算ロジックは型ごとに異なる場合があるため、距離計算関数（compare_distances 等）の変更時には本構造体の利用箇所全体に影響が及ぶ可能性があります。
