rusage

定義

  • 種別: struct
  • 定義箇所: src/include/port/win32/sys/resource.h:12

目的

Windows環境において、POSIX標準の getrusage 関数が使用するリソース使用状況を保持するための構造体を提供します。

シグネチャまたは形状

struct rusage {
    struct timeval ru_utime; /* ユーザー時間 */
    struct timeval ru_stime; /* システム時間 */
};

入力と出力

本構造体はデータコンテナであり、関数ではありません。getrusage 関数を呼び出す際、呼び出し元が確保した本構造体へのポインタを渡すことで、実装側が計測結果を書き込みます。

前提条件と不変条件

  • sys/time.h がインクルードされており、struct timeval が定義されている必要があります。
  • Windows環境におけるエミュレーションであるため、POSIX準拠の全フィールドが網羅されているとは限りません。

アルゴリズムまたは振る舞い

本構造体自体にロジックは含まれません。getrusage 実装（src/port/win32getrusage.c 等）において、Windowsの GetProcessTimes API等から取得した値が、ru_utime および ru_stime に格納されます。

呼び出し元と呼び出し先

  • **呼び出し元（参照元）**:
  • src/backend/tcop/postgres.c
  • src/include/utils/pg_rusage.h
  • src/backend/commands/indexcmds.c
  • **呼び出し先**:
  • 本構造体はデータ型であり、関数ではないため呼び出し先はありません。

関連データ構造

  • struct timeval: ru_utime および ru_stime の型として使用されます。

失敗モードと境界条件

  • getrusage に渡されるポインタが NULL の場合、実装側で適切に処理される必要があります（src/port/win32getrusage.c では NULL チェックが行われています）。

保守担当者向けメモ

  • 本構造体はWindows環境での互換性維持を目的としています。POSIX環境との差異については、src/include/port/win32/sys/resource.h の読者向けメモを参照してください。
  • 実際の計測ロジックは src/port/win32getrusage.c に依存しているため、フィールドの解釈を変更する場合は当該実装を確認してください。
