r_main_suffix

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_norwegian.c:157

目的

ノルウェー語のステミング処理において、単語の末尾から主要な接尾辞を検索し、削除または置換を行う。

シグネチャまたは形状

static int r_main_suffix(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング環境の状態を保持する構造体）
  • 出力: int（処理が成功した場合は 1、失敗した場合は 0、エラー発生時は負の値を返す）

前提条件と不変条件

  • z->I[1] に格納された領域境界（Region）以降の文字列に対してのみ処理を行う。
  • 処理対象の単語バッファ z->p が有効であること。

アルゴリズムまたは振る舞い

1. **境界チェック**: 現在のカーソル位置 z->c が領域境界 z->I[1] より前であれば終了する。 2. **接尾辞検索**: a_0 構造体を用いて、単語末尾から前方に向かって find_among_b を実行し、一致する接尾辞を特定する。 3. **条件分岐**:

  • case 1: 一致した接尾辞を削除する。
  • case 2: 特定の文字（k）の直前が母音であるか等の条件を確認し、条件を満たせば接尾辞を削除する。
  • case 3: 一致した接尾辞を別の文字列（s_0）に置換する。

    • 4. **終了**: 処理結果を呼び出し元に返す。

呼び出し元と呼び出し先

  • 呼び出し元: norwegian_ISO_8859_1_stem（src/backend/snowball/libstemmer/stem_ISO_8859_1_norwegian.c 内）
  • 呼び出し先: find_among_b, slice_del, slice_from_s, in_grouping_b, out_grouping_b（いずれもSnowballランタイムユーティリティ）

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_0: 接尾辞検索用の among 構造体（トライ木的データ構造）。

失敗モードと境界条件

  • z->c <= z->lb の場合、または検索対象の文字が条件に合致しない場合は 0 を返して終了する。
  • slice_del や slice_from_s が内部でエラー（負の値）を返した場合、その値をそのまま伝播させる。

保守担当者向けメモ

  • 本関数はSnowballコンパイラによる自動生成コードであるため、直接編集せず、元のSnowballソース（.sbl）を修正すること。
  • a_0 は stem_ISO_8859_1_norwegian.c 内で定義されている。
