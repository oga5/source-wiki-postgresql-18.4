MacaddrPGetDatum

定義

  • 種別: function
  • 定義箇所: src/include/utils/inet.h:152

目的

macaddr 構造体へのポインタを、PostgreSQL の関数マネージャ（fmgr）が扱う Datum 型へ変換します。

シグネチャまたは形状

static inline Datum MacaddrPGetDatum(const macaddr *X)

入力と出力

  • **入力**: const macaddr *X （MACアドレス構造体へのポインタ）
  • **出力**: Datum （変換されたデータム値）

前提条件と不変条件

  • 入力ポインタ X は有効な macaddr 構造体を指している必要があります。
  • 本関数は PointerGetDatum をラップしたインライン関数であり、メモリレイアウトの変換は行いません。

アルゴリズムまたは振る舞い

1. PointerGetDatum(X) を呼び出し、ポインタ値を Datum 型にキャストして返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/btree_gin/btree_gin.c
  • contrib/btree_gist/btree_macaddr.c
  • src/include/utils/inet.h 内の PG_RETURN_MACADDR_P マクロ
  • **呼び出し先**:
  • PointerGetDatum (fmgr.h)

関連データ構造

  • macaddr: MACアドレスを保持する固定長構造体。

失敗モードと境界条件

  • 入力ポインタが NULL の場合、PointerGetDatum の挙動に依存します。

保守担当者向けメモ

  • 本関数は macaddr 型専用の変換インターフェースです。macaddr8 型には Macaddr8PGetDatum を使用してください。
