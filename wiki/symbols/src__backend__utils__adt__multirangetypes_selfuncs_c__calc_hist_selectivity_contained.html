calc_hist_selectivity_contained

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/multirangetypes_selfuncs.c:1130

目的

レンジ型またはマルチレンジ型において、クエリ条件が指定された範囲に「含まれる（contained by）」場合の選択率を、境界値ヒストグラムとレンジ長ヒストグラムを用いて推定します。

シグネチャまたは形状

static double calc_hist_selectivity_contained(
    TypeCacheEntry *typcache,
    const RangeBound *lower, RangeBound *upper,
    const RangeBound *hist_lower, int hist_nvalues,
    Datum *length_hist_values, int length_hist_nvalues
)

入力と出力

  • **入力**:
  • typcache: 型の比較関数等を含むキャッシュエントリ。
  • lower, upper: クエリ条件の範囲境界。
  • hist_lower: 境界値ヒストグラムの配列。
  • hist_nvalues: ヒストグラムの要素数。
  • length_hist_values: レンジ長ヒストグラムのデータ。
  • length_hist_nvalues: レンジ長ヒストグラムの要素数。
  • **出力**: 0.0 から 1.0 の範囲で推定された選択率（double）。

前提条件と不変条件

  • hist_lower は昇順にソートされている必要があります。
  • typcache は対象の型に対して有効である必要があります。
  • 統計情報の仮定として、レンジの長さと下限値は独立しているものとして計算されます。

アルゴリズムまたは振る舞い

1. **境界探索**: rbound_bsearch を使用し、クエリの upper 境界がヒストグラムのどのビンに該当するかを特定します。 2. **範囲外判定**: upper がヒストグラムの範囲外（下限未満）であれば 0.0 を返します。 3. **ビン幅計算**: get_position を用いて、ヒストグラムのビン内での相対位置を線形補間により算出します。 4. **累積計算**: ヒストグラムのビンを逆順（upper_index から 0 まで）に走査します。

  • 各ビンにおいて、get_distance でクエリ範囲との距離を求めます。
  • calc_length_hist_frac を呼び出し、その距離内に収まるレンジ長の割合を積分計算します。
  • 算出した割合とビン幅を乗算し、合計（sum_frac）に加算します。

    • 5. **終了条件**: クエリの lower 境界を含むビンに到達した時点でループを終了します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/multirangetypes_selfuncs.c 内の統計計算ロジック。
  • src/backend/utils/adt/rangetypes_selfuncs.c 内の統計計算ロジック。
  • **呼び出し先**:
  • rbound_bsearch
  • get_position
  • range_cmp_bounds
  • get_distance
  • calc_length_hist_frac

関連データ構造

  • RangeBound: 範囲の境界値を保持する構造体。
  • TypeCacheEntry: 型固有の比較関数やサブタイプ情報を保持する構造体。

失敗モードと境界条件

  • upper_index < 0: クエリの範囲がヒストグラムの全範囲より小さい場合、0.0 を返します。
  • upper_index >= hist_nvalues - 1: ヒストグラムの範囲を超える場合、最後のビンを対象として計算を継続します。
  • bin_width が負になる場合、0.0 にクランプされます。

保守担当者向けメモ

  • 本関数はレンジ型とマルチレンジ型の両方で共通利用されています。
  • 精度は ANALYZE によって生成されたヒストグラムの品質に強く依存します。推定値が不自然な場合は、統計情報の更新状況を確認してください。
  • subdiff 関数が適切に定義されていない型では、線形補間の精度が低下します。
