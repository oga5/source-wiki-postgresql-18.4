show_debug

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/pgp-pgsql.c:161

目的

PGP処理に関連するデバッグメッセージをPostgreSQLのログシステム経由で出力します。

シグネチャまたは形状

static void show_debug(const char *msg)

入力と出力

  • **入力**: msg (出力対象のデバッグメッセージ文字列)
  • **出力**: なし（戻り値は void）

前提条件と不変条件

  • ereport マクロが利用可能なPostgreSQLの実行環境下で呼び出される必要があります。

アルゴリズムまたは振る舞い

1. 受け取った文字列 msg を引数として ereport を呼び出します。 2. NOTICE レベルのログとして、dbg: というプレフィックスを付与してメッセージを出力します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/pgcrypto/pgp-pgsql.c 内のデバッグ処理箇所。
  • **呼び出し先**:
  • ereport (PostgreSQLコアのログ出力関数)
  • **登録**: px_set_debug_handler(show_debug) を通じて、px モジュールのデバッグハンドラとして登録されることが確認されています。

関連データ構造

  • 特になし。

失敗モードと境界条件

  • msg が NULL の場合、ereport の挙動に依存します。

保守担当者向けメモ

  • 本関数は px ライブラリのデバッグ出力をフックするために使用されます。
  • 運用環境でログが過剰に出力される可能性があるため、デバッグ目的以外での利用は避けてください。
