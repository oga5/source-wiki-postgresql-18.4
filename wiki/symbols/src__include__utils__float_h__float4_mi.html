float4_mi

定義

  • 種別: function
  • 定義箇所: src/include/utils/float.h:169

目的

2つの float4 型の値を減算し、結果に対してオーバーフローチェックを行うインライン関数です。

シグネチャまたは形状

static inline float4 float4_mi(const float4 val1, const float4 val2)

入力と出力

  • **入力**: val1 (被減数), val2 (減数)
  • **出力**: 減算結果 (val1 - val2) を返す。

前提条件と不変条件

  • 演算結果が無限大（inf）になった場合、入力値 val1 および val2 のいずれもが無限大ではない場合に限り、オーバーフローとみなされます。

アルゴリズムまたは振る舞い

1. val1 から val2 を減算し、結果を result に格納します。 2. result が無限大であり、かつ val1 と val2 が共に無限大ではない場合、float_overflow_error() を呼び出して処理を中断します。 3. 上記の条件に該当しない場合、計算結果を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/adt/float.c:742 (PG_RETURN_FLOAT4 マクロ内で使用)
  • **呼び出し先**: float_overflow_error() (エラー発生時)

関連データ構造

  • float4 (PostgreSQLの単精度浮動小数点型)

失敗モードと境界条件

  • **オーバーフロー**: 演算結果が inf になり、かつ入力値が有限である場合に float_overflow_error() が発生します。

保守担当者向けメモ

  • 本関数はインライン関数として定義されており、パフォーマンスを最適化しつつ、PostgreSQLの要件に基づいたエラーハンドリングを提供します。
  • 比較演算とは異なり、算術演算ではNaNの扱いは標準的なC言語の挙動に依存します。
