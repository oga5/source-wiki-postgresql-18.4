pg_cryptohash_update

定義

  • 種別: function
  • 定義箇所: src/common/cryptohash_openssl.c:229

目的

OpenSSLの EVP_DigestUpdate を利用して、ハッシュ計算対象のデータをコンテキストに逐次投入する。

シグネチャまたは形状

int pg_cryptohash_update(pg_cryptohash_ctx *ctx, const uint8 *data, size_t len)

入力と出力

  • **入力**:
  • ctx: ハッシュ計算の状態を保持する pg_cryptohash_ctx 構造体へのポインタ。
  • data: ハッシュ計算に追加するデータのバッファ。
  • len: data のバイト長。
  • **出力**:
  • 成功時は 0 を返す。
  • 失敗時は -1 を返す。

前提条件と不変条件

  • ctx は pg_cryptohash_create によって初期化済みであること。
  • ctx が NULL の場合、関数は即座に -1 を返す。

アルゴリズムまたは振る舞い

1. EVP_DigestUpdate を呼び出し、指定されたデータを現在のハッシュコンテキストに適用する。 2. OpenSSLの戻り値が 1（成功）以外の場合、以下のエラー処理を行う：

  • ERR_get_error() を用いてOpenSSLのエラーメッセージを取得し、ctx->errreason に格納する。
  • ctx->error に PG_CRYPTOHASH_ERROR_OPENSSL を設定する。
  • -1 を返して終了する。

    • 3. 成功時は 0 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/uuid-ossp/uuid-ossp.c
  • src/backend/backup/backup_manifest.c
  • src/backend/libpq/auth-scram.c
  • src/backend/utils/adt/cryptohashfuncs.c
  • src/common/checksum_helper.c
  • src/common/cryptohash.c
  • **呼び出し先**:
  • EVP_DigestUpdate (OpenSSL)
  • ERR_get_error (OpenSSL)
  • SSLerrmessage (本モジュール内)

関連データ構造

  • pg_cryptohash_ctx: ハッシュ計算の状態を保持する構造体。内部でOpenSSLの EVP_MD_CTX を管理する。

失敗モードと境界条件

  • ctx が NULL の場合、処理を行わず -1 を返す。
  • OpenSSL内部でエラーが発生した場合、ctx にエラー理由とコードが記録され、-1 が返される。

保守担当者向けメモ

  • 本関数はOpenSSLがリンクされている環境でのみ有効な実装である。
  • エラー発生時の SSLerrmessage はOpenSSLの内部エラーキューから情報を取得するため、呼び出し側でエラー状態を適切に確認する必要がある。
