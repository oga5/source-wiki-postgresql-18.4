mic2sjis

定義

  • 種別: function
  • 定義箇所: src/backend/utils/mb/conversion_procs/euc_jp_and_sjis/euc_jp_and_sjis.c:301

目的

MULE_INTERNAL（MIC）形式の文字列をSJIS形式に変換する。

シグネチャまたは形状

int mic2sjis(const unsigned char *mic, unsigned char *p, int len, bool noError)

入力と出力

  • **入力**:
  • mic: 変換元のMULE_INTERNAL文字列へのポインタ。
  • p: 変換後のSJIS文字列を格納するバッファへのポインタ。
  • len: 変換対象のバイト長。
  • noError: 変換エラー発生時に処理を中断するか（true）、エラーを報告するか（false）を指定するフラグ。
  • **出力**:
  • 戻り値: 処理したバイト数。
  • pが指すバッファには変換後のSJIS文字列が格納され、末尾にヌル終端が付与される。

前提条件と不変条件

  • micは有効なMULE_INTERNALエンコーディングのバイト列であること。
  • pは変換結果を格納するのに十分な領域を確保していること。

アルゴリズムまたは振る舞い

1. 入力文字列を1文字ずつ走査する。 2. ASCII文字（高ビットが立っていない場合）はそのままコピーする。 3. マルチバイト文字の場合、pg_encoding_verifymbcharで妥当性を検証する。 4. 文字種に応じて以下の変換を行う：

  • LC_JISX0201K: 1バイトをそのままコピー。
  • LC_JISX0208: 計算式に基づきSJISの2バイトコードへ変換（UDC1領域を含む）。
  • LC_JISX0212: ibmkanjiテーブルを参照して変換。一致しない場合はPGSJISALTCODEを使用。

    • 5. 変換不能な文字や不正なバイト列に遭遇した場合、noErrorがfalseであればエラー報告関数を呼び出す。

呼び出し元と呼び出し先

  • **呼び出し元**: 同一ファイル内の関数（mic_to_sjis等）から呼び出される。
  • **呼び出し先**: pg_encoding_verifymbchar, report_invalid_encoding, report_untranslatable_char。

関連データ構造

  • ibmkanji: sjis.mapで定義されたIBM漢字変換テーブル。
  • PGSJISALTCODE: マッピング未定義時の代替コード。

失敗モードと境界条件

  • lenが0になるまでループを継続する。
  • 不正なマルチバイトシーケンスや変換不能文字に対し、noErrorがfalseの場合はreport_invalid_encodingまたはreport_untranslatable_charを介してエラーを発生させる。
  • noErrorがtrueの場合は、エラー発生時点でループを中断し、それまでに変換したバイト数を返す。

保守担当者向けメモ

  • 本関数は内部的な変換ロジックであり、直接的な公開インターフェースではない。
  • 新しい文字コードマッピングを追加する場合は、ibmkanjiテーブルの更新が必要となる可能性がある。
