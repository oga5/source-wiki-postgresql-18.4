PLyObject_ToBool

定義

  • 種別: function
  • 定義箇所: src/pl/plpython/plpy_typeio.c:881

目的

PythonオブジェクトをPostgreSQLのboolean型（Datum）に変換します。

シグネチャまたは形状

static Datum PLyObject_ToBool(PLyObToDatum *arg, PyObject *plrv, bool *isnull, bool inarray)

入力と出力

  • **入力**:
  • arg: 変換メタデータを保持する構造体ポインタ。
  • plrv: 変換対象のPythonオブジェクト。
  • isnull: NULL状態を格納するフラグへのポインタ。
  • inarray: 配列内での変換かどうかを示すフラグ。
  • **出力**:
  • 戻り値: 変換後のDatum値。
  • isnull: plrvがPy_Noneの場合にtrue、それ以外はfalseが設定されます。

前提条件と不変条件

  • plrvがPy_Noneの場合、isnullはtrueとなり、戻り値は(Datum) 0となります。
  • plrvがPy_None以外の場合、isnullはfalseとなります。

アルゴリズムまたは振る舞い

1. plrvがPy_Noneであるかを確認し、該当する場合はNULLとして処理を終了します。 2. Py_Noneでない場合、PyObject_IsTrue(plrv)を呼び出し、Pythonの真偽値評価結果を取得します。 3. 取得した結果をBoolGetDatumでPostgreSQLのDatum形式に変換して返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/pl/plpython/plpy_typeio.c 内で、型変換関数ポインタとして登録されています（arg->func = PLyObject_ToBool;）。
  • **呼び出し先**: Python C APIの PyObject_IsTrue を呼び出します。

関連データ構造

  • PLyObToDatum: 変換ロジックと型情報を保持する構造体。
  • PyObject: Pythonのオブジェクト型。

失敗モードと境界条件

  • PyObject_IsTrue がPythonの例外を発生させた場合、呼び出し元で適切に例外処理を行う必要があります。

保守担当者向けメモ

  • 本関数は PLyObToDatum 構造体の func フィールドに代入されるコールバック関数として設計されています。
  • inarray 引数は現在の実装では使用されていません。
