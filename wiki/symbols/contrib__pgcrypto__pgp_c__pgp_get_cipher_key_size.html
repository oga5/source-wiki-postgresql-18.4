pgp_get_cipher_key_size

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/pgp.c:136

目的

指定された暗号アルゴリズムコードに対応する鍵長（バイト単位）を取得します。

シグネチャまたは形状

int pgp_get_cipher_key_size(int code);

入力と出力

  • **入力**: code (int) - 暗号アルゴリズムを識別する内部コード。
  • **出力**: (int) - アルゴリズムに対応する鍵の長さ。該当するアルゴリズムが見つからない場合は 0 を返します。

前提条件と不変条件

  • get_cipher_info(code) が有効な cipher_info 構造体へのポインタ、または NULL を返すことを前提としています。
  • 返される鍵長は、cipher_info 構造体内の key_len フィールドに依存します。

アルゴリズムまたは振る舞い

1. get_cipher_info(code) を呼び出し、指定されたコードに対応するアルゴリズム情報を検索します。 2. 検索結果が NULL でない場合、その構造体内の key_len を返します。 3. 検索結果が NULL の場合、0 を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/pgcrypto/pgp-decrypt.c: 復号時の鍵長検証に使用。
  • contrib/pgcrypto/pgp-encrypt.c: 暗号化コンテキストのセッション鍵長設定に使用。
  • contrib/pgcrypto/pgp-s2k.c: S2K（String-to-Key）処理での鍵長決定に使用。
  • **呼び出し先**:
  • get_cipher_info (同ファイル内)

関連データ構造

  • struct cipher_info: 暗号アルゴリズムのメタデータ（鍵長を含む）を保持する構造体。

失敗モードと境界条件

  • 未知の code が渡された場合、get_cipher_info が NULL を返し、本関数は 0 を返します。

保守担当者向けメモ

  • 本関数は pgp.h で宣言されており、外部モジュールから参照可能です。
  • 新しい暗号アルゴリズムを追加する際は、get_cipher_info が参照するリストを更新する必要があります。
