TableFuncType

定義

  • 種別: enum
  • 定義箇所: src/include/nodes/primnodes.h:98

目的

テーブル関数（Table-valued function）の具体的な種類を識別するために使用されます。

シグネチャまたは形状

typedef enum TableFuncType
{
    TFT_XMLTABLE,
    TFT_JSON_TABLE,
} TableFuncType;

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: TableFunc 構造体内の functype フィールドとして保持される。

前提条件と不変条件

  • TFT_XMLTABLE または TFT_JSON_TABLE のいずれか一方の値をとる。

アルゴリズムまたは振る舞い

クエリ処理において、特定のテーブル関数がどの構文（XMLTABLE または JSON_TABLE）に基づいているかを判別し、実行時の評価ロジックを分岐させるための識別子として機能します。

呼び出し元と呼び出し先

  • 呼び出し元: src/include/nodes/primnodes.h:114 にて TableFunc 構造体のメンバとして参照されている。
  • 呼び出し先: 提供されたソース情報からは、具体的な関数呼び出しや参照先は特定できません。

関連データ構造

  • TableFunc: この列挙型をメンバとして保持する構造体。

失敗モードと境界条件

  • 定義外の値が代入された場合の挙動は未定義です。

保守担当者向けメモ

  • 新しいテーブル関数型を追加する場合は、この列挙型に値を追加し、それを利用する実行エンジン側の switch 文や条件分岐を更新する必要があります。
