Range

定義

  • 種別: struct
  • 定義箇所: src/backend/utils/adt/geo_spgist.c:103

目的

空間インデックス（SP-GiST）において、1次元の範囲（下限値と上限値）を保持するために使用される構造体です。

シグネチャまたは形状

typedef struct
{
    float8      low;
    float8      high;
} Range;

入力と出力

  • low: 範囲の下限値（float8）。
  • high: 範囲の上限値（float8）。

前提条件と不変条件

  • 一般的に low <= high であることが期待されますが、構造体自体による強制は行われません。

アルゴリズムまたは振る舞い

  • 本構造体は、geo_spgist.c 内で定義される RangeBox 構造体の構成要素として使用されます。
  • 2次元の矩形（BOX）を4次元空間の点として扱う際、各次元（x軸、y軸）の範囲を表現するために利用されます。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/adt/geo_spgist.c 内の空間インデックス処理（getRangeBox 等）から直接参照されます。
  • **呼び出し先**: 本構造体はデータ保持用であり、関数を呼び出すことはありません。
  • **注意**: ツールが提示した contrib/ や src/backend/access/ 等の他のファイルにおける「Range」という名称の言及は、本構造体とは異なる文脈（BRINインデックスやSQLのRange型など）を指している可能性が高く、直接的な関係性は確認できません。

関連データ構造

  • RangeBox: Range を2つ（x軸用とy軸用）保持する構造体。
  • RectBox: SP-GiSTの探索範囲を管理する構造体。

失敗モードと境界条件

  • low と high に NaN や無限大が含まれる場合の挙動は、呼び出し側の演算ロジックに依存します。

保守担当者向けメモ

  • 本構造体は geo_spgist.c 専用の内部構造体です。汎用的な範囲型（RangeType）とは異なるため、混同しないよう注意してください。
