special_uuid_value

定義

  • 種別: function
  • 定義箇所: contrib/uuid-ossp/uuid-ossp.c:202

目的

OSSP UUID ライブラリが提供する定義済み UUID（nil や名前空間 UUID など）を読み込み、PostgreSQL の uuid 型の Datum に変換して返します。

シグネチャまたは形状

static Datum special_uuid_value(const char *name);

入力と出力

  • **入力**: name (const char *) - 読み込む UUID の名前（例: "nil", "ns:DNS"）。
  • **出力**: Datum - PostgreSQL 内部表現としての UUID。

前提条件と不変条件

  • OSSP UUID ライブラリがリンクされている環境でのみ動作します。
  • get_cached_uuid_t(0) が有効な uuid_t オブジェクトを返す必要があります。

アルゴリズムまたは振る舞い

1. get_cached_uuid_t(0) を呼び出し、セッションキャッシュから UUID オブジェクトを取得します。 2. uuid_load を使用して、指定された name に対応する UUID をライブラリからロードします。 3. ロードに失敗した場合は pguuid_complain を呼び出し、エラーを報告します。 4. uuid_to_string を使用して、ロードした UUID を文字列形式に変換します。 5. DirectFunctionCall1 を介して uuid_in 関数を呼び出し、文字列を PostgreSQL の uuid 型（Datum）に変換して返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/uuid-ossp/uuid-ossp.c 内の各 UUID 生成関数（"nil", "ns:DNS", "ns:URL", "ns:OID", "ns:X500" を引数として呼び出し）。
  • **呼び出し先**:
  • get_cached_uuid_t
  • uuid_load (OSSP UUID ライブラリ)
  • pguuid_complain
  • uuid_to_string
  • uuid_in (PostgreSQL 組み込み関数)

関連データ構造

  • uuid_t (OSSP UUID ライブラリの型)
  • uuid_rc_t (OSSP UUID ライブラリのリターンコード型)
  • Datum (PostgreSQL の汎用データ型)

失敗モードと境界条件

  • uuid_load が UUID_RC_OK 以外の値を返した場合、pguuid_complain によりエラーがスローされます。
  • 無効な name が渡された場合、ライブラリ側でロード失敗が発生します。

保守担当者向けメモ

  • 本関数は OSSP UUID ライブラリに強く依存しています。OSSP 以外の UUID 実装を使用するビルド構成では、この関数は使用されません。
  • uuid_in への変換には文字列を経由するオーバーヘッドがあるため、頻繁に呼び出されるパスでのパフォーマンスに注意してください。
