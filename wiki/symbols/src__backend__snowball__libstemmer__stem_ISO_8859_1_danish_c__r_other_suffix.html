r_other_suffix

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_danish.c:212

目的

デンマーク語のステミング処理において、特定の接尾辞パターンを検索し、除去または置換を行う。

シグネチャまたは形状

static int r_other_suffix(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング環境の状態を保持する構造体）
  • 出力: 成功時に 1、失敗または処理不要時に 0、エラー発生時に負の値を返す。

前提条件と不変条件

  • z->I[1] に格納された領域境界情報が有効であること。
  • SN_env 構造体のカーソル位置 z->c および境界 z->lb が適切に管理されていること。

アルゴリズムまたは振る舞い

1. **第一段階**: 文字列の後方から特定の接尾辞（s_0, s_1）を検索し、一致すれば slice_del で削除する。 2. **第二段階**: z->I[1] で定義される領域内において、find_among_b を用いて配列 a_2 に含まれる接尾辞を検索する。 3. **条件分岐**:

  • among_var == 1: 一致した接尾辞を削除し、その後 r_consonant_pair を適用する。
  • among_var == 2: 一致した接尾辞を s_2 で定義される文字列に置換する。

呼び出し元と呼び出し先

  • 呼び出し元: danish_ISO_8859_1_stem（stem_ISO_8859_1_danish.c 内）
  • 呼び出し先: eq_s_b, slice_del, find_among_b, r_consonant_pair, slice_from_s

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_2: 接尾辞マッチングに使用される among 構造体の配列。

失敗モードと境界条件

  • slice_del や slice_from_s が負の値を返した場合、その値をそのまま伝播させる。
  • 検索対象の領域が不十分な場合や、接尾辞がマッチしない場合は 0 を返す。

保守担当者向けメモ

  • 本関数はSnowballコンパイラによる自動生成コードであるため、直接編集せず、元のSnowballスクリプトを修正すること。
  • among_var の各ケースは、デンマーク語の文法規則に基づいた接尾辞処理に対応している。
