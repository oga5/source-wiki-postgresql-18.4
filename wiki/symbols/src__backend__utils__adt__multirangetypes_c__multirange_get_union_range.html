multirange_get_union_range

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/multirangetypes.c:803

目的

指定されたマルチレンジ（MultirangeType）全体を包含する最小の単一レンジ（RangeType）を構築して返します。

シグネチャまたは形状

RangeType *multirange_get_union_range(TypeCacheEntry *rangetyp, const MultirangeType *mr)

入力と出力

  • **入力**:
  • rangetyp: レンジ型のキャッシュエントリ。
  • mr: 対象のマルチレンジ構造体。
  • **出力**:
  • マルチレンジの全範囲をカバーする RangeType へのポインタ。

前提条件と不変条件

  • mr は正規化（ソート済みかつ非重複）されている必要があります。
  • rangetyp は有効な型キャッシュエントリである必要があります。

アルゴリズムまたは振る舞い

1. MultirangeIsEmpty(mr) をチェックし、空の場合は make_empty_range を呼び出して空のレンジを返します。 2. 空でない場合、multirange_get_bounds を使用して以下の境界を取得します。

  • 最初のレンジの「下限（lower bound）」
  • 最後のレンジの「上限（upper bound）」

    • 3. 取得した下限と上限を用いて make_range を呼び出し、単一のレンジを生成して返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/rangetypes_gist.c:257
  • **呼び出し先**:
  • MultirangeIsEmpty (マクロ)
  • make_empty_range
  • multirange_get_bounds
  • make_range

関連データ構造

  • MultirangeType: マルチレンジのメモリ上の表現。
  • RangeType: 単一レンジのメモリ上の表現。
  • TypeCacheEntry: レンジ型のメタデータキャッシュ。
  • RangeBound: レンジの境界値を保持する構造体。

失敗モードと境界条件

  • mr が空の場合、make_empty_range を通じて空のレンジが返されます。
  • mr が単一のレンジのみを含む場合、そのレンジ自体が返されます。

保守担当者向けメモ

  • 本関数はマルチレンジの全範囲を包含する「バウンディングボックス」を計算する際に利用されます。
  • multirange_get_bounds を使用してインデックス 0 と mr->rangeCount - 1 にアクセスしているため、mr が正規化されていない（レンジ数が0など）場合、予期せぬ動作をする可能性があります。
