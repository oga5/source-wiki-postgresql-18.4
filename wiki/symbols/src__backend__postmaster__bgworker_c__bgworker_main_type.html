bgworker_main_type

定義

  • 種別: typedef (関数ポインタ型)
  • 定義箇所: src/include/postmaster/bgworker.h:72

目的

バックグラウンドワーカープロセスが実行するメインエントリポイント関数の型を定義します。

シグネチャまたは形状

typedef void (*bgworker_main_type) (Datum main_arg);

入力と出力

  • **入力**: Datum main_arg - ワーカー起動時に渡される引数。
  • **出力**: なし（void）。

前提条件と不変条件

  • この型を持つ関数は、バックグラウンドワーカーのメインループとして機能し、プロセス終了まで制御を保持することが期待されます。

アルゴリズムまたは振る舞い

  • bgworker_main_type は、PostgreSQLのバックグラウンドワーカーが起動した際に呼び出される関数を指します。
  • src/backend/postmaster/bgworker.c 内の InternalBGWorkers 配列や、LookupBackgroundWorkerFunction を通じて動的にロードされた関数がこの型に適合する必要があります。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/postmaster/bgworker.c 内のワーカー起動ロジック（BackgroundWorkerMain 等）から呼び出されます。
  • **呼び出し先**: ユーザー定義のバックグラウンドワーカー関数（例: logicalworker.c 等の実装）。

関連データ構造

  • BackgroundWorker 構造体（ワーカーの登録情報）
  • InternalBGWorkers（内部ワーカーの静的リスト）

失敗モードと境界条件

  • 該当する関数がロードできない場合や、シグネチャが一致しない場合は、プロセス起動時にエラーが発生します。

保守担当者向けメモ

  • この型は src/include/postmaster/bgworker.h で定義されており、拡張機能開発者がバックグラウンドワーカーを実装する際のインターフェースとなります。変更時はABI互換性に注意してください。
