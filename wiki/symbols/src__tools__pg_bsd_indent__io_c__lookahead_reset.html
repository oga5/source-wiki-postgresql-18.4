lookahead_reset

定義

  • 種別: function
  • 定義箇所: src/tools/pg_bsd_indent/io.c:319

目的

入力ストリームの先読み状態を初期化し、再スキャンを開始可能な状態にリセットします。

シグネチャまたは形状

void lookahead_reset(void)

入力と出力

  • 入力: なし（グローバル変数 lookahead_start、bp_save、lookahead_ptr、lookahead_bp_save を参照）
  • 出力: なし（グローバル変数の更新による副作用のみ）

前提条件と不変条件

  • lookahead_start が有効なバッファの先頭を指している必要があります。
  • bp_save は、現在の処理状態を保存したポインタ、または NULL である必要があります。

アルゴリズムまたは振る舞い

1. lookahead_ptr を lookahead_start に設定し、メインの先読みバッファをリセットします。 2. bp_save が NULL でない場合、lookahead_bp_save に bp_save を代入し、保存されたバッファ位置からスキャンを再開できるようにします。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/tools/pg_bsd_indent/lexi.c:168
  • 呼び出し先:
  • 情報なし（直接的な関数呼び出しは含まれていません）

関連データ構造

  • lookahead_ptr, lookahead_start, lookahead_bp_save, bp_save（いずれも indent_globs.h で定義されたグローバル変数）

失敗モードと境界条件

  • bp_save が NULL の場合、lookahead_bp_save にも NULL が代入されます。

保守担当者向けメモ

  • 本関数は、字句解析器が入力ストリームの特定位置から再スキャンを行う際に使用されます。
  • indent_globs.h で定義されたグローバルなバッファポインタを直接操作するため、呼び出しタイミングには注意が必要です。
