r_noun

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_KOI8_R_russian.c:509

目的

KOI8-Rエンコーディングのロシア語テキストに対し、名詞に関連する接尾辞を単語末尾から検索し、該当する場合は削除して語幹を抽出する。

シグネチャまたは形状

static int r_noun(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（Snowballステミング環境構造体）
  • **出力**: int（成功時に 1、失敗またはマッチしない場合に 0、内部エラー時に負の値を返す）

前提条件と不変条件

  • z が有効な SN_env 構造体を指していること。
  • z->c が単語内の処理対象位置を指していること。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を z->ket に設定する。 2. 文字列の境界チェックおよび、現在の文字コードに基づくビットマスク判定を行う。 3. a_5 配列（36個のパターン）を対象に、find_among_b を用いて単語末尾から前方へ接尾辞を検索する。 4. マッチした場合、その位置を z->bra に設定し、slice_del を呼び出して該当部分を削除する。 5. slice_del の戻り値が負であればその値を返し、成功時は 1 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_KOI8_R_russian.c:641
  • **呼び出し先**: find_among_b, slice_del

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_5: 名詞接尾辞パターンを格納した among 構造体の配列。

失敗モードと境界条件

  • z->c <= z->lb の場合（文字列の先頭に達している場合）、処理を中断し 0 を返す。
  • find_among_b がパターンにマッチしない場合、0 を返す。
  • slice_del が内部エラー（負の値）を返した場合、そのエラーコードを伝播させる。
