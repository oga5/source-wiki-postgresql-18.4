r_step_s1

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_greek.c:2539

目的

ギリシャ語のステミング処理における語尾削除ステップの一つとして、特定の接尾辞パターンを検索し、一致した文字列の置換または削除を行う。

シグネチャまたは形状

static int r_step_s1(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング処理の状態を保持する構造体）
  • 出力: 成功時に 1、失敗または処理対象外の場合に 0、エラー発生時に負の値を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • 内部で find_among_b を使用して文字列の末尾からマッチングを行うため、z->c（カーソル位置）が適切に設定されている必要がある。

アルゴリズムまたは振る舞い

1. a_3 配列に基づき、文字列末尾から find_among_b でパターンを検索する。一致すれば slice_del で削除する。 2. 続いて z->I[0] を 0 にリセットする。 3. a_2 配列に基づき、再度文字列末尾から find_among_b でパターンを検索する。 4. マッチした場合、z->c > z->lb（境界条件）を確認し、一致した among_var の値に応じて slice_from_s を用いて特定の文字列（s_35 または s_36）に置換する。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_greek.c:3483
  • 呼び出し先: find_among_b, slice_del, slice_from_s（いずれも Snowball ランタイム関数）

関連データ構造

  • struct SN_env: ステミング環境の状態管理。
  • a_2, a_3: 語尾削除ルールを定義する among 構造体の配列。

失敗モードと境界条件

  • find_among_b がマッチしない場合は 0 を返す。
  • slice_del または slice_from_s が負の値を返した場合、そのエラーコードをそのまま伝播させる。
  • z->c > z->lb の条件を満たさない場合、置換処理を行わずに 0 を返す。

  • a_2 および a_3 配列の定義内容については、ソースファイル内の該当する静的配列定義を参照すること。
