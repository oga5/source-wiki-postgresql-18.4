TimeTzADTPGetDatum

定義

  • 種別: static inline function
  • 定義箇所: src/include/utils/date.h:83

目的

TimeTzADT 構造体へのポインタを、PostgreSQLの関数呼び出しインターフェースで扱われる Datum 型へ変換する。

シグネチャまたは形状

Datum TimeTzADTPGetDatum(const TimeTzADT *X)

入力と出力

  • **入力**: const TimeTzADT *X （時刻とタイムゾーンオフセットを保持する構造体へのポインタ）
  • **出力**: Datum （ポインタ値をラップした Datum 型の表現）

前提条件と不変条件

  • 引数 X は有効な TimeTzADT 構造体を指している必要がある。
  • PointerGetDatum マクロを介して変換されるため、戻り値はポインタベースの Datum として扱われる。

アルゴリズムまたは振る舞い

1. 引数として受け取った TimeTzADT 構造体へのポインタ X を、PointerGetDatum マクロに渡す。 2. PointerGetDatum の結果（Datum 型）をそのまま呼び出し元へ返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/btree_gin/btree_gin.c
  • src/backend/executor/execExprInterp.c
  • src/backend/utils/adt/formatting.c
  • src/backend/utils/adt/jsonpath_exec.c
  • src/include/utils/date.h （PG_RETURN_TIMETZADT_P マクロ内）
  • **呼び出し先**:
  • PointerGetDatum （PostgreSQLのコアマクロ）

関連データ構造

  • TimeTzADT: 時刻とタイムゾーンオフセットを保持する構造体。
  • Datum: PostgreSQLの汎用データ型表現。

失敗モードと境界条件

  • X が NULL の場合、PointerGetDatum の振る舞いに依存する（通常は (Datum) 0 が返される）。

保守担当者向けメモ

  • 本関数はインライン関数として定義されており、パフォーマンス最適化のために src/include/utils/date.h に配置されている。
  • PG_RETURN_TIMETZADT_P マクロを通じてSQL関数から値を返す際に頻繁に使用されるため、変更時は注意が必要である。
