CreateTupleDescCopy

定義

  • 種別: function
  • 定義箇所: src/backend/access/common/tupdesc.c:251

目的

既存の TupleDesc の属性定義をコピーして、新しい TupleDesc を作成します。制約（TupleConstr）やデフォルト値はコピーされません。

シグネチャまたは形状

TupleDesc CreateTupleDescCopy(TupleDesc tupdesc)

入力と出力

  • **入力**: tupdesc (コピー元の TupleDesc 構造体)
  • **出力**: コピーされた新しい TupleDesc 構造体。

前提条件と不変条件

  • 入力 tupdesc は有効な TupleDesc である必要があります。
  • 制約情報（TupleConstr）はコピーされないため、戻り値の tdconst フィールドは NULL となります。

アルゴリズムまたは振る舞い

1. CreateTemplateTupleDesc を呼び出し、元の natts 数に基づいた新しい TupleDesc を割り当てます。 2. memcpy を使用して、属性配列（FormData_pg_attribute）をフラットコピーします。 3. 各属性に対して以下のフィールドをリセットします。

  • attnotnull = false
  • atthasdef = false
  • atthasmissing = false
  • attidentity = '\0'
  • attgenerated = '\0'

    • 4. 各属性に対して populate_compact_attribute を呼び出し、最適化用メタデータを再構築します。 5. 元の tdtypeid と tdtypmod を新しい記述子にコピーします。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/dblink/dblink.c
  • contrib/postgres_fdw/postgres_fdw.c
  • contrib/tablefunc/tablefunc.c
  • src/backend/access/spgist/spgutils.c
  • src/backend/commands/portalcmds.c
  • src/backend/commands/prepare.c
  • **呼び出し先**:
  • CreateTemplateTupleDesc
  • populate_compact_attribute

関連データ構造

  • TupleDesc: タプルの構造を定義する主要な構造体。
  • FormData_pg_attribute: 各列の属性情報を保持する構造体。

失敗モードと境界条件

  • メモリ不足時には CreateTemplateTupleDesc 内でエラーが発生し、処理が中断されます。
  • 制約やデフォルト値が必要な場合は、本関数ではなく CreateTupleDescCopyConstr を使用する必要があります。

保守担当者向けメモ

  • 本関数は制約を意図的に除外しています。制約を含めた完全なコピーが必要な場合は、CreateTupleDescCopyConstr を参照してください。
  • FormData_pg_attribute を直接操作しているため、将来的に属性構造体が拡張された場合は、コピー範囲やリセット対象フィールドの確認が必要です。
