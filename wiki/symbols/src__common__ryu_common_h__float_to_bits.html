float_to_bits

定義

  • 種別: function
  • 定義箇所: src/common/ryu_common.h:115

目的

浮動小数点数（float）を、そのビット表現を保持する符号なし32ビット整数（uint32）へ安全に変換します。

シグネチャまたは形状

static inline uint32 float_to_bits(const float f)

入力と出力

  • **入力**: const float f（変換対象の浮動小数点数）
  • **出力**: uint32（f と同じビットパターンを持つ整数）

前提条件と不変条件

  • sizeof(float) が sizeof(uint32) と等しい環境を前提としています。
  • memcpy を使用しているため、厳格なエイリアシング規則（Strict Aliasing Rule）に抵触することなくビット変換が行われます。

アルゴリズムまたは振る舞い

1. ローカル変数 bits を 0 で初期化します。 2. memcpy を用いて、f のメモリ領域から bits のメモリ領域へ sizeof(float) バイト分をコピーします。 3. コピーされたビットパターンを保持する bits を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/common/f2s.c:748 にて呼び出しを確認しています。
  • **呼び出し先**: 標準ライブラリ関数 memcpy を使用しています。

関連データ構造

  • uint32: PostgreSQL の共通型定義（c.h 経由）であり、本関数が戻り値として使用します。

失敗モードと境界条件

  • 特殊な浮動小数点値（NaN, Infinity, ゼロ）に対しても、そのビット表現をそのまま整数として抽出します。
  • 境界条件に関する明示的なエラーチェックは行われません。

保守担当者向けメモ

  • 本関数は static inline で定義されているため、ヘッダーインクルード時に各翻訳単位へ展開されます。
  • 浮動小数点数からビット表現への変換は、Ryu アルゴリズムによる文字列変換の初期ステップとして利用されます。
