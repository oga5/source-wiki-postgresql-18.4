llvm_inline_reset_caches

定義

  • 種別: function
  • 定義箇所: src/backend/jit/llvm/llvmjit_inline.cpp:155

目的

JIT コンパイルのインライン化処理で使用されるモジュールおよびインデックスのキャッシュをクリアし、メモリ上の不正な参照を防ぐ。

シグネチャまたは形状

void llvm_inline_reset_caches(void)

入力と出力

  • 入力: なし
  • 出力: なし（副作用としてグローバルなキャッシュ状態がリセットされる）

前提条件と不変条件

  • module_cache および summary_cache が初期化済みであること。
  • LLVM コンテキストの破棄やライフサイクルの終了時に呼び出される必要がある。

アルゴリズムまたは振る舞い

1. module_cache オブジェクトの clear() メソッドを呼び出し、保持されている LLVM モジュールを解放する。 2. summary_cache オブジェクトの clear() メソッドを呼び出し、保持されているビットコードインデックスを解放する。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/backend/jit/llvm/llvmjit.c:200
  • 呼び出し先:
  • module_cache->clear()
  • summary_cache->clear()

関連データ構造

  • ModuleCache
  • SummaryCache

失敗モードと境界条件

  • キャッシュオブジェクトが未初期化の状態で呼び出された場合、未定義の動作を引き起こす可能性がある。

保守担当者向けメモ

  • 本関数は llvm::ManagedStatic で管理されるキャッシュの生存期間を制御する重要な役割を持つ。LLVM コンテキストの破棄後にキャッシュが残存しないよう、適切なタイミングで呼び出すこと。
