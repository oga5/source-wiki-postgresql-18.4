json_out

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/json.c:125

目的

json 型のデータを、クライアントへ送信可能な C 文字列（cstring）形式に変換します。

シグネチャまたは形状

Datum json_out(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS を介して json 型の Datum を受け取ります。
  • **出力**: PG_RETURN_CSTRING を介して、JSON データの文字列表現を返します。

前提条件と不変条件

  • 入力データは json 型の Datum である必要があります。
  • text_to_cstring を使用するため、明示的な detoast（TOAST 展開）は不要です。

アルゴリズムまたは振る舞い

1. PG_GETARG_DATUM(0) を使用して、入力された json 型の Datum を取得します。 2. TextDatumGetCString を呼び出し、内部のテキスト表現を C 文字列に変換します。 3. 変換された文字列を PG_RETURN_CSTRING で返却します。

呼び出し元と呼び出し先

  • **呼び出し元**: PostgreSQL の関数マネージャ（SQL インターフェース経由）。
  • **呼び出し先**: TextDatumGetCString（PostgreSQL のコアユーティリティ関数）。

関連データ構造

  • Datum: PostgreSQL の汎用データ型表現。

失敗モードと境界条件

  • 入力が無効な Datum である場合、TextDatumGetCString の内部でエラーが発生する可能性があります。

保守担当者向けメモ

  • 本関数は json 型の標準的な出力関数です。jsonb 型の出力処理とは実装が異なるため、修正時は注意してください。
  • text_to_cstring が内部でメモリ割り当てを行うため、呼び出し元のメモリコンテキストが適切であることを確認してください。
