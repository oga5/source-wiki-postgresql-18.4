MJEvalInnerValues

定義

  • 種別: function
  • 定義箇所: src/backend/executor/nodeMergejoin.c:335

目的

マージ結合における内部リレーション（inner relation）のタプルから結合キーを抽出し、評価を行う。

シグネチャまたは形状

static MJEvalResult MJEvalInnerValues(MergeJoinState *mergestate, TupleTableSlot *innerslot)

入力と出力

  • **入力**:
  • mergestate: マージ結合の実行状態を保持する MergeJoinState 構造体。
  • innerslot: 評価対象のタプルを保持する TupleTableSlot。
  • **出力**:
  • MJEvalResult 型の列挙値（MJEVAL_MATCHABLE, MJEVAL_NONMATCHABLE, MJEVAL_ENDOFJOIN のいずれか）。

前提条件と不変条件

  • innerslot が NULL の場合、結合の終了を示す MJEVAL_ENDOFJOIN を返す。
  • 評価には mergestate->mj_InnerEContext を使用し、実行前に ResetExprContext でリセットされる。
  • 評価結果は MergeJoinClause 構造体の rdatum および risnull フィールドに格納される。

アルゴリズムまたは振る舞い

1. innerslot が空であれば MJEVAL_ENDOFJOIN を返す。 2. econtext をリセットし、メモリコンテキストを ecxt_per_tuple_memory に切り替える。 3. mergestate->mj_Clauses に含まれるすべての結合句について ExecEvalExpr を実行し、値を抽出する。 4. 評価中に NULL が検出された場合:

  • 最初の結合句かつ ssup_nulls_first が偽、かつ mj_FillInner が偽であれば、結合を早期終了（MJEVAL_ENDOFJOIN）させる。
  • それ以外の場合は、一致不可（MJEVAL_NONMATCHABLE）とマークする。

    • 5. 処理終了後、元のメモリコンテキストに戻して結果を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/executor/nodeMergejoin.c 内の ExecMergeJoin ループ内（行702, 880, 1048, 1114, 1304）。
  • **呼び出し先**: ExecEvalExpr（式評価用）、ResetExprContext、MemoryContextSwitchTo。

関連データ構造

  • MergeJoinState: 結合全体の実行状態を管理。
  • MergeJoinClause: 各結合キーの式や評価結果を保持。
  • TupleTableSlot: 評価対象のタプルデータ。
  • ExprContext: 式評価のためのコンテキスト。

失敗モードと境界条件

  • **NULL値**: 結合キーにNULLが含まれる場合、結合条件の不一致として扱われる。特定の条件下では結合自体の早期終了をトリガーする。
  • **メモリ**: ecxt_per_tuple_memory を使用するため、タプルごとのメモリ消費に依存する。

保守担当者向けメモ

  • 本関数は MJEvalOuterValues と対になる処理を行う。
  • 結合句の評価順序やNULLの扱いは、B-tree比較ロジック（ssup_nulls_first）と密接に関連しているため、変更時は注意が必要。
