RemoveStatisticsById

定義

  • 種別: function
  • 定義箇所: src/backend/commands/statscmds.c:785

目的

指定されたOIDを持つ拡張統計情報オブジェクトをシステムカタログから削除し、関連する統計データおよびキャッシュをクリーンアップする。

シグネチャまたは形状

void RemoveStatisticsById(Oid statsOid)

入力と出力

  • **入力**: statsOid (削除対象の統計オブジェクトのOID)
  • **出力**: なし（副作用としてカタログおよびキャッシュを更新）

前提条件と不変条件

  • 呼び出し元は、対象の統計オブジェクトに対する適切な権限チェックを済ませている必要がある。
  • トランザクション内で実行されることを前提としており、ロックはトランザクション終了まで保持される。

アルゴリズムまたは振る舞い

1. pg_statistic_ext カタログを RowExclusiveLock で開き、対象のタプルを STATEXTOID システムキャッシュから検索する。 2. 統計オブジェクトに関連付けられたテーブルのOID（stxrelid）を取得する。 3. 対象テーブルを ShareUpdateExclusiveLock でロックし、同時実行による競合を防ぐ。 4. RemoveStatisticsDataById を呼び出し、継承の有無にかかわらず関連する統計データ（pg_statistic_ext_data）を削除する。 5. CacheInvalidateRelcacheByRelid を呼び出し、対象テーブルのプランナキャッシュを無効化する。 6. CatalogTupleDelete を使用して pg_statistic_ext から該当タプルを削除する。 7. システムキャッシュを解放し、テーブルのロックを解除する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/catalog/dependency.c:1420 (依存関係管理経由)
  • **呼び出し先**:
  • table_open / table_close
  • SearchSysCache1 / ReleaseSysCache
  • RemoveStatisticsDataById
  • CacheInvalidateRelcacheByRelid
  • CatalogTupleDelete

関連データ構造

  • pg_statistic_ext: 統計オブジェクトの定義を保持するカタログ。
  • pg_statistic_ext_data: 統計情報の計算結果を保持するカタログ。
  • Form_pg_statistic_ext: カタログタプルの構造体。

失敗モードと境界条件

  • 指定された statsOid がシステムキャッシュに見つからない場合、elog(ERROR) によりトランザクションがアボートされる。

保守担当者向けメモ

  • 本関数はDDL操作の一部として実行される。統計情報の計算ロジック自体は ANALYZE 実行時に行われるため、本関数はあくまでメタデータの削除とキャッシュの整合性維持に専念する。
  • RemoveStatisticsDataById を2回呼び出しているのは、継承の有無によるデータ分離に対応するためである。
