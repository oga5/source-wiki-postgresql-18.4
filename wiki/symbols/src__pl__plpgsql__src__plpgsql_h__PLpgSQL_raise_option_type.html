PLpgSQL_raise_option_type

定義

  • 種別: enum
  • 定義箇所: src/pl/plpgsql/src/plpgsql.h:168

目的

PL/pgSQL の RAISE 文において、エラー発生時に付加できるオプションの種類を定義します。

シグネチャまたは形状

typedef enum PLpgSQL_raise_option_type
{
    PLPGSQL_RAISEOPTION_ERRCODE,
    PLPGSQL_RAISEOPTION_MESSAGE,
    PLPGSQL_RAISEOPTION_DETAIL,
    PLPGSQL_RAISEOPTION_HINT,
    PLPGSQL_RAISEOPTION_COLUMN,
    PLPGSQL_RAISEOPTION_CONSTRAINT,
    PLPGSQL_RAISEOPTION_DATATYPE,
    PLPGSQL_RAISEOPTION_TABLE,
    PLPGSQL_RAISEOPTION_SCHEMA,
} PLpgSQL_raise_option_type;

入力と出力

本シンボルは列挙型であり、入力や出力を持つ関数ではありません。各列挙子は RAISE 文のオプション指定を識別するために使用されます。

前提条件と不変条件

  • RAISE 文の構文解析および実行時に、指定されたオプションがどの属性（エラーコード、メッセージ、詳細など）を指すかを一意に決定するために使用されます。

アルゴリズムまたは振る舞い

  • RAISE 文の実行時、この列挙型を使用して、提供された式がどのエラー属性に割り当てられるべきかを制御します。

呼び出し元と呼び出し先

  • 呼び出し元: src/pl/plpgsql/src/plpgsql.h:898 にて、PLpgSQL_raise_option_type 型のメンバ opt_type が定義されています。
  • 呼び出し先: 提供されたソース excerpt 内には、本列挙型を直接呼び出す関数や、本列挙型が呼び出す関数に関する情報はありません。

関連データ構造

  • PLpgSQL_stmt (基底構造体): RAISE 文の実行に関連する構造体の一部として利用される可能性があります。

失敗モードと境界条件

  • 本列挙型に含まれないオプションを指定しようとした場合、構文解析フェーズでエラーとなります。

保守担当者向けメモ

  • 本列挙型は RAISE 文の拡張に伴い、新しいエラー属性が追加される可能性があります。変更時は pl_gram.y 等の構文解析ロジックとの整合性に注意してください。
