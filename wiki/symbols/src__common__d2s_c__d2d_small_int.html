d2d_small_int

定義

  • 種別: function
  • 定義箇所: src/common/d2s.c:961

目的

IEEE 754 倍精度浮動小数点数を、指数部が 0 以下の単純な整数として表現可能か判定し、可能な場合に floating_decimal_64 構造体へ変換結果を格納する。

シグネチャまたは形状

static inline bool d2d_small_int(const uint64 ieeeMantissa, const uint32 ieeeExponent, floating_decimal_64 *v)

入力と出力

  • **入力**:
  • ieeeMantissa: IEEE 754 倍精度浮動小数点数の仮数部（下位 52 ビット）。
  • ieeeExponent: IEEE 754 倍精度浮動小数点数の指数部。
  • v: 変換結果を格納する floating_decimal_64 構造体へのポインタ。
  • **出力**:
  • 成功時 true（v に値が設定される）、失敗時 false。

前提条件と不変条件

  • ieeeExponent は DOUBLE_BIAS および DOUBLE_MANTISSA_BITS を用いて正規化された指数 e2 を計算するために使用される。
  • 変換対象の数値が 2^52 <= m2 < 2^53 の範囲にあることを前提とする。

アルゴリズムまたは振る舞い

1. 指数 e2 を計算する。 2. e2 が [-DOUBLE_MANTISSA_BITS, 0] の範囲内にあるか確認する。 3. 範囲内の場合、仮数の下位ビットをマスクして端数（fraction）を抽出する。 4. 端数が 0 であれば、整数として表現可能と判断し、v->mantissa にシフト後の値を、v->exponent に 0 を設定して true を返す。 5. 上記条件を満たさない場合は false を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/common/d2s.c 内のメイン変換ロジック（d2d 相当の処理フロー）から呼び出される。
  • **呼び出し先**: 外部関数呼び出しはなし。

関連データ構造

  • floating_decimal_64: 変換後の仮数と指数を保持する構造体。

失敗モードと境界条件

  • e2 が指定範囲外の場合、または仮数に端数が含まれる（fraction != 0）場合は false を返す。
  • e2 < -DOUBLE_MANTISSA_BITS のような特殊なケースは、この関数に到達する前に除外される設計となっている。

保守担当者向けメモ

  • コンパイラによる過度なインライン展開を避けるため、return false を複数箇所に記述しないよう注意すること。
  • 128 ビット演算環境以外での挙動については、d2s.c 全体のビルド設定（RYU_ONLY_64_BIT_OPS）に依存する。
