RegisterCustomScanMethods

定義

  • 種別: function
  • 定義箇所: src/backend/nodes/extensible.c:87

目的

拡張機能が定義したカスタムスキャンメソッド構造体を、コアシステムのレジストリに登録する。

シグネチャまたは形状

void RegisterCustomScanMethods(const CustomScanMethods *methods)

入力と出力

  • **入力**: methods - 登録対象の CustomScanMethods 構造体へのポインタ。CustomName フィールドが必須。
  • **出力**: なし（登録に成功した場合）。

前提条件と不変条件

  • methods->CustomName が一意の識別子として設定されていること。
  • 内部ハッシュテーブル custom_scan_methods が初期化済みであること（RegisterExtensibleNodeEntry 内部で処理される）。

アルゴリズムまたは振る舞い

1. 内部の RegisterExtensibleNodeEntry 関数を呼び出す。 2. その際、ハッシュテーブルとして custom_scan_methods を指定する。 3. 識別ラベルとして "Custom Scan Methods" を使用し、methods->CustomName をキーとして methods を格納する。

呼び出し元と呼び出し先

  • **呼び出し元**: 拡張機能（外部モジュール）の初期化ルーチンから呼び出されることが想定される。
  • **呼び出し先**: RegisterExtensibleNodeEntry（src/backend/nodes/extensible.c 内）。

関連データ構造

  • CustomScanMethods: 拡張機能が提供するカスタムスキャンのメソッド群を定義する構造体。
  • custom_scan_methods: 登録されたメソッドを保持する内部ハッシュテーブル。

失敗モードと境界条件

  • RegisterExtensibleNodeEntry の仕様に基づき、同一の CustomName で既に登録が存在する場合、エラーが発生する可能性がある。

保守担当者向けメモ

  • 本関数は RegisterExtensibleNodeEntry のラッパーであり、CustomScanMethods 専用の登録インターフェースを提供する。
  • 登録されたメソッドは、後に GetCustomScanMethods を通じて取得される。
