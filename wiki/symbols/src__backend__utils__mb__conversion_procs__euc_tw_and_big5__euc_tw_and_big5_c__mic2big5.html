mic2big5

定義

  • 種別: function
  • 定義箇所: src/backend/utils/mb/conversion_procs/euc_tw_and_big5/euc_tw_and_big5.c:513

目的

MULE_INTERNAL (MIC) エンコーディングの文字列を BIG5 エンコーディングへ変換する。

シグネチャまたは形状

static int mic2big5(const unsigned char *mic, unsigned char *p, int len, bool noError)

入力と出力

  • **入力**:
  • mic: 変換元のMULE_INTERNAL文字列へのポインタ。
  • p: 変換後のBIG5文字列を格納するバッファ。
  • len: 入力文字列のバイト長。
  • noError: 変換エラー時に例外を投げず処理を中断するかを指定するフラグ。
  • **出力**:
  • 戻り値: 処理した入力バイト数。
  • p: 変換後の文字列（ヌル終端付き）が書き込まれる。

前提条件と不変条件

  • 入力文字列は PG_MULE_INTERNAL エンコーディングに従っている必要がある。
  • noError が false の場合、不正な文字や変換不能な文字に遭遇するとエラーを報告する。

アルゴリズムまたは振る舞い

1. 入力文字列を1文字ずつ走査する。 2. ASCII文字（高位ビットが立っていない文字）はそのまま出力バッファへコピーする。 3. pg_encoding_verifymbchar を使用して文字境界を検証する。 4. LC_CNS11643_1、LC_CNS11643_2、または LCPRV2_B プレフィックスを持つ文字を識別し、CNStoBIG5 関数を用いてBIG5コードへ変換する。 5. 変換後の2バイトを p に書き込む。 6. 処理終了後、出力バッファの末尾にヌル終端を付与し、消費した入力バイト数を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: 同一ファイル内の mic_to_big5 関数（PG_FUNCTION_INFO_V1 登録関数）から呼び出される。
  • **呼び出し先**:
  • pg_encoding_verifymbchar: 文字境界の検証。
  • CNStoBIG5: CNS11643からBIG5へのコード変換。
  • report_invalid_encoding / report_untranslatable_char: エラー報告。

関連データ構造

  • PG_MULE_INTERNAL, PG_BIG5: エンコーディング識別子。
  • LC_CNS11643_1, LC_CNS11643_2, LCPRV2_B: MULE_INTERNALにおける文字セット識別子。

失敗モードと境界条件

  • 入力文字列が不正なバイトシーケンスである場合、noError が false ならエラーを報告し、true ならその時点までのバイト数を返して終了する。
  • 変換不能な文字に遭遇した場合、同様にエラー報告または処理中断を行う。
  • 入力バイト長 len が0になった時点で処理を終了する。

保守担当者向けメモ

  • 本関数は変換の橋渡し役であり、具体的な変換テーブルは CNStoBIG5（big5.c 等に定義）に依存している。
  • noError フラグによる制御は COPY コマンド等の柔軟な処理をサポートするためのものである。
