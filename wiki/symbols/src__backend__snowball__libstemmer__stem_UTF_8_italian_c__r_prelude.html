r_prelude

定義

  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_italian.c:494

目的

イタリア語のステミング処理における前処理として、単語内の特定の文字置換および正規化を行う。

シグネチャまたは形状

static int r_prelude(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（Snowball ランタイム環境構造体）
  • 出力: 成功時に 1、失敗時に負の整数（エラーコード）を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • a_0 配列および g_v（母音グループ）が定義されていること。

アルゴリズムまたは振る舞い

1. **文字置換**: find_among を使用し、a_0 に定義されたルールに基づき、単語内の特定の文字列を置換する。一致しない場合は skip_utf8 で文字をスキップする。 2. **母音周辺の正規化**: 母音（g_v）に続く 'u' または 'i' を特定の文字に置換するループ処理を行う。 3. 処理中、z->bra（開始位置）と z->ket（終了位置）を更新し、slice_from_s を用いて文字列の置換を実行する。 4. 処理が完了すると 1 を返して終了する。

呼び出し元と呼び出し先

  • 呼び出し元: 提供されたソースコード内では明示的な呼び出し元は記述されていない。
  • 呼び出し先: find_among, slice_from_s, skip_utf8, in_grouping_U（いずれも Snowball ランタイム関数）。

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_0: 置換ルールを保持する among 構造体の配列。
  • g_v: 母音を定義するグループ。

失敗モードと境界条件

  • slice_from_s が負の値を返した場合、その値をそのまま呼び出し元へ伝播させる。
  • skip_utf8 が失敗した場合、lab0 または lab1 ラベルへジャンプし、現在の処理を中断してループを抜ける。
