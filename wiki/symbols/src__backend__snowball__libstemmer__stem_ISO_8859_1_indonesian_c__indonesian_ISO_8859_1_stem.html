indonesian_ISO_8859_1_stem

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_indonesian.c:312

目的

ISO-8859-1エンコーディングされたインドネシア語テキストに対し、接辞（接頭辞・接尾辞）を除去して語幹を抽出する。

シグネチャまたは形状

int indonesian_ISO_8859_1_stem(struct SN_env * z)

入力と出力

  • **入力**: z (Snowball環境構造体 SN_env)。内部の文字列バッファおよび整数ワークスペース I[0]（接頭辞フラグ）、I[1]（母音数）を使用。
  • **出力**: 成功時に 1、失敗時に 0 または負の値を返す。

前提条件と不変条件

  • z は indonesian_ISO_8859_1_create_env によって初期化されている必要がある。
  • 単語内の母音数（I[1]）が2以下の場合はステミングを行わず終了する。

アルゴリズムまたは振る舞い

1. **母音カウント**: 単語内の母音数を数え、z->I[1] に格納する。 2. **粒子・所有格除去**: 粒子（r_remove_particle）および所有格代名詞（r_remove_possessive_pronoun）を末尾から除去する。 3. **接辞除去**:

  • 1次接頭辞（r_remove_first_order_prefix）の除去を試みる。
  • 成功した場合は、さらに接尾辞（r_remove_suffix）と2次接頭辞（r_remove_second_order_prefix）の除去を順次試みる。
  • 1次接頭辞が除去できない場合は、2次接頭辞の除去を試みた後、接尾辞の除去を試みる。

    • 4. 処理の各段階で z->I[1]（母音数）が2以下になった場合、処理を中断する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/include/snowball/libstemmer/stem_ISO_8859_1_indonesian.h に宣言があり、外部から呼び出される。
  • **呼び出し先**: r_remove_particle, r_remove_possessive_pronoun, r_remove_first_order_prefix, r_remove_second_order_prefix, r_remove_suffix。

関連データ構造

  • struct SN_env: Snowballランタイムの環境構造体。
  • g_vowel: 母音判定用のグループ定義。

失敗モードと境界条件

  • 母音数が2以下の単語はステミング対象外となり、即座に終了する。
  • 内部関数が負の値を返した場合、その値をそのまま呼び出し元へ伝播させる。

保守担当者向けメモ

  • 本関数は自動生成コードであるため、直接編集せず元のSnowballスクリプトを修正すること。
  • I[0] は接頭辞の種類、I[1] は母音数を示す内部フラグとして使用されている。
