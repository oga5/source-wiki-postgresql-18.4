scalarineqsel_wrapper

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/selfuncs.c:1407

目的

不等号演算子（<, <=, >, >=）の選択率推定を行うためのラッパー関数です。クエリプランナが統計情報を利用して条件の選択率を算出する際、変数の位置調整や定数の検証といった前処理を共通化します。

シグネチャまたは形状

static Datum scalarineqsel_wrapper(PG_FUNCTION_ARGS, bool isgt, bool iseq)
  • PG_FUNCTION_ARGS: 関数呼び出しの標準引数。
  • isgt: true の場合、演算子が「より大きい（> または >=）」であることを示す。
  • iseq: true の場合、演算子が「等号を含む（>= または <=）」であることを示す。

入力と出力

  • **入力**:
  • root: プランナのコンテキスト情報。
  • operator: 評価対象の演算子 OID。
  • args: 制限句の引数リスト。
  • varRelid: 変数のリレーション ID。
  • **出力**:
  • float8: 推定された選択率（0.0〜1.0）。

前提条件と不変条件

  • args が変数と定数の比較式として解析可能であること。
  • 解析不能な場合や定数以外との比較の場合は、デフォルト値 DEFAULT_INEQ_SEL を返す。
  • 定数が NULL の場合、演算子は厳密（strict）であると見なし、選択率 0.0 を返す。

アルゴリズムまたは振る舞い

1. get_restriction_variable を呼び出し、式から変数と定数を抽出する。 2. 定数が Const 型でない場合はデフォルト値を返す。 3. 定数が NULL の場合は 0.0 を返す。 4. 変数が式の右辺にある場合、get_commutator を用いて演算子を反転させ、変数が左辺に来るように正規化する。反転できない場合はデフォルト値を返す。 5. 正規化された引数を用いて scalarineqsel を呼び出し、選択率を計算する。 6. ReleaseVariableStats を呼び出し、取得した統計情報を解放する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/selfuncs.c 内の各不等号推定関数（scalarltsel, scalartesel, scalargtsel, scalargesel）から呼び出される。
  • **呼び出し先**:
  • get_restriction_variable
  • get_commutator
  • scalarineqsel
  • ReleaseVariableStats

関連データ構造

  • VariableStatData: 統計情報取得時に使用される変数メタデータ構造体。
  • PlannerInfo: クエリプランナのグローバル情報。
  • Const: 定数ノード。

失敗モードと境界条件

  • **解析失敗**: 変数と定数の比較ではない場合、DEFAULT_INEQ_SEL を返す。
  • **NULL 定数**: 演算子が NULL を許容しないと仮定し、0.0 を返す。
  • **演算子反転失敗**: get_commutator が演算子を返せない場合、DEFAULT_INEQ_SEL を返す。

保守担当者向けメモ

  • 本関数は scalarineqsel の前処理層です。ロジックの変更は scalarineqsel 側で行うのが適切です。
  • 演算子の反転ロジックは、pg_operator カタログの整合性に依存しています。
