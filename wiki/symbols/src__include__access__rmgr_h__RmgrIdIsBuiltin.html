RmgrIdIsBuiltin

定義

  • 種別: static inline function
  • 定義箇所: src/include/access/rmgr.h:41

目的

指定されたリソースマネージャID（rmid）が、PostgreSQL標準の組み込みリソースマネージャに該当するかを判定します。

シグネチャまたは形状

static inline bool RmgrIdIsBuiltin(int rmid)

入力と出力

  • **入力**: int rmid （検証対象のリソースマネージャID）
  • **出力**: bool （組み込みIDであれば true、それ以外であれば false）

前提条件と不変条件

  • rmid は RmgrId 型（uint8）の範囲内であることを想定しています。
  • RM_MAX_BUILTIN_ID 定数に基づき、IDがこの値以下である場合に組み込みとみなします。

アルゴリズムまたは振る舞い

引数 rmid が RM_MAX_BUILTIN_ID 以下であるかを比較し、その結果を真偽値で返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/access/transam/rmgr.c:165: RmgrTable の情報取得時に使用。
  • src/bin/pg_waldump/rmgrdesc.c:91: WALダンプ出力時の条件分岐に使用。
  • src/include/access/rmgr.h:53: RmgrIdIsValid マクロ内で使用。
  • **呼び出し先**:
  • 本関数は直接的な関数呼び出しを行わず、定数 RM_MAX_BUILTIN_ID を参照します。

関連データ構造

  • RmgrId: WALレコード種別を識別する uint8 型のID。
  • RM_MAX_BUILTIN_ID: 組み込みリソースマネージャの最大IDを定義する定数。

失敗モードと境界条件

  • rmid が負の値や RM_MAX_BUILTIN_ID を超える値であっても、型定義の範囲内であれば論理的な判定結果（false）を返します。

保守担当者向けメモ

  • 本関数はインライン定義されており、頻繁に呼び出されるため、判定ロジックの変更はWAL処理全体のパフォーマンスに影響を与える可能性があります。
  • 新しい組み込みリソースマネージャを追加する場合は、RM_MAX_BUILTIN_ID の定義が適切に更新されていることを確認してください。
