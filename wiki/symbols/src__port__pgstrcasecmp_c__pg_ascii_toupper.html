pg_ascii_toupper

定義

  • 種別: function
  • 定義箇所: src/port/pgstrcasecmp.c:134

目的

ロケール設定に依存せず、ASCII文字の小文字を大文字に変換する。

シグネチャまたは形状

unsigned char pg_ascii_toupper(unsigned char ch);

入力と出力

  • **入力**: unsigned char ch（変換対象の文字）
  • **出力**: unsigned char（変換後の文字。小文字の 'a'-'z' であれば対応する大文字を返し、それ以外は入力をそのまま返す）

前提条件と不変条件

  • 入力値は unsigned char 型であること。
  • ASCII文字セットの範囲内でのみ変換が保証される。

アルゴリズムまたは振る舞い

1. 入力文字 ch が 'a' から 'z' の範囲内にあるか判定する。 2. 範囲内であれば、'A' - 'a' の差分を加算して大文字に変換する。 3. 範囲外であれば、入力値をそのまま返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/commands/event_trigger.c
  • src/backend/regex/regc_pg_locale.c
  • src/backend/utils/adt/formatting.c
  • **呼び出し先**: なし（標準演算のみで構成）

関連データ構造

なし。

失敗モードと境界条件

  • マルチバイト文字に対しては正しく動作しない（ASCII範囲外の文字は変換されずそのまま返される）。

保守担当者向けメモ

  • ロケールに左右されない厳密なASCII変換が必要な場合にのみ使用すること。
  • src/backend/parser/scansup.c の downcase_truncate_identifier() 等のロジックと整合性を保つ必要がある。
