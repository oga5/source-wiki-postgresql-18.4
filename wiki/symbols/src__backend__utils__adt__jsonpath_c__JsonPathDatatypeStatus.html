JsonPathDatatypeStatus

定義

  • 種別: enum
  • 定義箇所: src/backend/utils/adt/jsonpath.c:1254

目的

JSONパス式内のアイテムが保持するデータ型の性質（特に日時関連の特性）を分類し、評価や可変性（mutability）判定の際に利用する。

シグネチャまたは形状

enum JsonPathDatatypeStatus
{
    jpdsNonDateTime,        /* null, bool, numeric, string, array, object */
    jpdsUnknownDateTime,    /* unknown datetime type */
    jpdsDateTimeZoned,      /* timetz, timestamptz */
    jpdsDateTimeNonZoned,   /* time, timestamp, date */
};

入力と出力

本シンボルは列挙型であり、関数への引数や戻り値として、あるいは構造体メンバとして使用される。

前提条件と不変条件

  • jpdsNonDateTime は日時型以外のすべての型（null, bool, numeric, string, array, object）を包括する。
  • 日時型はタイムゾーンの有無により jpdsDateTimeZoned と jpdsDateTimeNonZoned に分類される。

アルゴリズムまたは振る舞い

JSONパス式の評価過程において、各アイテムが日時型であるか、またそのタイムゾーン特性が何であるかを追跡するために使用される。特に jspIsMutableWalker などの関数内で、パス式内の要素の状態を保持するために利用される。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • jspIsMutableWalker (src/backend/utils/adt/jsonpath.c:1272)
  • その他、パス式の走査を行う内部関数にて、状態変数として参照・代入される。
  • **呼び出し先**:
  • 本シンボルは列挙型定義であるため、直接的な呼び出し先は存在しない。

関連データ構造

  • JsonPathItem: パス式の各要素を表す構造体。
  • JsonPathMutableContext: 走査中の可変性コンテキスト。

失敗モードと境界条件

  • 該当なし（列挙型定義のため）。

保守担当者向けメモ

  • 日時型の分類ロジックに変更が生じた場合、本列挙型のメンバ追加や定義の見直しが必要となる可能性がある。
  • jspIsMutableWalker 等の走査ロジックと密接に関連しているため、変更時はパス式の評価エンジンへの影響を考慮すること。
