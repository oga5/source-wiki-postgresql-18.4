gbt_bpchargt

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_text.c:97

目的

PostgreSQLのGiSTインデックスにおいて、bpchar（空白埋め固定長文字）型に対する「より大きい（greater than）」比較操作を提供します。

シグネチャまたは形状

static bool gbt_bpchargt(const void *a, const void *b, Oid collation, FmgrInfo *flinfo)

入力と出力

  • **入力**:
  • a, b: 比較対象のデータへのポインタ。
  • collation: 照合順序（Collation）のOID。
  • flinfo: 関数マネージャ情報。
  • **出力**:
  • bool: a > b が真であれば true、そうでなければ false を返します。

前提条件と不変条件

  • a および b は bpchar 型として解釈可能なデータである必要があります。
  • 内部的に bpchargt 関数を呼び出すため、PostgreSQLの標準的な比較ルールに従います。

アルゴリズムまたは振る舞い

1. PointerGetDatum を使用して、入力ポインタ a および b を Datum 型に変換します。 2. DirectFunctionCall2Coll を介して、PostgreSQLのコア関数である bpchargt を呼び出します。 3. bpchargt の実行結果（Datum）を DatumGetBool で bool 型に変換して返します。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/btree_gist/btree_text.c 内の bptinfo 構造体（gbtree_vinfo 型）の初期化時に参照されています。
  • **呼び出し先**: PostgreSQLコアの bpchargt 関数。

関連データ構造

  • gbtree_vinfo: bpchar 型の比較関数群を保持する構造体の一部として使用されます。

失敗モードと境界条件

  • bpchargt 関数がエラーをスローした場合、本関数もそのエラーを伝播します。
  • 入力ポインタが不正な場合、PointerGetDatum およびその後の関数呼び出しで予期せぬ動作を引き起こす可能性があります。

保守担当者向けメモ

  • 本関数は bpchar 型専用の比較ロジックをカプセル化しています。
  • text 型の比較関数（gbt_textgt）と類似の構造を持っていますが、比較対象の型が異なるため、混同しないよう注意してください。
