get_gz_error

定義

  • 種別: function
  • 定義箇所: src/fe_utils/astreamer_gzip.c:217

目的

gzFile ハンドルに関連する直近のgzipエラーメッセージを取得する。

シグネチャまたは形状

static const char *get_gz_error(gzFile gzf)

入力と出力

  • **入力**: gzf (gzFile) - エラー状態を確認する対象のgzipファイルハンドル。
  • **出力**: const char * - エラー内容を示す文字列。

前提条件と不変条件

  • gzf は有効な gzFile ハンドルであること。
  • zlib ライブラリが初期化されている環境で使用されること。

アルゴリズムまたは振る舞い

1. gzerror(gzf, &errnum) を呼び出し、エラーコードとエラーメッセージを取得する。 2. エラーコードが Z_ERRNO（システムエラー）である場合、strerror(errno) を用いてシステムエラーメッセージを返す。 3. それ以外の場合は、gzerror が返した標準的なgzipエラーメッセージを返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/fe_utils/astreamer_gzip.c:136
  • src/fe_utils/astreamer_gzip.c:168
  • **呼び出し先**:
  • gzerror (zlib)
  • strerror (標準ライブラリ)

関連データ構造

  • gzFile: zlib で定義されるgzipファイル操作用ハンドル。

失敗モードと境界条件

  • gzf が無効な場合、gzerror の挙動に依存する。
  • errno が設定されていない状態で Z_ERRNO が返された場合、strerror(0) の結果が返される。

保守担当者向けメモ

  • 本関数は src/fe_utils/astreamer_gzip.c 内の静的ヘルパー関数であり、gzip操作中のエラーログ出力時に一貫したメッセージを取得するために使用される。
