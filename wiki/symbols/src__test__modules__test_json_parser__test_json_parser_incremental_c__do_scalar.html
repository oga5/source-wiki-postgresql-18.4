do_scalar

定義

  • 種別: function
  • 定義箇所: src/test/modules/test_json_parser/test_json_parser_incremental.c:337

目的

JSON パース中に検出されたスカラ値（文字列、数値、真偽値、null）を処理し、標準出力へエコーバックするコールバック関数です。

シグネチャまたは形状

static JsonParseErrorType do_scalar(void *state, char *token, JsonTokenType tokentype);

入力と出力

  • **入力**:
  • state: DoState 構造体へのポインタ（型消去済み）。
  • token: パースされたスカラ値を含む文字列。
  • tokentype: トークンの種類を示す JsonTokenType。
  • **出力**:
  • JSON_SUCCESS: 処理成功時に返される。

前提条件と不変条件

  • state は DoState 構造体としてキャスト可能であること。
  • lex_owns_tokens が偽の場合、token は呼び出し元によって割り当てられたメモリであり、本関数内で free() を実行する責任を持つ。

アルゴリズムまたは振る舞い

1. state を DoState 型にキャストする。 2. tokentype が JSON_TOKEN_STRING の場合:

  • _state->buf をリセットする。
  • escape_json を使用してトークンをエスケープし、バッファに書き込む。
  • バッファの内容を標準出力へ出力する。

    • 3. それ以外（数値、真偽値、null等）の場合:

  • token をそのまま標準出力へ出力する。

    • 4. lex_owns_tokens が偽であれば、token を解放する。 5. JSON_SUCCESS を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: JsonSemAction 構造体の .scalar フィールドを介して、pg_parse_json_incremental 内のパーサーから呼び出される。
  • **呼び出し先**:
  • resetStringInfo (lib/stringinfo.h)
  • escape_json (本ファイル内)
  • printf (標準ライブラリ)
  • free (標準ライブラリ)

関連データ構造

  • DoState: パース状態を保持する構造体。
  • JsonTokenType: トークンの種類を定義する列挙型。
  • JsonSemAction: コールバック関数テーブル。

失敗モードと境界条件

  • token が NULL の場合、あるいは state が不正なポインタである場合の挙動は定義されていない。
  • lex_owns_tokens の設定と token のメモリ管理責任が一致していない場合、ダブルフリーやメモリリークが発生する可能性がある。

保守担当者向けメモ

  • 本関数は JsonSemAction の一部として登録されるため、シグネチャの変更は common/jsonapi.h の定義と整合させる必要がある。
  • escape_json によるエスケープ処理は、JSON 構造の再構成において重要である。
