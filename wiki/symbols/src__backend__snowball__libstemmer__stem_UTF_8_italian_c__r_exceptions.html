イタリア語のステミング処理において、特定の例外単語を検出し、あらかじめ定義された置換文字列に変換する。

  • 出力: int（処理が成功し置換が行われた場合は 1、対象外または失敗した場合は 0 または負の整数）

前提条件と不変条件

  • z は有効な SN_env 構造体を指していること。
  • s_18（検索対象の例外単語）および s_19（置換後の文字列）が定義されていること。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を z->bra に設定する。 2. s_18 と一致するかを eq_s で確認する。一致しない場合は 0 を返す。 3. 現在のカーソル位置が文字列の末尾 z->l に達していない場合は、例外として処理せず 0 を返す。 4. z->ket を現在のカーソル位置に設定する。 5. slice_from_s を呼び出し、対象範囲を s_19 に置換する。 6. 置換結果を返し、成功時は 1 を戻り値とする。
