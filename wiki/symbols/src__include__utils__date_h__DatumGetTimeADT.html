DatumGetTimeADT

定義

  • 種別: static inline 関数
  • 定義箇所: src/include/utils/date.h:59

目的

PostgreSQLの内部データ表現である Datum 型から、SQLの TIME 型に対応する TimeADT 型へ変換を行うためのユーティリティ関数です。

シグネチャまたは形状

static inline TimeADT DatumGetTimeADT(Datum X)

入力と出力

  • **入力**: Datum X （変換対象のデータ）
  • **出力**: TimeADT （int64 としてキャストされた値）

前提条件と不変条件

  • 入力 Datum X は、有効な TimeADT 値を保持している必要があります。
  • 本関数は DatumGetInt64 を介して実装されており、TimeADT が int64 型であることを前提としています。

アルゴリズムまたは振る舞い

1. 引数として受け取った Datum X を DatumGetInt64 マクロで int64 型に変換します。 2. 変換された値を TimeADT 型にキャストして返します。

関連データ構造

  • TimeADT: int64 として定義される TIME 型の内部表現。
  • Datum: PostgreSQLの汎用データ型。

失敗モードと境界条件

  • 入力が TimeADT として正しくエンコードされていない Datum である場合、型変換後の値は無効となります。本関数自体は値の妥当性検証を行いません。

保守担当者向けメモ

  • 本関数はインライン化されており、パフォーマンスを重視した設計となっています。変更を加える際は、呼び出し元のバイナリ互換性に注意してください。
  • PG_GETARG_TIMEADT マクロが本関数をラップしているため、SQL関数インターフェースの変更時にはこのマクロと併せて確認が必要です。
