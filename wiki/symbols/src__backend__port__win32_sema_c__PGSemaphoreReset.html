PGSemaphoreReset

定義

  • 種別: function
  • 定義箇所: src/backend/port/win32_sema.c:115

目的

指定されたセマフォのカウントを0にリセットします。

シグネチャまたは形状

void PGSemaphoreReset(PGSemaphore sema)

入力と出力

  • **入力**: sema (PGSemaphore型) - リセット対象のセマフォハンドル。
  • **出力**: なし（void型）。

前提条件と不変条件

  • Win32 APIにはセマフォを直接リセットする機能が存在しないため、本関数はセマフォのカウントを強制的に0へ減らす操作を行います。

アルゴリズムまたは振る舞い

1. PGSemaphoreTryLock(sema) を呼び出し、セマフォのカウントが0より大きい間、ループを繰り返します。 2. PGSemaphoreTryLock が成功（セマフォを取得）する限り、カウントを減らし続けます。 3. PGSemaphoreTryLock が失敗（カウントが0）した時点でループを終了します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/storage/lmgr/proc.c (直接の呼び出しを確認)
  • src/backend/port/posix_sema.c および src/backend/port/sysv_sema.c (ソースコード内の言及を確認)
  • **呼び出し先**:
  • PGSemaphoreTryLock (ループ内で呼び出し)

関連データ構造

  • PGSemaphore: セマフォを管理するための抽象データ型。

失敗モードと境界条件

  • セマフォが既に0である場合、ループは一度も実行されず即座に終了します。
  • PGSemaphoreTryLock の振る舞いに依存しており、セマフォがロックできない状態（カウントが0）であれば安全に終了します。

保守担当者向けメモ

  • 本実装はWin32環境特有の回避策です。他のプラットフォーム（POSIX/System V）の実装と比較する際は、各プラットフォームの PGSemaphoreReset 定義を参照してください。
