pg_atomic_unlocked_write_u32_impl

定義

  • 種別: function
  • 定義箇所: src/include/port/atomics/generic.h:63

目的

アトミックな同期処理を必要としない状況において、pg_atomic_uint32 型の変数に対して直接値を代入する。

シグネチャまたは形状

static inline void pg_atomic_unlocked_write_u32_impl(volatile pg_atomic_uint32 *ptr, uint32 val)

入力と出力

  • ptr: 書き込み対象となる pg_atomic_uint32 型構造体へのポインタ（volatile 修飾あり）。
  • val: 書き込む uint32 型の値。
  • 戻り値: なし（void）。

前提条件と不変条件

  • 本関数は、メモリの可視性やアトミック性を保証するためのバリア処理を行わない。
  • 呼び出し側は、この操作が他のスレッドからのアクセスと競合しないことを保証する責任がある。

アルゴリズムまたは振る舞い

  • ptr が指す構造体の value メンバに対して、引数 val を直接代入する。

呼び出し元と呼び出し先

  • 呼び出し元: src/include/port/atomics.h:299 からの呼び出しが確認されている。
  • 呼び出し先: なし。

関連データ構造

  • pg_atomic_uint32: アトミック操作対象の 32 ビット整数を保持する構造体。

失敗モードと境界条件

  • 本関数自体は単純な代入操作であるため、関数内部での失敗モードは存在しない。
  • ptr が NULL である場合、未定義動作となる。

保守担当者向けメモ

  • 本関数は「unlocked」の名の通り、ロックやメモリバリアを伴わない非アトミックな書き込みを行う。競合が発生する可能性のある箇所では使用しないこと。
  • atomics.h を経由して使用されることを前提としている。
