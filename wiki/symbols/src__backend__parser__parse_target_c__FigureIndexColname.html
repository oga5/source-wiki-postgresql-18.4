FigureIndexColname

定義

  • 種別: function
  • 定義箇所: src/backend/parser/parse_target.c:1731

目的

インデックス定義や列定義の式から、デフォルトの列名を推論して取得します。

シグネチャまたは形状

char *FigureIndexColname(Node *node)

入力と出力

  • **入力**: node (推論対象となる式ノード)
  • **出力**: 推論された列名を表す char * 型の文字列。推論できない場合は NULL を返す可能性があります。

前提条件と不変条件

  • 入力 node は有効な構文解析ノードである必要があります。
  • 内部的に FigureColnameInternal を呼び出し、その結果を文字列として返します。

アルゴリズムまたは振る舞い

1. ローカル変数 name を NULL で初期化します。 2. FigureColnameInternal を呼び出し、node から列名を推論させます。推論結果は name に格納されます。 3. 推論された name を呼び出し元へ返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/parser/parse_utilcmd.c:3106 (ielem->indexcolname の設定に使用)
  • **呼び出し先**: FigureColnameInternal

関連データ構造

  • Node: 構文解析ツリーの基本ノード構造。

失敗モードと境界条件

  • FigureColnameInternal が列名を特定できない場合、name は NULL のまま返されます。呼び出し元はこの戻り値が NULL である可能性を考慮する必要があります。

保守担当者向けメモ

  • 本関数は FigureColnameInternal のラッパーです。列名の推論ロジックを変更する場合は、主に FigureColnameInternal を確認してください。
  • インデックス定義における列名生成に特化しており、parse_utilcmd.c からの利用が確認されています。
