GetHeapamTableAmRoutine

定義

  • 種別: function
  • 定義箇所: src/backend/access/heap/heapam_handler.c:2675

目的

ヒープテーブルアクセス手法（Heap Table AM）の全機能を定義した TableAmRoutine 構造体へのポインタを取得する。

シグネチャまたは形状

const TableAmRoutine *GetHeapamTableAmRoutine(void)

入力と出力

  • 入力: なし
  • 出力: ヒープAMのメソッドテーブル（heapam_methods）を指す定数ポインタ。

前提条件と不変条件

  • 戻り値の TableAmRoutine 構造体は静的に定義されており、呼び出しごとに同一のインスタンスを指す。

アルゴリズムまたは振る舞い

1. 内部で定義された heapam_methods 構造体のアドレスを返す。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/backend/access/heap/heapam.c:1349 (テーブルAMの整合性チェック)
  • src/backend/utils/cache/relcache.c:2034 (リレーションキャッシュの初期化)
  • 呼び出し先: なし（静的変数のアドレスを返すのみ）

関連データ構造

  • TableAmRoutine: テーブルアクセス手法のコールバック関数群を保持する構造体。
  • heapam_methods: 本関数が返す TableAmRoutine の実体。

失敗モードと境界条件

  • 常に有効なポインタを返すため、失敗モードは存在しない。

保守担当者向けメモ

  • 本関数はヒープAMの「エントリポイント」として機能する。新しいテーブルAMメソッドを追加または変更する場合は、heapam_methods 構造体の定義を更新する必要がある。
