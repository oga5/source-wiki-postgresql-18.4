help

定義

  • 種別: function
  • 定義箇所: src/bin/scripts/vacuumdb.c:1333

目的

vacuumdb コマンドの利用方法、利用可能なオプション、および接続オプションを標準出力に表示する。

シグネチャまたは形状

static void help(const char *progname)

入力と出力

  • **入力**: progname (実行中のプログラム名を示す文字列)
  • **出力**: なし（標準出力へのテキスト出力のみ）

前提条件と不変条件

  • progname は有効な文字列ポインタであること。
  • printf が標準出力に対して書き込み可能であること。

アルゴリズムまたは振る舞い

1. プログラム名を含む概要を表示する。 2. vacuumdb で使用可能なコマンドラインオプションの一覧をカテゴリ別（基本オプション、接続オプション）に表示する。 3. 最後にバグ報告先およびプロジェクトのホームページ情報を表示する。

呼び出し元と呼び出し先

  • **呼び出し元**: ソースコード内に直接的な呼び出し元の記述はありません。
  • **呼び出し先**: printf (標準ライブラリ関数)

関連データ構造

  • なし

失敗モードと境界条件

  • 標準出力がクローズされている場合や書き込みエラーが発生した場合、出力は失敗するが、本関数自体はエラーコードを返さない。

保守担当者向けメモ

  • 本関数は vacuumdb 専用のヘルプテキストを保持しています。オプションを追加または変更した際は、必ずこの関数内の記述も更新してください。
  • ツールが提示した contrib/oid2name/oid2name.c 等の他のファイルにおける help 関数は、本シンボルとは別個の定義であり、呼び出し関係にはありません。
