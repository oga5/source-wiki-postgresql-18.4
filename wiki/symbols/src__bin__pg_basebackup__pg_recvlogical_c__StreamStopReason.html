StreamStopReason

定義

  • 種別: enum
  • 定義箇所: src/bin/pg_basebackup/pg_recvlogical.c:33

目的

論理デコーディングストリームの終了理由を識別し、終了処理の挙動を制御するために使用されます。

シグネチャまたは形状

typedef enum
{
	STREAM_STOP_NONE,
	STREAM_STOP_END_OF_WAL,
	STREAM_STOP_KEEPALIVE,
	STREAM_STOP_SIGNAL
} StreamStopReason;

入力と出力

  • 入力: 該当なし（列挙型定義）
  • 出力: 終了処理関数 prepareToTerminate の引数として渡されます。

前提条件と不変条件

  • ストリームの終了時に、必ずいずれかの列挙値が選択される必要があります。

アルゴリズムまたは振る舞い

  • STREAM_STOP_NONE: ストリームが停止していない状態を示します。
  • STREAM_STOP_END_OF_WAL: WALの終端に到達したことによる停止を示します。
  • STREAM_STOP_KEEPALIVE: キープアライブ信号による停止を示します。
  • STREAM_STOP_SIGNAL: シグナル（終了要求）による停止を示します。

呼び出し元と呼び出し先

  • 呼び出し元: prepareToTerminate 関数がこの型を引数として受け取ります（src/bin/pg_basebackup/pg_recvlogical.c:1064）。
  • 呼び出し先: 情報なし。

関連データ構造

  • prepareToTerminate 関数において、終了処理のロジック分岐に使用されます。

失敗モードと境界条件

  • 該当なし。

保守担当者向けメモ

  • 新たな終了条件が必要になった場合は、この列挙型に値を追加し、prepareToTerminate 内の処理を更新してください。
