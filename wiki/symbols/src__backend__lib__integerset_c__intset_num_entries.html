intset_num_entries

定義

  • 種別: function
  • 定義箇所: src/backend/lib/integerset.c:348

目的

IntegerSet 構造体に現在格納されている整数の総数を取得する。

シグネチャまたは形状

uint64 intset_num_entries(IntegerSet *intset)

入力と出力

  • **入力**: intset (IntegerSet *) - 対象となる整数集合へのポインタ。
  • **出力**: uint64 - 集合に含まれる要素の総数。

前提条件と不変条件

  • intset は有効な IntegerSet 構造体を指している必要がある。

アルゴリズムまたは振る舞い

IntegerSet 構造体内の num_entries フィールドの値を直接返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/access/gist/gistvacuum.c:513
  • src/test/modules/test_integerset/test_integerset.c:226, 331, 408
  • **呼び出し先**:
  • なし（構造体フィールドへの直接アクセスのみ）。

関連データ構造

  • IntegerSet: 集合のルート構造体。本関数はこの構造体が保持する num_entries を参照する。

失敗モードと境界条件

  • intset が NULL の場合、セグメンテーション違反が発生する可能性がある（ソースコード上に明示的なチェックは存在しない）。

保守担当者向けメモ

  • 本関数は単なるゲッターであり、計算コストは O(1) である。
  • IntegerSet 構造体の内部状態（num_entries）が更新されるタイミングについては、intset_add_member 等の挿入ロジックを参照すること。
