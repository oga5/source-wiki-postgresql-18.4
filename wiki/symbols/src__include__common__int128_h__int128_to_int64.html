int128_to_int64

定義

  • 種別: function
  • 定義箇所: src/include/common/int128.h:93

目的

128ビット整数型 (INT128) を 64ビット整数型 (int64) へ変換します。

シグネチャまたは形状

static inline int64 int128_to_int64(INT128 val)

入力と出力

  • **入力**: INT128 val (変換対象の128ビット整数)
  • **出力**: int64 (変換後の64ビット整数)

前提条件と不変条件

  • 入力値 val が int64 の表現範囲を超えている場合、キャストによる切り捨てが発生します。

アルゴリズムまたは振る舞い

  • 入力された INT128 型の値を int64 型へ直接キャストして返します。
  • コンパイラがネイティブな int128 をサポートしている環境では、組み込みのキャストが実行されます。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/timestamp.c:2668
  • src/backend/utils/adt/timestamp.c:2681
  • **呼び出し先**:
  • 情報なし（言語組み込みのキャスト操作のため、関数呼び出しは発生しません）。

関連データ構造

  • INT128: 128ビット整数を保持する型。環境に応じてネイティブ型または構造体として定義されます。

失敗モードと境界条件

  • 変換時にオーバーフローが発生しても、本関数はエラーを報告せず、キャスト結果をそのまま返します。

保守担当者向けメモ

  • 本関数は static inline で定義されており、ヘッダーファイル内で完結しています。
  • ソフトウェア実装（USE_NATIVE_INT128 が無効な環境）においても、INT128 型のメモリレイアウトは int64 へのキャストが意図したビットパターンを保持するように設計されています。
