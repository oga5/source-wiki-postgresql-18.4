box_copy

定義

  • 種別: function
  • 定義箇所: src/backend/access/spgist/spgproc.c:81

目的

既存の BOX 構造体の内容を新しいメモリ領域に複製し、そのポインタを返します。

シグネチャまたは形状

BOX *box_copy(BOX *orig)

入力と出力

  • **入力**: orig (複製元となる BOX 構造体へのポインタ)
  • **出力**: 複製された BOX 構造体へのポインタ。メモリは palloc によって確保されます。

前提条件と不変条件

  • orig は有効な BOX 構造体を指している必要があります。
  • palloc を使用しているため、PostgreSQLのメモリコンテキスト内で呼び出される必要があります。

アルゴリズムまたは振る舞い

1. palloc を使用して BOX 構造体サイズ分のメモリを確保します。 2. 確保したメモリ領域に対して、orig が指す BOX の内容をコピーします。 3. 複製された領域へのポインタを返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/access/spgist/spgkdtreeproc.c:323
  • src/backend/access/spgist/spgquadtreeproc.c:283
  • **呼び出し先**:
  • palloc (PostgreSQLのメモリ管理関数)

関連データ構造

  • BOX: utils/geo_decls.h で定義される、軸平行なボックスを表す幾何データ型。

失敗モードと境界条件

  • palloc がメモリ不足により失敗した場合、PostgreSQLの標準的なエラー処理（elog または ereport）により例外が発生します。
