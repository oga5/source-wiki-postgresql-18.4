pgstat_bgwriter_snapshot_cb

定義

  • 種別: function
  • 定義箇所: src/backend/utils/activity/pgstat_bgwriter.c:102

目的

バックグラウンドライターの統計情報を共有メモリからローカルスナップショットへコピーし、統計リセット時刻に基づいた補正を行うためのコールバック関数です。

シグネチャまたは形状

void pgstat_bgwriter_snapshot_cb(void)

入力と出力

  • **入力**: 共有メモリ上の pgStatLocal.shmem->bgwriter に保持された統計データおよびリセットオフセット。
  • **出力**: pgStatLocal.snapshot.bgwriter に格納された、リセット補正済みの統計データ。

前提条件と不変条件

  • 呼び出し時に pgStatLocal.shmem が初期化されていること。
  • 統計情報のコピーには changecount を用いたロックフリーに近い手法が適用される。
  • リセットオフセットの取得には stats_shmem->lock による共有ロックが必要。

アルゴリズムまたは振る舞い

1. pgstat_copy_changecounted_stats を呼び出し、共有メモリからローカルスナップショットへ統計データをコピーする。 2. stats_shmem->lock を共有モードで取得し、reset_offset（リセット時点の統計値）をローカル変数 reset にコピーする。 3. BGWRITER_COMP マクロを使用し、スナップショットの各統計フィールド（buf_written_clean, maxwritten_clean, buf_alloc）から reset の値を減算する。これにより、統計リセット後の累積値が算出される。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/activity/pgstat.c:413（統計システムのスナップショット取得処理からコールバックとして参照）。
  • **呼び出し先**:
  • pgstat_copy_changecounted_stats
  • LWLockAcquire / LWLockRelease

関連データ構造

  • PgStatShared_BgWriter: 共有メモリ上のバックグラウンドライター統計構造体。
  • PgStat_BgWriterStats: 統計カウンタを保持する構造体。
  • pgStatLocal: プロセスローカルな統計管理用グローバル変数。

失敗モードと境界条件

  • 共有メモリへのアクセス時にロック競合が発生する可能性があるが、LWLock により制御されている。
  • 統計フィールドの追加・削除時は、BGWRITER_COMP マクロの適用対象を更新する必要がある。

保守担当者向けメモ

  • 本関数は統計システムのスナップショット取得フローの一部として実行されます。
  • 統計フィールドの定義変更時は、src/backend/utils/activity/pgstat_bgwriter.c 内の BGWRITER_COMP マクロ定義と整合性を取ってください。
