ginBeginBAScan

定義

  • 種別: function
  • 定義箇所: src/backend/access/gin/ginbulk.c:256

目的

BuildAccumulator 内の赤黒木（RBT）を走査するためのイテレータを初期化します。

シグネチャまたは形状

void ginBeginBAScan(BuildAccumulator *accum)

入力と出力

  • **入力**: accum - 走査対象の BuildAccumulator 構造体へのポインタ。
  • **出力**: なし（副作用として accum->tree_walk が初期化されます）。

前提条件と不変条件

  • accum が有効な BuildAccumulator 構造体を指していること。
  • accum->tree が初期化済みであること。

アルゴリズムまたは振る舞い

rbt_begin_iterate 関数を呼び出し、accum->tree に対して LeftRightWalk（左から右への順序）で走査を開始するように accum->tree_walk を設定します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/access/gin/ginfast.c
  • src/backend/access/gin/gininsert.c
  • **呼び出し先**:
  • rbt_begin_iterate（src/include/lib/rbt.h で定義された赤黒木走査関数）

関連データ構造

  • BuildAccumulator: GINインデックス構築中の状態を保持する構造体。
  • rbt_iterator（accum->tree_walk 内）: 赤黒木の走査状態を保持する構造体。

失敗モードと境界条件

  • accum が NULL の場合、セグメンテーション違反が発生する可能性があります。

保守担当者向けメモ

  • 本関数は ginGetBAEntry を使用した走査の準備段階として機能します。
  • LeftRightWalk を指定しているため、走査結果はキーの昇順で得られます。
