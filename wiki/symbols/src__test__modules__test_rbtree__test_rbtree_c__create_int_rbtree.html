create_int_rbtree

定義

  • 種別: function
  • 定義箇所: src/test/modules/test_rbtree/test_rbtree.c:79

目的

テスト用の赤黒木（RBTree）インスタンスを生成し、初期化を行う。

シグネチャまたは形状

static RBTree *create_int_rbtree(void);

入力と出力

  • **入力**: なし
  • **出力**: RBTree 構造体へのポインタ。

前提条件と不変条件

  • IntRBTreeNode 構造体サイズ、および irbt_cmp, irbt_combine, irbt_alloc, irbt_free コールバック関数が定義されている必要がある。

アルゴリズムまたは振る舞い

rbt_create を呼び出し、以下のコールバックを指定して赤黒木を構築する。

  • ノードサイズ: sizeof(IntRBTreeNode)
  • 比較関数: irbt_cmp
  • 重複キー結合関数: irbt_combine
  • メモリ確保関数: irbt_alloc
  • メモリ解放関数: irbt_free
  • 追加引数: NULL

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/test/modules/test_rbtree/test_rbtree.c 内の各テスト関数（test_rb_tree 等の内部ロジック）から、新しい木を作成する際に呼び出される。
  • **呼び出し先**:
  • rbt_create (PostgreSQL lib/rbtree.h API)

関連データ構造

  • RBTree: 赤黒木の管理構造体。
  • IntRBTreeNode: 本テストモジュールで使用される整数キーを保持するノード構造体。

失敗モードと境界条件

  • rbt_create 内部でメモリ確保に失敗した場合、PostgreSQL のエラーハンドリング機構により elog(ERROR, ...) が発生する可能性がある。

保守担当者向けメモ

  • 本関数はテスト専用のファクトリ関数である。赤黒木ライブラリの仕様変更に伴い、コールバック関数のシグネチャや rbt_create の引数が変更された場合は本関数の修正が必要となる。
