astreamer_verify_free

定義

  • 種別: function
  • 定義箇所: src/bin/pg_verifybackup/astreamer_verify.c:146

目的

検証用ストリーマーに関連付けられたリソースを解放し、メモリリークを防ぐ。

シグネチャまたは形状

static void astreamer_verify_free(astreamer *streamer)

入力と出力

  • **入力**: astreamer *streamer - 解放対象のストリーマー構造体へのポインタ。
  • **出力**: なし（void）。

前提条件と不変条件

  • streamer は astreamer_verify 構造体としてキャスト可能であること。
  • pfree を使用するため、PostgreSQL のメモリコンテキスト内で適切に割り当てられたメモリであること。

アルゴリズムまたは振る舞い

1. 引数の streamer を astreamer_verify 型にキャストする。 2. mystreamer->checksum_ctx が存在する場合、pfree を呼び出してチェックサム計算用コンテキストを解放する。 3. 最後に streamer 自体を pfree で解放する。

呼び出し元と呼び出し先

  • **呼び出し元**: astreamer_ops 構造体の .free メンバとして登録されており、ストリーマーのライフサイクル管理を行う上位の処理から呼び出される（直接的な呼び出し元関数の詳細はソース抜粋からは不明）。
  • **呼び出し先**: pfree（PostgreSQL のメモリ解放関数）。

関連データ構造

  • astreamer: 基底となるストリーマー構造体。
  • astreamer_verify: 本関数が操作する検証用ストリーマーの派生構造体。

失敗モードと境界条件

  • streamer が NULL の場合、キャストおよび pfree の挙動は未定義（本関数内での明示的な NULL チェックは存在しない）。

保守担当者向けメモ

  • 本関数は astreamer_ops インターフェースの一部として定義されている。新しいメンバを astreamer_verify 構造体に追加した場合は、必ず本関数内で解放処理を追加すること。
