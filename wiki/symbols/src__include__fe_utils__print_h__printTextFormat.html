printTextFormat

定義

  • 種別: struct
  • 定義箇所: src/include/fe_utils/print.h:81

目的

クエリ結果をテーブル形式で出力する際の、罫線や装飾文字のスタイル定義を保持するために使用されます。

シグネチャまたは形状

typedef struct printTextFormat
{
	const char *name;
	printTextLineFormat lrule[4];
	const char *midvrule_nl;
	const char *midvrule_wrap;
	const char *midvrule_blank;
	const char *header_nl_left;
	const char *header_nl_right;
	const char *nl_left;
	const char *nl_right;
	const char *wrap_left;
	const char *wrap_right;
	bool		wrap_right_border;
} printTextFormat;

入力と出力

本構造体はデータ構造であり、関数ではありません。出力整形処理において、スタイル設定の参照先として利用されます。

前提条件と不変条件

  • lrule 配列は enum printTextRule をインデックスとして使用します。
  • wrap_right_border は、border=0 の設定時にラップされたデータの右側に境界線を描画するかどうかを制御します。

アルゴリズムまたは振る舞い

テーブル出力処理において、get_line_style 関数などを通じて取得され、各行やセルの描画時に使用する罫線文字や装飾文字を決定するための静的な設定セットとして機能します。

呼び出し元と呼び出し先

  • **呼び出し元（参照元）**:
  • src/fe_utils/print.c: pg_asciiformat, pg_asciiformat_old, pg_utf8format の定義および get_line_style 関数内。
  • src/include/fe_utils/print.h: printTableOpt 構造体内の line_style フィールド。
  • **呼び出し先**: 本構造体はデータ保持用であり、直接関数を呼び出すことはありません。

関連データ構造

  • printTextLineFormat: lrule 配列の要素として使用される罫線定義構造体。
  • printTextRule: lrule 配列のインデックスとして使用される列挙型。
  • printTableOpt: 本構造体を参照して出力スタイルを管理する構造体。

失敗モードと境界条件

  • line_style が NULL の場合、デフォルトのスタイルが使用される設計となっています。
  • 各文字列フィールドに不正な文字コードや空文字列が設定された場合、出力結果のレイアウトが崩れる可能性があります。

保守担当者向けメモ

  • 新しい出力スタイルを追加する場合は、src/fe_utils/print.c にて本構造体のインスタンスを定義し、get_line_style で選択可能にする必要があります。
  • PGDLLIMPORT を介して外部公開されているインスタンス（pg_asciiformat 等）を変更する際は、バイナリ互換性に注意してください。
