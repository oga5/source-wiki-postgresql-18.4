pg_query_performance_counter

定義

  • 種別: function
  • 定義箇所: src/include/portability/instr_time.h:135

目的

Windows環境において、高精度なパフォーマンスカウンタの値を取得し、instr_time 構造体としてラップして提供する。

シグネチャまたは形状

static inline instr_time pg_query_performance_counter(void)

入力と出力

  • 入力: なし (void)
  • 出力: instr_time 構造体（内部の ticks メンバに QueryPerformanceCounter の結果が格納される）

前提条件と不変条件

  • Windowsプラットフォームでの使用を前提とする。
  • QueryPerformanceCounter がシステム上で利用可能であること。

アルゴリズムまたは振る舞い

1. LARGE_INTEGER 型の変数 tmp を宣言する。 2. Windows API の QueryPerformanceCounter(&tmp) を呼び出し、現在のカウンタ値を取得する。 3. tmp.QuadPart を instr_time 構造体の ticks メンバに代入する。 4. 取得した instr_time 構造体を返す。

呼び出し元と呼び出し先

  • 呼び出し先: QueryPerformanceCounter (Windows API)
  • 呼び出し元:
  • src/include/portability/instr_time.h 内の INSTR_TIME_SET_CURRENT マクロ等から間接的に呼び出される可能性がある。

関連データ構造

  • instr_time: 時間計測用の不透明な構造体。
  • LARGE_INTEGER: Windows API で使用される 64 ビット整数型。

失敗モードと境界条件

  • QueryPerformanceCounter が失敗した場合の戻り値やエラーハンドリングについては、本関数内では定義されていない。

保守担当者向けメモ

  • 本関数は static inline で定義されており、ヘッダファイル内で完結している。
  • Windows固有のAPIに依存しているため、プラットフォーム固有のビルド設定が適切であることを確認すること。
