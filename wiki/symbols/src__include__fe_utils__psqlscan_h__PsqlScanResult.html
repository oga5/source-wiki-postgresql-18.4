PsqlScanResult

定義

  • 種別: enum
  • 定義箇所: src/include/fe_utils/psqlscan.h:30

目的

SQL 文の字句解析処理において、解析器が到達した状態や検出したイベントを呼び出し元に通知するために使用されます。

シグネチャまたは形状

typedef enum
{
	PSCAN_SEMICOLON,
	PSCAN_BACKSLASH,
	PSCAN_INCOMPLETE,
	PSCAN_EOL,
} PsqlScanResult;

入力と出力

  • **出力**: psql_scan() 関数の戻り値として返されます。

前提条件と不変条件

  • psql_scan() の呼び出し結果として、常にいずれか一つの列挙値が返されます。

アルゴリズムまたは振る舞い

解析器の状態に応じて以下のいずれかを返します：

  • PSCAN_SEMICOLON: SQL コマンドを終了させるセミコロンを検出した。
  • PSCAN_BACKSLASH: バックスラッシュコマンドを検出した。
  • PSCAN_EOL: 行末に達し、SQL 文が完了している可能性がある。

呼び出し元と呼び出し先

  • **呼び出し元**: psql_scan() を実行する各プログラム（psql の mainloop.c、pgbench.c、および test_escape.c など）。
  • **呼び出し先**: 本シンボル自体は列挙型であり、呼び出し先はありません。

関連データ構造

  • PsqlScanState: 解析器の内部状態を保持する構造体。
  • psql_scan(): 本列挙型を返すメインの解析関数。

失敗モードと境界条件

  • 特記事項なし。列挙された値は、解析器が定義されたイベントを検出した際の正常な終了状態を表します。

保守担当者向けメモ

  • 新しい解析イベントを追加する場合は、この列挙型に値を追加し、psql_scan() の実装側で適切に返されるようにしてください。
