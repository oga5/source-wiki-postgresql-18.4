r_R2

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_irish.c:330

目的

現在のカーソル位置が、単語内の「R2」領域の開始位置以降にあるかどうかを判定します。

シグネチャまたは形状

static int r_R2(struct SN_env * z)

入力と出力

  • **入力**: z (Snowball 実行環境構造体)
  • **出力**: 判定結果を整数で返します。z->I[0]（R2領域の開始位置）が現在のカーソル位置 z->c 以下であれば真（1）、そうでなければ偽（0）を返します。

前提条件と不変条件

  • z->I[0] には、r_mark_regions 関数によって事前に計算された R2 領域の開始位置が格納されている必要があります。
  • z->c は現在の処理対象位置を指します。

アルゴリズムまたは振る舞い

1. 実行環境 z 内の整数配列 I のインデックス 0（R2領域の開始位置）を参照します。 2. 現在のカーソル位置 z->c と比較します。 3. z->I[0] <= z->c の評価結果を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: 複数の言語用ステマー（stem_ISO_8859_1_basque.c, stem_ISO_8859_1_catalan.c, stem_ISO_8859_1_dutch.c 等）から参照されています。
  • **呼び出し先**: 本関数は直接他の関数を呼び出しません。

関連データ構造

  • struct SN_env: Snowball ステマーの実行状態を保持する構造体。I 配列（領域境界）および c（カーソル位置）を保持します。

失敗モードと境界条件

  • z が NULL の場合、未定義の動作となります。
  • r_mark_regions が未実行で z->I[0] が適切に初期化されていない場合、誤った判定結果を返します。
