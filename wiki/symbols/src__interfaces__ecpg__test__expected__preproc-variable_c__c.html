ECPGプリプロセッサのテストスイートにおいて、C言語の文字列ポインタ型を抽象化し、SQLホスト変数として利用するための型定義です。

  • 入力: 文字列データ（char*）
  • 出力: c 型として定義された文字列ポインタ
