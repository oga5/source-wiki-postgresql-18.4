calc_hist_selectivity

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/rangetypes_selfuncs.c:372

目的

範囲型（Range Types）の統計情報（境界ヒストグラムおよび長さヒストグラム）に基づき、指定された演算子に対するクエリの選択率を推定するディスパッチャ関数です。

シグネチャまたは形状

static double calc_hist_selectivity(TypeCacheEntry *typcache, VariableStatData *vardata, const RangeType *constval, Oid operator);

入力と出力

  • **入力**:
  • typcache: 型固有のキャッシュ情報。
  • vardata: 対象列の統計情報（VariableStatData）。
  • constval: 比較対象の定数範囲値。
  • operator: 選択率計算に使用する範囲演算子のOID。
  • **出力**:
  • 推定された選択率（double 型）。統計情報が利用できない場合は -1.0 を返します。

前提条件と不変条件

  • vardata に有効な統計情報（STATISTIC_KIND_BOUNDS_HISTOGRAM）が含まれている必要があります。
  • 統計情報のセキュリティチェック（statistic_proc_security_check）を通過する必要があります。
  • 境界ヒストグラムには空の範囲が含まれていないことが前提です。

アルゴリズムまたは振る舞い

1. **セキュリティ検証**: 型キャッシュ内の比較関数および差分関数に対してセキュリティチェックを実行します。 2. **統計取得**: pg_statistic から境界ヒストグラムを取得し、RangeBound の配列にデシリアライズします。 3. **条件分岐**:

  • operator が CONTAINS または CONTAINED の場合、追加で長さヒストグラム（STATISTIC_KIND_RANGE_LENGTH_HISTOGRAM）を取得します。
  • それ以外の演算子では、境界ヒストグラムのみを使用します。

    • 4. **計算**: 演算子に応じて適切な計算ロジック（calc_hist_selectivity_scalar や calc_hist_selectivity_contains 等）を呼び出し、選択率を算出します。

  • &&（重なり）演算子などは、論理演算（NOT (A << B OR A >> B)）を用いて確率を合成します。

    • 5. **返却**: 計算結果を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/rangetypes_selfuncs.c 内の calc_rangesel から呼び出されます。
  • src/backend/utils/adt/multirangetypes_selfuncs.c からも呼び出されます。
  • **呼び出し先**:
  • calc_hist_selectivity_scalar
  • calc_hist_selectivity_contains
  • get_attstatsslot / free_attstatsslot
  • range_deserialize

関連データ構造

  • TypeCacheEntry: 型の比較関数や差分関数の情報。
  • VariableStatData: 列の統計情報（statsTuple を保持）。
  • RangeBound: 範囲の境界値（下限・上限）。
  • AttStatsSlot: 統計情報スロット。

失敗モードと境界条件

  • 統計情報が不足している場合（hslot.nvalues < 2 など）は -1.0 を返します。
  • セキュリティチェックに失敗した場合は -1 を返します。
  • 境界ヒストグラム内に空の範囲が含まれている場合は elog(ERROR) を発生させます。

保守担当者向けメモ

  • 本関数は「範囲の長さ」と「下限値」が独立しているという仮定に基づいています。
  • operator ごとの計算ロジックは、ヒストグラムのビン間を線形補間する幾何学的なアプローチをとっています。
  • カスタム範囲型を追加する場合、subdiff 関数が正しく実装されていないと推定精度が低下するため注意してください。
