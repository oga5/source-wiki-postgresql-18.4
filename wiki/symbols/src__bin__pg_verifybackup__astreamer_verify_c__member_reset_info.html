member_reset_info

定義

  • 種別: function
  • 定義箇所: src/bin/pg_verifybackup/astreamer_verify.c:422

目的

検証ストリーマーの状態をリセットし、アーカイブ内の次のメンバー（ファイル）を処理するための準備を行う。

シグネチャまたは形状

static void member_reset_info(astreamer *streamer)

入力と出力

  • 入力: astreamer *streamer - 検証対象のストリーマー構造体へのポインタ。
  • 出力: なし（副作用として astreamer_verify 構造体のメンバを初期化する）。

前提条件と不変条件

  • 引数 streamer は astreamer_verify 型にキャスト可能であること。
  • 本関数呼び出し後、検証状態は「メンバー未処理」の状態に遷移する。

アルゴリズムまたは振る舞い

1. streamer を astreamer_verify 型へキャストする。 2. 以下のメンバ変数をリセットする:

  • mfile を NULL に設定。
  • verify_checksum および verify_control_data フラグを false に設定。
  • checksum_bytes および control_file_bytes カウンタを 0 に設定。

呼び出し元と呼び出し先

  • 呼び出し元: src/bin/pg_verifybackup/astreamer_verify.c 内のストリーマー処理ロジック（行122付近）。
  • 呼び出し先: なし。

関連データ構造

  • astreamer_verify: 検証状態を保持する構造体。本関数はこの構造体の各フィールドをクリアする。

失敗モードと境界条件

  • 引数 streamer が NULL の場合、キャストおよびメンバアクセスにより未定義動作が発生する可能性がある。

保守担当者向けメモ

  • 本関数は astreamer_ops インターフェースの一部として、各メンバーの処理境界で呼び出されることを想定している。新しい検証状態変数を astreamer_verify に追加した場合は、必ず本関数でリセット処理を追加すること。
