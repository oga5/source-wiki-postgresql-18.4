JsonbHashScalarValueExtended

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/jsonb_util.c:1371

目的

JsonbValue 型のスカラ値（Null、文字列、数値、真偽値）に対して、指定されたシード値を用いてハッシュ値を計算し、既存のハッシュ値に累積的に反映させます。

シグネチャまたは形状

void JsonbHashScalarValueExtended(const JsonbValue *scalarVal, uint64 *hash, uint64 seed)

入力と出力

  • **入力**:
  • scalarVal: ハッシュ対象となる JsonbValue 構造体へのポインタ。
  • hash: 現在のハッシュ値が格納された変数へのポインタ。計算結果がここに上書きされます。
  • seed: ハッシュ計算に使用するシード値。
  • **出力**:
  • なし（hash ポインタ経由で結果を更新）。

前提条件と不変条件

  • scalarVal->type は jbvNull, jbvString, jbvNumeric, jbvBool のいずれかである必要があります。それ以外の場合は elog(ERROR) が発生します。
  • hash ポインタは有効なメモリ領域を指している必要があります。

アルゴリズムまたは振る舞い

1. scalarVal->type に基づき、以下の通り一時的なハッシュ値 tmp を算出します。

  • jbvNull: seed + 0x01
  • jbvString: hash_any_extended を使用して文字列から算出。
  • jbvNumeric: hash_numeric_extended を使用して数値から算出。
  • jbvBool: seed が非ゼロの場合 hashcharextended を使用。seed がゼロの場合は真なら 0x02、偽なら 0x04 を使用。

    • 2. 既存の *hash 値を ROTATE_HIGH_AND_LOW_32BITS マクロで回転させます。 3. 回転後の *hash と tmp の間で XOR 演算を行い、結果を *hash に格納します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/jsonb_op.c:323
  • **呼び出し先**:
  • hash_any_extended
  • DirectFunctionCall2 (内部で hash_numeric_extended または hashcharextended を呼び出し)

関連データ構造

  • JsonbValue: JSONB のノード表現。
  • uint64: ハッシュ値の型。

失敗モードと境界条件

  • scalarVal->type がサポート対象外（jbvObject, jbvArray 等）の場合、elog(ERROR) により処理が中断されます。

保守担当者向けメモ

  • 本関数は JsonbHashScalarValue の拡張版であり、シード値によるハッシュの制御が可能です。
  • ROTATE_HIGH_AND_LOW_32BITS マクロによるビット回転がハッシュの混合に寄与しているため、変更時はハッシュの衝突特性に注意してください。
