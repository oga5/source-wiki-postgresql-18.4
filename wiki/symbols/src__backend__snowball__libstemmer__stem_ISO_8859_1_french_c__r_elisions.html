r_elisions

目的

フランス語のステミング処理において、アポストロフィを含むエリジオン（母音省略）を検出し、該当箇所を削除する。

シグネチャまたは形状

static int r_elisions(struct SN_env * z)

入力と出力

  • 入力: SN_env 構造体へのポインタ（現在のステミング環境）。
  • 出力: 処理が成功した場合は 1、条件不一致やエラーにより処理を中断した場合は 0 または負の整数（slice_del の戻り値）。

前提条件と不変条件

  • z->c が単語内の現在位置を指していること。
  • SN_env 構造体が適切に初期化されていること。

アルゴリズムまたは振る舞い

1. 現在位置 z->c を開始点 z->bra に設定する。 2. g_elision_char グループに含まれる文字を確認するか、または特定の文字列 s_0 と一致するかを判定する。 3. 条件を満たした場合、現在位置が単語末尾でないこと、かつ現在の文字がアポストロフィ (') であることを確認する。 4. アポストロフィを確認後、その直後を終了点 z->ket に設定する。 5. slice_del(z) を呼び出し、該当するエリジオン部分を削除する。

呼び出し元と呼び出し先

  • 呼び出し先: in_grouping, eq_s, slice_del（いずれも Snowball ランタイム関数）。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • g_elision_char: エリジオン判定に使用される文字グループ。

失敗モードと境界条件

  • 単語の末尾に達している場合、または現在の文字がアポストロフィでない場合は 0 を返す。
  • slice_del が負の値を返した場合、その値をそのまま戻り値として伝播させる。
