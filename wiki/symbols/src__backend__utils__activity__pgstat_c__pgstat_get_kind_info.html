pgstat_get_kind_info

定義

  • 種別: function
  • 定義箇所: src/backend/utils/activity/pgstat.c:1436

目的

指定された統計種別（PgStat_Kind）に対応する統計情報構造体（PgStat_KindInfo）を取得します。

シグネチャまたは形状

const PgStat_KindInfo *pgstat_get_kind_info(PgStat_Kind kind)

入力と出力

  • **入力**: kind (統計種別を示す列挙型または識別子)
  • **出力**: 該当する PgStat_KindInfo 構造体へのポインタ。見つからない場合は NULL を返します。

前提条件と不変条件

  • 組み込みの統計種別は pgstat_kind_builtin_infos テーブルから取得されます。
  • カスタム統計種別は pgstat_kind_custom_infos 配列から取得されます。

アルゴリズムまたは振る舞い

1. pgstat_is_kind_builtin(kind) が真であれば、組み込みテーブル pgstat_kind_builtin_infos の該当インデックスを返します。 2. pgstat_is_kind_custom(kind) が真であれば、kind から PGSTAT_KIND_CUSTOM_MIN を引いた値をインデックスとして pgstat_kind_custom_infos 配列を参照します。 3. カスタム情報配列が初期化されていない、または該当インデックスが NULL の場合は NULL を返します。 4. いずれにも該当しない場合は NULL を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/activity/pgstat.c 内の統計処理関数群（行 651, 773, 855, 859, 877, 938, 1049, 1065, 1090, 1147, 1195, 1214）から頻繁に呼び出されています。
  • **呼び出し先**: pgstat_is_kind_builtin および pgstat_is_kind_custom を内部的に使用します。

関連データ構造

  • PgStat_KindInfo: 統計種別の振る舞いを定義する構造体。
  • pgstat_kind_builtin_infos: 組み込み統計種別の定義テーブル。
  • pgstat_kind_custom_infos: カスタム統計種別の定義を保持する配列。

失敗モードと境界条件

  • 未定義の kind が渡された場合、またはカスタム統計種別が登録されていないインデックスが指定された場合は NULL を返します。呼び出し元はこの戻り値が NULL でないことを前提とする箇所（Assert 等）があるため、注意が必要です。

保守担当者向けメモ

  • 本関数は統計システムの基盤であり、新しい統計種別を追加する際は、適切なテーブルへの登録が本関数の動作に直結します。
