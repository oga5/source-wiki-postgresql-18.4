bbsink_shell_begin_archive

定義

  • 種別: function
  • 定義箇所: contrib/basebackup_to_shell/basebackup_to_shell.c:309

目的

バックアップアーカイブの開始イベントを処理し、外部シェルコマンドの実行と後続のシンクへの通知を行う。

シグネチャまたは形状

static void bbsink_shell_begin_archive(bbsink *sink, const char *archive_name)

入力と出力

  • sink: bbsink 構造体へのポインタ。bbsink_shell 型としてキャストして使用される。
  • archive_name: 開始するアーカイブの名前。
  • 出力: なし（戻り値なし）。

前提条件と不変条件

  • sink は bbsink_shell 構造体の一部として初期化されている必要がある。
  • shell_run_command が内部で呼び出されるため、GUCパラメータ等で設定された外部コマンドが実行可能な状態であること。

アルゴリズムまたは振る舞い

1. 引数の sink を bbsink_shell 型にキャストする。 2. shell_run_command を呼び出し、指定された archive_name を用いて外部プロセスを起動する。 3. bbsink_forward_begin_archive を呼び出し、バックアップストリームの次のシンクへアーカイブ開始イベントを転送する。

呼び出し元と呼び出し先

  • 呼び出し元: bbsink フレームワークのイベントループ（bbsink_shell_ops 構造体を通じて参照される）。
  • 呼び出し先:
  • shell_run_command
  • bbsink_forward_begin_archive

関連データ構造

  • bbsink: バックアップターゲットの基底構造体。
  • bbsink_shell: 本モジュール固有の拡張構造体。

失敗モードと境界条件

  • shell_run_command 内で外部コマンドの実行に失敗した場合、バックアップ処理全体がエラーとして扱われる可能性がある。

保守担当者向けメモ

  • 本関数は bbsink インターフェースの一部であり、bbsink_shell_ops 構造体で関数ポインタとして登録されている。
  • 外部コマンドの実行結果（終了ステータス）の検証は、対となる bbsink_shell_end_archive で行われる。
