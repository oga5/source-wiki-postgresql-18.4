query_contains_extern_params

定義

  • 種別: function
  • 定義箇所: src/backend/parser/parse_param.c:330

目的

与えられたクエリツリー内に外部パラメータ（Param ノード）が含まれているかを判定します。

シグネチャまたは形状

bool query_contains_extern_params(Query *query)

入力と出力

  • **入力**: Query *query （解析対象のクエリツリー）
  • **出力**: bool （外部パラメータが見つかった場合は true、それ以外は false）

前提条件と不変条件

  • 入力される query は有効な Query 構造体へのポインタである必要があります。

アルゴリズムまたは振る舞い

query_tree_walker を使用してクエリツリー全体を走査します。走査の過程で query_contains_extern_params_walker を呼び出し、外部パラメータの存在を探索します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/parser/analyze.c:3215
  • **呼び出し先**: query_tree_walker（標準ライブラリ関数）、query_contains_extern_params_walker（内部ヘルパー関数）

関連データ構造

  • Query: PostgreSQLのクエリツリーを表現する主要な構造体。

失敗モードと境界条件

  • query が NULL の場合の挙動については、query_tree_walker の仕様に依存します。

保守担当者向けメモ

  • 本関数は単なるラッパーであり、実際の探索ロジックは query_contains_extern_params_walker に委譲されています。探索ロジックを変更する場合は、当該ウォーカー関数を確認してください。
