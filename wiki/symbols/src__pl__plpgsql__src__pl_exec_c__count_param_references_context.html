count_param_references_context

定義

  • 種別: struct
  • 定義箇所: src/pl/plpgsql/src/pl_exec.c:256

目的

式ツリー内における特定のパラメータ（Param）の出現回数をカウントし、最後に参照されたパラメータノードを追跡するためのコンテキスト構造体です。

シグネチャまたは形状

typedef struct count_param_references_context
{
    int         paramid;        /* 検索対象のパラメータID */
    int         count;          /* 検出された参照回数 */
    Param      *last_param;     /* 最後に検出されたParamノードへのポインタ */
} count_param_references_context;

入力と出力

  • **入力**: 検索対象の paramid を設定して初期化します。
  • **出力**: 走査完了後、count に総参照数が、last_param に最後に参照されたノードが格納されます。

前提条件と不変条件

  • 走査対象のノードツリーが有効であること。
  • count は 0 から開始されることが想定されます。

アルゴリズムまたは振る舞い

この構造体は、count_param_references 関数（src/pl/plpgsql/src/pl_exec.c:8531）の引数として渡され、ツリー走査中の状態保持に使用されます。走査中に特定の paramid と一致する Param ノードが見つかるたびに、count がインクリメントされ、last_param が更新されます。

呼び出し元と呼び出し先

  • **呼び出し元**: src/pl/plpgsql/src/pl_exec.c:8426 にてローカル変数としてインスタンス化されています。
  • **呼び出し先**: count_param_references 関数によって参照・更新されます。

関連データ構造

  • Param: PostgreSQL のノード構造体。式ツリー内のパラメータ参照を表します。

失敗モードと境界条件

  • 該当する paramid がツリー内に存在しない場合、count は 0 のままとなり、last_param は NULL となります。

保守担当者向けメモ

  • 本構造体は count_param_references 関数専用の補助的な状態保持用です。他の関数での再利用は想定されていません。
