SH_STATUS

定義

  • 種別: enum
  • 定義箇所: src/include/lib/simplehash.h:175

目的

simplehash.h によって生成されるハッシュテーブルにおいて、各バケットが現在使用中であるか、空であるかを識別するために使用されます。

シグネチャまたは形状

typedef enum SH_STATUS
{
	SH_STATUS_EMPTY = 0x00,
	SH_STATUS_IN_USE = 0x01
} SH_STATUS;

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: SH_STATUS_EMPTY (0x00) または SH_STATUS_IN_USE (0x01)

前提条件と不変条件

  • ハッシュテーブルの各バケットは、初期状態では SH_STATUS_EMPTY である必要があります。
  • 要素が挿入されると、対応するバケットの状態は SH_STATUS_IN_USE に遷移します。

アルゴリズムまたは振る舞い

  • 本列挙型は、オープンアドレス法ハッシュテーブルにおけるバケットの占有状態を管理します。
  • 検索や挿入の際、この状態を確認することで、そのバケットが有効なデータを含んでいるかを判定します。

呼び出し元と呼び出し先

  • 呼び出し元: simplehash.h 内で生成されるハッシュテーブル操作関数（SH_INSERT, SH_LOOKUP 等）が、バケットの状態確認のために参照します。
  • 呼び出し先: なし（列挙型のため）。

関連データ構造

  • SH_TYPE: 本列挙型を利用してバケット状態を管理するハッシュテーブル構造体。

失敗モードと境界条件

  • 該当なし。

保守担当者向けメモ

  • simplehash.h はマクロによるテンプレート生成を行うため、SH_STATUS は生成される各ハッシュテーブル型ごとに SH_MAKE_NAME(status) を通じて名前が解決されます。
  • ソースコード内では SH_STATUS マクロが定義・利用され、最終的に #undef されるため、利用時は生成されるコンテキストに注意してください。
