get_len_position

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/rangetypes_selfuncs.c:761

目的

ヒストグラムのビン内における特定の値の相対的な位置（0.0 から 1.0 の間の割合）を算出します。主に範囲型やマルチレンジ型の長さヒストグラムを用いた選択率推定に使用されます。

シグネチャまたは形状

static double get_len_position(double value, double hist1, double hist2)

入力と出力

  • **入力**:
  • value: 位置を特定したい対象の値。
  • hist1: ビンの下限値。
  • hist2: ビンの上限値。
  • **出力**:
  • double: ビン内での相対位置。通常は [0.0, 1.0] の範囲の値。

前提条件と不変条件

  • hist1 と hist2 はヒストグラムの境界値です。
  • value が有限値であり、かつ hist1 と hist2 も有限である場合、1.0 - (hist2 - value) / (hist2 - hist1) によって線形補間された位置が返されます。

アルゴリズムまたは振る舞い

1. **有限値の場合**: hist1 と hist2 が共に有限であれば、線形補間により位置を計算します。value が無限大の場合は 0.5 を返します。 2. **片方が無限大の場合**: hist1 が -inf で hist2 が有限であれば 1.0 を返します。 3. **両方が無限大の場合**: hist1 と hist2 が共に無限大であれば 0.0 を返します。 4. **その他の無限大の組み合わせ**: 両方の境界が無限大であるとみなされる場合は 0.5 を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/rangetypes_selfuncs.c (複数箇所)
  • src/backend/utils/adt/multirangetypes_selfuncs.c (複数箇所)
  • **呼び出し先**:
  • math.h の isinf マクロ。

関連データ構造

  • pg_statistic: 統計情報の格納に使用されるカタログ。本関数はここから取得されたヒストグラム値（length_hist_values 等）を引数として受け取ります。

失敗モードと境界条件

  • hist1 == hist2 の場合、除算によるゼロ除算が発生する可能性がありますが、コード内の条件分岐により無限大のケースが優先的に処理されます。
  • 不正なヒストグラム値が渡された場合でも、クラッシュを避けるために Assert を使用せず、デフォルト値（0.5）を返す設計になっています。

保守担当者向けメモ

  • 本関数は「範囲の長さ」と「下限値」が独立しているという仮定に基づいています。
  • multirangetypes_selfuncs.c と rangetypes_selfuncs.c の両方から参照されているため、変更時は両方のモジュールへの影響を考慮してください。
