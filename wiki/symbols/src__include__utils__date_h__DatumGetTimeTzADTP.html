DatumGetTimeTzADTP

定義

  • 種別: static inline 関数
  • 定義箇所: src/include/utils/date.h:65

目的

PostgreSQLの内部データ表現である Datum 型から、タイムゾーン付き時刻型 TimeTzADT へのポインタを安全に抽出します。

シグネチャまたは形状

static inline TimeTzADT *DatumGetTimeTzADTP(Datum X)

入力と出力

  • **入力**: Datum X（TimeTzADT 構造体を指すポインタを保持する Datum 値）
  • **出力**: TimeTzADT *（抽出された構造体へのポインタ）

前提条件と不変条件

  • 入力 Datum X は、有効な TimeTzADT 構造体を指すポインタとして格納されている必要があります。
  • 本関数は DatumGetPointer マクロを利用して型キャストを行うため、Datum がポインタ型として構築されていることが前提となります。

アルゴリズムまたは振る舞い

1. 引数 X を DatumGetPointer マクロに渡します。 2. 返されたポインタを TimeTzADT * 型にキャストして返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/btree_gist/btree_time.c
  • src/backend/utils/adt/json.c
  • src/backend/utils/adt/jsonpath_exec.c
  • src/backend/utils/adt/selfuncs.c
  • src/include/utils/date.h（PG_GETARG_TIMETZADT_P マクロ内）
  • **呼び出し先**:
  • DatumGetPointer（PostgreSQLのコア基盤マクロ）

関連データ構造

  • TimeTzADT: 時刻とタイムゾーンオフセットを保持する構造体。
  • Datum: PostgreSQLの汎用データ表現型。

失敗モードと境界条件

  • Datum が有効なポインタを保持していない場合、不正なメモリアドレスへのアクセスが発生する可能性があります。本関数自体には検証ロジックは含まれていません。

保守担当者向けメモ

  • 本関数はインライン化されており、パフォーマンスが重視されるパスで使用されます。変更を加える際は、Datum のポインタ変換規則に準拠していることを確認してください。
