pgstat_drop_database

定義

  • 種別: function
  • 定義箇所: src/backend/utils/activity/pgstat_database.c:43

目的

データベース削除時に、当該データベースに関連付けられた統計情報を破棄する。

シグネチャまたは形状

void pgstat_drop_database(Oid databaseid)

入力と出力

  • **入力**: databaseid (Oid) - 削除対象のデータベース識別子。
  • **出力**: なし。

前提条件と不変条件

  • 統計サブシステムが初期化されていること。
  • データベースの削除処理の一環として呼び出されることを想定している。

アルゴリズムまたは振る舞い

1. PGSTAT_KIND_DATABASE（データベース種別）を指定する。 2. pgstat_drop_transactional を呼び出し、指定された databaseid に紐づく統計情報をトランザクション的に破棄する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/commands/dbcommands.c:1644
  • src/backend/commands/dbcommands.c:1816
  • **呼び出し先**:
  • pgstat_drop_transactional（統計サブシステムの内部関数）

関連データ構造

  • Oid: データベース識別子として使用。
  • PGSTAT_KIND_DATABASE: 統計種別を識別するための定数。

失敗モードと境界条件

  • 存在しない databaseid が渡された場合の挙動については、本ソース抜粋からは判断できない。

保守担当者向けメモ

  • 本関数はデータベース単位の統計破棄を担当する。テーブルやインデックスなど、他のオブジェクトの統計破棄については pgstat_relation.c 等の関連モジュールを確認すること。
