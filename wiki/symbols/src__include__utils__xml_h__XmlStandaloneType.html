XmlStandaloneType

定義

  • 種別: enum
  • 定義箇所: src/include/utils/xml.h:25

目的

XML宣言における standalone 属性の状態を表現するための列挙型です。

シグネチャまたは形状

typedef enum
{
    XML_STANDALONE_YES,
    XML_STANDALONE_NO,
    XML_STANDALONE_NO_VALUE,
    XML_STANDALONE_OMITTED,
} XmlStandaloneType;

入力と出力

  • 入力: なし（列挙定数）
  • 出力: なし（列挙定数）

前提条件と不変条件

  • XML処理において、ドキュメントの独立性（standalone）を識別するために使用されます。

アルゴリズムまたは振る舞い

  • XML_STANDALONE_YES: standalone="yes" を示す。
  • XML_STANDALONE_NO: standalone="no" を示す。
  • XML_STANDALONE_NO_VALUE: 値が指定されていない（standalone のみ）状態を示す。
  • XML_STANDALONE_OMITTED: 属性自体が省略されている状態を示す。

呼び出し元と呼び出し先

  • src/backend/utils/adt/xml.c: 本列挙型を利用してXMLのパースや生成を行う実装ファイル。

失敗モードと境界条件
