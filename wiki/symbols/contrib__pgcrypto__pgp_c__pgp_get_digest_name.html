pgp_get_digest_name

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/pgp.c:125

目的

内部的なハッシュアルゴリズム識別子（整数コード）から、対応するアルゴリズムの名称（文字列）を取得します。

シグネチャまたは形状

const char *pgp_get_digest_name(int code);

入力と出力

  • **入力**: code (int) - 検索対象のハッシュアルゴリズム識別子。
  • **出力**: const char * - アルゴリズム名を表す文字列。該当するコードが見つからない場合は NULL を返します。

前提条件と不変条件

  • digest_list が有効な digest_info 構造体の配列であり、name メンバが NULL で終わる終端子を持つ配列であることを前提とします。

アルゴリズムまたは振る舞い

1. 静的に定義された digest_list 配列を先頭から順に走査します。 2. 各要素の code メンバが引数の code と一致するか確認します。 3. 一致した要素が見つかった場合、その name メンバを返します。 4. digest_list の終端（name が NULL の要素）に達するまで一致するものがない場合、NULL を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/pgcrypto/pgp.c 内のコードから参照されています。
  • **呼び出し先**: 外部関数は呼び出しません。digest_list 配列を参照するのみです。

関連データ構造

  • digest_info: ハッシュアルゴリズムのメタデータ（コードと名称）を保持する構造体。

失敗モードと境界条件

  • 指定された code に対応するアルゴリズムが digest_list に存在しない場合、NULL が返されます。呼び出し元はこの戻り値が NULL である可能性を考慮する必要があります。

保守担当者向けメモ

  • 本関数は digest_list の内容に完全に依存しています。サポートするハッシュアルゴリズムを追加・削除する場合は、digest_list の定義を更新してください。
