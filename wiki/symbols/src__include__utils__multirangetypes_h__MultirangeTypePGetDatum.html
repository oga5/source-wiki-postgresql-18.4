MultirangeTypePGetDatum

定義

  • 種別: static inline function
  • 定義箇所: src/include/utils/multirangetypes.h:59

目的

MultirangeType 構造体へのポインタを、PostgreSQL のデータ処理基盤である Datum 型に変換する。

シグネチャまたは形状

static inline Datum MultirangeTypePGetDatum(const MultirangeType *X)

入力と出力

  • **入力**: const MultirangeType *X （マルチレンジ型のメモリ表現へのポインタ）
  • **出力**: Datum （PostgreSQL の汎用データ表現）

前提条件と不変条件

  • 入力ポインタ X は有効な MultirangeType 構造体を指している必要がある。
  • MultirangeType は可変長データ（varlena）として扱われる。

アルゴリズムまたは振る舞い

PointerGetDatum(X) を呼び出し、渡されたポインタを Datum 型へキャストして返却する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/include/utils/multirangetypes.h:67 に定義されたマクロ PG_RETURN_MULTIRANGE_P(x) から呼び出される。
  • **呼び出し先**:
  • PointerGetDatum （PostgreSQL のコアマクロ）

関連データ構造

  • MultirangeType: マルチレンジ型のメモリ上の表現。
  • Datum: PostgreSQL における汎用的なデータ型。

失敗モードと境界条件

  • 入力ポインタが NULL の場合、PointerGetDatum の振る舞いに依存する。

保守担当者向けメモ

  • 本関数はインライン関数であり、マルチレンジ型を関数から返却する際の標準的な変換手段として使用される。
  • 構造体のメンバを直接操作するのではなく、提供されているアクセサやマクロ経由で扱うこと。
