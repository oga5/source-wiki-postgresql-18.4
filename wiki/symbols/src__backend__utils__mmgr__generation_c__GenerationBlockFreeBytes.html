GenerationBlockFreeBytes

定義

  • 種別: static inline function
  • 定義箇所: src/backend/utils/mmgr/generation.c:653

目的

指定された GenerationBlock 内で、現在割り当て可能な残りのバイト数を算出します。

シグネチャまたは形状

static inline Size GenerationBlockFreeBytes(GenerationBlock *block)

入力と出力

  • **入力**: GenerationBlock *block （対象となるメモリブロックへのポインタ）
  • **出力**: Size （ブロック内の空き領域のバイト数）

前提条件と不変条件

  • 引数 block は有効な GenerationBlock 構造体を指している必要があります。
  • block->endptr はブロックの末尾アドレスを、block->freeptr は次に割り当て可能なアドレスを指しているものとします。

アルゴリズムまたは振る舞い

ブロックの末尾アドレス (endptr) から現在の空き領域開始アドレス (freeptr) を減算することで、利用可能な残存バイト数を計算して返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/mmgr/generation.c:569 （GenerationAlloc 内でブロックの容量不足判定に使用）
  • src/backend/utils/mmgr/generation.c:578 （GenerationAlloc 内で再利用可能な freeblock の容量判定に使用）
  • **呼び出し先**: なし（算術演算のみを実行）

関連データ構造

  • GenerationBlock: メモリ管理の単位となる構造体。endptr および freeptr を保持します。

失敗モードと境界条件

  • block が NULL の場合、未定義の動作を引き起こします。
  • freeptr が endptr を超えている場合、負の値を返しますが、本コンテキストの設計上、そのような状態は想定されていません。

保守担当者向けメモ

  • 本関数は頻繁に呼び出されるため、static inline として定義されています。ロジックを変更する際は、パフォーマンスへの影響を考慮してください。
