fasthash_reduce32

定義

  • 種別: function
  • 定義箇所: src/include/common/hashfn_unstable.h:336

目的

64ビットのハッシュ値を32ビットに縮小します。上位ビットと下位ビットの両方の情報を保持するフェルマー剰余（Fermat residue）への変換を行います。

シグネチャまたは形状

static inline uint32 fasthash_reduce32(uint64 h)

入力と出力

  • **入力**: uint64 h （縮小対象の64ビットハッシュ値）
  • **出力**: uint32 （計算結果の32ビットハッシュ値）

前提条件と不変条件

  • 本関数は fasthash アルゴリズムの一部であり、プラットフォーム依存のエンディアンの影響を受けます。
  • 永続化（ディスク保存）を目的としたハッシュ計算には使用できません。

アルゴリズムまたは振る舞い

入力された64ビット値 h に対して、h - (h >> 32) を計算します。これにより、元の64ビット値の上位32ビットと下位32ビットの差分をとり、情報を圧縮します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/include/common/hashfn_unstable.h 内の fasthash_final64 の戻り値処理
  • src/include/common/hashfn_unstable.h 内の fasthash64 の戻り値処理
  • **呼び出し先**: なし（インライン関数として定義）

関連データ構造

  • なし

失敗モードと境界条件

  • 入力値が uint64 の範囲内であれば常に計算可能であり、特筆すべき失敗モードはありません。

保守担当者向けメモ

  • 本関数は「不安定」なハッシュ実装の一部です。インデックスやディスク上の構造体には絶対に使用しないでください。
  • 変更を加える際は、ハッシュの衝突耐性や分布特性への影響を考慮してください。
