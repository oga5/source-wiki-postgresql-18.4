r_noun

目的

ロシア語のステミング処理において、単語末尾の名詞接尾辞を特定し、削除することで語幹を抽出する。

シグネチャまたは形状

static int r_noun(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（Snowballステミング実行環境）
  • **出力**: int 型。接尾辞が削除された場合は 1 を返し、マッチしなかった場合は 0 を返す。エラー発生時は負の値を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体へのポインタであること。
  • find_among_b を使用するため、単語の末尾から前方に向かって検索が行われる。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を z->ket（削除対象の末尾）に設定する。 2. a_5 配列（36個のパターン）を用いて、後方検索 find_among_b を実行する。 3. マッチした場合、その位置を z->bra（削除対象の先頭）に設定する。 4. slice_del(z) を呼び出し、該当する接尾辞を削除する。 5. 削除処理の結果を呼び出し元に返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_UTF_8_russian.c 内のメイン処理フローから呼び出される。
  • **呼び出し先**: find_among_b（パターンマッチング）、slice_del（文字列削除）。

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_5: 名詞接尾辞のパターンを定義した among 構造体の配列。

失敗モードと境界条件

  • find_among_b がマッチしない場合、何も削除せずに 0 を返す。
  • slice_del が内部エラー（負の値）を返した場合、その値をそのまま伝播させる。
