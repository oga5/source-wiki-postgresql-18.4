CreateVariableSpace

定義

  • 種別: function
  • 定義箇所: src/bin/psql/variables.c:52

目的

psql の変数管理システムで使用される、空の変数リスト（VariableSpace）を初期化してメモリを割り当てる。

シグネチャまたは形状

VariableSpace CreateVariableSpace(void);

入力と出力

  • **入力**: なし
  • **出力**: 新たに割り当てられた VariableSpace（内部構造体 struct _variable へのポインタ）を返す。

前提条件と不変条件

  • pg_malloc を使用してメモリを確保するため、メモリ不足時にはプロセスが終了する可能性がある。
  • 返される構造体は、名前、値、および各フック（substitute_hook, assign_hook）が NULL で初期化された状態である。

アルゴリズムまたは振る舞い

1. pg_malloc を呼び出し、struct _variable サイズのメモリを確保する。 2. 確保した領域の各メンバ（name, value, substitute_hook, assign_hook, next）を NULL に設定する。 3. 初期化済みのポインタを呼び出し元へ返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/bin/psql/startup.c:1220（pset.vars の初期化時に呼び出される）
  • **呼び出し先**: pg_malloc（メモリ確保のため）

関連データ構造

  • VariableSpace: struct _variable 型のエイリアス。変数リストのヘッダーとして機能する。

失敗モードと境界条件

  • メモリ不足時には pg_malloc がエラーを発生させ、プログラムが異常終了する。

保守担当者向けメモ

  • 本関数はリストの先頭（ヘッダー）を作成するものであり、実際の変数データは後続の SetVariable 等によって追加される。
  • VariableSpace は単方向連結リストの起点となるため、メモリリークを防ぐために適切な破棄処理が必要となる。
