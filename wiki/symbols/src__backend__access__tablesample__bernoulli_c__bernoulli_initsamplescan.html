bernoulli_initsamplescan

定義

  • 種別: function
  • 定義箇所: src/backend/access/tablesample/bernoulli.c:126

目的

TABLESAMPLE BERNOULLI メソッドにおけるスキャン状態の初期化を行います。

シグネチャまたは形状

static void bernoulli_initsamplescan(SampleScanState *node, int eflags)

入力と出力

  • **入力**:
  • node: SampleScanState 構造体へのポインタ。
  • eflags: スキャンの実行フラグ（本関数内では未使用）。
  • **出力**: なし。node->tsm_state にメモリを割り当てます。

前提条件と不変条件

  • node が有効な SampleScanState を指している必要があります。
  • 実行後、node->tsm_state は BernoulliSamplerData 構造体として初期化（ゼロクリア）されます。

アルゴリズムまたは振る舞い

1. palloc0 を使用して BernoulliSamplerData サイズのメモリを動的に確保します。 2. 確保したメモリ領域を node->tsm_state に格納します。

呼び出し元と呼び出し先

  • **呼び出し元**: TsmRoutine 構造体の InitSampleScan フィールドを通じて、クエリ実行エンジンから呼び出されます（src/backend/access/tablesample/bernoulli.c:73 に登録あり）。
  • **呼び出し先**: palloc0（PostgreSQL メモリ管理関数）。

関連データ構造

  • SampleScanState: スキャン全体の実行状態を保持する構造体。
  • BernoulliSamplerData: 本関数によって初期化される、ベルヌーイサンプリング固有の状態保持用構造体。

失敗モードと境界条件

  • メモリ不足時には palloc0 がエラーを発生させ、処理が中断されます。

保守担当者向けメモ

  • 本関数は TsmRoutine インターフェースの一部です。tsm_bernoulli_handler を通じて登録されるため、インターフェースの変更時には注意してください。
