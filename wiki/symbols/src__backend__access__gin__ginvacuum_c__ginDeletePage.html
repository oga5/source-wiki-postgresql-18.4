ginDeletePage

定義

  • 種別: function
  • 定義箇所: src/backend/access/gin/ginvacuum.c:128

目的

GINインデックスのポストティングツリーにおいて、不要となったページを物理的に削除し、ツリー構造（親ノードのダウンリンクおよび左隣接ページの右リンク）を整合させる。

シグネチャまたは形状

static void ginDeletePage(GinVacuumState *gvs, BlockNumber deleteBlkno, BlockNumber leftBlkno,
                          BlockNumber parentBlkno, OffsetNumber myoff, bool isParentRoot)

入力と出力

  • **入力**:
  • gvs: バキューム処理の状態を保持する GinVacuumState 構造体。
  • deleteBlkno: 削除対象のブロック番号。
  • leftBlkno: 削除対象の左隣接ブロック番号。
  • parentBlkno: 削除対象の親ブロック番号。
  • myoff: 親ページ内での削除対象へのオフセット。
  • isParentRoot: 親がルートであるかを示すフラグ。
  • **出力**: なし（副作用としてインデックスページおよび統計情報を更新）。

前提条件と不変条件

  • 呼び出し元が、削除対象ページ、親ページ、および左隣接ページに対して排他的なクリーンアップロックを保持していること。
  • このロックにより、対象サブツリーへの挿入が発生しないことが保証されていること。

アルゴリズムまたは振る舞い

1. 指定された3つのページ（削除対象、左隣接、親）をバッファに読み込む。 2. PredicateLockPageCombine を呼び出し、削除対象ページへの挿入が右隣接ページへ転送されるよう調整する。 3. クリティカルセクションを開始する。 4. 左隣接ページの rightlink を削除対象ページの rightlink に書き換え、リンクを繋ぎ変える。 5. 親ページから削除対象ページへのダウンリンク（PostingItem）を削除する。 6. 削除対象ページに GinPageSetDeleted を適用し、削除時のトランザクションIDを記録する。 7. WALが必要な場合、ginxlogDeletePage を記録し、各ページのLSNを更新する。 8. バッファをダーティとしてマークし、解放する。 9. gvs->result の削除ページ統計をインクリメントする。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/access/gin/ginvacuum.c 内の ginScanToDelete から呼び出される。
  • **呼び出し先**: ReadBufferExtended, BufferGetPage, GinPageGetOpaque, PredicateLockPageCombine, START_CRIT_SECTION, GinPageDeletePostingItem, GinPageSetDeleted, GinPageSetDeleteXid, ReadNextTransactionId, MarkBufferDirty, RelationNeedsWAL, XLogBeginInsert, XLogRegisterBuffer, XLogRegisterData, XLogInsert, PageSetLSN, ReleaseBuffer, END_CRIT_SECTION。

関連データ構造

  • GinVacuumState: バキュームの実行状態と統計情報を保持。
  • ginxlogDeletePage: WALログ記録用の構造体。
  • PostingItem: 親ページ内のダウンリンク情報。

失敗モードと境界条件

  • 削除対象ページが他のトランザクションから参照されている場合、前提条件のロックが取得できない可能性がある。
  • WAL記録時、古いバージョンからのバイナリアップグレード等で pd_lower が正しく設定されていないページに対する処理に注意が必要（ソース内のコメント参照）。

保守担当者向けメモ

  • 本関数は、親ページが排他的クリーンアップロックを保持している状況でのみ実行される。
  • 削除対象ページの rightlink フィールドは、検索スキャンが正しく動作するように変更してはならない。
  • isParentRoot 引数はシグネチャに含まれているが、提供されたソース内では直接使用されていない。
