r_attached_pronoun

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_catalan.c:1281

目的

単語の末尾に付着した代名詞を検出し、R1領域内にある場合にそれを削除する。

シグネチャまたは形状

static int r_attached_pronoun(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z（ステミング環境の状態を保持する構造体）
  • 出力: int（成功時に 1、失敗または条件不一致時に 0、エラー時に負の値を返す）

前提条件と不変条件

  • z->c は現在処理中の単語内の位置を指す。
  • find_among_b を使用するため、後方検索が可能な状態であること。
  • 削除対象の代名詞は a_1 配列（39個の要素）に定義されている必要がある。

アルゴリズムまたは振る舞い

1. z->ket を現在位置 z->c に設定する。 2. 文字列の境界チェックおよび、特定の文字コード範囲（p[z->c - 1]）に基づく高速な事前フィルタリングを行う。 3. find_among_b(z, a_1, 39) を呼び出し、末尾に代名詞が存在するか確認する。 4. 存在する場合、z->bra を現在位置に設定する。 5. r_R1(z) を呼び出し、対象がR1領域内にあるか検証する。 6. 検証に成功した場合、slice_del(z) で該当部分を削除する。

呼び出し元と呼び出し先

  • 呼び出し元: catalan_UTF_8_stem（直接の呼び出し関係はソースコードの制御フローに基づく）。
  • 呼び出し先: find_among_b, r_R1, slice_del。

関連データ構造

  • struct SN_env: ステミング環境を管理する構造体。
  • a_1: 代名詞の接尾辞パターンを格納した among 構造体の配列。

失敗モードと境界条件

  • 文字列の先頭に達している場合や、R1領域外である場合は削除を行わず 0 を返す。
  • slice_del が内部エラー（負の値）を返した場合、その値をそのまま伝播させる。
