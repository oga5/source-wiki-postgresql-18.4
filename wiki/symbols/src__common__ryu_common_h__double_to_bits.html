double_to_bits

定義

  • 種別: function
  • 定義箇所: src/common/ryu_common.h:124

目的

double 型の浮動小数点数を、その内部ビット表現である uint64 型へ安全にキャスト（再解釈）します。

シグネチャまたは形状

static inline uint64 double_to_bits(const double d)

入力と出力

  • **入力**: d (const double) - ビット表現に変換する対象の浮動小数点数。
  • **出力**: uint64 - d のメモリ上のビットパターンを保持する 64 ビット符号なし整数。

前提条件と不変条件

  • sizeof(double) が sizeof(uint64) と等しい環境を想定しています。
  • memcpy を使用しているため、厳密なエイリアシング規則に抵触することなく安全にビット変換が行われます。

アルゴリズムまたは振る舞い

1. uint64 型の変数 bits を 0 で初期化します。 2. memcpy を用いて、d のメモリ領域から bits のメモリ領域へ sizeof(double) バイト分をコピーします。 3. コピーされたビットパターンを保持する bits を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/common/d2s.c:1021 にて呼び出しを確認しています。
  • **呼び出し先**: 標準ライブラリ関数 memcpy を使用しています。

関連データ構造

  • uint64: PostgreSQL の基本型定義（c.h 経由）に基づく 64 ビット符号なし整数。

失敗モードと境界条件

  • 特殊な浮動小数点値（NaN, Infinity, 0.0 など）が入力された場合でも、それらのビットパターンをそのまま uint64 として返します。エラー処理は行われません。

保守担当者向けメモ

  • 本関数は static inline で定義されており、ヘッダーインクルード時にインライン展開されます。
  • プラットフォーム間の浮動小数点表現が IEEE 754 に準拠していることを前提としています。
