AttInMetadata

定義

  • 種別: struct
  • 定義箇所: src/include/funcapi.h:35

目的

C 文字列からタプルを構築する際に必要な型情報および変換関数を保持し、データ変換処理を支援するために使用されます。

シグネチャまたは形状

typedef struct AttInMetadata
{
    TupleDesc   tupdesc;
    FmgrInfo   *attinfuncs;
    Oid        *attioparams;
    int32      *atttypmods;
} AttInMetadata;

入力と出力

本構造体は、タプル構築に必要なメタデータを保持するコンテナです。

  • **入力**: 構築対象のタプル記述子（TupleDesc）および各属性の型情報。
  • **出力**: 構築されたタプルを生成するための変換関数情報（FmgrInfo）および型パラメータ。

前提条件と不変条件

  • tupdesc は有効な TupleDesc である必要があります。
  • attinfuncs、attioparams、atttypmods は、tupdesc が持つ属性数に対応する配列である必要があります。

アルゴリズムまたは振る舞い

C 文字列の配列からタプルを構築する際、この構造体内の attinfuncs を参照して各属性の入力関数を特定し、attioparams や atttypmods を用いて型変換を実行します。

呼び出し元と呼び出し先

  • **呼び出し元（直接的な参照の証拠）**:
  • contrib/dblink/dblink.c
  • contrib/pgcrypto/pgp-pgsql.c
  • contrib/pgrowlocks/pgrowlocks.c
  • contrib/pgstattuple/pgstattuple.c
  • contrib/postgres_fdw/postgres_fdw.c
  • **呼び出し先**:
  • 本構造体自体はデータ保持用であり、具体的な関数呼び出し先は本ソース excerpt には含まれていません。

関連データ構造

  • TupleDesc: タプルの構造を記述するデータ構造。
  • FmgrInfo: 関数呼び出しのための情報構造体。

失敗モードと境界条件

  • 属性数と配列のサイズが一致しない場合、メモリ破壊や不正なアクセスが発生する可能性があります。
  • tupdesc が解放された後に本構造体を参照すると、ダングリングポインタが発生します。

保守担当者向けメモ

  • 本構造体は BuildTupleFromCStrings() 等の関数で利用されることを想定しています。
  • 拡張モジュール（contrib 配下）で頻繁に利用されているため、変更時は互換性に注意してください。
