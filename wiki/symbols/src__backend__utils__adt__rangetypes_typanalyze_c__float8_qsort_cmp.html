float8_qsort_cmp

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/rangetypes_typanalyze.c:94

目的

float8 型の値のペアを比較し、qsort 関数によるソート順序を決定します。

シグネチャまたは形状

static int float8_qsort_cmp(const void *a1, const void *a2, void *arg)

入力と出力

  • **入力**:
  • a1, a2: 比較対象の float8 値へのポインタ。
  • arg: 使用されない引数（NULL が渡される）。
  • **出力**:
  • *a1 < *a2 の場合: -1
  • *a1 == *a2 の場合: 0
  • *a1 > *a2 の場合: 1

前提条件と不変条件

  • a1 および a2 は、有効な float8 型のメモリ領域を指している必要があります。

アルゴリズムまたは振る舞い

1. a1 と a2 を float8 型のポインタとしてキャストします。 2. 指し示す値を比較し、標準的な比較結果（-1, 0, 1）を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/adt/rangetypes_typanalyze.c 内の統計処理ロジック（compute_range_stats 等）から、qsort の比較関数として参照されています。
  • **呼び出し先**: 呼び出し先に関する情報は提供されていません。

関連データ構造

  • float8: PostgreSQL の 8 バイト浮動小数点型。

失敗モードと境界条件

  • a1 または a2 が NULL ポインタである場合、デリファレンスにより未定義の動作を引き起こします。

保守担当者向けメモ

  • 本関数は qsort インターフェースに適合するように設計されています。
  • arg 引数は qsort のシグネチャに合わせるために存在しますが、本関数内では使用されていません。
