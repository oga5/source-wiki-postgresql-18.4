get_notclausearg

定義

  • 種別: static inline function
  • 定義箇所: src/include/nodes/nodeFuncs.h:135

目的

NOT 演算子を表すノードから、その引数（否定対象の式）を抽出します。

シグネチャまたは形状

static inline Expr *get_notclausearg(const void *notclause)

入力と出力

  • **入力**: notclause (const void *) - BoolExpr 構造体（NOT 演算子）へのポインタ。
  • **出力**: Expr * - NOT 演算子の引数リストの先頭要素。

前提条件と不変条件

  • 入力 notclause は、BoolExpr 構造体としてキャスト可能である必要があります。
  • notclause が指す BoolExpr の args リストは空であってはなりません（linitial を使用するため）。

アルゴリズムまたは振る舞い

1. 引数 notclause を BoolExpr * 型にキャストします。 2. BoolExpr 構造体の args フィールド（List）にアクセスします。 3. linitial マクロを使用して、リストの先頭要素を取得します。 4. 取得した要素を Expr * 型にキャストして返します。

  • **呼び出し先**:
  • linitial (リスト操作マクロ)

関連データ構造

  • BoolExpr: NOT 演算子を表現するノード構造体。
  • Expr: 式ノードの基底型。
  • List: 引数を保持するリスト構造。

失敗モードと境界条件

  • notclause が NULL の場合、または BoolExpr 構造体でない場合にアクセスするとセグメンテーション違反が発生します。
  • args リストが空の場合、linitial の内部実装によりアサーション失敗または未定義の動作を引き起こす可能性があります。

保守担当者向けメモ

  • 本関数はインライン化されており、パフォーマンスを重視しています。
  • BoolExpr の構造に依存しているため、parsenodes.h での BoolExpr 定義変更時には本関数の整合性を確認してください。
