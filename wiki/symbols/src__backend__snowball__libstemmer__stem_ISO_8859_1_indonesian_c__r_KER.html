目的

インドネシア語のステミング処理において、特定の文字パターン（母音に続く特定の2文字）を検証する条件判定を行う。

シグネチャまたは形状

static int r_KER(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（Snowballステミング環境構造体）
  • **出力**: int 型。条件を満たす場合は 1、満たさない場合は 0 を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • 内部で g_vowel（母音グループ）および s_0（特定の文字列定数）を参照する。

アルゴリズムまたは振る舞い

1. out_grouping を呼び出し、現在のカーソル位置の文字が母音（g_vowel、範囲 97-117）ではないことを確認する。母音であれば 0 を返して終了する。 2. eq_s を呼び出し、現在のカーソル位置から2文字が s_0 と一致するかを確認する。 3. 両方の条件を満たした場合にのみ 1 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_ISO_8859_1_indonesian.c および src/backend/snowball/libstemmer/stem_UTF_8_indonesian.c 内の among 構造体配列（a_4 関連）から参照されている。
  • **呼び出し先**: out_grouping および eq_s（Snowballランタイムのユーティリティ関数）。

関連データ構造

  • struct SN_env: ステミング処理のワークスペース。
  • among: パターンマッチングに使用される構造体。

失敗モードと境界条件

  • z が NULL の場合の挙動はランタイムの out_grouping および eq_s の実装に依存する。
  • 文字列の残りが2文字未満の場合、eq_s は一致しないと判定し 0 を返す。
