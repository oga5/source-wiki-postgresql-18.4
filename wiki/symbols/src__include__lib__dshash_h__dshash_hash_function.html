dshash_hash_function

定義

  • 種別: typedef
  • 定義箇所: src/include/lib/dshash.h:37

目的

dshash（動的共有ハッシュテーブル）において、格納するキーからハッシュ値を計算するためのコールバック関数型を定義します。

シグネチャまたは形状

typedef dshash_hash (*dshash_hash_function) (const void *v, size_t size, void *arg);

入力と出力

  • **入力**:
  • v: ハッシュ対象のキーデータへのポインタ。
  • size: キーデータのサイズ。
  • arg: ハッシュ関数に渡される追加のコンテキスト引数（dshash_parameters で指定されたもの）。
  • **出力**:
  • dshash_hash 型の計算されたハッシュ値。

前提条件と不変条件

  • 本関数はプロセス間で共有できないため、各プロセスが dshash_attach を呼び出す際に、改めて dshash_parameters を通じて再指定する必要があります。

アルゴリズムまたは振る舞い

  • 指定されたキーデータ v とそのサイズ size を基に、ハッシュテーブル内で使用される dshash_hash 値を生成します。
  • 内部的なハッシュテーブルのバケット選択やパーティション分割の計算に使用されます。

呼び出し元と呼び出し先

  • **呼び出し元**: dshash_parameters 構造体のメンバ hash_function として保持され、dshash の内部操作（検索、挿入、削除など）の過程で呼び出されます。
  • **呼び出し先**: 提供されたソース excerpt からは、具体的な実装関数や呼び出し先は特定できません。

関連データ構造

  • dshash_parameters: この関数ポインタを保持し、ハッシュテーブルの動作を定義します。
  • dshash_hash: 本関数が戻り値として返すハッシュ値の型。

失敗モードと境界条件

  • 呼び出し元で提供される関数実装が、v が指すメモリ領域の境界を超えてアクセスしないことが求められます。
  • 呼び出し元で提供される関数実装が、arg が指すコンテキストの有効性を保証する必要があります。

保守担当者向けメモ

  • この関数は dshash_parameters を介して設定されます。プロセス間で共有メモリを扱う性質上、関数ポインタの再設定が必須である点に注意してください。
