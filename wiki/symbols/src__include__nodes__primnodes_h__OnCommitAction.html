OnCommitAction

定義

  • 種別: enum
  • 定義箇所: src/include/nodes/primnodes.h:56

目的

トランザクション終了時（COMMIT時）に一時テーブルに対して実行すべきアクションを定義する列挙型です。

シグネチャまたは形状

typedef enum OnCommitAction
{
    ONCOMMIT_NOOP,          /* ON COMMIT 句なし（何もしない） */
    ONCOMMIT_PRESERVE_ROWS, /* ON COMMIT PRESERVE ROWS（何もしない） */
    ONCOMMIT_DELETE_ROWS,   /* ON COMMIT DELETE ROWS */
    ONCOMMIT_DROP,          /* ON COMMIT DROP */
} OnCommitAction;

入力と出力

  • 入力: なし（列挙定数）
  • 出力: なし

前提条件と不変条件

  • 一時テーブルの作成時や管理時に、トランザクション終了時の挙動を決定するために使用されます。

アルゴリズムまたは振る舞い

  • ONCOMMIT_NOOP: 明示的な指定がない場合のデフォルト動作。
  • ONCOMMIT_PRESERVE_ROWS: 行を保持する（実質的な処理は行わない）。
  • ONCOMMIT_DELETE_ROWS: トランザクション終了時にテーブル内の全行を削除する。
  • ONCOMMIT_DROP: トランザクション終了時にテーブル自体を削除する。
