indonesian_UTF_8_stem

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_indonesian.c:312

目的

インドネシア語の単語に対し、形態論に基づいた接辞（粒子、所有代名詞、接頭辞、接尾辞）の除去を行い、語幹を抽出する。

シグネチャまたは形状

int indonesian_UTF_8_stem(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（ステミング対象の単語バッファおよび状態管理用環境構造体）。
  • **出力**: int 型。成功時は 1、処理がスキップされた場合や失敗時は 0 または負の値を返す。

前提条件と不変条件

  • z->I[1] には単語内の母音数が格納される。
  • 母音数が 2 以下の単語はステミング対象外となり、即座に終了する。
  • 内部状態 z->I[0] は接辞除去の進行状況を追跡するために使用される。

アルゴリズムまたは振る舞い

1. **母音カウント**: 入力単語内の母音数をカウントし、z->I[1] に保存する。 2. **粒子・所有代名詞の除去**: r_remove_particle および r_remove_possessive_pronoun を呼び出し、該当する接辞を除去する。 3. **接頭辞・接尾辞の除去**:

  • 第一順位の接頭辞除去を試みる。成功した場合は、続けて接尾辞および第二順位の接頭辞除去を試みる。
  • 第一順位の接頭辞が除去できない場合は、第二順位の接頭辞除去を試みた後、接尾辞の除去を試みる。

    • 4. **状態更新**: slice_del 等のランタイム関数を介して、SN_env 内のバッファを直接更新する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_UTF_8_indonesian.c:8 および src/include/snowball/libstemmer/stem_UTF_8_indonesian.h:10 に宣言が存在する。具体的な呼び出し元コードの詳細は提供されていない。
  • **呼び出し先**: r_remove_particle, r_remove_possessive_pronoun, r_remove_first_order_prefix, r_remove_second_order_prefix, r_remove_suffix, out_grouping_U。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。I[] 配列（整数フラグ）、c（カーソル位置）、l（文字列長）などが含まれる。

失敗モードと境界条件

  • 母音数が 2 以下の単語は処理されず 0 を返す。
  • 内部関数（r_remove_*）が負の値を返した場合、その値をそのまま呼び出し元へ伝播させる。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードであるため、直接編集しないこと。ロジック変更は元の .sbl スクリプトに対して行う。
  • z->I[0] および z->I[1] の値は、ステミングのルール適用状態を制御する重要なフラグである。デバッグ時は among 配列の定義と照合すること。
