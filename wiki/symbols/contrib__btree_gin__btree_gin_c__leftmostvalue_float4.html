leftmostvalue_float4

定義

  • 種別: function
  • 定義箇所: contrib/btree_gin/btree_gin.c:233

目的

float4 型のデータにおいて、GINインデックスを用いた範囲検索（特に BTLess 系戦略）の際に、インデックスの走査を開始するための「最小値」を提供します。

シグネチャまたは形状

static Datum leftmostvalue_float4(void)

入力と出力

  • **入力**: なし。
  • **出力**: float4 型の負の無限大を表す Datum 値。

前提条件と不変条件

  • float4 型の演算において、負の無限大が最小値として扱われることを前提としています。

アルゴリズムまたは振る舞い

1. get_float4_infinity() を呼び出し、浮動小数点の無限大値を取得します。 2. 符号を反転させ、負の無限大を作成します。 3. Float4GetDatum() を介して、PostgreSQLの内部データ形式である Datum に変換して返します。

呼び出し元と呼び出し先

  • **呼び出し元**: GIN_SUPPORT(float4, false, leftmostvalue_float4, btfloat4cmp) マクロ内で参照されています。
  • **呼び出し先**: get_float4_infinity()（utils/float.h 由来）および Float4GetDatum()。

関連データ構造

  • Datum: PostgreSQLの汎用データ表現型。
  • GIN_SUPPORT: GINインデックスのサポート関数を定義するためのマクロ。

失敗モードと境界条件

  • 浮動小数点の表現範囲外の値や、プラットフォーム固有の無限大の扱いに依存する可能性があります。

保守担当者向けメモ

  • 本関数は GIN_SUPPORT マクロを通じて、GINインデックスの検索ロジックに組み込まれます。
  • float4 型の最小値の定義を変更する必要がある場合は、この関数を修正してください。
