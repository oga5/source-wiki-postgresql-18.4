ExecAssignScanProjectionInfoWithVarno

定義

  • 種別: function
  • 定義箇所: src/backend/executor/execScan.c:93

目的

スキャンノードに対して、指定された変数番号（varno）に基づいた射影（Projection）情報を設定します。

シグネチャまたは形状

void ExecAssignScanProjectionInfoWithVarno(ScanState *node, int varno)

入力と出力

  • **入力**:
  • node: 射影情報を設定する対象の ScanState 構造体。
  • varno: 射影対象を特定するための変数番号。
  • **出力**: なし（副作用として node->ps に射影情報が設定されます）。

前提条件と不変条件

  • node->ss_ScanTupleSlot が初期化されており、有効な tts_tupleDescriptor を保持している必要があります。

アルゴリズムまたは振る舞い

1. node->ss_ScanTupleSlot からタプル記述子（TupleDesc）を取得します。 2. 取得したタプル記述子と指定された varno を引数として、ExecConditionalAssignProjectionInfo を呼び出します。これにより、不要なプロジェクションを回避する最適化を伴う射影設定が行われます。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/executor/nodeCustom.c
  • src/backend/executor/nodeForeignscan.c
  • src/backend/executor/nodeIndexonlyscan.c
  • **呼び出し先**:
  • ExecConditionalAssignProjectionInfo（実行エンジンの内部関数）

関連データ構造

  • ScanState: スキャンノードの実行状態を保持する構造体。
  • TupleDesc: タプルの構造を記述するデータ構造。

失敗モードと境界条件

  • node が NULL の場合や、ss_ScanTupleSlot が適切に設定されていない場合はセグメンテーション違反が発生する可能性があります。

保守担当者向けメモ

  • 本関数は ExecAssignScanProjectionInfo のバリエーションであり、特定の varno を明示的に指定する必要があるスキャンノード（カスタムスキャンやインデックスオンリースキャン等）で使用されます。
  • 射影の最適化ロジックは ExecConditionalAssignProjectionInfo に依存しているため、動作の詳細についてはそちらを参照してください。
