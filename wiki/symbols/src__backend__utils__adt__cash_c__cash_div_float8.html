cash_div_float8

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/cash.c:129

目的

通貨型（Cash）の値を浮動小数点数（float8）で除算し、結果を再び通貨型として返します。

シグネチャまたは形状

static inline Cash cash_div_float8(Cash c, float8 f)

入力と出力

  • **入力**:
  • c: 除算される通貨値（Cash型、内部的にはint64）。
  • f: 除数（float8型）。
  • **出力**: 除算結果をCash型にキャストして返します。

前提条件と不変条件

  • 内部計算において float8 への変換と rint() による丸め処理が行われます。
  • 結果が int64 の範囲内に収まる必要があります。

アルゴリズムまたは振る舞い

1. c を float8 にキャストし、f で除算します。 2. rint() 関数を用いて結果を整数に丸めます。 3. 結果が NaN であるか、または int64 の範囲外である場合、ERRCODE_NUMERIC_VALUE_OUT_OF_RANGE エラーを発生させます。 4. 正常な場合は、丸められた値を Cash 型にキャストして返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/cash.c:767
  • src/backend/utils/adt/cash.c:807
  • **呼び出し先**:
  • float8_div (ソースコード内の記述より)
  • rint (標準数学ライブラリ)
  • ereport (エラー報告用マクロ)

関連データ構造

  • Cash: int64 の別名であり、通貨値を保持する内部表現。

失敗モードと境界条件

  • **ゼロ除算**: float8_div の挙動に依存します。
  • **範囲外**: 計算結果が int64 の最大値・最小値を超える場合、または NaN となる場合はエラーとなります。

保守担当者向けメモ

  • 本関数は static inline で定義されており、通貨演算の精度を維持するために rint() を使用しています。
  • ロケール設定による通貨の小数桁数（frac_digits）の扱いは、この関数自体ではなく、呼び出し側の文脈や他の通貨演算関数で考慮される必要があります。
