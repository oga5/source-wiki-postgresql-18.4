get_tablespace_page_costs

定義

  • 種別: function
  • 定義箇所: src/backend/utils/cache/spccache.c:181

目的

指定されたテーブルスペースのランダムおよびシーケンシャルページコストを取得します。テーブルスペース固有の設定が存在しない場合や無効な場合は、グローバルなデフォルト値を返します。

シグネチャまたは形状

void get_tablespace_page_costs(Oid spcid, double *spc_random_page_cost, double *spc_seq_page_cost)

入力と出力

  • spcid (Oid): 対象のテーブルスペースOID。
  • spc_random_page_cost (double*): ランダムページコストを格納する出力ポインタ。NULLの場合はスキップされる。
  • spc_seq_page_cost (double*): シーケンシャルページコストを格納する出力ポインタ。NULLの場合はスキップされる。

前提条件と不変条件

  • get_tablespace(spcid) が有効な TableSpaceCacheEntry を返すことを前提としています。
  • 内部で Assert(spc != NULL) を実行しており、キャッシュエントリが取得できない場合はアボートします。

アルゴリズムまたは振る舞い

1. get_tablespace(spcid) を呼び出し、キャッシュから対象テーブルスペースの情報を取得します。 2. spc_random_page_cost が指定されている場合：

  • キャッシュされたオプションが存在し、かつ random_page_cost が 0 以上であればその値を採用します。
  • それ以外（オプション未設定または負の値）の場合は、グローバル変数 random_page_cost を採用します。

    • 3. spc_seq_page_cost が指定されている場合：

  • キャッシュされたオプションが存在し、かつ seq_page_cost が 0 以上であればその値を採用します。
  • それ以外（オプション未設定または負の値）の場合は、グローバル変数 seq_page_cost を採用します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/tsm_system_time/tsm_system_time.c
  • src/backend/optimizer/path/costsize.c (複数箇所)
  • src/backend/utils/adt/selfuncs.c (複数箇所)
  • **呼び出し先**:
  • get_tablespace (内部関数)

関連データ構造

  • TableSpaceCacheEntry: キャッシュされたテーブルスペース情報およびオプションを保持する構造体。

失敗モードと境界条件

  • spcid に対応するテーブルスペースが存在しない場合、get_tablespace の仕様によりアボートする可能性があります。
  • オプション値が負の値としてキャッシュされている場合、グローバルなデフォルト値へフォールバックします。

保守担当者向けメモ

  • 本関数はプランナのコスト計算において頻繁に呼び出されるため、キャッシュのヒット率がパフォーマンスに直結します。
  • キャッシュの無効化は InvalidateTableSpaceCacheCallback によって行われます。
