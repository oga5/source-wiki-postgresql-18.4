r_i_verb_suffix

定義

  • **種別**: function
  • **定義箇所**: src/backend/snowball/libstemmer/stem_UTF_8_french.c:1006

目的

フランス語のステミング処理において、特定の動詞接尾辞を条件付きで除去します。

シグネチャまたは形状

static int r_i_verb_suffix(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z（ステミング環境を保持する構造体）。
  • **出力**: int（成功時に 1、失敗時に 0、エラー時に負の値を返します）。

前提条件と不変条件

  • z->I[2] に格納された領域（R1）の境界位置よりカーソル z->c が後ろにあること。
  • 処理対象の接尾辞が a_5 配列（35個の要素）に含まれていること。

アルゴリズムまたは振る舞い

1. **領域チェック**: カーソル位置が z->I[2]（R1領域）より前であれば終了します。 2. **接尾辞マッチング**: z->lb を z->I[2] に一時変更し、find_among_b を用いて後方から接尾辞を検索します。 3. **条件判定**:

  • 接尾辞の直前が 'H' である場合、処理を中断します。
  • g_v（母音グループ）に属する文字が接尾辞の直前にあることを確認します。

    • 4. **削除**: 条件を満たす場合、slice_del を呼び出して該当する接尾辞を削除します。 5. **復元**: 変更した z->lb を元の値に戻します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_UTF_8_french.c および stem_ISO_8859_1_french.c 内のメインステミング処理フローから呼び出されます。
  • **呼び出し先**: find_among_b（接尾辞検索）、out_grouping_b_U（母音判定）、slice_del（文字列削除）。

関連データ構造

  • struct SN_env: ステミング環境（カーソル位置、文字列バッファ、領域情報など）。
  • a_5: 接尾辞マッチングに使用される among 構造体の配列。
  • g_v: 母音判定用のグループ定義。

失敗モードと境界条件

  • カーソルが R1 領域より前にある場合は 0 を返します。
  • find_among_b が一致しない場合、または母音判定に失敗した場合は 0 を返します。
  • slice_del がメモリ不足等の理由で負の値を返した場合、その値をそのまま返します。
