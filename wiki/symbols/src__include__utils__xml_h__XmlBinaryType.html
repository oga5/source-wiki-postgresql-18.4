XmlBinaryType

定義

  • 種別: enum
  • 定義箇所: src/include/utils/xml.h:33

目的

XMLデータ型におけるバイナリデータのエンコーディング方式（Base64またはHex）を定義する列挙型です。

シグネチャまたは形状

typedef enum
{
	XMLBINARY_BASE64,
	XMLBINARY_HEX,
} XmlBinaryType;

入力と出力

  • 本シンボル自体は型定義であり、入力や出力は持ちません。
  • 関連するグローバル変数 xmlbinary がこの型を保持し、XML処理時のエンコーディング設定として機能します。

前提条件と不変条件

  • XMLBINARY_BASE64 または XMLBINARY_HEX のいずれかの値をとります。
  • PostgreSQLのGUC（Grand Unified Configuration）変数 xmlbinary と連動します。

アルゴリズムまたは振る舞い

  • XML処理において、バイナリデータをどのように表現・変換するかを決定するフラグとして使用されます。
  • 実行時の設定値に基づき、XML生成やパース時のエンコーディングロジックを切り替えます。

呼び出し元と呼び出し先

  • 呼び出し元: 提供されたソース情報からは、具体的な関数呼び出しの証拠は得られていません。
  • 呼び出し先: 本シンボルは型定義であるため、呼び出し先はありません。
  • 参照元: src/include/utils/xml.h:88 にて、GUC変数 xmlbinary の型定義（内部的な int 型としての保持）に関連して参照されています。

関連データ構造

  • xmltype: XMLデータを保持する型。
  • xmlbinary: 本列挙型を値として保持するグローバル変数（int 型として宣言）。

失敗モードと境界条件

  • 定義された列挙値以外の値が xmlbinary に設定された場合の挙動については、本ソース excerpt からは判断できません。

保守担当者向けメモ
