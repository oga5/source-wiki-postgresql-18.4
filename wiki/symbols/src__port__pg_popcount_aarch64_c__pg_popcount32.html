pg_popcount32

定義

  • 種別: function
  • 定義箇所: src/port/pg_popcount_aarch64.c:298

目的

32ビット符号なし整数のビットカウント（セットされているビット数）を計算します。

シグネチャまたは形状

int pg_popcount32(uint32 word)

入力と出力

  • **入力**: uint32 word（カウント対象の32ビット整数）
  • **出力**: int（入力値に含まれる1のビット数）

前提条件と不変条件

  • POPCNT_AARCH64 が定義されている環境で有効です。
  • 内部的に pg_popcount64 を呼び出すため、64ビット版のビットカウント実装が利用可能である必要があります。

アルゴリズムまたは振る舞い

入力された uint32 型の値を uint64 型にキャストし、pg_popcount64 関数へ渡してその戻り値を返します。

呼び出し元と呼び出し先

  • **呼び出し先**: pg_popcount64
  • **呼び出し元**:
  • src/backend/commands/user.c:2335
  • src/include/nodes/bitmapset.h:80 (マクロ bmw_popcount 経由)
  • src/port/pg_bitutils.c:191 および src/port/pg_bitutils.c:509

関連データ構造

  • なし

失敗モードと境界条件

  • 32ビット整数の全範囲（0〜4,294,967,295）に対して定義されています。
  • 失敗モードに関する特記事項はありません。

保守担当者向けメモ

  • 本関数は pg_popcount64 への単なるラッパーです。最適化の変更は pg_popcount64 側で行う必要があります。
  • src/port/pg_bitutils.c にて関数ポインタとして定義されており、実行時に最適な実装へ動的に切り替えられる可能性があります。
