portuguese_ISO_8859_1_stem

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_portuguese.c:880

目的

ISO-8859-1 エンコーディングのポルトガル語テキストに対し、接尾辞削除ルールを適用して語幹を抽出する。

シグネチャまたは形状

int portuguese_ISO_8859_1_stem(struct SN_env * z)

入力と出力

  • **入力**: z (Snowball 環境構造体 SN_env)。入力文字列および処理状態を保持。
  • **出力**: 成功時に 1、失敗時に負の値を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • 内部で r_mark_regions を呼び出し、単語内の領域（RV, R1, R2）を定義する。
  • 処理中、z->c（カーソル位置）や z->lb（境界）が適切に操作される。

アルゴリズムまたは振る舞い

1. **前処理**: r_prelude を実行し、特殊文字を内部表現に変換する。 2. **領域定義**: r_mark_regions を実行し、語幹抽出に必要な領域を計算する。 3. **ステミング**:

  • r_standard_suffix または r_verb_suffix を試行する。
  • 特定の条件（末尾が 'i' かつその前が 'c' など）に基づき、r_RV 領域内であれば接尾辞を削除する。
  • 失敗した場合は r_residual_suffix を試行する。

    • 4. **後処理**: r_residual_form を適用し、最後に r_postlude で文字を元の形式に戻す。

呼び出し元と呼び出し先

  • **呼び出し元**: ソースコード内での直接的な呼び出し元情報は提供されていない。
  • **呼び出し先**: r_prelude, r_mark_regions, r_standard_suffix, r_verb_suffix, r_RV, r_residual_suffix, r_residual_form, r_postlude, slice_del。

関連データ構造

  • struct SN_env: Snowball ランタイムの共通環境構造体。

失敗モードと境界条件

  • 各内部関数（r_prelude 等）が負の値を返した場合、その値をそのまま伝播させて終了する。
  • slice_del が失敗した場合も同様に負の値を返す。
