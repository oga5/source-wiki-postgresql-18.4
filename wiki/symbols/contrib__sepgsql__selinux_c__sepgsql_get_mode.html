sepgsql_get_mode

定義

  • 種別: function
  • 定義箇所: contrib/sepgsql/selinux.c:624

目的

SE-PostgreSQL 拡張機能の現在の動作モードを取得します。

シグネチャまたは形状

int sepgsql_get_mode(void)

入力と出力

  • 入力: なし
  • 出力: 現在の動作モードを示す整数値（sepgsql_mode 変数の値）を返します。

前提条件と不変条件

  • sepgsql_mode 変数が初期化されている必要があります。
  • 本関数は、SE-PostgreSQL の動作状態（Disabled, Default, Permissive, Internal）を判定するために使用されます。

アルゴリズムまたは振る舞い

グローバル変数 sepgsql_mode の値をそのまま返します。

呼び出し元と呼び出し先

  • 呼び出し元: contrib/sepgsql/uavc.c:399 において、モードが SEPGSQL_MODE_INTERNAL でないことを確認するために呼び出されています。
  • 呼び出し先: なし。

関連データ構造

  • sepgsql_mode: モジュールの動作状態を保持するグローバル変数。

失敗モードと境界条件

  • 特筆すべき失敗モードはありません。常に sepgsql_mode の現在の値を返します。

保守担当者向けメモ

  • 本関数は sepgsql.h で extern 宣言されており、モジュール全体から参照可能です。
  • 動作モードの変更が必要な場合は、対となる sepgsql_set_mode を参照してください。
