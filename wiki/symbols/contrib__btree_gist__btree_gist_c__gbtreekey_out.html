gbtreekey_out

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_gist.c:39

目的

GiST 索引内部で使用される gbtreekey 型の値をユーザーが直接表示しようとした際に、エラーを発生させて操作を拒否するためのスタブ関数です。

シグネチャまたは形状

Datum gbtreekey_out(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS（標準的な PostgreSQL 関数インターフェース）
  • **出力**: なし（関数内でエラーを投げるため、正常な戻り値は存在しません）

前提条件と不変条件

  • 本関数は、gbtreekey 型がユーザーによる直接的な表示（出力）を意図していないという設計上の制約に基づいています。
  • 実行されると必ず ERRCODE_FEATURE_NOT_SUPPORTED エラーが発生します。

アルゴリズムまたは振る舞い

1. ereport を呼び出し、ERRCODE_FEATURE_NOT_SUPPORTED エラーコードと共に「cannot display a value of type gbtreekey?」というメッセージを出力します。 2. コンパイラの警告を抑制するために PG_RETURN_VOID() を呼び出しますが、実際にはエラーにより処理が中断されるため到達しません。

呼び出し元と呼び出し先

  • **呼び出し元**: 直接的な呼び出し元は不明ですが、PostgreSQL の型システムを通じて間接的に呼び出される可能性があります。
  • **呼び出し先**: ereport（PostgreSQL のエラー報告マクロ）。

関連データ構造

  • gbtreekey: 本モジュールで内部的に使用される型。

失敗モードと境界条件

  • 本関数自体が「失敗」を意図した設計となっており、呼び出された時点で常にエラーを返します。

保守担当者向けメモ

  • 本関数は GiST 索引の仕様上要求されるインターフェースを満たすためのダミー実装です。
  • ユーザーがこの型を直接操作することは想定されていないため、実装を変更して表示を許可すべきではありません。
