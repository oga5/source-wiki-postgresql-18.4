GetPrepStmtNumber

定義

  • 種別: function
  • 定義箇所: contrib/postgres_fdw/connection.c:926

目的

リモートの PostgreSQL サーバー上で使用する準備済みステートメント（Prepared Statement）に対して、一意な識別番号を生成します。

シグネチャまたは形状

unsigned int GetPrepStmtNumber(PGconn *conn);

入力と出力

  • **入力**: PGconn *conn（リモートサーバーへの接続ハンドル。本関数内では使用されていません）
  • **出力**: unsigned int（インクリメントされた新しいステートメント番号）

前提条件と不変条件

  • 内部的な静的変数 prep_stmt_number が、呼び出されるたびにインクリメントされます。
  • 戻り値は呼び出しごとに単調増加します。

アルゴリズムまたは振る舞い

1. 静的変数 prep_stmt_number を 1 加算します。 2. 加算後の値を戻り値として返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/postgres_fdw/connection.c:927
  • contrib/postgres_fdw/postgres_fdw.c:4250
  • **呼び出し先**:
  • 本関数内から他の関数を呼び出すことはありません。

関連データ構造

  • PGconn: libpq の接続構造体。本関数は引数として受け取りますが、処理には利用していません。

失敗モードと境界条件

  • unsigned int の最大値を超えた場合のオーバーフロー挙動については、ソースコード内に明示的なガード条件は存在しません。

保守担当者向けメモ

  • 本関数は引数 conn を使用していませんが、インターフェースの整合性のために保持されています。
  • prep_stmt_number はファイルスコープの静的変数であり、セッション内での一意性を保証するために使用されます。
