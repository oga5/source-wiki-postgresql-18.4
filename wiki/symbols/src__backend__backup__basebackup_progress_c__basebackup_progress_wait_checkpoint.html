basebackup_progress_wait_checkpoint

定義

  • 種別: function
  • 定義箇所: src/backend/backup/basebackup_progress.c:185

目的

ベースバックアップの進捗状況を追跡するシステムビュー（pg_stat_progress_basebackup）に対し、現在のフェーズが「チェックポイント待機中」であることを通知・更新する。

シグネチャまたは形状

void basebackup_progress_wait_checkpoint(void)

入力と出力

  • 入力: なし
  • 出力: なし（副作用として統計情報を更新）

前提条件と不変条件

  • pgstat サブシステムが初期化されており、進捗追跡が開始されていること。

アルゴリズムまたは振る舞い

1. pgstat_progress_update_param 関数を呼び出す。 2. PROGRESS_BASEBACKUP_PHASE パラメータに対し、PROGRESS_BASEBACKUP_PHASE_WAIT_CHECKPOINT 定数を設定する。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/backup/basebackup.c:268
  • 呼び出し先: pgstat_progress_update_param（pgstat.h で定義）

関連データ構造

  • PROGRESS_BASEBACKUP_PHASE（commands/progress.h で定義）
  • PROGRESS_BASEBACKUP_PHASE_WAIT_CHECKPOINT（commands/progress.h で定義）

失敗モードと境界条件

  • pgstat が無効な場合や、進捗追跡が開始されていない状態で呼び出された場合の挙動は、pgstat_progress_update_param の実装に依存する。

保守担当者向けメモ

  • 本関数は basebackup.c からのバックアップ進行状況の通知を受け取るためのインターフェースである。
  • 統計情報の更新のみを行い、実際のチェックポイント処理自体には関与しない。
