range_sockaddr_AF_INET6

定義

  • 種別: function
  • 定義箇所: src/backend/libpq/ifaddr.c:77

目的

IPv6アドレスが指定されたネットワーク範囲（サブネット）に含まれるかどうかを判定します。

シグネチャまたは形状

static int range_sockaddr_AF_INET6(const struct sockaddr_in6 *addr,
                                   const struct sockaddr_in6 *netaddr,
                                   const struct sockaddr_in6 *netmask)

入力と出力

  • **入力**:
  • addr: 判定対象のIPv6アドレス。
  • netaddr: ネットワークアドレス。
  • netmask: ネットワークマスク。
  • **出力**:
  • 1: アドレスが指定範囲内にある場合。
  • 0: アドレスが指定範囲外にある場合。

前提条件と不変条件

  • 入力される各ポインタは有効な sockaddr_in6 構造体を指している必要があります。
  • IPv6アドレスの比較のため、内部で16バイト（128ビット）のループ処理を行います。

アルゴリズムまたは振る舞い

1. 16バイトのループを回し、各バイトに対して以下のビット演算を行います。 2. (addr[i] XOR netaddr[i]) AND netmask[i] を計算します。 3. この結果が 0 でない場合、そのバイト位置でネットワーク範囲外と判断し、直ちに 0 を返します。 4. 全てのバイトで一致が確認された場合、1 を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/libpq/ifaddr.c 内の関数（pg_range_sockaddr 等）から呼び出されます。
  • **呼び出し先**: 本関数は標準ライブラリや他の関数を呼び出さず、純粋なビット演算のみを実行します。

関連データ構造

  • struct sockaddr_in6: IPv6アドレスを保持する標準的なネットワーク構造体。

失敗モードと境界条件

  • ネットワークマスクが適切に設定されていない場合、判定結果はマスクの内容に依存します。
  • 本関数自体はエラーコードを返さず、範囲内か否かの二値のみを返します。

保守担当者向けメモ

  • 本関数は pg_range_sockaddr から IPv6 判定用として委譲される静的関数です。
  • 128ビットの比較をバイト単位で行う実装であるため、修正時はネットワークバイトオーダーの整合性に注意してください。
