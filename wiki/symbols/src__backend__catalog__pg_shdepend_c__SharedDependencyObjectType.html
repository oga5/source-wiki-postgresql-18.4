SharedDependencyObjectType

定義

  • 種別: enum
  • 定義箇所: src/backend/catalog/pg_shdepend.c:67

目的

共有依存関係システムにおいて、依存先オブジェクトがローカル（現在のデータベース内）、共有（クラスタ全体）、またはリモートのいずれであるかを識別するために使用されます。

シグネチャまたは形状

typedef enum
{
    LOCAL_OBJECT,
    SHARED_OBJECT,
    REMOTE_OBJECT,
} SharedDependencyObjectType;

入力と出力

  • 本シンボルは列挙型であり、関数への引数や構造体のメンバとして型指定に使用されます。

前提条件と不変条件

  • 共有依存関係の追跡において、オブジェクトのスコープを分類する唯一の手段として機能します。

アルゴリズムまたは振る舞い

  • 依存関係の記録や検索を行う際、この列挙値に基づいて処理対象のカタログやアクセス方法が決定されます。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/catalog/pg_shdepend.c 内の複数の関数（recordSharedDependencyOn 等）で、引数として利用されています。
  • 呼び出し先: 本シンボル自体は列挙型であり、関数を呼び出すことはありません。

関連データ構造

  • pg_shdepend カタログ: 共有依存関係を格納するシステムカタログ。

失敗モードと境界条件

  • 該当する情報はありません。

保守担当者向けメモ

  • 本列挙型に新しい値を追加する場合は、pg_shdepend.c 内の依存関係処理ロジック（特にスキャンや削除処理）がその型を正しく扱えるか確認してください。
