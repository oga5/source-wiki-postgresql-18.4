skip_utf8

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/utilities.c:27

目的

UTF-8 エンコードされた文字列バッファにおいて、現在のカーソル位置から指定された文字数分だけ前方へスキップし、新しいカーソル位置を返します。

シグネチャまたは形状

int skip_utf8(const symbol * p, int c, int limit, int n)

入力と出力

  • **入力**:
  • p: 文字列バッファへのポインタ。
  • c: 現在のカーソル位置（インデックス）。
  • limit: バッファの終了位置。
  • n: スキップする文字数。
  • **出力**:
  • 成功時: スキップ後の新しいカーソル位置（int）。
  • 失敗時: -1（n < 0 の場合、またはスキップ中にバッファの終端 limit に達した場合）。

前提条件と不変条件

  • n は非負である必要があります。
  • c は limit 以下である必要があります。
  • UTF-8 のマルチバイトシーケンス（10xxxxxx）は、先行バイト（11xxxxxx）の直後に続くものとして正しく処理されます。

アルゴリズムまたは振る舞い

1. n が負であれば即座に -1 を返します。 2. n 回のループを行い、各文字をスキップします。 3. 各文字の先頭バイトが 0xC0 以上（マルチバイト文字の開始）である場合、後続の継続バイト（0x80 以上 0xC0 未満）を limit に達するまで読み飛ばします。 4. 処理中に c が limit に達した場合は -1 を返します。 5. すべてのスキップが完了した時点で、最終的なカーソル位置 c を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: stem_UTF_8_arabic.c, stem_UTF_8_basque.c, stem_UTF_8_catalan.c, stem_UTF_8_danish.c, stem_UTF_8_dutch.c, stem_UTF_8_english.c 等の各言語ステミング実装ファイル。
  • **呼び出し先**: 外部関数への依存はありません。

関連データ構造

  • symbol: 文字列バッファの要素型（header.h で定義）。

失敗モードと境界条件

  • n < 0: 無効な引数として -1 を返します。
  • c >= limit: スキップ開始時点で既に終端に達している場合、-1 を返します。

保守担当者向けメモ

  • 本関数は UTF-8 のマルチバイト構造を認識してスキップするため、ASCII 専用のスキップ処理とは異なります。
  • 呼び出し元でカーソル位置 z->c を更新する際に、本関数の戻り値が -1 でないことを確認してください。
