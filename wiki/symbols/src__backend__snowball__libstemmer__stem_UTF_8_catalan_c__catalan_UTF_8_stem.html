catalan_UTF_8_stem

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_catalan.c:1400

目的

UTF-8エンコーディングされたカタルーニャ語の単語に対し、接尾辞の削除や正規化を行い、語幹を導出するメインエントリーポイント。

シグネチャまたは形状

int catalan_UTF_8_stem(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（Snowballランタイム環境構造体。処理対象の文字列データを含む）
  • **出力**: int（成功時は 1、失敗時は負の値を返す）

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • 内部で r_mark_regions を呼び出し、単語内のR1, R2領域を定義する。

アルゴリズムまたは振る舞い

1. r_mark_regions を実行し、単語内の領域をマークする。 2. r_attached_pronoun を適用し、付着した代名詞を削除する。 3. r_standard_suffix を試行し、成功しなければ r_verb_suffix を試行する（排他的な選択）。 4. r_residual_suffix を適用し、残余の接尾辞を処理する。 5. r_cleaning を適用し、アクセント記号等の正規化を行う。 6. 処理終了後、環境のカーソル位置を復元し 1 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/include/snowball/libstemmer/stem_UTF_8_catalan.h および src/backend/snowball/libstemmer/stem_UTF_8_catalan.c 内で宣言されているが、具体的な呼び出し元コードの詳細は本ソース excerpt には含まれていない。
  • **呼び出し先**: r_mark_regions, r_attached_pronoun, r_standard_suffix, r_verb_suffix, r_residual_suffix, r_cleaning。

関連データ構造

  • struct SN_env: Snowballのステミング環境を管理する構造体。

失敗モードと境界条件

  • 各サブ関数（r_mark_regions 等）が負の値を返した場合、即座にその値を戻り値として終了する。
  • r_standard_suffix と r_verb_suffix の選択ロジックにおいて、いずれも適用されない場合はそのまま後続の処理へ進む。

保守担当者向けメモ

  • 本関数はSnowballコンパイラによる自動生成コードであるため、直接編集せず元のSnowballスクリプトを修正すること。
  • 巨大な among 配列（a_0 ～ a_4）を用いたトライ木検索が各サブ関数内で利用されている。
