uuid_to_string

定義

  • 種別: function
  • 定義箇所: contrib/uuid-ossp/uuid-ossp.c:175

目的

OSSP UUID ライブラリの内部形式である uuid_t オブジェクトを、PostgreSQL で扱える文字列形式（UUID 文字列）に変換します。

シグネチャまたは形状

static char *uuid_to_string(const uuid_t *uuid)

入力と出力

  • **入力**: const uuid_t *uuid - 変換対象の UUID オブジェクトへのポインタ。
  • **出力**: char * - UUID 文字列を格納したメモリ領域へのポインタ（palloc により割り当て済み）。

前提条件と不変条件

  • UUID_LEN_STR + 1 バイトのメモリが確保可能であること。
  • 内部で使用する uuid_export 関数が正常に動作する環境であること。

アルゴリズムまたは振る舞い

1. palloc を使用して、UUID 文字列を保持するためのバッファ（UUID_LEN_STR + 1）を割り当てます。 2. uuid_export を呼び出し、uuid オブジェクトを UUID_FMT_STR（文字列形式）でバッファに書き出します。 3. uuid_export の戻り値 rc が UUID_RC_OK でない場合、pguuid_complain(rc) を呼び出してエラーを処理します。 4. 変換された文字列バッファを返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/uuid-ossp/uuid-ossp.c 内の各 UUID 生成関数（直接的な呼び出しが確認されています）。
  • **呼び出し先**:
  • palloc (PostgreSQL メモリ管理)
  • uuid_export (OSSP UUID ライブラリ)
  • pguuid_complain (エラーハンドリング)

関連データ構造

  • uuid_t: OSSP UUID ライブラリが使用する UUID の内部表現。

失敗モードと境界条件

  • palloc がメモリ不足により失敗した場合、PostgreSQL のエラーハンドリングにより処理が中断されます。
  • uuid_export が UUID_RC_OK 以外の値を返した場合、pguuid_complain が呼び出され、通常は PostgreSQL のエラーとして報告されます。

保守担当者向けメモ

  • 本関数は OSSP UUID ライブラリに依存しています。ビルド環境で OSSP UUID が有効な場合にのみ使用されます。
  • uuid_export の第 3 引数 ptr は、関数内でバッファの先頭を指すように初期化されています。
