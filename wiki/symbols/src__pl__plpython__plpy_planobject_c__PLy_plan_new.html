PLy_plan_new

定義

  • 種別: function
  • 定義箇所: src/pl/plpython/plpy_planobject.c:63

目的

新しい PLyPlanObject インスタンスを生成し、初期化するファクトリ関数です。

シグネチャまたは形状

PyObject *PLy_plan_new(void)

入力と出力

  • **入力**: なし
  • **出力**: 成功時は初期化された PLyPlanObject へのポインタ（PyObject * 型）、失敗時は NULL を返します。

前提条件と不変条件

  • PLy_PlanType がPythonの型システムに登録済みである必要があります。
  • Python 3.8未満の環境では、PLy_PlanType の参照カウントをインクリメントする回避策が適用されます。

アルゴリズムまたは振る舞い

1. PyObject_New を使用して PLyPlanObject 構造体のメモリを確保します。 2. 確保に失敗した場合は NULL を返します。 3. Python 3.8未満の場合、Py_INCREF を呼び出して型の参照カウントを調整します。 4. 構造体の各フィールド（plan, nargs, types, args, mcxt）を NULL または 0 で初期化します。 5. 初期化済みのオブジェクトを返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/pl/plpython/plpy_spi.c:57 にて呼び出されています。
  • **呼び出し先**: Python C API の PyObject_New および Py_INCREF を呼び出します。

関連データ構造

  • PLyPlanObject: 実行計画を保持する構造体。本関数で生成される主要なデータ構造です。

失敗モードと境界条件

  • PyObject_New がメモリ不足等で失敗した場合、NULL を返します。

保守担当者向けメモ

  • Python 3.8以降では Py_INCREF の回避策が不要であるため、条件付きコンパイルが適用されています。
  • 本関数はオブジェクトのメモリ確保と初期化のみを行い、SPI計画自体の生成は行いません。
