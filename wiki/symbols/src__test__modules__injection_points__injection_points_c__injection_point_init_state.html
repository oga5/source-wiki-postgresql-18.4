injection_point_init_state

定義

  • 種別: function
  • 定義箇所: src/test/modules/injection_points/injection_points.c:126

目的

共有メモリ上に配置された InjectionPointSharedState 構造体を初期化し、インジェクションポイント機能が使用する同期プリミティブおよび状態変数をリセットします。

シグネチャまたは形状

static void injection_point_init_state(void *ptr)

入力と出力

  • **入力**: ptr (共有メモリ領域を指す void * ポインタ)
  • **出力**: なし（副作用として ptr が指すメモリ領域を初期化）

前提条件と不変条件

  • ptr は InjectionPointSharedState 構造体としてキャスト可能な有効なメモリ領域を指している必要があります。
  • 本関数は共有メモリの初期化プロセスにおいて、排他制御が確立される前、あるいは初期化フェーズで呼び出されることを前提としています。

アルゴリズムまたは振る舞い

1. 引数 ptr を InjectionPointSharedState * 型にキャストします。 2. SpinLockInit を呼び出し、状態構造体内のスピンロックを初期化します。 3. memset を使用して、wait_counts 配列および name 配列を 0 で埋めます。 4. ConditionVariableInit を呼び出し、待機ポイント用の条件変数を初期化します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/test/modules/injection_points/injection_points.c:171 (injection_init_shmem 内)
  • src/test/modules/injection_points/injection_points.c:190 (構造体初期化の参照)
  • **呼び出し先**:
  • SpinLockInit
  • memset
  • ConditionVariableInit

関連データ構造

  • InjectionPointSharedState: 本関数が初期化対象とする共有メモリ上の管理構造体。

失敗モードと境界条件

  • ptr が NULL の場合、あるいは割り当てられたメモリサイズが InjectionPointSharedState 構造体より小さい場合、メモリ破壊やセグメンテーション違反が発生します。

保守担当者向けメモ

  • 本関数は共有メモリの初期化時にのみ使用されます。構造体のメンバ構成に変更があった場合は、memset の対象範囲や初期化ロジックを更新してください。
