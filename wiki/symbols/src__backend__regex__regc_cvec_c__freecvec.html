freecvec

定義

  • 種別: function
  • 定義箇所: src/backend/regex/regc_cvec.c:137

目的

正規表現コンパイル時に使用される文字セット構造体 cvec のメモリを解放します。

シグネチャまたは形状

static void freecvec(struct cvec *cv)

入力と出力

  • **入力**: cv (struct cvec *) - 解放対象の cvec 構造体へのポインタ。
  • **出力**: なし。

前提条件と不変条件

  • cv は newcvec 等で動的に確保された有効なメモリ領域を指している必要があります。
  • regguts.h で定義される FREE マクロを使用してメモリを解放します。

アルゴリズムまたは振る舞い

1. 引数として受け取った cv ポインタに対して FREE マクロを実行し、メモリを解放します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/regex/regc_cvec.c:126 (getcvec 内での再確保時)
  • src/backend/regex/regcomp.c:607, 609 (コンパイル終了時のクリーンアップ)
  • **呼び出し先**:
  • FREE マクロ（regguts.h で定義）

関連データ構造

  • struct cvec: 正規表現のブラケット式で使用される文字セットを保持する構造体。

失敗モードと境界条件

  • cv が NULL の場合の挙動は FREE マクロの実装に依存します。

保守担当者向けメモ

  • 本関数は regc_cvec.c 内で static として定義されており、regcomp.c にインクルードされて使用されます。
  • cvec 構造体内部の配列は構造体自体と一体で確保されているため、個別の解放は不要です。
