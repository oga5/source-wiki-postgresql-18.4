pg_atomic_clear_flag

定義

  • 種別: function
  • 定義箇所: src/include/port/atomics.h:206

目的

pg_atomic_test_set_flag() によって取得されたフラグ（ロック）を解放するために使用されます。

シグネチャまたは形状

static inline void pg_atomic_clear_flag(volatile pg_atomic_flag *ptr)

入力と出力

  • **入力**: ptr - 操作対象となる pg_atomic_flag 型のポインタ。volatile 修飾されています。
  • **出力**: なし（戻り値なし）。

前提条件と不変条件

  • ptr は適切にアライメントされている必要があります。
  • 本関数は、対応する pg_atomic_test_set_flag() によってロックが保持されている状態での呼び出しを想定しています。

アルゴリズムまたは振る舞い

内部的に pg_atomic_clear_flag_impl(ptr) を呼び出し、プラットフォーム固有の実装を通じてフラグをアトミックにクリアします。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/postmaster/autovacuum.c
  • src/test/regress/regress.c
  • **呼び出し先**:
  • pg_atomic_clear_flag_impl（プラットフォーム固有の実装へ委譲）

関連データ構造

  • pg_atomic_flag: アトミック操作対象のフラグ型。

失敗モードと境界条件

  • 不適切なポインタ（アライメント違反や NULL）が渡された場合の動作は未定義です。

保守担当者向けメモ

  • 本関数は static inline で定義されており、実際の処理は _impl 接尾辞を持つ関数に委譲されます。プラットフォーム固有の実装の詳細については、src/include/port/atomics/ 配下の各アーキテクチャヘッダーを参照してください。
