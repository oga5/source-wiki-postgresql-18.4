pgstat_create_function

定義

  • 種別: function
  • 定義箇所: src/backend/utils/activity/pgstat_function.c:44

目的

指定された関数（Oid）に対する統計情報の作成を、トランザクション管理下で開始します。

シグネチャまたは形状

void pgstat_create_function(Oid proid)

入力と出力

  • **入力**: proid (Oid) - 統計対象となる関数のオブジェクト識別子。
  • **出力**: なし（戻り値なし）。

前提条件と不変条件

  • MyDatabaseId が有効なデータベースコンテキストで呼び出される必要があります。
  • 統計サブシステムが初期化されている必要があります。

アルゴリズムまたは振る舞い

1. pgstat_create_transactional を呼び出します。 2. 統計種別として PGSTAT_KIND_FUNCTION を指定し、現在のデータベースID (MyDatabaseId) と対象の関数ID (proid) を渡して、トランザクションに関連付けられた統計エントリを作成します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/catalog/pg_proc.c:732
  • **呼び出し先**:
  • pgstat_create_transactional（統計サブシステムの内部関数）

関連データ構造

  • PGSTAT_KIND_FUNCTION: 関数統計であることを示す識別子。
  • Oid: PostgreSQLのオブジェクト識別子。

失敗モードと境界条件

  • 統計サブシステムが利用できない状態や、メモリ不足時には pgstat_create_transactional 経由でエラーが発生する可能性があります。

保守担当者向けメモ

  • 本関数は、関数が作成または登録された際に統計情報を追跡可能にするためのエントリポイントです。
  • トランザクション境界での整合性維持については pgstat_create_transactional の実装に依存しています。
