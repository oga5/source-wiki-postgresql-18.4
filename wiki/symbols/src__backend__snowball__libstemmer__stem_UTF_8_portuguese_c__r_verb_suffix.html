r_verb_suffix

  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_portuguese.c:804

目的

ポルトガル語のステミング処理において、単語の動詞活用語尾を特定し、条件を満たす場合に削除を行う。

シグネチャまたは形状

static int r_verb_suffix(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（Snowball ステミング環境構造体）
  • 出力: int（成功時に 1、失敗または条件不一致時に 0、エラー発生時に負の値を返す）

前提条件と不変条件

  • z->I[2] に格納された領域境界（R2領域など）が有効であること。
  • 現在のカーソル位置 z->c が z->I[2] 以降にあること。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置が z->I[2] より前であれば終了（0 を返す）。 2. 検索範囲の境界（lb）を z->I[2] に一時的に設定する。 3. a_6 テーブル（120個の動詞接尾辞パターン）を用いて、後方検索（find_among_b）を実行する。 4. 一致する接尾辞が見つかった場合、その範囲を削除（slice_del）する。 5. 検索境界を元に戻し、成功時は 1 を返す。

呼び出し元と呼び出し先

  • 呼び出し元: portuguese_UTF_8_stem（直接の呼び出し関係はソースコードの制御フローから推論可能だが、本抜粋には明示的な呼び出し元コードは含まれていない）。
  • 呼び出し先: find_among_b（接尾辞検索）、slice_del（文字列削除）。

関連データ構造

  • SN_env: ステミングの状態を保持する構造体。
  • a_6: 削除対象となる動詞接尾辞の定義テーブル。

失敗モードと境界条件

  • z->c < z->I[2] の場合、処理対象外として 0 を返す。
  • find_among_b が一致を見つけられない場合、0 を返す。
  • slice_del が内部エラー（負の値）を返した場合、その値をそのまま伝播させる。
