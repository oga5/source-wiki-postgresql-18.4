anl_init_selection_state

定義

  • 種別: function
  • 定義箇所: src/backend/utils/misc/sampling.c:280

目的

リザーバーサンプリング（Algorithm Z）を開始するための初期値 $W$ を計算し、必要に応じて乱数生成器の状態を初期化する。

シグネチャまたは形状

double anl_init_selection_state(int n)

入力と出力

  • **入力**: int n（サンプリング対象の数）
  • **出力**: double（Algorithm Zの初期化に使用される $W$ 値）

前提条件と不変条件

  • oldrs_initialized フラグにより、乱数生成器の状態（oldrs.randstate）は初回呼び出し時にのみ初期化される。
  • sampler_random_fract は 0 を返さないため、log() の引数は常に正の値となる。

アルゴリズムまたは振る舞い

1. **初期化**: oldrs_initialized が偽の場合、pg_global_prng_state から取得した値を用いて sampler_random_init_state を呼び出し、内部状態を初期化する。 2. **計算**: 以下の式に従い、乱数から $W$ を算出する。 $W = \exp(-\ln(\text{random\_fract}) / n)$ 3. **戻り値**: 計算された $W$ を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: 外部からの具体的な呼び出し元はソースコードの断片からは特定できない。
  • **呼び出し先**:
  • sampler_random_init_state
  • pg_prng_uint32
  • sampler_random_fract
  • exp / log (math.h)

関連データ構造

  • oldrs: 静的変数として保持されるリザーバーサンプリングの状態構造体。
  • pg_global_prng_state: グローバルな乱数生成器の状態。

失敗モードと境界条件

  • n が 0 または負の場合、log() の引数や除算において数学的な未定義動作が発生する可能性がある（ソースコード上でのガードは確認できない）。

保守担当者向けメモ

  • 本関数はレガシーな anl_ 系APIの一部である。新規実装では ReservoirState を使用する新しいAPIを優先すること。
  • oldrs は静的変数であるため、スレッドセーフではない点に注意が必要。
