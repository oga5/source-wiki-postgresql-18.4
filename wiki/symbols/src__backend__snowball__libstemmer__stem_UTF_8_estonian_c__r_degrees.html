r_degrees

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_estonian.c:1192

目的

エストニア語のステミング処理において、特定の接尾辞（degrees）を単語末尾から削除し、語幹を正規化する。

シグネチャまたは形状

static int r_degrees(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（Snowball ランタイム環境構造体）
  • **出力**: 成功時に 1、失敗または条件不一致時に 0 を返す。slice_del が失敗した場合は負の値を返す。

前提条件と不変条件

  • z->I[0] に格納された境界値以上の位置で処理を行う必要がある。
  • find_among_b を使用して a_10 配列内の接尾辞を後方検索する。

アルゴリズムまたは振る舞い

1. **境界チェック**: 現在のカーソル位置 z->c が z->I[0] 未満であれば終了する。 2. **検索**: z->lb を z->I[0] に一時的に設定し、a_10 配列を用いて単語末尾の接尾辞を検索する。 3. **条件分岐**:

  • among_var が 1 の場合: g_RV グルーピングに含まれない文字であることを確認し、slice_del で削除する。
  • among_var が 2 の場合: 条件確認なしで slice_del で削除する。

    • 4. **復元**: 変更した z->lb を元の値 mlimit1 に戻す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_UTF_8_estonian.c:1241（ステミングパイプラインの一部として呼び出される）。
  • **呼び出し先**: find_among_b, in_grouping_b_U, slice_del。

関連データ構造

  • struct SN_env: Snowball ランタイム環境。
  • a_10: 削除対象の接尾辞を保持する among 構造体の配列。
  • g_RV: 処理領域を定義する文字グループ。

失敗モードと境界条件

  • z->c < z->I[0] の場合、処理を行わず 0 を返す。
  • find_among_b で一致する接尾辞が見つからない場合、0 を返す。
  • slice_del が内部エラー（負の値）を返した場合、その値をそのまま伝播させる。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードであるため、直接編集せず元の .sbl ファイルを修正すること。
  • a_10 配列の定義や g_RV の構成に変更がある場合、本関数の挙動に直接影響する。
