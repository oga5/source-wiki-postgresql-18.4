r_Step_3

目的

セルビア語のステミングアルゴリズムにおける第3段階の語尾削除処理を実行する。

シグネチャまたは形状

static int r_Step_3(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング環境構造体）
  • 出力: 成功時に 1、失敗または条件不一致時に 0、エラー発生時に負の値を返す。

前提条件と不変条件

  • z->c が現在の処理位置を指していること。
  • z->lb（下限境界）および z->p（文字バッファ）が有効であること。

アルゴリズムまたは振る舞い

1. 現在位置 z->c を z->ket に設定する。 2. 文字バッファの末尾付近の文字が特定のビットマスク条件（3188642）を満たすか確認し、満たさない場合は終了する。 3. a_3 テーブル（26要素）を用いて後方検索（find_among_b）を行い、一致する語尾を特定する。 4. 一致した場合、その位置を z->bra に設定する。 5. r_R1 を呼び出し、対象が領域 R1 内にあるか検証する。 6. 検証に成功した場合、slice_from_s を用いて該当部分を削除（空文字列に置換）する。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_serbian.c:6530（serbian_UTF_8_stem 内の処理フローの一部）。
  • 呼び出し先: find_among_b, r_R1, slice_from_s。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_3: 語尾削除ルールを定義した among 構造体の配列。

失敗モードと境界条件

  • 文字列が z->lb 以下の場合、または文字条件に合致しない場合は 0 を返す。
  • find_among_b が一致しない場合は 0 を返す。
  • r_R1 または slice_from_s がエラー（負の値）を返した場合、その値をそのまま伝播させる。
