r_main_suffix

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_danish.c:162

目的

デンマーク語のステミング処理において、単語の末尾から主要な接尾辞を検索し、条件に応じて削除する。

シグネチャまたは形状

static int r_main_suffix(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z （Snowballステミング環境）
  • 出力: int （成功時に 1、失敗または処理対象外の場合に 0、エラー発生時に負の値を返す）

前提条件と不変条件

  • z->I[1] に格納された境界位置（R1相当）よりカーソル位置 z->c が後ろにあること。
  • z->p がUTF-8文字列バッファとして有効であること。

アルゴリズムまたは振る舞い

1. **領域制限**: z->I[1] を一時的な境界として設定し、処理範囲を制限する。 2. **接尾辞検索**: find_among_b を使用し、定義テーブル a_0 に基づいて後方から接尾辞をマッチングする。 3. **条件分岐**:

  • among_var == 1: マッチした接尾辞を削除する。
  • among_var == 2: g_s_ending グループに含まれる文字が直前にないことを確認し、条件を満たせば接尾辞を削除する。

    • 4. **復元**: 処理終了後、一時的な境界 mlimit1 を元の状態に戻す。

呼び出し元と呼び出し先

  • 呼び出し元: danish_UTF_8_stem（メインエントリーポイント）から呼び出される。
  • 呼び出し先: find_among_b（接尾辞検索）、slice_del（文字列削除）、in_grouping_b_U（文字グループ判定）。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_0: 除去対象となる接尾辞を定義した among テーブル。
  • g_s_ending: among_var == 2 の判定に使用される文字グループ。

失敗モードと境界条件

  • カーソル位置が境界 z->I[1] より前にある場合、即座に 0 を返す。
  • find_among_b で接尾辞が見つからない場合、0 を返す。
  • slice_del がメモリ不足等のエラーを返した場合、その負の値をそのまま伝播させる。

保守担当者向けメモ

  • 本関数は自動生成コードの一部である。ロジックの変更が必要な場合は、直接編集せず元のSnowballソース（.sbl）を修正すること。
  • デバッグ時は z->c（カーソル）と z->lb（境界）の遷移を追跡すること。
