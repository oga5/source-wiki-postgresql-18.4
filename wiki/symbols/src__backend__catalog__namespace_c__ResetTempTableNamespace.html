ResetTempTableNamespace

定義

  • 種別: function
  • 定義箇所: src/backend/catalog/namespace.c:4646

目的

セッション内の一時テーブル名前空間をリセットし、関連する一時リレーションを削除する。

シグネチャまたは形状

void ResetTempTableNamespace(void)

入力と出力

  • 入力: なし（グローバル変数 myTempNamespace を参照）
  • 出力: なし（副作用として一時リレーションの削除を実行）

前提条件と不変条件

  • myTempNamespace が有効な OID である場合のみ、削除処理が実行される。

アルゴリズムまたは振る舞い

1. OidIsValid(myTempNamespace) を評価し、現在の一時名前空間が割り当てられているかを確認する。 2. 有効な場合、RemoveTempRelations(myTempNamespace) を呼び出し、当該名前空間内のすべての一時リレーションを削除する。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/backend/commands/discard.c (DISCARD TEMP コマンド実行時)
  • 呼び出し先:
  • RemoveTempRelations (内部的に呼び出し)

関連データ構造

  • myTempNamespace (現在のセッションの一時名前空間 OID を保持するグローバル変数)

失敗モードと境界条件

  • myTempNamespace が InvalidOid の場合、本関数は何もしない（安全に終了する）。

保守担当者向けメモ

  • 本関数はセッションレベルでの一時オブジェクトのクリーンアップを担う。DISCARD TEMP コマンドの主要な実行ロジックとして機能する。
