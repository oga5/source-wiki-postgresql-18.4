macaddr_eq

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/mac.c:227

目的

2つの macaddr 型の値が等しいかどうかを判定し、SQLの等価演算子（=）の結果を返します。

シグネチャまたは形状

Datum macaddr_eq(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS を介して2つの macaddr * 型の引数を受け取ります。
  • **出力**: bool 型の Datum を返します。両者が等しい場合は true、そうでない場合は false となります。

前提条件と不変条件

  • 入力される2つの macaddr ポインタは有効である必要があります。
  • 内部比較ロジックは macaddr_cmp_internal に依存しています。

アルゴリズムまたは振る舞い

1. PG_GETARG_MACADDR_P マクロを使用して、引数から2つの macaddr 構造体へのポインタを取得します。 2. macaddr_cmp_internal を呼び出し、2つのMACアドレスを比較します。 3. 比較結果が 0（一致）であるかどうかを判定し、その結果を PG_RETURN_BOOL を通じて返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/btree_gist/btree_macaddr.c:44 から直接呼び出されています。
  • **呼び出し先**:
  • macaddr_cmp_internal（比較ロジックの実行）

関連データ構造

  • macaddr: 6バイトのEUI-48 MACアドレスを保持する構造体。

失敗モードと境界条件

  • 引数が NULL の場合、PG_GETARG_MACADDR_P の挙動に従い、PostgreSQLの関数呼び出し規約に基づいたエラーが発生する可能性があります。

保守担当者向けメモ

  • 本関数は macaddr_cmp_internal の結果に完全に依存しています。比較ロジックの変更が必要な場合は macaddr_cmp_internal を参照してください。
