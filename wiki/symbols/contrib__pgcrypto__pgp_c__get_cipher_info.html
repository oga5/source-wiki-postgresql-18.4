get_cipher_info

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/pgp.c:92

目的

指定された暗号アルゴリズムコードに対応する cipher_info 構造体を取得します。

シグネチャまたは形状

static const struct cipher_info *get_cipher_info(int code);

入力と出力

  • **入力**: code (int型) - 検索対象の暗号アルゴリズム識別コード。
  • **出力**: const struct cipher_info * - 見つかった場合は対応する構造体へのポインタ、見つからない場合は NULL。

前提条件と不変条件

  • cipher_list が有効な cipher_info 構造体の配列であり、末尾が name フィールドが NULL である要素で終端されている必要があります。

アルゴリズムまたは振る舞い

1. 静的な配列 cipher_list を先頭から順に走査します。 2. 各要素の name フィールドが NULL になるまでループを継続します。 3. 現在の要素の code フィールドが引数の code と一致した場合、その要素へのポインタを即座に返します。 4. 配列の終端に達しても一致するものが見つからない場合、NULL を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/pgcrypto/pgp.c:139
  • contrib/pgcrypto/pgp.c:149
  • contrib/pgcrypto/pgp.c:160
  • **呼び出し先**:
  • 外部関数への呼び出しはありません。

関連データ構造

  • struct cipher_info: 暗号アルゴリズムのメタデータ（名称、コード、鍵サイズ、ブロックサイズ等）を保持する構造体。

失敗モードと境界条件

  • 指定された code に一致する暗号アルゴリズムが cipher_list 内に存在しない場合、NULL を返します。呼び出し元はこの戻り値が NULL である可能性を考慮する必要があります。

保守担当者向けメモ

  • 本関数は cipher_list の線形探索を行うため、サポートする暗号アルゴリズムの数が増加した場合は性能への影響を考慮してください。
  • 戻り値のポインタは静的データ領域を指しているため、解放の必要はありません。
