ExecInitBitmapHeapScan

定義

  • 種別: function
  • 定義箇所: src/backend/executor/nodeBitmapHeapscan.c:332

目的

ビットマップヒープスキャンを実行するための BitmapHeapScanState 構造体を生成・初期化し、実行計画ノードをスキャン可能な状態に準備する。

シグネチャまたは形状

BitmapHeapScanState *ExecInitBitmapHeapScan(BitmapHeapScan *node, EState *estate, int eflags)

入力と出力

  • **入力**:
  • node: 実行対象の BitmapHeapScan 計画ノード。
  • estate: 実行状態（EState）。
  • eflags: 実行フラグ。
  • **出力**:
  • 初期化済みの BitmapHeapScanState 構造体へのポインタ。

前提条件と不変条件

  • eflags に EXEC_FLAG_BACKWARD または EXEC_FLAG_MARK を含めることはできない。
  • estate->es_snapshot は MVCC 準拠のスナップショットでなければならない。

アルゴリズムまたは振る舞い

1. BitmapHeapScanState ノードを割り当て、ExecBitmapHeapScan を実行関数として設定する。 2. 統計カウンタをゼロクリアし、初期状態（initialized = false, recheck = true）を設定する。 3. 式コンテキスト（ExprContext）を作成する。 4. ExecOpenScanRelation を呼び出し、対象のリレーションを開く。 5. ExecInitNode を呼び出し、子ノード（インデックススキャン等）を再帰的に初期化する。 6. スキャンタプルスロットを初期化し、結果型とプロジェクション情報を設定する。 7. ExecInitQual を使用して、ノードの qual および bitmapqualorig を初期化する。 8. 最後に、初期化済みの scanstate を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/executor/execProcnode.c 内の ExecInitNode から呼び出されることが確認されている。
  • **呼び出し先**: ExecAssignExprContext, ExecOpenScanRelation, ExecInitNode, ExecInitScanTupleSlot, ExecInitResultTypeTL, ExecAssignScanProjectionInfo, ExecInitQual 等の実行エンジンユーティリティ関数。

関連データ構造

  • BitmapHeapScanState: 本関数が生成する状態構造体。
  • BitmapHeapScan: 計画ノード。
  • EState: 実行状態。

失敗モードと境界条件

  • eflags に不正なフラグが含まれている場合、Assert により停止する。
  • MVCC 非準拠のスナップショットが渡された場合、Assert により停止する。

保守担当者向けメモ

  • 本関数はビットマップヒープスキャンの初期化のみを担当する。実行時の並列処理の初期化については ExecBitmapHeapInitializeDSM 等が別途関与する。
  • recheck フラグはデフォルトで true に設定されており、損失のあるビットマップに対する再検証が必須であることを示している。
