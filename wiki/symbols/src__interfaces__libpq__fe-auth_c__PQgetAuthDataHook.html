PQgetAuthDataHook

定義

  • 種別: function
  • 定義箇所: src/interfaces/libpq/fe-auth.c:1588

目的

現在設定されている認証データフック（PQauthDataHook）の関数ポインタを取得します。

シグネチャまたは形状

PQauthDataHook_type PQgetAuthDataHook(void)

入力と出力

  • 入力: なし
  • 出力: 現在有効な PQauthDataHook_type 型の関数ポインタを返します。

前提条件と不変条件

  • PQauthDataHook は、PQsetAuthDataHook を介して設定可能なグローバル変数または静的変数として保持されています。

アルゴリズムまたは振る舞い

1. 内部変数 PQauthDataHook の現在の値を参照します。 2. その値を呼び出し元へ返します。

呼び出し元と呼び出し先

  • 呼び出し元: src/interfaces/libpq-oauth/oauth-curl.c:2624 にて、フックを取得する用途で呼び出されています。
  • 呼び出し先: なし。

関連データ構造

  • PQauthDataHook_type: 認証データ取得フックの型定義。

失敗モードと境界条件

  • 認証データフックが設定されていない場合、NULL が返される可能性があります。

保守担当者向けメモ

  • 本関数は libpq の認証拡張用インターフェースの一部です。
  • 関連する設定関数 PQsetAuthDataHook およびデフォルト実装 PQdefaultAuthDataHook とセットで管理してください。
