anycompatiblerange_out

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/pseudotypes.c:222

目的

anycompatiblerange 型の値をテキスト形式に変換して出力します。

シグネチャまたは形状

Datum anycompatiblerange_out(PG_FUNCTION_ARGS)

入力と出力

  • 入力: PG_FUNCTION_ARGS（関数マネージャ経由で渡される anycompatiblerange 型のデータ）。
  • 出力: Datum（範囲型の文字列表現）。

前提条件と不変条件

  • 本関数は anycompatiblerange 型の出力関数として登録されることを前提としています。
  • 内部的に range_out を呼び出すため、range_out が要求するすべての前提条件（有効な Datum であること等）を満たす必要があります。

アルゴリズムまたは振る舞い

1. 受け取った fcinfo（関数呼び出し情報）をそのまま range_out 関数へ渡します。 2. range_out の戻り値をそのまま呼び出し元へ返却します。

呼び出し元と呼び出し先

  • 呼び出し先: range_out
  • 呼び出し元: ソースコード内での直接的な呼び出し元情報は提供されていません。

関連データ構造

  • Datum: PostgreSQL の汎用データ型表現。
  • range_out: 範囲型の出力を行うための基盤関数。

失敗モードと境界条件

  • range_out が失敗する場合（無効なデータ形式など）、本関数も同様にエラーを伝播します。

保守担当者向けメモ

  • 本関数は単なるラッパーであり、実質的なロジックは range_out に依存しています。
  • 擬似型に対する出力処理を統一的に扱うためのエントリポイントとして機能しています。
