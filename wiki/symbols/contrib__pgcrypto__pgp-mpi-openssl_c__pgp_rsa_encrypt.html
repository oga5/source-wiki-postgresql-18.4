pgp_rsa_encrypt

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/pgp-mpi-openssl.c:214

目的

OpenPGP 形式の公開鍵を用いた RSA 暗号化処理を行う。内部的には OpenSSL の BIGNUM ライブラリを利用してモジュロ演算を実行する。

シグネチャまたは形状

int pgp_rsa_encrypt(PGP_PubKey *pk, PGP_MPI *_m, PGP_MPI **c_p);

入力と出力

  • **入力**:
  • pk: 暗号化に使用する公開鍵（PGP_PubKey 構造体）。
  • _m: 暗号化対象のメッセージ（PGP_MPI 構造体）。
  • **出力**:
  • c_p: 暗号化された結果（PGP_MPI 構造体）へのポインタを格納するアドレス。
  • **戻り値**: 成功時は 0、失敗時は PXE_PGP_MATH_FAILED を返す。

前提条件と不変条件

  • pk には有効な RSA 公開鍵（指数 e および法 n）が含まれている必要がある。
  • 内部で確保された BIGNUM および BN_CTX は、関数終了時に必ず解放される（goto err によるクリーンアップ）。

アルゴリズムまたは振る舞い

1. mpi_to_bn を使用し、入力された PGP_MPI および公開鍵パラメータを OpenSSL の BIGNUM 形式に変換する。 2. BN_CTX_new で演算用コンテキストを確保する。 3. BN_mod_exp を呼び出し、$c = m^e \pmod n$ を計算する。 4. 計算結果を bn_to_mpi で PGP_MPI 形式に変換し、c_p に格納する。 5. 確保したリソース（BIGNUM オブジェクトおよび BN_CTX）を BN_clear_free および BN_CTX_free で解放する。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/pgcrypto/pgp-pubenc.c:176
  • **呼び出し先**:
  • mpi_to_bn (変換)
  • bn_to_mpi (変換)
  • BN_new, BN_CTX_new, BN_mod_exp, BN_clear_free, BN_CTX_free (OpenSSL API)

関連データ構造

  • PGP_PubKey: 公開鍵情報を保持する構造体。
  • PGP_MPI: 多倍長整数を保持する PGP 形式の構造体。
  • BIGNUM: OpenSSL の多倍長整数構造体。

失敗モードと境界条件

  • メモリ不足や OpenSSL の演算失敗が発生した場合、PXE_PGP_MATH_FAILED を返す。
  • 変換関数（mpi_to_bn 等）が NULL を返した場合、即座にエラー処理へ移行する。

保守担当者向けメモ

  • 本関数は pgcrypto における RSA 暗号化の数学的処理のブリッジである。
  • エラー発生時のメモリリークを防ぐため、新しい BIGNUM を追加する際は必ず err ラベルでの解放処理を確認すること。
