r_noun

目的

アルメニア語のステミング処理において、名詞に関連する特定の接尾辞を単語末尾から検索し、一致した場合に削除する。

シグネチャまたは形状

static int r_noun(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（ステミング環境を保持する構造体）
  • **出力**: int（成功時に 1、失敗時に 0、エラー発生時に負の値を返す）

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • a_2 配列（40個の要素を持つ among 構造体）が定義されている必要がある。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を z->ket に設定する。 2. find_among_b を使用し、a_2 配列に基づいて単語末尾から接尾辞を検索する。 3. 一致しない場合は 0 を返して終了する。 4. 一致した場合、現在のカーソル位置を z->bra に設定する。 5. slice_del を呼び出し、一致した接尾辞を単語から削除する。 6. slice_del の戻り値（成功時は 1、エラー時は負の値）を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_UTF_8_armenian.c:548（armenian_UTF_8_stem 内から呼び出される）。
  • **呼び出し先**: find_among_b（接尾辞検索）、slice_del（文字列削除）。

関連データ構造

  • SN_env: ステミング環境を管理する構造体。
  • a_2: 名詞の接尾辞パターンを保持する among 構造体の配列。

失敗モードと境界条件

  • find_among_b が一致を見つけられない場合、0 を返す。
  • slice_del がメモリ不足やその他の内部エラーで失敗した場合、負の値を返す。
