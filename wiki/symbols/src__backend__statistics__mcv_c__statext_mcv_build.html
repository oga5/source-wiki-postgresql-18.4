statext_mcv_build

定義

  • 種別: function
  • 定義箇所: src/backend/statistics/mcv.c:179

目的

サンプルデータから多変量MCV（Most Common Values）リストを構築し、列間の相関関係を考慮した統計情報を生成する。

シグネチャまたは形状

MCVList *statext_mcv_build(StatsBuildData *data, double totalrows, int stattarget)

入力と出力

  • **入力**:
  • data: 統計構築用のソースデータ（StatsBuildData）。
  • totalrows: テーブルの総行数。
  • stattarget: 統計ターゲット値（保持するMCVアイテム数の上限）。
  • **出力**:
  • 構築された MCVList 構造体へのポインタ。条件を満たすアイテムがない場合は NULL を返す。

前提条件と不変条件

  • data に含まれる行データは、build_sorted_items によってソート可能であること。
  • 生成される MCVList 内のアイテムは、出現頻度（frequency）が降順にソートされていること。
  • 各MCVアイテムの base_frequency は、各列が独立であると仮定した場合の積として計算される。

アルゴリズムまたは振る舞い

1. **ソートとグループ化**: build_sorted_items でデータをソートし、build_distinct_groups で頻出する値の組み合わせ（グループ）を抽出する。 2. **閾値判定**: get_mincount_for_mcv_list を用いて、統計的に有意な頻度閾値を算出する。 3. **リストの切り詰め**: 頻度が閾値を下回るグループを除外し、残ったグループを MCVList に格納する。 4. **頻度計算**: 各グループの実際の出現頻度と、各列の独立性を仮定した base_frequency を算出して格納する。 5. **メモリ解放**: 処理中に使用した一時的なソート済みアイテムやグループ配列を解放する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/statistics/extended_stats.c:209
  • **呼び出し先**: build_mss, build_sorted_items, build_distinct_groups, get_mincount_for_mcv_list, build_column_frequencies, bsearch_arg 等。

関連データ構造

  • MCVList: 多変量統計情報を保持するメイン構造体。
  • MCVItem: 個別の値の組み合わせと、その頻度情報を保持する構造体。
  • StatsBuildData: 統計構築の入力データ。
  • MultiSortSupport: 複数列のソートをサポートする構造体。

失敗モードと境界条件

  • 入力データが空の場合や、ソート済みアイテムが生成できない場合は NULL を返す。
  • 統計的に有意なグループが存在しない場合、nitems が 0 となり、MCVList は構築されない。
  • stattarget がグループ数より大きい場合、利用可能な全グループがリストに含まれる。

保守担当者向けメモ

  • 本関数は、単一列のMCV構築アルゴリズムとは異なり、列間の依存関係（base_frequency との乖離）を考慮したフィルタリングを行う。
  • MCVList の構築には palloc0 を使用しており、nitems に応じた可変長メモリ確保が行われる点に注意すること。
