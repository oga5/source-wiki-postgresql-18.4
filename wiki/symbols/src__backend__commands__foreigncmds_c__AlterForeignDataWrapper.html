AlterForeignDataWrapper

定義

  • 種別: function
  • 定義箇所: src/backend/commands/foreigncmds.c:684

目的

既存の外部データラッパー（FDW）定義を変更する。ハンドラ関数、バリデーター関数、およびオプションの更新をサポートする。

シグネチャまたは形状

ObjectAddress AlterForeignDataWrapper(ParseState *pstate, AlterFdwStmt *stmt)

入力と出力

  • **入力**:
  • pstate: パーサー状態。
  • stmt: 変更内容を保持する AlterFdwStmt 構造体。
  • **出力**:
  • 変更されたFDWの ObjectAddress を返す。

前提条件と不変条件

  • 実行にはスーパーユーザー権限が必要。
  • 対象のFDWがシステムカタログ pg_foreign_data_wrapper に存在すること。

アルゴリズムまたは振る舞い

1. pg_foreign_data_wrapper を RowExclusiveLock で開く。 2. スーパーユーザー権限を確認し、対象FDWをシステムキャッシュから検索する。 3. parse_func_options を呼び出し、指定されたハンドラ関数およびバリデーター関数の変更を解析する。 4. ハンドラまたはバリデーターが変更される場合、警告を発行する。 5. オプションが指定されている場合、transformGenericOptions を用いて既存のオプションを変換・検証する。 6. heap_modify_tuple と CatalogTupleUpdate を使用してシステムカタログを更新する。 7. ハンドラまたはバリデーターが変更された場合、既存の依存関係を削除し、新しい関数への依存関係を pg_depend に再登録する。 8. InvokeObjectPostAlterHook を呼び出し、変更を通知する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/tcop/utility.c（SQLコマンド実行時）。
  • **呼び出し先**: table_open, superuser, SearchSysCacheCopy1, parse_func_options, transformGenericOptions, heap_modify_tuple, CatalogTupleUpdate, deleteDependencyRecordsForClass, recordDependencyOn, InvokeObjectPostAlterHook, table_close 等。

関連データ構造

  • AlterFdwStmt: 変更対象のFDW名、関数オプション、および汎用オプションを保持する。
  • Form_pg_foreign_data_wrapper: システムカタログ pg_foreign_data_wrapper の行構造。
  • ObjectAddress: 変更されたオブジェクトの識別子。

失敗モードと境界条件

  • スーパーユーザーでない場合、ERRCODE_INSUFFICIENT_PRIVILEGE でエラー終了する。
  • 指定されたFDWが存在しない場合、ERRCODE_UNDEFINED_OBJECT でエラー終了する。

保守担当者向けメモ

  • 本関数はDDL操作を行うため、実行時のFDW動作（fdwapi.c 等）には直接関与しない。
  • ハンドラやバリデーターの変更は、既存の外部テーブルやサーバーに影響を与える可能性があるため、警告メッセージの維持が重要である。
