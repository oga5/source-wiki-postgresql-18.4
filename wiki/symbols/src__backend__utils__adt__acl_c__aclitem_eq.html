aclitem_eq

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/acl.c:748

目的

2つの AclItem 構造体が等しいかどうかを判定する。

シグネチャまたは形状

Datum aclitem_eq(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS を介して2つの AclItem ポインタ（a1, a2）を受け取る。
  • **出力**: 両者が等しい場合は true、そうでない場合は false を Datum 型として返す。

前提条件と不変条件

  • 入力される a1 および a2 は有効な AclItem 構造体へのポインタであること。

アルゴリズムまたは振る舞い

以下の3つのフィールドがすべて一致する場合にのみ true を返す。 1. ai_privs（権限ビット） 2. ai_grantee（被付与者） 3. ai_grantor（付与者）

呼び出し元と呼び出し先

  • **呼び出し元**: 不明（ソースコード上の直接的な呼び出し元情報は提供されていません）。
  • **呼び出し先**: PG_GETARG_ACLITEM_P および PG_RETURN_BOOL マクロを使用。

関連データ構造

  • AclItem: 権限付与の最小単位を保持する構造体。

失敗モードと境界条件

  • 入力ポインタが NULL の場合の挙動については、本ソースコード内では明示されていません。

保守担当者向けメモ

  • 本関数は AclItem の等価性比較を行う基本的なユーティリティです。比較ロジックを変更する場合は、AclItem 構造体の定義（src/include/utils/acl.h）との整合性に注意してください。
