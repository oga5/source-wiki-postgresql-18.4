pgp_get_unicode_mode

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/pgp.c:339

目的

PGP コンテキスト (PGP_Context) に設定されている Unicode モードの状態を取得します。

シグネチャまたは形状

int pgp_get_unicode_mode(PGP_Context *ctx);

入力と出力

  • **入力**: PGP_Context *ctx - PGP 処理の状態を保持するコンテキスト構造体へのポインタ。
  • **出力**: int - コンテキスト内の unicode_mode フィールドの値（真偽値またはフラグ）。

前提条件と不変条件

  • 引数 ctx は有効な PGP_Context 構造体を指している必要があります。

アルゴリズムまたは振る舞い

コンテキスト構造体 ctx が保持する unicode_mode メンバの値をそのまま返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/pgcrypto/pgp-pgsql.c:388
  • contrib/pgcrypto/pgp-pgsql.c:515
  • **呼び出し先**:
  • 本関数は他の関数を呼び出しません。

関連データ構造

  • PGP_Context: pgp.h で定義される PGP 処理の状態管理構造体。

失敗モードと境界条件

  • ctx が NULL の場合、セグメンテーション違反が発生する可能性があります。本関数自体には NULL チェックは含まれていません。

保守担当者向けメモ

  • 本関数は単純なゲッターです。Unicode モードの設定変更が必要な場合は、対応するセッター関数（pgp_set_unicode_mode 等）を確認してください。
