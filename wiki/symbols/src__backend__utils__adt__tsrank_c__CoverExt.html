CoverExt

定義

  • 種別: struct
  • 定義箇所: src/backend/utils/adt/tsrank.c:588

目的

全文検索における近接度ランキング（Cover Density）計算において、文書内のクエリ一致範囲を保持するための構造体です。

シグネチャまたは形状

typedef struct {
    int            pos;
    int            p;
    int            q;
    DocRepresentation *begin;
    DocRepresentation *end;
} CoverExt;

入力と出力

本構造体はデータ保持用であり、関数としての入出力は持ちません。Cover 関数において、計算結果の格納先として利用されます。

前提条件と不変条件

  • begin および end は、DocRepresentation 配列内の有効な範囲を指す必要があります。
  • p および q は、クエリのカバー範囲を示すインデックスとして使用されます。

アルゴリズムまたは振る舞い

Cover 関数内での再帰的な探索処理において、クエリをカバーする文書内の特定範囲（begin から end まで）を記録するために使用されます。これにより、単語間の距離に基づいたランキングスコアの算出を支援します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/adt/tsrank.c 内の Cover 関数および ts_rank_cd 実装関数（861行目、891行目など）から、計算用の一時変数としてインスタンス化されます。
  • **呼び出し先**: 本構造体自体はデータ保持用であり、直接的な関数呼び出しは行いません。

関連データ構造

  • DocRepresentation: 文書内の単語位置情報を保持する構造体。CoverExt はこの配列の範囲を指し示します。
  • QueryRepresentation: クエリの構造を保持するデータ構造。

失敗モードと境界条件

  • begin または end が NULL の状態でアクセスするとクラッシュの原因となります。
  • Cover 関数の再帰処理において、スタック深さが過大になるような複雑なクエリや長い文書では、計算が制限される可能性があります。

保守担当者向けメモ

  • Cover 関数の再帰的な探索ロジックと密接に結合しています。本構造体のフィールドを変更する場合は、Cover 関数内のポインタ操作および範囲計算ロジックへの影響を考慮してください。
