目的

ヒンディー語のステミング（語幹抽出）処理を実行するための実行環境（SN_env）を生成する。

シグネチャまたは形状

extern struct SN_env * hindi_UTF_8_create_env(void);

入力と出力

  • 入力: なし
  • 出力: 初期化された SN_env 構造体へのポインタを返す。

前提条件と不変条件

  • Snowballランタイムの SN_create_env 関数が利用可能であること。
  • 戻り値が NULL でない場合、呼び出し元は最終的に hindi_UTF_8_close_env を呼び出してメモリを解放する責任を負う。

アルゴリズムまたは振る舞い

内部的に SN_create_env(0, 0) を呼び出し、ヒンディー語ステマー用の環境を構築して返却する。

  • 呼び出し元: 提供されたソース内には明示的な呼び出し元は記載されていない。
  • 呼び出し先: SN_create_env（Snowballランタイム関数）。

関連データ構造

  • SN_env: ステミング処理の状態を保持する構造体。

失敗モードと境界条件

  • SN_create_env がメモリ不足等で失敗した場合、NULL を返す可能性がある。
