CopyToCSVOneRow

定義

  • 種別: function
  • 定義箇所: src/backend/commands/copyto.c:251

目的

CSV形式の COPY TO 操作において、単一のタプルを処理して出力バッファへ書き出すための関数です。

シグネチャまたは形状

static void CopyToCSVOneRow(CopyToState cstate, TupleTableSlot *slot)

入力と出力

  • **入力**:
  • cstate: COPY TO 操作の現在の状態を保持する CopyToState 構造体。
  • slot: 出力対象のデータを含む TupleTableSlot。
  • **出力**: なし（副作用として cstate が管理する出力先へデータが書き込まれます）。

前提条件と不変条件

  • cstate は有効な CopyToState である必要があります。
  • slot は処理対象のタプルを保持している必要があります。

アルゴリズムまたは振る舞い

内部的に CopyToTextLikeOneRow を呼び出し、CSVモード（true）を指定して処理を委譲します。これにより、CSV特有のエスケープ処理や区切り文字の適用が行われます。

呼び出し元と呼び出し先

  • **呼び出し元**: COPY TO の実行ループ内（CopyToRoutine 構造体の CopyToOneRow フィールド経由で参照）。
  • **呼び出し先**: CopyToTextLikeOneRow。

関連データ構造

  • CopyToState: COPY TO 操作の全状態を保持する構造体。
  • TupleTableSlot: クエリ実行エンジンがタプルを保持するために使用する構造体。

失敗モードと境界条件

  • メモリ不足や出力先（ファイル、パイプ、フロントエンド）への書き込みエラーが発生した場合、PostgreSQLの標準的なエラーハンドリングメカニズムにより中断されます。

保守担当者向けメモ

  • 本関数は CopyToTextOneRow と共通のロジックを共有するラッパーです。CSV形式の出力仕様を変更する場合は、CopyToTextLikeOneRow の実装を確認してください。
