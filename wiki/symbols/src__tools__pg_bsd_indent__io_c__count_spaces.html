count_spaces

定義

  • 種別: function
  • 定義箇所: src/tools/pg_bsd_indent/io.c:549

目的

指定されたバッファ内の文字列を走査し、タブ文字を考慮した現在のカラム位置を計算して返します。

シグネチャまたは形状

int count_spaces(int cur, char *buffer)

入力と出力

  • **入力**:
  • cur: 走査開始時のカラム位置。
  • buffer: 走査対象の文字列（ヌル終端文字列）。
  • **出力**:
  • 文字列をすべて処理した後の最終的なカラム位置を int 型で返します。

前提条件と不変条件

  • buffer は有効なヌル終端文字列である必要があります。
  • タブ幅の計算には、システムで定義されたタブ設定が暗黙的に使用されます。

アルゴリズムまたは振る舞い

内部的に count_spaces_until 関数を呼び出し、終端条件として NULL を渡すことで、バッファ全体を走査してカラム位置を算出します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/tools/pg_bsd_indent/io.c (複数箇所: s_lab や s_code のカラム計算に使用)
  • src/tools/pg_bsd_indent/pr_comment.c (コメント内のカラム位置計算に使用)
  • **呼び出し先**:
  • count_spaces_until

関連データ構造

  • s_lab, s_code: indent_globs.h で定義されたグローバルなバッファポインタ。

失敗モードと境界条件

  • buffer が NULL の場合の挙動は count_spaces_until の実装に依存します。
  • cur が負の値である場合の動作は定義されていません。

保守担当者向けメモ

  • 本関数は count_spaces_until のラッパーです。ロジックの変更が必要な場合は、まず count_spaces_until を確認してください。
