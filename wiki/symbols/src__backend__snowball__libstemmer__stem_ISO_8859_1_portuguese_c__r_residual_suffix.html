r_residual_suffix

  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_portuguese.c:814

目的

単語の末尾に残存する特定の接尾辞を、領域制約を満たす場合に削除する。

シグネチャまたは形状

static int r_residual_suffix(struct SN_env * z)

入力と出力

  • 入力: SN_env 構造体（現在の単語データおよび領域情報を含む）。
  • 出力: 処理が成功した場合は 1、失敗または条件不一致の場合は 0、エラー発生時は負の値を返す。

前提条件と不変条件

  • z が有効な SN_env 構造体を指していること。
  • a_7 配列に定義された接尾辞ルールが適用可能であること。
  • 削除対象の接尾辞が RV 領域内に存在すること。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置を ket に設定する。 2. a_7 配列に基づき、後方から接尾辞を検索する (find_among_b)。見つからない場合は 0 を返す。 3. マッチした接尾辞の開始位置を bra に設定する。 4. r_RV(z) を呼び出し、対象が RV 領域内にあるか検証する。検証に失敗した場合はその結果を返す。 5. slice_del(z) を呼び出し、該当する接尾辞を削除する。削除結果を返す。

呼び出し元と呼び出し先

  • 呼び出し元: stem_ISO_8859_1_portuguese.c および他の言語実装（catalan.c, french.c, spanish.c 等）のステミング関数から呼び出される。
  • 呼び出し先: find_among_b, r_RV, slice_del。

関連データ構造

  • struct SN_env: ステミング処理の環境状態を保持する構造体。
  • a_7: 接尾辞の検索ルールを定義した among 構造体の配列。

失敗モードと境界条件

  • find_among_b がマッチしない場合、0 を返して終了する。
  • r_RV が領域外と判定した場合、その戻り値（0 または負の値）を伝播する。
  • slice_del がメモリ操作等で失敗した場合、負の値を返す。
