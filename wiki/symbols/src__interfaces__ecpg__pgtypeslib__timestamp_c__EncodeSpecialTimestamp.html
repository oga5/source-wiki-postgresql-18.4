EncodeSpecialTimestamp

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/pgtypeslib/timestamp.c:190

目的

特殊なタイムスタンプ値（infinity や -infinity）を、対応する文字列形式に変換してバッファへ書き込みます。

シグネチャまたは形状

static void EncodeSpecialTimestamp(timestamp dt, char *str)

入力と出力

  • **入力**:
  • dt: 変換対象の timestamp 型値。
  • str: 結果の文字列を格納するためのバッファ（呼び出し元で十分な領域を確保している必要がある）。
  • **出力**: なし（str が指すメモリ領域に文字列が書き込まれる）。

前提条件と不変条件

  • dt は TIMESTAMP_IS_NOBEGIN または TIMESTAMP_IS_NOEND のいずれかの特殊値である必要があります。
  • str は、EARLY または LATE 定数文字列を格納できる十分なサイズが必要です。

アルゴリズムまたは振る舞い

1. TIMESTAMP_IS_NOBEGIN(dt) が真であれば、str に EARLY をコピーします。 2. TIMESTAMP_IS_NOEND(dt) が真であれば、str に LATE をコピーします。 3. 上記のいずれでもない場合、abort() を呼び出しプログラムを終了させます。

呼び出し元と呼び出し先

  • **呼び出し元**: src/interfaces/ecpg/pgtypeslib/timestamp.c:278 内で呼び出されています。
  • **呼び出し先**: strcpy（標準ライブラリ）、abort（標準ライブラリ）。
  • 注: ツールにより src/backend/utils/adt/timestamp.c 等の他ファイルからの参照も示唆されていますが、これらは本関数と同一名称の別定義（バックエンド側の実装）である可能性が高く、本ファイル内の static 関数としての呼び出し関係とは区別する必要があります。

関連データ構造

  • timestamp: int64（マイクロ秒単位）として定義されたデータ型。

失敗モードと境界条件

  • 特殊値以外の通常のタイムスタンプ値が渡された場合、abort() が実行されプロセスが異常終了します。

保守担当者向けメモ

  • 本関数は static であり、timestamp.c 内でのみ使用されます。
  • EARLY および LATE の具体的な文字列定義については、dt.h などの関連ヘッダーファイルを確認してください。
