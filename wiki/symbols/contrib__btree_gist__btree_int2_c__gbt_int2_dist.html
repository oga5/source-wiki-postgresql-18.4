gbt_int2_dist

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_int2.c:72

目的

2つの int2 型値間の距離を計算し、float8 型として返します。

シグネチャまたは形状

static float8 gbt_int2_dist(const void *a, const void *b, FmgrInfo *flinfo)

入力と出力

  • **入力**:
  • a, b: 比較対象の int2 型値へのポインタ。
  • flinfo: 関数管理情報（本関数内では未使用）。
  • **出力**:
  • 2値間の距離を表す float8 値。

前提条件と不変条件

  • 入力ポインタ a および b は、有効な int2 型のデータ領域を指している必要があります。
  • GET_FLOAT_DISTANCE マクロが int16 型に対して正しく定義されている環境である必要があります。

アルゴリズムまたは振る舞い

1. GET_FLOAT_DISTANCE マクロを呼び出し、int16 型として解釈した a と b の差分を計算します。 2. 計算された距離を float8 型に変換して返します。

呼び出し元と呼び出し先

  • **呼び出し元**: ソースコード内の参照情報に基づくと、本ファイル内の静的構造体（tinfo 等）から参照されている可能性がありますが、具体的な呼び出し箇所はソース抜粋からは特定できません。
  • **呼び出し先**: GET_FLOAT_DISTANCE マクロ（btree_utils_num.h 由来と推測される）を呼び出します。

関連データ構造

  • int16: PostgreSQL の 2バイト整数型。

失敗モードと境界条件

  • 入力値が NULL の場合の挙動は定義されていません。
  • 整数演算のオーバーフローについては、common/int.h の依存関係により適切に処理されることが期待されます。

保守担当者向けメモ

  • 本関数は btree_utils_num.c で提供される汎用的な距離計算ロジックを int2 型に適用するためのラッパーです。
  • 類似の関数として int2_dist が存在しますが、本関数は GiST インデックスの内部処理用として定義されています。
