shiftright128

定義

  • 種別: function
  • 定義箇所: src/common/d2s_intrinsics.h:47

目的

2つの 64ビット値を連結した 128ビット値に対し、指定されたビット数だけ右シフトを行うための低レベルな算術演算プリミティブです。

シグネチャまたは形状

static inline uint64 shiftright128(const uint64 lo, const uint64 hi, const uint32 dist)

入力と出力

  • **入力**:
  • lo: 128ビット値の下位 64ビット。
  • hi: 128ビット値の上位 64ビット。
  • dist: 右シフトを行うビット数。
  • **出力**:
  • 右シフト後の結果のうち、下位 64ビットを uint64 として返します。

前提条件と不変条件

  • dist < 64 である必要があります（Assert により検証済み）。
  • Ryu アルゴリズムの現在の実装では、dist は通常 [2, 59] または [49, 58] の範囲で利用されます。

アルゴリズムまたは振る舞い

1. dist が 64 未満であることをアサートします。 2. コンパイラ組み込み関数 __shiftright128(lo, hi, dist) を呼び出し、連結された 128ビット値に対する右シフトを実行します。 3. 結果の下位 64ビットを返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/common/d2s.c 内の複数の箇所で、浮動小数点数から文字列への変換処理中に呼び出されています。
  • **呼び出し先**: コンパイラ組み込み関数 __shiftright128 を呼び出します。

関連データ構造

  • uint64, uint32: 算術演算に使用される固定幅整数型。

失敗モードと境界条件

  • dist >= 64 の場合、アサーションエラーが発生します。
  • Ryu アルゴリズムの将来的な変更により、64ビット以上のシフトが必要となった場合は、本関数の実装を調整する必要があります。

保守担当者向けメモ

  • 本関数は src/common/d2s_intrinsics.h で定義されており、プラットフォーム固有の最適化を抽象化する役割を担います。
  • Assert(dist < 64) は現在の Ryu アルゴリズムの制約に基づいています。変更時は数学的な正当性とシフト範囲の再検証を行ってください。
