SessionEndType

定義

  • 種別: enum
  • 定義箇所: src/include/pgstat.h:53

目的

バックエンドセッションが終了した際の理由を分類・記録するための列挙型です。統計システムにおいて、セッション終了時の状態を識別するために使用されます。

シグネチャまたは形状

typedef enum SessionEndType
{
    DISCONNECT_NOT_YET,     /* まだアクティブ */
    DISCONNECT_NORMAL,      /* 正常終了 */
    DISCONNECT_CLIENT_EOF,  /* クライアントからのEOFによる切断 */
    DISCONNECT_FATAL,       /* 致命的エラーによる切断 */
    DISCONNECT_KILLED       /* 強制終了 */
} SessionEndType;

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: なし（列挙型定義）

前提条件と不変条件

  • pgStatSessionEndCause 変数の値として使用されます。
  • セッションがアクティブな間は DISCONNECT_NOT_YET が維持されることが想定されます。

アルゴリズムまたは振る舞い

セッション終了処理において、終了原因を本型のいずれかの値に設定することで、統計システムがセッションの終了状況を追跡可能にします。

呼び出し元と呼び出し先

  • 呼び出し元（参照元）:
  • src/backend/utils/activity/pgstat_database.c:32: pgStatSessionEndCause の初期値として使用。
  • src/include/pgstat.h:840: pgStatSessionEndCause の外部変数宣言で使用。
  • 呼び出し先:
  • 提供されたソース情報からは、本型を引数や戻り値として直接呼び出す関数は特定されていません。

関連データ構造

  • pgStatSessionEndCause: 本型を保持するグローバル変数（src/include/pgstat.h:840 で宣言）。

失敗モードと境界条件

  • 列挙型に含まれない値が代入された場合の挙動は未定義です。
  • DISCONNECT_NOT_YET から他の状態への遷移は、セッション終了のライフサイクルに依存します。

保守担当者向けメモ

  • 統計情報の互換性に影響するため、列挙子の追加や順序変更を行う際は注意してください。
  • セッション終了時の統計収集ロジック（pgstat_database.c 等）と密接に関連しているため、変更時は関連する統計処理への影響を確認してください。
