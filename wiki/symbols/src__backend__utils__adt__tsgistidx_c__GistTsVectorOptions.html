GistTsVectorOptions

定義

  • 種別: struct
  • 定義箇所: src/backend/utils/adt/tsgistidx.c:29

目的

tsvector に対する GiST インデックス作成時に指定可能なオプション（特にシグネチャ長 siglen）を保持するための構造体です。

シグネチャまたは形状

typedef struct
{
	int32		vl_len_;
	int			siglen;
} GistTsVectorOptions;

入力と出力

  • 入力: インデックス作成時のオプションパラメータ。
  • 出力: siglen フィールドを通じて、インデックスのシグネチャ長設定を保持します。

前提条件と不変条件

  • vl_len_ は varlena ヘッダーであり、直接操作してはなりません。
  • siglen はインデックスの精度とサイズを決定するパラメータとして使用されます。

アルゴリズムまたは振る舞い

本構造体は、init_local_reloptions を介してインデックスのオプションとして初期化され、インデックス操作中に PG_GET_OPCLASS_OPTIONS() を通じて siglen 値が参照されます。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/backend/utils/adt/tsgistidx.c:804: init_local_reloptions 内で構造体サイズとして参照。
  • src/backend/utils/adt/tsgistidx.c:807: offsetof を用いて siglen フィールドのオフセットが指定される。
  • src/backend/utils/adt/tsgistidx.c:38: PG_GET_OPCLASS_OPTIONS() の戻り値をキャストして siglen を取得。

関連データ構造

  • SignTSVector: GiST インデックス内に格納されるキー構造体。本オプションで設定された siglen に基づいて動作します。

失敗モードと境界条件

  • siglen の設定値が不適切な場合、インデックスのサイズや検索性能に影響を及ぼします。
  • varlena ヘッダーの不適切な操作はメモリ破壊を引き起こすため、直接の編集は禁止されています。

保守担当者向けメモ

  • siglen はデフォルト値（124バイト）から変更可能です。変更時は検索性能とインデックスサイズのトレードオフを考慮してください。
  • 本構造体は reloptions インフラストラクチャの一部として機能します。
