CloseMatViewIncrementalMaintenance

定義

  • 種別: function
  • 定義箇所: src/backend/commands/matview.c:975

目的

マテリアライズド・ビューの増分メンテナンス処理の終了を通知し、内部的なメンテナンス深度を減算する。

シグネチャまたは形状

static void CloseMatViewIncrementalMaintenance(void)

入力と出力

  • 入力: なし
  • 出力: なし（副作用として静的変数 matview_maintenance_depth を更新）

前提条件と不変条件

  • matview_maintenance_depth が 0 以上であることを保証するアサーションが存在する。

アルゴリズムまたは振る舞い

1. 静的変数 matview_maintenance_depth を 1 減算する。 2. matview_maintenance_depth が負の値になっていないかを確認する。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/commands/matview.c 内の増分メンテナンス処理終了箇所（例: 884行目付近）。
  • 呼び出し先: なし。

関連データ構造

  • matview_maintenance_depth: 増分メンテナンスのネスト状態を追跡する静的整数変数。

失敗モードと境界条件

  • matview_maintenance_depth が 0 の状態で呼び出された場合、Assert によりプロセスが停止する。

保守担当者向けメモ

  • 本関数は OpenMatViewIncrementalMaintenance と対で使用される。メンテナンス処理のネスト管理に使用されており、直接的なデータ操作は行わない。
