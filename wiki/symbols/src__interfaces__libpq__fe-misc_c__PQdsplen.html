PQdsplen

定義

  • 種別: function
  • 定義箇所: src/interfaces/libpq/fe-misc.c:1275

目的

指定されたエンコーディングにおいて、文字列が端末上で占有する表示幅（カラム数）を計算します。

シグネチャまたは形状

int PQdsplen(const char *s, int encoding);

入力と出力

  • **入力**:
  • s: 測定対象の文字列（ヌル終端文字列）。
  • encoding: 文字列のエンコーディングを示す整数値。
  • **出力**:
  • 文字列の表示幅を整数で返します。

前提条件と不変条件

  • s は有効な文字列ポインタである必要があります。
  • encoding は libpq がサポートする有効なエンコーディング識別子である必要があります。

アルゴリズムまたは振る舞い

内部的に pg_encoding_dsplen を呼び出し、指定されたエンコーディングに基づいた文字幅計算を委譲します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/bin/psql/prompt.c
  • src/fe_utils/mbprint.c
  • src/fe_utils/print.c
  • **呼び出し先**:
  • pg_encoding_dsplen（内部関数）

関連データ構造

  • 特になし。

失敗モードと境界条件

  • s が NULL の場合の挙動は、依存先の pg_encoding_dsplen の実装に依存します。

保守担当者向けメモ

  • 本関数は libpq-fe.h で公開されている API です。
  • 表示幅の計算ロジックを変更する場合は、pg_encoding_dsplen 側の実装を確認してください。
