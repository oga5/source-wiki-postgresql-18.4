xl_brin_desummarize

定義

  • 種別: struct
  • 定義箇所: src/include/access/brin_xlog.h:133

目的

BRIN（Block Range Index）インデックスのデサマライズ（要約解除）操作をWAL（Write-Ahead Logging）に記録するためのデータ構造です。

シグネチャまたは形状

typedef struct xl_brin_desummarize
{
    BlockNumber pagesPerRange;
    BlockNumber heapBlk;
    OffsetNumber regOffset;
} xl_brin_desummarize;

入力と出力

  • **入力**:
  • pagesPerRange: 範囲あたりのブロック数。
  • heapBlk: 無効化対象となるヒープブロック番号。
  • regOffset: 通常のインデックスページ内で削除対象となるアイテムのオフセット。
  • **出力**: なし（WALレコードの一部として永続化される）。

前提条件と不変条件

  • 本構造体は、BRINインデックスの特定の範囲に対する要約情報を無効化する際に使用されます。
  • SizeOfBrinDesummarize マクロにより、レコードサイズの計算が定義されています。

アルゴリズムまたは振る舞い

1. インデックスのデサマライズ操作時に、本構造体にメタデータを格納します。 2. XLogInsert を介してWALに書き込まれます。 3. リカバリ時には、brin_redo がこの構造体を読み取り、指定された heapBlk および regOffset に基づいてインデックスページ上の情報を無効化します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/access/brin/brin_revmap.c: デサマライズ処理の開始時に本構造体がインスタンス化されます。
  • **呼び出し先**:
  • src/backend/access/brin/brin_xlog.c: リカバリ時のRedo処理において、XLogRecGetData を通じて本構造体が読み取られます。
  • src/backend/access/rmgrdesc/brindesc.c: WALログのデバッグ出力（記述）時に参照されます。

関連データ構造

  • xl_brin_desummarize は brin_xlog.h で定義される他のWALレコード構造体（xl_brin_insert 等）と同様に、BRINのリカバリプロセスを構成します。

失敗モードと境界条件

  • WALレコードの読み取り時にデータが破損している場合、XLogRecGetData によるキャスト後の値が不正になる可能性があります。
  • regOffset が対象ページ内の有効なオフセット範囲外である場合、Redo処理でエラーが発生する可能性があります。

保守担当者向けメモ

  • 本構造体を変更した場合は、必ず SizeOfBrinDesummarize マクロの定義を更新してください。
  • リカバリ時の具体的なページ操作ロジックについては、対となる src/backend/access/brin/brin_xlog.c の実装を確認してください。
