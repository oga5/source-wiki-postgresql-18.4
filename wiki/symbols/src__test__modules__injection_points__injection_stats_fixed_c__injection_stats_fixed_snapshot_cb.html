injection_stats_fixed_snapshot_cb

定義

  • 種別: function
  • 定義箇所: src/test/modules/injection_points/injection_stats_fixed.c:96

目的

固定数統計（fixed-numbered statistics）のスナップショットを生成し、リセット後のオフセットを差し引いて現在の統計値を算出する。

シグネチャまたは形状

static void injection_stats_fixed_snapshot_cb(void)

入力と出力

  • **入力**: 共有メモリ上の PgStatShared_InjectionPointFixed（統計カウンタとリセットオフセット）。
  • **出力**: pgstat_get_custom_snapshot_data で取得したスナップショット領域への書き込み。

前提条件と不変条件

  • PGSTAT_KIND_INJECTION_FIXED 種別の統計が pgstat フレームワークに登録されていること。
  • 共有メモリ上の統計データは changecount によって整合性が保護されていること。

アルゴリズムまたは振る舞い

1. pgstat_get_custom_shmem_data および pgstat_get_custom_snapshot_data を介して共有メモリとスナップショット領域を取得する。 2. pgstat_copy_changecounted_stats を使用し、共有メモリからスナップショットへ統計値をコピーする。 3. LWLock（共有モード）を取得し、リセット時のオフセット値（reset_offset）をローカル変数にコピーしてロックを解放する。 4. FIXED_COMP マクロを用いて、スナップショット値から reset_offset の各フィールド（numattach, numdetach, numrun, numcached, numloaded）を減算し、補正を行う。

呼び出し元と呼び出し先

  • **呼び出し元**: pgstat フレームワークのコールバック構造体 injection_stats_fixed の .snapshot_cb フィールドとして登録されている。
  • **呼び出し先**: pgstat_get_custom_shmem_data, pgstat_get_custom_snapshot_data, pgstat_copy_changecounted_stats, LWLockAcquire, LWLockRelease。

関連データ構造

  • PgStatShared_InjectionPointFixed: 共有メモリ上の統計管理構造体。
  • PgStat_StatInjFixedEntry: 統計カウンタの各フィールドを保持する構造体。

失敗モードと境界条件

  • 共有メモリの初期化が完了していない状態で呼び出された場合、pgstat_get_custom_shmem_data が NULL を返す可能性がある。
  • LWLock 競合が発生した場合、スナップショット取得処理が待機状態となる。

保守担当者向けメモ

  • FIXED_COMP マクロはこの関数内でのみ定義・使用されるローカルな補正ロジックである。
  • 統計フィールドの追加・変更時は、本関数の減算処理にも反映させる必要がある。
