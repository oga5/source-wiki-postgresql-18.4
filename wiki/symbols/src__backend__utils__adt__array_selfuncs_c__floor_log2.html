floor_log2

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/array_selfuncs.c:1088

目的

与えられた32ビット符号なし整数 n に対し、その対数（底は2）の整数部分（$\lfloor \log_2(n) \rfloor$）を計算します。

シグネチャまたは形状

static int floor_log2(uint32 n)

入力と出力

  • **入力**: uint32 n （対象となる整数）
  • **出力**: int （n の2を底とする対数の整数部分。n=0 の場合は -1）

前提条件と不変条件

  • n が 0 の場合、関数は -1 を返します。
  • n が 1 以上の場合、戻り値は n の最上位ビットの位置（0から始まるインデックス）と一致します。

アルゴリズムまたは振る舞い

二分探索的なビットシフト操作を用いて、最上位ビットの位置を特定します。 1. n が 0 ならば -1 を即座に返します。 2. n を 16, 8, 4, 2, 1 ビットずつ右シフトし、その都度 logval に加算することで、最上位ビットがどの範囲にあるかを判定します。 3. 最終的な logval を返却します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/array_selfuncs.c:557 （統計情報の計算ロジック内で使用）
  • **呼び出し先**:
  • 特になし（標準的な算術演算のみを使用）

関連データ構造

  • なし

失敗モードと境界条件

  • n = 0: 戻り値は -1 となります。
  • n = 1: 戻り値は 0 となります。
  • n = 0xFFFFFFFF: 戻り値は 31 となります。

  • 本関数は static であり、array_selfuncs.c 内でのみ使用されます。
