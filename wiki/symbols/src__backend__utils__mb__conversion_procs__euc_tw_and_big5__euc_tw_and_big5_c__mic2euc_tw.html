mic2euc_tw

定義

  • 種別: function
  • 定義箇所: src/backend/utils/mb/conversion_procs/euc_tw_and_big5/euc_tw_and_big5.c:377

目的

MULE_INTERNAL (MIC) エンコーディングの文字列を EUC_TW エンコーディングへ変換します。

シグネチャまたは形状

static int mic2euc_tw(const unsigned char *mic, unsigned char *p, int len, bool noError)

入力と出力

  • **入力**:
  • mic: 変換元となる MULE_INTERNAL 文字列のポインタ。
  • p: 変換先バッファのポインタ。
  • len: 入力文字列のバイト長。
  • noError: 変換エラー時に例外を投げず中断するかどうかを指定するフラグ。
  • **出力**: 変換処理したバイト数を返します。変換後のバッファ p にはヌル終端が付与されます。

前提条件と不変条件

  • 入力文字列は MULE_INTERNAL 形式である必要があります。
  • p は変換後の文字列を格納するのに十分な領域を確保している必要があります。

アルゴリズムまたは振る舞い

1. 入力文字列を1文字ずつ走査します。 2. ASCII文字（高位ビットが未設定）はそのままコピーします。 3. マルチバイト文字の場合、pg_encoding_verifymbchar で妥当性を検証します。 4. MULE_INTERNAL のプレフィックス（LC_CNS11643_1 等）に基づき、以下の変換を行います。

  • LC_CNS11643_1: そのままコピー。
  • LC_CNS11643_2: SS2 + 0xa2 を付与して変換。
  • LCPRV2_B かつ LC_CNS11643_3〜7: SS2 + 変換後のコードを付与。

    • 5. 変換できない文字や不正なエンコーディングに遭遇した場合、noError が false ならエラーを報告し、true ならその時点までの変換結果を返して終了します。

呼び出し元と呼び出し先

  • **呼び出し元**: 同一ファイル内の mic_to_euc_tw 関数から呼び出されます。
  • **呼び出し先**: pg_encoding_verifymbchar、report_invalid_encoding、report_untranslatable_char を呼び出します。

関連データ構造

  • SS2: EUC_TWにおける補助文字セットを示すシングルシフト2。
  • LC_CNS11643_x: MULE_INTERNALにおける各文字セットの識別子。

失敗モードと境界条件

  • 入力文字列が不正なエンコーディングである場合、noError が false であればエラーを報告します。
  • 変換対象外の文字コードが含まれる場合、noError が false であればエラーを報告します。
  • noError が true の場合、エラー発生時に処理を中断し、そこまでに変換したバイト数を返します。

保守担当者向けメモ

  • 本関数は MULE_INTERNAL から EUC_TW への変換ロジックをカプセル化しています。
  • 変換テーブルや計算式は big5.c 等の外部依存ではなく、本関数内の条件分岐で直接制御されています。
