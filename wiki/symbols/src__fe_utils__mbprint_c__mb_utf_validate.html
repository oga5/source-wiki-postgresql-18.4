mb_utf_validate

定義

  • 種別: function
  • 定義箇所: src/fe_utils/mbprint.c:135

目的

UTF-8 文字列内の不正なバイトシーケンスを検出し、それらを除去して文字列をクリーンアップします。

シグネチャまたは形状

static void mb_utf_validate(unsigned char *pwcs)

入力と出力

  • **入力**: pwcs (ヌル終端された UTF-8 文字列へのポインタ)
  • **出力**: なし（入力されたバッファを直接書き換えるインプレース処理）

前提条件と不変条件

  • 入力文字列はヌル終端されている必要があります。
  • utf_charcheck 関数が UTF-8 の妥当性判定に使用可能である必要があります。

アルゴリズムまたは振る舞い

1. 文字列を先頭から走査します。 2. utf_charcheck を呼び出し、現在のバイト位置から始まる文字が妥当か判定します。 3. 妥当な場合（戻り値 > 0）、その文字をバッファの現在位置に保持（またはコピー）し、ポインタを進めます。 4. 不当な場合（戻り値 <= 0）、そのバイトをスキップして次のバイトへ進みます。 5. 走査終了後、書き込み位置が元の文字列の終端と異なる場合は、新しい終端にヌル文字を書き込みます。

呼び出し元と呼び出し先

  • **呼び出し元**: src/fe_utils/mbprint.c 内の関数から呼び出されています（行136および395）。
  • **呼び出し先**: utf_charcheck を呼び出します。

関連データ構造

  • 特になし（unsigned char 配列を直接操作）。

失敗モードと境界条件

  • 空文字列が渡された場合、何も行わずに終了します。
  • 不正なバイトシーケンスが連続する場合、それらはすべて除去されます。
  • インプレース処理のため、入力バッファのサイズは変更されませんが、文字列の長さが短くなる可能性があります。

保守担当者向けメモ

  • 本関数は UTF-8 特化です。他のエンコーディングへの対応が必要な場合は、ロジックの拡張または別関数の実装が必要です。
  • utf_charcheck の仕様変更が本関数の動作に直接影響するため、修正時は同期を確認してください。
