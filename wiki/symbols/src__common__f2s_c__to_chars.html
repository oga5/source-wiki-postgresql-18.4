to_chars

定義

  • 種別: function
  • 定義箇所: src/common/f2s.c:562

目的

Ryuアルゴリズムによって計算された浮動小数点数の仮数と指数を、文字列バッファへ書き出すためのフォーマット関数です。

シグネチャまたは形状

static inline int to_chars(const floating_decimal_32 v, const bool sign, char *const result)

入力と出力

  • **入力**:
  • v: 変換対象の floating_decimal_32 構造体（仮数と指数を保持）。
  • sign: 負数であることを示すフラグ。
  • result: 文字列を書き込むためのバッファ（呼び出し元で十分なサイズを確保する必要がある）。
  • **出力**:
  • int: 書き込まれた文字列の総バイト数。

前提条件と不変条件

  • result バッファは、符号、数値、小数点、指数部（e+xx）を格納するのに十分なサイズであること。
  • DIGIT_TABLE が利用可能であること。

アルゴリズムまたは振る舞い

1. **符号処理**: sign が真の場合、バッファ先頭に '-' を書き込みます。 2. **フォーマット選択**: 指数 exp が -4 以上 6 未満の範囲であれば、to_chars_f を呼び出し固定小数点表記で出力します。 3. **科学的表記**: 範囲外の場合、以下の手順で科学的表記（d.dde+ee）を行います。

  • v.exponent == 0 の場合、末尾の不要なゼロを除去して指数を調整します。
  • DIGIT_TABLE を使用して、数値を高速に文字列へ変換します。
  • olength > 1 の場合、小数点と指数部（e、符号、2桁の指数）を付与します。

    • 4. **戻り値**: 書き込んだ総文字数を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/common/f2s.c:769
  • **呼び出し先**:
  • decimalLength (マクロまたは関数)
  • to_chars_f
  • memcpy
  • DIGIT_TABLE (データテーブル)

関連データ構造

  • floating_decimal_32: 変換対象の数値を保持する構造体。
  • DIGIT_TABLE: 数値から文字列への高速変換用テーブル。

失敗モードと境界条件

  • result バッファのオーバーフローに対する保護は関数内には存在しません。
  • exp が負の場合、指数部に '-' を付与し、正の場合は '+' を付与します。

保守担当者向けメモ

  • to_chars_f と FLOAT_SHORTEST_DECIMAL_LEN は、固定小数点出力の閾値（-4 から 6）に依存しています。これらの閾値を変更する場合は注意してください。
  • DIGIT_TABLE のインデックス計算は、2桁単位の変換に基づいています。修正時はインデックスのオフセットに注意してください。
