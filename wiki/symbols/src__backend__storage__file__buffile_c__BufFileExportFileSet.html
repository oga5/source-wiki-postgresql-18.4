BufFileExportFileSet

定義

  • 種別: function
  • 定義箇所: src/backend/storage/file/buffile.c:393

目的

BufFile を他のバックエンドと共有するための準備として、バッファをフラッシュし、読み取り専用状態に移行させる。

シグネチャまたは形状

void BufFileExportFileSet(BufFile *file)

入力と出力

  • **入力**: file (BufFile *) - 共有対象となるバッファファイルへのポインタ。
  • **出力**: なし（副作用として file の状態を更新）。

前提条件と不変条件

  • file は FileSet に属している必要がある（file->fileset != NULL）。
  • file は読み取り専用状態であってはならない（!file->readOnly）。

アルゴリズムまたは振る舞い

1. BufFileFlush(file) を呼び出し、バッファ内の未書き込みデータを物理ファイルへフラッシュする。 2. file->readOnly を true に設定し、以降の書き込みを禁止する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/sort/logtape.c:1043
  • **呼び出し先**:
  • BufFileFlush

関連データ構造

  • BufFile: 物理ファイルセグメントを管理する論理ファイル構造体。
  • FileSet: 複数のバックエンド間で一時ファイルを共有するための管理構造体。

失敗モードと境界条件

  • file->fileset が NULL の場合、アサーション違反により異常終了する。
  • 既に読み取り専用の BufFile に対して呼び出した場合、アサーション違反により異常終了する。

保守担当者向けメモ

  • 本関数は、共有準備の最終段階で呼び出されることを想定している。
  • BufFileFlush の実行により、ディスクへの物理的な書き込みが発生するため、I/O 負荷に注意すること。
