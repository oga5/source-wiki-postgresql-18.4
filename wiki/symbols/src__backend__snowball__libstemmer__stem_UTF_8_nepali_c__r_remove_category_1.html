r_remove_category_1

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_nepali.c:284

目的

ネパール語のステミング処理において、カテゴリ1に分類される特定の接尾辞を識別し、必要に応じて文字列から削除する。

シグネチャまたは形状

static int r_remove_category_1(struct SN_env * z)

入力と出力

  • 入力: SN_env * z （Snowballランタイム環境構造体）
  • 出力: int （処理が成功した場合は1、マッチしなかった場合は0、エラー発生時は負の値を返す）

前提条件と不変条件

  • z は有効な SN_env 構造体を指していること。
  • ステミング処理は文字列の末尾から前方に向かって行われる。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を z->ket に設定する。 2. find_among_b を使用し、テーブル a_0（17要素）を用いて末尾から接尾辞を検索する。 3. マッチしなかった場合は0を返して終了する。 4. マッチした場合、現在のカーソル位置を z->bra に設定する。 5. マッチした接尾辞のインデックス（among_var）に応じて以下の処理を行う:

  • case 1: slice_del を呼び出し、マッチした部分を削除する。
  • case 2: 特定の条件（s_0 または s_1 との比較）に基づき、条件を満たさない場合にのみ slice_del を実行する。

    • 6. 処理結果（成功またはエラーコード）を返す。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_nepali.c 内のメイン処理（nepali_UTF_8_stem 等から参照される）。
  • 呼び出し先: find_among_b, slice_del, eq_s_b （いずれもSnowballランタイム関数）。

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_0: カテゴリ1の接尾辞を定義する among 構造体の配列。

失敗モードと境界条件

  • slice_del が負の値を返した場合、その値をそのまま呼び出し元へ伝播させる。
  • find_among_b でマッチが見つからない場合は0を返し、文字列の変更は行わない。

保守担当者向けメモ

  • 本関数はSnowballコンパイラによって自動生成されたコードである。直接編集せず、元のSnowballスクリプト（.sbl）を修正すること。
  • UTF-8マルチバイト文字を扱うため、among テーブル内のバイト列の整合性に注意が必要である。
