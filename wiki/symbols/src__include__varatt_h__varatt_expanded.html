varatt_expanded

定義

  • 種別: struct
  • 定義箇所: src/include/varatt.h:74

目的

メモリ上に展開されたオブジェクト（Expanded Object）を指し示すためのTOASTポインタ構造体です。インライン化されていないデータへの間接参照を提供します。

シグネチャまたは形状

typedef struct varatt_expanded
{
    ExpandedObjectHeader *eohptr;
} varatt_expanded;

入力と出力

  • **入力**: ExpandedObjectHeader へのポインタ。
  • **出力**: なし（構造体定義）。

前提条件と不変条件

  • 本構造体はTOASTポインタの一種として扱われます。
  • eohptr は有効な ExpandedObjectHeader を指している必要があります。

アルゴリズムまたは振る舞い

  • 本構造体は、データがメモリ上で展開された形式であることを示す識別子として機能します。
  • VARTAG_IS_EXPANDED マクロにより、対象のデータがこの形式であるかどうかが判定されます。
  • 外部ストレージにあるデータとは異なり、メモリ上のオブジェクトを直接参照する仕組みを提供します。

呼び出し元と呼び出し先

  • **呼び出し元（参照元）**:
  • src/include/utils/expandeddatum.h: EXPANDED_POINTER_SIZE の計算に使用。
  • src/include/varatt.h: VARTAG_IS_EXPANDED マクロ内でのサイズ判定に使用。
  • src/backend/utils/adt/expandeddatum.c: 構造体ポインタとして利用。
  • **呼び出し先**:
  • ExpandedObjectHeader（型定義のみ参照）。

関連データ構造

  • ExpandedObjectHeader: varatt_expanded が指し示す実体。
  • varatt_external: 他のTOASTポインタ形式。
  • vartag_external: TOASTポインタの種類を識別する列挙型。

失敗モードと境界条件

  • eohptr が NULL の場合、デリファレンス時にクラッシュする可能性があります。
  • 本構造体はアライメントが保証されない可能性があるため、直接アクセスする際は memcpy 等の適切な手段を用いる必要があります。

保守担当者向けメモ

  • 本構造体は src/include/utils/expandeddatum.h で定義される ExpandedObjectHeader に強く依存しています。
  • TOASTポインタのサイズ計算（EXPANDED_POINTER_SIZE）において、VARHDRSZ_EXTERNAL と組み合わせて使用される点に注意してください。
