IndexTupleSize

定義

  • 種別: function
  • 定義箇所: src/include/access/itup.h:71

目的

インデックスタプル（IndexTupleData）のメモリ上の総サイズをバイト単位で取得する。

シグネチャまたは形状

static inline Size IndexTupleSize(const IndexTupleData *itup)

入力と出力

  • **入力**: itup - サイズを取得対象とする IndexTupleData 構造体へのポインタ。
  • **出力**: Size 型のタプルサイズ。

前提条件と不変条件

  • itup は有効な IndexTupleData 構造体を指している必要がある。
  • t_info フィールドには、INDEX_SIZE_MASK でマスク可能なサイズ情報が格納されていること。

アルゴリズムまたは振る舞い

itup->t_info フィールドに対して INDEX_SIZE_MASK をビット論理積（AND）演算し、タプルのサイズ部分を抽出して返却する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/amcheck/verify_gin.c
  • contrib/amcheck/verify_nbtree.c
  • contrib/pageinspect/btreefuncs.c
  • **呼び出し先**:
  • 本関数はインライン関数であり、特定の関数を呼び出すことはない。

関連データ構造

  • IndexTupleData: インデックスタプルのヘッダー構造体。t_info フィールドを保持する。

失敗モードと境界条件

  • itup が NULL の場合、デリファレンスによりクラッシュする可能性がある。
  • t_info のサイズビットが不正な値である場合、誤ったサイズを返す。

保守担当者向けメモ

  • 本関数は頻繁に呼び出されるため、インライン化されている。
  • t_info のビットレイアウトを変更する場合は、INDEX_SIZE_MASK の定義と整合性が取れているか確認すること。
