isbn_cast_from_ean13

定義

  • 種別: function
  • 定義箇所: contrib/isn/isn.c:1058

目的

PostgreSQL の ean13 型から isbn 型への明示的な型変換（キャスト）を実行します。

シグネチャまたは形状

Datum isbn_cast_from_ean13(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_GETARG_EAN13(0) を介して ean13 型の値を受け取ります。
  • **出力**: 変換後の値を PG_RETURN_EAN13(result) を介して ean13 型（ISBN として解釈される値）として返します。

前提条件と不変条件

  • 入力値は ean13 型として有効である必要があります。
  • 内部的に ean2isn 関数を呼び出し、ISBN 形式への変換を試みます。

アルゴリズムまたは振る舞い

1. 引数から ean13 値を取得します。 2. ean2isn 関数を呼び出し、第2引数に false（厳密な検証）、第4引数に ISBN 型を指定して変換処理を行います。 3. 変換結果を result に格納し、呼び出し元へ返します。

呼び出し元と呼び出し先

  • **呼び出し元**: PostgreSQL の関数マネージャ（PG_FUNCTION_INFO_V1 マクロにより登録）。
  • **呼び出し先**: ean2isn（contrib/isn/isn.c 内の内部関数）。

関連データ構造

  • ean13: 内部ストレージ形式（uint64）。
  • ISBN: ean2isn に渡される識別子種別定数。

失敗モードと境界条件

  • ean2isn 内部で ISBN 形式への変換が不可能な場合（範囲外の値など）の挙動は、ean2isn の実装に依存します。

保守担当者向けメモ

  • 本関数は PostgreSQL の型キャストシステムから呼び出されることを想定しています。
  • ean2isn の第2引数が false であるため、変換時にチェックデジット等の厳密な検証が行われます。
