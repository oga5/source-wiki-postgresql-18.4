quote_if_needed

定義

  • 種別: function
  • 定義箇所: src/bin/psql/stringutils.c:291

目的

入力文字列が引用符（クォート）を必要とするか判定し、必要な場合はクォートで囲み、内部のエスケープ処理を施した新しい文字列を生成します。

シグネチャまたは形状

char *quote_if_needed(const char *source, const char *entails_quote,
                      char quote, char escape, bool force_quote,
                      int encoding);

入力と出力

  • **入力**:
  • source: 処理対象の文字列（NULL不可）。
  • entails_quote: クォートを強制する文字が含まれる文字列。
  • quote: 使用するクォート文字。
  • escape: エスケープ文字。
  • force_quote: 強制的にクォートするかどうかのフラグ。
  • encoding: 文字エンコーディング（PQmblenBounded用）。
  • **出力**:
  • クォートが必要と判断された場合は、クォート済みの新しい文字列（pg_mallocで確保）。
  • 不要な場合は NULL。

前提条件と不変条件

  • source は NULL であってはなりません。
  • quote は '\0' であってはなりません。
  • 戻り値が NULL でない場合、呼び出し元は free() を使用してメモリを解放する必要があります。

アルゴリズムまたは振る舞い

1. force_quote が true の場合、need_quotes を true に初期化します。 2. 入力文字列を走査し、以下の条件のいずれかに該当する場合、need_quotes を true に設定します。

  • 現在の文字が quote と一致する。
  • 現在の文字が escape と一致する。
  • 現在の文字が entails_quote に含まれる。

    • 3. PQmblenBounded を使用してマルチバイト文字を考慮しつつ、文字をコピーします。 4. 走査終了後、need_quotes が false であれば、確保したメモリを解放し NULL を返します。それ以外の場合は、クォートで囲んだ文字列を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/bin/psql/tab-complete.in.c:6269
  • src/bin/psql/tab-complete.in.c:6831
  • **呼び出し先**:
  • pg_malloc (メモリ確保)
  • strchr (文字検索)
  • PQmblenBounded (マルチバイト文字長判定)
  • free (メモリ解放)

関連データ構造

  • 特になし。

失敗モードと境界条件

  • pg_malloc が失敗した場合、メモリ不足によりプロセスが終了する可能性があります。
  • source が空文字列の場合、quote で囲まれた空の文字列（例: ''）が返されます。

保守担当者向けメモ

  • 本関数は PQescapeStringConn の代わりにはなりません。あくまで psql 内部の引数解析用として設計されています。
  • マルチバイト文字の扱いは PQmblenBounded に依存しています。
