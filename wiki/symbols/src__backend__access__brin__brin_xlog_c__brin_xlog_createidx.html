brin_xlog_createidx

定義

  • 種別: function
  • 定義箇所: src/backend/access/brin/brin_xlog.c:23

目的

BRINインデックス作成時のWALレコードをリプレイし、インデックスのメタページを初期化します。

シグネチャまたは形状

static void brin_xlog_createidx(XLogReaderState *record)

入力と出力

  • **入力**: XLogReaderState *record - リプレイ対象のWALレコード。
  • **出力**: なし（副作用としてバッファプール上のメタページを更新）。

前提条件と不変条件

  • record から取得したデータが xl_brin_createidx 構造体として正しく解釈可能であること。
  • XLogInitBufferForRedo が有効なバッファを返すこと。

アルゴリズムまたは振る舞い

1. XLogRecGetData を用いてWALレコードから xl_brin_createidx データを抽出します。 2. XLogInitBufferForRedo を呼び出し、インデックスのメタページを初期化するためのバッファを取得します。 3. brin_metapage_init を呼び出し、レコードに含まれる pagesPerRange および version を用いてメタページの内容を初期化します。 4. PageSetLSN を使用して、メタページのLSNをWALレコードの終了位置（record->EndRecPtr）に更新します。 5. MarkBufferDirty でバッファをダーティ状態にし、UnlockReleaseBuffer でロックを解放します。

呼び出し元と呼び出し先

  • **呼び出し元**: brin_redo（src/backend/access/brin/brin_xlog.c 内のディスパッチャ）。
  • **呼び出し先**:
  • XLogRecGetData
  • XLogInitBufferForRedo
  • brin_metapage_init
  • PageSetLSN
  • MarkBufferDirty
  • UnlockReleaseBuffer

関連データ構造

  • XLogReaderState: WALレコードの読み取り状態を保持。
  • xl_brin_createidx: インデックス作成時のWALレコード構造体（pagesPerRange および version を保持）。

失敗モードと境界条件

  • XLogInitBufferForRedo が失敗した場合、アサーション（Assert(BufferIsValid(buf))）により異常終了します。

保守担当者向けメモ

  • 本関数はインデックス作成時のメタページ初期化のみを担当します。REVMAP等の他の構造の初期化は別のWALレコード処理で行われます。
  • brin_metapage_init への依存が強いため、メタページのレイアウト変更時は本関数と brin_metapage_init の両方を確認してください。
