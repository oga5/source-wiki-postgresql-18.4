sv2cstr

定義

  • 種別: static inline 関数
  • 定義箇所: src/pl/plperl/plperl.h:88

目的

Perl のスカラー値 (SV) を PostgreSQL のメモリ管理下にあるデータベースエンコーディングの文字列 (char *) に変換します。

シグネチャまたは形状

static inline char *sv2cstr(SV *sv)

入力と出力

  • **入力**: SV *sv (変換対象の Perl スカラー値)
  • **出力**: char * (PostgreSQL のメモリコンテキストで確保された、データベースエンコーディングに変換済みの文字列)

前提条件と不変条件

  • 呼び出し側は、PostgreSQL のメモリコンテキストのライフサイクルを管理する必要があります。
  • データベースエンコーディングが SQL_ASCII の場合、エンコーディング変換はスキップされます。
  • Perl の SV が読み取り専用オブジェクトや型グロブ等の特殊な型である場合、内部で一時的なコピーが作成されます。

アルゴリズムまたは振る舞い

1. **保護**: SvREADONLY や特定の型（型グロブ等）を持つ sv の場合、newSVsv でコピーを作成します。それ以外の場合は参照カウントをインクリメントします。 2. **抽出**: データベースエンコーディングが SQL_ASCII であれば SvPV を使用し、それ以外の場合は SvPVutf8 を使用して Perl から文字列を取得します。 3. **変換**: utf_u2e を呼び出し、取得した文字列をデータベースエンコーディングへ変換します。 4. **解放**: 手順1でインクリメントした参照カウントを SvREFCNT_dec で減らし、一時的な SV を解放します。 5. **返却**: 変換後の文字列ポインタを返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/hstore_plperl/hstore_plperl.c
  • contrib/jsonb_plperl/jsonb_plperl.c
  • src/pl/plperl/plperl.c
  • **呼び出し先**:
  • utf_u2e (エンコーディング変換用)
  • Perl API (dTHX, SvPV, SvPVutf8, newSVsv, SvREFCNT_inc_simple_void, SvREFCNT_dec, SvREADONLY, isGV_with_GP, SvTYPE)
  • PostgreSQL API (GetDatabaseEncoding)

関連データ構造

  • SV (Perl のスカラー値構造体)
  • char * (PostgreSQL 側の文字列バッファ)

失敗モードと境界条件

  • SvPVutf8 は特定のオブジェクト（読み取り専用オブジェクト等）に対して予期せぬ動作をする可能性があるため、本関数内でコピーを作成して回避しています。
  • 埋め込み NULL バイトが含まれる場合、utf_u2e への変換プロセスを通じて適切にエラー処理が行われます。

保守担当者向けメモ

  • 本関数は palloc を使用してメモリを確保するため、呼び出し側でのメモリリークに注意してください。
  • postgres.h およびシステムヘッダーの後にインクルードする必要があります。
