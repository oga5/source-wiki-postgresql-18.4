float_compare_desc

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/array_selfuncs.c:1180

目的

float 型の値を降順（大きい順）にソートするための比較関数を提供します。

シグネチャまたは形状

static int float_compare_desc(const void *key1, const void *key2)

入力と出力

  • **入力**: 比較対象となる float 型へのポインタ key1 および key2。
  • **出力**:
  • key1 > key2 の場合: -1
  • key1 < key2 の場合: 1
  • key1 == key2 の場合: 0

前提条件と不変条件

  • 入力ポインタは有効な float 型の値を指している必要があります。
  • qsort 等の標準的な比較関数インターフェースに準拠しています。

アルゴリズムまたは振る舞い

1. key1 および key2 が指す値を float 型としてデリファレンスします。 2. 値を比較し、降順ソートを実現するために、標準的な昇順比較とは逆の値を返却します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/adt/array_selfuncs.c 内のソート処理（qsort 等）から参照されています。
  • **呼び出し先**: 特になし（標準ライブラリの演算のみ）。

関連データ構造

  • なし。

失敗モードと境界条件

  • 入力ポインタが NULL の場合の動作は定義されていません。
  • NaN（非数）が含まれる場合の比較結果は、標準的な float の比較規則に従います。

保守担当者向けメモ

  • 本関数は降順ソート専用です。昇順が必要な場合は別の比較関数を使用してください。
