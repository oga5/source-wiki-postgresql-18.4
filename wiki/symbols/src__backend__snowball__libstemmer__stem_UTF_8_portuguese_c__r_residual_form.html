r_residual_form

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_portuguese.c:833

目的

ポルトガル語のステミング処理において、特定の残余的な接尾辞パターンを検出し、条件に応じて削除または置換を行う。

シグネチャまたは形状

static int r_residual_form(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング処理の現在の状態を保持する構造体）
  • 出力: 成功時に 1、失敗または条件不一致時に 0、エラー発生時に負の値を返す。

前提条件と不変条件

  • z が有効な SN_env 構造体を指していること。
  • find_among_b を使用して、定義済みの接尾辞テーブル a_8 と照合を行う。

アルゴリズムまたは振る舞い

1. z->ket を現在のカーソル位置 z->c に設定し、a_8 テーブルを用いて後方検索を行う。 2. マッチした among_var の値に応じて分岐する。

  • **case 1**: r_RV(z) を呼び出し、RV領域内であることを確認後、slice_del(z) で削除する。その後、直前の文字が「u」かつ「g」の前、または「i」かつ「c」の前であるかを判定し、条件を満たせば再度 r_RV(z) を経て削除を行う。
  • **case 2**: slice_from_s(z, 1, s_10) を呼び出し、該当する接尾辞を別の文字列に置換する。

    • 3. 処理完了後、成功を示す 1 を返す。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_portuguese.c および src/backend/snowball/libstemmer/stem_ISO_8859_1_portuguese.c 内のステミング処理フローから呼び出される。
  • 呼び出し先: find_among_b, r_RV, slice_del, slice_from_s（いずれも Snowball ランタイム関数）。

関連データ構造

  • SN_env: ステミングの状態管理用構造体。
  • a_8: 削除対象となる接尾辞パターンを保持するテーブル。

失敗モードと境界条件

  • find_among_b がマッチしない場合、0 を返して終了する。
  • r_RV や slice_del 等の内部関数が負の値を返した場合、その値をそのまま伝播させて終了する。
  • 文字列の先頭や境界条件（z->lb）に達した場合、安全に処理を中断するガード条件が含まれている。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによって自動生成されたコードであるため、直接編集せず、元の .sbl ファイルを修正すること。
  • case 1 内の文字判定（'u', 'g', 'i', 'c'）はポルトガル語の綴り規則に基づいた特殊な削除条件である。
