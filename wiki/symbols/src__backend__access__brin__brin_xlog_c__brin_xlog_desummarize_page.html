brin_xlog_desummarize_page

定義

  • 種別: function
  • 定義箇所: src/backend/access/brin/brin_xlog.c:268

目的

BRINインデックスのページ要約解除（desummarize）操作をWALリプレイ時に再実行し、REVMAPの更新とインデックスページからのタプル削除を同期的に行う。

シグネチャまたは形状

static void brin_xlog_desummarize_page(XLogReaderState *record)

入力と出力

  • **入力**: XLogReaderState *record（リプレイ対象のWALレコード）。
  • **出力**: なし（副作用としてバッファの更新とLSNの書き込みを行う）。

前提条件と不変条件

  • record には xl_brin_desummarize 構造体が含まれている必要がある。
  • REVMAPページ（バッファ0）とインデックスページ（バッファ1）の順序でWALレコードが構成されていること。

アルゴリズムまたは振る舞い

1. **REVMAPの更新**: XLogReadBufferForRedo を使用してREVMAPページを取得する。BLK_NEEDS_REDO の場合、brinSetHeapBlockItemptr を呼び出し、対象ヒープブロックのポインタを無効化する。 2. **インデックスページの更新**: XLogReadBufferForRedo を使用してインデックスページを取得する。BLK_NEEDS_REDO の場合、PageIndexTupleDeleteNoCompact を使用して指定されたオフセットのタプルを削除する。 3. **コミット**: 各操作後に PageSetLSN でLSNを更新し、MarkBufferDirty でバッファをダーティにする。 4. **クリーンアップ**: 取得したバッファが有効な場合、UnlockReleaseBuffer を呼び出してロックを解放する。

呼び出し元と呼び出し先

  • **呼び出し元**: brin_redo（src/backend/access/brin/brin_xlog.c 内）。
  • **呼び出し先**: XLogRecGetData, XLogReadBufferForRedo, ItemPointerSetInvalid, brinSetHeapBlockItemptr, PageSetLSN, BufferGetPage, MarkBufferDirty, BufferIsValid, UnlockReleaseBuffer, PageIndexTupleDeleteNoCompact。

関連データ構造

  • xl_brin_desummarize: WALレコードのデータ構造。
  • XLogReaderState: WAL読み取り状態。
  • Buffer: バッファ管理用ハンドル。

失敗モードと境界条件

  • XLogReadBufferForRedo が BLK_NOTFOUND を返した場合、そのページに対する操作はスキップされる。
  • BufferIsValid が偽の場合、ロック解放は行われない。

保守担当者向けメモ

  • 本関数はREVMAPとインデックスページの2つのバッファを操作するため、リプレイ順序の整合性が重要である。
  • PageIndexTupleDeleteNoCompact を使用しているため、ページ内のタプル配置は再圧縮されない。
