pg_atomic_init_u32_impl

定義

  • 種別: static inline function
  • 定義箇所: src/include/port/atomics/generic.h:150

目的

pg_atomic_uint32 型の変数を指定された値で初期化する。

シグネチャまたは形状

static inline void pg_atomic_init_u32_impl(volatile pg_atomic_uint32 *ptr, uint32 val_)

入力と出力

  • ptr: 初期化対象となる volatile pg_atomic_uint32 構造体へのポインタ。
  • val_: 設定する uint32 型の初期値。
  • 戻り値: なし。

前提条件と不変条件

  • ptr は有効なメモリ領域を指している必要がある。
  • 本関数はアトミック操作の初期化用であり、他のスレッドが当該変数に対して並行して操作を行っていないタイミングで使用されることが前提である。

アルゴリズムまたは振る舞い

  • ptr が指す構造体の value メンバに、引数 val_ を直接代入する。

呼び出し元と呼び出し先

  • 呼び出し元: src/include/port/atomics.h:225
  • 呼び出し先: なし（直接的な代入操作のみ）。

関連データ構造

  • pg_atomic_uint32: アトミック操作対象となる32ビット整数を保持する構造体。

失敗モードと境界条件

  • 特筆すべき失敗モードはない。ただし、volatile 修飾されているため、コンパイラによる最適化で代入が省略されることはない。

保守担当者向けメモ

  • 本関数は初期化専用である。実行中のアトミックな更新には pg_atomic_write_u32_impl 等の適切なアトミック操作関数を使用すること。
  • 本ファイルは atomics.h からのみインクルードされる設計であるため、直接の呼び出しは避けること。
