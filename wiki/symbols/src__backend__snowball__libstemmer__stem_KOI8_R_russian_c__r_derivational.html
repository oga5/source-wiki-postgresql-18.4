r_derivational

定義

  • **種別**: 関数
  • **定義箇所**: src/backend/snowball/libstemmer/stem_KOI8_R_russian.c:520

目的

KOI8-Rエンコーディングのロシア語単語から、派生語尾（derivational suffix）を特定し、条件を満たす場合に削除する。

シグネチャまたは形状

static int r_derivational(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z（Snowballステミング環境）
  • **出力**: 成功時に 1、処理対象外または失敗時に 0、あるいは slice_del の戻り値（負の値）を返す。

前提条件と不変条件

  • z->c（現在のカーソル位置）が単語の末尾付近にあること。
  • a_6 配列に定義された派生語尾パターンが検索対象となる。
  • 削除対象の語尾は R2 領域内にある必要がある。

アルゴリズムまたは振る舞い

1. z->ket を現在のカーソル位置に設定する。 2. 単語の末尾から2文字目以降を確認し、KOI8-Rコード 212 または 216 で終わるか判定する。 3. find_among_b を使用し、a_6 配列内のパターンを後方から検索する。 4. マッチした場合、z->bra を設定し、r_R2 を呼び出して対象が R2 領域内にあるか検証する。 5. 検証に成功した場合、slice_del を実行して該当部分を削除する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_KOI8_R_russian.c および src/backend/snowball/libstemmer/stem_UTF_8_russian.c 内のメインステミング処理から呼び出される。
  • **呼び出し先**: find_among_b, r_R2, slice_del（いずれもSnowballランタイムユーティリティ）。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_6: 派生語尾パターンを格納した among 構造体の配列。

失敗モードと境界条件

  • 単語の長さが不足している場合や、末尾の文字コードが一致しない場合は 0 を返す。
  • r_R2 が失敗した場合、その戻り値をそのまま返す。
  • slice_del が失敗した場合、その戻り値をそのまま返す。

保守担当者向けメモ

  • 本関数はSnowballコンパイラによる自動生成コードである。直接編集せず、元のSnowballスクリプト（.sbl）を修正すること。
  • KOI8-Rの文字コード 212 および 216 は、ロシア語の特定の文字に対応する。デバッグ時はエンコーディングの不一致に注意が必要。
