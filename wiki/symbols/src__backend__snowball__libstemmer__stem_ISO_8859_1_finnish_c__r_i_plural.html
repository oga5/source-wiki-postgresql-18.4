r_i_plural

定義

  • **種別**: function
  • **定義箇所**: src/backend/snowball/libstemmer/stem_ISO_8859_1_finnish.c:511

目的

フィンランド語のステミング処理において、特定の複数形語尾（i を含むもの）を単語末尾から削除する。

シグネチャまたは形状

static int r_i_plural(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z （Snowballステミング環境）
  • **出力**: int （成功時に 1、失敗時に 0、エラー時に負の値を返す）

前提条件と不変条件

  • z->I[1] に格納された境界位置（R1領域など）より現在位置 z->c が後ろにあること。
  • 処理対象の単語末尾が文字コード 105 ('i') または 106 ('j') であること。

アルゴリズムまたは振る舞い

1. z->I[1] を現在の境界（lb）として設定し、処理範囲を制限する。 2. 単語末尾が 'i' または 'j' であるかを確認する。 3. a_8 配列に定義されたパターンと末尾を照合する（find_among_b）。 4. 照合に成功した場合、該当する語尾を slice_del で削除する。 5. 処理終了後、元の境界設定を復元する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_ISO_8859_1_finnish.c 内のメイン処理フローから呼び出される。
  • **呼び出し先**: find_among_b（パターン照合）、slice_del（文字列削除）。

関連データ構造

  • struct SN_env: ステミング実行時の状態を保持する構造体。
  • a_8: 複数形語尾の照合に使用される among 構造体配列。

失敗モードと境界条件

  • z->c < z->I[1] の場合、処理を行わず 0 を返す。
  • 単語末尾が 'i' または 'j' でない場合、または a_8 に一致しない場合は 0 を返す。
  • slice_del が失敗した場合、その戻り値（負の値）をそのまま返す。
