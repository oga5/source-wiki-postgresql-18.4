Macaddr8PGetDatum

定義

  • 種別: static inline 関数
  • 定義箇所: src/include/utils/inet.h:168

目的

macaddr8 型のポインタを、PostgreSQL の関数マネージャ（fmgr）が扱う Datum 型へ変換します。

シグネチャまたは形状

static inline Datum Macaddr8PGetDatum(const macaddr8 *X)

入力と出力

  • **入力**: const macaddr8 *X （変換対象の macaddr8 構造体へのポインタ）
  • **出力**: Datum （PostgreSQL 内部表現としての Datum 値）

前提条件と不変条件

  • 入力ポインタ X は有効な macaddr8 構造体を指している必要があります。
  • 本関数は PointerGetDatum を呼び出す単純なラッパーであり、メモリのコピーや変換は行いません。

アルゴリズムまたは振る舞い

1. 引数として受け取った macaddr8 構造体へのポインタ X を PointerGetDatum マクロに渡します。 2. PointerGetDatum の戻り値をそのまま返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/btree_gin/btree_gin.c
  • contrib/btree_gist/btree_macaddr8.c
  • src/include/utils/inet.h 内の PG_RETURN_MACADDR8_P マクロ
  • **呼び出し先**:
  • PointerGetDatum （PostgreSQL のコアマクロ）

関連データ構造

  • macaddr8: MACアドレスを格納する固定長構造体。

失敗モードと境界条件

  • 入力ポインタ X が NULL の場合、PointerGetDatum の仕様に従い (Datum) 0 が返されます。

保守担当者向けメモ

  • 本関数は macaddr8 型のデータ変換を抽象化する目的で提供されています。
  • macaddr8 は固定長データ型であるため、inet 型のような可変長ヘッダーの操作は不要です。
