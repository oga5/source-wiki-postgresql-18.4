is_scram_printable

定義

  • 種別: function
  • 定義箇所: src/backend/libpq/auth-scram.c:778

目的

与えられた文字列が、RFC 5802 で定義される SCRAM の「printable」な文字のみで構成されているかを検証する。

シグネチャまたは形状

static bool is_scram_printable(char *p)

入力と出力

  • **入力**: char *p (検証対象のヌル終端文字列)
  • **出力**: bool (すべてが printable な文字であれば true、それ以外は false)

前提条件と不変条件

  • 入力文字列 p はヌル終端されている必要がある。
  • SCRAM 仕様における printable 文字は、ASCII コードの 0x21 から 0x7E の範囲（ただし , (0x2C) を除く）と定義されている。

アルゴリズムまたは振る舞い

1. 文字列の各文字を順次走査する。 2. 各文字が以下のいずれかの条件に該当する場合、直ちに false を返す。

  • 0x21 未満（制御文字など）
  • 0x7E より大きい（拡張 ASCII など）
  • 0x2C である（カンマ）

    • 3. 文字列の終端まで走査が完了した場合、true を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/libpq/auth-scram.c:1107 において、クライアントから送信された nonce の検証に使用されている。
  • **呼び出し先**: なし（標準ライブラリ関数や他の関数を呼び出さない）。

関連データ構造

  • 特になし。

失敗モードと境界条件

  • 空文字列が渡された場合、ループが実行されず true を返す。
  • カンマ (,) が含まれている場合は false を返す。
  • 非 ASCII 文字や制御文字が含まれている場合は false を返す。

保守担当者向けメモ

  • 本関数は RFC 5802 の定義に厳密に従っている。将来的に SCRAM の仕様が拡張された場合、この printable の定義も変更が必要になる可能性がある。
