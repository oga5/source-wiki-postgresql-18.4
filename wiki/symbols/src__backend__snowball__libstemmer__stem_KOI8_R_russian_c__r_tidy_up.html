r_tidy_up

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_KOI8_R_russian.c:534

目的

ロシア語のステミング処理の最終段階として、特定の接尾辞パターンに基づき単語末尾の調整（削除）を行う。

シグネチャまたは形状

static int r_tidy_up(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z （Snowballのステミング環境状態）
  • **出力**: int 型。成功時は 1、処理対象外または失敗時は 0、あるいは slice_del 実行時のエラーコード（負の値）を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • a_7 配列が find_among_b の検索対象として定義されていること。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置（z->c）から末尾方向の文字を検査し、特定の文字コード条件（0xCE 等を含むパターン）を満たすか判定する。 2. find_among_b を使用して、配列 a_7 に定義された接尾辞パターンを後方から検索する。 3. マッチしたパターン（among_var）に応じて以下の処理を行う：

  • **case 1**: 接尾辞を削除し、さらに直前の特定の文字（0xCE）を2回連続で削除する。
  • **case 2**: 直前の特定の文字（0xCE）を1回削除する。
  • **case 3**: 単純に接尾辞を削除する。

    • 4. 各削除操作において slice_del を呼び出し、エラーが発生した場合はその値を伝播させる。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_KOI8_R_russian.c:671 および src/backend/snowball/libstemmer/stem_UTF_8_russian.c:667 から呼び出される。
  • **呼び出し先**: find_among_b（パターン検索）、slice_del（文字列削除）。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_7: among 構造体の配列（接尾辞パターン定義）。

失敗モードと境界条件

  • カーソル位置が境界（z->lb）以下の場合、または文字コード条件に一致しない場合は 0 を返して終了する。
  • slice_del が負の値を返した場合、そのエラーコードをそのまま呼び出し元に返す。

保守担当者向けメモ

  • 本関数はSnowballコンパイラによって自動生成されたコードである。直接編集せず、元のSnowballスクリプト（.sbl）を修正すること。
  • 0xCE などの16進数はKOI8-Rエンコーディングの文字コードである。デバッグ時はエンコーディングの整合性に注意が必要。
