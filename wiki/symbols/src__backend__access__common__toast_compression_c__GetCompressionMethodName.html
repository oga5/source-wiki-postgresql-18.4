GetCompressionMethodName

定義

  • 種別: function
  • 定義箇所: src/backend/access/common/toast_compression.c:303

目的

圧縮方式を示す内部識別子（char）を、人間が読み取り可能な文字列（"pglz" または "lz4"）に変換します。

シグネチャまたは形状

const char *GetCompressionMethodName(char method);

入力と出力

  • **入力**: method (char) - 圧縮方式を表す識別子。
  • **出力**: const char * - 対応する圧縮方式の名称文字列。

前提条件と不変条件

  • 入力 method は TOAST_PGLZ_COMPRESSION または TOAST_LZ4_COMPRESSION のいずれかである必要があります。

アルゴリズムまたは振る舞い

1. switch 文を用いて method を評価します。 2. TOAST_PGLZ_COMPRESSION の場合は "pglz" を返します。 3. TOAST_LZ4_COMPRESSION の場合は "lz4" を返します。 4. 上記以外の値が渡された場合、elog(ERROR) を呼び出し、処理を中断します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/commands/tablecmds.c:2787
  • src/backend/parser/parse_utilcmd.c:1249
  • **呼び出し先**:
  • elog (PostgreSQLのエラー報告関数)

関連データ構造

  • TOAST_PGLZ_COMPRESSION / TOAST_LZ4_COMPRESSION (圧縮方式を定義する定数)

失敗モードと境界条件

  • 未定義の圧縮方式が渡された場合、elog(ERROR) によりエラーが発生し、関数は正常な文字列を返さずに終了します。

保守担当者向けメモ

  • 本関数は switch 文で網羅的に処理を行っています。新しい圧縮アルゴリズムをサポートする場合は、この switch 文に新しい case を追加する必要があります。
