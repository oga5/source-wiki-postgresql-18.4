IsCatalogTextUniqueIndexOid

定義

  • 種別: function
  • 定義箇所: src/backend/catalog/catalog.c:155

目的

指定されたOIDが、システムカタログにおける特定の「テキスト型の一意インデックス」であるかを判定します。

シグネチャまたは形状

bool IsCatalogTextUniqueIndexOid(Oid relid)

入力と出力

  • **入力**: relid (Oid) - 判定対象のリレーションOID。
  • **出力**: bool - 対象が特定のテキスト一意インデックスOIDであれば true、そうでなければ false。

前提条件と不変条件

  • 本関数はカタログアクセスを行いません。
  • 判定対象として以下のOIDをハードコードされたリストとして保持しています。
  • ParameterAclParnameIndexId
  • ReplicationOriginNameIndex
  • SecLabelObjectIndexId
  • SharedSecLabelObjectIndexId

アルゴリズムまたは振る舞い

引数 relid を switch 文で評価し、定義済みの定数と一致するかを比較します。一致する場合は直ちに true を返し、いずれにも該当しない場合は false を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/storage/buffer/bufmgr.c:4155
  • src/test/regress/regress.c:731
  • **呼び出し先**:
  • 本関数は特定の定数比較のみを行うため、他の関数を呼び出しません。

関連データ構造

  • Oid: PostgreSQLのオブジェクト識別子。

失敗モードと境界条件

  • 該当しないOIDが渡された場合、副作用なく false を返します。
  • 境界条件として、システムカタログのOIDが変更・追加された場合、本関数のリストも更新する必要があります。

保守担当者向けメモ

  • 本関数はカタログアクセスを伴わないため、ロック取得中や低レイヤーの処理から安全に呼び出せます。
  • 新しいシステムカタログのテキスト一意インデックスが追加された際は、本関数の switch 文にケースを追加してください。
