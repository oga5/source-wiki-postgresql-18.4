gcd

定義

  • 種別: function
  • 定義箇所: contrib/tsm_system_time/tsm_system_time.c:315

目的

2つの符号なし32ビット整数 a と b の最大公約数（Greatest Common Divisor）を算出する。

シグネチャまたは形状

static uint32 gcd(uint32 a, uint32 b)

入力と出力

  • **入力**:
  • a (uint32): 比較対象の整数。
  • b (uint32): 比較対象の整数。
  • **出力**:
  • uint32: a と b の最大公約数。

前提条件と不変条件

  • 入力値は uint32 型の範囲内であること。
  • アルゴリズムの性質上、a または b が 0 の場合、もう一方の値を返す（gcd(x, 0) = x）。

アルゴリズムまたは振る舞い

ユークリッドの互除法を用いて最大公約数を求める。 1. a が 0 になるまで以下のループを繰り返す。

  • c に a を一時保存する。
  • a を b % a で更新する。
  • b を c で更新する。

    • 2. ループ終了後、b を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: 同一ファイル内の random_relative_prime（またはそれに準ずるブロック選択ロジック）から呼び出されることが確認されている。
  • **呼び出し先**: 本関数は標準的な算術演算（%）のみを使用し、他の関数を呼び出さない。
  • 注: contrib/pgcrypto や src/backend/utils/adt/int.c 等にも同名の関数や言及が存在するが、本シンボルは tsm_system_time.c 内の静的関数として独立している。

関連データ構造

  • なし。

失敗モードと境界条件

  • a = 0, b = 0 の場合、戻り値は 0 となる。
  • どちらか一方が 0 の場合、0 でない方の値が返される。

保守担当者向けメモ

  • 本関数は tsm_system_rows 等の他のモジュールでも類似の定義が重複して存在する可能性がある。修正を行う際は、本ファイル固有の要件であるか確認すること。
