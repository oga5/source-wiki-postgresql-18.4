italian_ISO_8859_1_stem

定義

  • **種別**: function
  • **定義箇所**: src/backend/snowball/libstemmer/stem_ISO_8859_1_italian.c:982

目的

ISO-8859-1 エンコーディングのイタリア語テキストに対し、接尾辞除去（ステミング）を行い、単語を語幹に正規化する。

シグネチャまたは形状

int italian_ISO_8859_1_stem(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（ステミング対象の文字列と現在の処理状態を保持する環境構造体）
  • **出力**: int（成功時に 1 を返し、エラー発生時に負の値を返す）

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • 処理対象の文字列は ISO-8859-1 エンコーディングであること。

アルゴリズムまたは振る舞い

1. **例外処理**: r_exceptions を呼び出し、例外リストに該当すれば即座に終了する。 2. **前処理**: r_prelude を実行し、特殊文字の置換や母音化処理を行う。 3. **領域分割**: r_mark_regions を呼び出し、ステミングのルール適用範囲（RV, R1, R2）を計算する。 4. **接尾辞除去**:

  • r_attached_pronoun を適用。
  • r_standard_suffix を試行し、失敗した場合は r_verb_suffix を適用する。
  • r_vowel_suffix を適用。

    • 5. **後処理**: r_postlude を実行し、文字の正規化を行う。

呼び出し元と呼び出し先

  • **呼び出し元**: 外部インターフェース（src/include/snowball/libstemmer/stem_ISO_8859_1_italian.h で宣言）。具体的な呼び出し元コードの詳細は提供されていません。
  • **呼び出し先**: r_exceptions, r_prelude, r_mark_regions, r_attached_pronoun, r_standard_suffix, r_verb_suffix, r_vowel_suffix, r_postlude。

関連データ構造

  • SN_env: ステミング処理の実行環境を保持する構造体。

失敗モードと境界条件

  • 各内部関数（r_ で始まる関数）が負の値を返した場合、その値をそのまま呼び出し元へ伝播させる。
  • r_standard_suffix と r_verb_suffix は、マッチングの成否に応じて 0 または 1 を返す。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードであるため、直接編集せず、元の .sbl ファイルを修正すること。
  • 領域（RV, R1, R2）の定義は r_mark_regions に依存しており、ステミングの精度に直結する。
