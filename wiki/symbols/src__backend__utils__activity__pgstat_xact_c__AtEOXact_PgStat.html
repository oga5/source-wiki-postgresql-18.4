AtEOXact_PgStat

定義

  • 種別: function
  • 定義箇所: src/backend/utils/activity/pgstat_xact.c:39

目的

トップレベルトランザクションの終了時（コミットまたはアボート）に、トランザクション内で発生した統計情報の変更を確定または破棄し、統計システムの状態をクリーンアップします。

シグネチャまたは形状

void AtEOXact_PgStat(bool isCommit, bool parallel)

入力と出力

  • isCommit (bool): true の場合、トランザクションのコミットとして統計変更を適用します。false の場合、アボートとして変更を破棄します。
  • parallel (bool): 並列ワーカーのトランザクション終了かどうかを示します。
  • 戻り値: なし。

前提条件と不変条件

  • 本関数はトップレベルトランザクションの終了時に呼び出されることを想定しています。
  • pgStatXactStack が存在する場合、そのネストレベルは 1 であり、スタックの先頭である必要があります。
  • 2相コミット（PREPARE）時には呼び出されません。

アルゴリズムまたは振る舞い

1. AtEOXact_PgStat_Database を呼び出し、データベース統計の終了処理を行います。 2. pgStatXactStack が存在する場合、以下の処理を実行します。

  • AtEOXact_PgStat_Relations を呼び出し、リレーション統計のコミット/アボート処理を行います。
  • AtEOXact_PgStat_DroppedStats を呼び出し、削除予定の統計アイテムを物理削除または破棄します。

    • 3. pgStatXactStack を NULL にリセットし、スタックを解放します。 4. pgstat_clear_snapshot を呼び出し、保持されている統計スナップショットを破棄します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/access/transam/xact.c (トランザクション終了処理)
  • src/backend/access/transam/twophase.c (2相コミット関連処理)
  • **呼び出し先**:
  • AtEOXact_PgStat_Database
  • AtEOXact_PgStat_Relations
  • AtEOXact_PgStat_DroppedStats
  • pgstat_clear_snapshot

関連データ構造

  • pgStatXactStack: トランザクション統計状態を保持するスタックのグローバルポインタ。
  • PgStat_SubXactStatus: トランザクションの統計状態を管理する構造体。

失敗モードと境界条件

  • pgStatXactStack が NULL の場合、リレーションおよび削除統計の処理はスキップされます。
  • Assert により、スタックのネストレベルが 1 でない場合や、スタックが正しく終了していない場合にエラーとなります。

保守担当者向けメモ

  • 本関数は統計情報の「作成」と「削除」をトランザクションの成否と逆の論理で処理します（例：コミット時は削除予定の統計を物理削除する）。
  • 2相コミット（PREPARE）時には本関数は呼ばれず、別途 AtPrepare_PgStat 等が使用される点に注意してください。
