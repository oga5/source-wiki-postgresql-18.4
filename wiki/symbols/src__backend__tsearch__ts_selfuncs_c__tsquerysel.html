tsquerysel

定義

  • 種別: function
  • 定義箇所: src/backend/tsearch/ts_selfuncs.c:151

目的

tsvector 列に対する tsquery の選択率（Selectivity）を推定します。統計情報（MCELEM）の有無に応じて、適切な計算ロジックを選択し、プランナに結果を返します。

シグネチャまたは形状

static Selectivity tsquerysel(VariableStatData *vardata, Datum constval)

入力と出力

  • **入力**:
  • vardata: 対象列の統計情報およびメタデータを含む VariableStatData 構造体。
  • constval: 評価対象の tsquery を保持する Datum。
  • **出力**:
  • Selectivity (double): 推定された選択率（0.0 から 1.0 の範囲）。

前提条件と不変条件

  • constval は TSQuery 型であることが呼び出し元で保証されている必要があります。
  • query->size == 0 の場合、選択率は常に 0.0 となります。
  • MCELEM 統計情報が存在する場合、計算結果に対して 1.0 - stanullfrac を乗算し、NULL 値を考慮した補正を行います。

アルゴリズムまたは振る舞い

1. DatumGetTSQuery を使用して constval を TSQuery 構造体に変換します。 2. vardata->statsTuple が有効な場合:

  • get_attstatsslot を用いて STATISTIC_KIND_MCELEM を取得します。
  • 取得できた場合、mcelem_tsquery_selec を呼び出して頻度に基づいた計算を行います。
  • 取得できない場合、tsquery_opr_selec_no_stats を呼び出して統計情報なしの計算を行います。
  • 最後に stanullfrac を用いて NULL 値による補正を適用します。

    • 3. vardata->statsTuple が無効な場合、tsquery_opr_selec_no_stats を使用して計算します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/tsearch/ts_selfuncs.c 内の tsmatchsel 関数から呼び出されます。
  • **呼び出し先**:
  • DatumGetTSQuery
  • get_attstatsslot / free_attstatsslot
  • mcelem_tsquery_selec
  • tsquery_opr_selec_no_stats

関連データ構造

  • VariableStatData: 列の統計情報を保持する構造体。
  • TSQuery: 全文検索クエリの構造体。
  • AttStatsSlot: pg_statistic から抽出された統計情報を保持する構造体。

失敗モードと境界条件

  • query->size == 0: 空のクエリに対しては即座に 0.0 を返します。
  • 統計情報が欠如している場合: tsquery_opr_selec_no_stats を使用し、デフォルトの確率計算にフォールバックします。

保守担当者向けメモ

  • 本関数は tsmatchsel の内部ロジックの一部です。統計情報の抽出と補正の責務を負っています。
  • 統計情報の更新（ANALYZE）が行われていない場合、推定精度が低下する可能性があります。
