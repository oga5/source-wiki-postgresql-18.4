get_mincount_for_mcv_list

定義

  • 種別: function
  • 定義箇所: src/backend/statistics/mcv.c:147

目的

多変量MCV（Most Common Values）リストを構築する際、統計的に有意とみなされる頻度の最小閾値を算出します。

シグネチャまたは形状

static double get_mincount_for_mcv_list(int samplerows, double totalrows)

入力と出力

  • **入力**:
  • samplerows (int): サンプリングされた行数。
  • totalrows (double): テーブルの総行数。
  • **出力**:
  • double: MCVリストに含めるべき最小頻度閾値。

前提条件と不変条件

  • samplerows および totalrows は正の数であることを想定しています。
  • samplerows が totalrows と等しく、かつ 1 である場合など、分母が 0 になるケースをガードしています。

アルゴリズムまたは振る舞い

1. サンプルサイズ n と総行数 N を用いて、以下の式で閾値を計算します。

  • numer = n * (N - n)
  • denom = N - n + 0.04 * n * (N - 1)

    • 2. denom が 0 の場合は 0.0 を返します。 3. それ以外の場合は numer / denom を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/statistics/mcv.c 内の statext_mcv_build 関数から呼び出されます（ソース内の記述より）。
  • **呼び出し先**: 数値演算のみを実行するため、他の関数呼び出しはありません。

関連データ構造

  • 特になし（数値計算のみを行うユーティリティ関数）。

失敗モードと境界条件

  • n = N = 1 のような極端なケースでは分母が 0 になる可能性がありますが、if (denom == 0.0) によるガードにより 0.0 を返すことで回避しています。

保守担当者向けメモ

  • この関数は、MCVリストの切り詰め処理において、統計的な信頼性を担保するための閾値を決定する重要な役割を担っています。計算式を変更する際は、統計的な妥当性を再検証してください。
