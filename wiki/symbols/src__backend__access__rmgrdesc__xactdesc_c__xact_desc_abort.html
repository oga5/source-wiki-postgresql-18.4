xact_desc_abort

定義

  • 種別: function
  • 定義箇所: src/backend/access/rmgrdesc/xactdesc.c:369

目的

トランザクションのアボート（Abort）に関するWALレコードを人間が読める形式に変換し、指定されたバッファへ出力する。

シグネチャまたは形状

static void xact_desc_abort(StringInfo buf, uint8 info, xl_xact_abort *xlrec, RepOriginId origin_id)

入力と出力

  • buf: 出力先の StringInfo バッファ。
  • info: WALレコードのフラグ情報。
  • xlrec: アボートレコードのデータ構造体。
  • origin_id: レプリケーションオリジンID。
  • 戻り値: なし。buf に整形済み文字列が追記される。

前提条件と不変条件

  • xlrec は有効な xl_xact_abort 構造体を指している必要がある。
  • ParseAbortRecord が内部で呼び出され、xlrec から xl_xact_parsed_abort 構造体へデータが展開される。

アルゴリズムまたは振る舞い

1. ParseAbortRecord を呼び出し、WALレコードからパース済み情報を取得する。 2. プリペアされたトランザクションである場合（parsed.twophase_xid が有効）、そのXIDをバッファに書き込む。 3. トランザクション時刻を文字列化してバッファに追加する。 4. xact_desc_relations を呼び出し、関連するリレーション情報を出力する。 5. xact_desc_subxacts を呼び出し、サブトランザクション情報を出力する。 6. XACT_XINFO_HAS_ORIGIN フラグが立っている場合、レプリケーションオリジン情報（ノードID、LSN、タイムスタンプ）を追記する。 7. xact_desc_stats を呼び出し、統計情報を出力する。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/access/rmgrdesc/xactdesc.c 内のメインディスパッチャ（xact_desc）。
  • 呼び出し先: ParseAbortRecord, appendStringInfo, appendStringInfoString, timestamptz_to_str, xact_desc_relations, xact_desc_subxacts, xact_desc_stats。

関連データ構造

  • xl_xact_abort: WALレコードの生データ構造。
  • xl_xact_parsed_abort: パース後の内部表現。
  • StringInfo: 文字列構築用バッファ。

失敗モードと境界条件

  • parsed.twophase_xid が無効な場合、XIDの出力はスキップされる。
  • レプリケーションオリジン情報がない場合、そのセクションは出力されない。

保守担当者向けメモ

  • 本関数は pg_waldump 等のツールから利用されるため、出力フォーマットを変更する際は互換性に注意すること。
  • ParseAbortRecord との密結合により、WALレコードの構造変更時には両方の修正が必要となる。
