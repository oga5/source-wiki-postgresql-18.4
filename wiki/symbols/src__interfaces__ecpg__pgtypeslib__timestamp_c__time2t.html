time2t

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/pgtypeslib/timestamp.c:20

目的

時・分・秒および小数秒（fsec_t）の各成分を、マイクロ秒単位の int64 値へ変換します。

シグネチャまたは形状

static int64 time2t(const int hour, const int min, const int sec, const fsec_t fsec);

入力と出力

  • **入力**:
  • hour: 時
  • min: 分
  • sec: 秒
  • fsec: 小数秒 (fsec_t)
  • **出力**:
  • 合計マイクロ秒数を表す int64 値。

前提条件と不変条件

  • MINS_PER_HOUR、SECS_PER_MINUTE、USECS_PER_SEC の各定数が dt.h で定義されている必要があります。
  • 入力値が各単位の範囲を超えた場合、計算結果は単純な加算ロジックに従います。

アルゴリズムまたは振る舞い

以下の数式に基づき、すべての単位をマイクロ秒に換算して合算します。 ((((hour * 60 + min) * 60) + sec) * 1,000,000) + fsec

呼び出し元と呼び出し先

  • **呼び出し元**: src/interfaces/ecpg/pgtypeslib/timestamp.c 内の tm2timestamp 関数から呼び出されます。
  • **呼び出し先**: 算術演算のみを行うため、他の関数呼び出しは含みません。
  • 注: src/backend/utils/adt/timestamp.c にも同名の関数が存在しますが、本ページは ecpg ライブラリ内の実装を対象としています。

関連データ構造

  • fsec_t: 小数秒を保持する型。
  • int64: 戻り値として使用される時刻の内部表現型。

失敗モードと境界条件

  • 入力値が極端に大きい場合、int64 の範囲を超えるオーバーフローが発生する可能性があります。本関数自体にはエラーチェックは含まれていません。

保守担当者向けメモ

  • 本関数は static であり、モジュール内でのみ使用されます。
  • 単位変換の定数は dt.h に依存しているため、精度変更の際は同ヘッダーを確認してください。
