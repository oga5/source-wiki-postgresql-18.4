IsToastNamespace

定義

  • 種別: function
  • 定義箇所: src/backend/catalog/catalog.c:260

目的

指定された名前空間OIDが、TOASTテーブルを格納するためのシステム名前空間（PG_TOAST_NAMESPACE）または一時的なTOAST名前空間であるかを判定します。

シグネチャまたは形状

bool IsToastNamespace(Oid namespaceId)

入力と出力

  • **入力**: namespaceId (Oid) - 判定対象の名前空間OID。
  • **出力**: bool - 対象がTOAST名前空間であれば true、そうでなければ false。

前提条件と不変条件

  • 本関数はカタログアクセスを行いません。ロック取得中や再帰的な呼び出しにおいても安全に使用可能です。

アルゴリズムまたは振る舞い

1. 引数 namespaceId が定数 PG_TOAST_NAMESPACE と一致するか確認します。 2. 一致しない場合、isTempToastNamespace(namespaceId) を呼び出し、一時的なTOAST名前空間であるかを判定します。 3. いずれかの条件を満たす場合に true を返します。

関連データ構造

  • Oid: PostgreSQLのオブジェクト識別子。

失敗モードと境界条件

  • 無効なOIDが渡された場合、PG_TOAST_NAMESPACE との比較および isTempToastNamespace の判定結果に従い、適切に false を返します。

保守担当者向けメモ
