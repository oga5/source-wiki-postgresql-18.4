DatumGetDateADT

定義

  • 種別: static inline function
  • 定義箇所: src/include/utils/date.h:53

目的

PostgreSQLの内部データ表現である Datum 型から、日付型 DateADT への型変換を行う。

シグネチャまたは形状

static inline DateADT DatumGetDateADT(Datum X)

入力と出力

  • **入力**: Datum X（変換対象のデータ）
  • **出力**: DateADT（int32 として表現される日付値）

前提条件と不変条件

  • 入力 Datum X は、DateADT 型（int32）として解釈可能な形式で格納されている必要がある。
  • DateADT は内部的に int32 として保持される。

アルゴリズムまたは振る舞い

1. DatumGetInt32(X) を呼び出し、Datum 型から int32 値を抽出する。 2. 抽出した値を DateADT 型にキャストして返却する。 3. インライン関数として定義されており、関数呼び出しのオーバーヘッドを最小限に抑える設計となっている。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/btree_gist/btree_utils_num.c
  • src/backend/utils/adt/date.c
  • src/backend/utils/adt/json.c
  • src/backend/utils/adt/jsonpath_exec.c
  • src/backend/utils/adt/rangetypes.c
  • **呼び出し先**:
  • DatumGetInt32（マクロ）

関連データ構造

  • DateADT: 日付の内部表現（int32）。
  • Datum: PostgreSQLの汎用データ型。

失敗モードと境界条件

  • DateADT は整数であるため、無限大（Infinity）は DATEVAL_NOBEGIN や DATEVAL_NOEND といった特定の定数値で表現される。本関数は値をそのまま変換するため、無限大の判定は呼び出し側で DATE_NOT_FINITE マクロを用いて行う必要がある。

保守担当者向けメモ
