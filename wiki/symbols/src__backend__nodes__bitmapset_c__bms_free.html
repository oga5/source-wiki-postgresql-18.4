bms_free

定義

  • 種別: function
  • 定義箇所: src/backend/nodes/bitmapset.c:238

目的

Bitmapset 構造体が占有しているメモリ領域を解放します。

シグネチャまたは形状

void bms_free(Bitmapset *a)

入力と出力

  • **入力**: a (解放対象の Bitmapset へのポインタ、または NULL)
  • **出力**: なし

前提条件と不変条件

  • Bitmapset は空集合の場合 NULL として扱われるため、本関数は NULL ポインタを渡されても安全に動作します。

アルゴリズムまたは振る舞い

1. 引数 a が NULL でないことを確認します。 2. NULL でない場合、pfree(a) を呼び出してメモリを解放します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/postgres_fdw/deparse.c
  • src/backend/access/heap/heapam.c

    • (上記ファイル内の各箇所から、不要になった Bitmapset を破棄するために呼び出されています)

  • **呼び出し先**:
  • pfree (PostgreSQLのメモリ管理関数)

関連データ構造

  • Bitmapset: 非負整数の集合を表現する構造体。

失敗モードと境界条件

  • a が NULL の場合、何もせず正常に終了します。

保守担当者向けメモ

  • 本関数は pfree をラップしたものです。Bitmapset を保持する変数は、本関数呼び出し後に再利用しないよう注意してください。
