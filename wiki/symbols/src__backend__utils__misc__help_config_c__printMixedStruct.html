printMixedStruct

定義

  • 種別: function
  • 定義箇所: src/backend/utils/misc/help_config.c:86

目的

GUC（Grand Unified Configuration）パラメータの情報を構造体から抽出し、タブ区切り形式で標準出力に整形して表示する。

シグネチャまたは形状

static void printMixedStruct(mixedStruct *structToPrint)

入力と出力

  • **入力**: mixedStruct *structToPrint - 表示対象のGUCパラメータを保持する共用体へのポインタ。
  • **出力**: なし（標準出力へ printf を通じて書き出しを行う）。

前提条件と不変条件

  • structToPrint は有効な mixedStruct を指している必要がある。
  • structToPrint->generic メンバが初期化されており、vartype に基づいて適切な共用体メンバ（_bool, integer, real, string, _enum）が有効であること。

アルゴリズムまたは振る舞い

1. パラメータ名、コンテキスト、グループ名を共通ヘッダーとして出力する。 2. generic.vartype に基づき switch 文で分岐する。

  • PGC_BOOL: 値を TRUE/FALSE で出力。
  • PGC_INT / PGC_REAL: 値、最小値、最大値を出力。
  • PGC_STRING: 設定値を出力。
  • PGC_ENUM: config_enum_lookup_by_value を用いて列挙値の文字列を取得し出力。

    • 3. 未知の型が指定された場合は write_stderr でエラーメッセージを出力する。 4. 最後に、設定パラメータの短い説明（short_desc）と長い説明（long_desc）をタブ区切りで出力する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/misc/help_config.c 内のループ処理（GucInfoMain からのフロー内）。
  • **呼び出し先**:
  • printf (標準ライブラリ)
  • _() (国際化マクロ)
  • config_enum_lookup_by_value (GUC列挙値解決用)
  • write_stderr (エラー出力用)

関連データ構造

  • mixedStruct: 各種GUC構造体を共通化するための共用体。
  • config_generic: 全GUC構造体の先頭に含まれる共通メンバ。

失敗モードと境界条件

  • vartype が定義された型（BOOL, INT, REAL, STRING, ENUM）のいずれにも該当しない場合、標準エラー出力に内部エラーを出力する。
  • short_desc や long_desc が NULL の場合は空文字列として出力する。

保守担当者向けメモ

  • 本関数は mixedStruct の各メンバへのアクセスに依存しているため、GUC構造体の定義変更時には switch 文の整合性を確認すること。
  • _() マクロによる翻訳対象となっているため、説明文の変更時は翻訳ファイルへの影響を考慮すること。
