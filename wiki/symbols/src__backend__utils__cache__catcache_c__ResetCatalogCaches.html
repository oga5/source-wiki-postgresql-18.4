ResetCatalogCaches

定義

  • 種別: function
  • 定義箇所: src/backend/utils/cache/catcache.c:808

目的

システムカタログキャッシュ全体をリセットするためのラッパー関数です。

シグネチャまたは形状

void ResetCatalogCaches(void)

入力と出力

  • 入力: なし
  • 出力: なし（副作用としてキャッシュの状態をリセット）

前提条件と不変条件

  • ResetCatalogCachesExt(false) を呼び出すことで、キャッシュのリセットを実行します。

アルゴリズムまたは振る舞い

1. ResetCatalogCachesExt を引数 false で呼び出します。 2. 実行中、デバッグログが有効な場合は CACHE_elog を通じて開始と終了のログが出力されます。

呼び出し元と呼び出し先

  • 呼び出し元: ソースコード内の直接的な呼び出し元情報は提供されていません。
  • 呼び出し先: ResetCatalogCachesExt

関連データ構造

  • CatCache: キャッシュ管理構造体（本関数によりリセット対象となるキャッシュの基盤）

失敗モードと境界条件

  • 特記事項なし。

保守担当者向けメモ

  • 本関数は ResetCatalogCachesExt の単純なラッパーです。キャッシュの完全なリセットが必要な場面で使用されます。
  • デバッグ時には CACHEDEBUG マクロを有効にすることで、本関数の呼び出し状況を追跡可能です。
