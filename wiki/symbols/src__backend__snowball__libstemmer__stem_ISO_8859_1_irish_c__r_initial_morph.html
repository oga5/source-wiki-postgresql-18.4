r_initial_morph

定義

  • **種別**: 関数
  • **定義箇所**: src/backend/snowball/libstemmer/stem_ISO_8859_1_irish.c:261

目的

アイルランド語の単語先頭に現れる形態論的変異（Initial mutation）を検出し、削除または置換を行うことで正規化の準備を行う。

シグネチャまたは形状

static int r_initial_morph(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（Snowball ステミング環境へのポインタ）
  • **出力**: 成功時に 1、マッチしなかった場合に 0、エラー発生時に負の整数を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体である必要がある。
  • find_among を使用して、定義済みの変異パターン（a_0、24個の候補）を検索する。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を開始点（z->bra）に設定する。 2. find_among を呼び出し、単語先頭の変異パターンを検索する。 3. マッチした場合、終了点（z->ket）を現在のカーソル位置に設定する。 4. マッチした変異の種類（among_var）に応じて以下の処理を行う：

  • case 1: slice_del で該当箇所を削除する。
  • case 2〜10: slice_from_s を使用し、対応する文字列（s_0〜s_8）に置換する。

    • 5. 各操作において、戻り値が負の場合はエラーとして即座に終了する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/snowball/libstemmer/stem_ISO_8859_1_irish.c 内のメインステミング関数。
  • src/backend/snowball/libstemmer/stem_UTF_8_irish.c 内のメインステミング関数。
  • **呼び出し先**:
  • find_among（Snowball ランタイム）
  • slice_del（Snowball ランタイム）
  • slice_from_s（Snowball ランタイム）

関連データ構造

  • struct SN_env: ステミング対象の文字列やカーソル位置を保持する環境構造体。
  • a_0: 検索対象となる変異パターンの配列（among 構造体）。

失敗モードと境界条件

  • find_among がマッチしない場合、何も変更せず 0 を返す。
  • slice_del または slice_from_s が内部エラー（メモリ不足や不正なインデックス等）を返した場合、その負の値をそのまま呼び出し元へ伝播させる。
