PQgetssl

定義

  • 種別: function
  • 定義箇所: src/interfaces/libpq/fe-secure-openssl.c:1625

目的

PGconn オブジェクトから、現在使用されている OpenSSL の SSL 構造体へのポインタを取得します。

シグネチャまたは形状

void *PQgetssl(PGconn *conn);

入力と出力

  • **入力**: PGconn *conn （libpq の接続オブジェクトへのポインタ）
  • **出力**: 成功した場合は SSL *（void * として返却）、接続が確立されていない場合や conn が NULL の場合は NULL を返します。

前提条件と不変条件

  • conn が NULL の場合、関数は安全に NULL を返します。
  • 返されるポインタは、PGconn 構造体内部の ssl メンバを直接参照します。

アルゴリズムまたは振る舞い

1. 引数 conn が NULL かどうかを判定し、NULL であれば即座に NULL を返します。 2. conn->ssl に格納されているポインタを返却します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/interfaces/libpq/fe-secure.c:428
  • src/interfaces/libpq/fe-secure-openssl.c:1626 （自己参照の可能性あり）
  • **呼び出し先**:
  • 本関数は内部で他の関数を呼び出しません。

関連データ構造

  • PGconn: libpq-int.h で定義される接続状態を保持する構造体。本関数はこの構造体の ssl メンバに依存します。

失敗モードと境界条件

  • conn が NULL の場合、エラーではなく NULL を返すことで境界条件を処理します。
  • SSL 接続が確立されていない場合、conn->ssl は NULL であるため、本関数も NULL を返します。

保守担当者向けメモ

  • 本関数は libpq-fe.h で公開されているパブリック API です。
  • 返り値は void * 型ですが、実際には OpenSSL の SSL オブジェクトを指しています。型安全性を確保するため、利用側で適切なキャストが必要です。
