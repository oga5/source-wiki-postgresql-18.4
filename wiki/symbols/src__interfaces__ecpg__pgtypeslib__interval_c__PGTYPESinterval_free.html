PGTYPESinterval_free

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/pgtypeslib/interval.c:996

目的

interval 構造体によって占有されているメモリ領域を解放します。

シグネチャまたは形状

void PGTYPESinterval_free(interval * intvl)

入力と出力

  • **入力**: intvl - 解放対象となる interval 構造体へのポインタ。
  • **出力**: なし（void）。

前提条件と不変条件

  • intvl は、PGTYPESinterval_new 等によって動的に確保された有効なメモリ領域を指している必要があります。

アルゴリズムまたは振る舞い

標準ライブラリの free() 関数を呼び出し、引数として渡された interval 構造体のメモリを解放します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/interfaces/ecpg/test/expected/pgtypeslib-dt_test.c
  • src/interfaces/ecpg/test/expected/pgtypeslib-dt_test2.c
  • **呼び出し先**:
  • 標準ライブラリの free()

関連データ構造

  • interval (src/interfaces/ecpg/include/pgtypes_interval.h で定義)

失敗モードと境界条件

  • intvl が NULL の場合、標準の free() の仕様に従い、何も行われません。
  • 解放済みのポインタを渡した場合の挙動は、標準ライブラリの free() に依存します。

保守担当者向けメモ

  • 本関数は PGTYPESinterval_new と対になるメモリ管理関数です。interval 型の動的確保を行うコードを追加した際は、必ず本関数による解放を検討してください。
