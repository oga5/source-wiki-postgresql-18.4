ResetAwaitedLock

定義

  • 種別: function
  • 定義箇所: src/backend/storage/lmgr/lock.c:1905

目的

現在バックエンドが待機しているロックの追跡状態を解除（リセット）します。

シグネチャまたは形状

void ResetAwaitedLock(void)

入力と出力

  • 入力: なし
  • 出力: なし（副作用としてグローバル変数 awaitedLock を更新）

前提条件と不変条件

  • 本関数は、呼び出し元がロックの待機状態を終了した（または待機する必要がなくなった）タイミングで呼び出されることを前提としています。

アルゴリズムまたは振る舞い

1. グローバル変数 awaitedLock に NULL を代入します。これにより、システムが現在どのロックを待機しているかという追跡情報をクリアします。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/backend/storage/lmgr/proc.c:866
  • 呼び出し先:
  • なし（直接的な関数呼び出しは含まれていません）

関連データ構造

  • awaitedLock: ロック待機状態を保持するグローバル変数（型定義はソース抜粋外）。

失敗モードと境界条件

  • 特筆すべき失敗モードはありません。

保守担当者向けメモ

  • 本関数はロック待機情報のクリーンアップに使用されます。デッドロック検出や待機状態の可視化に関連する処理の一部です。
