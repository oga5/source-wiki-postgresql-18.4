pgstat_get_slru_name

定義

  • 種別: function
  • 定義箇所: src/backend/utils/activity/pgstat_slru.c:117

目的

SLRU（Simple Least Recently Used）キャッシュのインデックス値に対応する名称（文字列）を取得します。

シグネチャまたは形状

const char *pgstat_get_slru_name(int slru_idx)

入力と出力

  • **入力**: slru_idx (int) - 取得対象のSLRUインデックス。
  • **出力**: 指定されたインデックスに対応するSLRU名の文字列ポインタ。インデックスが範囲外の場合は NULL を返します。

前提条件と不変条件

  • slru_idx は 0 以上 SLRU_NUM_ELEMENTS 未満である必要があります。

アルゴリズムまたは振る舞い

1. 引数 slru_idx が 0 未満、または SLRU_NUM_ELEMENTS 以上であるかを確認します。 2. 範囲外であれば NULL を返します。 3. 範囲内であれば、内部配列 slru_names の該当インデックスにある文字列を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/pgstatfuncs.c:1735 (SQL関数等からの統計取得処理)
  • **呼び出し先**:
  • 直接的な関数呼び出しは行いません（配列アクセスのみ）。

関連データ構造

  • slru_names: SLRU名を保持する静的配列（定義箇所は pgstat_slru.c 内）。

失敗モードと境界条件

  • slru_idx が負の値、または SLRU_NUM_ELEMENTS 以上の場合、関数は安全に NULL を返します。

保守担当者向けメモ

  • 本関数は slru_names 配列のインデックスと名前の対応を抽象化する役割を持ちます。新しいSLRUキャッシュタイプを追加する際は、slru_names 配列の更新が必要です。
