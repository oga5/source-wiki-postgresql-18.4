mic2latin

定義

  • 種別: function
  • 定義箇所: src/backend/utils/mb/conv.c:126

目的

Mule Internal Code (MIC) 形式の文字列を、指定された特定の Latin 系エンコーディングへ変換します。

シグネチャまたは形状

int mic2latin(const unsigned char *mic, unsigned char *p, int len,
              int lc, int encoding, bool noError)

入力と出力

  • **入力**:
  • mic: 変換元の MIC 文字列。
  • p: 変換後の文字列を格納するバッファ。
  • len: 入力文字列のバイト長。
  • lc: 変換対象の文字セットを識別するコード（Mule Internal Code のエスケープシーケンス等に対応）。
  • encoding: 変換先のエンコーディング識別子。
  • noError: 変換失敗時にエラーを報告せず中断するかどうかを指定するフラグ。
  • **出力**:
  • 戻り値: 処理した入力バイト数。
  • p: 変換結果が書き込まれ、末尾にヌル終端文字が付与されます。

前提条件と不変条件

  • len が 0 より大きい間、ループ処理が継続されます。
  • ASCII 文字（高位ビットがセットされていない文字）はそのままコピーされます。
  • マルチバイト文字の場合、lc と一致し、かつ 2 バイト構成である必要があります。

アルゴリズムまたは振る舞い

1. 入力文字列をバイト単位で走査します。 2. ASCII 文字（IS_HIGHBIT_SET が偽）であれば、そのまま出力バッファへコピーします。 3. マルチバイト文字の場合、pg_mule_mblen で長さを確認します。 4. 長さが 2 バイトであり、かつ指定された lc と一致する場合のみ、2 バイト目を出力バッファへコピーします。 5. 条件を満たさない場合、noError が偽であれば report_invalid_encoding または report_untranslatable_char を呼び出し、真であれば処理を中断します。 6. 最後に p の末尾に \0 を付与し、消費したバイト数を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/mb/conversion_procs/cyrillic_and_mic/cyrillic_and_mic.c
  • src/backend/utils/mb/conversion_procs/latin2_and_win1250/latin2_and_win1250.c
  • src/backend/utils/mb/conversion_procs/latin_and_mic/latin_and_mic.c
  • **呼び出し先**:
  • pg_mule_mblen
  • report_invalid_encoding
  • report_untranslatable_char

関連データ構造

  • PG_MULE_INTERNAL: MIC エンコーディングの定数。
  • PG_KOI8R, PG_LATIN1 等のエンコーディング識別子。

失敗モードと境界条件

  • 入力文字列が不正なエンコーディングである場合、または変換対象外の文字が含まれる場合、noError が偽であればエラーが報告されます。
  • len が不足している場合、不完全なマルチバイト文字としてエラー報告または中断が行われます。

保守担当者向けメモ

  • 本関数は src/include/mb/pg_wchar.h で宣言されています。
  • 変換ロジックは Mule Internal Code の構造に強く依存しているため、エンコーディング定義の変更時には注意が必要です。
