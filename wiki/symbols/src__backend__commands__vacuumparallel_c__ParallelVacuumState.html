ParallelVacuumState

定義

  • 種別: struct
  • 定義箇所: src/backend/commands/vacuumparallel.c:162

目的

PostgreSQL の VACUUM 処理において、インデックスのバルク削除およびクリーンアップを並列実行するための状態を保持するメイン構造体です。

シグネチャまたは形状

struct ParallelVacuumState {
    ParallelContext *pcxt;
    Relation        heaprel;
    Relation       *indrels;
    int             nindexes;
    PVShared       *shared;
    PVIndStats     *indstats;
    TidStore       *dead_items;
    BufferUsage    *buffer_usage;
    WalUsage       *wal_usage;
    bool           *will_parallel_vacuum;
    int             nindexes_parallel_bulkdel;
    int             nindexes_parallel_cleanup;
    int             nindexes_parallel_condcleanup;
    BufferAccessStrategy bstrategy;
    char           *relnamespace;
    char           *relname;
    char           *indname;
    PVIndVacStatus  status;
};

入力と出力

  • **入力**: parallel_vacuum_init により初期化され、リーダープロセスおよびワーカープロセス間で共有されるコンテキスト情報。
  • **出力**: 並列処理終了後、各インデックスの統計情報（IndexBulkDeleteResult）が DSM から回収され、ローカルメモリへコピーされます。

前提条件と不変条件

  • pcxt はワーカープロセスでは NULL となります。
  • indstats 配列は、並列処理の可否（will_parallel_vacuum）に関わらず、すべてのインデックスに対して確保されます。
  • maintenance_work_mem はワーカー間で共有・分配されます。

アルゴリズムまたは振る舞い

  • リーダープロセスが ParallelContext を作成し、DSM 上に共有リソース（PVShared、TidStore 等）を確保します。
  • インデックスの並列安全性に基づき、タスクを動的に割り当てます。
  • 各ワーカーは shared->idx をアトミックにインクリメントしながら、未処理のインデックスを順次処理します。
  • エラー発生時は parallel_vacuum_error_callback が呼び出され、失敗したインデックス名が報告されます。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/access/heap/vacuumlazy.c（Lazy Vacuum のメインロジック）。
  • **呼び出し先**: parallel_vacuum_init（初期化）、parallel_vacuum_end（終了・解放）、parallel_vacuum_get_dead_items（タプル情報取得）等の vacuumparallel.c 内関数。

関連データ構造

  • PVShared: 全ワーカーで共有される制御情報。
  • PVIndStats: インデックスごとの統計情報。
  • TidStore: 削除対象タプルを格納する共有領域。
  • ParallelContext: 並列実行のための基盤構造体。

失敗モードと境界条件

  • min_parallel_index_scan_size 未満の小さなインデックスは並列処理の対象外となり、リーダーが単独で処理します。
  • 並列ワーカー内でエラーが発生した場合、PVIndVacStatus を通じて状態が管理されます。

保守担当者向けメモ

  • 本構造体は src/backend/commands/vacuum.h にも定義が存在します。
  • 並列処理の可否は amparallelvacuumoptions に依存するため、インデックスの特性変更時には注意が必要です。
