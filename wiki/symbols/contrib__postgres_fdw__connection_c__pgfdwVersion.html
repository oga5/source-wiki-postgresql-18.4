pgfdwVersion

定義

  • 種別: enum
  • 定義箇所: contrib/postgres_fdw/connection.c:121

目的

postgres_fdw がリモートサーバーとの通信において使用する API バージョンを識別するために定義されています。

シグネチャまたは形状

enum pgfdwVersion
{
	PGFDW_V1_1 = 0,
	PGFDW_V1_2,
};

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: PGFDW_V1_1 (0) または PGFDW_V1_2 (1) のいずれかの値を保持します。

前提条件と不変条件

  • 列挙値は昇順で定義されており、新しいバージョンが追加される際は後続の整数値が割り当てられることが想定されます。

アルゴリズムまたは振る舞い

  • リモートサーバーとの接続確立時や、特定のプロトコル機能の有効性を判定する際のバージョンフラグとして機能します。

呼び出し元と呼び出し先

  • 呼び出し元（参照箇所）:
  • contrib/postgres_fdw/connection.c:184
  • contrib/postgres_fdw/connection.c:2169
  • 呼び出し先: なし（本シンボルはデータ定義です）

関連データ構造

  • ConnCacheEntry: 接続キャッシュ内で、その接続がどの API バージョンを使用しているかを管理するフィールドの型として利用される可能性があります。

失敗モードと境界条件

  • 定義されていない整数値をキャストして代入した場合の動作は未定義です。

保守担当者向けメモ

  • 新しいプロトコル機能や互換性変更を導入する際は、この列挙型に新しいバージョン値を追加し、各所での条件分岐を更新してください。
