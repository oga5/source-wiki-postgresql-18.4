left_offset

定義

  • 種別: static inline function
  • 定義箇所: src/common/binaryheap.c:89

目的

バイナリヒープの配列表現において、指定されたインデックス i にあるノードの左子ノードのインデックスを算出します。

シグネチャまたは形状

static inline int left_offset(int i)

入力と出力

  • **入力**: i (int) - ヒープ内のノードインデックス。
  • **出力**: int - 左子ノードのインデックス。

前提条件と不変条件

  • ヒープは 0 から始まる配列として格納されていることを前提とします。
  • 入力 i が有効なノードインデックスである場合、戻り値は 2 * i + 1 となります。

アルゴリズムまたは振る舞い

  • 0 始まりのインデックス体系に基づき、2 * i + 1 を計算して返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/common/binaryheap.c:324
  • src/backend/utils/sort/logtape.c:401
  • **呼び出し先**: なし（算術演算のみ）。

関連データ構造

  • binaryheap 構造体（src/include/lib/binaryheap.h で定義）。本関数は、この構造体が保持する配列内の位置計算に使用されます。

失敗モードと境界条件

  • 入力 i が負の値や、ヒープの最大サイズを超える値である場合のチェックは行われません。呼び出し側でインデックスの妥当性を保証する必要があります。

保守担当者向けメモ

  • 本関数は static inline で定義されており、コンパイル時にインライン展開されることを意図しています。
  • 関連する兄弟関数として right_offset（2 * i + 2）および parent_offset（(i - 1) / 2）が存在します。これらと一貫性を保つ必要があります。
