SetVariableBool

定義

  • src/bin/psql/variables.c:461

目的

指定された変数名に対して、値 "on" を設定する。

シグネチャまたは形状

bool SetVariableBool(VariableSpace space, const char *name)

入力と出力

  • **入力**:
  • space: 変数を格納する VariableSpace 構造体。
  • name: 設定対象の変数名（文字列）。
  • **出力**:
  • bool: 変数の設定が成功した場合は true、失敗した場合は false を返す。

前提条件と不変条件

  • space は初期化済みであること。
  • 変数の値は内部的に文字列として保持されるため、本関数は "on" という文字列を代入する。

アルゴリズムまたは振る舞い

内部的に SetVariable(space, name, "on") を呼び出す。これにより、SetVariable が提供する標準的な変数設定フロー（置換フックによる正規化および代入フックによる妥当性検証）が実行される。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/bin/psql/startup.c: AUTOCOMMIT, SHOW_ALL_RESULTS, ECHO_HIDDEN, QUIET, SINGLESTEP, SINGLELINE の各変数の初期化時に呼び出される。
  • **呼び出し先**:
  • SetVariable (同ファイル内)

関連データ構造

  • VariableSpace: 変数リストを管理するヘッダー構造体。

失敗モードと境界条件

  • SetVariable の内部ロジックに依存する。代入フックが登録されている場合、そのフックが false を返すと本関数も false を返す。

保守担当者向けメモ

  • 本関数はブール値設定のショートカットである。変数の値を on 以外（例: off）に設定する必要がある場合は、直接 SetVariable を使用すること。
