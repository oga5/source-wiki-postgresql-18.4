r_residual_suffix

定義

  • **種別**: function
  • **定義箇所**: src/backend/snowball/libstemmer/stem_ISO_8859_1_french.c:1067

目的

フランス語のステミング処理において、単語末尾に残存する特定の接尾辞を削除または置換し、語幹を正規化する。

シグネチャまたは形状

static int r_residual_suffix(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z （Snowballの実行環境状態）
  • **出力**: int （成功時は 1、失敗時は 0、エラー発生時は負の値を返す）

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • 内部処理において z->I[2]（領域境界情報）が初期化されていることを前提とする。

アルゴリズムまたは振る舞い

1. **第一段階（'s' の処理）**:

  • 単語末尾が 's' であり、かつ直前が特定の文字（g_keep_with_s グループ外）であるか、特定の文字列（s_31）に一致する場合、その 's' を削除する。

    • 2. **第二段階（テーブルベースの接尾辞処理）**:

  • z->I[2]（R2領域）以降の範囲で、a_7 テーブルを用いて接尾辞を検索する。
  • マッチした接尾辞の種類（among_var）に応じて以下の処理を行う:
  • **ケース 1**: R2領域内であることを確認し、末尾が 's' または 't' であれば削除する。
  • **ケース 2**: 特定の文字列（s_32）に置換する。
  • **ケース 3**: 単純に削除する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_ISO_8859_1_french.c および stem_UTF_8_french.c 等のステミングメイン関数から呼び出される。
  • **呼び出し先**: slice_del, slice_from_s, find_among_b, r_R2, eq_s_b, out_grouping_b 等のSnowballランタイム関数。

関連データ構造

  • struct SN_env: ステミングの実行状態を保持する構造体。
  • a_7: 接尾辞検索用の among 構造体配列。
  • g_keep_with_s: 's' 削除の条件判定に使用される文字グループ。

失敗モードと境界条件

  • z->c が z->lb 以下の場合、または領域境界条件を満たさない場合は処理を中断し 0 を返す。
  • slice_del 等のランタイム関数が負の値を返した場合、その値をそのまま伝播させる。

保守担当者向けメモ
