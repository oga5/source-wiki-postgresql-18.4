SimpleActionListCell

定義

  • 種別: struct
  • 定義箇所: src/bin/psql/startup.c:53

目的

psql 起動時に実行すべきアクション（コマンドやスクリプト実行など）を保持する連結リストの各ノードを定義します。

シグネチャまたは形状

typedef struct SimpleActionListCell
{
	struct SimpleActionListCell *next;
	enum _actions action;
	char	   *val;
} SimpleActionListCell;

入力と出力

  • next: リスト内の次のセルへのポインタ。
  • action: 実行するアクションの種類（enum _actions）。
  • val: アクションに関連付けられた文字列値（ファイルパスやクエリ文字列など）。

前提条件と不変条件

  • リストの末尾では next は NULL である必要があります。
  • val は動的に割り当てられた文字列を指すことが想定されます。

アルゴリズムまたは振る舞い

  • psql の起動引数解析において、実行すべき一連のアクションをキューイングするために使用されます。
  • simple_action_list_append 関数を介してリストに追加され、後に順次処理されます。

呼び出し元と呼び出し先

  • 呼び出し元: src/bin/psql/startup.c 内の引数解析およびリスト構築処理（pg_malloc_object(SimpleActionListCell) を使用してメモリ確保が行われる箇所）。
  • 呼び出し先: 本構造体自体はデータ保持用であり、直接的な関数呼び出しは行いません。

関連データ構造

  • SimpleActionList: 本セルを要素として管理するリスト構造（head および tail ポインタを保持）。
  • enum _actions: action フィールドで使用される列挙型。

失敗モードと境界条件

  • メモリ不足時: pg_malloc_object が失敗した場合、プログラムは終了します。
  • リストが空の場合: head ポインタが NULL となり、アクションは実行されません。

保守担当者向けメモ

  • val フィールドのメモリ管理（解放タイミング）は、リストの破棄処理と整合させる必要があります。
  • next ポインタの操作時は、リストの循環やメモリリークが発生しないよう注意してください。
