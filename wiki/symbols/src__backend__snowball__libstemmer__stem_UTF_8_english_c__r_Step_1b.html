r_Step_1b

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_english.c:548

目的

Porterステミングアルゴリズムの「Step 1b」を実装し、英語の単語末尾の接尾辞（"-eed", "-ed", "-ing" 等）を条件に応じて削除または置換する。

シグネチャまたは形状

static int r_Step_1b(struct SN_env * z)

入力と出力

  • 入力: SN_env 構造体（ステミング対象の文字列と現在のカーソル位置 c を保持）。
  • 出力: 処理が成功した場合は 1、失敗または適用対象外の場合は 0、エラー発生時は負の値を返す。

前提条件と不変条件

  • z->c が単語の末尾を指していること。
  • find_among_b を用いた後方検索により、特定の接尾辞パターンが検出される必要がある。
  • 一部の置換処理では r_R1（R1領域内であること）や r_shortv（短母音条件）の判定が必須となる。

アルゴリズムまたは振る舞い

1. **接尾辞検索**: a_4 テーブルを用いて単語末尾を検索。 2. **分岐処理**:

  • case 1 ("eed"): r_R1 を満たす場合に "-ee" に置換。
  • case 2 ("ed", "ing"):
  • 母音を含む場合に接尾辞を削除。
  • 削除後、単語末尾が "at", "bl", "iz" であれば "e" を付与。
  • 単語末尾が二重子音（"bb", "dd", "ff" 等）であれば最後の文字を削除。
  • 単語末尾が短母音＋子音（r_shortv）であれば "e" を付与。

呼び出し元と呼び出し先

  • 呼び出し元: stem_UTF_8_english.c および stem_ISO_8859_1_english.c 等のステミングメイン関数から呼び出される。
  • 呼び出し先: find_among_b, r_R1, r_shortv, slice_del, slice_from_s, out_grouping_b_U, in_grouping_b_U, skip_b_utf8。

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_4, a_3: 接尾辞検索用のテーブル。
  • g_v, g_aeo: 母音グループ定義。

失敗モードと境界条件

  • z->c が文字列の先頭（lb）に達した場合、処理を中断して 0 を返す。
  • slice_del や slice_from_s がメモリ不足や不正な操作で負の値を返した場合、その値をそのまま伝播させる。
