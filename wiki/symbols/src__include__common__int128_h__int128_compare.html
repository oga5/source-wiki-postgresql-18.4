int128_compare

定義

  • 種別: function
  • 定義箇所: src/include/common/int128.h:70

目的

2つの128ビット整数値を比較し、その大小関係に基づいた標準的な比較結果（-1, 0, 1）を返します。

シグネチャまたは形状

static inline int int128_compare(INT128 x, INT128 y)

入力と出力

  • **入力**: 比較対象となる2つの INT128 型の値 x および y。
  • **出力**:
  • x < y の場合: -1
  • x > y の場合: 1
  • x == y の場合: 0

前提条件と不変条件

  • INT128 型は、プラットフォームに応じてネイティブな128ビット整数型、または hi/lo 構造体として定義されます。本関数は、この型定義の抽象化レイヤー上で動作します。

アルゴリズムまたは振る舞い

1. x が y より小さい場合、-1 を返します。 2. x が y より大きい場合、1 を返します。 3. 上記のいずれでもない（等しい）場合、0 を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/timestamp.c (複数箇所)
  • src/tools/testint128.c (複数箇所)
  • **呼び出し先**:
  • 本関数は比較演算子（<, >）を使用します。これらはコンパイラが提供するネイティブな128ビット比較、または INT128 型に対する演算子のオーバーロード（または構造体比較の定義）に依存します。

関連データ構造

  • INT128: 比較対象となる128ビット整数型。

失敗モードと境界条件

  • 境界条件: 128ビット整数の最小値および最大値の比較においても、型定義が正しく行われていれば正常に動作します。
  • 失敗モード: 特筆すべき失敗モードはありません。

保守担当者向けメモ

  • 本関数は static inline で定義されており、ヘッダーファイル内で完結しています。
  • ソフトウェア実装（USE_NATIVE_INT128 が無効な場合）において、比較ロジックが INT128 構造体のメモリレイアウトと整合していることを確認してください。
