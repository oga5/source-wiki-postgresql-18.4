uuid_generate_internal

定義

  • 種別: function
  • 定義箇所: contrib/uuid-ossp/uuid-ossp.c:218

目的

OSSP UUID ライブラリを使用して UUID を生成し、PostgreSQL の内部形式（Datum）に変換して返す共通バックエンド関数です。

シグネチャまたは形状

static Datum uuid_generate_internal(int mode, const uuid_t *ns, const char *name, int len)

入力と出力

  • **入力**:
  • mode: 生成する UUID のバージョンやオプションを指定するフラグ（例: UUID_MAKE_V1, UUID_MAKE_V4）。
  • ns: 名前空間 UUID（v3/v5 生成時に使用）。
  • name: 名前空間内で一意性を確保するための文字列。
  • len: name の長さ。
  • **出力**:
  • PostgreSQL の uuid 型に対応する Datum 値。

前提条件と不変条件

  • get_cached_uuid_t(0) により取得されたキャッシュ済み uuid_t オブジェクトが有効であること。
  • OSSP ライブラリの uuid_make が正常に動作する環境であること。

アルゴリズムまたは振る舞い

1. get_cached_uuid_t(0) を呼び出し、セッション内で再利用される uuid_t オブジェクトを取得します。 2. uuid_make を呼び出し、指定された mode に基づいて UUID を生成します。 3. 生成結果がエラー（UUID_RC_OK 以外）の場合、pguuid_complain を呼び出して例外を発生させます。 4. uuid_to_string を使用して生成された UUID を文字列形式に変換します。 5. DirectFunctionCall1 を介して PostgreSQL の uuid_in 関数を呼び出し、文字列を Datum 型に変換して返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/uuid-ossp/uuid-ossp.c 内の各 SQL 関数インターフェース（uuid_generate_v1, uuid_generate_v4 等）。
  • **呼び出し先**:
  • get_cached_uuid_t
  • uuid_make
  • pguuid_complain
  • uuid_to_string
  • uuid_in (PostgreSQL 内部関数)

関連データ構造

  • uuid_t: OSSP ライブラリが提供する UUID オブジェクト。
  • uuid_rc_t: OSSP ライブラリの戻り値ステータスコード。
  • Datum: PostgreSQL の汎用データ型。

失敗モードと境界条件

  • uuid_make が UUID_RC_OK 以外の値を返した場合、pguuid_complain が呼び出され、処理が中断されます。

保守担当者向けメモ

  • 本関数は OSSP UUID ライブラリに強く依存しています。OSSP 以外のライブラリを使用するビルド構成では、別の実装パスが選択される可能性があります。
  • get_cached_uuid_t によるキャッシュの再利用は、MAC アドレス取得のオーバーヘッドを削減するための重要な最適化です。
