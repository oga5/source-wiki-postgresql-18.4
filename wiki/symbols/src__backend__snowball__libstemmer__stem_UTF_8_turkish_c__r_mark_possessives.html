r_mark_possessives

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_turkish.c:656

目的

トルコ語のステミング処理において、単語末尾の所有格接尾辞を識別し、必要に応じて母音を伴う接尾辞処理を適用する。

シグネチャまたは形状

static int r_mark_possessives(struct SN_env * z)

入力と出力

  • **入力**: z (Snowball ステミング環境構造体)
  • **出力**: 成功時に 1、失敗または条件不一致時に 0、エラー発生時に負の値を返す。

前提条件と不変条件

  • z->c（現在のカーソル位置）が z->lb（境界）より大きいこと。
  • 単語末尾の文字が特定のビットマスク条件（z->p[z->c - 1] >> 5 != 3 および 67133440 ビットマップによる判定）を満たすこと。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置の文字が、所有格接尾辞の候補として妥当かビットマスクで検証する。 2. find_among_b を使用し、a_0 配列に定義された接尾辞パターンを末尾から検索する。 3. マッチした場合、r_mark_suffix_with_optional_U_vowel を呼び出し、母音の有無を考慮した接尾辞のマーク処理を行う。 4. 呼び出し先の結果をそのまま戻り値として返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_UTF_8_turkish.c 内の複数箇所（行 32, 1204, 1261, 1411, 1598, 1696, 1793, 1836）から参照されている。
  • **呼び出し先**: find_among_b（ユーティリティ関数）、r_mark_suffix_with_optional_U_vowel。

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_0: 接尾辞マッチング用の among 構造体配列。

失敗モードと境界条件

  • カーソルが境界（z->lb）に達している場合、または文字判定条件を満たさない場合は 0 を返す。
  • find_among_b がパターンを見つけられない場合は 0 を返す。
  • r_mark_suffix_with_optional_U_vowel がエラー（負の値）を返した場合、その値をそのまま伝播させる。
