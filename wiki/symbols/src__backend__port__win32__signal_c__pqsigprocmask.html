pqsigprocmask

定義

  • 種別: function
  • 定義箇所: src/backend/port/win32/signal.c:170

目的

Windows環境において、POSIXの sigprocmask と同等のシグナルマスク制御機能を提供します。

シグネチャまたは形状

int pqsigprocmask(int how, const sigset_t *set, sigset_t *oset);

入力と出力

  • how: マスク変更方法（SIG_BLOCK, SIG_UNBLOCK, SIG_SETMASK）。
  • set: 変更対象のシグナルセット。NULL の場合は無視されます。
  • oset: 変更前のシグナルマスクを格納するポインタ。NULL でない場合に書き込まれます。
  • 戻り値: 成功時は 0、無効な how が指定された場合は -1 を返し、errno に EINVAL を設定します。

前提条件と不変条件

  • pg_signal_mask が現在のシグナルマスク状態を保持しています。
  • oset が指定されている場合、変更前の pg_signal_mask が必ずコピーされます。

アルゴリズムまたは振る舞い

1. oset が指定されていれば、現在の pg_signal_mask を保存します。 2. set が NULL であれば、何も変更せず 0 を返します。 3. how に基づき pg_signal_mask を更新します。

  • SIG_BLOCK: 現在のマスクに set を追加。
  • SIG_UNBLOCK: 現在のマスクから set を除去。
  • SIG_SETMASK: set でマスクを上書き。

    • 4. pgwin32_dispatch_queued_signals() を呼び出し、ブロック解除されたシグナルがあれば即座に処理を実行します。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/port/win32/signal.c:117 および src/include/libpq/pqsignal.h で sigprocmask マクロとして定義。
  • 呼び出し先: pgwin32_dispatch_queued_signals()。

関連データ構造

  • pg_signal_mask: シグナルマスクを保持する静的変数。
  • sigset_t: シグナルセット型。

失敗モードと境界条件

  • how に SIG_BLOCK, SIG_UNBLOCK, SIG_SETMASK 以外の値が渡された場合、errno を EINVAL に設定し -1 を返します。

保守担当者向けメモ

  • 本関数はWindows環境におけるPOSIX互換レイヤーの一部です。
  • pgwin32_dispatch_queued_signals() の呼び出しは、シグナルブロック解除直後の即時処理を保証するために重要です。
