DatumGetInetP

定義

  • 種別: static inline 関数
  • 定義箇所: src/include/utils/inet.h:138

目的

PostgreSQLの Datum 型として保持されているネットワークアドレスデータ（inet 型）を、C言語の構造体ポインタ（inet *）へ変換します。

シグネチャまたは形状

static inline inet *DatumGetInetP(Datum X)

入力と出力

  • **入力**: Datum X（inet 型のデータを含む Datum）
  • **出力**: inet *（展開済みデータへのポインタ）

前提条件と不変条件

  • 入力 Datum X は、有効な inet 型のデータである必要があります。
  • PG_DETOAST_DATUM マクロを使用するため、入力が TOAST 圧縮されている場合は自動的に展開されます。

アルゴリズムまたは振る舞い

1. PG_DETOAST_DATUM(X) を呼び出し、必要に応じてデータを展開（detoast）します。 2. 展開されたデータのポインタを inet * 型にキャストして返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/include/utils/inet.h:143 にて定義されるマクロ PG_GETARG_INET_P(n) から呼び出されます。
  • **呼び出し先**:
  • PG_DETOAST_DATUM（fmgr.h 経由で提供されるマクロ）

関連データ構造

  • inet: ネットワークアドレスを保持する構造体。
  • Datum: PostgreSQLの汎用データ型。

失敗モードと境界条件

  • 入力 X が inet 型として不正な場合、PG_DETOAST_DATUM の内部処理によりエラーが発生する可能性があります。

保守担当者向けメモ

  • 本関数はインライン関数として定義されており、パフォーマンスを考慮して fmgr.h のマクロを利用しています。
  • inet 型の内部表現（varlena 形式）に変更があった場合は、本関数のキャスト先構造体定義との整合性に注意してください。
