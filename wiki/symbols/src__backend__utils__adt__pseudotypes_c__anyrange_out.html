anyrange_out

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/pseudotypes.c:209

目的

擬似型 anyrange の出力関数として機能し、範囲型の値を文字列形式に変換して表示可能にする。

シグネチャまたは形状

Datum anyrange_out(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS（関数マネージャ経由で渡される anyrange 型のデータ）。
  • **出力**: Datum（範囲型の文字列表現）。

前提条件と不変条件

  • 本関数は anyrange 型の出力処理を担うが、実質的な処理は range_out に委譲される。
  • 呼び出しには PostgreSQL の関数呼び出し規約（fmgr）に従う必要がある。

アルゴリズムまたは振る舞い

1. fcinfo（関数呼び出し情報）をそのまま range_out 関数へ渡す。 2. range_out の戻り値をそのまま呼び出し元へ返す。

呼び出し元と呼び出し先

  • **呼び出し元**: PostgreSQL の型システム（pg_type の出力関数エントリ経由）。
  • **呼び出し先**: range_out（実体型の出力関数）。

関連データ構造

  • Datum: PostgreSQL の汎用データ表現。
  • RangeType: 範囲型の内部構造（range_out が処理する対象）。

失敗モードと境界条件

  • range_out が失敗した場合、そのエラーがそのまま伝播する。
  • 渡されたデータが有効な範囲型でない場合、range_out 内部でエラーが発生する可能性がある。

保守担当者向けメモ

  • 本関数は単なるラッパーであり、ロジックの変更が必要な場合は range_out を確認すること。
  • 擬似型に対する出力処理の委譲という設計パターンに従っている。
