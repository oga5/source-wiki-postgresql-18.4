gbt_float8ge

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_float8.c:34

目的

2つの float8 値を比較し、第一引数が第二引数以上であるか（>=）を判定します。

シグネチャまたは形状

bool gbt_float8ge(const void *a, const void *b, FmgrInfo *flinfo)

入力と出力

  • **入力**:
  • a: 比較対象の float8 値へのポインタ。
  • b: 比較対象の float8 値へのポインタ。
  • flinfo: 関数管理情報（本関数内では未使用）。
  • **出力**:
  • bool: *a >= *b が真であれば true、そうでなければ false を返します。

前提条件と不変条件

  • a および b は、有効な float8 型の値を指している必要があります。

アルゴリズムまたは振る舞い

1. ポインタ a および b が指す値を float8 型としてキャストし、デリファレンスします。 2. C言語の標準演算子 >= を用いて比較を行い、その結果を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/btree_gist/btree_float8.c 内の tinfo 構造体定義（92行目）において、比較関数として参照されています。
  • **呼び出し先**: 直接的な関数呼び出しはありません。

関連データ構造

  • float8: PostgreSQLの倍精度浮動小数点数型。

失敗モードと境界条件

  • 入力ポインタが NULL の場合、デリファレンスによりセグメンテーション違反が発生する可能性があります。
  • NaN（非数）が含まれる場合の比較結果は、標準的な浮動小数点数の比較規則に従います。

保守担当者向けメモ

  • 本関数は btree_gist モジュールにおける float8 型の比較ロジックの一部です。
  • tinfo 構造体を通じて btree_utils_num.c のユーティリティ関数から間接的に利用されることを前提としています。
