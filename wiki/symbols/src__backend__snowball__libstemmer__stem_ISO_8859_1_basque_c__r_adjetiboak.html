r_adjetiboak

定義

  • **種別**: function
  • **定義箇所**: src/backend/snowball/libstemmer/stem_ISO_8859_1_basque.c:1119

目的

バスク語のステミング処理において、単語末尾の形容詞に関連する接尾辞を識別し、削除または置換を行う。

シグネチャまたは形状

static int r_adjetiboak(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z （Snowballステミング環境）
  • **出力**: 処理が成功した場合は 1、失敗または対象外の場合は 0、エラー発生時は負の値を返す。

前提条件と不変条件

  • z->c が単語内の有効な位置を指していること。
  • find_among_b を用いた後方検索が可能な状態であること。

アルゴリズムまたは振る舞い

1. 現在位置から単語末尾の文字をチェックし、特定のビットマスク条件を満たすか判定する。 2. a_2 テーブルを使用して、単語末尾から形容詞接尾辞を後方検索する。 3. マッチした場合、among_var の値に応じて以下のいずれかを実行する。

  • **case 1**: r_RV(z) を呼び出し、RV領域内であれば接尾辞を削除 (slice_del) する。
  • **case 2**: 接尾辞を定数 s_10 に置換 (slice_from_s) する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_ISO_8859_1_basque.c および src/backend/snowball/libstemmer/stem_UTF_8_basque.c 内のメインステミング関数から呼び出される。
  • **呼び出し先**: find_among_b, r_RV, slice_del, slice_from_s。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_2: 形容詞接尾辞の探索テーブル。

失敗モードと境界条件

  • 単語末尾の文字が条件を満たさない場合、または find_among_b でマッチしない場合は 0 を返す。
  • r_RV や slice 操作が失敗した場合、そのエラーコード（負の値）をそのまま伝播させる。

保守担当者向けメモ

  • 本関数はSnowballコンパイラによる自動生成コードであるため、直接編集せず、元の .sbl ファイルを修正すること。
  • a_2 テーブルの定義や s_10 の内容は、生成元の言語定義ファイルに依存する。
