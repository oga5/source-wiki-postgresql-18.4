HeadMatchesImpl

定義

  • 種別: function
  • 定義箇所: src/bin/psql/tab-complete.in.c:1698

目的

入力された単語リストの先頭部分が、指定された可変長引数のパターンと一致するかを判定します。

シグネチャまたは形状

static bool HeadMatchesImpl(bool case_sensitive, int previous_words_count, char **previous_words, int narg, ...);

入力と出力

  • **入力**:
  • case_sensitive: 大文字小文字を区別するかどうか。
  • previous_words_count: previous_words 配列内の単語数。
  • previous_words: 比較対象となる単語の配列。
  • narg: 比較するパターンの数。
  • ...: 比較対象となる文字列パターンの可変長引数。
  • **出力**:
  • bool: パターンが一致した場合は true、それ以外は false を返します。

前提条件と不変条件

  • narg は 64 以下である必要があります（Assert により検証）。
  • previous_words_count が narg 未満の場合、一致しないものとして即座に false を返します。

アルゴリズムまたは振る舞い

1. 可変長引数から最大 64 個のパターン文字列を argarray に格納します。 2. previous_words_count が narg より小さい場合は false を返します。 3. HeadMatchesArray を呼び出し、previous_words の先頭 narg 個の単語と argarray の各要素を case_sensitive フラグに従って比較し、その結果を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/bin/psql/tab-complete.in.c 内のコード（1882行目、1887行目など）。
  • **呼び出し先**: HeadMatchesArray。

関連データ構造

  • previous_words: 入力行を単語単位に分解した文字列配列。

失敗モードと境界条件

  • narg が 64 を超える場合、Assert によりプログラムが停止します。
  • previous_words_count が narg より小さい場合、比較を行わずに false を返します。

保守担当者向けメモ

  • 本関数は HeadMatchesArray へのラッパーとして機能します。
  • gen_tabcomplete.pl による自動生成コードの一部である可能性があるため、直接編集する際はビルドプロセスへの影響を考慮してください。
