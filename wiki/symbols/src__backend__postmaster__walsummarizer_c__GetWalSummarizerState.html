GetWalSummarizerState

定義

  • 種別: function
  • 定義箇所: src/backend/postmaster/walsummarizer.c:450

目的

WAL Summarizer プロセスの現在の要約状態（タイムライン、LSN、プロセスID）を共有メモリから取得し、呼び出し元に提供する。

シグネチャまたは形状

void GetWalSummarizerState(TimeLineID *summarized_tli, XLogRecPtr *summarized_lsn,
                           XLogRecPtr *pending_lsn, int *summarizer_pid)

入力と出力

  • **入力**: 状態を格納するためのポインタ引数（summarized_tli, summarized_lsn, pending_lsn, summarizer_pid）。
  • **出力**: なし（引数経由で値を書き戻す）。

前提条件と不変条件

  • WALSummarizerLock を共有モードで取得して共有メモリ WalSummarizerCtl にアクセスする。
  • WalSummarizerCtl->initialized が false の場合、出力値はすべて初期値（0, InvalidXLogRecPtr, -1）に設定される。

アルゴリズムまたは振る舞い

1. WALSummarizerLock を取得する。 2. WalSummarizerCtl->initialized が偽であれば、各出力引数を無効値で初期化する。 3. initialized が真であれば、共有メモリから summarized_tli と summarized_lsn をコピーする。 4. summarizer_pgprocno が INVALID_PROC_NUMBER（プロセス終了済み）の場合、pending_lsn を summarized_lsn とし、summarizer_pid を -1 に設定する。 5. プロセスが稼働中の場合、pending_lsn を取得し、GetPGProcByNumber を介して現在の PID を取得・正規化（0以下を -1 に変換）する。 6. WALSummarizerLock を解放する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/backup/walsummaryfuncs.c:188
  • **呼び出し先**: GetPGProcByNumber（共有メモリ上のプロセス情報取得用）

関連データ構造

  • WalSummarizerCtl: 共有メモリ上の制御構造体。
  • LWLock: WALSummarizerLock を使用して排他制御を行う。

失敗モードと境界条件

  • WalSummarizerCtl が未初期化の場合、安全なデフォルト値が返される。
  • WAL Summarizer プロセスが終了している場合、summarizer_pid は -1 となり、pending_lsn は最後に要約された LSN と同値になる。

保守担当者向けメモ

  • 本関数は共有メモリのロックを取得するため、頻繁な呼び出しはパフォーマンスに影響を与える可能性がある。
  • summarizer_pid の取得において、PID が 0 以下の場合は -1 に正規化する仕様となっている。
