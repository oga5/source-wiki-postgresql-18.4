cnt_length

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/tsrank.c:53

目的

TSVector 内に含まれる全単語の出現位置情報の総数を計算します。この値は、主にランキング計算時の正規化処理において、文書の長さの指標として利用されます。

シグネチャまたは形状

static int cnt_length(TSVector t)

入力と出力

  • **入力**: TSVector t（解析対象の全文検索ベクトル）
  • **出力**: int 型。文書内の全単語の出現位置情報の合計数。

前提条件と不変条件

  • 引数 t は有効な TSVector 構造体である必要があります。
  • ARRPTR(t) および STRPTR(t) マクロにより、ベクトル内の WordEntry 配列の開始位置と終了位置が正しく特定できることを前提とします。

アルゴリズムまたは振る舞い

1. TSVector 内の各 WordEntry を先頭から末尾まで走査します。 2. 各 WordEntry に対して POSDATALEN(t, ptr) を呼び出し、その単語に関連付けられた位置情報の長さを取得します。 3. 位置情報の長さが 0 の場合は 1 を加算し、それ以外の場合はその長さを加算します。 4. 全ての WordEntry を走査した後の合計値を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/tsrank.c 内のランキング計算ロジック（ts_rank および ts_rank_cd 関連関数）から、正規化係数の算出や長さの取得のために複数回呼び出されています。
  • **呼び出し先**:
  • ARRPTR、STRPTR、POSDATALEN などの TSVector 操作用マクロ。

関連データ構造

  • TSVector: 全文検索用の文書表現。
  • WordEntry: TSVector 内の各単語のメタデータ（位置情報や重みを含む）を保持する構造体。

失敗モードと境界条件

  • TSVector が空の場合、ループは実行されず 0 を返します。
  • POSDATALEN が 0 を返す単語（位置情報を持たない単語）は、文書長として 1 としてカウントされます。

保守担当者向けメモ

  • 本関数は tsrank.c 内のランキング計算における正規化の基礎となるため、変更時は ts_rank および ts_rank_cd のスコア出力に影響が出る点に注意してください。
  • POSDATALEN の解釈（0 の場合に 1 を加算するロジック）は、全文検索のランキング仕様に直結しています。
