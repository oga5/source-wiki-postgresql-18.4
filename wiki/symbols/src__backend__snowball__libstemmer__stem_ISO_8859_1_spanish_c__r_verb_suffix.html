r_verb_suffix

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_spanish.c:901

目的

スペイン語のステミング処理において、単語末尾の動詞活用接尾辞を識別し、必要に応じて削除する。

シグネチャまたは形状

static int r_verb_suffix(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング環境を保持する構造体）
  • 出力: 処理が成功または該当なしの場合は 1、エラー発生時は負の値を返す。

前提条件と不変条件

  • z->I[2] に格納された境界位置（R2領域など）より現在位置 z->c が後ろにあること。
  • a_8 配列（96個の要素）を用いて接尾辞の検索を行う。

アルゴリズムまたは振る舞い

1. 現在位置 z->c が z->I[2] より前にある場合は即座に終了する。 2. z->lb を z->I[2] に一時的に設定し、find_among_b を用いて末尾から接尾辞を検索する。 3. マッチした接尾辞の種類（among_var）に応じて以下の処理を行う：

  • case 1: 接尾辞の直前が "gu" であるかを確認し、条件を満たせば接尾辞を削除する。
  • case 2: 条件判定なしで接尾辞を削除する。

    • 4. slice_del を呼び出し、該当する接尾辞を単語から切り離す。

呼び出し元と呼び出し先

  • 呼び出し元: spanish_ISO_8859_1_stem（直接の呼び出し元であることはソースコードのフローから明白）。また、Catalan, French, Italian, Portuguese等の各言語ステマーからも呼び出されている。
  • 呼び出し先: find_among_b（接尾辞検索）、slice_del（文字列削除）。

関連データ構造

  • SN_env: ステミング環境を管理する構造体。
  • a_8: 動詞接尾辞のパターンを保持する among 構造体の配列。

失敗モードと境界条件

  • z->c <= z->I[2] の場合、処理は行われない。
  • find_among_b で接尾辞が見つからない場合、処理は行われない。
  • slice_del が内部でエラーを返した場合、その値をそのまま呼び出し元へ伝播する。
