float_to_shortest_decimal_bufn

定義

  • 種別: function
  • 定義箇所: src/common/f2s.c:741

目的

IEEE 754 単精度浮動小数点数（float）を、その値を正確に表現する最短の10進数文字列に変換し、指定されたバッファに書き込みます。

シグネチャまたは形状

int float_to_shortest_decimal_bufn(float f, char *result)

入力と出力

  • **入力**:
  • f: 変換対象の単精度浮動小数点数。
  • result: 変換後の文字列を格納するバッファ（十分なサイズが必要）。
  • **出力**:
  • 戻り値: 書き込まれた文字列の長さ（バイト数）。

前提条件と不変条件

  • result バッファは、変換結果を格納するために十分なメモリ領域を確保している必要があります。
  • 入力値は IEEE 754 単精度浮動小数点数として解釈されます。

アルゴリズムまたは振る舞い

1. **ビット解析**: float_to_bits を使用して浮動小数点数を符号、仮数、指数に分解します。 2. **特殊値処理**: 指数が最大値（無限大またはNaN）の場合、または指数と仮数が共に0（ゼロ）の場合、copy_special_str を呼び出して即座に結果を返します。 3. **高速パス**: f2d_small_int を試し、整数として表現可能な場合は計算をスキップします。 4. **Ryu変換**: 上記以外の場合、f2d を使用してRyuアルゴリズムにより最短の10進数表現（仮数と指数）を計算します。 5. **フォーマット**: to_chars を使用して、計算された数値を文字列形式に変換し、result バッファへ書き込みます。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/common/f2s.c 内の他の関数から参照されています。
  • src/include/common/shortest_dec.h にプロトタイプ宣言が存在します。
  • **呼び出し先**:
  • float_to_bits
  • copy_special_str
  • f2d_small_int
  • f2d
  • to_chars

関連データ構造

  • floating_decimal_32: 変換過程で仮数と指数を保持するために使用される中間構造体。

失敗モードと境界条件

  • 特殊値（NaN, Infinity, 0）は copy_special_str によって個別に処理されます。
  • 非常に大きな値や小さな値は、Ryuアルゴリズムによって科学的表記法または固定小数点表記に適切に変換されます。

保守担当者向けメモ

  • 本関数は f2s.c のメインエントリポイントです。
  • 変換ロジックの詳細は f2d および to_chars に依存しています。
  • 32ビット環境と64ビット環境の両方で動作するよう設計されています。
