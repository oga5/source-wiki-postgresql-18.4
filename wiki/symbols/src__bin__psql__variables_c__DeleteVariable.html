DeleteVariable

定義

  • 種別: function
  • 定義箇所: src/bin/psql/variables.c:473

目的

指定された変数空間（VariableSpace）から、特定の名前を持つ変数を削除します。

シグネチャまたは形状

bool DeleteVariable(VariableSpace space, const char *name);

入力と出力

  • **入力**:
  • space: 変数を管理する VariableSpace 構造体。
  • name: 削除対象の変数名。
  • **出力**:
  • bool: 削除処理が成功した場合は true、失敗した場合は false を返します。

前提条件と不変条件

  • SetVariable のラッパーとして実装されており、内部的に SetVariable(space, name, NULL) を呼び出します。
  • 変数に代入フックが設定されている場合、そのフックの戻り値が削除の成否に影響します。

アルゴリズムまたは振る舞い

1. 引数として受け取った space と name をそのまま SetVariable 関数へ渡します。 2. SetVariable が内部で実行する代入フックの検証結果をそのまま戻り値として返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/bin/psql/startup.c:653
  • **呼び出し先**:
  • SetVariable (同ファイル内)

関連データ構造

  • VariableSpace: 変数リストを保持する構造体。

失敗モードと境界条件

  • 指定した名前の変数が存在しない場合や、代入フックによって削除が拒否された場合、SetVariable の挙動に従い false を返します。

保守担当者向けメモ

  • 本関数は SetVariable への単なるエイリアスです。変数の削除ロジックに変更を加える場合は、SetVariable の実装を確認してください。
