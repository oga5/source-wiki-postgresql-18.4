NumericGetDatum

定義

  • 種別: function
  • 定義箇所: src/include/utils/numeric.h:72

目的

Numeric 型の値を、PostgreSQLの関数マネージャ（fmgr）が扱う汎用的な Datum 型に変換します。

シグネチャまたは形状

static inline Datum NumericGetDatum(Numeric X)

入力と出力

  • **入力**: Numeric X （NumericData 構造体へのポインタ）
  • **出力**: Datum （変換された数値データ）

前提条件と不変条件

  • X は有効な Numeric ポインタである必要があります。
  • 本関数はインライン関数として定義されており、fmgr.h で定義される PointerGetDatum マクロを利用します。

アルゴリズムまたは振る舞い

Numeric ポインタを Datum 型へキャスト（変換）して返します。内部的には PointerGetDatum(X) を呼び出すだけの単純なラッパーです。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/btree_gin/btree_gin.c
  • contrib/btree_gist/btree_numeric.c
  • contrib/jsonb_plperl/jsonb_plperl.c
  • contrib/jsonb_plpython/jsonb_plpython.c
  • src/backend/executor/execExprInterp.c
  • src/backend/utils/adt/cash.c
  • **呼び出し先**:
  • PointerGetDatum（fmgr.h で定義されるマクロ）

関連データ構造

  • Numeric: NumericData 構造体への不透明ポインタ。
  • Datum: PostgreSQLの関数間で値をやり取りするための汎用型。

失敗モードと境界条件

  • 本関数自体は単純なキャストであるため、関数内部での失敗モードはありません。
  • 入力 X が NULL の場合、PointerGetDatum の仕様に従った挙動となります。

保守担当者向けメモ

  • 本関数は numeric.h で定義された公開インターフェースの一部です。
  • Numeric 型の内部構造を変更する場合でも、この変換インターフェースは維持される必要があります。
  • 演算結果をシステムへ返す際の標準的な変換手段として利用してください。
