testcustomrmgrs_desc

定義

  • 種別: function
  • 定義箇所: src/test/modules/test_custom_rmgrs/test_custom_rmgrs.c:90

目的

カスタムWALリソースマネージャのWALレコード内容を人間が読める形式に変換し、デバッグやログ出力用に文字列バッファへ追加する。

シグネチャまたは形状

void testcustomrmgrs_desc(StringInfo buf, XLogReaderState *record)

入力と出力

  • **入力**:
  • buf: 文字列情報を格納するための StringInfo バッファ。
  • record: 解析対象のWALレコード状態を保持する XLogReaderState 構造体。
  • **出力**: なし（buf を通じて文字列が追記される）。

前提条件と不変条件

  • record は有効な XLogReaderState を指している必要がある。
  • XLogRecGetData および XLogRecGetInfo を使用してレコードデータにアクセスするため、レコードが正しく読み込まれていることが前提となる。

アルゴリズムまたは振る舞い

1. XLogRecGetData を呼び出し、WALレコードのデータ部分を取得する。 2. XLogRecGetInfo を呼び出し、レコードのサブタイプ情報を取得する。この際、XLR_INFO_MASK を用いてマスク処理を行う。 3. レコードのサブタイプが XLOG_TEST_CUSTOM_RMGRS_MESSAGE と一致する場合:

  • xl_testcustomrmgrs_message 構造体としてデータをキャストする。
  • ペイロードのサイズ（message_size）を buf に追記する。
  • ペイロード本体（message）をバイナリ文字列として buf に追記する。

呼び出し元と呼び出し先

  • **呼び出し元**: WAL解析サブシステム（RmgrData 構造体の .rm_desc コールバックとして登録され、WAL読み込み時に呼び出される）。
  • **呼び出し先**:
  • XLogRecGetData (PostgreSQL内部関数)
  • XLogRecGetInfo (PostgreSQL内部関数)
  • appendStringInfo (PostgreSQL内部関数)
  • appendBinaryStringInfo (PostgreSQL内部関数)

関連データ構造

  • StringInfo: 文字列構築用バッファ。
  • XLogReaderState: WALレコード読み込み状態。
  • xl_testcustomrmgrs_message: 本モジュールで定義されたカスタムWALレコード構造体。

失敗モードと境界条件

  • info が XLOG_TEST_CUSTOM_RMGRS_MESSAGE 以外の場合、本関数は何も出力せず終了する。
  • record が不正なデータを含む場合、キャスト後の xlrec へのアクセスでメモリ安全性の問題が発生する可能性がある。

保守担当者向けメモ

  • 本関数は testcustomrmgrs_rmgr 構造体の rm_desc フィールドに割り当てられている。
  • 新しいWALレコードタイプを追加した場合は、本関数の if 文を拡張して対応する記述処理を追加すること。
