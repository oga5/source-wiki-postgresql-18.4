clearcvec

定義

  • 種別: function
  • 定義箇所: src/backend/regex/regc_cvec.c:68

目的

正規表現コンパイル時に使用される文字セット構造体（cvec）のカウンタをリセットし、再利用可能な状態にする。

シグネチャまたは形状

static struct cvec *clearcvec(struct cvec *cv)

入力と出力

  • **入力**: cv (struct cvec *) - 初期化またはリセット対象の文字セット構造体。
  • **出力**: struct cvec * - リセットされた cv へのポインタを返す。

前提条件と不変条件

  • cv は NULL であってはならない（assert による検証）。

アルゴリズムまたは振る舞い

1. cv->nchrs（保持する文字数）を 0 に設定する。 2. cv->nranges（保持する範囲数）を 0 に設定する。 3. cv->cclasscode を -1 に設定する。 4. 変更後の cv を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/regex/regc_cvec.c 内の getcvec 関数から呼び出される。
  • **呼び出し先**:
  • 特筆すべき外部関数呼び出しはない。

関連データ構造

  • struct cvec: 正規表現のブラケット式（[...]）解析時に使用される文字セットを保持する構造体。regguts.h で定義されている。

失敗モードと境界条件

  • cv が NULL の場合、アサーション失敗によりプログラムが終了する可能性がある。

保守担当者向けメモ

  • 本関数は regcomp.c にインクルードされる regc_cvec.c 内で定義されており、静的関数として最適化される。
  • cvec のメモリ管理は vars 構造体を通じて行われるため、本関数でメモリの解放は行わない。
