r_noun_sfx

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_irish.c:334

目的

アイルランド語のステミング処理において、単語末尾の名詞接尾辞を特定し、条件を満たす場合に削除を行う。

シグネチャまたは形状

static int r_noun_sfx(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z（Snowball ステミング環境へのポインタ）
  • **出力**: int 型。成功時は 1、ルールにマッチしない場合は 0、処理中にエラーが発生した場合は負の値を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • 内部で find_among_b を使用するため、対象の単語末尾が a_1 で定義された接尾辞リストと照合される。
  • 削除操作は r_R1 または r_R2 で定義された領域内でのみ許可される。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を z->ket に設定する。 2. find_among_b を用いて、a_1（16個のルール）から後方一致検索を行う。 3. マッチした among_var に応じて分岐する:

  • **case 1**: r_R1 領域内であることを確認し、slice_del で接尾辞を削除する。
  • **case 2**: r_R2 領域内であることを確認し、slice_del で接尾辞を削除する。

    • 4. 削除成功時に 1 を返し、失敗時はエラーコードを伝播させる。

呼び出し元と呼び出し先

  • **呼び出し元**: irish_ISO_8859_1_stem（stem_ISO_8859_1_irish.c および stem_UTF_8_irish.c 内のメイン処理フロー）。
  • **呼び出し先**: find_among_b, r_R1, r_R2, slice_del（いずれも header.h 経由で提供されるランタイム関数）。

関連データ構造

  • struct SN_env: ステミングの状態を保持する構造体。
  • a_1: 名詞接尾辞のパターンを保持する among 構造体の配列。

失敗モードと境界条件

  • find_among_b がマッチしない場合、即座に 0 を返す。
  • r_R1 または r_R2 のチェックで範囲外と判定された場合、その判定結果（負の値または0）をそのまま返す。
  • slice_del がメモリ不足等の理由で失敗した場合、負の値を返す。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードである。ロジックの変更が必要な場合は、直接編集せず元の .sbl ソースファイルを修正すること。
  • a_1 配列の定義やルール数（16）は、stem_ISO_8859_1_irish.c 内の他の定義と整合している必要がある。
