MatViewIncrementalMaintenanceIsEnabled

定義

  • 種別: function
  • 定義箇所: src/backend/commands/matview.c:963

目的

マテリアライズド・ビューの増分メンテナンス処理が現在有効であるかどうかを判定する。

シグネチャまたは形状

bool MatViewIncrementalMaintenanceIsEnabled(void)

入力と出力

  • 入力: なし
  • 出力: bool 型。増分メンテナンス処理中であれば true、そうでなければ false を返す。

前提条件と不変条件

  • 内部変数 matview_maintenance_depth が 0 より大きい場合に true を返す。

アルゴリズムまたは振る舞い

  • matview_maintenance_depth の値を参照し、その値が 0 より大きいかどうかを評価して結果を返す。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/backend/commands/matview.c:964
  • src/backend/executor/execMain.c:1111
  • 呼び出し先:
  • 情報なし（本関数は変数の参照のみを行う）

関連データ構造

  • matview_maintenance_depth: 本関数が参照する内部状態変数。

失敗モードと境界条件

  • 特になし。常に安全に呼び出し可能。

保守担当者向けメモ

  • 本関数は matview_maintenance_depth の値に依存している。この変数の増減を行う OpenMatViewIncrementalMaintenance および CloseMatViewIncrementalMaintenance とセットで管理すること。
