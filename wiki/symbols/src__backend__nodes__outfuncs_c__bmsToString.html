bmsToString

定義

  • 種別: function
  • 定義箇所: src/backend/nodes/outfuncs.c:821

目的

Bitmapset 構造体を、デバッグやシリアライズに適した文字列形式に変換します。

シグネチャまたは形状

char *bmsToString(const Bitmapset *bms)

入力と出力

  • **入力**: bms (変換対象の Bitmapset へのポインタ)
  • **出力**: 変換された文字列を保持する char * 型のメモリ領域。呼び出し元は、このメモリの解放に責任を持つ必要があります。

前提条件と不変条件

  • bms が NULL の場合、outBitmapset の実装に依存します。
  • StringInfo を使用して文字列を構築するため、メモリ不足時にはエラーが発生する可能性があります。

アルゴリズムまたは振る舞い

1. initStringInfo を呼び出し、文字列構築用のバッファ StringInfoData を初期化します。 2. outBitmapset を呼び出し、Bitmapset の内容をバッファに書き込みます。 3. 構築された文字列データ（str.data）を返却します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/executor/execPartition.c
  • src/backend/optimizer/plan/setrefs.c
  • **呼び出し先**:
  • initStringInfo (lib/stringinfo.h)
  • outBitmapset (src/backend/nodes/outfuncs.c)

関連データ構造

  • Bitmapset: 変換対象となるビットマップ集合構造体。
  • StringInfoData: 文字列構築用の中間バッファ。

失敗モードと境界条件

  • outBitmapset が内部で扱う Bitmapset の構造に依存します。
  • メモリ割り当て失敗時は、PostgreSQL のエラーハンドリング機構により中断されます。

保守担当者向けメモ

  • 本関数は Bitmapset を文字列化する標準的なインターフェースです。
  • outBitmapset の出力形式を変更する場合は、本関数経由で生成される文字列も影響を受けるため注意してください。
