r_tidy

定義

  • **種別**: function
  • **定義箇所**: src/backend/snowball/libstemmer/stem_UTF_8_finnish.c:576

目的

フィンランド語のステミング処理の最終段階として、単語末尾の不要な文字や二重母音の調整を行い、語幹を正規化する。

シグネチャまたは形状

static int r_tidy(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（Snowball ステミング環境構造体）
  • **出力**: 成功時に 1、処理対象外または変更なしの場合に 0、メモリ不足等のエラー時に負の値を返す。

前提条件と不変条件

  • z->I[1] に領域 R1 の開始位置が格納されていること。
  • z->c（現在のカーソル位置）が z->I[1] 以上であること（R1 領域内での処理が前提）。

アルゴリズムまたは振る舞い

1. **領域制限**: z->lb を z->I[1] に設定し、R1 領域内でのみ操作を制限する。 2. **長母音の処理**: r_LONG を呼び出し、該当する場合は末尾の文字を削除する。 3. **母音+子音の削除**: g_AEI（母音グループ）と g_C（子音グループ）に基づき、末尾のパターンを削除する。 4. **二重母音の調整**: 'j' で終わる単語に対し、直前が 'o' または 'u' である場合に 'j' を削除する。 5. **'oj' の削除**: 末尾が 'oj' である場合に削除する。 6. **最終正規化**: g_V1（母音グループ）と g_C（子音グループ）を用いて、単語末尾の特定のパターンを z->S[0] に保存された文字列と比較し、一致すれば削除する。 7. **復元**: z->lb を元の値に戻す。

呼び出し元と呼び出し先

  • **呼び出し元**: finnish_UTF_8_stem（src/backend/snowball/libstemmer/stem_UTF_8_finnish.c:714）
  • **呼び出し先**:
  • r_LONG
  • skip_b_utf8
  • slice_del
  • in_grouping_b_U
  • slice_to
  • eq_v_b

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • g_AEI, g_C, g_V1: 文字グループを定義するビットマップ配列。

失敗モードと境界条件

  • z->c < z->I[1] の場合、処理を行わず 0 を返す。
  • slice_to や slice_del 等のランタイム関数がエラーを返した場合、そのエラーコードを伝播させる。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードである。ロジックの変更が必要な場合は、直接編集せず Snowball ソースファイルを修正すること。
  • z->I[1] は領域 R1 の境界を示すため、この値の算出ロジック（r_mark_regions）との整合性に注意が必要。
