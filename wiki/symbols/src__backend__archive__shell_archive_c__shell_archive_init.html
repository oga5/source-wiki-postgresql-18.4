shell_archive_init

定義

  • 種別: function
  • 定義箇所: src/backend/archive/shell_archive.c:39

目的

PostgreSQLのアーカイブモジュールインターフェースを初期化し、シェルコマンド実行用のコールバック構造体へのポインタを返します。

シグネチャまたは形状

const ArchiveModuleCallbacks *shell_archive_init(void);

入力と出力

  • 入力: なし
  • 出力: shell_archive_callbacks を指す定数ポインタ（ArchiveModuleCallbacks *）。

前提条件と不変条件

  • 本関数は、アーカイブプロセスがモジュールをロードした際に呼び出されることを想定しています。
  • 返される ArchiveModuleCallbacks 構造体は静的に定義されており、呼び出しを通じて不変です。

アルゴリズムまたは振る舞い

1. 内部で定義された shell_archive_callbacks 構造体のアドレスを返します。 2. この構造体には、アーカイブプロセスがWALファイルを処理するために必要な関数ポインタ（設定確認、ファイルアーカイブ、シャットダウン等）が含まれています。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/postmaster/pgarch.c において、アーカイブモジュールの初期化時に参照されています。
  • 呼び出し先: なし（静的な構造体のアドレスを返すのみ）。

関連データ構造

  • ArchiveModuleCallbacks: アーカイブモジュールが実装すべきインターフェースを定義する構造体。
  • shell_archive_callbacks: 本モジュール内で定義されている、ArchiveModuleCallbacks の具体的な実装インスタンス。

失敗モードと境界条件

  • 本関数自体は常に成功し、有効なポインタを返します。エラー発生の条件はありません。

保守担当者向けメモ

  • 本関数はアーカイブモジュールのエントリポイントです。新しいコールバック関数を追加した場合は、shell_archive_callbacks 構造体を更新してください。
  • shell_archive.h にて外部公開（extern）されています。
