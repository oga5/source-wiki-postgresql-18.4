mulShift

定義

  • 種別: function
  • 定義箇所: src/common/f2s.c:119

目的

浮動小数点数の変換処理において、64ビットの factor と32ビットの m の乗算結果に対して右シフト演算を行い、結果の上位32ビットを抽出する。

シグネチャまたは形状

static inline uint32 mulShift(const uint32 m, const uint64 factor, const int32 shift)

入力と出力

  • **入力**:
  • m: 乗数（32ビット符号なし整数）
  • factor: 乗数（64ビット符号なし整数）
  • shift: 右シフト量（32ビット符号付き整数）
  • **出力**:
  • 計算結果の32ビット符号なし整数。

前提条件と不変条件

  • shift > 32 であること（Assert により検証）。
  • RYU_32_BIT_PLATFORM 定義の有無により、プラットフォームに適したビット操作が行われる。

アルゴリズムまたは振る舞い

1. m と factor を上位・下位の32ビットに分割し、乗算を実行する。 2. RYU_32_BIT_PLATFORM が定義されている場合:

  • 64ビットの右シフトを避け、32ビット単位の加算とシフトを組み合わせて上位32ビット相当の結果を算出する。

    • 3. それ以外の場合:

  • 64ビットの加算を行い、指定された shift - 32 分だけ右シフトする。

    • 4. 最終的に結果を uint32 にキャストして返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/common/f2s.c: float_to_shortest_decimal_bufn 内の変換ロジックから呼び出される。
  • src/common/d2s.c: 複数の箇所で呼び出されている（ただし、d2s.c での呼び出しは引数の型が異なるため、オーバーロードまたはマクロ的な利用の可能性がある）。
  • **呼び出し先**:
  • 特筆すべき関数呼び出しはない（算術演算のみ）。

関連データ構造

  • FLOAT_POW5_INV_SPLIT: f2s.c 内で定義される事前計算テーブル。
  • FLOAT_POW5_SPLIT: f2s.c 内で定義される事前計算テーブル。

失敗モードと境界条件

  • shift <= 32 の場合、Assert によりプログラムが停止する可能性がある。
  • RYU_32_BIT_PLATFORM 未定義環境において、shiftedSum が PG_UINT32_MAX を超える場合は Assert により停止する。

保守担当者向けメモ

  • MSVCコンパイラで __allmul ライブラリ関数が呼び出されるのを防ぐため、明示的なキャストが行われている。
  • 32ビット環境と64ビット環境で実装が分岐しているため、変更を加える際は両方のパスの整合性に注意すること。
