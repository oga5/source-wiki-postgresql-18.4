make_path_initial_array

定義

  • 種別: function
  • 定義箇所: src/backend/rewrite/rewriteSearchCycle.c:158

目的

再帰クエリの SEARCH または CYCLE 句において、パス追跡用の配列の初期状態（単一要素を持つ配列）を生成します。

シグネチャまたは形状

static Expr *make_path_initial_array(RowExpr *rowexpr)

入力と出力

  • **入力**: RowExpr *rowexpr - パスに含める初期値となる行式。
  • **出力**: Expr * - RECORDARRAYOID 型の ArrayExpr ノード。

前提条件と不変条件

  • 入力される rowexpr は有効な RowExpr ノードである必要があります。
  • 生成される配列は、要素型として RECORDOID を持ちます。

アルゴリズムまたは振る舞い

1. makeNode(ArrayExpr) を使用して新しい配列ノードを割り当てます。 2. 配列の型情報を RECORDARRAYOID に、要素の型情報を RECORDOID に設定します。 3. 位置情報 location を -1 に初期化します。 4. 入力された rowexpr を単一要素として含むリストを作成し、elements フィールドに格納します。 5. 構築した ArrayExpr を Expr * にキャストして返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/rewrite/rewriteSearchCycle.c:329
  • src/backend/rewrite/rewriteSearchCycle.c:344
  • **呼び出し先**:
  • makeNode (マクロ)
  • list_make1 (マクロ)

関連データ構造

  • ArrayExpr: PostgreSQL の配列式ノード。
  • RowExpr: 行式ノード。
  • RECORDARRAYOID, RECORDOID: catalog/pg_type_d.h で定義される内部型 OID。

失敗モードと境界条件

  • メモリ不足時には makeNode 経由でエラーが発生する可能性があります。
  • 入力 rowexpr が NULL の場合、list_make1 の挙動により予期せぬ動作となる可能性があります。

保守担当者向けメモ

  • 本関数は SEARCH および CYCLE 句の書き換えにおいて、再帰の初期ステップでパス配列を初期化するために使用されます。
  • 型定義（RECORDARRAYOID 等）を変更する場合は、本関数が生成するノードの整合性に注意してください。
