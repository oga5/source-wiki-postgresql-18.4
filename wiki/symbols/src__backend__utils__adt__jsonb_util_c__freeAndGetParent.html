freeAndGetParent

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/jsonb_util.c:1053

目的

現在の JsonbIterator をメモリから解放し、その親となるイテレータを返却することで、ネストされた JSON 構造の走査状態を一段階戻す。

シグネチャまたは形状

static JsonbIterator *freeAndGetParent(JsonbIterator *it);

入力と出力

  • **入力**: it (解放対象の JsonbIterator へのポインタ)
  • **出力**: it->parent (解放されたイテレータの親にあたる JsonbIterator へのポインタ)

前提条件と不変条件

  • 入力 it は有効な JsonbIterator 構造体であること。
  • it は palloc によってメモリ確保されている必要がある（pfree を実行するため）。

アルゴリズムまたは振る舞い

1. 引数 it が保持する parent メンバをローカル変数 v に退避する。 2. pfree(it) を呼び出し、現在のイテレータが使用していたメモリを解放する。 3. 退避しておいた親イテレータ v を呼び出し元へ返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/jsonb_util.c:902
  • src/backend/utils/adt/jsonb_util.c:956
  • **呼び出し先**:
  • pfree (PostgreSQL メモリ管理関数)

関連データ構造

  • JsonbIterator: JSONB コンテナを走査するための状態管理構造体。parent メンバを介して再帰的な走査をサポートする。

失敗モードと境界条件

  • it が NULL の場合、it->parent へのアクセスによりセグメンテーション違反が発生する可能性がある。
  • it がルートイテレータ（parent が NULL）である場合、戻り値として NULL が返される。

保守担当者向けメモ

  • 本関数は JsonbIterator のスタックを巻き戻すためのユーティリティである。イテレータのメモリ管理は呼び出し側で適切に制御する必要がある。
