pg_atomic_exchange_u32_impl

定義

  • 種別: function
  • 定義箇所: src/include/port/atomics/generic.h:159

目的

pg_atomic_uint32 型の値を新しい値でアトミックに置換し、置換前の値を返します。

シグネチャまたは形状

static inline uint32 pg_atomic_exchange_u32_impl(volatile pg_atomic_uint32 *ptr, uint32 xchg_)

入力と出力

  • **入力**:
  • ptr: 操作対象となる pg_atomic_uint32 へのポインタ。
  • xchg_: 書き込む新しい uint32 値。
  • **出力**:
  • 置換前の uint32 値。

前提条件と不変条件

  • ptr は volatile 修飾された pg_atomic_uint32 を指している必要があります。
  • 本関数は atomics.h を経由して呼び出されることを前提としています。

アルゴリズムまたは振る舞い

1. 現在の値を old 変数に読み込みます（この時点での読み込みはアトミックでなくても許容されます）。 2. pg_atomic_compare_exchange_u32_impl を使用して、ptr の値が old と一致するか確認し、一致すれば xchg_ で更新します。 3. 更新に失敗した場合（他スレッドによる割り込み等）、old が最新の値に更新されるため、成功するまでループを繰り返します。 4. 成功時に、更新前の値を返します。

関連データ構造

  • pg_atomic_uint32: アトミック操作対象の 32 ビット整数構造体。

失敗モードと境界条件

  • 本関数はループ構造を持つため、競合が極めて激しい環境では、ハードウェア固有の最適化が提供されていない場合、処理の完了まで待機時間が発生する可能性があります。

保守担当者向けメモ

  • 本関数はフォールバック実装です。アーキテクチャ固有の最適化（arch-*.h）が存在しない場合にのみ使用されます。
  • pg_atomic_compare_exchange_u32_impl の成否に依存するため、基盤となる比較交換操作の正確性に依存します。
