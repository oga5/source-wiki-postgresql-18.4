RmgrIdIsCustom

定義

  • 種別: static inline function
  • 定義箇所: src/include/access/rmgr.h:47

目的

指定されたリソースマネージャID（rmid）が、拡張機能用に予約されたカスタムID範囲内にあるかどうかを判定する。

シグネチャまたは形状

static inline bool RmgrIdIsCustom(int rmid)

入力と出力

  • **入力**: int rmid （検証対象のリソースマネージャID）
  • **出力**: bool （rmid がカスタム範囲内であれば true、それ以外は false）

前提条件と不変条件

  • rmid は RM_MIN_CUSTOM_ID から RM_MAX_CUSTOM_ID までの範囲に含まれる場合にカスタムIDとみなされる。
  • 本関数は rmgr.h 内で定義された定数に依存する。

アルゴリズムまたは振る舞い

rmid が RM_MIN_CUSTOM_ID 以上かつ RM_MAX_CUSTOM_ID 以下であるかを比較演算子で評価し、その結果を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/access/transam/rmgr.c:113
  • src/bin/pg_waldump/pg_waldump.c:692
  • src/bin/pg_waldump/pg_waldump.c:959
  • src/include/access/rmgr.h:53 （RmgrIdIsValid マクロ内）
  • **呼び出し先**:
  • 外部関数への呼び出しはなし。

関連データ構造

  • RmgrId: WALレコードを識別する uint8 型のID。

失敗モードと境界条件

  • rmid が負の値や、uint8 の範囲外である場合の挙動は、呼び出し側の入力値に依存する。
  • 組み込みID（RM_MAX_BUILTIN_ID 以下）は常に false を返す。

保守担当者向けメモ

  • 本関数はWALレコードの解析や検証において、組み込みリソースマネージャと拡張機能によるカスタムリソースマネージャを区別するために不可欠である。
  • ID範囲の変更が必要な場合は、rmgr.h で定義されている RM_MIN_CUSTOM_ID および RM_MAX_CUSTOM_ID の定数定義を更新すること。
