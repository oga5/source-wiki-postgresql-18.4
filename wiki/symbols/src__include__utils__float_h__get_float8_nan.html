get_float8_nan

定義

  • src/include/utils/float.h:122

目的

プラットフォーム間で一貫した float8 型の NaN（Not-a-Number）値を取得する。

シグネチャまたは形状

static inline float8 get_float8_nan(void)

入力と出力

  • 入力: なし
  • 出力: float8 型の NaN 値

前提条件と不変条件

  • C99 標準の NAN マクロが利用可能な環境ではそれを使用する。
  • NetBSD/MIPS のような一部の環境で (float8) NAN が正しく動作しないケースを考慮し、フォールバックとして 0.0 / 0.0 による NaN 生成を行う。

アルゴリズムまたは振る舞い

1. NAN マクロが定義されており、かつ NetBSD/MIPS 環境ではない場合、(float8) NAN をキャストして返す。 2. それ以外の場合、0.0 / 0.0 を実行して NaN を生成し、それを返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/access/hash/hashfunc.c
  • src/backend/access/spgist/spgproc.c
  • src/backend/utils/adt/float.c
  • **呼び出し先**:
  • 直接的な関数呼び出しはなし（算術演算またはマクロによる値生成）。

関連データ構造

  • float8 (PostgreSQLの倍精度浮動小数点型)

失敗モードと境界条件

  • 浮動小数点演算をサポートしない環境では、0.0 / 0.0 の挙動がプラットフォームの IEEE 754 実装に依存する。

保守担当者向けメモ

  • 本関数はインライン関数として定義されており、パフォーマンスを重視している。
  • NetBSD/MIPS 特有の制約が解消された場合、条件分岐を簡素化できる可能性がある。
