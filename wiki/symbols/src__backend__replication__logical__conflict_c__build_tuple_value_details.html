build_tuple_value_details

定義

  • 種別: function
  • 定義箇所: src/backend/replication/logical/conflict.c:317

目的

論理レプリケーションの競合発生時に、関連するタプル（ローカル行、リモート行、検索キー、レプリカアイデンティティ）の状態を可視化するための詳細文字列を構築する。

シグネチャまたは形状

static char *build_tuple_value_details(EState *estate, ResultRelInfo *relinfo,
                                      ConflictType type,
                                      TupleTableSlot *searchslot,
                                      TupleTableSlot *localslot,
                                      TupleTableSlot *remoteslot,
                                      Oid indexoid)

入力と出力

  • **入力**:
  • estate, relinfo: 実行状態およびリレーション情報。
  • type: 競合の種類（ConflictType）。
  • searchslot, localslot, remoteslot: 競合に関与する各タプルのスロット。
  • indexoid: 一意制約違反時に使用するインデックスのOID。
  • **出力**: 構築された詳細情報を含む文字列（char *）。情報がない場合は NULL を返す。

前提条件と不変条件

  • searchslot, localslot, remoteslot のうち少なくとも一つは非NULLであること。
  • 一意制約違反（CT_INSERT_EXISTS 等）の場合、indexoid が有効であり、かつ localslot が提供されていること。

アルゴリズムまたは振る舞い

1. StringInfo を初期化し、提供されたスロットの有無に応じて以下の順序で情報を連結する。 2. **一意制約違反**: indexoid を用いてキー値を文字列化し、Key ... として追加。 3. **ローカル行**: localslot があれば existing local row ... として追加。 4. **リモート行**: remoteslot があれば、挿入/更新列のビットマップを考慮して remote row ... として追加。 5. **検索スロット**: searchslot があれば、レプリカアイデンティティインデックスの有無に応じて replica identity ... または replica identity full ... として追加。 6. 最後にピリオドを付与して文字列を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/replication/logical/conflict.c 内の競合報告ロジック（例: ReportApplyConflict からの呼び出しが示唆される）。
  • **呼び出し先**: build_index_value_desc, ExecBuildSlotValueDescription, ExecGetInsertedCols, ExecGetUpdatedCols, GetRelationIdentityOrPK, initStringInfo, appendStringInfo 等。

関連データ構造

  • ConflictType: 競合の種類を定義。
  • TupleTableSlot: タプルデータ保持用。
  • ResultRelInfo: リレーションの実行時情報。

失敗モードと境界条件

  • すべての入力スロットが NULL の場合、Assert により停止する。
  • 構築する情報が一つも存在しない場合、NULL を返す。
  • track_commit_timestamp が無効な場合、関連するタイムスタンプ情報は含まれない。

保守担当者向けメモ

  • StringInfo を使用しているため、メモリ割り当てに注意すること。
  • modifiedCols の構築において、論理レプリケーションでは挿入列のビットマップが保持されないが、整合性のために ExecGetInsertedCols と ExecGetUpdatedCols を併用している。
