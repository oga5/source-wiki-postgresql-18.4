cat_str

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/preproc/util.c:219

目的

可変個の文字列引数を順次連結し、単一の文字列として構築する。

シグネチャまたは形状

char *cat_str(int count, ...);

入力と出力

  • **入力**:
  • count: 連結する文字列の総数。
  • ...: count 個の char * 型引数。
  • **出力**:
  • 連結された結果を保持する char * 型の文字列。

前提条件と不変条件

  • 戻り値の文字列は loc_alloc を介して確保された一時メモリ領域に配置されるため、呼び出し側で free を行う必要はない。
  • 解析サイクルの終了時に reclaim_local_storage() が呼び出されることで、戻り値のメモリ領域も一括解放される。

アルゴリズムまたは振る舞い

1. va_start を使用して可変引数リストを初期化する。 2. 最初の引数を初期文字列として取得する。 3. 残りの count - 1 個の引数に対し、cat2_str を繰り返し呼び出して文字列を順次連結する。 4. 最終的な連結結果を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: 提供されたソース情報からは具体的な呼び出し箇所は特定されていない。
  • **呼び出し先**: cat2_str（文字列連結の基盤関数）。

関連データ構造

  • loc_chunk: 一時メモリ管理用のチャンク構造体。本関数が生成する文字列は、このチャンク内に割り当てられる。

失敗モードと境界条件

  • count が 1 未満の場合の挙動は定義されていない。
  • cat2_str が内部的に loc_alloc を使用するため、メモリ不足時には mm_alloc 経由でエラー終了する可能性がある。

保守担当者向けメモ

  • 本関数は loc_alloc を利用するユーティリティであるため、戻り値を永続的に保持することはできない。解析の節目を越えて文字列を保持する必要がある場合は、別途 strdup 等でコピーを作成すること。
