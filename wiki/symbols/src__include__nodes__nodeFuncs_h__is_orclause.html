is_orclause

定義

  • 種別: static inline function
  • 定義箇所: src/include/nodes/nodeFuncs.h:117

目的

与えられたノードが OR 演算を表す BoolExpr ノードであるかを判定する。

シグネチャまたは形状

static inline bool is_orclause(const void *clause)

入力と出力

  • **入力**: clause (任意のノードへのポインタ)
  • **出力**: bool 型。ノードが OR 演算子を持つ BoolExpr であれば true、それ以外（NULL を含む）であれば false を返す。

前提条件と不変条件

  • clause が NULL の場合、安全に false を返す。
  • clause が指すメモリ領域は、IsA マクロによる型チェックが可能な有効なノード構造体である必要がある。

アルゴリズムまたは振る舞い

1. clause が NULL でないことを確認する。 2. IsA(clause, BoolExpr) マクロを使用して、ノードの型が T_BoolExpr であるかを確認する。 3. ノードを BoolExpr 構造体にキャストし、その boolop フィールドが OR_EXPR と一致するかを判定する。

  • **呼び出し元**:

  • **呼び出し先**:
  • IsA マクロ（ノード型判定用）

関連データ構造

  • BoolExpr: src/include/nodes/primnodes.h で定義される、論理演算（AND, OR, NOT）を保持するノード構造体。

失敗モードと境界条件

  • clause が BoolExpr 以外のノード型である場合、IsA マクロにより正しく false が返される。
  • clause が指すメモリが不正な場合、IsA マクロの内部でセグメンテーション違反が発生する可能性がある。
