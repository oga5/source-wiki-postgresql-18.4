pg_atomic_fetch_sub_u32_impl

定義

  • 種別: function
  • 定義箇所: src/include/port/atomics/generic-gcc.h:215

目的

pg_atomic_uint32 型の変数に対して、アトミックに減算を行い、減算前の値を返します。

シグネチャまたは形状

static inline uint32 pg_atomic_fetch_sub_u32_impl(volatile pg_atomic_uint32 *ptr, int32 sub_)

入力と出力

  • **入力**:
  • ptr: 操作対象となる pg_atomic_uint32 構造体へのポインタ（volatile 修飾あり）。
  • sub_: 減算する値（int32）。
  • **出力**:
  • 減算操作が行われる直前の ptr->value の値（uint32）。

前提条件と不変条件

  • 本関数は atomics.h を経由して呼び出されることを前提としています。
  • ptr は有効な pg_atomic_uint32 構造体を指している必要があります。

アルゴリズムまたは振る舞い

GCCの組み込み関数 __sync_fetch_and_sub を直接呼び出し、ptr->value に対してアトミックな減算を実行します。この操作はインライン展開されます。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/include/port/atomics.h:385
  • src/include/port/atomics/generic.h:186
  • src/include/port/atomics/generic.h:232
  • **呼び出し先**:
  • コンパイラ組み込み関数 __sync_fetch_and_sub

関連データ構造

  • pg_atomic_uint32: 原子操作対象の値を保持する構造体。

失敗モードと境界条件

  • 減算によるアンダーフローが発生した場合の挙動は、符号なし整数としてのラップアラウンド規則に従います。
  • 本実装は __sync 系組み込み関数に依存しているため、コンパイラがサポートしていない環境ではリンクエラーやコンパイルエラーとなる可能性があります。

保守担当者向けメモ

  • 本関数は generic-gcc.h 内で定義されており、より優先度の高いアーキテクチャ固有の実装（arch-*.h）が存在しない場合に利用されます。
  • __atomic 系組み込み関数が利用可能な環境であっても、本実装は __sync 系を使用しているため、メモリモデルの厳密な制御が必要な場合は __atomic への移行を検討してください。
