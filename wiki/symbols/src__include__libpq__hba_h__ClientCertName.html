ClientCertName

定義

  • 種別: enum
  • 定義箇所: src/include/libpq/hba.h:75

目的

クライアント証明書認証において、証明書のどのフィールドをユーザー名との照合に使用するかを指定するための列挙型です。

シグネチャまたは形状

typedef enum ClientCertName
{
	clientCertCN,
	clientCertDN,
} ClientCertName;

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: clientCertCN（Common Nameを使用）または clientCertDN（Distinguished Nameを使用）のいずれかの値を保持します。

前提条件と不変条件

  • pg_hba.conf の設定解析時に、認証方式が証明書関連である場合に選択されます。

アルゴリズムまたは振る舞い

  • clientCertCN: 証明書の Common Name を認証対象として利用します。
  • clientCertDN: 証明書全体の Distinguished Name を認証対象として利用します。

呼び出し元と呼び出し先

  • 呼び出し元: src/include/libpq/hba.h:126 にて HbaLine 構造体のメンバ clientcertname として参照されています。
  • 呼び出し先: 提供されたソース情報からは、具体的な関数による呼び出し先は特定できません。

関連データ構造

  • HbaLine: pg_hba.conf の各行を保持する構造体であり、この列挙型をメンバとして含みます。

失敗モードと境界条件

  • 該当なし（列挙型定義のため）。

保守担当者向けメモ

  • 本列挙型に値を追加する場合、src/backend/libpq/hba.c 内の対応するパースロジックおよび設定値の文字列変換処理を更新する必要があります。
