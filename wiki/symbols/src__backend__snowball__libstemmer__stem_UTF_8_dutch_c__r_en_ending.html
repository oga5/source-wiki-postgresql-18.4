r_en_ending

定義

  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_dutch.c:376

目的

オランダ語のステミング処理において、単語末尾の "-en" に関連する接尾辞を削除し、必要に応じて二重子音を解消する。

シグネチャまたは形状

static int r_en_ending(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z （Snowball ステミング環境）
  • 出力: int （成功時に 1、失敗または条件不一致時に 0、エラー時に負の値を返す）

前提条件と不変条件

  • r_R1(z) が成功（R1領域内にマッチ）すること。
  • g_v（母音グループ）の直前に "-en" が存在すること。

アルゴリズムまたは振る舞い

1. r_R1 を呼び出し、現在の位置が R1 領域内であることを確認する。 2. 現在位置から後方へ向かって母音グループ g_v を検索する。 3. 母音の直後に特定の文字列（s_10）が存在しないことを確認する。 4. 条件を満たす場合、slice_del で接尾辞を削除する。 5. 最後に r_undouble を呼び出し、削除後の単語末尾で二重子音の解消を行う。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_dutch.c および src/backend/snowball/libstemmer/stem_ISO_8859_1_dutch.c 内のステミングロジックから呼び出される。
  • 呼び出し先: r_R1, out_grouping_b_U, eq_s_b, slice_del, r_undouble。

関連データ構造

  • struct SN_env: ステミングの実行状態を保持する構造体。
  • g_v: 母音を定義するグループ。
  • s_10: 比較対象となる文字列定数。

失敗モードと境界条件

  • r_R1 が失敗した場合、即座にその戻り値を返す。
  • 母音が見つからない、または s_10 がマッチした場合は処理を中断し 0 を返す。
  • slice_del または r_undouble がエラーを返した場合は、その値を伝播させる。
