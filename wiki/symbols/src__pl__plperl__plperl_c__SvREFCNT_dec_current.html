SvREFCNT_dec_current

定義

  • 種別: static inline function
  • 定義箇所: src/pl/plperl/plperl.c:313

目的

Perl インタプリタのコンテキスト（dTHX）を明示的に取得した上で、Perl のスカラー値（SV）の参照カウントをデクリメントするラッパー関数です。

シグネチャまたは形状

static inline void SvREFCNT_dec_current(SV *sv)

入力と出力

  • **入力**: SV *sv（参照カウントを減らす対象の Perl スカラー値）
  • **出力**: なし（void）

前提条件と不変条件

  • 本関数を呼び出す前に、Perl インタプリタが適切に初期化され、アクティブな状態である必要があります。
  • sv が NULL である場合、Perl の SvREFCNT_dec マクロの仕様に従い、安全に処理されることが期待されます。

アルゴリズムまたは振る舞い

1. dTHX マクロを実行し、現在のスレッドの Perl インタプリタコンテキストを取得します。 2. SvREFCNT_dec(sv) を呼び出し、対象の SV の参照カウントを減らします。参照カウントがゼロに達した場合、Perl のメモリ管理機構によって当該オブジェクトが解放されます。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/pl/plperl/plperl.c 内の各所（関数実行後の戻り値やキャッシュされた記述子のクリーンアップ時など）。
  • 具体的な呼び出し箇所: 1963行目、1971行目、2514行目、2624行目、2626行目、2664行目、2707行目。
  • **呼び出し先**:
  • SvREFCNT_dec（Perl API マクロ）

関連データ構造

  • SV (Perl Scalar Value): Perl のデータ型を表現する構造体。

失敗モードと境界条件

  • Perl インタプリタがアクティブでない状態で呼び出された場合、dTHX によるコンテキスト取得が失敗し、セグメンテーションフォールト等の未定義動作を引き起こす可能性があります。

保守担当者向けメモ

  • 本関数は、PL/Perl 内で Perl API を直接呼び出す際のコンテキスト管理を簡略化するために定義されています。
  • Perl の SvREFCNT_dec は NULL を許容しますが、呼び出し元で NULL チェックが必要なケースがあるか注意してください。
