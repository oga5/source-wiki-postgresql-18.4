pg_saslprep

定義

  • 種別: function
  • 定義箇所: src/common/saslprep.c:1046

目的

RFC 4013 (SASLprep) に基づき、パスワード文字列の正規化および検証を行う。SCRAM認証の前処理として、文字列の互換性を確保する。

シグネチャまたは形状

pg_saslprep_rc pg_saslprep(const char *input, char **output)

入力と出力

  • **入力**: input (ヌル終端のUTF-8文字列)
  • **出力**: output (正規化されたUTF-8文字列へのポインタ。失敗時は NULL)
  • **戻り値**: pg_saslprep_rc 型のステータスコード（成功、メモリ不足、不正なUTF-8、禁止文字検出など）

前提条件と不変条件

  • 入力が純粋なASCII文字列である場合、正規化処理をスキップして複製を返す。
  • 処理後の文字列が空になることは許容されない。
  • メモリ確保失敗時は、*output を NULL に設定した上でエラーを返す。

アルゴリズムまたは振る舞い

1. **UTF-8変換**: 入力文字列をUnicodeコードポイントの配列に変換し、妥当性を検証する。 2. **マッピング**: non_ascii_space_ranges に該当する文字をスペース（0x0020）に置換し、commonly_mapped_to_nothing_ranges に該当する文字を削除する。 3. **正規化**: unicode_normalize を使用してNFKC形式へ変換する。 4. **禁止文字チェック**: prohibited_output_ranges および unassigned_codepoint_ranges に含まれる文字を検出する。 5. **双方向（Bidi）チェック**: RandALCat カテゴリの文字が含まれる場合、以下の制約を検証する。

  • LCat カテゴリの文字を含まないこと。
  • 先頭と末尾が RandALCat であること。

    • 6. **再エンコード**: 処理済みのコードポイントをUTF-8文字列に再変換して返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/libpq/auth-scram.c
  • src/interfaces/libpq/fe-auth-scram.c
  • **呼び出し先**:
  • pg_utf8_string_len, utf8_to_unicode, unicode_normalize, unicode_to_utf8, pg_utf_mblen, IS_CODE_IN_TABLE (マクロ)

関連データ構造

  • pg_saslprep_rc: 処理結果を示す列挙型。
  • pg_wchar: Unicodeコードポイントを保持する型。
  • 禁止・カテゴリテーブル: prohibited_output_ranges, unassigned_codepoint_ranges, RandALCat_codepoint_ranges, LCat_codepoint_ranges 等。

失敗モードと境界条件

  • **メモリ不足**: ALLOC 失敗時に oom ラベルへ遷移。
  • **不正なUTF-8**: pg_utf8_string_len が負の値を返した場合に SASLPREP_INVALID_UTF8 を返す。
  • **禁止文字・空文字**: 禁止文字の検出時や、マッピング後に文字列が空になった場合は prohibited ラベルへ遷移しエラーを返す。

保守担当者向けメモ

  • 本関数はフロントエンドとバックエンドの両方で利用されるため、メモリ確保関数（ALLOC/STRDUP）は環境に応じて抽象化されている。
  • 禁止文字やカテゴリの定義はRFC 3454に準拠しており、変更時は関連するテーブル定義との整合性に注意すること。
