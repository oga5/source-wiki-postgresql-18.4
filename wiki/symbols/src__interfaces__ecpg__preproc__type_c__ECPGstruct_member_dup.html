ECPGstruct_member_dup

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/preproc/type.c:12

目的

既存の ECPGstruct_member リストを複製し、新しいリストとして返します。構造体メンバーの型情報（ECPGtype）を再帰的にコピーすることで、独立した型定義のツリーを生成します。

シグネチャまたは形状

struct ECPGstruct_member *ECPGstruct_member_dup(struct ECPGstruct_member *rm)

入力と出力

  • **入力**: rm (複製対象の ECPGstruct_member リストの先頭ポインタ)
  • **出力**: 複製された ECPGstruct_member リストの先頭ポインタ。入力が NULL の場合は NULL を返します。

前提条件と不変条件

  • 入力 rm は有効な ECPGstruct_member リスト、または NULL である必要があります。
  • 複製処理において、各メンバーの型情報もメモリ上に新しく確保されます。

アルゴリズムまたは振る舞い

1. new ポインタを NULL で初期化します。 2. rm が指すリストを末尾まで走査します。 3. 各メンバーの type->type に基づき、以下のいずれかの関数を呼び出して型情報を複製します。

  • ECPGt_struct / ECPGt_union: ECPGmake_struct_type を使用。
  • ECPGt_array: 要素が構造体/共用体の場合は ECPGmake_struct_type を、それ以外は ECPGmake_array_type と ECPGmake_simple_type を組み合わせて使用。
  • その他: ECPGmake_simple_type を使用。

    • 4. 複製した型情報とメンバー名を用いて ECPGmake_struct_member を呼び出し、new リストに追加します。 5. リストの全要素を処理した後、new を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/interfaces/ecpg/preproc/type.c 内の ECPGmake_struct_type 等から呼び出されます（ソース内の直接的な参照として確認）。
  • **呼び出し先**:
  • ECPGmake_struct_type
  • ECPGmake_array_type
  • ECPGmake_simple_type
  • ECPGmake_struct_member

関連データ構造

  • ECPGstruct_member: 構造体のメンバー情報を保持するリスト構造。
  • ECPGtype: 変数の型情報を保持する構造体。

失敗モードと境界条件

  • 入力が NULL の場合、ループに入らず NULL を返します。
  • 構造体や配列がネストしている場合、再帰的な型構築が行われます。メモリ不足時の挙動は ECPGmake_* 関数群のメモリ確保ロジック（mm_alloc 等）に依存します。

保守担当者向けメモ

  • 本関数は型定義の深いコピーを行うため、元の型定義を変更しても複製されたリストには影響しません。
  • ECPGmake_struct_member はリストの先頭に要素を追加する挙動をとるため、結果としてリストの順序が逆転する可能性があります。必要に応じて呼び出し側で順序を確認してください。
