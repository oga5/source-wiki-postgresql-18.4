injection_stats_fixed_reset_all_cb

定義

  • 種別: function
  • 定義箇所: src/test/modules/injection_points/injection_stats_fixed.c:81

目的

固定数統計（fixed-numbered statistics）の累積値をリセットし、リセット時点のタイムスタンプを記録する。

シグネチャまたは形状

static void injection_stats_fixed_reset_all_cb(TimestampTz ts)

入力と出力

  • **入力**: ts (TimestampTz型) - 統計リセット時刻。
  • **出力**: なし。共有メモリ上の PgStatShared_InjectionPointFixed 構造体を直接更新する。

前提条件と不変条件

  • PGSTAT_KIND_INJECTION_FIXED に対応する共有メモリデータが初期化済みであること。
  • 共有メモリ上の stats_shmem->lock を排他制御（LW_EXCLUSIVE）で取得できること。

アルゴリズムまたは振る舞い

1. pgstat_get_custom_shmem_data を使用して、共有メモリ上の統計データ構造体を取得する。 2. LWLockAcquire により、統計データへの排他ロックを取得する。 3. pgstat_copy_changecounted_stats を呼び出し、現在の統計値（stats）をリセットオフセット（reset_offset）へコピーする。この際、changecount を用いて整合性を保証する。 4. 引数 ts を stat_reset_timestamp に代入し、リセット時刻を更新する。 5. LWLockRelease によりロックを解放する。

呼び出し元と呼び出し先

  • **呼び出し元**: pgstat フレームワークの統計リセット処理からコールバックとして呼び出される（injection_stats_fixed 構造体で定義）。
  • **呼び出し先**:
  • pgstat_get_custom_shmem_data
  • LWLockAcquire / LWLockRelease
  • pgstat_copy_changecounted_stats

関連データ構造

  • PgStatShared_InjectionPointFixed: 統計カウンタ、ロック、リセットオフセットを保持する共有メモリ構造体。

失敗モードと境界条件

  • 共有メモリが未初期化の場合、pgstat_get_custom_shmem_data が NULL を返す可能性がある（ただし、本モジュールの初期化フロー上では通常発生しない）。
  • ロック取得時に競合が発生した場合、待機状態となる。

保守担当者向けメモ

  • 本関数は pgstat フレームワークのコールバックとして登録されているため、直接呼び出すのではなく、統計リセットのインターフェース経由で実行されることを想定している。
  • 統計の更新には LWLock を使用しているため、高頻度なリセット操作はパフォーマンスに影響を与える可能性がある。
