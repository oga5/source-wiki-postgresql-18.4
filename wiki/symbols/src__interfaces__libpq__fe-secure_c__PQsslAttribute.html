PQsslAttribute

定義

  • 種別: function
  • 定義箇所: src/interfaces/libpq/fe-secure.c:439

目的

現在の接続に関連付けられたSSL/TLS属性情報を取得するためのインターフェースを提供します。

シグネチャまたは形状

const char *PQsslAttribute(PGconn *conn, const char *attribute_name);

入力と出力

  • **入力**:
  • conn: 接続状態を保持する PGconn 構造体へのポインタ。
  • attribute_name: 取得したい属性名（例: "library", "protocol", "key_bits", "cipher", "compression", "alpn"）。
  • **出力**:
  • 指定された属性値を示す文字列ポインタを返します。提供されたソースコードの定義上は常に NULL を返します。

前提条件と不変条件

  • 本関数は fe-secure.c 内で定義されていますが、実際の属性取得ロジックは src/interfaces/libpq/fe-secure-openssl.c に実装が存在することが示唆されています。

アルゴリズムまたは振る舞い

  • 提供されたソースコードの定義では、引数に関わらず即座に NULL を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/bin/psql/command.c 内の複数の箇所で、接続のSSL詳細情報を表示するために呼び出されています。
  • **呼び出し先**: 提供されたソースコードの範囲内では、他の関数を呼び出していません。

関連データ構造

  • PGconn: 接続状態を管理する主要な構造体。

失敗モードと境界条件

  • 属性が見つからない場合や、SSLが使用されていない場合、あるいは本実装のように未定義の状態では NULL を返します。
