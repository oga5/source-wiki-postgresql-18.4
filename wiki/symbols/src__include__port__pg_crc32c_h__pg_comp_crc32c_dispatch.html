pg_comp_crc32c_dispatch

定義

  • 種別: function
  • 定義箇所: src/include/port/pg_crc32c.h:69

目的

CRC-32C チェックサム計算において、入力データのサイズやコンパイル時の最適化条件に基づき、インライン展開による効率的な計算か、実行時のハードウェア判定を伴う関数呼び出しを選択する。

シグネチャまたは形状

static inline pg_crc32c pg_comp_crc32c_dispatch(pg_crc32c crc, const void *data, size_t len)

入力と出力

  • **入力**:
  • crc: 現在のチェックサム累積値。
  • data: 計算対象のデータバッファへのポインタ。
  • len: データ長（バイト単位）。
  • **出力**: 計算後の新しいチェックサム値（pg_crc32c 型）。

前提条件と不変条件

  • HAVE__BUILTIN_CONSTANT_P が定義されている環境では、len が定数かつ 32 バイト未満の場合にインライン計算が実行される。
  • 64 ビット環境（SIZEOF_VOID_P >= 8）では、8 バイト単位の処理に _mm_crc32_u64 が使用される。

アルゴリズムまたは振る舞い

1. **定数最適化パス**: コンパイラが len を定数と認識し、かつ len < 32 の場合、関数呼び出しのオーバーヘッドを避けるため、以下の順序でインライン展開された SSE 4.2 命令を実行する。

  • 8 バイト単位（64bit 環境のみ）: _mm_crc32_u64
  • 4 バイト単位: _mm_crc32_u32
  • 1 バイト単位: _mm_crc32_u8

    • 2. **汎用パス**: 上記条件を満たさない場合、または HAVE__BUILTIN_CONSTANT_P が未定義の場合は、pg_comp_crc32c を呼び出し、実行時のハードウェア判定（AVX-512 等の利用可否）に委ねる。

呼び出し元と呼び出し先

  • **呼び出し元**: src/include/port/pg_crc32c.h 内の COMP_CRC32C マクロから呼び出される。
  • **呼び出し先**:
  • _mm_crc32_u64, _mm_crc32_u32, _mm_crc32_u8（SSE 4.2 命令）
  • pg_comp_crc32c（実行時判定用関数）

関連データ構造

  • pg_crc32c: チェックサム値を保持する uint32 型のエイリアス。

失敗モードと境界条件

  • len が 0 の場合、計算を行わず入力された crc をそのまま返す。
  • 実行環境が SSE 4.2 命令をサポートしていない場合、pg_comp_crc32c への委譲先で適切なフォールバック（Slicing-by-8 等）が行われる必要がある。

保守担当者向けメモ

  • 本関数は static inline で定義されており、ヘッダーファイル内で完結する。
  • パフォーマンス最適化のため、__builtin_constant_p を活用している。修正時はコンパイル後のアセンブリコードを確認し、意図したインライン展開が行われているか検証すること。
