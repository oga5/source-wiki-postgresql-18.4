toast_decompress_datum

定義

  • 種別: function
  • 定義箇所: src/backend/access/common/detoast.c:470

目的

圧縮された varlena 形式のデータを、圧縮メソッドに応じて適切なアルゴリズムで展開し、復元されたデータへのポインタを返します。

シグネチャまたは形状

static struct varlena *toast_decompress_datum(struct varlena *attr);

入力と出力

  • **入力**: attr - 圧縮されたデータを含む varlena 構造体へのポインタ。
  • **出力**: 展開されたデータを含む varlena 構造体へのポインタ。

前提条件と不変条件

  • 入力 attr は VARATT_IS_COMPRESSED(attr) が真であること（アサーションにより強制）。
  • 圧縮メソッド ID が TOAST_PGLZ_COMPRESSION_ID または TOAST_LZ4_COMPRESSION_ID のいずれかであること。

アルゴリズムまたは振る舞い

1. 入力データの圧縮ヘッダーから ToastCompressionId を取得します。 2. switch 文により圧縮メソッドを判定します。

  • TOAST_PGLZ_COMPRESSION_ID の場合: pglz_decompress_datum(attr) を呼び出します。
  • TOAST_LZ4_COMPRESSION_ID の場合: lz4_decompress_datum(attr) を呼び出します。

    • 3. 判定されたメソッドに基づいて展開処理を実行し、その結果を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/access/common/detoast.c 内の各所（detoast_attr 等の復元処理フローから呼び出されます）。
  • **呼び出し先**: pglz_decompress_datum または lz4_decompress_datum。

関連データ構造

  • varlena: PostgreSQLの可変長データ型を表現する構造体。
  • ToastCompressionId: 圧縮アルゴリズムを識別するための列挙型。

失敗モードと境界条件

  • 未知の圧縮メソッド ID が指定された場合、elog(ERROR) を発行し、処理を中断します。

保守担当者向けメモ

  • 本関数は内部関数であり、呼び出し元でメモリの解放（pfree）を行う必要があります。
  • 新しい圧縮アルゴリズムを追加する場合は、switch 文に新しい case を追加してください。
