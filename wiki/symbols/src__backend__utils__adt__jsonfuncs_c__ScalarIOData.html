ScalarIOData

定義

  • 種別: struct
  • 定義箇所: src/backend/utils/adt/jsonfuncs.c:155

目的

JSONデータからPostgreSQLのスカラ型へ変換する際に必要な、型固有の入出力メタデータを保持するためのキャッシュ構造体です。

シグネチャまたは形状

typedef struct ScalarIOData
{
	Oid			typioparam;
	FmgrInfo	typiofunc;
} ScalarIOData;

入力と出力

  • **入力**: typioparam（型固有のパラメータOid）、typiofunc（入出力関数を呼び出すためのFmgrInfo）。
  • **出力**: なし（構造体定義）。

前提条件と不変条件

  • typiofunc は、対象となるPostgreSQL型に対応する入力関数を呼び出すために初期化されている必要があります。

アルゴリズムまたは振る舞い

  • JSON/JSONBからPostgreSQLの型へ値を変換する際、変換対象の型情報をキャッシュし、populate_scalar 関数内で効率的に型変換関数を呼び出すために使用されます。

呼び出し元と呼び出し先

  • **呼び出し元**: populate_scalar 関数（src/backend/utils/adt/jsonfuncs.c:451, 3125）がこの構造体を引数として受け取り、変換処理を実行します。
  • **呼び出し先**: 構造体内部の typiofunc を介して、PostgreSQLの型変換関数が呼び出されます。

関連データ構造

  • FmgrInfo: 関数マネージャが提供する関数呼び出し情報。
  • ColumnIOData: 関連するデータ構造として定義されています。

失敗モードと境界条件

  • 変換対象の型に対して適切な入力関数が見つからない場合、typiofunc の初期化に失敗する可能性があります。

保守担当者向けメモ

  • 本構造体は populate_scalar における変換のオーバーヘッドを削減するためのキャッシュとして機能します。変更を加える際は、populate_scalar での利用箇所と整合性を確認してください。
