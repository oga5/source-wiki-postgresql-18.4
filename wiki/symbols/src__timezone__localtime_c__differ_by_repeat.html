differ_by_repeat

定義

  • 種別: function
  • 定義箇所: src/timezone/localtime.c:169

目的

2つの時刻値（pg_time_t）が、タイムゾーンの繰り返し周期（SECSPERREPEAT）分だけ離れているかを判定する。

シグネチャまたは形状

static bool differ_by_repeat(const pg_time_t t1, const pg_time_t t0)

入力と出力

  • **入力**:
  • t1: 比較対象の時刻値。
  • t0: 基準となる時刻値。
  • **出力**:
  • bool: t1 - t0 が SECSPERREPEAT と等しい場合に true を返し、それ以外は false を返す。

前提条件と不変条件

  • pg_time_t 型のビット幅が SECSPERREPEAT_BITS 未満である場合、オーバーフローの可能性があるため常に false を返す。

アルゴリズムまたは振る舞い

1. pg_time_t 型のビット数から符号ビット分を差し引いた値が SECSPERREPEAT_BITS より小さい場合、計算を行わずに 0（false）を返す。 2. 上記条件を満たす場合、t1 と t0 の差分を計算し、その値が SECSPERREPEAT と一致するかを評価して結果を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/timezone/localtime.c:500
  • src/timezone/localtime.c:508
  • **呼び出し先**:
  • 情報なし。

関連データ構造

  • pg_time_t: 時刻を保持する型。

失敗モードと境界条件

  • pg_time_t の型定義が SECSPERREPEAT を表現するのに十分なビット幅を持たない場合、誤った結果を防ぐために安全策として false を返す。

保守担当者向けメモ

  • 本関数は localtime.c 内の遷移時刻（ats）の検証に使用される。
  • SECSPERREPEAT は private.h 等で定義されている定数であり、タイムゾーンデータの周期性を考慮した計算を行う。
