filter_object_type_name

定義

  • 種別: function
  • 定義箇所: src/bin/pg_dump/filter.c:81

目的

FilterObjectType 列挙型の値を、人間が読み取り可能な文字列形式に変換して返します。

シグネチャまたは形状

const char *filter_object_type_name(FilterObjectType fot);

入力と出力

  • **入力**: FilterObjectType fot（変換対象のオブジェクト型列挙値）
  • **出力**: const char *（オブジェクト型に対応する説明文字列）

前提条件と不変条件

  • 入力 fot は FilterObjectType 列挙型で定義されたいずれかの値である必要があります。
  • 戻り値は静的な文字列リテラルであり、呼び出し側で解放する必要はありません。

アルゴリズムまたは振る舞い

switch 文を使用して FilterObjectType の各列挙値を評価し、対応する文字列を返します。定義されていない値が渡された場合は pg_unreachable() を呼び出し、プログラムの実行を停止します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/bin/pg_dump/pg_dump.c
  • src/bin/pg_dump/pg_dumpall.c
  • src/bin/pg_dump/pg_restore.c
  • **呼び出し先**:
  • pg_unreachable()（定義外の値が渡された場合）

関連データ構造

  • FilterObjectType（src/bin/pg_dump/filter.h で定義）

失敗モードと境界条件

  • 定義されていない FilterObjectType 値が渡された場合、pg_unreachable() によりアボートします。

保守担当者向けメモ

  • 新しいオブジェクト型を FilterObjectType に追加した場合は、必ず本関数の switch 文にも対応するケースを追加してください。
