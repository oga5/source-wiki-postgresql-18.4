pg_logging_set_pre_callback

定義

  • 種別: function
  • 定義箇所: src/common/logging.c:195

目的

ログ出力処理の直前に実行されるコールバック関数を登録する。

シグネチャまたは形状

void pg_logging_set_pre_callback(void (*cb) (void))

入力と出力

  • 入力: cb - ログ出力直前に呼び出される関数ポインタ。
  • 出力: なし。

前提条件と不変条件

  • 登録されたコールバックは、ログ出力処理（pg_log_generic_v 等）の内部で実行される。

アルゴリズムまたは振る舞い

1. 引数として渡された関数ポインタ cb を、モジュール内の静的変数 log_pre_callback に代入する。

呼び出し元と呼び出し先

  • 呼び出し元: src/bin/psql/startup.c:134 からの呼び出しが確認されている。
  • 呼び出し先: 登録されたコールバック関数（実行時に決定される）。

関連データ構造

  • log_pre_callback: 本関数によって設定される静的関数ポインタ変数。

失敗モードと境界条件

  • 特記事項なし。

保守担当者向けメモ

  • 本関数は src/include/common/logging.h で公開されている。
  • ログ出力のフック機構の一部であり、主にフロントエンドプログラムでのログ整形や状態管理に使用される。
