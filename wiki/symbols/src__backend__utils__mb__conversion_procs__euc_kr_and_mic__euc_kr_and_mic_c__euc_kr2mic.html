euc_kr2mic

定義

  • 種別: function
  • 定義箇所: src/backend/utils/mb/conversion_procs/euc_kr_and_mic/euc_kr_and_mic.c:78

目的

EUC_KR エンコーディングのバイト列を MULE_INTERNAL (MIC) 形式へ変換する内部変換ロジックを提供します。

シグネチャまたは形状

static int euc_kr2mic(const unsigned char *euc, unsigned char *p, int len, bool noError)

入力と出力

  • **入力**:
  • euc: 変換元となる EUC_KR 文字列のポインタ。
  • p: 変換後の MULE_INTERNAL 文字列を格納するバッファ。
  • len: 変換対象のバイト長。
  • noError: 変換エラー時に例外を投げず処理を中断するかを指定するフラグ。
  • **出力**:
  • 戻り値: 処理した入力バイト数。

前提条件と不変条件

  • 入力文字列は EUC_KR エンコーディングに従っている必要があります。
  • p で指定されたバッファは、変換結果を格納するために十分なサイズ（入力長の最大1.5倍程度）を確保している必要があります。

アルゴリズムまたは振る舞い

1. 入力バイト列を順次走査します。 2. **1バイト文字（ASCII）**: IS_HIGHBIT_SET が偽の場合、そのまま出力バッファへコピーします。0 バイトは不正として扱われます。 3. **2バイト文字**: IS_HIGHBIT_SET が真の場合、pg_encoding_verifymbchar を用いて EUC_KR としての妥当性を検証します。

  • 検証成功時、LC_KS5601 プレフィックスを付与し、続く2バイトをコピーして計3バイトの MULE_INTERNAL 形式に変換します。

    • 4. 変換終了後、出力バッファの末尾にヌル終端文字を付与します。 5. noError が真の場合、不正な文字や終端条件に遭遇した時点で処理を中断し、それまでに変換したバイト数を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: 同一ファイル内の euc_kr_to_mic 関数（行54）から呼び出されます。
  • **呼び出し先**:
  • pg_encoding_verifymbchar: 文字の妥当性検証に使用。
  • report_invalid_encoding: エラー発生時の通知に使用。

関連データ構造

  • LC_KS5601: MULE_INTERNAL における KS X 1001 文字セットの識別子。
  • PG_EUC_KR: エンコーディング識別子。

失敗モードと境界条件

  • len が 0 に達した時点でループを終了します。
  • 不正なマルチバイトシーケンスや 0 バイトを検出した場合、noError が偽であれば report_invalid_encoding を呼び出し、真であればその時点までの処理結果を返して終了します。

保守担当者向けメモ

  • 本関数は euc_kr_to_mic から呼び出される内部ロジックです。
  • MULE_INTERNAL 形式への変換において、LC_KS5601 プレフィックスの付与が必須である点に注意してください。
