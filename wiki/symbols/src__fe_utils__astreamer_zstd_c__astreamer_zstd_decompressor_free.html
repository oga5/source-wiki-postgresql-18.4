astreamer_zstd_decompressor_free

定義

  • 種別: function
  • 定義箇所: src/fe_utils/astreamer_zstd.c:359

目的

zstd 伸長用ストリーマーに関連付けられたリソースを解放し、メモリリークを防ぐ。

シグネチャまたは形状

static void astreamer_zstd_decompressor_free(astreamer *streamer)

入力と出力

  • **入力**: astreamer *streamer - 解放対象のストリーマー構造体へのポインタ。
  • **出力**: なし。

前提条件と不変条件

  • streamer は astreamer_zstd_frame 構造体としてキャスト可能であること。
  • streamer->bbs_next、mystreamer->dctx、および streamer->bbs_buffer.data は、それぞれ適切に初期化・割り当て済みであること。

アルゴリズムまたは振る舞い

1. astreamer_free(streamer->bbs_next) を呼び出し、パイプラインの後続ストリーマーを再帰的に解放する。 2. ZSTD_freeDCtx(mystreamer->dctx) を呼び出し、zstd 伸長コンテキストを破棄する。 3. pfree(streamer->bbs_buffer.data) を呼び出し、内部バッファを解放する。 4. pfree(streamer) を呼び出し、ストリーマー構造体自体を解放する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/fe_utils/astreamer_zstd.c 内のストリーマー定義構造体（.free メンバ）から参照されている。
  • **呼び出し先**:
  • astreamer_free (後続ストリーマーの解放)
  • ZSTD_freeDCtx (zstd ライブラリ関数)
  • pfree (メモリ解放)

関連データ構造

  • astreamer: ストリーマーの基底構造体。
  • astreamer_zstd_frame: 伸長コンテキストとバッファを保持する内部構造体。

失敗モードと境界条件

  • streamer が NULL の場合、pfree 等でセグメンテーション違反が発生する可能性がある。
  • 内部ポインタが既に解放済み（二重解放）である場合、未定義動作を引き起こす。

保守担当者向けメモ

  • 本関数は astreamer インフラストラクチャの解放インターフェースの一部です。新しいメンバを astreamer_zstd_frame に追加した場合は、必ず本関数で解放処理を追加してください。
