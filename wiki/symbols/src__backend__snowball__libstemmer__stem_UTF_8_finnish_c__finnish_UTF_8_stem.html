finnish_UTF_8_stem

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_finnish.c:662

目的

フィンランド語の単語に対し、接尾辞の削除および語形変化の正規化を行い、語幹を抽出する。

シグネチャまたは形状

int finnish_UTF_8_stem(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z （Snowball 実行環境。処理対象の文字列や領域情報を含む）
  • 出力: int （成功時に 1、失敗時に負の値を返す）

前提条件と不変条件

  • z は有効な SN_env 構造体であること。
  • 内部的に z->I[0]（R2領域）および z->I[1]（R1領域）が r_mark_regions によって適切に設定されること。
  • 処理中、z->c（カーソル位置）および z->l（文字列長）が適宜操作され、処理終了時には元の状態に復元される。

アルゴリズムまたは振る舞い

1. **領域特定**: r_mark_regions を呼び出し、単語内のステミング境界を定義する。 2. **順次適用**: 以下の順序でルールを適用し、各ステップでカーソル位置を保持・復元する。

  • 粒子（r_particle_etc）
  • 所有接尾辞（r_possessive）
  • 格変化（r_case_ending）
  • その他の語尾（r_other_endings）

    • 3. **複数形処理**: 格変化の適用有無（z->I[2]）に基づき、r_i_plural または r_t_plural のいずれかを選択的に実行する。 4. **後処理**: r_tidy を呼び出し、不要な文字や二重母音の調整を行う。 5. **終了**: カーソルを初期位置に戻し、成功を返す。

呼び出し元と呼び出し先

  • 呼び出し元: PostgreSQL の全文検索エンジン（詳細は提供されたソース内では明示されていない）。
  • 呼び出し先: r_mark_regions, r_particle_etc, r_possessive, r_case_ending, r_other_endings, r_i_plural, r_t_plural, r_tidy。

関連データ構造

  • struct SN_env: Snowball ランタイム環境。I[] 配列（整数インデックス）、c（カーソル）、l（文字列長）などを保持。

失敗モードと境界条件

  • 呼び出し先の各関数が負の値を返した場合、即座にその値を返して終了する。
  • z->I[2] のフラグ状態により、複数形処理の分岐が決定される。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードであるため、直接編集せず、Snowball 言語のソースファイルを修正して再生成すること。
  • z->I[2] は格変化の適用状況を追跡する内部フラグとして使用されている。
