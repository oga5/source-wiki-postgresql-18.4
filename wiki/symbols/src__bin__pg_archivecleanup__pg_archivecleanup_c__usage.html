usage

定義

  • 種別: function
  • 定義箇所: src/bin/pg_archivecleanup/pg_archivecleanup.c:257

目的

pg_archivecleanup ユーティリティのコマンドライン引数、使用方法、および postgresql.conf における archive_cleanup_command への設定例を標準出力に表示する。

シグネチャまたは形状

static void usage(void)

入力と出力

  • 入力: なし（グローバル変数 progname を参照）
  • 出力: なし（標準出力へヘルプメッセージを出力）

前提条件と不変条件

  • グローバル変数 progname が適切に初期化されていること。
  • PACKAGE_BUGREPORT、PACKAGE_NAME、PACKAGE_URL がビルド時に定義されていること。

アルゴリズムまたは振る舞い

1. ユーティリティの概要と基本的な使用構文を表示する。 2. 利用可能なオプション（-b, -d, -n, -V, -x, -?）の一覧と説明を表示する。 3. postgresql.conf での archive_cleanup_command としての推奨設定例を表示する。 4. スタンドアロンツールとしての実行例を表示する。 5. バグ報告先およびプロジェクトのホームページ情報を表示する。

呼び出し元と呼び出し先

  • 呼び出し元: 提供されたソースコード内では、main 関数から呼び出されることが想定される（ソースコードの文脈に基づく推論）。
  • 呼び出し先: printf（標準ライブラリ）。
  • 補足: ツールが提示した contrib/ 配下の各ファイルにおける usage への言及は、本シンボルとは無関係な用語としての「使用法（usage）」を指しており、本関数への呼び出しではない。

関連データ構造

  • なし

失敗モードと境界条件

  • 標準出力への書き込みが失敗した場合の挙動は printf の仕様に依存する。

保守担当者向けメモ

  • オプションを追加した際は、必ず本関数内のヘルプメッセージを更新すること。
  • postgresql.conf の設定例は、ユーザーがコピー＆ペーストして利用することを想定しているため、正確性を維持すること。
