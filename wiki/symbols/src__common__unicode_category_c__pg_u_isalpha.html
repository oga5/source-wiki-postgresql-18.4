pg_u_isalpha

定義

  • 種別: function
  • 定義箇所: src/common/unicode_category.c:219

目的

指定された Unicode コードポイントがアルファベット（Alphabetic）プロパティを持つかどうかを判定します。

シグネチャまたは形状

bool pg_u_isalpha(pg_wchar code)

入力と出力

  • **入力**: pg_wchar code（判定対象の Unicode コードポイント）
  • **出力**: bool（アルファベットであれば true、そうでなければ false を返す）

前提条件と不変条件

  • 入力されるコードポイントは UTF-8 エンコーディングを前提としています。
  • Unicode 規格の安定性ポリシーに基づき、判定基準となるカテゴリ値は固定されています。

アルゴリズムまたは振る舞い

内部的に pg_u_prop_alphabetic(code) を呼び出し、その結果をそのまま返します。

呼び出し元と呼び出し先

  • **呼び出し先**: pg_u_prop_alphabetic
  • **呼び出し元**:
  • src/backend/regex/regc_pg_locale.c
  • src/common/unicode/category_test.c
  • src/common/unicode_category.c（内部の他の判定関数から参照）

関連データ構造

  • pg_wchar: ワイド文字型。
  • src/common/unicode_category_table.h: 判定に使用される静的データテーブル（pg_u_prop_alphabetic 経由で間接的に参照）。

失敗モードと境界条件

  • 0x80 未満の ASCII 文字については、unicode_opt_ascii テーブルを参照して定数時間で判定されます。
  • それ以上のコードポイントについては、unicode_category_table.h のデータに基づき二分探索が行われます。

保守担当者向けメモ

  • 本関数は pg_u_prop_alphabetic のラッパーです。ロジックの変更が必要な場合は、unicode_category_table.h の生成スクリプトを確認してください。
