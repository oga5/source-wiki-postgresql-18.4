r_remove_suffix

定義

目的

インドネシア語のステミング処理において、特定の接尾辞を単語末尾から除去する。

シグネチャまたは形状

static int r_remove_suffix(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（ステミング環境構造体）。現在の単語データおよびワークスペースを含む。
  • **出力**: int 型。成功した場合は 1、処理が適用されなかった場合や失敗した場合は 0 または負の値を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体を指していること。
  • a_2 配列に定義された接尾辞パターンが検索対象となる。
  • 処理の副作用として、z->I[1]（単語内の母音数を示す内部フラグ）がデクリメントされる。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を z->ket に設定する。 2. 単語の末尾が 'i' (105) または 'n' (110) で終わっているかを確認する。条件を満たさない場合は 0 を返す。 3. find_among_b を使用し、a_2 配列内のパターンと後方一致検索を行う。一致しない場合は 0 を返す。 4. 一致した接尾辞の開始位置を z->bra に設定する。 5. slice_del(z) を呼び出し、該当する接尾辞を単語から削除する。削除に失敗した場合はそのエラーコードを返す。 6. 内部フラグ z->I[1] を 1 減算し、成功を示す 1 を返す。

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_2: 接尾辞パターンを定義した among 構造体の配列。

失敗モードと境界条件

  • 単語が空、またはカーソル位置が境界（z->lb）にある場合は処理を中断し 0 を返す。
  • slice_del が負の値を返した場合、そのエラー状態を伝播させる。
