BoolGetDatum

定義

  • 種別: static inline 関数
  • 定義箇所: src/include/postgres.h:106

目的

C 言語の bool 型の値を、PostgreSQL の汎用データ表現である Datum 型に変換する。

シグネチャまたは形状

static inline Datum BoolGetDatum(bool X)

入力と出力

  • **入力**: bool X （真偽値）
  • **出力**: Datum 型にキャストされた値（true の場合は 1、false の場合は 0）

前提条件と不変条件

  • Datum 型は sizeof(void *) と同じサイズであると仮定されている。
  • 本関数はインライン化されており、関数呼び出しのオーバーヘッドを最小限に抑える設計となっている。

アルゴリズムまたは振る舞い

引数 X を評価し、真であれば 1 を、偽であれば 0 を Datum 型にキャストして返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/btree_gin/btree_gin.c
  • contrib/btree_gist/btree_utils_num.c
  • contrib/pageinspect/brinfuncs.c
  • contrib/pageinspect/btreefuncs.c
  • contrib/pageinspect/gistfuncs.c
  • contrib/pageinspect/hashfuncs.c
  • **呼び出し先**:
  • 本関数は単一の式で構成されており、他の関数を呼び出さない。

関連データ構造

  • Datum: PostgreSQL の実行エンジン内でデータを効率的に受け渡すための汎用型。

失敗モードと境界条件

  • 特筆すべき失敗モードはない。入力された bool 値は常に 1 または 0 の Datum に変換される。

保守担当者向けメモ

  • 本関数は postgres.h で定義されており、バックエンドのソースコード全体で利用される。
  • 移植性を維持するため、bool 型から Datum への変換には必ず本関数を使用すること。直接的なキャストは避けるべきである。
