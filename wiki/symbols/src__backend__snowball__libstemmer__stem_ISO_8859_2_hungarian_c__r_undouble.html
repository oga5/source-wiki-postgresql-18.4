r_undouble

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_2_hungarian.c:550

目的

単語の末尾付近にある重複文字を削除し、ステミング処理における正規化を行う。

シグネチャまたは形状

static int r_undouble(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング環境構造体）
  • 出力: 成功時に 1、境界条件等で処理を中断した際に 0、slice_del 実行時にエラーが発生した場合は負の値を返す。

前提条件と不変条件

  • z->c（現在のカーソル位置）が z->lb（下限境界）より大きい必要がある。
  • 処理対象の文字が少なくとも2文字分存在すること。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を1つ戻す。 2. z->ket（削除対象の末尾）を現在のカーソル位置に設定する。 3. さらにカーソルを1つ戻し、その位置を z->bra（削除対象の開始）に設定する。 4. slice_del(z) を呼び出し、bra から ket までの範囲を削除する。 5. 削除操作の結果を呼び出し元に返す。

呼び出し元と呼び出し先

  • 呼び出し元: stem_ISO_8859_2_hungarian.c および stem_ISO_8859_1_danish.c, stem_ISO_8859_1_dutch.c, stem_UTF_8_danish.c 等の各ステミング関数から参照されている。
  • 呼び出し先: slice_del（Snowball ランタイム関数）。

関連データ構造

  • struct SN_env: ステミング処理の状態（カーソル位置、境界、バッファ等）を保持する構造体。

失敗モードと境界条件

  • z->c <= z->lb の場合、処理を行わず 0 を返す。
  • z->c - 1 < z->lb の場合、削除範囲を確保できないため 0 を返す。
  • slice_del が負の値を返した場合、その値をそのまま伝播させる。
