IsVowel

定義

  • 種別: function
  • 定義箇所: contrib/fuzzystrmatch/dmetaphone.c:291

目的

指定された位置の文字が母音（A, E, I, O, U, Y）であるかを判定する。

シグネチャまたは形状

int IsVowel(metastring *s, int pos)

入力と出力

  • **入力**:
  • s: 検査対象の metastring 構造体へのポインタ。
  • pos: 検査する文字のインデックス（整数）。
  • **出力**:
  • 母音であれば 1 を返す。
  • 母音でない、またはインデックスが範囲外であれば 0 を返す。

前提条件と不変条件

  • s は有効な metastring 構造体を指している必要がある。
  • s->str は大文字に変換済みであることを前提とした判定ロジックである。

アルゴリズムまたは振る舞い

1. pos が 0 未満、または s->length 以上である場合、即座に 0 を返す（境界チェック）。 2. s->str の pos 番目の文字を取得する。 3. 取得した文字が 'A', 'E', 'I', 'O', 'U', 'Y' のいずれかと一致するかを比較する。 4. 一致すれば 1 を、それ以外は 0 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: DoubleMetaphone 関数内の各所（contrib/fuzzystrmatch/dmetaphone.c の 475, 723, 800, 903, 904, 945, 1192, 1307, 1311, 1325 行目など）。
  • **呼び出し先**: 本関数は他の関数を呼び出さない。

関連データ構造

  • metastring: 文字列バッファと現在の長さを保持する構造体。

失敗モードと境界条件

  • pos が負の値、または s->length 以上の場合、エラーにはならず安全に 0 を返す。

保守担当者向けメモ

  • 本関数は DoubleMetaphone アルゴリズムにおける発音ルールの判定基盤である。
  • 判定対象の文字はすべて大文字であることを前提としているため、入力文字列が小文字を含む場合は事前に MakeUpper 等で変換されている必要がある。
