make_icu_collator

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/pg_locale_icu.c:319

目的

指定されたロケール文字列およびカスタム照合ルールに基づき、ICUの UCollator オブジェクトを生成します。

シグネチャまたは形状

static UCollator *make_icu_collator(const char *iculocstr, const char *icurules)

入力と出力

  • **入力**:
  • iculocstr: ICUロケール識別子（例: "en_US.UTF-8"）。
  • icurules: 追加の照合ルール文字列（NULL可）。
  • **出力**:
  • 成功時: 初期化された UCollator* を返却。
  • 失敗時: ereport(ERROR, ...) によりトランザクションを中断。

前提条件と不変条件

  • icurules が指定された場合、標準ロケールのルールとカスタムルールを結合して新しい照合器を作成します。
  • メモリ割り当て失敗時は、確保済みのリソースを解放しエラーを報告します。

アルゴリズムまたは振る舞い

1. icurules がNULLの場合、pg_ucol_open を呼び出し、標準ロケールの照合器を返却します。 2. icurules が存在する場合:

  • icu_to_uchar を用いてルールをUTF-16へ変換します。
  • pg_ucol_open で標準ロケールの照合器を開き、ucol_getRules でそのルールを取得します。
  • 標準ルールとカスタムルールを連結した新しいルールセットを作成します。
  • ucol_openRules を用いて、結合されたルールから新しい UCollator を生成します。
  • 一時的なルールバッファおよび中間照合器を解放します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/adt/pg_locale_icu.c 内のロケール初期化処理（行81, 196）。
  • **呼び出し先**: pg_ucol_open, icu_to_uchar, ucol_getRules, u_strlen, u_strcpy, u_strcat, ucol_close, ucol_openRules, palloc_array_extended, pfree, ereport。

関連データ構造

  • UCollator: ICUの照合器構造体。
  • UChar: ICUのUTF-16文字型。

失敗モードと境界条件

  • **メモリ不足**: palloc_array_extended が失敗した場合、ereport(ERROR, ...) を発生させます。
  • **ルール解析エラー**: ucol_openRules が U_FAILURE を返した場合、エラーメッセージと共にトランザクションを中断します。

保守担当者向けメモ

  • 本関数は pg_locale_icu.c 内の照合順序生成の基盤です。
  • ICUのバージョン差異による挙動の変化については、pg_ucol_open 側の実装と併せて確認してください。
