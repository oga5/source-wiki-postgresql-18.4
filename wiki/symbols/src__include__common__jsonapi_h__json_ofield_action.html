json_ofield_action

定義

  • 種別: typedef
  • 定義箇所: src/include/common/jsonapi.h:128

目的

JSON オブジェクトのフィールド処理（開始または終了）時に呼び出されるコールバック関数の型定義です。

シグネチャまたは形状

typedef JsonParseErrorType (*json_ofield_action) (void *state, char *fname, bool isnull);

入力と出力

  • **入力**:
  • void *state: ユーザー定義の解析状態を保持するポインタ。
  • char *fname: 現在処理中のフィールド名。
  • bool isnull: フィールド値が JSON の null であるかどうかを示すフラグ。
  • **出力**:
  • JsonParseErrorType: 解析の継続可否を示す列挙型。成功時は JSON_SUCCESS を返します。

前提条件と不変条件

  • 本関数は JsonSemAction 構造体を通じてパーサーに登録されます。
  • 戻り値として JSON_SEM_ACTION_FAILED が返された場合、エラーの詳細はコールバック関数内で報告済みであると見なされます。

アルゴリズムまたは振る舞い

JSON 解析中にオブジェクトのフィールドに到達した際、パーサーは登録された json_ofield_action 型の関数を呼び出します。これにより、ユーザーはフィールド名や値の有無に基づいた動的なデータ抽出や検証を行うことができます。

呼び出し元と呼び出し先

  • **呼び出し元**: src/common/jsonapi.c 内の JSON 解析ロジックが、JsonSemAction に設定された関数ポインタを介して呼び出します。
  • **呼び出し先**: ユーザーが定義した、JsonSemAction に登録された具体的なコールバック関数。

関連データ構造

  • JsonSemAction: 本関数型をフィールド（object_field_start, object_field_end）として保持する構造体。
  • JsonParseErrorType: 戻り値として使用されるエラー型。

失敗モードと境界条件

  • 解析中にエラーが発生した場合、JsonParseErrorType を通じてパーサーに通知する必要があります。
  • フィールド名 fname のメモリ寿命は、パーサーの設定（setJsonLexContextOwnsTokens 等）に依存します。
