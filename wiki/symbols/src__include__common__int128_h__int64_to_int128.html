int64_to_int128

定義

  • 種別: function
  • 定義箇所: src/include/common/int128.h:83

目的

64ビット整数（int64）を128ビット整数型（INT128）へ変換します。

シグネチャまたは形状

static inline INT128 int64_to_int128(int64 v)

入力と出力

  • **入力**: int64 v（変換対象の64ビット整数）
  • **出力**: INT128（変換後の128ビット整数値）

前提条件と不変条件

  • INT128 型が定義されている環境で利用可能です。
  • 変換は型キャストによって行われます。

アルゴリズムまたは振る舞い

入力された int64 型の値を INT128 型へキャストして返します。プラットフォームがネイティブな128ビット整数をサポートしている場合はその型へ、サポートしていない場合は hi/lo 構造体へ適切に変換されます。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/timestamp.c:2559
  • src/backend/utils/adt/timestamp.c:2580
  • **呼び出し先**:
  • 本関数は単一のキャスト操作のみを行うため、他の関数を呼び出しません。

関連データ構造

  • INT128: PostgreSQLにおける128ビット整数を表現する型。

失敗モードと境界条件

  • 特筆すべき失敗モードはありません。入力値の範囲は int64 の全範囲を保持可能です。

保守担当者向けメモ

  • 本関数はインライン関数として定義されており、パフォーマンスへの影響は最小限です。
  • ソフトウェア実装（USE_NATIVE_INT128 が無効な場合）においても、キャストによる変換は INT128 型の定義に従って正しく処理されます。
