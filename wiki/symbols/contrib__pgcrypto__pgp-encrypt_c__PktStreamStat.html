PktStreamStat

定義

  • 種別: struct
  • 定義箇所: contrib/pgcrypto/pgp-encrypt.c:232

目的

PGP パケットストリームの生成過程における状態を保持するための構造体です。

シグネチャまたは形状

struct PktStreamStat
{
    int final_done;
    int pkt_block;
};

入力と出力

本構造体は関数内部でローカル変数としてインスタンス化され、フィルタ処理の状態管理に使用されます。

前提条件と不変条件

  • final_done: ストリームの終了処理が完了したかどうかを示すフラグ。
  • pkt_block: パケットのブロック状態を管理する整数値。

アルゴリズムまたは振る舞い

pgp-encrypt.c 内のフィルタ処理において、priv ポインタを通じて本構造体にアクセスし、パケットストリームの進行状況を追跡します。

呼び出し元と呼び出し先

  • 呼び出し元: contrib/pgcrypto/pgp-encrypt.c 内の各フィルタ関連関数（241行目、257行目、283行目、300行目付近で参照）。
  • 呼び出し先: 本構造体自体はデータ保持用であり、関数を呼び出すことはありません。

関連データ構造

  • PushFilter: 本構造体を利用するフィルタチェーンの基盤となる構造体。

失敗モードと境界条件

  • 構造体へのポインタが NULL の状態でアクセスされた場合、未定義の動作を引き起こします。

保守担当者向けメモ

  • 本構造体は pgp-encrypt.c 内のプライベートな状態管理用です。外部モジュールからは参照されません。
