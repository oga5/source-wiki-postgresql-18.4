目的

Porterステミングアルゴリズムのステップ1cを実行し、単語末尾の 'y' または 'Y' を 'i' に置換する。

シグネチャまたは形状

static int r_Step_1c(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z （Snowballの実行環境状態）
  • 出力: 成功時に 1、失敗または条件不一致時に 0、メモリ操作エラー時に負の値を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体を指していること。
  • out_grouping_b_U 関数により、対象文字の直前が母音（g_v）であることが確認される必要がある。

アルゴリズムまたは振る舞い

1. 現在位置（z->c）を z->ket に設定する。 2. 文字列を後方から走査し、'y' または 'Y' を検索する。見つからない場合は終了する。 3. z->bra を現在の位置に設定する。 4. out_grouping_b_U を使用し、対象文字の直前が母音であるかを確認する。 5. 条件を満たす場合、slice_from_s を呼び出し、該当箇所を s_8（'i'）に置換する。

呼び出し元と呼び出し先

  • 呼び出し元: stem_UTF_8_english.c 内のメイン処理フロー（english_UTF_8_stem からの呼び出しチェーン）、および stem_ISO_8859_1_english.c 等の各言語実装ファイル。
  • 呼び出し先: out_grouping_b_U（母音判定）、slice_from_s（文字列置換）。

関連データ構造

  • struct SN_env: ステミング処理の環境状態を保持する構造体。
  • g_v: 母音グループを定義するデータ構造。

失敗モードと境界条件

  • 文字列の先頭に達した場合や、直前が母音でない場合は 0 を返して終了する。
  • slice_from_s がメモリ不足等の理由で負の値を返した場合、その値をそのまま伝播させる。
