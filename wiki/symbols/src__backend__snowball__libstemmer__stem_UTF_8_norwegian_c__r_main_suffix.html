r_main_suffix

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_norwegian.c:159

目的

ノルウェー語のステミング処理において、単語の主要な接尾辞を特定し、削除または置換を行う。

シグネチャまたは形状

static int r_main_suffix(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（ステミング実行環境。単語データおよび現在のカーソル位置 z->c、境界 z->lb、領域情報 z->I を含む）
  • **出力**: int（成功時は 1、失敗または処理対象外の場合は 0、エラー発生時は負の値を返す）

前提条件と不変条件

  • z->I[1] に格納された領域境界よりカーソル位置 z->c が後方にあること。
  • 処理対象の単語は UTF-8 エンコーディングであること。
  • find_among_b による後方検索が可能な状態であること。

アルゴリズムまたは振る舞い

1. **領域チェック**: 現在のカーソル位置が z->I[1]（R1領域）より前であれば終了する。 2. **接尾辞検索**: z->lb を z->I[1] に一時的に設定し、a_0 配列を用いて後方から接尾辞を検索する。 3. **条件分岐**: 検索結果（among_var）に応じて以下の処理を行う。

  • case 1: slice_del を実行し、接尾辞を削除する。
  • case 2: g_s_ending グループに含まれるか判定し、条件を満たせば slice_del を実行する。条件を満たさない場合、特定の文字（'k'）と母音の組み合わせを確認した上で削除する。
  • case 3: slice_from_s を実行し、接尾辞を別の文字列（s_0）に置換する。

    • 4. **復元**: 変更した z->lb を元の値に戻して終了する。

呼び出し元と呼び出し先

  • **呼び出し元**: norwegian_UTF_8_stem（src/backend/snowball/libstemmer/stem_UTF_8_norwegian.c 内）。
  • **呼び出し先**: find_among_b, slice_del, slice_from_s, in_grouping_b_U, out_grouping_b_U。

関連データ構造

  • SN_env: ステミング実行環境を保持する構造体。
  • a_0: 接尾辞マッチングに使用される among 構造体の配列。
  • g_s_ending, g_v: 文字グループ定義。

失敗モードと境界条件

  • z->c が z->I[1] より前にある場合、即座に 0 を返す。
  • find_among_b で一致する接尾辞が見つからない場合、0 を返す。
  • slice_del や slice_from_s が内部でエラー（負の値）を返した場合、その値を伝播させる。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードであるため、直接編集せず元の .sbl スクリプトを修正すること。
  • UTF-8 処理のため、文字単位ではなくバイト単位での操作が行われている点に注意が必要。
  • z->lb の操作はスタック的な挙動を伴うため、後方検索の前後で正しく復元されていることを確認すること。
