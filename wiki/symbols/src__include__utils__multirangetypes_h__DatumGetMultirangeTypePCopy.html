DatumGetMultirangeTypePCopy

定義

  • 種別: function
  • 定義箇所: src/include/utils/multirangetypes.h:53

目的

Datum 形式のマルチレンジ型データを、メモリ上にコピーして MultirangeType ポインタとして取得します。

シグネチャまたは形状

static inline MultirangeType *DatumGetMultirangeTypePCopy(Datum X)

入力と出力

  • **入力**: Datum X（マルチレンジ型の Datum）
  • **出力**: MultirangeType *（デトーストおよびコピーされたマルチレンジ型のポインタ）

前提条件と不変条件

  • 入力 X は有効なマルチレンジ型の Datum である必要があります。
  • PG_DETOAST_DATUM_COPY を使用するため、戻り値は呼び出し元で適切にメモリ管理（解放）を行う必要があります。

アルゴリズムまたは振る舞い

1. 引数 X に対して PG_DETOAST_DATUM_COPY を実行します。 2. これにより、必要に応じて TOAST されたデータが展開（デトースト）され、新しいメモリ領域にコピーされます。 3. その結果を MultirangeType * にキャストして返却します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/include/utils/multirangetypes.h 内の #define PG_GETARG_MULTIRANGE_P_COPY(n) マクロが本関数を呼び出します。
  • **呼び出し先**: PG_DETOAST_DATUM_COPY マクロ（PostgreSQLのコアメモリ管理機能）。

関連データ構造

  • MultirangeType: マルチレンジ型のメモリ上の表現。
  • Datum: PostgreSQLの汎用データ型表現。

失敗モードと境界条件

  • メモリ不足時には PG_DETOAST_DATUM_COPY 経由でエラーが発生する可能性があります。
  • 入力が有効なマルチレンジ型でない場合、予期せぬ動作を引き起こす可能性があります。

保守担当者向けメモ

  • 本関数はコピーを作成するため、メモリリークを防ぐために戻り値のポインタが不要になった時点で適切に解放されているか確認してください。
  • MultirangeType 構造体の内部メンバに直接アクセスせず、提供されているアクセサ関数を使用してください。
