PQmblen

定義

  • 種別: function
  • 定義箇所: src/interfaces/libpq/fe-misc.c:1254

目的

指定されたエンコーディングにおける、マルチバイト文字列の先頭文字のバイト長を返します。

シグネチャまたは形状

int PQmblen(const char *s, int encoding);

入力と出力

  • **入力**:
  • s: 検査対象の文字列へのポインタ。
  • encoding: 文字列のエンコーディングを示す整数値。
  • **出力**:
  • 文字列 s の先頭文字のバイト長（整数）。

前提条件と不変条件

  • s は有効な文字列ポインタである必要があります。
  • encoding は、PostgreSQL がサポートする有効なエンコーディング識別子である必要があります。

アルゴリズムまたは振る舞い

内部的に pg_encoding_mblen 関数を呼び出し、指定されたエンコーディングルールに基づいて文字列の先頭文字の長さを算出します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/bin/psql/prompt.c
  • src/fe_utils/mbprint.c
  • src/fe_utils/print.c
  • src/fe_utils/string_utils.c
  • **呼び出し先**:
  • pg_encoding_mblen (内部的な処理委譲)

関連データ構造

  • mb/pg_wchar.h (エンコーディング関連の定義が含まれるヘッダー)

失敗モードと境界条件

  • s が指す文字列が空の場合や、エンコーディングが不正な場合の挙動は、依存先である pg_encoding_mblen の実装に依存します。

保守担当者向けメモ

  • 本関数は libpq-fe.h で公開されている API です。
  • 文字列の長さ制限を考慮した処理が必要な場合は、関連シンボルである PQmblenBounded の使用を検討してください。
