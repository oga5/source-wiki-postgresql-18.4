rt_box_union

定義

  • 種別: function
  • 定義箇所: src/backend/access/gist/gistproc.c:54

目的

2つの矩形（BOX）を包含する最小境界矩形（MBR）を計算し、結果を格納します。

シグネチャまたは形状

static void rt_box_union(BOX *n, const BOX *a, const BOX *b)

入力と出力

  • **入力**:
  • a, b: 結合対象となる2つの BOX 構造体へのポインタ（const）。
  • **出力**:
  • n: 結合結果のMBRが書き込まれる BOX 構造体へのポインタ。

前提条件と不変条件

  • n, a, b は有効な BOX 構造体を指している必要があります。
  • BOX 構造体は high（最大座標）と low（最小座標）のフィールドを持つ幾何データ型です。

アルゴリズムまたは振る舞い

1. a と b の各軸の最大値を比較し、n->high.x および n->high.y に設定します。 2. a と b の各軸の最小値を比較し、n->low.x および n->low.y に設定します。 3. 比較には float8_max および float8_min マクロを使用します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/access/gist/gistproc.c:101（gist_box_union 内での利用が確認されています）
  • **呼び出し先**:
  • float8_max
  • float8_min

関連データ構造

  • BOX: GiSTインデックスで扱われる2次元矩形データ型。

失敗モードと境界条件

  • 入力ポインタが NULL の場合の動作は定義されていません。
  • 座標値が NaN や無限大である場合の挙動は、float8_max/min の実装に依存します。

保守担当者向けメモ

  • 本関数は gistproc.c 内の内部ヘルパー関数であり、GiSTインデックスにおける矩形結合の基本演算を提供します。
  • 幾何データ型のインデックス構築において、MBRの拡張が必要な箇所で再利用されています。
