test_custom_rmgrs_insert_wal_record

定義

  • 種別: function
  • 定義箇所: src/test/modules/test_custom_rmgrs/test_custom_rmgrs.c:119

目的

SQLインターフェースを通じて、任意の文字列データをカスタムWALレコードとしてPostgreSQLのWAL（Write-Ahead Log）に書き込むための関数です。

シグネチャまたは形状

Datum test_custom_rmgrs_insert_wal_record(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS を介して受け取る text 型の引数（WALに書き込むペイロード）。
  • **出力**: XLogRecPtr（LSN: Log Sequence Number）を PG_RETURN_LSN マクロで返却。

前提条件と不変条件

  • モジュールが shared_preload_libraries に設定され、_PG_init によりカスタムRmgrが登録済みであること。
  • RM_TESTCUSTOMRMGRS_ID が有効なカスタムRmgr IDとしてシステムに認識されていること。

アルゴリズムまたは振る舞い

1. 引数からペイロードのデータとサイズを取得する。 2. xl_testcustomrmgrs_message 構造体にメッセージサイズを格納する。 3. XLogBeginInsert() を呼び出し、WAL挿入シーケンスを開始する。 4. XLogRegisterData() を使用して、ヘッダー（xlrec）とペイロードをWALバッファに登録する。 5. XLogSetRecordFlags(XLOG_MARK_UNIMPORTANT) を呼び出し、当該レコードを重要でないものとしてマークする。 6. XLogInsert() を実行し、RM_TESTCUSTOMRMGRS_ID および XLOG_TEST_CUSTOM_RMGRS_MESSAGE 型としてWALに書き込む。 7. 書き込まれたレコードのLSNを返す。

呼び出し元と呼び出し先

  • **呼び出し元**: SQLインターフェース（PG_FUNCTION_INFO_V1 マクロにより登録）。
  • **呼び出し先**:
  • XLogBeginInsert, XLogRegisterData, XLogSetRecordFlags, XLogInsert（PostgreSQL WAL API）。
  • VARDATA_ANY, VARSIZE_ANY_EXHDR（PostgreSQL 可変長データ操作マクロ）。

関連データ構造

  • xl_testcustomrmgrs_message: WALレコードのヘッダーとして使用される構造体。
  • RmgrData: カスタムRmgrの登録に使用されるデータ構造。

失敗モードと境界条件

  • 引数がNULLの場合、PG_GETARG_TEXT_PP の挙動に従いエラーとなる可能性がある。
  • WAL書き込み時にシステムリソースやログ領域が不足している場合、XLogInsert がエラーを発生させる可能性がある。

保守担当者向けメモ

  • 本関数はテスト用モジュールの一部であり、実運用環境でカスタムRmgrを実装する際のテンプレートとして利用可能です。
  • XLOG_MARK_UNIMPORTANT を使用しているため、リカバリ時の重要度は低く設定されています。
