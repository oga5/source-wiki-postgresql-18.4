pow5Factor

定義

  • 種別: function
  • 定義箇所: src/common/f2s.c:80

目的

与えられた整数が5で何回割り切れるか（5の冪乗の指数）を計算する。

シグネチャまたは形状

static inline uint32 pow5Factor(uint32 value)

入力と出力

  • **入力**: value (uint32) - 5で割る対象の正の整数。
  • **出力**: count (uint32) - value を5で割り切れる回数。

前提条件と不変条件

  • 入力 value は 0 であってはならない（Assert(value != 0) により検証される）。

アルゴリズムまたは振る舞い

1. カウンタ count を 0 に初期化する。 2. ループ内で value を 5 で除算し、商 q と剰余 r を求める。 3. 剰余 r が 0 でない場合、ループを終了する。 4. 剰余 r が 0 の場合、value を q で更新し、count をインクリメントして手順2へ戻る。 5. 最終的な count を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/common/f2s.c:104 で呼び出されている。また、src/common/d2s.c 内の複数の箇所で pow5Factor が参照されている（ただし、d2s.c での呼び出しは uint64 型を引数に取るため、本関数とはシグネチャが異なる可能性がある）。
  • **呼び出し先**: 特になし（標準的な算術演算のみ）。

関連データ構造

  • なし。

失敗モードと境界条件

  • value が 0 の場合、Assert によりプログラムが停止する可能性がある。
  • value が 5 で一度も割り切れない場合、戻り値は 0 となる。

保守担当者向けメモ

  • 本関数は static inline で定義されており、コンパイル時にインライン展開されることが期待されている。
  • d2s.c でも同様の名称の関数が参照されているが、引数の型が異なる点に注意すること。
