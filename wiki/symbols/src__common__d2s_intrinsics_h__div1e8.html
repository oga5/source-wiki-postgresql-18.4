div1e8

定義

  • 種別: function
  • 定義箇所: src/common/d2s_intrinsics.h:168

目的

浮動小数点数から文字列への変換アルゴリズム（Ryu）において、10^8 による除算を定数乗算とビットシフトに置き換えて実行する。

シグネチャまたは形状

static inline uint64 div1e8(const uint64 x)

入力と出力

  • **入力**: x (64ビット符号なし整数)
  • **出力**: x / 10^8 の計算結果（64ビット符号なし整数）

前提条件と不変条件

  • 本関数は、コンパイル時に RYU_32_BIT_PLATFORM が定義されている環境において、高コストな除算命令を回避するために使用される。
  • 使用されるマジックナンバー 0xABCC77118461CEFD は、10^8 による除算を乗算とシフトで近似するための定数である。

アルゴリズムまたは振る舞い

入力値 x に対して、マジックナンバー 0xABCC77118461CEFD を用いた umulh（上位64ビット積の取得）を行い、その結果を右に 26 ビットシフトすることで除算結果を得る。

呼び出し元と呼び出し先

  • **呼び出し元**: src/common/d2s.c (690行目および862行目)
  • **呼び出し先**: umulh

関連データ構造

  • uint64: 64ビット符号なし整数型。

失敗モードと境界条件

  • 本関数はマジックナンバーを用いた近似計算であるため、入力値が特定の範囲を超えた場合に精度が維持されるかは、マジックナンバーの数学的妥当性に依存する。
  • 32ビット環境以外での動作については、コンパイル時のマクロ定義（HAS_64_BIT_INTRINSICS 等）に依存する。

保守担当者向けメモ

  • 本関数で使用しているマジックナンバーは Clang x64 が生成するコードと同等である。変更が必要な場合は、数学的な正当性の検証が不可欠である。
  • 呼び出し元と呼び出し先の情報は、提供されたソースコードの範囲内でのみ確認されている。
