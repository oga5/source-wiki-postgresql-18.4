r_remove_first_order_prefix

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_indonesian.c:193

目的

インドネシア語のステミング処理において、単語の先頭から1次接頭辞（first-order prefix）を特定し、除去または置換を行う。

シグネチャまたは形状

static int r_remove_first_order_prefix(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（ステミング環境構造体。単語データおよびワークスペース I[0], I[1] を含む）
  • **出力**: int（成功時に 1、失敗または対象外の場合に 0、エラー発生時に負の値を返す）

前提条件と不変条件

  • z->p が処理対象の文字列を保持していること。
  • z->c が現在のカーソル位置を示すこと。
  • 処理の過程で z->I[0]（接頭辞の種類フラグ）および z->I[1]（母音数カウント）が更新される。

アルゴリズムまたは振る舞い

1. 現在位置の次（z->c + 1）が 'i' または 'e' であるかを確認し、条件を満たさない場合は終了する。 2. a_3 配列を用いて find_among を実行し、接頭辞のパターンマッチングを行う。 3. マッチしたパターン（among_var）に応じて以下の処理を行う：

  • **ケース1, 2**: slice_del で接頭辞を除去し、z->I[0] を更新、z->I[1] をデクリメントする。
  • **ケース3, 4**: slice_from_s で接頭辞を別の文字列に置換し、z->I[0] を更新、z->I[1] をデクリメントする。
  • **ケース5, 6**: 母音の有無に応じて slice_del または slice_from_s を選択的に実行し、z->I[0] を更新、z->I[1] をデクリメントする。

    • 4. 成功時に 1 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: indonesian_ISO_8859_1_stem（src/backend/snowball/libstemmer/stem_ISO_8859_1_indonesian.c:352）および stem_UTF_8_indonesian.c 内の同名関数。
  • **呼び出し先**: find_among, slice_del, slice_from_s, in_grouping（いずれもSnowballランタイムユーティリティ）。

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_3: 1次接頭辞を定義した among 構造体の配列。

失敗モードと境界条件

  • 文字列長が不足している場合や、パターンに一致しない場合は 0 を返す。
  • slice_del や slice_from_s が負の値を返した場合、そのエラーコードをそのまま伝播させる。

保守担当者向けメモ

  • 本関数は自動生成コードであるため、アルゴリズムの変更は元のSnowballスクリプト（.sbl）に対して行うこと。
  • z->I[0] は接頭辞の種類を識別するために使用されており、後続の処理（2次接頭辞除去など）に影響を与える。
