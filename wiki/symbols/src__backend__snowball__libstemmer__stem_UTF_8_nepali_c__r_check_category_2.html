r_check_category_2

定義

  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_nepali.c:319

目的

ネパール語のステミング処理において、カテゴリ2に分類される接尾辞の削除条件を満たしているかを確認する。

シグネチャまたは形状

static int r_check_category_2(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z （Snowballランタイム環境）
  • **出力**: int （条件を満たせば 1、それ以外は 0 を返す）

前提条件と不変条件

  • z->c は処理対象の文字列内のカーソル位置を指す。
  • find_among_b を用いた後方検索を行うため、カーソルは文字列の末尾付近にあることが想定される。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を z->ket に設定する。 2. 文字列の境界条件（z->lb）および、現在の文字の特定のビットパターン（z->p[z->c - 1]）を検証する。 3. a_1 テーブルを使用して、末尾からカテゴリ2に該当する接尾辞を後方検索（find_among_b）する。 4. マッチした場合、その位置を z->bra に設定し、成功（1）を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_UTF_8_nepali.c 内のメイン処理ループ（390行目付近）。
  • **呼び出し先**: find_among_b（Snowballランタイム関数）。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_1: カテゴリ2の接尾辞を定義する among 構造体配列。

失敗モードと境界条件

  • 文字列が短すぎて接尾辞を保持できない場合（z->c - 2 <= z->lb）、0 を返す。
  • 文字のビットパターンが一致しない場合、0 を返す。
  • find_among_b で接尾辞が見つからない場合、0 を返す。
