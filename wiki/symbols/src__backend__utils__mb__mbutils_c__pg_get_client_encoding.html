pg_get_client_encoding

定義

  • 種別: function
  • 定義箇所: src/backend/utils/mb/mbutils.c:344

目的

現在セッションで設定されているクライアントエンコーディングの識別子（ID）を取得します。

シグネチャまたは形状

int pg_get_client_encoding(void)

入力と出力

  • **入力**: なし
  • **出力**: 現在のクライアントエンコーディングを示す整数値（int）を返します。

前提条件と不変条件

  • ClientEncoding グローバル変数が初期化済みである必要があります。
  • 本関数は、バックエンドのエンコーディング管理状態を参照するため、セッションのライフサイクル内で有効な値を返します。

アルゴリズムまたは振る舞い

内部的に保持されている ClientEncoding 構造体（またはポインタ）から、encoding メンバの値を直接読み取って返します。

呼び出し元と呼び出し先

  • **呼び出し元（直接的な証拠）**:
  • src/backend/commands/copyfrom.c:1686
  • src/backend/commands/copyto.c:888
  • src/backend/utils/adt/xml.c:448
  • **呼び出し先**:
  • 外部関数呼び出しは行わず、グローバル変数へのアクセスのみを行います。

関連データ構造

  • ClientEncoding: 現在のクライアントエンコーディング設定を保持する内部構造体。

失敗モードと境界条件

  • 本関数自体は単純な値の読み取りであるため、関数内部での失敗モードはありません。
  • ClientEncoding が初期化されていない状態で呼び出された場合の挙動は、呼び出し側のコンテキストに依存します。

保守担当者向けメモ

  • 本関数は頻繁に呼び出される可能性があるため、実装は極めて軽量に保たれています。
  • エンコーディングの識別子を変更・拡張する場合は、src/include/mb/pg_wchar.h の定義と整合性を保つ必要があります。
