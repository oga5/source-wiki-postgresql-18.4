r_consonant_pair

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_norwegian.c:202

目的

単語の末尾から特定の子音ペアを検出し、その一部を削除することでノルウェー語のステミング処理を行う。

シグネチャまたは形状

static int r_consonant_pair(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング実行環境）
  • 出力: 成功時に 1、失敗または条件不一致時に 0、メモリ操作エラー時に負の値を返す。

前提条件と不変条件

  • z->I[1] に格納された領域境界（R1等）よりカーソル位置 z->c が後ろにあること。
  • z->p[z->c - 1] が文字コード 116（'t'）であること。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置を保存し、z->I[1] を境界として後方検索を行う。 2. a_1 配列を参照し、find_among_b を用いて特定の子音ペアを検索する。 3. マッチした場合、skip_b_utf8 で1文字分戻り、その位置から slice_del を実行して該当箇所を削除する。 4. 処理終了後、カーソル位置を元の位置に復元する。

呼び出し元と呼び出し先

  • 呼び出し元: norwegian_UTF_8_stem（src/backend/snowball/libstemmer/stem_UTF_8_norwegian.c 内）
  • 呼び出し先: find_among_b, skip_b_utf8, slice_del（いずれも header.h 経由で提供されるランタイム関数）

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_1: 子音ペアの定義を含む among 構造体の配列。

失敗モードと境界条件

  • カーソル位置が z->I[1] より前にある場合、即座に 0 を返す。
  • find_among_b がマッチしない場合、境界を復元して 0 を返す。
  • slice_del が内部エラー（負の値）を返した場合、その値をそのまま伝播させる。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードであるため、直接編集せず元の .sbl スクリプトを修正すること。
  • UTF-8 処理のため、文字単位ではなくバイト単位の操作が行われている点に注意が必要。
