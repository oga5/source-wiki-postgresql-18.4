r_vowel_suffix

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_italian.c:934

目的

イタリア語のステミング処理において、単語末尾の特定の母音や文字の組み合わせを削除し、語幹を正規化する。

シグネチャまたは形状

static int r_vowel_suffix(struct SN_env * z)

入力と出力

  • 入力: SN_env 構造体（ステミング環境）。
  • 出力: 成功時に 1 を返し、失敗またはエラー時に 0 または負の値を返す。

前提条件と不変条件

  • SN_env 構造体が初期化されていること。
  • r_RV 関数により、対象文字が RV 領域内にあることが検証される。

アルゴリズムまたは振る舞い

以下の2つの主要な処理を順次実行する。 1. **母音の削除**:

  • 単語末尾が g_AEIO グループの母音である場合、r_RV を経てその母音を削除する。
  • 続いて、末尾が 'i' である場合、同様に r_RV を経て削除する。

    • 2. **'h' を含む接尾辞の削除**:

  • 単語末尾が 'h' であり、その直前が g_CG グループ（'c' または 'g'）である場合、r_RV を経て 'h' を削除する。

呼び出し元と呼び出し先

  • 呼び出し元: italian_UTF_8_stem（stem_UTF_8_italian.c 内）から呼び出される。
  • 呼び出し先: r_RV（領域判定）、slice_del（文字列削除）、in_grouping_b_U（文字グループ判定）。

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • g_AEIO, g_CG: 文字グループを定義する静的配列。

失敗モードと境界条件

  • r_RV が失敗した場合、処理は中断され、元の状態に復元される（goto lab0 または lab1）。
  • slice_del がエラー（負の値）を返した場合、その値をそのまま呼び出し元へ伝播させる。
  • 文字列の境界（z->lb）に達した場合、削除処理は行われない。

保守担当者向けメモ

  • 本関数は Snowball コンパイラにより自動生成されているため、直接編集せず、元の Snowball ソース（.sbl）を修正すること。
  • r_RV による領域チェックがステミングの精度を決定するため、変更時は RV 領域の定義に注意が必要。
