pg_get_timezone_name

定義

  • 種別: function
  • 定義箇所: src/timezone/localtime.c:1988

目的

指定されたタイムゾーン構造体から、そのタイムゾーン名（略称）を取得する。

シグネチャまたは形状

const char *pg_get_timezone_name(pg_tz *tz);

入力と出力

  • **入力**: pg_tz *tz （タイムゾーン情報を保持する構造体へのポインタ）
  • **出力**: const char * （タイムゾーン名を表す文字列へのポインタ。tz が NULL の場合は NULL を返す）

前提条件と不変条件

  • 引数 tz が有効な pg_tz 構造体を指しているか、あるいは NULL である必要がある。
  • 返される文字列は pg_tz 構造体内部のメモリを指しており、呼び出し側で解放してはならない。

アルゴリズムまたは振る舞い

1. 引数 tz が NULL でないかを確認する。 2. NULL であれば NULL を返す。 3. NULL でなければ、tz->TZname メンバに格納されている文字列ポインタを返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/commands/variable.c
  • src/backend/utils/adt/date.c
  • src/backend/utils/adt/datetime.c
  • **呼び出し先**:
  • 本関数は内部で他の関数を呼び出さない。

関連データ構造

  • pg_tz: タイムゾーン情報を保持する構造体。本関数はこの構造体の TZname フィールドに直接アクセスする。

失敗モードと境界条件

  • tz が NULL の場合、安全に NULL を返す。

保守担当者向けメモ

  • 本関数は src/include/pgtime.h で宣言されており、外部モジュールから広く利用されるアクセサである。
  • TZname フィールドのライフサイクルは pg_tz 構造体に依存するため、構造体が破棄された後に返された文字列ポインタを参照しないよう注意すること。
