pg_popcount_avx512_available

定義

  • 種別: function
  • 定義箇所: src/port/pg_popcount_avx512.c:97

目的

実行環境が Intel AVX-512 命令セットを用いた高速なビットカウント（popcount）機能をサポートしているかを判定します。

シグネチャまたは形状

bool pg_popcount_avx512_available(void)

入力と出力

  • 入力: なし
  • 出力: AVX-512 popcount が利用可能な場合は true、それ以外は false を返します。

前提条件と不変条件

  • 本関数は USE_AVX512_POPCNT_WITH_RUNTIME_CHECK マクロが定義されている環境でのみ有効です。
  • 実行時の CPUID チェックを通じて、OS および CPU が必要な拡張命令をサポートしていることを検証します。

アルゴリズムまたは振る舞い

以下の3つの条件がすべて真である場合にのみ true を返します。 1. xsave_available(): XSAVE 命令が利用可能であること。 2. zmm_regs_available(): 512ビット幅の ZMM レジスタが利用可能であること。 3. avx512_popcnt_available(): AVX-512 VPOPCNTDQ 命令セットが利用可能であること。

呼び出し元と呼び出し先

  • 呼び出し元: src/port/pg_bitutils.c:179（汎用的な pg_popcount 実装を選択する際の判定ロジックとして使用）。
  • 呼び出し先: xsave_available(), zmm_regs_available(), avx512_popcnt_available() を呼び出します。

関連データ構造

  • なし

失敗モードと境界条件

  • コンパイル時にターゲット CPU が AVX-512 をサポートしていない場合でも、本関数による実行時チェックにより安全にフォールバックされる設計となっています。

保守担当者向けメモ

  • 本関数は pg_attribute_target を使用してコンパイルされたコードパスの一部です。
  • 呼び出し先である各 _available 関数は、同ファイル内で定義されている CPU 機能判定用のヘルパー関数です。
