r_prelude

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_english.c:361

目的

英語のステミング処理における前処理（prelude）を実行し、単語内の特定の文字（アポストロフィや 'y'）の置換や削除を行う。

シグネチャまたは形状

static int r_prelude(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z （Snowballランタイム環境）
  • 出力: 成功時に 1、メモリ操作等のエラー発生時に負の値を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • 内部状態 z->I[2] は本関数の開始時に 0 に初期化される。

アルゴリズムまたは振る舞い

1. **アポストロフィの削除**: 単語の先頭にアポストロフィがある場合、それを削除する。 2. **'y' の置換**: 単語内の 'y' を 'Y'（s_0）に置換し、フラグ z->I[2] を 1 に設定する。 3. **母音後の 'y' の置換**: 母音（g_v）の直後にある 'y' をすべて 'Y'（s_1）に置換し、フラグ z->I[2] を 1 に設定する。 4. 処理中、slice_del や slice_from_s を用いて文字列操作を行い、エラー時は即座にリターンする。

呼び出し元と呼び出し先

  • 呼び出し元: english_UTF_8_stem（直接の記述はないが、モジュール内の制御フローに基づく）。
  • 呼び出し先: slice_del, slice_from_s, in_grouping_U, skip_utf8。

関連データ構造

  • struct SN_env: ステミング処理の環境変数、カーソル位置、文字列バッファを保持する。
  • g_v: 母音グループを定義する定数。

失敗モードと境界条件

  • slice_del または slice_from_s が失敗した場合、負の戻り値を伝播させる。
  • 文字列の終端に達した場合、ポインタ操作は安全に停止する。

保守担当者向けメモ

  • 本関数はSnowballコンパイラによる自動生成コードであるため、直接編集せず .sbl 定義ファイルを修正すること。
  • z->I[2] は、この前処理で 'y' が変換されたかどうかを示すフラグとして後続のステップで使用される可能性がある。
