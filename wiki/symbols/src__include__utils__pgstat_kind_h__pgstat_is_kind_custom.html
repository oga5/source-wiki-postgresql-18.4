pgstat_is_kind_custom

定義

  • 種別: static inline function
  • 定義箇所: src/include/utils/pgstat_kind.h:66

目的

与えられた統計種別（PgStat_Kind）が、拡張機能等で利用可能な「カスタム統計種別」の範囲内にあるかどうかを判定する。

シグネチャまたは形状

static inline bool pgstat_is_kind_custom(PgStat_Kind kind)

入力と出力

  • **入力**: PgStat_Kind kind（判定対象の統計種別ID）
  • **出力**: bool（カスタム統計種別であれば true、それ以外は false）

前提条件と不変条件

  • PgStat_Kind は uint32 型のエイリアスである。
  • カスタム統計種別の有効範囲は PGSTAT_KIND_CUSTOM_MIN 以上かつ PGSTAT_KIND_CUSTOM_MAX 以下である。

アルゴリズムまたは振る舞い

引数 kind が PGSTAT_KIND_CUSTOM_MIN 以上かつ PGSTAT_KIND_CUSTOM_MAX 以下であるかを比較演算し、その結果を論理値で返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/activity/pgstat.c: 統計情報の処理分岐や判定に使用。
  • src/include/utils/pgstat_internal.h: 統計種別の妥当性検証（Assert）に使用。
  • **呼び出し先**: なし（インライン関数として定義）。

関連データ構造

  • PgStat_Kind: 統計種別を識別するための型。
  • PGSTAT_KIND_CUSTOM_MIN / PGSTAT_KIND_CUSTOM_MAX: カスタム統計種別の範囲を定義する定数。

失敗モードと境界条件

  • kind が PGSTAT_KIND_INVALID（0）の場合、範囲外として false を返す。
  • 組み込み統計種別（PGSTAT_KIND_BUILTIN_MIN ～ MAX）が渡された場合、範囲外として false を返す。

保守担当者向けメモ

  • 本関数は統計システムのルーティングや検証において頻繁に使用されるため、変更時は統計種別のID割り当て範囲（PGSTAT_KIND_MAX 等）との整合性に注意すること。
