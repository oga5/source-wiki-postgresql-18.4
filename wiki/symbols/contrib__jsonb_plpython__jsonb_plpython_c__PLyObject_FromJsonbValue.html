PLyObject_FromJsonbValue

定義

  • 種別: function
  • 定義箇所: contrib/jsonb_plpython/jsonb_plpython.c:94

目的

PostgreSQL の JsonbValue 構造体を、対応する Python オブジェクト（PyObject）へ変換する。

シグネチャまたは形状

static PyObject *PLyObject_FromJsonbValue(JsonbValue *jsonbValue)

入力と出力

  • **入力**: JsonbValue *jsonbValue （変換対象の JSONB 値）
  • **出力**: 変換後の PyObject *。成功時は新しい Python オブジェクトへの参照を返し、失敗時は PostgreSQL のエラーとして elog(ERROR, ...) を発生させる。

前提条件と不変条件

  • jsonbValue は有効なポインタであること。
  • Python インタプリタが初期化されており、decimal_constructor が利用可能であること。

アルゴリズムまたは振る舞い

jsonbValue->type に基づいて以下の変換を行う： 1. jbvNull: Python の None を返す。 2. jbvBinary: PLyObject_FromJsonbContainer を呼び出し、再帰的にコンテナ（オブジェクト/配列）を変換する。 3. jbvNumeric: numeric_out を介して文字列化し、decimal_constructor を用いて Python の Decimal オブジェクトを生成する。 4. jbvString: PLyUnicode_FromJsonbValue を呼び出し、Python の文字列型へ変換する。 5. jbvBool: jsonbValue->val.boolean に応じて Python の True または False を返す。 6. その他: 未知の型に対しては elog(ERROR) を発行する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/jsonb_plpython/jsonb_plpython.c:159
  • contrib/jsonb_plpython/jsonb_plpython.c:176
  • contrib/jsonb_plpython/jsonb_plpython.c:220
  • **呼び出し先**:
  • PLyObject_FromJsonbContainer
  • PLyUnicode_FromJsonbValue
  • numeric_out (PostgreSQL 内部関数)
  • PyObject_CallFunction (Python C API)

関連データ構造

  • JsonbValue: PostgreSQL 側の JSONB 値表現。
  • PyObject: Python 側のオブジェクト表現。

失敗モードと境界条件

  • 未知の JsonbValue 型が渡された場合、elog(ERROR) により処理が中断される。
  • 数値変換時やコンテナ変換時にメモリ不足等のエラーが発生した場合、Python C API 経由で例外が発生し、PostgreSQL のエラーハンドリングへ移行する。

保守担当者向けメモ

  • 本関数は再帰的な変換の一部を担う。jbvBinary 型の処理において PLyObject_FromJsonbContainer を呼び出すため、循環参照や深いネスト構造に対するスタックオーバーフローの可能性については、呼び出し元のコンテキストに依存する。
