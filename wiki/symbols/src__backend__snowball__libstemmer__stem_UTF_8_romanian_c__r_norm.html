r_norm

定義

  • **種別**: function
  • **定義箇所**: src/backend/snowball/libstemmer/stem_UTF_8_romanian.c:551

目的

ルーマニア語のステミング処理における前処理として、入力文字列内の特殊文字を正規化する。

シグネチャまたは形状

static int r_norm(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z（Snowball ステミング環境へのポインタ）
  • **出力**: 成功時に 1 を返し、slice_from_s 等の内部処理でエラーが発生した場合は負の値を返す。

前提条件と不変条件

  • z->p に処理対象の文字列が格納されていること。
  • z->c が現在のカーソル位置を指していること。
  • a_0 テーブルが定義されており、find_among によるマッチングが可能であること。

アルゴリズムまたは振る舞い

1. 文字列全体を走査し、特定のバイトパターン（159 または 163 を含むシーケンス）を検索する。 2. find_among を使用して a_0 テーブルと照合し、マッチした場合は slice_from_s を用いて文字列を置換する。 3. マッチしない場合は skip_utf8 を使用して次の文字へ進む。 4. 処理完了後、カーソル位置を初期状態（c1）に復元する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_UTF_8_romanian.c 内のメイン処理（行972付近）。
  • **呼び出し先**: find_among（ユーティリティ関数）、slice_from_s（置換関数）、skip_utf8（UTF-8 文字スキップ関数）。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_0: 正規化対象の接尾辞や文字パターンを定義した among 構造体の配列。

失敗モードと境界条件

  • slice_from_s がメモリ不足やその他の理由で負の値を返した場合、その値をそのまま呼び出し元へ伝播させる。
  • 文字列の終端に達した場合、ループを終了して正常終了する。

保守担当者向けメモ

  • 本関数は自動生成コードの一部である。ロジックの変更が必要な場合は、直接編集せず Snowball 言語のソース（.sbl）を修正すること。
  • z->bra と z->ket を使用して置換範囲を動的に制御している点に注意。
