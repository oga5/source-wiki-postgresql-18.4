check_special_value

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/ecpglib/data.c:100

目的

文字列から浮動小数点数の特殊値（NaN, Infinity, -Infinity）を検出し、対応する C 言語の double 値へ変換します。

シグネチャまたは形状

static bool check_special_value(char *ptr, double *retval, char **endptr);

入力と出力

  • **入力**:
  • ptr: 解析対象の文字列ポインタ。
  • retval: 特殊値が検出された場合に値を格納する double 型のポインタ。
  • endptr: 解析終了後のポインタ位置を格納する char ** 型のポインタ。
  • **出力**:
  • 戻り値: 特殊値が検出された場合は true、それ以外は false。
  • retval および endptr: 特殊値が検出された場合のみ更新されます。

前提条件と不変条件

  • ptr は有効なヌル終端文字列を指している必要があります。
  • retval および endptr は書き込み可能な有効なメモリ領域を指している必要があります。

アルゴリズムまたは振る舞い

1. pg_strncasecmp を使用して、大文字小文字を区別せずに文字列の先頭が "NaN", "Infinity", "-Infinity" のいずれかに一致するかを順次判定します。 2. 一致した場合、対応する特殊値を get_float8_nan() または get_float8_infinity() を用いて取得し、retval に代入します。 3. endptr に、一致した文字列の長さ分だけ進めたポインタを格納します。 4. いずれにも一致しない場合は false を返し、ポインタの更新は行いません。

呼び出し元と呼び出し先

  • **呼び出し元**: src/interfaces/ecpg/ecpglib/data.c 内の ecpg_get_data 関数から呼び出されます。
  • **呼び出し先**:
  • pg_strncasecmp
  • get_float8_nan
  • get_float8_infinity

関連データ構造

  • double: 浮動小数点値の格納に使用。

失敗モードと境界条件

  • 特殊値に該当しない文字列が渡された場合、false を返して終了します。
  • -Infinity の判定において、負号を含めた 9 文字の比較が正しく行われます。

保守担当者向けメモ

  • 本関数は ecpg_get_data における浮動小数点型のパース処理の一部として機能します。
  • 特殊値の定義を変更または追加する場合は、get_float8_nan および get_float8_infinity の実装と整合性を取る必要があります。
