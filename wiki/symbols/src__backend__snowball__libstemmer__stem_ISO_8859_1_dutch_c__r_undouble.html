r_undouble

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_dutch.c:327

目的

単語末尾の重複文字を解消し、語幹抽出を正規化する。

シグネチャまたは形状

static int r_undouble(struct SN_env * z)

入力と出力

  • 入力: SN_env 構造体（現在のカーソル位置 c、文字列バッファ p を含む）。
  • 出力: 成功時に 1、失敗または条件不一致時に 0、エラー時に負の値を返す。

前提条件と不変条件

  • z->c が有効な文字列範囲内であること。
  • a_2 に定義された特定の文字パターンが、現在のカーソル位置の直前に存在すること。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置から a_2 に定義されたパターンを後方検索する。 2. パターンが見つかった場合、その位置を ket（削除開始点）としてマークする。 3. カーソルを1つ戻し、その位置を bra（削除終了点）としてマークする。 4. slice_del(z) を呼び出し、該当する重複文字を削除する。

呼び出し元と呼び出し先

  • 呼び出し元: stem_ISO_8859_1_dutch.c 内のメイン処理フロー（dutch_ISO_8859_1_stem 経由の各ステップ）。
  • 呼び出し先: find_among_b（パターン検索）、slice_del（文字列削除）。

関連データ構造

  • struct SN_env: Snowball ランタイム環境。
  • a_2: 重複解消の対象となる文字パターンを保持する静的配列。

失敗モードと境界条件

  • カーソルがバッファの境界（lb）に達している場合、処理を中断し 0 を返す。
  • find_among_b がパターンを見つけられない場合、0 を返す。
  • slice_del が内部エラーを返した場合、その値をそのまま伝播させる。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードである。ロジックの変更が必要な場合は、直接編集せず元の .sbl ファイルを修正すること。
  • a_2 配列の定義内容が、この関数の適用範囲を決定する。
