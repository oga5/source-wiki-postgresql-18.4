crosstab_cat_desc

定義

  • 種別: struct
  • 定義箇所: contrib/tablefunc/tablefunc.c:122

目的

クロス集計（crosstab）処理において、カテゴリ名とそれに対応する属性インデックスを関連付けるためのデータ構造です。

シグネチャまたは形状

typedef struct crosstab_cat_desc
{
	char	   *catname;		/* カテゴリ名 */
	uint64		attidx;			/* 0から始まる属性インデックス */
} crosstab_cat_desc;

入力と出力

本構造体はデータ保持用であり、関数としての入力・出力は持ちません。メモリ確保時には palloc を介してインスタンス化されます。

前提条件と不変条件

  • catname はカテゴリ名を指す文字列ポインタです。
  • attidx は 0 始まりのインデックスとして定義されています。

アルゴリズムまたは振る舞い

本構造体自体はアルゴリズムを持ちませんが、crosstab 関連関数において、入力データのカテゴリを特定の出力列（属性）へマッピングするためのメタデータとして利用されます。

呼び出し元と呼び出し先

  • 呼び出し元（参照箇所）:
  • contrib/tablefunc/tablefunc.c:164
  • contrib/tablefunc/tablefunc.c:753
  • contrib/tablefunc/tablefunc.c:769 (メモリ確保)
  • contrib/tablefunc/tablefunc.c:870
  • 呼び出し先:
  • 構造体定義のため、特定の関数を呼び出すことはありません。

関連データ構造

  • crosstab_HashEnt: カテゴリ名と属性インデックスを管理するハッシュテーブルエントリ。

失敗モードと境界条件

  • palloc によるメモリ確保失敗時にエラーが発生する可能性があります。
  • catname が適切に初期化されていない場合、メモリ参照エラーを引き起こす可能性があります。

保守担当者向けメモ

  • 本構造体は crosstab 関数群の内部ロジックに密接に関連しています。構造体のフィールドを変更する場合は、関連するハッシュテーブル操作およびメモリ管理箇所（palloc 呼び出し元）の整合性を確認してください。
