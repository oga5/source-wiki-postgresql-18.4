ghstore_out

定義

  • 種別: function
  • 定義箇所: contrib/hstore/hstore_gist.c:107

目的

ghstore 型の値をテキスト形式に変換しようとした際に、その操作がサポートされていないことを通知する。

シグネチャまたは形状

Datum ghstore_out(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS（PostgreSQL の関数呼び出し規約に基づく引数）
  • **出力**: なし（常にエラーを発生させる）

前提条件と不変条件

  • 本関数は ghstore 型の出力関数として登録されているが、実質的な変換処理は行わない。

アルゴリズムまたは振る舞い

1. ereport を呼び出し、ERRCODE_FEATURE_NOT_SUPPORTED エラーを発生させる。 2. エラーメッセージとして「cannot display a value of type ghstore」を出力する。 3. PG_RETURN_VOID() を呼び出す（コンパイラの警告回避用であり、実際には到達しない）。

呼び出し元と呼び出し先

  • **呼び出し元**: PostgreSQL の関数管理システム（PG_FUNCTION_INFO_V1(ghstore_out) により登録）。
  • **呼び出し先**: ereport（PostgreSQL のエラー報告用マクロ）。

関連データ構造

  • ghstore: GiST インデックス内部で使用される hstore のシグネチャ型。

失敗モードと境界条件

  • 本関数は呼び出された時点で常に失敗（エラー発生）するように設計されている。

保守担当者向けメモ

  • 本関数は ghstore 型の表示を意図的に禁止するために存在している。将来的にデバッグ目的などで表示が必要になった場合を除き、この動作を変更すべきではない。
