mbms_int_members

定義

  • 種別: function
  • 定義箇所: src/backend/nodes/multibitmapset.c:99

目的

2つのマルチビットマップセット（List のリスト）の積集合を計算し、左辺のリストを更新する。

シグネチャまたは形状

List *mbms_int_members(List *a, const List *b)

入力と出力

  • **入力**:
  • a: 更新対象となるマルチビットマップセット（List）。
  • b: 積集合の計算元となるマルチビットマップセット（const List）。
  • **出力**:
  • 積集合の結果を保持する List ポインタ。引数 a が直接更新される。

前提条件と不変条件

  • a および b は Bitmapset を要素として持つ List であること。
  • a はインプレースで変更されるため、呼び出し側は戻り値を使用してリストを更新する必要がある。

アルゴリズムまたは振る舞い

1. list_truncate を使用し、a の長さを b の長さに合わせる（b より長い部分は不要なため削除）。 2. forboth マクロを用いて、a と b の各要素（Bitmapset）を同時に走査する。 3. 各ペアに対して bms_int_members を呼び出し、積集合を計算する。 4. 計算結果を a の該当セルに格納する。 5. 更新された a を返す。

  • **呼び出し先**:
  • list_truncate

関連データ構造

  • List: マルチビットマップセットを構成するコンテナ。
  • Bitmapset: 各リスト要素が保持するビット集合。

失敗モードと境界条件

  • a が b より長い場合、list_truncate により超過分が切り捨てられる。
  • a が b より短い場合、forboth の仕様により短い方の長さに合わせて処理が終了する。
  • リスト内の要素が NULL である場合、lfirst_node の使用によりエラーが発生する可能性がある。
