JsonbToCStringIndent

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/jsonb.c:481

目的

JsonbContainer 構造を、インデント（整形）を有効にした状態で文字列形式に変換します。

シグネチャまたは形状

char *JsonbToCStringIndent(StringInfo out, JsonbContainer *in, int estimated_len)

入力と出力

  • **入力**:
  • out: 文字列を書き込むための StringInfo バッファ。
  • in: 変換対象の JsonbContainer へのポインタ。
  • estimated_len: 推定される出力長。
  • **出力**:
  • 変換後の文字列を指す char *。

前提条件と不変条件

  • out は初期化済みの StringInfo である必要があります。
  • in は有効な JsonbContainer を指している必要があります。

アルゴリズムまたは振る舞い

内部的に JsonbToCStringWorker を呼び出し、第4引数（indent）に true を渡すことで、インデント付きの整形出力処理を実行します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/jsonfuncs.c:4612
  • **呼び出し先**:
  • JsonbToCStringWorker (同一ファイル内)

関連データ構造

  • StringInfo: 文字列構築用のバッファ構造体。
  • JsonbContainer: jsonb 型の内部バイナリ表現。

失敗モードと境界条件

  • out が NULL の場合や、in が不正なメモリ領域を指している場合は未定義の動作となります。
  • メモリ不足時には、PostgreSQL のエラーハンドリング機構により例外が発生します。

保守担当者向けメモ

  • 本関数は JsonbToCStringWorker のラッパーです。整形ロジックの変更が必要な場合は JsonbToCStringWorker を参照してください。
