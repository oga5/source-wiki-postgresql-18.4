fb

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/test/expected/preproc-init.c:91

目的

ECPG（Embedded SQL for C）の WHENEVER 文の動作検証用スタブ関数です。SQL エラー発生時に呼び出されるアクションとして機能します。

シグネチャまたは形状

static int fb(int x);

入力と出力

  • **入力**: int x （テスト用の整数値）
  • **出力**: 引数として受け取った x をそのまま返します。

前提条件と不変条件

  • 本関数は static スコープであり、定義ファイル内でのみ参照可能です。
  • 呼び出し時に標準出力へ "in fb (%d)\n" という形式でログを出力します。

アルゴリズムまたは振る舞い

1. 受け取った整数 x をフォーマット文字列に埋め込み、printf で標準出力に書き出します。 2. 受け取った引数 x を戻り値として返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/interfaces/ecpg/test/expected/preproc-init.c:165: int g = fb ( 2 ) ; として直接呼び出されています。
  • src/interfaces/ecpg/test/expected/preproc-init.c:215: EXEC SQL WHENEVER sqlerror do fb ( 20 ) ; 文の展開先として指定されています。
  • **呼び出し先**:
  • printf (標準ライブラリ)

関連データ構造

  • 特になし。

失敗モードと境界条件

  • printf が失敗した場合の挙動は標準ライブラリに依存します。
  • 引数 x にどのような整数値が渡されても、そのまま出力および返却する単純なスタブとして動作します。
