WalSummarizerShmemSize

定義

  • 種別: function
  • 定義箇所: src/backend/postmaster/walsummarizer.c:173

目的

WAL Summarizer プロセスが共有メモリ上で使用する制御構造体 WalSummarizerData のサイズを算出する。

シグネチャまたは形状

Size WalSummarizerShmemSize(void)

入力と出力

  • 入力: なし
  • 出力: Size 型の共有メモリ所要バイト数

前提条件と不変条件

  • 共有メモリの割り当て前に呼び出される必要がある。
  • 返されるサイズは sizeof(WalSummarizerData) と等しい。

アルゴリズムまたは振る舞い

WalSummarizerData 構造体のサイズをそのまま返す。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/backend/postmaster/walsummarizer.c: WalSummarizerShmemInit 内で共有メモリ確保のために呼び出される。
  • src/backend/storage/ipc/ipci.c: CreateSharedMemoryAndSemaphores 内でシステム全体の共有メモリサイズ計算の一部として呼び出される。
  • 呼び出し先: なし

関連データ構造

  • WalSummarizerData: 本関数がサイズを算出する対象の共有メモリ制御構造体。

失敗モードと境界条件

  • 特になし（常に固定値を返す）。

保守担当者向けメモ

  • WalSummarizerData 構造体の定義が変更された場合、本関数は自動的に正しいサイズを返すため、通常は修正不要である。
  • 共有メモリのレイアウト変更時には、WalSummarizerShmemInit との整合性に注意すること。
