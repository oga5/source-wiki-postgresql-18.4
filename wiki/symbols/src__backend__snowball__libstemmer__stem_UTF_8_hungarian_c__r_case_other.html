r_case_other

目的

ハンガリー語のステミング処理において、特定の接尾辞パターン（a_6 配列で定義）を単語末尾から検出し、R1領域内であれば削除または置換を行う。

シグネチャまたは形状

static int r_case_other(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z（ステミング実行環境）
  • 出力: int（成功時は 1、失敗または処理対象外の場合は 0、エラー時は負の値）

前提条件と不変条件

  • z->c（現在のカーソル位置）が単語の末尾付近にあること。
  • 単語の末尾から3文字目以降に文字が存在すること。
  • 単語の末尾（z->p[z->c - 1]）が文字コード 108（'l'）であること。

アルゴリズムまたは振る舞い

1. z->ket を現在のカーソル位置に設定。 2. find_among_b を使用して、後方から a_6 配列内の接尾辞を検索。 3. 一致した場合、z->bra を設定し、r_R1(z) を呼び出して一致箇所がR1領域内にあるか検証。 4. 検証成功後、among_var の値に応じて以下のいずれかのアクションを実行：

  • 1: slice_del(z) で接尾辞を削除。
  • 2: slice_from_s(z, 1, s_4) で接尾辞を s_4 に置換。
  • 3: slice_from_s(z, 1, s_5) で接尾辞を s_5 に置換。

    • 5. 成功時に 1 を返して終了。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_hungarian.c および src/backend/snowball/libstemmer/stem_ISO_8859_2_hungarian.c 内のメインステミング処理フローから呼び出される。
  • 呼び出し先: find_among_b, r_R1, slice_del, slice_from_s。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_6: 接尾辞マッチングに使用される among 構造体の配列。

失敗モードと境界条件

  • 単語長が不足している場合や、末尾が 'l' でない場合は 0 を返す。
  • find_among_b で一致が見つからない場合は 0 を返す。
  • r_R1 またはスライス操作（slice_del, slice_from_s）がエラーを返した場合は、その負の値をそのまま返す。
