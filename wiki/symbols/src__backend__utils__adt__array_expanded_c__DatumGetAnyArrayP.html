DatumGetAnyArrayP

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/array_expanded.c:400

目的

与えられた Datum から、展開済み配列（ExpandedArrayHeader）または通常のフラットな配列（ArrayType）のいずれかとしてアクセス可能なポインタを取得します。

シグネチャまたは形状

AnyArrayType *DatumGetAnyArrayP(Datum d)

入力と出力

  • **入力**: Datum d（配列データを含む Datum）。
  • **出力**: AnyArrayType *（展開済み配列ヘッダー、またはデトーストされたフラットな配列へのポインタ）。

前提条件と不変条件

  • 入力が展開済み配列である場合、eah->ea_magic が EA_MAGIC と一致している必要があります。

アルゴリズムまたは振る舞い

1. 入力 d が外部展開済み配列（VARATT_IS_EXTERNAL_EXPANDED）であるかを確認します。 2. 展開済み配列であれば、DatumGetEOHP(d) を介して ExpandedArrayHeader を取得し、そのポインタを返します。 3. 展開済み配列でない場合は、PG_DETOAST_DATUM(d) を呼び出し、必要に応じてデトースト（TOAST解除）されたフラットな配列ポインタを返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/arrayfuncs.c:3205
  • src/include/utils/array.h:274（マクロ PG_GETARG_ANY_ARRAY_P 経由）
  • **呼び出し先**:
  • DatumGetPointer
  • DatumGetEOHP
  • PG_DETOAST_DATUM

関連データ構造

  • ExpandedArrayHeader: 展開済み配列のヘッダー構造体。
  • AnyArrayType: 展開済み配列とフラットな配列を抽象化する型。
  • ArrayType: 通常のフラットな配列構造体。

失敗モードと境界条件

  • Assert(eah->ea_magic == EA_MAGIC) により、展開済み配列の整合性が検証されます。このアサーションに失敗した場合、プログラムは異常終了します。

保守担当者向けメモ

  • 本関数は、配列操作において「展開済み」か「フラット」かを意識せずに共通の型として扱うための抽象化レイヤーです。
  • 戻り値の型 AnyArrayType は、呼び出し側で適切にキャストして使用する必要があります。
