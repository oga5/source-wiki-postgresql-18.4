CopyOnErrorChoice

定義

  • 種別: enum
  • 定義箇所: src/include/commands/copy.h:37

目的

COPY コマンド実行中にエラーが発生した際の挙動を制御するための列挙型です。

シグネチャまたは形状

typedef enum CopyOnErrorChoice
{
    COPY_ON_ERROR_STOP = 0,
    COPY_ON_ERROR_IGNORE,
} CopyOnErrorChoice;

入力と出力

  • **値**:
  • COPY_ON_ERROR_STOP: エラー発生時に即座に処理を停止し、エラーをスローします（デフォルト）。
  • COPY_ON_ERROR_IGNORE: エラーを無視して処理を継続します。

前提条件と不変条件

  • COPY コマンドのオプション解析時に設定され、実行状態を保持する構造体の一部として利用されます。

アルゴリズムまたは振る舞い

  • COPY 処理中のデータ行の読み込みや解析において、エラー発生時の継続可否を決定するフラグとして参照されます。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/commands/copy.c 内の関数（DefElem から値を取得する処理など）で参照されています。
  • **呼び出し先**: 本シンボル自体は列挙型定義であり、直接関数を呼び出すことはありません。

関連データ構造

  • CopyFormatOptions: COPY コマンドのオプションを保持する構造体であり、そのメンバ on_error として本型が使用されています。

失敗モードと境界条件

  • 特になし。定義された2つの値以外の値が割り当てられた場合の挙動は未定義です。

保守担当者向けメモ

  • 本シンボルは COPY コマンドの拡張機能やエラーハンドリングの仕様変更に伴い、新しい選択肢が追加される可能性があります。変更時は src/backend/commands/copy.c 側の処理ロジックも併せて確認してください。
