print_rt

定義

  • 種別: function
  • 定義箇所: src/backend/nodes/print.c:253

目的

クエリの範囲テーブル（Range Table）の内容を人間が読みやすい形式で標準出力にダンプするデバッグ用ユーティリティです。

シグネチャまたは形状

void print_rt(const List *rtable)

入力と出力

  • **入力**: rtable - RangeTblEntry ノードを要素として持つ List へのポインタ。
  • **出力**: なし（標準出力へフォーマットされたテキストを出力）。

前提条件と不変条件

  • 引数 rtable は有効な List である必要があります。
  • 各リスト要素は RangeTblEntry 構造体としてキャスト可能である必要があります。

アルゴリズムまたは振る舞い

1. 出力ヘッダー（resno, refname, relid, inFromCl）を表示します。 2. rtable リストを反復処理し、各 RangeTblEntry の rtekind を判定します。 3. rtekind に応じて、テーブル、サブクエリ、結合、関数、CTE などの種別を文字列として出力します。 4. 各エントリについて、継承フラグ（inh）および inFromCl フラグの状態を併せて出力します。

呼び出し元と呼び出し先

  • **呼び出し元**: 提供されたソース情報からは、特定の呼び出し元は特定されていません。
  • **呼び出し先**: printf（標準ライブラリ）、lfirst（リスト操作マクロ）。

関連データ構造

  • List: ノードのリスト構造。
  • RangeTblEntry: 範囲テーブルの各エントリを定義する構造体。

失敗モードと境界条件

  • rtable が空の場合、ヘッダーのみが表示されます。
  • rtekind が定義されたケースに該当しない場合、[unknown rtekind] と表示されます。

保守担当者向けメモ

  • 本関数はデバッグ専用です。本番環境のコードから呼び出さないでください。
  • RangeTblEntry の構造体定義が変更された場合、switch 文の各ケースを更新する必要があります。
