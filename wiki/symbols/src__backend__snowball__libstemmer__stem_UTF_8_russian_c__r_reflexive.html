r_reflexive

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_russian.c:470

目的

ロシア語のステミング処理において、単語末尾の再帰動詞（reflexive）接尾辞を検出し、削除する。

シグネチャまたは形状

static int r_reflexive(struct SN_env * z)

入力と出力

  • 入力: SN_env 構造体（ステミング実行環境）。
  • 出力: 成功時に 1、失敗または処理対象外の場合に 0、内部エラー発生時に負の値を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体であること。
  • find_among_b を使用するため、後方検索が可能な状態であること。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を z->ket に設定する。 2. 単語の長さが十分か（z->c - 3 > z->lb）、かつ末尾の文字が特定のバイト値（140 または 143）であるかを確認する。条件を満たさない場合は 0 を返す。 3. a_3 配列を用いて、後方から再帰動詞の接尾辞を検索する（find_among_b）。 4. マッチした場合、その位置を z->bra に設定し、slice_del を呼び出して接尾辞を削除する。 5. 削除結果を返し、成功時は 1 を返す。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_russian.c および src/backend/snowball/libstemmer/stem_KOI8_R_russian.c 内のメインステミング処理フローから呼び出される。
  • 呼び出し先: find_among_b（接尾辞検索）、slice_del（文字列削除）。

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_3: 再帰動詞の接尾辞パターンを定義する among 構造体の配列。

失敗モードと境界条件

  • 単語が短すぎる場合や、末尾の文字が対象外の場合は即座に 0 を返す。
  • slice_del が内部エラー（負の値）を返した場合、その値をそのまま伝播させる。
