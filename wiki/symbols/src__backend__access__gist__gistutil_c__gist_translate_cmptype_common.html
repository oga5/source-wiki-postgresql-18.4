gist_translate_cmptype_common

定義

  • 種別: function
  • 定義箇所: src/backend/access/gist/gistutil.c:1064

目的

GiSTインデックスにおける内部的な比較型（CompareType）を、PostgreSQLの戦略番号（StrategyNumber）へ変換します。

シグネチャまたは形状

Datum gist_translate_cmptype_common(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_GETARG_INT32(0) を介して CompareType 列挙値を受け取ります。
  • **出力**: 対応する StrategyNumber（uint16）を PG_RETURN_UINT16 で返します。

前提条件と不変条件

  • 入力値が定義済みの CompareType（COMPARE_EQ, COMPARE_LT 等）のいずれかである必要があります。

アルゴリズムまたは振る舞い

1. 引数として受け取った cmptype に対して switch 文を実行します。 2. 各 CompareType に対応する戦略番号（例: COMPARE_EQ → RTEqualStrategyNumber）を返します。 3. 定義されていない値が渡された場合、デフォルトとして InvalidStrategy を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: 提供されたソース情報からは特定の呼び出し元は特定できません。
  • **呼び出し先**: PostgreSQLの標準的なマクロ（PG_GETARG_INT32, PG_RETURN_UINT16）を使用します。

関連データ構造

  • CompareType（列挙型）
  • StrategyNumber（uint16）

失敗モードと境界条件

  • 未定義の CompareType が渡された場合、InvalidStrategy を返して安全に終了します。

保守担当者向けメモ

  • 本関数はGiSTの比較ロジックを戦略番号にマッピングする単純な変換器です。新しい比較演算子がGiSTに追加された場合は、本関数の switch 文にケースを追加する必要があります。
