AV_to_JsonbValue

定義

  • 種別: function
  • 定義箇所: contrib/jsonb_plperl/jsonb_plperl.c:130

目的

Perl の配列 (AV) を PostgreSQL の Jsonb 構造体へ変換する際の再帰的な処理の一部を担います。

シグネチャまたは形状

static JsonbValue *AV_to_JsonbValue(AV *in, JsonbParseState **jsonb_state)

入力と出力

  • **入力**:
  • in: 変換対象の Perl 配列 (AV *)。
  • jsonb_state: Jsonb 構築状態を保持する JsonbParseState へのポインタ。
  • **出力**:
  • JsonbValue *: 配列の終了を示す WJB_END_ARRAY トークンを含む JsonbValue 構造体。

前提条件と不変条件

  • dTHX マクロにより、Perl インタプリタのコンテキストが初期化されている必要があります。
  • jsonb_state は有効な JsonbParseState を指している必要があります。

アルゴリズムまたは振る舞い

1. pushJsonbValue を呼び出し、WJB_BEGIN_ARRAY トークンを jsonb_state に追加して配列の開始を宣言します。 2. av_len(in) + 1 で取得した要素数分だけループ処理を行います。 3. av_fetch を使用して各要素 (SV) を取得し、存在する場合は SV_to_JsonbValue を再帰的に呼び出して jsonb_state に追加します。 4. 最後に pushJsonbValue を呼び出し、WJB_END_ARRAY トークンを追加して配列の終了を確定させ、その結果を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/jsonb_plperl/jsonb_plperl.c 内の SV_to_JsonbValue から呼び出されることが示唆されています（ソースコードの文脈より）。
  • **呼び出し先**: SV_to_JsonbValue（再帰呼び出し）、pushJsonbValue、av_len、av_fetch。

関連データ構造

  • AV: Perl の配列型。
  • JsonbParseState: Jsonb 構築中に使用される状態管理構造体。
  • JsonbValue: Jsonb の各ノードを表現する構造体。

失敗モードと境界条件

  • av_fetch が NULL を返した場合、その要素はスキップされます。
  • Perl のデータ構造が深くネストしている場合、再帰呼び出しによりスタックオーバーフローが発生する可能性があります。

保守担当者向けメモ

  • 本関数は SV_to_JsonbValue からの再帰的な呼び出しを前提としています。
  • メモリ管理は PostgreSQL の palloc と Perl のメモリ管理が混在するため、変換プロセス全体でのリークに注意してください。
