RelationMapInitializePhase2

定義

  • 種別: function
  • 定義箇所: src/backend/utils/cache/relmapper.c:670

目的

PostgreSQLの起動プロセスにおいて、共有カタログのマッピング情報を保持するマップファイルをメモリ上に読み込む。

シグネチャまたは形状

void RelationMapInitializePhase2(void)

入力と出力

  • 入力: なし
  • 出力: なし（内部状態として共有マップをメモリにロードする）

前提条件と不変条件

  • 呼び出し時にブートストラップモード（IsBootstrapProcessingMode()）である場合、マップファイルは存在しないため何も行わない。

アルゴリズムまたは振る舞い

1. IsBootstrapProcessingMode() を確認し、真であれば即座にリターンする。 2. load_relmap_file(true, false) を呼び出し、共有マップファイルを読み込む。 3. 読み込みに失敗した場合はエラーを発生させ、プロセスを終了させる。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/utils/cache/relcache.c:4055
  • 呼び出し先: load_relmap_file（内部関数）

関連データ構造

  • RelMapFile: マップファイルの内容を保持する構造体。

失敗モードと境界条件

  • マップファイルが破損している、または読み込み権限がない場合、load_relmap_file 内で致命的なエラー（FATAL）が発生し、プロセスが停止する。

保守担当者向けメモ

  • 本関数は初期化フェーズの第2段階を担当する。第1段階および第3段階の初期化関数と連携して動作する。
  • load_relmap_file の第1引数に true を渡すことで、共有マップの読み込みを指定している。
