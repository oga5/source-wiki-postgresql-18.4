textarray_to_stringlist

定義

  • 種別: function
  • 定義箇所: src/backend/catalog/pg_subscription.c:240

目的

text 型の PostgreSQL 配列を、メモリ上の List（文字列のリスト）へ変換します。

シグネチャまたは形状

List *textarray_to_stringlist(ArrayType *textarray)

入力と出力

  • **入力**: ArrayType *textarray（変換対象のテキスト配列）
  • **出力**: List *（makeString で生成された文字列要素を含むリスト。空配列の場合は NIL）

前提条件と不変条件

  • 入力 textarray は text 型の配列である必要があります。
  • 戻り値のリストは、呼び出し元で適切にメモリ管理（解放）される必要があります。

アルゴリズムまたは振る舞い

1. deconstruct_array_builtin を使用して、入力配列を Datum 配列に分解します。 2. 配列が空（nelems == 0）の場合は NIL を返します。 3. 配列内の各要素に対して TextDatumGetCString を呼び出し、C 文字列に変換します。 4. 変換した文字列を makeString でラップし、lappend を用いてリストに追加して返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/catalog/pg_subscription.c 内の GetSubscription 関数から呼び出されています。
  • **呼び出し先**: deconstruct_array_builtin, TextDatumGetCString, makeString, lappend を呼び出します。

関連データ構造

  • ArrayType: PostgreSQL の配列データ構造。
  • List: PostgreSQL の汎用リスト構造。

失敗モードと境界条件

  • 入力が空配列の場合、NIL を返して正常終了します。
  • メモリ不足時には、PostgreSQL の標準的なエラーハンドリングによりアボートする可能性があります。

保守担当者向けメモ

  • 本関数は pg_subscription カタログ内の配列フィールドを解析するために使用されています。
  • TextDatumGetCString を使用しているため、戻り値のリストに含まれる文字列は palloc されたメモリ領域を指しています。リスト自体の解放だけでなく、リスト内の各要素のメモリ管理にも注意してください。
