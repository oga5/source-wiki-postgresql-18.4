GetDatabaseEncodingName

定義

  • 種別: function
  • 定義箇所: src/backend/utils/mb/mbutils.c:1392

目的

現在設定されているデータベースのエンコーディング名を文字列として取得する。

シグネチャまたは形状

const char *GetDatabaseEncodingName(void)

入力と出力

  • 入力: なし
  • 出力: データベースエンコーディング名を表すヌル終端文字列へのポインタ。

前提条件と不変条件

  • データベースのエンコーディング設定が初期化済みであること。
  • 戻り値の文字列は静的または永続的なメモリ領域を指しており、呼び出し側で解放してはならない。

アルゴリズムまたは振る舞い

内部的に保持されている DatabaseEncoding 構造体から name フィールドを参照し、その値をそのまま返す。

呼び出し元と呼び出し先

  • 呼び出し元（直接的な参照の例）:
  • contrib/dblink/dblink.c
  • contrib/postgres_fdw/connection.c
  • src/backend/catalog/namespace.c
  • src/backend/commands/collationcmds.c
  • src/backend/commands/variable.c
  • src/backend/replication/libpqwalreceiver/libpqwalreceiver.c
  • src/backend/snowball/dict_snowball.c
  • src/backend/utils/adt/regproc.c
  • src/backend/utils/init/postinit.c
  • src/backend/utils/mb/mbutils.c
  • 呼び出し先: なし（データ構造のフィールド参照のみ）

関連データ構造

  • DatabaseEncoding: データベースのエンコーディング情報を保持するグローバルな構造体（詳細は mbutils.c 内で管理）。

失敗モードと境界条件

  • データベースエンコーディングが未設定の状態で呼び出された場合の挙動は、DatabaseEncoding ポインタの初期化状態に依存する。

保守担当者向けメモ

  • 本関数は単なるゲッターであり、エンコーディング名の変更は SetDatabaseEncoding 等の関数を通じて行われる。
  • 戻り値は定数文字列であるため、呼び出し側での変更は避けること。
