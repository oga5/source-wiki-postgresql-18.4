TrimExtension

定義

  • 種別: function
  • 定義箇所: src/bin/pg_archivecleanup/pg_archivecleanup.c:75

目的

ファイル名の末尾に特定の拡張子が含まれている場合、その部分を削除してファイル名を正規化する。

シグネチャまたは形状

static void TrimExtension(char *filename, char *extension)

入力と出力

  • **入力**:
  • filename: 変更対象のファイル名（文字列）。関数内で直接書き換えられる。
  • extension: 除去対象の拡張子（文字列）。
  • **出力**: なし（戻り値は void）。

前提条件と不変条件

  • extension が NULL の場合、何も行わずに終了する。
  • filename の長さが extension の長さより短い場合、ファイル名は変更されない。

アルゴリズムまたは振る舞い

1. extension および filename の長さを取得する。 2. filename の末尾が extension と一致するかを strcmp で比較する。 3. 一致する場合、filename の該当位置にヌル終端文字（\0）を代入し、文字列を切り詰める。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/bin/pg_archivecleanup/pg_archivecleanup.c:116
  • src/bin/pg_archivecleanup/pg_archivecleanup.c:188
  • **呼び出し先**:
  • strlen (標準ライブラリ)
  • strcmp (標準ライブラリ)

関連データ構造

  • 特になし。

失敗モードと境界条件

  • filename が extension より短い場合、または一致しない場合は何も変更されない。
  • extension が NULL の場合は安全に無視される。

保守担当者向けメモ

  • 本関数は filename を破壊的に変更する。呼び出し側で元のファイル名が必要な場合は、事前にコピーを作成しておく必要がある。
