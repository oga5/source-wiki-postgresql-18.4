PSQL_ECHO_HIDDEN

定義

  • 種別: enum
  • 定義箇所: src/bin/psql/settings.h:49

目的

psql における「隠しコマンド（hidden commands）」の表示設定を制御するための列挙型です。

シグネチャまたは形状

typedef enum
{
	PSQL_ECHO_HIDDEN_OFF,
	PSQL_ECHO_HIDDEN_ON,
	PSQL_ECHO_HIDDEN_NOEXEC,
} PSQL_ECHO_HIDDEN;

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: PSQL_ECHO_HIDDEN_OFF（無効）、PSQL_ECHO_HIDDEN_ON（有効）、PSQL_ECHO_HIDDEN_NOEXEC（実行せず表示のみ）のいずれかの状態値。

前提条件と不変条件

  • PsqlSettings 構造体内の echo_hidden フィールドの型として使用されます。

アルゴリズムまたは振る舞い

  • PSQL_ECHO_HIDDEN_OFF: 隠しコマンドの表示を行いません。
  • PSQL_ECHO_HIDDEN_ON: 隠しコマンドを標準出力に表示します。
  • PSQL_ECHO_HIDDEN_NOEXEC: 隠しコマンドを表示しますが、実際の実行は行いません。

呼び出し元と呼び出し先

  • 呼び出し元: src/bin/psql/settings.h:177 にて PsqlSettings 構造体のメンバ echo_hidden として参照されています。
  • 呼び出し先: 提供されたソース情報からは、具体的な関数呼び出しや参照先は特定できません。

関連データ構造

  • PsqlSettings: psql のグローバル設定を保持する構造体であり、本型をメンバとして保持します。

失敗モードと境界条件

  • 該当なし（列挙型定義のため）。

保守担当者向けメモ

  • PsqlSettings 構造体を通じて設定値が管理されます。設定値の変更が必要な場合は、variables.c 等で実装されているフック関数経由で更新されているか確認してください。
