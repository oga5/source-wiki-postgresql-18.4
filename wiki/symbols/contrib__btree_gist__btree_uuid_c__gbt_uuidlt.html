gbt_uuidlt

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_uuid.c:60

目的

2つの pg_uuid_t 値を比較し、最初の値が2番目の値より小さいかどうかを判定します。

シグネチャまたは形状

static bool gbt_uuidlt(const void *a, const void *b, FmgrInfo *flinfo)

入力と出力

  • **入力**:
  • a: 比較対象の UUID（pg_uuid_t へのポインタとしてキャストして使用）。
  • b: 比較対象の UUID（pg_uuid_t へのポインタとしてキャストして使用）。
  • flinfo: 関数管理情報（本関数内では未使用）。
  • **出力**:
  • bool: a < b であれば true、そうでなければ false を返します。

前提条件と不変条件

  • 入力ポインタ a および b は、有効な pg_uuid_t 構造体を指している必要があります。
  • UUID の比較は、uuid_internal_cmp を介したバイト列の比較に基づきます。

アルゴリズムまたは振る舞い

1. a と b を pg_uuid_t 型のポインタにキャストします。 2. uuid_internal_cmp を呼び出し、その戻り値が 0 未満（a < b）であるかを評価します。 3. 評価結果をブール値として返します。

呼び出し元と呼び出し先

  • **呼び出し先**: uuid_internal_cmp
  • **呼び出し元**:
  • contrib/btree_gist/btree_uuid.c 内の静的配列（GiST 演算子クラスの定義の一部として参照されています）。

関連データ構造

  • pg_uuid_t: PostgreSQL の UUID 型を表現する構造体。

失敗モードと境界条件

  • 入力ポインタが NULL の場合の動作は定義されていません。
  • uuid_internal_cmp の仕様に依存し、バイト列としての辞書順比較が行われます。

保守担当者向けメモ

  • 本関数は btree_gist における UUID 型の比較演算子クラスの一部です。
  • flinfo 引数は現在の実装では使用されていませんが、GiST のインターフェース規約に従い保持されています。
