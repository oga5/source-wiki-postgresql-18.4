PGTYPESdate_dayofweek

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/pgtypeslib/datetime.c:137

目的

指定された日付（date型）が週の何曜日であるかを計算し、整数値として返します。

シグネチャまたは形状

int PGTYPESdate_dayofweek(date dDate)

入力と出力

  • **入力**: dDate (date型) - 2000年1月1日を基準としたオフセット値。
  • **出力**: int型 - 曜日を表す整数（0: 日曜日, 1: 月曜日, ..., 6: 土曜日）。

前提条件と不変条件

  • 入力される dDate は、PostgreSQLの内部的な日付表現（2000年1月1日を基準とする整数値）である必要があります。

アルゴリズムまたは振る舞い

1. date2j(2000, 1, 1) を用いて基準日からのユリウス日オフセットを算出します。 2. 入力された dDate に基準日のユリウス日と1を加算します。 3. その合計値を7で割った余りを計算し、曜日を特定します。

  • 0: 日曜日
  • 1: 月曜日
  • 2: 火曜日
  • 3: 水曜日
  • 4: 木曜日
  • 5: 金曜日
  • 6: 土曜日

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/interfaces/ecpg/compatlib/informix.c
  • src/interfaces/ecpg/pgtypeslib/datetime.c
  • src/interfaces/ecpg/pgtypeslib/timestamp.c
  • src/interfaces/ecpg/test/expected/pgtypeslib-dt_test.c
  • **呼び出し先**:
  • date2j（日付をユリウス日に変換する関数）

関連データ構造

  • date: ECPG内で使用される日付型（内部的には整数値）。

失敗モードと境界条件

  • 入力値が有効な日付範囲外である場合の挙動については、本ソースコード内では明示的なエラーチェックは行われていません。

保守担当者向けメモ

  • 本関数は2000年1月1日を基準とした計算を行っています。日付計算のロジックを変更する際は、dt_common.c 等で定義されている基準値との整合性に注意してください。
