pg_category_range

定義

目的

Unicodeコードポイントの範囲と、それに対応する一般的なカテゴリ（General Category）をマッピングするためのデータ構造を提供します。

シグネチャまたは形状

typedef struct
{
	uint32		first;			/* Unicode codepoint */
	uint32		last;			/* Unicode codepoint */
	uint8		category;		/* General Category */
} pg_category_range;

入力と出力

本シンボルはデータ構造の定義であり、関数ではないため直接的な入力・出力はありません。この構造体のインスタンスは、Unicodeカテゴリ判定のための静的テーブル（unicode_categories 等）の要素として使用されます。

前提条件と不変条件

  • first は範囲の開始コードポイント、last は終了コードポイントを表します。
  • first <= last であることが期待されます。
  • category は common/unicode_category.h で定義されたカテゴリ定数に対応します。

アルゴリズムまたは振る舞い

本構造体自体はロジックを持ちません。この構造体の配列に対して、外部の処理関数が二分探索を行うことで、任意のコードポイントがどのカテゴリに属するかを高速に判定します。

呼び出し元と呼び出し先

  • 呼び出し元: src/include/common/unicode_category_table.h 内の静的テーブル unicode_categories が本構造体を使用して定義されています。
  • 呼び出し先: 情報なし。

関連データ構造

  • unicode_categories: 本構造体の配列として定義された、全Unicodeコードポイントをカバーする範囲テーブル。
  • common/unicode_category.h: category フィールドで使用されるカテゴリ定数の定義元。

失敗モードと境界条件

  • 構造体定義自体に失敗モードはありません。
  • 範囲テーブルの構築において first > last となるような不正なデータが格納された場合、二分探索アルゴリズムが正しく機能しない可能性があります。
