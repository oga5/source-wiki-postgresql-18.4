px_gen_salt

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/px-crypt.c:155

目的

指定されたアルゴリズム名に基づき、パスワードハッシュ用のランダムなソルト文字列を生成する。

シグネチャまたは形状

int px_gen_salt(const char *salt_type, char *buf, int rounds);

入力と出力

  • **入力**:
  • salt_type: 生成するソルトのアルゴリズム名（例: "des", "md5", "bf"）。
  • buf: 生成されたソルト文字列を格納するバッファ。
  • rounds: アルゴリズムの反復回数（0の場合はデフォルト値を使用）。
  • **出力**:
  • 成功時: 生成されたソルト文字列の長さ（int）。
  • 失敗時: 負の定数（PXE_UNKNOWN_SALT_ALGO, PXE_BAD_SALT_ROUNDS, PXE_NO_RANDOM）。

前提条件と不変条件

  • CheckBuiltinCryptoMode() により、暗号化モードが安全であることが保証されている必要がある。
  • buf は PX_MAX_SALT_LEN 以上のサイズを持つ必要がある。
  • rounds が指定された場合、そのアルゴリズムの min_rounds から max_rounds の範囲内である必要がある。

アルゴリズムまたは振る舞い

1. gen_list テーブルから salt_type に一致するアルゴリズムを検索する。 2. rounds が 0 の場合、アルゴリズムのデフォルト値で初期化する。 3. pg_strong_random を使用して、アルゴリズムが必要とするバイト数分の乱数を生成する。 4. 対応するアルゴリズムの生成関数（g->gen）を呼び出し、buf にソルトを書き込む。 5. 一時的な乱数バッファを px_memset でゼロクリアする。 6. 生成された文字列の長さを返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/pgcrypto/pgcrypto.c (行175, 198)
  • **呼び出し先**:
  • CheckBuiltinCryptoMode()
  • pg_strcasecmp()
  • pg_strong_random()
  • g->gen() (アルゴリズム固有の生成関数)
  • px_memset()
  • strlen()

関連データ構造

  • struct generator: アルゴリズム名、ラウンド数の制限、生成関数ポインタ、入力長を保持する構造体。
  • gen_list: px-crypt.c 内で定義された generator 構造体の静的配列。

失敗モードと境界条件

  • salt_type が gen_list に存在しない場合: PXE_UNKNOWN_SALT_ALGO を返す。
  • rounds が許容範囲外の場合: PXE_BAD_SALT_ROUNDS を返す。
  • 乱数生成に失敗した場合: PXE_NO_RANDOM を返す。
  • アルゴリズム固有の生成関数が失敗した場合: PXE_BAD_SALT_ROUNDS を返す。

保守担当者向けメモ

  • gen_list のエントリを追加・変更する際は、min_rounds および max_rounds の妥当性を確認すること。
  • px_memset による乱数バッファのクリアは、セキュリティ上の機密保持のために必須である。
