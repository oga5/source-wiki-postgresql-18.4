date_test_strdate

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/test/expected/compat_informix-rfmtdate.c:26

目的

Informix 互換ライブラリの文字列から日付への変換機能（rstrdate）および日付から文字列への変換機能（rdatestr）の動作を検証するテスト関数です。

シグネチャまたは形状

static void date_test_strdate(const char *input)

入力と出力

  • **入力**: input (日付を表す文字列)
  • **出力**: なし（標準出力へ変換結果またはエラーコードを出力）

前提条件と不変条件

  • input は rstrdate が解析可能な形式である必要があります。
  • 内部静的変数 i がテストの回数カウントとして使用されます。

アルゴリズムまたは振る舞い

1. rstrdate(input, &d) を呼び出し、入力文字列を内部日付型 d に変換します。 2. 変換が成功（戻り値 r == 0）した場合:

  • rdatestr(d, dbuf) を呼び出し、内部日付を文字列 dbuf に再変換します。
  • 再変換も成功（戻り値 q == 0）した場合、変換結果を標準出力に表示します。

    • 3. 変換または再変換が失敗した場合:

  • check_return(r) を呼び出し、エラーコードを標準出力へ出力します。

呼び出し元と呼び出し先

  • **呼び出し元**: main 関数（compat_informix-rfmtdate.c:111, 112 など）から直接呼び出されます。
  • **呼び出し先**:
  • rstrdate (Informix 互換ライブラリ)
  • rdatestr (Informix 互換ライブラリ)
  • check_return (同ファイル内のヘルパー関数)
  • printf (標準ライブラリ)

関連データ構造

  • date: 内部日付表現型。
  • dbuf: 変換後の文字列を格納する char[11] 型のバッファ。

失敗モードと境界条件

  • rstrdate が 0 以外の値を返した場合、check_return を通じてエラーが報告されます。
  • rdatestr が 0 以外の値を返した場合、エラーコードのみが出力され、日付文字列の表示はスキップされます。

保守担当者向けメモ

  • 本関数は回帰テスト用であり、テストケースの追加は main 関数から本関数を呼び出す形で行ってください。
  • dbuf のサイズは 11 バイトであり、日付文字列の長さに依存するため、フォーマット変更時にはバッファオーバーフローに注意してください。
