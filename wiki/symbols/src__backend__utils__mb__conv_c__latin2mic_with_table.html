latin2mic_with_table

定義

  • 種別: function
  • 定義箇所: src/backend/utils/mb/conv.c:193

目的

単一バイトの文字セットから Mule Internal Code (MIC) への変換を、指定された変換テーブルを用いて実行します。

シグネチャまたは形状

int latin2mic_with_table(const unsigned char *l,
                         unsigned char *p,
                         int len,
                         int lc,
                         int encoding,
                         const unsigned char *tab,
                         bool noError)

入力と出力

  • **入力**:
  • l: 変換元文字列（単一バイトエンコーディング）。
  • p: 変換先バッファ。
  • len: 変換対象のバイト長。
  • lc: MICの文字セット識別子。
  • encoding: 現在のエンコーディング種別。
  • tab: 変換テーブル（tab[c1 - HIGHBIT] で変換先を検索）。
  • noError: 変換失敗時にエラーを報告せず中断するかどうか。
  • **出力**:
  • 戻り値: 処理されたバイト数。
  • p: 変換結果が格納され、末尾に \0 が付与されます。

前提条件と不変条件

  • tab は、高ビットがセットされた文字（c1 >= 0x80）をインデックスとして変換先を保持している必要があります。
  • p は、変換後の文字列を格納するのに十分な領域を確保している必要があります。

アルゴリズムまたは振る舞い

1. 入力文字列をバイト単位で走査します。 2. c1 が 0 の場合、noError が真であれば終了し、偽であればエラーを報告します。 3. c1 の高ビットがセットされていない場合、そのまま出力バッファへコピーします。 4. 高ビットがセットされている場合、tab を参照して変換先 c2 を取得します。

  • c2 が存在する場合、lc と c2 を出力バッファへ書き込みます。
  • c2 が存在しない場合、noError が真であれば終了し、偽であれば変換不可エラーを報告します。

    • 5. 最後に p の末尾にヌル文字を書き込み、処理したバイト数を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/mb/conversion_procs/cyrillic_and_mic/cyrillic_and_mic.c
  • src/backend/utils/mb/conversion_procs/latin2_and_win1250/latin2_and_win1250.c
  • **呼び出し先**:
  • report_invalid_encoding（エラー時）
  • report_untranslatable_char（変換失敗時）

関連データ構造

  • pg_mb_radix_tree（本関数はテーブル引きを使用しますが、モジュール全体ではこの構造体が変換の基盤となります）

失敗モードと境界条件

  • **不正なエンコーディング**: c1 == 0 が検出された場合、noError が偽であれば report_invalid_encoding を呼び出します。
  • **変換不可文字**: tab による検索結果が 0 の場合、noError が偽であれば report_untranslatable_char を呼び出します。
  • **バッファ境界**: len に基づいてループし、処理終了後に p の末尾に \0 を書き込みます。

保守担当者向けメモ

  • 本関数は単一バイト文字セットの変換に特化しています。
