comparecost

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/tsquery_gist.c:157

目的

GiST インデックスの分割処理において、コスト計算結果（SPLITCOST）に基づき、エントリをソートするための比較関数を提供します。

シグネチャまたは形状

static int comparecost(const void *a, const void *b)

入力と出力

  • **入力**: 比較対象となる2つの SPLITCOST 構造体へのポインタ（const void *）。
  • **出力**: a のコストが b より小さい場合は負の値、等しい場合は 0、大きい場合は正の値を返します。

前提条件と不変条件

  • 入力ポインタ a および b は、SPLITCOST 構造体へのキャストが可能である必要があります。
  • SPLITCOST 構造体には cost メンバが含まれている必要があります。

アルゴリズムまたは振る舞い

1. 引数 a と b を SPLITCOST 型のポインタにキャストします。 2. それぞれの cost メンバを pg_cmp_s32 関数に渡し、その結果を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: qsort 関数の比較関数として、tsquery_gist.c および他の GiST 実装ファイル（contrib/hstore/hstore_gist.c, contrib/intarray/_int_gist.c, contrib/pg_trgm/trgm_gist.c 等）から呼び出されます。
  • **呼び出し先**: pg_cmp_s32 を呼び出します。

関連データ構造

  • SPLITCOST: GiST インデックスの分割コストを保持する構造体。

失敗モードと境界条件

  • 入力ポインタが NULL の場合、未定義の動作となります。

保守担当者向けメモ

  • 本関数は複数の contrib モジュールや他の GiST 実装と共通のロジックとして利用されています。変更を加える際は、これら関連ファイルへの影響を考慮してください。
