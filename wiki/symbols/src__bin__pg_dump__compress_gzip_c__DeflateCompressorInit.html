DeflateCompressorInit

定義

  • 種別: function
  • 定義箇所: src/bin/pg_dump/compress_gzip.c:54

目的

CompressorState 構造体に対し、GZIP 圧縮のための内部状態（GzipCompressorState）を初期化し、zlib の deflate ストリームをセットアップする。

シグネチャまたは形状

static void DeflateCompressorInit(CompressorState *cs)

入力と出力

  • **入力**: CompressorState *cs（圧縮状態を保持する構造体。compression_spec.level を参照する）
  • **出力**: なし（副作用として cs->private_data に初期化済みの GzipCompressorState を格納する）

前提条件と不変条件

  • cs->compression_spec.level は 0 以外であること（Assert により検証）。
  • zlib ライブラリが利用可能であること。

アルゴリズムまたは振る舞い

1. GzipCompressorState 構造体をメモリ確保し、z_stream を初期化する。 2. zlib のメモリ管理関数（zalloc, zfree, opaque）を Z_NULL に設定する。 3. 出力バッファ outbuf を DEFAULT_IO_BUFFER_SIZE + 1 バイト確保する（末尾のゼロバイト付与用）。 4. deflateInit を呼び出し、指定された圧縮レベルでストリームを初期化する。失敗時は pg_fatal で終了する。 5. z_stream の出力先ポインタとサイズを初期化する。 6. 最終的に cs->private_data に生成した状態を保持させる。

呼び出し元と呼び出し先

  • **呼び出し元**: src/bin/pg_dump/compress_gzip.c 内の関数（直接の呼び出し箇所は src/bin/pg_dump/compress_gzip.c:254 等）。
  • **呼び出し先**: pg_malloc0, pg_malloc, deflateInit, pg_fatal。

関連データ構造

  • CompressorState: 圧縮サブシステムの基盤となる状態構造体。
  • GzipCompressorState: GZIP 固有の圧縮状態を保持する構造体。
  • z_stream: zlib が使用する圧縮ストリーム構造体。

失敗モードと境界条件

  • pg_malloc によるメモリ確保失敗時はプロセスが終了する。
  • deflateInit が Z_OK 以外の値を返した場合、pg_fatal によりエラーメッセージを出力して異常終了する。

保守担当者向けメモ

  • 本関数は zlib に依存しているため、ビルド時に HAVE_LIBZ が定義されている環境でのみ有効に機能する。
  • outsize に 1 バイト加算しているのは、後続の処理でバッファ末尾にゼロバイトを書き込むケースがあるためである。
