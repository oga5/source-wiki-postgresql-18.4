system_time_initsamplescan

定義

  • 種別: function
  • 定義箇所: contrib/tsm_system_time/tsm_system_time.c:179

目的

TABLESAMPLE SYSTEM_TIME メソッドにおけるスキャン状態の初期化を行います。

シグネチャまたは形状

static void system_time_initsamplescan(SampleScanState *node, int eflags)

入力と出力

  • **入力**:
  • node: SampleScanState 構造体へのポインタ。
  • eflags: スキャンの実行フラグ。
  • **出力**: なし（副作用として node->tsm_state を割り当てます）。

前提条件と不変条件

  • node->tsm_state は初期化前である必要があります。
  • 割り当てられた SystemTimeSamplerData 構造体の step フィールドは、この関数終了時点で 0 に初期化されます。

アルゴリズムまたは振る舞い

1. palloc0 を使用して SystemTimeSamplerData 構造体分のメモリをゼロクリアして確保します。 2. 確保したメモリ領域を node->tsm_state に格納します。

呼び出し元と呼び出し先

  • **呼び出し元**: tsm_system_time_handler 内で TsmRoutine 構造体の InitSampleScan フィールドに割り当てられ、PostgreSQL のテーブルサンプリング実行エンジンから呼び出されます。
  • **呼び出し先**: palloc0（メモリ割り当て関数）。

関連データ構造

  • SampleScanState: スキャン全体の実行状態を保持する構造体。
  • SystemTimeSamplerData: tsm_system_time メソッド固有のサンプリング状態を保持する構造体。

失敗モードと境界条件

  • メモリ不足時には palloc0 によりエラーが発生します。

保守担当者向けメモ

  • 本関数は tsm_system_time_handler を通じて登録されるコールバックです。
  • tsm_state->step が 0 で初期化される点は、後続の system_time_nextsampleblock におけるブロック選択ロジックの前提となります。
