get_first_loaded_module

定義

  • 種別: function
  • 定義箇所: src/backend/utils/fmgr/dfmgr.c:424

目的

現在ロードされている動的ライブラリのリスト（DynamicFileList）の先頭要素へのポインタを取得します。

シグネチャまたは形状

DynamicFileList *get_first_loaded_module(void)

入力と出力

  • 入力: なし
  • 出力: DynamicFileList 構造体へのポインタ。リストが空の場合は NULL を返します。

前提条件と不変条件

  • file_list は dfmgr.c 内で管理される静的な連結リストです。
  • 本関数は、このリストのヘッドを直接参照します。

アルゴリズムまたは振る舞い

1. モジュール内で保持されているグローバル変数 file_list を参照します。 2. その値を呼び出し元へ返します。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/backend/commands/extension.c:2983 (リストの走査に使用)
  • 呼び出し先:
  • なし（単純なゲッター関数です）

関連データ構造

  • DynamicFileList: ロード済みライブラリを管理する連結リスト構造体。

失敗モードと境界条件

  • リストが初期化されていない、またはライブラリが一つもロードされていない場合、NULL を返します。

保守担当者向けメモ

  • 本関数は file_list への直接的なアクセスを提供します。リストの内容を走査する際は、呼び出し元で DynamicFileList の構造に従って次要素へ辿る必要があります。
