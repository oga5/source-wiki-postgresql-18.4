pg_get_client_encoding_name

定義

  • 種別: function
  • 定義箇所: src/backend/utils/mb/mbutils.c:353

目的

現在設定されているクライアントエンコーディングの名称（文字列）を取得します。

シグネチャまたは形状

const char *pg_get_client_encoding_name(void)

入力と出力

  • 入力: なし
  • 出力: 現在のクライアントエンコーディング名を表す const char * 型の文字列を返します。

前提条件と不変条件

  • ClientEncoding グローバル変数が初期化されている必要があります。
  • 返される文字列は、PostgreSQLのエンコーディング管理システムによって保持されている静的または永続的なメモリ領域を指します。

アルゴリズムまたは振る舞い

1. グローバル変数 ClientEncoding が指す構造体へアクセスします。 2. その構造体内の name メンバを直接参照し、その値を呼び出し元へ返します。

呼び出し元と呼び出し先

  • 呼び出し元: src/include/mb/pg_wchar.h にて外部公開されており、システム内の各所から参照される可能性があります。
  • 呼び出し先: 本関数は内部で他の関数を呼び出しません。

関連データ構造

  • ClientEncoding: 現在のクライアントエンコーディング情報を保持する内部構造体（mbutils.c 内で管理）。

失敗モードと境界条件

  • ClientEncoding が初期化されていない状態で呼び出された場合、ヌルポインタ参照によるクラッシュの可能性があります。

保守担当者向けメモ

  • 本関数は非常に軽量なゲッターです。エンコーディング名の変更が必要な場合は、SetClientEncoding() を通じて ClientEncoding 構造体の状態を更新してください。
