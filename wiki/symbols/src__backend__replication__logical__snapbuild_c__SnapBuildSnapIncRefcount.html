SnapBuildSnapIncRefcount

定義

  • 種別: function
  • 定義箇所: src/backend/replication/logical/snapbuild.c:315

目的

指定されたスナップショット（Snapshot）の参照カウントをインクリメントし、スナップショットの生存期間を延長します。

シグネチャまたは形状

static void SnapBuildSnapIncRefcount(Snapshot snap)

入力と出力

  • **入力**: snap (Snapshot型) - 参照カウントを増加させる対象のスナップショット構造体へのポインタ。
  • **出力**: なし（副作用として snap->active_count が更新されます）。

前提条件と不変条件

  • snap は有効な Snapshot 構造体を指している必要があります。
  • snap->active_count は、この関数によってインクリメントされる整数値です。

アルゴリズムまたは振る舞い

1. 引数として渡された snap 構造体の active_count メンバを 1 増加させます。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/replication/logical/snapbuild.c 内の以下の箇所から呼び出されています。
  • 588, 668, 675, 784, 1085, 1091, 1909 行目（主に builder->snapshot を引数として使用）。
  • **呼び出し先**: なし。

関連データ構造

  • Snapshot: utils/snapshot.h で定義される、MVCCスナップショットを保持する構造体。active_count メンバを含みます。

失敗モードと境界条件

  • snap が NULL の場合、セグメンテーション違反が発生する可能性があります（ソースコード上でのチェックは行われていません）。

保守担当者向けメモ

  • 本関数は SnapBuildSnapDecRefcount と対になって使用されます。スナップショットのライフサイクル管理において、参照カウントが正しく増減されているか確認してください。
