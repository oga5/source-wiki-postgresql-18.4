目的

Snowball ステミングアルゴリズムにおいて、単語内の「R1」領域の境界条件を判定します。

シグネチャまたは形状

static int r_R1(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z （Snowball の実行環境構造体）
  • 出力: int 型。条件を満たす場合は 1、満たさない場合は 0 を返します。

前提条件と不変条件

  • z->I[1] には、事前に r_mark_regions 等で計算された R1 領域の開始位置が格納されている必要があります。
  • z->c は現在のカーソル位置を指します。

アルゴリズムまたは振る舞い

現在のカーソル位置 z->c が、R1 領域の開始位置 z->I[1] 以降にあるかどうかを比較します。

  • z->I[1] <= z->c が真であれば 1 を返し、偽であれば 0 を返します。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_ISO_8859_1_basque.c、catalan.c、dutch.c 等の各言語ステマーから、接尾辞除去の可否判定として呼び出されます。
  • 呼び出し先: なし（純粋な比較演算のみを実行）。

関連データ構造

  • struct SN_env: Snowball ランタイムの共通環境構造体。I 配列（整数型パラメータ）および c（カーソル）を保持します。

失敗モードと境界条件

  • z が NULL の場合、デリファレンスによりクラッシュする可能性があります（呼び出し元での保証に依存）。
  • z->I[1] が適切に初期化されていない場合、誤った判定結果を返します。
