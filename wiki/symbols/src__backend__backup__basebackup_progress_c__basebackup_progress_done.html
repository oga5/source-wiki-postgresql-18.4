basebackup_progress_done

定義

  • 種別: function
  • 定義箇所: src/backend/backup/basebackup_progress.c:238

目的

ベースバックアップの進捗追跡を終了し、統計システムから当該コマンドの情報を削除する。

シグネチャまたは形状

void basebackup_progress_done(void)

入力と出力

  • 入力: なし
  • 出力: なし（副作用として統計状態を更新）

前提条件と不変条件

  • pgstat_progress_start_command によって進捗追跡が既に開始されている必要がある。

アルゴリズムまたは振る舞い

1. pgstat_progress_end_command() を呼び出し、現在のバックアッププロセスに関連付けられた進捗統計の記録を終了する。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/backup/basebackup.c:678
  • 呼び出し先: pgstat_progress_end_command (PostgreSQL 統計サブシステム)

関連データ構造

  • pgstat (統計サブシステム)

失敗モードと境界条件

  • 追跡が開始されていない状態で呼び出した場合の挙動は pgstat_progress_end_command の実装に依存する。

保守担当者向けメモ

  • 本関数はバックアップ処理の最終段階で呼び出される。
  • bbsink インターフェースの一部ではなく、basebackup.c から直接呼び出される進捗管理用のユーティリティ関数である。
