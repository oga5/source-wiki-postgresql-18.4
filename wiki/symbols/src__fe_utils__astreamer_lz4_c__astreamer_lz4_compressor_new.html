astreamer_lz4_compressor_new

定義

  • 種別: function
  • 定義箇所: src/fe_utils/astreamer_lz4.c:71

目的

LZ4圧縮を行うための astreamer インスタンスを生成し、初期化します。

シグネチャまたは形状

astreamer *astreamer_lz4_compressor_new(astreamer *next, pg_compress_specification *compress)

入力と出力

  • **入力**:
  • next: 圧縮後のデータを受け取る後続の astreamer インターフェース。
  • compress: 圧縮レベルを含む圧縮設定情報。
  • **出力**: 初期化された astreamer 構造体へのポインタ。

前提条件と不変条件

  • next は NULL であってはならない。
  • USE_LZ4 が定義されている環境でビルドされている必要がある。定義されていない場合、本関数はエラーを報告し終了する。

アルゴリズムまたは振る舞い

1. astreamer_lz4_frame 構造体用のメモリを割り当てます。 2. bbs_ops を astreamer_lz4_compressor_ops に設定し、bbs_next に next を格納します。 3. bbs_buffer を初期化し、header_written フラグを false に設定します。 4. LZ4F_preferences_t を設定し、ブロックサイズを LZ4F_max256KB に指定します。 5. LZ4F_createCompressionContext を呼び出し、LZ4圧縮コンテキストを作成します。作成に失敗した場合は pg_fatal でエラーを報告します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/bin/pg_basebackup/pg_basebackup.c:1205
  • **呼び出し先**:
  • palloc0 (メモリ確保)
  • initStringInfo (バッファ初期化)
  • LZ4F_createCompressionContext (LZ4ライブラリ関数)
  • pg_fatal (エラー報告)

関連データ構造

  • astreamer_lz4_frame: 本関数が生成する内部状態構造体。
  • astreamer: 基底ストリーマーインターフェース。
  • LZ4F_preferences_t: LZ4圧縮設定を保持する構造体。

失敗モードと境界条件

  • next が NULL の場合、Assert により異常終了します。
  • LZ4F_createCompressionContext がエラーを返した場合、pg_fatal によりプロセスが終了します。
  • USE_LZ4 が未定義の場合、実行時に pg_fatal でエラーとなります。

保守担当者向けメモ

  • 本関数は astreamer パイプラインの初期化ステップを担当します。
  • 圧縮レベルの変更やブロックサイズの調整が必要な場合は、prefs の設定箇所を確認してください。
