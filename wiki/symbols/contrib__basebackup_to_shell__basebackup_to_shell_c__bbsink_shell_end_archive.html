bbsink_shell_end_archive

定義

  • 種別: function
  • 定義箇所: contrib/basebackup_to_shell/basebackup_to_shell.c:333

目的

バックアップアーカイブの終了処理を行い、外部シェルコマンドの実行を完了させるとともに、後続のシンクへ終了イベントを伝播させる。

シグネチャまたは形状

static void bbsink_shell_end_archive(bbsink *sink)

入力と出力

  • **入力**: sink (bbsink *) - 現在のバックアップシンクインスタンス。
  • **出力**: なし（void）。

前提条件と不変条件

  • sink は bbsink_shell 型の構造体としてキャスト可能であること。
  • shell_finish_command が内部で呼び出され、外部プロセスとのパイプが適切にクローズされること。

アルゴリズムまたは振る舞い

1. 引数 sink を bbsink_shell * 型の mysink にキャストする。 2. shell_finish_command(mysink) を呼び出し、実行中の外部コマンドの終了待機およびパイプのクローズを行う。 3. bbsink_forward_end_archive(sink) を呼び出し、バックアップストリームの終了イベントを次のシンクへ転送する。

呼び出し元と呼び出し先

  • **呼び出し元**: bbsink フレームワークのバックアップ終了処理フローから呼び出される。
  • **呼び出し先**:
  • shell_finish_command: 外部コマンドの終了処理を実行。
  • bbsink_forward_end_archive: 次のシンクへのイベント転送。

関連データ構造

  • bbsink: バックアップターゲットの基底構造体。
  • bbsink_shell: bbsink を継承した、シェルコマンド実行状態を保持する拡張構造体。

失敗モードと境界条件

  • shell_finish_command 内で外部コマンドが非ゼロの終了コードを返した場合、バックアップ処理全体がエラーとして扱われる。

保守担当者向けメモ

  • 本関数は bbsink_shell_ops 構造体を通じて関数ポインタとして登録されている。
  • 外部コマンドの終了ステータス検証は shell_finish_command に委譲されているため、エラーハンドリングの変更はそちらを確認すること。
