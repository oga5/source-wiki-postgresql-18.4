multipleOfPowerOf5

定義

  • 種別: function
  • 定義箇所: src/common/f2s.c:101

目的

指定された数値が、5の指定されたべき乗で割り切れるかどうかを判定します。

シグネチャまたは形状

static inline bool multipleOfPowerOf5(const uint32 value, const uint32 p)

入力と出力

  • **入力**:
  • value: 判定対象の符号なし32ビット整数。
  • p: 5の指数（5の何乗で割り切れるかを判定する基準）。
  • **出力**:
  • bool: value が $5^p$ で割り切れる場合に true、それ以外の場合に false を返します。

前提条件と不変条件

  • pow5Factor 関数が正しく定義されており、数値に含まれる5の因数の数を正確に算出できる必要があります。

アルゴリズムまたは振る舞い

1. pow5Factor(value) を呼び出し、value を素因数分解した際の5の指数を取得します。 2. 取得した指数が引数 p 以上であるかを比較し、その結果を返します。

関連データ構造

  • なし

失敗モードと境界条件

  • value が0の場合の挙動は pow5Factor の実装に依存します。
  • p が非常に大きい場合、pow5Factor の戻り値との比較により常に false となります。

保守担当者向けメモ

  • 本関数は static inline で定義されており、Ryuアルゴリズムの変換処理における末尾ゼロの判定に使用されます。
  • d2s.c でも同様の名称の関数が使用されていますが、引数の型（uint32 vs uint64）が異なる点に注意してください。
