List

定義

  • 種別: typedef
  • 定義箇所: src/backend/utils/adt/jsonb_gin.c:141

目的

PostgreSQLの汎用的なリスト構造を指す型定義です。jsonb_gin.c 内では、主に JsonPathGinExtractNodesFunc のシグネチャにおいて、ノードのリストを保持・操作するために使用されます。

シグネチャまたは形状

typedef List *(*JsonPathGinExtractNodesFunc) (JsonPathGinContext *cxt,
                                              JsonPathGinPath path,
                                              JsonbValue *scalar,
                                              List *nodes);

※ List 自体の構造体定義は本ファイル外（PostgreSQLのコアライブラリ）に依存します。

入力と出力

  • JsonPathGinExtractNodesFunc において、List *nodes は入力として既存のノードリストを受け取り、戻り値として更新または追加された List * を返します。

前提条件と不変条件

  • PostgreSQLの標準的なリスト管理機能（pg_list.h等）に準拠している必要があります。

アルゴリズムまたは振る舞い

  • JsonPathGinExtractNodesFunc の実装において、nodes リストに対して lappend 等の操作を行い、JSONPathの抽出結果を蓄積するために利用されます。

呼び出し元と呼び出し先

  • **呼び出し元**: jsonb_gin.c 内の JsonPathGinExtractNodesFunc 型を使用する関数群。
  • **呼び出し先**: PostgreSQLのリスト操作関数（lappend 等）。
  • 注: 提供されたソース excerpt には、具体的な呼び出し元・呼び出し先の関数名は含まれていません。

関連データ構造

  • JsonPathGinContext
  • JsonPathGinPath
  • JsonbValue

失敗モードと境界条件

  • メモリ不足時には、リスト操作関数がエラーを発生させる可能性があります。

保守担当者向けメモ

  • 本シンボルはPostgreSQLの汎用的な List 型へのエイリアスです。jsonb_gin.c 固有の構造体ではないため、リスト操作の詳細は src/include/nodes/pg_list.h を参照してください。
