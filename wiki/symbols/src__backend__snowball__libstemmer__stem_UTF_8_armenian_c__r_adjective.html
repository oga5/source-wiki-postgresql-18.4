目的

アルメニア語のステミング処理において、単語末尾の形容詞に関連する接尾辞を検索し、一致した場合に削除する。

static int r_adjective(struct SN_env * z)

入力と出力

  • **入力**: z (ステミング環境を保持する SN_env 構造体へのポインタ)
  • **出力**: 成功時は 1、接尾辞が見つからない場合は 0、削除処理中にエラーが発生した場合は負の値を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • 内部で find_among_b を使用するため、ステミング環境のカーソル位置が適切に設定されている必要がある。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を z->ket に設定する。 2. a_0 配列（23個の要素）を用いて、単語末尾から接尾辞を検索する (find_among_b)。 3. 一致した場合、その位置を z->bra に設定する。 4. slice_del(z) を呼び出し、一致した接尾辞を単語から削除する。 5. 削除処理の結果を呼び出し元に返す。

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_UTF_8_armenian.c:542 (メインのステミング処理内)
  • **呼び出し先**: find_among_b, slice_del (いずれも Snowball ランタイム関数)

関連データ構造

  • SN_env: ステミング環境を管理する構造体。
  • a_0: 形容詞の接尾辞パターンを定義した among 構造体の配列。

失敗モードと境界条件

  • find_among_b が一致を見つけられない場合、0 を返して終了する。
  • slice_del がメモリ不足や内部エラー等で負の値を返した場合、その値をそのまま伝播させる。
