GetPGVariable

定義

  • 種別: function
  • 定義箇所: src/backend/utils/misc/guc_funcs.c:381

目的

指定されたGUC（Grand Unified Configuration）パラメータの値を検索し、結果を DestReceiver を介して出力する。

シグネチャまたは形状

void GetPGVariable(const char *name, DestReceiver *dest)

入力と出力

  • name (入力): 取得対象のGUCパラメータ名。文字列 "all" を指定した場合は全設定が対象となる。
  • dest (入力): 結果を受け取るための DestReceiver 構造体へのポインタ。
  • 戻り値: なし。

前提条件と不変条件

  • name は有効なGUCパラメータ名、または "all" である必要がある。
  • dest は有効な DestReceiver オブジェクトを指している必要がある。

アルゴリズムまたは振る舞い

1. guc_name_compare を使用して name が "all" と一致するか判定する。 2. "all" の場合、ShowAllGUCConfig(dest) を呼び出し、全設定情報を出力する。 3. それ以外の場合、ShowGUCConfigOption(name, dest) を呼び出し、指定された単一のパラメータ情報を出力する。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/backend/replication/walsender.c:2205
  • src/backend/tcop/utility.c:882
  • 呼び出し先:
  • ShowAllGUCConfig
  • ShowGUCConfigOption
  • guc_name_compare

関連データ構造

  • DestReceiver: SQLコマンドの結果を受け取り、クライアントや他の処理系へ転送するためのインターフェース構造体。

失敗モードと境界条件

  • name が存在しないパラメータ名である場合、ShowGUCConfigOption 内でエラー処理が行われる（本関数自体は判定ロジックのみを保持）。

保守担当者向けメモ

  • 本関数は SHOW コマンドのディスパッチャーとして機能する。
  • パラメータの検索ロジックや出力形式の変更が必要な場合は、呼び出し先の ShowAllGUCConfig または ShowGUCConfigOption を参照すること。
