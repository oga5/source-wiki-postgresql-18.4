CopyLogVerbosityChoice

定義

  • 種別: enum
  • 定義箇所: src/include/commands/copy.h:46

目的

COPY コマンド実行時におけるログ出力の冗長性レベルを定義します。

シグネチャまたは形状

typedef enum CopyLogVerbosityChoice
{
    COPY_LOG_VERBOSITY_SILENT = -1,
    COPY_LOG_VERBOSITY_DEFAULT = 0,
    COPY_LOG_VERBOSITY_VERBOSE,
} CopyLogVerbosityChoice;

入力と出力

  • 入力: COPY コマンドのオプション解析時に DefElem から抽出されます。
  • 出力: CopyFormatOptions 構造体の log_verbosity フィールドとして保持されます。

前提条件と不変条件

  • COPY_LOG_VERBOSITY_DEFAULT がデフォルト値（0）として割り当てられています。
  • COPY_LOG_VERBOSITY_SILENT は -1 として定義されています。

アルゴリズムまたは振る舞い

  • COPY_LOG_VERBOSITY_SILENT: ログ出力を抑制します。
  • COPY_LOG_VERBOSITY_DEFAULT: 追加のメッセージを出力しません。
  • COPY_LOG_VERBOSITY_VERBOSE: 追加のメッセージを出力します。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/commands/copy.c 内の COPY オプション解析処理（DefElem からの抽出）。
  • 呼び出し先: 情報なし。

関連データ構造

  • CopyFormatOptions (本列挙型を log_verbosity フィールドとして保持)

失敗モードと境界条件

  • 該当なし。

保守担当者向けメモ

  • 本列挙型は COPY コマンドのログ制御専用です。新しい冗長性レベルを追加する場合は、既存の数値割り当て（-1, 0, 1...）との整合性に注意してください。
