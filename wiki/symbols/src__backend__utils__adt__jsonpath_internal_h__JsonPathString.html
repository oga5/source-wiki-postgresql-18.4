JsonPathString

定義

  • 種別: struct
  • 定義箇所: src/backend/utils/adt/jsonpath_internal.h:18

目的

JSONPath の字句解析（スキャナ）と構文解析（パーサ）の間で、解析対象の文字列データとそのメタデータを共有するために使用されます。

シグネチャまたは形状

typedef struct JsonPathString
{
	char	   *val;
	int			len;
	int			total;
} JsonPathString;

入力と出力

  • **入力**: 構造体の各フィールド（val は文字列ポインタ、len は現在の長さ、total は確保された容量など）。
  • **出力**: 解析器間で受け渡されるデータ構造体としての役割。

前提条件と不変条件

  • 本構造体は jsonpath_internal.h で定義されており、Flex/Bison 生成コード間で共有されることを前提としています。

アルゴリズムまたは振る舞い

  • 字句解析器（jsonpath_yylex）が解析した文字列情報を保持し、構文解析器（jsonpath_yyparse）へ渡すためのコンテナとして機能します。

呼び出し元と呼び出し先

  • **呼び出し元**: 提供されたソース情報からは、具体的な関数レベルの呼び出し元は特定できません。
  • **呼び出し先**: 提供されたソース情報からは、具体的な関数レベルの呼び出し先は特定できません。

関連データ構造

  • yyscan_t: スキャナ状態を保持する不透明ポインタ。
  • JsonPathParseResult: 解析結果を蓄積する構造体。

失敗モードと境界条件

  • 文字列ポインタ val が適切に初期化されていない場合や、len が実際のメモリ領域を超過している場合にメモリ安全性の問題が発生する可能性があります。

保守担当者向けメモ

  • 本構造体は jsonpath_gram.y（Bison ソース）と密接に関連しています。構造体の変更を行う際は、字句解析器と構文解析器の両方で整合性が取れていることを確認してください。
