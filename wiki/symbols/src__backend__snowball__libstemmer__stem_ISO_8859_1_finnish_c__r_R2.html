目的

単語内の現在のカーソル位置が、定義済みの「R2領域」の開始位置以降にあるかどうかを判定する。

シグネチャまたは形状

static int r_R2(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z（Snowball環境構造体）
  • 出力: int 型。条件を満たす場合は 1（真）、満たさない場合は 0（偽）を返す。

前提条件と不変条件

  • z->I[0] には、事前に r_mark_regions 等によって計算された R2 領域の開始インデックスが格納されている必要がある。
  • z->c は現在のカーソル位置を示す。

アルゴリズムまたは振る舞い

1. 環境構造体 z 内の整数配列 I のインデックス 0（R2の境界位置）と、現在のカーソル位置 c を比較する。 2. z->I[0] <= z->c の評価結果を返す。

  • struct SN_env: Snowballランタイムの環境を保持する構造体。I 配列（整数領域）および c（カーソル）を含む。

失敗モードと境界条件

  • z が NULL の場合、デリファレンスによりクラッシュする可能性がある。
