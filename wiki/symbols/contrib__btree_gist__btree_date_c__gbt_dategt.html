gbt_dategt

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_date.c:29

目的

date 型の2つの値に対して「より大きい（greater than）」という比較を行い、その結果を真偽値で返す。

シグネチャまたは形状

static bool gbt_dategt(const void *a, const void *b, FmgrInfo *flinfo)

入力と出力

  • **入力**:
  • a, b: 比較対象の DateADT 型へのポインタ（void * として渡される）。
  • flinfo: 関数マネージャ情報（本関数内では直接使用されない）。
  • **出力**:
  • bool: a > b であれば true、そうでなければ false を返す。

前提条件と不変条件

  • a および b は有効な DateADT 型の値を指している必要がある。
  • PostgreSQL のコア関数 date_gt が利用可能であること。

アルゴリズムまたは振る舞い

1. a と b を DateADT 型としてキャストし、DateADTGetDatum を用いて PostgreSQL の Datum 形式に変換する。 2. DirectFunctionCall2 を介して PostgreSQL コアの date_gt 関数を呼び出す。 3. date_gt の結果（Datum）を DatumGetBool で bool 型に変換して返す。

呼び出し元と呼び出し先

  • **呼び出し先**: date_gt（PostgreSQL コア関数）
  • **呼び出し元**: ソースコード内の静的解析によれば、同ファイル内の構造体初期化（contrib/btree_gist/btree_date.c:107）において、比較演算テーブルの一部として参照されている。

関連データ構造

  • DateADT: PostgreSQL の日付型を表す内部表現。

失敗モードと境界条件

  • 入力ポインタが NULL の場合、デリファレンスによりクラッシュする可能性がある。
  • date_gt 関数が内部でエラーを発生させた場合、PostgreSQL のエラー処理メカニズムにより処理が中断される。

保守担当者向けメモ

  • 本関数は btree_gist における date 型の比較ロジックのラッパーである。
  • btree_utils_num.c が提供する汎用的な数値処理ロジックと組み合わせて使用されることを想定している。
