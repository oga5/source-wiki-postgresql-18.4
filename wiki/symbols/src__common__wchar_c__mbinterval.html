mbinterval

定義

  • 種別: struct
  • 定義箇所: src/common/wchar.c:591

目的

Unicodeコードポイントの範囲（開始値と終了値）を保持し、特定のコードポイントがその範囲内に含まれるかを判定するためのデータ構造です。

シグネチャまたは形状

struct mbinterval
{
	unsigned int first;
	unsigned int last;
};

入力と出力

  • 入力: first（範囲の開始コードポイント）、last（範囲の終了コードポイント）。いずれも unsigned int 型。
  • 出力: なし（構造体定義）。

前提条件と不変条件

  • first <= last であることが期待されます。
  • この構造体の配列は、mbbisearch 関数による二分探索を前提として、コードポイント順にソートされている必要があります。

アルゴリズムまたは振る舞い

本構造体は、mbbisearch 関数において、対象のコードポイントが範囲内にあるかを判定するための比較対象として利用されます。

呼び出し元と呼び出し先

  • 呼び出し元（参照元）:
  • src/common/wchar.c:599 (mbbisearch 関数内)
  • src/common/wchar.c:668, 673 (配列サイズの計算)
  • 呼び出し先: なし（データ構造のため）。

関連データ構造

  • nonspacing: src/include/common/unicode_nonspacing_table.h で定義される mbinterval 型の配列。
  • east_asian_fw: src/include/common/unicode_east_asian_fw_table.h で定義される mbinterval 型の配列。

失敗モードと境界条件

  • mbbisearch で使用する際、配列がソートされていない場合や、max 引数に誤った配列サイズを指定した場合、探索結果が不正となります。

保守担当者向けメモ

  • Unicodeの仕様変更に伴い、nonspacing や east_asian_fw テーブルを更新する際は、本構造体の定義に従い、必ずコードポイント順にソートされた状態を維持してください。
