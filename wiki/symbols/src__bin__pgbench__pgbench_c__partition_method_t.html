partition_method_t

定義

  • 種別: enum
  • 定義箇所: src/bin/pgbench/pgbench.c:227

目的

pgbench におけるテーブルパーティショニングの方式を識別するために使用される列挙型です。

シグネチャまたは形状

typedef enum
{
    PART_NONE,  /* パーティショニングなし */
    PART_RANGE, /* 範囲パーティショニング */
    PART_HASH,  /* ハッシュパーティショニング */
} partition_method_t;

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: PART_NONE, PART_RANGE, PART_HASH のいずれかの値

前提条件と不変条件

  • pgbench 内でパーティショニング方式を保持する変数の型として使用されます。

アルゴリズムまたは振る舞い

  • PART_NONE: パーティショニングを行わないデフォルトの状態を示します。
  • PART_RANGE: 範囲に基づいたパーティショニング方式を示します。
  • PART_HASH: ハッシュに基づいたパーティショニング方式を示します。

呼び出し元と呼び出し先

  • 呼び出し元: src/bin/pgbench/pgbench.c:234 にて、静的変数 partition_method の型として参照されています。
  • 呼び出し先: 情報なし。

関連データ構造

  • pgbench.c 内の静的変数 partition_method（初期値 PART_NONE）。

失敗モードと境界条件

  • 該当なし。
