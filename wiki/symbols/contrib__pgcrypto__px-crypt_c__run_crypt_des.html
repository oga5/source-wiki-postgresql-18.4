run_crypt_des

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/px-crypt.c:37

目的

DES アルゴリズムを用いたパスワードハッシュ化処理を pgcrypto の統一インターフェースに適合させるためのラッパー関数。

シグネチャまたは形状

static char *run_crypt_des(const char *psw, const char *salt, char *buf, unsigned len);

入力と出力

  • **入力**:
  • psw: ハッシュ化対象のパスワード文字列。
  • salt: 使用するソルト文字列。
  • buf: 結果を格納するための出力バッファ。
  • len: buf のサイズ（バイト単位）。
  • **出力**:
  • 成功時: buf へのポインタ。
  • 失敗時: NULL。

前提条件と不変条件

  • buf は len バイト以上の書き込み可能な領域であること。
  • 内部で呼び出される px_crypt_des が有効なハッシュ文字列を返すこと。

アルゴリズムまたは振る舞い

1. px_crypt_des(psw, salt) を呼び出し、ハッシュ化された文字列を取得する。 2. 取得した結果が NULL であるか、結果の長さが len - 1 を超える場合は失敗として NULL を返す。 3. 上記の条件を満たす場合、strcpy を用いて buf に結果をコピーし、buf を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/pgcrypto/px-crypt.c 内の静的テーブル px_crypt_list から参照されている。
  • **呼び出し先**: px_crypt_des（外部関数）。

関連データ構造

  • px_crypt_list: px-crypt.c 内で定義される、アルゴリズムのプレフィックスと実行関数を紐付けるテーブル。

失敗モードと境界条件

  • px_crypt_des が NULL を返した場合、本関数も NULL を返す。
  • ハッシュ結果がバッファサイズ len に収まらない場合（終端文字を含む）、切り詰めを行わず NULL を返す。

保守担当者向けメモ

  • 本関数は px_crypt_list のエントリとして使用されており、直接的な呼び出しよりもテーブル経由でのディスパッチが主目的である。
  • px_crypt_des の実装詳細については contrib/pgcrypto/crypt-des.c を参照すること。
