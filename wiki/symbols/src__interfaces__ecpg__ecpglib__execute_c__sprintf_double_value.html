sprintf_double_value

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/ecpglib/execute.c:455

目的

double 型の値を文字列に変換し、指定された区切り文字（delim）を付加してバッファへ書き込みます。主に ECPG におけるホスト変数の値のシリアライズに使用されます。

シグネチャまたは形状

static void sprintf_double_value(char *ptr, double value, const char *delim)

入力と出力

  • ptr (char *): 出力先のバッファポインタ。
  • value (double): 文字列化する対象の値。
  • delim (const char *): 値の末尾に付加する文字列（区切り文字）。
  • 出力: なし（ptr が指すメモリ領域に結果が書き込まれます）。

前提条件と不変条件

  • ptr は、変換後の文字列と delim を格納するのに十分なメモリ領域を指している必要があります。
  • value が NaN または Infinity の場合、それぞれ "NaN"、"Infinity"、"-Infinity" という文字列が優先的に使用されます。

アルゴリズムまたは振る舞い

1. isnan(value) が真の場合、"NaN" と delim を連結して ptr に書き込みます。 2. isinf(value) が真の場合、符号に応じて "-Infinity" または "Infinity" と delim を連結して ptr に書き込みます。 3. 上記以外の場合、%.15g フォーマット指定子を使用して value を文字列化し、delim を連結して ptr に書き込みます。

呼び出し元と呼び出し先

  • 呼び出し元: src/interfaces/ecpg/ecpglib/execute.c 内の関数から呼び出されています。
  • sprintf_double_value(mallocedval + strlen(mallocedval), ((double *) var->value)[element], ",");
  • sprintf_double_value(mallocedval, *((double *) var->value), "");
  • 呼び出し先: 標準ライブラリの sprintf、isnan、isinf を使用します。

関連データ構造

  • 特になし。

失敗モードと境界条件

  • ptr が指すバッファのサイズが不足している場合、バッファオーバーフローが発生する可能性があります。
  • delim が NULL の場合、sprintf 内部で未定義の動作を引き起こす可能性があります。

保守担当者向けメモ

  • %.15g フォーマットは、double 型の精度を維持しつつ冗長なゼロを省略するために選択されています。
  • 本関数は static であり、execute.c 内でのみ使用される補助関数です。
