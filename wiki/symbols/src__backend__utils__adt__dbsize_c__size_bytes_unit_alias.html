size_bytes_unit_alias

定義

  • 種別: struct
  • 定義箇所: src/backend/utils/adt/dbsize.c:60

目的

人間が読みやすい形式のサイズ単位（例: "kB", "MB"）と、内部的な数値計算で使用される単位インデックスを紐付けるためのエイリアス定義構造体です。

シグネチャまたは形状

struct size_bytes_unit_alias
{
	const char *alias;
	int			unit_index;		/* corresponding size_pretty_units element */
};

入力と出力

  • **入力**: alias（単位を表す文字列）、unit_index（size_pretty_units 配列内のインデックス）。
  • **出力**: なし（データ保持用構造体）。

前提条件と不変条件

  • alias は NULL 終端文字列である必要があります。
  • unit_index は、size_pretty_units 配列の有効なインデックスを参照している必要があります。

アルゴリズムまたは振る舞い

この構造体は、pg_size_bytes 関数内で文字列からバイト数への変換を行う際、入力された単位文字列を検索するための静的配列 size_bytes_aliases の要素として使用されます。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/dbsize.c:67: size_bytes_aliases 配列の初期化時に使用。
  • src/backend/utils/adt/dbsize.c:844: pg_size_bytes 関数内のループ処理で、入力文字列と一致するエイリアスを検索するために使用。
  • **呼び出し先**: なし（データ構造体のため）。

関連データ構造

  • size_pretty_units: この構造体の unit_index が参照する、サイズ単位の定義テーブル。

失敗モードと境界条件

  • size_bytes_aliases 配列の末尾は、alias が NULL である要素で終了している必要があります。検索処理はこの NULL を終端条件としてループを終了します。

保守担当者向けメモ

  • 新しい単位をサポートする場合は、size_pretty_units を更新した上で、この構造体を用いた size_bytes_aliases 配列に新しいエイリアスを追加してください。
