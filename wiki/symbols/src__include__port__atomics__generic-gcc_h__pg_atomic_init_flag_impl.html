pg_atomic_init_flag_impl

定義

  • 種別: static inline function
  • 定義箇所: src/include/port/atomics/generic-gcc.h:148

目的

pg_atomic_flag 型の変数を初期化し、フラグをクリア状態に設定する。

シグネチャまたは形状

static inline void pg_atomic_init_flag_impl(volatile pg_atomic_flag *ptr)

入力と出力

  • **入力**: ptr - 初期化対象となる pg_atomic_flag 構造体への揮発性ポインタ。
  • **出力**: なし（副作用として ptr が指すメモリが更新される）。

前提条件と不変条件

  • ptr は有効な pg_atomic_flag 構造体を指している必要がある。
  • 本関数は atomics.h を経由して呼び出されることを前提としている。

アルゴリズムまたは振る舞い

内部的に pg_atomic_clear_flag_impl(ptr) を呼び出し、フラグをクリア状態に設定することで初期化を行う。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/include/port/atomics.h:172
  • src/include/port/atomics/generic.h:77
  • src/include/port/atomics/generic.h:114
  • **呼び出し先**: pg_atomic_clear_flag_impl

関連データ構造

  • pg_atomic_flag: 原子操作対象のフラグを保持する構造体。

失敗モードと境界条件

  • ptr が NULL の場合、未定義の動作を引き起こす可能性がある。

保守担当者向けメモ

  • 本関数は static inline で定義されており、コンパイル時に呼び出し元へ展開される。
  • pg_atomic_clear_flag_impl への依存があるため、実装を変更する際は当該関数の仕様と整合性を保つこと。
