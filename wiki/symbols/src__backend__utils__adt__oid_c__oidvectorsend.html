oidvectorsend

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/oid.c:253

目的

oidvector 型のデータをバイナリ形式に変換し、クライアントへ送信するための関数です。

シグネチャまたは形状

Datum oidvectorsend(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS を介して oidvector 型のデータを受け取ります。
  • **出力**: Datum 型としてバイナリ形式のデータ（bytea 相当）を返します。

前提条件と不変条件

  • array_send を呼び出すため、check_valid_oidvector による検証は明示的に行われません。

アルゴリズムまたは振る舞い

内部的に array_send(fcinfo) を呼び出し、汎用的な配列のバイナリ送信処理に委譲します。

呼び出し元と呼び出し先

  • **呼び出し元**: 不明（ソースコード上の直接的な呼び出し元は特定されていません）。
  • **呼び出し先**: array_send

関連データ構造

  • oidvector: Oid の固定長配列を表現する構造体。
  • Datum: PostgreSQL の汎用データ型。

失敗モードと境界条件

  • array_send の失敗モードに依存します。

保守担当者向けメモ

  • 本関数は oidvector をバイナリ形式へ変換する役割を担います。
  • array_send を利用しているため、oidvector 特有の制約（1次元であること等）の検証は array_send 側の挙動に依存します。
