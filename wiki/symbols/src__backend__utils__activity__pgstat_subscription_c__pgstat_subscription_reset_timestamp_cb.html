pgstat_subscription_reset_timestamp_cb

定義

  • 種別: function
  • 定義箇所: src/backend/utils/activity/pgstat_subscription.c:127

目的

サブスクリプション統計情報の最終リセット時刻を更新するコールバック関数です。

シグネチャまたは形状

void pgstat_subscription_reset_timestamp_cb(PgStatShared_Common *header, TimestampTz ts)

入力と出力

  • header: 更新対象の共有統計情報の共通ヘッダー（PgStatShared_Common 型）。
  • ts: 設定するリセット時刻（TimestampTz 型）。
  • 出力: なし（副作用として共有メモリ上の値を更新）。

前提条件と不変条件

  • header は PgStatShared_Subscription 型の構造体としてキャスト可能である必要があります。

アルゴリズムまたは振る舞い

1. 引数 header を PgStatShared_Subscription 型へキャストします。 2. キャストした構造体内の stats.stat_reset_timestamp フィールドに、引数 ts で渡された時刻を代入します。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/utils/activity/pgstat.c:363 において、統計システム初期化時にコールバックテーブルへ登録されています。
  • 呼び出し先: なし。

関連データ構造

  • PgStatShared_Subscription: サブスクリプション統計を保持する共有メモリ上の構造体。
  • PgStatShared_Common: 統計共有構造体の共通ヘッダー。

失敗モードと境界条件

  • header が NULL の場合、または PgStatShared_Subscription 型と互換性のない構造体を指している場合、メモリ破壊や不正なメモリアクセスが発生する可能性があります。

保守担当者向けメモ

  • 本関数は統計サブシステムの汎用的なリセット処理から呼び出されるコールバックです。構造体のレイアウト変更時には、キャストの整合性に注意してください。
