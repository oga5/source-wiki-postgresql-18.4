getOwnedSequences

定義

  • 種別: function
  • 定義箇所: src/backend/catalog/pg_depend.c:1008

目的

指定されたリレーション（テーブル等）に所有されているシーケンスのリストを取得します。

シグネチャまたは形状

List *getOwnedSequences(Oid relid)

入力と出力

  • **入力**: relid (Oid) - 所有関係を調査する対象リレーションのOID。
  • **出力**: List * - 所有されているシーケンスのOIDリスト。

前提条件と不変条件

  • relid は有効なリレーションを指している必要があります。
  • 内部的に getOwnedSequences_internal を呼び出し、所有関係の検索を行います。

アルゴリズムまたは振る舞い

1. 引数として受け取った relid を使用して、getOwnedSequences_internal(relid, 0, 0) を呼び出します。 2. 内部関数が pg_depend カタログをスキャンし、該当するシーケンスのリストを返却します。 3. 呼び出し元へそのリストをそのまま返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/commands/tablecmds.c:2067
  • src/backend/commands/tablecmds.c:6039
  • **呼び出し先**:
  • getOwnedSequences_internal (同一ファイル内)

関連データ構造

  • List: PostgreSQLの標準的なリスト構造体。
  • Oid: オブジェクト識別子。

失敗モードと境界条件

  • 指定された relid に所有シーケンスが存在しない場合、空のリストが返されます。
  • データベースのカタログアクセス権限やシステム状態に依存します。

保守担当者向けメモ

  • 本関数はラッパーであり、実際の検索ロジックは getOwnedSequences_internal に集約されています。ロジックの変更が必要な場合は getOwnedSequences_internal を参照してください。
  • src/include/catalog/dependency.h で宣言されており、外部モジュールから利用可能です。
