PLy_subtransaction_init_type

定義

  • 種別: function
  • 定義箇所: src/pl/plpython/plpy_subxactobject.c:57

目的

PL/Pythonのサブトランザクション用Python型オブジェクト（PLy_SubtransactionType）を初期化し、Pythonの型システムへ登録する。

シグネチャまたは形状

void PLy_subtransaction_init_type(void)

入力と出力

  • 入力: なし
  • 出力: なし（副作用としてグローバル変数 PLy_SubtransactionType を設定）

前提条件と不変条件

  • Pythonのインタープリタが初期化されていること。
  • PLySubtransaction_spec が有効な型定義として定義されていること。

アルゴリズムまたは振る舞い

1. PyType_FromSpec を呼び出し、PLySubtransaction_spec に基づいてPython型オブジェクトを生成する。 2. 生成されたオブジェクトを PLy_SubtransactionType に格納する。 3. 生成に失敗した場合、elog(ERROR, ...) を呼び出し、PostgreSQLの例外メカニズムを通じて処理を中断する。

呼び出し元と呼び出し先

  • 呼び出し元: src/pl/plpython/plpy_plpymodule.c:151（plpy モジュールの初期化プロセス内）
  • 呼び出し先: PyType_FromSpec（Python C API）、elog（PostgreSQLエラー報告）

関連データ構造

  • PLy_SubtransactionType: 初期化対象となるグローバルな PyTypeObject ポインタ。
  • PLySubtransaction_spec: 型の仕様を定義する PyType_Spec 構造体。

失敗モードと境界条件

  • PyType_FromSpec が NULL を返した場合、elog によりエラーが報告され、PostgreSQLのトランザクションがアボートされる。

保守担当者向けメモ

  • 本関数はPL/Pythonモジュールのロード時に一度だけ実行されることを想定している。
  • 型定義の変更が必要な場合は、PLySubtransaction_spec の定義箇所を確認すること。
