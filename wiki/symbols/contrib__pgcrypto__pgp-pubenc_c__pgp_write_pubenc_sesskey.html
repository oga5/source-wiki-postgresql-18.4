pgp_write_pubenc_sesskey

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/pgp-pubenc.c:189

目的

PGP公開鍵暗号方式を用いたセッション鍵パケット（Public-Key Encrypted Session Key Packet）を構築し、出力ストリームへ書き出します。

シグネチャまたは形状

int pgp_write_pubenc_sesskey(PGP_Context *ctx, PushFilter *dst);

入力と出力

  • **入力**:
  • ctx: PGPコンテキスト（公開鍵情報 pub_key を保持）。
  • dst: 書き出し先の出力フィルタ（PushFilter）。
  • **出力**:
  • 成功時は 0、失敗時は負のPGPエラーコードを返します。

前提条件と不変条件

  • ctx->pub_key が NULL であってはなりません（NULL の場合は PXE_BUG を返します）。
  • 公開鍵のアルゴリズムは、サポートされている RSA または ElGamal である必要があります。

アルゴリズムまたは振る舞い

1. PGPパケットライターを初期化し、パケットヘッダー（バージョン 3、鍵ID、アルゴリズムID）を書き出します。 2. 公開鍵のアルゴリズム（PGP_PUB_ELG_ENCRYPT、PGP_PUB_RSA_ENCRYPT、または PGP_PUB_RSA_ENCRYPT_SIGN）を判定します。 3. アルゴリズムに応じて、encrypt_and_write_elgamal または encrypt_and_write_rsa を呼び出し、暗号化されたセッション鍵データをパケットへ書き出します。 4. 最後にパケットのフラッシュを行い、使用したパケットライターを解放します。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/pgcrypto/pgp-encrypt.c:636
  • **呼び出し先**:
  • pgp_create_pkt_writer
  • pushf_write
  • encrypt_and_write_elgamal
  • encrypt_and_write_rsa
  • pushf_flush
  • pushf_free

関連データ構造

  • PGP_Context: 暗号化コンテキスト。
  • PGP_PubKey: 公開鍵情報。
  • PushFilter: 出力ストリーム制御用構造体。

失敗モードと境界条件

  • pk == NULL: PXE_BUG を返します。
  • パケット生成や書き出し処理（pushf_write 等）でエラーが発生した場合、そのエラーコードを返し、確保したリソースを解放して終了します。

保守担当者向けメモ

  • 新しい公開鍵アルゴリズムを追加する場合は、switch 文にケースを追加し、対応する encrypt_and_write_... 関数を実装してください。
  • 処理終了時には必ず pushf_free が呼ばれるよう、エラーハンドリングの goto err 構造が維持されています。
