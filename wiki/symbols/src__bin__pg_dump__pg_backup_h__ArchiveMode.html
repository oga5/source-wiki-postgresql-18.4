ArchiveMode

定義

  • 種別: enum
  • 定義箇所: src/bin/pg_dump/pg_backup.h:48

目的

PostgreSQLのバックアップおよびリストア処理における、アーカイブファイルの操作モード（読み込み、書き込み、追記）を定義する。

シグネチャまたは形状

typedef enum _archiveMode
{
	archModeAppend,
	archModeWrite,
	archModeRead,
} ArchiveMode;

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: archModeAppend（追記）、archModeWrite（書き込み）、archModeRead（読み込み）のいずれかの値。

前提条件と不変条件

  • アーカイバの初期化時やファイル操作のコンテキストにおいて、有効なモードとして指定される必要がある。

アルゴリズムまたは振る舞い

  • pg_dump や pg_restore の処理において、アーカイブファイルを開く際の挙動を決定するフラグとして利用される。
  • parseArchiveFormat 関数などで、ユーザー指定のフォーマットやモードを解析する際に引数として渡される。

呼び出し元と呼び出し先

  • 呼び出し元（参照箇所）:
  • src/bin/pg_dump/pg_dump.c: parseArchiveFormat 関数の引数および archiveMode 変数として使用。
  • src/bin/pg_dump/pg_backup_archiver.c: アーカイバの各種関数（bool dosync, ArchiveMode mode を引数に持つ関数群）で使用。
  • src/bin/pg_dump/pg_backup_archiver.h: Archive 構造体内のファイルモード保持用メンバとして使用。
  • 呼び出し先: なし（列挙型のため）。

関連データ構造

  • Archive: アーカイバのメインコンテキストを保持する構造体。本列挙型をメンバとして保持する。

失敗モードと境界条件

  • 該当なし（列挙型定義のため）。

保守担当者向けメモ

  • src/include/access/xlog.h にも同名の ArchiveMode が定義されているため、ヘッダーのインクルード順序や名前空間の衝突に注意すること。
  • pg_backup_archiver.h 内のコメントによれば、ファイルモード（r または w）として機能する。
