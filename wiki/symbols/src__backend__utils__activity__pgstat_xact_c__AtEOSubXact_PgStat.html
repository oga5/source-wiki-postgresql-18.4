AtEOSubXact_PgStat

定義

  • **種別**: 関数
  • **定義箇所**: src/backend/utils/activity/pgstat_xact.c:112

目的

サブトランザクションの終了（コミットまたはアボート）に伴い、そのサブトランザクション内で発生した統計情報の変更を親トランザクションへマージ、あるいは破棄するために呼び出されます。

シグネチャまたは形状

void AtEOSubXact_PgStat(bool isCommit, int nestDepth)

入力と出力

  • **入力**:
  • isCommit (bool): true の場合はコミット処理（変更を適用）、false の場合はアボート処理（変更を破棄）として動作します。
  • nestDepth (int): 現在のサブトランザクションのネストレベル。
  • **出力**: なし（void）。

前提条件と不変条件

  • pgStatXactStack が現在のサブトランザクション状態を保持している必要があります。
  • nestDepth が現在のスタックレベルと一致する場合にのみ処理が実行されます。

アルゴリズムまたは振る舞い

1. 現在のサブトランザクション状態（pgStatXactStack）を取得します。 2. スタックのネストレベルが指定された nestDepth 以上であるか確認します。 3. 条件を満たす場合、スタックから現在の xact_state を切り離します。 4. AtEOSubXact_PgStat_Relations を呼び出し、リレーション関連の統計変更を処理します。 5. AtEOSubXact_PgStat_DroppedStats を呼び出し、削除予定の統計アイテムを処理します。 6. 最後に pfree を使用して xact_state を解放します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/access/transam/xact.c（トランザクション管理のメインロジックから、サブトランザクションのコミット/アボート時に呼び出されます）
  • **呼び出し先**:
  • AtEOSubXact_PgStat_Relations
  • AtEOSubXact_PgStat_DroppedStats
  • pfree

関連データ構造

  • PgStat_SubXactStatus: サブトランザクションごとの統計状態を保持する構造体。
  • pgStatXactStack: トランザクションおよびサブトランザクションの統計状態を管理するスタックの先頭ポインタ。

失敗モードと境界条件

  • pgStatXactStack が NULL の場合、またはスタックのネストレベルが nestDepth 未満の場合は何も行われません。

保守担当者向けメモ

  • 本関数は TopTransactionContext 上のメモリを操作するため、スタックの整合性が重要です。
  • 統計の「作成」と「削除」はトランザクションの成否と逆の挙動をとるため、isCommit フラグの扱いに注意してください。
