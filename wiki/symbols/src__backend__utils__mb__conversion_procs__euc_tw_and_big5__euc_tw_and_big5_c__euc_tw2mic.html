euc_tw2mic

定義

  • 種別: function
  • 定義箇所: src/backend/utils/mb/conversion_procs/euc_tw_and_big5/euc_tw_and_big5.c:310

目的

EUC_TW エンコーディングの文字列を MULE_INTERNAL (MIC) エンコーディングへ変換します。

シグネチャまたは形状

static int euc_tw2mic(const unsigned char *euc, unsigned char *p, int len, bool noError)

入力と出力

  • **入力**:
  • euc: 変換元となる EUC_TW 文字列のポインタ。
  • p: 変換後の MIC 文字列を格納するバッファ。
  • len: 入力文字列のバイト長。
  • noError: 変換エラー時に例外を投げず中断するかどうかを指定するフラグ。
  • **出力**:
  • 戻り値: 処理した入力バイト数。
  • p が指すバッファには、変換後の MIC 文字列がヌル終端付きで書き込まれます。

前提条件と不変条件

  • euc は有効な EUC_TW エンコーディングのバイト列である必要があります。
  • p は変換後の文字列を格納するのに十分なメモリ領域を確保している必要があります。

アルゴリズムまたは振る舞い

1. 入力文字列をバイト単位で走査します。 2. ASCII文字（高位ビットが未セット）はそのまま p にコピーします。 3. マルチバイト文字（高位ビットがセット）の場合:

  • pg_encoding_verifymbchar で妥当性を検証します。不正な場合は noError に応じて中断またはエラー報告を行います。
  • SS2 (0x8e) で始まる場合、CNS 11643 の各プレーン（1, 2, またはそれ以外）に応じた MIC プレフィックスを付与し、コードポイントを変換します。
  • それ以外の場合は CNS 11643-1 として処理し、対応する MIC プレフィックスを付与します。

    • 4. 最後に p の末尾にヌル文字を付与します。

呼び出し元と呼び出し先

  • **呼び出し元**: 同一ファイル内の euc_tw_to_mic 関数から呼び出されます。
  • **呼び出し先**: pg_encoding_verifymbchar (妥当性検証)、report_invalid_encoding (エラー報告)。

関連データ構造

  • PG_EUC_TW: エンコーディング識別子。
  • LC_CNS11643_x: MULE_INTERNAL における文字セット識別子。
  • SS2: EUC_TW における補助文字セットを示すシングルシフトコード。

失敗モードと境界条件

  • 入力文字列が不正なマルチバイトシーケンスである場合、noError が false ならばエラーを報告し、true ならばその時点までの変換結果を返して終了します。
  • 入力文字列にヌルバイトが含まれる場合、不正なエンコーディングとして扱われます。

保守担当者向けメモ

  • 本関数は EUC_TW から MIC への変換ロジックをカプセル化しています。
  • 変換テーブルや計算式は MULE_INTERNAL の仕様に依存しています。修正時は pg_wchar.h の定義と整合性を確認してください。
