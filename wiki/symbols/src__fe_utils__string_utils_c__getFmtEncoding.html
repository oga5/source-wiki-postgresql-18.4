getFmtEncoding

定義

  • 種別: function
  • 定義箇所: src/fe_utils/string_utils.c:77

目的

識別子処理関数（fmtId 等）で使用される現在の文字エンコーディングを取得します。

シグネチャまたは形状

static int getFmtEncoding(void)

入力と出力

  • **入力**: なし
  • **出力**: 現在設定されているエンコーディングを示す整数値（int）。設定されていない場合は PG_UTF8 を返します。

前提条件と不変条件

  • 内部変数 fmtIdEncoding が -1 以外の値に初期化されていることが期待されます。
  • アサーションビルド（Assert）では、エンコーディングが未設定の場合にプログラムが停止します。

アルゴリズムまたは振る舞い

1. グローバル変数 fmtIdEncoding が -1 でないか確認します。 2. -1 でない場合、その値を返します。 3. -1 の場合、アサーションビルドではエラーを発生させ、それ以外のビルドではデフォルト値として PG_UTF8 を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/fe_utils/string_utils.c:250 (fmtId 内)
  • src/fe_utils/string_utils.c:298 (fmtQualifiedId 内)
  • **呼び出し先**:
  • なし（直接的な関数呼び出しは行いませんが、マクロ PG_UTF8 を参照します）

関連データ構造

  • fmtIdEncoding: エンコーディング状態を保持する静的変数。

失敗モードと境界条件

  • エンコーディングが明示的に設定されていない場合、デフォルトの PG_UTF8 にフォールバックします。
  • アサーション有効時には、未設定状態は致命的なエラーとして扱われます。

保守担当者向けメモ

  • 本関数は fmtId や fmtQualifiedId がエンコーディングを明示的に指定しない場合に呼び出されます。
  • 可能な限り、呼び出し側で fmtIdEnc のようにエンコーディングを明示する関数を使用することが推奨されます。
