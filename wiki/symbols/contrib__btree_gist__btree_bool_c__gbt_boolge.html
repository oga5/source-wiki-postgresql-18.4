gbt_boolge

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_bool.c:31

目的

2つの bool 型の値に対し、「以上（greater than or equal to）」の比較演算を行い、その真偽値を返します。

シグネチャまたは形状

static bool gbt_boolge(const void *a, const void *b, FmgrInfo *flinfo)

入力と出力

  • **入力**:
  • a: 比較対象の bool 値へのポインタ。
  • b: 比較対象の bool 値へのポインタ。
  • flinfo: 関数管理情報（本関数内では未使用）。
  • **出力**:
  • bool: *a >= *b の評価結果。

前提条件と不変条件

  • a および b は、有効な bool 型の値（true または false）を指している必要があります。
  • bool 型は内部的に数値（0/1）として扱われるため、標準的な比較演算子が適用可能です。

アルゴリズムまたは振る舞い

1. 引数 a と b を const bool * にキャストします。 2. それぞれのポインタが指す値をデリファレンスし、C言語の >= 演算子を用いて比較します。 3. 比較結果をそのまま返します。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/btree_gist/btree_bool.c:76 にて、比較関数テーブルの一部として参照されています。
  • **呼び出し先**: 直接的な関数呼び出しはありません。

関連データ構造

  • bool: 比較対象となるPostgreSQLのブール型。

失敗モードと境界条件

  • a または b が NULL ポインタである場合、デリファレンス時にセグメンテーション違反が発生します。

保守担当者向けメモ

  • 本関数は btree_utils_num.h が提供する数値系ユーティリティと組み合わせて使用されることを想定した、薄いラッパー関数です。
  • bool 型の比較ロジックに変更が必要な場合は、本関数および他の比較関数（gbt_boolgt 等）を併せて確認してください。
