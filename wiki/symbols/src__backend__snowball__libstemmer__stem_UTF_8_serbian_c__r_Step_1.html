目的

セルビア語のステミング処理における第1段階の語尾削除を実行します。

シグネチャまたは形状

static int r_Step_1(struct SN_env * z)

入力と出力

  • 入力: SN_env 構造体（ステミング環境）。
  • 出力: 成功時に 1 または 0、エラー（メモリ不足等）時に負の値を返します。

前提条件と不変条件

  • z->c が単語内の現在位置を指していること。
  • a_1 テーブル（130個の要素）が定義されていること。
  • z->I[1] が領域（R1）の判定フラグとして使用されていること。

アルゴリズムまたは振る舞い

1. z->ket を現在位置 z->c に設定します。 2. 文字列の末尾付近で、a_1 テーブルに基づき find_among_b を用いて後方一致検索を行います。 3. 一致が見つかった場合、z->bra を更新し、一致したルール（among_var）に応じて slice_from_s を呼び出し、該当する語尾を置換または削除します。 4. 一部のルール（ケース7および31）では、z->I[1] が真である場合にのみ削除を実行します。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_serbian.c 内のメイン処理（serbian_UTF_8_stem 等から間接的に呼び出される）。
  • 呼び出し先: find_among_b（ルール検索）、slice_from_s（文字列操作）。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_1: ステミングルールを定義した among 構造体の配列。

失敗モードと境界条件

  • find_among_b が一致を返さない場合は 0 を返して終了します。
  • 文字列の長さが不十分な場合や、slice_from_s が負の値を返した場合は、そのエラーコードを伝播させます。
