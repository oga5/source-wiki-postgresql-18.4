pg_sjis_mblen

定義

  • 種別: function
  • 定義箇所: src/port/path.c:201

目的

Shift-JIS エンコーディングにおけるマルチバイト文字列の文字長を判定します。特にパス操作において、Shift-JIS の 2 バイト文字（\ を含む可能性があるもの）を正しくスキップするために使用されます。

シグネチャまたは形状

int pg_sjis_mblen(const unsigned char *s)

入力と出力

  • **入力**: s (Shift-JIS 文字列の先頭ポインタ)
  • **出力**: int (現在の文字のバイト数。1 または 2 を返す)

前提条件と不変条件

  • 入力ポインタ s は有効なメモリ領域を指している必要があります。
  • 2 バイト文字の判定時、s[1] が存在することを前提としています。

アルゴリズムまたは振る舞い

1. *s が 0xa1 から 0xdf の範囲（半角カナ）であれば、長さ 1 を返します。 2. IS_HIGHBIT_SET(*s) が真かつ s[1] がヌル終端文字でない場合、2 バイト文字（漢字等）と見なし、長さ 2 を返します。 3. 上記以外（ASCII 文字等）は、長さ 1 を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/port/path.c:177 (パス走査ループ内)
  • src/common/wchar.c:1456
  • src/common/wchar.c:2122 (エンコーディング定義テーブル)
  • src/common/wchar.c:2128 (エンコーディング定義テーブル)
  • **呼び出し先**:
  • IS_HIGHBIT_SET マクロ（詳細な定義は本ソース外）

関連データ構造

  • src/common/wchar.c 内のエンコーディング定義テーブルで使用されています。

失敗モードと境界条件

  • s が指す文字列の末尾で s[1] が \0 の場合、2 バイト文字の判定がスキップされ、長さ 1 が返されます。
  • src/common/wchar.c に存在する同名の関数とは実装が異なる可能性がある旨が src/port/path.c 内で言及されています。

保守担当者向けメモ

  • 本関数は src/port/path.c 内でのパス操作に特化しています。他のエンコーディング対応を追加する際は、\（0x5C）問題への影響を考慮してください。
