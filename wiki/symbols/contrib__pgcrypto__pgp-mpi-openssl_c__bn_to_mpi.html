bn_to_mpi

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/pgp-mpi-openssl.c:55

目的

OpenSSL の BIGNUM 構造体を、OpenPGP 形式の PGP_MPI 構造体に変換する。

シグネチャまたは形状

static PGP_MPI *bn_to_mpi(BIGNUM *bn)

入力と出力

  • **入力**: BIGNUM *bn (変換対象の OpenSSL 多倍長整数)
  • **出力**: 成功時は変換後の PGP_MPI *、失敗時は NULL を返す。

前提条件と不変条件

  • bn は有効な BIGNUM ポインタである必要がある。
  • 内部で pgp_mpi_alloc を呼び出すため、呼び出し元は返された PGP_MPI を適切に解放する責任を持つ。

アルゴリズムまたは振る舞い

1. BN_num_bits(bn) を用いて必要なビット長を算出し、pgp_mpi_alloc で PGP_MPI 構造体を確保する。 2. 確保に失敗した場合は NULL を返す。 3. BN_num_bytes(bn) と確保した n->bytes を比較し、不一致の場合はデバッグログを出力して pgp_mpi_free でメモリを解放し、NULL を返す。 4. BN_bn2bin を使用して BIGNUM のデータを PGP_MPI のデータ領域へコピーする。 5. 変換後の PGP_MPI ポインタを返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/pgcrypto/pgp-mpi-openssl.c 内の以下の関数から呼び出される:
  • ElGamal 暗号化/復号処理 (pgp_elgamal_encrypt, pgp_elgamal_decrypt)
  • RSA 暗号化/復号処理 (pgp_rsa_encrypt, pgp_rsa_decrypt)
  • **呼び出し先**:
  • pgp_mpi_alloc
  • pgp_mpi_free
  • BN_num_bits
  • BN_num_bytes
  • BN_bn2bin
  • px_debug

関連データ構造

  • BIGNUM (OpenSSL)
  • PGP_MPI (pgp.h で定義)

失敗モードと境界条件

  • pgp_mpi_alloc がメモリ不足等で失敗した場合、NULL を返す。
  • BN_num_bytes(bn) と n->bytes が一致しない場合、変換は中断され NULL が返される。

保守担当者向けメモ

  • 本関数は pgcrypto モジュールにおける OpenSSL と PGP 形式のブリッジとして機能する。
  • 変換失敗時のメモリリークを防ぐため、エラーパスでの pgp_mpi_free 呼び出しが必須となっている。
