r_nu

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_estonian.c:1146

目的

エストニア語のステミング処理において、特定の接尾辞（a_8 配列で定義）を単語末尾から削除する。

シグネチャまたは形状

static int r_nu(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（Snowball ランタイム環境構造体）
  • 出力: 成功時に 1、失敗または条件不一致時に 0、内部エラー時に負の値を返す。

前提条件と不変条件

  • z->I[0] に格納された境界値（領域）より現在のカーソル位置 z->c が後ろにあること。
  • 削除対象の接尾辞が a_8 配列に含まれていること。
  • 削除対象の直前の文字が 'a' (97) または 'u' (117) であること。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置が z->I[0] より前であれば終了する。 2. 検索範囲を z->I[0] から現在の位置までに制限する。 3. 文字列の末尾（ket）から逆方向に検索を行い、a_8 に定義された接尾辞と、直前の文字が 'a' または 'u' であるかを確認する。 4. 条件に合致した場合、該当箇所を slice_del で削除する。 5. 処理終了後、制限していた検索範囲を元に戻す。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_estonian.c:1253（関数内での直接的な呼び出しを確認）
  • 呼び出し先: find_among_b（接尾辞検索）、slice_del（文字列削除）

関連データ構造

  • struct SN_env: ステミング処理の状態を保持する構造体。
  • a_8: 削除対象となる接尾辞を格納した among 構造体の配列。

失敗モードと境界条件

  • カーソル位置が z->I[0] より前にある場合、処理は行われず 0 を返す。
  • a_8 に一致する接尾辞が見つからない場合、または直前の文字が 'a'/'u' でない場合、0 を返す。
  • slice_del が失敗した場合、その戻り値（負の値）をそのまま返す。
