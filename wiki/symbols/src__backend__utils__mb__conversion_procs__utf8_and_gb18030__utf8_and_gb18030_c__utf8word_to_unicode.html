utf8word_to_unicode

定義

  • 種別: function
  • 定義箇所: src/backend/utils/mb/conversion_procs/utf8_and_gb18030/utf8_and_gb18030.c:93

目的

UTF-8 エンコーディング形式で表現されたマルチバイト文字（uint32 型のワード）を、対応する Unicode コードポイントへ変換します。

シグネチャまたは形状

static inline uint32 utf8word_to_unicode(uint32 c)

入力と出力

  • **入力**: c (UTF-8 バイト列を uint32 に格納したもの)
  • **出力**: ucs (変換後の Unicode コードポイント)

前提条件と不変条件

  • 入力 c は有効な UTF-8 バイトシーケンスの断片であると想定されます。
  • 1バイトから4バイトまでの UTF-8 表現に対応します。

アルゴリズムまたは振る舞い

入力された c のバイト長に応じてビット演算を行い、Unicode コードポイントを抽出します。

  • **1バイト (0x7F以下)**: そのまま値を返します。
  • **2バイト (0xFFFF以下)**: 上位バイトから 5 ビット、下位バイトから 6 ビットを抽出して結合します。
  • **3バイト (0xFFFFFF以下)**: 各バイトからそれぞれ 4, 6, 6 ビットを抽出し、12, 6, 0 ビットシフトして結合します。
  • **4バイト**: 各バイトからそれぞれ 3, 6, 6, 6 ビットを抽出し、18, 12, 6, 0 ビットシフトして結合します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/mb/conversion_procs/utf8_and_gb18030/utf8_and_gb18030.c 内の変換ロジック（例: 160行目付近）から呼び出されます。
  • **呼び出し先**: 外部関数への依存はありません。

関連データ構造

  • 特になし（プリミティブな uint32 型のみを使用）。

失敗モードと境界条件

  • 入力が不正な UTF-8 シーケンスである場合、本関数はそれを検証せず、ビット演算の結果をそのまま返します。

保守担当者向けメモ

  • 本関数は static inline で定義されており、変換プロシージャ内でのパフォーマンスを重視しています。
  • UTF-8 のビットパターンに基づいた変換ロジックであるため、UTF-8 の仕様変更がない限り修正は不要です。
