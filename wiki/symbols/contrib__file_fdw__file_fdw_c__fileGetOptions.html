fileGetOptions

定義

  • 種別: function
  • 定義箇所: contrib/file_fdw/file_fdw.c:378

目的

外部テーブルに関連付けられた FDW オプションから、ファイルパス（またはプログラムコマンド）とその他の設定項目を抽出・分離します。

シグネチャまたは形状

static void fileGetOptions(Oid foreigntableid, char **filename, bool *is_program, List **other_options)

入力と出力

  • **入力**:
  • foreigntableid: 対象となる外部テーブルの OID。
  • **出力**:
  • *filename: 抽出されたファイルパスまたはプログラム文字列。
  • *is_program: filename がプログラム実行を指す場合は true、ファイルパスの場合は false。
  • *other_options: filename または program 以外の残りのオプションリスト。

前提条件と不変条件

  • foreigntableid に対応する外部テーブル、サーバー、および FDW 定義が存在すること。
  • 呼び出し前に file_fdw_validator による妥当性検証が行われていることが期待されます。

アルゴリズムまたは振る舞い

1. GetForeignTable 等を使用して、FDW、サーバー、テーブルの各レベルで定義されたオプションを収集し、属性オプションと結合します。 2. 収集したリストから filename または program オプションを検索します。 3. 見つかったオプションをリストから削除し、*filename に値を格納します。program の場合は *is_program を true に設定します。 4. filename または program が見つからない場合は elog(ERROR) を発生させます。 5. 残りのオプションを *other_options に格納して返します。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/file_fdw/file_fdw.c 内の複数の関数（fileGetForeignRelSize, fileBeginForeignScan, file_acquire_sample_rows 等）から呼び出されています。
  • **呼び出し先**: GetForeignTable, GetForeignServer, GetForeignDataWrapper, get_file_fdw_attribute_options, defGetString, foreach_delete_current 等。

関連データ構造

  • ForeignTable, ForeignServer, ForeignDataWrapper: PostgreSQL のカタログ情報構造体。
  • DefElem: オプションの定義要素。

失敗モードと境界条件

  • filename または program オプションが指定されていない場合、エラーが発生します。
  • 複数の filename や program が混在する場合、最初に見つかったものが優先され、残りは other_options に含まれます。

保守担当者向けメモ

  • 本関数は COPY コマンドのインフラストラクチャに依存しています。オプションの仕様変更時は file_fdw_validator との整合性に注意してください。
  • 現在の実装ではユーザーマッピング（User Mapping）を無視していますが、将来的に必要となる可能性がある場合は考慮が必要です。
