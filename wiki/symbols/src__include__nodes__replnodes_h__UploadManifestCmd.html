UploadManifestCmd

定義

  • 種別: struct
  • 定義箇所: src/include/nodes/replnodes.h:127

目的

レプリケーションプロトコルにおける UPLOAD_MANIFEST コマンドを表現するためのパースノードです。

シグネチャまたは形状

typedef struct UploadManifestCmd
{
	NodeTag		type;
} UploadManifestCmd;

入力と出力

  • 入力: レプリケーションプロトコル経由で受信した UPLOAD_MANIFEST コマンドの構文解析結果。
  • 出力: 実行エンジンがレプリケーション処理を行うためのノード構造体。

前提条件と不変条件

  • NodeTag 型の type フィールドを先頭に持ち、PostgreSQLのノードシステムの一部として扱われます。

アルゴリズムまたは振る舞い

本構造体はデータ保持専用のノードであり、構文解析器によってインスタンス化され、実行エンジンへコマンド情報を引き渡すために使用されます。

呼び出し元と呼び出し先

  • 呼び出し元: 提供されたソース情報からは特定できません。
  • 呼び出し先: 提供されたソース情報からは特定できません。

関連データ構造

  • NodeTag: PostgreSQLのノードシステムにおける識別タグ。

失敗モードと境界条件

  • 特記事項なし（データ構造定義のみのため）。

保守担当者向けメモ
