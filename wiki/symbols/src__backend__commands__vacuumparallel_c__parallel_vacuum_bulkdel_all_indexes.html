parallel_vacuum_bulkdel_all_indexes

定義

  • 種別: function
  • 定義箇所: src/backend/commands/vacuumparallel.c:499

目的

並列 VACUUM 処理において、全インデックスに対するバルク削除フェーズを開始する。

シグネチャまたは形状

void parallel_vacuum_bulkdel_all_indexes(ParallelVacuumState *pvs, long num_table_tuples, int num_index_scans)

入力と出力

  • pvs: 並列 VACUUM の状態を保持する ParallelVacuumState 構造体へのポインタ。
  • num_table_tuples: テーブル内のタプル数（近似値）。
  • num_index_scans: 実行対象となるインデックススキャンの数。
  • 戻り値: なし。

前提条件と不変条件

  • 本関数はリーダープロセスによって呼び出される必要がある（IsParallelWorker() が偽であることをアサートする）。

アルゴリズムまたは振る舞い

1. 共有状態 pvs->shared にテーブルのタプル数 (reltuples) を設定し、近似値であることを示すフラグ (estimated_count) を立てる。 2. parallel_vacuum_process_all_indexes を呼び出し、インデックスのバルク削除処理を開始する。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/access/heap/vacuumlazy.c
  • 呼び出し先: parallel_vacuum_process_all_indexes

関連データ構造

  • ParallelVacuumState: 並列 VACUUM の実行状態を管理する構造体。

失敗モードと境界条件

  • 並列ワーカープロセスから呼び出された場合、アサーション違反により異常終了する。

保守担当者向けメモ

  • 本関数はリーダープロセス専用のインターフェースである。
  • num_table_tuples は現時点では近似値として扱われる仕様である。
