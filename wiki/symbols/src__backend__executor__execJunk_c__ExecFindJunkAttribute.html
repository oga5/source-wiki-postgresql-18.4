ExecFindJunkAttribute

定義

  • 種別: function
  • 定義箇所: src/backend/executor/execJunk.c:209

目的

JunkFilter が保持するターゲットリストから、指定された名前を持つジャンク属性の属性番号（AttrNumber）を検索します。

シグネチャまたは形状

AttrNumber ExecFindJunkAttribute(JunkFilter *junkfilter, const char *attrName)

入力と出力

  • **入力**:
  • junkfilter: 検索対象のジャンクフィルタ構造体へのポインタ。
  • attrName: 検索するジャンク属性の名前（文字列）。
  • **出力**:
  • 該当する属性の AttrNumber を返します。

前提条件と不変条件

  • junkfilter は初期化済みである必要があります。
  • 内部的に junkfilter->jf_targetList を参照するため、このリストが有効であることが前提です。

アルゴリズムまたは振る舞い

1. junkfilter が保持するターゲットリスト（jf_targetList）を引数として、ExecFindJunkAttributeInTlist を呼び出します。 2. ExecFindJunkAttributeInTlist の検索結果をそのまま呼び出し元へ返します。

呼び出し元と呼び出し先

  • **呼び出し先**: ExecFindJunkAttributeInTlist
  • **呼び出し元**: ソースコード内の言及として src/backend/executor/execJunk.c および src/backend/executor/functions.c での利用が示唆されていますが、具体的な関数呼び出しの連鎖については提供された情報では不完全です。

関連データ構造

  • JunkFilter: ジャンク属性の管理に使用される構造体。
  • AttrNumber: 属性番号を表す型。

失敗モードと境界条件

  • 該当する名前の属性が見つからない場合の挙動については、本ソース抜粋からは判断できません（ExecFindJunkAttributeInTlist の仕様に依存します）。

保守担当者向けメモ

  • 本関数は ExecFindJunkAttributeInTlist への単なるラッパーです。
  • JunkFilter 構造体の詳細や、属性名と属性番号のマッピングに関する詳細は src/backend/executor/execJunk.c の他の関数および executor.h を参照してください。
