TimeOffset

定義

  • 種別: typedef
  • 定義箇所: src/include/datatype/timestamp.h:40

目的

PostgreSQLにおける時間的な差分やオフセット値を表現するためのデータ型です。

シグネチャまたは形状

typedef int64 TimeOffset;

入力と出力

  • 入力: int64 型の整数値。
  • 出力: int64 型の整数値。

前提条件と不変条件

  • int64 型として定義されており、マイクロ秒単位の整数演算を前提としています。

アルゴリズムまたは振る舞い

  • 内部的には int64 として扱われ、時間的なオフセットやラグ（遅延）の計算に使用されます。
  • 浮動小数点演算を避け、整数演算による精度維持を目的としています。

呼び出し元と呼び出し先

  • 呼び出し元（参照箇所）:
  • src/backend/backup/basebackup_throttle.c
  • src/backend/replication/walsender.c
  • src/backend/utils/adt/date.c
  • 呼び出し先:
  • 情報なし（本シンボルは型定義であり、関数ではないため呼び出し先は存在しません）。

関連データ構造

  • Timestamp: 絶対時間を表す int64 型。
  • TimestampTz: タイムゾーン付きの絶対時間を表す int64 型。

失敗モードと境界条件

  • int64 型の範囲制限に依存します。極端に大きなオフセット値を扱う場合、オーバーフローの可能性があります。

保守担当者向けメモ

  • 本型は timestamp.h で定義されており、日時計算の基盤となる型です。変更を行う際は、int64 ベースの演算を行っている他の日時関連関数への影響を考慮してください。
