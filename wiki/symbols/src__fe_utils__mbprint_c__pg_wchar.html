pg_wchar

定義

  • 種別: typedef
  • 定義箇所: src/fe_utils/mbprint.c:31

目的

PostgreSQL におけるワイド文字（マルチバイト文字を内部的に表現するための型）を定義します。

シグネチャまたは形状

typedef unsigned int pg_wchar;

入力と出力

  • 入力: unsigned int 型の値を保持するデータ型です。
  • 出力: なし（型定義のため）。

前提条件と不変条件

  • 符号なし整数として定義されており、マルチバイト文字を内部処理するための抽象化された単位として使用されます。

アルゴリズムまたは振る舞い

  • 本シンボル自体は型定義であり、具体的なロジックは持ちません。
  • 関連する処理（pg_wcssize や pg_wcsformat 等）において、マルチバイト文字列を走査・変換する際の基本単位として利用されます。

呼び出し元と呼び出し先

  • 呼び出し元: ツールによる解析では、contrib/pg_trgm/trgm_regexp.c 等の複数のソースファイルで mb/pg_wchar.h を経由して参照されていることが示唆されています。
  • 呼び出し先: 型定義であるため、直接的な呼び出し先はありません。

関連データ構造

  • trgm_mb_char (関連する変換処理で使用)
  • lineptr (本モジュール内の整形処理で使用)

失敗モードと境界条件

  • 型定義であるため、本シンボル自体に失敗モードはありません。ただし、この型を扱う関数において、エンコーディングの不一致や不正なバイトシーケンスの処理が境界条件となります。

保守担当者向けメモ

  • 本ファイル (src/fe_utils/mbprint.c) は pg_wchar.h を直接インクルードしません。これはコンパイル時のヘッダーとリンク時の libpq.so のバージョン不一致によるエンコーディングIDの齟齬を避けるための設計上の判断です。
  • pg_wchar を使用するロジックを変更する際は、mbprint.c 内の表示幅計算ロジックと整合性が取れているか確認してください。
