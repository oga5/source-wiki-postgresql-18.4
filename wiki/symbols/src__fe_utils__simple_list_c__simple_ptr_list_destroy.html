simple_ptr_list_destroy

定義

  • 種別: function
  • 定義箇所: src/fe_utils/simple_list.c:180

目的

汎用ポインタを保持する単方向連結リスト（SimplePtrList）の各セルをメモリから解放する。

シグネチャまたは形状

void simple_ptr_list_destroy(SimplePtrList *list)

入力と出力

  • 入力: SimplePtrList *list（破棄対象のリスト構造体へのポインタ）
  • 出力: なし（void）

前提条件と不変条件

  • 引数 list は有効な SimplePtrList 構造体を指している必要がある。
  • リスト内の各セルは pg_malloc 等で確保されていることが前提となる。

アルゴリズムまたは振る舞い

1. リストの先頭（list->head）から走査を開始する。 2. 各セルについて、次のセルへのポインタを一時保存する。 3. pg_free を呼び出し、現在のセルを解放する。 4. 保存しておいた次のセルへ移動し、リストが空になるまで繰り返す。

呼び出し元と呼び出し先

  • 呼び出し元: src/bin/pg_verifybackup/pg_verifybackup.c:863
  • 呼び出し先: pg_free（メモリ解放用）

関連データ構造

  • SimplePtrList: リストの管理構造体（head を保持）。
  • SimplePtrListCell: リストの各要素（next ポインタを保持）。

失敗モードと境界条件

  • リストが空（list->head == NULL）の場合、ループは実行されず即座に終了する。
  • **注意**: 本関数はリストのセル自体を解放するのみであり、各セルが保持する汎用ポインタ（ptr）が指し示す先のメモリは解放されない。

保守担当者向けメモ

  • 本関数はポインタリスト専用である。文字列リストや OID リストにはそれぞれの専用破棄関数を使用すること。
  • リスト構造体自体（list ポインタそのもの）の解放は呼び出し元で行う必要がある。
