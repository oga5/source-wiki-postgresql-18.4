pgp_set_pubkey

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/pgp-pubkey.c:564

目的

MBuf から PGP 公開鍵または秘密鍵パケットを読み込み、解析した鍵情報を PGP_Context に設定します。

シグネチャまたは形状

int pgp_set_pubkey(PGP_Context *ctx, MBuf *keypkt, const uint8 *key, int key_len, int pubtype)

入力と出力

  • **入力**:
  • ctx: 鍵を格納する PGP_Context 構造体へのポインタ。
  • keypkt: 鍵データを含む MBuf。
  • key: 鍵の復号に使用するパスフレーズ（秘密鍵の場合）。
  • key_len: パスフレーズの長さ。
  • pubtype: 鍵の種類を示す識別子。
  • **出力**:
  • 成功時は 0 を返します。
  • 失敗時は負の整数（エラーコード）を返します。

前提条件と不変条件

  • keypkt は有効な MBuf である必要があります。
  • 内部で internal_read_key を呼び出し、パケットの解析を行います。
  • 成功した場合、解析された PGP_PubKey 構造体は ctx->pub_key に割り当てられます。

アルゴリズムまたは振る舞い

1. pullf_create_mbuf_reader を使用して keypkt から PullFilter を作成します。 2. internal_read_key を呼び出し、パケットから鍵情報を抽出します。 3. pullf_free で PullFilter を解放します。 4. 解析が成功した場合、抽出された PGP_PubKey 構造体を ctx->pub_key に格納します。 5. 処理結果（成功なら 0、失敗なら負の値）を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/pgcrypto/pgp-pgsql.c (行412, 499)
  • **呼び出し先**:
  • pullf_create_mbuf_reader
  • internal_read_key
  • pullf_free

関連データ構造

  • PGP_Context: 鍵情報を保持するコンテキスト。
  • MBuf: 鍵データが格納されたメモリバッファ。
  • PGP_PubKey: 解析された鍵情報を保持する構造体。
  • PullFilter: MBuf からデータを読み出すためのフィルタ。

失敗モードと境界条件

  • pullf_create_mbuf_reader が失敗した場合、そのエラーコードを返します。
  • internal_read_key が失敗した場合、そのエラーコードを返します。

保守担当者向けメモ

  • 本関数は PGP バージョン 4 パケット形式を前提としています。
  • internal_read_key の内部で実行される秘密鍵の復号や整合性チェック（SHA1/チェックサム）が失敗した場合、本関数もエラーを返します。
  • 鍵のメモリ管理は PGP_PubKey 構造体を通じて行われるため、構造体の変更時には pgp_key_free 等の解放処理との整合性に注意してください。
