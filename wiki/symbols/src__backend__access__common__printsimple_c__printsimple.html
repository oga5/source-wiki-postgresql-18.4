printsimple

定義

  • 種別: function
  • 定義箇所: src/backend/access/common/printsimple.c:58

目的

カタログアクセスが制限されている環境において、TupleTableSlot 内のタプルをフロントエンドへ DataRow メッセージとして送信する。

シグネチャまたは形状

bool printsimple(TupleTableSlot *slot, DestReceiver *self)

入力と出力

  • **入力**:
  • slot: 送信対象のデータを含む TupleTableSlot。
  • self: DestReceiver 構造体（本関数内では直接使用されない）。
  • **出力**:
  • 戻り値: 常に true を返す。
  • 副作用: フロントエンドへ DataRow メッセージを送信する。

前提条件と不変条件

  • プロトコルバージョン 3.0 で動作することを前提とする。
  • カタログアクセスを必要としないため、型変換はハードコードされたロジックに依存する。
  • サポート対象外の型が渡された場合、elog(ERROR) により処理が中断される。

アルゴリズムまたは振る舞い

1. slot_getallattrs(slot) を呼び出し、タプルを完全に分解する。 2. pq_beginmessage で PqMsg_DataRow メッセージを開始し、列数を送信する。 3. 各列に対して以下の処理を行う:

  • 値が NULL の場合、-1 を送信する。
  • 値が存在する場合、atttypid に基づき switch 文で型を判定する。
  • TEXTOID, INT4OID, INT8OID, OIDOID のいずれかであれば、対応する変換関数（pg_ltoa 等）で文字列化し、pq_sendcountedtext で送信する。
  • 上記以外の型の場合はエラーを発生させる。

    • 4. pq_endmessage でメッセージを終了する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/tcop/dest.c において DestReceiver インターフェースの一部として参照されている。
  • **呼び出し先**:
  • slot_getallattrs
  • pq_beginmessage, pq_sendint16, pq_sendint32, pq_sendcountedtext, pq_endmessage
  • DatumGetTextPP, DatumGetInt32, DatumGetInt64, ObjectIdGetDatum
  • pg_ltoa, pg_lltoa, pg_ultoa_n
  • elog

関連データ構造

  • TupleTableSlot: 送信されるタプルデータ。
  • TupleDesc: 列定義情報。
  • DestReceiver: 出力先インターフェース。
  • StringInfoData: メッセージ構築用バッファ。

失敗モードと境界条件

  • **サポート外の型**: atttypid が TEXTOID, INT4OID, INT8OID, OIDOID 以外の場合、elog(ERROR) が発生する。
  • **NULL 値**: slot->tts_isnull[i] が真の場合、値の代わりに -1 が送信される。

保守担当者向けメモ

  • 本関数はカタログを参照できない環境（walsender 等）専用であるため、新規の型を追加する場合は switch 文へのハードコードが必要となる。
  • pg_ltoa 等の変換関数は utils/builtins.h に依存している。
