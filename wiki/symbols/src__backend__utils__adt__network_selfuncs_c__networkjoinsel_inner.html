networkjoinsel_inner

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/network_selfuncs.c:282

目的

ネットワーク型（inet/cidr）の結合における選択率（Selectivity）を、統計情報（MCVおよびヒストグラム）に基づいて推定します。

シグネチャまたは形状

static Selectivity networkjoinsel_inner(Oid operator, int opr_codenum, VariableStatData *vardata1, VariableStatData *vardata2)

入力と出力

  • **入力**:
  • operator: 評価対象の演算子OID。
  • opr_codenum: 演算子を包含関係の数値（-2〜2）に変換した値。
  • vardata1, vardata2: 結合対象となる2つのカラムの統計情報構造体。
  • **出力**:
  • Selectivity (double): 推定された選択率（0.0〜1.0）。

前提条件と不変条件

  • vardata1 および vardata2 に含まれる statsTuple が有効な場合、統計情報（MCV、ヒストグラム）を読み込みます。
  • MAX_CONSIDERED_ELEMS（1024）を超えるMCVは切り捨てられ、計算対象から除外されます。
  • 統計情報が不足している場合は、デフォルトの選択率推定値が使用されます。

アルゴリズムまたは振る舞い

1. **統計情報の取得**: get_attstatsslot を使用し、両カラムのMCVおよびヒストグラムを取得します。 2. **MCV vs MCV**: 両方にMCVが存在する場合、inet_mcv_join_sel で選択率を算出します。 3. **MCV vs ヒストグラム**: 片方がMCV、もう片方がヒストグラムの場合、inet_mcv_hist_sel を適用し、NULLフラクションとMCVを除いた母集団の割合でスケーリングします。 4. **ヒストグラム vs ヒストグラム**: 両方にヒストグラムが存在する場合、inet_hist_inclusion_join_sel を適用し、同様にスケーリングします。 5. **デフォルト推定**: 統計情報が不十分な場合、DEFAULT_SEL(operator) にNULLフラクションを考慮した補正を加えて算出します。 6. **リソース解放**: 取得した統計スロットを free_attstatsslot で解放します。

呼び出し元と呼び出し先

  • **呼び出し元**: networkjoinsel（src/backend/utils/adt/network_selfuncs.c 内）から呼び出されます。
  • **呼び出し先**: get_attstatsslot, mcv_population, inet_mcv_join_sel, inet_mcv_hist_sel, inet_hist_inclusion_join_sel, free_attstatsslot 等。

関連データ構造

  • VariableStatData: カラムの統計情報を保持する構造体。
  • AttStatsSlot: pg_statistic から抽出された個別の統計データ（MCV値、ヒストグラム値など）。

失敗モードと境界条件

  • 統計情報が全く存在しない場合、デフォルトの選択率推定値が適用されます。
  • mcv_length は MAX_CONSIDERED_ELEMS によって制限され、大規模な統計データに対しても計算コストを一定に保ちます。

保守担当者向けメモ

  • 本関数は networkjoinsel の内部ロジックを分離したものです。
  • 包含関係の推定精度は inet_hist_inclusion_join_sel に依存しています。推定値が不自然な場合は、対象カラムの pg_stats の内容を確認してください。
