bloom_free

定義

  • 種別: function
  • 定義箇所: src/backend/lib/bloomfilter.c:125

目的

ブルームフィルタ構造体に関連付けられたメモリを解放します。

シグネチャまたは形状

void bloom_free(bloom_filter *filter)

入力と出力

  • **入力**: bloom_filter *filter - 解放対象のブルームフィルタ構造体へのポインタ。
  • **出力**: なし（void）。

前提条件と不変条件

  • filter は bloom_create によって割り当てられた有効なポインタである必要があります。

アルゴリズムまたは振る舞い

pfree を呼び出し、指定された bloom_filter 構造体が使用しているメモリを解放します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/amcheck/verify_nbtree.c:596
  • src/backend/utils/adt/acl.c:5253
  • src/test/modules/test_bloomfilter/test_bloomfilter.c:100
  • **呼び出し先**:
  • pfree（PostgreSQL メモリ管理関数）

関連データ構造

  • bloom_filter: 本関数が解放対象とするデータ構造。

失敗モードと境界条件

  • filter が NULL の場合の挙動は、pfree の実装に依存します。

保守担当者向けメモ

  • 本関数は単に pfree をラップしているのみです。フィルタ内部で個別に動的メモリ確保を行っているメンバが存在しないことを前提としています。
