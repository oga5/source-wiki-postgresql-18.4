basic_archive_configured

定義

  • 種別: function
  • 定義箇所: contrib/basic_archive/basic_archive.c:124

目的

アーカイブモジュールが使用可能かどうかを判定する。具体的には、アーカイブ先ディレクトリが設定されているかを確認する。

シグネチャまたは形状

static bool basic_archive_configured(ArchiveModuleState *state)

入力と出力

  • **入力**: ArchiveModuleState *state（アーカイブモジュールの状態構造体）
  • **出力**: bool 型。設定が有効であれば true、無効であれば false を返す。

前提条件と不変条件

  • GUC変数 archive_directory が初期化されていること。
  • archive_directory が空文字列でない場合、設定済みとみなされる。

アルゴリズムまたは振る舞い

1. archive_directory が NULL ではなく、かつ空文字列でないかを確認する。 2. 条件を満たす場合は true を返す。 3. 条件を満たさない場合は、arch_module_check_errdetail を呼び出してエラー詳細を設定し、false を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: basic_archive_callbacks 構造体（check_configured_cb フィールド）を介して、PostgreSQLのアーカイブプロセスから呼び出される。
  • **呼び出し先**: arch_module_check_errdetail（エラー詳細設定用関数）。

関連データ構造

  • ArchiveModuleState: アーカイブモジュールの状態を保持する構造体。
  • archive_directory: アーカイブ先ディレクトリパスを保持するGUC変数。

失敗モードと境界条件

  • archive_directory が未設定（NULL）または空文字列の場合、アーカイブは構成されていないと判断され、エラー詳細が記録される。

保守担当者向けメモ

  • 本関数はアーカイブモジュールの初期化チェックの一部として機能する。
  • arch_module_check_errdetail を使用して、設定不足の理由をログに出力する設計となっている。
