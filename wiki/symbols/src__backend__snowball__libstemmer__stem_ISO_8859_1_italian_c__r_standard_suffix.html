r_standard_suffix

定義

  • **種別**: function
  • **定義箇所**: src/backend/snowball/libstemmer/stem_ISO_8859_1_italian.c:744

目的

イタリア語のステミング処理において、単語の末尾から標準的な接尾辞を特定し、除去または置換を行うことで語幹を正規化します。

シグネチャまたは形状

static int r_standard_suffix(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（Snowball ステミング環境構造体）
  • **出力**: int（成功時に 1、失敗または処理中断時に 0 または負の値を返す）

前提条件と不変条件

  • z->c（現在のカーソル位置）が単語の末尾付近を指している必要があります。
  • 内部で r_R1 や r_R2 を呼び出し、接尾辞の除去が定義された領域（R1, R2）内で行われることを保証します。

アルゴリズムまたは振る舞い

1. find_among_b を使用し、配列 a_6 に定義された 51 個の接尾辞パターンを単語末尾から逆方向に検索します。 2. マッチした接尾辞の種類（among_var）に応じて switch 文で分岐します。 3. 各ケースにおいて、以下の処理を順次実行します：

  • 領域判定（r_R2 または r_RV）を行い、条件を満たす場合のみ処理を継続します。
  • slice_del（削除）または slice_from_s（置換）を実行します。

    • 4. 一部のケース（ケース 2, 7, 8, 9）では、接尾辞除去後にさらなる後続の接尾辞マッチング（eq_s_b や find_among_b）を試みる再帰的なロジックが含まれます。

呼び出し元と呼び出し先

  • **呼び出し元**: italian_ISO_8859_1_stem（直接的または間接的）
  • **呼び出し先**:
  • find_among_b（接尾辞検索）
  • r_R1, r_R2, r_RV（領域判定）
  • slice_del, slice_from_s（文字列操作）
  • eq_s_b（文字列比較）

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_6: 接尾辞検索用の among 構造体配列。

失敗モードと境界条件

  • find_among_b がマッチしない場合、直ちに 0 を返して終了します。
  • 領域判定（r_R1 等）が失敗した場合、その時点での戻り値（0 または負の値）を伝播させます。
  • slice_del 等の文字列操作関数がエラー（負の値）を返した場合、そのエラーコードを呼び出し元に返します。
