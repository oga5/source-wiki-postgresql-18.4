PLy_warning

定義

  • 種別: function
  • 定義箇所: src/pl/plpython/plpy_plpymodule.c:301

目的

Python スクリプトから呼び出された際に、PostgreSQL の WARNING レベルのログメッセージを出力するインターフェースを提供します。

シグネチャまたは形状

static PyObject *PLy_warning(PyObject *self, PyObject *args, PyObject *kw);

入力と出力

  • **入力**:
  • self: Python モジュールオブジェクト。
  • args: Python から渡された位置引数（タプル）。
  • kw: Python から渡されたキーワード引数（辞書）。
  • **出力**:
  • PyObject *: PLy_output の戻り値（通常は Py_None）。

前提条件と不変条件

  • 本関数は plpy モジュールのメソッドとして登録されている必要があります。
  • 内部で呼び出される PLy_output が PostgreSQL のエラー報告メカニズム（ereport）と正しく連携できる状態である必要があります。

アルゴリズムまたは振る舞い

1. 引数として受け取った self、args、kw をそのまま PLy_output 関数に渡します。 2. その際、ログレベルとして WARNING を指定します。 3. PLy_output が実行した ereport(WARNING, ...) の結果を呼び出し元に返します。

呼び出し元と呼び出し先

  • **呼び出し元**: Python インタプリタ（plpy.warning() メソッド経由）。
  • **呼び出し先**: PLy_output（src/pl/plpython/plpy_plpymodule.c 内の共通ログ出力関数）。

関連データ構造

  • PyObject: Python C API の汎用オブジェクト型。

失敗モードと境界条件

  • 引数の解析に失敗した場合や、PostgreSQL 側でエラーが発生した場合は、PLy_output を通じて Python 例外が送出されます。

保守担当者向けメモ

  • 本関数は PLy_output への単なるラッパーです。ログ出力のロジックを変更する場合は PLy_output を参照してください。
  • src/pl/plpython/plpy_plpymodule.c:63 にて METH_VARARGS | METH_KEYWORDS フラグ付きでメソッドテーブルに登録されています。
