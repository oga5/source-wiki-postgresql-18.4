ExecEndProjectSet

定義

  • 種別: function
  • 定義箇所: src/backend/executor/nodeProjectSet.c:327

目的

ProjectSet 実行ノードの終了処理を行い、関連するリソースを解放する。

シグネチャまたは形状

void ExecEndProjectSet(ProjectSetState *node)

入力と出力

  • **入力**: ProjectSetState *node - 終了対象となる ProjectSet ノードの実行時状態構造体。
  • **出力**: なし（void）。

前提条件と不変条件

  • 引数 node は有効な ProjectSetState へのポインタであること。
  • 本関数は、当該ノードの実行が完了した後に呼び出されることを想定している。

アルゴリズムまたは振る舞い

1. outerPlanState(node) を介して、ProjectSet ノードが保持する子プランノード（サブプラン）を取得する。 2. ExecEndNode を呼び出し、子プランノードの終了処理を再帰的に実行する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/executor/execProcnode.c:593
  • **呼び出し先**:
  • ExecEndNode（子プランノードの終了処理）

関連データ構造

  • ProjectSetState: ProjectSet ノードの実行時状態を保持する構造体。

失敗モードと境界条件

  • 子プランノードが NULL の場合、ExecEndNode の実装に依存する（通常は安全に処理される）。

保守担当者向けメモ

  • 本関数は主に子プランのクリーンアップを委譲する役割を持つ。ProjectSet ノード固有のメモリコンテキスト（argcontext 等）の明示的な解放処理が ExecEndNode 側で完結しているか、あるいは本関数で追加の解放が必要かを確認すること。
