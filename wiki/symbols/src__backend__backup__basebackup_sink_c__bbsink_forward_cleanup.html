bbsink_forward_cleanup

定義

  • 種別: function
  • 定義箇所: src/backend/backup/basebackup_sink.c:120

目的

bbsink チェーンにおけるクリーンアップ処理を、現在のシンクから後続のシンク（bbs_next）へ転送する。

シグネチャまたは形状

void bbsink_forward_cleanup(bbsink *sink)

入力と出力

  • **入力**: sink (操作対象の bbsink オブジェクトへのポインタ)
  • **出力**: なし（void）

前提条件と不変条件

  • sink->bbs_next が NULL であってはならない（Assert により検証される）。

アルゴリズムまたは振る舞い

1. sink->bbs_next が存在することを確認する。 2. bbsink_cleanup(sink->bbs_next) を呼び出し、チェーン内の次のシンクに対してクリーンアップ処理を委譲する。

呼び出し元と呼び出し先

  • **呼び出し元**: 以下の各実装において bbsink_ops 構造体の .cleanup フィールドとして参照されている。
  • contrib/basebackup_to_shell/basebackup_to_shell.c
  • src/backend/backup/basebackup_gzip.c
  • src/backend/backup/basebackup_progress.c
  • src/backend/backup/basebackup_server.c
  • src/backend/backup/basebackup_throttle.c
  • **呼び出し先**: bbsink_cleanup（sink->bbs_next を引数として実行）

関連データ構造

  • bbsink: バックアップデータの転送パイプラインを構成する抽象構造体。

失敗モードと境界条件

  • sink->bbs_next が NULL の場合、アサーション違反によりプロセスが異常終了する。

保守担当者向けメモ

  • 本関数は、独自のシンクを実装する際に、自身で特別なクリーンアップ処理が不要で、単に後続へ処理を流すだけでよい場合に利用する。
