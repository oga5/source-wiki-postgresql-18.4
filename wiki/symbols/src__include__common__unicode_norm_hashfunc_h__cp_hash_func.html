cp_hash_func

目的

Unicode 正規化処理（分解および再合成）において、文字コードをハッシュ値へ変換するための関数ポインタ型を定義します。

シグネチャまたは形状

typedef int (*cp_hash_func) (const void *key);

入力と出力

  • **入力**: key（const void * 型）。正規化対象の文字コードを指すポインタ。
  • **出力**: int 型。計算されたハッシュ値（ルックアップテーブルのインデックスとして使用）。

前提条件と不変条件

  • 本型に基づく関数は、特定のデータセットに対して衝突が発生しない「完全ハッシュ（Perfect Hash）」として設計される必要があります。
  • 入力 key は、対応する正規化テーブルの仕様に従った有効な文字コードである必要があります。

アルゴリズムまたは振る舞い

  • 入力された文字コードに対して、乗算と加算を用いた計算を行い、静的配列 h のオフセット値を取得するためのハッシュ値を生成します。
  • 実際の計算ロジックは、この型を実装する具体的な関数（Decomp_hash_func や Recomp_hash_func）に依存します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/include/common/unicode_norm_hashfunc.h 内の構造体（UnicodeDecompInfo や UnicodeRecompInfo）が本型を保持しています。具体的な呼び出し元コードの詳細は提供されていません。
  • **呼び出し先**: 不明。

関連データ構造

  • UnicodeDecompInfo: 分解処理用の情報を保持する構造体。
  • UnicodeRecompInfo: 再合成処理用の情報を保持する構造体。
  • h 配列: ハッシュ値を用いて参照される静的ルックアップテーブル。

失敗モードと境界条件

  • 完全ハッシュ関数であるため、設計上のデータセット内では衝突が発生しない前提ですが、範囲外の入力に対する挙動は定義されていません。
