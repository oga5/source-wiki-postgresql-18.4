r_Prefix_Step2

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_arabic.c:1082

目的

アラビア語のステミング処理において、特定の接頭辞パターンを識別し、条件を満たす場合にそれを除去する。

シグネチャまたは形状

static int r_Prefix_Step2(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（Snowball ステミング環境構造体）
  • **出力**: int 型。処理が成功し接頭辞が除去された場合は 1、条件不一致や処理対象外の場合は 0、エラー発生時は負の値を返す。

前提条件と不変条件

  • z->c（現在のカーソル位置）が有効な範囲にあること。
  • z->p（文字列バッファ）が UTF-8 エンコーディングであること。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を z->bra に設定する。 2. カーソル位置の次（z->c + 1）の文字コードが 129 または 136 であるかを確認し、該当しない場合は終了する。 3. a_5 テーブルを用いて接頭辞の検索を行い、一致しない場合は終了する。 4. 一致した場合、現在のカーソル位置を z->ket に設定する。 5. 文字列全体の長さが 3 文字以下である場合は終了する。 6. 特定の文字列 s_58 と一致するかを確認し、一致する場合は除去を行わずに終了する。 7. 上記条件をすべて満たす場合、slice_del(z) を呼び出して該当部分を削除し、成功として 1 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_UTF_8_arabic.c:1613（直接の呼び出しを確認）
  • **呼び出し先**: find_among（テーブル検索）、len_utf8（文字数判定）、eq_s（文字列比較）、slice_del（文字列削除）。

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_5: 接頭辞マッチングに使用される among 構造体のテーブル。

失敗モードと境界条件

  • 文字列長が 3 文字以下の場合は削除を行わない。
  • a_5 テーブルに一致するパターンがない場合は削除を行わない。
  • slice_del が内部エラー（負の戻り値）を返した場合、その値をそのまま伝播させる。
