conditional_stack_reset

定義

  • 種別: function
  • 定義箇所: src/fe_utils/conditional.c:29

目的

ConditionalStack 内のすべての要素をスタックから取り除き、解放する。スタック構造体自体のメモリは解放しない。

シグネチャまたは形状

void conditional_stack_reset(ConditionalStack cstack)

入力と出力

  • **入力**: cstack (操作対象の ConditionalStack 構造体へのポインタ)
  • **出力**: なし（void）

前提条件と不変条件

  • cstack が NULL の場合、関数は何もせず正常に終了する。
  • 実行後、スタックは空の状態（conditional_stack_empty(cstack) が真を返す状態）となる。

アルゴリズムまたは振る舞い

1. cstack が NULL かどうかを確認し、該当する場合は即座にリターンする。 2. conditional_stack_pop(cstack) が真を返す（スタックに要素が存在する）限り、ループ内で pop を繰り返し実行する。 3. スタックが空になった時点で終了する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/bin/pgbench/pgbench.c:4117 にて呼び出しを確認。
  • **呼び出し先**: conditional_stack_pop をループ内で呼び出す。

関連データ構造

  • ConditionalStack: \if 文のネスト状態を管理するスタック構造体。

失敗モードと境界条件

  • cstack が NULL の場合、安全に無視される。
  • スタックが既に空である場合、ループは一度も実行されず即座に終了する。

保守担当者向けメモ

  • 本関数はスタック内の要素をクリアするのみであり、スタック構造体自体のメモリ解放が必要な場合は conditional_stack_destroy を使用すること。
