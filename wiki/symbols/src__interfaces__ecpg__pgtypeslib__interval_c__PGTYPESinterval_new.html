PGTYPESinterval_new

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/pgtypeslib/interval.c:986

目的

ECPG ライブラリにおいて、interval 構造体を格納するためのメモリを動的に確保する。

シグネチャまたは形状

interval *PGTYPESinterval_new(void)

入力と出力

  • **入力**: なし
  • **出力**: 確保された interval 構造体へのポインタ。メモリ不足時は NULL を返す。

前提条件と不変条件

  • 戻り値が NULL でない場合、呼び出し元は PGTYPESinterval_free を使用してメモリを解放する責任を負う。

アルゴリズムまたは振る舞い

1. pgtypes_alloc を呼び出し、sizeof(interval) 分のメモリ領域を確保する。 2. 確保した領域へのポインタを返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/interfaces/ecpg/test/expected/pgtypeslib-dt_test2.c
  • src/interfaces/ecpg/test/expected/sql-array.c
  • **呼び出し先**:
  • pgtypes_alloc (内部的なメモリ確保関数)

関連データ構造

  • interval: ECPG で使用される時間間隔の内部表現構造体。

失敗モードと境界条件

  • メモリ不足時には pgtypes_alloc が NULL を返すため、本関数も NULL を返す。

保守担当者向けメモ

  • 本関数は interval 型のメモリ管理における標準的なエントリポイントである。
  • pgtypes_alloc の実装に依存しているため、メモリ管理ポリシーの変更時には注意が必要である。
