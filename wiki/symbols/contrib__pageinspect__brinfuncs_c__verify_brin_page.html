verify_brin_page

定義

  • 種別: function
  • 定義箇所: contrib/pageinspect/brinfuncs.c:92

目的

入力された生のページデータ（bytea）が、指定された型と一致する有効な BRIN インデックスページであるかを検証する。

シグネチャまたは形状

static Page verify_brin_page(bytea *raw_page, uint16 type, const char *strtype)

入力と出力

  • **入力**:
  • raw_page: 検査対象の生のページデータ。
  • type: 期待される BRIN ページの型（BRIN_PAGETYPE_REGULAR 等）。
  • strtype: エラーメッセージに使用する型名の文字列。
  • **出力**:
  • 成功時: 変換された Page ポインタを返す。
  • 失敗時: ereport(ERROR, ...) により処理を中断する。

前提条件と不変条件

  • get_page_from_raw を介して raw_page が Page 構造体に変換可能であること。
  • ページが空（PageIsNew）でない場合、PageGetSpecialSize が BrinSpecialSpace のサイズと一致しなければならない。
  • ページの BrinPageType が引数 type と一致しなければならない。

アルゴリズムまたは振る舞い

1. get_page_from_raw を呼び出し、bytea を Page に変換する。 2. ページが新規（PageIsNew）であれば、検証をスキップしてそのまま返す。 3. ページの特殊領域サイズが BrinSpecialSpace のサイズと一致するか確認し、不一致ならエラーを投げる。 4. ページの型が引数 type と一致するか確認し、不一致ならエラーを投げる。 5. 検証済みページを返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/pageinspect/brinfuncs.c 内の以下の関数から呼び出される。
  • brin_page_items (regular ページ検証)
  • brin_metapage_info (metapage 検証)
  • brin_revmap_data (revmap 検証)
  • **呼び出し先**:
  • get_page_from_raw
  • PageIsNew
  • PageGetSpecialSize
  • BrinPageType
  • ereport

関連データ構造

  • Page: PostgreSQL のページ構造体。
  • BrinSpecialSpace: BRIN インデックスページ固有の特殊領域構造体。

失敗モードと境界条件

  • PageIsNew の場合は検証を通過する。
  • 特殊領域サイズが期待値と異なる場合、ERRCODE_INVALID_PARAMETER_VALUE でエラーとなる。
  • ページ型が期待値と異なる場合、ERRCODE_INVALID_PARAMETER_VALUE でエラーとなる。

保守担当者向けメモ

  • 本関数は pageinspect 拡張の各 BRIN 関数における共通のバリデーションロジックである。
  • 権限チェックは本関数の呼び出し元（SQL 関数レベル）で行われるため、本関数内では実施されない。
