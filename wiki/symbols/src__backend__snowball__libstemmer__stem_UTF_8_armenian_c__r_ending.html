r_ending

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_armenian.c:505

目的

アルメニア語のステミング処理において、単語末尾の特定の接尾辞を検索し、条件を満たす場合にそれを削除する。

シグネチャまたは形状

static int r_ending(struct SN_env * z)

入力と出力

  • **入力**: z (ステミング環境を保持する SN_env 構造体へのポインタ)
  • **出力**: 成功した場合は 1、失敗した場合は 0、エラー発生時は負の値を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要がある。
  • find_among_b を使用するため、単語の末尾から前方に向かって処理が行われる。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置を z->ket に設定する。 2. a_3 配列（57個の要素）を用いて、単語末尾から接尾辞を検索する (find_among_b)。 3. 一致した場合、その位置を z->bra に設定する。 4. r_R2(z) を呼び出し、一致した箇所が R2 領域内にあるかを確認する。 5. R2 領域内であれば、slice_del(z) を実行して該当する接尾辞を削除する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_UTF_8_armenian.c 内のメインステミング処理（armenian_UTF_8_stem 等）から呼び出される。
  • **呼び出し先**:
  • find_among_b: 接尾辞のパターンマッチング。
  • r_R2: 領域判定。
  • slice_del: 文字列の削除。

関連データ構造

  • SN_env: ステミング環境を管理する構造体。
  • a_3: 接尾辞のパターンを格納した among 構造体の配列。

失敗モードと境界条件

  • find_among_b が一致しない場合、0 を返して終了する。
  • r_R2 または slice_del がエラー（負の値）を返した場合、その値をそのまま伝播させる。
