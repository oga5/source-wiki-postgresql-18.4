macaddr_cmp_internal

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/mac.c:181

目的

2つの macaddr 型データ（EUI-48）を比較し、その順序関係を判定する内部関数です。

シグネチャまたは形状

static int macaddr_cmp_internal(macaddr *a1, macaddr *a2)

入力と出力

  • **入力**: 比較対象となる2つの macaddr 構造体へのポインタ (a1, a2)。
  • **出力**: 比較結果を示す整数。
  • -1: a1 < a2
  • 1: a1 > a2
  • 0: a1 == a2

前提条件と不変条件

  • 入力ポインタ a1 および a2 は有効な macaddr 構造体を指している必要があります。
  • 比較は hibits（上位ビット）および lobits（下位ビット）マクロを使用して行われます。

アルゴリズムまたは振る舞い

1. hibits(a1) と hibits(a2) を比較し、大小関係があれば即座に結果を返します。 2. 上位ビットが等しい場合、lobits(a1) と lobits(a2) を比較し、その結果を返します。 3. 両方のビット群が等しい場合は 0 を返します。

  • 6バイトのMACアドレスを2つの32ビット値に分割して比較することで、効率的な比較を実現しています。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • macaddr_cmp (SQL関数 macaddr_cmp の実装)
  • macaddr_lt, macaddr_le, macaddr_eq, macaddr_ge, macaddr_gt, macaddr_ne (各比較演算子)
  • その他、src/backend/utils/adt/mac.c 内の比較を必要とする関数群。
  • **呼び出し先**:
  • hibits() マクロ
  • lobits() マクロ

関連データ構造

  • macaddr: 6バイトのMACアドレスを保持する構造体。

失敗モードと境界条件

  • 入力ポインタが NULL の場合の挙動は定義されていません。
  • 6バイトすべてが一致する場合、正しく 0 を返します。

保守担当者向けメモ

  • 本関数は macaddr 型のすべての比較演算の基盤となるため、変更時はソート順序への影響に注意してください。
  • hibits および lobits マクロの定義が、プラットフォームのエンディアンに対して適切に機能していることを前提としています。
