macaddr_cmp

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/mac.c:196

目的

2つの macaddr 型の値を比較し、その順序関係を整数値で返します。

シグネチャまたは形状

Datum macaddr_cmp(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS を介して2つの macaddr 構造体へのポインタ（a1, a2）を受け取ります。
  • **出力**: 比較結果を int32 型で返します（a1 < a2 なら負、a1 == a2 なら 0、a1 > a2 なら正）。

前提条件と不変条件

  • 入力される macaddr ポインタは有効である必要があります。
  • 内部的に macaddr_cmp_internal を呼び出し、6バイトのMACアドレスを比較します。

アルゴリズムまたは振る舞い

1. PG_GETARG_MACADDR_P マクロを使用して、引数から macaddr 構造体へのポインタを取得します。 2. macaddr_cmp_internal 関数を呼び出し、比較ロジックを実行します。 3. PG_RETURN_INT32 を使用して、比較結果を Datum 型として返します。

呼び出し元と呼び出し先

  • **呼び出し先**: macaddr_cmp_internal
  • **呼び出し元**:
  • contrib/btree_gin/btree_gin.c (GINインデックスサポート)
  • contrib/btree_gist/btree_macaddr.c (GiSTインデックスサポート)

関連データ構造

  • macaddr: 6バイトのMACアドレスを保持する構造体。

失敗モードと境界条件

  • 入力ポインタが NULL の場合、PG_GETARG_MACADDR_P の挙動に依存します（通常はエラーとなります）。

保守担当者向けメモ

  • 本関数は単なるラッパーであり、実際の比較ロジックは macaddr_cmp_internal に集約されています。変更が必要な場合は、まず macaddr_cmp_internal の実装を確認してください。
