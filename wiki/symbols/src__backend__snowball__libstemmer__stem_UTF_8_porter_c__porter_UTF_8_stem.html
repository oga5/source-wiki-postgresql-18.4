porter_UTF_8_stem

定義

  • **種別**: 関数
  • **定義箇所**: src/backend/snowball/libstemmer/stem_UTF_8_porter.c:561

目的

Porter Stemming アルゴリズムに基づき、UTF-8 エンコードされた英語単語を語幹（ステミング）に変換する。

シグネチャまたは形状

int porter_UTF_8_stem(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（ステミング処理に必要な状態、バッファ、カーソル位置を保持する構造体）。
  • **出力**: 成功時に 1 を返し、エラー発生時に負の値を返す。

前提条件と不変条件

  • SN_env 構造体は、porter_UTF_8_create_env 等で適切に初期化されている必要がある。
  • 処理対象の文字列は UTF-8 エンコードされていること。
  • 内部的に SN_env の I 配列（整数ワークスペース）および c（カーソル）、bra/ket（置換範囲）が操作される。

アルゴリズムまたは振る舞い

1. **前処理**: 単語内の 'y' を 'Y' に置換し、母音・子音のパターンに基づいて領域 R1, R2 を特定する。 2. **段階的処理**: 以下の順序で接尾辞の削除・置換を行う。

  • r_Step_1a, r_Step_1b, r_Step_1c
  • r_Step_2, r_Step_3, r_Step_4
  • r_Step_5a, r_Step_5b

    • 3. **後処理**: 前処理で置換した 'Y' を元の 'y' に戻す。 4. 各ステップは SN_env の状態を更新しながら、接尾辞のマッチングと slice_from_s による置換を実行する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/include/snowball/libstemmer/stem_UTF_8_porter.h および src/backend/snowball/libstemmer/stem_UTF_8_porter.c に宣言あり。具体的な呼び出し元コードの詳細は提供されていない。
  • **呼び出し先**: r_Step_1a ～ r_Step_5b（各ステミング段階）、slice_from_s、skip_utf8、in_grouping_U、out_grouping_U。

関連データ構造

  • struct SN_env: ステミング環境を保持する主要な構造体。

失敗モードと境界条件

  • slice_from_s や skip_utf8 等の内部関数が負の値を返した場合、そのエラーコードをそのまま呼び出し元へ伝播させる。
  • 文字列の末尾やカーソル位置が不正な場合、各ステップ内のガード節により処理がスキップされる。
