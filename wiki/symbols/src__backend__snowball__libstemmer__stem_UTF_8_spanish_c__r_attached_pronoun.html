r_attached_pronoun

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_spanish.c:656

目的

単語の末尾に付加されたスペイン語の代名詞を特定し、削除または置換を行うことで、語幹抽出の前処理を行う。

シグネチャまたは形状

static int r_attached_pronoun(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（Snowballランタイム環境構造体）
  • **出力**: int（成功時に 1、失敗または条件不一致時に 0、エラー時に負の値を返す）

前提条件と不変条件

  • z->c（現在のカーソル位置）が有効な文字列範囲内にあること。
  • r_RV 関数により、対象領域がRV（語幹の開始位置）内にあることが検証される。

アルゴリズムまたは振る舞い

1. **後方検索**: find_among_b を使用し、代名詞候補（a_1）を単語末尾から検索する。 2. **検証**: 代名詞の直前にある文字が特定の条件（a_2 に定義された文字セット）を満たすか確認する。 3. **領域判定**: r_RV を呼び出し、対象がRV領域内であることを保証する。 4. **置換・削除**: among_var の値に基づき、以下のいずれかの処理を行う。

  • slice_from_s: 特定の文字列へ置換する（ケース1～5）。
  • slice_del: 文字列を削除する（ケース6）。
  • slice_del（'u'を伴う場合）: 末尾の'u'を削除した上で、さらに文字列を削除する（ケース7）。

呼び出し元と呼び出し先

  • **呼び出し元**: spanish_UTF_8_stem（stem_UTF_8_spanish.c 内のメイン関数）から呼び出される。また、同名の関数が stem_ISO_8859_1_catalan.c, stem_ISO_8859_1_italian.c, stem_ISO_8859_1_spanish.c 等の各言語ステミング実装からも参照されている。
  • **呼び出し先**: find_among_b, r_RV, slice_from_s, slice_del（いずれも header.h 経由で提供されるランタイム関数）。

関連データ構造

  • SN_env: ステミング処理の状態を保持する構造体。
  • a_1, a_2: 接尾辞や置換ルールを定義した among 構造体の配列。

失敗モードと境界条件

  • 文字列が短すぎる場合や、find_among_b で一致が見つからない場合は 0 を返す。
  • r_RV の判定に失敗した場合、その戻り値をそのまま返す。
  • slice_from_s や slice_del がメモリ操作等で失敗した場合は負の値を返す。

保守担当者向けメモ

  • 本関数は自動生成コードであるため、直接修正せず、Snowballのソース（.sbl）を修正すること。
  • among_var の各ケースは、スペイン語の代名詞付加ルールに対応している。ロジックの変更時は a_1, a_2 テーブルの定義と整合性を確認すること。
