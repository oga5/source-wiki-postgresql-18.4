calc_hist_selectivity

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/multirangetypes_selfuncs.c:455

目的

境界値ヒストグラム（STATISTIC_KIND_BOUNDS_HISTOGRAM）およびレンジ長ヒストグラム（STATISTIC_KIND_RANGE_LENGTH_HISTOGRAM）を利用して、マルチレンジ型に対する演算子の選択率を推定します。

シグネチャまたは形状

static double calc_hist_selectivity(TypeCacheEntry *typcache, VariableStatData *vardata, const MultirangeType *constval, Oid operator);

入力と出力

  • **入力**:
  • typcache: 型キャッシュエントリ。
  • vardata: 統計情報を含む変数データ。
  • constval: 比較対象のマルチレンジ定数。
  • operator: 評価対象の演算子OID。
  • **出力**:
  • 成功時: 推定された選択率（double 型、0.0〜1.0）。
  • 失敗時: -1.0（統計情報不足やセキュリティチェック失敗時）。

前提条件と不変条件

  • vardata->statsTuple が有効であること。
  • 境界値ヒストグラムには空のレンジが含まれていないこと（含まれている場合はエラーをスロー）。
  • 統計処理に使用されるサブタイプ比較関数および差分関数がセキュリティチェックを通過すること。

アルゴリズムまたは振る舞い

1. **セキュリティ検証**: statistic_proc_security_check を用いて、型に関連する比較・差分関数が安全か確認します。 2. **統計取得**: pg_statistic から境界値ヒストグラムを取得します。 3. **ヒストグラム変換**: 取得したレンジヒストグラムを、下限値と上限値の配列に分解します。 4. **レンジ長ヒストグラム**: 包含演算子（@> 等）の場合、追加でレンジ長ヒストグラムを取得します。 5. **定数分解**: multirange_get_bounds を使用して、定数マルチレンジの最初と最後の境界値を抽出します。 6. **選択率計算**: 演算子に応じて calc_hist_selectivity_scalar を呼び出し、境界値の比較に基づいた確率を算出します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/adt/multirangetypes_selfuncs.c 内の calc_multirangesel から呼び出されます。
  • **呼び出し先**:
  • calc_hist_selectivity_scalar: 境界値の比較による選択率計算。
  • range_deserialize: レンジ型の分解。
  • get_attstatsslot: 統計情報の取得。
  • multirange_get_bounds: マルチレンジ定数の境界値取得。

関連データ構造

  • TypeCacheEntry: 型情報のキャッシュ。
  • VariableStatData: 統計情報のコンテナ。
  • RangeBound: レンジの境界値表現。
  • AttStatsSlot: 統計スロットデータ。

失敗モードと境界条件

  • 統計情報が取得できない場合や、ヒストグラムの要素数が2未満の場合は -1.0 を返します。
  • セキュリティチェックに失敗した場合、-1.0 を返します。
  • ヒストグラム内に空のレンジが含まれている場合、elog(ERROR) により処理を中断します。

保守担当者向けメモ

  • 本関数は rangetypes_selfuncs.c にある同名関数とロジックを共有しています。
  • 包含演算子（@> 等）の計算にはレンジ長ヒストグラムが必須ですが、それ以外の演算子では memset で初期化された空のスロットが使用されます。
  • 統計的な仮定として、レンジの長さと下限値は独立しているものとして扱われます。
