目的

単語内の現在のカーソル位置が、定義済みの領域「R1」の開始位置以降にあるかどうかを判定する。

シグネチャまたは形状

static int r_R1(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z （Snowball実行環境へのポインタ）
  • 出力: int （条件を満たす場合は 1、それ以外は 0）

前提条件と不変条件

  • z->I[1] には、r_mark_regions によって事前に計算された R1 領域の開始インデックスが格納されている必要がある。
  • z->c は現在のカーソル位置を示す。

アルゴリズムまたは振る舞い

z->I[1]（R1の開始位置）と z->c（現在のカーソル位置）を比較し、z->I[1] <= z->c であれば真（1）を返す。これにより、現在の処理対象が R1 領域内にあるか、またはそれ以降にあるかを判定する。

  • 呼び出し先: なし（算術比較のみを実行）。
  • 補足: 他の言語実装（Basque, Catalan, Dutch等）でも同名の関数が定義されており、共通のインターフェースとして利用されている。

関連データ構造

  • struct SN_env: Snowballの実行環境。整数配列 I を保持し、I[1] に領域 R1 の境界情報が格納される。
