pgstat_reset_slru

定義

  • 種別: function
  • 定義箇所: src/backend/utils/activity/pgstat_slru.c:44

目的

指定された名前のSLRU（Simple Least Recently Used）キャッシュに関連する統計情報をリセットします。

シグネチャまたは形状

void pgstat_reset_slru(const char *name)

入力と出力

  • **入力**: name (SLRUの名前を示す文字列へのポインタ)
  • **出力**: なし（副作用として共有メモリ上の統計データが更新されます）

前提条件と不変条件

  • name は NULL であってはなりません（Assert により検証されます）。
  • 指定された name は、システムが認識可能なSLRU名である必要があります。

アルゴリズムまたは振る舞い

1. GetCurrentTimestamp() を呼び出し、現在時刻を取得します。 2. pgstat_get_slru_index(name) を呼び出し、名前から対応するSLRUインデックスを特定します。 3. 取得したインデックスと時刻を引数として pgstat_reset_slru_counter_internal を呼び出し、統計カウンタをリセットします。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/pgstatfuncs.c:1998 (SQL関数経由の呼び出し)
  • **呼び出し先**:
  • GetCurrentTimestamp()
  • pgstat_get_slru_index(const char *)
  • pgstat_reset_slru_counter_internal(int, TimestampTz)

関連データ構造

  • TimestampTz: リセット時刻を記録するために使用されます。

失敗モードと境界条件

  • name が NULL の場合、アサーション違反によりプロセスが異常終了します。
  • pgstat_get_slru_index が未知の名前を受け取った場合の挙動は、当該関数の実装（「その他」用インデックスへの集約）に依存します。

保守担当者向けメモ

  • 本関数は統計リセットのインターフェースであり、実際のカウンタ操作は pgstat_reset_slru_counter_internal に委譲されています。
  • 統計の整合性を保つため、共有メモリへのアクセスは内部で適切にロック制御されている必要があります。
