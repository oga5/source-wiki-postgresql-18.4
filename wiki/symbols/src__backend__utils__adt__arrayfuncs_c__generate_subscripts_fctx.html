generate_subscripts_fctx

定義

  • 種別: struct
  • 定義箇所: src/backend/utils/adt/arrayfuncs.c:5918

目的

generate_subscripts 関数が実行中に保持する、配列の添字生成のための状態コンテキストを格納します。

シグネチャまたは形状

typedef struct generate_subscripts_fctx
{
    int32       lower;
    int32       upper;
    bool        reverse;
} generate_subscripts_fctx;

入力と出力

  • 本構造体自体はデータ保持用であり、関数呼び出しの引数や戻り値として直接やり取りされるものではありません。
  • 内部の各メンバは、添字の範囲（lower, upper）と走査方向（reverse）を定義します。

前提条件と不変条件

  • lower および upper は配列の次元境界に基づいた整数値である必要があります。
  • palloc を用いてメモリが割り当てられることが想定されています。

アルゴリズムまたは振る舞い

  • generate_subscripts 関数の実行において、現在の走査状態を維持するために使用されます。
  • reverse フラグが true の場合、添字を降順で生成するロジックの制御に利用されます。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/adt/arrayfuncs.c 内の generate_subscripts 関数が、この構造体のインスタンスを palloc で生成し、状態管理に使用しています。
  • **呼び出し先**: 提供されたソース情報からは、本構造体から直接呼び出される関数や、本構造体を操作する他の関数についての情報は得られません。

関連データ構造

  • ArrayType: 配列の内部表現を定義する構造体。本構造体は、この ArrayType から抽出された次元情報に基づいて初期化されます。

失敗モードと境界条件

  • メモリ不足時には palloc がエラーを発生させます。
  • lower が upper を超えるような不正な次元設定が与えられた場合の挙動は、この構造体ではなく、これを利用する generate_subscripts 関数のロジックに依存します。

保守担当者向けメモ

  • 本構造体は generate_subscripts 関数専用のコンテキストです。変更を加える際は、同関数内での添字計算ロジックへの影響を確認してください。
