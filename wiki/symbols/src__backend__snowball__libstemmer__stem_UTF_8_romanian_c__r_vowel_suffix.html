r_vowel_suffix

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_romanian.c:957

目的

単語の末尾にある特定の母音接尾辞を検出し、それが RV 領域内にある場合に削除することで、ルーマニア語の語幹抽出処理を完了させる。

シグネチャまたは形状

static int r_vowel_suffix(struct SN_env * z)

入力と出力

  • **入力**: z (Snowball ステミング環境構造体)
  • **出力**: 成功時に 1、失敗または削除対象外の場合は 0、エラー発生時は負の値を返す。

前提条件と不変条件

  • z->c は現在のカーソル位置を指す。
  • a_6 テーブルに定義された母音接尾辞のいずれかが、単語の末尾（z->ket）に存在する必要がある。
  • 削除対象の接尾辞は、r_RV 関数によって定義される RV 領域内に含まれていなければならない。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置を z->ket に設定する。 2. find_among_b を使用し、末尾から a_6 テーブルの接尾辞を検索する。見つからない場合は 0 を返して終了する。 3. 見つかった接尾辞の開始位置を z->bra に設定する。 4. r_RV を呼び出し、対象範囲が RV 領域内にあるか検証する。検証に失敗した場合はその戻り値を返す。 5. slice_del を呼び出し、z->bra から z->ket までの範囲を削除する。 6. 成功時に 1 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_UTF_8_romanian.c 内のメイン処理フローから呼び出される。また、stem_ISO_8859_1_italian.c および stem_UTF_8_italian.c からの参照が確認されている。
  • **呼び出し先**: find_among_b (Snowball ランタイム), r_RV, slice_del。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_6: 母音接尾辞のマッチングに使用される among 構造体のテーブル。

失敗モードと境界条件

  • find_among_b が一致しない場合、処理は何も行わず 0 を返す。
  • r_RV が 0 または負の値を返した場合、その結果をそのまま伝播させる。
  • slice_del がメモリ不足等の理由で失敗した場合、負の値を返す。

保守担当者向けメモ

  • z->bra および z->ket の操作は Snowball の接尾辞削除の標準的なイディオムである。
