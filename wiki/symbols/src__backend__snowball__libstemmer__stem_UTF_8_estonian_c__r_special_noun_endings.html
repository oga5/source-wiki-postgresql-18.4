r_special_noun_endings

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_estonian.c:987

目的

エストニア語のステミング処理において、特定の特殊な名詞語尾を検出し、対応する置換ルールを適用して語幹を正規化する。

シグネチャまたは形状

static int r_special_noun_endings(struct SN_env * z)

入力と出力

  • **入力**: z (Snowball ランタイム環境構造体)
  • **出力**: 成功時に 1、失敗または条件不一致時に 0、エラー発生時に負の整数を返す。

前提条件と不変条件

  • z->I[0] に格納された境界位置よりも現在のカーソル位置 z->c が後ろにあること。
  • 処理対象の単語末尾が、a_4 配列で定義された特定の接尾辞パターンに一致すること。

アルゴリズムまたは振る舞い

1. **領域制限**: z->I[0] を一時的な境界として設定し、処理範囲を限定する。 2. **パターンマッチング**: find_among_b を使用し、a_4 配列（12個のパターン）に対して後方一致検索を行う。 3. **条件判定**: 検索対象の文字コードおよびビット演算による事前チェックを行い、一致しない場合は即座に終了する。 4. **置換処理**: 一致した among_var の値（1〜3）に応じて、slice_from_s を呼び出し、対応する文字列（s_1, s_2, s_3）で接尾辞を置換する。 5. **後処理**: 変更された環境を維持し、成功を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_UTF_8_estonian.c:1223
  • **呼び出し先**: find_among_b, slice_from_s（いずれも Snowball ランタイム関数）

関連データ構造

  • struct SN_env: ステミング処理の環境状態を保持。
  • a_4: 検索対象となる among 構造体の配列。
  • s_1, s_2, s_3: 置換用の静的文字列。

失敗モードと境界条件

  • カーソル位置が z->I[0] より前にある場合、または find_among_b で一致が見つからない場合は 0 を返す。
  • slice_from_s が内部エラー（メモリ不足や不正なインデックス）を返した場合、その負の値をそのまま伝播させる。
