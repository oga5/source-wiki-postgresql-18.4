目的

ハンガリー語のステミング処理において、特定の接尾辞（a_5 配列に関連するもの）を R1 領域内で検索し、置換または削除を行う。

シグネチャまたは形状

static int r_case_special(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z（ステミング実行環境）
  • 出力: 成功時に 1、失敗または条件不一致時に 0、エラー発生時に負の値を返す。

前提条件と不変条件

  • z->c が単語の末尾を指していること。
  • r_R1(z) が呼び出され、対象の接尾辞が R1 領域内にあることが検証される。

アルゴリズムまたは振る舞い

1. 現在位置 z->c を z->ket に設定する。 2. 単語の末尾2文字が 'n' (110) または 't' (116) でない場合、処理を中断して 0 を返す。 3. find_among_b を用いて a_5 配列から接尾辞を後方検索する。 4. 一致した場合、z->bra を現在位置に設定し、r_R1(z) を呼び出して領域制約を検証する。 5. among_var の値に応じて、slice_from_s を使用して接尾辞を置換する。

  • 呼び出し先: find_among_b, r_R1, slice_from_s。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_5: 接尾辞マッチングに使用される among 構造体の配列。

失敗モードと境界条件

  • 単語長が不足している場合（z->c - 2 <= z->lb）、0 を返す。
  • 接尾辞が一致しない場合、0 を返す。
  • r_R1 の検証に失敗した場合、その戻り値をそのまま返す。
  • slice_from_s がエラー（負の値）を返した場合、その値を返す。
