JsonbToCString

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/jsonb.c:472

目的

JsonbContainer 構造体を JSON 文字列形式に変換し、指定されたバッファへ書き出すためのラッパー関数です。

シグネチャまたは形状

char *JsonbToCString(StringInfo out, JsonbContainer *in, int estimated_len);

入力と出力

  • **入力**:
  • out: 文字列を格納する StringInfo バッファ（NULL 指定可能）。
  • in: 変換対象の JsonbContainer へのポインタ。
  • estimated_len: バッファの初期割り当てサイズを決定するための推定長。
  • **出力**:
  • 変換後の文字列を指す char * を返します。

前提条件と不変条件

  • JsonbToCStringWorker を呼び出し、インデントを行わない（pretty = false）設定で処理を実行します。

アルゴリズムまたは振る舞い

1. JsonbToCStringWorker を呼び出し、in で指定されたバイナリツリーを走査します。 2. 走査結果を out バッファに書き出します。 3. 処理完了後、生成された文字列へのポインタを返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/jsonb.c 内の各所（jsonb_out 等）
  • src/backend/utils/adt/jsonfuncs.c 内の各所（jsonb_to_record 等）
  • **呼び出し先**:
  • JsonbToCStringWorker

関連データ構造

  • StringInfo: 文字列構築用バッファ。
  • JsonbContainer: Jsonb の内部バイナリ表現。

失敗モードと境界条件

  • out が NULL の場合、JsonbToCStringWorker の内部で新しい StringInfo が割り当てられます。
  • estimated_len が不適切な場合、バッファの再割り当てが発生する可能性があります。

保守担当者向けメモ

  • 本関数はインデントなしの標準的な出力用です。インデント付きの出力が必要な場合は JsonbToCStringIndent を使用してください。
  • 内部実装の詳細は JsonbToCStringWorker に依存しています。
