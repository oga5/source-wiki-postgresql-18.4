r_consonant_pair

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_swedish.c:205

目的

単語末尾における特定の子音ペアを検出し、その一方を削除することで語幹の正規化を行う。

シグネチャまたは形状

static int r_consonant_pair(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング実行環境）
  • 出力: 処理が成功した場合は 1、失敗または条件不一致の場合は 0、エラー発生時は負の値を返す。

前提条件と不変条件

  • z->I[1]（領域 R1 の開始位置）が設定されていること。
  • 現在のカーソル位置 z->c が z->I[1] 以降にあること。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置が z->I[1] 未満であれば終了する。 2. 検索範囲の境界（lb）を z->I[1] に一時的に設定する。 3. 直前の文字が特定の子音グループ（ビットマスク判定）に含まれるか確認する。 4. a_1 配列を用いて、末尾から前方へ向かって find_among_b によるマッチングを行う。 5. マッチした場合、skip_b_utf8 で文字をスキップし、slice_del を呼び出して該当部分を削除する。 6. 終了時に lb を元の値に復元する。

呼び出し元と呼び出し先

  • 呼び出し元: swedish_UTF_8_stem（スウェーデン語ステミングのメイン処理）から呼び出される。また、デンマーク語およびノルウェー語のステミング実装（stem_UTF_8_danish.c, stem_UTF_8_norwegian.c 等）からも参照されている。
  • 呼び出し先: find_among_b（接尾辞検索）、skip_b_utf8（UTF-8文字スキップ）、slice_del（文字列削除）。

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_1: 子音ペアのマッチングに使用される among 構造体の配列。

失敗モードと境界条件

  • z->c < z->I[1] の場合、処理を行わず 0 を返す。
  • find_among_b がマッチしない場合、0 を返す。
  • skip_b_utf8 または slice_del がエラー（負の値）を返した場合、その値をそのまま伝播させる。
