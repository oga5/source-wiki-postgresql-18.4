pg_popcount_sve

定義

  • 種別: function
  • 定義箇所: src/port/pg_popcount_aarch64.c:108

目的

AArch64 SVE（Scalable Vector Extension）命令セットを利用し、メモリバッファ内のビットがセットされている数（popcount）を高速に計算します。

シグネチャまたは形状

uint64 pg_popcount_sve(const char *buf, int bytes)

入力と出力

  • **入力**:
  • buf: 処理対象のメモリ領域へのポインタ。
  • bytes: 処理するバイト数。
  • **出力**:
  • uint64: バッファ内の全ビットの合計数。

前提条件と不変条件

  • 実行環境がAArch64 SVE命令セットをサポートしている必要があります。
  • buf は有効なメモリ領域を指している必要があります。

アルゴリズムまたは振る舞い

1. **初期化**: SVEレジスタ幅（svcntb()）に基づき、4つのアキュムレータを初期化します。 2. **メインループ**: 4つのSVEレジスタ幅分を一度に処理するループを展開し、命令レベルの並列性を高めてビット数を加算します。 3. **中間ループ**: 残りのデータが2レジスタ幅以上ある場合、追加の反復処理を行います。 4. **集約**: 4つのアキュムレータの値を合計します。 5. **端数処理**: 残りのデータに対して、svwhilelt_b8_s32 を用いてマスク付きロードを行い、残りのビット数を加算します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/port/pg_popcount_aarch64.c 内の関数ポインタ pg_popcount_optimized に代入されます。
  • **呼び出し先**: arm_sve.h で定義されたSVE組み込み関数（svptrue_b64, svdup_u64, svcntb, svld1_u64, svadd_u64_x, svcnt_u64_x, svaddv_u64, svwhilelt_b8_s32, svld1_u8, svaddv_u8）。

関連データ構造

  • svbool_t: SVE述語レジスタ。
  • svuint64_t, svuint8_t: SVEベクトルレジスタ。

失敗モードと境界条件

  • bytes が 0 の場合、ループは実行されず 0 を返します。
  • bytes が SVEレジスタ幅の倍数でない場合、最後のループでマスク処理により適切に端数が計算されます。

保守担当者向けメモ

  • 本関数は POPCNT_AARCH64 が定義されている環境でのみ使用可能です。
  • パイプライン効率を最大化するため、4つのアキュムレータを使用する設計となっています。変更時は依存関係の解消に注意してください。
