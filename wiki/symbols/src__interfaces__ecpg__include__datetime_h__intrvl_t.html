intrvl_t

定義

  • typedef interval intrvl_t;
  • 定義箇所: src/interfaces/ecpg/include/datetime.h:11

目的

ECPG（Embedded SQL for C）において、期間（interval）データを保持するための Informix 互換の型定義を提供します。

シグネチャまたは形状

interval 型の別名です。

入力と出力

本シンボルは型定義であり、関数ではないため、直接的な入力や出力はありません。ECPG プリプロセッサが生成する C コード内で、期間データを格納する変数として使用されます。

前提条件と不変条件

  • _ECPGLIB_H が定義されている場合、再定義を防ぐために本定義はスキップされます。
  • interval 型の定義が事前に読み込まれている必要があります。

アルゴリズムまたは振る舞い

本シンボル自体はデータ構造の宣言であり、実行時のアルゴリズムは持ちません。操作については src/interfaces/ecpg/pgtypeslib/datetime.c に実装されている pgtypeslib の関数群が担います。

呼び出し元と呼び出し先

  • 呼び出し元: ECPG プリプロセッサが生成する C コード。
  • 呼び出し先: 情報なし。

関連データ構造

  • interval: 本型の基盤となる型。
  • src/interfaces/ecpg/include/pgtypes_interval.h: interval 型の定義元。

失敗モードと境界条件

  • 型定義のため、実行時の失敗モードはありません。
  • 境界条件として、ecpglib との重複定義が発生した場合、コンパイルエラーを避けるためのガードマクロ（_ECPGLIB_H）による制御が行われます。

保守担当者向けメモ

  • 本型は Informix 互換モードでの利用を想定しています。
  • 型の操作や詳細な構造については pgtypeslib の関連関数を参照してください。
