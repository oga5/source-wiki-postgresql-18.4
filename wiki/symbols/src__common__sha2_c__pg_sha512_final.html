pg_sha512_final

定義

  • 種別: function
  • 定義箇所: src/common/sha2.c:904

目的

SHA-512 ハッシュ計算の最終処理を行い、ダイジェストを生成してコンテキストを破棄します。

シグネチャまたは形状

void pg_sha512_final(pg_sha512_ctx *context, uint8 *digest)

入力と出力

  • context: SHA-512 計算状態を保持する pg_sha512_ctx 構造体へのポインタ。
  • digest: 計算結果を格納するバッファへのポインタ（NULL の場合はダイジェスト生成をスキップ）。

前提条件と不変条件

  • context は初期化および更新済みの有効な状態であること。
  • digest が NULL でない場合、PG_SHA512_DIGEST_LENGTH 分の書き込み領域が確保されていること。

アルゴリズムまたは振る舞い

1. digest が NULL でない場合:

  • SHA512_Last(context) を呼び出し、パディングとビット長情報の埋め込みを含む最終変換を実行する。
  • 非ビッグエンディアン環境の場合、REVERSE64 マクロを用いて context->state の各要素をホストバイトオーダーに変換する。
  • memcpy を使用して context->state から digest へ結果をコピーする。

    • 2. 最後に memset を呼び出し、context 全体をゼロクリアして機密情報を破棄する。

呼び出し元と呼び出し先

  • 呼び出し元: src/common/cryptohash.c:225
  • 呼び出し先: SHA512_Last（内部関数）

関連データ構造

  • pg_sha512_ctx: SHA-512 の計算状態を保持する構造体。

失敗モードと境界条件

  • digest が NULL の場合、ハッシュ値の出力は行われず、コンテキストのゼロクリアのみが実行される。

保守担当者向けメモ
