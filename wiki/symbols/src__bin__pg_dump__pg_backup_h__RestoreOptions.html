RestoreOptions

定義

  • 種別: struct
  • 定義箇所: src/bin/pg_dump/pg_backup.h:96

目的

pg_restore 実行時におけるリストア処理の詳細な動作オプションを保持・管理するための構造体です。

シグネチャまたは形状

typedef struct _restoreOptions {
    /* 実行フラグおよび設定値 */
    int createDB, noOwner, noTableAm, noTablespace, disable_triggers, use_setsessauth;
    char *superuser, *use_role;
    int dropSchema, disable_dollar_quoting, dump_inserts, column_inserts, if_exists;
    int no_comments, no_policies, no_publications, no_security_labels, no_subscriptions, strict_names;
    const char *filename;
    int dumpSections, verbose, aclsSkip;
    const char *lockWaitTimeout;
    int include_everything;
    int tocSummary;
    char *tocFile;
    int format;
    char *formatName;
    /* 選択フィルタ */
    int selTypes, selIndex, selFunction, selTrigger, selTable;
    SimpleStringList indexNames, functionNames, schemaNames, schemaExcludeNames, triggerNames, tableNames;
    /* 接続およびトランザクション設定 */
    int useDB;
    ConnParams cparams;
    int noDataForFailedTables, exit_on_error;
    pg_compress_specification compression_spec;
    int suppressDumpWarnings;
    bool single_txn;
    int txn_size;
    bool *idWanted;
    int enable_row_security, sequence_data, binary_upgrade;
    /* 派生フラグ */
    bool dumpSchema, dumpData, dumpStatistics;
    char *restrict_key;
} RestoreOptions;

入力と出力

  • **入力**: pg_restore のコマンドライン引数や設定ファイルから読み込まれた値が格納されます。
  • **出力**: Archive 構造体を通じて、各バックアップ形式（カスタム、tar等）のリストアロジックへ設定値を提供します。

前提条件と不変条件

  • useDB が真の場合、cparams（ConnParams 型）に有効な接続パラメータが設定されている必要があります。
  • SimpleStringList 型の各メンバは、リストア対象または除外対象のオブジェクト名を保持します。

アルゴリズムまたは振る舞い

  • リストア処理の開始前に NewRestoreOptions() で初期化され、SetArchiveOptions() を通じて Archive ハンドルに適用されます。
  • idWanted 配列は、ダンプされた各オブジェクト（DumpId）をリストアするか否かの判定に使用されます。
  • dumpSchema, dumpData, dumpStatistics は、ユーザー指定のフラグから派生して計算される制御用フラグです。

呼び出し元と呼び出し先

  • **呼び出し元**: pg_restore のメイン処理や pg_backup_archiver.c 内の初期化関数。
  • **呼び出し先**: SetArchiveOptions（設定の適用）、dumpOptionsFromRestoreOptions（DumpOptions への変換）、StrictNamesCheck（名前の検証）。

関連データ構造

  • Archive: リストア処理のメインコンテキスト。
  • ConnParams: データベース接続設定。
  • SimpleStringList: オブジェクト名フィルタリング用のリスト構造。
  • pg_compress_specification: 圧縮アルゴリズムおよびレベルの定義。

失敗モードと境界条件

  • txn_size が 0 以下の場合、トランザクション分割の挙動が変化します。
  • idWanted が適切に割り当てられていない場合、リストア対象の判定が正しく行われません。

保守担当者向けメモ

  • 本構造体は pg_dump と pg_restore の両方に関連する設定を保持する可能性があるため、フィールド追加時は DumpOptions との整合性に注意してください。
  • SimpleStringList のメモリ管理は fe_utils/simple_list.h の仕様に従います。
