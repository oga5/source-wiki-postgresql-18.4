pow5Factor

定義

  • 種別: function
  • 定義箇所: src/common/d2s.c:73

目的

与えられた整数が 5 で何回割り切れるか（5 の冪乗の指数）を計算する。

シグネチャまたは形状

static inline uint32 pow5Factor(uint64 value)

入力と出力

  • **入力**: uint64 value（0 以外の正の整数）
  • **出力**: uint32 型のカウント値。value を 5 で繰り返し割った際の回数を返す。

前提条件と不変条件

  • 入力 value は 0 であってはならない（Assert(value != 0) により検証される）。

アルゴリズムまたは振る舞い

1. カウンタ count を 0 に初期化する。 2. 無限ループ内で以下を繰り返す：

  • div5(value) を用いて value を 5 で除算し、商 q を得る。
  • value から 5 * q を引くことで剰余 r を算出する。
  • r が 0 でない場合（5 で割り切れない場合）、ループを終了する。
  • r が 0 の場合、value を q で更新し、count をインクリメントする。

    • 3. 最終的な count を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/common/d2s.c: 浮動小数点変換の精度判定ロジック内で複数回参照されている。
  • src/common/f2s.c: 単精度浮動小数点変換のロジック内で参照されている。
  • **呼び出し先**:
  • div5: 内部で 5 による除算を行うために呼び出される（定義元は ryu_common.h 等の依存先と推測される）。

関連データ構造

  • 特になし。プリミティブな整数型のみを使用する。

失敗モードと境界条件

  • value が 0 の場合、Assert によりプログラムが停止する可能性がある。
  • 入力が 5 の倍数でない場合、戻り値は 0 となる。

保守担当者向けメモ

  • 本関数は Ryu アルゴリズムの移植版の一部であり、浮動小数点数の最短 10 進数表現を決定するための重要な判定ロジックである。
  • div5 の実装が 128 ビット演算に依存している場合があるため、環境によるパフォーマンス特性に注意すること。
