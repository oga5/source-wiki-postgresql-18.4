r_fix_gd

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_lithuanian.c:741

目的

リトアニア語のステミング処理において、語幹抽出後の特定の綴り（gd）を正規化する。

シグネチャまたは形状

static int r_fix_gd(struct SN_env * z)

入力と出力

  • 入力: SN_env * z （ステミング環境の状態を保持する構造体）
  • 出力: 成功時に 1、条件不一致または失敗時に 0 または負の整数（slice_from_s の戻り値）を返す。

前提条件と不変条件

  • z->c（現在のカーソル位置）が有効な範囲内であること。
  • z->p（文字列バッファ）の z->c - 1 位置の文字が 100（ASCIIの 'd'）であること。

アルゴリズムまたは振る舞い

1. z->ket を現在のカーソル位置 z->c に設定する。 2. カーソルの直前が 'd' でない場合、または a_4 テーブルに定義されたパターンにマッチしない場合は終了する。 3. マッチした場合、z->bra を現在のカーソル位置に設定する。 4. slice_from_s を呼び出し、マッチした部分を s_10 で定義された文字列に置換する。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_lithuanian.c:818
  • 呼び出し先: find_among_b（パターンマッチング用）、slice_from_s（文字列置換用）

関連データ構造

  • SN_env: ステミング環境を管理する構造体。
  • a_4: find_among_b で使用される接尾辞/パターンのテーブル。

失敗モードと境界条件

  • カーソル位置が文字列の先頭に近い場合（z->c - 1 <= z->lb）、処理を中断する。
  • find_among_b がマッチしない場合、処理を中断する。
  • slice_from_s がメモリ不足等の理由で負の値を返した場合、その値をそのまま返す。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードであるため、直接編集せず、元の .sbl スクリプトを修正すること。
  • z->p へのポインタ演算を行っているため、デバッグ時は SN_env のカーソル位置と境界条件に注意すること。
