tar_get_file_size

定義

  • 種別: function
  • 定義箇所: src/bin/pg_basebackup/walmethods.c:1006

目的

tar 形式の WAL 書き込みメソッドにおいて、指定されたパスのファイルサイズを取得します。

シグネチャまたは形状

ssize_t tar_get_file_size(WalWriteMethod *wwmethod, const char *pathname)

入力と出力

  • **入力**:
  • wwmethod: WalWriteMethod 構造体へのポインタ。
  • pathname: サイズを確認する対象のファイルパス（文字列）。
  • **出力**:
  • 常に -1 を返します。

前提条件と不変条件

  • 呼び出し時に clear_error(wwmethod) が実行され、既存のエラー状態がリセットされます。

アルゴリズムまたは振る舞い

1. clear_error を呼び出し、wwmethod のエラー状態をクリアします。 3. -1 を返して終了します。現在、この機能はサポートされていません。

呼び出し元と呼び出し先

  • **呼び出し元**: src/bin/pg_basebackup/walmethods.c 内の WalWriteMethodOps 構造体（get_file_size フィールド）で参照されています。
  • **呼び出し先**: clear_error を呼び出します。

関連データ構造

  • WalWriteMethod: WAL 書き込みの抽象インターフェースを保持する構造体。

失敗モードと境界条件

  • 本関数は常に失敗（-1 を返却）し、ENOSYS エラーを発生させます。

保守担当者向けメモ

  • 本関数は現在使用されておらず、サポートもされていません。将来的に tar メソッドでファイルサイズ取得が必要になった場合に実装を検討してください。
