r_factive

定義

  • **種別**: function
  • **定義箇所**: src/backend/snowball/libstemmer/stem_ISO_8859_2_hungarian.c:654

目的

ハンガリー語のステミング処理において、特定の接尾辞（factive）を単語末尾から削除し、必要に応じて単語の正規化を行う。

シグネチャまたは形状

static int r_factive(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z（ステミング環境の状態を保持する構造体）
  • **出力**: 処理が成功した場合は 1、失敗または条件不一致の場合は 0、エラー発生時は負の値を返す。

前提条件と不変条件

  • z->c（現在のカーソル位置）が有効範囲内であること。
  • 単語末尾の文字が 225 または 233（ISO-8859-2における特定の文字）であること。
  • a_7 で定義された接尾辞パターンが単語末尾にマッチすること。
  • r_R1 による領域判定が成功すること。

アルゴリズムまたは振る舞い

1. z->ket を現在のカーソル位置に設定する。 2. 文字コードのチェックを行い、条件を満たさない場合は終了する。 3. find_among_b を使用し、a_7 配列に基づいて末尾から接尾辞を検索する。 4. z->bra をマッチした接尾辞の開始位置に設定する。 5. r_R1 を呼び出し、対象領域が R1 内にあるか確認する。 6. r_double を呼び出し、重複文字の処理を行う。 7. slice_del を呼び出し、マッチした接尾辞を削除する。 8. r_undouble を呼び出し、削除後の単語を正規化する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_ISO_8859_2_hungarian.c:827 および src/backend/snowball/libstemmer/stem_UTF_8_hungarian.c:833 からの呼び出しが確認されている。
  • **呼び出し先**: find_among_b, r_R1, r_double, slice_del, r_undouble を呼び出す。

関連データ構造

  • struct SN_env: ステミング環境を管理する構造体。
  • a_7: among 構造体の配列（接尾辞パターン定義）。

失敗モードと境界条件

  • z->c <= z->lb: カーソルが境界を超えている場合は 0 を返す。
  • find_among_b がマッチしない場合、0 を返す。
  • r_R1, r_double, slice_del, r_undouble の各関数が負の値を返した場合、その値をそのまま伝播させて終了する。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードである。ロジックの変更が必要な場合は、直接ソースを編集せず、元の Snowball スクリプト（.sbl）を修正すること。
  • a_7 配列の定義や r_R1 の判定ロジックに変更がある場合、本関数の挙動に直接影響する。
