unicode_linestyle

定義

  • 種別: enum
  • 定義箇所: src/include/fe_utils/print.h:99

目的

Unicode罫線を用いたテーブル出力において、罫線のスタイル（単線または二重線）を指定するために使用されます。

シグネチャまたは形状

typedef enum unicode_linestyle
{
	UNICODE_LINESTYLE_SINGLE = 0,
	UNICODE_LINESTYLE_DOUBLE,
} unicode_linestyle;

入力と出力

  • 入力: 列挙型定数 UNICODE_LINESTYLE_SINGLE または UNICODE_LINESTYLE_DOUBLE。
  • 出力: なし（設定値として保持される）。

前提条件と不変条件

  • UNICODE_LINESTYLE_SINGLE は 0、UNICODE_LINESTYLE_DOUBLE は 1 として扱われます。

アルゴリズムまたは振る舞い

テーブルの境界線、カラム区切り、ヘッダー区切りの描画スタイルを決定する設定値として機能します。

呼び出し元と呼び出し先

  • 呼び出し元: src/bin/psql/command.c:5027 にて参照されています。
  • 呼び出し先: 情報なし。
  • 関連する構造体メンバ: printTableOpt 構造体内の unicode_border_linestyle、unicode_column_linestyle、unicode_header_linestyle で使用されています。

関連データ構造

  • printTableOpt: テーブル出力設定を保持する構造体。

失敗モードと境界条件

  • 定義された値以外の整数値を代入した場合の挙動は未定義です。

保守担当者向けメモ

  • 本列挙型は printTableOpt 構造体を通じて、出力フォーマットのカスタマイズに使用されます。新しい罫線スタイルを追加する場合は、この列挙型を拡張し、対応する描画ロジックを src/fe_utils/print.c に実装する必要があります。
