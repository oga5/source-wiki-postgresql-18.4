pgstat_fetch_stat_dbentry

定義

  • 種別: function
  • 定義箇所: src/backend/utils/activity/pgstat_database.c:277

目的

指定されたデータベースOIDに対応する統計情報エントリ（PgStat_StatDBEntry）を取得します。

シグネチャまたは形状

PgStat_StatDBEntry *pgstat_fetch_stat_dbentry(Oid dboid)

入力と出力

  • **入力**: dboid (Oid) - 取得対象のデータベース識別子。
  • **出力**: PgStat_StatDBEntry* - 該当するデータベース統計構造体へのポインタ。見つからない場合は NULL を返します。

前提条件と不変条件

  • 統計サブシステムが初期化されている必要があります。
  • 戻り値の有効性は、呼び出し元のコンテキストにおける統計スナップショットの生存期間に依存します。

アルゴリズムまたは振る舞い

内部的に pgstat_fetch_entry を呼び出し、PGSTAT_KIND_DATABASE 型の統計エントリを検索して返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/postmaster/autovacuum.c (複数箇所)
  • src/backend/utils/adt/pgstatfuncs.c (統計情報表示関数群)
  • **呼び出し先**:
  • pgstat_fetch_entry (統計サブシステム内部関数)

関連データ構造

  • PgStat_StatDBEntry: データベース単位の統計情報を保持する構造体。
  • Oid: PostgreSQLのオブジェクト識別子。

失敗モードと境界条件

  • 指定した dboid に対応する統計エントリが存在しない場合、NULL を返します。

保守担当者向けメモ

  • 本関数は pgstat_fetch_entry のラッパーです。統計情報の取得ロジックに変更がある場合は、pgstat_internal.h で定義される基盤側の実装を確認してください。
