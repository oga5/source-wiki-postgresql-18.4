ISO8601IntegerWidth

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/pgtypeslib/interval.c:80

目的

ISO 8601 形式の文字列において、符号を除いた数値部分の連続する桁数を算出する。

シグネチャまたは形状

static int ISO8601IntegerWidth(const char *fieldstart)

入力と出力

  • **入力**: fieldstart (数値文字列の開始位置を示すポインタ)
  • **出力**: 数値部分の桁数（int 型）

前提条件と不変条件

  • fieldstart は有効な文字列を指している必要がある。
  • 先頭に負符号（-）が存在する場合、それをスキップして桁数を計算する。

アルゴリズムまたは振る舞い

1. fieldstart の先頭文字が '-' であるかを確認し、該当する場合はポインタを1つ進める。 2. strspn を使用し、ポインタ以降の文字列において '0' から '9' までの文字が連続する長さを計算して返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/interfaces/ecpg/pgtypeslib/interval.c:174
  • src/interfaces/ecpg/pgtypeslib/interval.c:257
  • **呼び出し先**:
  • 標準ライブラリ関数 strspn

*注: ツールにより src/backend/utils/adt/datetime.c 内での呼び出しも示唆されているが、本ファイル（interval.c）の文脈外であるため、直接の依存関係としては扱わない。*

関連データ構造

なし。

失敗モードと境界条件

  • 文字列が数値で始まらない場合、strspn の仕様に従い 0 を返す。
  • fieldstart が空文字列または '-' のみの場合は 0 を返す。

保守担当者向けメモ

  • 本関数は PostgreSQL バックエンドのロジックを移植したものであり、src/backend/utils/adt/datetime.c に同名の関数が存在する。
  • strspn の戻り値は size_t であるが、本関数は int を返すため、極端に長い文字列を扱う場合は注意が必要である。
