ClearBufferTag

定義

  • 種別: function
  • 定義箇所: src/include/storage/buf_internals.h:147

目的

BufferTag 構造体の各フィールドを無効値（InvalidOid, InvalidBlockNumber 等）で初期化し、バッファタグをリセットする。

シグネチャまたは形状

static inline void ClearBufferTag(BufferTag *tag)

入力と出力

  • **入力**: BufferTag *tag （初期化対象のバッファタグへのポインタ）
  • **出力**: なし（引数で渡された tag の内容を直接書き換える）

前提条件と不変条件

  • 引数 tag は有効な BufferTag 構造体を指している必要がある。

アルゴリズムまたは振る舞い

以下の順序でフィールドを無効値に設定する： 1. tag->spcOid を InvalidOid に設定。 2. tag->dbOid を InvalidOid に設定。 3. BufTagSetRelForkDetails を呼び出し、リレーション番号とフォーク番号をそれぞれ InvalidRelFileNumber および InvalidForkNumber に設定。 4. tag->blockNum を InvalidBlockNumber に設定。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/storage/buffer/buf_init.c
  • src/backend/storage/buffer/bufmgr.c
  • src/backend/storage/buffer/localbuf.c
  • **呼び出し先**:
  • BufTagSetRelForkDetails

関連データ構造

  • BufferTag: バッファが保持するページを特定するための識別子。

失敗モードと境界条件

  • 引数 tag が NULL の場合、未定義の動作を引き起こす可能性がある。

保守担当者向けメモ

  • 本関数は static inline で定義されており、ヘッダーファイル内で完結している。
  • BufferTag 構造体の定義に変更があった場合、本関数による初期化漏れがないか確認すること。
