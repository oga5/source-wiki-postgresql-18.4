maxcolor

定義

  • 種別: function
  • 定義箇所: src/backend/regex/regc_color.c:171

目的

現在使用されている最大の色番号（color number）を取得します。

シグネチャまたは形状

static color maxcolor(struct colormap *cm)

入力と出力

  • **入力**: cm (カラーマップ構造体へのポインタ)
  • **出力**: 現在の最大色番号を color 型で返します。エラー状態の場合は COLORLESS を返します。

前提条件と不変条件

  • CISERR() マクロにより、正規表現コンパイルのエラー状態がチェックされます。
  • cm は有効な colormap 構造体を指している必要があります。

アルゴリズムまたは振る舞い

1. CISERR() を呼び出し、エラーが発生しているか確認します。エラーがある場合は直ちに COLORLESS を返します。 2. エラーがない場合、cm->max に格納されている値を color 型にキャストして返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/regex/regc_nfa.c:3559: cnfa->ncolors の算出に使用。
  • **呼び出し先**:
  • CISERR() (マクロ)

関連データ構造

  • colormap: カラーマップの管理構造体。max フィールドに現在の最大色番号を保持します。
  • color: 色番号を表す型。

失敗モードと境界条件

  • 正規表現コンパイル中にエラーが発生している場合、COLORLESS が返されます。

保守担当者向けメモ

  • 本関数は regcomp.c からも参照されるため、変更時は影響範囲に注意してください。
  • cm->max はカラーマップの動的な更新に伴い変化するため、呼び出しタイミングによって値が異なる可能性があります。
