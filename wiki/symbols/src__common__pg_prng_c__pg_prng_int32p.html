pg_prng_int32p

定義

  • 種別: function
  • 定義箇所: src/common/pg_prng.c:253

目的

擬似乱数生成器の状態から、32ビット符号付き整数を生成して返します。

シグネチャまたは形状

int32 pg_prng_int32p(pg_prng_state *state)

入力と出力

  • **入力**: pg_prng_state *state（乱数生成器の状態を保持する構造体へのポインタ）
  • **出力**: int32（生成された32ビット符号付き乱数値）

前提条件と不変条件

  • state は初期化済みである必要があります。
  • 本関数は暗号学的に安全な乱数生成を目的としていません。

アルゴリズムまたは振る舞い

1. 内部関数 xoroshiro128ss(state) を呼び出し、64ビットの乱数値を生成します。 2. 生成された64ビット値 v を右に33ビットシフトします。 3. シフト後の値を int32 型にキャストして返します。

呼び出し元と呼び出し先

  • **呼び出し先**: xoroshiro128ss
  • **呼び出し元**: src/test/modules/test_bloomfilter/test_bloomfilter.c:88 にて、グローバルな乱数生成器の状態を参照して呼び出されています。

関連データ構造

  • pg_prng_state: 乱数生成器の状態を保持する構造体。

失敗モードと境界条件

  • state がすべて0のビットで初期化されている場合、生成器は0を返し続けます。

保守担当者向けメモ

  • 本関数は暗号学的な強度が不要な用途（シミュレーションやテストデータ生成など）専用です。パスワードや暗号鍵の生成には使用しないでください。
