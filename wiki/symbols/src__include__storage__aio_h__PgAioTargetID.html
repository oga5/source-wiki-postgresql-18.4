PgAioTargetID

定義

  • 種別: enum
  • 定義箇所: src/include/storage/aio.h:116

目的

非同期 I/O (AIO) 操作の対象となるサブシステムやコンポーネントを識別するために使用されます。

シグネチャまたは形状

typedef enum PgAioTargetID
{
    PGAIO_TID_INVALID = 0,
    PGAIO_TID_SMGR,
} PgAioTargetID;

入力と出力

  • 入力: pgaio_io_set_target 関数の引数として使用されます。
  • 出力: 該当なし（列挙型定数）。

前提条件と不変条件

  • PGAIO_TID_INVALID は 0 に設定されており、メモリがゼロ初期化された状態を検知するために利用されます。

アルゴリズムまたは振る舞い

  • AIO ハンドルに対して、どのターゲット（例: SMGR）に対する I/O 操作であるかを指定します。
  • ターゲット固有の振る舞いは、src/backend/storage/aio/aio_target.c に実装されることが想定されています。

呼び出し元と呼び出し先

  • 呼び出し元: pgaio_io_set_target(PgAioHandle *ioh, PgAioTargetID targetid) がこの型を引数として受け取ります。
  • 呼び出し先: 該当なし。

関連データ構造

  • PgAioHandle: AIO 操作のコンテキストを保持する構造体。本列挙型は PgAioHandle のターゲット設定に使用されます。

失敗モードと境界条件

  • PGAIO_TID_INVALID が指定された場合、または未設定のまま操作が開始された場合の挙動については、ソースコードの断片からは明示されていません。

保守担当者向けメモ

  • 新しい AIO ターゲットを追加する場合は、この列挙型に定数を追加し、対応するロジックを src/backend/storage/aio/aio_target.c に実装してください。
