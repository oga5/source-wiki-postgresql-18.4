n_choose_k

定義

  • 種別: function
  • 定義箇所: src/backend/statistics/mvdistinct.c:549

目的

二項係数 $nCk$（$n$ 個から $k$ 個を選ぶ組み合わせの数）を計算する。

シグネチャまたは形状

static int n_choose_k(int n, int k)

入力と出力

  • **入力**:
  • n: 全要素数（整数）
  • k: 選択する要素数（整数）
  • **出力**:
  • 組み合わせの総数（整数）

前提条件と不変条件

  • k > 0 かつ n >= k であること（Assert により検証）。

アルゴリズムまたは振る舞い

1. 二項係数の対称性（$nCk = nC(n-k)$）を利用し、計算量を減らすために k を Min(k, n - k) に更新する。 2. $r = 1$ から開始し、d を 1 から k までインクリメントしながら、以下の計算を繰り返す。

  • $r = r \times n$
  • $r = r / d$
  • $n$ をデクリメントする。

    • 3. 最終的な r を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/statistics/mvdistinct.c 内の statext_ndistinct_build（state->ncombinations = n_choose_k(n, k);）から呼び出される。
  • **呼び出し先**: Min マクロを使用。

関連データ構造

  • 特になし。

失敗モードと境界条件

  • n や k が非常に大きい場合、戻り値の int 型でオーバーフローが発生する可能性がある。
  • 前提条件を満たさない場合、Assert によりアボートする。

保守担当者向けメモ

  • 本関数は mvdistinct.c 内の統計オブジェクト構築プロセスにおいて、必要な組み合わせ数を算出するために使用される。
  • 計算ロジックは整数除算を利用しているため、ループ内での順序（乗算してから除算）が正確な結果を得るために重要である。
