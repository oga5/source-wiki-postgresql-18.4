JsonbIteratorInit

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/jsonb_util.c:820

目的

JsonbContainer を走査するための JsonbIterator を初期化します。

シグネチャまたは形状

JsonbIterator *JsonbIteratorInit(JsonbContainer *container);

入力と出力

  • **入力**: 走査対象となる JsonbContainer *container。
  • **出力**: 初期化された JsonbIterator *。

前提条件と不変条件

  • 内部で iteratorFromContainer を呼び出し、親イテレータを NULL として初期化します。
  • 走査中にネストされたコンテナに遭遇した場合、内部で新しいイテレータが palloc されます。

アルゴリズムまたは振る舞い

1. 引数として受け取った container を iteratorFromContainer 関数に渡します。 2. 親イテレータが存在しない（ルートレベルの）初期状態としてイテレータを生成し、返却します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/jsonb_plperl/jsonb_plperl.c
  • contrib/jsonb_plpython/jsonb_plpython.c
  • src/backend/utils/adt/jsonb.c
  • src/backend/utils/adt/jsonb_gin.c
  • src/backend/utils/adt/jsonb_op.c
  • **呼び出し先**:
  • iteratorFromContainer (内部関数)

関連データ構造

  • JsonbContainer: jsonb のバイナリ表現を保持する構造体。
  • JsonbIterator: 走査状態を保持する構造体。

失敗モードと境界条件

  • メモリ不足時に palloc が失敗する可能性があります。
  • 呼び出し元は、ネストされたコンテナを処理する際に生成されるイテレータの親子関係を適切に管理し、メモリリークを防ぐ必要があります。

保守担当者向けメモ

  • 本関数は JsonbIteratorNext と組み合わせて使用されます。
  • 複雑な JSON 構造を扱う際は、再帰的な処理によるスタックオーバーフローに注意してください。
