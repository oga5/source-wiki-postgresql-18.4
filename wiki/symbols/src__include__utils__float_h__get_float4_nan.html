get_float4_nan

定義

  • 種別: function
  • 定義箇所: src/include/utils/float.h:110

目的

プラットフォームに依存せず、一貫した方法で float4 型の NaN（Not-a-Number）値を取得する。

シグネチャまたは形状

static inline float4 get_float4_nan(void)

入力と出力

  • 入力: なし
  • 出力: float4 型の NaN 値

前提条件と不変条件

  • C99 標準が利用可能な環境では NAN マクロを使用する。
  • NAN マクロが定義されていない環境では、0.0 / 0.0 の演算結果として NaN を生成する。

アルゴリズムまたは振る舞い

1. コンパイル環境で NAN マクロが定義されているかを確認する。 2. 定義されている場合、NAN を float4 にキャストして返す。 3. 定義されていない場合、0.0 / 0.0 を実行し、その結果（NaN）を float4 にキャストして返す。

呼び出し元と呼び出し先

  • 呼び出し元（直接的な証拠）:
  • src/backend/utils/adt/float.c:230
  • src/backend/utils/numeric.c:4854
  • 呼び出し先:
  • 外部関数への依存なし（標準的な浮動小数点演算のみを使用）。

関連データ構造

  • float4: PostgreSQL で定義された単精度浮動小数点型。

失敗モードと境界条件

  • 本関数自体は失敗しない。
  • NaN の生成および返却は、IEEE 754 準拠の浮動小数点演算を前提としている。

保守担当者向けメモ

  • 本関数はインライン関数として定義されており、パフォーマンスを重視している。
  • PostgreSQL において NaN は「すべての非 NaN 値より大きく、かつ NaN 同士は等しい」と定義されているため、比較演算を行う際は本関数で取得した値と一貫性のあるロジックを用いること。
