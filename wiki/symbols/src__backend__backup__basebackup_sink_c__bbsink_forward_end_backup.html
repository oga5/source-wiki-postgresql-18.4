bbsink_forward_end_backup

定義

  • src/backend/backup/basebackup_sink.c:110

目的

バックアップ終了処理（end_backup）の呼び出しを、bbsink チェーン内の次のシンクへ転送する。

シグネチャまたは形状

void bbsink_forward_end_backup(bbsink *sink, XLogRecPtr endptr, TimeLineID endtli)

入力と出力

  • sink: 現在の bbsink オブジェクト。
  • endptr: バックアップ終了時の WAL 位置（XLogRecPtr）。
  • endtli: バックアップ終了時のタイムライン ID（TimeLineID）。
  • 戻り値: なし。

前提条件と不変条件

  • sink->bbs_next が NULL であってはならない（後続のシンクが存在すること）。

アルゴリズムまたは振る舞い

1. sink->bbs_next が存在することをアサートする。 2. bbsink_end_backup を呼び出し、後続のシンクに対して endptr および endtli を渡して処理を委譲する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/backup/basebackup_zstd.c
  • 各種 bbsink 実装（basebackup_to_shell.c, basebackup_gzip.c, basebackup_lz4.c, basebackup_progress.c, basebackup_server.c, basebackup_throttle.c）において、bbsink_ops 構造体の .end_backup フィールドに割り当てられている。
  • **呼び出し先**:
  • bbsink_end_backup（sink->bbs_next が指すシンクのメソッド）。

関連データ構造

  • bbsink: バックアップデータの転送先を抽象化した構造体。

失敗モードと境界条件

  • sink->bbs_next が NULL の場合、アサーション失敗により異常終了する。

保守担当者向けメモ

  • 本関数は、独自の bbsink 実装において、終了処理を単に後続へ流すだけでよい場合に利用するヘルパー関数である。
