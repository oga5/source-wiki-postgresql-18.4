r_remove_possessive_pronoun

定義

  • **種別**: function
  • **定義箇所**: src/backend/snowball/libstemmer/stem_ISO_8859_1_indonesian.c:135

目的

インドネシア語のステミング処理において、単語末尾から所有格代名詞（nya, ku, mu 等）を除去します。

シグネチャまたは形状

static int r_remove_possessive_pronoun(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z （Snowballステミング環境構造体）
  • **出力**: int （成功時に 1、失敗または条件不一致時に 0、内部エラー発生時に負の値を返す）

前提条件と不変条件

  • z->c は現在のカーソル位置を指していること。
  • z->p は処理対象の文字列バッファであること。
  • 処理対象の単語末尾が所有格代名詞の候補（a または u で終わる）である必要がある。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を z->ket に保存する。 2. 単語の末尾文字が a (ASCII 97) または u (ASCII 117) であるかを確認し、条件を満たさない場合は 0 を返す。 3. find_among_b を使用し、a_1 配列（所有格代名詞の定義）に基づいて末尾からパターンマッチングを行う。 4. マッチした場合、z->bra を更新し、slice_del を呼び出して該当部分を削除する。 5. 削除成功後、環境内の母音数カウンタ z->I[1] を 1 減算し、1 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/snowball/libstemmer/stem_ISO_8859_1_indonesian.c
  • src/backend/snowball/libstemmer/stem_UTF_8_indonesian.c
  • **呼び出し先**:
  • find_among_b (Snowballランタイム)
  • slice_del (Snowballランタイム)

関連データ構造

  • SN_env: ステミング処理の状態を保持する構造体。
  • a_1: 所有格代名詞のパターンを定義した among 構造体の配列。

失敗モードと境界条件

  • 単語長が不足している場合（z->c - 1 <= z->lb）、処理を中断し 0 を返す。
  • 末尾文字が a または u 以外の場合、0 を返す。
  • find_among_b が一致するパターンを見つけられない場合、0 を返す。
  • slice_del が内部エラー（負の値）を返した場合、その値をそのまま伝播させる。

保守担当者向けメモ

  • 本関数は自動生成コードの一部です。ロジックの変更が必要な場合は、元のSnowballスクリプト（.sbl）を修正してください。
  • I[1]（母音数カウンタ）の更新は、ステミングの妥当性判定に影響するため、削除処理と同期している必要があります。
