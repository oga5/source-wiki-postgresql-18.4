binaryheap_reset

定義

  • 種別: function
  • 定義箇所: src/common/binaryheap.c:62

目的

既存のバイナリヒープ構造体の状態をリセットし、要素が空の状態へ初期化します。

シグネチャまたは形状

void binaryheap_reset(binaryheap *heap)

入力と出力

  • **入力**: binaryheap *heap - 操作対象のヒープ構造体へのポインタ。
  • **出力**: なし（副作用として heap のメンバを更新）。

前提条件と不変条件

  • heap は binaryheap_allocate 等によって既にメモリが確保され、初期化されている必要があります。
  • この関数はヒープのメモリ解放は行いません。

アルゴリズムまたは振る舞い

1. heap->bh_size を 0 に設定し、ヒープ内の要素数をリセットします。 2. heap->bh_has_heap_property を true に設定し、ヒーププロパティが維持されている状態としてマークします。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/executor/nodeGatherMerge.c:468
  • src/backend/executor/nodeMergeAppend.c:390
  • src/backend/postmaster/pgarch.c:686
  • **呼び出し先**: なし（本関数は他の関数を呼び出しません）。

関連データ構造

  • binaryheap: src/include/lib/binaryheap.h で定義されるヒープ管理構造体。

失敗モードと境界条件

  • heap が NULL の場合、セグメンテーション違反が発生します。

保守担当者向けメモ

  • 本関数はメモリを解放しません。メモリの解放が必要な場合は binaryheap_free を使用してください。
  • bh_has_heap_property を true に戻すことで、リセット後の空の状態を「ヒーププロパティを満たしている」と定義しています。
