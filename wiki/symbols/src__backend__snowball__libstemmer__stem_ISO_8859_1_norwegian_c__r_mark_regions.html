r_mark_regions

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_norwegian.c:133

目的

単語内の処理対象範囲（Region）を特定し、接尾辞削除が許可される境界位置（I[1]）を計算する。

シグネチャまたは形状

static int r_mark_regions(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z（ステミング環境の状態を保持する構造体）
  • 出力: 成功時に 1、失敗時に 0 を返す。内部状態 z->I[0] および z->I[1] を更新する。

前提条件と不変条件

  • z が有効な SN_env 構造体を指していること。
  • g_v（母音グループ）が定義されていること。

アルゴリズムまたは振る舞い

1. z->I[1] を単語の末尾（z->l）に初期化する。 2. 単語の先頭から3文字目以降の位置を計算し、z->I[0] に格納する。 3. 母音（g_v）の出現位置を検索する。

  • out_grouping で最初の母音をスキップし、続く in_grouping で母音以外の文字を検索する。
  • 見つかった位置を z->I[1] に設定する。

    • 4. z->I[1] が z->I[0] より小さい場合、z->I[1] を z->I[0] に更新する。 5. 最終的に 1 を返して終了する。

呼び出し元と呼び出し先

  • 呼び出し元: 提供されたソース抜粋には記載なし。
  • 呼び出し先: out_grouping, in_grouping（Snowballランタイム関数）。

関連データ構造

  • struct SN_env: ステミング環境の状態（単語バッファ、カーソル位置、整数配列 I など）を保持する。

失敗モードと境界条件

  • z->c + 3 が単語長 z->l を超える場合、0 を返して終了する。
  • 母音が見つからない場合、out_grouping または in_grouping の戻り値により 0 を返して終了する。

保守担当者向けメモ

  • 本関数はSnowballコンパイラにより自動生成されている。ロジックの変更が必要な場合は、対応するSnowballソース（.sbl）を修正すること。
  • z->I[0] および z->I[1] は、後続の接尾辞削除処理における境界条件として使用される。
