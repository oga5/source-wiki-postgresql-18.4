TupleDescCopy

定義

  • 種別: function
  • 定義箇所: src/backend/access/common/tupdesc.c:427

目的

既存のタプル記述子（src）の内容を、別のタプル記述子（dst）へコピーする。制約やデフォルト値はコピーせず、属性定義のフラットコピーと最適化用メタデータの再構築を行う。

シグネチャまたは形状

void TupleDescCopy(TupleDesc dst, TupleDesc src)

入力と出力

  • dst: コピー先の TupleDesc（事前に十分なメモリが確保されている必要がある）。
  • src: コピー元の TupleDesc。
  • 戻り値: なし。

前提条件と不変条件

  • dst は src の内容を保持するのに十分なサイズで割り当てられていること。
  • 本関数は制約（TupleConstr）をコピーしないため、実行後に dst->constr は NULL に設定される。
  • 参照カウント（tdrefcount）はコピーされず、-1（参照カウント管理対象外）に初期化される。

アルゴリズムまたは振る舞い

1. memcpy を使用し、src のヘッダーおよび属性配列を dst へフラットコピーする。 2. 各属性について、制約に関連するフィールド（attnotnull, atthasdef, atthasmissing, attidentity, attgenerated）を初期値（false または '\0'）にリセットする。 3. 各属性に対して populate_compact_attribute を呼び出し、CompactAttribute を再構築する。 4. dst->constr を NULL に設定する。 5. dst->tdrefcount を -1 に設定する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/access/common/indextuple.c:600
  • src/backend/utils/cache/typcache.c:2929
  • **呼び出し先**:
  • memcpy（標準ライブラリ）
  • TupleDescSize（マクロ）
  • TupleDescAttr（マクロ）
  • populate_compact_attribute（内部関数）

関連データ構造

  • TupleDesc: タプルの構造を定義する主要なデータ構造。
  • Form_pg_attribute: 各列の属性情報を保持する構造体。

失敗モードと境界条件

  • dst が src よりも小さいメモリ領域を指している場合、memcpy によりバッファオーバーランが発生する。
  • 本関数はメモリ割り当てを行わないため、呼び出し側で dst の領域確保を適切に行う必要がある。

保守担当者向けメモ

  • 本関数は制約をコピーしないため、制約を含めたコピーが必要な場合は CreateTupleDescCopyConstr を検討すること。
  • CompactAttribute の整合性を保つため、属性情報の変更後は必ず populate_compact_attribute が実行される設計となっている。
