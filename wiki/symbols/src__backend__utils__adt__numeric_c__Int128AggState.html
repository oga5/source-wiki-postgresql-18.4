Int128AggState

定義

  • 種別: struct
  • 定義箇所: src/backend/utils/adt/numeric.c:5586

目的

集約関数（SUM、AVG、STDDEV 等）の計算過程において、int128 型の数値データの中間状態を保持するために使用されます。

シグネチャまたは形状

typedef struct Int128AggState
{
    bool        calcSumX2;      /* sumX2 を計算するかどうか */
    int64       N;              /* 処理された数値の個数 */
    int128      sumX;           /* 処理された数値の合計 */
    int128      sumX2;          /* 処理された数値の二乗和 */
} Int128AggState;

入力と出力

  • **入力**: do_int128_accum 関数を介して int128 型の新しい値を受け取り、状態を更新します。
  • **出力**: numeric_poly_stddev_internal 等の関数が、蓄積された N、sumX、sumX2 を参照して最終的な集約結果を算出します。

前提条件と不変条件

  • N は処理された数値の個数を表し、負の値にはなりません。
  • calcSumX2 が true の場合のみ、sumX2 が有効な計算値として扱われます。

アルゴリズムまたは振る舞い

  • palloc0 を使用してメモリが割り当てられ、各フィールドが初期化されます。
  • do_int128_accum により、新しい値が加算され、N がインクリメントされます。
  • do_int128_discard により、既存の状態から値を取り除く（減算する）操作が可能です。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/adt/numeric.c 内の集約処理関数群。
  • **呼び出し先**: palloc0（メモリ確保）。
  • 注: 呼び出し元の詳細な関数名や、本構造体を直接操作する外部モジュールの有無については、提供されたソース情報からは網羅的に特定できません。

関連データ構造

  • PolyNumAggState: Int128AggState の別名として定義されています（typedef Int128AggState PolyNumAggState;）。

失敗モードと境界条件

  • int128 の範囲を超える加算が行われた場合のオーバーフローについては、本構造体自体には保護機能は含まれていません。

保守担当者向けメモ

  • 本構造体は numeric.c 内の集約ロジックに密結合しています。変更を加える際は、do_int128_accum および do_int128_discard の実装への影響を確認してください。
