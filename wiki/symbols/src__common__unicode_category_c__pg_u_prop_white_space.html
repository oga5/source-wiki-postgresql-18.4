pg_u_prop_white_space

定義

  • 種別: function
  • 定義箇所: src/common/unicode_category.c:169

目的

指定された Unicode コードポイントが、Unicode 規格における「空白文字（White Space）」プロパティを持つかどうかを判定します。

シグネチャまたは形状

bool pg_u_prop_white_space(pg_wchar code)

入力と出力

  • **入力**: pg_wchar code（判定対象の Unicode コードポイント）
  • **出力**: bool（該当する場合は true、それ以外は false）

前提条件と不変条件

  • 入力コードポイントは有効な Unicode コードポイントである必要があります。
  • Unicode 規格の安定性ポリシーに基づき、判定基準となるカテゴリ値は固定されています。

アルゴリズムまたは振る舞い

1. **ASCII 範囲の最適化**: コードポイントが 0x80 未満の場合、unicode_opt_ascii テーブルを参照し、PG_U_PROP_WHITE_SPACE ビットマスクを用いて定数時間で判定結果を返します。 2. **広域範囲の検索**: 0x80 以上のコードポイントについては、unicode_white_space テーブルに対して range_search を実行し、該当範囲に含まれるかを二分探索で判定します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/common/unicode/category_test.c:73
  • src/common/unicode_category.c:313
  • **呼び出し先**:
  • range_search（内部ユーティリティ）

関連データ構造

  • unicode_opt_ascii: ASCII 文字のプロパティを保持する静的テーブル。
  • unicode_white_space: 空白文字の範囲を定義する静的テーブル（unicode_category_table.h で定義）。

失敗モードと境界条件

  • 境界条件: 0x7F と 0x80 の境界で検索アルゴリズムが切り替わりますが、一貫した結果を返すよう設計されています。
  • 失敗モード: 特筆すべき失敗モードはありません。
