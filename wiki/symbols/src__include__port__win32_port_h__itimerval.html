itimerval

定義

  • 種別: struct
  • 定義箇所: src/include/port/win32_port.h:181

目的

Windows 環境において POSIX 準拠のインターバルタイマー機能を提供するためのデータ構造です。

シグネチャまたは形状

struct itimerval
{
    struct timeval it_interval; /* 次回以降のタイマー間隔 */
    struct timeval it_value;    /* 次のタイマー満了までの時間 */
};

入力と出力

  • 入力: setitimer 関数の引数として、タイマーの設定値（it_interval および it_value）を受け取ります。
  • 出力: setitimer 関数の引数として、以前のタイマー設定値が格納されます。

前提条件と不変条件

  • struct timeval 型のフィールドを保持します。
  • Windows 環境専用の定義であり、POSIX 互換レイヤーの一部として機能します。

アルゴリズムまたは振る舞い

  • it_value はタイマーが満了するまでの残り時間を指定します。
  • it_interval はタイマーが満了した後に再設定される間隔を指定します。
  • setitimer 関数を通じて、タイマーの開始や停止、状態取得に使用されます。

呼び出し元と呼び出し先

  • 呼び出し元（直接的な参照）:
  • src/backend/port/win32/timer.c
  • src/backend/postmaster/fork_process.c
  • src/backend/utils/misc/timeout.c
  • src/bin/psql/command.c
  • 呼び出し先:
  • setitimer 関数（src/include/port/win32_port.h:187 で宣言）が本構造体を利用します。

関連データ構造

  • struct timeval: タイマーの時間を保持するために使用されます。

失敗モードと境界条件

  • メモリ操作（MemSet 等）において、構造体サイズが正しく計算されている必要があります。
  • Windows のタイマー精度や実装上の制限により、POSIX 環境と挙動が厳密に一致しない可能性があります。

保守担当者向けメモ

  • 本構造体は src/include/port/win32_port.h で定義されており、Windows ビルド時にのみ有効です。
  • setitimer の実装（src/backend/port/win32/timer.c）と密接に関連しているため、変更時は両方の整合性を確認してください。
