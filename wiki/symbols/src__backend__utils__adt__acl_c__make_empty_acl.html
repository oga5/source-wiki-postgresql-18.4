make_empty_acl

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/acl.c:448

目的

要素を一つも含まない空のアクセス制御リスト（ACL）構造体を生成し、初期化します。

シグネチャまたは形状

Acl *make_empty_acl(void);

入力と出力

  • **入力**: なし。
  • **出力**: 空の Acl 構造体へのポインタ。

前提条件と不変条件

  • 内部的に allocacl(0) を呼び出し、メモリ割り当てを行います。
  • 返される Acl 構造体は、有効な空の ACL として後続の ACL 操作関数で使用可能です。

アルゴリズムまたは振る舞い

1. allocacl(0) を呼び出し、要素数 0 の ACL 構造体をメモリ上に確保します。 2. 確保されたポインタを呼び出し元に返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/catalog/aclchk.c:1175
  • **呼び出し先**:
  • allocacl (内部関数)

関連データ構造

  • Acl: PostgreSQL のアクセス制御リストを保持する可変長構造体。

失敗モードと境界条件

  • メモリ不足時には allocacl の挙動に従い、メモリ割り当てエラーが発生する可能性があります。

保守担当者向けメモ

  • 本関数は src/include/utils/acl.h で外部公開されています。
  • ACL の初期化が必要な箇所で一貫して使用してください。
