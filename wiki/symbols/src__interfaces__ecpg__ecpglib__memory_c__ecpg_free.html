ecpg_free

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/ecpglib/memory.c:12

目的

標準の free 関数をラップし、ECPGライブラリ内でのメモリ解放処理を統一的に提供します。

シグネチャまたは形状

void ecpg_free(void *ptr)

入力と出力

  • **入力**: ptr (解放対象のメモリ領域を指すポインタ)
  • **出力**: なし

前提条件と不変条件

  • ptr は malloc 等で確保された有効なメモリ領域、または NULL である必要があります。
  • free の仕様に従い、ptr が NULL の場合は何も行われません。

アルゴリズムまたは振る舞い

1. 引数として受け取ったポインタ ptr を標準の free() 関数に渡して実行します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/interfaces/ecpg/ecpglib/connect.c 内の複数の箇所で、接続キャッシュや接続情報（name, dbname, host, port, options, realname 等）の解放に使用されています。
  • **呼び出し先**: 標準ライブラリの free()。

関連データ構造

  • 特になし（標準のメモリ管理機能に依存）。

失敗モードと境界条件

  • free() の動作に準拠します。無効なポインタや二重解放（double free）を行った場合の挙動は、標準の free() の実装に依存します。

保守担当者向けメモ

  • 本関数は単なる free のラッパーです。ECPGライブラリが提供する「自動メモリ追跡機能」を利用して確保されたメモリを解放する場合は、本関数ではなく ECPGfree_auto_mem を使用してください。
