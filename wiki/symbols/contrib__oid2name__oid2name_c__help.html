help

定義

  • 種別: function
  • 定義箇所: contrib/oid2name/oid2name.c:200

目的

oid2name コマンドの利用方法、利用可能なオプション、および接続パラメータを標準出力に表示する。

シグネチャまたは形状

static void help(const char *progname)

入力と出力

  • **入力**: progname (実行ファイル名を示す文字列ポインタ)
  • **出力**: なし（標準出力へのテキスト書き出しのみ）

前提条件と不変条件

  • progname は有効な文字列ポインタであること。
  • PACKAGE_BUGREPORT、PACKAGE_NAME、PACKAGE_URL マクロがビルド環境で定義されていること。

アルゴリズムまたは振る舞い

1. printf を使用し、ツールの概要、コマンドライン引数のリスト、接続オプションのリストを整形して出力する。 2. 出力文字列には、引数として渡された progname が埋め込まれる。 3. 最後に、バグ報告先およびプロジェクトのホームページ URL を表示して終了する。

呼び出し元と呼び出し先

  • **呼び出し元**: main 関数内において、ユーザーが --help または -? オプションを指定した場合に呼び出される。
  • **呼び出し先**: 標準ライブラリの printf を使用。

関連データ構造

  • なし

失敗モードと境界条件

  • 標準出力への書き込みが失敗した場合の挙動は printf の仕様に依存する。

保守担当者向けメモ

  • オプションの追加や変更を行った際は、本関数の出力内容も同期して更新する必要がある。
