SeqType

定義

  • 種別: enum
  • 定義箇所: src/bin/pg_dump/pg_dump.c:109

目的

PostgreSQLのシーケンスオブジェクトが保持するデータ型（smallint, integer, bigint）を識別するために使用されます。

シグネチャまたは形状

typedef enum SeqType
{
	SEQTYPE_SMALLINT,
	SEQTYPE_INTEGER,
	SEQTYPE_BIGINT,
} SeqType;

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: なし

前提条件と不変条件

  • pg_dump において、シーケンスのデータ型を表現する列挙値として利用されます。

アルゴリズムまたは振る舞い

  • シーケンスの定義情報を処理する際、対応するデータ型をこの列挙型にマッピングして保持します。
  • src/bin/pg_dump/pg_dump.c 内の静的インライン関数において、整数値から SeqType へのキャスト変換が行われています。

呼び出し元と呼び出し先

  • 呼び出し元: src/bin/pg_dump/pg_dump.c 内のシーケンス処理ロジック（SeqType seqtype メンバ変数として利用）。
  • 呼び出し先: 不明（ソース抜粋からは直接の関数呼び出し関係は確認できません）。

関連データ構造

  • SequenceItem: シーケンス情報を保持する構造体で、SeqType 型のメンバ seqtype を持ちます。

失敗モードと境界条件

  • 整数値から SeqType へのキャスト時に、定義外の値が渡された場合の挙動は、キャスト先のロジックに依存します。

保守担当者向けメモ

  • 本シンボルは pg_dump におけるシーケンスの型定義を抽象化するものです。新しいシーケンス型が追加された場合は、この列挙型を拡張する必要があります。
