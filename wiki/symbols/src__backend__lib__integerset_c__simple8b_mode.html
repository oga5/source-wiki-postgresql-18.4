simple8b_mode

定義

  • 種別: struct
  • 定義箇所: src/backend/lib/integerset.c:820

目的

Simple-8b アルゴリズムにおける各エンコーディングモードの構成を定義する構造体です。1つの 64 ビットコードワード内に格納される整数の数と、各整数に割り当てるビット数を管理します。

シグネチャまたは形状

struct simple8b_mode {
    uint8 bits_per_int;
    uint8 num_ints;
};

入力と出力

  • 入力: なし（静的な構成定義として使用）
  • 出力: なし

前提条件と不変条件

  • simple8b_modes 配列の要素として定義されており、合計 17 種類のモードが存在します。
  • 各モードは、64 ビットのワード長に収まるように bits_per_int と num_ints が調整されています。

アルゴリズムまたは振る舞い

  • 本構造体は、simple8b_encode および simple8b_decode 関数において、圧縮対象の整数列をどのモードでパッキングするかを選択するための参照テーブルとして機能します。
  • bits_per_int は各整数を表現するために必要なビット幅を示し、num_ints はそのビット幅で 64 ビットワード内に格納可能な整数の個数を示します。

呼び出し元と呼び出し先

  • 呼び出し元: simple8b_encode および simple8b_decode（推測される利用箇所）。
  • 呼び出し先: 情報なし。

関連データ構造

  • simple8b_modes: simple8b_mode 型の 17 要素からなる静的配列。

失敗モードと境界条件

  • 境界条件: simple8b_modes 配列のインデックスは 0 から 16 の範囲内である必要があります。範囲外のアクセスは未定義の動作を引き起こします。

保守担当者向けメモ

  • 本構造体は integerset.c 内の圧縮ロジックの根幹を成すため、変更時には Simple-8b のビットパッキング計算式との整合性を厳密に確認してください。
