check_final_sigma

定義

  • 種別: function
  • 定義箇所: src/common/unicode_case.c:311

目的

ギリシャ語の最終シグマ（Final Sigma）の文脈依存変換を判定する。対象文字が単語の末尾にあるか、あるいは「Cased」な文字に後続されない位置にあるかを検証する。

シグネチャまたは形状

static bool check_final_sigma(const unsigned char *str, size_t len, size_t offset)

入力と出力

  • str: UTF-8 エンコードされた文字列へのポインタ。
  • len: 文字列のバイト長。
  • offset: 判定対象となる文字の開始位置（バイトオフセット）。
  • 戻り値: 最終シグマへ変換すべき場合は true、そうでない場合は false を返す。

前提条件と不変条件

  • 入力文字列 str は有効な UTF-8 であると仮定される。
  • offset は文字列内の有効な文字境界を指している必要がある。

アルゴリズムまたは振る舞い

1. **前方走査**: offset より前のバイトを逆方向に走査し、直近の「Cased」な文字を探す。

  • Case-Ignorable な文字はスキップする。
  • Cased な文字が見つかった時点で走査を終了する。
  • Cased ではない文字（かつ Case-Ignorable でない）に遭遇した場合、最終シグマではないと判断し false を返す。

    • 2. **後方走査**: offset 以降のバイトを走査し、直近の「Cased」な文字を探す。

  • Cased な文字が見つかった場合、単語末尾ではないと判断し false を返す。
  • Case-Ignorable な文字はスキップする。

    • 3. 上記の条件を満たし、単語の末尾と判定された場合に true を返す。

呼び出し元と呼び出し先

  • 呼び出し元: src/common/unicode_case.c 内の特殊ケースマッピング処理から呼び出される。
  • 呼び出し先: utf8_to_unicode, pg_u_prop_case_ignorable, pg_u_prop_cased を利用して文字プロパティを判定する。

関連データ構造

  • pg_wchar: Unicode コードポイントを保持する型。

失敗モードと境界条件

  • offset == 0: 文字列の先頭であるため、直前に Cased 文字が存在しないとみなされ false を返す。
  • offset == len: 文字列の末尾であるため、直後に Cased 文字が存在しないとみなされ true を返す。
  • 無効な UTF-8 シーケンス: Assert(false) により異常終了する。

保守担当者向けメモ

  • 本関数はギリシャ語の特殊なケースマッピングに特化している。
  • pg_u_prop_cased および pg_u_prop_case_ignorable の定義は src/common/unicode_category.c に依存している。
