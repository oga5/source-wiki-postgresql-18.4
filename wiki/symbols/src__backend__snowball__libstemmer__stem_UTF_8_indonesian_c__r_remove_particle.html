r_remove_particle

定義

  • **種別**: function
  • **定義箇所**: src/backend/snowball/libstemmer/stem_UTF_8_indonesian.c:123

目的

インドネシア語のステミング処理において、単語末尾から粒子（particle）を除去する。

シグネチャまたは形状

static int r_remove_particle(struct SN_env * z);

入力と出力

  • **入力**: struct SN_env * z（ステミング環境の状態を保持する構造体）
  • **出力**: int（成功時に 1、失敗または条件不一致時に 0、エラー発生時に負の値を返す）

前提条件と不変条件

  • z->c（現在のカーソル位置）が有効な範囲にあること。
  • z->lb（下限境界）よりもカーソル位置が十分後ろにあること。
  • 除去対象の粒子が a_0 配列に定義されているパターンと一致すること。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を z->ket に設定する。 2. カーソル位置が境界に近いか、末尾の文字が 'h' (104) または 'n' (110) でない場合は処理を中断し 0 を返す。 3. find_among_b を使用し、a_0 配列に基づいて後方から粒子を検索する。一致しない場合は 0 を返す。 4. 一致した粒子の開始位置を z->bra に設定する。 5. slice_del を呼び出し、該当する粒子をバッファから削除する。削除に失敗した場合はそのエラーコードを返す。 6. 内部状態 z->I[1]（ステミングの進行状況を示す整数）を 1 減算する。 7. 成功として 1 を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/snowball/libstemmer/stem_UTF_8_indonesian.c および src/backend/snowball/libstemmer/stem_ISO_8859_1_indonesian.c 内のメインステミング処理フローから呼び出される。
  • **呼び出し先**: find_among_b（パターンマッチング用）、slice_del（バッファ操作用）。

関連データ構造

  • struct SN_env: ステミング環境を管理する構造体。
  • a_0: 粒子除去のパターンマッチングに使用される among 構造体の配列。

失敗モードと境界条件

  • カーソル位置が z->lb + 2 以下の場合、処理は行われない。
  • 粒子が特定できない場合、または slice_del が内部エラー（負の戻り値）を返した場合に失敗する。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによって自動生成されているため、直接編集せず、元の .sbl スクリプトを修正すること。
  • z->I[1] の更新はステミングの進行状態に直結するため、変更時は他の接辞除去ロジックとの整合性に注意が必要。
