pgstat_archiver_init_shmem_cb

定義

  • 種別: function
  • 定義箇所: src/backend/utils/activity/pgstat_archiver.c:65

目的

WAL アーカイバ統計情報を保持する共有メモリ領域の初期化を行う。

シグネチャまたは形状

void pgstat_archiver_init_shmem_cb(void *stats)

入力と出力

  • **入力**: void *stats - 初期化対象となる共有メモリ上の PgStatShared_Archiver 構造体へのポインタ。
  • **出力**: なし。

前提条件と不変条件

  • 引数 stats は PgStatShared_Archiver 型としてキャスト可能であること。
  • 共有メモリの割り当てが完了した後に呼び出されること。

アルゴリズムまたは振る舞い

1. 引数 stats を PgStatShared_Archiver * 型にキャストする。 2. 構造体内の lock メンバに対して LWLockInitialize を実行し、LWTRANCHE_PGSTATS_DATA トランシェを使用してロックを初期化する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/activity/pgstat.c:395 において、統計サブシステムの初期化プロセスからコールバックとして参照されている。
  • **呼び出し先**: LWLockInitialize（PostgreSQL のロック管理関数）。

関連データ構造

  • PgStatShared_Archiver: アーカイバ統計を格納する共有メモリ構造体。
  • LWLock: 統計データへのアクセスを制御するための軽量ロック。

失敗モードと境界条件

  • 引数が NULL の場合、構造体へのアクセス時にセグメンテーション違反が発生する可能性がある。

保守担当者向けメモ

  • 本関数は統計サブシステムの共有メモリ初期化フェーズで呼び出される。ロックの初期化のみを行うため、統計値自体のリセットは行わない。
