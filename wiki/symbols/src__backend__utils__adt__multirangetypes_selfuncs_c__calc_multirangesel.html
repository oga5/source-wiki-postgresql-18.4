calc_multirangesel

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/multirangetypes_selfuncs.c:290

目的

マルチレンジ型に対する演算子の選択率（selectivity）を、統計情報（ヒストグラム、NULL率、空集合率）に基づいて推定します。

シグネチャまたは形状

static double calc_multirangesel(TypeCacheEntry *typcache, VariableStatData *vardata, const MultirangeType *constval, Oid operator);

入力と出力

  • **入力**:
  • typcache: 型キャッシュエントリ。
  • vardata: 統計情報を含む変数データ。
  • constval: 比較対象の定数マルチレンジ値。
  • operator: 評価対象の演算子OID。
  • **出力**: double 型の選択率（0.0 から 1.0 の範囲）。

前提条件と不変条件

  • pg_statistic から取得した統計情報（NULL率、空集合率、境界値ヒストグラム）を利用します。
  • 統計情報が欠如している場合は、NULL率と空集合率を 0.0 と仮定して計算します。
  • 最終的な選択率は CLAMP_PROBABILITY マクロにより 0.0〜1.0 の範囲に収められます。
  • すべてのマルチレンジ演算子は「厳密（strict）」であるとみなされます。

アルゴリズムまたは振る舞い

1. **統計取得**: vardata から NULL率と空集合率を取得します。 2. **空集合の処理**: constval が空の場合、演算子に応じて固定値（0.0, 1.0, または empty_frac）を返します。 3. **ヒストグラム計算**: constval が空でない場合、calc_hist_selectivity を呼び出し、境界値ヒストグラムを用いて選択率を算出します。失敗した場合は default_multirange_selectivity にフォールバックします。 4. **統合**: 空集合の割合とヒストグラムによる推定値を合成します。 5. **補正**: 最後に NULL率を考慮して値を調整し、確率範囲に収めて返却します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/adt/multirangetypes_selfuncs.c 内の関数（multirangesel 等から参照される）。
  • **呼び出し先**: calc_hist_selectivity, default_multirange_selectivity, get_attstatsslot, free_attstatsslot, MultirangeIsEmpty。

関連データ構造

  • TypeCacheEntry: 型情報のキャッシュ。
  • VariableStatData: 統計情報の保持用構造体。
  • MultirangeType: マルチレンジ型の内部表現。
  • AttStatsSlot: 統計スロット。

失敗モードと境界条件

  • get_attstatsslot で期待される統計情報が取得できない場合、デフォルト値（0.0）として扱います。
  • 未定義の演算子が渡された場合、elog(ERROR) を発生させます。
  • sslot.nnumbers が 1 でない場合、統計情報の不整合としてエラーを発生させます。

保守担当者向けメモ

  • 本関数はレンジの長さと下限値が独立しているという統計的仮定に基づいています。
  • 演算子の追加や変更時には、switch 文の網羅性を確認してください。
  • 積分計算を含む calc_hist_selectivity の精度が推定結果に直結するため、統計データの品質（ANALYZE の実行状況）が重要です。
