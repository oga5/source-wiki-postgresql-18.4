jsonb_subscript_assign

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/jsonbsubs.c:260

目的

jsonb 型のデータ構造に対して、添字（subscript）を用いた要素の挿入または更新を行う。

シグネチャまたは形状

static void jsonb_subscript_assign(ExprState *state, ExprEvalStep *op, ExprContext *econtext)

入力と出力

  • **入力**:
  • op->d.sbsref.state: 添字操作の状態を保持する SubscriptingRefState。
  • op->resnull / op->resvalue: 操作対象となる jsonb データ（Datum）。
  • **出力**:
  • op->resvalue: 更新後の jsonb データが格納される。

前提条件と不変条件

  • op->d.sbsref.state から取得される workspace（JsonbSubWorkspace）が初期化済みであること。
  • sbsrefstate->replacevalue には、挿入または更新対象となる値が含まれていること。

アルゴリズムまたは振る舞い

1. **値の準備**: sbsrefstate->replacevalue を JsonbValue 形式に変換する。 2. **コンテナの初期化**: 入力 jsonb が NULL の場合、workspace->expectArray の値に応じて空の配列または空のオブジェクトを生成する。 3. **要素の更新**: jsonb_set_element を呼び出し、指定されたパス（workspace->index）に対して値を設定する。 4. **結果の格納**: 更新された jsonb を op->resvalue に書き戻す。結果は常に非 NULL として扱われる。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/jsonbsubs.c:392（jsonb_subscript_handler 構造体の sbs_assign フィールドに登録）。
  • **呼び出し先**:
  • JsonbToJsonbValue
  • JsonbValueToJsonb
  • jsonb_set_element
  • DatumGetJsonbP

関連データ構造

  • JsonbSubWorkspace: 添字の型情報や変換後の値を保持する。
  • SubscriptingRefState: 実行時の添字操作状態を管理する。
  • JsonbValue: jsonb の内部ノード表現。

失敗モードと境界条件

  • 入力コンテナが NULL の場合、expectArray のフラグに基づき、自動的に空の配列またはオブジェクトとして初期化される。
  • jsonb_set_element が失敗した場合、その内部ロジックに従いエラーが発生する可能性がある。

保守担当者向けメモ

  • 本関数は jsonb の添字操作フレームワークの一部であり、スライス操作はサポートしていない。
  • jsonb_set_element の戻り値が jsonb 構造全体を再構築するため、大規模な jsonb データに対する頻繁な更新はコストが高くなる可能性がある。
