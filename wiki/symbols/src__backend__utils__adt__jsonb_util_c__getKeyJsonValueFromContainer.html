getKeyJsonValueFromContainer

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/jsonb_util.c:401

目的

JsonbContainer（オブジェクト）内から、指定されたキー文字列に一致する値を検索し、JsonbValue 構造体として取得します。

シグネチャまたは形状

JsonbValue *getKeyJsonValueFromContainer(JsonbContainer *container, const char *keyVal, int keyLen, JsonbValue *res);

入力と出力

  • **入力**:
  • container: 検索対象の JsonbContainer。
  • keyVal: 検索するキーの文字列ポインタ。
  • keyLen: キーの長さ。
  • res: 結果を格納する JsonbValue 構造体へのポインタ（NULL の場合は内部で palloc される）。
  • **出力**:
  • キーが見つかった場合は res（または割り当てられたメモリ）、見つからない場合は NULL を返します。

前提条件と不変条件

  • container はオブジェクト型である必要があります（Assert(JsonContainerIsObject(container)) により検証）。
  • container 内のキーはソートされている前提でバイナリサーチが行われます。

アルゴリズムまたは振る舞い

1. コンテナが空の場合、即座に NULL を返します。 2. オブジェクトのバイナリ構造（JEntry のペア）に対してバイナリサーチを実行します。 3. lengthCompareJsonbString を用いて、候補となるキーと検索キーを比較します。 4. 一致した場合、対応する値のインデックスを特定し、fillJsonbValue を呼び出して res に値を書き込みます。 5. 検索キーが候補より小さい場合は左側、大きい場合は右側を探索します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/jsonb_util.c (複数箇所)
  • src/backend/utils/adt/jsonfuncs.c (複数箇所)
  • src/include/utils/jsonb.h (外部公開インターフェース)
  • **呼び出し先**:
  • JsonContainerSize
  • getJsonbOffset
  • getJsonbLength
  • lengthCompareJsonbString
  • fillJsonbValue
  • palloc (必要時)

関連データ構造

  • JsonbContainer: JSONB のバイナリコンテナ。
  • JsonbValue: メモリ上の JSONB ノード表現。
  • JEntry: コンテナ内の各要素のメタデータ（オフセットや長さ）。

失敗モードと境界条件

  • container がオブジェクト型でない場合、アサーションエラーが発生します。
  • キーが見つからない場合は NULL を返します。
  • res が NULL でメモリ確保が必要な際、palloc が失敗した場合はエラー（elog 等）が発生します。

保守担当者向けメモ

  • 本関数はバイナリサーチを利用しているため、JsonbContainer 内のキーがソートされているという不変条件が維持されていることが重要です。
  • res が NULL の場合に palloc を行うため、呼び出し側でメモリの解放が必要になるケースがある点に注意してください。
