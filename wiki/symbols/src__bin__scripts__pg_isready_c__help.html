help

定義

  • 種別: function
  • 定義箇所: src/bin/scripts/pg_isready.c:222

目的

pg_isready コマンドの利用方法、利用可能なオプション、およびバグ報告先や公式ホームページの情報を標準出力に表示する。

シグネチャまたは形状

static void help(const char *progname);

入力と出力

  • **入力**: progname（実行中のプログラム名を示す文字列）。
  • **出力**: なし（標準出力へのテキスト出力のみ）。

前提条件と不変条件

  • progname は有効な文字列ポインタであること。
  • DEFAULT_CONNECT_TIMEOUT、PACKAGE_BUGREPORT、PACKAGE_NAME、PACKAGE_URL が定義されている環境で実行されること。

アルゴリズムまたは振る舞い

1. progname を用いてプログラムの概要と使用法（Usage）を表示する。 2. 利用可能なコマンドラインオプション（-d, -q, -V, -?, -h, -p, -t, -U）の一覧を整形して表示する。 3. 最後にバグ報告先とプロジェクトのホームページ URL を表示する。 4. 各出力行は国際化（_() マクロ）に対応している。

呼び出し元と呼び出し先

  • **呼び出し元**: 提供されたソースコード内には、この関数を呼び出している箇所は明示されていません。
  • **呼び出し先**: 標準ライブラリの printf を呼び出します。
  • **注意**: ツールが検出した contrib/oid2name/oid2name.c 等の他のファイルにある help 関数は、名前が同一ですが本シンボルとは別物です。

関連データ構造

  • なし。

失敗モードと境界条件

  • 標準出力がクローズされている場合や書き込みエラーが発生した場合、printf の振る舞いに依存します。

保守担当者向けメモ

  • オプションの追加や変更を行った際は、本関数内のヘルプメッセージも必ず同期して更新してください。
  • DEFAULT_CONNECT_TIMEOUT などの定数は、pg_isready.c の他の箇所と整合性が取れていることを確認してください。
