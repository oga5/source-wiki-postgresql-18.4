r_deriv

目的

アイルランド語のステミング処理において、単語の派生語接尾辞を検索し、必要に応じて削除または置換を行う。

シグネチャまたは形状

static int r_deriv(struct SN_env * z)

入力と出力

  • 入力: SN_env 構造体（ステミング環境の状態を保持）。
  • 出力: 処理が成功した場合は 1、ルールにマッチしなかった場合や処理が中断された場合は 0、エラー発生時は負の値を返す。

前提条件と不変条件

  • z が有効な SN_env 構造体を指していること。
  • find_among_b を使用して、後方から派生語接尾辞の検索を行う。

アルゴリズムまたは振る舞い

1. z->ket を現在のカーソル位置 z->c に設定する。 2. a_2 配列（25個の要素）に対して find_among_b を実行し、接尾辞を検索する。 3. マッチした場合、z->bra を現在のカーソル位置に設定し、among_var の値に応じて以下の処理を行う。

  • case 1: r_R2 を呼び出し、領域 R2 内であれば slice_del で削除する。
  • case 2〜6: slice_from_s を使用し、マッチした接尾辞を対応する文字列（s_9〜s_13）に置換する。

    • 4. マッチしなかった場合は 0 を返す。

呼び出し元と呼び出し先

  • 呼び出し元: irish_ISO_8859_1_stem（src/backend/snowball/libstemmer/stem_ISO_8859_1_irish.c:453 および stem_UTF_8_irish.c:453）。
  • 呼び出し先: find_among_b, r_R2, slice_del, slice_from_s（いずれも Snowball ランタイム関数）。

関連データ構造

  • SN_env: ステミング環境を管理する構造体。
  • a_2: 派生語接尾辞の検索に使用される among 構造体の配列。

失敗モードと境界条件

  • find_among_b がマッチしない場合は 0 を返す。
  • r_R2 や slice_del 等の内部関数が負の値を返した場合、その値をそのまま呼び出し元へ伝播させる。
