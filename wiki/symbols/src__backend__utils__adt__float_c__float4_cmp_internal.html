float4_cmp_internal

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/float.c:815

目的

2つの float4 型の値を比較し、その大小関係を標準的な比較結果（-1, 0, 1）として返します。

シグネチャまたは形状

int float4_cmp_internal(float4 a, float4 b)

入力と出力

  • **入力**:
  • a: 比較対象の float4 値
  • b: 比較対象の float4 値
  • **出力**:
  • 1: a > b の場合
  • -1: a < b の場合
  • 0: a == b の場合

前提条件と不変条件

  • float4_gt および float4_lt 関数が正しく定義されている環境で動作します。

アルゴリズムまたは振る舞い

1. float4_gt(a, b) を呼び出し、真であれば 1 を返します。 2. float4_lt(a, b) を呼び出し、真であれば -1 を返します。 3. 上記のいずれでもない場合、0 を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/btree_gist/btree_float4.c
  • src/backend/utils/adt/float.c（複数の箇所で呼び出しあり）
  • **呼び出し先**:
  • float4_gt
  • float4_lt

関連データ構造

  • float4: PostgreSQLの単精度浮動小数点型。

失敗モードと境界条件

  • NaN（非数）が含まれる場合の挙動は、依存する float4_gt および float4_lt の実装に準拠します。

保守担当者向けメモ

  • 本関数は src/include/utils/float.h にて extern 宣言されており、モジュール外からも参照可能です。
  • 比較ロジックを変更する場合は、float4_gt および float4_lt との整合性に注意してください。
