SV_to_JsonbValue

定義

  • 種別: function
  • 定義箇所: contrib/jsonb_plperl/jsonb_plperl.c:176

目的

Perl のスカラー値 (SV) を PostgreSQL の JsonbValue 構造体に変換します。

シグネチャまたは形状

static JsonbValue *SV_to_JsonbValue(SV *in, JsonbParseState **jsonb_state, bool is_elem)

入力と出力

  • **入力**:
  • in: 変換対象の Perl スカラー (SV*)。
  • jsonb_state: Jsonb 構築状態を保持するポインタのポインタ。
  • is_elem: 配列の要素として処理するかどうかを示すフラグ。
  • **出力**:
  • 変換後の JsonbValue* を返します。jsonb_state が提供されている場合は、構築状態に追加された値へのポインタを返し、そうでない場合はメモリ確保した値を返します。

前提条件と不変条件

  • in がリファレンスである場合、再帰的にデリファレンスされます。
  • jsonb 型の仕様上、無限大 (inf) および NaN は変換できず、エラーとなります。
  • Perl の UV (符号なし整数) は、64ビットを超える可能性があるため、文字列変換を経由して numeric 型として扱われます。

アルゴリズムまたは振る舞い

1. **デリファレンス**: SvROK が真の間、SvRV を用いて再帰的にデリファレンスします。 2. **型判定**: SvTYPE に基づき処理を分岐します。

  • SVt_PVAV: AV_to_JsonbValue を呼び出します。
  • SVt_PVHV: HV_to_JsonbValue を呼び出します。
  • その他:
  • NULL: jbvNull を設定。
  • UV: 文字列変換後 numeric_in を介して jbvNumeric に変換。
  • IV: int64_to_numeric を介して jbvNumeric に変換。
  • NV: float8_numeric を介して jbvNumeric に変換（isinf/isnan チェックあり）。
  • PV: sv2cstr を介して jbvString に変換。

    • 3. **構築**: jsonb_state が存在すれば pushJsonbValue を呼び出し、存在しなければ palloc でメモリを確保して結果を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/jsonb_plperl/jsonb_plperl.c 内の各箇所（配列やハッシュの変換処理、およびトップレベルの変換関数から呼び出されます）。
  • **呼び出し先**:
  • AV_to_JsonbValue
  • HV_to_JsonbValue
  • pushJsonbValue
  • numeric_in, int64_to_numeric, float8_numeric (PostgreSQL 内部関数)

関連データ構造

  • JsonbValue: PostgreSQL の JSONB 内部表現。
  • JsonbParseState: JSONB 構築時の状態管理構造体。
  • SV, AV, HV: Perl のデータ構造体。

失敗モードと境界条件

  • **エラー**: サポートされていない Perl 型が渡された場合、ERRCODE_FEATURE_NOT_SUPPORTED でエラーを報告します。
  • **エラー**: 数値が inf または NaN の場合、ERRCODE_NUMERIC_VALUE_OUT_OF_RANGE でエラーを報告します。

保守担当者向けメモ

  • Perl のメモリ管理と PostgreSQL の palloc が混在するため、メモリリークに注意してください。
  • 再帰的な構造を持つため、深いネストのデータ構造に対してはスタックオーバーフローのリスクがあります。
