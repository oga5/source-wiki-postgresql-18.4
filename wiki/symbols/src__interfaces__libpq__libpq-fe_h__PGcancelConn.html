PGcancelConn

定義

  • 種別: struct
  • 定義箇所: src/interfaces/libpq/libpq-fe.h:207

目的

データベースサーバーに対するクエリのキャンセル操作を安全に実行するための、接続情報を保持する不透明な構造体です。

シグネチャまたは形状

typedef struct pg_cancel_conn PGcancelConn;

入力と出力

  • 本構造体は不透明ポインタとして扱われます。
  • PQcancelCreate 等の関数によって生成され、PGcancelConn* 型として返されます。

前提条件と不変条件

  • 内部メンバへの直接アクセスは禁止されています。
  • PGconn とは異なる構造体ですが、実装上は pg_cancel_conn のラッパーとして機能します。

アルゴリズムまたは振る舞い

  • 実行中のクエリを安全にキャンセルするために使用されます。
  • PGconn と PGcancelConn は相互に変換可能な関係として扱われる場合があります（fe-cancel.c の実装に基づく）。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/fe_utils/connect_utils.c:164 (PQcancelCreate を介して取得)
  • src/include/libpq/libpq-be-fe-helpers.h:388
  • **呼び出し先**:
  • src/interfaces/libpq/fe-cancel.c 内の関数群が本構造体の生成および操作を行います。

関連データ構造

  • PGconn: データベース接続を管理する主要な構造体。
  • pg_cancel_conn: PGcancelConn の実体となる内部構造体。

失敗モードと境界条件

  • エラー発生時に PGcancelConn をリークさせないよう管理する必要があります。

保守担当者向けメモ

  • 本構造体は libpq の公開 API の一部です。メンバの変更は後方互換性に影響するため注意してください。
  • fe-cancel.c に実装の詳細が含まれています。
