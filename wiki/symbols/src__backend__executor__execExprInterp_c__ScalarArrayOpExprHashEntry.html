ScalarArrayOpExprHashEntry

定義

  • 種別: struct
  • 定義箇所: src/backend/executor/execExprInterp.c:203

目的

ScalarArrayOpExpr（配列に対する演算子式）の評価において、比較対象の値を効率的に検索するためのハッシュテーブルの各要素を保持する。

シグネチャまたは形状

typedef struct ScalarArrayOpExprHashEntry
{
    Datum       key;            /* 比較対象のキー値 */
    uint32      status;         /* ハッシュテーブル内でのエントリ状態 */
    uint32      hash;           /* キャッシュされたハッシュ値 */
} ScalarArrayOpExprHashEntry;

入力と出力

本構造体はデータ保持用であり、関数ではないため直接の入出力はない。ハッシュテーブル操作を通じて、Datum 型のキー値が格納・参照される。

前提条件と不変条件

  • lib/simplehash.h を用いたハッシュテーブル実装の要素として使用される。
  • hash フィールドには、key から計算されたハッシュ値がキャッシュとして保持される必要がある。

アルゴリズムまたは振る舞い

  • src/backend/executor/execExprInterp.c 内で SH_ELEMENT_TYPE として定義されており、simplehash.h が提供するハッシュテーブル操作関数群（挿入、検索、削除等）の対象となる。
  • 配列演算の実行時に、配列内の各要素をハッシュテーブルに格納し、高速なルックアップを実現するために利用される。

呼び出し元と呼び出し先

  • 呼び出し元および呼び出し先の具体的な関数情報は、提供されたソース抜粋からは特定できない。
  • src/backend/executor/execExprInterp.c 内のハッシュテーブル操作ロジックによって参照される。

関連データ構造

  • ScalarArrayOpExprHashTable: 本構造体を要素として管理するハッシュテーブル構造体。
  • lib/simplehash.h: 本構造体を利用したハッシュテーブルの実装基盤。

失敗モードと境界条件

  • メモリ不足時、ハッシュテーブルの拡張や要素の追加に失敗する可能性がある。
  • key が Datum 型であるため、データ型に応じた適切なハッシュ関数と等価性比較関数がハッシュテーブル側で設定されている必要がある。

保守担当者向けメモ

  • 本構造体は simplehash.h のマクロ定義と密接に関連している。構造体のフィールドを変更する場合は、simplehash.h を利用している関連マクロ定義（SH_ELEMENT_TYPE 等）との整合性に注意すること。
