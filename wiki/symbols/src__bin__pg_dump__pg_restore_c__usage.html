usage

定義

  • 種別: function
  • 定義箇所: src/bin/pg_dump/pg_restore.c:532

目的

pg_restore コマンドの利用方法（ヘルプメッセージ）を標準出力に表示し、プログラムを終了させるための補助関数です。

シグネチャまたは形状

static void usage(const char *progname);

入力と出力

  • 入力: progname (実行ファイル名を示す文字列)
  • 出力: なし（標準出力へヘルプテキストを出力）

前提条件と不変条件

  • progname は有効な文字列である必要があります。
  • 本関数は呼び出された後、プログラムの実行を継続せず終了させることを前提としています。

アルゴリズムまたは振る舞い

1. pg_restore の概要と基本的な使用法（コマンドライン構文）を表示します。 2. 一般オプション、復元制御オプション、接続オプションをカテゴリ別に分類して出力します。 3. 複数のオブジェクト指定に関する注意書きを表示します。 4. 入力ファイルが指定されない場合の挙動（標準入力の使用）を説明します。 5. バグ報告先およびプロジェクトのホームページ情報を表示して終了します。

呼び出し元と呼び出し先

  • 呼び出し元: 提供されたソース抜粋内には、本関数を呼び出している箇所は含まれていません。
  • 呼び出し先: printf を使用してメッセージを出力します。

関連データ構造

  • なし

失敗モードと境界条件

  • 標準出力への書き込みが失敗した場合の挙動については、本関数内では定義されていません。

保守担当者向けメモ

  • 本関数は pg_restore のコマンドライン引数が増減するたびに更新する必要があります。
  • 翻訳対象の文字列は _() マクロで囲まれており、国際化（i18n）に対応しています。
