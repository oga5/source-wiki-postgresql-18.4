conditional_stack_depth

定義

  • 種別: function
  • 定義箇所: src/fe_utils/conditional.c:83

目的

指定された ConditionalStack が現在保持しているネストされた \if ブロックの深さを取得します。

シグネチャまたは形状

int conditional_stack_depth(ConditionalStack cstack)

入力と出力

  • **入力**: cstack (ConditionalStack 型) - 調査対象のスタック構造体。
  • **出力**: int - スタック内の要素数。cstack が NULL の場合は -1 を返します。

前提条件と不変条件

  • cstack が有効なポインタである場合、スタック内の head から始まるリンクリストを辿り、要素数をカウントします。

アルゴリズムまたは振る舞い

1. cstack が NULL かどうかを確認し、該当する場合は -1 を返します。 2. cstack が有効な場合、スタックの先頭 (head) から開始し、next ポインタを辿りながら要素を順次カウントします。 3. 全ての要素を走査した後、累積した深さを返します。

呼び出し元と呼び出し先

  • **呼び出し元**: 提供されたソース excerpt 内では具体的な呼び出し元は特定されていません。
  • **呼び出し先**: 内部で cstack->head および p->next を参照し、スタックのリンクリストを走査します。

関連データ構造

  • ConditionalStack: fe_utils/conditional.h で定義されるスタック管理構造体。
  • IfStackElem: スタックの各要素を構成するリンクリストノード。

失敗モードと境界条件

  • cstack が NULL の場合、エラーとして -1 を返します。
  • スタックが空の場合、ループは実行されず 0 を返します。

保守担当者向けメモ

  • 本関数はスタックの全要素を走査するため、スタックの深さに比例した計算量（O(N)）を要します。頻繁に呼び出す場合は注意してください。
