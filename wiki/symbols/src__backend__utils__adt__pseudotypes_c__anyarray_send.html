anyarray_send

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/pseudotypes.c:163

目的

擬似型である anyarray 型のデータを、クライアントへ送信するためのバイナリ形式に変換します。

シグネチャまたは形状

Datum anyarray_send(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS (関数マネージャ経由で渡される anyarray 型のデータ)。
  • **出力**: Datum (バイナリ形式に変換された bytea 型のデータ)。

前提条件と不変条件

  • 本関数は anyarray 型のバイナリ送信をサポートするために使用されます。
  • 内部的に array_send へ処理を委譲するため、array_send が要求する前提条件（有効な配列データであること等）に従います。

アルゴリズムまたは振る舞い

1. 受け取った fcinfo (関数呼び出し情報) をそのまま array_send 関数に渡します。 2. array_send の戻り値をそのまま呼び出し元へ返却します。

呼び出し元と呼び出し先

  • **呼び出し先**: array_send (PostgreSQL の配列型バイナリ送信関数)。
  • **呼び出し元**: 提供されたソース情報からは特定できません。

関連データ構造

  • Datum: PostgreSQL の汎用データ型。
  • PG_FUNCTION_ARGS: 関数呼び出しの引数コンテキスト。

失敗モードと境界条件

  • array_send が失敗する場合（無効な配列構造など）、本関数も同様にエラーを発生させます。

保守担当者向けメモ

  • 本関数は単なるラッパーであり、実質的なロジックは array_send に依存しています。
  • anyarray は擬似型ですが、本関数によりバイナリ送信が可能となっています。
