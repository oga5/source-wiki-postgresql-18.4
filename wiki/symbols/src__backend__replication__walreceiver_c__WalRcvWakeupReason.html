WalRcvWakeupReason

定義

  • 種別: enum
  • 定義箇所: src/backend/replication/walreceiver.c:118

目的

walreceiver プロセスが次に実行すべきタスクを決定し、待機状態から復帰する際の理由を識別するために使用されます。

シグネチャまたは形状

typedef enum WalRcvWakeupReason
{
    WALRCV_WAKEUP_TERMINATE,    /* プロセスの終了 */
    WALRCV_WAKEUP_PING,         /* Keepalive の送信 */
    WALRCV_WAKEUP_REPLY,        /* 受信状況の報告 */
    WALRCV_WAKEUP_HSFEEDBACK,   /* Hot Standby フィードバックの送信 */
    #define NUM_WALRCV_WAKEUPS (WALRCV_WAKEUP_HSFEEDBACK + 1)
} WalRcvWakeupReason;

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: WalRcvComputeNextWakeup 関数において、次回のウェイクアップ時刻を計算するための引数として利用されます。

前提条件と不変条件

  • NUM_WALRCV_WAKEUPS は、定義された理由の総数を保持する定数として機能します。
  • 各列挙値は、walreceiver が定期的に実行すべきバックグラウンドタスクと一対一で対応しています。

アルゴリズムまたは振る舞い

walreceiver プロセスは、この列挙値を用いて WalRcvComputeNextWakeup を呼び出し、各タスクの実行間隔に基づいた次回の起動時刻を管理します。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/replication/walreceiver.c 内の WalRcvComputeNextWakeup 関数がこの型を引数として受け取ります。
  • 呼び出し先: 情報なし。

関連データ構造

  • WalRcv: 共有メモリ上で walreceiver の状態を管理する構造体。

失敗モードと境界条件

  • 列挙値の範囲外の値を WalRcvComputeNextWakeup に渡した場合の挙動は定義されていません。

保守担当者向けメモ

  • 新たなウェイクアップ理由を追加する場合は、NUM_WALRCV_WAKEUPS の定義が正しく更新されるよう注意してください。
