l_ptr

定義

  • 種別: function
  • 定義箇所: src/include/jit/llvmjit_emit.h:38

目的

指定された LLVM 型に対するポインタ型を生成するラッパー関数です。LLVM C API の LLVMPointerType を呼び出す際の定型的な記述を簡略化します。

シグネチャまたは形状

static inline LLVMTypeRef l_ptr(LLVMTypeRef t)

入力と出力

  • **入力**: t (LLVMTypeRef) - ポインタの参照先となる LLVM 型。
  • **出力**: LLVMTypeRef - 参照先型 t を指すポインタ型。

前提条件と不変条件

  • 本関数は USE_LLVM が定義されている環境でのみ有効です。
  • アドレス空間は常に 0（デフォルト）として生成されます。

アルゴリズムまたは振る舞い

1. 引数として受け取った LLVMTypeRef 型 t を保持します。 2. LLVM C API の LLVMPointerType(t, 0) を呼び出し、アドレス空間 0 のポインタ型を生成して返します。 3. インライン関数として定義されており、呼び出し側のコンテキストで直接展開されます。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/jit/llvm/llvmjit_deform.c
  • src/backend/jit/llvm/llvmjit_expr.c
  • **呼び出し先**:
  • LLVMPointerType (LLVM C API)

関連データ構造

  • LLVMTypeRef: LLVM IR 上の型を表現する不透明なポインタ型。

失敗モードと境界条件

  • LLVM C API の仕様に依存します。無効な LLVMTypeRef を渡した場合の挙動は LLVM 側の実装に準拠します。

保守担当者向けメモ

  • 本関数は llvmjit_emit.h 内の他のヘルパー関数（l_ptr_const 等）と組み合わせて使用されることが一般的です。
  • アドレス空間を明示的に指定する必要が生じた場合は、本関数の拡張または新規ヘルパーの作成を検討してください。
