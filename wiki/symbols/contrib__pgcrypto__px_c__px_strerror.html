px_strerror

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/px.c:111

目的

pgcrypto モジュール内で使用される数値エラーコードを、人間が判読可能な説明文字列に変換します。

シグネチャまたは形状

const char *px_strerror(int err);

入力と出力

  • **入力**: err (int型) - pgcrypto 内部で定義されたエラーコード。
  • **出力**: const char * - エラーコードに対応する説明文字列。

前提条件と不変条件

  • px_err_list が有効な error_desc 構造体の配列として定義されている必要があります。
  • px_err_list は desc メンバが NULL である要素で終端されている必要があります。

アルゴリズムまたは振る舞い

1. px_err_list 配列を先頭から順に走査します。 2. 現在の要素の err メンバが入力された err と一致するか確認します。 3. 一致した場合は、その要素の desc 文字列を返します。 4. 配列の終端（desc が NULL）に達しても一致するものが見つからない場合、文字列 "Bad error code" を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/pgcrypto/pgcrypto.c (複数箇所: gen_salt, encrypt, decrypt, encrypt_iv, decrypt_iv 等の処理中)
  • contrib/pgcrypto/px.c (内部エラー報告時)
  • **呼び出し先**:
  • 本関数は直接的な関数呼び出しを行わず、静的なエラーリストを参照します。

関連データ構造

  • struct error_desc: エラーコードと説明文字列のペアを保持する構造体。px_err_list 配列の要素として使用されます。

失敗モードと境界条件

  • **境界条件**: 未定義のエラーコードが渡された場合、エラーを発生させる代わりに "Bad error code" という文字列を返します。

保守担当者向けメモ

  • 新しいエラーコードを追加する場合は、px_err_list に対応するエントリを追加してください。
  • 本関数は静的な文字列ポインタを返すため、呼び出し側でメモリ解放を行う必要はありません。
