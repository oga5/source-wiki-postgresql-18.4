initStringInfoExt

定義

  • 種別: function
  • 定義箇所: src/common/stringinfo.c:110

目的

指定された初期サイズで StringInfoData 構造体を初期化します。

シグネチャまたは形状

void initStringInfoExt(StringInfo str, int initsize)

入力と出力

  • **入力**:
  • str: 初期化対象となる StringInfo 構造体へのポインタ。
  • initsize: バッファの初期確保サイズ（バイト単位）。
  • **出力**: なし（str が指す構造体の内部状態が更新されます）。

前提条件と不変条件

  • str は有効な StringInfoData 構造体を指している必要があります。
  • 内部的に palloc（またはフロントエンド環境での malloc）を使用するため、メモリ不足時にはエラーが発生します。

アルゴリズムまたは振る舞い

1. 内部関数 initStringInfoInternal(str, initsize) を呼び出します。 2. StringInfoData の各フィールド（data, len, maxlen, cursor）を、指定された initsize に基づいて初期化します。

呼び出し元と呼び出し先

  • **呼び出し先**: initStringInfoInternal
  • **呼び出し元**: ソースコード内の参照情報に基づくと、src/include/lib/stringinfo.h 等から呼び出されることが想定されます。具体的な呼び出し箇所の詳細は提供されていません。

関連データ構造

  • StringInfoData: data（文字列バッファ）、len（現在長）、maxlen（最大容量）、cursor（走査位置）を保持する構造体。

失敗モードと境界条件

  • initsize に 0 以下の値を指定した場合の挙動は、内部の initStringInfoInternal の実装に依存します。
  • メモリ割り当てが失敗した場合、システムはエラーを報告し、呼び出し元へは戻りません。

保守担当者向けメモ

  • 本関数は initStringInfoInternal のラッパーです。ロジックの変更が必要な場合は、initStringInfoInternal の実装を確認してください。
