pg_md5_final

定義

  • 種別: function
  • 定義箇所: src/common/md5.c:431

目的

MD5 ハッシュ計算の最終処理を行い、計算されたメッセージダイジェストを出力バッファに書き出します。

シグネチャまたは形状

void pg_md5_final(pg_md5_ctx *ctx, uint8 *dest)

入力と出力

  • ctx: MD5 計算状態を保持する pg_md5_ctx 構造体へのポインタ。
  • dest: 計算結果（16バイトのダイジェスト）を格納するための uint8 配列へのポインタ。戻り値はありません。

前提条件と不変条件

  • ctx は pg_md5_init によって初期化され、pg_md5_update によってデータが供給されている必要があります。
  • dest は少なくとも16バイトのメモリ領域を指している必要があります。

アルゴリズムまたは振る舞い

1. md5_pad(ctx) を呼び出し、メッセージのパディング（ビット長情報の追加を含む）を実行します。 2. md5_result(dest, ctx) を呼び出し、内部状態から最終的なハッシュ値を抽出し、dest が指すバッファへ書き込みます。

呼び出し元と呼び出し先

  • 呼び出し元: src/common/cryptohash.c:185
  • 呼び出し先: md5_pad, md5_result

関連データ構造

  • pg_md5_ctx: src/common/md5_int.h で定義される、ハッシュ計算の状態を保持する構造体。

失敗モードと境界条件

  • ctx が NULL の場合、または dest が十分なメモリを指していない場合の動作は未定義です。

保守担当者向けメモ

  • 本関数は RFC 1321 に準拠したフォールバック実装です。
  • 現代のセキュリティ要件（衝突耐性など）が求められる用途には適していません。
  • ビッグエンディアン環境では、内部でバイト順序の調整が行われます。
