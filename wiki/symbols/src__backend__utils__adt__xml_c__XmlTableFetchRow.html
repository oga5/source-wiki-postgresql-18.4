XmlTableFetchRow

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/xml.c:4880

目的

XMLTABLE 演算子の実行において、XPath評価結果から次の行を取得し、処理対象が存在するかを判定します。

シグネチャまたは形状

static bool XmlTableFetchRow(TableFuncScanState *state)

入力と出力

  • **入力**: TableFuncScanState *state （XMLTABLE 実行状態を保持する構造体）。
  • **出力**: bool （次の行が存在する場合は true、それ以外は false）。

前提条件と不変条件

  • USE_LIBXML が定義されている環境でのみ動作します。
  • GetXmlTableBuilderPrivateData を介して、state から XmlTableBuilderData が取得可能である必要があります。
  • libxml2 のエラーハンドラが xml_errorHandler に設定されます。

アルゴリズムまたは振る舞い

1. GetXmlTableBuilderPrivateData を呼び出し、内部コンテキストを取得します。 2. xmlSetStructuredErrorFunc を使用して、現在のエラーコンテキストを libxml2 に伝播させます。 3. xpathobj が未初期化の場合、xmlXPathCompiledEval を実行してXPathを評価します。評価失敗時またはエラー発生時は xml_ereport を呼び出し、エラーを報告します。 4. 評価結果が XPATH_NODESET 型である場合、row_count をインクリメントし、現在の行数がノードセットの総数以下であれば true を返します。 5. 上記以外の場合は false を返します。 6. USE_LIBXML が未定義の場合は NO_XML_SUPPORT() を呼び出し、false を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/adt/xml.c 内の XmlTableRoutine 構造体（.FetchRow メンバ）から参照されています。
  • **呼び出し先**:
  • GetXmlTableBuilderPrivateData
  • xmlSetStructuredErrorFunc
  • xmlXPathCompiledEval
  • xml_ereport

関連データ構造

  • TableFuncScanState: 実行ノードの状態を保持する構造体。
  • XmlTableBuilderData: XMLTABLE 実行に必要なXPathオブジェクトやノードセット情報を保持する内部構造体。

失敗モードと境界条件

  • xpathobj の生成に失敗した場合、ERRCODE_INVALID_ARGUMENT_FOR_XQUERY を伴うエラーが発生します。
  • libxml2 側でエラーが発生した場合、xmlerrcxt->err_occurred フラグにより検知され、PostgreSQLのエラーとして報告されます。

保守担当者向けメモ

  • 本関数は libxml2 のグローバルなエラーハンドラを一時的に書き換えるため、PG_TRY/PG_CATCH ブロック内での適切なエラーハンドリングが前提となります。
  • USE_LIBXML が定義されていない環境では機能しないため、ビルド構成に依存します。
