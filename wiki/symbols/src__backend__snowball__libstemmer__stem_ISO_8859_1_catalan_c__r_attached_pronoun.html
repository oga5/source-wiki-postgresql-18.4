r_attached_pronoun

定義

  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_catalan.c:1278

目的

カタルーニャ語のステミング処理において、単語末尾に付加された代名詞を特定し、削除する。

シグネチャまたは形状

static int r_attached_pronoun(struct SN_env * z)

入力と出力

  • 入力: SN_env 構造体（ステミング環境）。
  • 出力: 処理が成功した場合は 1、失敗または条件不一致の場合は 0、エラー発生時は負の値を返す。

前提条件と不変条件

  • z->c が単語の末尾を指していること。
  • r_R1 関数によって定義される領域（R1）内での削除操作が許可されていること。

アルゴリズムまたは振る舞い

1. 現在位置（z->c）を z->ket に設定する。 2. 文字コードのビット演算による事前チェックを行い、対象外の文字であれば終了する。 3. a_1 テーブル（39個の要素）を用いて、末尾の代名詞を find_among_b で後方検索する。 4. マッチした場合、現在位置を z->bra に設定する。 5. r_R1 を呼び出し、削除対象が R1 領域内にあるか検証する。 6. slice_del を呼び出し、該当する代名詞部分を文字列から削除する。

呼び出し元と呼び出し先

  • 呼び出し元: catalan_ISO_8859_1_stem（直接的または間接的）。
  • 呼び出し先: find_among_b, r_R1, slice_del。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_1: 代名詞の探索に使用される among 構造体の配列。

失敗モードと境界条件

  • z->c - 1 <= z->lb: 文字列の境界を超えた場合は 0 を返す。
  • find_among_b が一致する代名詞を見つけられない場合は 0 を返す。
  • r_R1 が領域外と判断した場合、または slice_del が失敗した場合は、その戻り値をそのまま返す。
