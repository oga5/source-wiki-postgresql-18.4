EOHPGetRODatum

定義

  • 種別: static inline function
  • 定義箇所: src/include/utils/expandeddatum.h:144

目的

展開済みオブジェクトのヘッダから、読み取り専用（R/O）の Datum 表現を抽出する。

シグネチャまたは形状

Datum EOHPGetRODatum(const struct ExpandedObjectHeader *eohptr)

入力と出力

  • **入力**: eohptr - 展開済みオブジェクトの共通ヘッダ構造体へのポインタ。
  • **出力**: eohptr が保持する読み取り専用ポインタを Datum 型に変換して返す。

前提条件と不変条件

  • eohptr は有効な ExpandedObjectHeader を指している必要がある。
  • 戻り値の Datum は、当該オブジェクトの読み取り専用表現として利用可能であること。

アルゴリズムまたは振る舞い

1. 引数として受け取った ExpandedObjectHeader 構造体の eoh_ro_ptr メンバにアクセスする。 2. PointerGetDatum マクロを使用して、そのポインタを Datum 型に変換して返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/expandeddatum.c:107
  • src/include/utils/expandedrecord.h:151
  • **呼び出し先**:
  • PointerGetDatum (マクロ)

関連データ構造

  • ExpandedObjectHeader: 展開済みオブジェクトの共通ヘッダ。本関数はこのヘッダから読み取り専用ポインタを取得する。

失敗モードと境界条件

  • eohptr が NULL の場合、未定義の動作（セグメンテーション違反など）を引き起こす可能性がある。

保守担当者向けメモ

  • 本関数はインライン関数であり、パフォーマンスを重視して設計されている。
  • 読み取り書き込み可能なポインタを取得する EOHPGetRWDatum と対になる関数である。
