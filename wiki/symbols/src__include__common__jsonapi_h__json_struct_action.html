json_struct_action

定義

  • 種別: typedef
  • 定義箇所: src/include/common/jsonapi.h:127

目的

JSON 解析プロセスにおいて、オブジェクトや配列の開始・終了といった構造的なイベントが発生した際に実行されるコールバック関数の型を定義します。

シグネチャまたは形状

typedef JsonParseErrorType (*json_struct_action) (void *state);

入力と出力

  • **入力**: void *state（解析中に状態を保持するためのユーザー定義データ構造へのポインタ）。
  • **出力**: JsonParseErrorType 型の値を返します。解析が正常に完了した場合は成功を示す値を返し、エラーが発生した場合は対応するエラーコードを返します。

前提条件と不変条件

  • コールバック関数は、JSON パーサーが構造的要素（オブジェクトまたは配列）を検出したタイミングで呼び出されます。
  • 戻り値として JSON_SEM_ACTION_FAILED が返された場合、エラーの詳細はコールバック関数内で報告済みであると見なされます。

アルゴリズムまたは振る舞い

パーサーが JSON 構造の境界（開始または終了）を認識すると、JsonSemAction 構造体に登録されたこの型の関数ポインタを介して、ユーザー定義のロジックが実行されます。これにより、JSON 構造のネストに応じたデータ抽出や検証が可能になります。

呼び出し元と呼び出し先

  • **呼び出し元**: src/common/jsonapi.c 内の JSON 解析ロジック（pg_parse_json 等）が、解析イベントに応じてこれらの関数ポインタを呼び出します。
  • **呼び出し先**: ユーザーが定義した、JsonSemAction 構造体に登録された具体的なコールバック関数が実行されます。

関連データ構造

  • JsonSemAction: 本型をメンバとして保持し、解析イベントとコールバック関数を紐付けるテーブル構造体。
  • JsonParseErrorType: 本型が戻り値として使用する列挙型。

失敗モードと境界条件

  • コールバック関数内でエラーが発生した場合、JsonParseErrorType を通じてパーサーに通知する必要があります。
  • state ポインタが適切に初期化されていない場合、コールバック実行時に未定義の動作を引き起こす可能性があります。

保守担当者向けメモ

  • JsonSemAction 構造体の object_start, object_end, array_start, array_end メンバとして使用されています。
  • 本型は src/common/jsonapi.c において、パーサーのイベントハンドリングの基盤として利用されています。
