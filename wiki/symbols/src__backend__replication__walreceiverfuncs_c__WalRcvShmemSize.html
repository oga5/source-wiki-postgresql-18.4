WalRcvShmemSize

定義

  • 種別: function
  • 定義箇所: src/backend/replication/walreceiverfuncs.c:43

目的

WALレシーバープロセスが共有メモリ上で状態管理を行うために必要なメモリサイズを算出します。

シグネチャまたは形状

Size WalRcvShmemSize(void)

入力と出力

  • 入力: なし
  • 出力: Size 型の共有メモリサイズ（バイト単位）

前提条件と不変条件

  • 算出されるサイズは WalRcvData 構造体のサイズに基づきます。
  • add_size を使用して安全に計算されます。

アルゴリズムまたは振る舞い

1. 初期サイズを 0 に設定します。 2. WalRcvData 構造体のサイズを add_size 関数を用いて加算します。 3. 計算された合計サイズを返します。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/backend/replication/walreceiverfuncs.c (内部での初期化処理)
  • src/backend/storage/ipc/ipci.c:141 (PostgreSQLの共有メモリ初期化プロセスの一部)
  • 呼び出し先:
  • add_size (PostgreSQLのメモリサイズ計算ユーティリティ)

関連データ構造

  • WalRcvData: WALレシーバーの状態を保持する共有メモリ上の構造体。

失敗モードと境界条件

  • メモリサイズの計算においてオーバーフローが発生する場合、add_size の仕様に従い最大値が返されます。

保守担当者向けメモ

  • WalRcvData 構造体の定義が変更された場合、本関数が返すサイズも自動的に追従しますが、共有メモリのレイアウト変更時には注意が必要です。
