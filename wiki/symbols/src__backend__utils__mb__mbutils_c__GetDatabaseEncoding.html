GetDatabaseEncoding

定義

  • 種別: function
  • 定義箇所: src/backend/utils/mb/mbutils.c:1386

目的

現在設定されているデータベースのエンコーディング識別子（整数値）を取得します。

シグネチャまたは形状

int GetDatabaseEncoding(void)

入力と出力

  • 入力: なし
  • 出力: データベースのエンコーディングを示す整数値（int）を返します。

前提条件と不変条件

  • 内部的に保持されている DatabaseEncoding グローバル変数が初期化されている必要があります。
  • バックエンドの起動プロセスにおいて、InitializeClientEncoding() 等を通じてエンコーディング設定が確定した後に呼び出されることが前提です。

アルゴリズムまたは振る舞い

1. グローバル変数 DatabaseEncoding が指す構造体から、encoding メンバの値を直接読み取ります。 2. 読み取った値を呼び出し元に返します。

呼び出し元と呼び出し先

  • 呼び出し元（直接的な証拠）:
  • contrib/dblink/dblink.c
  • contrib/pg_stat_statements/pg_stat_statements.c
  • contrib/pg_trgm/trgm_op.c
  • contrib/pgcrypto/pgp-pgsql.c
  • src/backend/access/transam/parallel.c
  • src/backend/catalog/namespace.c
  • src/backend/catalog/pg_collation.c
  • src/backend/commands/collationcmds.c
  • 呼び出し先:
  • 外部関数への呼び出しはありません。構造体メンバへの直接アクセスのみです。

関連データ構造

  • DatabaseEncoding: データベースのエンコーディング情報を保持する内部構造体（詳細は mbutils.c 内で管理）。

失敗モードと境界条件

  • データベースのエンコーディングが初期化される前に呼び出された場合、未定義の動作や不正なメモリ参照が発生する可能性があります。

保守担当者向けメモ

  • 本関数は非常に軽量なアクセサです。パフォーマンスへの影響は無視できます。
  • 戻り値は pg_wchar.h で定義されたエンコーディング定数と照合可能です。
