anycompatiblearray_send

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/pseudotypes.c:183

目的

anycompatiblearray 型のバイナリ送信（シリアライズ）処理を実装します。

シグネチャまたは形状

Datum anycompatiblearray_send(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS（関数マネージャ経由で渡される引数群）。
  • **出力**: Datum 型として、バイナリ形式に変換された配列データ（bytea）を返します。

前提条件と不変条件

  • 本関数は anycompatiblearray 型の送信関数として登録されることを前提としています。
  • 内部的に array_send を呼び出すため、array_send が要求する引数構造（fcinfo）が正しく渡されている必要があります。

アルゴリズムまたは振る舞い

1. 受け取った fcinfo をそのまま array_send 関数へ引き渡します。 2. array_send の戻り値をそのまま呼び出し元へ返却します。

呼び出し元と呼び出し先

  • **呼び出し先**: array_send（src/backend/utils/adt/arrayfuncs.c に定義されている標準的な配列送信関数）。
  • **呼び出し元**: 不明（PostgreSQL の関数マネージャ経由で呼び出されます）。

関連データ構造

  • Datum: PostgreSQL の汎用データ型。
  • PG_FUNCTION_ARGS: 関数呼び出し時の引数コンテキスト。

失敗モードと境界条件

  • array_send が失敗した場合（メモリ不足や不正な配列データなど）、そのエラーがそのまま伝播します。

保守担当者向けメモ

  • 本関数は単なるラッパーであり、実質的なロジックは array_send に依存しています。将来的に anycompatiblearray 固有のバイナリ形式が必要となった場合は、この委譲を解除して独自の実装を行う必要があります。
