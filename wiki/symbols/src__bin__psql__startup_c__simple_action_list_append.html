simple_action_list_append

定義

  • 種別: function
  • 定義箇所: src/bin/psql/startup.c:752

目的

SimpleActionList 構造体に新しいアクション（SimpleActionListCell）を追加し、リストの末尾に連結します。

シグネチャまたは形状

static void simple_action_list_append(SimpleActionList *list, enum _actions action, const char *val);

入力と出力

  • **入力**:
  • list: 操作対象の SimpleActionList 構造体へのポインタ。
  • action: 追加するアクションの種類を示す enum _actions 値。
  • val: アクションに関連付けられた文字列（オプション）。NULL の場合、セル内の値は NULL に設定されます。
  • **出力**: なし（void）。

前提条件と不変条件

  • list は有効な SimpleActionList 構造体を指している必要があります。
  • list->head および list->tail は、リストが空の場合は NULL で初期化されている必要があります。
  • pg_malloc_object および pg_strdup がメモリ割り当てに成功することを前提としています。

アルゴリズムまたは振る舞い

1. pg_malloc_object を使用して新しい SimpleActionListCell を割り当てます。 2. セルの next ポインタを NULL に初期化し、action を設定します。 3. val が提供されている場合、pg_strdup で複製して cell->val に格納します。提供されていない場合は NULL を格納します。 4. リストの tail が存在する場合、現在の末尾セルの next を新しいセルに更新します。 5. リストが空（tail が NULL）の場合、head を新しいセルに設定します。 6. 最後に、リストの tail ポインタを新しいセルに更新します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/bin/psql/startup.c 内の複数箇所（parse_psql_options 等の引数解析処理から呼び出されます）。
  • **呼び出し先**:
  • pg_malloc_object（メモリ割り当て）
  • pg_strdup（文字列複製）

関連データ構造

  • SimpleActionList: アクションリストのヘッドとテイルを保持する構造体。
  • SimpleActionListCell: リストの各要素を構成する構造体。
  • enum _actions: 実行可能なアクションの種類を定義する列挙型。

失敗モードと境界条件

  • メモリ不足が発生した場合、pg_malloc_object または pg_strdup 内部でエラーが発生し、プログラムが終了する可能性があります。
  • val が NULL の場合でも、適切に処理され、セルには NULL が格納されます。

保守担当者向けメモ

  • 本関数は static であり、src/bin/psql/startup.c 内でのみ使用されます。
  • SimpleActionList のメモリ管理は呼び出し元に依存せず、本関数内で pg_malloc_object を用いて確保されるため、リストの解放処理が別途必要になる点に注意してください。
