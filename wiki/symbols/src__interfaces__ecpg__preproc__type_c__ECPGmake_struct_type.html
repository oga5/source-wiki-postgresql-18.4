ECPGmake_struct_type

定義

  • **種別**: function
  • **定義箇所**: src/interfaces/ecpg/preproc/type.c:96

目的

ECPGプリプロセッサにおいて、構造体型の情報を保持する ECPGtype 構造体を生成・初期化するファクトリ関数です。

シグネチャまたは形状

struct ECPGtype *ECPGmake_struct_type(struct ECPGstruct_member *rm, enum ECPGttype type, const char *type_name, const char *struct_sizeof);

入力と出力

  • **入力**:
  • rm: 構造体のメンバーリスト (ECPGstruct_member 構造体へのポインタ)。
  • type: 型タグ (enum ECPGttype)。
  • type_name: 構造体の型名。
  • struct_sizeof: 構造体のサイズを表す文字列。
  • **出力**:
  • 構築された ECPGtype 構造体へのポインタを返します。

前提条件と不変条件

  • mm_strdup を使用して文字列を複製するため、呼び出し元は返された構造体の解放時にメモリ管理に注意する必要があります。
  • 内部で ECPGmake_simple_type および ECPGstruct_member_dup を呼び出します。

アルゴリズムまたは振る舞い

1. ECPGmake_simple_type を呼び出し、基本となる ECPGtype 構造体を生成します。 2. 引数として渡された type_name および struct_sizeof を mm_strdup で複製し、構造体のフィールドに格納します。 3. ECPGstruct_member_dup を使用して、メンバーリスト rm を複製し、構造体の u.members に割り当てます。 4. 構築した ECPGtype 構造体を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/interfaces/ecpg/preproc/type.c (複数箇所)
  • src/interfaces/ecpg/preproc/variable.c (複数箇所)
  • **呼び出し先**:
  • ECPGmake_simple_type
  • ECPGstruct_member_dup
  • mm_strdup

関連データ構造

  • ECPGtype: 型情報を保持する主要な構造体。
  • ECPGstruct_member: 構造体のメンバー定義を保持する構造体。

失敗モードと境界条件

  • メモリ割り当て失敗時の挙動は mm_strdup 等のメモリ管理ラッパーの仕様に依存します。

保守担当者向けメモ

  • 本関数は ECPGtype の構造体関連フィールドを初期化する役割を持ちます。メンバーリストの複製には ECPGstruct_member_dup が使用されているため、メンバー構造の変更時は同関数との整合性に注意してください。
