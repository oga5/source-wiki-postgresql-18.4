strnlen

定義

  • 種別: function
  • 定義箇所: src/port/strnlen.c:25

目的

標準ライブラリで strnlen が提供されていない環境において、指定された上限を超えない範囲で安全に文字列の長さを取得するフォールバック実装を提供します。

シグネチャまたは形状

size_t strnlen(const char *str, size_t maxlen);

入力と出力

  • **入力**:
  • str: 測定対象のヌル終端文字列へのポインタ。
  • maxlen: 走査する最大文字数。
  • **出力**:
  • ヌル終端文字が見つかった場合はその位置までの長さ（size_t）。
  • ヌル終端文字が見つかる前に maxlen に達した場合は maxlen。

前提条件と不変条件

  • str は有効なメモリ領域を指している必要があります。
  • 戻り値は常に maxlen 以下となります。

アルゴリズムまたは振る舞い

1. ポインタ p を str の先頭に初期化します。 2. maxlen をデクリメントしながら、*p がヌル文字（\0）でない間、ポインタ p をインクリメントします。 3. ループ終了後、現在のポインタ位置 p から開始位置 str を減算し、その差分を返します。

関連データ構造

  • なし。

失敗モードと境界条件

  • str が NULL の場合、未定義の動作となります。
  • maxlen が 0 の場合、ループは実行されず 0 が返されます。

保守担当者向けメモ

  • POSIX 標準の strnlen と同一の挙動を再現しています。
  • 現代のシステムでは標準ライブラリに含まれることが多いため、主に古い環境や特定のプラットフォームでのビルド互換性維持のために使用されます。
