BIG5toCNS

定義

  • 種別: function
  • 定義箇所: src/backend/utils/mb/conversion_procs/euc_tw_and_big5/big5.c:291

目的

BIG5エンコーディングの文字コードを、CNS 11643-1992（Mule内部コード）へ変換します。

シグネチャまたは形状

unsigned short BIG5toCNS(unsigned short big5, unsigned char *lc)

入力と出力

  • **入力**:
  • big5: 変換対象のBIG5コード（unsigned short）。
  • lc: 変換後のCNSプレーン情報を格納するポインタ（unsigned char *）。
  • **出力**:
  • 変換後のCNSコード（unsigned short）。変換不能な場合は '?' を返します。
  • lc には対応するCNSプレーン（LC_CNS11643_1〜4）が設定されます。

前提条件と不変条件

  • 変換には静的配列 b1c4、b2c3 および BinarySearchRange 関数による範囲変換テーブルを使用します。
  • 戻り値のCNSコードには、成功時に 0x8080 がビット論理和で付加されます。

アルゴリズムまたは振る舞い

1. **レベル1判定 (big5 < 0xc940U)**:

  • b1c4 テーブルを走査し、一致すれば LC_CNS11643_4 を設定して返します。
  • 一致しない場合、BinarySearchRange を用いて big5Level1ToCnsPlane1 を検索し、成功すれば LC_CNS11643_1 を設定します。

    • 2. **例外処理**:

  • big5 == 0xc94aU の場合、LC_CNS11643_1 を設定し、固定値 0x4442 を返します。

    • 3. **レベル2判定**:

  • b2c3 テーブルを走査し、一致すれば LC_CNS11643_3 を設定して返します。
  • 一致しない場合、BinarySearchRange を用いて big5Level2ToCnsPlane2 を検索し、成功すれば LC_CNS11643_2 を設定します。

    • 4. **終了処理**:

  • いずれの検索も失敗した場合、lc に 0 を設定し '?' を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/mb/conversion_procs/euc_tw_and_big5/euc_tw_and_big5.c 内の変換処理から呼び出されます。
  • **呼び出し先**:
  • BinarySearchRange 関数。

関連データ構造

  • b1c4, b2c3: 例外的なマッピングを保持する静的配列。
  • big5Level1ToCnsPlane1, big5Level2ToCnsPlane2: 範囲変換用のテーブル。

失敗モードと境界条件

  • マッピングが見つからない場合、lc に 0 が書き込まれ、戻り値として '?' が返されます。

保守担当者向けメモ

  • 本関数はフロントエンドとバックエンドの両方で利用可能です。
  • 新規マッピングの追加は、まず b1c4 や b2c3 配列を確認し、範囲外であれば BinarySearchRange 用のテーブルを調整してください。
