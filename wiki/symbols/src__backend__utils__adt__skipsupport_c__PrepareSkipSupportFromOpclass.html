PrepareSkipSupportFromOpclass

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/skipsupport.c:29

目的

B-Treeインデックスのスキップスキャン（Skip Scan）において、次の探索値を計算するためのサポート構造体（SkipSupportData）を初期化します。

シグネチャまたは形状

SkipSupport PrepareSkipSupportFromOpclass(Oid opfamily, Oid opcintype, bool reverse)

入力と出力

  • **入力**:
  • opfamily (Oid): 対象の演算子ファミリー。
  • opcintype (Oid): 対象のデータ型。
  • reverse (bool): 降順スキャンを行うか否か。
  • **出力**:
  • 初期化済みの SkipSupport 構造体へのポインタ。サポート関数が見つからない場合は NULL を返します。

前提条件と不変条件

  • 指定された opfamily と opcintype に対して BTSKIPSUPPORT_PROC が定義されている必要があります。
  • 呼び出し元は、返却された SkipSupport 構造体のメモリ管理（pfree）に責任を持ちます。

アルゴリズムまたは振る舞い

1. get_opfamily_proc を使用し、演算子ファミリーから BTSKIPSUPPORT_PROC を検索します。 2. 関数が見つからない場合は NULL を返します。 3. palloc で SkipSupportData を確保し、OidFunctionCall1 を介して演算子クラス側の初期化関数を呼び出します。 4. reverse が true の場合、降順スキャン用に以下の入れ替えを行います。

  • low_elem と high_elem をスワップ。
  • increment 関数と decrement 関数をスワップ。

    • 5. 初期化済みの構造体を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/access/nbtree/nbtpreprocesskeys.c:1979
  • **呼び出し先**:
  • get_opfamily_proc (システムカタログ検索)
  • OidFunctionCall1 (関数呼び出し)
  • palloc (メモリ確保)

関連データ構造

  • SkipSupportData: スキップスキャンに必要な境界値と増減関数を保持する構造体。
  • BTSKIPSUPPORT_PROC: B-Tree演算子クラスが提供するプロシージャID。

失敗モードと境界条件

  • BTSKIPSUPPORT_PROC が定義されていない演算子ファミリーに対しては NULL を返します。
  • メモリ不足時には palloc がエラーを発生させます。

保守担当者向けメモ

  • 本関数はスキップスキャン最適化の基盤です。データ型固有の増減ロジックは演算子クラス側の実装に依存します。
  • reverse フラグによる入れ替えは、スキャンエンジンが常に一貫したインターフェースで「次の値」を計算できるようにするための重要な調整ステップです。
