r_mark_regions

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_french.c:541

目的

フランス語のステミング処理において、単語内の領域（R1, R2, RV）を特定し、SN_env 構造体の整数配列 I にその境界位置を格納する。

シグネチャまたは形状

static int r_mark_regions(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z（ステミング環境）。現在の文字列バッファ p とカーソル位置 c を含む。
  • **出力**: 常に 1 を返す。副作用として z->I[0] (RV), z->I[1] (R2), z->I[2] (R1) が更新される。

前提条件と不変条件

  • z は有効な SN_env 構造体を指していること。
  • g_v（母音グループ）が定義されていること。
  • a_0（特定の接尾辞配列）が利用可能であること。

アルゴリズムまたは振る舞い

1. **初期化**: I[0], I[1], I[2] を文字列の末尾 z->l に初期化する。 2. **RV領域の特定**:

  • 特定の条件（母音2連続、または特定の接尾辞マッチ、または子音後の母音）に基づき、RVの開始位置を探索し z->I[2] に設定する。

    • 3. **R1/R2領域の特定**:

  • 母音と子音の遷移を追跡し、最初の母音の後の子音の直後を R1 (z->I[1])、さらにその後の母音の後の子音の直後を R2 (z->I[0]) として設定する。

    • 4. 処理中、カーソル位置 z->c は一時的に移動するが、終了時には元の位置に復元される。

呼び出し元と呼び出し先

  • **呼び出し元**: french_UTF_8_stem（直接の言及はないが、モジュール内の制御フローより推測される）。
  • **呼び出し先**: in_grouping_U, out_grouping_U, skip_utf8, find_among。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_0: among 構造体の配列（RV領域判定に使用）。
  • g_v: 母音を定義するグループ。

失敗モードと境界条件

  • 文字列が短い場合や、母音・子音のパターンが条件に合致しない場合、領域は文字列末尾に留まる。
  • skip_utf8 やグループ判定関数が失敗した場合、ラベル（lab0〜lab4）へジャンプし、部分的な領域設定をスキップして終了する。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードである。ロジックの変更が必要な場合は、対応する .sbl ファイルを修正すること。
  • I[0]〜I[2] のインデックスと領域の対応関係（RV, R1, R2）は、フランス語のステミングルールに直結しているため、変更時は注意が必要。
