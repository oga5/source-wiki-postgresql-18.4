intset_memory_usage

定義

  • 種別: function
  • 定義箇所: src/backend/lib/integerset.c:357

目的

指定された IntegerSet が現在消費しているメモリ量をバイト単位で取得します。

シグネチャまたは形状

uint64 intset_memory_usage(IntegerSet *intset)

入力と出力

  • **入力**: IntegerSet *intset - 対象となる整数集合の構造体ポインタ。
  • **出力**: uint64 - 構造体が保持する mem_used フィールドの値（バイト数）。

前提条件と不変条件

  • intset は有効な IntegerSet 構造体を指している必要があります。
  • IntegerSet 構造体内の mem_used フィールドは、集合の操作に伴い適切に更新されていることが前提となります。

アルゴリズムまたは振る舞い

IntegerSet 構造体内に保持されている mem_used メンバ変数の値をそのまま返します。計算処理は行いません。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/test/modules/test_integerset/test_integerset.c: テストモジュール内でメモリ使用量の検証および出力に使用されています。
  • **呼び出し先**:
  • 本関数は他の関数を呼び出しません。

関連データ構造

  • IntegerSet: 集合のルート構造体。本関数はこの構造体の mem_used フィールドを参照します。

失敗モードと境界条件

  • intset が NULL の場合、セグメンテーション違反が発生する可能性があります。

保守担当者向けメモ

  • 本関数は単なるゲッターであり、IntegerSet の内部状態（mem_used）が正確に維持されていることに依存しています。将来的にメモリ管理ロジックを変更する場合は、mem_used の更新箇所が正しく同期されているか確認してください。
