is_visible_fxid

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/xid8funcs.c:186

目的

指定された64ビットトランザクションID（FullTransactionId）が、提供されたスナップショット（pg_snapshot）において可視であるかどうかを判定します。

シグネチャまたは形状

static bool is_visible_fxid(FullTransactionId value, const pg_snapshot *snap)

入力と出力

  • **入力**:
  • value: 判定対象の FullTransactionId。
  • snap: 判定基準となる pg_snapshot 構造体へのポインタ。
  • **出力**:
  • bool: 可視であれば true、不可視（進行中または将来のトランザクション）であれば false を返します。

前提条件と不変条件

  • snap は有効な pg_snapshot 構造体を指している必要があります。
  • snap->xmin および snap->xmax は、トランザクションの可視性判定における境界として機能します。

アルゴリズムまたは振る舞い

1. **下限判定**: value が snap->xmin より前であれば、既に完了しており可視とみなして true を返します。 2. **上限判定**: value が snap->xmax 以上であれば、将来のトランザクションであり不可視とみなして false を返します。 3. **進行中判定**: xmin と xmax の間にある場合、スナップショット内の進行中トランザクション配列（xip）を走査します。

  • USE_BSEARCH_IF_NXIP_GREATER が定義され、かつ配列サイズが閾値を超える場合は bsearch を使用します。
  • それ以外の場合は線形探索を行います。
  • 配列内に value が存在すれば進行中（不可視）として false を返し、存在しなければ true を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/adt/xid8funcs.c 内の関数から呼び出されています（例: 行560）。
  • **呼び出し先**:
  • FullTransactionIdPrecedes, FullTransactionIdEquals: 比較演算。
  • bsearch (条件付き): 配列探索。
  • cmp_fxid: bsearch 用の比較関数。

関連データ構造

  • FullTransactionId: 64ビットのトランザクションID。
  • pg_snapshot: トランザクションの可視性情報を保持する構造体。

失敗モードと境界条件

  • snap->xip 配列が空の場合、範囲内にあるトランザクションはすべて可視と判定されます。
  • value が xmin と xmax の境界値と一致する場合の挙動は、FullTransactionIdPrecedes の定義に依存します。

保守担当者向けメモ

  • 本関数はサブトランザクションを考慮していません。
  • USE_BSEARCH_IF_NXIP_GREATER の閾値設定は、スナップショット内の進行中トランザクション数に応じたパフォーマンス最適化を目的としています。
