r_mark_regions

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_catalan.c:1183

目的

単語内のステミング処理における境界領域（R1およびR2）を特定し、SN_env 構造体の整数配列 I にそのオフセットを記録する。

シグネチャまたは形状

static int r_mark_regions(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z（ステミング環境の状態を保持する構造体）
  • **出力**: 常に 1 を返す。
  • **副作用**: z->I[1]（R1の開始位置）および z->I[0]（R2の開始位置）が更新される。

前提条件と不変条件

  • z が有効な SN_env 構造体を指していること。
  • g_v（母音グループ）が定義されていること。

アルゴリズムまたは振る舞い

1. z->I[1]（R1）と z->I[0]（R2）を初期値として単語の末尾（z->l）に設定する。 2. 現在のカーソル位置 z->c を保存する。 3. 母音と子音の連続パターンを検索する：

  • 母音 (out_grouping) の後に子音 (in_grouping) が続く箇所を検索し、その直後を R1 の開始位置 (z->I[1]) とする。
  • さらにその後に続く母音と子音のペアを検索し、その直後を R2 の開始位置 (z->I[0]) とする。

    • 4. 検索に失敗した場合は、保存したカーソル位置に復帰する。

呼び出し元と呼び出し先

  • **呼び出し元**: catalan_ISO_8859_1_stem から呼び出される（直接的な呼び出し関係はソースコードの制御フローに基づく）。
  • **呼び出し先**: out_grouping, in_grouping（Snowballランタイムのユーティリティ関数）。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。I 配列に領域境界を格納する。

失敗モードと境界条件

  • 単語内に母音・子音のパターンが見つからない場合、R1 および R2 の位置は単語の末尾に留まる。
  • out_grouping または in_grouping が負の値を返した場合、検索は中断され、カーソル位置は初期状態に復元される。

保守担当者向けメモ

  • 本関数はSnowballコンパイラによって自動生成されたコードである。ロジックの変更が必要な場合は、直接編集せず、対応する .sbl ファイルを修正すること。
  • g_v は ISO-8859-1 エンコーディングにおける母音セットを指す。
