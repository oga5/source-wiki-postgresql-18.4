r_mark_regions

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_swedish.c:150

目的

スウェーデン語のステミング処理において、語幹抽出の基準となる領域（R1, R2）の境界位置を計算し、環境構造体 SN_env の整数配列 I に記録する。

シグネチャまたは形状

static int r_mark_regions(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z （ステミング実行環境）
  • **出力**: 成功時に 1、失敗時に 0 を返す。
  • **副作用**: z->I[0] および z->I[1] に計算された境界オフセットが格納される。

前提条件と不変条件

  • z は有効な SN_env 構造体へのポインタであること。
  • g_v（母音グループ）が定義されていること。

アルゴリズムまたは振る舞い

1. z->I[1] を文字列の末尾位置で初期化する。 2. 文字列の先頭から3文字スキップした位置を z->I[0] に設定する。 3. 母音（g_v）が出現するまで走査し、その直後の位置を z->I[1] に更新する。 4. z->I[1] が z->I[0] より小さい場合、z->I[1] を z->I[0] に引き上げる（領域の最小値を保証）。

呼び出し元と呼び出し先

  • **呼び出し元**: swedish_UTF_8_stem（src/backend/snowball/libstemmer/stem_UTF_8_swedish.c 内）。
  • **呼び出し先**: skip_utf8, out_grouping_U, in_grouping_U（いずれも Snowball ランタイムユーティリティ）。

関連データ構造

  • struct SN_env: ステミングの状態を保持する構造体。I 配列（整数領域）および c（現在のカーソル位置）、l（文字列長）を使用する。

失敗モードと境界条件

  • skip_utf8 が負の値を返した場合、処理を中断し 0 を返す。
  • out_grouping_U または in_grouping_U がマッチに失敗した場合、処理を中断し 0 を返す。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによって自動生成されている。ロジックの変更が必要な場合は、対応する .sbl ファイルを修正し、再生成すること。
  • z->I[0] は R1 領域の開始点、z->I[1] は R2 領域の開始点として後続の接尾辞処理で使用される。
