deccmp

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/compatlib/informix.c:166

目的

2つの decimal 型変数を比較し、その結果を返します。Informix 互換の算術演算インターフェースの一部です。

シグネチャまたは形状

int deccmp(decimal *arg1, decimal *arg2)

入力と出力

  • **入力**: 比較対象となる2つの decimal 構造体へのポインタ (arg1, arg2)。
  • **出力**: 比較結果を示す整数値。

前提条件と不変条件

  • 入力される decimal ポインタは有効なメモリ領域を指している必要があります。
  • 内部的に PGTYPESnumeric_cmp を利用するため、decimal 型が numeric 型として正しく解釈可能である必要があります。

アルゴリズムまたは振る舞い

1. 内部ラッパー関数 deccall2 を呼び出します。 2. deccall2 を介して、PostgreSQL の pgtypes ライブラリ関数である PGTYPESnumeric_cmp に処理を委譲します。 3. PGTYPESnumeric_cmp の実行結果をそのまま呼び出し元へ返却します。

呼び出し元と呼び出し先

  • **呼び出し先**: deccall2
  • **呼び出し元**:
  • src/interfaces/ecpg/test/expected/compat_informix-dec_test.c (テストコード内での利用が確認されています)
  • その他、Informix 互換 API を利用する ECPG 生成コードから呼び出される可能性があります。

関連データ構造

  • decimal: Informix 互換の数値型構造体。

失敗モードと境界条件

  • deccall2 経由で呼び出される PGTYPESnumeric_cmp の挙動に依存します。
  • 入力ポインタが NULL の場合の動作については、本ソースコード単体では明示されていません。

保守担当者向けメモ

  • 本関数は deccall2 を利用した単純なラッパーです。ロジックの変更が必要な場合は、deccall2 の実装または PGTYPESnumeric_cmp の仕様を確認してください。
