set_spins_per_delay

定義

  • 種別: function
  • 定義箇所: src/backend/storage/lmgr/s_lock.c:206

目的

プロセスローカルなスピンロックの待機閾値（spins_per_delay）を、共有された値で更新します。

シグネチャまたは形状

void set_spins_per_delay(int shared_spins_per_delay)

入力と出力

  • **入力**: shared_spins_per_delay (int型) - 新たに設定するスピン回数の閾値。
  • **出力**: なし（副作用としてグローバル変数 spins_per_delay を更新）。

前提条件と不変条件

  • 本関数は、スピンロックの待機戦略を調整する際に呼び出されます。
  • spins_per_delay は、マルチプロセッサ環境におけるスピンロックのパフォーマンスを最適化するための重要なパラメータです。

アルゴリズムまたは振る舞い

1. 引数として受け取った shared_spins_per_delay を、プロセスローカルな変数 spins_per_delay に代入します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/storage/lmgr/proc.c:434
  • src/backend/storage/lmgr/proc.c:642
  • **呼び出し先**:
  • 特になし。

関連データ構造

  • spins_per_delay: プロセスローカルなスピン回数の閾値を保持する変数。

失敗モードと境界条件

  • 供給されたソースコード上では、入力値の妥当性チェックは行われていません。

保守担当者向けメモ

  • 本関数は src/include/storage/s_lock.h で外部公開されています。
  • spins_per_delay の値が不適切である場合、マルチプロセッサ環境において不必要なカーネル呼び出しが発生し、システム全体の性能が低下する可能性があります。
