pgxml_result_to_text

定義

  • 種別: function
  • 定義箇所: contrib/xml2/xpath.c:435

目的

libxml2 の xmlXPathObjectPtr 型の結果を、PostgreSQL の text 型へ変換する。

シグネチャまたは形状

text *pgxml_result_to_text(xmlXPathObjectPtr res, xmlChar *toptag, xmlChar *septag, xmlChar *plainsep)

入力と出力

  • **入力**:
  • res: 変換対象の xmlXPathObjectPtr。
  • toptag, septag, plainsep: ノードセット変換時に使用されるタグや区切り文字の設定。
  • **出力**: 変換後の text * 型データ。res が NULL の場合は NULL を返す。

前提条件と不変条件

  • res が XPATH_NODESET の場合、内部で pgxmlNodeSetToText を呼び出す。
  • res が XPATH_STRING の場合、文字列を複製して変換する。
  • 上記以外の型が渡された場合、NOTICE レベルのログを出力し、<unsupported/> という文字列を返す。

アルゴリズムまたは振る舞い

1. res が NULL か確認し、該当すれば即座に NULL を返す。 2. res->type に基づき分岐する。

  • XPATH_NODESET: pgxmlNodeSetToText を用いてノードセットをテキスト化する。
  • XPATH_STRING: xmlStrdup を用いて文字列を複製する。
  • その他: elog(NOTICE, ...) を発行し、デフォルト値として <unsupported/> を割り当てる。

    • 3. cstring_to_text を使用して、取得した xmlChar * 文字列を PostgreSQL の text 型に変換する。 4. xmlFree を呼び出し、一時的に確保した xmlChar * バッファを解放する。 5. 変換後の text * を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/xml2/xpath.c 内の各関数（xpath_nodeset, xpath_list, xpath_string 等の評価結果処理箇所）。
  • **呼び出し先**:
  • pgxmlNodeSetToText
  • xmlStrdup
  • cstring_to_text
  • xmlFree
  • elog

関連データ構造

  • xmlXPathObjectPtr (libxml2)
  • text (PostgreSQL)

失敗モードと境界条件

  • res が NULL の場合、安全に NULL を返す。
  • 未サポートの XPath 型が渡された場合、エラーで停止せず NOTICE ログを出力し、<unsupported/> を返すことで処理を継続する。

保守担当者向けメモ

  • 本関数は libxml2 が確保したメモリ（xmlChar *）を xmlFree で解放する責任を持つ。
  • cstring_to_text は PostgreSQL のメモリコンテキストで新しい text オブジェクトを割り当てるため、xmlFree 後も結果は有効である。
