DatumGetMacaddrP

定義

  • 種別: static inline 関数
  • 定義箇所: src/include/utils/inet.h:146

目的

PostgreSQLの内部表現である Datum 型から、MACアドレス構造体（macaddr *）へのポインタを取得します。

シグネチャまたは形状

static inline macaddr *DatumGetMacaddrP(Datum X)

入力と出力

  • **入力**: Datum X （MACアドレスデータを含む Datum）
  • **出力**: macaddr * （macaddr 構造体へのポインタ）

前提条件と不変条件

  • 入力 Datum X は、有効な macaddr 構造体へのポインタを保持している必要があります。
  • 本関数は DatumGetPointer マクロを利用してキャストを行うため、型安全性の検証は呼び出し側の責任となります。

アルゴリズムまたは振る舞い

1. DatumGetPointer(X) を呼び出し、Datum を汎用ポインタ型に変換します。 2. 変換されたポインタを macaddr * 型にキャストして返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/mac.c: macaddr 型の操作関数内で利用。
  • src/backend/utils/adt/network.c: macaddr 型を扱う関数内で利用。
  • src/include/utils/inet.h: マクロ PG_GETARG_MACADDR_P(n) の定義内で利用。
  • **呼び出し先**:
  • DatumGetPointer（PostgreSQLのコアマクロ）

関連データ構造

  • macaddr: MACアドレスを格納する固定長構造体。

失敗モードと境界条件

  • Datum が NULL または不正なポインタを指している場合、呼び出し先でセグメンテーション違反が発生する可能性があります。本関数自体はポインタの妥当性を検証しません。

保守担当者向けメモ

  • 本関数は macaddr 型専用の単純なキャスト関数です。inet 型のような可変長データ（varlena）の展開（detoast）は行いません。
  • macaddr は固定長データ型であるため、inet 型のような複雑なメモリ管理は不要です。
