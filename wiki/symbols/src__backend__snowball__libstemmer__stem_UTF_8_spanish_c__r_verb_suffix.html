r_verb_suffix

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_spanish.c:905

目的

スペイン語のステミング処理において、単語末尾の動詞活用語尾を特定し、条件に応じて削除する。

シグネチャまたは形状

static int r_verb_suffix(struct SN_env * z)

入力と出力

  • **入力**: struct SN_env * z （Snowballのランタイム環境構造体）
  • **出力**: int （処理が成功した場合は1、失敗した場合は0、エラー時は負の値を返す）

前提条件と不変条件

  • z->I[2] に格納された領域境界（R2領域の開始位置など）が設定されていること。
  • z->c （現在のカーソル位置）が z->I[2] 以上であること。

アルゴリズムまたは振る舞い

1. **領域チェック**: 現在位置が z->I[2] 未満であれば即座に終了する。 2. **後方検索**: a_8 配列（96個のルール）を用いて、単語末尾から動詞接尾辞を find_among_b で検索する。 3. **条件分岐**:

  • **ケース1**: 特定の接尾辞（'u'で終わる等）の場合、直前の文字が 'g' であるかを確認し、条件を満たせば slice_del で削除する。
  • **ケース2**: その他の接尾辞の場合、そのまま slice_del で削除する。

    • 4. **終了**: 削除が成功した場合は1を返し、失敗した場合は0を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: spanish_UTF_8_stem 関数（直接の呼び出し元はソースコードの制御フローに基づく）。
  • **呼び出し先**: find_among_b, slice_del （いずれも header.h で定義された共通ユーティリティ）。
  • **注記**: 他の言語（カタラン語、フランス語、イタリア語、ポルトガル語）のステミングファイルからも同名の関数が参照されているが、これらは各言語のコンテキストで定義された別個の関数である可能性がある。

関連データ構造

  • struct SN_env: ステミング処理の環境状態を保持する構造体。
  • a_8: 動詞接尾辞のパターンを定義した among 構造体の配列。

失敗モードと境界条件

  • z->c < z->I[2]: 処理対象領域外のため0を返す。
  • find_among_b が一致を見つけられない場合: 0を返す。
  • slice_del が負の値を返した場合: そのエラーコードをそのまま伝播させる。

  • a_8 配列の定義や検索ロジックは、スペイン語の文法ルールに基づいている。
