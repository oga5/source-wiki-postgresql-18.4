init_ttinfo

定義

  • 種別: function
  • 定義箇所: src/timezone/localtime.c:107

目的

タイムゾーン情報構造体 ttinfo の各フィールドを初期化し、特定の期間におけるオフセットや夏時間フラグを設定する。

シグネチャまたは形状

static void init_ttinfo(struct ttinfo *s, int32 utoff, bool isdst, int desigidx)

入力と出力

  • **入力**:
  • s: 初期化対象の ttinfo 構造体へのポインタ。
  • utoff: UTC からのオフセット（秒単位）。
  • isdst: 夏時間（DST）であるかを示すブール値。
  • desigidx: タイムゾーン略称文字列へのインデックス。
  • **出力**: なし（s が指す構造体を直接更新）。

前提条件と不変条件

  • s は有効な struct ttinfo へのポインタであること。
  • 初期化後、tt_ttisstd および tt_ttisut は常に false に設定される。

アルゴリズムまたは振る舞い

1. 引数 utoff を s->tt_utoff に代入する。 2. 引数 isdst を s->tt_isdst に代入する。 3. 引数 desigidx を s->tt_desigidx に代入する。 4. s->tt_ttisstd を false に設定する。 5. s->tt_ttisut を false に設定する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/timezone/localtime.c 内の複数箇所（POSIX タイムゾーン文字列の解析処理等において、sp->ttis 配列の要素を初期化するために使用）。
  • **呼び出し先**:
  • 本関数は他の関数を呼び出さない。

関連データ構造

  • struct ttinfo: タイムゾーンの遷移情報を保持する構造体（pgtz.h で定義）。

失敗モードと境界条件

  • ポインタ s が NULL の場合、セグメンテーション違反が発生する可能性がある。

保守担当者向けメモ

  • 本関数は tzparse 等の POSIX タイムゾーン解析ロジックから頻繁に呼び出される。
  • tt_ttisstd および tt_ttisut を常に false に固定する仕様であるため、これらを動的に変更する必要がある場合は本関数の拡張が必要となる。
