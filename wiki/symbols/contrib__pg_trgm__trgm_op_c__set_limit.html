set_limit

定義

  • 種別: function
  • 定義箇所: contrib/pg_trgm/trgm_op.c:233

目的

pg_trgm.similarity_threshold GUC（設定変数）を更新し、現在の設定値を返します。

シグネチャまたは形状

Datum set_limit(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_GETARG_FLOAT4(0) により、新しい類似度閾値（float4）を受け取ります。
  • **出力**: 設定後の similarity_threshold の値を float4 として返します。

前提条件と不変条件

  • 入力値は float4 型である必要があります。
  • SetConfigOption を介して pg_trgm.similarity_threshold を PGC_USERSET（ユーザー設定可能）かつ PGC_S_SESSION（セッションスコープ）で更新します。

アルゴリズムまたは振る舞い

1. 入力された float4 値を getTypeOutputInfo および OidOutputFunctionCall を用いて文字列形式に変換します。 2. SetConfigOption を呼び出し、pg_trgm.similarity_threshold を指定された値で更新します。 3. 現在の similarity_threshold の値を PG_RETURN_FLOAT4 で返します。

呼び出し元と呼び出し先

  • **呼び出し元**: SQL インターフェース経由で呼び出されます（PG_FUNCTION_INFO_V1(set_limit) による登録あり）。
  • **呼び出し先**: getTypeOutputInfo, OidOutputFunctionCall, SetConfigOption などの PostgreSQL 内部 API を呼び出します。

関連データ構造

  • similarity_threshold: 本関数が更新対象とする GUC 変数。

失敗モードと境界条件

  • 入力値が float4 として解釈できない場合、PostgreSQL の型変換エラーが発生します。
  • SetConfigOption が失敗した場合、例外が発生し処理が中断されます。

保守担当者向けメモ

  • 本関数は SQL からの set_limit() 関数呼び出しに対応しています。
  • similarity_threshold の値はモジュール内の類似度計算ロジックで参照されるため、設定変更は即座に後続の類似度判定に影響します。
