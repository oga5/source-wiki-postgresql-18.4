gbt_float8lt

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_float8.c:49

目的

GiSTインデックスにおいて、2つの float8 値を比較し、左辺が右辺より小さいかどうかを判定します。

シグネチャまたは形状

bool gbt_float8lt(const void *a, const void *b, FmgrInfo *flinfo)

入力と出力

  • **入力**:
  • a: 比較対象の float8 値へのポインタ。
  • b: 比較対象の float8 値へのポインタ。
  • flinfo: 関数管理情報（本関数内では未使用）。
  • **出力**:
  • bool: *a < *b が真であれば true、それ以外は false を返します。

前提条件と不変条件

  • ポインタ a および b は、有効な float8 型の値を指している必要があります。

アルゴリズムまたは振る舞い

1. ポインタ a および b を float8 型としてキャストします。 2. C言語の標準的な小なり演算子 (<) を用いて値を比較します。 3. 比較結果をブール値として返します。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/btree_gist/btree_float8.c 内の tinfo 構造体定義（95行目）において、比較関数として参照されています。
  • **呼び出し先**: 直接的な関数呼び出しはありません。

関連データ構造

  • float8: PostgreSQLの倍精度浮動小数点数型。

失敗モードと境界条件

  • 入力ポインタが NULL の場合、セグメンテーション違反が発生する可能性があります。
  • float8 の値が NaN（非数）である場合の比較動作は、標準的な浮動小数点数の比較規則に従います。

保守担当者向けメモ

  • 本関数は btree_utils_num が提供する汎用的なGiSTインデックス構築ロジックから呼び出される比較用ラッパーです。
  • tinfo 構造体を通じて利用されるため、シグネチャの変更には注意が必要です。
