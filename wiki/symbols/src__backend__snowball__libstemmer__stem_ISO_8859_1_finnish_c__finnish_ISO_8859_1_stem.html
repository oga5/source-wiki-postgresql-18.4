finnish_ISO_8859_1_stem

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_finnish.c:658

目的

ISO-8859-1エンコーディングのフィンランド語テキストに対し、語尾削除ルールを適用して正規化された語幹を生成する。

シグネチャまたは形状

int finnish_ISO_8859_1_stem(struct SN_env * z)

入力と出力

  • 入力: SN_env 構造体（ステミング対象の文字列および処理環境を含む）。
  • 出力: 成功時に 1 を返し、エラー発生時に負の値を返す。

前提条件と不変条件

  • z は有効な SN_env 構造体へのポインタであること。
  • 入力テキストは ISO-8859-1 エンコーディングであること。
  • 内部状態 z->I[2] は、格語尾の削除状況を追跡するために使用される。

アルゴリズムまたは振る舞い

1. **領域定義**: r_mark_regions を呼び出し、単語内の処理対象領域（R1, R2）を特定する。 2. **順次適用**: 以下の順序でルールを適用し、各ステップで文字列を置換・削除する。

  • r_particle_etc（接辞）
  • r_possessive（所有接尾辞）
  • r_case_ending（格語尾）
  • r_other_endings（その他の語尾）

    • 3. **条件分岐**: 格語尾が削除されたか（z->I[2] が真か）に応じて、r_i_plural または r_t_plural を選択的に実行する。 4. **最終処理**: r_tidy を実行し、二重母音や不要な文字を調整して語幹を確定させる。

呼び出し元と呼び出し先

  • 呼び出し元: 提供されたソース情報からは特定不可。
  • 呼び出し先: r_mark_regions, r_particle_etc, r_possessive, r_case_ending, r_other_endings, r_i_plural, r_t_plural, r_tidy。

関連データ構造

  • SN_env: ステミング処理の環境を保持する構造体。
  • among 構造体配列（a_0 ～ a_9）: 語尾パターン照合に使用される。

失敗モードと境界条件

  • 各内部関数（r_...）が負の値を返した場合、その値をそのまま呼び出し元へ伝播させる。
  • r_mark_regions が失敗した場合、処理を中断してエラーを返す。

保守担当者向けメモ

  • 本関数はSnowballコンパイラによる自動生成コードである。ロジックの変更が必要な場合は、直接編集せず元のSnowballスクリプト（.sbl）を修正すること。
  • UTF-8環境で使用する場合は、本関数ではなく stem_UTF_8_finnish.c を使用すること。
