degtorad

定義

  • 種別: function
  • 定義箇所: contrib/earthdistance/earthdistance.c:33

目的

度（degrees）単位の数値をラジアン（radians）単位に変換する。

シグネチャまたは形状

static double degtorad(double degrees)

入力と出力

  • 入力: double degrees（度単位の角度）
  • 出力: double（ラジアン単位に変換された角度）

前提条件と不変条件

  • TWO_PI 定数が定義されている環境で動作する。
  • 入力値は浮動小数点数として扱われる。

アルゴリズムまたは振る舞い

入力された degrees を 360.0 で除算し、それに TWO_PI を乗算することでラジアン値を算出する。

呼び出し元と呼び出し先

  • 呼び出し元: contrib/earthdistance/earthdistance.c 内の geo_distance_internal 関数から、Point 構造体の経度（x）および緯度（y）を変換するために呼び出される。
  • 呼び出し先: なし（算術演算のみを実行）。

関連データ構造

  • なし

失敗モードと境界条件

  • 浮動小数点演算の精度に依存する。
  • 非常に大きな値や極端な値に対するオーバーフロー・アンダーフローの明示的なチェックは含まれていない。

保守担当者向けメモ

  • 本関数は static であり、このファイル内でのみ使用される。
  • 単位変換のロジックは単純な比例計算であるため、将来的に定数定義が変更された場合の影響範囲に注意すること。
