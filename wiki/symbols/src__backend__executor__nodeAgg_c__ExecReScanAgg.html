ExecReScanAgg

定義

  • 種別: function
  • 定義箇所: src/backend/executor/nodeAgg.c:4465

目的

集約ノード（AggState）の実行状態をリセットし、再スキャン（ReScan）を可能にする。

シグネチャまたは形状

void ExecReScanAgg(AggState *node)

入力と出力

  • **入力**: AggState *node（集約ノードの状態構造体）
  • **出力**: なし（副作用として node の内部状態を初期化する）

前提条件と不変条件

  • node は初期化済みであること。
  • AGG_HASHED の場合、ハッシュテーブルが未構築であれば即座に終了する。
  • AGG_HASHED かつ特定の条件（ハッシュテーブルがスピルしておらず、パラメータ変更がない等）を満たす場合、ハッシュテーブルを再利用してリセットする。

アルゴリズムまたは振る舞い

1. **ハッシュ集約の最適化**: AGG_HASHED で条件を満たす場合、ハッシュイテレータをリセットして終了する。 2. **リソース解放**:

  • 開いている tuplesort を終了し、sortstates をクリアする。
  • グループごとの式コンテキスト（aggcontexts）をリセットする。
  • 保持している grp_firstTuple を解放する。

    • 3. **状態リセット**:

  • 集約値（ecxt_aggvalues）と NULL フラグ（ecxt_aggnulls）をクリアする。

    • 4. **ハッシュ/混合集約の再構築**:

  • AGG_HASHED または AGG_MIXED の場合、スピル状態をリセットし、ハッシュテーブルを再構築（build_hash_tables）する。

    • 5. **ソート集約の初期化**:

  • AGG_HASHED 以外の場合、グループごとの状態をクリアし、フェーズ 1 に戻す。

    • 6. **下位ノードの再スキャン**:

  • 下位プランの chgParam が NULL の場合、ExecReScan を呼び出して下位ノードを再スキャンする。

関連データ構造

  • AggState: 集約ノードの実行状態を保持。
  • AggStatePerTrans: 遷移関数ごとの状態。
  • AggStatePerGroupData: グループごとの集約状態。
  • ExprContext: 式評価コンテキスト。

失敗モードと境界条件

  • node が NULL の場合、セグメンテーションフォールトが発生する可能性がある（ソースコード上でのチェックは明示されていない）。
  • メモリ不足時、build_hash_tables 等の内部関数がエラーを投げる可能性がある。

保守担当者向けメモ

  • AGG_HASHED におけるハッシュテーブルの再利用条件（outerPlan->chgParam == NULL 等）は、パフォーマンスに直結するため変更時は注意が必要。
  • AGG_HASHED または AGG_MIXED の場合、hashagg_reset_spill_state を通じてディスクスピル状態を確実にクリアすること。
