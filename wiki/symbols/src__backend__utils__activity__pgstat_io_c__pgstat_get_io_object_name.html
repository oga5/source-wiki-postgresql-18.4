pgstat_get_io_object_name

定義

  • 種別: function
  • 定義箇所: src/backend/utils/activity/pgstat_io.c:260

目的

I/O統計に関連する列挙型 IOObject の値を、対応する人間が読み取り可能な文字列名に変換します。

シグネチャまたは形状

const char *pgstat_get_io_object_name(IOObject io_object)

入力と出力

  • **入力**: IOObject 型の列挙値（IOOBJECT_RELATION, IOOBJECT_TEMP_RELATION, IOOBJECT_WAL のいずれか）。
  • **出力**: 対応する文字列リテラル（"relation", "temp relation", "wal" のいずれか）。

前提条件と不変条件

  • 入力される io_object は、定義済みの IOObject 列挙型のいずれかである必要があります。

アルゴリズムまたは振る舞い

1. switch 文を用いて io_object の値を判定します。 2. IOOBJECT_RELATION の場合は "relation" を返します。 3. IOOBJECT_TEMP_RELATION の場合は "temp relation" を返します。 4. IOOBJECT_WAL の場合は "wal" を返します。 5. 上記以外の値が渡された場合、elog(ERROR) を呼び出し、処理を中断します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/pgstatfuncs.c:1460 にて、統計関数から呼び出されています。
  • **呼び出し先**:
  • elog（エラー報告用マクロ）を呼び出します。

関連データ構造

  • IOObject: I/O操作の対象を識別するための列挙型。

失敗モードと境界条件

  • 定義されていない IOObject 値が渡された場合、ERROR レベルのログを出力して処理を停止します。

保守担当者向けメモ

  • 新しい IOObject が追加された場合は、本関数の switch 文にも対応するケースを追加する必要があります。
