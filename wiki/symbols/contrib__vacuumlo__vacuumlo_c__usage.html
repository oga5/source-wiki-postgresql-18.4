usage

定義

  • 種別: function
  • 定義箇所: contrib/vacuumlo/vacuumlo.c:413

目的

vacuumlo コマンドの利用方法、コマンドラインオプション、およびサポート情報を標準出力に表示する。

シグネチャまたは形状

void usage(const char *progname);

入力と出力

  • **入力**: progname (実行ファイル名を示す文字列)
  • **出力**: なし（標準出力へヘルプメッセージを出力して終了する）

前提条件と不変条件

  • progname は有効な文字列ポインタであること。
  • 本関数が呼び出された後、プログラムは終了することが想定される。

アルゴリズムまたは振る舞い

1. プログラム名を含む概要を表示する。 2. 利用可能なコマンドラインオプション（操作オプションおよび接続オプション）の一覧を出力する。 3. バグ報告先およびプロジェクトのホームページ URL を出力する。

呼び出し元と呼び出し先

  • **呼び出し元**: ソースコード内で本関数を呼び出している箇所は、提供された抜粋内には存在しません。
  • **呼び出し先**: 標準ライブラリの printf を使用してメッセージを出力します。

関連データ構造

  • なし

失敗モードと境界条件

  • 標準出力への書き込みが失敗した場合の挙動については、本関数内では制御していません。

保守担当者向けメモ

  • ツールが提示した他のファイル（contrib/oid2name 等）内の「usage」という単語を含むコメントは、本関数とは無関係な文脈（用語としての「使用法」）であるため、混同しないこと。
  • 新しいコマンドラインオプションを追加した際は、本関数内の printf 文を更新してヘルプメッセージを同期させる必要があります。
