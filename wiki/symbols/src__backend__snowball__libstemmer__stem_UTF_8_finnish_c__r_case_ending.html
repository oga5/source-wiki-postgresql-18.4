r_case_ending

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_finnish.c:413

目的

フィンランド語のステミング処理において、単語の格変化接尾辞を特定し、該当する場合はそれを削除して語幹を正規化する。

シグネチャまたは形状

static int r_case_ending(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（Snowball ステミング環境）
  • 出力: 成功時に 1、失敗時に 0、エラー発生時に負の値を返す。

前提条件と不変条件

  • z->I[1] にはステミング対象領域（R1）の開始位置が格納されている必要がある。
  • 処理対象の単語は UTF-8 エンコーディングであること。

アルゴリズムまたは振る舞い

1. **領域制限**: z->I[1]（R1領域）より前の文字は処理対象外とする。 2. **接尾辞検索**: a_6 配列を用いて、単語末尾から格変化接尾辞を検索する。 3. **条件分岐**:

  • case 1-4: 特定の母音（a, e, i, o）で終わる接尾辞を削除。
  • case 5-6: 特定の文字列（s_2, s_3）と一致する場合に削除。
  • case 7: r_LONG の条件判定や特定の文字スキップを経て削除。
  • case 8: 母音と子音のグループ（g_V1, g_C）の条件を満たす場合に削除。

    • 4. **削除実行**: slice_del(z) を呼び出し、特定された接尾辞を単語から除去する。 5. **状態更新**: 成功時に z->I[2] を 1 に設定し、処理済みであることを示す。

呼び出し元と呼び出し先

  • 呼び出し元: finnish_UTF_8_stem（src/backend/snowball/libstemmer/stem_UTF_8_finnish.c）
  • 呼び出し先: find_among_b, eq_s_b, r_LONG, skip_b_utf8, in_grouping_b_U, slice_del

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_6: 格変化接尾辞を定義した among 構造体の配列。
  • g_V1, g_C: 母音および子音のグループ定義。

失敗モードと境界条件

  • z->c < z->I[1] の場合、領域外のため即座に 0 を返す。
  • find_among_b で接尾辞が見つからない場合、0 を返す。
  • slice_del が失敗した場合、負の値を返す。
  • z->c <= z->lb の場合、境界条件により処理を中断する。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードである。アルゴリズムの変更は直接編集せず、Snowball 言語のソースファイルを修正すること。
  • z->I[2] は、この関数が適用されたかどうかのフラグとして後続の処理で使用される可能性がある。
