unicodeStyleColumnFormat

定義

  • 種別: struct
  • 定義箇所: src/fe_utils/print.c:108

目的

Unicode罫線を用いた表形式出力において、列の境界線や交差点を描画するための文字セットを定義する構造体です。

シグネチャまたは形状

typedef struct unicodeStyleColumnFormat
{
	const char *vertical;
	const char *vertical_and_horizontal[2];
	const char *up_and_horizontal[2];
	const char *down_and_horizontal[2];
} unicodeStyleColumnFormat;

入力と出力

本構造体はデータ入力ではなく、出力フォーマットの構成要素として使用されます。各メンバは罫線を描画するための文字列（UTF-8文字）を保持します。

前提条件と不変条件

  • 各メンバは、表の描画ロジックにおいて適切な罫線文字を指す必要があります。
  • 配列メンバ（vertical_and_horizontal 等）は、特定の交差状態に応じたバリエーションを保持します。

アルゴリズムまたは振る舞い

print.c 内の出力関数が、この構造体を参照して表の列境界を描画します。特に、Unicode罫線スタイルを選択した際に、垂直線や水平線との交差部分を動的に選択するために利用されます。

呼び出し元と呼び出し先

  • 呼び出し元: src/fe_utils/print.c:3697 にて、const unicodeStyleColumnFormat *column として参照されています。
  • 呼び出し先: 本構造体自体はデータ保持用であり、直接関数を呼び出すことはありません。
  • その他: src/fe_utils/print.c:129 にて column_style[2] の型として定義されています。

関連データ構造

  • unicodeStyleRowFormat: 行の境界線定義。
  • unicodeStyleBorderFormat: 表の外枠定義。
  • unicodeStyleFormat: 上記の各フォーマットを統合する上位構造体。

失敗モードと境界条件

  • 適切なUnicode文字が設定されていない場合、出力される表の罫線が崩れる可能性があります。
  • 端末がUTF-8をサポートしていない環境では、表示が乱れる可能性があります。

保守担当者向けメモ

  • 本構造体は unicodeStyleFormat の一部として使用されます。新しい罫線スタイルを追加する際は、この構造体のメンバがすべての交差パターンを網羅しているか確認してください。
