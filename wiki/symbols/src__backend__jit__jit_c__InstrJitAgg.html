InstrJitAgg

定義

  • 種別: function
  • 定義箇所: src/backend/jit/jit.c:181

目的

JIT コンパイルに関連する統計情報（JitInstrumentation）を、ソースからターゲットへ集計・加算するユーティリティ関数です。

シグネチャまたは形状

void InstrJitAgg(JitInstrumentation *dst, JitInstrumentation *add)

入力と出力

  • **入力**:
  • dst: 集計先となる JitInstrumentation 構造体へのポインタ。
  • add: 加算元となる JitInstrumentation 構造体へのポインタ。
  • **出力**: なし（dst が指す構造体の内容が更新されます）。

前提条件と不変条件

  • dst および add は有効な JitInstrumentation 構造体を指している必要があります。

アルゴリズムまたは振る舞い

add が保持する各統計値を dst に加算します。 1. created_functions（生成された関数数）を単純加算します。 2. 以下の各計測時間（instr_time）を INSTR_TIME_ADD マクロを用いて加算します。

  • generation_counter（生成時間）
  • deform_counter（デフォーム時間）
  • inlining_counter（インライン化時間）
  • optimization_counter（最適化時間）
  • emission_counter（コード生成時間）

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/commands/explain.c: EXPLAIN コマンド実行時の統計集計に使用。
  • src/backend/executor/execParallel.c: パラレルクエリ実行時の統計集計に使用。
  • **呼び出し先**:
  • INSTR_TIME_ADD マクロ（portability/instr_time.h で定義）。

関連データ構造

  • JitInstrumentation: JIT の実行統計を保持する構造体。

失敗モードと境界条件

  • ポインタが NULL の場合、セグメンテーション違反が発生します。

保守担当者向けメモ

  • 本関数は単なる加算処理であり、スレッドセーフではありません。呼び出し側で必要に応じて排他制御を行う必要があります。
