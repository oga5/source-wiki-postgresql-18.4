GetSQLCurrentDate

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/date.c:316

目的

SQL標準の CURRENT_DATE を実装し、現在の日付を DateADT 型で取得します。

シグネチャまたは形状

DateADT GetSQLCurrentDate(void)

入力と出力

  • **入力**: なし。
  • **出力**: DateADT（エポックからの経過日数を示す int32 値）。

前提条件と不変条件

  • GetCurrentDateTime を介してシステム時刻を取得します。
  • 内部的に静的変数（cache_year, cache_mon, cache_mday, cache_date）を保持しており、日付が変更されない限りキャッシュされた値を返します。

アルゴリズムまたは振る舞い

1. GetCurrentDateTime(&tm) を呼び出し、現在の時刻情報を取得します。 2. 取得した時刻の年・月・日がキャッシュと一致するか確認します。 3. 一致しない場合（日付が変わった場合）、date2j 関数を使用してユリウス日を計算し、POSTGRES_EPOCH_JDATE を減算して DateADT 形式に変換します。 4. 計算結果をキャッシュに保存し、その値を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/executor/execExprInterp.c:3217
  • **呼び出し先**:
  • GetCurrentDateTime
  • date2j

関連データ構造

  • DateADT: 日付を表現する int32 型。
  • struct pg_tm: 時刻情報を保持する構造体。

失敗モードと境界条件

  • 内部的な日付計算（date2j）において、システム時刻が極端な値である場合の挙動については本ソース内では明示されていません。

保守担当者向けメモ

  • 本関数は CURRENT_DATE の実装として機能します。
  • キャッシュ機構により、同一セッション内で日付が変わらない限り、高コストな date2j 呼び出しを抑制する設計となっています。
