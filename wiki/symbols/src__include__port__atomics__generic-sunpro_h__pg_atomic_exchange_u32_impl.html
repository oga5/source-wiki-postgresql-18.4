pg_atomic_exchange_u32_impl

定義

  • 種別: function
  • 定義箇所: src/include/port/atomics/generic-sunpro.h:85

目的

Solaris 環境において、32ビット符号なし整数に対するアトミックな値の交換操作を提供します。

シグネチャまたは形状

static inline uint32 pg_atomic_exchange_u32_impl(volatile pg_atomic_uint32 *ptr, uint32 newval)

入力と出力

  • **入力**:
  • ptr: 操作対象となる pg_atomic_uint32 構造体へのポインタ。
  • newval: ptr が指すメモリ位置に書き込む新しい値。
  • **出力**:
  • 交換前のメモリ位置に格納されていた値を uint32 型で返します。

前提条件と不変条件

  • 本関数は Solaris コンパイラ環境（SunPro）でのみ使用可能です。
  • ptr は volatile 修飾された pg_atomic_uint32 構造体を指している必要があります。

アルゴリズムまたは振る舞い

1. Solaris 標準の atomic_swap_32 関数を呼び出します。 2. ptr->value が指すメモリ位置の内容を newval でアトミックに置換します。 3. 置換前の値を戻り値として返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/include/port/atomics.h:334 にて、PostgreSQL の共通インターフェース経由で呼び出されています。
  • **呼び出し先**: Solaris システムライブラリの atomic_swap_32 を直接呼び出します。

関連データ構造

  • pg_atomic_uint32: アトミック操作対象の値を保持する構造体。

失敗モードと境界条件

  • 本関数はアトミック操作であるため、呼び出し自体が失敗することはありません。
  • ptr が指すアドレスが適切にアライメントされていない場合、基盤となる atomic_swap_32 の仕様に従い、ハードウェア例外や未定義動作が発生する可能性があります。

保守担当者向けメモ

  • 本関数は Solaris 環境専用の実装です。他プラットフォーム向けの実装については generic-gcc.h や generic-msvc.h を参照してください。
  • atomic_swap_32 は Solaris の標準的なアトミック操作関数であり、本関数はそのラッパーとして機能しています。
