copy_data_source_cb

定義

  • 種別: typedef
  • 定義箇所: src/include/commands/copy.h:96

目的

COPY FROM コマンド実行時に、外部ソースからデータを読み込むための抽象化されたコールバック関数型です。

シグネチャまたは形状

typedef int (*copy_data_source_cb) (void *outbuf, int minread, int maxread);

入力と出力

  • **入力**:
  • outbuf: 読み込んだデータを格納するバッファへのポインタ。
  • minread: 読み込むべき最小バイト数。
  • maxread: 読み込み可能な最大バイト数。
  • **出力**:
  • 戻り値: 実際に読み込まれたバイト数を返します。

前提条件と不変条件

  • outbuf は少なくとも maxread バイト以上のメモリ領域を指している必要があります。
  • 戻り値は 0 以上 maxread 以下である必要があります。

アルゴリズムまたは振る舞い

この関数は、COPY FROM 処理のループ内で呼び出され、指定されたバッファにデータを供給します。具体的な実装は、ファイル、標準入力、あるいはプログラムからのパイプなど、データソースの種類に応じて異なります。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/commands/copyfrom.c 内の処理フローから参照されています。
  • **呼び出し先**: 不明（この型は関数ポインタであり、具体的な実装関数は実行時に割り当てられます）。

関連データ構造

  • src/include/commands/copyfrom_internal.h: copy_data_source_cb 型のフィールドを持つ構造体が定義されています。
  • src/include/commands/copy.h: COPY コマンドの実行状態を管理する構造体群。

失敗モードと境界条件

  • 読み込みエラーが発生した場合の挙動については、本シンボルの定義からは特定できません。
  • minread と maxread の関係性や、EOF（ファイル終端）到達時の戻り値の仕様については、実装側の規約に依存します。
