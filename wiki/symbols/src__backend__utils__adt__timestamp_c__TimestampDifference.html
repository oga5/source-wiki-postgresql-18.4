TimestampDifference

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/timestamp.c:1724

目的

2つのタイムスタンプ（TimestampTz）の差分を計算し、秒（secs）とマイクロ秒（microsecs）の単位に分解して出力する。

シグネチャまたは形状

void TimestampDifference(TimestampTz start_time, TimestampTz stop_time, long *secs, int *microsecs)

入力と出力

  • **入力**:
  • start_time: 開始時刻（TimestampTz）
  • stop_time: 終了時刻（TimestampTz）
  • **出力**:
  • *secs: 差分の秒数（long へのポインタ）
  • *microsecs: 差分のマイクロ秒数（int へのポインタ）

前提条件と不変条件

  • TimestampTz はマイクロ秒単位の int64 として扱われる。
  • USECS_PER_SEC（1秒あたりのマイクロ秒数）が計算の基底となる。

アルゴリズムまたは振る舞い

1. stop_time から start_time を減算し、差分 diff を算出する。 2. diff が 0 以下の場合、*secs および *microsecs に 0 を代入する。 3. diff が正の場合、diff を USECS_PER_SEC で除算した商を *secs に、剰余を *microsecs に格納する。

  • src/backend/utils/activity/pgstat_database.c
  • **呼び出し先**:
  • 直接的な関数呼び出しは含まれない（算術演算のみ）。

関連データ構造

  • TimestampTz: int64 型のタイムスタンプ表現。

失敗モードと境界条件

  • stop_time が start_time より小さい場合、結果は 0 に切り捨てられる。
  • diff が非常に大きい場合、long 型へのキャスト時にオーバーフローが発生する可能性がある（環境依存）。

保守担当者向けメモ

  • 本関数は経過時間の計測やログ出力の計算によく利用される。
  • 負の差分を許容しない設計であるため、逆転した時刻入力を扱う場合は注意が必要。
