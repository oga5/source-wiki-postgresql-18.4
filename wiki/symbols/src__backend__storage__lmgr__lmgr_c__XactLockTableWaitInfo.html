XactLockTableWaitInfo

定義

  • 種別: struct
  • 定義箇所: src/backend/storage/lmgr/lmgr.c:54

目的

トランザクションの終了待機処理（XactLockTableWait）において、待機対象のコンテキスト情報や操作内容を保持するために使用される構造体です。

シグネチャまたは形状

typedef struct XactLockTableWaitInfo
{
	XLTW_Oper	oper;
	Relation	rel;
	ItemPointer ctid;
} XactLockTableWaitInfo;

入力と出力

本構造体は関数間の引数受け渡しに使用されます。

  • oper: 実行する操作の種類（XLTW_Oper 型）。
  • rel: 待機に関連するリレーション情報。
  • ctid: 待機に関連するタプルの物理位置（ItemPointer）。

前提条件と不変条件

  • 構造体の各フィールドは、待機処理を開始する前に適切に初期化されている必要があります。

アルゴリズムまたは振る舞い

本構造体は、待機処理のコールバックやエラーコンテキスト内で、どのリレーションやタプルに対して待機が発生しているかを特定するために参照されます。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/storage/lmgr/lmgr.c 内の XactLockTableWait 関連処理にてインスタンス化されます。
  • 呼び出し先: 待機処理の引数として渡され、内部で参照されます。
  • 詳細なコールグラフは提供されていません。

関連データ構造

  • XLTW_Oper: 待機操作の種類を定義する列挙型（本構造体のメンバ）。
  • Relation: PostgreSQLのリレーション記述子。
  • ItemPointer: タプルの物理位置を示すデータ構造。

失敗モードと境界条件

  • 構造体内のポインタ（rel）がNULLの場合、参照時にクラッシュする可能性があります。

保守担当者向けメモ

  • 本構造体は XactLockTableWait の実行コンテキストを保持する役割を持つため、待機ロジックに変更を加える際は、本構造体のメンバが十分な情報を保持しているか確認してください。
