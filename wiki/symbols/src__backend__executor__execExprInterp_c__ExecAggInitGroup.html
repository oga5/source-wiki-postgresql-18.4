ExecAggInitGroup

定義

  • 種別: function
  • 定義箇所: src/backend/executor/execExprInterp.c:5686

目的

集約関数のグループ処理において、遷移値（transition value）を初期化する。

シグネチャまたは形状

void ExecAggInitGroup(AggState *aggstate, AggStatePerTrans pertrans, AggStatePerGroup pergroup, ExprContext *aggcontext)

入力と出力

  • **入力**:
  • aggstate: 集約実行状態。
  • pertrans: 集約遷移関数の状態情報。
  • pergroup: 初期化対象のグループ状態。
  • aggcontext: 実行コンテキスト（メモリ管理に使用）。
  • **出力**: なし（pergroup のメンバを直接更新）。

前提条件と不変条件

  • 集約の入力型と遷移型（transtype）がバイナリ互換であること。
  • pergroup->transValue は初期化前であり、解放不要であること。

アルゴリズムまたは振る舞い

1. aggcontext->ecxt_per_tuple_memory にメモリコンテキストを切り替える。 2. pertrans->transfn_fcinfo から初期値を取得し、datumCopy を用いて pergroup->transValue にコピーする。 3. pergroup->transValueIsNull を false に設定する。 4. pergroup->noTransValue を false に設定する。 5. 元のメモリコンテキストへ復帰する。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/executor/execExprInterp.c 内の複数箇所（行2131, 2192）。
  • **呼び出し先**: datumCopy, MemoryContextSwitchTo（PostgreSQL 内部ユーティリティ）。

関連データ構造

  • AggState: 集約実行の全体状態。
  • AggStatePerTrans: 遷移関数ごとのメタデータ。
  • AggStatePerGroup: 特定グループの集約状態。
  • ExprContext: 式評価の実行コンテキスト。

失敗モードと境界条件

  • メモリ不足時に datumCopy がエラーを発生させる可能性がある。
  • 参照渡し（pass-by-ref）のデータ型の場合、メモリコンテキストの切り替えが正しく行われないとメモリリークや不正アクセスが発生する。

保守担当者向けメモ

  • 本関数は集約処理のホットパスの一部である。
  • pertrans->transtypeByVal および transtypeLen に基づいてコピーが行われるため、これらの設定が正しいことを前提としている。
