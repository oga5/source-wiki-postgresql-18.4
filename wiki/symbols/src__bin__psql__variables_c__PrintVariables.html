PrintVariables

定義

  • 種別: function
  • 定義箇所: src/bin/psql/variables.c:255

目的

VariableSpace 内に保持されているすべての変数を、標準出力へ一覧表示します。

シグネチャまたは形状

void PrintVariables(VariableSpace space)

入力と出力

  • **入力**: VariableSpace space （変数リストのヘッダー構造体）
  • **出力**: なし（副作用として標準出力へ name = 'value' 形式で出力）

前提条件と不変条件

  • space が NULL の場合、関数は何もせず即座に復帰します。
  • 変数は連結リストとして保持されており、本関数はリストを順次走査します。
  • ptr->value が存在（非NULL）する変数のみが出力対象となります。

アルゴリズムまたは振る舞い

1. space が NULL であれば終了します。 2. リストの先頭から順に各変数を走査します。 3. 各変数について、値が設定されている場合に printf を用いて 変数名 = '値' の形式で出力します。 4. 走査中にグローバル変数 cancel_pressed が真になった場合、ループを中断して終了します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/bin/psql/command.c:2887 にて pset.vars を引数として呼び出されています。
  • **呼び出し先**: 標準ライブラリの printf を使用しています。

関連データ構造

  • VariableSpace: 変数リストのヘッダー構造体。
  • struct _variable: リストの各要素を構成する構造体（name および value フィールドを持つ）。

失敗モードと境界条件

  • space が NULL の場合は安全に処理を終了します。

保守担当者向けメモ

  • 本関数は psql の \set コマンド等で設定された変数のデバッグや確認に使用されます。
  • 出力フォーマットを変更する場合は、src/bin/psql/variables.c 内の当該箇所を修正してください。
