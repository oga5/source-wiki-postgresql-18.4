ArchiveModuleInit

定義

  • 種別: typedef
  • 定義箇所: src/include/archive/archive_module.h:55

目的

アーカイブモジュールがロードされた際に、コアシステムに対してモジュールが提供するコールバック関数のセット（ArchiveModuleCallbacks）を渡すための初期化関数の型を定義します。

シグネチャまたは形状

typedef const ArchiveModuleCallbacks *(*ArchiveModuleInit) (void);

入力と出力

  • 入力: なし（void）
  • 出力: ArchiveModuleCallbacks 構造体への定数ポインタを返します。

前提条件と不変条件

  • 本関数は、アーカイブライブラリがロードされた直後に呼び出されることを想定しています。
  • 返される ArchiveModuleCallbacks 構造体は、モジュールの生存期間中、有効かつ不変である必要があります。

アルゴリズムまたは振る舞い

1. システムが archive_library GUC で指定された動的ライブラリをロードします。 2. ライブラリ内の初期化関数（_PG_archive_module_init 等）がこの型に従って呼び出されます。 3. モジュールは実装済みのコールバック関数を格納した ArchiveModuleCallbacks 構造体のアドレスを返します。 4. コアシステムは受け取ったポインタを介して、アーカイブ処理に必要な各コールバック関数にアクセスします。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/postmaster/pgarch.c 内のロジックが、ロードしたライブラリからこの型の関数を取得し実行します。
  • 呼び出し先: 外部アーカイブモジュール内の初期化関数。

関連データ構造

  • ArchiveModuleCallbacks: 本関数が返すコールバック関数の集合を保持する構造体。

失敗モードと境界条件

  • 供給されたソース情報には、本関数自体の失敗モードに関する記述はありません。

保守担当者向けメモ

  • 本関数はアーカイブモジュールのエントリポイントとなるため、シグネチャの変更はバイナリ互換性に影響を与えます。
  • 戻り値の ArchiveModuleCallbacks 構造体には、必須の ArchiveFileCB が含まれている必要があります。
