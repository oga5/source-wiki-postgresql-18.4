DatumGetMacaddr8P

定義

  • 種別: static inline 関数
  • 定義箇所: src/include/utils/inet.h:162

目的

PostgreSQLの Datum 型として保持されている値を、macaddr8 構造体へのポインタとしてキャストして取得します。

シグネチャまたは形状

static inline macaddr8 *DatumGetMacaddr8P(Datum X)

入力と出力

  • **入力**: Datum X（macaddr8 型のデータを含む Datum）
  • **出力**: macaddr8 *（キャストされた構造体へのポインタ）

前提条件と不変条件

  • 入力 Datum X は、有効な macaddr8 構造体へのポインタを保持している必要があります。
  • 本関数は DatumGetPointer マクロを利用しており、Datum がポインタ型として格納されていることを前提としています。

アルゴリズムまたは振る舞い

1. DatumGetPointer(X) を呼び出し、Datum から汎用ポインタ（void *）を取り出します。 2. 取り出したポインタを macaddr8 * 型にキャストして返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/network.c:1507
  • src/include/utils/inet.h:174（PG_GETARG_MACADDR8_P マクロ内）
  • **呼び出し先**:
  • DatumGetPointer（PostgreSQLのコアマクロ）

関連データ構造

  • macaddr8: MACアドレスを格納するための固定長構造体。

失敗モードと境界条件

  • Datum が NULL または不正なメモリ領域を指している場合、呼び出し元でセグメンテーション違反が発生する可能性があります。本関数自体はポインタの妥当性チェックを行いません。

保守担当者向けメモ

  • 本関数は macaddr8 型のデータアクセスを抽象化するインライン関数です。変更を加える際は、macaddr8 構造体のメモリレイアウトとの整合性に注意してください。
