r_e_ending

目的

オランダ語のステミング処理において、単語末尾の 'e' を条件付きで削除し、必要に応じて重複文字の解消を行う。

シグネチャまたは形状

static int r_e_ending(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z (ステミング環境構造体)
  • **出力**: int (成功時に 1、失敗または条件不一致時に 0、エラー時に負の値を返す)

前提条件と不変条件

  • z->c が単語の末尾を指していること。
  • r_R1 関数によって定義される領域 R1 内での処理であること。
  • 削除対象の 'e' の直前に母音（g_v グループ）が存在すること。

アルゴリズムまたは振る舞い

1. z->I[3] を 0 に初期化する。 2. 現在のカーソル位置の文字が 'e' であるかを確認し、一致しなければ 0 を返す。 3. r_R1(z) を呼び出し、削除対象が R1 領域内にあるか検証する。 4. out_grouping_b を使用し、'e' の直前が母音（g_v）であることを確認する。 5. slice_del(z) を実行して 'e' を削除する。 6. z->I[3] を 1 に設定し、r_undouble(z) を呼び出して語幹の重複文字を解消する。 7. 成功時に 1 を返す。

  • src/backend/snowball/libstemmer/stem_UTF_8_dutch.c 内のメインロジック（行 438, 538）

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • g_v: 母音を定義するビットマップ（in_grouping / out_grouping 用）。

失敗モードと境界条件

  • z->c <= z->lb: カーソルが境界外にある場合は 0 を返す。
  • r_R1 または slice_del が負の値を返した場合、その値をそのまま伝播させる。
  • 直前の文字が母音でない場合、削除を行わず 0 を返す。
