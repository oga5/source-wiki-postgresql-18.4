SkipPages

定義

  • 種別: enum
  • 定義箇所: contrib/amcheck/verify_heapam.c:62

目的

ヒープリレーションの検証処理において、どの種類のページを走査対象から除外するかを指定するための列挙型です。

シグネチャまたは形状

typedef enum SkipPages
{
	SKIP_PAGES_ALL_FROZEN,
	SKIP_PAGES_ALL_VISIBLE,
	SKIP_PAGES_NONE,
} SkipPages;

入力と出力

  • 入力: なし（列挙定数）
  • 出力: なし

前提条件と不変条件

  • SKIP_PAGES_NONE がデフォルトの選択肢として使用されます。
  • 各オプションは、ヒープ走査時のページスキップ判定ロジックにおいて、検証の範囲を制御するために使用されます。

アルゴリズムまたは振る舞い

  • SKIP_PAGES_ALL_FROZEN: すべてのタプルが凍結されているページをスキップします。
  • SKIP_PAGES_ALL_VISIBLE: すべてのタプルが可視であるページをスキップします。
  • SKIP_PAGES_NONE: いかなるページもスキップせず、全ページを検証対象とします。

呼び出し元と呼び出し先

  • 呼び出し元: verify_heapam 関数内で skip_option として初期化され、検証ロジックの制御に使用されます。
  • 呼び出し先: 情報なし。

関連データ構造

  • HeapCheckContext: 検証のコンテキストを保持する構造体であり、この中で skip_option フィールドとして利用されます。

失敗モードと境界条件

  • 特筆すべき失敗モードはありません。

保守担当者向けメモ

  • 本列挙型は verify_heapam における走査の最適化を目的としています。新しいスキップ条件を追加する場合は、heapcheck_read_stream_next_unskippable 等の走査ロジックとの整合性に注意してください。
