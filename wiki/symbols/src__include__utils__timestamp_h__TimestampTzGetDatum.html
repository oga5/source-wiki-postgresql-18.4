TimestampTzGetDatum

定義

  • 種別: function
  • 定義箇所: src/include/utils/timestamp.h:51

目的

内部的なタイムスタンプ表現（TimestampTz）を、PostgreSQLの関数マネージャ（FMGR）が扱う Datum 型へ変換する。

シグネチャまたは形状

static inline Datum TimestampTzGetDatum(TimestampTz X)

入力と出力

  • **入力**: TimestampTz X （内部的な64ビット整数表現のタイムスタンプ）
  • **出力**: Datum （X を格納した Datum 型の値）

前提条件と不変条件

  • TimestampTz は int64 として実装されているため、本関数は Int64GetDatum を介して値をラップする。
  • インライン関数として定義されており、呼び出し時のオーバーヘッドを最小限に抑える設計となっている。

アルゴリズムまたは振る舞い

引数として受け取った TimestampTz 型の値 X を、Int64GetDatum(X) を呼び出すことで Datum 型に変換して返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/pg_stat_statements/pg_stat_statements.c
  • src/backend/access/transam/commit_ts.c
  • src/backend/access/transam/twophase.c
  • src/backend/access/transam/xlogprefetcher.c
  • src/backend/commands/prepare.c
  • src/backend/executor/execExprInterp.c
  • src/backend/replication/logical/launcher.c
  • **呼び出し先**:
  • Int64GetDatum （fmgr.h で定義されるマクロ）

関連データ構造

  • TimestampTz: 内部的な日時表現（int64）。
  • Datum: PostgreSQLの汎用データ型。

失敗モードと境界条件

  • 特記事項なし。int64 の値を Datum に変換する単純なラッパーであるため、本関数自体が失敗することはない。

保守担当者向けメモ

  • 本関数は src/include/utils/timestamp.h に定義されたインライン関数であり、パフォーマンスを重視して設計されている。変更を加える際は、Datum 変換のオーバーヘッドが増加しないよう注意すること。
