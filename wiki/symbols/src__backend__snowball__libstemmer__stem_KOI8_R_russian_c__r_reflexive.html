r_reflexive

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_KOI8_R_russian.c:466

目的

ロシア語のステミング処理において、単語末尾の再帰動詞（reflexive）接尾辞を特定し、削除する。

シグネチャまたは形状

static int r_reflexive(struct SN_env * z)

入力と出力

  • 入力: SN_env 構造体（ステミング環境）。
  • 出力: 処理が成功した場合は 1、失敗または条件不一致の場合は 0、内部エラー発生時は負の値を返す。

前提条件と不変条件

  • z->c が現在のカーソル位置を指していること。
  • a_3 配列に定義された再帰動詞の接尾辞パターンが利用可能であること。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置 z->c を z->ket に設定する。 2. 単語末尾の文字が KOI8-R コードで 209 または 216 であるかを確認し、条件を満たさない場合は終了する。 3. find_among_b を使用し、a_3 に定義されたパターンを末尾から検索する。 4. マッチした場合、その位置を z->bra に設定し、slice_del を呼び出して該当部分を削除する。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_KOI8_R_russian.c および src/backend/snowball/libstemmer/stem_UTF_8_russian.c 内のステミングメイン処理から呼び出される。
  • 呼び出し先: find_among_b（パターンマッチング）、slice_del（文字列削除）。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • a_3: 再帰動詞の接尾辞パターンを格納した among 構造体の配列。

失敗モードと境界条件

  • 単語の長さが不足している場合や、末尾の文字が指定されたコード（209, 216）と一致しない場合は 0 を返す。
  • slice_del が内部エラー（負の値）を返した場合、その値をそのまま伝播させる。

保守担当者向けメモ

  • 本関数はSnowballコンパイラによる自動生成コードである。直接編集せず、元のSnowballスクリプト（.sbl）を修正すること。
  • KOI8-R エンコーディング特有の文字コード（209, 216）に依存しているため、エンコーディング変更時には注意が必要である。
