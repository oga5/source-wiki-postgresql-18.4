FlushLocalBuffer

定義

  • 種別: function
  • 定義箇所: src/backend/storage/buffer/localbuf.c:181

目的

一時テーブル用バッファの内容を物理ストレージへ書き出し、バッファのダーティ状態を解除します。

シグネチャまたは形状

void FlushLocalBuffer(BufferDesc *bufHdr, SMgrRelation reln)

入力と出力

  • **入力**:
  • bufHdr: 書き出し対象のバッファ記述子。
  • reln: 対象リレーションのストレージマネージャハンドル（NULL の場合は内部で取得）。
  • **出力**: なし（副作用としてディスクへの書き込みと統計情報の更新を行う）。

前提条件と不変条件

  • 対象バッファはピン留めされている必要があります（LocalRefCount が 0 より大きいこと）。
  • StartLocalBufferIO が成功することが前提であり、失敗した場合はエラーを発生させます。

アルゴリズムまたは振る舞い

1. LocalBufHdrGetBlock を用いてバッファ内のページデータを取得します。 2. StartLocalBufferIO を呼び出し、I/O 操作を開始します。 3. reln が NULL の場合、BufTagGetRelFileLocator を使用してストレージマネージャを開きます。 4. PageSetChecksumInplace を実行し、ページチェックサムを更新します。 5. smgrwrite を呼び出し、物理ストレージへページを書き出します。 6. pgstat_count_io_op_time を呼び出し、I/O 統計を記録します。 7. TerminateLocalBufferIO を呼び出し、バッファのダーティフラグをクリアします。 8. pgBufferUsage.local_blks_written をインクリメントします。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/storage/buffer/bufmgr.c:4972
  • src/backend/storage/buffer/localbuf.c:289
  • src/test/modules/test_aio/test_aio.c:489
  • **呼び出し先**:
  • StartLocalBufferIO
  • smgropen
  • PageSetChecksumInplace
  • smgrwrite
  • TerminateLocalBufferIO
  • pgstat_prepare_io_time / pgstat_count_io_op_time

関連データ構造

  • BufferDesc: バッファのメタデータおよびタグ情報。
  • SMgrRelation: ストレージマネージャの操作対象リレーション。
  • LocalRefCount: ローカルバッファの参照カウント配列。

失敗モードと境界条件

  • StartLocalBufferIO が失敗した場合、elog(ERROR) により処理が中断されます。

保守担当者向けメモ

  • 本関数は一時テーブル専用であり、共有バッファマネージャの BufferAccessStrategy は使用しません。
  • 呼び出し元で reln を提供しない場合、smgropen が内部で実行されるため、呼び出し頻度が高い場合は注意が必要です。
