uuid_ns_oid

定義

  • 種別: function
  • 定義箇所: contrib/uuid-ossp/uuid-ossp.c:459

目的

名前空間ベースの UUID 生成（v3/v5）において、OID（Object Identifier）名前空間を表す UUID 値を返します。

シグネチャまたは形状

Datum uuid_ns_oid(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS（PostgreSQL の関数呼び出し規約に基づく引数）。
  • **出力**: Datum 型として、OID 名前空間の UUID 値を返します。

前提条件と不変条件

  • コンパイル時の環境設定（HAVE_UUID_OSSP の有無）により、内部的な取得経路が分岐します。
  • 返される値は RFC 4122 で定義された OID 名前空間の固定 UUID（6ba7b812-9dad-11d1-80b4-00c04fd430c8）に準拠します。

アルゴリズムまたは振る舞い

1. HAVE_UUID_OSSP が定義されている場合:

  • special_uuid_value("ns:OID") を呼び出し、OSSP ライブラリ経由で名前空間 UUID を取得します。

    • 2. HAVE_UUID_OSSP が定義されていない場合:

  • uuid_generate_internal を呼び出し、OID 名前空間の固定 UUID 文字列を直接渡して生成します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/uuid-ossp/uuid-ossp.c:113 (PG_FUNCTION_INFO_V1 マクロによる登録)
  • **呼び出し先**:
  • special_uuid_value（OSSP 有効時）
  • uuid_generate_internal（OSSP 無効時）

関連データ構造

  • Datum: PostgreSQL の汎用データ型。

失敗モードと境界条件

  • 外部ライブラリ（OSSP）がリンクされていない環境では、フォールバックとして uuid_generate_internal が使用されます。
  • 呼び出し元からの引数検証に関する情報は提供されていません。

保守担当者向けメモ

  • 本関数は名前空間の定数を返すためのラッパーです。名前空間 UUID の値自体を変更する必要がある場合は、RFC 4122 の仕様との整合性に注意してください。
