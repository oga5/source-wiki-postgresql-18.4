float4_dist

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_float4.c:94

目的

2つの float4 型の値の絶対差分（距離）を計算し、PostgreSQLの関数インターフェースとして提供する。

シグネチャまたは形状

Datum float4_dist(PG_FUNCTION_ARGS)

入力と出力

  • 入力: PG_FUNCTION_ARGS を介して2つの float4 値を受け取る。
  • 出力: Datum 型として、入力値の差の絶対値（float4）を返す。

前提条件と不変条件

  • 入力値は PG_GETARG_FLOAT4 マクロで取得可能であること。
  • 演算結果がオーバーフローした場合、入力値自体が無限大でない限り float_overflow_error() が呼び出される。

アルゴリズムまたは振る舞い

1. PG_GETARG_FLOAT4 を用いて第1引数 a と第2引数 b を取得する。 2. r = a - b を計算する。 3. r が無限大であり、かつ元の a と b がいずれも無限大ではない場合、浮動小数点オーバーフローエラーを発生させる。 4. fabsf(r) を計算し、その結果を PG_RETURN_FLOAT4 を介して返す。

呼び出し元と呼び出し先

  • 呼び出し元: PostgreSQLの関数マネージャ（PG_FUNCTION_INFO_V1 マクロにより登録）。
  • 呼び出し先: float_overflow_error()（エラー発生時）、fabsf()（標準Cライブラリ）。
  • 注: ソースコードの文脈外からの具体的な呼び出し元情報は提供されていない。

関連データ構造

  • float4 (PostgreSQLの単精度浮動小数点型)

失敗モードと境界条件

  • 演算結果が float4 の表現範囲を超える場合、float_overflow_error() によりエラー終了する。

保守担当者向けメモ

  • 本関数はインデックス外でも利用可能な距離計算関数としてエクスポートされている。
  • btree_gist モジュール内の他の数値型距離計算関数と一貫したエラーハンドリングを行っている。
