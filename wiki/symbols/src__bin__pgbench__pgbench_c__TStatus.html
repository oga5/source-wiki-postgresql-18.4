TStatus

定義

  • 種別: enum
  • 定義箇所: src/bin/pgbench/pgbench.c:470

目的

pgbench におけるトランザクション実行状態を表現するための列挙型です。

シグネチャまたは形状

typedef enum TStatus
{
	TSTATUS_IDLE,        /* 待機中 */
	TSTATUS_IN_BLOCK,    /* トランザクションブロック内 */
	TSTATUS_CONN_ERROR,  /* 接続エラー発生 */
	TSTATUS_OTHER_ERROR, /* その他のエラー発生 */
} TStatus;

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: TStatus 型の値を返します。

前提条件と不変条件

  • pgbench の実行ループ内において、クライアントの現在のトランザクション状態を追跡するために使用されます。

アルゴリズムまたは振る舞い

  • TSTATUS_IDLE: トランザクションが開始されていない状態。
  • TSTATUS_IN_BLOCK: BEGIN コマンド発行後、COMMIT または ROLLBACK が完了するまでの状態。
  • TSTATUS_CONN_ERROR: データベース接続の切断や通信障害が発生した状態。
  • TSTATUS_OTHER_ERROR: 実行中のスクリプトやコマンド処理で予期せぬエラーが発生した状態。

呼び出し元と呼び出し先

  • 呼び出し元: src/bin/pgbench/pgbench.c 内の関数（3586 行目、4112 行目、4267 行目など）で、ローカル変数 tstatus として参照されています。
  • 呼び出し先: 本シンボルは列挙型であるため、直接的な呼び出し先はありません。

関連データ構造

  • CState: クライアントの状態を保持する構造体。
  • TState: スレッドの状態を保持する構造体。

失敗モードと境界条件

  • エラー状態（TSTATUS_CONN_ERROR, TSTATUS_OTHER_ERROR）に遷移した場合、後続のトランザクション実行が停止または再試行の対象となります。

保守担当者向けメモ

  • 本シンボルは pgbench.c 内のステートマシン制御において、トランザクションの整合性を維持するために重要です。状態遷移のロジックを変更する際は、advanceConnectionState() 関数との整合性に注意してください。
