IsWalSummaryFilename

定義

  • 種別: function
  • 定義箇所: src/backend/backup/walsummary.c:262

目的

与えられたファイル名が、WAL要約ファイルの命名規則（TLI + StartLSN + EndLSN を16進数で連結し、.summary 拡張子を持つ）に従っているかを判定します。

シグネチャまたは形状

static bool IsWalSummaryFilename(char *filename)

入力と出力

  • **入力**: filename (char型ポインタ) - 検査対象のファイル名文字列。
  • **出力**: bool 型。命名規則に合致する場合は true、それ以外は false を返します。

前提条件と不変条件

  • ファイル名の先頭40文字が16進数（0-9, A-F）で構成されている必要があります。
  • 41文字目以降が .summary という文字列で終わる必要があります。

アルゴリズムまたは振る舞い

1. strspn を使用し、ファイル名の先頭から16進数文字のみが連続する長さを計測します。この長さが正確に40文字であることを確認します。 2. strcmp を使用し、40文字目以降の文字列が .summary と一致するかを確認します。 3. 上記2つの条件を両方満たす場合にのみ true を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/backup/walsummary.c 内の GetWalSummaries 関数から呼び出され、ディレクトリ内のファイルがWAL要約ファイルであるかをフィルタリングするために使用されています。
  • **呼び出し先**: 標準Cライブラリの strspn および strcmp を呼び出します。

関連データ構造

  • なし。

失敗モードと境界条件

  • ファイル名が40文字未満、または40文字を超えていても .summary で終わらない場合は false を返します。
  • 16進数以外の文字が含まれている場合、strspn の結果が40未満となり false を返します。

保守担当者向けメモ

  • 本関数はファイル名の形式のみを検証します。ファイルの内容や、実際にファイルが存在するかどうかは検証しません。
  • 命名規則の変更（例えばファイル名の長さや拡張子の変更）が発生した場合は、本関数のマジックナンバー（40および .summary）を更新する必要があります。
