PLpgSQL_label_type

定義

  • 種別: enum
  • 定義箇所: src/pl/plpgsql/src/plpgsql.h:52

目的

PL/pgSQL におけるラベル付き構造（ブロックやループなど）の種類を識別し、名前空間管理や構文解析時のコンテキストを区別するために使用されます。

シグネチャまたは形状

typedef enum PLpgSQL_label_type
{
    PLPGSQL_LABEL_BLOCK,        /* DECLARE/BEGIN ブロック */
    PLPGSQL_LABEL_LOOP,         /* ループ構造 */
    PLPGSQL_LABEL_OTHER,        /* その他 */
} PLpgSQL_label_type;

入力と出力

  • 入力: なし（列挙型定義）
  • 出力: なし

前提条件と不変条件

  • PLPGSQL_LABEL_BLOCK は DECLARE または BEGIN ブロックに関連付けられます。
  • PLPGSQL_LABEL_LOOP はループ構造に関連付けられます。
  • 名前空間管理において、ラベルに関連する項目（itemno）としてこの値が格納されることがあります。

アルゴリズムまたは振る舞い

本シンボルは、名前空間スタックへのプッシュ操作や、構文解析時のラベルの分類に使用されます。特定のラベルがどの構造に属するかを判定する際の型識別子として機能します。

呼び出し元と呼び出し先

  • 呼び出し元（直接的な使用例）:
  • src/pl/plpgsql/src/pl_funcs.c:54: plpgsql_ns_push 関数が引数として受け取ります。
  • 呼び出し先:
  • 本シンボルは列挙型であり、関数を呼び出す側ではありません。

関連データ構造

  • PLpgSQL_nsitem: 名前空間内の項目を管理する構造体で、ラベルの種類を保持するために本型が参照されます。

失敗モードと境界条件

  • 定義外の値をキャストして代入した場合の動作は未定義です。
  • PLPGSQL_LABEL_OTHER は、ブロックやループ以外のラベル付き構造に対してフォールバックとして使用されます。

保守担当者向けメモ
