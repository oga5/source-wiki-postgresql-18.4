TimeADTGetDatum

定義

  • 種別: static inline function
  • 定義箇所: src/include/utils/date.h:77

目的

TimeADT 型の値を、PostgreSQLの関数呼び出しインターフェース（fmgr）で扱われる汎用的な Datum 型に変換する。

シグネチャまたは形状

static inline Datum TimeADTGetDatum(TimeADT X)

入力と出力

  • **入力**: TimeADT X （時刻の内部表現である int64 値）
  • **出力**: Datum （int64 を格納した Datum 型の値）

前提条件と不変条件

  • TimeADT は内部的に int64 として定義されている。
  • 本関数は Int64GetDatum を呼び出すことで変換を行う。

アルゴリズムまたは振る舞い

引数として受け取った TimeADT 値を、Int64GetDatum マクロを介して Datum 型へキャストして返す。インライン関数として定義されており、呼び出し時のオーバーヘッドを最小化している。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/btree_gin/btree_gin.c
  • contrib/btree_gist/btree_time.c
  • contrib/btree_gist/btree_utils_num.c
  • src/backend/executor/execExprInterp.c
  • src/backend/utils/adt/formatting.c
  • src/backend/utils/adt/jsonpath_exec.c
  • src/include/utils/date.h (マクロ PG_RETURN_TIMEADT 経由)
  • **呼び出し先**:
  • Int64GetDatum (PostgreSQLの Datum 変換マクロ)

関連データ構造

  • TimeADT: 時刻を保持する int64 型のエイリアス。
  • Datum: PostgreSQLの汎用データ型。

失敗モードと境界条件

  • 特記事項なし。入力された int64 値をそのまま Datum に変換する。

保守担当者向けメモ

  • 本関数はパフォーマンス向上のため static inline で定義されている。変更を加える際は、インライン化の特性に注意すること。
  • TimeADT の内部表現（int64）に変更があった場合、本関数の実装も追従する必要がある。
