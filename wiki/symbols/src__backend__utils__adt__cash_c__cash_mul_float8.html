cash_mul_float8

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/cash.c:116

目的

通貨型（Cash）の値に浮動小数点数（float8）を乗算し、結果を通貨型として返します。

シグネチャまたは形状

static inline Cash cash_mul_float8(Cash c, float8 f)

入力と出力

  • **入力**:
  • c: 乗算対象の通貨値（Cash 型）。
  • f: 乗数（float8 型）。
  • **出力**:
  • 乗算結果を Cash 型にキャストして返します。

前提条件と不変条件

  • 内部的に rint() を使用して浮動小数点演算結果を整数へ丸めます。
  • 結果が int64 の範囲内に収まる必要があります。

アルゴリズムまたは振る舞い

1. c を float8 にキャストし、f と乗算します。 2. rint() 関数を用いて結果を整数値に丸めます。 3. 結果が NaN であるか、または int64 の範囲を超えている場合、ERRCODE_NUMERIC_VALUE_OUT_OF_RANGE エラーを発生させます。 4. 正常な場合は、丸められた値を Cash 型にキャストして返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/adt/cash.c 内の複数の箇所から呼び出されています（行 741, 754, 780, 793）。これらは主に SQL インターフェース経由の演算処理です。
  • **呼び出し先**:
  • rint() (標準数学ライブラリ)
  • float8_mul()
  • ereport() (エラー報告)

関連データ構造

  • Cash: int64 の別名であり、通貨型の内部表現。

失敗モードと境界条件

  • **オーバーフロー**: 乗算結果が int64 の範囲（FLOAT8_FITS_IN_INT64）を超えた場合、エラーとなります。
  • **無効な値**: 乗算結果が NaN の場合、エラーとなります。

保守担当者向けメモ

  • 本関数は static inline で定義されており、パフォーマンスを考慮してヘッダーではなくソースファイル内で定義されています。
  • int64 へのキャスト時に精度が失われる可能性があるため、丸め処理（rint）の挙動には注意してください。
