makeInteger

定義

  • 種別: function
  • 定義箇所: src/backend/nodes/value.c:22

目的

整数値を保持する Integer 型のノードを生成し、初期化して返却するファクトリ関数です。

シグネチャまたは形状

Integer *makeInteger(int i)

入力と出力

  • **入力**: int i （ノードに格納する整数値）
  • **出力**: Integer * （初期化された Integer 構造体へのポインタ）

前提条件と不変条件

  • makeNode(Integer) マクロにより、PostgreSQL のメモリコンテキスト上で適切なメモリ領域が確保されている必要があります。

アルゴリズムまたは振る舞い

1. makeNode(Integer) を呼び出し、Integer 構造体用のメモリを確保します。 2. 引数 i を、確保した構造体の ival メンバに代入します。 3. 初期化済みのノードポインタを返却します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/postgres_fdw/postgres_fdw.c
  • src/backend/commands/tsearchcmds.c
  • src/backend/nodes/read.c
  • **呼び出し先**:
  • makeNode マクロ（メモリ確保用）

関連データ構造

  • Integer: src/include/nodes/value.h で定義される、整数値を保持するためのノード構造体。

失敗モードと境界条件

  • メモリ不足時には makeNode の内部実装に従い、エラーが発生する可能性があります。

保守担当者向けメモ

  • 本関数は value.c 内の他の make* 関数群と設計思想を共有しています。
  • Integer ノードは単純な値の保持に使用されるため、メモリ管理は PostgreSQL のノードシステムに依存します。
