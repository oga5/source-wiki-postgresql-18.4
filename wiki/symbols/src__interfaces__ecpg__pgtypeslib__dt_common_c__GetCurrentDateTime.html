GetCurrentDateTime

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/pgtypeslib/dt_common.c:1059

目的

現在の日時を取得し、指定された struct tm 構造体に格納する。

シグネチャまたは形状

void GetCurrentDateTime(struct tm *tm)

入力と出力

  • **入力**: tm (現在日時を書き込むための struct tm へのポインタ)
  • **出力**: なし（戻り値は void）

前提条件と不変条件

  • 引数 tm は有効なメモリ領域を指している必要がある。
  • 内部で time(NULL) を呼び出すため、システム時刻が正しく設定されている必要がある。

アルゴリズムまたは振る舞い

1. time(NULL) を呼び出し、現在のシステム時刻（エポック秒）を取得する。 2. 取得した時刻を abstime2tm 関数に渡し、タイムゾーン情報（tz）および分解された日時情報（tm）へ変換する。 3. 変換結果を引数 tm に格納する。

呼び出し元と呼び出し先

  • **呼び出し先**: time (標準ライブラリ), abstime2tm (内部関数)
  • **呼び出し元**:
  • src/interfaces/ecpg/pgtypeslib/datetime.c:152
  • その他、src/backend/utils/adt/ 配下の各ファイル（date.c, datetime.c）から参照されているが、これらはバックエンド側の実装であり、本ライブラリ（ECPG）の文脈における直接的な呼び出し関係は datetime.c が該当する。

関連データ構造

  • struct tm: C標準ライブラリの日時構造体。

失敗モードと境界条件

  • tm が NULL の場合、セグメンテーションフォールトが発生する可能性がある。
  • time(NULL) が失敗した場合の挙動は、標準ライブラリの time 関数の仕様に依存する。

保守担当者向けメモ

  • 本関数は abstime2tm を介して時刻変換を行うため、タイムゾーンの扱いは abstime2tm の実装に依存する。
  • バックエンド側の datetime.c でも同名の関数が定義されているため、リンク時の競合や参照先の混同に注意すること。
