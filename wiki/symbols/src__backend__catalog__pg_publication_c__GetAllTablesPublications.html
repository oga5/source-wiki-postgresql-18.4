GetAllTablesPublications

定義

  • 種別: function
  • 定義箇所: src/backend/catalog/pg_publication.c:822

目的

システムカタログ pg_publication を検索し、「全テーブルを対象とする（puballtables が真である）」パブリケーションの OID リストを取得する。

シグネチャまたは形状

List *GetAllTablesPublications(void)

入力と出力

  • **入力**: なし。
  • **出力**: List * 型。条件に合致するパブリケーションの OID を格納したリストを返す。

前提条件と不変条件

  • pg_publication システムカタログに対する AccessShareLock を取得して読み取りを行う。
  • 呼び出し元は、返されたリストに対して適切なメモリ管理（解放）を行う必要がある。

アルゴリズムまたは振る舞い

1. pg_publication リレーションを AccessShareLock で開く。 2. Anum_pg_publication_puballtables 属性が true であることを条件とするスキャンキーを初期化する。 3. systable_beginscan を使用してシステムテーブルのスキャンを開始する。 4. systable_getnext をループで呼び出し、各タプルからパブリケーションの OID を抽出して result リストに追加する。 5. スキャンを終了し、リレーションを閉じてからリストを返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/utils/cache/relcache.c:5854
  • **呼び出し先**:
  • table_open
  • ScanKeyInit
  • systable_beginscan
  • systable_getnext
  • lappend_oid
  • systable_endscan
  • table_close

関連データ構造

  • pg_publication: パブリケーション情報を保持するシステムカタログ。
  • List: PostgreSQL のリスト構造。

失敗モードと境界条件

  • 対象となるパブリケーションが存在しない場合、空のリスト（NIL）を返す。
  • システムカタログへのアクセス権限がない場合、エラーが発生する可能性がある。

保守担当者向けメモ

  • 本関数は puballtables フラグのみを基準に検索を行う。特定のテーブルに関連付けられたパブリケーションを検索する他の関数（GetRelationPublications 等）とは検索ロジックが異なる点に注意すること。
