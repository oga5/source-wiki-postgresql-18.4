ts_copychar_with_len

定義

  • 種別: static inline 関数
  • 定義箇所: src/include/tsearch/ts_locale.h:41

目的

指定された長さのバイト列を、ソースメモリ領域からデスティネーションメモリ領域へコピーし、コピーしたバイト数を返します。

シグネチャまたは形状

static inline int ts_copychar_with_len(void *dest, const void *src, int length)

入力と出力

  • dest: コピー先のメモリ領域へのポインタ。
  • src: コピー元のメモリ領域へのポインタ。
  • length: コピーするバイト数。
  • 戻り値: コピーしたバイト数（入力された length と同一）。

前提条件と不変条件

  • dest および src が指すメモリ領域は、少なくとも length バイト以上のサイズを確保している必要があります。
  • memcpy を使用しているため、メモリ領域の重複がある場合の動作は保証されません。

アルゴリズムまたは振る舞い

1. memcpy を用いて、src から dest へ length バイトをコピーします。 2. 引数として受け取った length をそのまま戻り値として返します。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/backend/tsearch/spell.c (複数箇所で文字列操作に使用)
  • src/include/tsearch/ts_locale.h (インライン関数 ts_copychar_cstr 内から呼び出し)
  • 呼び出し先:
  • memcpy (標準ライブラリ関数)

関連データ構造

  • 特になし（汎用的なメモリコピー関数として機能）。

失敗モードと境界条件

  • length が 0 の場合、何もコピーせず 0 を返します。
  • dest または src が NULL の場合、memcpy の仕様に従い未定義動作となります。

保守担当者向けメモ

  • 本関数は ts_locale.h 内でインライン定義されており、主に全文検索用の辞書やトークン処理における高速な文字コピーを目的としています。
  • ts_copychar_cstr は本関数を内部で呼び出し、pg_mblen_cstr を用いて文字長を自動計算するラッパーとして機能しています。
