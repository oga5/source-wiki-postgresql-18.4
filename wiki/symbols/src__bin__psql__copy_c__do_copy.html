do_copy

定義

  • 種別: function
  • 定義箇所: src/bin/psql/copy.c:267

目的

psql の \copy メタコマンドを実行し、ローカルファイル、パイプ、または標準入出力とデータベースサーバー間でのデータ転送を仲介する。

シグネチャまたは形状

bool do_copy(const char *args);

入力と出力

  • **入力**: args（\copy コマンドに続く引数文字列）。
  • **出力**: 処理が成功した場合は true、失敗した場合は false を返す。

前提条件と不変条件

  • parse_slash_copy が有効な copy_options 構造体を返すこと。
  • pset.copyStream は本関数の実行中のみ一時的に設定され、終了時に NULL にリセットされる。
  • PROGRAM オプション使用時は、シグナルハンドリング（sigpipe）の制御が一時的に変更される。

アルゴリズムまたは振る舞い

1. **解析**: parse_slash_copy を呼び出し、引数を解析してオプションを生成する。 2. **ストリーム準備**: options に基づき、ファイル、パイプ（popen）、または標準入出力（stdin/stdout）をオープンする。 3. **検証**: ファイルの場合、ディレクトリでないことを fstat で確認する。 4. **クエリ構築**: PQExpBuffer を使用して COPY ... FROM STDIN または COPY ... TO STDOUT クエリを生成する。 5. **実行**: pset.copyStream にストリームを割り当て、SendQuery でサーバーへコマンドを送信する。 6. **クリーンアップ**: ストリームをクローズ（パイプの場合は pclose）し、リソースを解放して結果を返す。

呼び出し元と呼び出し先

  • **呼び出し元**: src/bin/psql/command.c:972
  • **呼び出し先**:
  • parse_slash_copy, canonicalize_path_enc, popen, fopen, fstat, initPQExpBuffer, printfPQExpBuffer, appendPQExpBufferStr, SendQuery, pclose, fclose, free_copy_options, pg_log_error, SetShellResultVariables, wait_result_to_str

関連データ構造

  • copy_options: \copy の設定を保持する構造体。
  • PQExpBufferData: クエリ文字列構築用バッファ。
  • pset: psql のグローバル設定構造体。

失敗モードと境界条件

  • 引数の解析失敗時に false を返す。
  • ファイルのオープン失敗、または fstat によるディレクトリ判定失敗時にエラーログを出力し false を返す。
  • SendQuery が失敗した場合、処理を中断する。
  • pclose が非ゼロを返した場合、外部コマンドの実行結果としてエラーを報告する。

保守担当者向けメモ

  • PROGRAM オプション使用時の popen 呼び出し前後で sigpipe トラップの制御を行っている点に注意すること。
  • pset.copyStream を介して SendQuery がデータ転送を行うため、本関数は転送のセットアップと後処理に集中している。
