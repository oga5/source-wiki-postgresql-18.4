r_standard_suffix

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_spanish.c:715

目的

スペイン語のステミング処理において、名詞や形容詞に関連する標準的な接尾辞を単語末尾から特定し、削除または置換を行う。

シグネチャまたは形状

static int r_standard_suffix(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（Snowballの実行環境状態を保持する構造体）
  • 出力: int（成功時は 1、失敗時は 0、エラー発生時は負の値を返す）

前提条件と不変条件

  • z->c（現在のカーソル位置）が単語の末尾を指していること。
  • find_among_b による後方検索が成功すること。
  • 接尾辞の削除・置換操作は、r_R1 または r_R2 で定義された領域境界条件を満たす必要がある。

アルゴリズムまたは振る舞い

1. 単語末尾の文字が特定のビットマスク条件を満たすか確認し、満たさない場合は即座に終了する。 2. a_6 配列を用いて、単語末尾から最長一致する接尾辞を find_among_b で検索する。 3. 検索された接尾辞の種類（among_var）に応じて以下の処理を行う：

  • case 1, 3, 4, 5: r_R2 領域内であることを確認し、接尾辞の削除または指定文字列への置換を行う。
  • case 2, 7, 8, 9: r_R2 領域内の削除に加え、特定の条件（eq_s_b）に合致する場合に限り、追加の接尾辞削除を行う。
  • case 6: r_R1 領域内の削除を行い、さらに特定の接尾辞が続く場合は r_R2 領域内であることを条件に削除を行う。

呼び出し元と呼び出し先

  • 呼び出し元: spanish_UTF_8_stem（直接の呼び出し関係はソースコードの制御フローより推論）。
  • 呼び出し先: find_among_b, r_R1, r_R2, slice_del, slice_from_s, eq_s_b。
  • 注記: 他ファイル（stem_ISO_8859_1_catalan.c 等）にも同名のシンボルが存在するが、本ページは stem_UTF_8_spanish.c 内の定義を対象としている。

関連データ構造

  • SN_env: ステミング処理の環境状態を保持。
  • among: 接尾辞の検索テーブル（a_3, a_4, a_5, a_6）。

失敗モードと境界条件

  • r_R1 や r_R2 の領域条件を満たさない場合、接尾辞の削除は行われない。
  • find_among_b が一致する接尾辞を見つけられない場合、関数は 0 を返す。
  • メモリ操作やスライス処理でエラーが発生した場合、負の戻り値が返される。

保守担当者向けメモ

  • 本関数は自動生成コードであるため、直接の修正は推奨されない。ロジックの変更が必要な場合は、対応するSnowballソース（.sbl）を修正すること。
  • among_var による分岐は、Snowballコンパイラが生成した最適化されたテーブル検索に基づいている。
