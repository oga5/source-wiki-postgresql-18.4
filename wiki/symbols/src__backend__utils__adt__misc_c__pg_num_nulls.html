pg_num_nulls

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/misc.c:161

目的

SQL関数として、可変長引数リスト内のNULL値の個数をカウントして返します。

シグネチャまたは形状

Datum pg_num_nulls(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: PG_FUNCTION_ARGS（SQL呼び出し時の引数情報）。
  • **出力**: int32 型のNULL値の個数。引数が無効な場合はNULLを返します。

前提条件と不変条件

  • count_nulls 関数が正常に引数リストを解析できる必要があります。

アルゴリズムまたは振る舞い

1. count_nulls 関数を呼び出し、引数リストの解析とNULLカウントを実行します。 2. count_nulls が失敗（falseを返却）した場合、PG_RETURN_NULL() を実行して終了します。 3. 成功した場合、カウントされたNULLの数（nulls）を PG_RETURN_INT32(nulls) により返却します。

呼び出し元と呼び出し先

  • **呼び出し元**: SQLインターフェース経由で呼び出されます（具体的なSQLクエリ等の呼び出し元情報はソースから特定できません）。
  • **呼び出し先**: count_nulls 関数を呼び出します。

関連データ構造

  • Datum: PostgreSQLの汎用データ型。
  • PG_FUNCTION_ARGS: 関数呼び出しの引数コンテキスト。

失敗モードと境界条件

  • count_nulls が失敗した場合、関数はSQLレベルのNULLを返します。

保守担当者向けメモ

  • 本関数は count_nulls へのラッパーです。ロジックの変更が必要な場合は、まず count_nulls の実装を確認してください。
