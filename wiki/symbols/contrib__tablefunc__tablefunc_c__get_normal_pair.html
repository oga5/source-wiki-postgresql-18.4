get_normal_pair

定義

  • 種別: function
  • 定義箇所: contrib/tablefunc/tablefunc.c:286

目的

Box-Muller 法を用いて、標準正規分布に従う2つの独立した乱数のペアを生成します。

シグネチャまたは形状

static void get_normal_pair(float8 *x1, float8 *x2)

入力と出力

  • **入力**:
  • x1: 生成された1つ目の乱数を格納する float8 型へのポインタ。
  • x2: 生成された2つ目の乱数を格納する float8 型へのポインタ。
  • **出力**: なし（引数ポインタ経由で値を書き込みます）。

前提条件と不変条件

  • pg_global_prng_state が初期化されている必要があります。
  • 生成される乱数は標準正規分布（平均0、分散1）に従います。

アルゴリズムまたは振る舞い

1. pg_prng_double を用いて、[0, 1) の範囲の乱数 u1, u2 を取得します。 2. これを [-1, 1] の範囲に変換した v1, v2 を計算し、s = v1^2 + v2^2 を求めます。 3. s >= 1.0 または s == 0 となる場合は、条件を満たすまで再試行します（棄却サンプリング）。 4. s が有効な場合、s = sqrt((-2.0 * log(s)) / s) を計算し、x1 = v1 * s, x2 = v2 * s を代入して正規分布乱数を得ます。

呼び出し元と呼び出し先

  • **呼び出し元**: contrib/tablefunc/tablefunc.c 内の normal_rand 関数から呼び出されます。
  • **呼び出し先**: pg_prng_double (PostgreSQL 乱数生成器), sqrt, log (標準数学ライブラリ)。

関連データ構造

  • pg_global_prng_state: 乱数生成器の状態を保持するグローバル変数。

失敗モードと境界条件

  • s == 0 の場合、除算を避けるために x1, x2 に 0 を代入して終了します。
  • s >= 1.0 の場合はループを継続し、有効な値が得られるまで再試行します。

保守担当者向けメモ

  • 本関数は normal_rand 関数専用の内部ヘルパーです。
  • 乱数生成の品質は pg_global_prng_state に依存します。
