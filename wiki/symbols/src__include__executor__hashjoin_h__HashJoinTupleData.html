HashJoinTupleData

定義

  • 種別: struct
  • 定義箇所: src/include/executor/hashjoin.h:78

目的

ハッシュ結合におけるハッシュテーブル内のタプルをリンクするためのヘッダー構造体です。

シグネチャまたは形状

typedef struct HashJoinTupleData
{
    union
    {
        struct HashJoinTupleData *unshared;
        dsa_pointer shared;
    } next;
    uint32 hashvalue;
} HashJoinTupleData;

入力と出力

  • **入力**: 構造体の直後に MinimalTuple 形式のタプルデータが MAXALIGN 境界で配置されます。
  • **出力**: next フィールドを通じて、同一バケット内の後続タプルへの参照を提供します。

前提条件と不変条件

  • タプルデータは構造体の直後に配置される必要があります。
  • next フィールドは、共有メモリ環境では dsa_pointer を、ローカルメモリ環境ではポインタを使用します。

アルゴリズムまたは振る舞い

  • ハッシュテーブルのバケットチェーンを構成するためのリンクリスト要素として機能します。
  • hashvalue には、対応するタプルのハッシュコードが格納され、検索時の比較に使用されます。

呼び出し元と呼び出し先

  • 呼び出し元および呼び出し先の具体的な関数情報は、提供されたソース抜粋には含まれていません。

関連データ構造

  • HashJoinTableData: ハッシュ結合の全体状態を管理する構造体。
  • MinimalTuple: 本構造体の直後に格納される実際のタプルデータ形式。

失敗モードと境界条件

  • HJTUPLE_OVERHEAD マクロ（MAXALIGN(sizeof(HashJoinTupleData))）により、タプルデータのオフセット計算が行われます。このアライメント境界を逸脱したアクセスは未定義動作となります。

保守担当者向けメモ

  • 本構造体は並列ハッシュ結合をサポートするため、dsa_pointer を含む共用体を採用しています。メモリレイアウトを変更する際は、MAXALIGN 境界の維持に注意してください。
