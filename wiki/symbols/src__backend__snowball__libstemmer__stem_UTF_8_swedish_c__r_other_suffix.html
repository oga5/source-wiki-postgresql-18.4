r_other_suffix

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_swedish.c:229

目的

スウェーデン語のステミング処理において、単語末尾の特定の接尾辞を検索し、削除または置換を行う。

シグネチャまたは形状

static int r_other_suffix(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（ステミング実行環境）
  • 出力: 成功時に 1、失敗または処理対象外の場合に 0、内部エラー発生時に負の値を返す。

前提条件と不変条件

  • z->I[1]（領域 R1 の開始位置）以降の文字が処理対象となる。
  • z->c が z->I[1] 未満の場合は即座に終了する。
  • 処理中、一時的に z->lb を z->I[1] に設定し、処理後に復元する。

アルゴリズムまたは振る舞い

1. **境界チェック**: 現在位置が R1 領域内にあるか確認する。 2. **マッチング**: find_among_b を使用し、配列 a_2 に定義された接尾辞を末尾から検索する。 3. **条件分岐**: マッチした接尾辞の種類（among_var）に応じて以下の処理を行う。

  • case 1: slice_del を呼び出し、該当部分を削除する。
  • case 2: g_ost_ending グループに含まれないことを確認後、slice_from_s で文字列を置換する。
  • case 3: slice_from_s で文字列を置換する。

    • 4. **終了**: 変更を反映し、成功値 1 を返す。

呼び出し元と呼び出し先

  • 呼び出し元: swedish_UTF_8_stem（src/backend/snowball/libstemmer/stem_UTF_8_swedish.c）
  • 呼び出し先: find_among_b, slice_del, slice_from_s, in_grouping_b_U

関連データ構造

  • SN_env: ステミング環境を保持する構造体。
  • a_2: 接尾辞マッチングに使用される among 構造体の配列。
  • g_ost_ending: case 2 の条件判定に使用される文字グループ。

失敗モードと境界条件

  • z->c が R1 領域より前にある場合、0 を返す。
  • find_among_b で接尾辞がマッチしない場合、0 を返す。
  • slice_del や slice_from_s が負の値を返した場合、そのエラーコードをそのまま伝播させる。

保守担当者向けメモ

  • 本関数は Snowball コンパイラによる自動生成コードである。ロジックの変更が必要な場合は、直接編集せず元の .sbl ソースファイルを修正すること。
  • among_var の各ケースは、スウェーデン語の文法ルールに基づいた接尾辞の正規化を定義している。
