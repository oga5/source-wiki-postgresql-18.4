GetCurrentTimestamp

定義

  • 種別: function
  • 定義箇所: src/backend/utils/adt/timestamp.c:1644

目的

システムクロックから現在時刻を取得し、PostgreSQLの内部表現である TimestampTz 型に変換して返す。

シグネチャまたは形状

TimestampTz GetCurrentTimestamp(void)

入力と出力

  • **入力**: なし（void）
  • **出力**: TimestampTz 型（マイクロ秒単位の int64 値）

前提条件と不変条件

  • TimestampTz はマイクロ秒単位の精度で保持される。
  • POSTGRES_EPOCH_JDATE と UNIX_EPOCH_JDATE の差分に基づき、UNIXエポックからPostgreSQLエポックへのオフセット調整が行われる。

アルゴリズムまたは振る舞い

1. gettimeofday を呼び出し、現在のシステム時刻を struct timeval に取得する。 2. tv_sec（秒）からエポック差分を減算し、PostgreSQL基準の秒数に変換する。 3. 秒数をマイクロ秒に換算し、tv_usec（マイクロ秒）を加算して最終的な TimestampTz 値を算出する。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/dblink/dblink.c
  • contrib/pg_prewarm/autoprewarm.c
  • contrib/pg_stat_statements/pg_stat_statements.c
  • contrib/postgres_fdw/connection.c
  • **呼び出し先**:
  • gettimeofday（システムコール）

関連データ構造

  • TimestampTz: 日時を保持する int64 型。
  • struct timeval: システム時刻を保持する標準構造体。

失敗モードと境界条件

  • gettimeofday が失敗した場合の挙動については、本ソースコード内では明示的なエラーハンドリングは行われていない。
  • システム時刻が極端な値である場合、int64 の範囲を超える可能性がある。

保守担当者向けメモ

  • 本関数はシステム時刻に直接依存する。タイムゾーンの考慮は含まれておらず、UTCベースの内部表現を生成する。
  • TimestampTz の計算において、エポック間の変換定数が正しく適用されていることを確認すること。
