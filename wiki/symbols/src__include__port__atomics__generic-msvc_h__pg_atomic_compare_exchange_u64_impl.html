pg_atomic_compare_exchange_u64_impl

定義

  • 種別: function
  • 定義箇所: src/include/port/atomics/generic-msvc.h:82

目的

MSVC環境において、64ビット整数に対するアトミックな比較交換（Compare-and-Swap）操作を提供します。

シグネチャまたは形状

static inline bool pg_atomic_compare_exchange_u64_impl(volatile pg_atomic_uint64 *ptr, uint64 *expected, uint64 newval)

入力と出力

  • **入力**:
  • ptr: 操作対象となる pg_atomic_uint64 構造体へのポインタ。
  • expected: 期待値が格納された変数へのポインタ。操作後、実際の現在の値で更新されます。
  • newval: 交換先の新しい値。
  • **出力**:
  • bool: 比較が成功（ptr の値が expected と一致）した場合は true、失敗した場合は false を返します。

前提条件と不変条件

  • ptr はアトミック操作のために適切にアライメントされている必要があります。
  • 本関数は _InterlockedCompareExchange64 を利用するため、64ビットビルド環境での使用を前提としています。

アルゴリズムまたは振る舞い

1. _InterlockedCompareExchange64 を呼び出し、ptr->value と *expected を比較します。 2. 一致する場合、ptr->value を newval に更新します。 3. 比較前の ptr->value の値を current に保持します。 4. current と元の *expected を比較し、結果を ret に格納します。 5. *expected を current で更新します。 6. ret を返します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/include/port/atomics.h および src/include/port/atomics/generic.h 内の汎用アトミック関数から呼び出されます。
  • **呼び出し先**: MSVC 組み込み関数 _InterlockedCompareExchange64 を呼び出します。

関連データ構造

  • pg_atomic_uint64: アトミック操作対象の値を保持する構造体。

失敗モードと境界条件

  • ptr が指すメモリが適切にアライメントされていない場合、_InterlockedCompareExchange64 の仕様により未定義動作を引き起こす可能性があります。
  • 比較が失敗した場合、*expected は ptr->value の現在の値で上書きされます。

保守担当者向けメモ

  • 本関数は MSVC 特有の実装です。他のプラットフォーム向けの実装については、同ディレクトリ内の arch-*.h や generic-*.h を参照してください。
  • _InterlockedCompareExchange64 はメモリバリアを内包する点に留意してください。
