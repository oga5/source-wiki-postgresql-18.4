PGTYPESdate_from_timestamp

定義

  • 種別: function
  • 定義箇所: src/interfaces/ecpg/pgtypeslib/datetime.c:30

目的

timestamp 型の値を date 型（日付）に変換します。

シグネチャまたは形状

date PGTYPESdate_from_timestamp(timestamp dt)

入力と出力

  • **入力**: timestamp dt（変換対象のタイムスタンプ値）
  • **出力**: date（変換後の日付値。内部的には整数として表現される）

前提条件と不変条件

  • 入力 dt が有限値（!TIMESTAMP_NOT_FINITE(dt)）である場合にのみ、計算が行われます。
  • 基準となる日付計算は、dt をマイクロ秒単位から日数単位へ変換することで行われます。

アルゴリズムまたは振る舞い

1. dDate を 0 で初期化します。 2. 入力された dt が有限値かどうかを判定します。 3. 有限値である場合、dt を USECS_PER_DAY（1日あたりのマイクロ秒数）で除算し、その結果を dDate に代入します。 4. 有限値でない場合は、初期値の 0 が返されます。 5. 最終的に計算された dDate を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:

  • **呼び出し先**:
  • ソースコード上、明示的な関数呼び出しは行われていません（マクロ TIMESTAMP_NOT_FINITE および定数 USECS_PER_DAY を使用）。

関連データ構造

  • date: ECPG内で使用される日付型。
  • timestamp: 変換元の時刻型。

失敗モードと境界条件

  • TIMESTAMP_NOT_FINITE(dt) が真となる場合（無限大のタイムスタンプなど）、計算は行われず 0 が返されます。

保守担当者向けメモ

  • 本関数は dt を単純にマイクロ秒から日数へ変換するロジックです。日付の基準（エポック）については、datetime.c のファイル概要にある通り、2000年1月1日を基準としたオフセット値として扱われる点に留意してください。
