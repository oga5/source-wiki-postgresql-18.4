SendCopyDone

定義

  • 種別: function
  • 定義箇所: src/backend/backup/basebackup_copy.c:330

目的

クライアントに対して、COPYプロトコルにおけるデータ転送の終了（CopyDone）を通知する。

シグネチャまたは形状

static void SendCopyDone(void)

入力と出力

  • 入力: なし
  • 出力: なし（ネットワーク経由で CopyDone メッセージを送信）

前提条件と不変条件

  • 本関数を呼び出す前に、COPYモードが開始されている必要がある。

アルゴリズムまたは振る舞い

pq_putemptymessage を使用して、PqMsg_CopyDone メッセージをクライアントへ送信する。これにより、COPYストリームの終了をクライアントに伝える。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/backup/basebackup_copy.c 内の bbsink_copystream_end_backup（推測される文脈）および同ファイル内の他の終了処理箇所。
  • 呼び出し先: pq_putemptymessage（libpqのメッセージ送信関数）。

関連データ構造

  • PqMsg_CopyDone: 送信されるメッセージタイプを定義する定数。

失敗モードと境界条件

  • ネットワークエラーが発生した場合、pq_putemptymessage 内部でエラーが報告される可能性がある。

保守担当者向けメモ

  • 本関数は basebackup_copy.c におけるCOPYプロトコル終了処理の標準的な手段である。
  • bbsink インターフェースの終了処理フローの一部として利用される。
