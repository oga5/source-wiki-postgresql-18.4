InstrInit

定義

  • 種別: function
  • 定義箇所: src/backend/executor/instrument.c:57

目的

クエリ実行計画ノードの統計収集用構造体（Instrumentation）を初期化し、計測オプションを設定します。

シグネチャまたは形状

void InstrInit(Instrumentation *instr, int instrument_options)

入力と出力

  • **入力**:
  • instr: 初期化対象となる Instrumentation 構造体へのポインタ。
  • instrument_options: 計測対象を指定するビットマスク（INSTRUMENT_BUFFERS, INSTRUMENT_WAL, INSTRUMENT_TIMER）。
  • **出力**: なし（instr が指すメモリ領域が直接更新されます）。

前提条件と不変条件

  • instr は有効な Instrumentation 構造体を指している必要があります。
  • 実行後、instr の全メンバはゼロクリアされ、指定されたオプションに基づいてフラグが設定されます。

アルゴリズムまたは振る舞い

1. memset を使用して、instr が指す構造体全体を 0 で初期化します。 2. instrument_options ビットマスクを評価し、以下のフラグを instr に反映します。

  • need_bufusage: INSTRUMENT_BUFFERS が含まれる場合。
  • need_walusage: INSTRUMENT_WAL が含まれる場合。
  • need_timer: INSTRUMENT_TIMER が含まれる場合。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/executor/execParallel.c:811（並列クエリ実行時の初期化）
  • **呼び出し先**:
  • 標準ライブラリの memset を使用。

関連データ構造

  • Instrumentation: 実行統計を保持する主要構造体。

失敗モードと境界条件

  • instr が NULL の場合、セグメンテーション違反が発生します。

保守担当者向けメモ

  • 本関数は Instrumentation 構造体のメモリレイアウトに依存します。構造体定義に変更を加える際は、memset による初期化が意図通りか確認してください。
