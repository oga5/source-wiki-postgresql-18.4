avlUpdateHeight

定義

  • 種別: function
  • 定義箇所: src/bin/psql/crosstabview.c:471

目的

AVL木のノードの height プロパティを、左右の子ノードの高さに基づいて再計算し、更新する。

シグネチャまたは形状

static void avlUpdateHeight(avl_node *n)

入力と出力

  • 入力: avl_node *n （高さを更新する対象のノード）
  • 出力: なし（n->height を直接更新する）

前提条件と不変条件

  • 引数 n は有効な avl_node 構造体へのポインタであること。
  • n->children[0] および n->children[1] が有効なノード（または NULL ではない構造）を指していることが前提となっている（コード内で直接 height メンバにアクセスしているため）。

アルゴリズムまたは振る舞い

1. 左右の子ノードの height を比較する。 2. 大きい方の高さに 1 を加算した値を、現在のノード n の height に代入する。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/bin/psql/crosstabview.c:472
  • src/bin/psql/crosstabview.c:488
  • src/bin/psql/crosstabview.c:520
  • 呼び出し先: なし（本関数は計算のみを行う）

関連データ構造

  • avl_node: AVL木のノードを定義する構造体。height および children 配列を保持する。

失敗モードと境界条件

  • n が NULL の場合、または n->children が NULL を指している場合、セグメンテーション違反が発生する可能性がある。

保守担当者向けメモ

  • 本関数は AVL 木の平衡維持処理の一部として呼び出される。
  • avl_node 構造体の定義において、children 配列の要素が常に有効なノードを指すか、あるいは NULL チェックが必要かを確認すること。現在の実装では NULL チェックなしで height にアクセスしている。
