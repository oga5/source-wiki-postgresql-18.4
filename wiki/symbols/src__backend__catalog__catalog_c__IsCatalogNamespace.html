IsCatalogNamespace

定義

  • 種別: function
  • 定義箇所: src/backend/catalog/catalog.c:242

目的

指定された名前空間（OID）が、システムカタログを格納する pg_catalog 名前空間であるかを判定します。

シグネチャまたは形状

bool IsCatalogNamespace(Oid namespaceId)

入力と出力

  • **入力**: namespaceId (Oid) - 判定対象の名前空間OID。
  • **出力**: bool - namespaceId が PG_CATALOG_NAMESPACE と一致する場合は true、それ以外は false を返します。

前提条件と不変条件

  • 本関数はカタログアクセスを行わないため、ロック取得中や再帰的な呼び出しにおいても安全に使用可能です。

アルゴリズムまたは振る舞い

  • 渡された namespaceId が定数 PG_CATALOG_NAMESPACE と等しいかを比較する単純な判定を行います。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/catalog/heap.c
  • src/backend/catalog/pg_publication.c
  • src/backend/commands/tablecmds.c
  • src/backend/utils/cache/relcache.c
  • **呼び出し先**:
  • 外部関数への依存はありません。

関連データ構造

  • Oid: PostgreSQLのオブジェクト識別子型。

失敗モードと境界条件

  • 入力値が有効なOIDであるかどうかの検証は行いません。無効なOIDが渡された場合、単に false を返します。

保守担当者向けメモ

  • 本関数はシステムカタログの判定ロジックの一部であり、src/include/catalog/catalog.h でプロトタイプ宣言されています。
  • 類似の判定関数（IsToastNamespace など）と併用されることが多いため、変更時は関連する名前空間判定ロジックとの整合性に注意してください。
