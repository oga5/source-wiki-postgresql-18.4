CheckFIPSMode

定義

  • 種別: function
  • 定義箇所: contrib/pgcrypto/openssl.c:843

目的

実行中の OpenSSL ライブラリにおいて、FIPS モードが有効化されているかどうかを判定する。

シグネチャまたは形状

bool CheckFIPSMode(void)

入力と出力

  • **入力**: なし
  • **出力**: bool 型。FIPS モードが有効な場合は true、それ以外は false を返す。

前提条件と不変条件

  • OpenSSL のバージョンに応じた適切な API（EVP_default_properties_is_fips_enabled または FIPS_mode）が利用可能であること。

アルゴリズムまたは振る舞い

1. コンパイル時の OpenSSL バージョン（OPENSSL_VERSION_NUMBER）をチェックする。 2. OpenSSL 3.0 以降の場合: EVP_default_properties_is_fips_enabled(NULL) を呼び出し、結果を取得する。 3. OpenSSL 3.0 未満の場合: FIPS_mode() を呼び出し、結果を取得する。 4. 取得した値が 1（有効）であるかを判定し、その真偽値を返す。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/pgcrypto/openssl.c:885
  • contrib/pgcrypto/pgcrypto.c:492
  • **呼び出し先**:
  • OpenSSL API: EVP_default_properties_is_fips_enabled (OpenSSL 3.0+) または FIPS_mode (OpenSSL 1.x)

関連データ構造

  • なし

失敗モードと境界条件

  • OpenSSL のバージョンが 3.0 未満かつ FIPS 対応でない環境での挙動は、リンクされている OpenSSL ライブラリの実装に依存する。

保守担当者向けメモ

  • 本関数は OpenSSL のバージョン差異を吸収するためのラッパーである。将来的に OpenSSL の API が変更された場合は、条件分岐の更新が必要となる。
