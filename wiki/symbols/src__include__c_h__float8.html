float8

定義

  • 種別: typedef
  • 定義箇所: src/include/c.h:609

目的

PostgreSQL における 8 バイト（倍精度）浮動小数点数を表現するための標準的な型定義です。

シグネチャまたは形状

typedef double float8;

入力と出力

  • 入力: C 言語の double 型と互換性のある数値。
  • 出力: double 型として評価される値。

前提条件と不変条件

  • プラットフォームの double 型が 8 バイトであることを前提としています。
  • PostgreSQL のデータ構造において、浮動小数点演算の標準的な型として使用されます。

アルゴリズムまたは振る舞い

  • C 言語の double 型のエイリアスとして機能します。
  • 算術演算や比較演算は、コンパイラおよびハードウェアの double 型の仕様に従います。

呼び出し元と呼び出し先

  • 呼び出し元（参照箇所）:
  • contrib/btree_gin/btree_gin.c: GIN インデックスのサポート関数定義。
  • contrib/btree_gist/btree_cash.c, btree_date.c, btree_float4.c: GiST インデックスのサポート関数内での変数宣言。
  • contrib/btree_gist/btree_float8.c: 比較演算（>、>=、==、<=、<）の実装において、ポインタ経由で値を参照し比較を行う。
  • 呼び出し先:
  • C 言語標準の浮動小数点演算および型キャスト操作。

関連データ構造

  • 特になし（基本型として定義）。

失敗モードと境界条件

  • double 型の仕様に依存する（オーバーフロー、アンダーフロー、NaN、無限大の扱いなど）。
  • ポインタ経由でアクセスする際は、アライメントが適切であることを前提とします。

保守担当者向けメモ

  • 本シンボルは c.h で定義されており、PostgreSQL 全体で一貫した浮動小数点型として使用されます。
  • 変更を加える際は、PostgreSQL の全モジュールに影響を与えるため注意してください。
