makeString

定義

  • 種別: function
  • 定義箇所: src/backend/nodes/value.c:62

目的

文字列データを保持する String 型のノードを生成し、初期化するファクトリ関数です。

シグネチャまたは形状

String *makeString(char *str)

入力と出力

  • **入力**: char *str - ノードに格納する文字列へのポインタ。
  • **出力**: String * - 初期化された String ノードへのポインタ。

前提条件と不変条件

  • 引数 str は、呼び出し元によって palloc 等で適切にメモリ確保されている必要があります。
  • 生成されたノードの寿命は、格納される文字列の寿命と同期することが期待されます。

アルゴリズムまたは振る舞い

1. makeNode(String) マクロを使用して、PostgreSQL のメモリコンテキスト上に String 構造体用のメモリを確保します。 2. 確保した構造体の sval メンバに、引数として渡された str を代入します。 3. 初期化済みのノードポインタを返却します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • contrib/file_fdw/file_fdw.c
  • contrib/postgres_fdw/postgres_fdw.c
  • src/backend/access/common/reloptions.c
  • src/backend/catalog/namespace.c
  • src/backend/catalog/objectaddress.c
  • **呼び出し先**:
  • makeNode (マクロ)

関連データ構造

  • String: src/include/nodes/value.h で定義される、文字列を保持するためのノード構造体。

失敗モードと境界条件

  • makeNode 内部でメモリ不足が発生した場合、PostgreSQL のエラーハンドリング機構によりアボートされます。
  • 引数 str が NULL の場合、そのまま sval に代入されます。

保守担当者向けメモ

  • 本関数は文字列のコピーを作成しません。引数として渡されたポインタを直接保持するため、呼び出し元でのメモリ管理（所有権の移譲）に注意してください。
