digest_info

定義

  • 種別: struct
  • 定義箇所: contrib/pgcrypto/pgp.c:53

目的

OpenPGP で使用されるダイジェスト（ハッシュ）アルゴリズムの名称と、内部で使用される識別コードを紐付けるためのメタデータ構造体です。

シグネチャまたは形状

struct digest_info
{
	const char *name;
	int			code;
};

入力と出力

  • name: アルゴリズムの名称を表す文字列（例: "sha1"）。
  • code: 内部処理で使用される整数型の識別子。

前提条件と不変条件

  • name は pgcrypto モジュール内で一意である必要があります。
  • code は px レイヤーのハッシュ関数と整合している必要があります。

アルゴリズムまたは振る舞い

この構造体は、文字列ベースのアルゴリズム指定を内部的な数値コードへ変換するためのルックアップテーブル（digest_list）の要素として使用されます。

呼び出し元と呼び出し先

  • 呼び出し元: contrib/pgcrypto/pgp.c 内の digest_list 定義（68行目）および、アルゴリズム検索を行う関数（106行目、128行目付近）から参照されています。
  • 呼び出し先: 本構造体はデータ保持用であり、関数を呼び出すことはありません。

関連データ構造

  • digest_list: digest_info 型の静的配列。サポートされる全ダイジェストアルゴリズムのカタログを保持します。

失敗モードと境界条件

  • 該当する name が digest_list に存在しない場合、検索処理は失敗します。

保守担当者向けメモ

  • 新しいダイジェストアルゴリズムを追加する場合は、digest_list に本構造体のインスタンスを追加してください。
  • 実際のハッシュ計算ロジックは本構造体には含まれず、px レイヤーに委譲されています。
