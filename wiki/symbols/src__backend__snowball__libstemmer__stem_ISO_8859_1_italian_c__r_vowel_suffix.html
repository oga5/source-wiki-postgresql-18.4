r_vowel_suffix

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_italian.c:926

目的

イタリア語のステミング処理において、単語末尾に残存する特定の母音や接尾辞を削除し、語幹を正規化します。

シグネチャまたは形状

static int r_vowel_suffix(struct SN_env * z)

入力と出力

  • **入力**: SN_env * z（Snowball ステミング環境構造体）
  • **出力**: int 型。成功時は 1 を返し、エラー発生時は負の値を返します。

前提条件と不変条件

  • z は有効な SN_env 構造体を指している必要があります。
  • 内部で呼び出される r_RV 関数により、対象の文字が RV 領域内にあることが削除の条件となります。

アルゴリズムまたは振る舞い

本関数は以下の2つの主要な削除ロジックを順次実行します。

1. **母音および 'i' の削除**:

  • g_AEIO グループ（母音）に該当する文字が末尾にある場合、RV 領域内であれば削除します。
  • 続いて、末尾が 'i' である場合、これも RV 領域内であれば削除します。

    • 2. **'h' を伴う接尾辞の削除**:

  • 末尾が 'h' であり、その直前が g_CG グループ（'c' または 'g'）である場合、RV 領域内であれば削除します。

各ステップで条件を満たさない場合は lab0 または lab1 へジャンプし、処理をスキップします。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/backend/snowball/libstemmer/stem_ISO_8859_1_italian.c
  • src/backend/snowball/libstemmer/stem_UTF_8_italian.c
  • src/backend/snowball/libstemmer/stem_UTF_8_romanian.c
  • **呼び出し先**:
  • r_RV (領域判定)
  • slice_del (文字列削除)
  • in_grouping_b (文字グループ判定)

関連データ構造

  • SN_env: ステミング処理の環境状態を保持する構造体。
  • g_AEIO, g_CG: 文字グループを定義する静的配列（ソースファイル内で定義）。

失敗モードと境界条件

  • r_RV や slice_del が負の値を返した場合、その値をそのまま呼び出し元へ伝播させます。
  • z->c（現在のカーソル位置）が z->lb（下限）以下になる場合、または対象文字が一致しない場合は削除を行わずに処理を継続します。
