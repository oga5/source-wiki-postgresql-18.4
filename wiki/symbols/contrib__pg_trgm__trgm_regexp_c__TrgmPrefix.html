TrgmPrefix

定義

  • 種別: struct
  • 定義箇所: contrib/pg_trgm/trgm_regexp.c:291

目的

正規表現のグラフ変換プロセスにおいて、トリグラムの接頭辞（Prefix）を表現し、状態間の包含関係やマッチング可能性を判定するために使用されます。

シグネチャまたは形状

typedef struct
{
	TrgmColor	colors[2];
} TrgmPrefix;

入力と出力

  • 入力: TrgmColor 型の要素を2つ保持する構造体。
  • 出力: なし（データ保持用構造体）。

前提条件と不変条件

  • colors 配列は、正規表現のグラフ走査における特定の状態遷移や色付けに関連する情報を保持します。

アルゴリズムまたは振る舞い

  • 本構造体は主に prefixContains 関数内で、2つの TrgmPrefix インスタンス間の包含関係を比較するために利用されます。

呼び出し元と呼び出し先

  • 呼び出し元: contrib/pg_trgm/trgm_regexp.c 内の prefixContains 関数が引数として受け取ります。また、同ファイル内の関数内でローカル変数として宣言されています。
  • 呼び出し先: 構造体自体はデータ保持用であり、直接的な関数呼び出しは行いません。

関連データ構造

  • TrgmColor: TrgmPrefix の構成要素として使用される型。

失敗モードと境界条件

  • 構造体定義自体に起因する失敗モードはありません。メモリ上の配置や TrgmColor の値の妥当性は、操作する関数側に依存します。

保守担当者向けメモ

  • 本構造体は正規表現からトリグラムへの変換ロジックにおいて、グラフの簡略化や最適化の判断基準として使用されます。変更を加える際は、prefixContains による包含判定ロジックへの影響を考慮してください。
