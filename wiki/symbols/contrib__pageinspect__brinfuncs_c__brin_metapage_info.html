brin_metapage_info

定義

  • 種別: function
  • 定義箇所: contrib/pageinspect/brinfuncs.c:342

目的

BRIN（Block Range Index）インデックスのメタページの内容を抽出し、SQLで参照可能な形式で返します。

シグネチャまたは形状

Datum brin_metapage_info(PG_FUNCTION_ARGS)

入力と出力

  • **入力**: bytea 型の生ページデータ（PG_GETARG_BYTEA_P(0)）。
  • **出力**: メタ情報を含む複合型（magic、version、pagesPerRange、lastRevmapPage）の Datum。

前提条件と不変条件

  • 実行にはスーパーユーザー権限が必要です。
  • 入力されたページは verify_brin_page によって BRIN_PAGETYPE_META 型であることが検証されます。
  • ページが新規（PageIsNew）の場合、結果は NULL となります。

アルゴリズムまたは振る舞い

1. 権限チェックを行い、スーパーユーザーでない場合はエラーを発生させます。 2. verify_brin_page を呼び出し、ページが有効なメタページであることを確認します。 3. ページが空であれば NULL を返します。 4. 呼び出し元の期待する戻り値の型（複合型）を取得し、BlessTupleDesc で準備します。 5. ページの内容を BrinMetaPageData 構造体としてキャストし、以下の値を抽出します。

  • brinMagic: 16進数文字列
  • brinVersion: 整数
  • pagesPerRange: 整数
  • lastRevmapPage: 64ビット整数

    • 6. 抽出した値を heap_form_tuple でタプルに変換し、結果として返します。

呼び出し元と呼び出し先

  • **呼び出し元**: SQLインターフェース（PG_FUNCTION_INFO_V1 マクロによる登録）。
  • **呼び出し先**: superuser, verify_brin_page, get_call_result_type, BlessTupleDesc, PageGetContents, psprintf, CStringGetTextDatum, Int32GetDatum, Int64GetDatum, heap_form_tuple, HeapTupleGetDatum。

関連データ構造

  • BrinMetaPageData: メタページの物理レイアウトを定義する構造体。

失敗モードと境界条件

  • スーパーユーザー権限がない場合、ERRCODE_INSUFFICIENT_PRIVILEGE でエラーとなります。
  • 戻り値の型が複合型でない場合、elog によりエラーとなります。
  • verify_brin_page が失敗した場合、内部でエラーが発生します。

保守担当者向けメモ

  • 本関数は生のページデータを直接操作するため、インデックスの破損調査やデバッグ用途に限定してください。
  • 戻り値の構造（values 配列の順序と型）を変更する場合は、対応する SQL 側の型定義も更新する必要があります。
