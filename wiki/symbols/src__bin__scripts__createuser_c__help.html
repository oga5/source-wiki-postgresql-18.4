help

定義

  • 種別: function
  • 定義箇所: src/bin/scripts/createuser.c:413

目的

createuser コマンドの利用方法、利用可能なオプション、および接続設定に関する情報を標準出力に表示する。

シグネチャまたは形状

void help(const char *progname)

入力と出力

  • **入力**: progname (実行ファイル名を示す文字列)
  • **出力**: なし（標準出力へのメッセージ表示のみ）

前提条件と不変条件

  • progname は有効な文字列ポインタであること。
  • printf が標準出力に対して書き込み可能であること。

アルゴリズムまたは振る舞い

1. progname を使用してコマンドの概要と使用法（Usage）を表示する。 2. createuser がサポートする各オプション（ロール属性、接続オプション等）の一覧をカテゴリ別に表示する。 3. PACKAGE_BUGREPORT、PACKAGE_NAME、PACKAGE_URL を用いて、バグ報告先および公式ホームページの情報を表示する。

呼び出し元と呼び出し先

  • **呼び出し元**: 提供されたソース excerpt 内には、本関数を呼び出している箇所は存在しません。
  • **呼び出し先**: 標準ライブラリの printf を呼び出します。
  • **注意**: ツールが提示した contrib/oid2name/oid2name.c 内の help 関数は同名ですが、本ファイル（src/bin/scripts/createuser.c）の help とは別個の実装です。

関連データ構造

  • なし

失敗モードと境界条件

  • 標準出力がクローズされている場合、出力は失敗しますが、本関数自体はエラーコードを返しません。

保守担当者向けメモ

  • 本関数は createuser のオプション定義と同期している必要があります。新しいオプションを main 関数に追加した際は、必ず本関数の出力にも反映してください。
