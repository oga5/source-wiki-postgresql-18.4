gbt_intv_dist

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_interval.c:82

目的

2つの Interval 型の値の間の距離を計算し、float8 型として返します。

シグネチャまたは形状

static float8 gbt_intv_dist(const void *a, const void *b, FmgrInfo *flinfo)

入力と出力

  • **入力**:
  • a, b: 比較対象の Interval 型へのポインタ（void * として渡される）。
  • flinfo: 関数管理情報（本関数内では未使用）。
  • **出力**:
  • 2つのインターバル間の絶対距離を float8 型で返します。

前提条件と不変条件

  • 入力ポインタ a および b は、有効な Interval 構造体を指している必要があります。
  • intr2num 関数が、Interval 型を距離計算可能な数値表現に変換できることを前提としています。

アルゴリズムまたは振る舞い

1. intr2num を使用して、入力された2つの Interval 値をそれぞれ数値に変換します。 2. 変換された数値の差を求め、fabs 関数を用いてその絶対値を算出します。 3. 得られた絶対値を距離として返します。

呼び出し元と呼び出し先

  • **呼び出し元**: ソースコード内の直接的な呼び出し元は、contrib/btree_gist/btree_interval.c:108 で参照されています。
  • **呼び出し先**:
  • intr2num (内部変換関数)
  • fabs (標準ライブラリ)

関連データ構造

  • Interval: PostgreSQL の時間間隔型。

失敗モードと境界条件

  • 入力ポインタが NULL の場合、未定義の動作を引き起こす可能性があります。
  • Interval の値が極端に大きい場合、intr2num での変換時や差分計算時にオーバーフローが発生する可能性があります。

保守担当者向けメモ

  • 本関数は btree_gist の距離計算ロジックの一部です。距離計算の精度や定義を変更する場合は、intr2num の実装と併せて確認してください。
