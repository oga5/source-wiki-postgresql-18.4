r_en_ending

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_ISO_8859_1_dutch.c:366

目的

オランダ語のステミング処理において、単語末尾の "-en" 関連の接尾辞を除去し、必要に応じて重複文字を解消する。

シグネチャまたは形状

static int r_en_ending(struct SN_env * z)

入力と出力

  • 入力: struct SN_env * z （Snowball ステミング環境へのポインタ）
  • 出力: int （成功時に 1、失敗または条件不一致時に 0 または負の値を返す）

前提条件と不変条件

  • r_R1(z) が呼び出され、現在のカーソル位置が R1 領域内にあることが条件となる。
  • g_v（母音グループ）に基づいた文字チェックが行われる。

アルゴリズムまたは振る舞い

1. r_R1(z) を実行し、R1 領域外であれば終了する。 2. 現在のカーソル位置から後方へ向かって母音（g_v）を検索する。 3. 母音が見つかった場合、その直後に特定の文字列（s_10）が存在しないことを確認する。 4. 条件を満たす場合、slice_del(z) で接尾辞を削除する。 5. 最後に r_undouble(z) を呼び出し、重複文字の解消を行う。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_ISO_8859_1_dutch.c および src/backend/snowball/libstemmer/stem_UTF_8_dutch.c 内のステミング処理フローから参照されている。
  • 呼び出し先: r_R1, out_grouping_b, eq_s_b, slice_del, r_undouble。

関連データ構造

  • struct SN_env: ステミング環境を保持する構造体。
  • g_v: 母音を定義するビットマップ配列。
  • s_10: 比較対象となる文字列定数。

失敗モードと境界条件

  • r_R1 が 0 以下を返した場合、処理を中断する。
  • slice_del が負の値を返した場合、エラーとして伝播させる。
  • 母音が見つからない、または s_10 が存在する場合、削除処理は行われない。
