ForBothCellState

定義

  • 種別: struct
  • 定義箇所: src/include/nodes/pg_list.h:86

目的

2つの List を同時に走査する際の反復状態を保持するために使用されます。

シグネチャまたは形状

typedef struct ForBothCellState
{
	const List *l1;
	const List *l2;
	int			i1;
	int			i2;
} ForBothCellState;

入力と出力

  • **入力**: 走査対象となる2つの List ポインタ。
  • **出力**: 構造体内部の i1, i2 が現在の走査インデックスとして更新されます。

前提条件と不変条件

  • l1 および l2 は有効な List 構造体（または NIL）を指す必要があります。
  • 構造体内の i1, i2 は、対応するリストの要素インデックスを追跡します。

アルゴリズムまたは振る舞い

  • 2つのリストを並行して走査するためのイテレータとして機能します。
  • src/include/nodes/pg_list.h 内の for マクロ（foreach 系）の内部実装において、ループの状態管理に使用されます。

呼び出し元と呼び出し先

  • **呼び出し元**: src/include/nodes/pg_list.h 内の for ループマクロ（for(ForBothCellState ...)）から利用されます。
  • **呼び出し先**: 直接的な関数呼び出しは行いませんが、リストの走査ロジックの一部として組み込まれます。

関連データ構造

  • List: 走査対象となるリスト構造体。
  • ForEachState, ForThreeState 等: 同様の目的を持つ他の反復状態構造体。

失敗モードと境界条件

  • リストが NIL の場合、またはリストの長さが異なる場合の挙動は、この構造体を利用するマクロ側の実装に依存します。

保守担当者向けメモ

  • 本構造体は foreach マクロ等の反復処理の基盤です。構造体のメンバを変更する場合は、関連するマクロ定義（pg_list.h 内）との整合性に注意してください。
