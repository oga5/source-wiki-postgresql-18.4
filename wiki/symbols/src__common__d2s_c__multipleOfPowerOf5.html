multipleOfPowerOf5

定義

  • 種別: function
  • 定義箇所: src/common/d2s.c:94

目的

指定された整数 value が 5 の p 乗で割り切れるかどうかを判定します。

シグネチャまたは形状

static inline bool multipleOfPowerOf5(const uint64 value, const uint32 p)

入力と出力

  • **入力**:
  • value: 判定対象の 64 ビット符号なし整数。
  • p: 5 の指数（5 の p 乗で割り切れるかを確認する）。
  • **出力**:
  • bool: value が 5 の p 乗の倍数であれば true、そうでなければ false を返します。

前提条件と不変条件

  • pow5Factor(value) が value を割り切る 5 の最大指数を返すことを前提としています。

アルゴリズムまたは振る舞い

1. pow5Factor(value) を呼び出し、value を割り切る 5 の最大指数を取得します。 2. 取得した指数が p 以上であるかを比較し、その結果を返します。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/common/d2s.c 内の浮動小数点数から 10 進数への変換処理（mv や mv - 1 - mmShift 等の計算結果に対して呼び出し）。
  • **呼び出し先**:
  • pow5Factor（同ファイル内の関数）。

関連データ構造

  • なし。

失敗モードと境界条件

  • value が 0 の場合、pow5Factor の実装に依存します。
  • p が 0 の場合、任意の整数は 5 の 0 乗（1）で割り切れるため、常に true を返します。

保守担当者向けメモ

  • 本関数は static inline で定義されており、パフォーマンスを考慮して pow5Factor の結果を直接評価しています。
  • src/common/f2s.c にも同名の関数が存在しますが、引数の型が uint32 である点が異なります。変更時は両方の実装への影響を考慮してください。
