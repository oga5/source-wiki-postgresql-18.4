dshash_copy_function

定義

  • 種別: typedef
  • 定義箇所: src/include/lib/dshash.h:41

目的

dshash（動的共有メモリハッシュテーブル）において、ハッシュテーブルの要素をコピーするためのカスタム関数を定義するための型です。

シグネチャまたは形状

typedef void (*dshash_copy_function) (void *dest, const void *src, size_t size, void *arg);

入力と出力

  • dest: コピー先のメモリ領域へのポインタ。
  • src: コピー元のメモリ領域へのポインタ。
  • size: コピーするデータのサイズ（バイト単位）。
  • arg: コピー処理に渡される追加のコンテキスト情報（ユーザー定義の引数）。
  • 戻り値: なし（void）。

前提条件と不変条件

  • 本関数は、dshash_parameters 構造体を通じてハッシュテーブルに登録されます。
  • プロセス間で共有メモリを扱うため、この関数ポインタ自体はプロセス間で共有できません。各プロセスが dshash_attach を行う際に、改めて適切な関数を再指定する必要があります。

アルゴリズムまたは振る舞い

  • 指定された src から dest へ、size バイト分のデータをコピーします。
  • arg を使用することで、コピー処理中に必要な外部状態やコンテキストを関数内で参照可能です。

呼び出し元と呼び出し先

  • 呼び出し元: src/include/lib/dshash.h 内の dshash_parameters 構造体でフィールド copy_function として参照されています。
  • 呼び出し先: 提供されたソース情報からは、具体的な呼び出し先関数は特定できません。

関連データ構造

  • dshash_parameters: ハッシュテーブルの挙動を定義する構造体であり、本型をメンバとして保持します。

失敗モードと境界条件

  • dest または src が NULL の場合の挙動は、実装側の定義に依存します。
  • size がコピー先のバッファサイズを超過する場合のメモリ保護については、本型の定義範囲外です。

保守担当者向けメモ

  • 本関数は dshash の操作において、要素の複製が必要な場面（例：テーブルの拡張や再配置時）で使用されることを想定しています。
  • 関数ポインタのプロセス間共有に関する制約（「読者向けメモ」参照）を考慮し、実装時には各プロセスでの再設定が確実に行われるよう注意してください。
