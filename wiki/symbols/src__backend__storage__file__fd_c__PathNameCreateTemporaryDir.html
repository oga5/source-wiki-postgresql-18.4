PathNameCreateTemporaryDir

定義

  • 種別: function
  • 定義箇所: src/backend/storage/file/fd.c:1663

目的

一時ファイル格納用のディレクトリおよびその親ディレクトリ（ベースディレクトリ）を、必要に応じて作成する。

シグネチャまたは形状

void PathNameCreateTemporaryDir(const char *basedir, const char *directory)

入力と出力

  • basedir: 作成対象の親ディレクトリパス。
  • directory: 作成対象のディレクトリパス。
  • 戻り値: なし（成功時）。失敗時は ereport(ERROR, ...) によりプロセスを中断する。

前提条件と不変条件

  • 指定されたパスに対してディレクトリ作成権限が必要である。
  • 既にディレクトリが存在する場合、エラーとはならず正常終了する（冪等性）。

アルゴリズムまたは振る舞い

1. directory の作成を試みる。成功または errno == EEXIST の場合は終了する。 2. directory の作成に失敗した場合、basedir の作成を試みる。 3. basedir の作成に失敗し、かつ errno != EEXIST の場合はエラーを報告する。 4. basedir の作成後、再度 directory の作成を試みる。 5. 再度の試行でも作成に失敗し、かつ errno != EEXIST の場合はエラーを報告する。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/backend/storage/file/fileset.c:109
  • 呼び出し先:
  • MakePGDirectory (共通ユーティリティ)
  • ereport (エラー報告)

関連データ構造

  • 特になし。

失敗モードと境界条件

  • ディレクトリ作成権限がない場合、ereport によりエラーが発生する。
  • 競合するプロセスが同時にディレクトリを作成しようとした場合、EEXIST を許容することで正常に処理を継続する。

保守担当者向けメモ

  • 本関数は src/include/storage/fd.h で公開されている。
  • MakePGDirectory の戻り値と errno を組み合わせて、競合と権限エラーを区別している。
