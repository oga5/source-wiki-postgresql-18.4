mdinit

定義

  • 種別: function
  • 定義箇所: src/backend/storage/smgr/md.c:179

目的

磁気ディスク（Magnetic Disk）ストレージマネージャのプライベートな状態を初期化する。

シグネチャまたは形状

void mdinit(void)

入力と出力

  • 入力: なし
  • 出力: なし（副作用としてグローバル変数 MdCxt を初期化する）

前提条件と不変条件

  • TopMemoryContext が既に初期化されている必要がある。
  • 本関数はストレージマネージャの初期化プロセスの一部として呼び出される。

アルゴリズムまたは振る舞い

1. AllocSetContextCreate を呼び出し、TopMemoryContext を親として MdSmgr という名称のメモリコンテキストを作成する。 2. 作成したメモリコンテキストをグローバル変数 MdCxt に割り当てる。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/storage/smgr/smgr.c において、smgr_init 関数ポインタとして参照されている。
  • 呼び出し先: AllocSetContextCreate（メモリ管理サブシステム）。

関連データ構造

  • MdCxt: 本関数で初期化されるメモリコンテキスト。MdfdVec オブジェクトの管理に使用される。

失敗モードと境界条件

  • メモリ不足時には AllocSetContextCreate がエラーを発生させる可能性がある。

保守担当者向けメモ

  • 本関数はストレージマネージャのライフサイクル管理の起点である。
  • MdCxt は md.c 内でセグメント管理を行うためのメモリ領域として利用されるため、本関数の実行順序は重要である。
