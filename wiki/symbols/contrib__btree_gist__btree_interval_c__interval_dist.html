interval_dist

定義

  • 種別: function
  • 定義箇所: contrib/btree_gist/btree_interval.c:127

目的

2つの interval 型の値の間の絶対距離を計算します。

シグネチャまたは形状

Datum interval_dist(PG_FUNCTION_ARGS)

入力と出力

  • 入力: PG_FUNCTION_ARGS を介して受け取る2つの interval 型の Datum。
  • 出力: 2つの入力値の差の絶対値を示す interval 型の Datum。

前提条件と不変条件

  • 入力された Datum は有効な interval 型として解釈可能である必要があります。
  • 内部で interval_mi（減算）および abs_interval を使用するため、これらが正常に動作する環境である必要があります。

アルゴリズムまたは振る舞い

1. DirectFunctionCall2 を使用して、第1引数から第2引数を減算する interval_mi を呼び出します。 2. 得られた差分（diff）を abs_interval 関数に渡し、負の値を正の値に正規化します。 3. 正規化された絶対距離を PG_RETURN_INTERVAL_P マクロを介して返します。

呼び出し元と呼び出し先

  • 呼び出し元: ソースコード内に明示的な呼び出し元は記載されていません。
  • 呼び出し先:
  • interval_mi (PostgreSQL 内部関数)
  • abs_interval (contrib/btree_gist/btree_interval.c 内で定義)
  • DatumGetIntervalP (PostgreSQL 内部マクロ)

関連データ構造

  • interval: PostgreSQL の時間間隔型。

失敗モードと境界条件

  • 入力値が interval 型として不正な場合、基盤となる interval_mi や abs_interval の挙動に従いエラーが発生する可能性があります。

保守担当者向けメモ

  • 本関数は btree_gist モジュールにおける距離計算ロジックの一部です。
  • 距離の正規化には abs_interval が必須であるため、計算ロジックを変更する際は abs_interval の仕様と整合性を取る必要があります。
