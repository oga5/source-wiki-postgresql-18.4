local_destroy

定義

  • 種別: function
  • 定義箇所: src/bin/pg_rewind/local_source.c:181

目的

ローカルデータソースに関連付けられたメモリリソースを解放する。

シグネチャまたは形状

static void local_destroy(rewind_source *source);

入力と出力

  • 入力: rewind_source *source （解放対象のソース構造体へのポインタ）
  • 出力: なし

前提条件と不変条件

  • 引数 source は、init_local_source によって割り当てられた有効なメモリ領域を指している必要がある。

アルゴリズムまたは振る舞い

1. pfree(source) を呼び出し、渡された rewind_source 構造体のメモリを解放する。

呼び出し元と呼び出し先

  • 呼び出し元: init_local_source 内で、rewind_source 構造体の destroy 関数ポインタとして本関数が登録されている。
  • 呼び出し先: pfree （PostgreSQL のメモリ管理関数）。

関連データ構造

  • rewind_source: src/bin/pg_rewind/rewind_source.h で定義される抽象インターフェース。

失敗モードと境界条件

  • source が NULL の場合、pfree の仕様に依存する（通常はクラッシュまたはエラー）。

保守担当者向けメモ

  • 本関数は rewind_source インターフェースの破棄処理を実装している。将来的に local_source 構造体内部で追加のメモリ割り当て（パス文字列の保持など）が行われるようになった場合は、pfree を呼ぶ前にそれらのメンバを解放する処理を追加する必要がある。
