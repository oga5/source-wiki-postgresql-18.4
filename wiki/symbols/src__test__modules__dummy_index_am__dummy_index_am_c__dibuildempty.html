dibuildempty

定義

  • 種別: function
  • 定義箇所: src/test/modules/dummy_index_am/dummy_index_am.c:154

目的

インデックスの初期化（空のインデックス作成）時に呼び出されるスタブ関数です。

シグネチャまたは形状

static void dibuildempty(Relation index)

入力と出力

  • 入力: Relation index（初期化対象となるインデックスのリレーション情報）
  • 出力: なし（void）

前提条件と不変条件

  • 本関数はダミーのインデックス・アクセスメソッドの一部として実装されています。
  • ダミーインデックスであるため、初期化フォーク（init fork）の構築は不要という前提で動作します。

アルゴリズムまたは振る舞い

  • 関数内部では処理を行わず、空のインデックス構築をシミュレートします。

呼び出し元と呼び出し先

  • 呼び出し元: IndexAmRoutine 構造体の ambuildempty フィールドを通じて、PostgreSQLのインデックス管理システムから呼び出されます。
  • 呼び出し先: なし。

関連データ構造

  • Relation: PostgreSQLのカタログリレーション情報を保持する構造体。
  • IndexAmRoutine: 本関数が登録されるアクセスメソッドのルーチンテーブル。

失敗モードと境界条件

  • 特筆すべき失敗モードはありません。

保守担当者向けメモ

  • 本関数は「何もしない」実装です。新しいアクセスメソッドを開発する際のテンプレートとして利用してください。
