BitmapHeapScanDescData

定義

  • 種別: struct
  • 定義箇所: src/include/access/heapam.h:102

目的

ビットマップヒープスキャン（Bitmap Heap Scan）実行時の状態を保持するための構造体です。

シグネチャまたは形状

typedef struct BitmapHeapScanDescData
{
	HeapScanDescData rs_heap_base;

	/* Holds no data */
} BitmapHeapScanDescData;

入力と出力

  • **入力**: HeapScanDescData を継承しており、ヒープスキャンの基底状態を保持します。
  • **出力**: 本構造体自体はデータを保持しませんが、BitmapHeapScanDesc（本構造体へのポインタ）としてスキャン状態の管理に使用されます。

前提条件と不変条件

  • rs_heap_base フィールドを構造体の先頭に持つため、HeapScanDescData へのキャストが可能です。

アルゴリズムまたは振る舞い

  • 本構造体は、ビットマップインデックススキャンにおいてヒープページを読み取る際のスキャン記述子として機能します。
  • 現状、本構造体自体には追加のデータフィールドは定義されておらず、基底クラスである HeapScanDescData の機能を利用してスキャンを制御します。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/access/heap/heapam.c:1124 にて、palloc を用いてメモリ確保が行われています。
  • **呼び出し先**: 構造体定義のため、直接的な関数呼び出し先はありません。

関連データ構造

  • HeapScanDescData: 基底となるスキャン状態構造体。
  • BitmapHeapScanDesc: 本構造体へのポインタ型定義。

失敗モードと境界条件

  • メモリ確保失敗時（palloc 失敗時）にエラーが発生する可能性があります。

保守担当者向けメモ

  • 本構造体は現在データフィールドを持たない拡張用プレースホルダーですが、将来的にビットマップスキャン固有の状態管理が必要になった場合にフィールドが追加される可能性があります。
