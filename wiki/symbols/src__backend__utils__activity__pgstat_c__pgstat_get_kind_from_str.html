pgstat_get_kind_from_str

定義

  • **種別**: function
  • **定義箇所**: src/backend/utils/activity/pgstat.c:1402

目的

統計種別を表す文字列（kind_str）を、対応する内部列挙型 PgStat_Kind に変換します。

シグネチャまたは形状

PgStat_Kind pgstat_get_kind_from_str(char *kind_str)

入力と出力

  • **入力**: kind_str (統計種別名を示す文字列)
  • **出力**: 一致する PgStat_Kind 値。一致しない場合はエラーを報告し、戻り値は到達しません。

前提条件と不変条件

  • pgstat_kind_builtin_infos が初期化されている必要があります。
  • pgstat_kind_custom_infos が存在する場合、その各要素は有効な統計種別情報を指している必要があります。

アルゴリズムまたは振る舞い

1. 組み込み統計種別（PGSTAT_KIND_BUILTIN_MIN から PGSTAT_KIND_BUILTIN_MAX）を走査し、pgstat_kind_builtin_infos 内の名称と大文字小文字を区別せずに比較します。 2. 一致するものが見つかればその PgStat_Kind を返します。 3. 見つからない場合、カスタム統計種別（pgstat_kind_custom_infos）が存在すれば同様に走査し、一致する名称を探します。 4. いずれにも一致しない場合、ERRCODE_INVALID_PARAMETER_VALUE でエラーを発生させます。

呼び出し元と呼び出し先

  • **呼び出し元**: src/backend/utils/adt/pgstatfuncs.c:2269
  • **呼び出し先**: pg_strcasecmp（文字列比較用）

関連データ構造

  • PgStat_Kind: 統計種別を識別する列挙型。
  • pgstat_kind_builtin_infos: 組み込み統計種別の定義テーブル。
  • pgstat_kind_custom_infos: カスタム統計種別の定義配列。

失敗モードと境界条件

  • **失敗**: 指定された文字列がどの統計種別とも一致しない場合、ereport(ERROR, ...) により処理が中断されます。
  • **境界条件**: pgstat_kind_custom_infos が NULL の場合、カスタム統計の検索はスキップされます。

保守担当者向けメモ

  • 本関数は pgstat.h で公開されており、SQL 関数層から統計種別を解決するために使用されます。
  • 新しい統計種別を追加した際は、対応する name フィールドが適切に設定されていることを確認してください。
