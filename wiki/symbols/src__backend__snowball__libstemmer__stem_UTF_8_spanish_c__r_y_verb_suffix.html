r_y_verb_suffix

定義

  • 種別: function
  • 定義箇所: src/backend/snowball/libstemmer/stem_UTF_8_spanish.c:887

目的

スペイン語のステミング処理において、特定の動詞接尾辞（'y'を含むもの）を単語末尾から削除する。

シグネチャまたは形状

static int r_y_verb_suffix(struct SN_env * z)

入力と出力

  • 入力: SN_env * z（Snowballランタイム環境構造体）
  • 出力: 成功時に 1、失敗時に 0、エラー発生時に負の整数を返す。

前提条件と不変条件

  • z->I[2] に格納された領域境界（R2領域の開始位置など）より現在のカーソル位置 z->c が後ろにあること。
  • 削除対象の接尾辞が a_7 配列（12個の要素）に定義されていること。

アルゴリズムまたは振る舞い

1. 現在のカーソル位置が z->I[2] より前であれば終了する。 2. z->lb を z->I[2] に一時的に設定し、後方検索（find_among_b）を用いて a_7 に定義された接尾辞を検索する。 3. 接尾辞が見つかった場合、その直前の文字が 'u' であることを確認する。 4. 条件を満たす場合、slice_del を呼び出して該当する接尾辞を削除する。 5. 処理終了後、z->lb を元の値に戻す。

呼び出し元と呼び出し先

  • 呼び出し元: src/backend/snowball/libstemmer/stem_UTF_8_spanish.c および src/backend/snowball/libstemmer/stem_ISO_8859_1_spanish.c 内のステミングメイン処理。
  • 呼び出し先: find_among_b（接尾辞検索）、slice_del（文字列削除）。

関連データ構造

  • SN_env: ステミング処理の環境状態を保持する構造体。
  • a_7: 削除対象となる接尾辞パターンを保持する among 構造体の配列。

失敗モードと境界条件

  • カーソル位置が境界条件を満たさない場合、または接尾辞が一致しない場合は 0 を返す。
  • slice_del が内部エラー（負の戻り値）を返した場合、その値をそのまま伝播させる。
  • 単語末尾の文字が 'u' でない場合は削除を行わない。

保守担当者向けメモ

  • 本関数は自動生成コードの一部であるため、直接修正せず、Snowballのソース（.sbl）を修正すること。
  • a_7 配列の定義変更は、ステミングのルール全体に影響を与えるため注意が必要。
