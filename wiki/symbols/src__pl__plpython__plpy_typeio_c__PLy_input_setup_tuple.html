PLy_input_setup_tuple

定義

  • 種別: function
  • 定義箇所: src/pl/plpython/plpy_typeio.c:164

目的

SQLタプル（TupleDesc）の構造に合わせて、PL/Pythonのデータ変換構造体（PLyDatumToOb）を初期化または更新します。

シグネチャまたは形状

void PLy_input_setup_tuple(PLyDatumToOb *arg, TupleDesc desc, PLyProcedure *proc)

入力と出力

  • arg: 変換対象の PLyDatumToOb 構造体（入力/出力）。
  • desc: 変換対象となるタプルの記述子（TupleDesc）。
  • proc: 現在実行中のPL/Pythonプロシージャ情報。
  • 戻り値: なし。

前提条件と不変条件

  • arg->func が PLyDict_FromComposite であることがアサーションにより保証されている必要があります。
  • arg が指す構造体は、既に初期化済みである必要があります。

アルゴリズムまたは振る舞い

1. **メタデータ保存**: arg が匿名レコード型（RECORDOID かつ typmod < 0）の場合、desc を arg->u.tuple.recdesc に保存します。 2. **属性配列の管理**: arg 内の属性数（natts）が desc->natts と一致しない場合、既存の属性配列を解放し、新しいサイズで再割り当て（MemoryContextAllocZero）を行います。 3. **再帰的セットアップ**: 各属性をループ処理し、以下の条件を満たす場合に PLy_input_setup_func を呼び出して再帰的に変換関数をセットアップします。

  • 属性が削除（attisdropped）されていないこと。
  • 型情報（typoid, typmod）が既存のセットアップと一致しないこと。

呼び出し元と呼び出し先

  • **呼び出し元**:
  • src/pl/plpython/plpy_cursorobject.c
  • src/pl/plpython/plpy_exec.c
  • src/pl/plpython/plpy_spi.c
  • src/pl/plpython/plpy_typeio.c
  • **呼び出し先**:
  • PLy_input_setup_func（各属性の型変換設定のため）

関連データ構造

  • PLyDatumToOb: SQLからPythonへの変換ロジックを保持する構造体。
  • TupleDesc: SQLタプルの構造情報。
  • PLyProcedure: PL/Pythonプロシージャの実行コンテキスト。

失敗モードと境界条件

  • 属性数が変更された場合、メモリの再割り当てが発生します。
  • 削除された属性（attisdropped）はスキップされ、変換対象から除外されます。
  • 匿名レコード型の場合、typmod が負の値であるという条件で recdesc が更新されます。

保守担当者向けメモ

  • 本関数は再帰的に PLy_input_setup_func を呼び出すため、メモリコンテキストの管理には注意が必要です。
  • arg->mcxt を使用してメモリを割り当てているため、呼び出し側で適切なメモリコンテキストが維持されていることを確認してください。
