SavedTransactionCharacteristics

定義

  • 種別: struct
  • 定義箇所: src/include/access/xact.h:152

目的

現在のトランザクション特性（分離レベル、読み取り専用設定、遅延可能設定）を一時的に保存および復元するためのデータ構造です。

シグネチャまたは形状

typedef struct SavedTransactionCharacteristics
{
	int			save_XactIsoLevel;
	bool		save_XactReadOnly;
	bool		save_XactDeferrable;
} SavedTransactionCharacteristics;

入力と出力

  • 入力: 本構造体へのポインタが SaveTransactionCharacteristics および RestoreTransactionCharacteristics 関数に渡されます。
  • 出力: SaveTransactionCharacteristics は、現在のグローバルなトランザクション状態を構造体の各フィールドに書き込みます。RestoreTransactionCharacteristics は、構造体の値を用いてグローバルなトランザクション状態を上書きします。

前提条件と不変条件

  • 本構造体は、トランザクション制御のコンテキスト内でのみ使用されることを想定しています。
  • save_XactIsoLevel は、PostgreSQLの有効なトランザクション分離レベルを保持する必要があります。

アルゴリズムまたは振る舞い

  • SaveTransactionCharacteristics(SavedTransactionCharacteristics *s): 現在のグローバルなトランザクション特性を構造体 s にコピーします。
  • RestoreTransactionCharacteristics(const SavedTransactionCharacteristics *s): 構造体 s に保存された値を読み取り、現在のトランザクション特性をその値に復元します。

呼び出し元と呼び出し先

  • 呼び出し元:
  • src/backend/access/transam/xact.c
  • src/backend/executor/spi.c
  • 呼び出し先:
  • 本構造体自体はデータ保持用であり、直接関数を呼び出しません。操作は SaveTransactionCharacteristics および RestoreTransactionCharacteristics 関数を介して行われます。

関連データ構造

  • XactIsoLevel (グローバル変数): 本構造体の save_XactIsoLevel が参照・設定する対象です。

失敗モードと境界条件

  • 構造体へのポインタが NULL の場合、関数呼び出し時に未定義の動作を引き起こす可能性があります。
  • 復元時に不正な分離レベル値が構造体に格納されていた場合、トランザクションの挙動が予期せぬものとなる可能性があります。

保守担当者向けメモ

  • 本構造体は、SPI（Server Programming Interface）などでトランザクション特性を一時的に変更し、終了後に元に戻す必要がある際に利用されます。
  • フィールドを追加する場合は、Save および Restore 関数の両方を更新し、整合性を維持してください。
