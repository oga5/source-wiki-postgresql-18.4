contrib/hstore_plpython

システム内での役割

本モジュールは、PostgreSQL の hstore 型と Python の辞書型（dict）間の相互変換機能を提供するブリッジ拡張です。ディレクトリ境界に基づき構成されており、hstore および plpython 拡張の両方に依存する独立した共有ライブラリとして機能します。

責務

  • **動的リンク**: 実行時に hstore および plpython モジュールの関数ポインタを解決し、クロスモジュール呼び出しを実現する。
  • **型変換**: hstore データを Python の dict へ、また Python のマッピングオブジェクトを hstore 形式へ変換するパイプラインの提供。
  • **メモリ管理**: Python オブジェクトの参照カウントと PostgreSQL のメモリコンテキストの整合性維持。

このモジュールのファイル

  • contrib/hstore_plpython/hstore_plpython.c

主要なデータ構造

  • PyObject, Py_ssize_t: Python C API 関連の型。
  • Datum: PostgreSQL の汎用データ表現。
  • hstoreCheckKeyLen_t, hstoreCheckValLen_t: hstore 構造の検証用型。

主なフロー

1. **初期化**: _PG_init が load_external_function を介して必要な外部関数アドレスを解決。 2. **hstore → Python**: hstore 構造を走査し、Python 文字列へ変換後、PyDict_SetItem で辞書を構築。 3. **Python → hstore**: PyMapping_Items で要素を取得し、重複排除（hstoreUniquePairs）を経て HStore 構造体を生成。エラー発生時は PG_TRY/PG_FINALLY でメモリリークを防止。

外部依存

  • postgres.h, fmgr.h (PostgreSQL コア)
  • hstore/hstore.h (hstore 型定義)
  • plpy_typeio.h, plpy_util.h (plpython ユーティリティ)
