contrib/jsonb_plpython

システム内での役割

本モジュールは、PostgreSQL の jsonb 型と Python のオブジェクト間での相互変換機能を提供する plpython 拡張のサブモジュールです。ディレクトリ境界に基づき構成されています。

責務

  • PostgreSQL の jsonb データと Python の辞書・リスト・数値型等の間での再帰的な相互変換。
  • plpython 拡張との動的な連携による型変換ロジックの提供。
  • JSON 仕様と PostgreSQL の制約（数値の NaN/Infinity 判定等）を遵守した整合性の維持。

このモジュールのファイル

  • contrib/jsonb_plpython/jsonb_plpython.c

主要なデータ構造

  • PyObject (Python オブジェクト)
  • JsonbValue / JsonbContainer (PostgreSQL 側の JSONB 構造)
  • JsonbParseState (JSONB 構築用状態管理)
  • PLy_elog_impl_t

主なフロー

1. **初期化**: _PG_init にて plpython 本体から必要な関数ポインタを動的にロード。 2. **JSONB → Python**: jsonb_to_plpython がイテレータを介して JSONB を走査し、PLyObject_FromJsonbContainer が再帰的に Python オブジェクトを生成。 3. **Python → JSONB**: plpython_to_jsonb が Python オブジェクトを解析し、PLyObject_ToJsonbValue を経由して JsonbParseState を用いて JSONB 構造を構築。
