contrib/hstore_plperl

システム内での役割

PostgreSQL の hstore 型と Perl のハッシュリファレンス間の相互変換を行うアダプター層です。PL/Perl 環境において、hstore データを Perl の連想配列としてシームレスに操作可能にします。本モジュールはディレクトリ境界に基づき定義されています。

責務

  • hstore 構造体から Perl の HV (Hash Value) への変換。
  • Perl のハッシュリファレンスから hstore 構造体への変換。
  • hstore モジュールおよび plperl モジュール間の動的リンクと関数ポインタの解決。

このモジュールのファイル

  • contrib/hstore_plperl/hstore_plperl.c

主要なデータ構造

  • Datum: PostgreSQL の汎用データ型。
  • hstoreCheckKeyLen_t / hstoreCheckValLen_t: hstore のキーおよび値の長さを検証するための型。
  • HV (Perl): Perl のハッシュ構造体。

主なフロー

1. **初期化**: _PG_init が実行時に load_external_function を介して hstore モジュールの関数ポインタを解決・格納します。 2. **hstore → Perl**: hstore_to_plperl が hstore エントリを走査し、Perl API を使用してハッシュを構築します。 3. **Perl → hstore**: plperl_to_hstore が Perl ハッシュを走査してペア配列を構築し、重複排除を経て hstore 構造体を生成します。

外部依存

  • postgres.h / fmgr.h: PostgreSQL のコアインターフェース。
  • hstore/hstore.h: hstore 型の定義および操作用マクロ。
  • plperl.h: Perl インタープリタとの連携用ヘッダー。
