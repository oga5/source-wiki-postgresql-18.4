src/test/modules/test_rbtree

システム内での役割

本モジュールは、PostgreSQL の赤黒木（Red-Black Tree）ライブラリ（src/include/lib/rbtree.h）の正当性を検証するためのテスト用拡張機能です。ディレクトリ境界に基づき構成されています。

責務

  • 赤黒木ライブラリの挿入、検索、削除、走査操作の網羅的な検証。
  • 乱数を用いた動的な木構造の構築と、再構成時の整合性チェック。
  • SQL インターフェースを通じたテスト実行環境の提供。

このモジュールのファイル

  • src/test/modules/test_rbtree/test_rbtree.c: テストロジックおよび赤黒木ライブラリ用コールバックの実装。

主要なデータ構造

  • IntRBTreeNode: RBTNode を継承し、整数キーを保持するテスト用ノード構造体。

主なフロー

1. **初期化**: SQL から test_rb_tree(int) を呼び出し、検証を開始。 2. **構築**: Fisher-Yates シャッフルで生成した順序に基づき、rbt_populate で木を構築。 3. **検証**: 順方向・逆方向の走査、検索（rbt_find 等）、削除操作を順次実行し、期待値と一致するかを elog(ERROR, ...) で確認。 4. **終了**: 全てのテストが成功すれば正常終了。

外部依存

  • postgres.h, fmgr.h: PostgreSQL 基本機能および関数管理。
  • lib/rbtree.h: 検証対象の赤黒木ライブラリ。
  • common/pg_prng.h: 乱数生成（テスト順序のランダム化）。
  • utils/memutils.h: メモリ管理。

内部構成

  • 子モジュールなし。単一のソースファイルで構成されるテストスイート。

推奨読書順

1. src/test/modules/test_rbtree/test_rbtree.c: テストの全体像とコールバック実装の確認。
