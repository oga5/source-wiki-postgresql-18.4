contrib/ltree_plpython

システム内での役割

本モジュールは、PostgreSQLの ltree 型と plpython 拡張の橋渡しを行うアダプターです。ディレクトリ境界に基づき構成されており、ltree 階層パスデータを Python スクリプト内で扱いやすいリスト形式へ変換する機能を提供します。

責務

  • plpython が提供する関数ポインタの動的ロードと解決。
  • ltree 構造体の走査および各ラベルの抽出。
  • 抽出したラベルを Python の文字列リスト（list）へ変換する型変換処理。
  • Python C API を利用したメモリ管理とエラーハンドリング。

このモジュールのファイル

  • contrib/ltree_plpython/ltree_plpython.c

主要なデータ構造

  • Py_ssize_t: Python オブジェクトのサイズやインデックスを扱うための型。

主なフロー

1. **初期化**: _PG_init が実行時に load_external_function を介して plpython の関数ポインタを取得・保持します。 2. **変換**: ltree_to_plpython が呼び出されると、ltree の各レベルを走査し、PyList_New で確保した Python リストへ各ラベルを格納します。 3. **返却**: 構築された Python リストを Datum として PostgreSQL 側に返却します。

外部依存

  • postgres.h, fmgr.h: PostgreSQL コアおよび関数管理。
  • ltree/ltree.h: ltree 型の定義と操作マクロ。
  • plpy_util.h: plpython 関連ユーティリティ。
