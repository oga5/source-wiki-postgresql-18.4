src/test/modules/ldap_password_func

システム内での役割

本モジュールは、PostgreSQLのLDAP認証プロセスにおけるパスワード変換フックの動作検証を目的としたテスト用拡張機能です。ディレクトリ境界に基づき定義されています。

責務

  • PostgreSQLの認証フック機構を利用し、LDAPバインドパスワードを動的に加工する。
  • テスト目的で、入力されたパスワードに対してROT13変換を適用する。

このモジュールのファイル

  • src/test/modules/ldap_password_func/ldap_password_func.c

主要なデータ構造

  • ldap_password_hook: libpq/auth.h で定義された、パスワード変換関数を保持するためのグローバルな関数ポインタ。

主なフロー

1. **初期化**: _PG_init が実行され、ldap_password_hook に rot13_passphrase 関数を登録する。 2. **フック実行**: LDAP認証時にコアシステムがフックを呼び出し、rot13_passphrase が起動する。 3. **変換処理**: 入力パスワードを palloc で確保したメモリにコピーし、ROT13アルゴリズムでアルファベットを置換する。 4. **返却**: 変換後の文字列ポインタをコアシステムへ返し、認証処理を継続させる。

外部依存

  • postgres.h
  • fmgr.h
  • libpq/auth.h
  • float.h, stdio.h
