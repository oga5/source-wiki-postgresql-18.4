contrib/jsonb_plperl

システム内での役割

PostgreSQL の jsonb 型と Perl のデータ構造（スカラー、配列、ハッシュ）を相互変換するためのブリッジ機能を提供します。PL/Perl 関数内で JSON データを扱う際の型変換ロジックを実装する拡張モジュールです。

責務

  • Jsonb 構造体から Perl の SV (Scalar Value) への再帰的な変換。
  • Perl のデータ構造から Jsonb 形式への再帰的な構築。
  • PostgreSQL と Perl 間での型整合性（数値、文字列、真偽値、NULL）の維持。

このモジュールのファイル

  • contrib/jsonb_plperl/jsonb_plperl.c

主要なデータ構造

  • Jsonb / JsonbValue: PostgreSQL 側の JSONB 内部表現。
  • SV / AV / HV: Perl 側のスカラー、配列、ハッシュ構造。
  • JsonbParseState: jsonb 構築時に使用する状態管理構造体。

主なフロー

1. **変換開始**: jsonb_to_plperl または plperl_to_jsonb がエントリポイントとなり、Perl インタプリタのコンテキストを初期化。 2. **走査と変換**:

  • Jsonb からの変換時は JsonbIterator を使用してノードを順次処理。
  • Perl からの変換時は SvTYPE を判定し、再帰的に JsonbValue へ変換。

    • 3. **構築**: pushJsonbValue を用いて JsonbParseState を更新し、最終的に JsonbValueToJsonb で確定。

外部依存

  • postgres.h, fmgr.h: PostgreSQL コアおよび関数インターフェース。
  • plperl.h: Perl インタプリタ連携。
  • utils/jsonb.h: Jsonb 操作 API。
  • math.h: 数値の妥当性チェック（isinf, isnan）。

内部構成

  • 子モジュール数: 0（ディレクトリ境界に基づき、jsonb_plperl.c が単一の変換エンジンとして機能）

推奨読書順

1. contrib/jsonb_plperl/jsonb_plperl.c
