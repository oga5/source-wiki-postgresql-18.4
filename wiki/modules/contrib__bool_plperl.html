contrib/bool_plperl

システム内での役割

PostgreSQLの boolean 型と Perl のスカラ値（SV）間の型変換を仲介する拡張モジュールです。PL/Perl を使用した関数定義において、ブール値のシームレスな受け渡しを実現します。本モジュールはディレクトリ境界に基づき定義されています。

責務

  • PostgreSQL の bool 型を Perl の真偽値（PL_sv_yes / PL_sv_no）へ変換する。
  • Perl のスカラ値（SV）を Perl の真偽値判定ルール（SvTRUE）に基づき PostgreSQL の bool 型へ変換する。

このモジュールのファイル

  • contrib/bool_plperl/bool_plperl.c

主要なデータ構造

  • SV* (Perl スカラ値ポインタ): Perl インタプリタとのデータ交換に使用。
  • dTHX (Perl スレッドコンテキスト): マルチスレッド環境下で正しいインタプリタコンテキストを特定するために使用。

主なフロー

1. **bool_to_plperl**: PostgreSQL の bool を取得し、Perl の真偽値定数へマッピングして返却。 2. **plperl_to_bool**: Perl の SV* を取得し、SvTRUE マクロを用いて PostgreSQL の bool へ変換して返却。

外部依存

  • postgres.h: PostgreSQL コア定義。
  • fmgr.h: 関数マネージャ（PG_FUNCTION_ARGS 等）。
  • plperl.h: Perl インタプリタ連携用ヘッダー。
