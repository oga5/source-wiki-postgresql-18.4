src/test/modules/test_tidstore

システム内での役割

本モジュールは、PostgreSQL の TidStore データ構造の機能および整合性を検証するためのテスト用拡張モジュールです。ディレクトリ境界に基づき構成されており、TidStore API の利用例および回帰テストとしての役割を担います。

責務

  • TidStore のライフサイクル（作成・破棄）の管理。
  • SQL インターフェースを通じたデータ挿入、検索、イテレーション操作のシミュレーション。
  • 挿入データと TidStore 内の保持データの整合性検証。
  • TidStore のメモリ使用量計測機能の正常性確認。

このモジュールのファイル

  • src/test/modules/test_tidstore/test_tidstore.c

主要なデータ構造

  • ItemArray: テストの正当性を検証するために、挿入済みデータを保持・比較するための補助構造体。

主なフロー

1. **初期化**: test_create により TidStore インスタンスをメモリ上に確保。 2. **データ挿入**: do_set_block_offsets を介して TidStore への書き込みと検証用配列へのコピーを実行。 3. **検証**: check_set_block_offsets を実行し、TidStoreIsMember による個別確認および TidStoreBeginIterate による全走査を行い、期待値との不整合がないか確認。 4. **終了**: test_destroy によりメモリを解放。

外部依存

  • access/tidstore.h (TidStore API)
  • storage/itemptr.h, storage/block.h, storage/lwlock.h
  • utils/array.h, utils/memutils.h
  • postgres.h, fmgr.h

内部構成

  • 子モジュールなし（単一のソースファイルで構成）。
