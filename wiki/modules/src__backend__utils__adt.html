src/backend/utils/adt

システム内での役割

src/backend/utils/adt は、PostgreSQL のデータ型（Abstract Data Types）に対する SQL レベルの演算、変換、および集約関数の実装を統括するディレクトリです。SQL クエリで利用可能な組み込み関数と、それらを支える内部的なデータ操作ロジックの大部分がここに集約されています。

責務

  • 各データ型（数値、日時、文字列、幾何、JSON 等）の入出力（I/O）およびバイナリ変換の提供。
  • SQL 演算子（+, -, ->, @@ 等）に対応するバックエンド関数の実装。
  • 統計情報やプランナが利用する選択率推定・コスト計算関数の提供。
  • 複雑なデータ型（配列、レンジ、JSONB）の内部メモリレイアウトの管理と操作。

このモジュールのファイル

114個のソースファイルで構成され、主なカテゴリは以下の通りです：

  • **数値・算術**: numeric.c, int.c, int8.c, float.c
  • **テキスト・可変長**: varlena.c, formatting.c, like.c, regexp.c
  • **日時**: timestamp.c, date.c, datetime.c
  • **JSON/JSONB**: json.c, jsonb.c, jsonfuncs.c, jsonpath_exec.c
  • **幾何**: geo_ops.c, geo_selfuncs.c
  • **配列・レンジ**: arrayfuncs.c, rangetypes.c, multirangetypes.c
  • **その他**: acl.c (権限), ruleutils.c (逆コンパイル), xml.c (XML), selfuncs.c (プランナ統計)

主要なデータ構造

  • **可変長データ型**: varlena ヘッダーを持つ全ての型（text, bytea, numeric, array 等）。
  • **JSONB**: JsonbValue, JsonbIterator 等のバイナリ走査構造体。
  • **日時**: Timestamp, Interval, DateADT, TimeADT。
  • **範囲型**: RangeType, RangeBound, MultirangeType。
  • **プランナ用**: VariableStatData（統計情報）、deparse_context（逆コンパイル用）。

主なフロー

1. **入力**: SQL 文字列が _in 関数（例: numeric_in）に渡され、Parse ルーチンを経て内部バイナリ表現に変換される。 2. **演算**: SQL 演算子が呼び出されると、PG_FUNCTION_ARGS を介して Datum を受け取り、各型の実装関数が算術・論理演算を実行する。 3. **プランナ連携**: selfuncs.c が pg_statistic を参照し、演算子の選択率を推定してプランナに返す。 4. **出力**: 内部表現が _out 関数（例: numeric_out）により SQL 文字列へシリアライズされる。

外部依存

  • postgres.h: 基本的な型定義とマクロ。
  • utils/fmgr.h: 関数マネージャ（PG_FUNCTION_ARGS 等）。
  • utils/typcache.h: 型情報の動的キャッシュ。
  • common/int.h: オーバーフロー保護付き算術演算。
  • libxml2 (xml.c のみ): XML 処理用。

内部構成

ディレクトリ境界に基づくモジュールであり、サブディレクトリは存在しません。各ファイルは特定のデータ型や機能に特化しており、共通のユーティリティ（datetime.c や varlena.c）を介して相互に連携しています。

推奨読書順

1. src/backend/utils/adt/varlena.c (可変長型の基本) 2. src/backend/utils/adt/numeric.c (数値型の実装) 3. src/backend/utils/adt/arrayfuncs.c (配列操作) 4. src/backend/utils/adt/jsonb.c / jsonfuncs.c (JSONB) 5. src/backend/utils/adt/timestamp.c (日時型) 6. src/backend/utils/adt/selfuncs.c (プランナ統計) 7. src/backend/utils/adt/ruleutils.c (逆コンパイル)
