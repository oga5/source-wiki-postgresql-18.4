src/backend/utils/mb/conversion_procs/cyrillic_and_mic

システム内での役割

本モジュールは、PostgreSQL のエンコーディング変換フレームワークの一部として、キリル文字系エンコーディング（KOI8-R, ISO-8859-5, WIN1251, WIN866）と MULE_INTERNAL 間の相互変換機能を提供します。

責務

  • 各キリル文字エンコーディング間のマッピングテーブル（変換用配列）の保持。
  • PostgreSQL の fmgr インターフェースに準拠した変換関数の提供。
  • 変換時の引数検証、バイト数計算、およびエラーハンドリングの実行。

このモジュールのファイル

  • src/backend/utils/mb/conversion_procs/cyrillic_and_mic/cyrillic_and_mic.c

    • ※本モジュールはディレクトリ境界に基づき定義されています。

主要なデータ構造

  • **変換テーブル**: iso2koi, koi2iso, win12512koi, koi2win1251 等の static const unsigned char 配列。各エンコーディング間の文字コード対応を定義します。

主なフロー

1. **呼び出し**: SQL の CONVERT 関数やクライアント・サーバー間の通信時に変換関数が起動。 2. **検証**: CHECK_ENCODING_CONVERSION_ARGS マクロにより、エンコーディング ID の整合性を確認。 3. **変換実行**:

  • MULE_INTERNAL への変換には latin2mic 系関数を使用。
  • MULE_INTERNAL からの変換には mic2latin 系関数を使用。
  • エンコーディング間の直接変換には local2local を使用。

    • 4. **返却**: 変換後のバイト数を PG_RETURN_INT32 で返却。

外部依存

  • postgres.h: 基本型およびマクロ定義。
  • fmgr.h: 関数マネージャインターフェース。
  • mb/pg_wchar.h: マルチバイト文字処理の共通定義および変換ユーティリティ。
