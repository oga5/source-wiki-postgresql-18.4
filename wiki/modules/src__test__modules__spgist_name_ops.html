src/test/modules/spgist_name_ops

システム内での役割

本モジュールは、PostgreSQLのSP-GiSTインデックスにおいて name 型をサポートするためのテスト用演算子クラス（opclass）を提供します。ディレクトリ境界に基づき構成されたテスト用拡張モジュールです。

責務

  • name 型の値を SP-GiST インデックスで効率的に扱うためのロジック実装。
  • 内部的に text 型の処理ロジックを再利用し、照合順序を考慮しないバイナリ比較によるインデックス操作の提供。

このモジュールのファイル

  • src/test/modules/spgist_name_ops/spgist_name_ops.c: インデックス構成、データ変換、ツリー走査ロジックの実装。

主要なデータ構造

  • spgist_name_config: インデックスのメタデータおよび型定義を保持する構造体。

主なフロー

1. **挿入時**: spgist_name_compress により name 型を text 型へ変換し、spgist_name_choose がツリー内の配置場所（ノード選択や分割）を決定します。 2. **検索時**: spgist_name_inner_consistent がツリーを再帰的に走査して候補ノードを絞り込み、最終的に spgist_name_leaf_consistent が実際の値とクエリを比較して結果を返します。

外部依存

  • postgres.h, access/spgist.h, catalog/pg_type.h, utils/datum.h, varatt.h

内部構成

  • 子モジュールなし。コアの spgtextproc.c のロジックをベースに実装されています。

推奨読書順

1. src/test/modules/spgist_name_ops/spgist_name_ops.c 2. src/backend/access/spgist/spgtextproc.c（比較対象として）
