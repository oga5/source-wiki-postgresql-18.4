src/test/locale/test-ctype.c

概要

test-ctype.c は、現在のロケール設定における標準Cライブラリの文字分類関数（<ctype.h>）の挙動を検証するためのテストプログラムです。ASCIIコード 0 から 255 までの全文字に対して、各種分類フラグの状態を一覧表示します。

モジュール内での役割

src/test/locale モジュールにおける検証用ツールです。システムが正しくロケールを認識し、それに基づいた文字属性（大文字・小文字・空白・句読点など）を正しく判定できているかを確認する役割を担います。

責務

  • **ロケールの初期化**: setlocale(LC_ALL, "") を呼び出し、環境変数に基づくロケール設定を試行する。
  • **文字属性の網羅的テスト**: 0〜255の全文字に対して、isalnum, isalpha, isspace 等の標準関数を適用する。
  • **結果の可視化**: 各文字の属性を整形された表形式で標準出力に出力する。

重要なシンボル

  • main(): プログラムのエントリポイント。ロケールの設定とループ処理を制御する。
  • describe_char(int c): 特定の文字コード c に対し、各種 is* 関数の結果を判定し、一行のレポートを出力する。
  • flag(int b): 真偽値を受け取り、表示用の記号（+ または ）を返すヘルパー関数。

追加の索引済みシンボル

  • describe_char

制御フローとデータフロー

1. **初期化**: main が setlocale を実行。失敗した場合はエラーメッセージを出力して終了する。 2. **ループ処理**: 0 から 255 までの整数を順次 describe_char に渡す。 3. **属性判定**: describe_char 内で toupper, tolower および各種 is* マクロを呼び出し、現在のロケールに基づいた文字の性質を取得する。 4. **出力**: 判定結果を printf でフォーマットし、コンソールへ出力する。

直接依存

  • <stdio.h>: 標準入出力（printf, fprintf）に使用。
  • <locale.h>: ロケール設定（setlocale）に使用。
  • <ctype.h>: 文字分類および変換関数（isalnum, toupper 等）に使用。
  • ctype.h
  • locale.h
  • stdio.h

関連コード

  • src/test/locale/sort-test.py: 同ディレクトリ内の関連ファイル。ロケール依存のソート順序などを検証する際に併用される可能性がある。

読者向けメモ

  • **ロケール依存**: 本プログラムの出力は、実行環境の環境変数（LANG, LC_CTYPE）に強く依存します。期待通りの結果を得るには、適切なロケールがシステムにインストールされている必要があります。
  • **マクロの挙動**: LONG_FLAG マクロを定義することで、フラグの出力を + / から yes / no に切り替えることが可能です。
  • **移植性**: このコードは標準的なCライブラリの挙動を確認するためのものであり、特定のプラットフォーム固有の拡張機能は含んでいません。
