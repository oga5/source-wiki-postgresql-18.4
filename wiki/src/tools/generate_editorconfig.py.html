src/tools/generate_editorconfig.py

概要

本スクリプトは、リポジトリ内の .gitattributes ファイルを解析し、対応する .editorconfig ファイルを自動生成するためのユーティリティです。開発環境間でのコーディングスタイルの統一を支援します。

モジュール内での役割

src/tools ディレクトリに配置された補助ツールの一つです。ビルドシステムの一部ではなく、リポジトリのメンテナンスや開発環境のセットアップを補助する役割を担います。

責務

  • リポジトリルートへの移動と環境の特定。
  • .gitattributes に定義されたホワイトスペースルール（whitespace 属性）の読み取り。
  • Git のルールを .editorconfig 形式（indent_style, trim_trailing_whitespace 等）へ変換。
  • 特定のファイル拡張子に対するハードコードされたインデント設定の適用。

重要なシンボル

  • space_based_indent_sizes: Python や XML など、タブではなくスペースインデントを強制するファイル拡張子とサイズの対応表。
  • main(): スクリプトのメインロジック。.gitattributes をパースし、.editorconfig を生成する一連の処理を実行します。
  • cd_to_repo_root(): スクリプトの実行場所に関わらず、リポジトリのルートディレクトリへ移動する初期化関数。

制御フローとデータフロー

1. **初期化**: cd_to_repo_root によりリポジトリルートへ移動。 2. **読み込み**: .gitattributes を読み込み、各行を走査。 3. **変換**: 各行の whitespace 属性を解析し、trim_trailing_whitespace や indent_style などの .editorconfig 用ルール文字列を構築。 4. **補完**: space_based_indent_sizes を参照し、特定のファイルタイプに対して明示的なインデントサイズを付与。 5. **出力**: 構築したルールを .editorconfig ファイルとして書き出し。

直接依存

  • os: ファイルパスの操作およびディレクトリ移動に使用。

関連コード

  • .gitattributes: 本スクリプトの入力ソースとなるファイル。
  • .editorconfig: 本スクリプトの出力先。エディタの設定を制御する標準的な設定ファイル。

読者向けメモ

  • 本スクリプトは、.gitattributes の whitespace 属性の定義に強く依存しています。.gitattributes のフォーマットが変更された場合、このスクリプトのパースロジックも追従する必要があります。
  • Perl ファイル（*.pl, *.pm）など、一部のファイルタイプについては .gitattributes から自動推論できないため、スクリプト内にハードコードされた例外ルールが存在します。修正が必要な場合は main 関数内の条件分岐を確認してください。
