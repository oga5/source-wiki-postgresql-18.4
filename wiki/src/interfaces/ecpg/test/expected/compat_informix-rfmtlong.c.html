src/interfaces/ecpg/test/expected/compat_informix-rfmtlong.c

概要

本ファイルは、ECPG（Embedded SQL for C）における Informix 互換ライブラリの rfmtlong 関数の動作検証を行う回帰テスト用の期待値ソースコードです。

モジュール内での役割

src/interfaces/ecpg/test/expected ディレクトリ群の一部として、ECPG プリプロセッサが生成する C コードの正当性を確認するためのテストスイートを構成します。Informix 互換機能のテストカテゴリに属します。

責務

  • rfmtlong 関数に対し、多様なフォーマット文字列と数値を入力し、変換結果が期待通りであるかを検証する。
  • 変換エラー発生時の戻り値（ECPG_INFORMIX_* 系）が正しくハンドリングされることを確認する。

重要なシンボル

  • fmtlong(): テスト対象の rfmtlong を呼び出し、結果を標準出力に整形して出力するラッパー関数。
  • rfmtlong(): Informix 互換ライブラリが提供する、長整数をフォーマット文字列に従って文字列に変換する関数。
  • check_return(): 変換失敗時に返されるエラーコードを人間が判読可能な文字列に変換して出力するヘルパー関数。

追加の索引済みシンボル

  • check_return
  • fmtlong
  • main

制御フローとデータフロー

1. main() 関数が開始され、ECPGdebug を有効化してデバッグ出力を設定します。 2. fmtlong() を異なるフォーマット文字列（#, $, *, + 等の修飾子を含む）で繰り返し呼び出します。 3. fmtlong() 内で rfmtlong() が実行され、結果が buf に格納されます。 4. 変換が成功すれば buf の内容を、失敗すれば check_return() を経由してエラー内容を標準出力へ書き出します。

直接依存

  • ecpglib.h, ecpgerrno.h, sqlca.h: ECPG の基本ライブラリおよびエラー定義。
  • ecpg_informix.h: Informix 互換機能の定義。
  • pgtypes_error.h, sqltypes.h: データ型およびエラーコードの定義。
  • stdio.h, stdlib.h: 標準入出力および汎用ユーティリティ。

関連コード

  • src/interfaces/ecpg/include/ecpg_informix.h: rfmtlong の宣言元。
  • src/interfaces/ecpg/pgtypeslib/: rfmtlong が実装されているライブラリ本体。
  • src/interfaces/ecpg/test/expected/compat_informix-rfmtdate.c: 同様の Informix 互換フォーマットテスト（日付関連）。

読者向けメモ

  • 本ファイルはプリプロセッサによって生成された「期待値」ファイルです。手動で編集するものではなく、テスト実行時に生成される出力との比較に使用されます。
  • ECPGdebug(1, stderr) が設定されているため、実行時には詳細なデバッグログが標準エラー出力に出力されます。
  • rfmtlong のフォーマット指定子の挙動を理解するためのリファレンスとして活用できます。
