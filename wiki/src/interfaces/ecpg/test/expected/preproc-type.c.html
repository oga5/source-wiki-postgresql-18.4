src/interfaces/ecpg/test/expected/preproc-type.c

概要

本ファイルは、ECPG（Embedded SQL for C）プリプロセッサの型定義機能およびカスタム型マッピングの動作を検証するための回帰テスト用ソースコードです。

モジュール内での役割

src/interfaces/ecpg/test/expected ディレクトリ内のテストスイートの一部であり、プリプロセッサが EXEC SQL TYPE 文をどのように解釈し、C言語の型定義や構造体へ変換するかを確認する役割を担います。

責務

  • typedef を用いたカスタム型（mmInteger, mmChar 等）の宣言と、それらを用いた構造体定義の検証。
  • EXEC SQL TYPE 指令による、SQL型とC言語型（string, c 等）のエイリアス定義のテスト。
  • プリプロセッサが生成したコードにおける、ホスト変数とデータベース型間のデータバインディングの正当性確認。

重要なシンボル

  • TBempl: テスト用に定義された構造体。カスタム型メンバを含む。
  • string, c: EXEC SQL TYPE によって定義されたカスタム型エイリアス。
  • ECPGdo: ECPGライブラリのコア関数。SQL文の実行とホスト変数のバインドを行う。
  • sqlca: SQL通信領域。エラーハンドリングに使用。

追加の索引済みシンボル

  • access
  • access_renamed
  • main
  • mmChar
  • mmInteger
  • varchar

制御フローとデータフロー

1. **初期化**: ECPGdebug でデバッグ出力を有効化し、データベースへ接続。 2. **DDL実行**: create table を発行し、テスト用テーブルを作成。 3. **データ挿入**: insert 文でホスト変数をバインドし、データを投入。 4. **データ取得**: select 文を実行し、複数のホスト変数（構造体メンバ、配列、ポインタ、varchar型）に結果をマッピング。 5. **検証**: 取得した値を printf で出力し、期待値と一致するかを確認。 6. **終了**: ECPGdisconnect で接続を解除し、メモリを解放して終了。

直接依存

  • ecpglib.h: ECPGランタイムライブラリの関数定義。
  • ecpgerrno.h: エラーコード定義。
  • sqlca.h: SQL通信領域の構造体定義。
  • stdio.h, stdlib.h: 標準入出力およびメモリ管理。

関連コード

  • src/interfaces/ecpg/test/expected/preproc-variable.c: ホスト変数の宣言と使用に関する基本的なテスト。
  • src/interfaces/ecpg/test/expected/preproc-array_of_struct.c: 構造体配列の取り扱いに関するテスト。
  • src/interfaces/ecpg/test/expected/sql-sqlda.c: 動的SQLにおける型定義のより高度なテスト。

読者向けメモ

  • 本ファイルはプリプロセッサによって生成された「期待される出力」であるため、手動で編集するものではありません。
  • EXEC SQL TYPE は、C言語の型にSQLのセマンティクスを付与する強力な機能ですが、プリプロセッサの変換ルールを理解する上で重要なサンプルです。
  • ECPGdo 関数の引数リストは、プリプロセッサが生成した型情報とバインド変数のマッピングを直接示しており、ECPGの内部動作を理解するのに適しています。
