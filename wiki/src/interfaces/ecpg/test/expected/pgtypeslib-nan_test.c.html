src/interfaces/ecpg/test/expected/pgtypeslib-nan_test.c

概要

本ファイルは、ECPG（Embedded SQL for C）における浮動小数点数（float8）および数値型（numeric）の特殊値（NaN、infinity）の取り扱いを検証するための回帰テスト用ソースコードです。

モジュール内での役割

src/interfaces/ecpg/test/expected ディレクトリ内のテストスイートの一部であり、ECPGプリプロセッサが生成するCコードの期待値（出力結果の検証用）として機能します。

責務

  • float8 型の NaN および infinity 値のデータベースへの挿入とフェッチの検証。
  • numeric 型における NaN 値のサポート確認。
  • ECPGのカーソル操作を通じた、特殊値を含むデータの整合性チェック。

重要なシンボル

  • main: テストの実行エントリポイント。接続からテーブル作成、データ操作、切断までの一連のフローを制御します。
  • ECPGdo: ECPGが生成する、SQL文を実行するための主要なライブラリ関数。
  • numeric: pgtypes_numeric.h で定義される、PostgreSQLの数値型を扱う構造体。
  • isnan, isinf: C標準ライブラリの関数。フェッチした double 型の値が特殊値であるかを判定します。

制御フローとデータフロー

1. **初期化**: ECPGconnect でデータベースに接続し、テスト用テーブル nantest1 および nantest2 を作成します。 2. **float8 テスト**:

  • NaN や infinity を含むデータを挿入し、カーソルを用いてフェッチします。
  • isnan() や isinf() で値を判定し、結果を標準出力へ書き出します。

    • 3. **numeric テスト**:

  • PGTYPESnumeric_new() でメモリを確保し、numeric 型の NaN を扱います。
  • num->sign を参照して NUMERIC_NAN フラグを確認し、型変換の整合性を検証します。

    • 4. **終了処理**: PGTYPESnumeric_free() でメモリを解放し、トランザクションのロールバックと切断を行います。

直接依存

  • ecpglib.h, sqlca.h: ECPGライブラリの基本機能とエラーハンドリング。
  • pgtypes_numeric.h, decimal.h: numeric 型の操作用ヘッダー。
  • math.h, float.h: 浮動小数点数の特殊値判定用。
  • ecpgerrno.h
  • stdio.h
  • stdlib.h

関連コード

  • src/interfaces/ecpg/pgtypeslib/: numeric 型や decimal 型の内部実装が含まれるディレクトリ。
  • src/interfaces/ecpg/test/expected/pgtypeslib-num_test.c: numeric 型の基本的な算術演算や変換を検証する関連テスト。

読者向けメモ

  • 本ファイルは ecpg プリプロセッサによって生成されたコードの「期待値」です。直接編集するものではなく、ECPGの動作変更時に生成結果がこのファイルと一致するかを確認するために使用されます。
  • ECPGdebug(1, stderr) が有効になっており、実行時には詳細なデバッグログが標準エラー出力に出力されます。
