概要

本ファイルは、ECPG（Embedded SQL for C）における pgtypes ライブラリの数値型（numeric）操作を検証するための回帰テスト用ソースコードです。ECPGプリプロセッサによって生成されたCコードの期待値として機能します。

モジュール内での役割

src/interfaces/ecpg/test/expected ディレクトリ群の一部として、ECPGの型変換機能およびデータベースとの数値型データのやり取りが正しく動作することを保証するテストスイートの役割を担います。

責務

  • pgtypes_numeric.h で定義された数値操作関数の動作検証。
  • ECPGの EXEC SQL 文を通じたデータベースへの数値型データの挿入および取得のテスト。
  • 数値型とC言語の基本型（int, long, double）間の相互変換の検証。

重要なシンボル

  • main: テストの実行エントリポイント。一連の数値演算とDB操作を順次実行します。
  • print_double: プラットフォーム依存性を排除し、浮動小数点数の出力を正規化するためのヘルパー関数。
  • ECPGdebug: ECPGライブラリのデバッグ出力を制御するマクロ。

制御フローとデータフロー

1. **初期化**: ECPGconnect でデータベースに接続し、テスト用テーブル test を作成します。 2. **数値演算テスト**: PGTYPESnumeric_from_asc 等を用いてメモリ上で数値演算（加算、減算、乗算、除算）を行い、結果を printf で出力します。 3. **DB連携**: EXEC SQL 文を使用して、メモリ上の numeric 型変数をデータベースへ挿入し、その後 SELECT で再取得して値の整合性を確認します。 4. **型変換テスト**: PGTYPESnumeric_to_int や to_long を使用し、数値型からC言語の整数型への変換境界値（INT_MIN, INT_MAX）を検証します。 5. **終了処理**: トランザクションのロールバックとデータベース接続の切断を行います。
