src/interfaces/ecpg/test/expected/compat_informix-rfmtdate.c

概要

Informix 互換ライブラリにおける日付フォーマット変換機能（rfmtdate, rdefmtdate, rstrdate）の動作を検証するための回帰テスト用ソースコードです。

モジュール内での役割

src/interfaces/ecpg/test/expected ディレクトリ群の一部として、ECPG の Informix 互換モードにおける日付処理の正確性を保証します。プリプロセッサによって生成されるテストコードの期待値（expected）としての役割を持ちます。

責務

  • 様々な日付文字列形式から内部表現（date 型）への変換テスト。
  • 内部表現から指定フォーマットへの文字列変換テスト。
  • 異常系入力に対するエラーコード（ECPG_INFORMIX_*）の正確な返却確認。

重要なシンボル

  • rstrdate: 文字列から日付への変換。
  • rdefmtdate: 指定フォーマットに基づいた文字列から日付への変換。
  • rfmtdate: 日付から指定フォーマットの文字列への変換。
  • check_return: 変換処理で発生したエラーコードを判別し、標準出力へ出力するヘルパー関数。

追加の索引済みシンボル

  • date_test_defmt
  • date_test_fmt
  • date_test_strdate
  • main

制御フローとデータフロー

1. **初期化**: main 関数にて rmdyjul を使用し、基準となる日付を生成します。 2. **テスト実行**:

  • date_test_strdate: 文字列入力から日付変換を行い、結果を rdatestr で出力します。
  • date_test_fmt: 内部日付を様々なフォーマット文字列で変換し、結果を表示します。
  • date_test_defmt: 複雑なフォーマット指定と入力文字列の組み合わせをテストします。

    • 3. **エラーハンドリング**: 各変換関数が 0 以外の値を返した場合、check_return が呼び出され、対応するエラー定数が特定されます。

直接依存

  • ecpglib.h, ecpg_informix.h: ECPG および Informix 互換機能のコア定義。
  • sqltypes.h, pgtypes_error.h: ECPG で使用される型定義とエラーコード定義。
  • stdio.h, stdlib.h: 標準入出力およびユーティリティ。
  • ecpgerrno.h
  • sqlca.h

関連コード

  • src/interfaces/ecpg/pgtypeslib/: 日付変換ロジックの実装本体が含まれるディレクトリ。
  • src/interfaces/ecpg/test/expected/compat_informix-rfmtlong.c: 数値フォーマット変換の検証を行う関連テスト。

読者向けメモ

  • 本ファイルは ecpg プリプロセッサの回帰テスト用です。直接的なビジネスロジックではなく、ライブラリの API 仕様に対する期待値のリストとして参照してください。
  • ECPGdebug マクロが再定義されており、デバッグ出力が標準エラーに出力されるようになっています。
  • date_test_defmt に渡されるフォーマット文字列には、意図的に冗長な文字が含まれており、パーサーの堅牢性を確認する意図があります。
