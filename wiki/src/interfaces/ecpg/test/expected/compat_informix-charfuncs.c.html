src/interfaces/ecpg/test/expected/compat_informix-charfuncs.c

概要

本ファイルは、ECPG（Embedded SQL for C）における Informix 互換モードの文字列操作関数の動作を検証するための回帰テスト用ソースコードです。プリプロセッサによって生成された C コードの期待値（expected output）として機能します。

モジュール内での役割

src/interfaces/ecpg/test/expected ディレクトリ群の一部であり、ECPG の Informix 互換ライブラリが提供する文字列ユーティリティ関数が、仕様通りに動作することを保証する役割を担います。

責務

  • Informix 互換の文字列操作関数（rupshift, ldchar, byleng）のテスト。
  • 固定長文字列や空白を含む文字列に対する、期待される変換結果の出力。
  • ECPG プリプロセッサの出力結果が、Informix 互換モードで正しく生成されているかの確認。

重要なシンボル

  • rupshift(): 文字列を大文字に変換する Informix 互換関数。
  • ldchar(): 文字列から末尾の空白を除去してコピーする関数。
  • byleng(): 文字列の末尾の空白を除いた長さを計算する関数。
  • ECPGdebug: デバッグ出力を制御するマクロ（本ファイルではラップして使用）。

追加の索引済みシンボル

  • main

制御フローとデータフロー

1. **初期化**: テスト対象の文字列 t1 を定義し、標準出力へ初期状態を出力します。 2. **変換処理**: rupshift を呼び出し、文字列を大文字へ変換します。 3. **検証処理**: k の値を変化させながら ldchar と byleng を繰り返し呼び出し、結果を標準出力へ書き出します。 4. **終了**: 処理完了後、正常終了コード 0 を返します。

直接依存

  • ecpglib.h, ecpgerrno.h, sqlca.h: ECPG のコア機能およびエラーハンドリング。
  • ecpg_informix.h: Informix 互換関数のプロトタイプ定義。
  • stdio.h, stdlib.h, sqltypes.h: 標準入出力および SQL 型定義。

関連コード

  • src/interfaces/ecpg/include/ecpg_informix.h: 本ファイルで使用されている Informix 互換関数の定義元。
  • src/interfaces/ecpg/ecpglib/informix.c: 上記関数の実装が含まれるライブラリ本体。

読者向けメモ

  • 本ファイルは ecpg プリプロセッサによって自動生成されたコードの「期待値」です。手動で編集するものではなく、テスト実行時に生成されるコードとの比較対象となります。
  • ECPGdebug マクロが再定義されており、デバッグレベルを調整する工夫が見られます。
  • Informix 互換機能のテストであるため、標準的な PostgreSQL の SQL 動作とは異なる挙動（特に文字列の空白処理）に焦点を当てて読むと理解が深まります。
