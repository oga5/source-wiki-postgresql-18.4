src/interfaces/ecpg/test/expected/pgtypeslib-num_test2.c

概要

本ファイルは、ECPG（Embedded SQL for C）における pgtypes ライブラリの数値型（numeric および decimal）の変換および演算機能を検証するための回帰テスト用ソースコードです。

モジュール内での役割

src/interfaces/ecpg/test/expected ディレクトリ群の一部として、ECPGのプリプロセッサが生成するコードの期待値、あるいはその動作確認用テストスイートとして機能します。

責務

  • 文字列から numeric 型への変換（PGTYPESnumeric_from_asc）の検証。
  • numeric 型と long, int, double, decimal 型間の相互変換の検証。
  • numeric 型に対する四則演算（加減乗除）の正確性の検証。
  • エラーハンドリング（errno）の網羅的なチェック。

重要なシンボル

  • nums[]: テスト対象となる数値文字列の配列。正数、負数、指数表記、無効な値などを含む。
  • main(): テストのメインループ。変換テストと演算テストを順次実行する。
  • print_double(): プラットフォーム間で一貫した浮動小数点数の出力を保証するためのヘルパー関数。
  • check_errno(): 発生したエラーコードを判別し、人間が読める形式で出力するデバッグ用関数。

追加の索引済みシンボル

  • check_errno
  • main
  • print_double

制御フローとデータフロー

1. **初期化**: nums 配列を走査し、各文字列を numeric 型に変換して numarr に格納する。 2. **変換テスト**: 各 numeric 値に対し、to_asc（文字列変換）、to_long、to_int、to_decimal などの変換を行い、結果を標準出力へ書き出す。 3. **演算テスト**: numarr に格納された全ペアに対して、PGTYPESnumeric_add/sub/mul/div を実行し、計算結果を検証する。 4. **終了処理**: 確保したメモリを解放し、プログラムを終了する。

直接依存

  • ecpglib.h, sqlca.h: ECPGライブラリの基本機能。
  • pgtypes_numeric.h, decimal.h: 数値型操作のためのコアライブラリ。
  • pgtypes_error.h: エラーコード定義。
  • stdio.h, stdlib.h: 標準入出力およびメモリ管理。
  • ecpgerrno.h

関連コード

  • src/interfaces/ecpg/pgtypeslib/: 本テストが検証対象としている pgtypes ライブラリの実装本体。
  • src/interfaces/ecpg/test/expected/pgtypeslib-num_test.c: 同様の数値テストの別バリエーション。

読者向けメモ

  • 本ファイルは回帰テスト用であるため、出力結果は特定のフォーマットに固定されています。
  • print_double 関数は、Windows環境での指数表記の差異を吸収するためのハックを含んでおり、クロスプラットフォームでのテスト安定性を担保しています。
  • ECPGdebug(1, stderr) が有効になっており、実行時に詳細なデバッグログが標準エラー出力に出力されます。
