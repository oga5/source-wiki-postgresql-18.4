src/interfaces/ecpg/include/pgtypes_timestamp.h

概要

ECPG（Embedded C for PostgreSQL）において、タイムスタンプ型を扱うためのデータ型定義および関数プロトタイプを定義するヘッダーファイルです。

モジュール内での役割

src/interfaces/ecpg/include モジュールの一部として、ECPGアプリケーションがデータベースのタイムスタンプ型とC言語の変数間で相互運用を行うためのインターフェースを提供します。

責務

  • タイムスタンプを表現するC言語の型（timestamp, TimestampTz）の定義。
  • 文字列とタイムスタンプ間の変換、およびタイムスタンプに対する算術演算（加算・減算）を行う関数の宣言。

重要なシンボル

  • timestamp: タイムスタンプを保持する int64 型のエイリアス。
  • TimestampTz: タイムゾーン付きタイムスタンプを保持する int64 型のエイリアス。
  • PGTYPEStimestamp_from_asc / to_asc: 文字列とタイムスタンプの相互変換関数。
  • PGTYPEStimestamp_add_interval / sub_interval: タイムスタンプとインターバル型の演算関数。

制御フローとデータフロー

本ファイルは定義のみを提供するため、直接的な制御フローは持ちません。実際のデータフローは、ECPGライブラリがこれらの関数を呼び出すことで発生します。

  • **変換フロー**: 文字列入力 → PGTYPEStimestamp_from_asc → timestamp 型変数。
  • **演算フロー**: timestamp 型変数 + interval 型変数 → PGTYPEStimestamp_add_interval → 結果の timestamp 型変数。

直接依存

  • pgtypes.h: 基本的な型定義を提供。
  • pgtypes_interval.h: タイムスタンプ演算に必要な interval 型の定義を提供。

関連コード

  • src/interfaces/ecpg/pgtypeslib/timestamp.c: 本ヘッダーで宣言された関数の実装が含まれています。
  • src/interfaces/ecpg/include/pgtypes_interval.h: 演算で使用されるインターバル型の詳細が定義されています。

読者向けメモ

  • 本ファイルで定義される timestamp 型は、内部的には int64 として扱われます。
  • PGTYPEStimestamp_fmt_asc や defmt_asc を使用することで、特定のフォーマット文字列に基づいた柔軟な入出力が可能です。
  • C++から利用する場合を考慮し、extern "C" ブロックで囲まれています。
