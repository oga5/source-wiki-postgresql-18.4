src/common/unicode_category.c

概要

Unicode 文字の一般カテゴリ（General Category）および各種プロパティを判定するためのユーティリティ関数群を提供します。UTF-8 エンコーディングを前提とし、pg_wchar 型のコードポイントを引数として受け取ります。

モジュール内での役割

src/common 内において、文字列処理や文字種判定の基盤を担います。フロントエンド（クライアント）とバックエンド（サーバー）の両方から利用可能であり、ロケールに依存しない一貫した Unicode 処理を提供します。

責務

  • Unicode コードポイントに対する一般カテゴリの取得。
  • 文字のプロパティ（アルファベット、数字、空白、句読点など）の判定。
  • POSIX 互換および標準的な Unicode 互換プロパティの提供。
  • カテゴリ名や略称の文字列変換。

重要なシンボル

  • unicode_category(pg_wchar code): 指定コードポイントの一般カテゴリを返します。
  • pg_u_prop_* 系関数: alphabetic, lowercase, uppercase 等、特定の Unicode プロパティを判定します。
  • pg_u_is* 系関数: isdigit, isalpha, isspace 等、POSIX や標準に準拠した文字種判定を行います。
  • PG_U_*_MASK: カテゴリをビット演算で効率的に比較するためのマクロ群。

制御フローとデータフロー

  • **高速化**: 0x80 未満の ASCII 文字については、unicode_opt_ascii テーブルを参照して定数時間で判定します。
  • **検索**: それ以上のコードポイントに対しては、unicode_category_table.h で定義されたテーブルに対し、二分探索（range_search）を実行してプロパティを特定します。
  • **判定**: pg_u_is* 関数は、内部で unicode_category() や pg_u_prop_* を呼び出し、必要に応じてビットマスクによる論理演算を行って結果を返します。

直接依存

  • postgres.h / postgres_fe.h: 環境に応じた共通定義。
  • common/unicode_category.h: 公開インターフェースと列挙型定義。
  • common/unicode_category_table.h: 文字カテゴリとプロパティの静的データテーブル。

関連コード

  • src/common/unicode_case.c: 大文字・小文字変換に関連する処理。
  • src/common/unicode_norm.c: Unicode 正規化に関連する処理。
  • src/common/wchar.c: ワイド文字処理のユーティリティ。

読者向けメモ

  • 本ファイルは Unicode 規格の安定性ポリシーに基づき、カテゴリ値が固定されていることを前提としています。
  • posix 引数を取る関数（pg_u_isdigit など）は、標準的な Unicode 動作と POSIX 互換動作の差異を吸収するためのものです。
  • 内部テーブルは非常に大きくなる可能性があるため、修正が必要な場合は unicode_category_table.h の生成スクリプトを確認してください。
