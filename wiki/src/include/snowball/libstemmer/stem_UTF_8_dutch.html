概要

Snowball 言語で記述されたオランダ語用ステミング（語幹抽出）アルゴリズムのヘッダーファイルです。Snowball コンパイラによって自動生成されており、外部モジュールからオランダ語のステミング機能を利用するためのインターフェースを定義しています。

モジュール内での役割

libstemmer ライブラリの一部として、特定の言語（オランダ語）およびエンコーディング（UTF-8）に特化したステミング処理の公開 API を提供します。このヘッダーは、対応する実装ファイル src/backend/snowball/libstemmer/stem_UTF_8_dutch.c と対になっています。

責務

  • 語幹抽出を実行するメイン関数 dutch_UTF_8_stem の宣言。
  • C++ からの呼び出しを考慮した extern "C" による名前修飾の抑制。

重要なシンボル

  • dutch_UTF_8_create_env(): ステミング処理に必要なメモリ領域と状態を保持する SN_env 構造体を初期化します。
  • dutch_UTF_8_close_env(): SN_env が保持するリソースを解放します。
  • dutch_UTF_8_stem(): 入力された文字列に対してオランダ語のステミングアルゴリズムを適用します。

制御フローとデータフロー

本ファイルはインターフェース定義のみを行うため、具体的な制御フローは実装ファイル側に存在します。 1. 利用側は create_env で環境を作成します。 2. SN_env 内のバッファに処理対象の単語をセットします。 3. dutch_UTF_8_stem を呼び出し、アルゴリズムを実行します。 4. 処理完了後、close_env で環境を破棄します。
