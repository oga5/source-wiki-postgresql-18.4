src/include/snowball/libstemmer/stem_UTF_8_serbian.h

概要

Snowball コンパイラによって自動生成された、セルビア語向けのステミング（語幹抽出）アルゴリズムのヘッダーファイルです。UTF-8 エンコーディングされたセルビア語テキストの語幹を抽出するためのインターフェースを定義します。

モジュール内での役割

libstemmer ライブラリの一部として、特定の言語（セルビア語）に特化したステミング機能を提供します。他の言語用ヘッダーと同様に、共通の SN_env 構造体を利用してステミング処理を実行するための API を公開します。

責務

  • セルビア語ステミング処理に必要な環境の生成・破棄関数の宣言。
  • ステミング実行関数 serbian_UTF_8_stem の公開。
  • C++ との互換性を保つための extern "C" ブロックの提供。

重要なシンボル

  • serbian_UTF_8_create_env: ステミング処理に必要なメモリ領域と状態を保持する SN_env 構造体を初期化します。
  • serbian_UTF_8_close_env: create_env で確保されたリソースを解放します。
  • serbian_UTF_8_stem: 入力された SN_env 内の文字列に対して、セルビア語のステミングルールを適用します。

制御フローとデータフロー

本ファイルは宣言のみを提供します。実際の処理はペアとなる src/backend/snowball/libstemmer/stem_UTF_8_serbian.c に記述されています。 1. 呼び出し元が create_env で環境を作成。 2. SN_env 内のバッファにターゲット文字列をセット。 3. stem 関数を呼び出し、アルゴリズム（キリル文字からラテン文字への変換、接尾辞の削除など）を適用。 4. close_env で終了処理。
