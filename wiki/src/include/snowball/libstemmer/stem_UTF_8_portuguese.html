src/include/snowball/libstemmer/stem_UTF_8_portuguese.h

概要

Snowball コンパイラによって生成された、ポルトガル語向けのステミング（語幹抽出）アルゴリズムの公開インターフェースを定義するヘッダーファイルです。

モジュール内での役割

libstemmer モジュールの一部として、特定の言語（ポルトガル語）に対するステミング処理の入り口を提供します。実装の詳細を隠蔽し、外部モジュールがポルトガル語の語幹抽出機能を利用するための API を公開します。

責務

  • ポルトガル語ステマーの環境生成 (create_env) および破棄 (close_env) の宣言。
  • ステミング処理のメイン関数 (portuguese_UTF_8_stem) の宣言。
  • C++ からの呼び出しを可能にするための extern "C" ブロックの提供。

重要なシンボル

  • portuguese_UTF_8_stem: 入力された SN_env 構造体内の文字列に対して、ポルトガル語のステミング規則を適用するメイン関数。
  • portuguese_UTF_8_create_env: ステミングに必要なメモリ領域と状態を保持する SN_env 構造体を初期化・確保します。
  • portuguese_UTF_8_close_env: ステミング環境に関連付けられたリソースを解放します。

制御フローとデータフロー

本ファイルはインターフェース定義のみであるため、実際の制御フローは対応する .c ファイルに依存します。 1. 呼び出し元が create_env で環境を生成。 2. SN_env 構造体に処理対象の文字列を設定。 3. portuguese_UTF_8_stem を呼び出し、内部で定義されたルール（接尾辞の削除など）に基づき語幹を抽出。 4. 処理終了後、close_env でリソースを解放。

直接依存

  • SN_env: libstemmer の共通ヘッダー（header.h 等）で定義されているステミング環境構造体。

関連コード

  • src/backend/snowball/libstemmer/stem_UTF_8_portuguese.c: 本ヘッダーで宣言された関数の実装および、ポルトガル語特有の接尾辞テーブル（among 構造体）を含む実体ファイル。
  • src/include/snowball/libstemmer/header.h: SN_env 構造体や共通の型定義を保持する基底ヘッダー。

読者向けメモ

  • 本ファイルは Snowball コンパイラによる自動生成コードです。手動での編集は推奨されません。
  • ポルトガル語のステミング規則を変更したい場合は、Snowball 言語のソースファイルを修正し、コンパイラで再生成する必要があります。
  • UTF_8 という名称の通り、入力文字列は UTF-8 エンコーディングであることを前提としています。
