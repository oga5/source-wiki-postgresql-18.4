src/include/common/unicode_category_table.h

概要

Unicode 文字の分類およびプロパティ判定のための静的データテーブルを定義するヘッダーファイルです。Perl スクリプト（src/common/unicode/generate-unicode_category_table.pl）によって自動生成されるため、手動編集は禁止されています。

モジュール内での役割

src/include/common モジュールにおける Unicode 処理層の基盤データを提供します。unicode_norm.h や unicode_case.h といった上位の Unicode ユーティリティが、文字のカテゴリやプロパティを高速に検索するための参照テーブルとして機能します。

責務

  • **文字分類データの保持**: Unicode コードポイント範囲とカテゴリの対応付け。
  • **プロパティ定義**: 文字が持つ属性（アルファベット、大文字・小文字、空白、結合制御など）のビットフラグ定義。
  • **高速検索用テーブルの提供**: ASCII 範囲の直接参照テーブルおよび、全 Unicode 範囲をカバーする範囲検索用テーブルの提供。

重要なシンボル

データ構造

  • pg_category_range: コードポイント範囲とカテゴリ ID を保持。
  • pg_unicode_range: コードポイント範囲のみを保持（プロパティ検索用）。
  • pg_unicode_properties: カテゴリとプロパティフラグのセット。

プロパティフラグ (ビットマスク)

  • PG_U_PROP_ALPHABETIC: アルファベット文字。
  • PG_U_PROP_LOWERCASE / PG_U_PROP_UPPERCASE: 大文字・小文字。
  • PG_U_PROP_CASED: 大文字・小文字の区別がある文字。
  • PG_U_PROP_CASE_IGNORABLE: 大文字・小文字変換で無視される文字。
  • PG_U_PROP_WHITE_SPACE: 空白文字。
  • PG_U_PROP_JOIN_CONTROL: 結合制御文字。
  • PG_U_PROP_HEX_DIGIT: 16 進数文字。

静的テーブル

  • unicode_opt_ascii: ASCII 範囲（0-127）のプロパティ直接参照テーブル。
  • unicode_categories: 全 Unicode 範囲のカテゴリ検索用テーブル。
  • unicode_alphabetic, unicode_lowercase, unicode_uppercase 等: 各プロパティに対応する範囲テーブル。

制御フローとデータフロー

本ファイルはロジックを含まず、データ定義のみを提供します。 1. **ASCII 検索**: unicode_opt_ascii[codepoint] を用いて O(1) でプロパティを取得。 2. **範囲検索**: unicode_categories や各プロパティテーブル（unicode_alphabetic 等）に対して、二分探索（Binary Search）を行うことで、特定のコードポイントが範囲内に含まれるかを判定します。

直接依存

  • common/unicode_category.h: カテゴリ ID の列挙型定義 (PG_U_CONTROL 等) をインポート。

関連コード

  • src/common/unicode/generate-unicode_category_table.pl: 本ファイルを生成するソース。
  • src/include/common/unicode_norm.h: Unicode 正規化機能。
  • src/include/common/unicode_case.h: 大文字・小文字変換機能。

読者向けメモ

  • **インクルードガードの欠如**: 本ファイルは意図的に #ifndef ガードを設けていません。これは、複数のソースファイルからインクルードされることを想定した設計です。
  • **メモリレイアウト**: すべてのテーブルは static const で定義されており、プログラムのデータセグメントに配置されます。実行時のメモリ割り当ては発生しません。
  • **パフォーマンス**: ASCII 範囲外の検索は二分探索に依存するため、頻繁に呼び出される場合は検索結果のキャッシュを検討してください。
  • **拡張性**: プロパティフラグは uint8 に収まるように設計されています。プロパティの種類が増加した場合は、フラグの型を uint16 等へ拡張する必要があります。
