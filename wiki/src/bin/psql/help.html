src/bin/psql/help.h

概要

src/bin/psql/help.h は、psql クライアントのヘルプ機能および使用方法の表示に関するインターフェースを定義するヘッダーファイルです。

モジュール内での役割

psql のユーザーインターフェースの一部として、コマンドライン引数、バックスラッシュコマンド、特殊変数、および SQL 構文に関するヘルプ情報を出力する機能を提供します。

責務

  • ヘルプ表示用関数のプロトタイプ宣言。
  • ユーザーが psql の操作方法を理解するための情報提供機能の公開。
  • ページャー（出力制御）の利用を前提とした表示インターフェースの提供。

重要なシンボル

  • usage(unsigned short int pager): psql の起動オプション（コマンドライン引数）を表示します。
  • slashUsage(unsigned short int pager): psql 内で使用可能なバックスラッシュコマンドの一覧を表示します。
  • helpVariables(unsigned short int pager): psql の動作を制御する特殊変数の一覧を表示します。
  • helpSQL(const char *topic, unsigned short int pager): 指定された SQL コマンドに関するヘルプを表示します。
  • print_copyright(void): PostgreSQL の著作権情報を表示します。

制御フローとデータフロー

各関数は unsigned short int pager 引数を受け取ります。これは、出力が長い場合にページャー（less 等）を介して表示するかどうかを制御するフラグとして機能します。実際の出力処理は help.c 内で PQExpBuffer を用いて文字列を構築し、必要に応じてページャーへ渡されます。

直接依存

本ファイルは純粋なヘッダーファイルであり、他のヘッダーへの直接的な依存関係はありません。

関連コード

  • src/bin/psql/help.c: 各関数の具体的な実装が含まれています。
  • src/bin/psql/sql_help.h: helpSQL 関数が参照する SQL ヘルプデータ構造の定義が含まれています。
  • ref/psql-ref.sgml: help.c 内のヘルプテキストを更新する際は、このドキュメントも同期させる必要があります。

読者向けメモ

  • ヘルプテキストを修正する際は、必ず help.c とドキュメント（psql-ref.sgml）の両方を更新してください。
  • pager 引数は、ユーザー設定や出力の長さに応じて適切に制御されることが期待されています。
  • このモジュールは psql の対話的側面を支える重要なコンポーネントであり、ユーザー体験に直結します。
