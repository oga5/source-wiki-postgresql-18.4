src/backend/optimizer/prep/prepjointree.c

概要

本ファイルは、PostgreSQLのクエリ最適化における「ジョインツリー（JoinTree）の変形」を担う中心的なモジュールです。パーサが生成した論理的なジョインツリーを、物理プランナが効率的にパスを探索できる形式へ正規化・簡略化します。具体的には、サブクエリの引き上げ（Pull-up）、外結合の削減（Reduce Outer Joins）、不要なリレーションの除去などを行います。

モジュール内での役割

src/backend/optimizer/prep モジュールの一部として、クエリの構造を物理プランニングに適したフラットな状態に変換します。prepagg.c や prepunion.c が特定の演算ノードの最適化を行うのに対し、本ファイルはクエリ全体の結合構造を再構築する役割を持ちます。

責務

  • **サブクエリの引き上げ**: RTE_SUBQUERY を親のジョインツリーに統合し、結合の再配置を可能にする。
  • **外結合の削減**: LEFT JOIN や FULL JOIN を、制約条件に基づいて INNER JOIN や ANTI JOIN へと強度を下げ、探索空間を広げる。
  • **ジョインツリーの正規化**: RTE_RESULT の除去や、単一の子を持つ FromExpr の平坦化。
  • **MERGE文の変換**: MERGE コマンドの論理構造を、プランナが処理可能な JOIN 構造へ変換する。
  • **生成列の展開**: 仮想生成列（Virtual Generated Columns）の参照を、定義式への置換によって展開する。

重要なシンボル

変換・最適化関数

  • pull_up_subqueries: ジョインツリー内のサブクエリを親クエリへ引き上げるメインエントリ。
  • pull_up_sublinks: ANY/EXISTS サブリンクをセミジョイン/アンチセミジョインへ変換する。
  • reduce_outer_joins: 外結合を内結合へ削減する。
  • remove_useless_result_rtes: 不要な RTE_RESULT を削除し、ジョインツリーを簡略化する。
  • transform_MERGE_to_join: MERGE 文を JOIN 構造に書き換える。

コンテキスト構造体

  • pullup_replace_vars_context: サブクエリ引き上げ時に、親クエリ側の変数参照をサブクエリの出力式へ置換するためのコンテキスト。
  • reduce_outer_joins_pass1_state / pass2_state: 外結合削減の2パス処理で、ツリーの各ノードの結合状態を保持する。

制御フローとデータフロー

1. **初期化**: replace_empty_jointree により、空のジョインツリーを RTE_RESULT で埋める。 2. **サブクエリ引き上げ**: pull_up_subqueries_recurse がジョインツリーを再帰的に探索し、引き上げ可能なサブクエリを特定。perform_pullup_replace_vars を通じて、親クエリ内の変数参照をサブクエリのターゲットリストへ置換する。 3. **外結合削減**: reduce_outer_joins が2パスで動作。1パス目で結合構造と外結合の有無を収集し、2パス目で WHERE 句の制約（strictness）に基づき結合タイプを変換する。 4. **クリーンアップ**: remove_useless_results_recurse が、引き上げ後に不要となった RTE_RESULT や冗長な FromExpr を削除し、ツリーを整理する。

読者向けメモ

  • **メモリ管理**: 本モジュールは PlannerInfo を通じてメモリを管理し、再帰的なツリー操作を行うため、check_stack_depth() によるスタックオーバーフロー対策が随所に施されています。
  • **不変条件**: pull_up_subqueries 実行中、ジョインツリーの構造を維持しながら変数を置換する必要があるため、perform_pullup_replace_vars はツリー構造自体を破壊しないよう慎重に設計されています。
  • **制約**: LATERAL サブクエリの引き上げは、結合順序やスコープの制約により非常に複雑です。is_simple_subquery 内の条件分岐がその複雑性を物語っています。
  • **省略された詳細**: MERGE 文の変換ロジックは、トリガー付きビュー等の特殊ケースを扱うため非常に複雑ですが、本ファイルでは主にジョインツリーへの変換に焦点を当てています。
