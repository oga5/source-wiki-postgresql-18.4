src/backend/utils/adt/varlena.c

概要

PostgreSQL の可変長データ型（text, bytea, unknown 等）に対する基本的な操作、変換、および集約関数を実装する中心的なファイルです。文字列の連結、部分文字列抽出、検索、置換、およびバイナリデータの入出力処理を担います。

モジュール内での役割

src/backend/utils/adt モジュールにおいて、可変長データ型の「標準的な振る舞い」を定義する基盤です。他のデータ型実装（numeric.c や arrayfuncs.c など）が文字列操作を必要とする際に呼び出されるユーティリティ群を提供します。

責務

  • **データ変換**: C 文字列と text 型の相互変換、および bytea の入出力（エスケープ/HEX）。
  • **文字列操作**: 連結、部分文字列抽出、置換、検索（Boyer-Moore-Horspool アルゴリズム等）。
  • **比較・ソート**: ロケールを考慮した文字列比較および SortSupport インターフェースの提供。
  • **集約関数**: string_agg の状態遷移、シリアライズ、および最終化処理。
  • **Unicode 処理**: 正規化、コードポイント変換、エスケープシーケンスのデコード。

重要なシンボル

データ構造

  • TextPositionState: text_position 関連関数の検索状態を保持。BMH アルゴリズム用のスキップテーブルを含む。
  • VarStringSortSupport: ソート時の比較キャッシュや HyperLogLog 状態を保持する構造体。
  • SplitTextOutputData: split_text の結果を配列またはテーブルへ出力するためのコンテキスト。

型変換・I/O

  • cstring_to_text, text_to_cstring: C 文字列と text 型の変換。
  • byteain, byteaout: bytea 型の入出力（HEX/エスケープ形式）。
  • textin, textout: text 型の入出力。

文字列操作・検索

  • text_substring, bytea_substring: 部分文字列の抽出。
  • text_position: 文字列内のパターン検索。
  • replace_text, replace_text_regexp: 文字列の置換。

ソートサポート

  • varstr_sortsupport: ソート最適化のための初期化関数。
  • varstrfastcmp_c, varlenafastcmp_locale: ロケールに応じた高速比較関数。

制御フローとデータフロー

  • **検索フロー**: text_position は text_position_setup でスキップテーブルを構築し、text_position_next が内部の text_position_next_internal を呼び出してマッチングを行います。
  • **ソートフロー**: varstr_sortsupport が SortSupport オブジェクトを初期化し、比較時に varlenafastcmp_locale 等が呼ばれます。Abbreviated Key（省略キー）が有効な場合、varstr_abbrev_convert で変換が行われ、varstr_abbrev_abort で統計に基づき最適化の継続可否が判断されます。
  • **集約フロー**: string_agg は string_agg_transfn で StringInfo にデータを蓄積し、string_agg_finalfn で最終的な text 型に変換します。

読者向けメモ

  • **メモリ管理**: 本ファイル内の関数は palloc を多用します。特にソートサポートや集約関数では、MemoryContext の切り替えに注意が必要です。
  • **TOAST**: DatumGetVarStringPP 等のマクロは、TOAST されたデータを必要に応じて展開（detoast）します。パフォーマンス向上のため、可能な限り detoast を避ける実装（toast_raw_datum_size の利用など）が随所に見られます。
  • **ロケール**: 文字列比較は pg_locale_t を介して行われます。C ロケールの場合、memcmp を用いた高速パスが選択されます。
  • **省略**: levenshtein.c の詳細なアルゴリズム実装は、本ファイルからインクルードされる別ファイルであるため、本ファイル内では展開されていません。
