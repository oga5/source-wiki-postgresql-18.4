src/backend/utils/mb/wstrncmp.c

概要

本ファイルは、PostgreSQLにおけるワイド文字（pg_wchar）配列を扱うための文字列比較および長さ計算ユーティリティを提供します。フロントエンドとバックエンドの両方から利用可能です。

モジュール内での役割

src/backend/utils/mb モジュールの一部として、マルチバイト文字を内部表現であるワイド文字に変換した後の、メモリ上の比較操作を補助します。標準の strncmp や strlen に相当する機能を、pg_wchar 型に対して安全に実行する役割を担います。

責務

  • pg_wchar 配列同士の指定長比較。
  • char 配列（マルチバイト文字列のバイト表現）と pg_wchar 配列の比較。
  • pg_wchar 配列の終端（ヌル文字）までの長さ計算。

重要なシンボル

  • pg_wchar_strncmp: 2つの pg_wchar 配列を最大 n 文字まで比較します。
  • pg_char_and_wchar_strncmp: char 型の文字列と pg_wchar 型の文字列を比較します。型変換を伴う比較が必要な場面で使用されます。
  • pg_wchar_strlen: pg_wchar 配列の長さを返します。

制御フローとデータフロー

  • 各関数は n（比較上限）が 0 の場合に即座に 0 を返します。
  • ループ内では、ポインタをインクリメントしながら各要素を比較し、不一致が発生した時点で差分を返します。
  • ヌル文字（0）に到達した場合は、比較の途中であってもループを終了し、一致とみなして 0 を返します。
  • pg_char_and_wchar_strncmp では、char 型を unsigned char にキャストした上で pg_wchar に昇格させ、比較を行っています。

直接依存

  • postgres_fe.h: フロントエンド/バックエンド共通の基本定義。
  • mb/pg_wchar.h: pg_wchar 型の定義および関連するマクロ。

関連コード

  • src/backend/utils/mb/wstrcmp.c: wstrncmp.c と対になるファイルで、こちらは長さ制限のない strcmp 相当の機能を提供します。
  • src/backend/utils/mb/mbutils.c: 文字コード変換やマルチバイト処理のメインロジックが含まれており、本ファイルの関数が呼び出される文脈を提供します。

読者向けメモ

  • 本ファイルの実装は標準的な C ライブラリの strncmp 等の挙動を模倣していますが、対象が pg_wchar である点に注意してください。
  • pg_char_and_wchar_strncmp は、型が異なる配列を直接比較する特殊なケースでのみ使用されます。一般的な文字列比較には pg_wchar_strncmp を使用してください。
  • このコードは非常に軽量であり、ロジックは標準的な文字列操作アルゴリズムに基づいています。複雑な依存関係はないため、単体での理解が容易です。
