src/backend/utils/activity/backend_progress.c

概要

本ファイルは、PostgreSQLにおける長時間実行コマンド（VACUUM, CREATE INDEX等）の進捗状況を追跡・公開するためのインフラストラクチャを提供します。各バックエンドプロセスが自身の進捗パラメータを共有メモリ上の PgBackendStatus に書き込むためのインターフェースを定義しています。

モジュール内での役割

src/backend/utils/activity モジュールの一部として、バックエンドの状態管理（backend_status.c）と統計情報収集（pgstat.c）の橋渡しをします。本ファイルは「進捗」という特定のコンテキストに特化した書き込み操作を担当します。

責務

  • **進捗状態のライフサイクル管理**: コマンドの開始（start）から終了（end）までの状態遷移を管理します。
  • **パラメータの更新**: 進捗状況を示す最大20個の int64 型パラメータ（st_progress_param）の更新、加算、一括更新を提供します。
  • **並列処理のサポート**: パラレルワーカーがリーダープロセスに対して進捗更新を通知する仕組みを提供します。

重要なシンボル

  • pgstat_progress_start_command: コマンド開始時に呼び出し、進捗パラメータを初期化します。
  • pgstat_progress_update_param: 特定の進捗パラメータを更新します。
  • pgstat_progress_parallel_incr_param: パラレルワーカーがリーダー経由で進捗を更新するためのメッセージング処理を行います。
  • pgstat_progress_update_multi_param: 複数のパラメータをアトミックに更新します。
  • pgstat_progress_end_command: コマンド終了時に進捗状態をリセットします。

追加の索引済みシンボル

  • pgstat_progress_incr_param

制御フローとデータフロー

1. **書き込み**: 各関数は MyBEEntry（現在のバックエンドの共有メモリ状態）を取得し、PGSTAT_BEGIN_WRITE_ACTIVITY マクロでロック（またはメモリバリア）を確保します。 2. **更新**: 共有メモリ上の st_progress_param 配列を直接操作します。 3. **並列処理**: pgstat_progress_parallel_incr_param は、自身がワーカーであれば PqMsg_Progress メッセージを送信し、リーダーであれば直接更新を行います。 4. **終了**: PGSTAT_END_WRITE_ACTIVITY マクロで書き込みを完了させます。

直接依存

  • utils/backend_status.h: 共有メモリ上の PgBackendStatus 構造体定義および MyBEEntry へのアクセス。
  • libpq/pqformat.h: パラレルワーカーからの進捗通知メッセージング用。
  • access/parallel.h: IsParallelWorker() チェック用。
  • postgres.h
  • utils/backend_progress.h

関連コード

  • src/include/utils/backend_progress.h: 本ファイルのヘッダー。ProgressCommandType 列挙型が定義されています。
  • src/include/commands/progress.h: 各コマンド（VACUUM等）がどのインデックスに何を格納するかという「意味論」が定義されています。
  • src/backend/utils/activity/backend_status.c: PgBackendStatus の管理基盤。

読者向けメモ

  • **意味論の分離**: 本ファイルは進捗パラメータの「格納場所」を提供するだけで、各インデックス（0〜19）が何を意味するかは関知しません。意味論は各コマンドの実装側に依存します。
  • **アトミック性**: pgstat_progress_update_multi_param を使用すると、複数のパラメータを一度のロック期間で更新できるため、読み取り側（ビューなど）から見て不整合な状態が見えることを防げます。
  • **パフォーマンス**: pgstat_track_activities が無効な場合、これらの関数は即座にリターンするため、オーバーヘッドは最小限に抑えられています。
