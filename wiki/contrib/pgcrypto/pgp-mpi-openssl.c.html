contrib/pgcrypto/pgp-mpi-openssl.c

概要

OpenPGP の多倍長整数（MPI: Multi-Precision Integers）形式と、OpenSSL の BIGNUM ライブラリ間の変換および演算を行うブリッジモジュールです。

モジュール内での役割

pgcrypto モジュールにおいて、公開鍵暗号（RSA および ElGamal）の数学的処理を OpenSSL に委譲するためのアダプターとして機能します。

責務

  • **データ変換**: PGP_MPI 構造体と OpenSSL の BIGNUM 間の相互変換。
  • **暗号演算の実行**: OpenSSL の BN_mod_exp 等を利用した、RSA および ElGamal 暗号の暗号化・復号処理。
  • **パラメータ管理**: ElGamal 暗号化に必要な乱数成分 $k$ のビット長決定。

重要なシンボル

  • mpi_to_bn / bn_to_mpi: 内部表現と OpenSSL 表現を変換する静的ヘルパー関数。
  • pgp_elgamal_encrypt / pgp_elgamal_decrypt: ElGamal 暗号の数学的処理。
  • pgp_rsa_encrypt / pgp_rsa_decrypt: RSA 暗号の数学的処理。
  • decide_k_bits: ElGamal 暗号化時の乱数 $k$ のビット長を計算するヒューリスティック関数。

制御フローとデータフロー

1. **入力変換**: 公開鍵やメッセージ（PGP_MPI）を mpi_to_bn で BIGNUM に変換します。 2. **演算実行**: OpenSSL の BN_CTX を確保し、モジュロ演算（BN_mod_exp 等）を実行します。 3. **出力変換**: 演算結果を bn_to_mpi で PGP_MPI に戻し、呼び出し元へ返します。 4. **リソース解放**: goto err パターンを用いて、関数終了時に確保したすべての BIGNUM および BN_CTX を確実に解放します。

直接依存

  • postgres.h: PostgreSQL の基本定義。
  • <openssl/bn.h>: OpenSSL の多倍長整数演算ライブラリ。
  • pgp.h: PGP 関連のデータ構造（PGP_MPI, PGP_PubKey 等）。
  • px.h: pgcrypto 共通のユーティリティおよびエラーコード。
  • openssl/bn.h

関連コード

  • pgp-mpi.c: MPI のメモリ管理（pgp_mpi_alloc, pgp_mpi_free）を定義。
  • pgp-pubenc.c / pgp-pubdec.c: 本ファイルの関数を呼び出し、公開鍵暗号処理の全体フローを制御する上位レイヤー。

読者向けメモ

  • **エラー処理**: 本ファイル内の関数は、OpenSSL の演算失敗やメモリ不足時に PXE_PGP_MATH_FAILED を返します。
  • **メモリ管理**: BIGNUM は動的確保されるため、エラー発生時を含め、すべてのポインタを BN_clear_free で解放する厳格なクリーンアップ処理が記述されています。
  • **ElGamal の乱数**: decide_k_bits は GnuPG の挙動を模倣しており、セキュリティとパフォーマンスのバランスを考慮した経験的なビット長計算を行っています。
