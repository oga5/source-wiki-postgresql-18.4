contrib/pgcrypto/px-crypt.c

概要

px-crypt.c は、PostgreSQL の pgcrypto モジュールにおいて、パスワードハッシュ化およびソルト生成のための抽象化レイヤーを提供するソースファイルです。

モジュール内での役割

pgcrypto がサポートする複数の暗号アルゴリズム（DES, MD5, Blowfish, SHA-256/512）を統一的なインターフェースで呼び出すためのディスパッチャーとして機能します。

責務

  • **アルゴリズムのディスパッチ**: ソルト文字列のプレフィックスに基づき、適切なハッシュ関数を選択して実行する。
  • **ソルト生成の管理**: 指定されたアルゴリズムに応じた適切なソルト生成関数を呼び出し、ラウンド数や乱数生成のバリデーションを行う。
  • **抽象化**: 個別のアルゴリズム実装（crypt-*.c）を隠蔽し、上位層に対して一貫した API を提供する。

重要なシンボル

  • px_crypt(): パスワードとソルトを受け取り、ハッシュ化された文字列を返すメイン関数。
  • px_gen_salt(): 指定されたアルゴリズム名に基づき、ランダムなソルトを生成する関数。
  • px_crypt_list: ソルトのプレフィックスと実行関数を紐付ける静的テーブル。
  • gen_list: ソルト生成アルゴリズムのメタデータ（ラウンド数制限や生成関数）を保持するテーブル。

制御フローとデータフロー

1. **px_crypt**:

  • px_crypt_list を走査し、入力された salt の先頭文字列（例: $1$）と一致するアルゴリズムを特定する。
  • 特定された関数（run_crypt_*）を呼び出し、結果をバッファに格納する。

    • 2. **px_gen_salt**:

  • gen_list からアルゴリズムを検索し、ラウンド数の妥当性をチェックする。
  • pg_strong_random で安全な乱数を生成し、対応する _crypt_gensalt_*_rn 関数でソルト文字列を構築する。

直接依存

  • postgres.h: PostgreSQL の基本型およびマクロ定義。
  • px-crypt.h: 公開インターフェースおよび定数定義。
  • px.h: pgcrypto 共通のユーティリティ関数（CheckBuiltinCryptoMode 等）。

関連コード

  • crypt-blowfish.c, crypt-des.c, crypt-md5.c, crypt-sha.c: 各ハッシュアルゴリズムの具体的な実装。
  • crypt-gensalt.c: ソルト生成の低レベルな実装。

読者向けメモ

  • 本ファイルは単なるラッパーであり、実際の暗号計算ロジックは含まれていません。アルゴリズムの詳細を追う場合は、crypt-*.c を参照してください。
  • px_crypt_list の順序は重要です。プレフィックスが短いもの（空文字など）が後に来るように配置されています。
  • CheckBuiltinCryptoMode() は、モジュールが安全なモードで動作しているかを確認するガード節として機能しています。
