contrib/bool_plperl/bool_plperl.c

概要

PostgreSQLの boolean 型と Perl のスカラ値（SV）間の型変換を仲介する拡張モジュールです。PL/Perl を使用した関数定義において、ブール値の受け渡しを可能にします。

モジュール内での役割

contrib/bool_plperl モジュールの唯一のソースファイルであり、PostgreSQL の型システムと Perl インタプリタのデータ表現を橋渡しするアダプターとして機能します。

責務

  • **PostgreSQL から Perl への変換**: bool 型を Perl の真偽値定数（&PL_sv_yes / &PL_sv_no）へ変換する。
  • **Perl から PostgreSQL への変換**: Perl のスカラ値（SV）を SvTRUE マクロで評価し、PostgreSQL の bool 型へ変換する。

重要なシンボル

  • bool_to_plperl: PostgreSQL の bool を Perl の真偽値オブジェクトに変換する関数。
  • plperl_to_bool: Perl の SV を PostgreSQL の bool に変換する関数。
  • dTHX: Perl インタプリタのコンテキスト（Thread Context）を取得するマクロ。

制御フローとデータフロー

1. **bool_to_plperl**:

  • PG_GETARG_BOOL で PostgreSQL 側の引数を取得。
  • 三項演算子により、値に応じて Perl のグローバル変数 PL_sv_yes または PL_sv_no へのポインタを選択。
  • PointerGetDatum で PostgreSQL に戻す。

    • 2. **plperl_to_bool**:

  • PG_GETARG_POINTER で Perl の SV* を取得。
  • SvTRUE(in) を呼び出し、Perl の真偽値判定ルールに従って bool に変換。
  • PG_RETURN_BOOL で結果を返す。

直接依存

  • postgres.h: PostgreSQL のコア定義。
  • fmgr.h: 関数マネージャ（PG_FUNCTION_ARGS 等）。
  • plperl.h: Perl インタプリタとの連携用ヘッダー。

関連コード

  • src/pl/plperl: PL/Perl 言語ハンドラの本体。本ファイルは、このハンドラが型変換を行う際に呼び出されることを想定しています。

読者向けメモ

  • 本ファイルは非常にシンプルですが、Perl の dTHX マクロが必須である点に注意してください。これはマルチスレッド環境下で正しい Perl インタプリタコンテキストを特定するために必要です。
  • SvTRUE は Perl の真偽値判定ルール（undef や空文字列、数値の 0 が偽となる等）に従うため、PostgreSQL 側の bool 変換時もこのルールが適用される点に留意してください。
